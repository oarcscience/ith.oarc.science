| <span id="an_050_009">9</span> ‏وَنَزَّلْنَا مِنَ ٱلسَّمَآءِ مَآءًۭ مُّبَرَكًۭا فَأَنۢبَتْنَا بِهِۦ جَنَّتٍۢ وَحَبَّ ٱلْحَصِيدِ ‎                            | wanaz:alna' mina 'ls:ama'ʔʔi ma'ʔʔanm m:ubarakanm' fa'ʔanmbatna' bihiy jan:atinm waḥab:a 'lḥaṣiydi                     |
| <span id="an_050_010">10</span> ‏وَٱلنَّخْلَ بَاسِقَتٍۢ لَّهَا طَلْعٌۭ نَّضِيدٌۭ ‎                                                                             | wa'ln:axla ba'siqatinm l:aha' ṭalʕunm n:aḍiydunm                                                                       |
| <span id="an_050_011">11</span> ‏رِّزْقًۭا لِّلْعِبَادِ ۖ وَأَحْيَيْنَا بِهِۦ بَلْدَةًۭ مَّيْتًۭا ۚ كَذَلِكَ ٱلْخُرُوجُ ‎                                      | r:izqanm' l:ilʕiba'di sl' wa'ʔaḥyayna' bihiy baldatanm m:aytanm' j kaðalika 'lxuruwju                                  |
| <span id="an_050_012">12</span> ‏كَذَّبَتْ قَبْلَهُمْ قَوْمُ نُوحٍۢ وَأَصْحَبُ ٱلرَّسِّ وَثَمُودُ ‎                                                            | kað:abat qablahum qawmu nuwḥinm wa'ʔaṣḥabu 'lr:as:i waθamuwdu                                                          |
| <span id="an_050_013">13</span> ‏وَعَادٌۭ وَفِرْعَوْنُ وَإِخْوَنُ لُوطٍۢ ‎                                                                                     | waʕa'dunm wafirʕawnu wa'ʔixwanu luwṭinm                                                                                |
| <span id="an_050_014">14</span> ‏وَأَصْحَبُ ٱلْأَيْكَةِ وَقَوْمُ تُبَّعٍۢ ۚ كُلٌّۭ كَذَّبَ ٱلرُّسُلَ فَحَقَّ وَعِيدِ ‎                                         | wa'ʔaṣḥabu 'l'ʔaykati waqawmu tub:aʕinm j kul:unm kað:aba 'lr:usula faḥaq:a waʕiydi                                    |
| <span id="an_050_015">15</span> ‏أَفَعَيِينَا بِٱلْخَلْقِ ٱلْأَوَّلِ ۚ بَلْ هُمْ فِى لَبْسٍۢ مِّنْ خَلْقٍۢ جَدِيدٍۢ ‎                                          | 'ʔafaʕayiyna' bi'lxalqi 'l'ʔaw:ali j bal hum fiy labsinm m:in xalqinm jadiydinm                                        |
| <span id="an_050_016">16</span> ‏وَلَقَدْ خَلَقْنَا ٱلْإِنسَنَ وَنَعْلَمُ مَا تُوَسْوِسُ بِهِۦ نَفْسُهُۥ ۖ وَنَحْنُ أَقْرَبُ إِلَيْهِ مِنْ حَبْلِ ٱلْوَرِيدِ ‎ | walaqad xalaqna' 'l'ʔinsana wanaʕlamu ma' tuwaswisu bihiy nafsuhuw sl' wanaḥnu 'ʔaqrabu 'ʔilayhi min ḥabli 'lwariydi   |
| <span id="an_050_017">17</span> ‏إِذْ يَتَلَقَّى ٱلْمُتَلَقِّيَانِ عَنِ ٱلْيَمِينِ وَعَنِ ٱلشِّمَالِ قَعِيدٌۭ ‎                                                | 'ʔið yatalaq:ay 'lmutalaq:iya'ni ʕani 'lyamiyni waʕani 'lʃ:ima'li qaʕiydunm                                            |
| <span id="an_050_018">18</span> ‏مَّا يَلْفِظُ مِن قَوْلٍ إِلَّا لَدَيْهِ رَقِيبٌ عَتِيدٌۭ ‎                                                                   | m:a' yalfiẓu min qawlin 'ʔil:a' ladayhi raqiybun ʕatiydunm                                                             |
| <span id="an_050_019">19</span> ‏وَجَآءَتْ سَكْرَةُ ٱلْمَوْتِ بِٱلْحَقِّ ۖ ذَلِكَ مَا كُنتَ مِنْهُ تَحِيدُ ‎                                                  | waja'ʔʔat sakratu 'lmawti bi'lḥaq:i sl' ðalika ma' kunta minhu taḥiydu                                                 |
| <span id="an_050_020">20</span> ‏وَنُفِخَ فِى ٱلصُّورِ ۚ ذَلِكَ يَوْمُ ٱلْوَعِيدِ ‎                                                                            | wanufixa fiy 'lṣ:uwri j ðalika yawmu 'lwaʕiydi                                                                         |
| <span id="an_050_021">21</span> ‏وَجَآءَتْ كُلُّ نَفْسٍۢ مَّعَهَا سَآئِقٌۭ وَشَهِيدٌۭ ‎                                                                      | waja'ʔʔat kul:u nafsinm m:aʕaha' sa'ʔyʔiqunm waʃahiydunm                                                               |
| <span id="an_050_022">22</span> ‏لَّقَدْ كُنتَ فِى غَفْلَةٍۢ مِّنْ هَذَا فَكَشَفْنَا عَنكَ غِطَآءَكَ فَبَصَرُكَ ٱلْيَوْمَ حَدِيدٌۭ ‎                          | l:aqad kunta fiy ɣaflatinm m:in haða' fakaʃafna' ʕanka ɣiṭa'ʔʔaka fabaṣaruka 'lyawma ḥadiydunm                         |
| <span id="an_050_023">23</span> ‏وَقَالَ قَرِينُهُۥ هَذَا مَا لَدَىَّ عَتِيدٌ ‎                                                                                | waqa'la qariynuhuw haða' ma' laday:a ʕatiydun                                                                          |
| <span id="an_050_024">24</span> ‏أَلْقِيَا فِى جَهَنَّمَ كُلَّ كَفَّارٍ عَنِيدٍۢ ‎                                                                             | 'ʔalqiya' fiy jahan:ama kul:a kaf:a'rin ʕaniydinm                                                                      |
| <span id="an_050_025">25</span> ‏مَّنَّاعٍۢ لِّلْخَيْرِ مُعْتَدٍۢ مُّرِيبٍ ‎                                                                                   | m:an:a'ʕinm l:ilxayri muʕtadinm m:uriybin                                                                              |
| <span id="an_050_026">26</span> ‏ٱلَّذِى جَعَلَ مَعَ ٱللَّهِ إِلَهًا ءَاخَرَ فَأَلْقِيَاهُ فِى ٱلْعَذَابِ ٱلشَّدِيدِ ‎                                         | 'l:aðiy jaʕala maʕa 'll:ahi 'ʔilahan' ʔa'xara fa'ʔalqiya'hu fiy 'lʕaða'bi 'lʃ:adiydi                                   |
| <span id="an_050_027">27</span> ‏ قَالَ قَرِينُهُۥ رَبَّنَا مَآ أَطْغَيْتُهُۥ وَلَكِن كَانَ فِى ضَلَلٍۭ بَعِيدٍۢ ‎                                            | qa'la qariynuhuw rab:ana' ma'ʔ 'ʔaṭɣaytuhuw walakin ka'na fiy ḍalalinm baʕiydinm                                       |
| <span id="an_050_028">28</span> ‏قَالَ لَا تَخْتَصِمُوا۟ لَدَىَّ وَقَدْ قَدَّمْتُ إِلَيْكُم بِٱلْوَعِيدِ ‎                                                     | qa'la la' taxtaṣimuw' laday:a waqad qad:amtu 'ʔilaykum bi'lwaʕiydi                                                     |
| <span id="an_050_029">29</span> ‏مَا يُبَدَّلُ ٱلْقَوْلُ لَدَىَّ وَمَآ أَنَا۠ بِظَلَّمٍۢ لِّلْعَبِيدِ ‎                                                       | ma' yubad:alu 'lqawlu laday:a wama'ʔ 'ʔana' biẓal:aminm l:ilʕabiydi                                                    |
| <span id="an_050_030">30</span> ‏يَوْمَ نَقُولُ لِجَهَنَّمَ هَلِ ٱمْتَلَأْتِ وَتَقُولُ هَلْ مِن مَّزِيدٍۢ ‎                                                    | yawma naquwlu lijahan:ama hali 'mtala'ʔti wataquwlu hal min m:aziydinm                                                 |
| <span id="an_050_031">31</span> ‏وَأُزْلِفَتِ ٱلْجَنَّةُ لِلْمُتَّقِينَ غَيْرَ بَعِيدٍ ‎                                                                       | wa'ʔuzlifati 'ljan:atu lilmut:aqiyna ɣayra baʕiydin                                                                    |
| <span id="an_050_032">32</span> ‏هَذَا مَا تُوعَدُونَ لِكُلِّ أَوَّابٍ حَفِيظٍۢ ‎                                                                              | haða' ma' tuwʕaduwna likul:i 'ʔaw:a'bin ḥafiyẓinm                                                                      |
| <span id="an_050_033">33</span> ‏مَّنْ خَشِىَ ٱلرَّحْمَنَ بِٱلْغَيْبِ وَجَآءَ بِقَلْبٍۢ مُّنِيبٍ ‎                                                            | m:an xaʃiya 'lr:aḥmana bi'lɣaybi waja'ʔʔa biqalbinm m:uniybin                                                          |

| <span id="an_050_040">40</span> ‏وَمِنَ ٱلَّيْلِ فَسَبِّحْهُ وَأَدْبَرَ ٱلسُّجُودِ ‎                                                                           | wamina 'l:ayli fasab:iḥhu wa'ʔadbara 'ls:ujuwdi                                                                        |
