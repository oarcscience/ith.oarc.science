#### SONG OF THE EARTH

**NAVAJO**

      

<div style="margin-left: 32px">

All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

Now the Mother Earth  
And the Father Sky,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

Sisnajinni,  
Tsodsichl,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

Now Doko-oslid  
And Depenitsa,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

And the night of darkness  
And the dawn of light,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,

</div>

<span id="page_119">p. 119</span>

      

<div style="margin-left: 32px">

All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

Now Hastyeyalli  
And Hastyehogan,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

And the white corn  
And the yellow corn,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

And the corn-pollen  
And the Ripener,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

<div style="margin-left: 32px">

Life-that-never-passeth,  
Happiness-of-all-things,  
    Meeting, joining one another,  
    Helpmates ever, they.  
      All is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>

      

<div style="margin-left: 32px">

Now all is beautiful,  
      All is beautiful,  
      All is beautiful, indeed.

</div>
