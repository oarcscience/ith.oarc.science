Production Notes

This section is based on texts transcribed by the Christian Classics Ethereal Library. [External Site]

These files were generated from XML source files at the CCEL, using procedures and programs developed at Intangible Textual Heritage. Some of the files in each text are blank; however, no actual text is missing, these are simply places where a section break occurred.

Volumes 4, 5 and 6 in the Ante-Nicene Fathers were not yet available in XML source form when this section was developed, so these have some formatting problems: notably, missing greek text. We will rebuild these files from XML once CCEL has completed them, at which time their file breakdown may change.
