This is the Ralph T.H. Griffith English translation of the Rig Veda. This was one of the first etexts developed for this site. Each page of this is cross-linked with the Sanskrit text of the Rig Veda. Both this and the Sanskrit Rig Veda require browser support for Unicode. If you have trouble seeing non-Latin characters in this text, refer to the Intangible Textual Heritage Unicode page.
