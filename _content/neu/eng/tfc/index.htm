Tales from Chaucer

by Charles Cowden Clarke

[1833]
Contents    Start Reading    Page Index    Text [Zipped]

This is a modern retelling of the highlights of Chaucer's Canterbury Tales. While it is written for a younger audience, readers of any age can enjoy these timeless stories, illustrated by witty fairy-tale style color drawings by Arthur Szyk.
