### Aramaic Bible: 2 Corinthians

<span id="an_1:1">1:1</span> ܦܘܠܘܤ ܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܒܨܒܝܢܗ ܕܐܠܗܐ
ܘܛܝܡܬܐܘܤ ܐܚܐ ܠܥܕܬܐ ܕܐܠܗܐ ܕܐܝܬ ܒܩܘܪܢܬܘܤ ܘܠܟܠܗܘܢ ܩܕܝܫܐ ܕܐܝܬ ܒܐܟܐܝܐ ܟܠܗ ܀

<span id="an_1:2">1:2</span> ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ
ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:3">1:3</span> ܡܒܪܟ ܗܘ ܐܠܗܐ ܐܒܘܗܝ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܒܐ
ܕܪܚܡܐ ܘܐܠܗܐ ܕܟܠ ܒܘܝܐ ܀

<span id="an_1:4">1:4</span> ܗܘ ܕܡܒܝܐ ܠܢ ܒܟܠܗܘܢ ܐܘܠܨܢܝܢ ܕܐܦ ܚܢܢ ܢܫܟܚ
ܢܒܝܐ ܠܐܝܠܝܢ ܕܒܟܠ ܐܘܠܨܢܝܢ ܐܢܘܢ ܒܗܘ ܒܘܝܐܐ ܕܚܢܢ ܡܬܒܝܐܝܢܢ ܡܢ ܐܠܗܐ ܀

<span id="an_1:5">1:5</span> ܐܝܟܢܐ ܓܝܪ ܕܡܬܝܬܪܝܢ ܒܢ ܚܫܘܗܝ ܕܡܫܝܚܐ ܗܟܢܐ ܒܝܕ
ܡܫܝܚܐ ܡܬܝܬܪ ܐܦ ܒܘܝܐܢ ܀

<span id="an_1:6">1:6</span> ܐܦܢ ܕܝܢ ܡܬܐܠܨܝܢܢ ܥܠ ܐܦܝ ܒܘܝܐܟܘܢ ܗܘ ܘܥܠ ܐܦܝ
ܚܝܝܟܘܢ ܡܬܐܠܨܝܢܢ ܘܐܢ ܡܬܒܝܐܝܢܢ ܡܛܠ ܕܐܢܬܘܢ ܬܬܒܝܐܘܢ ܘܬܗܘܐ ܒܟܘܢ ܚܦܝܛܘܬܐ
ܕܬܤܝܒܪܘܢ ܐܢܘܢ ܠܚܫܐ ܗܢܘܢ ܕܐܦ ܚܢܢ ܚܫܝܢܢ ܠܗܘܢ ܀

<span id="an_1:7">1:7</span> ܘܤܒܪܢ ܕܥܠܝܟܘܢ ܫܪܝܪ ܗܘ ܝܕܥܝܢܢ ܓܝܪ ܕܐܢ ܫܘܬܦܝܢ
ܐܢܬܘܢ ܒܚܫܐ ܫܘܬܦܝܢ ܐܢܬܘܢ ܐܦ ܒܒܘܝܐܐ ܀

<span id="an_1:8">1:8</span> ܨܒܝܢܢ ܕܝܢ ܕܬܕܥܘܢ ܐܚܝܢ ܥܠ ܐܘܠܨܢܐ ܕܗܘܐ ܠܢ
ܒܐܤܝܐ ܕܪܘܪܒܐܝܬ ܐܬܐܠܨܢ ܝܬܝܪ ܡܢ ܚܝܠܢ ܥܕܡܐ ܕܩܪܝܒܝܢ ܗܘܘ ܚܝܝܢ ܠܡܬܛܠܩܘ ܀

<span id="an_1:9">1:9</span> ܘܥܠ ܢܦܫܢ ܦܤܩܢ ܡܘܬܐ ܕܠܐ ܢܗܘܐ ܠܢ ܬܘܟܠܢܐ ܥܠ
ܢܦܫܢ ܐܠܐ ܥܠ ܐܠܗܐ ܗܘ ܕܡܩܝܡ ܡܝܬܐ ܀

<span id="an_1:10">1:10</span> ܗܘ ܕܡܢ ܡܘܬܐ ܚܤܝܢܐ ܦܪܩܢ ܘܬܘܒ ܡܤܒܪܝܢܢ ܕܦܪܩ
ܠܢ ܀

<span id="an_1:11">1:11</span> ܒܡܥܕܪܢܘܬܐ ܕܒܥܘܬܟܘܢ ܕܥܠ ܐܦܝܢ ܕܬܗܘܐ ܡܘܗܒܬܗ
ܕܠܘܬܢ ܛܝܒܘܬܐ ܕܥܒܝܕܐ ܒܐܦܝ ܤܓܝܐܐ ܘܤܓܝܐܐ ܢܘܕܘܢ ܠܗ ܥܠ ܐܦܝܢ ܀

<span id="an_1:12">1:12</span> ܫܘܒܗܪܢ ܓܝܪ ܗܢܘ ܤܗܕܘܬܐ ܕܪܥܝܢܢ ܕܒܦܫܝܛܘܬܐ
ܘܒܕܟܝܘܬܐ ܘܒܛܝܒܘܬܐ ܕܐܠܗܐ ܐܬܗܦܟܢ ܒܥܠܡܐ ܘܠܐ ܒܚܟܡܬܐ ܕܦܓܪܐ ܘܝܬܝܪܐܝܬ ܠܘܬܟܘܢ
ܕܝܠܟܘܢ ܀

<span id="an_1:13">1:13</span> ܠܐ ܗܘܐ ܐܚܪܢܝܢ ܟܬܒܝܢܢ ܠܟܘܢ ܐܠܐ ܐܝܠܝܢ
ܕܝܕܥܝܢ ܐܢܬܘܢ ܐܦ ܡܫܬܘܕܥܝܢ ܐܢܬܘܢ ܬܟܝܠ ܐܢܐ ܕܝܢ ܕܥܕܡܐ ܠܐܚܪܝܬܐ ܬܫܬܘܕܥܘܢ ܀

<span id="an_1:14">1:14</span> ܐܝܟܢܐ ܕܐܦ ܐܫܬܘܕܥܬܘܢ ܩܠܝܠ ܡܢ ܤܓܝ ܕܫܘܒܗܪܟܘܢ
ܚܢܢ ܐܝܟ ܕܐܦ ܐܢܬܘܢ ܕܝܠܢ ܒܝܘܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:15">1:15</span> ܘܒܗܢܐ ܬܘܟܠܢܐ ܨܒܐ ܗܘܝܬ ܡܢ ܩܕܝܡ ܕܐܬܐ ܠܘܬܟܘܢ
ܕܐܥܝܦܐܝܬ ܬܩܒܠܘܢ ܛܝܒܘܬܐ ܀

<span id="an_1:16">1:16</span> ܘܐܥܒܪ ܥܠܝܟܘܢ ܠܡܩܕܘܢܝܐ ܘܬܘܒ ܡܢ ܡܩܕܘܢܝܐ
ܠܘܬܟܘܢ ܐܬܐ ܘܐܢܬܘܢ ܬܠܘܘܢܢܝ ܠܝܗܘܕ ܀

<span id="an_1:17">1:17</span> ܗܕܐ ܗܟܝܠ ܕܐܬܪܥܝܬ ܠܡܐ ܐܝܟ ܡܤܪܗܒܐ ܐܬܪܥܝܬ ܐܘ
ܕܠܡܐ ܕܒܤܪ ܐܢܝܢ ܐܝܠܝܢ ܕܡܬܪܥܐ ܐܢܐ ܡܛܠ ܕܘܠܐ ܗܘܐ ܕܢܗܘܐ ܒܗܝܢ ܐܝܢ ܐܝܢ ܘܠܐ ܠܐ ܀

<span id="an_1:18">1:18</span> ܡܗܝܡܢ ܗܘ ܐܠܗܐ ܕܠܐ ܗܘܬ ܡܠܬܢ ܕܠܘܬܟܘܢ ܐܝܢ
ܘܠܐ ܀

<span id="an_1:19">1:19</span> ܒܪܗ ܓܝܪ ܕܐܠܗܐ ܝܫܘܥ ܡܫܝܚܐ ܗܘ ܕܒܐܝܕܢ ܐܬܟܪܙ
ܠܟܘܢ ܒܝ ܘܒܤܠܘܢܘܤ ܘܒܛܝܡܬܐܘܤ ܠܐ ܗܘܐ ܐܝܢ ܘܠܐ ܐܠܐ ܐܝܢ ܗܘܐ ܒܗ ܀

<span id="an_1:20">1:20</span> ܟܠܗܘܢ ܓܝܪ ܡܘܠܟܢܘܗܝ ܕܐܠܗܐ ܒܗ ܗܘ ܒܡܫܝܚܐ ܐܝܢ
ܗܘܘ ܡܛܠ ܗܢܐ ܒܐܝܕܗ ܝܗܒܝܢܢ ܐܡܝܢ ܠܬܫܒܘܚܬܗ ܕܐܠܗܐ ܀

<span id="an_1:21">1:21</span> ܐܠܗܐ ܗܘ ܕܝܢ ܡܫܪܪ ܠܢ ܥܡܟܘܢ ܒܡܫܝܚܐ ܗܘ ܕܗܘ
ܡܫܚܢ ܀

<span id="an_1:22">1:22</span> ܘܚܬܡܢ ܘܝܗܒ ܪܗܒܘܢܐ ܕܪܘܚܗ ܒܠܒܘܬܢ ܀

<span id="an_1:23">1:23</span> ܐܢܐ ܕܝܢ ܠܐܠܗܐ ܡܤܗܕ ܐܢܐ ܥܠ ܢܦܫܝ ܕܡܛܠ ܕܚܐܤ
ܐܢܐ ܥܠܝܟܘܢ ܠܐ ܐܬܝܬ ܠܩܘܪܢܬܘܤ ܀

<span id="an_1:24">1:24</span> ܠܐ ܗܘܐ ܡܛܠ ܕܡܪܝ ܗܝܡܢܘܬܟܘܢ ܚܢܢ ܐܠܐ ܡܥܕܪܢܐ
ܚܢܢ ܕܚܕܘܬܟܘܢ ܒܗܝܡܢܘܬܐ ܗܘ ܓܝܪ ܩܝܡܝܢ ܐܢܬܘܢ ܀

<span id="an_2:1">2:1</span> ܕܢܬ ܕܝܢ ܗܕܐ ܒܢܦܫܝ ܕܠܐ ܬܘܒ ܒܟܪܝܘܬܐ ܐܬܐ
ܠܘܬܟܘܢ ܀

<span id="an_2:2">2:2</span> ܐܢ ܐܢܐ ܓܝܪ ܡܟܪܐ ܐܢܐ ܠܟܘܢ ܡܢܘ ܢܚܕܝܢܝ ܐܠܐ ܗܘ
ܕܐܢܐ ܐܟܪܝܬ ܠܗ ܀

<span id="an_2:3">2:3</span> ܘܕܟܬܒܬ ܠܟܘܢ ܗܝ ܗܕܐ ܕܠܐ ܟܕ ܐܬܐ ܢܟܪܘܢ ܠܝ ܗܢܘܢ
ܐܝܠܝܢ ܕܘܠܐ ܗܘܐ ܠܗܘܢ ܕܗܢܘܢ ܢܚܕܘܢܢܝ ܬܟܝܠ ܐܢܐ ܕܝܢ ܥܠ ܟܠܟܘܢ ܕܚܕܘܬܝ ܕܟܠܟܘܢ ܗܝ
܀

<span id="an_2:4">2:4</span> ܘܡܢ ܐܘܠܨܢܐ ܪܒܐ ܘܡܢ ܐܢܘܤܝܐ ܕܠܒܐ ܟܬܒܬ ܠܟܘܢ
ܗܠܝܢ ܒܕܡܥܐ ܤܓܝܐܬܐ ܠܐ ܡܛܠ ܕܬܟܪܐ ܠܟܘܢ ܐܠܐ ܡܛܠ ܕܬܕܥܘܢ ܚܘܒܐ ܝܬܝܪܐ ܕܐܝܬ ܠܝ
ܠܘܬܟܘܢ ܀

<span id="an_2:5">2:5</span> ܐܢ ܕܝܢ ܐܢܫ ܐܟܪܝ ܠܐ ܗܘܐ ܠܝ ܐܟܪܝ ܐܠܐ ܒܨܝܪܐ
ܩܠܝܠ ܠܟܠܟܘܢ ܕܠܐ ܬܐܩܪ ܡܠܬܐ ܥܠܝܟܘܢ ܀

<span id="an_2:6">2:6</span> ܟܕܘ ܠܗ ܕܝܢ ܠܗܢܐ ܟܐܬܐ ܕܡܢ ܤܓܝܐܐ ܀

<span id="an_2:7">2:7</span> ܘܡܟܝܠ ܐܚܪܢܝܐܝܬ ܘܠܐ ܕܬܫܒܩܘܢ ܠܗ ܘܬܒܝܐܘܢܗ ܕܠܡܐ
ܒܟܪܝܘܬܐ ܝܬܝܪܬܐ ܢܬܒܠܥ ܠܗ ܗܘ ܡܢ ܕܗܟܢܐ ܗܘ ܀

<span id="an_2:8">2:8</span> ܡܛܠ ܗܢܐ ܒܥܐ ܐܢܐ ܡܢܟܘܢ ܕܬܫܪܪܘܢ ܒܗ ܚܘܒܟܘܢ ܀

<span id="an_2:9">2:9</span> ܡܛܠ ܗܢܐ ܓܝܪ ܟܬܒܬ ܐܦ ܕܐܕܥ ܒܢܤܝܢܐ ܐܢ ܒܟܠܡܕܡ
ܡܫܬܡܥܝܢ ܐܢܬܘܢ ܀

<span id="an_2:10">2:10</span> ܠܡܢ ܕܐܢܬܘܢ ܕܝܢ ܫܒܩܝܢ ܐܢܬܘܢ ܐܦ ܐܢܐ ܐܦ ܐܢܐ
ܓܝܪ ܡܕܡ ܕܫܒܩܬ ܠܡܢ ܕܫܒܩܬ ܡܛܠܬܟܘܢ ܗܘ ܫܒܩܬ ܒܦܪܨܘܦܗ ܕܡܫܝܚܐ ܀

<span id="an_2:11">2:11</span> ܕܠܐ ܢܥܠܒܢ ܤܛܢܐ ܝܕܥܝܢܢ ܓܝܪ ܡܚܫܒܬܗ ܀

<span id="an_2:12">2:12</span> ܟܕ ܐܬܝܬ ܕܝܢ ܠܛܪܘܐܤ ܒܤܒܪܬܗ ܕܡܫܝܚܐ ܘܐܬܦܬܚ
ܠܝ ܬܪܥܐ ܒܡܪܝܐ ܀

<span id="an_2:13">2:13</span> ܠܐ ܗܘܐ ܠܝ ܢܝܚܐ ܒܪܘܚܝ ܕܠܐ ܐܫܟܚܬ ܠܛܛܘܤ ܐܚܝ
ܐܠܐ ܫܪܝܬ ܐܢܘܢ ܘܢܦܩܬ ܠܝ ܠܡܩܕܘܢܝܐ ܀

<span id="an_2:14">2:14</span> ܛܝܒܘ ܕܝܢ ܠܐܠܗܐ ܕܒܟܠ ܙܒܢ ܚܙܬܐ ܥܒܕ ܠܢ
ܒܡܫܝܚܐ ܘܓܠܐ ܒܢ ܪܝܚܐ ܕܝܕܥܬܗ ܒܟܠ ܐܬܪ ܀

<span id="an_2:15">2:15</span> ܪܝܚܐ ܚܢܢ ܓܝܪ ܒܤܝܡܐ ܒܡܫܝܚܐ ܠܐܠܗܐ ܒܐܝܠܝܢ
ܕܚܐܝܢ ܘܒܐܝܠܝܢ ܕܐܒܕܝܢ ܀

<span id="an_2:16">2:16</span> ܠܐܝܠܝܢ ܕܪܝܚܐ ܕܡܘܬܐ ܠܡܘܬܐ ܘܠܐܝܠܝܢ ܕܪܝܚܐ
ܕܚܝܐ ܠܚܝܐ ܘܠܗܠܝܢ ܡܢܘ ܢܫܘܐ ܀

<span id="an_2:17">2:17</span> ܠܐ ܗܘܝܢ ܓܝܪ ܐܝܟ ܫܪܟܐ ܕܡܡܙܓܝܢ ܡܠܘܗܝ ܕܐܠܗܐ
ܐܠܐ ܐܝܟ ܕܒܫܪܪܐ ܘܐܝܟ ܕܡܢ ܐܠܗܐ ܩܕܡ ܐܠܗܐ ܒܡܫܝܚܐ ܡܡܠܠܝܢܢ ܀

<span id="an_3:1">3:1</span> ܡܫܪܝܢܢ ܠܢ ܬܘܒ ܡܢ ܕܪܝܫ ܕܢܚܘܝܟܘܢ ܡܢܐ ܚܢܢ ܐܘ
ܕܠܡܐ ܤܢܝܩܝܢܢ ܐܝܟ ܐܚܪܢܐ ܕܐܓܪܬܐ ܕܦܘܩܕܐ ܢܬܟܬܒܢ ܠܟܘܢ ܥܠܝܢ ܐܘ ܕܐܢܬܘܢ ܬܟܬܒܘܢ
ܬܦܩܕܘܢ ܥܠܝܢ ܀

<span id="an_3:2">3:2</span> ܐܓܪܬܢ ܕܝܢ ܕܝܠܢ ܐܢܬܘܢ ܐܢܘܢ ܕܟܬܝܒܐ ܒܠܒܢ
ܘܝܕܝܥܐ ܘܡܬܩܪܝܐ ܡܢ ܟܠܢܫ ܀

<span id="an_3:3">3:3</span> ܝܕܥܝܬܘܢ ܓܝܪ ܕܐܓܪܬܗ ܐܢܬܘܢ ܕܡܫܝܚܐ ܕܐܫܬܡܫܬ ܡܢܢ
ܕܟܬܝܒܐ ܠܐ ܒܕܝܘܬܐ ܐܠܐ ܒܪܘܚܐ ܕܐܠܗܐ ܚܝܐ ܠܐ ܒܠܘܚܐ ܕܟܐܦܐ ܐܠܐ ܒܠܘܚܐ ܕܠܒܐ ܕܒܤܪܐ
܀

<span id="an_3:4">3:4</span> ܬܘܟܠܢܐ ܕܝܢ ܗܟܢܐ ܐܝܬ ܠܢ ܒܡܫܝܚܐ ܠܘܬ ܐܠܗܐ ܀

<span id="an_3:5">3:5</span> ܠܘ ܕܚܢܢ ܤܦܩܝܢܢ ܕܢܬܪܥܐ ܡܕܡ ܐܝܟ ܕܡܢ ܢܦܫܢ ܐܠܐ
ܚܝܠܢ ܡܢ ܐܠܗܐ ܗܘ ܀

<span id="an_3:6">3:6</span> ܗܘ ܕܐܫܘܝܢ ܕܢܗܘܐ ܡܫܡܫܢܐ ܕܕܝܬܩܐ ܚܕܬܐ ܠܐ ܒܟܬܒܐ
ܐܠܐ ܒܪܘܚܐ ܟܬܒܐ ܓܝܪ ܩܛܠ ܪܘܚܐ ܕܝܢ ܡܚܝܐ ܀

<span id="an_3:7">3:7</span> ܐܢ ܕܝܢ ܬܫܡܫܬܐ ܕܡܘܬܐ ܒܟܬܒܐ ܐܬܪܫܡܬ ܒܟܐܦܐ ܘܗܘܬ
ܒܬܫܒܘܚܬܐ ܐܝܟܢܐ ܕܠܐ ܢܫܟܚܘܢ ܒܢܝ ܐܝܤܪܝܠ ܠܡܚܪ ܒܦܪܨܘܦܗ ܕܡܘܫܐ ܡܛܠ ܫܘܒܚܐ
ܕܦܪܨܘܦܗ ܗܘ ܕܐܬܒܛܠ ܀

<span id="an_3:8">3:8</span> ܐܝܟܢܐ ܗܟܝܠ ܬܫܡܫܬܐ ܕܪܘܚܐ ܠܐ ܝܬܝܪܐܝܬ ܬܗܘܐ
ܒܫܘܒܚܐ ܀

ܡܗܝܡܢܝܢ ܕܠܐ ܢܕܢܚ ܠܗܘܢ ܢܘܗܪܗ ܕܐܘܢܓܠܝܘܢ ܕܫܘܒܚܗ ܕܡܫܝܚܐ ܕܗܘܝܘ ܕܡܘܬܗ ܕܐܠܗܐ ܀

<span id="an_4:5">4:5</span> ܠܐ ܗܘܐ ܓܝܪ ܢܦܫܢ ܡܟܪܙܝܢܢ ܐܠܐ ܠܡܫܝܚܐ ܝܫܘܥ ܡܪܢ

<span id="an_4:6">4:6</span> ܡܛܠ ܕܐܠܗܐ ܗܘ ܕܐܡܪ ܕܡܢ ܚܫܘܟܐ ܢܘܗܪܐ ܢܕܢܚ ܗܘ
ܕܢܚ ܒܠܒܘܬܢ ܕܢܬܢܗܪ ܒܝܕܥܬܐ ܕܫܘܒܚܗ ܕܐܠܗܐ ܒܦܪܨܘܦܗ ܕܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_4:7">4:7</span> ܐܝܬ ܠܢ ܕܝܢ ܤܝܡܬܐ ܗܕܐ ܒܡܐܢܐ ܕܚܨܦܐ ܕܪܒܘܬܐ
ܕܚܝܠܐ ܡܢ ܐܠܗܐ ܬܗܘܐ ܘܠܐ ܡܢܢ ܀

<span id="an_4:8">4:8</span> ܒܟܠܡܕܡ ܓܝܪ ܡܬܐܠܨܝܢܢ ܐܠܐ ܠܐ ܡܬܚܢܩܝܢܢ

<span id="an_4:9">4:9</span> ܡܬܪܕܦܝܢܢ ܐܠܐ ܠܐ ܡܫܬܒܩܝܢܢ ܡܤܬܚܦܝܢܢ ܐܠܐ ܠܐ

<span id="an_4:11">4:11</span> ܐܢ ܚܢܢ ܓܝܪ ܚܝܐ ܠܡܘܬܐ ܡܫܬܠܡܝܢܢ ܡܛܠ ܝܫܘܥ
ܗܟܢܐ ܐܦ ܚܝܘܗܝ ܕܝܫܘܥ ܢܬܓܠܘܢ ܒܦܓܪܢ ܗܢܐ ܕܡܐܬ ܀

<span id="an_4:13">4:13</span> ܐܦ ܚܢܢ ܗܟܝܠ ܕܚܕܐ ܗܝ ܪܘܚܐ ܕܗܝܡܢܘܬܐ ܐܝܬ ܠܢ
ܐܝܟ ܕܟܬܝܒ ܕܗܝܡܢܬ ܡܛܠ ܗܢܐ ܐܦ ܡܠܠܬ ܡܗܝܡܢܝܢܢ ܡܛܠ ܗܢܐ ܐܦ ܡܡܠܠܝܢܢ ܀

<span id="an_4:14">4:14</span> ܘܝܕܥܝܢܢ ܕܗܘ ܡܢ ܕܐܩܝܡ ܠܡܪܢ ܝܫܘܥ ܐܦ ܠܢ ܒܝܕ

<span id="an_4:15">4:15</span> ܟܠܡܕܡ ܓܝܪ ܡܛܠܬܟܘܢ ܗܘ ܕܟܕ ܡܬܝܬܪܐ ܛܝܒܘܬܐ
ܒܝܕ ܤܓܝܐܐ ܬܤܓܐ ܬܘܕܝܬܐ ܠܫܘܒܚܗ ܕܐܠܗܐ ܀

<span id="an_4:16">4:16</span> ܡܛܠ ܗܢܐ ܠܐ ܡܐܢܐ ܠܢ ܐܦ ܐܢ ܒܪܢܫܢ ܓܝܪ ܒܪܝܐ
ܡܬܚܒܠ ܐܠܐ ܕܡܢ ܠܓܘ ܡܬܚܕܬ ܝܘܡ ܡܢ ܝܘܡ ܀

ܫܘܒܚܐ ܪܒܐ ܕܠܐ ܤܟܐ ܠܥܠܡ ܥܠܡܝܢ ܡܛܝܒ ܠܢ ܀

ܡܬܚܙܝܢ ܕܡܬܚܙܝܢ ܓܝܪ ܕܙܒܢܐ ܐܢܝܢ ܕܠܐ ܡܬܚܙܝܢ ܕܝܢ ܕܠܥܠܡ ܐܢܝܢ ܀

ܝܘܩܪܗ ܘܠܐ ܨܒܝܢܢ ܠܡܫܠܚܗ ܐܠܐ ܕܢܠܒܫ ܠܥܠ ܡܢܗ ܕܬܬܒܠܥ ܡܝܬܘܬܗ ܒܚܝܐ ܀

<span id="an_5:5">5:5</span> ܘܕܡܥܬܕ ܠܢ ܠܗ ܠܗܕܐ ܐܠܗܐ ܗܘ ܗܘ ܕܝܗܒ ܠܢ ܪܗܒܘܢܐ

ܠܦܓܪܢ ܐܠܐ ܒܟܠ ܡܕܡ ܐܬܐܠܨܢ ܡܢ ܠܒܪ ܩܪܒܐ ܘܡܢ ܠܓܘ ܕܚܠܬܐ ܀

ܒܟܘܢ ܤܒܪܢ ܓܝܪ ܥܠ ܚܘܒܟܘܢ ܕܠܘܬܢ ܘܥܠ ܐܒܠܟܘܢ ܘܛܢܢܟܘܢ ܕܥܠ ܐܦܝܢ ܘܟܕ ܫܡܥܬ ܚܕܘܬܐ

ܬܘܝܐ ܗܘܬ ܚܙܐ ܐܢܐ ܓܝܪ ܕܗܝ ܐܓܪܬܐ ܐܦܢ ܕܫܥܬܐ ܐܟܪܝܬ ܠܟܘܢ ܀

ܐܠܐ ܡܛܠ ܕܟܪܝܘܬܟܘܢ ܠܬܝܒܘܬܐ ܐܝܬܝܬܟܘܢ ܟܪܝܬ ܠܟܘܢ ܓܝܪ ܒܕܐܠܗܐ ܐܝܟܢܐ ܕܒܡܕܡ ܠܐ

ܗܦܟܐ ܘܡܦܢܝܐ ܠܚܝܐ ܟܪܝܘܬܐ ܕܝܢ ܕܥܠܡܐ ܡܘܬܐ ܗܘ ܥܒܕܐ ܀

ܠܚܕܘܬܗܘܢ ܘܥܘܡܩܐ ܕܡܤܟܢܘܬܗܘܢ ܐܬܝܬܪ ܒܥܘܬܪܐ ܕܦܫܝܛܘܬܗܘܢ ܀

<span id="an_13:4">13:4</span> ܐܦܢ ܐܙܕܩܦ ܓܝܪ ܒܡܚܝܠܘܬܐ ܐܠܐ ܚܝ ܗܘ ܒܚܝܠܐ
ܕܐܠܗܐ ܘܐܦ ܚܢܢ ܡܚܝܠܝܢܢ ܥܡܗ ܐܠܐ ܚܝܝܢ ܚܢܢ ܥܡܗ ܒܚܝܠܐ ܕܐܠܗܐ ܕܒܟܘܢ ܀

<span id="an_13:5">13:5</span> ܢܦܫܟܘܢ ܒܩܘ ܐܢ ܒܗ ܒܗܝܡܢܘܬܐ ܩܝܡܝܢ ܐܢܬܘܢ
ܢܦܫܟܘܢ ܐܤܘ ܐܘ ܠܐ ܡܫܬܘܕܥܝܬܘܢ ܕܝܫܘܥ ܡܫܝܚܐ ܒܟܘܢ ܗܘ ܘܐܢ ܠܐ ܡܤܠܝܐ ܐܢܬܘܢ ܀

<span id="an_13:6">13:6</span> ܡܤܒܪ ܐܢܐ ܕܝܢ ܕܬܕܥܘܢ ܕܚܢܢ ܠܐ ܗܘܝܢ ܡܤܠܝܢ ܀

<span id="an_13:7">13:7</span> ܒܥܐ ܐܢܐ ܕܝܢ ܡܢ ܐܠܗܐ ܕܠܐ ܢܗܘܐ ܒܟܘܢ ܡܕܡ
ܕܒܝܫ ܐܝܟ ܕܒܘܩܝܢ ܕܝܠܢ ܢܬܚܙܐ ܐܠܐ ܕܐܢܬܘܢ ܬܗܘܘܢ ܤܥܪܝܢ ܛܒܬܐ ܘܚܢܢ ܢܗܘܐ ܐܝܟ
ܡܤܠܝܐ ܀

<span id="an_13:8">13:8</span> ܠܐ ܓܝܪ ܡܫܟܚܝܢܢ ܕܢܥܒܕ ܡܕܡ ܠܘܩܒܠ ܩܘܫܬܐ ܐܠܐ
ܚܠܦ ܩܘܫܬܐ ܀

<span id="an_13:9">13:9</span> ܚܕܝܢܢ ܕܝܢ ܡܐ ܕܚܢܢ ܟܪܝܗܝܢ ܘܐܢܬܘܢ ܚܝܠܬܢܝܢ
ܗܕܐ ܓܝܪ ܐܦ ܡܨܠܝܢܢ ܕܐܢܬܘܢ ܬܬܓܡܪܘܢ ܀

<span id="an_13:10">13:10</span> ܡܛܠ ܗܢܐ ܟܕ ܪܚܝܩ ܐܢܐ ܗܠܝܢ ܟܬܒ ܐܢܐ ܕܠܐ ܡܐ
ܕܐܬܝܬ ܩܫܝܐܝܬ ܐܤܥܘܪ ܐܝܟ ܫܘܠܛܢܐ ܕܝܗܒ ܠܝ ܡܪܝ ܠܒܢܝܢܟܘܢ ܘܠܐ ܠܤܘܚܦܟܘܢ ܀

<span id="an_13:11">13:11</span> ܡܟܝܠ ܐܚܝ ܚܕܘ ܘܐܬܓܡܪܘ ܘܐܬܒܝܐܘ ܘܐܘܝܘܬܐ
ܘܫܝܢܐ ܢܗܘܐ ܒܟܘܢ ܘܐܠܗܐ ܕܚܘܒܐ ܘܕܫܠܡܘܬܐ ܢܗܘܐ ܥܡܟܘܢ ܀

<span id="an_13:12">13:12</span> ܫܐܠܘ ܒܫܠܡܐ ܚܕ ܕܚܕ ܒܢܘܫܩܬܐ ܩܕܝܫܬܐ ܀

<span id="an_13:13">13:13</span> ܫܐܠܝܢ ܒܫܠܡܟܘܢ ܟܠܗܘܢ ܩܕܝܫܐ ܀

<span id="an_13:14">13:14</span> ܫܠܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܚܘܒܗ ܕܐܠܗܐ
ܘܫܘܬܦܘܬܐ ܕܪܘܚܐ ܕܩܘܕܫܐ ܥܡ ܟܠܟܘܢ ܐܡܝܢ ܀
