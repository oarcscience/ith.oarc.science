### Aramaic Bible: 1 Corinthians

<span id="an_1:1">1:1</span> ܦܘܠܘܤ ܩܪܝܐ ܘܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܒܨܒܝܢܗ ܕܐܠܗܐ
ܘܤܘܤܬܢܤ ܐܚܐ ܀

<span id="an_1:2">1:2</span> ܠܥܕܬܐ ܕܐܠܗܐ ܕܒܩܘܪܢܬܘܤ ܩܪܝܐ ܘܩܕܝܫܐ ܕܒܝܫܘܥ
ܡܫܝܚܐ ܡܩܕܫܝܢ ܘܠܟܠܗܘܢ ܐܝܠܝܢ ܕܩܪܝܢ ܫܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܒܟܠ ܐܬܪ ܕܝܠܗܘܢ
ܘܕܝܠܢ ܀

<span id="an_1:3">1:3</span> ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ
ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:4">1:4</span> ܡܘܕܐ ܐܢܐ ܠܐܠܗܝ ܒܟܠܙܒܢ ܚܠܦܝܟܘܢ ܥܠ ܛܝܒܘܬܐ
ܕܐܠܗܐ ܕܐܬܝܗܒܬ ܠܟܘܢ ܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:5">1:5</span> ܕܒܟܠ ܡܕܡ ܥܬܪܬܘܢ ܒܗ ܒܟܠ ܡܠܐ ܘܒܟܠ ܝܕܥܬܐ ܀

<span id="an_1:6">1:6</span> ܐܝܟ ܕܤܗܕܘܬܗ ܕܡܫܝܚܐ ܐܫܬܪܪܬ ܒܟܘܢ ܀

<span id="an_1:7">1:7</span> ܕܠܐ ܐܬܒܨܪܬܘܢ ܒܚܕܐ ܡܢ ܡܘܗܒܬܗ ܐܠܐ ܡܤܟܝܢ ܐܢܬܘܢ
ܠܓܠܝܢܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:8">1:8</span> ܕܗܘ ܢܫܪܪܟܘܢ ܥܕܡܐ ܠܐܚܪܝܬܐ ܕܕܠܐ ܪܫܝܢ ܬܗܘܘܢ
ܒܝܘܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:9">1:9</span> ܡܗܝܡܢ ܗܘ ܐܠܗܐ ܕܒܐܝܕܗ ܐܬܩܪܝܬܘܢ ܠܫܘܬܦܘܬܐ ܕܒܪܗ
ܝܫܘܥ ܡܫܝܚܐ ܡܪܢ ܀

<span id="an_1:10">1:10</span> ܒܥܐ ܐܢܐ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܒܫܡܗ ܕܡܪܢ ܝܫܘܥ
ܡܫܝܚܐ ܕܬܗܘܐ ܚܕܐ ܡܠܬܐ ܠܟܠܟܘܢ ܘܠܐ ܢܗܘܝܢ ܒܟܘܢ ܦܠܓܘܬܐ ܐܠܐ ܬܗܘܘܢ ܓܡܝܪܝܢ ܒܚܕܐ
ܬܪܥܝܬܐ ܘܒܚܕ ܪܥܝܢܐ ܀

<span id="an_1:11">1:11</span> ܫܠܚܘ ܠܝ ܓܝܪ ܥܠܝܟܘܢ ܐܚܝ ܡܢ ܒܝܬ ܟܠܐܐ ܕܚܪܝܢܐ
ܐܝܬ ܒܝܢܬܟܘܢ ܀

<span id="an_1:12">1:12</span> ܗܕܐ ܕܝܢ ܐܡܪ ܐܢܐ ܕܐܝܬ ܡܢܟܘܢ ܕܐܡܪ ܐܢܐ
ܕܦܘܠܘܤ ܐܢܐ ܘܐܝܬ ܕܐܡܪ ܐܢܐ ܕܐܦܠܘ ܐܢܐ ܘܐܝܬ ܕܐܡܪ ܐܢܐ ܕܟܐܦܐ ܐܢܐ ܘܐܝܬ ܕܐܡܪ ܐܢܐ
ܕܡܫܝܚܐ ܐܢܐ ܀

<span id="an_1:13">1:13</span> ܕܠܡܐ ܐܬܦܠܓ ܠܗ ܡܫܝܚܐ ܐܘ ܠܡܐ ܦܘܠܘܤ ܐܙܕܩܦ ܥܠ
ܐܦܝܟܘܢ ܐܘ ܒܫܡܗ ܕܦܘܠܘܤ ܥܡܕܬܘܢ ܀

<span id="an_1:14">1:14</span> ܡܘܕܐ ܐܢܐ ܠܐܠܗܝ ܕܠܐܢܫ ܡܢܟܘܢ ܠܐ ܐܥܡܕܬ ܐܠܐ
ܠܟܪܝܤܦܘܤ ܘܠܓܐܝܘܤ ܀

<span id="an_1:15">1:15</span> ܕܠܐ ܐܢܫ ܢܐܡܪ ܕܒܫܡܝ ܐܥܡܕܬ ܀

<span id="an_1:16">1:16</span> ܐܥܡܕܬ ܕܝܢ ܐܦ ܠܒܝܬܗ ܕܐܤܛܦܢܐ ܬܘܒ ܕܝܢ ܠܐ ܝܕܥ
ܐܢܐ ܐܢ ܠܐܢܫ ܐܚܪܝܢ ܐܥܡܕܬ ܀

<span id="an_1:17">1:17</span> ܠܐ ܓܝܪ ܫܕܪܢܝ ܡܫܝܚܐ ܠܡܥܡܕܘ ܐܠܐ ܠܡܤܒܪܘ ܠܐ
ܒܚܟܡܬ ܡܠܐ ܕܠܐ ܢܤܬܪܩ ܙܩܝܦܗ ܕܡܫܝܚܐ ܀

<span id="an_1:18">1:18</span> ܡܠܬܐ ܓܝܪ ܕܙܩܝܦܐ ܠܐܒܝܕܐ ܫܛܝܘܬܐ ܗܝ ܠܢ ܕܝܢ
ܠܐܝܠܝܢ ܕܚܝܝܢܢ ܚܝܠܐ ܗܝ ܕܐܠܗܐ ܀

<span id="an_1:19">1:19</span> ܟܬܝܒ ܓܝܪ ܕܐܘܒܕ ܚܟܡܬܐ ܕܚܟܝܡܐ ܘܐܓܠܘܙ ܬܪܥܝܬܐ
ܕܤܟܘܠܬܢܐ ܀

<span id="an_1:20">1:20</span> ܐܝܟܐ ܗܘ ܚܟܝܡܐ ܐܘ ܐܝܟܐ ܗܘ ܤܦܪܐ ܐܘ ܐܝܟܐ ܗܘ
ܕܪܘܫܐ ܕܥܠܡܐ ܗܢܐ ܠܐ ܗܐ ܐܫܛܝܗ ܐܠܗܐ ܚܟܡܬܗ ܕܥܠܡܐ ܗܢܐ ܀

<span id="an_1:21">1:21</span> ܡܛܠ ܓܝܪ ܕܒܚܟܡܬܗ ܕܐܠܗܐ ܠܐ ܝܕܥ ܥܠܡܐ ܒܚܟܡܬܐ
ܠܐܠܗܐ ܨܒܐ ܐܠܗܐ ܕܒܫܛܝܘܬܐ ܕܟܪܘܙܘܬܐ ܢܚܐ ܠܐܝܠܝܢ ܕܡܗܝܡܢܝܢ ܀

<span id="an_1:22">1:22</span> ܡܛܠ ܕܝܗܘܕܝܐ ܐܬܘܬܐ ܫܐܠܝܢ ܘܐܪܡܝܐ ܚܟܡܬܐ ܒܥܝܢ
܀

<span id="an_1:23">1:23</span> ܚܢܢ ܕܝܢ ܡܟܪܙܝܢܢ ܡܫܝܚܐ ܟܕ ܙܩܝܦ ܬܘܩܠܬܐ
ܠܝܗܘܕܝܐ ܘܠܐܪܡܝܐ ܫܛܝܘܬܐ ܀

<span id="an_1:24">1:24</span> ܠܗܢܘܢ ܕܝܢ ܕܩܪܝܢ ܝܗܘܕܝܐ ܘܐܪܡܝܐ ܡܫܝܚܐ ܚܝܠܐ
ܕܐܠܗܐ ܘܚܟܡܬܐ ܕܐܠܗܐ ܀

<span id="an_1:25">1:25</span> ܡܛܠ ܕܫܛܝܘܬܗ ܕܐܠܗܐ ܚܟܝܡܐ ܗܝ ܡܢ ܒܢܝܢܫܐ
ܘܟܪܝܗܘܬܗ ܕܐܠܗܐ ܚܝܠܬܢܝܐ ܗܝ ܡܢ ܒܢܝ ܐܢܫܐ ܀

<span id="an_1:26">1:26</span> ܚܙܘ ܓܝܪ ܐܦ ܩܪܝܬܟܘܢ ܐܚܝ ܕܠܐ ܤܓܝܐܝܢ ܒܟܘܢ
ܚܟܝܡܐ ܒܒܤܪ ܘܠܐ ܤܓܝܐܝܢ ܒܟܘܢ ܚܝܠܬܢܐ ܘܠܐ ܤܓܝܐܝܢ ܒܟܘܢ ܒܢܝ ܛܘܗܡܐ ܪܒܐ ܀

<span id="an_1:27">1:27</span> ܐܠܐ ܓܒܐ ܐܠܗܐ ܠܤܟܠܘܗܝ ܕܥܠܡܐ ܕܢܒܗܬ ܠܚܟܝܡܐ
ܘܓܒܐ ܟܪܝܗܘܗܝ ܕܥܠܡܐ ܕܢܒܗܬ ܠܚܝܠܬܢܐ ܀

<span id="an_1:28">1:28</span> ܘܓܒܐ ܠܐܝܠܝܢ ܕܒܨܝܪ ܛܘܗܡܗܘܢ ܒܥܠܡܐ ܘܠܡܤܠܝܐ
ܘܠܐܝܠܝܢ ܕܠܝܬ ܐܢܘܢ ܕܢܒܛܠ ܠܐܝܠܝܢ ܕܐܝܬܝܗܘܢ ܀

<span id="an_1:29">1:29</span> ܕܠܐ ܢܫܬܒܗܪ ܟܠ ܒܤܪ ܩܕܡܘܗܝ ܀

<span id="an_1:30">1:30</span> ܐܦ ܐܢܬܘܢ ܕܝܢ ܡܢܗ ܐܢܬܘܢ ܒܝܫܘܥ ܡܫܝܚܐ ܗܘ
ܕܗܘܐ ܠܢ ܚܟܡܬܐ ܡܢ ܐܠܗܐ ܘܙܕܝܩܘܬܐ ܘܩܕܝܫܘܬܐ ܘܦܘܪܩܢܐ ܀

<span id="an_1:31">1:31</span> ܐܝܟ ܕܟܬܝܒ ܕܡܢ ܕܡܫܬܒܗܪ ܒܡܪܝܐ ܢܫܬܒܗܪ ܀

<span id="an_2:1">2:1</span> ܘܐܢܐ ܐܚܝ ܟܕ ܐܬܝܬ ܠܘܬܟܘܢ ܠܐ ܒܡܡܠܠܐ ܪܘܪܒܐ
ܐܦܠܐ ܒܚܟܡܬܐ ܤܒܪܬܟܘܢ ܐܪܙܗ ܕܐܠܗܐ ܀

<span id="an_2:2">2:2</span> ܘܠܐ ܕܢܬ ܢܦܫܝ ܒܝܢܬܟܘܢ ܐܝܟ ܕܡܕܡ ܝܕܥ ܐܢܐ ܐܠܐ
ܐܢ ܠܝܫܘܥ ܡܫܝܚܐ ܐܦ ܠܗ ܟܕ ܙܩܝܦ ܀

<span id="an_2:3">2:3</span> ܘܐܢܐ ܒܕܚܠܬܐ ܤܓܝܐܬܐ ܘܒܪܬܝܬܐ ܗܘܝܬ ܠܘܬܟܘܢ ܀

<span id="an_2:4">2:4</span> ܘܡܠܬܝ ܘܟܪܘܙܘܬܝ ܠܐ ܗܘܬ ܒܡܦܝܤܢܘܬܐ ܕܡܠܐ ܕܚܟܡܬܐ
ܐܠܐ ܒܬܚܘܝܬܐ ܕܪܘܚܐ ܘܕܚܝܠܐ ܀

<span id="an_2:5">2:5</span> ܕܠܐ ܬܗܘܐ ܗܝܡܢܘܬܟܘܢ ܒܚܟܡܬܐ ܕܒܢܝܢܫܐ ܐܠܐ ܒܚܝܠܐ
ܕܐܠܗܐ ܀

<span id="an_2:6">2:6</span> ܚܟܡܬܐ ܕܝܢ ܡܡܠܠܝܢܢ ܒܓܡܝܪܐ ܚܟܡܬܐ ܠܐ ܕܗܢܐ ܥܠܡܐ
ܐܦܠܐ ܕܫܠܝܛܢܘܗܝ ܕܥܠܡܐ ܗܢܐ ܕܡܬܒܛܠܝܢ ܀

<span id="an_2:7">2:7</span> ܐܠܐ ܡܡܠܠܝܢܢ ܚܟܡܬܐ ܕܐܠܗܐ ܒܐܪܙ ܗܝ ܕܡܟܤܝܐ ܗܘܬ
ܘܩܕܡ ܗܘܐ ܦܪܫܗ ܐܠܗܐ ܡܢ ܩܕܡ ܥܠܡܐ ܠܫܘܒܚܐ ܕܝܠܢ ܀

<span id="an_2:8">2:8</span> ܗܝ ܕܚܕ ܡܢ ܫܠܝܛܢܐ ܕܥܠܡܐ ܗܢܐ ܠܐ ܝܕܥܗ ܐܠܘ ܓܝܪ
ܝܕܥܘܗ ܠܘ ܠܡܪܗ ܕܬܫܒܘܚܬܐ ܙܩܦܝܢ ܗܘܘ ܀

<span id="an_2:9">2:9</span> ܐܠܐ ܐܝܟ ܕܟܬܝܒ ܕܥܝܢܐ ܠܐ ܚܙܬ ܘܐܕܢܐ ܠܐ ܫܡܥܬ
ܘܥܠ ܠܒܐ ܕܒܪܢܫܐ ܠܐ ܤܠܩ ܡܕܡ ܕܛܝܒ ܐܠܗܐ ܠܐܝܠܝܢ ܕܪܚܡܝܢ ܠܗ ܀

<span id="an_2:10">2:10</span> ܠܢ ܕܝܢ ܓܠܐ ܐܠܗܐ ܒܪܘܚܗ ܪܘܚܐ ܓܝܪ ܟܠܡܕܡ ܒܨܝܐ
ܐܦ ܥܘܡܩܘܗܝ ܕܐܠܗܐ ܀

<span id="an_2:11">2:11</span> ܐܝܢܐ ܗܘ ܓܝܪ ܒܪܢܫܐ ܕܝܕܥ ܕܒܒܪܢܫܐ ܐܠܐ ܐܢ
ܪܘܚܐ ܕܒܪ ܕܒܗ ܗܟܢܐ ܐܦ ܕܒܐܠܗܐ ܐܢܫ ܠܐ ܝܕܥ ܐܠܐ ܐܢ ܪܘܚܗ ܕܐܠܗܐ ܀

<span id="an_2:12">2:12</span> ܚܢܢ ܕܝܢ ܠܐ ܗܘܐ ܪܘܚܐ ܕܥܠܡܐ ܢܤܒܢ ܐܠܐ ܪܘܚܐ
ܕܡܢ ܐܠܗܐ ܕܢܕܥ ܡܘܗܒܬܐ ܕܡܢ ܐܠܗܐ ܐܬܝܗܒ ܠܢ ܀

<span id="an_2:13">2:13</span> ܐܝܠܝܢ ܕܐܦ ܡܡܠܠܝܢܢ ܠܐ ܗܘܐ ܒܝܘܠܦܢܐ ܕܡܠܐ
ܕܚܟܡܬܐ ܕܒܢܝܢܫܐ ܐܠܐ ܒܝܘܠܦܢܐ ܕܪܘܚܐ ܘܠܪܘܚܢܐ ܪܘܚܢܝܬܐ ܡܦܚܡܝܢܢ ܀

<span id="an_2:14">2:14</span> ܒܪܢܫܐ ܓܝܪ ܕܒܢܦܫ ܗܘ ܠܐ ܡܩܒܠ ܪܘܚܢܝܬܐ ܫܛܝܘܬܐ
ܐܢܝܢ ܓܝܪ ܠܗ ܘܠܐ ܡܫܟܚ ܕܢܕܥ ܕܒܪܘܚ ܡܬܕܝܢ ܀

<span id="an_2:15">2:15</span> ܪܘܚܢܐ ܕܝܢ ܟܠ ܡܕܡ ܕܐܢ ܘܗܘ ܡܢ ܐܢܫ ܠܐ ܡܬܕܝܢ
܀

<span id="an_2:16">2:16</span> ܡܢܘ ܓܝܪ ܝܕܥ ܪܥܝܢܗ ܕܡܪܝܐ ܕܢܠܦܝܘܗܝ ܠܢ ܕܝܢ
ܪܥܝܢܐ ܕܡܫܝܚܐ ܐܝܬ ܠܢ ܀

<span id="an_3:1">3:1</span> ܘܐܢܐ ܐܚܝ ܠܐ ܐܫܟܚܬ ܕܐܡܠܠ ܥܡܟܘܢ ܐܝܟ ܕܥܡ ܪܘܚܢܐ
ܐܠܐ ܐܝܟ ܕܥܡ ܦܓܪܢܐ ܘܐܝܟ ܕܠܝܠܘܕܐ ܒܡܫܝܚܐ ܀

<span id="an_3:2">3:2</span> ܚܠܒܐ ܐܫܩܝܬܟܘܢ ܘܠܐ ܝܗܒܬ ܠܟܘܢ ܡܐܟܘܠܬܐ ܠܐ ܓܝܪ
ܥܕܟܝܠ ܡܫܟܚܝܢ ܗܘܝܬܘܢ ܐܠܐ ܐܦܠܐ ܗܫܐ ܡܫܟܚܝܢ ܐܢܬܘܢ ܀

<span id="an_3:3">3:3</span> ܥܕܟܝܠ ܓܝܪ ܒܒܤܪ ܐܢܬܘܢ ܐܝܟܐ ܓܝܪ ܕܐܝܬ ܒܟܘܢ
ܚܤܡܐ ܘܚܪܝܢܐ ܘܦܠܓܘܬܐ ܠܐ ܗܐ ܦܓܪܢܐ ܐܢܬܘܢ ܘܒܒܤܪ ܡܗܠܟܝܢ ܐܢܬܘܢ ܀

<span id="an_3:4">3:4</span> ܡܐ ܕܐܡܪ ܓܝܪ ܐܢܫ ܐܢܫ ܡܢܟܘܢ ܐܢܐ ܕܦܘܠܘܤ ܐܢܐ
ܘܐܚܪܢܐ ܐܡܪ ܐܢܐ ܕܐܦܠܘ ܐܢܐ ܠܐ ܗܐ ܦܓܪܢܐ ܐܢܬܘܢ ܀

<span id="an_3:5">3:5</span> ܡܢܘ ܓܝܪ ܦܘܠܘܤ ܐܘ ܡܢܘ ܐܦܠܘ ܐܠܐ ܡܫܡܫܢܐ
ܕܒܐܝܕܝܗܘܢ ܗܝܡܢܬܘܢ ܘܐܢܫ ܐܢܫ ܐܝܟ ܕܝܗܒ ܠܗ ܡܪܝܐ ܀

<span id="an_3:6">3:6</span> ܐܢܐ ܢܨܒܬ ܘܐܦܠܘ ܐܫܩܝ ܐܠܐ ܐܠܗܐ ܪܒܝ ܀

<span id="an_3:7">3:7</span> ܠܐ ܗܟܝܠ ܗܘ ܕܢܨܒ ܐܝܬܘܗܝ ܡܕܡ ܘܠܐ ܗܘ ܕܡܫܩܐ ܐܠܐ
ܐܠܗܐ ܕܡܪܒܐ ܀
