### Aramaic Bible: Jude

<span id="an_1:1">1:1</span> ܝܗܘܕܐ ܥܒܕܐ ܕܝܫܘܥ ܡܫܝܚܐ ܐܚܐ ܕܝܢ ܕܝܥܩܘܒ ܠܥܡܡܐ
ܩܪܝܐ ܕܒܐܠܗܐ ܐܒܐ ܪܚܝܡܝܢ ܘܒܝܫܘܥ ܡܫܝܚܐ ܢܛܝܪܝܢ ܀

<span id="an_1:2">1:2</span> ܪܚܡܐ ܘܫܠܡܐ ܘܚܘܒܐ ܢܤܓܐ ܠܟܘܢ ܀

<span id="an_1:3">1:3</span> ܚܒܝܒܝ ܟܕ ܟܠܗ ܝܨܝܦܘܬܐ ܥܒܕ ܐܢܐ ܠܡܟܬܒ ܠܟܘܢ ܥܠ
ܚܝܐ ܕܝܠܢ ܕܓܘܐ ܐܢܢܩܐ ܐܝܬ ܠܝ ܠܡܟܬܒ ܠܟܘܢ ܟܕ ܡܦܝܤ ܐܢܐ ܕܐܓܘܢܐ ܬܥܒܕܘܢ ܚܠܦ
ܗܝܡܢܘܬܐ ܐܝܕܐ ܕܚܕܐ ܙܒܢ ܐܫܬܠܡܬ ܠܩܕܝܫܐ ܀

<span id="an_1:4">1:4</span> ܩܢܘ ܓܝܪ ܐܢܫܝܢ ܡܥܠܢܘܬܐ ܐܝܠܝܢ ܕܡܢ ܫܘܪܝܐ ܩܕܡܘ
ܐܬܟܬܒܘ ܒܚܘܝܒܐ ܗܢܐ ܐܢܫܐ ܪܫܝܥܐ ܕܠܛܝܒܘܬܗ ܕܐܠܗܢ ܡܗܦܟܝܢ ܠܛܢܦܘܬܐ ܘܒܗܘ ܕܐܝܬܘܗܝ
ܒܠܚܘܕܘܗܝ ܡܪܐ ܐܠܗܐ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܟܦܪܝܢ ܀

<span id="an_1:5">1:5</span> ܠܡܥܗܕܘܬܟܘܢ ܕܝܢ ܨܒܐ ܐܢܐ ܟܕ ܝܕܥܝܢ ܐܢܬܘܢ ܟܠܗܝܢ
ܕܐܠܗܐ ܟܕ ܚܕܐ ܙܒܢ ܠܥܡܐ ܡܢ ܡܨܪܝܢ ܦܪܩ ܗܝ ܕܬܪܬܝܢ ܠܐܝܠܝܢ ܕܠܐ ܗܝܡܢܘ ܐܘܒܕ ܀

<span id="an_1:6">1:6</span> ܘܠܡܠܐܟܐ ܐܝܠܝܢ ܕܠܐ ܢܛܪܘ ܪܝܫܢܘܬܗܘܢ ܐܠܐ ܫܒܩܘ
ܥܘܡܪܐ ܕܝܠܗܘܢ ܠܕܝܢܐ ܕܝܘܡܐ ܪܒܐ ܒܐܤܘܪܐ ܠܐ ܝܕܝܥܐ ܬܚܝܬ ܥܡܛܢܐ ܢܛܪ ܀

<span id="an_1:7">1:7</span> ܐܝܟܢܐ ܕܤܕܘܡ ܘܥܡܘܪܐ ܘܡܕܝܢܬܐ ܕܚܕܪܝܗܝܢ ܕܒܗ
ܒܕܡܘܬܐ ܕܗܠܝܢ ܙܢܝ ܘܐܙܠ ܒܬܪ ܒܤܪܐ ܐܚܪܢܐ ܤܝܡܢ ܬܚܝܬ ܬܚܘܝܬܐ ܕܢܘܪܐ ܕܠܥܠܡ ܟܕ
ܡܚܝܒܢ ܠܕܝܢܐ ܀

<span id="an_1:8">1:8</span> ܒܗ ܒܕܡܘܬܐ ܐܦ ܗܠܝܢ ܕܒܚܠܡܐ ܡܫܬܪܓܪܓܝܢ ܠܒܤܪܐ ܡܢ
ܡܛܢܦܝܢ ܠܡܪܘܬܐ ܕܝܢ ܛܠܡܝܢ ܘܠܬܫܒܘܚܬܐ ܡܓܕܦܝܢ ܀

<span id="an_1:9">1:9</span> ܡܝܟܐܝܠ ܕܝܢ ܪܝܫ ܡܠܐܟܐ ܗܘ ܕܥܡ ܐܟܠܩܪܨܐ ܟܕ ܕܐܢ
ܡܡܠܠ ܗܘܐ ܡܛܠ ܦܓܪܗ ܕܡܘܫܐ ܠܐ ܐܡܪܚ ܕܢܝܬܐ ܥܠܘܗܝ ܕܝܢܐ ܕܓܘܕܦܐ ܐܠܐ ܐܡܪ ܕܢܓܥܘܪ
ܒܟ ܡܪܝܐ ܀

<span id="an_1:10">1:10</span> ܗܠܝܢ ܕܝܢ ܒܐܝܠܝܢ ܕܠܐ ܝܕܥܝܢ ܡܓܕܦܝܢ ܒܐܝܠܝܢ
ܕܝܢ ܕܟܝܢܐܝܬ ܐܝܟ ܚܝܘܬܐ ܚܪܫܬܐ ܡܦܤܝܢ ܒܗܝܢ ܡܬܚܒܠܝܢ ܀

<span id="an_1:11">1:11</span> ܘܝ ܠܗܘܢ ܕܒܐܘܪܚܗ ܕܩܐܝܢ ܐܙܠܘ ܘܒܬܪ ܛܥܝܘܬܗ
ܕܒܠܥܡ ܒܐܓܪܐ ܐܫܬܪܚܘ ܘܒܥܨܝܝܘܬܗ ܕܩܘܪܚ ܐܒܕܘ ܀

<span id="an_1:12">1:12</span> ܗܠܝܢ ܐܝܬܝܗܘܢ ܐܝܠܝܢ ܕܒܢܝܚܬܗܘܢ ܟܕ ܡܟܬܡܝܢ
ܡܬܦܪܦܥܝܢ ܟܕ ܕܠܐ ܕܚܠܬܐ ܢܦܫܗܘܢ ܪܥܝܢ ܥܢܢܐ ܕܠܐ ܡܛܪܐ ܕܡܢ ܪܘܚܐ ܦܗܝܢ ܐܝܠܢܐ
ܕܐܘܦܝ ܐܒܗܘܢ ܕܐܝܬܝܗܘܢ ܕܠܐ ܦܐܪܐ ܕܡܝܬܘ ܬܢܝܢܘܬ ܘܤܠܩܘ ܡܢ ܥܩܪܝܗܘܢ ܀

<span id="an_1:13">1:13</span> ܓܠܠܐ ܥܙܝܙܐ ܕܝܡܐ ܕܒܝܕ ܪܘܥܬܗܘܢ ܡܚܘܝܢ
ܒܗܬܬܗܘܢ ܟܘܟܒܐ ܡܛܥܝܢܐ ܠܗܠܝܢ ܕܥܡܛܢܐ ܕܚܫܘܟܐ ܠܥܠܡ ܠܗܘܢ ܢܛܝܪ ܀

<span id="an_1:14">1:14</span> ܐܬܢܒܝ ܕܝܢ ܐܦ ܠܗܠܝܢ ܗܘ ܕܐܝܬܘܗܝ ܕܫܒܥܐ ܡܢ
ܐܕܡ ܚܢܘܟ ܟܕ ܐܡܪ ܕܗܐ ܡܪܝܐ ܐܬܐ ܒܪܒܘܬܐ ܕܩܕܝܫܐ ܀

<span id="an_1:15">1:15</span> ܕܢܥܒܕ ܕܝܢܐ ܥܠ ܟܠ ܘܠܡܟܤܘ ܠܟܠ ܢܦܫܬܐ ܡܛܠ
ܟܠܗܘܢ ܥܒܕܐ ܐܝܠܝܢ ܕܐܪܫܥܘ ܘܡܛܠ ܟܠܗܝܢ ܡܠܐ ܩܫܝܬܐ ܐܝܠܝܢ ܕܡܠܠܘ ܚܛܝܐ ܪܫܝܥܐ ܀

<span id="an_1:16">1:16</span> ܗܠܝܢ ܐܢܘܢ ܕܡܪܛܢܝܢ ܘܥܕܠܝܢ ܒܟܠ ܨܒܘܢ ܟܕ ܐܝܟ
ܪܓܝܓܬܐ ܕܝܠܗܘܢ ܡܗܠܟܝܢ ܘܦܘܡܗܘܢ ܡܡܠܠ ܓܢܝܚܬܐ ܘܡܫܒܚܝܢ ܠܦܪܨܘܦܐ ܡܛܠ ܝܘܬܪܢܐ ܀

<span id="an_1:17">1:17</span> ܐܢܬܘܢ ܕܝܢ ܚܒܝܒܝ ܐܬܕܟܪܘ ܠܡܠܐ ܐܝܠܝܢ ܕܩܕܡ
ܐܬܐܡܪ ܡܢ ܫܠܝܚܘܗܝ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:18">1:18</span> ܕܐܡܪܝܢ ܗܘܘ ܠܟܘܢ ܕܒܚܪܬܐ ܕܙܒܢܐ ܢܗܘܘܢ ܐܝܠܝܢ
ܕܡܒܙܚܝܢ ܕܐܝܟ ܪܓܝܓܬܐ ܕܝܠܗܘܢ ܐܙܠܝܢ ܒܬܪ ܪܘܫܥܐ ܀

<span id="an_1:19">1:19</span> ܗܠܝܢ ܐܝܬܝܗܘܢ ܐܝܠܝܢ ܕܡܦܪܫܝܢ ܢܦܫܢܝܐ ܕܪܘܚܐ
ܠܝܬ ܠܗܘܢ ܀

<span id="an_1:20">1:20</span> ܐܢܬܘܢ ܕܝܢ ܚܒܝܒܝ ܒܗܝܡܢܘܬܐ ܕܝܠܟܘܢ ܩܕܝܫܬܐ
ܐܬܒܢܘ ܡܢ ܕܪܝܫ ܒܪܘܚܐ ܩܕܝܫܐ ܟܕ ܡܨܠܝܢ ܀

<span id="an_1:21">1:21</span> ܢܦܫܢ ܕܝܢ ܒܚܘܒܐ ܕܐܠܗܐ ܢܛܪ ܟܕ ܡܤܟܝܢܢ ܠܚܢܢܗ
ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܠܚܝܐ ܕܝܠܢ ܕܠܥܠܡ ܀

<span id="an_1:22">1:22</span> ܘܠܡܢܗܘܢ ܡܢ ܡܢ ܢܘܪܐ ܚܛܘܦܘ ܀

<span id="an_1:23">1:23</span> ܟܕ ܕܝܢ ܡܬܬܘܝܢ ܐܬܪܚܡܘ ܥܠܝܗܘܢ ܒܕܚܠܬܐ ܟܕ
ܤܢܝܢ ܐܢܬܘܢ ܐܦ ܠܟܘܬܝܢܐ ܕܡܢ ܒܤܪܐ ܕܡܟܬܡܐ ܀

<span id="an_1:24">1:24</span> ܠܗܘ ܕܝܢ ܕܡܫܟܚ ܕܢܢܛܪ ܠܢ ܕܠܐ ܫܘܪܥܬܐ ܘܕܠܐ
ܟܘܬܡܐ ܘܢܩܝܡ ܕܠܐ ܡܘܡܐ ܀

<span id="an_1:25">1:25</span> ܒܠܚܘܕܘܗܝ ܐܠܗܐ ܦܪܘܩܢ ܒܝܕ ܝܫܘܥ ܡܫܝܚܐ ܡܪܢ
ܩܕܡ ܬܫܒܘܚܬܗ ܒܚܕܘܬܐ ܠܗ ܫܘܒܚܐ ܘܐܘܚܕܢܐ ܘܐܝܩܪܐ ܘܪܒܘܬܐ ܐܦ ܗܫܐ ܘܒܟܠܗܘܢ ܥܠܡܐ
ܐܡܝܢ ܀
