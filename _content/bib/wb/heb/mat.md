### Modern Hebrew Bible: Matthew

<span id="an_1:1">1:1</span> ספר תולדת ישוע המשיח בן דוד בן אברהם׃

<span id="an_1:2">1:2</span> אברהם הוליד את יצחק ויצחק הוליד את יעקב
ויעקב הוליד את יהודה ואת אחיו׃

<span id="an_1:3">1:3</span> ויהודה הוליד את פרץ ואת זרח מתמר ופרץ הוליד
את חצרון וחצרון הוליד את רם׃

<span id="an_1:4">1:4</span> ורם הוליד את עמינדב ועמינדב הוליד את נחשון
ונחשון הוליד את שלמון׃

<span id="an_1:5">1:5</span> ושלמון הוליד את בעז מרחב ובעז הוליד את עובד
מרות ועובד הוליד את ישי׃

<span id="an_1:6">1:6</span> וישי הוליד את דוד המלך ודוד המלך הוליד את
שלמה מאשת אוריה׃

<span id="an_1:7">1:7</span> ושלמה הוליד את רחבעם ורחבעם הוליד את אביה
ואביה הוליד את אסא׃

<span id="an_1:8">1:8</span> ואסא הוליד את יהושפט ויהושפט הוליד את יורם
ויורם הוליד את עזיהו׃

<span id="an_1:9">1:9</span> ועזיהו הוליד את יותם ויותם הוליד את אחז
ואחז הוליד את יחזקיהו׃

<span id="an_1:10">1:10</span> ויחזקיהו הוליד את מנשה ומנשה הוליד את
אמון ואמון הוליד את יאשיהו׃

<span id="an_1:11">1:11</span> ויאשיהו הוליד את יכניהו ואת אחיו לעת גלות
בבל׃

<span id="an_1:12">1:12</span> ואחרי גלותם בבלה הוליד יכניה את שאלתיאל
ושאלתיאל הוליד את זרבבל׃

<span id="an_1:13">1:13</span> וזרבבל הוליד את אביהוד ואביהוד הוליד את
אליקים ואליקים הוליד את עזור׃

<span id="an_1:14">1:14</span> ועזור הוליד את צדוק וצדוק הוליד את יכין
ויכין הוליד את אליהוד׃

<span id="an_1:15">1:15</span> ואליהוד הוליד את אלעזר ואלעזר הוליד את
מתן ומתן הוליד את יעקב׃

<span id="an_1:16">1:16</span> ויעקב הוליד את יוסף בעל מרים אשר ממנה
נולד ישוע הנקרא משיח׃

<span id="an_1:17">1:17</span> והנה כל הדרות מן אברהם עד דוד ארבעה עשר
דרות ומן דוד עד גלות בבל ארבעה עשר דרות ומעת גלות בבל עד המשיח ארבעה עשר
דרות׃

<span id="an_1:18">1:18</span> והלדת ישוע המשיח כה היתה מרים אמו ארשה
ליוסף ובטרם יבא אליה נמצאת הרה לרוח הקדש׃

<span id="an_1:19">1:19</span> ויוסף בעלה היה איש צדיק ולא אבה לתתה לבוז
ויאמר בלבו לשלחה בסתר׃

<span id="an_1:20">1:20</span> הוא חשב כזאת והנה מלאך יהוה נראה אליו
בחלום ויאמר יוסף בן דוד אל תירא מכנוס אליך את מרים אשתך כי אשר הרה בה
מרוח הקדש הוא׃

<span id="an_1:21">1:21</span> והיא ילדת בן וקראת את שמו ישוע כי הוא
יושיע את עמו מעונותיהם׃

<span id="an_1:22">1:22</span> ותהי כל זאת למלאת את אשר דבר יהוה ביד
הנביא לאמר׃

<span id="an_1:23">1:23</span> הנה העלמה הרה וילדת בן וקראו שמו עמנואל
אשר פרושו האל עמנו׃

<span id="an_1:24">1:24</span> וייקץ יוסף משנתו ויעש כאשר צוהו מלאך יהוה
ויקח אליו את אשתו׃

<span id="an_1:25">1:25</span> ולא ידעה עד כי ילדה בן את בכורה ויקרא את
שמו ישוע׃

<span id="an_2:1">2:1</span> ויהי כאשר נולד ישוע בבית לחם יהודה בימי
הורדוס המלך ויבאו מגושים מארץ מזרח ירושלימה לאמר׃

<span id="an_2:2">2:2</span> איה מלך היהודים הנולד כי ראינו את כוכבו
במזרח ונבא להשתחות לו׃

<span id="an_2:3">2:3</span> ויהי כשמע הורדוס המלך את דבריהם ויבהל הוא
וכל ירושלים עמו׃

<span id="an_2:4">2:4</span> ויקהל את כל ראשי הכהנים וסופרי העם וידרש
מאתם לאמר איפה יולד המשיח׃

<span id="an_2:5">2:5</span> ויאמרו לו בבית לחם יהודה כי כן כתוב ביד
הנביא׃

<span id="an_2:6">2:6</span> ואתה בית לחם ארץ יהודה אינך צעיר באלפי
יהודה כי ממך יצא מושל אשר ירעה את עמי ישראל׃

<span id="an_2:7">2:7</span> אז קרא הורדוס למגושים בסתר ויחקר אתם לדעת
עת הראות הכוכב׃

<span id="an_2:8">2:8</span> וישלחם בית לחם ויאמר לכו חקרו היטב על דבר
הילד והיה כי תמצאון אתו והגדתם לי ואבאה להשתחות לו גם אני׃

<span id="an_2:9">2:9</span> ויהי כשמעם את דברי המלך וילכו והנה הכוכב
אשר ראו במזרח דרך לפניהם עד באו אל מקום אשר שם הילד ויעמד ממעל לו׃

<span id="an_2:10">2:10</span> ויראו את הכוכב וישמחו שמחה גדולה עד מאד׃

<span id="an_2:11">2:11</span> ויבאו הביתה וימצאו את הילד עם מרים אמו
ויפל על פניהם וישתחוו לו ויפתחו את אוצרותם ויקריבו לו מנחה זהב ולבונה
ומר׃

<span id="an_2:12">2:12</span> ויצוו בחלום לבלתי שוב אל הורדוס וילכו
בדרך אחר אל ארצם׃

<span id="an_2:13">2:13</span> הם הלכו משם והנה מלאך יהוה נראה אל יוסף
בחלום לאמר קום קח את הילד ואת אמו וברח לך מצרימה והיה שם עד אשר אמר אליך
כי הורדוס צדה את הילד לאבדו׃

<span id="an_2:14">2:14</span> ויקם ויקח את הילד ואת אמו בלילה ויברח
מצרימה׃

<span id="an_2:15">2:15</span> ויהי שם עד מות הורדוס למלאת את אשר דבר
יהוה ביד הנביא לאמר ממצרים קראתי לבני׃

<span id="an_2:16">2:16</span> וירא הורדוס כי התלו בו המגושים ויקצף מאד
וישלח וימת את כל הילדים אשר בבית לחם ובכל גבוליה מבן שנתים ולמטה כפי העת
אשר חקרה מפי המגושים׃

<span id="an_2:17">2:17</span> וימלא הדבר הנאמר בפי ירמיה הנביא לאמר׃

<span id="an_2:18">2:18</span> קול ברמה נשמע נהי ובכי תמרורים רחל מבכה
על בניה מאנה להנחם כי אינם׃

<span id="an_2:19">2:19</span> ויהי אחרי מות הורדוס והנה מלאך יהוה נראה
בחלום אל יוסף בארץ מצרים׃

<span id="an_2:20">2:20</span> ויאמר אליו קום קח את הילד ואת אמו ולך שוב
אל ארץ ישראל כי מתו המבקשים את נפש הילד׃

<span id="an_2:21">2:21</span> ויקם ויקח את הילד ואת אמו ויבא ארצה
ישראל׃

<span id="an_2:22">2:22</span> וכשמעו כי ארקלוס מלך ביהודה תחת הורדוס
אביו ירא ללכת שמה ויצוה בחלום וילך לו אל ארצות הגליל׃

<span id="an_2:23">2:23</span> ויבא וישב בעיר אשר שמה נצרת למלאת הדבר
הנאמר על פי הנביאים כי נצרי יקרא לו׃

<span id="an_3:1">3:1</span> בימים ההם בא יוחנן המטביל ויהי קרא במדבר
יהודה לאמר׃

<span id="an_3:2">3:2</span> שובו כי מלכות השמים הגיעה׃

<span id="an_3:3">3:3</span> הלא זה הוא אשר נבא עליו ישעיהו הנביא לאמר
קול קורא במדבר פנו דרך יהוה ישרו מסלותיו׃

<span id="an_3:4">3:4</span> והוא יוחנן לבושו שער גמלים ואזור עור במתניו
ומאכלו חגבים ודבש היער׃

<span id="an_3:5">3:5</span> ויצאו אליו ישבי ירושלים וכל יהודה וכל ככר
הירדן׃

<span id="an_3:6">3:6</span> ויטבלו על ידו בירדן מתודים את חטאתם׃

<span id="an_3:7">3:7</span> ויהי כראותו רבים מן הפרושים והצדוקים נגשים
לטבילתו ויאמר להם ילדי הצפעונים מי השכיל אתכם להמלט מן הקצף הבא׃

<span id="an_3:8">3:8</span> לכן עשו פרי הראוי לתשובה׃

<span id="an_3:9">3:9</span> ואל תחשבו בלבבכם לאמר אברהם הוא אבינו כי
אני אמר לכם כי מן האבנים האלה יכול האלהים להקים בנים לאברהם׃

<span id="an_3:10">3:10</span> וכבר הושם הגרזן על שרש העצים והנה כל עץ
אשר איננו עשה פרי טוב יגדע וישלך באש׃

<span id="an_3:11">3:11</span> הן אנכי טובל אתכם במים לתשובה והבא אחרי
חזק הוא ממני אשר אינני כדי לשאת נעליו והוא יטבל אתכם ברוח הקדש ובאש׃

<span id="an_3:12">3:12</span> אשר בידו המזרה וזרה את גרנו ואסף את דגנו
אל אוצרו ואת המץ ישרפנו באש אשר לא תכבה׃

<span id="an_3:13">3:13</span> ויבא ישוע מן הגליל הירדנה אל יוחנן להטבל
על ידו׃

<span id="an_3:14">3:14</span> ויוחנן חשך אותו לאמר אני צריך להטבל על
ידך ואתה בא אלי׃

<span id="an_3:15">3:15</span> ויען ישוע ויאמר אליו הניחה לי כי כן נאוה
לנו למלא כל הצדקה וינח לו׃

<span id="an_3:16">3:16</span> ויהי כאשר נטבל ישוע וימהר לעלות מן המים
והנה נפתחו לו השמים וירא את רוח אלהים יורדת כיונה ונחה עליו׃

<span id="an_3:17">3:17</span> והנה קול מן השמים אומר זה בני ידידי אשר
רצתה נפשי בו׃

<span id="an_4:1">4:1</span> אז נשא הרוח את ישוע המדברה למען ינסהו השטן׃

<span id="an_4:2">4:2</span> ויהי אחרי צומו ארבעים יום וארבעים לילה
וירעב׃

<span id="an_4:3">4:3</span> ויגש אליו המנסה ויאמר אם בן האלהים אתה אמר
לאבנים האלה ותהיין ללחם׃

<span id="an_4:4">4:4</span> ויען ויאמר הן כתוב לא על הלחם לבדו יחיה
האדם כי על כל מוצא פי יי׃

<span id="an_4:5">4:5</span> וישאהו השטן אל עיר הקדש ויעמידהו על פנת בית
המקדש׃

<span id="an_4:6">4:6</span> ויאמר אליו אם בן האלהים אתה התנפל למטה כי
כתוב כי מלאכיו יצוה לך ועל כפים ישאונך פן תגף באבן רגלך׃

<span id="an_4:7">4:7</span> ויאמר אליו ישוע ועוד כתוב לא תנסה את יהוה
אלהיך׃

<span id="an_4:8">4:8</span> ויוסף השטן וישאהו אל הר גבה מאד ויראהו את
כל ממלכות תבל וכבודן׃

<span id="an_4:9">4:9</span> ויאמר אליו את כל זאת לך אתננה אם תקד
ותשתחוה לי׃

<span id="an_4:10">4:10</span> ויאמר אליו ישוע סור ממני השטן כי כתוב
ליהוה אלהיך תשתחוה ואותו לבדו תעבד׃

<span id="an_4:11">4:11</span> וירף ממנו השטן והנה נגשו אליו מלאכים
וישרתוהו׃

<span id="an_4:12">4:12</span> ויהי כשמעו כי הסגירו את יוחנן וילך לו ארץ
הגליל׃

<span id="an_4:13">4:13</span> ויעזב את נצרת ויבא וישב בכפר נחום אשר על
שפת הים בגבול זבלון ונפתלי׃

<span id="an_4:14">4:14</span> למלאת הנאמר על פי ישעיהו הנביא לאמר׃

<span id="an_4:15">4:15</span> ארצה זבלון וארצה נפתלי דרך הים עבר הירדן
גליל הגוים׃

<span id="an_4:16">4:16</span> העם ההלכים בחשך ראו אור גדול וישבי בארץ
צלמות אור נגה עליהם׃

<span id="an_4:17">4:17</span> מן העת ההיא החל ישוע לקרא קרוא ואמור שובו
כי הגיעה מלכות השמים׃

<span id="an_4:18">4:18</span> ויהי בהתהלכו על יד ים הגליל וירא שני
אנשים אחים את שמעון הנקרא פטרוס ואת אנדרי אחיו והמה משליכים מצודה בים כי
דיגים היו׃

<span id="an_4:19">4:19</span> ויאמר אליהם לכו אחרי ואשימכם לדיגי אנשים׃

<span id="an_4:20">4:20</span> ויעזבו מהרה את המכמרות וילכו אחריו׃

<span id="an_4:21">4:21</span> ויהי כעברו משם וירא שני אנשים אחים אחרים
את יעקב בן זבדי ואת יוחנן אחיו באניה עם זבדי אביהם מתקנים את מכמרותם
ויקרא אליהם׃

<span id="an_4:22">4:22</span> ויעזבו מיד את האניה ואת אביהם וילכו
אחריו׃

<span id="an_4:23">4:23</span> ויסב ישוע בכל הגליל וילמד בבתי כנסיותיהם
ויבשר בשורת המלכות וירפא כל מחלה וכל מדוה בעם׃

<span id="an_4:24">4:24</span> ויצא שמעו בכל ארץ סוריא ויביאו אליו את כל
החולים המענים בכל חלים ומכאובים ואחוזי שדים ומכי ירח ונכי אברים וירפאם׃

<span id="an_4:25">4:25</span> וילכו אחריו המנים המנים מן הגליל ומן עשר
הערים ומירושלים ויהודה ומעבר לירדן׃

<span id="an_5:1">5:1</span> ויהי כראותו את המון העם ויעל ההרה וישב שם
ויגשו אליו תלמידיו׃

<span id="an_5:2">5:2</span> ויפתח את פיהו וילמד אותם לאמר׃

<span id="an_5:3">5:3</span> אשרי עניי רוח כי להם מלכות השמים׃

<span id="an_5:4">5:4</span> אשרי האבלים כי הם ינחמו׃

<span id="an_5:5">5:5</span> אשרי הענוים כי המה יירשו הארץ׃

<span id="an_5:6">5:6</span> אשרי הרעבים והצמאים לצדקה כי הם ישבעו׃

<span id="an_5:7">5:7</span> אשרי הרחמנים כי הם ירחמו׃

<span id="an_5:8">5:8</span> אשרי ברי לבב כי הם יחזו את האלהים׃

<span id="an_5:9">5:9</span> אשרי רדפי שלום כי בני אלהים יקראו׃

<span id="an_5:10">5:10</span> אשרי הנרדפים על דבר הצדקה כי להם מלכות
השמים׃

<span id="an_5:11">5:11</span> אשריכם אם יחרפו וירדפו אתכם וידברו בשקר
עליכם כל רע בעבורי׃

<span id="an_5:12">5:12</span> שמחו וגילו כי שכרכם רב בשמים כי כן רדפו
