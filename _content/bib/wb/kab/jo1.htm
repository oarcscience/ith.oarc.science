Kabyle Bible: 1 John

1:1 Ɣef wayen yeɛnan win yellan d awal n tudert, i gellan si tazwara : nesla-yas yerna neẓra-t s wallen-nneɣ, nnulent ifassen-nneɣ yerna nerra ddehn-nneɣ ɣuṛ-es.

1:2 Tudert tban-ed, nwala-ț, nețcehhid fell-as, yerna nețbecciṛ-awen-d tudert n dayem yellan ɣer tama n Sidi Ṛebbi, tura tban-aɣ-ed.

1:3 Ayen nwala d wayen nesla nxebbeṛ-awen-t-id, iwakken ula d kunwi aț-țekkim di tdukkli-nneɣ, akken nella nukni d Baba-tneɣ Ṛebbi akk-d Mmi-s, Ɛisa Lmasiḥ.

1:4 Uriɣ-awen ayagi iwakken lfeṛḥ nneɣ ad innekmal.

1:5 Ayen nesla s ɣuṛ Ɛisa Lmasiḥ newwi-yawen-t-id : Sidi Ṛebbi ț-țafat, ur yelli ara deg-s ṭṭlam.

1:6 Ma neqqaṛ deg webrid n Ṛebbi i nella yili mazal-aɣ nteddu di ṭṭlam, neskaddeb nexḍa i webrid n tideț.

1:7 Meɛna ma yella nteddu di tafat am akken i gella nețța di tafat, ad ddukklen wulawen-nneɣ, yerna ddnubat-nneɣ ad iriden s idammen n Mmi-s Ɛisa Lmasiḥ.

1:8 Ma neqqaṛ ur yelli deg-nneɣ ddnub, atan neskaddeb ɣef yiman-nneɣ, tideț ur telli deg-nneɣ.

1:9 Meɛna ma nețqiṛṛi s ddnubat-nneɣ i Sidi Ṛebbi, nețța yellan d aḥeqqi, nezmer a nețkel fell-as a ɣ-isameḥ ddnubat-nneɣ, a ɣ-isellek si lbaṭel.

1:10 Ma neqqaṛ mačči d imednuben i nella, nerra Sidi Ṛebbi d akeddab, awal-is ur yelli ara deg-nneɣ.

2:1 Ay arraw-iw imeẓyanen, uriɣ awen ayagi iwakken aț-țexḍum i ddnub, meɛna ma yeɣli yiwen di ddnub, nesɛa win yețḥuddun fell-aɣ ɣer Baba Ṛebbi, d Ɛisa Lmasiḥ yellan d aḥeqqi.

2:2 Mačči ɣef ddemma n ddnubat nneɣ kan i gefka iman-is d asfel meɛna ula ɣef ddemma n ddnubat n yemdanen n ddunit meṛṛa.

2:3 Ihi ma nețțaɣ awal i Sidi Ṛebbi yerna nḥerrez lumuṛat-is, d ayagi ara d-isbeggnen belli nessen-it.

2:4 Win ara yinin : ssneɣ Ṛebbi yili ur yeṭṭif ara di lumuṛat-is, d akeddab, ulac deg-s tideț.

2:5 Ma d win yețțaɣen awal-is, iḥemmel Sidi Ṛebbi seg-wul-is. S wakka ara d-iban belli d ayla n Sidi Ṛebbi i nella.

2:6 Win ara yinin ṭṭfeɣ di Sidi Ṛebbi, ilaq ad ilḥu am akken yelḥa Lmasiḥ.

2:7 A wid eɛzizen, di tebṛaț-agi mačči d lameṛ ajdid i wen-d-wwiɣ, meɛna d lexbaṛ-nni i kkun-id-iwḍen si tazwara ; lameṛ-agi d awal-nni i wumi teslam yakan.

2:8 Meɛna zemreɣ a d-iniɣ belli lameṛ-agi d ajdid. Tideț-agi tban-ed di Lmasiḥ, tban-ed daɣen deg-wen ; imi ṭṭlam ițwexxiṛ, tafat n tideț tfeǧǧeǧ yakan.

2:9 Win yeqqaṛen leḥḥuɣ di tafat, yili nețța yekṛeh gma-s, mazal-it di ṭṭlam.

2:10 Win iḥemmlen gma-s, di tafat i gețțili, ur yelli ara deg-s wayen ara t-isseɣlin.

2:11 Ma d win ikeṛhen gma-s, di ṭṭlam i gella ; iteddu di ṭṭlam ur yeẓri anda ileḥḥu, axaṭer ṭṭlam isderɣel-as allen.

2:12 Uriɣ-awen ayagi ay ilmeẓyen imi i wen-țwaɛfan ddnubat-nwen ɣef ddemma n yisem-is.

2:13 Uriɣ-awen ay ibabaten, axaṭer tesnem win yellan si tazwara. Uriɣ-awen ay ilmeẓyen, axaṭer tɣelbem amcum. Akken i wen-d-nniɣ yakan ay ilmeẓyen, uriɣ-awen axaṭer tesnem Baba Ṛebbi.

2:14 Uriɣ-awen ay ibabaten, axaṭer tesnem win yellan si tazwara. Uriɣ-awen ay ilmeẓyen, axaṭer tǧehdem, awal n Sidi Ṛebbi yezdeɣ deg-wen yerna tɣelbem amcum.

2:15 Ur ḥemmlet ara ddunit d wayen yellan deg-s. Ma yella walebɛaḍ ihemmel ddunit, leḥmala n baba Ṛebbi ur telli ara deg-s ;

2:16 axaṭer ayen akk yellan di ddunit, lebɣi n tnefsit, ṭṭmeɛ di kra n wayen i gɛeǧben i wallen akk-d zzux i d ițekken si cci n ddunit, mačči s ɣuṛ Baba Ṛebbi i d-kkan lameɛna si ddunit.

2:17 Ddunit d wayen yellan deg-s ad iɛeddi, kullec ad ikfu ; ma d win ixeddmen lebɣi n Sidi Ṛebbi ad yidir i dayem.

2:18 Ay arrac imeẓyanen, ț-țaswiɛt taneggarut ; am akken teslam belli aɛdaw n Lmasiḥ iteddu-d, atan tura aṭas n yeɛdawen n Lmasiḥ i d-ibanen ; s wakka ara neɛqel belli d lweqt aneggaru.

2:19 Sṭaxṛen fell-aɣ axaṭer ur llin ara seg-neɣ ; lemmer llin seg-neɣ, tili qqimen yid-nneɣ ; ṭṭaxṛen fell-aneɣ iwakken a d-iban belli ur llin ara seg-neɣ.

2:20 Ma d kunwi, yesmar-ed fell awen Lmasiḥ Ṛṛuḥ iqedsen, yerna tessnem akk tideț.

2:21 Uriɣ-awen mačči imi ur tessinem ara tideț, lameɛna imi ț-tesnem axaṭer ulac lekdeb i d itekken si tideț.

2:22 Anwa i d akeddab anagar win inekkṛen belli Ɛisa d Lmasiḥ. Aɛdaw n Lmasiḥ, d winna i gnekkṛen Baba Ṛebbi akk-d Mmi-s.

2:23 Win ara inekṛen Mmi-s n Ṛebbi yenkeṛ daɣen Baba Ṛebbi, ma d win iqeblen Mmi-s n Ṛebbi yeqbel Baba Ṛebbi.

2:24 Ma d kunwi, ayen teslam si tazwara ilaq aț-țeṭṭfem deg-s ; ma yella tjemɛem-t deg ulawen-nwen, ula d kunwi aț-țilim di tdukkli akk-d Baba Ṛebbi d Mmi-s.

2:25 Atan lweɛd i ɣ-d-ifka Lmasiḥ : ț-țudert yețdumun.

2:26 Uriɣ-awen ayagi ɣef wid yețnadin a kkun-ɣuṛṛen.

2:27 Ma d kunwi, Ṛṛuḥ iqedsen i d-yesmar fell-awen Lmasiḥ yezga deg-wen, ur teḥwaǧem ara win ara kkun-islemden ; axaṭer d Ṛṛuḥ iqedsen i d-immaren fell-awen i kkun-isselmaden. Ayen i wen-yesselmad ț-țideț, ur yeskiddib ara. Sṭfet ihi di Lmasiḥ am akken i wen-t-isselmed Ṛṛuḥ iqedsen.

2:28 Ihi tura ay arrac imeẓyanen ṭṭfet di Lmasiḥ iwakken asm'ara d-iban a nesɛu lețkal, ur nțeffer ara fell-as si lḥecma ass n tuɣalin-is.

2:29 Imi teẓram belli Lmasiḥ d aḥeqqi, ilaq aț-țeẓrem daɣen kra n win ixeddmen lḥeqq s ɣuṛ-es i d-ikka.

3:1 Walit acḥal i ɣ-iḥemmel Baba Ṛebbi, armi nețțusemma d arraw-is ! yerna ț-țideț d arraw-is i nella. Ddunit ur aɣ-teɛqil ara d arraw n Ṛebbi imi ur tessin ara Sidi Ṛebbi.

3:2 A wid eɛzizen, aql-aɣ tura d arraw n Ṛebbi, ayen akken ara nuɣal werɛad i d-yedṛi, lameɛna neẓra belli m'ara d-yedṛu, a nili am nețța, axaṭer a t-nwali akken yella.

3:3 Kra n win yessaramen deg-s ad yeṣfu akken yeṣfa Lmasiḥ.

3:4 Kra n win idenben yerẓa ccariɛa n Sidi Ṛebbi, axaṭer ddnub ț-țaṛuẓi n ccariɛa.

3:5 Teẓram belli Lmasiḥ yusa-d iwakken ad ikkes ddnub nețța ur nesɛi ddnub.

3:6 Kra n win yeṭṭfen deg-s ixeṭṭu i ddnub ; kra n win yețɛicin di ddnub ur t-iwala ur t-yessin.

3:7 Ay arrac imeẓyanen, ɣuṛ-wat win ara kkun-iɣuṛṛen. Win iteddun s lḥeqq d aḥeqqi akken yella Sidna Ɛisa d aḥeqqi.

3:8 Win yețɛicin di ddnub yedda d Cciṭan, axaṭer d nețța i d amednub amezwaru. Ɣef ddemma n wayagi i d-yusa Mmi-s n Ṛebbi iwakken ad yessenger lecɣal n Cciṭan.

3:9 Kra n wid yellan d arraw n Ṛebbi ur țɛicin ara di ddnub, axaṭer tudert i d-itekken s ɣuṛ Sidi Ṛebbi tella deg-sen ; ur zmiren ara ad ɛicen di ddnub imi s ɣuṛ Ṛebbi i d-ɛawden talalit.

3:10 Atah wayen i d-ițbegginen lxilaf yellan ger warraw n Sidi Ṛebbi d wid n Cciṭan : kra n win ur nteddu ara s lḥeqq neɣ ur nḥemmel ara gma-s ur yelli ara si tarwa n Sidi Ṛebbi.

3:11 Atan lameṛ i wumi teslam si tazwara : ilaq a nemyeḥmal wway gar-aneɣ.

3:12 Mačči am Kahin yellan d amcum yenɣa gma-s, yezla-t axaṭer nețța lecɣal-is diri-ten, ma d gma-s iteddu s lḥeqq. f+

3:13 Ur wehhmet ara ay atmaten ma yella keṛhen-kkun wat ddunit.

3:14 Nukni nɛedda-d si lmut ɣer tudert axaṭer nḥemmel atmaten. Win ur nḥemmel ara, mazal-it di lmut.

3:15 Kra n win ikeṛhen gma-s d aqettal, yerna teẓram belli win ineqqen, ulac deg-s tudert n dayem.

3:16 Leḥmala n tideț, nessen-iț s Ɛisa Lmasiḥ yefkan tudert-is fell-aneɣ. Ilaq-aɣ ula d nukni, a nefk tudert nneɣ ɣef watmaten-nneɣ.

3:17 Ma yella walebɛaḍ yesɛan cci, iwala gma-s di lexṣaṣ ur d-iḥunn ara fell-as, amek ara tezdeɣ leḥmala n Sidi Ṛebbi deg wul-is.

3:18 Ay arrac imeẓyanen, ur ḥemmlet ara s wawal kan d yiles aẓidan, lameɛna leḥmala n tideț a d-tban s lecɣal-nwen.

3:19 S wakka ara nẓer belli di tideț i nella, yerna ad yethedden wul-nneɣ zdat Sidi Ṛebbi,

3:20 ɣas yesseḍlem-aɣ wul-nneɣ : Sidi Ṛebbi sennig wul nneɣ i gella yerna yeẓra kullec.

3:21 A wid eɛzizen, ma yella ur aɣ-isseḍlem ara wul-nneɣ, nesɛa lețkal zdat Sidi Ṛebbi.

3:22 Kra n wayen ara nessuter, ițțak-aɣ-t-id axaṭer neḥrez lumuṛat-is yerna nxeddem lebɣi-s.

3:23 Atan wayen i ɣ-d-yumeṛ : a namen s yisem n Mmi-s Ɛisa Lmasiḥ yerna a nemyeḥmal wway gar-aneɣ am akken yella di lameṛ i ɣ-d-yefka. f+

3:24 Win yețțaṭṭafen di Sidi Ṛebbi, iḥerzen lumuṛat-is, Sidi Ṛebbi yețțili deg wul-is ; yerna a neɛqel belli Sidi Ṛebbi yella deg ul-nneɣ s Ṛṛuḥ iqedsen i ɣ-d-yefka.

4:1 A wid eɛzizen, ḥadret iman-nwen ɣef kra n win ara wen-d-yinin heddṛeɣ s Ṛṛuḥ n Sidi Ṛebbi, lameɛna jeṛṛbet-țen iwakken aț-țeẓrem ma yella s ɣuṛ Sidi Ṛebbi i d-kkan axaṭer, aṭas n lenbiya n lekdeb i gellan di ddunit.

4:2 Atan wamek ara tɛeqlem ma yella yiwen yesɛa Ṛṛuḥ n Sidi Ṛebbi : kra n win ara icehden belli Ɛisa Lmasiḥ yusa-d ɣer ddunit s ṣṣifa n wemdan, s ɣuṛ Sidi Ṛebbi i d-yusa.

4:3 Ma d win ara inekkṛen ayagi ɣef Sidna Ɛisa, mačči s ɣuṛ Ṛebbi i d-yekka lameɛna yekka-d si Ṛṛuḥ n weɛdaw n Lmasiḥ i ɣef teslam a d-yas ɣer ddunit, atan tura yella yakan di ddunit.

4:4 Kunwi ay arrac imeẓyanen, d arraw n Sidi Ṛebbi i tellam, tɣelbem wid yesselmaden lekdeb, axaṭer Ṛṛuḥ yellan deg-wen yugar win yellan di ddunit.

4:5 Nutni n ddunit, daymi i țmeslayen s lɛeqliya n wat ddunit, yerna at ddunit smeḥsisen-asen.

4:6 Nukni d ayla n Sidi Ṛebbi ; win yebɣan ad yissin Sidi Ṛebbi yesmeḥsis-aɣ-d ; win ur nelli ara n Sidi Ṛebbi ur a ɣ-d-ismeḥsis ara : s wakka ara neɛqel Ṛṛuḥ n tideț akk-d Ṛṛuḥ n lekdeb.

4:7 A wid eɛzizen, ilaq a nemyeḥmal wway gar-aneɣ axaṭer Sidi Ṛebbi s yiman-is, d leḥmala ; kra n wid yesɛan leḥmala deg ulawen-nsen, d arraw n Sidi Ṛebbi i llan yerna ssnen Sidi Ṛebbi.

4:8 Kra n win ur nesɛi ara leḥmala deg wul-is ur yessin ara Sidi Ṛebbi, axaṭer leḥmala s ɣuṛ-es i d-tekk.

4:9 Atah wamek i d-isban Sidi Ṛebbi belli iḥemmel-aɣ : Sidi Ṛebbi iceggeɛ-ed Mmi-s awḥid ɣer ddunit iwakken yis a nesɛu tudert.

4:10 Ațah d acu i d leḥmala : mačči d nukni i gḥemmlen Sidi Ṛebbi lameɛna d nețța i ɣ-iḥemmlen armi i d iceggeɛ Mmi-s ad yefk iman-is d asfel i leɛfu n ddnubat-nneɣ.

4:11 A wid eɛzizen, ma yella Sidi Ṛebbi iḥemmel-aɣ annect-agi, ilaq ula d nukni a nemyeḥmal wway gar-aneɣ.

4:12 Ulac win iwalan Sidi Ṛebbi s wallen-is, meɛna ma yella nemyeḥmal wway gar-aneɣ, Baba Ṛebbi yețțili deg-nneɣ, leḥmala-s tețban-ed deg-nneɣ.

4:13 Atah wamek ara neɛqel nukni nețțili deg-s nețța yețțili deg-nneɣ : imi i ɣ-d-yefka Ṛṛuḥ-ines.

4:14 Nukni s yiman-nneɣ netɛeǧǧeb deg-s, daymi nezmer a d-neched belli Baba-tneɣ Ṛebbi iceggeɛ-ed Mmi-s, i leslak n ddunit.

4:15 Kra n win ara icehden belli Ɛisa d Mmi-s n Ṛebbi, Ṛebbi ad yețțili deg wul-is, nețța ad iṭṭef di Sidi Ṛebbi.

4:16 Nukni nessen leḥmala i ɣ-iḥemmel Sidi Ṛebbi, yerna numen yis. Sidi Ṛebbi, d leḥmala ; win yesɛan leḥmala deg wul-is yeṭṭef di Sidi Ṛebbi, Sidi Ṛebbi yezdeɣ deg ul-is.

4:17 Ma yella tennekmal leḥmala deg ulawen-nneɣ, a nesɛu lețkal i wass n lḥisab, axaṭer ayen yețɛeddin fell-aɣ di ddunit-a, d ayen yesɛedda Lmasiḥ s yiman-is.

4:18 Win yesɛan leḥmala n tideț ur t-tkeččem ara tugdi ; leḥmala n tideț tessufuɣ tugdi, axaṭer tugdi tețțas-ed si lxuf n lɛiqab, win yețɛicin di tugdi werɛad yessaweḍ ɣer leḥmala n tideț.

4:19 Ma d nukni, nḥemmel Sidi Ṛebbi axaṭer d nețța i ɣ-iḥemmlen d amezwaru.

4:20 Kra n win yeqqaṛen : ḥemmleɣ Sidi Ṛebbi yili yekṛeh gma-s, d akeddab, axaṭer win ur nḥemmel ara gma-s i gețwali, ur izmir ara ad iḥemmel Sidi Ṛebbi ur yețwali ara.

4:21 Atan ihi lameṛ i ɣ-d-iǧǧa Lmasiḥ : win iḥemmlen Sidi Ṛebbi ad iḥemmel daɣen gma-s. f+

5:1 Kra n wid yumnen belli d Ɛisa i d Lmasiḥ, uɣalen d arraw n Ṛebbi ; daɣen wid iḥemmlen Baba Ṛebbi, ḥemmlen ula d arraw-is.

5:2 Atah wamek ara nẓer belli nḥemmel arraw n Ṛebbi : ma nḥemmel Sidi Ṛebbi yerna nxeddem lumuṛat-is.

5:3 Axaṭer ma nḥemmel Sidi Ṛebbi ilaq a neḥrez lumuṛat-is ; lumuṛat-is ur ẓẓayit ara,

5:4 axaṭer arraw n Sidi Ṛebbi ɣelben ddunit, ayen s wacu i neɣleb ddunit, d liman-nneɣ di Sidna Ɛisa.

5:5 Anwa i gɣelben ddunit ? D wid yumnen belli Ɛisa d Mmi-s n Ṛebbi.

5:6 Ɛisa Lmasiḥ, d win akken i d-yusan ɣer ddunit s waman d idammen, mačči kan s waman n weɣḍas-is lameɛna s waman akk-d idammen-is yuzzlen mi gemmut fell-aneɣ ; yerna d Ṛṛuḥ iqedsen i d-icehden fell-as axaṭer Ṛṛuḥ iqedsen, d Ṛṛuḥ n tideț.

5:7  Ṛṛuḥ iqedsen, aman d idammen, di tlata yid-sen beggnen-d tideț-agi.

5:9 Ma yella cchada n yemdanen tețwaqbel, cchada n Sidi Ṛebbi tugar tin n yemdanen ; axaṭer d Sidi Ṛebbi s yiman-is i d-icehden ɣef Mmi-s.

5:10 Win yumnen s Mmi-s n Ṛebbi, yesɛa țbut deg wul-is, ma d win ur numin ara, yerra Sidi Ṛebbi d akeddab imi ur yumin ara s wayen i d-iched Ṛebbi ɣef Mmi-s.

5:11 Cchada-yagi ț-țudert n dayem i ɣ-d-yefka Sidi Ṛebbi, tudert-agi di Mmi-s i tella.

5:12 Win yumnen s Mmi-s n Ṛebbi yesɛa tudert ; win ur numin ara s Mmi-s n Ṛebbi ur yesɛi ara tudert.

5:13 Ayagi uriɣ-awen-t-id iwakken aț-țeẓrem belli tesɛam tudert n ṣṣeḥ yețdumun, kunwi yumnen s Mmi-s n Ṛebbi.

5:14 Tura atah lețkal i nesɛa ɣuṛ-es : ma nessuter-as kra yellan di lebɣi-s, yețḥessis-aɣ-d ;

5:15 ma yella neẓra belli yețḥessis-aɣ-d, kra n wayen ara s nessuter, neḥsa belli yewweḍ-aɣ-d.

5:16 Ma yella win yeẓran gma-s yedneb yili ddnub-is ur yețțawi ara ɣer lmut, ilaq ad yedɛu ɣer Sidi Ṛebbi, nețța a s-d-yefk tudert ; nniɣ-ed ayagi ɣef wid ixeddmen ddnub ur nețțawi ara ɣer lmut. Yella ddnub yețțawin ɣer lmut lameɛna mačči ɣef wid ixeddmen ddnub-agi i wen-d-nniɣ aț-țedɛum.

5:17 Kra yellan d lbaṭel, d ddnub, meɛna llan ddnubat ur nețțawi ara ɣer lmut.

5:18 Neẓra belli kra n wid yellan d arraw n Sidi Ṛebbi ur țɛicin ara di ddnub ; meɛna Mmi-s n Ṛebbi yețɛassa fell-asen yerna Cciṭan ur yezmir ara a ten-yennal.

5:19 Neẓra belli d arraw n Sidi Ṛebbi i nella, ma d ddunit teɣli ger ifassen n wemcum.

5:20 Neẓra daɣen Mmi-s n Ṛebbi yusa-d ɣer ddunit yefka-yaɣ-d lefhama s wayes ara nissin Ṛebbi n țideț. Deg-s i neṭṭef s yisem n Mmi-s Ɛisa Lmasiḥ ; d nețța i d Ṛebbi n ṣṣeḥ, i ț-țudert yețdumun.

5:21 Ay ilmeẓyen, ḥadret aț-țɛebdem iṛebbiten n lekdeb.
