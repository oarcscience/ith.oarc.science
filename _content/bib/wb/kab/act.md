### Kabyle Bible: Acts

<span id="an_1:1">1:1</span> A Tawfilus, di tektabt-inu tamezwarut,
ḥkiɣ-ed ayen ixdem Sidna Ɛisa d wayen yesselmed seg wass amezwaru armi d
ass i deg yețwarfed ɣer igenni.

<span id="an_1:2">1:2</span> Uqbel ad ițwarfed, yefka lumuṛat-is s Ṛṛuḥ
iqedsen i wid i gextaṛ ad ilin d ṛṛusul.

<span id="an_1:3">1:3</span> Mi d-iḥya si ger lmegtin, i nutni daɣen
iwumi i d-isban iman-is belli yedder ; ṛebɛin wussan nețța ițban asen-d
yerna ițmeslay-asen-d ɣef wayen yeɛnan tagelda n Sidi Ṛebbi.

<span id="an_1:4">1:4</span> Yiwen wass, mi gella itețț yid-sen,
yefka-yasen lameṛ-agi : Ur țeffɣet ara si temdint n Lquds, lameɛna eṛǧut
lemɛahda n Baba Ṛebbi, tin akken i wen-d nniɣ ;

<span id="an_1:5">1:5</span> « Yeḥya isseɣḍas imdanen deg aman, ma d
kunwi, di kra n wussan aț-țețwaɣeḍsem s Ṛṛuḥ iqedsen.»

<span id="an_1:6">1:6</span> Mi llan ṛṛusul nnejmaɛen yid-es, steqsan-t
: A Sidi, deg ussan-agi ara d terreḍ tageldit i wegdud n wat Isṛail ?

<span id="an_1:7">1:7</span> Yerra-yasen : Ur țnadit ara aț-țeẓrem ayen
yeɛnan lewqat d wussan i ghegga Baba Ṛebbi s lḥekma-ines si zik, ayagi
mačči d ccɣel-nwen.

<span id="an_1:8">1:8</span> Kunwi a wen-d-tețțunefk tezmert n Ṛṛuḥ
iqedsen ara d-yersen fell-awen, imiren aț-țilim d inigan-iw di temdint n
Lquds, di tmurt n Yahuda ț-țmurt n Samarya, aț-țawḍem alamma d ixfawen n
ddunit.

<span id="an_1:9">1:9</span> Akken i d-yenna imeslayen-agi, Sidna Ɛisa
yețwarfed ɣer igenni nutni țmuqulen deg-s armi i t-tɣumm tagut.

<span id="an_1:10">1:10</span> Mazal ṛeṣṣan allen-nsen deg-s ɣer tegnaw,
mi sen-d-dehṛen sin yergazen s llebsa tamellalt, nnan-asen :

<span id="an_1:11">1:11</span> Ay imezdaɣ n tmurt n Jlili, acuɣeṛ
mazal-ikkun tețmuqqulem ɣer tegnaw ? Akken i twalam Sidna Ɛisa yuli ɣer
igenni, akken daɣen ara t-twalim asmi ara d-yuɣal.

<span id="an_1:12">1:12</span> Seg iɣil uzemmur, uɣalen ɣer temdint n
Lquds ibeɛden azal n tikli n wass n westeɛfu.

<span id="an_1:13">1:13</span> Mi wwḍen ɣer temdint n Lquds, ulin ɣer
texxamt anda nnumen țnejmaɛen. Yella gar-asen Buṭrus, Yuḥenna, Yeɛqub,
Andriyus, Filibas, Suma, Bartelmay, Matta, Yeɛqub mmi-s n Ḥalfi, Semɛun
awaṭani akk-d Yahuda mmi-s n Yeɛqub.

<span id="an_1:14">1:14</span> Țnejmaɛen s yiwen n ṛṛay iwakken ad
deɛɛun ɣer Sidi Ṛebbi nutni d kra n tlawin, llan yid-sen daɣen watmaten
n Sidna Ɛisa akk-d yemma-s, Meryem.

<span id="an_1:15">1:15</span> Deg ussan-nni, llan nnejmaɛen azal n
meyya uɛecrin n wayetmaten, yekker-ed Buṭrus gar-asen, yenna-d :

<span id="an_1:16">1:16</span> Ay atmaten, ilaq a d-yedṛu wayen i
d-yenna Ṛṛuḥ iqedsen si zik seg yimi n Sidna Dawed, ɣef wayen yeɛnan
Yudas i d-yezwaren zdat n wid i geṭṭfen Sidna Ɛisa.

<span id="an_1:17">1:17</span> Yudas-agi yella d yiwen seg neɣ, yerna
ițekki di leqdic-nneɣ ɣef Sidi Ṛebbi.

<span id="an_1:18">1:18</span> Argaz-agi yuɣ akal s yedrimen-nni n
lexdeɛ, yuɣal yeɣli ɣef wudem, ifelleq uɛebbuḍ-is, mmaren-d akk
ijuɣdan-is.

<span id="an_1:19">1:19</span> Imezdaɣ n temdint n Lquds slan akk s
wannect-agi, daymi i semman i wemkan-nni s tmeslayt-nsen : Ḥaqel damma ,
yeɛni iger n idammen.

<span id="an_1:20">1:20</span> Am akken yura di tektabt n Sabur :
Axxam-is ad yexlu, ur t-izeddeɣ yiwen. Yura daɣen : Wayeḍ ad yeṭṭef
amkan-is.

<span id="an_1:21">1:21</span> Ilaq-aɣ ihi a d-nextiṛ yiwen seg wid
yellan yid-nneɣ mi nțeddu d Sidna Ɛisa di mkul amkan,

<span id="an_1:22">1:22</span> seg wasmi i t-isseɣḍes Yeḥya aɣeṭṭas armi
d ass i deg yețwarfed ɣer igenni ; iwakken ad yili am nukkni d inigi n
ḥeggu-ines.

<span id="an_1:23">1:23</span> Sbedden-d sin yergazen : Yusef
yețțusemman Barsaba iwumi qqaṛen daɣen « Aḥeqqi » akk-d Matyas.

<span id="an_1:24">1:24</span> Dɛan ɣer Sidi Ṛebbi s imeslayen agi : A
Sidi Ṛebbi, kečč yesnen ulawen n yemdanen meṛṛa, sbeggen aɣ-ed anwa
deg-sen i gwulmen

<span id="an_1:25">1:25</span> iwakken ad iṭṭef amkan di lecɣal n ṛṛusul
i ɣef yeṭṭaxeṛ Yudas, imi nețța iṛuḥ ɣer wemkan i s-ilaqen.

<span id="an_1:26">1:26</span> Gren tasɣaṛt, tṣaḥ-ed i Matyas ; dɣa
rnan-t-id ɣer ḥdac-nni n ṛṛusul.

<span id="an_2:1">2:1</span> Ass n Lɛid n wass wis xemsin, inelmaden
llan nnejmaɛen akk deg yiwen n wemkan.

<span id="an_2:2">2:2</span> Taswiɛt kan, yekka-d yiwen n lḥess ameqqran
seg igenni am waḍu iǧehden, yeččuṛ akk axxam-nni anda qqimen.

<span id="an_2:3">2:3</span> Sehṛen-asen-d am akken d ilsawen n tmes
ițemfaṛaqen wway gar-asen, rsen-d ɣef yal yiwen deg sen.

<span id="an_2:4">2:4</span> Sčuṛen akk s Ṛṛuḥ iqedsen, bdan țmeslayen s
tutlayin nniḍen, yal yiwen seg-sen iheddeṛ s tutlayt i s-d-yefka Ṛṛuḥ
iqedsen.

<span id="an_2:5">2:5</span> Deg ussan-nni, di temdint n Lquds, llan
yemdanen n wat Isṛail yețḍuɛun Sidi Ṛebbi, i d-yusan si mkul tamurt n
ddunit.

<span id="an_2:6">2:6</span> Mi slan i lḥess-nni, lɣaci uzzlen d meṛṛa ;
dehcen, axaṭer sellen-asen țmeslayen-d s tutlayt n tmurt i deg țɛicin.

<span id="an_2:7">2:7</span> Wehmen, țɛeǧben, qqaṛen wway gar-asen :
Imdanen-agi yețmeslayen mačči n tmurt n Jlili ?

<span id="an_2:8">2:8</span> Amek armi yal yiwen deg-nneɣ isell-asen
țmeslayen s tmeslayt-is ?

<span id="an_2:9">2:9</span> Llan gar-aneɣ yemdanen i d yusan si tmurt n
Bartas, si tmurt n Midyas, si tmurt n Ilam, kra nniḍen si tmurt n
Mizubuṭamya, si tmurt n Yahuda akk-d tmurt n Kafadusya, wiyaḍ si tmurt n
Tqenṭert, si tmurt n Asya,

<span id="an_2:10">2:10</span> si tmurt n Frijya akk-d Bamfilya. Usan-d
daɣen si tmurt n Maṣer, si leǧwahi n Qiṛwan di tmurt n Libya, widak i
d-yusan si temdint n Ṛuma ;

<span id="an_2:11">2:11</span> llan daɣen wat Isṛail akk-d wid itabaɛen
di ddin nsen, llan wid n tegzirt n Kritus akk-d waɛṛaben. Amek armi i
sen-nesla țmeslayen s tutlayin-nneɣ meṛṛa ɣef leɛǧayeb n Sidi Ṛebbi ?

<span id="an_2:12">2:12</span> Dehcen akk, ur ẓrin d acu ara xemmemen,
qqaṛen wway gar-asen : D acu-t wayagi ?

<span id="an_2:13">2:13</span> Ma d wiyaḍ stehzayen fell-asen qqaṛen : D
askaṛ i sekkṛen !

<span id="an_2:14">2:14</span> Buṭrus yellan akk-d ḥdac n ṛṛusul-nni,
issaɛli taɣect-is yenna i lɣaci : Ay imezdaɣ n temdint n Lquds akk-d
kunwi meṛṛa isɛeddayen ussan di temdint-agi, slet-ed mliḥ i wayen ara
wen-d-iniɣ, ilaq aț-țeẓrem ayagi :

<span id="an_2:15">2:15</span> irgazen-agi ur skiṛen ara am akken i
ɣilen kra seg-wen, axaṭer ațan d țesɛa n ṣṣbeḥ kan !

<span id="an_2:16">2:16</span> Lameɛna ayagi yedṛan d ayen i d-icar nnbi
Yuwil :

<span id="an_2:17">2:17</span> Sidi Ṛebbi yenna-d : Deg ussan ineggura a
d-sserseɣ Ṛṛuḥ-iw ɣef yemdanen meṛṛa ; arrac-nwen d yessi-twen a
d-țcirin ayen ara sen-d-țxebbiṛeɣ, ilmeẓyen-nwen a sen-d-țweḥḥiɣ,
imɣaṛen-nwen a sen-d-mmeslayeɣ di tirga.

<span id="an_2:18">2:18</span> Anɛam, deg ussan-nni a d-smireɣ Ṛṛuḥ-iw
ɣef yiqeddacen-iw ț-țqeddacin-iw, a d-ciren s wayen ara sen-d-țxebbiṛeɣ.

<span id="an_2:19">2:19</span> Deg igenni ad xedmeɣ leɛǧayeb, di lqaɛa a
d-sekneɣ licaṛat yessewhamen : idammen, times d usigna n ddexxan.

<span id="an_2:20">2:20</span> Iṭij ad yuɣal d ṭṭlam, aggur ad yuɣal am
idammen uqbel a d-yas wass n Sidi Ṛebbi ; ass-nni d ass ameqqran,
yeččuṛen d lɛaḍima.

<span id="an_2:21">2:21</span> Imiren kra n win ineddhen s yisem n Sidi
Ṛebbi ad ițțusellek.

<span id="an_2:22">2:22</span> Ay imdanen n wat Isṛail, ḥesset-ed mliḥ i
wayen ara wen-d iniɣ : ɣef wayen yeɛnan Ɛisa anaṣari, teẓram akk belli
Sidi Ṛebbi isbeggen ed tazmert-is mi gexdem zdat-wen lbeṛhanat, licaṛat
d leɛǧubat, am akken i teẓram s yiman-nwen.

<span id="an_2:23">2:23</span> Akken yella si zik di leqsed n Sidi Ṛebbi
; argaz-agi tezzenzem-t, d kunwi i t-yenɣan mi t-tsellmem ger ifassen n
ikafriwen i t-isemmṛen ɣef lluḥ.

<span id="an_2:24">2:24</span> Lameɛna Sidi Ṛebbi yesseḥya t-id,
isellek-it-id si tesraft n lmut axaṭer d lmuḥal a t-teṭṭef ɣuṛ-es.

<span id="an_2:25">2:25</span> Sidna Dawed yenna-d fell-as : Țwaliɣ kull
ass Sidi Ṛebbi zdat-i axaṭer yețțili ɣer uyeffus-iw iwakken ur ɣelliɣ
ara ;

<span id="an_2:26">2:26</span> daymi i gefṛeḥ wul-iw, i tcennu taɣect-iw
; ula d iɣsan-iw ad ṛtiḥen di laman ;

<span id="an_2:27">2:27</span> axaṭer ur iyi-tețțaǧǧaḍ ara di laxeṛt,ur
tețțaǧaḍ ara win i k-iḍuɛen ad yerku deg uẓekka.

<span id="an_2:28">2:28</span> Teldiḍ-iyi iberdan n tudert ; ul-iw
yeččuṛ d lfeṛḥ imi tețțiliḍ yid-i.

<span id="an_2:29">2:29</span> Ay atmaten ilaq-iyi a wen d-mmeslayeɣ
ɛinani ɣef tideț yeɛnan jeddi-tneɣ Sidna Dawed. Teẓram belli yemmut,
yețwamḍel, yerna aẓekka ines mazal-it ar ass-a gar-aneɣ.

<span id="an_2:30">2:30</span> Lameɛna imi d nnbi i gella, yeẓra belli
Sidi Ṛebbi ad iṭṭef di lemɛahda i s-d-yefka mi s-d-yenna : D yiwen si
tarwa-k ara sbeddeɣ d agellid deg wemkan-ik.

<span id="an_2:31">2:31</span> Ɣef ḥeggu n Lmasiḥ i d-icar mi d-yenna
lehduṛ-agi ; axaṭer ț-țideț, ur t-yeǧǧi di laxeṛt, ur t-yeǧǧi ad yerku.+

<span id="an_2:32">2:32</span> Win ɣef i wen-d-țmeslayeɣ, d Ɛisa-agi i
d-isseḥya Sidi Ṛebbi si ger lmegtin, aql-aɣ akk d inagan.

<span id="an_2:33">2:33</span> Mi gețwarfed ɣer uyeffus n Sidi Ṛebbi,
yețțunefk-as Ṛṛuḥ iqedsen i s-yewɛed Baba Ṛebbi, nețța ismar-it-id
fell-aɣ ; d ayen i teslam, i twalam.

<span id="an_2:34">2:34</span> Mačči d Sidna Dawed i gulin ɣer igenni,
axaṭer d nețța i d-yennan : Sidi Ṛebbi yenna-d i Ssid-iw :

<span id="an_2:35">2:35</span> Sṭef amkan n lḥekma ɣer uyeffus-iw alamma
rriɣ iɛdawen-ik seddaw idaṛṛen-ik.+ +

<span id="an_2:36">2:36</span> Ilaq ihi at Isṛail meṛṛa ad ẓren s tideț
belli Ɛisa-nni i tsemmṛem ɣef lluḥ, d nețța i gerra Sidi Ṛebbi d Agellid
ameqqran, d Lmasiḥ.

<span id="an_2:37">2:37</span> Mi slan i wayen i d-yenna Buṭrus, lɣaci
meṛṛa saxen wulawen-nsen, nnan i Buṭrus d ṛṛusul nniḍen : Ay atmaten, d
acu ara nexdem ?

<span id="an_2:38">2:38</span> Buṭrus yerra-yasen : Tubet, beddlet tikli
! Mkul yiwen deg-wen ad yețwaɣḍes s yisem n Sidna Ɛisa Lmasiḥ iwakken a
wen-țwasemmḥen ddnubat-nwen, yerna a wen-d yețțunefk Ṛṛuḥ iqedsen.

<span id="an_2:39">2:39</span> Axaṭer lweɛd-agi n Sidi Ṛebbi,
yețțunefk-ed i kunwi d warraw-nwen akk-d wid meṛṛa n leǧnas nniḍen iwumi
ara d-issiwel Sidi Ṛebbi Illu-nneɣ.

<span id="an_2:40">2:40</span> Buṭrus inehhu-ten s waṭas n yimeslayen
nniḍen, iwakken ad amnen, yeqqaṛ-asen : Xḍut i lǧil-agi iɛewjen iwakken
aț-țețțusellkem !

<span id="an_2:41">2:41</span> Wid iqeblen ayen i d-yenna Buṭrus, qeblen
ad țwaɣedṣen. Ass-nni rnan-d wazal n tlata alaf n yemdanen i gumnen.

<span id="an_2:42">2:42</span> Seg imiren, zgan țnejmaɛen am atmaten
iwakken ad slen i wselmed n ṛṛusul ; beṭṭun aɣṛum n usmekti n Lmasiḥ,
yerna deɛɛun ɣer Sidi Ṛebbi s yiwen n wul.

<span id="an_2:43">2:43</span> Tɛeǧǧben, tekcem-iten akk tugdi mi țwalin
ṛṛusul xeddmen aṭas n lbeṛhanat d leɛǧayeb.

<span id="an_2:44">2:44</span> Wid akk yumnen, țɛicin s tdukli yerna
beṭṭun wway gar-asen ayen akk sɛan.

<span id="an_2:45">2:45</span> Znuzun ayla-nsen meṛṛa, ayen akken i
d-jemɛen, feṛqen-t wway gar-asen mkul yiwen ɣef leḥsab n wakken yeḥwaǧ.

<span id="an_2:46">2:46</span> Kull ass, s yiwen ṛṛay, țnejmaɛen di
lǧameɛ iqedsen, țnejmaɛen daɣen deg ixxamen-nsen, beṭṭun aɣṛum iwakken
ad mektin Lmasiḥ, tețțen lqut-nsen s nneya d lfeṛḥ ;

<span id="an_2:47">2:47</span> țḥemmiden Sidi Ṛebbi, imdanen meṛṛa
țqadaṛen-ten ; Sidi Ṛebbi irennu-d kull ass ɣer tejmaɛt-nsen wid
ițțusellken.

<span id="an_3:1">3:1</span> Aț-țili d leǧwahi n tlata n tmeddit, Buṭrus
d Yuḥenna ddukklen, ulin ɣer lǧameɛ iqedsen iwakken ad dɛun ɣer Sidi
Ṛebbi.

<span id="an_3:2">3:2</span> Țțawin-d mkul ass yiwen n wergaz i d-ilulen
d aquḍar ɣer zdat n tewwurt n lǧameɛ, tin iwumi qqaṛen « tawwurt tucbiḥt
»; yețɣimi dinna iwakken ad issuter tin n Ṛebbi i wid i gkeččmen ɣer
lǧameɛ-nni.

<span id="an_3:3">3:3</span> Mi gwala Buṭrus d Yuḥenna țeddun ad kecmen,
issuter-asen tin n Ṛebbi.

<span id="an_3:4">3:4</span> Buṭrus d Yuḥenna, ṛeṣṣan allen-nsen
fell-as, nnan-as : Muqel-ed ɣuṛ-nneɣ !

<span id="an_3:5">3:5</span> Argaz-nni imuqel-iten, iḍmeɛ deg-sen a
s-d-fken kra.

<span id="an_3:6">3:6</span> Dɣa Buṭrus yenna-yas : Ur sɛiɣ idrimen n
lfeṭṭa, ur sɛiɣ wid n ddheb, lameɛna ayen sɛiɣ a k-t-fkeɣ : s yisem n
Sidna Ɛisa Lmasiḥ anaṣari, ekker telḥuḍ !

<span id="an_3:7">3:7</span> Iṭṭef-it-id seg ufus ayeffus,
issekker-it-id. Imiren kan iḍaṛṛen-is ț-țkeɛburin n idaṛṛen-is ḥlan.

<span id="an_3:8">3:8</span> Ibedd ɣef yidaṛṛen-is, ineggez, yebda
ileḥḥu. Ikcem yid-sen ɣer lǧameɛ, ileḥḥu ițjellib, ițḥemmid Sidi Ṛebbi.

<span id="an_3:9">3:9</span> Lɣaci akk walan-t mi gleḥḥu ițḥemmid Sidi
Ṛebbi.

<span id="an_3:10">3:10</span> Mi t-ɛeqlen qqaṛen : Mačči d win akken
yețɣimin ɣer tewwurt tucbiḥt n lǧameɛ iqedsen win akken issuturen tin n
Ṛebbi ? Dehcen meṛṛa, wehmen deg wayen yedṛan yid-es.

<span id="an_3:11">3:11</span> Argaz-nni iteddu d Buṭrus akk-d Yuḥenna,
ur yeṭṭixiṛ ara fell-asen, lɣaci-nni meṛṛa wehmen, uzzlen-d ɣuṛ-sen ɣer
usqif iwumi qqaṛen « Asqif n Sidna Sliman ».

<span id="an_3:12">3:12</span> Buṭrus mi gwala ayagi, yenṭeq ɣer lɣaci
yenna : Ay at Isṛail, acuɣeṛ akka twehmem deg wayen yedṛan ? Acuɣeṛ i
d-țmuqulem ɣuṛ-nneɣ, am akken d nukkni i gesselḥan argaz-agi s
tezmert-nneɣ neɣ s ṭṭaɛa-nneɣ ?

<span id="an_3:13">3:13</span> D Sidi Ṛebbi Illu n Sidna Ibṛahim, n
Isḥaq d Yeɛqub, Ṛebbi n lejdud-nneɣ, d nețța i d-isbeggnen lɛaḍima n
Sidna Ɛisa aqeddac-is, win akken i tnekṛem, i tsellmem i lḥakem Bilaṭus
i gebɣan a s-d-iserreḥ.

<span id="an_3:14">3:14</span> Kunwi tnekṛem win akken yellan d azedgan,
d aḥeqqi ; tessutrem a d-serrḥen i bu tmegṛaḍ deg umkan-is.

<span id="an_3:15">3:15</span> Tenɣam Mmi-s n ugellid n tudert, meɛna
Sidi Ṛebbi yesseḥya-t-id si ger lmegtin , aql-aɣ d inigan.

<span id="an_3:16">3:16</span> Ț-țazmert yellan deg isem n Sidna Ɛisa i

»

<span id="an_7:7">7:7</span> Lameɛna Sidi Ṛebbi yenna-d : Ad uɣalen a

iṭeḥheṛ-as deg wass wis tmanya n tlalit-is. Isḥaq daɣen yexdem akken i

<span id="an_7:10">7:10</span> Lameɛna Sidi Ṛebbi yella yid-es,

<span id="an_7:11">7:11</span> Yeɣli-d laẓ ɣef tmurt n Maṣeṛ akk ț-țmurt

<span id="an_7:16">7:16</span> Wwin-d iɣsan-nsen ɣer taddart n Sicem,

<span id="an_7:17">7:17</span> Mi d-yewweḍ lweqt i deg ara yedṛu wayen i

innerna s waṭas di tmurt n Maṣer ;

ara Yusef.

<span id="an_7:19">7:19</span> Agellid-nni iḥqeṛ lɣaci-nneɣ, yețqehhiṛ

kan ara d-lalen, iwakken ur țidiren ara.

<span id="an_7:20">7:20</span> Di lweqt-nni i d-ilul Sidna Musa ; icbeḥ

<span id="an_7:21">7:21</span> uɣalen rran-t zdaxel n tqecwalt,

<span id="an_7:22">7:22</span> Ikker-ed ger imaṣriyen, yelmed akk

<span id="an_7:23">7:23</span> Mi geṣṣaweḍ ṛebɛin iseggasen di

<span id="an_7:24">7:24</span> Yiwen wass iwala s wallen-is yiwen umaṣri

<span id="an_7:26">7:26</span> Azekka-nni iwala sin yergazen n wat

<span id="an_7:33">7:33</span> Sidi Ṛebbi yenna-yas : Kkes arkasen seg
iḍaṛṛen-ik axaṭer amkan-agi anda tbeddeḍ d amkan iqedsen.

<span id="an_7:36">7:36</span> D nețța i ten-id-issufɣen si tmurt n

<span id="an_7:37">7:37</span> D nețța daɣen i gennan i wat Isṛail :

<span id="an_7:42">7:42</span> Sidi Ṛebbi yeṭṭaxeṛ fell-asen, yeǧǧa-ten
ad ɛebden itran n yigenni , d ayen i guran di tektabt n lenbiya : Ay

ṛebɛin iseggasen-nni i tesɛeddam deg unezṛuf ?

<span id="an_7:43">7:43</span> Xaṭi meɛna tbubbem aqiḍun n Mulux akk-d
itri n Rufan i tḥesbem d iṛebbiten-nwen ;+ lmeṣnuɛat nni i txedmem

<span id="an_7:44">7:44</span> Deg unezṛuf, lejdud-nneɣ sɛan aqiḍun i

s-t-id-yemla Sidi Ṛebbi i Sidna Musa mi s-d-yenna : Xdem-it am akken i
t-twalaḍ di lemtel i k-d-ssekneɣ.

<span id="an_7:45">7:45</span> Mi d-yewweḍ uqiḍun-nni ɣer ifassen n

Aqiḍun-nni yeqqim dinna armi d zzman n Sidna Dawed.

k-bnuɣ lǧameɛ iqedsen ara yilin ț-țanezduɣt-ik !

<span id="an_7:47">7:47</span> Lameɛna d Sidna Sliman i s yebnan

ara deg ixxamen ițwabnan s ufus n wemdan, akken i d-yenna nnbi Iceɛya :

<span id="an_7:49">7:49</span> Igenni d amkan n lḥekma-w, ma d lqaɛa

<span id="an_7:50">7:50</span> Eɛni mačči d afus-iw i d-ixelqen kullec ?

<span id="an_7:51">7:51</span> A wid isbeddayen iɛenqiqen nsen !

Ṛṛuḥ iqedsen am kunwi am lejdud-nwen !

<span id="an_7:53">7:53</span> Tewweḍ-ikkun-id ccariɛa n Sidi Ṛebbi s

ibedden ɣer uyeffus-is.

<span id="an_7:56">7:56</span> Imiren iɛeggeḍ, yenna : Atan țwaliɣ

<span id="an_7:59">7:59</span> M'akken i t-ṛeǧmen, Stifan ideɛɛu ɣer

<span id="an_8:2">8:2</span> Kra yergazen iḍuɛen Sidi Ṛebbi, wwin-d

<span id="an_8:3">8:3</span> Ma d Caɛul ixeddem akk lmejhud-is iwakken

<span id="an_8:7">8:7</span> axaṭer leǧnun țeffɣen s leɛyaḍ seg waṭas n
yemdanen ițțumelken, aṭas daɣen n iquḍaṛen akk-d wid ikerfen i geḥlan.

Sidna Ɛisa akk-d ḥeggu n lmegtin.

acu-t uselmed agi ajdid i ɣ-d-tewwiḍ ?

<span id="an_17:21">17:21</span> Imezdaɣ n temdint n Atinya akk-d

<span id="an_17:22">17:22</span> Bulus ibedd di tlemmast n Laryufaj

tettabaɛem ddin di mkul ḥaǧa;

<span id="an_17:24">17:24</span> D nețța i d-ixelqen ddunit d wayen

ifassen n yemdanen.

<span id="an_17:25">17:25</span> Ur yeḥwaǧ ara ad xedmen fell-as
yemdanen am akken yella wayen i t-ixuṣṣen, nețța i d-ițțaken i yemdanen
meṛṛa tudert, ṛṛuḥ d wayen akk yellan.

<span id="an_17:26">17:26</span> Seg yiwen wemdan i d-ixleq akk leǧnas,
yeɛmeṛ yis-sen ddunit ; yefka-yasen lewqat i tudert-nsen akk-d tilisa i
tmura-nsen.

<span id="an_17:27">17:27</span> Sidi Ṛebbi ixdem annect-agi meṛṛa
iwakken ad nadin fell-as ; ahat a t-afen s usferfed, nețța ur nebɛid ara
ɣef mkul yiwen deg-nneɣ.

<span id="an_17:28">17:28</span> Yis i nesɛa tudert, i nteddu, i nețțili
; d ayen i d-nnan daɣen kra seg imedyazen-nwen : « ula d nukkni,

lfeṭṭa, neɣ n wezṛu yețwanejṛen s ufus ț-țmusni n wemdan.

<span id="an_17:30">17:30</span> Sidi Ṛebbi isemmeḥ-asen i yemdanen asmi
akken ur t-ssinen ara; tura issawel-asen-d akk di mkul amkan iwakken ad
beddlen tikli, a d-uɣalen ɣuṛ-es.

<span id="an_17:31">17:31</span> Axaṭer yextaṛ ass i deg ara iḥaseb

t-id-isseḥya si ger lmegtin.

<span id="an_17:32">17:32</span> Mi slan iheddeṛ-ed ɣef ḥeggu n lmegtin,

nniḍen...

n Laryufaj, yiwet n tmeṭṭut isem-is Damaris akk-d wiyaḍ.

<span id="an_18:3">18:3</span> Imi Brisila d Akilas xeddmen iqiḍunen am

<span id="an_18:4">18:4</span> Mkul ass n westeɛfu, Bulus ițmeslay di

Isṛail, iwakken ad amnen.

<span id="an_18:5">18:5</span> Mi d-usan Silas d Timuti si tmurt n

ed i wat Isṛail belli d Ɛisa i d Lmasiḥ.

<span id="an_18:6">18:6</span> Imi i t-xulfen yerna regmen-t, Bulus

ad ṛuḥeɣ ɣer wat leǧnas ur nelli ara seg warraw n wat Isṛail.

<span id="an_18:7">18:7</span> Mi d-iffeɣ syenna, ikcem ɣer wexxam n Tit

lǧameɛ n wat Isṛail.

<span id="an_18:8">18:8</span> Krisbus, ameqqran n lǧameɛ yumen s Ɛisa

mi slan s wayen i d-yenna Bulus, dɣa țwaɣeḍsen deg waman.

<span id="an_18:9">18:9</span> Yiwen n yiḍ, Sidi Ṛebbi iweḥḥa yas-ed di

!

<span id="an_18:12">18:12</span> Di lweqt-nni i deg yella Galyun d

wwin-t ɣer wexxam n ccṛeɛ, nnan :

s webrid ixulfen ccariɛa.

<span id="an_18:14">18:14</span> Mi gekker Bulus ad ihdeṛ, inṭeq Galyun

a wen ḥesseɣ am akken yella di lqanun ay at Isṛail ;

<span id="an_18:15">18:15</span> meɛna imi d amjadel ɣef yimeslayen, ɣef

ḥekmeɣ ɣef wannect-a.

<span id="an_18:16">18:16</span> Imiren issufeɣ-iten-id syenna.

<span id="an_18:17">18:17</span> Dɣa lɣaci mmɣen ɣef Sustin, ameqqran n

ara lmeɛna.

<span id="an_18:19">18:19</span> Mi wwḍen ɣer temdint n Ifasus, mfaṛaqen

<span id="an_18:20">18:20</span> Ḥellelen-t ad yernu kra wussan yid-sen,

<span id="an_18:21">18:21</span> Yenna-yasen : A d-uɣaleɣ ɣuṛ-wen ma

<span id="an_18:22">18:22</span> Yerkeb di lbabuṛ si temdint n Ifasus

<span id="an_18:23">18:23</span> Mi gesɛedda kra n wussan di temdint n

yesseǧhad inelmaden di Liman.

<span id="an_18:24">18:24</span> Di lweqt-nni, yiwen wergaz n wat Isṛail

Maṣeṛ, yusa-d ɣer temdint n Ifasus ; yezweṛ deg wawal yerna yessen mliḥ
tira iqedsen.

<span id="an_18:25">18:25</span> Argaz-agi yelmed abrid n Sidi Ṛebbi,
iǧhed di liman, ițbecciṛ, yesselmad s teḥqiq ayen i d-qqaṛent tira
iqedsen ɣef Ɛisa Lmasiḥ , ɣas akken d aɣḍas n Yeḥya kan i gessen.

<span id="an_18:26">18:26</span> Yebda ițmeslay ɛinani di lǧameɛ n wat

webrid n Sidi Ṛebbi akken tella.

<span id="an_18:27">18:27</span> Imi i gebɣa Abulus ad iɛeddi ɣer tmurt
n Akaya, atmaten kksen-as akukru, uran tabṛaț i inelmaden n temdint n
Kurintus iwakken ad stṛeḥben yis. Mi gewweḍ ɣer dinna, s ṛṛeḥma n Sidi
Ṛebbi iɛawen aṭas widak yumnen ;

<span id="an_18:28">18:28</span> axaṭer yețṛuẓu ɛinani awal n lecyux n
wat Isṛail zdat lɣaci meṛṛa ; yerna ițbeggin-ed s tira iqedsen belli d
Ɛisa i d Lmasiḥ.

tumnem ? Rran-as : Leɛmeṛ nesli s Ṛṛuḥ-agi iqedsen.

<span id="an_19:3">19:3</span> Yenna-yasen : Anwa aɣḍas i-s-tețwaɣeḍsem

deffir-es, yeɛni s Sidna Ɛisa.

<span id="an_19:5">19:5</span> Mi slan i yimeslayen-agi, țwaɣeḍsen s
yisem n Sidna Ɛisa.

<span id="an_19:6">19:6</span> Akken yessers Bulus ifassen-is fell-asen,

țcaren-d ayen i sen-d-yusan s ɣuṛ Sidi Ṛebbi.

<span id="an_19:7">19:7</span> Imdanen-agi ad ilin di tnac yid-sen.

wayen yeɛnan tagelda n Sidi Ṛebbi.

<span id="an_19:9">19:9</span> Lameɛna kra deg-sen yeqquṛ wul-nsen ugin
ad amnen, rnan kkaten deg webrid n Sidna Ɛisa zdat lɣaci. Mi gwala
annect-agi, Bulus iṭṭaxeṛ-asen, yewwi yid-es inelmaden, isselmad-iten
kull ass deg wexxam n yiwen wergaz isem-is Tiranus.

<span id="an_19:10">19:10</span> Yeqqim dinna sin iseggasen armi imezdaɣ
meṛṛa n tmurt n Asya, ama d at Isṛail ama d iyunaniyen, slan i wawal n
Sidi Ṛebbi.

<span id="an_19:11">19:11</span> Sidi Ṛebbi ixeddem lbeṛhanat imeqqranen
s ifassen n Bulus ;

<span id="an_19:12">19:12</span> lɣaci wwḍen armi srusun ɣef yimuḍan ula

yețțumelken țeffɣen ten leǧnun.

<span id="an_19:13">19:13</span> Kra iseḥḥaren n wat Isṛail, ileḥḥun seg

qqaṛen-asen : S yisem n Sidna Ɛisa, win akken i gețbecciṛ Bulus, effɣet
seg-es !

<span id="an_19:14">19:14</span> Widak ixeddmen annect-agi llan di sebɛa
yid-sen, d arraw n Ciba yellan d yiwen si lmuqedmin imeqqranen n at
Isṛail.

<span id="an_19:15">19:15</span> Lǧen-nni yerra-yasen-d : Ɛisa ssneɣ-t,
ssneɣ daɣen anwa i d Bulus, ma d kunwi d acu-kkun ?

<span id="an_19:16">19:16</span> Argaz-nni i gezdeɣ lǧen, izḍem
fell-asen yekkat-iten armi rewlen seg wexxam d iɛeryanen, yerna jerḥen.

<span id="an_19:17">19:17</span> At Isṛail akk-d iyunaniyen meṛṛa

isem n Sidna Ɛisa.

<span id="an_19:18">19:18</span> Aṭas seg wid i gumnen, usan-d ad qiṛṛen
zdat wiyaḍ, ḥkan-d ayen xedmen.

<span id="an_19:19">19:19</span> Aṭas seg wid yellan xeddmen ssḥur,
wwin-d tiktabin-nsen s wayes xeddmen ssḥur, sseṛɣen-tent zdat medden.
Ufan belli azal n xemsin alef n twiztin i d ssuma-nsent.

<span id="an_19:20">19:20</span> S wakka i gewweḍ wawal n Sidi Ṛebbi ɣer
mkul amkan, ițnerni s tezmert n Sidi Ṛebbi.

<span id="an_19:21">19:21</span> Mi gɛedda wayagi akk, Bulus yeqsed ad
iṛuḥ ɣer temdint n Lquds, ad iɛeddi si tmura n Masidunya akk-d Akaya ;
ixemmem deg iman-is yenna : « M'ara awḍeɣ ɣer dinna, ilaq-iyi daɣen ad
ṛuḥeɣ ɣer temdint n Ṛuma.»

<span id="an_19:22">19:22</span> Iceggeɛ ɣer tmurt n Masidunya sin seg
imɛawnen-is, Timuti d Irastus ; ma d nețța yerna kra n wussan di tmurt n
Asya.

<span id="an_19:23">19:23</span> Di lweqt-nni, ɣef ddemma n webrid n
Sidi Ṛebbi ikker ccwal d ameqqran di temdint n Ifasus.

<span id="an_19:24">19:24</span> Yella yiwen uḥeddad isem-is Dimitriyus,
iṣenneɛ s lfeṭṭa leǧwameɛ imecṭaḥ i d-yețmettilen lǧameɛ n Artimis,
yesserbeḥ aṭas ixeddamen-is.

<span id="an_19:25">19:25</span> Isnejmaɛ-iten nutni d wid i gxeddmen
lfeṭṭa am nețța, yenna yasen : Ay atmaten ! Teẓram belli rrbeḥ-nneɣ
itekk-ed si lxedma-agi ;

<span id="an_19:26">19:26</span> teẓram yerna teslam daɣen ɣef Bulus-agi
issamnen aṭas n lɣaci di temdint n Ifasus d waṭas n tmura n Asya.
Yessamen lɣaci, yewwi-ten d webrid-is mi sen-iqqaṛ : « Iṛebbiten agi
ițwaxedmen s ifassen n wemdan mačči d iṛebbiten n tideț » ;

<span id="an_19:27">19:27</span> ur nuggad ara kan aț-țeɣli ṣṣenɛa-nneɣ,
lameɛna nuggad ad sseɣlin lqima i lǧameɛ n Artimis tameqqrant,
taṛebbiț-nneɣ, qṛib a s-kksen lɛaḍima s wayes tețwaɛbed di tmurt n Asya
d ddunit meṛṛa.

<span id="an_19:28">19:28</span> Mi slan i lehduṛ-agi, ikcem-iten
wurrif, bdan țɛeggiḍen : Ț-țameqqrant Artimis, taṛebbiț n at Ifasus !

<span id="an_19:29">19:29</span> Tețțucewwel akk temdint, lɣaci uzzlen
ɣer wexxam n umezgun, seddan yid-sen Gayus akk-d Aristark yellan d
imasiduniyen, d iṛfiqen n Bulus, ițṣafaṛen yid-es.

<span id="an_19:30">19:30</span> Bulus yebɣa ad ikcem di tlemmast n
lɣaci-nni, lameɛna ugin-as inelmaden.

<span id="an_19:31">19:31</span> Ula d kra n lḥukkam imeqqranen n tmurt
n Asya yellan d imdukkal-is, ceggɛen ɣuṛ-es iwakken a t-ḥellelen ur
ițṛuḥ ara ɣer wexxam n umezgun.

<span id="an_19:32">19:32</span> Anejmaɛ-nni yexṛeb, kra țɛeggiḍen akka,
wiyaḍ akken nniḍen yerna aṭas deg-sen ur ẓrin ara acu i d sebba n
unejmaɛ-nni.

<span id="an_19:33">19:33</span> At Isṛail sbedden-d Alixandru ad

<span id="an_20:20">20:20</span> Țbecciṛeɣ-awen rniɣ slemdeɣ kkun ama

<span id="an_20:21">20:21</span> Țbecciṛeɣ i wat Isṛail d iyunaniyen a

<span id="an_20:23">20:23</span> Si temdint ɣer tayeḍ, Ṛṛuḥ iqedsen

<span id="an_20:24">20:24</span> Lameɛna tudert-iw ur s-fkiɣ ara lqima,

<span id="an_20:25">20:25</span> Tura ẓriɣ belli ur tețțuɣalem ara

leqsed n Sidi Ṛebbi, ur ffireɣ ula d acemma fell-awen.

Ṛebbi i d-iḥerr s idammen n Mmi-s.

<span id="an_21:18">21:18</span> Azekka-nni, yedda yid-nneɣ Bulus ɣer

<span id="an_21:26">21:26</span> Azekka-nni, Bulus yewwi yid-es ṛebɛa

lǧameɛ iqedsen iwakken ad ixebbeṛ lmuqeddem ɣef lweqt i deg ara fakken
wussan n uzizdeg-nsen, yeɛni ass i deg ara yefk mkul yiwen seg-sen

<span id="an_21:35">21:35</span> Mi gewweḍ Bulus ɣer tseddaṛin, refden-t

<span id="an_21:36">21:36</span> Axaṭer lɣaci-nni ttabaɛen-t țɛeggiḍen :
« Enɣ-it ! Enɣ-it !... »

yenna-yasen :
