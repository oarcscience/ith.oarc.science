1:2 Sidna Ibṛahim yeǧǧa-d Isḥaq, Isḥaq yeǧǧa-d Yeɛqub, Yeɛqub yeǧǧa-d Yahuda d watmaten-is.

1:8 Asaf yeǧǧa-d Yehucafat ; Yehucafat yeǧǧa-d Yihuram ; Yihuram yeǧǧa-d Ɛuzya ;

1:9 Ɛuzya yeǧǧa-d Yuṭam ; Yuṭam yeǧǧa-d Aḥaz ; Aḥaz yeǧǧa-d Ḥizeqya ;

1:10 Ḥizeqya yeǧǧa-d Mennac ; Mennac yeǧǧa-d Ɛamun ; Ɛamun yeǧǧa-d Yuciya ;

1:11 Yuciya yeǧǧa-d Yixunya akk-d watmaten-is, di lweqt i deg iqaldiyen wwin at Isṛail d imeḥbas ɣer tmurt n Babilun.

1:12 Mbeɛd lweqt n usgiǧǧi ɣer tmurtn Babilun, Yixunya yeǧǧa-d Calatyel ; Calatyel yeǧǧa-d Zurubabil .

1:13 Zurubabil yeǧǧa-d Abihud. Abihud yeǧǧa-d Ilyaqim ; Ilyaqim yeǧǧa-d Ɛazuṛ ;

1:14 Ɛazuṛ yeǧǧa-d Saduq ; Saduq yeǧǧa-d Yaxin ; Yaxin yeǧǧa-d Ilihud.

1:15 Ilhud yeǧǧa-d Ilɛazaṛ ; Ilɛazaṛ yeǧǧa-d Mattan ; Mattan yeǧǧa-d Yeɛqub.

1:16 Yeɛqub yeǧǧa-d Yusef, d Yusef agi i d argaz n Meryem i d-yeǧǧan Sidna Ɛisa i gețțusemman Lmasiḥ.

1:17 Si lweqt n Sidna Ibṛahim armi d lweqt n Sidna Dawed, ɛeddan ṛbeɛṭac n leǧyal. Ɛeddan daɣen ṛbeɛṭac n leǧyal si lweqt n Sidna Dawed armi d lweqt n usgiǧǧi ɣer tmurt n Babilun. Si lweqt-agi n usgiǧǧi armi d lweqt n Lmasiḥ ɛeddan daɣen ṛbeɛṭac n leǧyal.

1:18 Atan wamek i d-ilul Sidna Ɛisa : yemma-s Meryem tella tețwaxḍeb i yiwen wergaz ițțusemman Yusef. S tezmert n Ṛṛuḥ iqedsen, terfed s tadist uqbel aț-țeddu ț-țislit.

1:19 Yusef axḍib-is, yellan d argaz n ṣṣwab, ur yebɣi ara a ț-țicemmet, dɣa yeqsed ad imsefṛaq yid-es s tuffra.

1:20 Akken yețxemmim i waya, yers-ed ɣuṛ-es lmelk n Ṛebbi di targit yenna-yas : A Yusef ! A mmi-s n Dawed ! Ur țțaggad ara aț-țesɛuḍ Meryem ț-țameṭṭut-ik, axaṭer llufan yellan di tɛebbuṭ-is, yusa-yas-d s Ṛṛuḥ iqedsen.

1:21 A d-tesɛu aqcic, a s-tsemmiḍ Ɛisa, axaṭer d nețța ara iselken lumma-s si ddnubat-nsen.

2:1 Sidna Ɛisa ilul di taddart n Bitelḥem, di tmurt n Yahuda di lweqt n ugellid Hiṛudus ameqqran, kra n imusnawen usan-d si cceṛq ɣer temdint n Lquds, iwakken ad steqsin :

2:4 Hiṛudus yesnejmaɛ-ed akk lmuqedmin d lɛulama n wegdud, isteqsa-ten ɣef wemkan anda ara d-ilal Lmasiḥ ara d-yasen.

2:6 I kemm a taddart n Bitelḥem, ur telliḍ ara ț-țaneggarut ger temdinin n Yahuda, axaṭer seg-em ara d-iffeɣ ugellid ara yeksen at Isṛail agdud-iw.

2:7 Hiṛudus yessawel i imusnawen nni s tuffra, isteqsa-ten si melmi i walan itri-nni.

2:8 Dɣa iceggeɛ-iten ɣer Bitelḥem, yenna-yasen : Nadit lexbaṛ n ṣṣeḥ ɣef weqcic-agi, m'ara t-tafem, init-iyi-d iwakken ad ṛuḥeɣ ula d nekk a t-ɛabdeɣ.

2:12 Sidi Ṛebbi ixebbeṛ-iten-id di targit ur țțuɣalen ara ɣer Hiṛudus, dɣa uɣalen ɣer tmurt-nsen seg ubrid nniḍen.

2:13 Mi ṛuḥen, lmelk n Sidi Ṛebbi idheṛ-as-ed i Yusef di targit, yenna-yas-ed :-- Kker, ddem aqcic akk-d yemma-s, rewlet ɣer tmurt n Maṣeṛ, qqimet dinna alamma nniɣ-ak-ed a d-tuɣaleḍ, axaṭer Hiṛudus ad iqelleb ɣef weqcic-agi iwakken a t-ineɣ.

2:14 Yusef ikker yewwi aqcic-nni akk-d yemma-s deg iḍ, yerwel ɣer tmurt n Maṣeṛ.

2:15 Yeqqim dinna armi yemmut Hiṛudus, iwakken ad idṛu wayen i d-yenna Sidi Ṛebbi seg imi n nnbi Huceɛ : Ssawleɣ-as i Mmi a d-iffeɣ si tmurt n Maṣer.

2:16 Mi gwala Hiṛudus belli kelxen-t imusnawen-nni, ikcem-it zzɛaf d ameqqran, iceggeɛ ɣer taddart n Bitelḥem akk-d ț-țmura i s-d-izzin, ad nɣen arrac meṛṛa yesɛan si sin iseggasen d akessar, s wakken yella leḥsab i s-d-fkan imusnawen-nni.

2:17 Dɣa yețwakemmel wayen i d yenna nnbi Irmiya :

2:18 Yekker leɛyaḍ di taddart n Rama, nesla i imeṭṭawen d umeǧǧed, d Ṛaḥil i gețrun ɣef warraw-is, tegguma aț-țesbeṛ fell-asen, axaṭer mmuten.

2:19 Mi gemmut Hiṛudus ameqqran, lmelk n Sidi Ṛebbi iweḥḥa-d i Yusef di targit mi gella di tmurt n Maṣer,

2:20 yenna-yas : Kker, ddem aqcic d yemma-s tuɣaleḍ ɣer tmurt n wat Isṛail, axaṭer wid yebɣan ad nɣen aqcic-nni, mmuten.

2:21 Yusef iddem aqcic-nni d yemma-s, yuɣal ɣer tmurt n wat Isṛail

2:22 Meɛna mi gesla belli d Aṛxilyus mmi-s n Hiṛudus, i gellan d agellid ɣef tmurt n Yahuda, yuggad. SSidi Ṛebbi iɛeggen-as-ed di targit, iṛuḥ ɣer tmurt n Jlili,

2:23 izdeɣ deg yiwet n taddart ițțusemman Naṣaret, iwakken ad yedṛu wayen i d-nnan lenbiya : Ad ițțusemmi Anaṣari.

3:1 Deg wussan-nni, iban-ed Yeḥya aɣeṭṭas, yețbecciṛ deg unezṛuf n tmurt n Yahuda,

3:3 Fell-as i d-immeslay nnbi Iceɛya mi d-yenna : Ț-țaɣect n win ițɛeggiḍen di lxali : Heggit abrid n Sidi Ṛebbi, ssemsawit iberdan-is.

5:25 Ma yella iḍlem-ik yiwen, tedduklem di sin ɣer wexxam n ccṛeɛ, ɣiwel msefham kečč yid-es deg ubrid, m'ulac axṣim-ik a k-yawi ɣer lḥakem, lḥakem a k-yefk i wɛessas, aɛessas-nni, a k-yerr ɣer lḥebs.

5:27 Teslam belli qqaṛen-d : ur xeddem ara asekkak yeɛni zzna.

5:28 Lameɛna, nekkini a wen-d-iniɣ : win ara imuqlen tameṭṭut, imenna deg ul-is ad iznu yid-es, atan am akken izna yid-es.

5:29 Daymi ma yella ț-țiṭ-ik tayeffust i k-ițawin ɣer leḥṛam, qleɛ-iț, ḍeggeṛ-iț akkin fell-ak. Axaṭer axiṛ-ak a k-iṛuḥ yiwen si lemfaṣel-ik wala aț-țkecmeḍ s lekmal-ik ɣer ǧahennama.

5:30 Ma yella s ufus-ik ayeffus i txeddmeḍ leḥṛam, gzem-it, tḍegṛeḍ-t akkin fell-ak, axiṛ-ak a k-ixaṣ ufus wala ma tkecmeḍ s lekmal-ik ɣer ǧahennama.

5:31 Qqaṛen-d daɣen : Win ara yebrun i tmeṭṭut-is, a s-yefk lkaɣeḍ n berru .

5:33 Teslam daɣen s wayen i d-nnan i lejdud-nneɣ : Ur ḥennet ara, lameɛna ayen i tɛuhdeḍ s limin zdat Sidi Ṛebbi, ilaq a t-txedmeḍ.

5:34 Meɛna nekk a wen-iniɣ : ur țgallat ara maḍi ! Ur țgallat s igenni, axaṭer d amkan n lḥekma n Sidi Ṛebbi,

5:35 ur țgallat s lqaɛa axaṭer fell-as i gesrusu iḍaṛṛen-is, neɣ s Lquds imi ț-țamdint n ugellid ameqqran.

5:36 Ur țgallat ara daɣen : « s yixef n uqeṛṛuy-nwen » axaṭer ur tezmirem ara aț-țerrem ula d yiwen wenẓad d aberkan neɣ d amellal.

5:37 Init « ih » ma ilaq aț-ținim ih, neɣ « xaṭi » ma ilaq aț-ținim xaṭi ; ayen akk ara ternum d zzyada, s ɣuṛ Cciṭan i d-ițas.

5:38 Teslam s wayen i d-qqaṛen : Tiṭ s tiṭ, tuɣmest s tuɣmest.

5:39 Meɛna nekk a wen-iniɣ : ma yella win i wen-ixedmen cceṛ ur țțarat ara țțaṛ. Ma iwwet-ik yiwen ɣer lḥenk ayeffus, sken-as lḥenk nniḍen.

5:40 Ma yebɣa yiwen a k-isiweḍ ɣer ccṛeɛ iwakken a k-ikkes aqenduṛ-ik, rnu-yas ula d abeṛnus-ik.

5:41 Ma yella iḥṛes-ik walebɛaḍ a s-tbibbeḍ taɛkumt azgen webrid-is, kečč awi-yas-ț armi d axxam.

5:42 Efk-as ayen yeḥwaǧ i win i k-d-issutren, ur reggel ara ɣef win ibɣan ad yerḍel s ɣuṛ-ek.

5:43 Teslam i wayen i d-qqaṛen : Ḥemmel amdakkel-ik, tkeṛheḍ aɛdaw ik.

5:44 Nekk a wen-iniɣ : ḥemmlet iɛdawen-nwen i kkun ițqehhiṛen. BBarket wid i kkun-ineɛlen, xedmet lxiṛ i wid i kkun-ikeṛhen, dɛut s lxiṛ i wid i kkun-iḍelmen, akk-d wid i kkun-ițqehhiṛen.

5:45 S wakka ara d-tbanem d arraw n Baba-twen yellan deg igenwan, axaṭer nețța icṛeq-ed iṭij-is ama ɣef wid yelhan ama ɣef yemcumen, yețțak-ed ageffur ( lehwa ) ama i wid ixeddmen lxiṛ, ama i wid ixeddmen cceṛ.

5:46 Ma yella tḥemmlem kan wid i kkun-iḥemmlen, d acu n lfayda ara tesɛum ? Ula d imekkasen n tebzert ( leɣṛama ) xeddmen akenni.

5:47 Ma yella tețțemsalamem d watmaten-nwen kan, d acu n lxiṛ i txedmem ? Ula d imednuben xeddmen akken.

6:1 Ayen txeddmem i Sidi Ṛebbi, ur t-id-sbegginet ara iwakken a kkun-walin medden, m'ulac ur tseɛɛum ara lfayda ɣer Baba-twen yellan deg igenwan.

6:2 M'ara tseddqeḍ, ur țberriḥ ara am akken xeddmen at sin wudmawen di leǧwameɛ neɣ deg iberdan, iwakken ad sɛun ccan ɣer yemdanen. A wen iniɣ tideț : d ayagi kan i d lfayda-nsen.

6:3 Ma d kečč m'ara tseddqeḍ, ilaq afus-ik ayeffus ur ițwali ara wayen ixeddem ufus-ik azelmaḍ ;

6:4 iwakken ssadaqa-inek aț-țeqqim di sser, akka ara tețwaqebleḍ ɣer Baba Ṛebbi yețwalin ayen txeddmeḍ di sser.

6:5 M'ara tețẓallam, ur xeddmet ara am at sin wudmawen iḥemmlen ad ẓallen s ibeddi di leǧwameɛ neɣ deg iberdan iwakken a ten-walin medden. AA wen-iniɣ tideț : d ayagi kan i d lfayda-nsen.

6:6 Ma d kečč m'ara tebɣuḍ aț-țẓalleḍ, ṛuḥ ekcem ɣer texxamt-ik, sekkeṛ tawwurt, tedɛuḍ di sser ɣer Baba Ṛebbi, nețța yețwalin ayen yellan di sser, a k-id-iqbel.

6:7 M'ara tețẓallam, ur sṭuqqutet ara lehduṛ am akken xeddmen at ddunit, ɣilen s lketṛa n imeslayen ara d țwaqeblen.

6:8 Ur ten-țɛanadet ara, axaṭer Baba Ṛebbi yeẓra d acu teḥwaǧem uqbel a s-t-tessutrem.

6:9 Atah wamek ilaq aț-țețẓallam : AA Baba-tneɣ yellan deg igenwan, iisem-ik ad ițwaqeddes,

6:10 lḥekma n tgeldit-ik a d-tass, llebɣi-k ad idṛu di lqaɛa aakken yedṛa deg igenwan.

6:11 Efk-aɣ-d mkul ass tamɛict-nneɣ.

6:12 Semmeḥ-aɣ ddnubat-nneɣ aakken i nețsamaḥ i wid i ɣ-iḍelmen.

6:13 Ssebɛed fell-aɣ ajeṛṛeb, ssellek-aɣ si tḥila n Cciṭan. AAxaṭer ɣuṛ-ek i tella tgeldit, ttazmert d lɛaḍima i dayem. Amin !

6:14 Ma yella tețsamaḥem i yemdanen ii wen-ixeddmen cceṛ, Baba Ṛebbi yyellan deg igenwan a wen-isameḥ uula i kunwi.

6:15 Lameɛna ma yella kkunwi ur tețsamaḥem ara i wiyaḍ, aatan Baba Ṛebbi ur a wen-ițsamaḥ aara ula i kunwi ddnubat-nwen.

6:16 Asmi ara tuẓummem, ur d tesbegginet ara iman-nwen tḥeznem am at sin wudmawen issexsaṛen udmawen-nsen iwakken a ten-țwalin yemdanen belli uẓamen. A wen-iniɣ tideț : d ayagi kan i d lfayda-nsen.

6:17 Ma d kečč, m'ara tuẓumeḍ, ssired udem-ik, tdehneḍ aqeṛṛuy-ik s rriḥa,

6:18 iwakken yiwen ur k-iɛeqqel belli tuẓameḍ. Ilaq ayagi ad iqqim gar-ak d Baba Ṛebbi ițwalin ayen yellan di sser, nețța a k-id-iqbel.

6:19 Ur ssexzanet ara igerrujen di ddunit anda ara ten-yečč ubeɛɛuc d ṣṣdiḍ, anda ara ten-akren wid ifettken leḥyuḍ.

6:20 Meɛna sxeznet axiṛ igerrujen deg igenwan anda ur llin ibeɛɛac d ṣṣdiḍ isserkuyen, anda ur zmiren ara imakaren ad fetken neɣ ad akren.

6:21 Axaṭer anda yella ugerruj-ik, dinna ara yili wul-ik.

6:22 Allen, ț-țiftilin n lǧețța, ma yella seḥḥant wallen-ik, aț-țilliḍ s lekmal-ik di tafat.

6:23 Lameɛna, ma yella allen-ik ur seḥḥant ara, aț-țiliḍ di ṭṭlam. Ma yella dɣa allen-ik ur seḥḥant ara, tafat yellan deg-k d ṭṭlam, acḥal ihi berrik ṭṭlam i deg telliḍ.

6:24 Ur izmir yiwen ad iqdec ɣef sin imɛellmen deg yiwet n tikkelt. Ma iḥemmel yiwen, ad ikṛeh wayeḍ ; ma yeṭṭef deg yiwen, ad iḥqeṛ wayeḍ. Ur tezmirem ara ihi aț-țɛebdem Ṛebbi akk-d idrimen !

6:25 Daymi i wen-d-qqaṛeɣ : ur țḥebbiṛet ara i tudert-nwen, ɣef wayen ara teččem d wayen ara teswem akk-d wayen ara telsem. Acu i gesɛan azal, ț-țudert neɣ d lqut ? D lǧețța neɣ d llebsa ? Tudert tugar lqut, lǧețța tugar llebsa !

6:26 Walit igṭaṭ ( ifṛax ) deg igenwan : ur zerrɛen, ur meggren, ur jemmɛen ula d acemma ɣer ikuffan, meɛna Baba-twen yellan deg igenwan, yețțak-asen-d tamɛict-nsen. Eɛni ur teswim ara akteṛ n yefṛax ?

6:27 Anwa i gzemren s uḥebbeṛ, ad yernu kra n wussan i leɛmeṛ-is ?

6:28 Iwacu ara tḥebbṛem ɣef llebsa ? Walit amek i gemmun ijeǧǧigen n lexla : ur zeṭṭen, ur țellmen,

6:29 lameɛna a wen-iniɣ : ula d agellid Sliman s yiman-is di ccan-is ameqqran, ur yelsi am yiwen seg-sen.

6:30 Ma yella Ṛebbi islusuy akka leḥcic yellan di lexla ass-agi, azekka ad ițwadeggeṛ ɣer tmes, amek ur kkun-islusu ara ula d kunwi ay imdanen ixuṣṣen di liman ?

6:31 Kkset anezgum i yiman-nwen, ur qqaṛet ara : d acu ara nečč, d acu ara nsew neɣ d acu ara nels ?

6:32 Axaṭer ayagi d imejhal i gețḥebbiṛen fell-as. Ma d Baba-twen yellan deg igenwan, yeẓra ayen akk teḥwaǧem !

6:33 Nadit uqbel ɣef tgeldit n Sidi Ṛebbi d wayen yellan d lḥeqq, ayen nniḍen meṛṛa d nețța ara wen-t-id yefken.

6:34 Ur țḥebbiṛet ara ihi i uzekka, axaṭer azekka s unezgum-is weḥd-es. Yal ass s leɛtab-ines !

7:1 Ur țḥasabet ara wiyaḍ iwakken ur kkun-ițḥasab ula d yiwen.

7:18 Ttejṛa yelhan ur tezmir ara a d-tefk yir lfakya, akken daɣen yir ttejṛa ur tezmir ara a d-tefk lfakya yelhan.

7:19 Yal ttejṛa ur d-nețțak ara lfakya lɛali aț-țețwagzem, aț-țețwaḍeggeṛ ɣer tmes.

7:22 Aṭas ara yi-d-yinin ass n lḥisab : « a Sidi a Sidi, s yisem-ik i nbecceṛ imeslayen s ɣuṛ Ṛebbi, s yisem-ik i nessufeɣ leǧnun, s yisem-ik i nexdem aṭas n lbeṛhanat ! »

7:26 Ma d win ismeḥsisen i wawal iw, ur nxeddem ara wayen i d-qqaṛeɣ, icuba ɣer wemdan ur nețxemmim ara, i gebnan axxam-is ɣef ṛṛmel ;

7:27 yeɣli-d ugeffur, ḥemlen-d isaffen, ihubb-ed waḍu ɣef wexxam-nni, imiren yeɣli. Lexsaṛa-s ț-țameqqrant !

8:1 Mi d-yuder Sidna Ɛisa seg udrar, aṭas n lɣaci i t-id-itebɛen.

8:7 Sidna Ɛisa yenna-yas : Ad ṛuḥeɣ a t-sseḥluɣ.

9:13 Meyzet ɣef lmeɛna n wawal-agi yellan di tira iqedsen : D ulawen yeṣfan i bɣiɣ mačči d iseflawen n lmal. AAxaṭer ur d-usiɣ ara ad ssiwleɣ i iḥeqqiyen iwakken a d uɣalen ɣer webrid, meɛna i imednuben.

9:14 Inelmaden n Yeḥya aɣeṭṭas usan-d ɣer Sidna Ɛisa nnan-as : Acuɣeṛ nukkni akk-d ifariziyen nețțuẓum ma d inelmaden-ik ur țțuẓumen ara ?

9:15 Sidna Ɛisa yerra-yasen : Eɛni inebgawen n tmeɣṛa zemren ad ilin di leḥzen skud yella yesli yid-sen ? A d-yas wass i deg ara sen-ițwakkes yesli, imiren ad uẓummen.

9:16 Ur nezmir ara a nxiḍ tafaweț tajḍiṭ i uceṭṭiḍ aqdim, axaṭer tafaweț-nni tajḍiṭ aț-țejbed aceṭṭiḍ-nni aqdim, acerrig-nni ad innerni.

9:17 Ur nețɛemmiṛ ara daɣen iyeddiden iqdimen s ccṛab ajdid, neɣ m'ulac ad fellqen, ccṛab ad inɣel, iyeddiden ad xesṛen ; meɛna a nerr ccṛab ajdid deg yeddiden ijdiden, s wakkenni ccṛab d uyeddid ad țwaḥerzen.

9:18 Mazal Sidna Ɛisa ițmeslay, yusa-d yiwen n ccix n lǧameɛ iseǧǧed zdat-es, yenna-yas : Tura kan i temmut yelli, di leɛnaya-k as-ed aț-țserseḍ afus-ik fell-as, a d-teḥyu !

9:19 Sidna Ɛisa ikker yedda yid-es nețța d inelmaden-is.

9:20 Ațaya yiwet n tmeṭṭut yuḍnen ssiq tnac iseggasen aya, tekka-yas-d si deffir, tennul tacḍaṭ n ubeṛnus-is,

9:21 axaṭer teqqaṛ deg ul-is : « lemmer zemreɣ ad nnaleɣ ɣas ula ț-țacḍaṭ n ubeṛnus-is, ad ḥluɣ. »

9:22 Sidna Ɛisa yezzi ɣer deffir, mi ț-iwala yenna-yas : Ur țțaggad a yell-i, liman-im isseḥla-kkem. SSeg imiren tameṭṭut-nni teḥla.

9:23 Mi gewweḍ Sidna Ɛisa ɣer wexxam n ccix-nni, yufa lɣaci țmeǧǧiden, yerna nhewwalen meṛṛa. Yenna-yasen :

9:24 Ffɣet syagi, taqcict ur temmut ara, d iḍes kan i teṭṭes. DDɣa țmesxiṛen fell-as.

9:25 Mi ssufɣen lɣaci, ikcem, yeṭṭef-ed taqcict-nni seg ufus, dɣa tekker-ed.

9:26 Yeffeɣ lexbaṛ-agi di tmurt meṛṛa.

9:27 Mi gṛuḥ Sidna Ɛisa syenna, tebɛen-t sin iderɣalen, țɛeggiḍen : A mmi-s n Sidna Dawed, ḥunn fell-aɣ !

9:28 Mi gewweḍ ɣer wexxam, iderɣalen-nni kecmen ɣuṛ-es, dɣa Sidna Ɛisa yenna-yasen : Tumnem belli zemreɣ a wen xedmeɣ ayen i yi-d-tessutrem ? RRran-as : Ih a Sidi numen !

9:29 Innul allen-nsen, yenna : A wen-ițwaxdem s wakken yella liman-nwen !

9:30 Dɣa yuɣal-asen-d yeẓri. Sidna Ɛisa yeggul deg-sen : Ɣuṛ-wat ad iffeɣ lexbaṛ ɣef wayagi !

9:31 Meɛna akken kan ffɣen, bdan aberreḥ, lexbaṛ-nni yekka-d meṛṛa tamurt.

9:32 Mi ṛuḥen sin-nni, a ten-aya wiyaḍ wwin-as-d yiwen wergaz i gesgugem uṛuḥani.

9:33 Akken issufeɣ Sidna Ɛisa aṛuḥani-nni, agugam-nni yebda ițmeslay. Lɣaci akk tɛeǧben, qqaṛen : Leɛmeṛ neẓri ayagi di tmurt n wat Isṛail !

9:34 Lameɛna ifariziyen qqaṛen : S tezmert n ugellid n leǧnun i ten-issufuɣ.

9:35 Sidna Ɛisa yekka-d ɣef temdinin ț-țudrin meṛṛa, yesselmad di leǧwameɛ nsen, yețbecciṛ lexbaṛ n lxiṛ n tgeldit n Sidi Ṛebbi, isseḥlay mkul aṭan d mkul lɛib.

9:36 Sidna Ɛisa tegzem tasa-s mi gwala annect-nni n lɣaci ɛyan, feclen, ḥeznen am ulli ur nesɛi ameksa .

10:1 Sidna Ɛisa yessawel i tnac inelmaden-is, yefka-yasen tazmert s wayes ara ssufuɣen leǧnun, ara seḥluyen mkul aṭan d mkul lɛib.

10:2 Atnan yismawen n tnac-nni inelmaden n Sidna Ɛisa :Amezwaru d Semɛun ițțusemman Buṭrus akk-d gma-s Andriyus, Yeɛqub mmi-s n Zabadi akk-d gma-s Yuḥenna,

10:3 Filibus d Bartelmay, Suma akk-d Matta amekkas n tebzert, Yeɛqub mmi-s n Ḥalfi akk-d Taddi ;

10:4 Semɛun awaṭani, Yudas n Qeṛyut, win akken ixedɛen Sidna Ɛisa.

10:19 M'ara kkun-ṭṭfen, ur țḥebbiṛet ara ɣef wamek ara temmeslayem, neɣ ɣef wayen ara d-tinim, imeslayen ara d-tinim, a wen-d-țțunefken di teswiɛt nni kan.

10:28 Ur țțaggadet ara wid ineqqen lǧețța lameɛna ur zmiren ara ad nnɣen ṛṛuḥ, aggadet win izemren ad iḍeggeṛ ṛṛuḥ akk-d lǧețța ɣer ǧahennama.

10:30 Ula d anẓaden uqeṛṛuy-nwen țwaḥesben irkulli.

10:31 Ihi ur țțaggadet ara : tesɛam azal akteṛ n waṭas n iẓiwcen !

10:33 Kra win ara yi-nekṛen zdat yemdanen, a t-nekkṛeɣ ula d nekk zdat Baba yellan deg igenwan.

10:34 Ɣuṛ-wat aț-țɣilem d lehna i d wwiɣ ɣer ddunit ; mačči d lehna i d-wwiɣ, meɛna d ccwal i d-wwiɣ.

10:38 Kra win ur neqbil ara ad inɛețțab ɣef ddemma n yisem-iw iwakken ad iyi-itbeɛ, ur yuklal ara ad yili d anelmad-iw.

10:39 Win iḥerzen taṛwiḥt-is, a s-tṛuḥ, ma d win ara isebblen taṛwiḥt-is ɣef ddemma-w ad yuɣal a ț-yaf.

18:1 Di lweqt-nni, inelmaden n Sidna Ɛisa qeṛṛben ɣuṛ-es nnan-as : -- Anwa i d ameqqran di tgelda n igenwan ?

18:2 Sidna Ɛisa yessawel i yiwen weqcic isbedd-it di tlemmast-nsen, yenna :

18:3 A wen-d-iniɣ tideț ; ma yella ur d-tuɣalem ara ɣer webrid, m'ur teqbilem ara aț-țilim .am arrac imecṭuḥen, ur tkeččmem ara ɣer tgelda n igenwan !

18:4 Daymi, win ara yeṣṣemẓin iman-is am weqcic-agi amecṭuḥ, d nețța ara yilin d ameqqran di tgelda n igenwan.

18:5 Kra win ara yestṛeḥben s weqcic am-agi ɣef ddemma n yisem-iw, yis-i i gesṭerḥeb.

18:6 Win ara yesseɣlin di ddnub ula d yiwen seg imecṭuḥen-agi yumnen yis-i, axiṛ-as a s-cudden aɣaṛef n tsirt ɣer temgeṛṭ-is, a t-ḍeggṛen ɣer lebḥeṛ ad iɣṛeq.

18:7 A tawaɣit n ddunit-a, imi qwant tsebbiwin n tuccḍa ! Axaṭer tisebbiwin n tuccḍa ulamek ur d-țilint ara. Meɛna a nnger n win ara yilin d sebba-nsent.

18:8 Ma yella d afus-ik neɣ d aḍaṛ-ik ara k-yesseɣlin di ddnub, gzem-iten, ḍeggeṛ-iten akkin fell-ak, axiṛ-ak aț-țiliḍ di ddunit d aɛiban wala aț-țesɛuḍ sin iḍaṛṛen neɣ sin ifassen, aț-țwaḍeggṛeḍ ɣer tmes ur nxețți.

18:9 Ma yella daɣen ț-țiṭ-ik ara k-yawin ɣer ddnub, qleɛ-iț ḍeggeṛ-iț akkin fell-ak ; axiṛ-ik aț-țiliḍ di ddunit s yiwet n tiṭ wala aț-țesɛuḍ snat wallen, aț- țwaḍeggṛeḍ ɣer tmes n ǧahennama.

18:10 Ɣuṛ-wat aț-țḥeqṛem yiwen seg imecṭuḥen-agi axaṭer aql-i nniɣ awen-t : lmalayekkat-nsen deg igenwan, zgant daymen zdat Baba Ṛebbi.

18:11 [ Axaṭer Mmi-s n bunadem yusa-d ad isellek wid i gḍaɛen ! ]

18:12 D acu twalam ? Ma yella win yesɛan meyya wulli dɣa teɛṛeq-as yiwet deg-sent, ur ițțaǧa ara țesɛa uțesɛin-nni nniḍen di lexla iwakken a d-inadi ɣef tin i s-iɛeṛqen ?

18:13 A wen-iniɣ tideț : ma yufa-ț, ad ifṛeḥ yis akteṛ n țesɛa uțesɛin-nni nniḍen ur neɛṛiq ara.

18:14 Akken daɣen Baba-twen Ṛebbi yellan deg igenwan, ur yebɣi ara a s-iṛuḥ ula d yiwen seg imecṭuḥen-agi.

18:15 Ma yella twalaḍ gma-k idneb, ṛuḥ ṭṭef-it weḥd-es, nhu-t. Ma yuɣ-ak awal, atan trebḥeḍ-ed gma-k !

18:16 Meɛna ma igumma a k-d-isel, awi yiwen neɣ sin yemdanen yid-ek, iwakken ad ilin d inigan m'ara tefru temsalt , akken i d-tenna ccariɛa.

18:17 Ma yugi a sen-isel, ssiweḍ tamsalt ɣer tejmaɛt n watmaten, m'ur iḥess ara ula i tejmaɛt-nni, ḥseb-it am akken d akafriw neɣ d amekkas axabit !

18:18 A wen-iniɣ tideț : ayen akk ara teqnem neɣ iwumi ara tserrḥem di ddunit, ad yețwaqbel deg igenwan.

18:19 A wen-iniɣ daɣen s tideț : ma yella sin seg-wen ddukklen iwakken ad ssutren kra di tẓallit, Baba Ṛebbi yellan deg igenwan, a sen-t-id-yefk.

18:20 Axaṭer anda ara dduklen sin neɣ tlata s yisem-iw, ad ḥedṛeɣ gar-asen !

18:21 Imiren Buṭrus iqeṛṛeb ɣuṛ-es yenna-yas : A Sidi, acḥal n tikkal ilaq ad semḥeɣ i gma ma yeḍlem-iyi, armi d sebɛa n tikkal ? SSidna Ɛisa yerra yas :

18:22 Xaṭi ! Ur k-d-nniɣ ara alamma d sebɛa tikkal, meɛna alamma d sebɛin iberdan sebɛa tikkal !

18:23 Daymi, tagelda n igenwan tețțemcabi ɣer yiwen ugellid yebɣan ad imḥasab nețța d iqeddacen-is.

18:24 Akken yebda leḥsab, wwin-as-d yiwen uqeddac iwumi yețțalas imelyunen.

18:25 Imi ur yesɛi ara s wacu ara ixelleṣ, agellid-nni ifka lameṛ a ten-zzenzen d aklan, s nețța s tmeṭṭut-is, s warraw-is d wayen akk yesɛa, iwakken ad ixelleṣ ṭṭlaba-ines.

18:26 Aqeddac-nni yeɣli ɣer iḍaṛṛen n ugellid, iḥellel-it yenna-yas : «` A Sidi, ṣṣbeṛ kra wussan, ad uɣaleɣ a k-xellṣeɣ ayen akk i yi tețțalaseḍ. »

18:27 Imi i t-iɣaḍ aṭas, agellid-nni isumeḥ-as ṭṭlaba-ines iserreḥ-as ad iṛuḥ.

18:28 Akken kan i d-iffeɣ uqeddac-nni immuger-ed yiwen seg imdukkal-is ixeddmen yid-es, iwumi yețțalas kra n iṣurdiyen. IIkker ɣuṛ-es yeṭṭef-it si temgeṛṭ yeqqaṛ-as : « Err-iyi-d ayen i k țțalaseɣ ! »

18:29 Amdakkel-is yeɣli ɣer iḍaṛṛen-is iḥellel-it, yenna-yas : « Sṣbeṛ kra n wussan, ad uɣaleɣ a k-xellṣeɣ ! »

18:30 Lameɛna ur yeqbil ara ad isbeṛ, yerra-t ɣer lḥebs, alamma tețwaxelleṣ ṭṭlaba-ines.

18:31 Iqeddacen nniḍen mi walan ayen yedṛan, iɣaḍ-iten lḥal, dɣa ṛuḥen ṣṣawḍen lexbaṛ i ugellid.

18:32 Agellid iceggeɛ ɣer uqeddac-nni akken a d-yas, dɣa yenna-yas : « A yir amdan ! Mi tețḥellileḍ deg-i, eɛni ur k-sumḥeɣ ara akk ṭṭlaba-inek ?

18:33 Acuɣeṛ ihi ur as-tsumḥeḍ ara i wemdakkel-ik akken i k-sumḥeɣ nekkini ? »

18:34 Agellid-nni yezɛef aṭas, yerra aqeddac-nni ɣer lḥebs, iwakken ad inɛețțab alamma tețwaxelleṣ ṭṭlaba ines.

19:1 Mi gekfa lewṣayat-agi, Sidna Ɛisa iṛuḥ si tmurt n Jlili ɣer tmurt n Yahuda ɣer leǧwahi n wasif n Urdun.

19:3 Kra ifariziyen qeṛṛben-d ɣuṛ-es iwakken a t-jeṛben, nnan-as : Eɛni ccariɛa-nneɣ tsumeḥ-as i wergaz ad yebru i tmeṭṭut-is ɣef yal sebba yellan ?

19:4 Yerra-yasen : Ur teɣṛim ara ayen i d-teqqaṛ ccariɛa : di tazwara Sidi Ṛebbi ixleq-ed argaz ț-țmeṭṭut,

19:5 yenna-d : AArgaz ad yeǧǧ baba-s d yemma-s, iiwakken ad iɛic nețța ț-țmeṭṭut-is, ddɣa di sin yid-sen, ad uɣalen d yiwen.

19:6 S wakka ur llin ara d sin, meɛna uɣalen d yiwen. Ur as-ilaq ara ihi i wemdan ad ifṛeq ayen yesdukkel Sidi Ṛebbi !

19:7 Nnan-as : Acuɣeṛ ihi Sidna Musa yeǧǧa yaɣ-d lqanun belli m'ara yebru wergaz i tmeṭṭut-is, a s-yefk lkaɣeḍ n berru ?

19:9 Ma d nekk a wen-d-iniɣ : kra n win ara yebrun i tmeṭṭut-is yili ur tezni ara, iwakken ad yaɣ tayeḍ, d nețța i geznan !

19:10 Inelmaden nnan-as : Ma yella akka i d lḥala n wergaz ɣer tmeṭṭut, ulac lfayda di zzwaǧ !

19:11 Sidna Ɛisa yerra-yasen : Mačči d imdanen meṛṛa i gzemren ad qeblen aselmed agi lameɛna anagar wid iwumi i d-ifka Sidi Ṛebbi lefhama i gzemren ad qeblen annect-agi.

19:12 Aṭas n sebbat i gqeḍɛen zzwaǧ i wemdan. Llan wid i d-ilulen si tɛebbuṭ n yemma-tsen ur sɛin ara ddunit, wiyaḍ d imdanen i sen-ț-ikksen. LLlan daɣen wid iḥeṛmen iman-nsen ɣef zwaǧ, ɣef ddemma n tgelda n igenwan. Win i gzemren ad ifhem aselmed agi a t-yefhem !

19:13 Kra yemdanen țawin-d i Sidna Ɛisa arrac imecṭuḥen iwakken ad issers afus-is fell-asen a ten-ibarek, meɛna inelmaden-is țțaran-ten.

19:14 Sidna Ɛisa yenna-yasen : Anfet-asen i warrac a d-asen ɣuṛ-i, ur sen-qeṭṭɛet ara abrid, axaṭer tagelda n igenwan thegga i nutni akk-d wid yellan am nutni !

19:20 Aqcic-nni yerra-yas-ed : Ayagi akk xedmeɣ-t, d acu i yi-ixuṣṣen ?

19:22 Mi gesla i yimeslayen-agi, ilemẓi-nni yeḥzen aṭas dɣa iṛuḥ, axaṭer d ameṛkanti ameqqran i gella.

19:23 Sidna Ɛisa yenna i inelmaden-is : A wen-iniɣ s tideț : iwɛeṛ aṭas i umeṛkanti ad ikcem ɣer tgelda n igenwan !

19:24 A wen-iniɣ daɣen : ma yella iwɛeṛ i welɣem ad iɛeddi di tiṭ n tissegnit, iwɛeṛ akteṛ i umeṛkanti ad ikcem ɣer tgelda n igenwan !

19:25 Mi slan i imeslayen-agi, inelmaden-is dehcen aṭas, nnan-as : Anwa i gzemren ad ițțusellek ihi ?

19:27 Buṭrus inṭeq, yenna-yas : Aql-aɣ nukkni neǧǧa kullec iwakken a k-netbeɛ, d acu ara yedṛun yid-nneɣ ?

19:28 Sidna Ɛisa yerra-yasen : A wen-iniɣ s tideț, asm'ara d-ḥyun lmegtin ɣer ddunit tajḍiṭ, asm'ara yeqqim Mmi-s n bunadem ɣef wukersi n lḥekma di lɛaḍima-s, ula d kunwi yeddan yid-i, aț-țeqqimem di tnac yid-wen ɣef yikersiyen n lḥekma iwakken aț-țḥasbem tnac n leɛṛac n wat Isṛail.

19:29 Yerna, kra win ara yeǧǧen ɣef ddemma n yisem-iw axxam-is neɣ atmaten-is, yessetma-s neɣ baba-s, yemma-s, arraw-is neɣ tiferkiwin-is, ad as-d-yețțunefk akteṛ n wayen yeǧǧa, yerna a t-id-tṣaḥ tudert n dayem.

19:30 Aṭas seg wid yellan d imezwura ad uɣalen d ineggura, aṭas daɣen seg wid yellan d ineggura ad uɣalen d imezwura.

20:4 yenna-yasen : Ṛuḥet ula d kunwi aț-țxedmem di tfeṛṛant-iw, a wen-fkeɣ ayen tuklalem.

26:19 Inelmaden-nni xedmen ayen i sen-yenna Sidna Ɛisa, heggan imensi n lɛid n Izimer n leslak.

26:20 Mi d-teɣli tmeddit, Sidna Ɛisa yeqqim ad yečč nețța d inelmaden-is.

26:21 Mi llan tețțen yenna-yasen : A wen-iniɣ tideț : yiwen seg-wen ad iyi-izzenz.

26:22 Mi d-yenna ayagi, ikcem-iten akk leḥzen d ameqqran, bdan qqaṛen as yiwen yiwen : Neɣ mačči d nekk a Sidi ?

26:23 Yerra-yasen : Win igren afus-is yid-i deg uḍebsi, d nețța ara yi-izzenzen.

26:24 Ad yedṛu d Mmi-s n bunadem akken yura fell-as ; mmeɛna a nnger n win ara t-izzenzen, lemmer ur d-ilul ara axiṛ-is.

26:25 Yudas ara t-ixedɛen yenṭeq yenna-yas : Eɛni d nekk a Sidi ? SSidna Ɛisa yerra-yas : Atan tenniḍ-t-id !

26:26 Mi llan tețțen, Sidna Ɛisa yeddem-d aɣṛum, iḥmed Ṛebbi, icekkeṛ-it, dɣa yebḍa-t, ifṛeq-it i inelmaden-is, yenna-yasen : Eččet aɣṛum-agi : d lǧețța-w.

26:27 Iddem-ed daɣen taqbuct, iḥmed Ṛebbi, icekkeṛ-it, yefka-yasen-ț, yenna-yasen : Swet akk si teqbuct-agi,

26:28 axaṭer ayagi d idammen-iw, d idammen n lemɛahda ara yazzlen ɣef yizumal n lɣaci ɣef ddemma n leɛfu n ddnubat-nsen.

26:29 A wen-d-iniɣ : ur țțuɣaleɣ ara ad sweɣ seg waman-agi n tẓurin alamma d ass i deg ara ten-sweɣ d ijdiden yid-wen, di tgelda n Baba Ṛebbi.

26:30 Mi ḥemden Sidi Ṛebbi s ccnawi d isefra, ṛuḥen ulin ɣer yiɣil n uzemmur,

26:31 dinna Sidna Ɛisa yenna-yasen : Iḍ-agi ad a wen-iliɣ akk d sebba n tuccḍa, axaṭer yura di tira iqedsen ad wwteɣ ameksa, ulli n tqeḍɛit-is ad rewlent ɣer yal tama .

26:32 Meɛna m'ara d-ḥyuɣ si ger lmegtin, ad iyi-tafem zwareɣ ɣer tmurt n Jlili.

26:33 Buṭrus yenṭeq, yenna : Ɣas a sen-tiliḍ d sebba n tuccḍa i wiyaḍ akk, nekk d lmuḥal ad ccḍeɣ.

26:34 Sidna Ɛisa yenna-yas : A k-iniɣ tideț : deg iḍ-agi uqbel ad iskkuɛ uyaziḍ, ad iyi-tnekṛeḍ tlata iberdan.

26:35 Buṭrus yerra-yas : Ɣas lukan d lmut ara mmteɣ yid-ek, d lmuḥal a k-nekkṛeɣ ! UUla d inelmaden nniḍen, nnan-as am nețța.

26:36 Imiren Sidna Ɛisa iṛuḥ yid-sen ɣer yiwen wemkan ițțusemman Jitsimani, yenna i inelmaden-is : Qqimet dagi, nekk ad ṛuḥeɣ ad dɛuɣ ɣer Sidi Ṛebbi.

26:37 Yewwi yid-es Buṭrus akk-d sin warraw-nni n Zabadi ; yebda ikcem-it leḥzen d leɣben d ameqqran.

26:38 Dɣa yenna-yasen : Ḥusseɣ s leḥzen n lmut, qqimet dagi, ɛiwzet yid-i !

26:39 Iṛuḥ yeɣli ɣef wudem, ibda yețẓalla yeqqaṛ : A Baba Ṛebbi, ma yella wamek , ssebɛed fell-i leɛtab-agi ! Meɛna akken tebɣiḍ keččini, mačči akken bɣiɣ nekkini.

26:40 Yuɣal-ed ɣer inelmaden-is, yufa-ten-id ṭṭsen, yenna i Buṭrus : Ur tezmirem ara aț-țɛiwzem yid-i ula d yiwet n ssaɛa ?

26:41 Ɛiwzet, ẓallet, iwakken ur tɣellim ara deg ujeṛṛeb ! Axaṭer ṛṛuḥ n wemdan yeǧhed ma d lǧețța ur tezmir ara !

26:42 Ibɛed akkin tikkelt tis snat iwakken ad iẓẓal, yenna : A Baba, ma yella ulamek ara tesbeɛdeḍ fell-i leɛtab-agi , ma ilaq ad iyi-d-yaweḍ ! Ad yedṛu lebɣi-k !

26:43 Mi d-yuɣal, yufa-ten-id daɣen ṭṭsen, iɣleb-iten nuddam.

26:44 Yeǧǧa-ten, issebɛed iman-is daɣen fell-asen, yedɛa tikkelt tis tlata am tikkelt-nni tamezwarut.

26:45 Mi d-yuɣal ɣer inelmaden-is, yufa-ten-id mazal-iten ṭṭsen ; yenna yasen : Lweqt i deg ara yețwasellem Mmi-s n bunadem ɣer ifassen n yemcumen iwweḍ-ed, kunwi mazal ikkun teṭṭsem ?

26:46 Dɣa yenna-yasen : Ekkret fell-awen, ataya iteddu-d win i yi-izzenzen !

26:47 Mazal-it ițmeslay mi d-iwweḍ Yudas yellan d yiwen si tnac-nni inelmaden. Ddan-d yid-es aṭas n lɣaci s iɛekkzan d ijenwiyen, ceggɛen ten-id imeqqranen n lmuqedmin akk-d imeqqranen n wegdud.

26:48 Win akken i t-izzenzen, imsefham yid-sen, yenna-yasen : Win i ɣef ara selmeɣ, d win ara teṭṭfem !

26:49 Imiren, iqeṛṛeb ɣer Sidna Ɛisa yenna-yas : Sslam fell-ak a Sidi ! DDɣa isellem fell-as.

26:50 Sidna Ɛisa yenna-yas : Ay amdakkel , ayen ɣef i d-tusiḍ a t-txedmeḍ, xdem-it ! DDɣa lɣaci-nni qeṛṛben ɣer Sidna Ɛisa ṭṭfen-t.

26:51 Yiwen seg wid yeddan d Sidna Ɛisa ijbed-ed ajenwi-ines, yewwet aqeddac n lmuqeddem ameqqran, igzem-as-d ameẓẓuɣ-is.

26:52 Sidna Ɛisa yenṭeq yenna-yas : Err ajenwi-inek ɣer wemkan-is, axaṭer kra win ara irefden ajenwi, ad immet s ujenwi !

26:53 Eɛni tɣileḍ ur zmireɣ ara ad ssutreɣ i Baba ad iyi-d-iceggeɛ luluf n lmalayekkat ?

26:54 Amek ihi ara țwakemlent tira iqedsen i d-ixebbṛen belli ilaq a d-yedṛu wayagi meṛṛa ?

26:55 Imiren Sidna Ɛisa yenna i lɣaci : Kull ass țțiliɣ yid-wen di lǧameɛ sselmadeɣ, ur iyi-teṭṭifem ara, ass-agi tusam-d ɣuṛ-i s iɛekkzan d ijenwiyen am akken d bu tmegṛaḍ i lliɣ !

26:56 Meɛna annect-agi meṛṛa yedṛa-d iwakken ad țwakemlent tira n lenbiya. IImiren, inelmaden meṛṛa ǧǧan-t, yal yiwen ɣer wanda yerwel.

26:57 Wid yeṭṭfen Sidna Ɛisa wwin-t ɣer Kayef, ameqqran n lmuqedmin, anda i nnejmaɛen lɛulama n ccariɛa d imeqqranen n wegdud.

26:58 Buṭrus iḍfeṛ-it, ikcem ɣer wefrag n lmuqeddem ameqqran, yeqqim akk-d iɛessasen n lǧameɛ iwakken ad iẓer amek ara tefru temsalt.

26:59 Imiren lmuqedmin imeqqranen akk-d usqamu n imeqqranen n ccariɛa, țnadin inigan ara icehden fell-as s ẓẓuṛ, iwakken ad ḥekmen fell-as s lmut,

26:60 meɛna ur s-d-ufin ara sebba, ɣas akken aṭas i d-icehden fell-as.

26:61 A ten-aya sin yergazen qeṛṛben d nnan : D wagi i d-yennan : « zemreɣ ad huddeɣ lǧameɛ iqedsen n Sidi Ṛebbi, a s-ɛiwdeɣ lebni di tlata wussan.»

26:62 Ameqqran n lmuqedmin ikker, yenna-yas : Ulac ayen ara d-terreḍ ɣef wayen akka i d-ccetkan fell-ak ?

26:63 Sidna Ɛisa yessusem. Dɣa ameqqran n lmuqedmin yenna-yas : Wekkleɣ-ak Sidi Ṛebbi yeddren, m'ur aɣ-d-tenniḍ ma d kečč i d Lmasiḥ, Mmi-s n Sidi Ṛebbi !

26:64 Sidna Ɛisa yerra-yas : Tenniḍ-t-id ! Yerna a wen-iniɣ, ssya d asawen : Mmi-s n bunadem aa d iḥkem ɣɣ er uyeffus n Bab n tezmert, a t-twalim iteddu-d ɣef wusigna seg yigenni.

26:65 Imiren ameqqran n lmuqedmin icerreg aqenduṛ-is yenna : Ikfeṛ ! Iwumi neḥwaǧ inigan tura ? Teslam akk d acu i d-yenna ! D acu tennam ?

26:66 Rran-as-d : Yuklal lmut !

26:67 Dɣa bdan ssusufen-t, țțaken-as tiyitiwin d ibeqqayen,

26:68 qqaṛen-as : Eɛqel a Lmasiḥ, ini-yaɣ-d anwa i k-iwwten !

26:69 Buṭrus yella yeqqim deg ufrag, tqeṛṛeb-ed ɣuṛ-es yiwet n tqeddact tenna-yas : Ula d kečč telliḍ d Ɛisa ajlili !

26:70 Meɛna yenkeṛ zdat-sen meṛṛa yenna-yas : Ur ẓriɣ ara d acu tebɣiḍ a d tiniḍ !

26:71 Akken ițeddu ɣer tewwurt, twala-t tqeddact nniḍen, tenna i wid akk yellan dinna : Wagi yella akk-d Ɛisa anaṣari !

26:72 Inkeṛ daɣen s limin : Qqaṛeɣ-awen ur ssineɣ ara argaz-agi !

26:73 Taswiɛt kan, wid yellan dinna qeṛṛben daɣen ɣer Buṭrus nnan-as : Mbla ccekk, kečč telliḍ seg-sen, tameslayt-ik tbeggen-ed belli d ajlili i telliḍ !

26:74 Ibda yețgalla irennu : Di leɛmeṛ ur ssineɣ argaz-agi ! IImiren kan yeskkuɛ uyaziḍ.

26:75 Dɣa Buṭrus immekta-d imeslayen-nni i s-d-yenna Sidna Ɛisa : uqbel ad iskkuɛ uyaziḍ, ad iyi-tnekṛeḍ tlata n tikkal . Yeffeɣ ɣer beṛṛa s nndama tameqqrant, iṭṭerḍeq d imeṭṭawen.

27:1 Taṣebḥit, nnejmaɛen-d akk imeqqranen n lmuqedmin akk-d imeqqranen n wegdud, mcawaṛen iwakken ad nɣen Sidna Ɛisa.

27:2 Dɣa urzen-t, wwin-t i Bilaṭus, yellan d lḥakem n Ṛuman yesteɛemṛen tamurt.

27:3 Mi gwala ḥekmen ɣef Sidna Ɛisa, Yudas-nni i t-ixedɛen yendem. DDi nndama-nni-ines tameqqrant, iṛuḥ ɣer imeqqranen n lmuqedmin d imeqqranen n wegdud, yerra-yasen tlatin-nni twiztin n lfeṭṭa, dɣa

27:4 yenna yasen : Denbeɣ, nekk i wen-d-izzenzen amdan aḥeqqi. RRran-as-d : Ur aɣ-d-tewqiɛ ara lmeɛna, wagi d ccɣel-ik.

27:5 Yudas iḍeggeṛ tiwiztin-nni di lǧameɛ iqedsen, imiren iṛuḥ iɛelleq iman-is.

27:6 Imeqqranen n lmuqedmin leqḍen-d tiwiztin-nni n lfeṭṭa, nnan : Ur yeḥlil ara a tent-nerr ɣer usenduq n lǧameɛ iqedsen, imi d lexlaṣ n temgeṛt.

27:7 Mi mcawaṛen d acu ara xedmen s twiztin-nni, msefhamen iwakken ad aɣen iger n yiwen wergaz ixeddmen afexxaṛ, iwakken a t-erren ț-țimeqbeṛt i ibeṛṛaniyen.

27:8 Daymi iger-agi ițțusemma ar ass-a : « Iger n idammen ».

27:9 S wakka i gedṛa wayen i d-yenna nnbi Irmiya : Ddmen-d tlatin n twiztin n lfeṭṭa, d wagi i d azal i s-fkan warraw n wat Isṛail ;

27:10 uɣen-d yis iger ɣer win ixeddmen afexxaṛ, akken i yi-d-yumeṛ Sidi Ṛebbi.

27:11 Sidna Ɛisa ibedd zdat lḥakem Bilaṭus. Yesteqsa-t lḥakem-nni yenna-yas : D kečč i d agellid n wat Isṛail ? SSidna Ɛisa yerra-yas : Tenniḍ-t-id !

27:12 Lmuqedmin imeqqranen akk-d imeqqranen n wegdud sseḍlamen deg-s, ma d nețța ur sen-d-yerri ula d acemma.

27:13 Bilaṭus yenna i Sidna Ɛisa : Eɛni ur tesliḍ ara ayen akk ɣef i d-țcetkin fell-ak ?

27:14 Sidna Ɛisa yessusem, ur as-d yerri ula d yiwen wawal, ayagi d ayen iswehmen lḥakem.

27:15 Lḥakem yennum di mkul lɛid n Tuffɣa ițserriḥ-ed i yiwen umeḥbus i gextaṛ lɣaci.

27:16 Yella yakan yiwen umeḥbus mechuṛen, ițțusemman Barabas.

27:17 Mi d-nnejmaɛen akk lɣaci, Bilaṭus isteqsa-ten : Anwa i tebɣam a s-d-serrḥeɣ : i Barabas neɣ i Ɛisa ițțusemman Lmasiḥ ?

27:18 Axaṭer yeẓra s tismin i s-d-wwin Ɛisa.

27:19 Mi gella Bilaṭus ițcaṛaɛ, tceggeɛ-as-d tmeṭṭut-is, tenna-yas-d : Ur ssekcam ara iman-ik di temsalt n uḥeqqi-agi, axaṭer ass-agi targit urgaɣ fell-as terwi-yi.

27:20 Imeqqranen n lmuqedmin akk-d imeqqranen n wegdud sḥeṛcen-d lɣaci iwakken ad ssutren a d-serrḥen i Barabas, ad nɣen Sidna Ɛisa.

27:21 Lḥakem yenna-yasen : Anwa ger sin-agi iwumi tebɣam ad serrḥeɣ ? RRran-as : I Barabas ! Serreḥ-ed i Barabas !

27:22 Bilaṭus yenna-yasen : D acu ara xedmeɣ ihi i Ɛisa ițțusemman Lmasiḥ ? RRran-as-d akk : Ad ițțusemmeṛ ɣef wumidag !

27:23 Lḥakem yenna : D acu i gexdem n diri ? RRnan țɛeggiḍen : Semmeṛ-it ɣef wumidag !

27:24 Mi gwala lɣaci ugin a s-d ḥessen, yerna ccwal simmal yețțimɣuṛ, Bilaṭus yeddem-ed aman, issared ifassen-is zdat lɣaci, yenna : Nekk ur țekkiɣ ara di tazzla n idammen n uḥeqqi-agi, wagi d ccɣel-nwen.

27:25 Lɣaci meṛṛa nnan-as : Tazzla n idammen-is a d-tuɣal ɣef yirawen-nneɣ d yirawen n warraw-nneɣ.

27:26 Imiren Bilaṭus iserreḥ-ed i Barabas, jelkḍen Sidna Ɛisa, dɣa yefka lameṛ a t-semmṛen ɣef wumidag.

27:27 Iɛsekṛiyen n lḥakem ssufɣen Sidna Ɛisa ɣer ufrag, snejmaɛen-d akk iɛsekṛiyen nniḍen, zzin-as.

27:28 Imiren kksen-as llebsa-s, sselsen as abeṛnus azeggaɣ.

27:29 Zḍan taɛeṣṣabt s isennanen, ssersen-as-ț ɣef wuqeṛṛuy is, fkan-as aɣanim ɣer ufus-is ayeffus. Bdan kennun zdat-es, stehzayen fell-as, qqaṛen-as : Sslam fell-ak ay agellid n wat Isṛail !

27:30 Ssusufen-t, kkaten-t s uɣanim ɣer uqeṛṛuy.

27:31 Mi ɛyan deg ustehzi fell-as, kksen-as abeṛnus-nni, rran-as llebsa-s, wwin-t a t-semmṛen ɣef wumidag.

27:32 Mi d-ffɣen, mlalen-d yiwen wergaz n tmurt n Qiṛwan, isem-is semɛun, ḥettmen fell-as ad ibbib amidag n Sidna Ɛisa.

27:33 Mi wwḍen ɣer yiwen wemkan ițțusemman Gulguṭa, yeɛni « amkan n uxecxuc »,

27:34 fkan-as ad isew ccṛab i deg yexleḍ yiwen uɛeqqaṛ aṛzagan. Iɛreḍ-it, yugi a t-isew.

27:35 Mi t-semmṛen ɣef wumidag, gren tasɣaṛt iwakken ad feṛqen llebsa-ines, s wakka, yedṛa-d wayen yuran di tira iqedsen : feṛqen llebsa-w, gren tasɣart ɣef wuqenduṛ-iw.

27:36 Imiren qqimen a t-ɛassen,

27:37 ɛelqen sennig uqeṛṛuy-is talwiḥt i ɣef uran sebba n lmut-is : Wagi d Ɛisa, agellid n wat Isṛail

27:38 Țwasemmṛen yid-es sin imcumen, yiwen ɣer uyeffus-is wayeḍ ɣer uẓelmaḍ-is.

27:39 Imsebriden țhuzzun iqeṛṛay-nsen reggmen-t,

27:40 qqaṛen-as : A win ițhuddun lǧameɛ iqedsen ibennu-t di tlata wussan, sellek iman-ik ma d Mmi-s n Ṛebbi i telliḍ, ers-ed seg umidag-nni.

27:41 Imeqqranen n lmuqedmin, lɛulama n ccariɛa akk-d imeqqranen n wegdud, stehzayen fell-as ula d nutni, qqaṛen :

27:42 Isellek wiyaḍ ur yezmir ad isellek iman-is ! Ma d agellid n wat Isṛail i gella, a d-yers ihi seg umidag iwakken a namen yis.

27:43 Yețkel ɣef Ṛebbi, a t-isellek ihi Ṛebbi ma iḥemmel-it aaxaṭer yenna-d : « Nekk d Mmi-s n Ṛebbi.»

27:44 Ula d imcumen-nni ițțusemmṛen yid-es, regmen-t am wiyaḍ.

27:45 Deg uzzal si tnac armi d țlata n tmeddit, yeɣli-d ṭṭlam ɣef ddunit meṛṛa.

27:46 Ɣef leǧwahi n țlata, Sidna Ɛisa iɛeggeḍ s tɛibṛanit, yenna : Ilya, Ilya, lama sabaqtani ? Yeɛni : Illu-iw, Illu-iw, acuɣeṛ i yi-teǧǧiḍ ?

27:47 Kra seg wid iḥedṛen dinna slan d acu i genna, qqaṛen : Issawal i nnbi Ilyas !

27:48 Imiren kan yiwen seg-sen yuzzel yeddem-ed ameččim n taḍuṭ isselxes-it di lxell, icudd-it ɣef wuɣanim yefka-yas ad isew.

27:49 Wiyaḍ qqaṛen : Anfet-as, a nwali ma yella a d-yas nnbi Ilyas a t-isellek.

27:50 Sidna Ɛisa iɛeggeḍ tikkelt nniḍen, dɣa iffeɣ-it Ṛṛuḥ.

27:51 Imiren kan abeḥnuq n leḥjab yellan di lǧameɛ iqedsen icerreg ɣef sin, seg ixef ufella armi d akessar. Lqaɛa tergagi, idurar ceqqeqen,

27:52 iẓekwan ldin, aṭas seg iḥeqqiyen yemmuten, ḥyan-d

27:53 ffɣen-d seg iẓekwan-nsen. Mi d-iḥya Sidna Ɛisa, kecmen ɣer temdint iqedsen, aṭas n lɣaci iwumi i d-ḍehṛen.

27:54 Lḥakem yellan d aqeṛṛuy ɣef meyya iɛsekṛiyen akk-d iɛsekṛiyen iɛussen Sidna Ɛisa, tekcem-iten tugdi mi walan tergagi lqaɛa d wayen akk yedṛan, nnan : Ț-țideț, argaz-agi d Mmi-s n Ṛebbi !

27:55 Aṭas n tilawin i gellan dinna, țmuqulent-ed si lebɛid. Ț-țiggad-nni i d-yeddan d Sidna Ɛisa si tmurt n Jlili iwakken a s-qedcent.

27:56 Tella gar-asent Meryem tamagdalit, Meryem yemma-s n Yeɛqub d Yusef akk-d d yemma-s n warraw n Zabadi.

27:57 Mi d-tewweḍ tmeddit, yusa-d yiwen umeṛkanti n temdint n Arimati isem-is Yusef, ula d nețța d anelmad n Sidna Ɛisa.

27:58 Iṛuḥ ɣer Bilaṭus yessuter-as lǧețța n Sidna Ɛisa. Bilaṭus yumeṛ ad a s-ț-id-fken.

27:59 Yusef yeddem-ed lǧețța n Sidna Ɛisa, ițțel-iț di lekfen zeddigen,

27:60 issers-iț deg yiwen uẓekka ajdid, i geɣza deg uzṛu. Yeɣleq tawwurt n uẓekka-nni s weblaḍ ameqqran, dɣa iṛuḥ.

27:61 Meryem tamagdalit akk-d Meryem-nni nniḍen, llant dinna qqiment, qublent-ed aẓekka.

27:62 Azekka nni yellan d ass n uheggi n wass n westeɛfu, imeqqranen n lmuqedmin d ifariziyen ṛuḥen ɣer Bilaṭus

27:63 nnan-as : A Sidi, necfa ɣef wayen i d-yenna ukeddab-agi asmi yedder, yenna-d : Ad ɛeddin tlata wussan, a d-ḥyuɣ si ger lmegtin .

27:64 Ceggeɛ iɛessasen ad ɛassen aẓekka-s alamma d ass wis tlata, iwakken inelmaden-is ur d-țasen ara ad akren lǧețța-s, ad inin i wegdud : « iḥya-d si ger lmegtin. » Neɣ m'ulac lekdeb-agi aneggaru ad yaggar amezwaru.

27:65 Bilaṭus yenna-yasen-d : Awit iɛessasen, ṛuḥet ɛasset-ț akken tebɣam.

27:66 Ṛuḥen, zemmemen aken ilaq tawwurt n uẓekka-nni s weblaḍ, imiren sbedden taɛessast zdat-es.

28:1 Mi gɛedda wass n westeɛfu, taṣebḥit n wass amezwaru n dduṛt, Meryem tamagdalit akk-d meryem-nni nniḍen, ṛuḥent a d-ẓuṛent aẓekka n Sidna Ɛisa.

28:2 Deg yiwet n teswiɛt, tergagi lqaɛa ; ataya yiwen n lmelk n Sidi Ṛebbi, iṣubb-ed seg igenni, iqeṛṛeb ɣer uẓekka, iswexxeṛ ablaḍ-nni, yeqqim fell-as.

28:3 Ițfeǧǧiǧ am lebṛaq, llebsa-s ț-țamellalt am udfel.

28:4 Iɛessasen yellan dinna, ikcem-iten lxuf d ameqqran, qquṛen amzun mmuten.

28:5 Yenṭeq lmelk ɣer tilawin-nni, yenna-yasent : Kunemti ur țțaggademt ara, axaṭer ẓriɣ ɣef Ɛisa i yețțusemmṛen ɣef wumidag i tețnadimt.

28:6 Ulac-it dagi, iḥya-d si ger lmegtin akken i t-id-yenna. Asemt-ed aț-țwalimt amkan anda akken yeḍleq !

28:7 Azlemt aț-ținimt i inelmaden-is belli yeḥya-d si ger lmegtin. Atan a kkun-yezwir ɣer tmurt n Jlili ; dinna ara t-teẓrem. AAtan nniɣ-akunt-ț-id !

28:8 Imiren kan, ṛuḥent seg uẓekka-nni s lfeṛḥ d ameqqran ɣas akken uggadent. Uzzlent ad awint lexbaṛ i inelmaden n Sidna Ɛisa.

28:9 Ataya Sidna Ɛisa imuger-itent-id, yenna-yasent : Sslam fell-akunt ! QQeṛṛbent, seǧǧdent zdat-es, ssudnent iḍaṛṛen-is.

28:10 Imiren Sidna Ɛisa yenna-yasent : Ur țțaggademt ! Ṛuḥemt inimt i watmaten-iw ad ṛuḥen ɣer tmurt n Jlili, dinna ara yi-ẓren.

28:11 Mi țeddunt deg ubrid, kra seg iɛessasen-nni n uẓekka kecmen ɣer temdint, xebbṛen imeqqranen n lmuqedmin ɣef wayen akk yedṛan.

28:12 Mi nnejmaɛen imeqqranen n wegdud akk-d imeqqranen n lmuqedmin, msefhamen ad fken i iɛessasen-nni ssuma tameqqrant n yedrimen,

28:13 nnan-asen : Init belli d inelmaden-is i d-yusan deg yiḍ, ukren lǧețța-s mi nella neṭṭes.

28:14 Ma yella yewweḍ lexbaṛ-agi ɣer lḥakem, d nukkni ara t-iqablen iwakken ur awen-ixeddem ula d acemma.

28:15 Iɛsekṛiyen-nni ṭṭfen idrimen-nni, xedmen ayen akk i sen-d-nnan lmuqedmin. Tamsalt-agi țɛawaden-ț-id wat Isṛail ar ass-a.

28:16 Ḥdac inelmaden-nni ṛuḥen ɣer tmurt n Jlili, ɣer wedrar-nni anda i sen-yenna Sidna Ɛisa ad mlilen.

28:17 Mi t-walan seǧǧden zdat-es, meɛna kra seg-sen mazal țcukkuten.

28:18 Sidna Ɛisa iqeṛṛeb-ed ɣuṛ-sen, yenna-yasen : Tețțunefk-iyi-d tezmert ama di lqaɛa ama deg igenwan.

28:19 Ṛuḥet ɣer tmura meṛṛa n ddunit, slemdet imdanen iwakken ad uɣalen d inelmaden-iw, ad țwaɣedṣen s yisem n Baba Ṛebbi, s yisem n Mmi-s akk-d yisem n Ṛṛuḥ iqedsen.

28:20 Slemdet-asen ad tebɛen ayen akk i wen-slemdeɣ. MMa d nekk ad iliɣ yid-wen yal ass alamma ț-țaggara n ddunit.
