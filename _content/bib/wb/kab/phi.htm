Kabyle Bible: Philippians

1:1 S ɣuṛ Bulus akk-d Timuti iqeddacen n Ɛisa Lmasiḥ, i watmaten yumnen s Ɛisa Lmasiḥ, i imeqqranen n tqeḍɛit akk-d wid i ten-ițɛassan yellan di temdint n Filibus.

1:2 Ṛṛeḥma ț-țalwit a wen-d țțunefkent s ɣuṛ baba-tneɣ Ṛebbi akk-d Ɛisa Lmasiḥ Ssid-nneɣ !

1:3 ?ḥemmideɣ țcekkiṛeɣ Sidi Ṛebbi mkul tikkelt m'ara kkun-id-mmektiɣ.

1:4 Yal ass deɛɛuɣ fell-awen s lxiṛ di tẓallit-iw s lfeṛḥ d ameqqran,

1:5 ɣef ddemma n lexbaṛ n lxiṛ i deg tețțekkim seg wass amezwaru armi ț-țura.

1:6 Umneɣ s tideț belli win yebdan deg-wen ccɣel-agi yelhan, a t-ikemmel alamma d ass n tuɣalin n Sidna Ɛisa Lmasiḥ.

1:7 D lḥeqq imi țxemmimeɣ akkagi fell-awen axaṭer tezgam deg ul-iw yerna tețțekkim irkulli di tejmilt i yi-d-ifka Sidi Ṛebbi, ama mi lliɣ di lḥebs ama mi țbecciṛeɣ lexbaṛ n lxiṛ s tezmert ț-țissas.

1:8 Atan Ṛebbi d inigi, acḥal i kkun ccedhaɣ axaṭer ḥemmleɣ-kkun s leḥnana n Ɛisa Lmasiḥ ;

1:9 yerna ayen i ssutureɣ di tẓallit-iw d anerni n leḥmala-nwen di tmusni akk-d lefhama.

1:10 S wakka aț-țissinem amek ara textiṛem ayen ifazen, iwakken aț-țilim teṣfam, ulac ayen ara wen-ssukksen i wass n tuɣalin n Lmasiḥ ;

1:11 yerna tudert-nwen aț-țeččaṛ s wayen yellan d lḥeqq i gxeddem yis-wen Ɛisa Lmasiḥ, i tmanegt akk-d lḥemd n Sidi Ṛebbi.

1:12 Ay atmaten, bɣiɣ aț-țeẓrem belli ayen idṛan yid-i, d ayen i gesnernan abecceṛ n lexbaṛ n lxiṛ.

1:13 Axaṭer deg ufrag n ccṛeɛ akk-d imukan nniḍen meṛṛa, ẓran akk ɣef ddemma n Lmasiḥ i țwaḥebseɣ.

1:14 Aṭas n watmaten i gțeklen ɣef Ssid-nneɣ, yerna ɣas akken ẓran-iyi țwarzeɣ s snasel ur kukran ara, țbecciṛen awal n Ṛebbi mbla tugdi.

1:15 S tideț, llan kra țbecciṛen Lmasiḥ s ṭṭmeɛ akk-d zzux ; meɛna wiyaḍ țbecciṛen s wul yeṣfan d neyya.

1:16 Widak-agi xeddmen ayagi s lmaḥibba, imi ẓran țwaxtaṛeɣ iwakken ad dafɛeɣ ɣef lexbaṛ n lxiṛ.

1:17 Ma d wiyaḍ, țbecciṛen Lmasiḥ s wul yeččuṛen ț-țiḥila ; ixemmimen nsen ur ṣfin ara, ɣilen s wakka ara rnun lqeṛḥ i wul-iw m'akka lliɣ di lḥebs.

1:18 Ur i yi-tewqiɛ ara, axaṭer akken yebɣu yili lḥal, ama s tḥila ama s wul yeṣfan Lmasiḥ yețwabecceṛ ; ɣef wayagi feṛḥeɣ yerna mazal ad feṛḥeɣ,

1:19 axaṭer ẓriɣ ayagi meṛṛa i nnfeɛ inu, s tẓallit-nwen akk-d lemɛawna n Ṛṛuḥ n Sidna Ɛisa.

1:20 Bɣiɣ yerna ssarameɣ ur d-yețțili wacemma s wayes ara ssetḥiɣ ; meɛna am ass-a am uzekka, s lețkal ameqqran, a d-tban deg-i tmanegt n Lmasiḥ ama s tudert-iw ama s lmut-iw ;

1:21 axaṭer ɣuṛ-i, Lmasiḥ ț-țudert-iw ma d lmut d rrbeḥ.

1:22 Meɛna ma yella tudert-iw di ddunit, d lfayda ɣef ddemma n leqdic-iw ɣef lexbaṛ n lxiṛ , d acu ara xtiṛeɣ ? Ur ẓriɣ ara !

1:23 Di snat lǧihat-agi ɛewqeɣ : deg yiwet n tama, bɣiɣ ad ṛuḥeɣ iwakken ad iliɣ akk-d Lmasiḥ, axaṭer d ayen ifazen aṭas ,

1:24 di tama nniḍen, ɣef ddemma-nwen yenfeɛ axiṛ ad idireɣ di ddunit ; ayagi tḥeqqeqeɣ deg-s.

1:25 ?riɣ ad idireɣ yerna ad qqimeɣ gar-awen meṛṛa, iwakken aț-tennernim yerna aț-țfeṛḥem di liman.

1:26 Tuɣalin-iw ɣuṛ-wen, a wen-d-tefk sebba tajḍiṭ, s wayes ara tzuxxem s Ɛisa Lmasiḥ ɣef ddemma-inu.

1:27 Acu kan, ddut s tikli yuklal lexbaṛ n lxiṛ n Lmasiḥ iwakken ad sleɣ fell-awen, ama usiɣ-ed a kkun-ẓreɣ ama ɣabeɣ fell-awen, tǧehdem, tedduklem s yiwen n Ṛṛuḥ, tețḥarabem s yiwen wul ɣef liman i d-ițțak lexbaṛ n lxiṛ.

1:28 ?uṛ-wat aț-țeǧǧem iɛdawen a kkun-ssagden, ayagi ad yili i nutni d țbut n nnger-nsen, i kunwi d țbut n leslak-nwen ; annect-agi s ɣuṛ Ṛebbi i d-yekka ;

1:29 axaṭer tețțunefk-awen-d ṛṛeḥma mačči kan iwakken aț-țamnem s Lmasiḥ, meɛna iwakken aț-țenɛețțabem daɣen ɣef ddemma-ines

1:30 m'ara tețḥarabem akken twalam țḥarabeɣ nekkini, yerna mazal țḥarabeɣ ar tura am akken i teslam yis.

2:1 Eɛni ur tufim ara ṣṣbeṛ di Lmasiḥ, asseǧhed di leḥmala, tagmaț di Ṛṛuḥ iqedsen wway-gar-awen, ur tufim ara leḥnana d ṛṛeḥma ?

2:2 Kemmlet ihi lfeṛḥ-iw, sɛut yiwen uxemmem, sɛut yiwet n leḥmala, ddukklet s yiwen n ṛṛuḥ, d yiwen uxemmem.

2:3 Ur xeddmet acemma s umḥizwer neɣ iwakken aț-țesɛum ccan ur nesɛi lqima, meɛna anzet, ḥesbet wiyaḍ am akken sɛan lqima akteṛ-nwen.

2:4 Ilaq yal yiwen deg-wen ur ițnadi ara kan ɣef nnfeɛ-ines, lameɛna ilaq ad inadi ɣef nnfeɛ n wiyaḍ.

2:5 Sɛut deg ulawen nwen ixemmimen yellan di Ɛisa Lmasiḥ :

2:6 Nețța yellan d ṣṣifa n Ṛebbi, ur inuda ara ad isseɛdel iman-is akk-d Ṛebbi.

2:7 Meɛna isellem kullec s yiman-is mi guɣal d aqeddac. Isseɛdel iman-is d yemdanen mi d-iban s ṣṣifa n wemdan ;

2:8 yessanez iman-is mi gedda s ṭṭaɛa armi d lmut, lmut ɣef wumidag.

2:9 Daymi Sidi Ṛebbi issuli-t sennig n kra wayen yellan yerna ifka-yas isem yellan sennig n yismawen meṛṛa,

2:10 iwakken s yisem n Ɛisa, yal tagecrirt aț-țeknu deg igenwan, di lqaɛa, ddaw lqaɛa,

2:11 iwakken yal iles a d-icehhed belli Ɛisa Lmasiḥ, d nețța i gellan sennig kullec i tmanegt n Ṛebbi baba-tneɣ.

2:12 A wid eɛzizen fell-i, yal ass tețțaɣem awal ; sbeggnet-ed ihi leslak-nwen s lecɣal, s ṭṭaɛa d wannuz, mačči am asm'akken lliɣ yid-wen, meɛna tura akteṛ imi ɣabeɣ fell-awen.

2:13 Axaṭeṛ d Ṛebbi i wen-d-ițțaken lebɣi akk-d tezmert s wayes ara txedmem lebɣi-s yelhan.

2:14 Xedmet kullec mbla asmermeg neɣ asṭuqqet n wawal,

2:15 iwakken yiwen ur kkun-ițlumu, aț-țilim teṣfam, d arraw n Ṛebbi, ur tseɛɛum ara lɛib ger lǧil-agi n ixeddaɛen, aț-țfeǧǧeǧem gar-asen am teftilin di ddunit

2:16 aț-țețbecciṛem awal n tudert ; s wakka, ass n tuɣalin n Lmasiḥ, ayagi ad yili d sebba n ccan-iw imi ur uzzileɣ ur nɛețțabeɣ baṭel.

2:17 ?as akken ad țțusebbleɣ d asfel ara yernun ɣef wesfel n liman-nwen yețțunefken d lweɛda i Sidi Ṛebbi, d ayen i yi isfeṛḥen yerna feṛḥeɣ yid-wen meṛṛa .

2:18 Ula d kunwi feṛḥet am nekk, feṛḥet yid-i.

2:19 S lețkal i sɛiɣ di Sidna Ɛisa, ssarameɣ qṛib a wen-ceggɛeɣ Timuti iwakken a yi-issefṛeḥ s lexbaṛ ara yi-d-yawi s ɣuṛ-wen.

2:20 Ur sɛiɣ ula d yiwen ara yi-iɛawnen deg ixemmimen-iw, iwakken ad iḥebbeṛ fell-awen seg ul ;

2:21 axaṭer akken ma llan țnadin ɣef nnfeɛ-nsen, mačči ɣef nnfeɛ n Ɛisa Lmasiḥ.

2:22 Tessnem tikli n Timuṭi, am weqcic ɣer baba-s, yefka tudert-is yid-i i ubecceṛ n lexbaṛ n lxiṛ.

2:23 Ihi akken kan ara tefru temsalt-iw, a wen-t-in-ceggɛeɣ

2:24 yerna sɛiɣ lețkal di Ssid-nneɣ belli qṛib a n-aseɣ s yiman-iw ɣuṛ-wen.

2:25 Walaɣ ilaq-iyi a wen-ceggɛeɣ gma-tneɣ Ebafrudit, amdakkul-iw di lxedma d uḥareb ɣef ddemma n lexbaṛ n lxiṛ, win akken i yi-d-tceggɛem yewwi-yi-d s ɣuṛ-wen ayen akk ḥwaǧeɣ ;

2:26 iccedha-kkun meṛṛa yerna yeḥzen aṭas imi teslam yis yehlek.

2:27 ?-țideț ihlek armi yewweḍ lmut ; meɛna iɣaḍ Sidi Ṛebbi yerna mačči kan d nețța i t-iɣaḍen, ɣaḍeɣ-t ula d nekk iwakken ur d-iɣelli ara fell-i leḥzen ɣef wayeḍ.

2:28 Ceggɛeɣ-t-in s lemɣawla, iwakken aț-țfeṛḥem m'ara t-twalim yerna ula d nekk leḥzen-iw ad yifsus.

2:29 Sṭerḥbet yis s lfeṛḥ d ameqqran, s wakken yuklal yal yiwen yellan deg ubrid n Lmasiḥ ; qadṛet imdanen am wigi

2:30 axaṭer ɣef ddemma n leqdic ɣef Lmasiḥ i gewweḍ ɣer lmut, mi gsebbel ṛṛuḥ-is iwakken ad ixdem ayen akken ur tezmirem ara a yi-t-txedmem kunwi.

3:1 Ihi ay atmaten, feṛḥet imi tedduklem akk-d Ssid-nneɣ. Ur ɛeyyuɣ ara di tira n wayen i wen-n-uriɣ yakan, yerna ayagi a wen-yili d ṭṭmana i liman-nwen.

3:2 ?adret iman nwen deg wuccanen, ḥadret iman nwen ɣef yemcumen-nni iḍehṛen s lekdeb.

3:3 Axaṭeṛ d nukni i gesɛan ṭṭhara n ṣṣeḥ imi nɛebbed Sidi Ṛebbi s Ṛṛuḥ-is ; yerna nețzuxxu s Ɛisa Lmasiḥ nețkel fell-as, ur nețkil ara ɣef limaṛat n yemdanen.

3:4 Ula d nekk zemreɣ ad țekleɣ ɣef limaṛat n yemdanen. Ma yella win i gɣilen izmer ad ițkel ɣef limaṛat agi, nekk zemreɣ ad țekleɣ akteṛ !

3:5 Axaṭer dehṛeɣ ass wis tmanya, kkiɣ-ed si ccetla n wat Isṛail, n wedrum n Benyamin, d aɛibṛani mmi--s n uɛibṛani.

3:6 ?ef wayen yeɛnan ccariɛa, d afarizi, ma d zzwaṛa, d win ițqehhiṛen tajmaɛt n imasiḥiyen ; ɣef wayen yeɛnan lqanun n ccariɛa n Sidna Musa, ur yelli wayen i yi-ixuṣṣen.

3:7 Meɛna ayen akken i yi-illan d lfayda, ḥesbeɣ-t tura d lexṣara ɣef ddemma n Lmasiḥ.

3:8 Zemreɣ a d iniɣ : ḥesbeɣ kullec d lexṣara ɣef ddemma n wayen ifazen, yellan ț-țamusni n Ɛisa Lmasiḥ Ssid-iw. ?ef ddemma-s qebleɣ a yi-iṛuḥ kullec, ḥesbeɣ kullec d afṛaden, iwakken a d-rebḥeɣ Lmasiḥ

3:9 yerna ad sɛuɣ amkan ɣuṛ-es, ad uɣaleɣ d aḥeqqi mačči mi ḍuɛeɣ ccariɛa n Sidna Musa, meɛna imi uɣaleɣ d aḥeqqi s liman yellan di Lmasiḥ, s lḥeqq-nni i d-yețțak Sidi Ṛebbi i wid yesɛan liman deg-s.

3:10 S wakka ad issineɣ Lmasiḥ, ad issineɣ tazmert n ḥeggu-ines, ad țekkiɣ di leɛtab-is, ad iliɣ am nețța di lmut-is iwakken

3:11 akken yebɣu yili lḥal ad awḍeɣ s wayagi ɣer ḥeggu si ger lmegtin.

3:12 Ur d-nniɣ ara tikli-inu di Lmasiḥ tennekmal neɣ wwḍeɣ ɣer lmeqsud, meɛna mazal țțazzaleɣ iwakken a t-awḍeɣ, imi ula d nekk yewwi-yi Ɛisa Lmasiḥ.

3:13 Ay atmaten, ur ḥsibeɣ ara iman-iw wwḍeɣ ɣer lmeqṣud-agi ; lameɛna tella yiwet lḥaǧa xeddmeɣ-ț : tețțuɣ ayen iɛeddan, ssarameɣ ɣer wayen yellan ɣer zdat,

3:14 țțazzaleɣ ɣer lmeqṣud-agi iwakken ad awḍeɣ ɣer wayen i ɣ-ihegga Sidi Ṛebbi deg igenwan s Ɛisa Lmasiḥ.

3:15 Nukni yellan d imdanen meqqṛen di liman ilaq a nesɛu akk axemmem-agi ; ma yella tesɛam axemmem yemxalafen ɣef kra n temsalin, d Sidi Ṛebbi ara kkun-d-isfehmen ɣef wayagi.

3:16 S wakka, imi newweḍ di tudert-nneɣ ɣer tegniț-agi, ilaq a nkemmlet tikli-nneɣ s tdukli.

3:17 Ɛandet-iyi ay atmaten, țmuqulet ɣer wid i twalam leḥḥun di liman am nukni,

3:18 axaṭer aṭas i gteddun am iɛdawen n umidag n Lmasiḥ ; nniɣ-awen-t-id acḥal d abrid, tura daɣen a wen-t-id ɛiwdeɣ s imeṭṭawen :

3:19 Taggara-nsen d nnger ; ṛebbi-nsen d iɛebbaḍ-nsen, țzuxxun s wayen i glaq ad nneḥcamen, țxemmimen anagar ɣef leḥwayeǧ n ddunit.

3:20 Ma d nukni tanezduɣt-nneɣ deg igenwan, syenna i nețṛaǧu s lḥir amsellek-nneɣ Sidna Ɛisa Lmasiḥ

3:21 ara ibeddlen lǧețțat-nneɣ timeḥquṛin, a tent-yerr am lǧețța-s yeččuṛen d lɛaḍima, s tezmert s wayes ițțarra kullec seddaw lḥekma-s.

4:1 Daymi, ay atmaten-iw eɛzizen fell-i, kunwi i ccedhaɣ a kkun ẓreɣ, kunwi yellan d lfeṛḥ-iw, d ccan-iw , akka i glaq aț-țeṭṭfem s lǧehd di Ssid-nneɣ, a wid eɛzizen fell-i !

4:2 Bɣiɣ ad weṣṣiɣ tiyessetmatin nneɣ Iwudya akk-d Sinticya, iwakken ad ddukklent deg yiwen uxemmem di Ssid-nneɣ.

4:3 Ula d kečč a gma Sizugus unṣiḥ, ḍelbeɣ-ak a tent-tɛiwneḍ nutenti i ḥurben yid-i ɣef ddemma n lexbaṛ n lxiṛ, akk-d Klimun d imdukkal-iw nniḍen di leqdic iwumi uran yismawen nsen di tektabt n tudert.

4:4 Ilit dayem di lfeṛḥ imi tedduklem akk-d Ssid-nneɣ ; a wen-t-id-ɛiwdeɣ : ilit dayem di lfeṛḥ.

4:5 Ilaq aț-țiḥninem ɣer yemdanen meṛṛa, Ssid-nneɣ qṛib a d-yuɣal.

4:6 Ur țḥebbiṛet ɣef wacemma, meɛna di yal lḥaǧa, ssutret ayen teḥwaǧem i Sidi Ṛebbi s tẓallit, s uḥellel akk-d leḥmadi ;

4:7 imiren lehna n Sidi Ṛebbi i gugaren yal tamusni, aț-țeḥrez ulawen-nwen d ixemmimen nwen di Ɛisa Lmasiḥ.

4:8 Taggara ay atmaten a wen-d iniɣ : ayen akk yellan d ṣṣeḥ, d lḥeṛma, d lḥeqq, d ṣṣfa, ayen akk yuklalen leḥmala d wayen yuklalen leqdeṛ akk-d leḥmadi, ad ilin d sebba n ixemmimen-nwen.

4:9 Ayen tlemdem s ɣuṛ-i yerna tqeblem-t, ayen twalam xeddmeɣ-t d wayen teslam nniɣ-t-id xedmet-eț ula d kunwi, dɣa Sidi Ṛebbi Bab n lehna ad yili yid-wen.

4:10 Feṛḥeɣ aṭas am akken ț-țukci i yi-d-ifka Ssid-nneɣ mi walaɣ ul-nwen iɛawed iǧǧuǧeg-ed ɣuṛ-i, tețḥebbiṛem fell-i ; ẓriɣ tețḥebbiṛem fell-i , lameɛna ur wen-d-teɣli ara teswiɛt iwakken a yi-t-id-tbeggnem.

4:11 Ur d-nniɣ ara ayagi imi xuṣṣeɣ, axaṭer lemdeɣ ad steqniɛeɣ di yal taswiɛt i deg lliɣ.

4:12 Ssneɣ ad ɛiceɣ di lexṣaṣ, am akken ssneɣ ad ɛiceɣ di tawant. Di yal amkan, di kullec lemdeɣ leqnaɛa d lexṣaṣ, lemdeɣ tawant d laẓ.

4:13 Zemreɣ i kullec s win i yi-sseǧhaden.

4:14 Lameɛna d ayen ilhan i txedmem imi tețțekkim di leɛtab-iw.

4:15 Kunwi teẓram ay atmaten n temdint n Filibus belli asmi bdiɣ abecceṛ n lexbaṛ n lxiṛ, mi ṛuḥeɣ si tmurt n Masidunya. Ula d yiwet n tejmaɛt ur temḥasab yid-i deg wayen i ț-id ikeččmen d wayen i tețțak d lemɛawna i wiyaḍ, anagar tajmaɛt-nwen ;

4:16 anagar kunwi i gxedmen ayagi, yerna mi lliɣ di temdint n Tisalunikya, snat n tikkal i yi-d-tceggɛem ayen akk ḥwaǧeɣ.

4:17 Mačči ɣef lweɛda i țnadiɣ, meɛna țnadiɣ ɣef lfayda tameqqrant ara yilin di nnfeɛ-nwen.

4:18 Yewweḍ-iyi-d kullec, aql-i di tawant, ur iyi-ixuṣṣ wacemma imi i yi-d-iwweḍ wayen i d-tefkam i gma tneɣ Ibafrudit ; ayen i d-yusan s ɣuṛ-wen am leɛṭeṛ n rriḥa yelhan, am wesfel i geqbel Sidi Ṛebbi yerna iɛǧeb-as.

4:19 Sidi Ṛebbi a wen-d-yefk ayen akk teḥwaǧem s wakken tella lɛaḍima n lbaṛakat-is, di Ɛisa Lmasiḥ.

4:20 Tamanegt i Sidi Ṛebbi baba tneɣ, si lǧil ɣer lǧil, amin !

4:21 Sellmet ɣef wegdud n Sidi Ṛebbi, wid akk yumnen s Ɛisa Lmasiḥ.

4:22 Atmaten akk yellan yid-i țsellimen fell-awen. Wid akk yellan seg ugdud n Sidi Ṛebbi țsellimen fell-awen, abeɛda wid yellan d iqeddacen n Qayṣer.

4:23 Ṛṛeḥma n Sidna Ɛisa Lmasiḥ aț-țili yid-wen.
