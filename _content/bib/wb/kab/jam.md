### Kabyle Bible: James

<span id="an_1:1">1:1</span> Nekk Yeɛqub aqeddac n Sidi Ṛebbi akk-d
Sidna Ɛisa Lmasiḥ, sslam-iw i tnac leɛṛac n wegdud n Sidi Ṛebbi
yenwezzaɛen di ddunit meṛṛa !

<span id="an_1:2">1:2</span> Ay atmaten, feṛḥet m'ara țɛeddayent
fell-awen teswiɛin ujeṛṛeb,

<span id="an_1:3">1:3</span> axaṭer teẓram belli ajeṛṛeb n liman nwen
yețțawi-d ṣṣbeṛ.

<span id="an_1:4">1:4</span> Meɛna ilaq ṣṣbeṛ-nwen a kkun issiweḍ
aț-țxedmem lecɣal yelhan iwakken aț-țilim d wid yennekmalen, aț-țǧehdem
ur kkun-yețxaṣṣa ula d acemma.

<span id="an_1:5">1:5</span> Ma yella yiwen deg-wen ixuṣṣ di lefhama, a
ț-yessuter i Sidi Ṛebbi i d-yețțaken i yemdanen meṛṛa mbla cceḥḥa d
ulummu, a s-ț-id-yefk.

<span id="an_1:6">1:6</span> Lameɛna Ilaq a ț-yessuter s liman mbla
ccekk ; axaṭer win yețcukkun icuba ɣer lemwaji n lebḥeṛ i greffed
yesrusu waḍu.

<span id="an_1:7">1:7</span> Amdan am wagi ɣas ur iṭṭamaɛ ara di Sidi
Ṛebbi a s-d-yefk kra,

<span id="an_1:8">1:8</span> axaṭer d amdan ur nesɛi ara yiwen n ṛṛay,
lecɣal-is ur seḥḥan ara.

<span id="an_1:9">1:9</span> Lameɛna igellil ilaq ad ifṛeḥ imi i t-irfed
Sidi Ṛebbi ;

<span id="an_1:10">1:10</span> ma d ameṛkanti ad ifṛeḥ imi t-id-issers
Sidi Ṛebbi axaṭer ad ifnu akken ifennu ujeǧǧig n lexla.

<span id="an_1:11">1:11</span>  M'ara d-yali yiṭij s leḥmu-ines
yesseṛɣayen, imɣi ad iqqaṛ, ajeǧǧig-is ad iɣli, ccbaḥa-ines a s-tṛuḥ.
Akka ara tedṛu d umeṛkanti , ad iṛuḥ a d-yeǧǧ deffir-es wayen akk yesɛa.

<span id="an_1:12">1:12</span> D aseɛdi wemdan i gteṭṭfen deg ujeṛṛeb ur
iɣelli ara ; axaṭer m'ara iɛeddi seg ujeṛṛeb, a s-d-tețțunefk tɛeṣṣabt n
tudert i gewɛed Sidi Ṛebbi i wid i t-iḥemmlen.

<span id="an_1:13">1:13</span> M'ara tețwaɣuṛṛem, ɣuṛ-wat aț-ținim : d
Ṛebbi i ɣ-iɣuṛṛen axaṭer Ṛebbi ur yezmir ara a t-iɣuṛṛ wemcum yerna
Nețța s yiman-is ur yețɣuṛṛu ula yiwen.

<span id="an_1:14">1:14</span> Lameɛna yal yiwen deg-nneɣ yețɛedday deg
ujeṛṛeb axaṭer d ccehwat n tnefsit-nneɣ i ɣ-ițɣuṛṛun.

<span id="an_1:15">1:15</span> Lebɣi n tnefsit yețțawi-d ddnub, ddnub mi
gețwaxdem, yețțawi-d lmut.

<span id="an_1:16">1:16</span> Ay atmaten-iw eɛzizen ur ɣellṭet ara deg
wayagi :

<span id="an_1:17">1:17</span> kra n tikci yelhan tețțas-ed seg igenwan,
s ɣuṛ baba Ṛebbi i d-ixelqen tafat n igenwan, win akken ur nețbeddil, ur
nețɣab am yiṭij iwakken a d-yeǧǧ ṭṭlam.

<span id="an_1:18">1:18</span> Yehwa-yas a ɣ-d-yefk tudert s wawal-is n
tideț, iwakken a nili d imezwura deg wayen akk i d-yexleq.

<span id="an_1:19">1:19</span> Ay atmaten-iw eɛzizen, teẓram belli yal
yiwen deg-wen ilaq ad yesmeḥsis, ur ilaq ara ad ireffu neɣ ad yețḥawal
ameslay ;

<span id="an_1:20">1:20</span> axaṭer amdan ireffun ur ixeddem ara ayen
yellan d lḥeqq ɣer Sidi Ṛebbi.

<span id="an_1:21">1:21</span> Daymi i glaq aț-țḍeggṛem seg ulawen-nwen
ayen akk ur nzeddig ara, d wayen yellan d cceṛ deg-wen ; meɛna qeblet s
wannuz awal i geẓẓan deg-wen i gzemren ad isellek lerwaḥ nwen.

<span id="an_1:22">1:22</span> Mačči d asmeḥses kan n wawal n Sidi Ṛebbi
i ɣer tella, lameɛna sbeggnet-ț-id di lecɣal-nwen ; neɣ m'ulac
ț-țikellax kan i tețkellixem iman-nwen.

<span id="an_1:23">1:23</span> Axaṭer win yesmeḥsisen i wawal n Sidi
Ṛebbi ur t-yexdim ara, icuba ɣer wemdan yețwalin udem-is di lemri akken
yella.

<span id="an_1:24">1:24</span> Mi gwala iman-is, iṛuḥ, yețțu imiren kan
amek akken i gella.

<span id="an_1:25">1:25</span> Ma d win yețmeyyizen akken ilaq ccariɛa
iṣeḥḥan, ccariɛa i d yețțaken tilelli, iteṭṭfen deg-s mačči s usmeḥses
kan ad yuɣal a ț-yețțu, meɛna ixeddem ayen i d-tenna, wagi ad yili d
aseɛdi di lecɣal-is meṛṛa.

<span id="an_1:26">1:26</span> Ma yella win iḥesben iman-is yetbeɛ abrid
n Sidi Ṛebbi yili ur yețțaṭṭaf ara iles-is, yesseɣlaḍ deg iman-is,
abrid-agi i gewwi ur iṣeḥḥa ara.

<span id="an_1:27">1:27</span> Atan webrid iṣeḥḥan yellan ț-țideț zdat
Sidi Ṛebbi baba-tneɣ : d m'ara terzum ɣef igujilen ț-țuǧǧal m'ara ilin
di lmeḥna, d uḥader n yiman-nwen ɣef wayen n diri ara d-yekken si
ddunit.

<span id="an_2:1">2:1</span> Ay atmaten, ur xeddmet ara lxilaf ger
watmaten, d ayen ur nlaq ara ad yili ɣer wid yețțamnen s Ssid-nneɣ
ameqqran, Ɛisa Lmasiḥ.

<span id="an_2:2">2:2</span> Yezmer lḥal a d-kecmen ɣer tejmaɛt-nwen sin
yemdanen, yiwen d ameṛkanti yelsa llebsa ifazen yeqqen taxatemt n ddheb,
wayeḍ d igellil llebsa-ines tcerreg ;

<span id="an_2:3">2:3</span> ma yella tuzzlem ɣer umeṛkanti-nni yelsan
llebsa ifazen tefkam-as lqima tennam-as : eyya-d aț-țeqqimeḍ dagi, d
amkan n lɛali ; ma d igellil-nni tennam-as : kečč bedd kan dagi neɣ
qqim-ed dagi ɣer iḍaṛṛen-nneɣ.

<span id="an_2:4">2:4</span> Ma yella akka, atan txeddmem lxilaf wway
gar-awen, eɛni mačči d lbaṭel i d itekken seg yir ixemmimen-nwen ?

<span id="an_2:5">2:5</span> Ay atmaten-iw eɛzizen, ḥesset-ed : Sidi
Ṛebbi yextaṛ igellilen n ddunit agi iwakken a ten-yerr d imeṛkantiyen di
liman, yerna ad weṛten tagelda i gweɛɛed i wid akk i t-iḥemmlen.

<span id="an_2:6">2:6</span> Ma d kunwi tḥeqṛem igellil ! Tqudṛem
imeṛkantiyen, wid i kkun yeṭṭalamen i kkun-yeẓẓuɣuṛen ɣer yexxamen n
ccṛeɛ.

<span id="an_2:7">2:7</span> D nutni daɣen i greggmen isem eɛzizen s
wayes i tneddhem ɣer Sidi Ṛebbi.

<span id="an_2:8">2:8</span> Ma yella txeddmem ayen i d-tenna ccariɛa n
Sidi Ṛebbi : ?emmel wiyaḍ am yiman-ik, d ayen yelhan i txedmem.

<span id="an_2:9">2:9</span> Meɛna ma yella txeddmem lxilaf ger
yemdanen, tdenbem yerna ccariɛa n Sidi Ṛebbi teḥkem fell-awen axaṭer
tɛuṣam-ț.

<span id="an_2:10">2:10</span> Ma yella win i gteddun deg webrid n
ccariɛa, ixulef ulamma d yiwen si lumuṛat-is, ihi yerẓa akk lumuṛat n
ccariɛa.

<span id="an_2:11">2:11</span> Win i d-yennan : Ur zennu ara, yenna-d
daɣen : Ur tneqqeḍ ara tamgeṛṭ. Ihi ma yella ur tezniḍ ara meɛna tenɣiḍ
tamgerṭ, atan teṛẓiḍ ccariɛa n Sidi Ṛebbi.

<span id="an_2:12">2:12</span> Ilaq aț-țeẓrem belli Sidi Ṛebbi a
kkun-iḥaseb s ccariɛa-ines yețțaken tilelli ɣef kra n wayen i
tețmeslayem d wayen i txeddmem ;

<span id="an_2:13">2:13</span> Sidi Ṛebbi ur yețḥunnu ara ɣef win ur
nețḥunnu ɣef wiyaḍ meɛna win yețḥunnun ɣef wiyaḍ, ur yețțagad ara ass n
lḥisab.

<span id="an_2:14">2:14</span> Ay atmaten, d acu n lfayda ara yesɛu
wemdan ma yenna-d umneɣ s Ṛebbi, m'ur t-id-isbeggen ara s lecɣal-is ?
Eɛni liman am wagi a t-isellek ?

<span id="an_2:15">2:15</span> Yezmer lḥal ad yili yiwen seg watmaten
neɣ si tyessetmatin xuṣṣen di llebsa, lluẓen ur sɛin ara ayen ara ččen ;

<span id="an_2:16">2:16</span> ma yella yiwen deg-wen yenna-yasen kan :
« qqimet di lehna, sseḥmut teččem aț-țeṛwum » yili nețța ur sen-yefki
ara ayen ḥwaǧen, d acu n lfayda yellan deg imeslayen-agi ?

<span id="an_2:17">2:17</span> Liman ɣer Ṛebbi akka i gella : ma yella
ur t-id-sbeggnen ara lecɣal nwen, liman-agi yemmut

<span id="an_2:18">2:18</span> Meɛna ma yenna walebɛaḍ : kečč tesɛiḍ
liman ma d nekk xeddmeɣ lecɣal ; nekk a s-rreɣ i wagi : sken-iyi-d
liman-inek mbla lecɣal, nekk a k-ed ssekneɣ liman-iw s lecɣal-iw yelhan.

<span id="an_2:19">2:19</span> Tumneḍ belli yiwen n Ṛebbi kan i gellan ?
d ayen yelhan ; ula d ccwaṭen umnen yis yerna țergigin seg-s.

<span id="an_2:20">2:20</span> Teggumam aț-țfehmem ay imdanen ! Ilaq
aț-țeẓrem belli liman mbla lecɣal n lɛali, ur infiɛ ara.

<span id="an_2:21">2:21</span> Mmektit-ed Ibṛahim baba-tneɣ yețwaḥesben
d aḥeqqi ɣef ddemma n lecɣal-is m'akken i d-iddem mmi-s Isḥaq a t-yezlu
d asfel i Sidi Ṛebbi.

<span id="an_2:22">2:22</span> Twalam belli liman-is d lecɣal is
ddukklen, s lecɣal-is i gennekmal liman-is.

<span id="an_2:23">2:23</span> Akka i gedṛa wayen d-nnant tira iqedsen
fell-as : Ibṛahim yesɛa liman di Ṛebbi, daymi yenneḥsab d aḥeqqi ;
yețțusemma d aḥbib n Ṛebbi.

<span id="an_2:24">2:24</span> Twalam ihi ! Amdan yețban-ed d aḥeqqi
zdat Sidi Ṛebbi s lecɣal yelhan mačči kan s liman-is.

<span id="an_2:25">2:25</span> Ula d Raḥab yellan d yir tameṭṭut,
tețwaḥseb ț-țaḥeqqit zdat Sidi Ṛebbi ɣef ddemma n ccɣel-nni n lɛali i
texdem asm'akken i tesṭerḥeb s yemceggɛen n at Isṛail mi sen-temla abrid
nniḍen ara yawin iwakken ad ṛuḥen.

<span id="an_2:26">2:26</span> Lǧețța ur nesɛi ara Ṛṛuḥ, temmut, akken
daɣen, liman ur nesɛi ara lecɣal yelhan yemmut.

<span id="an_3:1">3:1</span> Ay atmaten-iw, ur țnadit ara aț-țuɣalem
meṛṛa d wid yesselmaden, axaṭer teẓram belli d nukni s wid yesselmaden,
ara yețțuḥasben akteṛ n wiyaḍ,

<span id="an_3:2">3:2</span> axaṭer nɣelleḍ meṛṛa, yal yiwen deg-nneɣ
amek i gɣelleḍ. Win ur nɣelleḍ ara deg umeslay-is d amdan ikemlen, ula d
lǧețța-s yezmer ad iḥkem deg-s.

<span id="an_3:3">3:3</span> Ma yella nerra aleggam deg yimi n uɛewdiw
iwakken a ɣ-yaɣ awal, lǧețța-ines meṛṛa txeddem akken nebɣa.

<span id="an_3:4">3:4</span> Walit daɣen lbabuṛ, ɣas d ameqqran, yețsuḍu
deg-s waḍu iǧehden, meɛna s yiwen ujeggu d ameẓyan, lqebṭan-ines
issedduy-it ɣer wanda yebɣa.

<span id="an_3:5">3:5</span> Akken daɣen ula d iles ; ɣas mecṭuḥ deg
yiman-is, yețzuxxu s lecɣal imeqqranen. Walit times tamecṭuḥt i gzemren
aț-țesseṛɣ tiẓgi tameqqrant.

<span id="an_3:6">3:6</span> Iles daɣen ț-țimes, d bab n lbaṭel, yesɛa
amkan di lǧețța-nneɣ yerna yessamas akk taṛwiḥt-nneɣ, yesseṛɣay akk
tudert-nneɣ s tmes n ǧahennama i seg d-yuɣ aẓar.

<span id="an_3:7">3:7</span> Amdan yezmer ad iḥeṛṛeb yal ṣṣenf n lewḥuc
: ama d wid iteddun ɣef ṛebɛa iḍaṛṛen, ama d ifṛax, ama d izerman ama d
lewḥuc n lebḥeṛ yerna ț-țideț iɣleb-iten meṛṛa.

<span id="an_3:8">3:8</span> Ma d iles, ulac amdan i gzemren a t-yeɣleb.
Yeččuṛ d ssem ineqqen ; d lehlak ur nesɛi ddwa.

<span id="an_3:9">3:9</span> Yis i nḥemmed Sidi Ṛebbi baba tneɣ, yis
daɣen i nenneɛɛel imdanen i d-ixleq Sidi Ṛebbi ɣer ccbiha-s.

<span id="an_3:10">3:10</span> Seg yimi-agi i d-țeffɣent ddeɛwat n lxiṛ
d nneɛlat. Ay atmaten-iw, ur ilaq ara ad yili wakka.

<span id="an_3:11">3:11</span> Eɛni yezmer yiwen n lɛinseṛ a d-yefk aman
ḥlawen akk-d wid ṛẓagen ?

<span id="an_3:12">3:12</span> Ay atmaten-iw, ulac taneqleț ara d-yefken
azemmur, neɣ tara ara d-yefken tibexsisin, lɛinseṛ meṛṛiɣen ur yezmir
ara a d-yefk aman yelhan.

<span id="an_3:13">3:13</span> Ma yella yiwen deg-wen yefhem yetɛeqqel,
ilaq a t-id-isbeggen s tikli-ines yelhan akk-d lecɣal i gxeddem s tmusni
d wannuz.

<span id="an_3:14">3:14</span> Meɛna ma yella ul-nwen yeččuṛ ț-țismin
tirẓaganin yerna tebɣam a d-tekkem sennig wiyaḍ ; ur xeddmet ara ccan i
yiman-nwen, ur skiddibet ara, ur nekkṛet ara tideț.

<span id="an_3:15">3:15</span> Lefhama am tagi ur d-tekki ara s ɣuṛ
Ṛebbi meɛna d lefhama n ddunit akk-d yemdanen, tekka-d daɣen s ɣuṛ
Cciṭan.

<span id="an_3:16">3:16</span> Axaṭer anda llant tismin d wemḥizwer,
yella ussexṛeb d mkul ccɣel n diri yellan.

<span id="an_3:17">3:17</span> Lefhama n bab igenwan uqbel kullec teṣfa,
ț-țamhennit, ț-țuṛẓint, teččuṛ d leḥnana akk-d lecɣal yelhan, ur tesɛi
timrurit ur tesɛi sin wudmawen.

<span id="an_3:18">3:18</span> Imawlan n lehna țeẓẓun s lehna ayen i
d-issemɣayen ayen yellan d lḥeqq ɣer Sidi Ṛebbi.

<span id="an_4:1">4:1</span> Seg uxemmem i tețxemmimem ɣef yiman-nwen
kan i d-itekk umennuɣ akk-d imenɣi gar-awen.

<span id="an_4:2">4:2</span> Tețmennim aṭas leḥwayeǧ lameɛna ur tezmirem
ara tent-tesɛum ; tneqqem, tețțasmem ur d-tețțawim ayen tebɣam ;
tețnaɣem yerna tețțemyenɣam, ur kkun-id-ițțaweḍ ara wayen tebɣam axaṭer
ur tessuturem ara ɣer Sidi Ṛebbi.

<span id="an_4:3">4:3</span> M'ara tessutrem kra ur a wen-d- țunefk ara,
axaṭer ur tessutrem ara akken ilaq, tessutrem kan wayen yellan d
nnfeɛ-nwen.

<span id="an_4:4">4:4</span> Ay at leɛṛuṛ ! Ur teẓrim ara belli win
iḥemmlen ddunit yekṛeh Sidi Ṛebbi ? Ihi, win iḥemmlen ddunit-agi d aɛdaw
n Sidi Ṛebbi.

<span id="an_4:5">4:5</span> Tɣilem ayen akken yuran di tira iqedsen ur
yesɛi ara lqima : belli Sidi Ṛebbi iḥemmel Ṛṛuḥ iqedsen i d-iceggeɛ ad
izdeɣ deg-wen armi yețțasem seg win ara iṭṭfen amkan-is ?

<span id="an_4:6">4:6</span> Lameɛna Sidi Ṛebbi yețțak-ed ṛṛeḥma
tameqqrant, akken yura di tira iqedsen : S idi Ṛebbi ur iqebbel ara wid
yessemɣaṛen iman-nsen meɛna yețțak-ed ṛṛeḥma-s i wid yessanazen
iman-nsen.

<span id="an_4:7">4:7</span> Ihi ḍuɛet Sidi Ṛebbi, ur țaǧǧat ara Cciṭan
a kkun-yeɣleb dɣa ad yerwel fell-awen.

<span id="an_4:8">4:8</span> Qeṛṛbet ɣer Sidi Ṛebbi, nețța a d-iqeṛṛeb
ɣuṛ-wen. Ssizdeget ifassen-nwen ay imednuben, ṣṣfut ulawen-nwen, ay at
sin udmawen.

<span id="an_4:9">4:9</span> Walit lɣelṭat-nwen, meǧdet, ilaq taḍsa-nwen
aț-țuɣal d imeṭṭi, lfeṛḥ nwen ad yuɣal d leḥzen.

<span id="an_4:10">4:10</span> Anzet i Sidi Ṛebbi, nețța a kkun yerfed.

<span id="an_4:11">4:11</span> Ay atmaten, ur kkatet ara wa deg wa, win
ara yewten di gma-s neɣ iḥuseb-it, yewwet-ed di ccariɛa yerna iḥuseb-iț.
Ihi ma yella tewtem di ccariɛa, ur tellim ara seg wid ixeddmen wayen i
d-tenna, meɛna seg wid yekkaten deg-s.

<span id="an_4:12">4:12</span> Kečč yețḥasaben gma-k d acu i tḥesbeḍ
iman-ik ? D Sidi Ṛebbi kan i gețḥasaben i d-yefkan ccariɛa, d nețța kan
i gețselliken i gețțaǧan i lmut.

<span id="an_4:13">4:13</span> Slet-ed tura kunwi i geqqaṛen : ass-a neɣ
azekka a nṛuḥ ɣer temdint ihina, a neqqim aseggas, a netjaṛ iwakken a
d-nerbeh idrimen !

<span id="an_4:14">4:14</span> Kunwi ur neẓri ara d acu ara wen-yedṛun
azekka ! Kunwi yecban tagut i d-ițalin tɣelli !

<span id="an_4:15">4:15</span> Meɛna ilaq aț-țeqqaṛem : ma yebɣa Sidi
Ṛebbi, a nɛic am akka, ma yebɣa daɣen, a nexdem aya neɣ aya.

<span id="an_4:16">4:16</span> Meɛna kunwi tesɛam ixemmimen n zzux deg
ulawen-nwen. Kra yellan d zzux am wagi diri-t.

<span id="an_4:17">4:17</span> Ma yella win yessnen ad ixdem ayen yelhan
ur t-yexdim ara, yedneb.

<span id="an_5:1">5:1</span> Slet-ed tura ula d kunwi ay imeṛkantiyen :
meǧdet ɣef ddemma n txeṣṣarin ara d-yeɣlin fell-awen !

<span id="an_5:2">5:2</span> Cci-nwen yerka, llebsa-nwen tečča-ț tumeṭ !

<span id="an_5:3">5:3</span> Ddheb d lfeṭṭa-nwen ṣeddeḍen ; d ṣḍiḍ-nsen
ara icehden fell-awen, ara kkun-iččen am tmes. Tjemmɛem cci deg
ussan-agi ineggura n ddunit !

<span id="an_5:4">5:4</span> Teččam lḥeqq n wid imegren igran-nwen,
slet-asen i wigi ițcetkin ɣef lbaṭel-nwen, Sidi Ṛebbi Bab n tezmert
yesla-d i leɛyaḍ-nsen.

<span id="an_5:5">5:5</span> Ddunit-nwen tesɛeddam-ț di lxiṛ, txedmem
ayen akk mennan wulawen nwen ; tṛebbam tassemt i wass aneggaru.

<span id="an_5:6">5:6</span> Tḥekmem, tenɣam iḥeqqiyen yerna ur
kkun-xuṣmen ara.

<span id="an_5:7">5:7</span> Ay atmaten, ṣebṛet ihi alamma d asm' ara
d-yuɣal Ssid-nneɣ. Walit amek i gețṛaǧu ufellaḥ lɣella n wakal-is,
iṣebbeṛ alamma yewwet-ed ugeffur n lexṛif ț-țefsut.

<span id="an_5:8">5:8</span> Ula d kunwi ṣebṛet, sǧehdet iman nwen,
axaṭer tuɣalin n Ssid-nneɣ teqṛeb-ed.

<span id="an_5:9">5:9</span> Ur țemcetkayet ara wway gar-awen iwakken ur
kkun-ițḥasab ara Sidi Ṛebbi ; axaṭer Bab n lḥisab, ibedd-ed ɣer tewwurt
!

<span id="an_5:10">5:10</span> Ay atmaten-iw, mmektit-ed lenbiya i
d-iceggeɛ Sidi Ṛebbi tecbum-d ṣṣbeṛ akk-d liman i sɛan di teswiɛin n
leɛtab.

<span id="an_5:11">5:11</span> Ihi amarezg n wid yeṭṭfen ur fcilen ara.
Teslam ayen i d-qqaṛen ɣef ṣṣbeṛ n Sidna Yub, yerna twalam daɣen d acu i
geqsed Sidi Ṛebbi a s-t-id-yefk, axaṭer Sidi Ṛebbi yeččuṛ d leḥnana
akk-d ṛṛeḥma.

<span id="an_5:12">5:12</span> Ay atmaten-iw, ur țgallat s igenni, ur
țgallat s lqaɛa neɣ s wayen nniḍen, meɛna init kan ih neɣ ala, iwakken
ur kkun-ițḥasab ara Sidi Ṛebbi.

<span id="an_5:13">5:13</span> Ma yella yenneɛtab yiwen deg-wen, ad
yedɛu ɣer Sidi Ṛebbi. Ma yella win yellan di lfeṛḥ, ad yeḥmed Sidi Ṛebbi
s ccnawi.

<span id="an_5:14">5:14</span> Ma yella yiwen deg-wen yehlek, ad issiwel
i imeqqranen n tejmaɛt, ad dɛun ɣer Ṛebbi fell-as, ad dehnen aqeṛṛuy-is
s zzit s yisem n Sidi Ṛebbi ;

<span id="an_5:15">5:15</span> ma yella dɛan ɣer Sidi Ṛebbi s liman,
țeklen fell-as, amuḍin-nni ad yețțusellek ; Sidi Ṛebbi a t-yesseḥlu
yerna a s-isemmeḥ ddnubat-is.

<span id="an_5:16">5:16</span> ?emqiṛit ddnubat-nwen wway gar-awen, wa
ad yedɛu i wa s lxiṛ ɣer Sidi Ṛebbi iwakken aț-țeḥlum. Ddeɛwat n win
yellan d aḥeqqi zdat Sidi Ṛebbi, sɛant tazmert tameqqrant.

<span id="an_5:17">5:17</span> Nnbi Ilyas yella d argaz am nukni : meɛna
mi gessuter ɣer Sidi Ṛebbi s wul-is iwakken ur d-tɣelli ara lehwa, ur
d-teɣli ara tlata iseggasen d sețța waguren

<span id="an_5:18">5:18</span> Yessuter daɣen a d-teɣli lehwa dɣa
teɣli-d, lqaɛa tefka-d leṛẓaq-is.

<span id="an_5:19">5:19</span> Ay atmaten-iw, ma yella yiwen deg-wen
iɛṛeq-as webrid n Sidi Ṛebbi, yerra-t-id wayeḍ ɣer webrid,

<span id="an_5:20">5:20</span> ilaq aț-țeẓrem belli win ara d-yerren
yiwen umednub ɣer webrid, isellek-it-id si lmut yerna aṭas si ddnubat-is
ara yețțusemḥen.
