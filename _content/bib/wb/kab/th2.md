### Kabyle Bible: 2 Thessalonians

<span id="an_1:1">1:1</span> Nekk Bulus, Silwan akk-d Timuti, i tejmaɛt
n watmaten n temdint n Tiṣalunik yumnen s Sidi Ṛebbi d Ssid-nneɣ Ɛisa
Lmasiḥ.

<span id="an_1:2">1:2</span> Ṛṛeḥma ț-țalwit a wen-d țțunefkent s ɣuṛ
Sidi Ṛebbi.

<span id="an_1:3">1:3</span> Ay atmaten eɛzizen, imi liman-nwen simmal
yețnerni, simmal tețțemyeḥmalem wway gar-awen, eɛni mačči d lḥeqq ma
nețḥemmid Sidi Ṛebbi fell-awen am iḍ am ass ?

<span id="an_1:4">1:4</span> Newweḍ armi nețzuxxu yis-wen di yal tajmaɛt
n watmaten, nețțawi-kkun d lemtel axaṭer ɣas akken tenneɛtabem
tețwaqehṛem, tṣebrem, teṭṭfem di liman-nwen.

<span id="an_1:5">1:5</span> Sidi Ṛebbi d bab n lḥeqq, ma yella iɛemmed
aț-țenneɛtabem akkagi, yebɣa a kkun-yerr d wid yuklalen tagelda-ines ;
ț-țagelda-agi i ɣef tenneɛtabem.

<span id="an_1:6">1:6</span> Axaṭer Sidi Ṛebbi yellan d Bab n lḥeqq, a
d-yerr leqheṛ ɣef wid i kkun-ițqehhiṛen,

<span id="an_1:7">1:7</span> a kkun-iṣebber yid-nneɣ, kunwi yețwaqehṛen.
Ayagi a d-idṛu asm'ara d-yuɣal Ssid-nneɣ Ɛisa Lmasiḥ, m'ara d-iṣubb seg
igenwan nețța d lmalayekkat-is

<span id="an_1:8">1:8</span> di tlemmast n uḥeǧǧaǧu n tmes iwakken ad
iɛaqeb wid iɛuṣan Sidi Ṛebbi, ur neqbil ara lexbaṛ n lxiṛ n Sidna Ɛisa.

<span id="an_1:9">1:9</span> Lɛiqab ara ten-id-yasen, ț-țawaɣit ur
nețfaka, Sidi Ṛebbi a ten-iḥeṛṛem ɣef udem-is, ur țwalin ara lɛaḍima-s.

<span id="an_1:10">1:10</span> Ayagi a d-idṛu asm'ara d-yas Sidna Ɛisa ;
ass-nni wid yumnen yis ad tɛeǧǧben deg-s, a t-ḥemden m'ara t-walin di
lɛaḍima-s ! Ula d kunwi aț-țilim gar-asen axaṭer tumnem s lexbaṛ n lxiṛ
i wen-nbecceṛ.

<span id="an_1:11">1:11</span> ?ef ayagi i ndeɛɛu ɣer Sidi Ṛebbi
fell-awen am iḍ am ass, iwakken a kkun-yerr d wid yuklalen tudert i ɣer
wen-d-issawel. Nessutur-as daɣen a wen-d-ifk tazmert, iwakken aț-țxedmem
ayen akken tessaramem n wayen yelhan, yerna ad ibarek lecɣal txeddmem
deg webrid n Sidna Ɛisa.

<span id="an_1:12">1:12</span> S wakka imdanen ad ḥemden isem-is ɣef
ddemma n tikli-nwen yelhan, kunwi daɣen aț-țwaɛuzzem s ṛṛeḥma n Sidi
Ṛebbi akk-d ṛṛeḥma n Ssid-nneɣ Ɛisa Lmasiḥ.

<span id="an_2:1">2:1</span> ?ef wayen yeɛnan tuɣalin n Ssid nneɣ Ɛisa
Lmasiḥ ț-țemlilit-nneɣ yid-es, di leɛnaya-nwen ay atmaten :

<span id="an_2:2">2:2</span> ḥadret a kkun-iffeɣ leɛqel neɣ a wen-ikcem
ukukru, ma yella win i wen-d-yennan belli ass n tuɣalin n Sidna Ɛisa
yewweḍ-ed ; yezmer lḥal ad yili win i wen-t-id-yennan neɣ win i t-iwalan
deg uweḥḥi, neɣ daɣen a wen-arun tabṛaț am akken d nukni i
wen-țin-yuran.

<span id="an_2:3">2:3</span> Akken yebɣu yili lḥal ur țțaǧat yiwen a
kkun-ikellex, axaṭer ass-nni ur d-ițaweḍ ara uqbel ussan n tijehli,
uqbel a d-iban wemcum ijehlen i gețṛaǧu nnger.

<span id="an_2:4">2:4</span> Win akken ara yessimɣuṛen iman is sennig
wayen meṛṛa iɛebbden yemdanen, ad yekcem ɣer lǧameɛ iqedsen n Ṛebbi ad
yerr iman-is deg wemkan n Ṛebbi. f+

<span id="an_2:5">2:5</span> Ur tecfim ara belli xebbṛeɣ kkun id yakan
ɣef ayagi asm'akken lliɣ yid-wen ?

<span id="an_2:6">2:6</span> Teẓram acu i t-iṭṭfen ur d-ițban ara tura
alamma yewweḍ-ed lweqt-is !

<span id="an_2:7">2:7</span> Tazmert n cceṛ yellan d lbaḍna, tebda
ccɣel-is, ilaq kan ayen i ț-iṭṭfen ar tura ad ițwakkes.

<span id="an_2:8">2:8</span> Imiren amcum-nni amejhul a d- ban, dɣa
Sidna Ɛisa a t-iṣuḍ s nnefs-is a t-issenger, a t-imḥu s ufeǧǧeǧ n
lɛaḍima-s asm'ara d-yuɣal.

<span id="an_2:9">2:9</span> Amcum-agi amejhul a d-yas s tezmert n
Cciṭan, ad ixdem leɛǧubat d lbeṛhanat s wayes ara ikellex,

<span id="an_2:10">2:10</span> ad issexdem lekdeb d lbaṭel iwakken ad
ikellex wid iḍaɛen, axaṭer ur qbilen ara leḥmala n tideț s wayes ara
țwaselken.

<span id="an_2:11">2:11</span> Daymi Sidi Ṛebbi iceggeɛ-asen-d tazmert n
ṭṭlam aț-țexdem deg-sen, iwakken ad amnen s wayen yellan d lekdeb.

<span id="an_2:12">2:12</span> S wakka wid akk ur numin ara s tideț, ad
țțuɛaqben imi țțafen lfeṛḥ-nsen anagar di lbaṭel.

<span id="an_2:13">2:13</span> Ay atmaten ɛzizen ɣer Ṛebbi, ilaq-aɣ a
nețḥemmid kull ass Sidi Ṛebbi fell-awen, axaṭer ixtaṛ-ikkun d imezwura
iwakken aț-țețțusellkem s Ṛṛuḥ iqedsen i kkun-iṣeffan s liman di tideț.

<span id="an_2:14">2:14</span> Sidi Ṛebbi issawel-awen-d aț țekkim di
tmanegt n Ssid-nneɣ Ɛisa Lmasiḥ, s lexbaṛ-agi n lxiṛ i wen-d newwi.

<span id="an_2:15">2:15</span> Ihi ay atmaten ḥerzet, ṭṭfet deg uselmed
i wen-d-newwi, ama s yimi nneɣ ama s tebṛatin-nneɣ.

<span id="an_2:16">2:16</span> Nessutur di Ssid-nneɣ Ɛisa Lmasiḥ akk-d
baba-tneɣ Ṛebbi i ɣ-iḥemmlen, i ɣ-d-ifkan s ṛṛeḥma ines ṣṣbeṛ d usirem
ițdumun,

<span id="an_2:17">2:17</span> ad țțuṣebbṛen wulawen-nwen, a
wen-d-tețțunefk tezmert s wayes ara txeddmem anagar ayen yelhan ama di
lecɣal-nwen ama deg imeslayen-nwen.

<span id="an_3:1">3:1</span> ?ef wayen nniḍen ay atmaten, dɛut Sidi
Ṛebbi fell-aneɣ iwakken awal-is ad ițțubecceṛ, ad yaweḍ ɣer yal amkan,
ad ițwaɛuzz am akken i t-tɛuzzem kunwi.

<span id="an_3:2">3:2</span> Dɛut daɣen iwakken a ɣ-isellek si ger
ifassen n yemcumen i ɣ-ibɣan cceṛ, axaṭer llan wid ur nețțamen ara.

<span id="an_3:3">3:3</span> Lameɛna Sidi Ṛebbi d bab n lețkal, a
kkun-isseǧhed, a kkun iḥader ɣef wemcum.

<span id="an_3:4">3:4</span> Nesɛa laman ɣuṛ-es fell-awen belli txeddmem
yerna aț-țkemlem aț-țxedmem ayen i ɣef i kkun-nweṣṣa.

<span id="an_3:5">3:5</span> Ndeɛɛu ɣer Sidi Ṛebbi iwakken ad iččaṛ
ulawen-nwen s leḥmala-ines, iwakken aț-țesɛum ṣṣbeṛ n Lmasiḥ. f+

<span id="an_3:6">3:6</span> Ay atmaten a kkun-nenhu s yisem n Sidna
Ɛisa Lmasiḥ : sbeɛdet iman-nwen ɣef atmaten imeɛdazen, ițɛicin akken
bɣan, mačči am akken i wen-t-nesselmed.

<span id="an_3:7">3:7</span> Teẓram s yiman nwen amek i glaq a
ɣ-tɛandem, axaṭer nukni ur nɛac ara deg ussexṛeb asmi nella yid-wen.

<span id="an_3:8">3:8</span> Ur neḍmiɛ ula deg yiwen aɣṛum-nneɣ, lameɛna
nețɛețțib iman-nneɣ di lxedma iḍ d wass, iwakken ur nețțili ț-țaɛekkumt
ula i yiwen.

<span id="an_3:9">3:9</span> Yerna lemmer nessuter-awen a ɣ-tɛiwnem, d
lḥeqq fell-aɣ ; meɛna nexdem akkenni iwakken a wen-nili d lemtel.

<span id="an_3:10">3:10</span> Asmi nella gar-awen nweṣṣa kkun belli «
win ur nebɣi ara ad ixdem ur ilaq ara ad yečč. »

<span id="an_3:11">3:11</span> Axaṭer nesla belli llan gar-awen
imeɛdazen ur nebɣi ara ad xedmen meɛna ggaren iman-nsen di lecɣal i
ten-ixḍan.

<span id="an_3:12">3:12</span> Imdanen-agi a ten-nweṣṣi s yisem n
Ssid-nneɣ Ɛisa Lmasiḥ : ad xedmen akken ilaq iwakken a d-ḥeṛṛen aɣṛum
nsen.

<span id="an_3:13">3:13</span> Ma d kunwi ay atmaten xedmet lxiṛ, ur
ɛeggut ara !

<span id="an_3:14">3:14</span> Win ur neqbil ara ayen i d-nenna di
tebṛaț-agi, jebdet iman-nwen fell-as, ur țeddut ara yid-es iwakken ad
inneḥcam, ad issetḥi.

<span id="an_3:15">3:15</span> Ur t-ḥețțbet ara d aɛdaw, meɛna nhut-eț
am yiwen seg watmaten-nwen.

<span id="an_3:16">3:16</span> Nessutur di Sidi Ṛebbi bab n talwit a
wen-d-yefk lehna kull ass, di yal tagniț ! Sidi Ṛebbi ad yili yid-wen
meṛṛa !

<span id="an_3:17">3:17</span> Nekk Bulus, uriɣ-awen sslam-agi s
ufus-iw, ț-țagi i ț-țira-inu di yal tabṛaț i wen-ceggɛeɣ.

<span id="an_3:18">3:18</span> Ṛṛeḥma n Ssid-nneɣ Ɛisa Lmasiḥ a d-ters
fell-awen meṛṛa !
