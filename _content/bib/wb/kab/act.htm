Kabyle Bible: Acts

1:1 A Tawfilus, di tektabt-inu tamezwarut, ḥkiɣ-ed ayen ixdem Sidna Ɛisa d wayen yesselmed seg wass amezwaru armi d ass i deg yețwarfed ɣer igenni.

1:2 Uqbel ad ițwarfed, yefka lumuṛat-is s Ṛṛuḥ iqedsen i wid i gextaṛ ad ilin d ṛṛusul.

1:3 Mi d-iḥya si ger lmegtin, i nutni daɣen iwumi i d-isban iman-is belli yedder ; ṛebɛin wussan nețța ițban asen-d yerna ițmeslay-asen-d ɣef wayen yeɛnan tagelda n Sidi Ṛebbi.

1:4 Yiwen wass, mi gella itețț yid-sen, yefka-yasen lameṛ-agi : Ur țeffɣet ara si temdint n Lquds, lameɛna eṛǧut lemɛahda n Baba Ṛebbi, tin akken i wen-d nniɣ ;

1:5 « Yeḥya isseɣḍas imdanen deg aman, ma d kunwi, di kra n wussan aț-țețwaɣeḍsem s Ṛṛuḥ iqedsen.»

1:6 Mi llan ṛṛusul nnejmaɛen yid-es, steqsan-t : A Sidi, deg ussan-agi ara d terreḍ tageldit i wegdud n wat Isṛail ?

1:7 Yerra-yasen : Ur țnadit ara aț-țeẓrem ayen yeɛnan lewqat d wussan i ghegga Baba Ṛebbi s lḥekma-ines si zik, ayagi mačči d ccɣel-nwen.

1:8 Kunwi a wen-d-tețțunefk tezmert n Ṛṛuḥ iqedsen ara d-yersen fell-awen, imiren aț-țilim d inigan-iw di temdint n Lquds, di tmurt n Yahuda ț-țmurt n Samarya, aț-țawḍem alamma d ixfawen n ddunit.

1:9 Akken i d-yenna imeslayen-agi, Sidna Ɛisa yețwarfed ɣer igenni nutni țmuqulen deg-s armi i t-tɣumm tagut.

1:10 Mazal ṛeṣṣan allen-nsen deg-s ɣer tegnaw, mi sen-d-dehṛen sin yergazen s llebsa tamellalt, nnan-asen :

1:11 Ay imezdaɣ n tmurt n Jlili, acuɣeṛ mazal-ikkun tețmuqqulem ɣer tegnaw ? Akken i twalam Sidna Ɛisa yuli ɣer igenni, akken daɣen ara t-twalim asmi ara d-yuɣal.

1:12 Seg iɣil uzemmur, uɣalen ɣer temdint n Lquds ibeɛden azal n tikli n wass n westeɛfu.

1:13 Mi wwḍen ɣer temdint n Lquds, ulin ɣer texxamt anda nnumen țnejmaɛen. Yella gar-asen Buṭrus, Yuḥenna, Yeɛqub, Andriyus, Filibas, Suma, Bartelmay, Matta, Yeɛqub mmi-s n Ḥalfi, Semɛun awaṭani akk-d Yahuda mmi-s n Yeɛqub.

1:14 Țnejmaɛen s yiwen n ṛṛay iwakken ad deɛɛun ɣer Sidi Ṛebbi nutni d kra n tlawin, llan yid-sen daɣen watmaten n Sidna Ɛisa akk-d yemma-s, Meryem.

1:15 Deg ussan-nni, llan nnejmaɛen azal n meyya uɛecrin n wayetmaten, yekker-ed Buṭrus gar-asen, yenna-d :

1:16 Ay atmaten, ilaq a d-yedṛu wayen i d-yenna Ṛṛuḥ iqedsen si zik seg yimi n Sidna Dawed, ɣef wayen yeɛnan Yudas i d-yezwaren zdat n wid i geṭṭfen Sidna Ɛisa.

1:17 Yudas-agi yella d yiwen seg neɣ, yerna ițekki di leqdic-nneɣ ɣef Sidi Ṛebbi.

1:18 Argaz-agi yuɣ akal s yedrimen-nni n lexdeɛ, yuɣal yeɣli ɣef wudem, ifelleq uɛebbuḍ-is, mmaren-d akk ijuɣdan-is.

1:19 Imezdaɣ n temdint n Lquds slan akk s wannect-agi, daymi i semman i wemkan-nni s tmeslayt-nsen : Ḥaqel damma , yeɛni iger n idammen.

1:20 Am akken yura di tektabt n Sabur : Axxam-is ad yexlu, ur t-izeddeɣ yiwen. Yura daɣen : Wayeḍ ad yeṭṭef amkan-is.

1:21 Ilaq-aɣ ihi a d-nextiṛ yiwen seg wid yellan yid-nneɣ mi nțeddu d Sidna Ɛisa di mkul amkan,

1:22 seg wasmi i t-isseɣḍes Yeḥya aɣeṭṭas armi d ass i deg yețwarfed ɣer igenni ; iwakken ad yili am nukkni d inigi n ḥeggu-ines.

1:23 Sbedden-d sin yergazen : Yusef yețțusemman Barsaba iwumi qqaṛen daɣen « Aḥeqqi » akk-d Matyas.

1:24 Dɛan ɣer Sidi Ṛebbi s imeslayen agi : A Sidi Ṛebbi, kečč yesnen ulawen n yemdanen meṛṛa, sbeggen aɣ-ed anwa deg-sen i gwulmen

1:25 iwakken ad iṭṭef amkan di lecɣal n ṛṛusul i ɣef yeṭṭaxeṛ Yudas, imi nețța iṛuḥ ɣer wemkan i s-ilaqen.

1:26 Gren tasɣaṛt, tṣaḥ-ed i Matyas ; dɣa rnan-t-id ɣer ḥdac-nni n ṛṛusul.

2:1 Ass n Lɛid n wass wis xemsin, inelmaden llan nnejmaɛen akk deg yiwen n wemkan.

2:2 Taswiɛt kan, yekka-d yiwen n lḥess ameqqran seg igenni am waḍu iǧehden, yeččuṛ akk axxam-nni anda qqimen.

2:3 Sehṛen-asen-d am akken d ilsawen n tmes ițemfaṛaqen wway gar-asen, rsen-d ɣef yal yiwen deg sen.

2:4 Sčuṛen akk s Ṛṛuḥ iqedsen, bdan țmeslayen s tutlayin nniḍen, yal yiwen seg-sen iheddeṛ s tutlayt i s-d-yefka Ṛṛuḥ iqedsen.

2:5 Deg ussan-nni, di temdint n Lquds, llan yemdanen n wat Isṛail yețḍuɛun Sidi Ṛebbi, i d-yusan si mkul tamurt n ddunit.

2:6 Mi slan i lḥess-nni, lɣaci uzzlen d meṛṛa ; dehcen, axaṭer sellen-asen țmeslayen-d s tutlayt n tmurt i deg țɛicin.

2:7 Wehmen, țɛeǧben, qqaṛen wway gar-asen : Imdanen-agi yețmeslayen mačči n tmurt n Jlili ?

2:8 Amek armi yal yiwen deg-nneɣ isell-asen țmeslayen s tmeslayt-is ?

2:9 Llan gar-aneɣ yemdanen i d yusan si tmurt n Bartas, si tmurt n Midyas, si tmurt n Ilam, kra nniḍen si tmurt n Mizubuṭamya, si tmurt n Yahuda akk-d tmurt n Kafadusya, wiyaḍ si tmurt n Tqenṭert, si tmurt n Asya,

2:10 si tmurt n Frijya akk-d Bamfilya. Usan-d daɣen si tmurt n Maṣer, si leǧwahi n Qiṛwan di tmurt n Libya, widak i d-yusan si temdint n Ṛuma ;

2:11 llan daɣen wat Isṛail akk-d wid itabaɛen di ddin nsen, llan wid n tegzirt n Kritus akk-d waɛṛaben. Amek armi i sen-nesla țmeslayen s tutlayin-nneɣ meṛṛa ɣef leɛǧayeb n Sidi Ṛebbi ?

2:12 Dehcen akk, ur ẓrin d acu ara xemmemen, qqaṛen wway gar-asen : D acu-t wayagi ?

2:13 Ma d wiyaḍ stehzayen fell-asen qqaṛen : D askaṛ i sekkṛen !

2:14 Buṭrus yellan akk-d ḥdac n ṛṛusul-nni, issaɛli taɣect-is yenna i lɣaci : Ay imezdaɣ n temdint n Lquds akk-d kunwi meṛṛa isɛeddayen ussan di temdint-agi, slet-ed mliḥ i wayen ara wen-d-iniɣ, ilaq aț-țeẓrem ayagi :

2:15 irgazen-agi ur skiṛen ara am akken i ɣilen kra seg-wen, axaṭer ațan d țesɛa n ṣṣbeḥ kan !

2:16 Lameɛna ayagi yedṛan d ayen i d-icar nnbi Yuwil :

2:17 Sidi Ṛebbi yenna-d : Deg ussan ineggura a d-sserseɣ Ṛṛuḥ-iw ɣef yemdanen meṛṛa ; arrac-nwen d yessi-twen a d-țcirin ayen ara sen-d-țxebbiṛeɣ, ilmeẓyen-nwen a sen-d-țweḥḥiɣ, imɣaṛen-nwen a sen-d-mmeslayeɣ di tirga.

2:18 Anɛam, deg ussan-nni a d-smireɣ Ṛṛuḥ-iw ɣef yiqeddacen-iw ț-țqeddacin-iw, a d-ciren s wayen ara sen-d-țxebbiṛeɣ.

2:19 Deg igenni ad xedmeɣ leɛǧayeb, di lqaɛa a d-sekneɣ licaṛat yessewhamen : idammen, times d usigna n ddexxan.

2:20 Iṭij ad yuɣal d ṭṭlam, aggur ad yuɣal am idammen uqbel a d-yas wass n Sidi Ṛebbi ; ass-nni d ass ameqqran, yeččuṛen d lɛaḍima.

2:21 Imiren kra n win ineddhen s yisem n Sidi Ṛebbi ad ițțusellek.

2:22 Ay imdanen n wat Isṛail, ḥesset-ed mliḥ i wayen ara wen-d iniɣ : ɣef wayen yeɛnan Ɛisa anaṣari, teẓram akk belli Sidi Ṛebbi isbeggen ed tazmert-is mi gexdem zdat-wen lbeṛhanat, licaṛat d leɛǧubat, am akken i teẓram s yiman-nwen.

2:23 Akken yella si zik di leqsed n Sidi Ṛebbi ; argaz-agi tezzenzem-t, d kunwi i t-yenɣan mi t-tsellmem ger ifassen n ikafriwen i t-isemmṛen ɣef lluḥ.

2:24 Lameɛna Sidi Ṛebbi yesseḥya t-id, isellek-it-id si tesraft n lmut axaṭer d lmuḥal a t-teṭṭef ɣuṛ-es.

2:25 Sidna Dawed yenna-d fell-as : Țwaliɣ kull ass Sidi Ṛebbi zdat-i axaṭer yețțili ɣer uyeffus-iw iwakken ur ɣelliɣ ara ;

2:26 daymi i gefṛeḥ wul-iw, i tcennu taɣect-iw ; ula d iɣsan-iw ad ṛtiḥen di laman ;

2:27 axaṭer ur iyi-tețțaǧǧaḍ ara di laxeṛt,ur tețțaǧaḍ ara win i k-iḍuɛen ad yerku deg uẓekka.

2:28 Teldiḍ-iyi iberdan n tudert ; ul-iw yeččuṛ d lfeṛḥ imi tețțiliḍ yid-i.

2:29 Ay atmaten ilaq-iyi a wen d-mmeslayeɣ ɛinani ɣef tideț yeɛnan jeddi-tneɣ Sidna Dawed. Teẓram belli yemmut, yețwamḍel, yerna aẓekka ines mazal-it ar ass-a gar-aneɣ.

2:30 Lameɛna imi d nnbi i gella, yeẓra belli Sidi Ṛebbi ad iṭṭef di lemɛahda i s-d-yefka mi s-d-yenna : D yiwen si tarwa-k ara sbeddeɣ d agellid deg wemkan-ik.

2:31 Ɣef ḥeggu n Lmasiḥ i d-icar mi d-yenna lehduṛ-agi ; axaṭer ț-țideț, ur t-yeǧǧi di laxeṛt, ur t-yeǧǧi ad yerku.+

2:32 Win ɣef i wen-d-țmeslayeɣ, d Ɛisa-agi i d-isseḥya Sidi Ṛebbi si ger lmegtin, aql-aɣ akk d inagan.

2:33 Mi gețwarfed ɣer uyeffus n Sidi Ṛebbi, yețțunefk-as Ṛṛuḥ iqedsen i s-yewɛed Baba Ṛebbi, nețța ismar-it-id fell-aɣ ; d ayen i teslam, i twalam.

2:34 Mačči d Sidna Dawed i gulin ɣer igenni, axaṭer d nețța i d-yennan : Sidi Ṛebbi yenna-d i Ssid-iw :

2:35 Sṭef amkan n lḥekma ɣer uyeffus-iw alamma rriɣ iɛdawen-ik seddaw idaṛṛen-ik.+ +

2:36 Ilaq ihi at Isṛail meṛṛa ad ẓren s tideț belli Ɛisa-nni i tsemmṛem ɣef lluḥ, d nețța i gerra Sidi Ṛebbi d Agellid ameqqran, d Lmasiḥ.

2:37 Mi slan i wayen i d-yenna Buṭrus, lɣaci meṛṛa saxen wulawen-nsen, nnan i Buṭrus d ṛṛusul nniḍen : Ay atmaten, d acu ara nexdem ?

2:38 Buṭrus yerra-yasen : Tubet, beddlet tikli ! Mkul yiwen deg-wen ad yețwaɣḍes s yisem n Sidna Ɛisa Lmasiḥ iwakken a wen-țwasemmḥen ddnubat-nwen, yerna a wen-d yețțunefk Ṛṛuḥ iqedsen.

2:39 Axaṭer lweɛd-agi n Sidi Ṛebbi, yețțunefk-ed i kunwi d warraw-nwen akk-d wid meṛṛa n leǧnas nniḍen iwumi ara d-issiwel Sidi Ṛebbi Illu-nneɣ.

2:40 Buṭrus inehhu-ten s waṭas n yimeslayen nniḍen, iwakken ad amnen, yeqqaṛ-asen : Xḍut i lǧil-agi iɛewjen iwakken aț-țețțusellkem !

2:41 Wid iqeblen ayen i d-yenna Buṭrus, qeblen ad țwaɣedṣen. Ass-nni rnan-d wazal n tlata alaf n yemdanen i gumnen.

2:42 Seg imiren, zgan țnejmaɛen am atmaten iwakken ad slen i wselmed n ṛṛusul ; beṭṭun aɣṛum n usmekti n Lmasiḥ, yerna deɛɛun ɣer Sidi Ṛebbi s yiwen n wul.

2:43 Tɛeǧǧben, tekcem-iten akk tugdi mi țwalin ṛṛusul xeddmen aṭas n lbeṛhanat d leɛǧayeb.

2:44 Wid akk yumnen, țɛicin s tdukli yerna beṭṭun wway gar-asen ayen akk sɛan.

2:45 Znuzun ayla-nsen meṛṛa, ayen akken i d-jemɛen, feṛqen-t wway gar-asen mkul yiwen ɣef leḥsab n wakken yeḥwaǧ.

2:46 Kull ass, s yiwen ṛṛay, țnejmaɛen di lǧameɛ iqedsen, țnejmaɛen daɣen deg ixxamen-nsen, beṭṭun aɣṛum iwakken ad mektin Lmasiḥ, tețțen lqut-nsen s nneya d lfeṛḥ ;

2:47 țḥemmiden Sidi Ṛebbi, imdanen meṛṛa țqadaṛen-ten ; Sidi Ṛebbi irennu-d kull ass ɣer tejmaɛt-nsen wid ițțusellken.

3:1 Aț-țili d leǧwahi n tlata n tmeddit, Buṭrus d Yuḥenna ddukklen, ulin ɣer lǧameɛ iqedsen iwakken ad dɛun ɣer Sidi Ṛebbi.

3:2 Țțawin-d mkul ass yiwen n wergaz i d-ilulen d aquḍar ɣer zdat n tewwurt n lǧameɛ, tin iwumi qqaṛen « tawwurt tucbiḥt »; yețɣimi dinna iwakken ad issuter tin n Ṛebbi i wid i gkeččmen ɣer lǧameɛ-nni.

3:3 Mi gwala Buṭrus d Yuḥenna țeddun ad kecmen, issuter-asen tin n Ṛebbi.

3:4 Buṭrus d Yuḥenna, ṛeṣṣan allen-nsen fell-as, nnan-as : Muqel-ed ɣuṛ-nneɣ !

3:5 Argaz-nni imuqel-iten, iḍmeɛ deg-sen a s-d-fken kra.

3:6 Dɣa Buṭrus yenna-yas : Ur sɛiɣ idrimen n lfeṭṭa, ur sɛiɣ wid n ddheb, lameɛna ayen sɛiɣ a k-t-fkeɣ : s yisem n Sidna Ɛisa Lmasiḥ anaṣari, ekker telḥuḍ !

3:7 Iṭṭef-it-id seg ufus ayeffus, issekker-it-id. Imiren kan iḍaṛṛen-is ț-țkeɛburin n idaṛṛen-is ḥlan.

3:8 Ibedd ɣef yidaṛṛen-is, ineggez, yebda ileḥḥu. Ikcem yid-sen ɣer lǧameɛ, ileḥḥu ițjellib, ițḥemmid Sidi Ṛebbi.

3:9 Lɣaci akk walan-t mi gleḥḥu ițḥemmid Sidi Ṛebbi.

3:10 Mi t-ɛeqlen qqaṛen : Mačči d win akken yețɣimin ɣer tewwurt tucbiḥt n lǧameɛ iqedsen win akken issuturen tin n Ṛebbi ? Dehcen meṛṛa, wehmen deg wayen yedṛan yid-es.

3:11 Argaz-nni iteddu d Buṭrus akk-d Yuḥenna, ur yeṭṭixiṛ ara fell-asen, lɣaci-nni meṛṛa wehmen, uzzlen-d ɣuṛ-sen ɣer usqif iwumi qqaṛen « Asqif n Sidna Sliman ».

3:12 Buṭrus mi gwala ayagi, yenṭeq ɣer lɣaci yenna : Ay at Isṛail, acuɣeṛ akka twehmem deg wayen yedṛan ? Acuɣeṛ i d-țmuqulem ɣuṛ-nneɣ, am akken d nukkni i gesselḥan argaz-agi s tezmert-nneɣ neɣ s ṭṭaɛa-nneɣ ?

3:13 D Sidi Ṛebbi Illu n Sidna Ibṛahim, n Isḥaq d Yeɛqub, Ṛebbi n lejdud-nneɣ, d nețța i d-isbeggnen lɛaḍima n Sidna Ɛisa aqeddac-is, win akken i tnekṛem, i tsellmem i lḥakem Bilaṭus i gebɣan a s-d-iserreḥ.

3:14 Kunwi tnekṛem win akken yellan d azedgan, d aḥeqqi ; tessutrem a d-serrḥen i bu tmegṛaḍ deg umkan-is.

3:15 Tenɣam Mmi-s n ugellid n tudert, meɛna Sidi Ṛebbi yesseḥya-t-id si ger lmegtin , aql-aɣ d inigan.

3:16 Ț-țazmert yellan deg isem n Sidna Ɛisa i d-yerran lǧehd i wergaz agi i tețwalim, yerna tessnem-t. D Sidna Ɛisa s wayes numen i t-isseḥlan zdat-wen meṛṛa.

3:17 Ay atmaten, ẓriɣ s lqella n lefhama i txedmem annect-nni, am kunwi am imeqranen-nwen.

3:18 Meɛna s wakka i gesnekmal Sidi Ṛebbi ayen i d-yenna si zik seg imawen n lenbiya meṛṛa : yenna-d belli Lmasiḥ-ines ad inɛețțab.

3:19 Ndemmet ihi, beddlet tikli iwakken ad mḥun ddnubat-nwen,

3:20 iwakken a kkun-id-asen wussan n ṛṛaḥa s ɣuṛ Sidi Ṛebbi, a wen-d iceggeɛ Ɛisa Lmasiḥ, win akken i wen-ihegga.

3:21 Tura atan deg igenni alamma yusa-d lweqt i deg ara tuɣal txelqit meṛṛa ț-țajḍiṭ, akken i t-id-yenna Sidi Ṛebbi si zik seg imawen n lenbiya.

3:22 Sidna Musa yenna i wat Isṛail : Sidi Ṛebbi Illu-nwen a wen-d iceggeɛ si ger watmaten-nwen,+ yiwen n nnbi am nekk, a s-tḥessem deg wayen akk ara wen-d-yini.+

3:23 Kra n win ur nețḥessis ara i nnbi-yagi, ad yețwakkes si ddunit ger watmaten-is.

3:24 Lenbiya meṛṛa, si zzman n nnbi Samwil d wid meṛṛa i d-yernan deffir-es, caren-d daɣen ɣef wussan agi i deg nețɛici.

3:25 D kunwi i d arraw n lenbiya, i d arraw n leɛqed-nni i gexdem Sidi Ṛebbi d lejdud-nneɣ, mi s-yenna i Sidna Ibṛahim : S dderya n dderya-k ara barkeɣ tiwaculin meṛṛa n ddunit+ .

3:26 I kunwi d imezwura iwumi i d iceggeɛ Sidi Ṛebbi aqeddac-is, isseḥya t-id iwakken a kkun-ibarek yerna a d-issebɛed mkul yiwen deg-wen seg yir n lecɣal-is.

4:1 Buṭrus d Yuḥenna mazal-iten heddṛen i lɣaci mi d-wwḍen kra n lmuqedmin, yedda-d yid-sen umeqqran n iɛessasen n lǧameɛ iqedsen akk-d kra n yergazen si tejmaɛt n isaduqiyen

4:2 Zeɛfen mi walan ṛṛusul sselmaden lɣaci yerna ssefhamen-ten ɣef ḥeggu n lmegtin qqaṛen-asen : -- Akken i d-yeḥya Sidna Ɛisa si ger lmegtin, akken daɣen ara d-ḥyun lmegtin.

4:3 Sṭfen-ten, rran-ten ɣer lḥebs armi d azekka-nni axaṭer imiren yeɣli-d yiḍ.

4:4 Lameɛna aṭas seg wid yeslan i wawal, umnen. Ad ilin azal n xemsa alaf n yemdanen i gumnen.

4:5 Azekka-nni, lḥukkam, lecyux akk-d lɛulama n ccariɛa yellan di temdint n Lquds, nnejmaɛen.

4:6 Yella lmuqeddem ameqqran ițusemman Hannan akk-d wid i s-yețțilin, ḥedṛen daɣen : Kayef, Yuḥenna d Alixandru.

4:7 Wwin-d Buṭrus d Yuḥenna, sbedden-ten di tlemmast-nsen akken a ten-beḥten, qqaṛen-asen : Ansi i wen-d-tekka tezmert-agi, anwa isem s wayes i tneddhem armi txedmem ayagi ?

4:8 Dɣa Buṭrus yeččuṛen d Ṛṛuḥ iqedsen, yenna-yasen :

4:9 A lḥukkam d lecyux n wat Isṛail ! Imi ɣef lxiṛ i nexdem i wergaz-agi aɛiban d wamek i geḥla i ɣ-d-testeqsayem ass-agi,

4:10 ilaq aț-țeẓrem kunwi d wat Isṛail meṛṛa, belli s yisem n Sidna Ɛisa anaṣari, win akken i tsemmṛem ɣef wumidag, win akken i d-yesseḥya Sidi Ṛebbi si ger lmegtin, s tezmert-is i gbedd wergaz agi tura zdat-wen yeḥla.

4:11 D nețța i ɣef d-nnant tira iqedsen : D azṛu-nni i tḍeggṛem kunwi ay ibennayen, i guɣalen d azṛu i geṭṭfen lebni. +

4:12 Deg-s i gella leslak ; ulac di ddunit meṛṛa isem nniḍen s wayes i nezmer a nețțusellek.

4:13 Mi walan belli Buṭrus d Yuḥenna heddṛen s laman mbla akukru, wehmen axaṭer ẓran d imdanen ur neɣṛi ara ; ɛeqlen-ten seg wid yeddan d Sidna Ɛisa.

4:14 Lameɛna imi walan argaz-nni yeḥlan ibedd zdat-sen, ur ufin d acu ara d-inin.

4:15 Umṛen-ten ad ffɣen seg unejmaɛ n ccṛeɛ iwakken ad mcawaṛen wway gar-asen,

4:16 nnan : D acu ara nexdem s yemdanen agi ? Imezdaɣ n temdint n Lquds akk wwin-d s lexbaṛ belli xedmen lbeṛhan yerna ț-țideț ur nezmir ara a nenkeṛ ayagi.

4:17 Lameɛna iwakken lexbaṛ-agi ur iteffeɣ ara ger lɣaci, a neggal deg-sen ur țțuɣalen ara ad mmeslayen ula i yiwen s yisem-agi.

4:18 Uɣalen ssawlen-asen, gullen deg-sen iwakken ur țțuɣalen ara ad mmeslayen neɣ ad slemden s yisem-agi n Ɛisa.

4:19 Lameɛna Buṭrus d Yuḥenna nnan-asen : D acu i d lḥeqq zdat Ṛebbi, a wen-naɣ awal i kunwi neɣ a naɣ awal i Sidi Ṛebbi ?

4:20 Walit s yiman nwen ! Nukni ur nezmir ara a nessusem ɣef wayen neẓra d wayen nesla.

4:21 Rnan gullen deg-sen, lameɛna serrḥen-asen imi ur d-ufin ara sebba s wayes ara ten-ɛaqben, yerna lɣaci akk țḥemmiden Sidi Ṛebbi ɣef wayen yedṛan,

4:22 axaṭer argaz-agi i geḥlan s lbeṛhan di leɛmeṛ-is akteṛ n ṛebɛin iseggasen.

4:23 Mi sen-serrḥen, ṛuḥen ɣer watmaten, ḥkan-asen akk ayen i sen-d-nnan lmuqedmin imeqqranen d lecyux.

4:24 Mi slan watmaten-nni i wayen i d-yenna Buṭrus, akken ma llan serrḥen i taɣect-nsen, nedhen ɣer Sidi Ṛebbi nnan : A Sidi Ṛebbi, kečč i gxelqen igenni, lqaɛa, lebḥeṛ d wayen akk yellan.

4:25 D kečč i d-ineṭqen s Ṛṛuḥ-ik iqedsen seg yimi n jeddi-tneɣ Dawed aqeddac-ik, mi d-yenna : Acuɣeṛ ccwal-agi ger leǧnas, acuɣeṛ igduden ggaren-d anagar yir ixemmimen ?

4:26 Igelliden n ddunit heggan iman nsen, imeqqranen ddukklen akk, ffɣen-d d iɛdawen n Sidi Ṛebbi akk-d Lmasiḥ-ines.

4:27 Axaṭer ț-țideț, di temdint-agi, Hiṛudus d Bilaṭus mcawaṛen, ddukklen nutni d ibeṛṛaniyen akk-d wat Isṛail, uɣalen d iɛdawen n Sidna Ɛisa imqeddes i textaṛeḍ d Lmasiḥ.

4:28 Ayen akka xedmen, d ayen akken i tqesdeḍ keččini di lebɣi-inek, s tezmert-ik, si zik.

4:29 I tura, a Sidi Ṛebbi, wali-d amek akka i ɣ ssaggaden ; efk-aɣ-d a Sidi Ṛebbi tazmert s wayes ara nbecceṛ awal-ik s ṭṭmana mbla akukru.

4:30 Ssers-ed afus-ik iwakken ad ḥlun imuḍan, iwakken a d-ilint licaṛat d lbeṛhanat s yisem n Sidna Ɛisa aqeddac-ik imqeddes.

4:31 Mi fukken taẓallit, amkan-nni anda nnejmaɛen yergagi ; ččuṛen akk s Ṛṛuḥ iqedsen, bdan țbecciṛen awal n Sidi Ṛebbi s ṭṭmana mbla tugdi.

4:32 Wid akk yumnen s Lmasiḥ, ddukklen akk s yiwen n wul, s yiwen n ṛṛuḥ, ula d yiwen deg-sen ur yeḥsib ayen yesɛa d ayla-s i yiman-is kan, meɛna cerken kullec wway gar-asen.

4:33 Ṛṛusul țcehhiden-d ɣef ḥeggu n Sidna Ɛisa s tezmert tameqqrant yerna ṛṛeḥma n Sidi Ṛebbi txeddem deg-sen s lǧehd.

4:34 Ulac gar-asen win ixuṣṣen ; wid akk yesɛan igran neɣ ixxamen zzenzen-ten,

4:35 idrimen-nni i d-țțawin deg-sen, țțawin-ten ɣer ifassen n ṛṛusul i ten-ifeṛṛqen i mkul yiwen ɣef leḥsab n wakken yeḥwaǧ.

4:36 Yusef illan n at Lewwi iwumi semman ṛṛusul Barnabas, ( yeɛni argaz inehhun ) i glulen di tegzirt n Qubṛus,

4:37 yesɛa yiwen n yiger, izzenz-it, yewwi-d idrimen-nni, yefka-ten i ṛṛusul.

5:1 Yiwen wergaz isem-is Ananyas, nețța ț-țmeṭṭut-is Safira zzenzen yiwen n weḥric seg wakal-nsen ; msefhamen ad kksen yiwen n umur si ssuma n lbiɛ-nni ;

5:2 amur nniḍen yewwi-t-id ɣer ifassen n ṛṛusul.

5:3 Buṭrus inṭeq ɣuṛ-es yenna : Ay Ananyas, acuɣeṛ i teǧǧiḍ Cciṭan yeččuṛ ul-ik ? Acuɣeṛ i teskadbeḍ i Ṛṛuḥ iqedsen mi tekkseḍ yiwen umur n yedrimen n wakal i tezzenzeḍ ? Uqbel a t- tezzenzeḍ, akal-nni yella d ayla-k !

5:4 Mi t-tezzenzeḍ, idrimen-nni daɣen d ayla-k ! Amek armi i teṣṣawḍed aț-țxeḍmeḍ yiwet lḥaǧa am tagi ? Ihi mačči i yemdanen iwumi teskadbeḍ, lameɛna i Sidi Ṛebbi !

5:5 Mi gesla i yimeslayen-nni, Ananyas yeɣli, yeffeɣ-it ṛṛuḥ, yemmut. Wid akk yellan dinna, ikcem-iten akk lxuf, țergigin.

5:6 Kkren ilmeẓyen kefnen-t, wwin-t a t-meḍlen.

5:7 Ɛeddant wazal n tlata sswayeɛ, mi d-tekcem tmeṭṭut n Ananyas, nețțat ur tesli ara s wayen yedṛan.

5:8 Buṭrus yenṭeq ɣuṛ-es yenna-yas : Ini-yi-d, ma s ssuma-yagi i tezzenzem akal-nwen ? Terra-yas-d : Ih, s ssuma-yagi i t-nezzenz.

5:9 Dɣa Buṭrus yenna-yas : Amek armi i temsefhamem iwakken aț-țjeṛṛbem Ṛṛuḥ n Sidi Ṛebbi ? Wid imeḍlen argaz-im atnan ɣer tewwurt, a kkem-awin ula d kemm !

5:10 Imiren kan teɣli ɣer idaṛṛen-is yeffeɣ-iț ṛṛuḥ, temmut. Ilmeẓyen-nni mi d-kecmen ufan-ț-id temmut, wwin-ț meḍlen-ț ɣer tama n wergaz-is.

5:11 Yeɣli-d ṛṛɛac ( lxuf ) d ameqqran ɣef tejmaɛt n watmaten d wid akk yeslan s wayen yedṛan.

5:12 Aṭas n lbeṛhanat d lmuɛǧizat i xeddmen ṛṛusul ger lɣaci. Wid yumnen țnejmaɛen akk s yiwen ṛṛay deg wesqif n Sidna Sliman.

5:13 Lɣaci meṛṛa țcekkiṛen-ten meɛna kukran akk a d-rnun ɣuṛ-sen.

5:14 Leḥsab n yergazen ț-țilawin yumnen s Sidna Ɛisa yețzad irennu.

5:15 Țțawin-d imuḍan ɣer iberdan, srusun-ten ɣef tgertyal iwakken m'ara d-iɛeddi Buṭrus, ad innal yiwen seg-sen ulamma s tili-ines.

5:16 Lɣaci țțazzalen-d daɣen ula si temdinin nniḍen iqeṛben tamdint n Lquds ; țțawin-d imuḍan akk-d wid yețwazedɣen, ḥellun akken ma llan.

5:17 Ɣef ddemma n wayen ideṛṛun kkrent-asen-d tismin i lmuqeddem ameqqran akk-d isaduqiyen-nni yellan yid-es,

5:18 ṭṭfen ṛṛusul, rran-ten ɣer lḥebs n baylek.

5:19 Lameɛna deg yiḍ-nni, yiwen lmelk n Sidi Ṛebbi yeldi tiwwura n lḥebs, issufeɣ-iten-id, yenna-yasen :

5:20 Ṛuḥet beccṛet di lǧameɛ iqedsen, slemdet i lɣaci awal n tudert.

5:21 Mi slan i wayen i sen-d-yenna lmelk-nni, ṛuḥen ṣṣbeḥ zik ɣer lǧameɛ iqedsen, bdan sselmaden. Ma d lmuqeddem ameqqran d wid akk yellan yid-es, snejmaɛen-d lecyux akk-d imeqqranen n wat Isṛail ɣer unejmaɛ n ccṛeɛ ; imiren ceggɛen a d-awin ṛṛusul-nni si lḥebs.

5:22 Mi wwḍen iɛessasen ɣer lḥebs, ur ten-ufin ara dinna ; uɣalen-d ɣer unejmaɛ, nnan-asen :

5:23 Nufa lḥebs isekkeṛ, iɛessasen bedden zdat n tewwurt, lameɛna mi nekcem, ur nufi ula d yiwen zdaxel.

5:24 Mi ten-yewweḍ lexbaṛ-agi, lqebṭan n lǧameɛ akk-d lmuqedmin imeqqranen ɛewqen ! Ur ẓrin ara ɣer wanda ara taweḍ temsalt-agi.

5:25 Iṛuḥ-ed ɣuṛ-sen yiwen yenna yasen : Irgazen-nni i terram ɣer lḥebs, atnan deg ufrag n lǧameɛ iqedsen yerna sselmaden lɣaci.

5:26 Lqebṭan n lǧameɛ d iɛessasen ṛuḥen wwin-ten-id lameɛna ḥudren ten axaṭer uggaden a ten-ṛejmen lɣaci.

5:27 Mi ten-id-wwin ɣer zdat usqamu n ccṛeɛ, lmuqeddem ameqqran isteqsa ten yenna-yasen :

5:28 Neggul deg-wen ur tețțuɣalem aț-țeslemdem s yisem-agi ; lameɛna kunwi tessufɣem aselmed nwen di temdint n Lquds meṛṛa ! Tura tebɣam a d-terrem ɣer yirawen nneɣ tamgeṛṭ n wergaz-agi !

5:29 Buṭrus d ṛṛusul nniḍen erran asen : Ilaq a naɣ awal i Sidi Ṛebbi, mačči i yemdanen !

5:30 Ṛebbi, Illu n lejdud-nneɣ yesseḥya-d Sidna Ɛisa, win akken i tenɣam, i tsemmṛem ɣef lluḥ.

5:31 D nețța i gessuli Sidi Ṛebbi ɣer uyeffus-is d Agellid, d Amsellek, iwakken a d-yefk i wat Isṛail leɛfu n ddnubat m'ara beddlen tikli.

5:32 Aql-aɣ d inagan ɣef wayagi, nukkni akk-d Ṛṛuḥ iqedsen i d-yefka Sidi Ṛebbi i wid i t-iḍuɛen.

5:33 Mi sen-slan, ikcem-iten wurrif d ameqqran armi bɣan a ten-nɣen.

5:34 Meɛna Gamalyel, yiwen n lɛalem n ccariɛa yellan d afarizi, ḥemmlen-t akk lɣaci ; ikker-ed deg unejmaɛ-nni, yefka lameṛ ad ssufɣen ṛṛusul-nni kra n lweqt.

5:35 Dɣa yenṭeq yenna : Ay at Isṛail, ḥadret iman-nwen ɣef wayen akka tebɣam a t-txedmem i yemdanen-agi !

5:36 Mmektit-ed, mačči aṭas aya ikker-ed yiwen yețțuseman Tudas yerran iman-is d ameqqran. Azal n ṛebɛa meyya n yergazen i ddan yid-es; asmi i t-nɣan, wid akk i geddan yid-es mfaṛaqen, ur d-yeqqim lateṛ-nsen.

5:37 Deffir Tudas, ikker-ed daɣen Yahuda ajlili di lweqt-nni n ujerred n wegdud ; aṭas daɣen n wid i geddan yid-es, meɛna ula d nețța mi gemmut, wid akk yeddan yid-es mfaṛaqen.

5:38 Ihi tura a wen-d-iniɣ : ur ceɣlet ara d yemdanen-agi, anfet-asen ad ṛuḥen. Ma yella aselmed-agi-nsen yekka-d s ɣuṛ-sen, ur yețdumu ara ;

5:39 ma yella s ɣuṛ Sidi Ṛebbi i d-ikka, ur tețțizmirem ara a t-tekksem ! Ḥadret iman-nwen ihi, neɣ m'ulac aț-țuɣalem d iɛdawen n Sidi Ṛebbi !

5:40 Mi mcawaṛen, uɣen ṛṛay-is. Uɣalen skecmen ṛṛusul-nni, fkan lameṛ a ten-wwten s iɛekkzan, rnan gullen deg-sen ur țțuɣalen ara ad mmeslayen s yisem-agi n Ɛisa, dɣa serrḥen-asen.

5:41 Mi d-ffɣen ṛṛusul seg unejmaɛ n ccṛeɛ, feṛḥen imi țwaḥesben d wid yuklalen ad țewten, ad țwaregmen ɣef ddemma n yisem n Sidna Ɛisa.

5:42 Dɣa kull ass sselmaden țbecciṛen di lǧameɛ iqedsen akk-d ixxamen, lexbaṛ n lxiṛ yeɛnan Ɛisa Lmasiḥ.

6:1 Deg ussan-nni, leḥsab n widak yumnen s Sidna Ɛisa yețzid irennu. At Isṛail i d-yusan si tmura nniḍen, yețmeslayen s tyunanit, ccetkan ɣef wat Isṛail yellan di tmurt, axaṭer m'ara feṛqen lmakla, ur d-țelhayen ara ț-țuǧǧal-nsen.

6:2 Tnac ṛṛusul-nni snejmaɛen-d inelmaden meṛṛa, nnan-asen : Mačči d lḥeqq a neǧǧ Awal n Sidi Ṛebbi di rrif iwakken a nelhi d wefṛaq n lqut i mkul yiwen.

6:3 Ihi ay atmaten, xtiṛet-ed si gar-awen sebɛa yergazen i deg sɛan laman yemdanen n tmurt meṛṛa, yeččuṛen d Ṛṛuḥ iqedsen akk-d leɛqel, d nutni ara nwekkel ɣef ccɣel-agi.

6:4 Ma d nukkni, lweqt-nneɣ a t-nesɛeddi di tẓallit akk-d uselmed n wawal n Sidi Ṛebbi.

6:5 Tajmaɛt akk iɛeǧǧeb-as ṛṛay-agi. Kkren xtaṛen-d Stifan yellan d argaz yeččuṛen d liman akk-d Ṛṛuḥ iqedsen. Xtaṛen daɣen Filbas, Brukurus, Nikanur, Timun, Barmenas, akk-d Nikulus n temdint n Antyuc, i gkecmen si zik di ddin n wat Isṛail.

6:6 Wwin-ten-id, sbedden-ten zdat n ṛṛusul. Ṛṛusul ssersen ifassen-nsen fell-asen, dɛan-asen s lxiṛ ɣer Sidi Ṛebbi.

6:7 Yuɣal, awal n Sidi Ṛebbi iffeɣ yețwassen, leḥsab n wid yumnen s Lmasiḥ yețzid irennu di temdint n Lquds yerna aṭas n lmuqedmin i gumnen, qeblen abrid n Lmasiḥ.

6:8 Stifan yeččuṛen d ṛṛeḥma ț-țezmert n Sidi Ṛebbi, ixeddem lbeṛhanat d leɛǧayeb timeqqranin ger lɣaci.

6:9 Kra n yemdanen n tejmaɛt n iḥeṛṛiyen i d-yusan si tmurt n Qiṛwan, si tmurt n Skandriya, si tmurt n Silisya akk-d wid i d-yusan si tmurt n Asya, usan-d ad mjadalen nutni d Stifan ;

6:10 lameɛna ur zmiren ara a t-ɣelben deg awal axaṭer ițmeslay s ṣṣwab d leɛqel i s-d-yefka Ṛṛuḥ iqedsen.

6:11 Imiren fkan tajɛelt i kra n yergazen iwakken a d-inin : « Nesla yas yekfeṛ ɣef Sidna Musa akk-d Sidi Ṛebbi ! »

6:12 Sekkren ccwal ger lɣaci d lecyux akk-d lɛulama n ccariɛa , dɣa ṭṭfen Stifan ewwtent, wwin-t ɣer unejmaɛ n usqamu n ccṛeɛ.

6:13 Sbedden-d inigan i d-icehden fell-as s ẓẓur, nnan-d : Argaz-agi, yezga yekkat deg wemkan-agi iqedsen akk-d ccariɛa n Sidna Musa !

6:14 Nesla-yas mi geqqaṛ : « Ɛisa-nni Anaṣari, ad ihudd amkan agi yerna ad ibeddel lɛaddat i ɣ-d yeǧǧa Sidna Musa. »

6:15 Wid akk yellan deg unejmaɛ n ccṛeɛ, ṛeṣṣan allen-nsen ɣef Stifan ; țwalin udem-is am akken d udem n lmalayekkat.

7:1 Lmuqeddem ameqqran isteqsa Stifan yenna-yas : Ț-țideț wayen akka i d-nnan ?

7:2 Stifan yerra-yas : Ay atmaten, ay imɣaṛen n wat Isṛail, ḥesset-iyi-d ! Sidi Ṛebbi di lɛaḍima-s tameqqrant, yesbeggen-ed iman-is i jeddi-tneɣ Sidna Ibṛahim, mi gella di tmurt n Mizubuṭamya uqbel ad yezdeɣ di temdint n Haran.

7:3 Yenna-yas : Ffeɣ si tmurt-ik, si tmurt n imawlan-ik, tṛuḥeḍ ɣer tmurt ara k-d-ssekneɣ.

7:4 Iffeɣ ihi si tmurt n iqaldiyen, iṛuḥ ad izdeɣ di temdint n Haran. Mi gemmut baba-s, Sidi Ṛebbi yewwi-t-id ɣer tmurt-agi i deg tzedɣem tura.

7:5 Sidi Ṛebbi ur s-yefki ara di tmurt-agi ula d azal n terdast i deg ara yesres aḍar-is ; meɛna iwɛed-as a s-yefk tamurt-agi i nețța d dderya n dderya-s ɣas akken imiren mazal ur yesɛi ara dderya, Sidi Ṛebbi yenna-yas :

7:6 Dderya n dderya-k aț-țezdeɣ deg yiwet n tmurt tabeṛṛanit, dinna a ten-erren d aklan, ad sɛeddin fell-asen lḥif azal n ṛebɛa meyya iseggasen.

7:7 Lameɛna Sidi Ṛebbi yenna-d : Ad uɣalen a d-ffɣen si tmurt-nni, ad iyi-ɛebden deg wemkan-agi ; ma d lǧens-nni i ten-yerran d aklan, d nekk ara ten-iḥasben.

7:8 Dɣa Sidi Ṛebbi yefka-d i Sidna Ibṛahim leɛqed yeɛnan ṭṭhaṛa. S wakka, mi d-yesɛa mmi-s Isḥaq, Sidna Ibṛahim iṭeḥheṛ-as deg wass wis tmanya n tlalit-is. Isḥaq daɣen yexdem akken i mmi-s yeɛqub ; ula d Yeɛqub iṭeḥheṛ i tnac-nni warraw-is yellan d lejdud-nneɣ.

7:9 Ihi arraw-agi n yeɛqub usmen ɣef gma-tsen Yusef, kkren zenzen-t ɣer tmurt n Maṣer anda yuɣal d akli.

7:10 Lameɛna Sidi Ṛebbi yella yid-es, ițsellik-it si tegnatin n ccedda ; issers-ed fell-as ṛṛeḥma yerna-yas-d leɛqel s wayes ara yeɛziz ɣer Ferɛun agellid n tmurt n Maṣeṛ. Ferɛun isbedd-it d lḥakem ɣef tmurt-is akk-d wexxam-is meṛṛa.

7:11 Yeɣli-d laẓ ɣef tmurt n Maṣeṛ akk ț-țmurt n Kenɛan, d lweqt n cedda tameqqrant ; lejdud-nneɣ ur ufin ara ayen ara ččen.

7:12 Mi gesla Yeɛqub belli llan yirden di tmurt n Maṣer, iceggeɛ ɣer dinna tikkelt tamezwarut, arraw-is yellan d lejdud-nneɣ.

7:13 Mi uɣalen tikkelt tis snat, Yusef isbeggen-ed iman-is ɣer watmaten-is, ɛeqlen-t d nețța. Yuɣal Ferɛun yeẓra laṣel n Yusef ansi-t.

7:14 Yusef iceggeɛ a d-awin baba-s Yeɛqub akk-d wat wexxam-is meṛṛa i gellan di xemsa usebɛin yid-sen.

7:15 Yeɛqub iṣubb ɣer tmurt n Maṣer ikemmel dinna ussan-is armi yemmut, am nețța am lejdud-nneɣ.

7:16 Wwin-d iɣsan-nsen ɣer taddart n Sicem, meḍlen-ten deg uẓekka i guɣ Sidna Ibṛahim s yedrimen ɣef warraw n Hamur n taddart n Sicem.

7:17 Mi d-yewweḍ lweqt i deg ara yedṛu wayen i gewɛed Sidi Ṛebbi i Sidna Ibṛahim s limin, agdud-nneɣ yufa-t-id lḥal innerna s waṭas di tmurt n Maṣer ;

7:18 imiren ikker-ed ugellid nniḍen ur nessin ara Yusef.

7:19 Agellid-nni iḥqeṛ lɣaci-nneɣ, yețqehhiṛ lejdud nneɣ, yeṣṣaweḍ armi i gefka lameṛ ad ǧǧen arraw-nsen i cceṛ akken kan ara d-lalen, iwakken ur țidiren ara.

7:20 Di lweqt-nni i d-ilul Sidna Musa ; icbeḥ yerna eɛziz ɣer Sidi Ṛebbi. Yețțuṛebba deg wexxam n imawlan-is tlata wagguren s tuffra ;

7:21 uɣalen rran-t zdaxel n tqecwalt, serrḥen-as deg wasif ; d yelli-s n Ferɛun i t-id-ijemɛen, tṛebba-t-id am mmi-s.

7:22 Ikker-ed ger imaṣriyen, yelmed akk tamusni-nsen ifazen. Yuɣal d argaz amusnaw, izewṛen ama deg yimeslayen ama di lecɣal.

7:23 Mi geṣṣaweḍ ṛebɛin iseggasen di leɛmeṛ-is, yemmekta-d atmaten-is at Isṛail, iṛuḥ ad iẓer acu n lḥala i deg țɛicin.

7:24 Yiwen wass iwala s wallen-is yiwen umaṣri ikkat yiwen seg wat Isṛail ; ikker a t-iḥudd, yerra-as-d țțar ; yewwet amaṣri-nni, yenɣa-t.

7:25 Yenwa ad fehmen watmaten-is belli s ufus-is ara ten-isellek Sidi Ṛebbi ; lameɛna nutni ur fhimen ara.

7:26 Azekka-nni iwala sin yergazen n wat Isṛail țnaɣen wway gar-asen, iger iman-is, yeɛṛed a ten-ifru, yenna-yasen : Eɛni mačči d atmaten i tellam ? Acimi i tețțnaɣem wway gar-awen ?

7:27 Lameɛna win akken iḍelmen wayeḍ idemmer Musa yenna-yas : Anwa i k-yerran d aqeṛṛuy neɣ d lḥakem fell-aɣ ?

7:28 Eɛni tebɣiḍ a yi-tenɣeḍ akken tenɣiḍ iḍelli amaṣri-nni ?

7:29 Ɣef ddemma n yimeslayen-agi Sidna Musa yerwel, iṛuḥ ad izdeɣ di tmurt n Madyan. Dinna i gesɛa sin n warrac.

7:30 Mi ɛeddan ṛebɛin n iseggasen, idheṛ-as-ed yiwen n lmelk deg yiwen uḥeǧǧaǧu n tmes iceɛlen deg inijel, mi gella deg unezṛuf n wedrar n Sinay.

7:31 Sidna Musa yewhem deg wayen yețwali ; akken kan iqeṛṛeb ɣuṛ-es, Sidi Ṛebbi yessawel-as-d seg uḥeǧǧaǧu-nni n tmes, yenna-yas-d :

7:32 D nekk i d Illu n lejdud-ik, d Illu n Ibṛahim, n Isḥaq akk-d Yeɛqub. Sidna Musa yeqqim ițergigi, yuggad ad yerfed ula d allen-is.

7:33 Sidi Ṛebbi yenna-yas : Kkes arkasen seg iḍaṛṛen-ik axaṭer amkan-agi anda tbeddeḍ d amkan iqedsen.

7:34 Walaɣ lbaṭel ideṛṛun d wegdud-iw di tmurt n Maṣer ; sliɣ-ed i unazeɛ-nsen ; ṣubbeɣ-d iwakken a ten-sellkeɣ. Ṛuḥ tura a k-ceggɛeɣ ɣer tmurt n Maṣeṛ.

7:35 Musa-nni i nekṛen m'akken i s-nnan : Anwa i k-yerran d amdebbar neɣ d lḥakem ? D nețța i d-iceggeɛ Sidi Ṛebbi d aqeṛṛuy, d amcafeɛ n wegdud ; s lemɛawna n lmelk i s-d-idehṛen deg udarnu ireqqen.

7:36 D nețța i ten-id-issufɣen si tmurt n Maṣer. Yexdem aṭas n lbeṛhanat d leɛǧayeb di tmurt-nni, di lebḥeṛ azeggaɣ, deg unezṛuf n Sinay, anda akken qqimen ṛebɛin iseggasen.

7:37 D nețța daɣen i gennan i wat Isṛail : Sidi Ṛebbi a wen-d-yefk si ger watmaten-nwen yiwen n nnbi am nekk.

7:38 Asm'akken i gennejmaɛ wegdud deg unezṛuf, d nețța iwumi d yemmeslay lmelk-nni deg wedrar n Sinay, d nețța iwumi d-yefka Sidi Ṛebbi awal n tudert iwakken a ɣ-t-id issiweḍ.

7:39 Lameɛna lejdud-nneɣ ugin a s-aɣen awal ; ur t-qbilen ara axaṭer deg ulawen-nsen bɣan ad uɣalen ɣer tmurt n Maṣer.

7:40 Nnan i Haṛun gma-s n Sidna Musa : Xdem-aɣ-d iṛebbiten am wid n imaṣriyen, ara nezzwer zdat-nneɣ, axaṭer Musa-agi i ɣ-d-yessufɣen si tmurt n Maṣeṛ, ur neẓri ara d acu i gedṛan yid-es.

7:41 Dɣa deg ussan-nni, jemɛen-d ddheb, ssefsin-t xedmen yis yiwen ugenduz. Imiren zlan-as asfel, xedmen-as tameɣṛa, feṛḥen s wayen xedmen yifassen-nsen.

7:42 Sidi Ṛebbi yeṭṭaxeṛ fell-asen, yeǧǧa-ten ad ɛebden itran n yigenni , d ayen i guran di tektabt n lenbiya : Ay agdud n wat Isṛail, eɛni i nekk iwumi d-tefkam lewɛadi d iseflawen, di ṛebɛin iseggasen-nni i tesɛeddam deg unezṛuf ?

7:43 Xaṭi meɛna tbubbem aqiḍun n Mulux akk-d itri n Rufan i tḥesbem d iṛebbiten-nwen ;+ lmeṣnuɛat nni i txedmem iwakken a ten-tɛebdem ! Daymi ara kkun-nfuɣ ɣer lebɛid, akkin i temdint n Babilun !

7:44 Deg unezṛuf, lejdud-nneɣ sɛan aqiḍun i deg yella leɛqed n Sidi Ṛebbi ; aqiḍun-nni ițwaxedmen akken i s-t-id-yemla Sidi Ṛebbi i Sidna Musa mi s-d-yenna : Xdem-it am akken i t-twalaḍ di lemtel i k-d-ssekneɣ.

7:45 Mi d-yewweḍ uqiḍun-nni ɣer ifassen n lejdud-nneɣ , wwin-t-id yid-sen mi i ten-issekcem Yacuɛa ɣer tmurt-nni i sen-yewɛed Sidi Ṛebbi , mi ssufɣen imezdaɣ-is s tezmert n Sidi Ṛebbi. Aqiḍun-nni yeqqim dinna armi d zzman n Sidna Dawed.

7:46 Sidna Dawed eɛzizen ɣer Sidi Ṛebbi, yedɛa ɣuṛ-es yenna : A Sidi Ṛebbi ! A Yillu n jeddi-tneɣ Yeɛqub ! Eǧǧ-iyi ad a k-bnuɣ lǧameɛ iqedsen ara yilin ț-țanezduɣt-ik !

7:47 Lameɛna d Sidna Sliman i s yebnan lǧameɛ-nni !

7:48 Ɣas akken Sidi Ṛebbi ɛlayen ur izeddeɣ ara deg ixxamen ițwabnan s ufus n wemdan, akken i d-yenna nnbi Iceɛya :

7:49 Igenni d amkan n lḥekma-w, ma d lqaɛa ț-țaḥsiṛt i ɣef srusuɣ idaṛṛen-iw. Sidi Ṛebbi yenna-d : Acu n wexxam i tzemrem a yi-tebnum, neɣ anda-t wemkan i deg zemreɣ ad steɛfuɣ ?

7:50 Eɛni mačči d afus-iw i d-ixelqen kullec ?

7:51 A wid isbeddayen iɛenqiqen nsen ! Tesɣaṛayem ulawen-nwen, tqeflem imeẓẓuɣen-nwen, tețțeddum di nneqma i Ṛṛuḥ iqedsen am kunwi am lejdud-nwen !

7:52 Anwa nnbi ur qhiṛen ara lejdud-nwen ? Nɣan wid akk ițbecciṛen si zik ɣef Uḥeqqi ara d-yasen ; tura d kunwi i t-ixedɛen yerna tenɣam-t.

7:53 Tewweḍ-ikkun-id ccariɛa n Sidi Ṛebbi s ufus n lmalayekkat, lameɛna kunwi tugim a s-taɣem awal !

7:54 Mi slan i yemslayen-agi, lmuqedmin iffeɣ-iten leɛqel, qeṛcen tuɣmas-nsen fell-as.

7:55 Lameɛna Stifan yeččuṛen d Ṛṛuḥ iqedsen, yerfed allen-is ɣer igenni, iwala tamanegt n Sidi Ṛebbi akk-d Sidna Ɛisa ibedden ɣer uyeffus-is.

7:56 Imiren iɛeggeḍ, yenna : Atan țwaliɣ igenwan ldin, țwaliɣ Mmi-s n bunadem ibedd ɣer tama tayeffust n Ṛebbi !

7:57 Qeflen imeẓẓuɣen-nsen, bdan țɛeggiḍen, ẓedmen fell-as akken ma llan.

7:58 Zzuɣṛen-t beṛṛa n temdint, ṛejmen-t s iblaḍen iwakken a t-nɣen. Inigan ǧǧan leḥwayeǧ-nsen ɣer yiwen ilemẓi isem-is Caɛul, iwakken a ten-iɛas.

7:59 M'akken i t-ṛeǧmen, Stifan ideɛɛu ɣer Sidi Ṛebbi yeqqaṛ : A Sidna Ɛisa, atan ṛṛuḥ-iw ger ifassen-ik !

7:60 Imiren yeɣli ɣef tgecrar, iɛeggeḍ s lǧehd-is : A baba Ṛebbi ur ten-țḥasab ara ɣef ddnub-agi ! Akken kan i d-yenna imeslayen agi, iffeɣ-it ṛṛuḥ.

8:1 Caɛul, d yiwen seg wid iqeblen lmut n Stifan. Ass-nni kkren-d wid ibdan țqehhiṛen tajmaɛt n wid yețțamnen s Lmasiḥ yellan di temdint n Lquds. Wid akk yumnen s Lmasiḥ rewlen ɣer tudrin n tmurt n Yahuda armi ț-țamurt n Samarya ; anagar ṛṛusul i d-yeqqimen di temdint n Lquds.

8:2 Kra yergazen iḍuɛen Sidi Ṛebbi, wwin-d Stifan meḍlen-t, ḥeznen aṭas, țrun fell-as.

8:3 Ma d Caɛul ixeddem akk lmejhud-is iwakken ad ihudd tajmaɛt n wid yumnen s Sidna Ɛisa ; issufuɣ ed seg ixxamen-nsen, irgazen ț-țilawin yumnen s Lmasiḥ, issekcam iten ɣer lḥebs.

8:4 Ihi wid akk i grewlen, mfaṛaqen, țṛuḥun seg wemkan ɣer wayeḍ, țbecciṛen lexbaṛ n lxiṛ n wawal n Sidi Ṛebbi.

8:5 Filbas iṣubb ɣer temdint n Samarya, ibecceṛ dinna lexbaṛ n lxiṛ yeɛnan Lmasiḥ.

8:6 Lɣaci meṛṛa rran ddehn-nsen ɣer Filbas, mi slan i wayen i d-yeqqaṛ yerna walan lbeṛhanat i gxeddem ;

8:7 axaṭer leǧnun țeffɣen s leɛyaḍ seg waṭas n yemdanen ițțumelken, aṭas daɣen n iquḍaṛen akk-d wid ikerfen i geḥlan.

8:8 Dɣa imezdaɣ n temdint-nni meṛṛa feṛḥen mačči d kra.

8:9 Yuɣ lḥal di temdint-nni yella yiwen wergaz isem-is Semɛun, issewham imezdaɣ n tmurt n Samarya s ssḥur i gxeddem ; iḥețțeb iman-is d win yesɛan tazmert.

8:10 Imdanen akk am umeẓyan am umeqqran, țțaran ddehn-nsen ɣuṛ-es qqaṛen : « Argaz-agi ț-țazmert n Ṛebbi s yiman-is ; ț-țazmert-nni iwumi neqqaṛ tazmert yessewhamen ! »

8:11 Țțamnen yis axaṭer si zik i ten-issewham s ssḥur i gxeddem ;

8:12 lameɛna, asmi umnen s wayen i sen-ibecceṛ Filbas ɣef wayen yeɛnan tagelda n Sidi Ṛebbi d yisem n Sidna Ɛisa Lmasiḥ, usan-d akk ad țwaɣeḍsen deg waman am yergazen am tlawin.

8:13 Semɛun s yiman-is yumen ; ula d nețța yețwaɣḍes deg waman, seg wass-nni ur yettixiṛ ara ɣef Filbas ; ițțawham deg-s mi gețwali leɛǧayeb d lbeṛhanat timeqqranin i gxeddem.

8:14 Mi slan belli lɣaci n tmurt n Samarya qeblen awal n Sidi Ṛebbi, ṛṛusul yeqqimen di temdint n Lquds ceggɛen-asen Buṭrus d Yuḥenna.

8:15 Mi wwḍen ɣuṛ-sen, dɛan fell asen ɣer Sidi Ṛebbi iwakken a sen-d-yefk Ṛṛuḥ iqedsen ;

8:16 axaṭer mazal ur d-iris ula ɣef yiwen deg-sen, țwaɣeḍsen kan deg aman s yisem n Sidna Ɛisa.

8:17 Buṭrus akk-d Yuḥenna ssersen ifassen-nsen fell-asen, dɣa yețțunefk asen-d Ṛṛuḥ iqedsen.

8:18 Mi gwala Semɛun yețțunefk-ed Ṛṛuḥ iqedsen i lɣaci imi ssersen ṛṛusul ifassen-nsen fell-asen, yewwi-yasen-d idrimen yenna-yasen :

8:19 Ffket-iyi ula d nekk tazmert agi iwakken kra n win i ɣef ara sserseɣ ifassen-iw, a t-ikcem Ṛṛuḥ iqedsen.

8:20 Lameɛna Buṭrus yenna-yas : Ad negren yedrimen-ik ad glun yis-ek imi tɣileḍ Ṛṛuḥ iqedsen n Sidi Ṛebbi yețnuzu s idrimen !

8:21 Ul-ik ur yeṣfi ara ɣer Sidi Ṛebbi ! Ur k-id-iṣaḥ umur, ur tesɛiḍ lweṛt di ccɣel-agi.

8:22 Endem di yir ṛṛay-ik, tedɛuḍ ɣer Sidi Ṛebbi iwakken ma yella wamek, a k-yeɛfu ɣef yir axemmem-agi i d-yeffɣen seg ul-ik !

8:23 Atan walaɣ teččuṛeḍ d ddɣel, imlek-ik lbaṭel !

8:24 Semɛun yerra-yasen : Dɛut kunwi s yiman-nwen fell-i ɣer Sidi Ṛebbi iwakken ur d-ideṛṛu ara yid-i wayen akka i d-tennam.

8:25 Buṭrus d Yuḥenna kemmlen cehden ɣef Sidna Ɛisa, beccṛen awal n Sidi Ṛebbi, mbeɛd uɣalen ɣer temdint n Lquds. Deg webrid-nsen, țbecciṛen lexbaṛ n lxiṛ deg waṭas n tudrin n tmurt n Samarya.

8:26 Yiwen lmelk n Sidi Ṛebbi inṭeq-ed ɣer Filbas yenna-yas : Kker, ṛuḥ ɣer lǧiha n usammer, ɛeddi seg webrid-nni yexlan i gețṣubbun si temdint n Lquds ɣer tmurt n Ɣaza.

8:27 Ikker, iṛuḥ. Immuger-ed deg webrid yiwen n wergaz, d aneɣlaf n Kendyas tagellit n tmurt n Ityubya ; d nețța i d lewkil ɣef lmelk-is meṛṛa , lameɛna d argaz ur nesɛi ddunit ; yusa-d ɣer temdint n Lquds iwakken ad iɛbed Sidi Ṛebbi.

8:28 Mi gțeddu ɣef tkeṛṛust-is iwakken ad yuɣal ɣer wexxam-is, yufa-t-id lḥal yeqqaṛ di tektabt n nnbi Iceɛya.

8:29 Ṛṛuḥ iqedsen yenna i Filbas : Ṛuḥ qeṛṛeb ɣer tkeṛṛust-ihina.

8:30 Filbas yuzzel ɣuṛ-es, yesla i uneɣlaf-nni n tmurt n Ityubya yeqqaṛ di tektabt n nnbi Iceɛya. Yenna-yas : Tfehmeḍ ayen akka i teqqaṛeḍ ?

8:31 Yerra-yas : Amek ara fehmeɣ ma ulac win ara yi-d-isfehmen ? Dɣa yesserkeb-ed Filbas ɣer tama-s.

8:32 Atah wayen akken yella yeqqaṛ di tektabt-nni : Am izimer m'ara t-awin a t-zlun, am tixsi iggugmen zdat win i ț-ițellsen, ur d-yeldi imi-is.

8:33 Yețwaḥqeṛ, ițwačč lḥeqq-is ! Anwa ara d-yeḥkun ɣef wayen yedṛan di lǧil-ines ? Anwa ara d-yeḥkun ɣef tudert-is yețwakksen si ddunit ?

8:34 Aneɣlaf-nni yenna-yas i Filbas : Di leɛnaya-k ssefhem-iyi-d ɣef wanwa i d-ițmeslay nnbi-yagi, ɣef yiman-is neɣ ɣef wayeḍ ?

8:35 Dɣa Filbas s wawal-agi i gebda a s-ițbecciṛ lexbaṛ n lxiṛ yeɛnan Sidna Ɛisa.

8:36 Mi teddun deg webrid-nsen, wwḍen ɣer yiwen wemkan i deg llan waman. Aneɣlaf-nni yenna-yas : Atnan waman ! D acu ara yi-d iḥebsen iwakken ad țwaɣedṣeɣ ?

8:37 Filbas yenna-yas : Ma tumneḍ seg ul-ik, tzemreḍ aț-tețwaɣeḍseḍ. Aneɣlaf-nni yerra-yas : Umneɣ belli Sidna Ɛisa Lmasiḥ d Mmi-s n Ṛebbi.

8:38 Imiren yumeṛ ad ḥebsen takeṛṛust-is. Filbas d uneɣlaf-nni rsen-d, kecmen i sin ɣer waman, dɣa Filbas isseɣḍes-it.

8:39 Mi d-ffɣen seg waman, Ṛṛuḥ n Sidi Ṛebbi yerfed Filbas, yexfa ger wallen uneɣlaf-nni. Aneɣlaf-nni ikemmel abrid-is s lfeṛḥ ;

8:40 ma d Filbas yufa-d iman-is di temdint n Acdud ; syenna ikemmel abrid-is, ițeddu ițbecciṛ awal n tudert di tudrin meṛṛa i ɣef iɛedda, armi i gewweḍ ɣer temdint n Qiṣarya.

9:1 Caɛul mazal ițqehhiṛ inelmaden n Sidna Ɛisa, ițgalla deg-sen a ten-ineɣ. Iṛuḥ ɣer lmuqeddem ameqqran,

9:2 issuter-as a s-d-ixdem lkaɣeḍ ara yawi ɣer leǧwameɛ meṛṛa n temdint n Dimecq, iwakken ma yufa deg-sen wid ittabaɛen abrid n Ɛisa Lmasiḥ, ama d argaz ama ț-țameṭṭut, a ten-id-yawi țwarzen ɣer temdint n Lquds.

9:3 Yella iteddu deg webrid, qṛib ad yaweḍ ɣer temdint n Dimecq. Taswiɛt kan, tekka-d yiwet n tafat seg igenni tfeǧǧeǧ-ed fell-as, tezzi yas-d.

9:4 Yeɣli ɣer lqaɛa, yesla i yiwen n ṣṣut i s-d-yeqqaṛen : Caɛul ! A Caɛul ! Acuɣeṛ i yi tețqehhiṛeḍ ?

9:5 Yenna-yas : Anwa-k a Sidi ? Sṣut-nni yerra-yas-d : D nekk i d Ɛisa, win akken i tețqehhiṛeḍ !

9:6 Kker fell-ak ! Ṛuḥ ɣer temdint n Dimecq, dinna ara k-d-inin acu ara txedmeḍ.

9:7 Irgazen i geddan yid-es bedden, dehcen, ggugmen ; slan i taɣect-nni lameɛna ur walan ula d yiwen.

9:8 Caɛul ikker-ed si lqaɛa, ɣas akken ldint wallen-is, ur yețwali ara. Sṭfen-as afus, wwin-t ɣer temdint n Dimecq.

9:9 Yeqqim tlata wussan ur iwala, ur ičči, ur yeswi. Caɛul yumen s Ɛisa Lmasiḥ

9:10 Di temdint n Dimecq, yella yiwen unelmad isem-is Ananyas ; Sidi Ṛebbi yessawel-as-d deg uweḥḥi, Ananyas yerra-yas : Aql-iyi a Sidi.

9:11 Sidi Ṛebbi yenna-yas : Kker tṛuḥeḍ ɣer webrid ițusemman « Abrid ayeffus », ɣer wexxam n Yuda ; dinna steqsi ɣef yiwen wergaz isem-is Caɛul n temdint n Sars. Atan ideɛɛu ɣer Sidi Ṛebbi,

9:12 yețwali deg uweḥḥi ikcem ɣuṛ-es yiwen wergaz isem-is Ananyas, yessers ifassen-is fell-as iwakken a t-id-yuɣal yeẓri.

9:13 Ananyas yerra-yas : A Sidi, acḥal n yemdanen i d yeḥkan fell-as ițqehhiṛ inelmaden-ik imqedsen di temdint n Lquds !

9:14 Yusa-d ɣer dagi daɣen s țțesriḥ n lmuqedmin imeqqranen iwakken ad yerr ɣer lḥebs wid akk ineddhen s yisem-ik.

9:15 Lameɛna Sidna Ɛisa yenna-yas : Ṛuḥ, axaṭer argaz-agi d nekk i t-yextaṛen iwakken ad ibecceṛ isem-iw i wat Isṛail, i leǧnas nniḍen akk-d igelliden-nsen.

9:16 Yerna a s ssekneɣ acḥal ara yeɛteb ɣef ddemma n yisem-iw.

9:17 Ananyas iṛuḥ ; mi gewweḍ ɣer wexxam-nni, yessers ifassen-is ɣef Caɛul yenna-yas : A Caɛul a gma, d Sidna Ɛisa-nni i k-d-iḍehṛen deg-webrid ansi i d-tusiḍ, i yi-d-iceggɛen iwakken a k-id-yuɣal yeẓri yerna aț-țeččaṛeḍ d Ṛṛuḥ iqedsen.

9:18 Imiren kan ɣlin-d seg wallen-is am akken d iqecṛan, dɣa yuɣal-it-id yeẓri ; yekker, yețwaɣḍes.

9:19 Mi gečča, yuɣal-it-id lǧehd ; isɛedda kra wussan ɣer inelmaden i gellan di temdint n Dimecq.

12:3 Mi gwala d ayen isfeṛḥen at Isṛail, yerna daɣen isseḥbes Butṛus, deg wussan n lɛid n leslak.

12:4 Mi t-id-ṭṭfen, rran-t ɣer lḥebs. Isbedd-as taɛessast n ṛebɛa trebbaɛ n lɛeskeṛ, mkul tarbaɛt deg-s ṛebɛa iɛeskṛiwen ; yebɣa a t-icaṛeɛ zdat lɣaci meṛṛa, m'ara tɛeddi Tfaska n yizimer.

12:5 Butṛus ihi yella di lḥebs ; ma ț-țajmaɛt n watmaten tezga di tẓallit, tdeɛɛu fell-as ɣer Sidi Ṛebbi.

12:6 Iḍ-nni uqbel a t-icaṛeɛ Hiṛudus, Butṛus ițwarzen s snat snasel, yella iṭṭes ger sin iɛeskṛiwen , wiyaḍ ɛussen tawwurt n lḥebs.

12:7 Taswiɛt kan, ataya lmelk n Sidi Ṛebbi iḍheṛ-as-d ; yiwet n tafat tameqqrant tceɛceɛ-ed di lḥebs-nni. Lmelk-nni yennul idis n Buṭrus, issendekwal-it-id yenna-yas : Ɣiwel Kker ! Imiren kan, qqeṛsent snasel-nni ɣlint-ed seg ifassen-is.

12:8 Lmelk-nni yenna-yas : Els-ed arkasen-ik tbeggseḍ ɣef yiman-ik ! Buṭrus ixdem akken i s-d-yenna, lmelk yenna-yas-d daɣen : Sburr-ed abeṛnus-ik tettebɛeḍ-iyi-d !

12:9 Butṛus ikker itbeɛ-it ; ur ifaq ara belli lmelk nni ț-țideț iḍheṛ-as-ed, inwa d aweḥḥi n Sidi Ṛebbi.

12:10 Ɛeddan zdat tɛessast tamezwarut, rnan tis snat, wwḍen ɣer tewwurt n wuzzal yessufuɣen ɣer temdint, teldi weḥd-es ; ffɣen ɛeddan deg yiwet n tezniqt, imiren lmelk yexfa ger wallen n Buṭrus.

12:11 Mi d-yufa iman-is di beṛṛa, ifaq belli ț-țideț, Sidi Ṛebbi iceggeɛ-d lmelk-is, isellek-it-id seg-ufus n Hiṛudus akk-d lbaṭel i bɣan a s-t xedmen wat Isṛail.

12:12 Mi gefhem lḥala i deg yella, ikker iṛuḥ ɣer wexxam n Meryem yemma-s n Yuḥenna iwumi qqaṛen daɣen Maṛqus, nnejmaɛen dinna waṭas n watmaten, deɛɛun ɣer Sidi Ṛebbi.

12:13 Mi gesqeṛbeb di tewwurt n beṛṛa, tusa-d yiwet n tqeddact isem-is Rudya, tqeṛṛeb ɣer tewwurt aț-țesmeḥses ;

12:14 mi teɛqel taɣect n Buṭrus, di lfeṛḥ-nni i deg tella ur teldi ara tawwurt, tuzzel aț-țissiweḍ lexbaṛ belli Butṛus atan ibedd zdat tewwurt.

12:15 Nnan-as : Waqila iffeɣ-ikem leɛqel ! Lameɛna teṭṭef deg wayen i d tenna.

12:16 Nnan-as : Ahat d lexyal-is ihi ! Llan țemsteqsayen, ma d Butṛus mazal-it isqeṛbub-ed ; mi d-ldin tawwurt wehmen mi t-walan.

12:17 Buṭrus iwehha-yasen s ufus-is ad ssusmen, yeḥka-yasen amek i t-id issufeɣ Sidi Ṛebbi si lḥebs, yenna yasen : Sṣiwḍet lexbaṛ-agi i Yeɛqub d watmaten. Dɣa iṛuḥ ibeddel amkan.

12:18 Mi guli wass, lɛeskeṛ iɛussen Butṛus tqelqen, țmesteqsayen amek i geffeɣ si lḥebs, anwa i t-yessufɣen ?

12:19 Hiṛudus iceggeɛ a t-id-qellben, meɛna ur t-ufin ara. Yebḥet iɛessasen-nni, dɣa yefka lameṛ a ten-nɣen ; imiren iṛuḥ si tmurt n Yahuda ɣer tmurt n Qiṣarya, anda yesɛedda kra n wussan.

12:20 Hiṛudus yerfa ɣef yimezdaɣ n temdinin n Sur akk d Sidun, lameɛna nutni mcawaṛen, msefhamen akk-d Blastus yellan d lewkil-is ; dɣa ceggɛen wid ara s-issutren lehna, axaṭer tamurt-nsen tețɛic seg wayen i d-itekken si tmurt n ugellid Hiṛudus.

12:21 Yewweḍ-ed wass i deg ara d-yini Hiṛudus lxeṭba i lɣaci. Mi d-yewweḍ wass n temlilit, Hiṛudus yelsa llebsa n ugellid, yuli ɣer wemkan n lxeṭba ad immeslay i lɣaci.

12:22 Mi gebda ameslay, lɣaci meṛṛa qqaṛen : D Ṛebbi i d-ițmeslayen, mačči d amdan !

12:23 Imiren kan lmelk n Sidi Ṛebbi yesseɣli-d fell-as aṭan, axaṭer yeqbel a t-ḥesben yemdanen am Ṛebbi, iceggeɛ-as-d abeɛɛuc i t-yeččan dɣa yemmut ;

12:24 ma d awal n Sidi Ṛebbi țbecciṛen-t, yewweḍ ɣer mkul amkan.

12:25 Barnabas akk-d Caɛul mi fukken lxedma-nsen di temdint n Lquds uɣalen, wwin yid-sen Yuḥenna ițusemman Maṛqus.

13:1 Di tejmaɛt n temdint n Antyuc llan lenbiya d lɛulama : Barnabas, Semɛun ițusemman Aberkan, Lusyus n temdint n Qiṛwan, Manahen i d-ițțuṛebban d Hiṛudus Antibas, akk-d Caɛul.

13:2 Yiwen wass atmaten nnejmaɛen iwakken ad uẓummen yerna ad dɛun ɣer Sidi Ṛebbi. Ṛṛuḥ iqedsen yenna-yasen-d : Swejdet-iyi-d Barnabas akk-d Caɛul i ccɣel-nni iwumi i ten-heggaɣ.

13:3 Mi ẓummen, ssersen ifassen-nsen fell-asen, rnan dɛan ɣer Sidi Ṛebbi, dɣa ǧǧan-ten ad ṛuḥen.

13:4 Barnabas akk-d Caɛul i gceggeɛ Ṛṛuḥ iqedsen, ṣubben ɣer lmeṛṣa n Slukya ; syenna rekben di lbabuṛ ɣer tegzirt n Qubṛus.

13:5 Mi wwḍen ɣer lmeṛṣa n Salamin, beccṛen awal n Ṛebbi di leǧwameɛ n wat Isṛail ; Yuḥenna yella yid-sen, yețɛawan-iten.

13:6 Mi zegren akk tigzirt-nni, wwḍen ɣer temdint n Bafus, mlalen-d d yiwen useḥḥar n wat Isṛail i gețțarran iman-is d nnbi, isem-is Barɛisa.

13:7 Ițțili ɣer lḥakem Serjyus Bulus, yellan d argaz ifehmen. Serjyus iceggeɛ ɣer Barnabas d Caɛul, axaṭer yebɣa ad isel i wawal n Ṛebbi.

13:8 Meɛna Ilimas aseḥḥar-nni, ( akkagi i gețwaṭerjem yisem-is s tyunanit ) yețxaṣam-iten, yețnadi ad isseɛṛeq abrid n liman i lḥakem-nni.

13:9 Caɛul, ițusemman daɣen Bulus, yeččuṛen d Ṛṛuḥ iqedsen, issers allen-is fell-as,

13:10 yenna-yas : A mmi-s n Cciṭan yeččuṛen ț-țiḥila d lexdeɛ ! Ay aɛdaw n lḥeqq ! Ar melmi ara tesseɛwajeḍ akka iberdan n lewqam n Sidi Ṛebbi ?

13:11 Tura atan ufus n Sidi Ṛebbi fell-ak : aț-țedderɣleḍ, aț-țeqqimeḍ kra n wussan ur tețwaliḍ ara tafat n yiṭij. Imiren kan, Ilimas idderɣel, yeɣli-d fell-as ṭṭlam ; yeqqim isferfud, ițnadi anwa ara s-yeṭṭfen afus.

13:12 Lḥakem-nni mi gwala ayen yedṛan, itɛeǧǧeb deg wawal n Sidi Ṛebbi dɣa yumen.

13:13 Bulus d yeṛfiqen-is zegren tigzirt n Qubṛus, wwḍen ɣer temdint n Bafus, syenna rekben di lbabuṛ iwakken ad zegren ɣer temdint n Barja yellan di tmurt n Bamfilya.

13:14 Mi wwḍen ɣer dinna, Yuḥenna yeǧǧa-ten yuɣal ɣer temdint n Lquds ; ma d nutni kemmlen abrid-nsen si temdint n Barja, wwḍen ɣer temdint n Antyuc yellan di tmurt n Bisidya. Deg wass n westeɛfu, i deg țnejmaɛen wat Isṛail di leǧwameɛ, kecmen ɣer lǧameɛ, qqimen.

13:15 Mi d-ɣṛan tiktabin n Sidna Musa akk-d Lenbiya, imeqqranen n lǧameɛ nni nnan-asen : Ay atmaten, ma yella tesɛam kra n wawal s wayes ara tenhum lɣaci-agi, ɛeddit-ed aț-țhedṛem !

13:16 Bulus ikker-ed, iwehha-yasen s ufus-is, yenna : Ay at Isṛail ! A kunwi iḍuɛen Ṛebbi, ḥesset-iyi-d !

13:17 Sidi Ṛebbi Illu n wat Isṛail, yextaṛ lejdud-nneɣ, yessefti-ten mi llan di tmurt n Maṣeṛ, yessufeɣ iten id s tezmert-is tameqqrant.

13:18 Iḥuder-iten azal n ṛebɛin iseggasen deg unezṛuf n Sinay.

13:19 Issenger sebɛa leǧnas di tmurt n Kenɛan, iwakken a sen-yefk tamurt-nni i lejdud-nneɣ.

13:21 Dɣa ssutren i Sidi Ṛebbi a sen-d-yefk agellid ; yefka-yasen-d Caɛul, mmi-s n Qic n wedrum n Benyamin, yuɣal d agellid fell-asen azal n ṛebɛin iseggasen.

13:22 Lameɛna Sidi Ṛebbi yekkes leḥkum i Caɛul, yerra Sidna Dawed d agellid deg umkan-is. Atan wayen i d-yenna fell-as : Ufiɣ Dawed, mmi-s n Yassa, d argaz yellan akken i t-yebɣa wul-iw, ara ixedmen lebɣi-w.

13:23 Tura, si dderya n Sidna Dawed i d-ifka Sidi Ṛebbi amsellek i wat Isṛail akken i t-id-yewɛed ; amsellek-agi d Ɛisa.

13:24 Uqbel a d-yas Sidna Ɛisa, Yeḥya ibecceṛ țțuba s weɣḍas n waman i wegdud n wat Isṛail.

13:25 Uqbel a t-ṭṭfen ɣer lḥebs iwakken a t-nɣen, Yeḥya yenna i lɣaci : Nekk ur lliɣ ara d win akken i tɣilem ; lameɛna a d-yas deffir-i yiwen, ur uklaleɣ ara ad fsiɣ ula d lexyuḍ n warkasen-is.

13:26 Ay atmaten ! Ay arraw n Sidna Ibṛahim d wid akk ițțaggaden Sidi Ṛebbi ! I nukkni iwumi i d-ițțuceggeɛ wawal-agi n leslak.

13:27 Axaṭer imezdaɣ n temdint n Lquds d imeqqranen-nsen, ur fhimen ara d acu-t Ɛisa-agi, ur fhimen ara daɣen imeslayen i d qqaṛen di tektabt n lenbiya mkul ass n westeɛfu ; lameɛna xedmen akken i gura di tira iqedsen, mi ḥekmen ɣef Sidna Ɛisa s lmut.

13:28 Ɣas akken ur s-d-ufin ara sebba s wayes ara ḥekmen fell-as s lmut, ssutren i Bilaṭus ad yefk lameṛ iwakken a t-semmṛen ɣef lluḥ.

13:29 Mi gedṛa wayen akk ițwaketben fell-as, ṣubben-t-id seg umidag, rran-t ɣer uẓekka.

13:30 Lameɛna Sidi Ṛebbi isseḥya-t-id si ger lmegtin.

13:31 Acḥal wussan nețța ițbeggin-ed iman-is i widak-nni i d-yeddan yid-es si tmurt n Jlili armi ț-țamdint n Lquds ; tura d nutni i d inigan-is zdat n yemdanen meṛṛa.

13:32 Daymi, nukkni tura newwi yawen-d lexbaṛ-agi n lxiṛ. Ayen akken i gewɛed Sidi Ṛebbi i lejdud nneɣ, yedṛa-d tura gar-aneɣ, nukkni yellan d dderya-nsen.

13:33 Mi d-yesseḥya Sidi Ṛebbi Sidna Ɛisa, akken yura di Sabur wis-sin : Kečč d mmi, d nekk i k-id-yefkan ɣer ddunit ass-a !

13:34 Ihi Sidi Ṛebbi isseḥya-t-id si ger lmegtin iwakken ur irekku ara, akken i t-id-yenna di tira iqedsen : Si Dawed ara wen-d-fkeɣ lbaṛakat yeṣfan, lbaṛakat n tideț i wen weɛdeɣ. +

13:35 Yura daɣen : Ur tețțaǧǧaḍ ara ameɛzuz-ik ad yerku zdaxel n uẓekka .

13:36 Di lweqt-is, Sidna Dawed yexdem lebɣi n Sidi Ṛebbi ; mi gemmut, yețwamḍel ger lejdud-is, tura lǧețța-s terka deg uẓekka.

13:37 Lameɛna win i d-yesseḥya Sidi Ṛebbi, ur t-yeǧǧi ara ad yerku deg uẓekka.

13:38 Ilaq ihi aț-țeẓrem ay atmaten, belli s yisem-is i wen-d-ițțubecceṛ leɛfu n ddnubat.

13:39 Kra n win yumnen yis, a s-țwasemmḥen ddnubat-is, ddnubat-nni i ɣef ur tezmirem ara aț-țețwasemmḥem s ccariɛa n Musa.

13:40 Ihi ɣuṛ-wat a d-yedṛu yid-wen wayen i d-nnan lenbiya :

13:41 Ay imdanen yeččuṛen d zzux, iḥeqqṛen wiyaḍ, dehcet, tenfum syagi ! Axaṭer atan ad xedmeɣ di zzman-nwen yiwet lḥaǧa s wayes ur tețțamnem ara ma yella ḥkan awen-ț-id.

13:42 Mi kkren ad ffɣen si lǧameɛ, ḥellelen Bulus d Barnabas iwakken a d-uɣalen ass n ssebt i d-iteddun a d-rnun a d-mmeslayen ɣef wannect nni.

13:43 Mi mfaṛaqen, aṭas n wat Isṛail akk-d wid i gkecmen ddin-nsen, tebɛen-d Bulus akk-d Barnabas i sen-ițmeslayen, i ten-inehhun ad ṭṭfen di ṛṛeḥma n Sidi Ṛebbi.

13:44 Ass n westeɛfu nniḍen, qṛib tamdint meṛṛa i d-innejmaɛen iwakken ad slen i wawal n Ṛebbi.

13:45 Mi walan annect-nni n lɣaci, at Isṛail usmen, sseɛwajen s rregmat ayen i d-yeqqaṛ Bulus.

13:46 Lameɛna Bulus d Barnabas nnan-asen ɛinani : I kunwi d imezwura iwumi ilaq ad ițțubecceṛ wawal n Sidi Ṛebbi, meɛna imi ur t-teqbilem ara, tḥusbem iman-nwen ur tuklalem ara tudert n dayem, ihi a nezzi ɣer wid ur nelli ara n wat Isṛail.

13:47 Axaṭer atah wayen i d-yenna Sidi Ṛebbi : Sbeddeɣ-k aț-țiliḍ ț-țafat i leǧnas, aț-țessiwḍeḍ leslak alamma d ixfawen n ddunit.

13:48 Leǧnas ur nelli ara n wat Isṛail feṛḥen mi slan s wannect-agi, țḥemmiden awal n Ṛebbi ; wid akk yețțuheggan i tudert n dayem umnen.

13:49 Awal n Sidi Ṛebbi yețțubecceṛ di mkul amkan n tmurt.

13:50 Lameɛna at Isṛail sḥeṛcen imeqqranen n temdint akk-d d kra n tilawin timeṛkantiyin i gettabaɛen ddin n wat Isṛail ; sekkren-d aqehheṛ ameqqran ɣef Bulus d Barnabas, ssufɣen-ten si tmurt-nsen.

13:51 Bulus d Barnabas zwin aɣebbaṛ n iḍaṛṛen-nsen , ṛuḥen ɣer temdint n Ikunyum.

13:52 Ma d inelmaden ijdiden, ččuṛen d lfeṛḥ akk-d Ṛṛuḥ iqedsen.

14:1 Di temdint n Ikunyum, Bulus d Barnabas kecmen daɣen ɣer lǧameɛ n wat Isṛail, beccṛen-asen lexbaṛ n lxiṛ ; aṭas n Iyunaniyen akk-d wat Isṛail i gumnen.

14:2 Lameɛna kra seg wat Isṛail i gugin ad amnen, ɛemṛen-asen aqeṛṛuy i wat n leǧnas niḍen sḥeṛcen-ten iwakken ad xedmen lbaṭel i watmaten ;

14:3 ɣas akken, Bulus d Barnabas ɛeṭṭlen di temdint n Ikunyum , țeklen ɣef Sidi Ṛebbi, heddṛen s țțbut mbla akukru ɣef ṛṛeḥma-ines. Sidi Ṛebbi ițbeggin-ed s lbeṛhanat d licaṛat belli ayen țbecciṛen ț-țideț.

14:4 Imezdaɣ n temdint mfaṛaqen : kra deg-sen tebɛen at Isṛail, ma d wiyaḍ ddan d ṛṛusul.

14:5 At Isṛail d iyunaniyen akk-d imeqqranen-nsen ɛewlen ad xedmen lbaṭel i ṛṛusul, bɣan a ten nɣen s weṛjam.

14:6 Bulus d Barnabas mi ten-yewweḍ lexbaṛ, beddlen amkan ṛuḥen ɣer tmurt n Likawnya, ɣer temdinin n Listra, d Derba akk-d tudrin i sent-id yezzin ;

14:7 țbecciṛen dinna lexbaṛ n lxiṛ n Sidi Ṛebbi.

14:8 Di temdint n Listra, yella yiwen wergaz ikerfen seg-wasmi i d-ilul, di leɛmeṛ-is ur yelḥi.

14:9 Yesmeḥsis i wayen i d-yeqqaṛ Bulus ; Bulus iṛeṣṣa allen-is fell-as, iwala belli yesɛa liman iwakken ad yeḥlu,

14:10 dɣa yenṭeq ɣur-es s ṣṣut ɛlayen yenna-yas : Ekker, bedd ɣef yidaṛṛen-ik ! Imiren kan yekker, yebda ițeddu.

14:11 Mi walan ayen yexdem Bulus, lɣaci n tmurt n Likawnya țɛeggiḍen s tutlayt nsen : Iṛebbiten ṣubben-d ɣuṛ-nneɣ s ṣṣifa n yemdanen !

14:12 Semman-as i Barnabas « SSus », ma d Bulus « Hermas » axaṭer d nețța i d-ițmeslayen.

14:13 Lmuqeddem n lǧameɛ n SSus yellan ɣer tewwurt n temdint, yewwi-d izgaren akk-d ijeǧǧigen zdat tewwura n lǧameɛ ; yebɣa a ten-yezlu d asfel nețța d lɣaci i Bulus akk-d Barnabas.

14:14 Ṛṛusul mi slan s wannect nni cerrgen llebsa-nsen, uzzlen ɣer tlemmast n lɣaci, țɛeggiḍen :

14:15 Ay irgazen, acuɣeṛ i txeddmem akka ? Nukni d imdanen kan i nella am kunwi ! D lexbaṛ n lxiṛ i wen-d newwi iwakken aț-țeṭṭixṛem i leɛbada n wayen ur nesɛi lmeɛna ; aț-tebɛem abrid n Sidi Ṛebbi, Bab n tudert i d-ixelqen igenni d lqaɛa, lebḥeṛ d wayen akk yellan deg-s.

14:16 Ɣas akken di zzmanat iɛeddan yeǧǧa imdanen n leǧnas meṛṛa ad tebɛen abrid i sen-yehwan ;

14:17 nețța yețbeggin-ed iman-is s lxiṛat i gxeddem : d nețța i wen-d-ițțaken ageffur ( lehwa ), d nețța i wen-d ițțaken lɣellat di lweqt-nsent, d nețța daɣen i wen-d-ițakken tamɛict-nwen, ula d ulawen-nwen d nețța i ten issefṛaḥen.

14:18 Ɣas akken ṛṛusul nnan asen imeslayen-agi, s lḥif ameqqran i ḥebsen lɣaci-nni iwakken ur sen zellun ara asfel.

14:19 Dɣa usan-d kra seg wat Isṛail si temdinin n Antyuc akk-d Ikunyum, sekkren ccwal ger lɣaci, sḥeṛcen-ten ad ṛejmen Bulus iwakken a t-nɣen. Mi t-ṛejmen, zzuɣṛen-t ɣer beṛṛa n temdint, axaṭer ɣilen yemmut.

14:20 Lameɛna nnejmaɛen-d ɣuṛ-es inelmaden, ṭṭfen-as afus, yuɣal yekker yekcem ɣer temdint. Azekka-nni, nețța d Barnabas ṛuḥen ɣer temdint n Derba.

14:21 Mi beccṛen lexbaṛ n lxiṛ di temdint-nni n Derba, aṭas inelmaden i d-yernan ; dɣa uɣalen ɣer temdinin n Listra, Ikunyum d Antyuc

14:22 iwakken ad sǧehden ulawen n inelmaden. Nehhun-ten ad ṭṭfen di liman, qqaṛen-asen : S waṭas n leɛtab ara nekcem ɣer ddewla n Sidi Ṛebbi.

14:23 Sbedden imḍebbṛen di mkul tajmaɛt ; mi uẓamen, rnan dɛan ɣer Sidi Ṛebbi, ǧǧan-ten seddaw leɛnaya n Sidi Ṛebbi s wayes umnen.

14:24 Zegren si tmurt n Bisidya, wwḍen-d ɣer tmurt n Bamfilya,

14:25 beccṛen awal n Ṛebbi di temdint n Barja, imiren ṣubben-d ɣer temdint n Aṭalya.

14:26 Syenna rekben lbabuṛ, uɣalen ɣer temdint n Antyuc ansi i ten wekklen i Ṛṛeḥma n Sidi Ṛebbi ɣef cceɣl-nni xedmen.

14:27 Mi wwḍen, snejmaɛen-d atmaten, ḥkan-asen ayen akk i gexdem Sidi Ṛebbi yis-sen, d wamek i d-wwin ɣer webrid n Sidi Ṛebbi at leǧnas ur nelli ara n wat Isṛail.

14:28 Sɛeddan aṭas n wussan akk-d inelmaden.

15:39 Ikker yiwen lxilaf gar-asen armi mfaṛaqen. Barnabas yewwi yid-es Maṛqus, rekben di lbabuṛ ɣer tegzirt n Qubṛus,

15:41 Yekka-d nețța d Silas si tmura n Surya akk-d Silisya, isseǧhad tijmaɛin n watmaten.

16:1 Bulus d Silas wwḍen ɣer temdinin n Derba d Listra ; ufan dinna yiwen unelmad isem-is Timuti, baba-s d ayunani, ma d yemma-s n wat Isṛail, nețțat daɣen tumen s Lmasiḥ.

16:2 Atmaten yellan di temdint n Listra ț-țemdint n Ikunyum ḥemmlen-t yerna țcekkiṛen-t.

16:3 Bulus yebɣa ad yawi yid-es Timuti, daymi i s-iḍehheṛ ɣef ddemma n wat Isṛail yellan dinna, axaṭer ẓran akk belli baba-s d ayunani.

16:4 Di yal tamdint i deg ɛeddan, țxebbiṛen atmaten s lewṣayat yellan di tebṛaț i d-fkan ṛṛusul d imeqqranen n temdint n Lquds, nehhun-ten ad tebɛen lewṣayat-agi.

16:5 Tijmaɛin n watmaten țimɣuṛent di liman, leḥsab n widak yumnen s Lmasiḥ yețnerni kull ass.

16:6 Ṛṛuḥ iqedsen ur sen-iserreḥ ara ad ṛuḥen ad beccṛen awal n Sidi Ṛebbi di tmurt n Asya, dɣa zegren timura n Frijya d Galasya.

16:7 Mi wwḍen ɣer tmurt n Misya, ɛeṛden ad ṛuḥen ɣer tmurt n Bitinya ; lameɛna Ṛṛuḥ n Sidna Ɛisa ur sen-iserreḥ ara daɣen ad ṛuḥen ɣer dinna.

16:8 Ɛeddan si tmurt n Misya, ṣubben ɣer temdint n Truwas.

16:9 Deg iḍ, iweḥḥa-yas-ed Ṛebbi i Bulus, iwala yiwen umezdaɣ n Masidunya ibedd-ed ɣuṛ-es, ițḥellil-it ad iɛeddi ɣuṛ-sen ɣer tmurt n Masidunya a ten-iɛiwen.

16:10 Mbeɛd ayen i s-d-iweḥḥa Sidi Ṛebbi i Bulus, nennuda amek ara nṛuḥ ɣer tmurt n Masidunya, axaṭer nefhem belli d Sidi Ṛebbi i ɣ-d yessawlen iwakken a nbecceṛ Lexbeṛ n lxiṛ di tmurt-nni.

16:11 Mi nerkeb di lbabuṛ si temdint n Truwas, nṛuḥ qbala ɣer tegzirt n Samutras. Azekka-nni nkemmel ɣer temdint n Nabulis.

16:12 Syenna nṛuḥ ɣer temdint n Filibus, illan țamdint tamezwarut n Masidunya i gesteɛmeṛ Ṛuman ; nesɛedda kra n wussan dinna.

16:13 Ass n westeɛfu, neffeɣ si temdint nṛuḥ ɣer rrif n wasif, nɣil a naf amkan anda țnejmaɛen i tẓallit. Nufa dinna kra n tilawin nnejmaɛent, neqqim iwakken a nemmeslay yid sent.

16:14 Tella dinna yiwet n tmeṭṭut isem-is Lidya n temdint n Tyatir, teznuzu lkețțan azeggaɣ ɣlayen ; ț-țameṭṭut iḍuɛen Sidi Ṛebbi. M'akken i d-tesmeḥsis, Sidi Ṛebbi yeldi-yas ul-is iwakken aț-țerr ddehn is ɣer wayen i d-yeqqaṛ Bulus.

16:15 Mi tețwaɣḍes deg waman nețțat d wat wexxam-is, teɛṛeḍ-aɣ, tenna-d : ma twalam umneɣ seg ul-iw s Sidi Ṛebbi, eyyaw aț-țesɛeddim kra n wussan deg wexxam-iw. Dɣa teḥreṣ-aɣ iwakken a nekcem.

16:16 Yiwen wass mi nteddu ɣer wemkan-nni n tẓallit, temmuger-aɣ-d yiwet n tqeddact ițwamelken, ț- țaderwict, tețțawi-d aṭas n lfayda i yemɛellmen i ɣef txeddem.

16:17 Tettabaɛ-aɣ-d nekk d Bulus, tețɛeggiḍ teqqaṛ : Imdanen-agi d iqeddacen n Ṛebbi ɛlayen, țbecciṛen-awen abrid n leslak.

16:19 Imɛellmen-is mi walan dayen ur sen-d-tețțawi ara lfayda, ṭṭfen Bulus d Silas, zzuɣṛen-ten ɣer wezniq n temdint anda țnejmaɛen lɣaci zdat lḥukkam.

16:22 Lɣaci kkren-d ɣuṛ-sen s leɛyaḍ, dɣa lḥukkam n Ṛuman umṛen a sen-cerrgen llebsa-nsen, a ten wwten s ujelkkaḍ ( s ucelliṭ ).

16:25 Di țnaṣfa n yiḍ, Bulus d Silas deɛɛun ɣer Sidi Ṛebbi, cennun, țḥemmiden-t, imeḥbas nniḍen smeḥsisen-asen.

16:26 Taswiɛt kan, tezlez tmurt armi yenhezz ula d lsas n lḥebs ; imiren kan ldint tewwura n lḥebs, snasel icudden imeḥbas meṛṛa qelɛent-ed.

16:27 Aɛessas-nni ikker-ed ; mi gwala tiwwura n lḥebs ldint, ijbed-ed asekkin is iwakken ad ineɣ iman-is ; yenwa rewlen imeḥbas-nni.

16:28 Lameɛna Bulus iɛeggeḍ s lǧehd is, yenna-yas : Eḥbes ! Ḥader aț-țenɣeḍ iman ik ! Aql-aɣ akk dagi.

16:29 Dɣa aɛessas yumeṛ a s-d-ceɛlen tafat, yuzzel ɣer daxel n lḥebs ; yeɣli ɣer idaṛṛen n Bulus d Silas, ițergigi.

16:30 Yessufeɣ-iten-id si lḥebs, yenna yasen : A ssyad-i, d acu i glaqen a t xedmeɣ iwakken ad țțusellkeɣ ?

16:34 Yuɣal nețța d Bulus d Silas ɣer wexxam-is, issers-asen-d ad ččen. Yefṛeḥ aṭas nețța d wat wexxam-is imi umnen s Sidi Ṛebbi.

16:38 Wid-nni i d-ițțuceggɛen ssawḍen imeslayen-nni i lḥukkam n Ṛuman. Lḥukkam ikcem-iten lxuf mi slan belli Bulus d Silas d iṛumaniyen i llan. Usan-d ḍelben deg-sen ssmaḥ, serrḥen-asen si lḥebs,

16:40 Mi ffɣen si lḥebs, Bulus d Silas ṛuḥen ɣer wexxam n Lidya, anda nnejmaɛen d watmaten, nhan-ten, imiren ṛuḥen.

17:1 Bulus d Silas ɛeddan si temdinin n Anfibulis akk-d Abulunis, wwḍen ɣer temdint n Tiṣalunik anda sɛan wat Isṛail lǧameɛ.

17:2 Akken yennum, Bulus yekcem ɣer lǧameɛ ; tlata smanat, mkul ass n westeɛfu, ițmeslay yid-sen ɣef wawal n Ṛebbi.

17:3 Issefham-asen, ițbeggin-asen-d di Tira belli Lmasiḥ ilaq ad yeɛteb yerna a d-yeḥyu si ger lmegtin, iqqaṛ-asen : Ɛisa-yagi ɣef i wen-d-hedṛeɣ, d nețța i d Lmasiḥ.

17:4 Kra deg-sen umnen ; aṭas n iyunaniyen iḍuɛen Sidi Ṛebbi d waṭas n tilawin n laṣel i geddan daɣen d Bulus akk-d Silas.

17:5 Lameɛna at Isṛail usmen, dɣa wwin yid-sen kra n yergazen n diri ițmenṭaṛen deg iberdan, sekkren ccwal ger lɣaci, cewwlen tamdint meṛṛa, zeḍmen ɣer wexxam n Yazun, qellben ɣef Bulus d Silas iwakken a ten-caṛɛen zdat lɣaci.

17:6 Imi ur ten-ufin ara, zuɣṛen-d Yazun d kra n watmaten ɣer lḥukkam n temdint, bdan țɛeggiḍen : D irgazen-agi i gerwin ddunit, atan usan-d armi d ɣuṛ-nneɣ,

17:7 Yazun isṭerḥeb yis-sen. Xulfen lumuṛ n Qayṣer, qqaṛen belli yella ugellid nniḍen ițusemman Ɛisa.

17:8 S yimeslayen-agi, sḥeṛcen lɣaci akk-d lḥakem n temdint.

17:9 Ugin a sen-serrḥen i Yazun d watmaten nniḍen armi xellṣen lexṭiya.

17:10 Mi d-yewweḍ yiḍ, atmaten srewlen Bulus d Silas ɣer temdint n Biri. Mi wwḍen, kecmen ɣer lǧameɛ n wat Isṛail.

17:11 Lɣaci n dinna sṭerḥben yis-sen axiṛ n wid n temdint n Tiṣalunik , qeblen awal n Ṛebbi s wul yeṣfan. Kull ass țnadin di Tira iqedsen iwakken ad walin ma yella ț-țideț, yella di tira wayen i d-qqaṛen.

17:12 Aṭas seg-sen i gumnen ; ger iyunaniyen, aṭas n tilawin n laṣel d yergazen nniḍen i gumnen daɣen.

17:13 Lameɛna mi slan wat Isṛail n Tiṣalunik belli Bulus ițbecciṛ awal n Ṛebbi di temdint n Biri, uzzlen-d ad rwin lɣaci, a ten-cewwlen.

17:20 Axaṭer nesla-yak tețmeslayeḍ-ed ɣef temsalin yessewhamen. Nebɣa a nissin lmeɛna-nsent.

17:21 Imezdaɣ n temdint n Atinya akk-d ibeṛṛaniyen i gzedɣen dinna, ḥemmlen ad sɛeddin lweqt-nsen anagar deg umeslay d usmeḥses i lexbaṛat ijdiden i d-ițeffɣen.

17:22 Bulus ibedd di tlemmast n Laryufaj yenna : Ay irgazen n temdint n Atinya, ẓriɣ-kkun tẓewṛem yerna tettabaɛem ddin di mkul ḥaǧa;

17:23 axaṭer ḥewwseɣ akk tamdint-nwen, walaɣ lmeṣnuɛat akk i tɛebdem ; ufiɣ daɣen yiwen n idebder anda yura : « I Ṛebbi-nni ur nețwassen ara. » Win akken i tɛebbdem mbla ma tessnem-t, d nețța i wen-țbecciṛeɣ.

17:24 D nețța i d-ixelqen ddunit d wayen yellan deg-s, d Bab igenwan d lqaɛa, ur yezdiɣ ara di leǧwameɛ i bnan ifassen n yemdanen.

17:25 Ur yeḥwaǧ ara ad xedmen fell-as yemdanen am akken yella wayen i t-ixuṣṣen, nețța i d-ițțaken i yemdanen meṛṛa tudert, ṛṛuḥ d wayen akk yellan.

17:26 Seg yiwen wemdan i d-ixleq akk leǧnas, yeɛmeṛ yis-sen ddunit ; yefka-yasen lewqat i tudert-nsen akk-d tilisa i tmura-nsen.

17:27 Sidi Ṛebbi ixdem annect-agi meṛṛa iwakken ad nadin fell-as ; ahat a t-afen s usferfed, nețța ur nebɛid ara ɣef mkul yiwen deg-nneɣ.

17:28 Yis i nesɛa tudert, i nteddu, i nețțili ; d ayen i d-nnan daɣen kra seg imedyazen-nwen : « ula d nukkni, darraw-is ! »

17:29 Ihi imi nella d arraw n Sidi Ṛebbi, ur ilaq ara a nɣil belli Ṛebbi yețțemcabi ɣer lmeṣnuɛ n ddheb, neɣ n lfeṭṭa, neɣ n wezṛu yețwanejṛen s ufus ț-țmusni n wemdan.

17:30 Sidi Ṛebbi isemmeḥ-asen i yemdanen asmi akken ur t-ssinen ara; tura issawel-asen-d akk di mkul amkan iwakken ad beddlen tikli, a d-uɣalen ɣuṛ-es.

17:31 Axaṭer yextaṛ ass i deg ara iḥaseb imdanen s lḥeqq, s wemdan i gextaṛ yerna yefka-yaɣ-d akk țțbut mi t-id-isseḥya si ger lmegtin.

17:32 Mi slan iheddeṛ-ed ɣef ḥeggu n lmegtin, kra țmesxiṛen, wiyaḍ qqaṛen-as : A k-nḥess ɣef wannect-agi tikkelt nniḍen...

17:33 Imiren, Bulus iṛuḥ yeǧǧa-ten.

17:34 Ɣas akken aṭas i gețmesxiṛen, llan kra deg-sen umnen, ddan yid-es . Yella gar-asen Dunisus i gțekkin di tejmaɛt n Laryufaj, yiwet n tmeṭṭut isem-is Damaris akk-d wiyaḍ.

18:1 Bulus iṛuḥ si temdint n Atinya ɣer temdint n Kurintus.

18:2 Yufa dinna yiwen wergaz n wat Isṛail, isem-is Akilas, laṣel-is n tmurt n Tqenṭert ; yewweḍ-ed deg ussan-nni kan si tmurt n Selyan nețța ț-țmeṭṭut-is Brisila, axaṭer Qayṣer Kludyus yumeṛ i wat Isṛail meṛṛa ad ffɣen si temdint n Ṛuma.

18:3 Imi Brisila d Akilas xeddmen iqiḍunen am Bulus, yeqqim ɣuṛ-sen xeddmen s tdukli.

18:4 Mkul ass n westeɛfu, Bulus ițmeslay di lǧameɛ, ițbecciṛ i wat Isṛail akk-d iyunaniyen ttabaɛen ddin n wat Isṛail, iwakken ad amnen.

18:5 Mi d-usan Silas d Timuti si tmurt n Masidunya, Bulus yuɣal ițțelhay kan d ubecceṛ n wawal n Ṛebbi, ițbeggin ed i wat Isṛail belli d Ɛisa i d Lmasiḥ.

18:6 Imi i t-xulfen yerna regmen-t, Bulus yezwi icuḍaḍ-is, yenna-yasen : Cfut ur teqbilem ara leslak-agi, ddnub i yirawen-nwen ! Nekk ayen i yi-d-yewwi lḥal xedmeɣ-t. Ihi sya d asawen, ad ṛuḥeɣ ɣer wat leǧnas ur nelli ara seg warraw n wat Isṛail.

18:7 Mi d-iffeɣ syenna, ikcem ɣer wexxam n Tit Justus, i gellan d amdan iḍuɛen Sidi Ṛebbi ; axxam-is yezga-d tama n lǧameɛ n wat Isṛail.

18:8 Krisbus, ameqqran n lǧameɛ yumen s Ɛisa Lmasiḥ nețța d wat wexxam-is meṛṛa ; aṭas n wat kurintus daɣen i gumnen mi slan s wayen i d-yenna Bulus, dɣa țwaɣeḍsen deg waman.

18:9 Yiwen n yiḍ, Sidi Ṛebbi iweḥḥa yas-ed di targit i Bulus yenna-yas-d : Ur țțaggad ara, kemmel ehdeṛ ur țsusum ara !

18:10 Axaṭer aqli-n yid-ek ! Yiwen ur izmir ad issers afus-is fell-ak iwakken a k-yexdem cceṛ, axaṭer aṭas i gellan d ayla-w di temdint-agi.

18:11 Bulus yeqqim aseggas d sețța wagguren ger wat kurintus, isselmad asen awal n Sidi Ṛebbi.

18:12 Di lweqt-nni i deg yella Galyun d lḥakem ɣef tmurt n Lakaya, at Isṛail msefhamen, ddukklen ɣef Bulus, wwin-t ɣer wexxam n ccṛeɛ, nnan :

18:13 Argaz-agi issamaṛ lɣaci ad ɛebden Ṛebbi s webrid ixulfen ccariɛa.

18:14 Mi gekker Bulus ad ihdeṛ, inṭeq Galyun yenna i wat Isṛail : Lemmer d lbaṭel neɣ d ayen n diri i gexdem, zemreɣ a wen ḥesseɣ am akken yella di lqanun ay at Isṛail ;

18:15 meɛna imi d amjadel ɣef yimeslayen, ɣef yismawen akk-d ccariɛa n Musa, ayagi d ccɣel-nwen ! Ur bɣiɣ ara ad ḥekmeɣ ɣef wannect-a.

18:16 Imiren issufeɣ-iten-id syenna.

18:17 Dɣa lɣaci mmɣen ɣef Sustin, ameqqran n lǧameɛ n wat Isṛail, ewten-t zdat wexxam n ccṛeɛ , Galyun ur s-tewqiɛ ara lmeɛna.

18:18 Bulus yerna aṭas n wussan di temdint n Kurintus ; syenna yeǧǧa atmaten di lehna, yerkeb lbabuṛ ɣer tmurt n Surya ; Brisila d Akilas ddan yid-es. Uqbel ad iṛuḥ si lmeṛṣa n temdint n Senkriya, iseṭṭel aqeṛṛuy-is axaṭer yefka lemɛahda i Sidi Ṛebbi.

18:19 Mi wwḍen ɣer temdint n Ifasus, mfaṛaqen : Bulus yeǧǧa iṛfiqen-is Akilas d Brisila, ma d nețța ikcem ɣer lǧameɛ n wat Isṛail, ițmeslay yid-sen.

18:20 Ḥellelen-t ad yernu kra wussan yid-sen, lameɛna ur yeqbil ara.

18:21 Yenna-yasen : A d-uɣaleɣ ɣuṛ-wen ma yebɣa Ṛebbi.

18:22 Yerkeb di lbabuṛ si temdint n Ifasus ɣer temdint n Qiṣarya ; syenna iṛuḥ a d-iẓer tajmaɛt n watmaten yellan di temdint n Lquds ; imiren iṣubb ɣer temdint n Antyuc.

18:23 Mi gesɛedda kra n wussan di temdint n Antyuc, yuɣal yeṭṭef abrid ɣer tmura n Galasya d Frijya, di mkul amkan yesseǧhad inelmaden di Liman.

18:24 Di lweqt-nni, yiwen wergaz n wat Isṛail yețțusemman Abulus, ilulen di temdint n Skandriya yellan di tmurt n Maṣeṛ, yusa-d ɣer temdint n Ifasus ; yezweṛ deg wawal yerna yessen mliḥ tira iqedsen.

18:25 Argaz-agi yelmed abrid n Sidi Ṛebbi, iǧhed di liman, ițbecciṛ, yesselmad s teḥqiq ayen i d-qqaṛent tira iqedsen ɣef Ɛisa Lmasiḥ , ɣas akken d aɣḍas n Yeḥya kan i gessen.

18:26 Yebda ițmeslay ɛinani di lǧameɛ n wat Isṛail. Akilas d Brisila mi s-slan wwin-t yid-sen, sfehmen-as tideț n webrid n Sidi Ṛebbi akken tella.

18:27 Imi i gebɣa Abulus ad iɛeddi ɣer tmurt n Akaya, atmaten kksen-as akukru, uran tabṛaț i inelmaden n temdint n Kurintus iwakken ad stṛeḥben yis. Mi gewweḍ ɣer dinna, s ṛṛeḥma n Sidi Ṛebbi iɛawen aṭas widak yumnen ;

18:28 axaṭer yețṛuẓu ɛinani awal n lecyux n wat Isṛail zdat lɣaci meṛṛa ; yerna ițbeggin-ed s tira iqedsen belli d Ɛisa i d Lmasiḥ.

19:2 Yers-ed fell-awen Ṛṛuḥ iqedsen asmi i tumnem ? Rran-as : Leɛmeṛ nesli s Ṛṛuḥ-agi iqedsen.

19:3 Yenna-yasen : Anwa aɣḍas i-s-tețwaɣeḍsem ihi ? Rran-as : S weɣḍas n Yeḥya.

19:4 Bulus yenna-yasen : Yeḥya yesseɣḍes wid akk iqeblen ad beddlen tikli, yeqqaṛ i wegdud ad amnen s win ara d-yasen deffir-es, yeɛni s Sidna Ɛisa.

19:5 Mi slan i yimeslayen-agi, țwaɣeḍsen s yisem n Sidna Ɛisa.

19:6 Akken yessers Bulus ifassen-is fell-asen, ikcem-iten Ṛṛuḥ iqedsen, bdan heddṛen s tutlayin ur nețwassen ara, yerna țcaren-d ayen i sen-d-yusan s ɣuṛ Sidi Ṛebbi.

19:7 Imdanen-agi ad ilin di tnac yid-sen.

19:8 Dɣa Bulus ikcem ɣer lǧameɛ n wat Isṛail. Azal n tlata wagguren ițmeslay yid-sen, ițqenniɛ-iten iwakken ad amnen s wayen yeɛnan tagelda n Sidi Ṛebbi.

19:9 Lameɛna kra deg-sen yeqquṛ wul-nsen ugin ad amnen, rnan kkaten deg webrid n Sidna Ɛisa zdat lɣaci. Mi gwala annect-agi, Bulus iṭṭaxeṛ-asen, yewwi yid-es inelmaden, isselmad-iten kull ass deg wexxam n yiwen wergaz isem-is Tiranus.

19:10 Yeqqim dinna sin iseggasen armi imezdaɣ meṛṛa n tmurt n Asya, ama d at Isṛail ama d iyunaniyen, slan i wawal n Sidi Ṛebbi.

19:11 Sidi Ṛebbi ixeddem lbeṛhanat imeqqranen s ifassen n Bulus ;

19:12 lɣaci wwḍen armi srusun ɣef yimuḍan ula ț-țimucwaṛin neɣ iceṭṭiḍen yennulen Bulus, dɣa imuḍan ḥellun, wid akk yețțumelken țeffɣen ten leǧnun.

19:13 Kra iseḥḥaren n wat Isṛail, ileḥḥun seg wemkan ɣer wayeḍ, ɛerden ad ssuffɣen leǧnun seg wid yețwamelken, qqaṛen-asen : S yisem n Sidna Ɛisa, win akken i gețbecciṛ Bulus, effɣet seg-es !

19:14 Widak ixeddmen annect-agi llan di sebɛa yid-sen, d arraw n Ciba yellan d yiwen si lmuqedmin imeqqranen n at Isṛail.

19:15 Lǧen-nni yerra-yasen-d : Ɛisa ssneɣ-t, ssneɣ daɣen anwa i d Bulus, ma d kunwi d acu-kkun ?

19:16 Argaz-nni i gezdeɣ lǧen, izḍem fell-asen yekkat-iten armi rewlen seg wexxam d iɛeryanen, yerna jerḥen.

19:17 At Isṛail akk-d iyunaniyen meṛṛa izedɣen tamdint n Ifasus slan s wayagi, ikcem-iten meṛṛa lxuf, ḥemmden isem n Sidna Ɛisa.

19:18 Aṭas seg wid i gumnen, usan-d ad qiṛṛen zdat wiyaḍ, ḥkan-d ayen xedmen.

19:19 Aṭas seg wid yellan xeddmen ssḥur, wwin-d tiktabin-nsen s wayes xeddmen ssḥur, sseṛɣen-tent zdat medden. Ufan belli azal n xemsin alef n twiztin i d ssuma-nsent.

19:20 S wakka i gewweḍ wawal n Sidi Ṛebbi ɣer mkul amkan, ițnerni s tezmert n Sidi Ṛebbi.

19:21 Mi gɛedda wayagi akk, Bulus yeqsed ad iṛuḥ ɣer temdint n Lquds, ad iɛeddi si tmura n Masidunya akk-d Akaya ; ixemmem deg iman-is yenna : « M'ara awḍeɣ ɣer dinna, ilaq-iyi daɣen ad ṛuḥeɣ ɣer temdint n Ṛuma.»

19:22 Iceggeɛ ɣer tmurt n Masidunya sin seg imɛawnen-is, Timuti d Irastus ; ma d nețța yerna kra n wussan di tmurt n Asya.

19:23 Di lweqt-nni, ɣef ddemma n webrid n Sidi Ṛebbi ikker ccwal d ameqqran di temdint n Ifasus.

19:24 Yella yiwen uḥeddad isem-is Dimitriyus, iṣenneɛ s lfeṭṭa leǧwameɛ imecṭaḥ i d-yețmettilen lǧameɛ n Artimis, yesserbeḥ aṭas ixeddamen-is.

19:25 Isnejmaɛ-iten nutni d wid i gxeddmen lfeṭṭa am nețța, yenna yasen : Ay atmaten ! Teẓram belli rrbeḥ-nneɣ itekk-ed si lxedma-agi ;

19:26 teẓram yerna teslam daɣen ɣef Bulus-agi issamnen aṭas n lɣaci di temdint n Ifasus d waṭas n tmura n Asya. Yessamen lɣaci, yewwi-ten d webrid-is mi sen-iqqaṛ : « Iṛebbiten agi ițwaxedmen s ifassen n wemdan mačči d iṛebbiten n tideț » ;

19:27 ur nuggad ara kan aț-țeɣli ṣṣenɛa-nneɣ, lameɛna nuggad ad sseɣlin lqima i lǧameɛ n Artimis tameqqrant, taṛebbiț-nneɣ, qṛib a s-kksen lɛaḍima s wayes tețwaɛbed di tmurt n Asya d ddunit meṛṛa.

19:28 Mi slan i lehduṛ-agi, ikcem-iten wurrif, bdan țɛeggiḍen : Ț-țameqqrant Artimis, taṛebbiț n at Ifasus !

19:29 Tețțucewwel akk temdint, lɣaci uzzlen ɣer wexxam n umezgun, seddan yid-sen Gayus akk-d Aristark yellan d imasiduniyen, d iṛfiqen n Bulus, ițṣafaṛen yid-es.

19:30 Bulus yebɣa ad ikcem di tlemmast n lɣaci-nni, lameɛna ugin-as inelmaden.

19:31 Ula d kra n lḥukkam imeqqranen n tmurt n Asya yellan d imdukkal-is, ceggɛen ɣuṛ-es iwakken a t-ḥellelen ur ițṛuḥ ara ɣer wexxam n umezgun.

19:32 Anejmaɛ-nni yexṛeb, kra țɛeggiḍen akka, wiyaḍ akken nniḍen yerna aṭas deg-sen ur ẓrin ara acu i d sebba n unejmaɛ-nni.

19:33 At Isṛail sbedden-d Alixandru ad immeslay i lɣaci. Iwehha-yasen ufus-is ad ssusmen, iwakken a d-ihdeṛ,

19:34 lameɛna mi t-ɛeqlen n wat Isṛail i gella, bdan țɛeggiḍen meṛṛa ɣef tikkelt azal n snat sswayeɛ : Ț-țameqqrant Artimis n at Ifasus !

19:35 Lameɛna lamin n temdint ihedden lɣaci, yenna-yasen : Ay at Ifasus, anwa ur neẓri ara belli tamdint n Ifasus ț-țaɛessast n lǧameɛ Artimis tameqqrant akk-d lmeṣnuɛ-ines i d-iɣlin seg igenni ?

19:36 Imi ayagi yiwen ur yezmir a t-yenkeṛ, ilaq ihi aț-țhennim iman-nwen, ur xeddmet acemma s tuffɣa n leɛqel !

19:37 Axaṭer irgazen-agi i d-tewwim ur ukiren lǧameɛ, ur rgimen Artimis taṛebbiț-nneɣ.

19:38 Ihi ma yella Dimitriyus d imdukkal-is sɛan sebba n uccetki ɣef yiwen, llan wussan n ccṛeɛ am akken llan lḥukkam ara ten-icaṛɛen , ad ṛuḥen ad ccetkin fell-asen.

19:39 Ma yella wayen nniḍen i tessuturem deg-sen, ad ḥekmen fell-asen deg unejmaɛ zdat lḥukkam.

19:40 Atan nuggad ad ccetkin fell-aɣ ɣef wayen yedṛan ass-agi imi ulac sebba ara nefk ɣef wunejmaɛ-agi. Mi gfukk ameslay, iserreḥ i lɣaci, yefra unejmaɛ.

20:1 Mi tethedden lḥala ifuk ccwal, imiren Bulus isnejmaɛ-ed inelmaden, yenha-ten, dɣa yeǧǧa-ten di lehna, yeṭṭef abrid ɣer tmurt n Masidunya.

20:2 Mi gezger tamurt n Masidunya, inehhu inelmaden, isseǧhad-iten s waṭas yimeslayen ; syenna ikemmel abrid ɣer tmurt n iyunaniyen anda yeqqim tlata wagguren.

20:3 Mi gțeddu ad yerkeb di lbabuṛ ɣer tmurt n Surya, yesla belli at Isṛail mcawaṛen iwakken a t-xedɛen ; dɣa ibeddel ṛṛay, iṭṭef abrid ad yuɣal ɣer tmurt n Masidunya.

20:4 Ddan yid-es : Subatir, mmi-s n Birus si temdint n Biri ; Arisṭark akk-d Skundus n temdint n Tiṣalunik, Gayus n temdint n Derba ; yedda daɣen Timuti, Tucik akk-d Trufim n tmurt n Asya.

20:5 Wigi zwaren ɣer zdat, eṛǧan-aɣ di temdint n Truwas,

20:6 ma d nukni mi gɛedda Lɛid n weɣṛum mbla iɣes ( tamtunt ), nerkeb lbabuṛ si temdint n Filibus ; xemsa wussan mbeɛd, nelḥeq-iten ɣer temdint n Truwas, anda nesɛedda sebɛa wussan.

20:7 Ass amezwaru n dduṛt nennejmaɛ a nebḍu aɣṛum iwakken ad nemekti Lmasiḥ, Bulus ițmeslay nețța d inelmaden. Imi ilaq ad iṛuḥ azekka-nni, ikemmel yid-sen awal armi i gneṣṣef yiḍ ;

20:8 taɣuṛfeț anda i nennejmaɛ, ceɛlent deg-s aṭas n teftilin.

20:9 Yiwen ilemẓi, isem-is Utikus, yella yeqqim ɣef yiri n ṭṭaq. Imi i gɛeṭṭel Bulus deg umeslay, Utikus yewwi-t nuddam ; yeɣli-d si leɛli wis tlata armi d lqaɛa, mi t-id-refden, ufan-t yemmut.

20:10 Bulus iṣubb-ed, yekna ɣuṛ-es, iddem-it ger iɣallen-is yenna : Ur țțaggadet ara, mazal deg-s taṛwiḥt, idder.

20:11 Syenna yuɣal yuli ar tɣuṛfeț, yebḍa aɣṛum ; mi ččan, ikemmel yid-sen ameslay armi d lefjer, dɣa iṛuḥ.

20:12 Ilemẓi-nni, wwin-t yedder ; dɣa yers-ed lfeṛḥ deg ulawen-nsen.

20:13 Nukni nerkeb lbabuṛ, nezwar Bulus ɣer temdint n Assus anda ara nemyagar yid-es axaṭer yebɣa ad iṛuḥ ɣef wuḍaṛ.

20:14 Mi i ɣ-d-yelḥeq di temdint n Assus, yerkeb yid-nneɣ di lbabuṛ ɣer temdint n Mitilan ;

20:15 syenna nkemmel abrid di lebḥeṛ, newweḍ azekka-nni zdat temdint n Ciyu, sellazekka-nni newweḍ ɣer temdint n Samus ; ass wis ṛebɛa newweḍ ɣer temdint n Mili.

20:16 Bulus yebɣa ad iɛeddi rrif n temdint n Ifasus mbla ma yeḥbes, iwakken ur isṛuḥuy ara lweqt di tmurt n Asya. Ițɣawal iwakken ma yella wamek, ad yaweḍ ɣer temdint n Lquds ass n lɛid n wass wis xemsin.

20:17 Si temdint n Mili, Bulus iceggeɛ ɣer imeqqranen n tejmaɛt n temdint n Ifasus iwakken a d-asen.

20:18 Mi d-wwḍen ɣuṛ-es, yenna-yasen : Tesnem si zik tikli-inu yid-wen seg wass amezwaru mi d-kecmeɣ tamurt n Asya.

20:19 Qedceɣ ɣef Sidi Ṛebbi s wannuz d imeṭṭawen ger iɣeblan d lemḥayen i yi-sseṛwan wat Isṛail.

20:20 Țbecciṛeɣ-awen rniɣ slemdeɣ kkun ama zdat lɣaci ama deg ixxamen-nwen, mbla ma ffreɣ acemma seg wayen i kkun-inefɛen.

20:21 Țbecciṛeɣ i wat Isṛail d iyunaniyen a d-uɣalen ɣer webrid n Sidi Ṛebbi yerna ad amnen s Sidna Ɛisa.

20:22 Tura, Ṛṛuḥ iqedsen iḥṛes-iyi ad ṭṭfeɣ abrid ɣer temdint n Lquds, ur ẓriɣ ara d acu ara yedṛun yid-i.

20:23 Si temdint ɣer tayeḍ, Ṛṛuḥ iqedsen ițxebbiṛ-iyi-d belli țṛaǧun-iyi leḥbus d wussan n ddiq.

20:24 Lameɛna tudert-iw ur s-fkiɣ ara lqima, ur ț-ḥsibeɣ ara eɛzizet fell-i ; a wi yufan kan ad kemmleɣ tikli-inu akk-d Ṛebbi, ad xedmeɣ s lfeṛḥ ayen i yi-d-ifka Sidna Ɛisa, iwakken ad beccṛeɣ lexbaṛ n lxiṛ n ṛṛeḥma n Sidi Ṛebbi.

20:25 Tura ẓriɣ belli ur tețțuɣalem ara aț-țeẓrem udem-iw, kunwi iwumi beccṛeɣ tageldit n Sidi Ṛebbi.

20:26 Daymi i wen-d-qqaṛeɣ ass-agi, ma yella yiwen yeffeɣ i webrid n Sidi Ṛebbi, ddnub i yiri-s,

20:27 imi i wen-beccṛeɣ ayen akk yellan di leqsed n Sidi Ṛebbi, ur ffireɣ ula d acemma fell-awen.

20:28 Ɛasset ɣef yiman-nwen, ɛasset ɣef tqeḍɛit ɣef i kkun-iwekkel Ṛṛuḥ iqedsen. Ilit d iɛessasen ɣef tejmaɛt n Ṛebbi i d-iḥerr s idammen n Mmi-s.

20:29 Sriɣ belli m'ara ṛuḥeɣ, a d-kecmen gar-awen wuccanen iweɛṛen, wid ur nețsamaḥ ara i tqeḍɛit ;

20:30 a d-kkren daɣen gar-awen yergazen ara islemden leḥwayeǧ iɛewjen iwakken ad jebden inelmaden ɣuṛ-sen.

20:31 Ɣuṛ-wat ihi, ɛasset iman-nwen ! Mmektit-ed belli azal n tlata iseggasen, ur ḥbiseɣ am yiḍ am ass, nehhuɣ mkul yiwen seg-wen s imeṭṭawen.

20:32 Tura a kkun-ǧǧeɣ ger ifassen n Sidi Ṛebbi d ṛṛeḥma n wawal-is; D nețța i gzemren a kkun-isseǧhed, a wen-d-yefk lweṛt i kunwi akk yellan d ayla-s.

20:33 Ur ḍmiɛeɣ, ama d ddheb ama d idrimen neɣ leḥwayeǧ n yiwen deg-wen.

20:34 Teẓram s yiman-nwen belli s ifassen-iw i xeddmeɣ iwakken ad ɛeyyceɣ iman-iw akk-d wid yellan yid-i.

20:35 Di mkul lḥaǧa, ssekneɣ-awen amek ara txedmem iwakken aț-țɛiwnem wid ur nezmir ara. Mmektit-ed imeslayen n Sidna Ɛisa mi d-yenna : « Win ițseddiqen ițțubarek akteṛ n win iteṭṭfen ssadaqa ».

20:36 Mbeɛd imeslayen-agi, yuɣal ɣef tgecrar, yedɛa ɣer Sidi Ṛebbi yid-sen.

20:37 Sṭeṛḍqen s imeṭṭawen, ṭṭfen-t ger iɣallen-nsen, selmen akk fell-as ;

20:38 ḥeznen aṭas imi i sen-d-yenna : « Ur tețțuɣalem ara aț-țeẓrem udem- iw. » Syenna ddan yid-es armi d lbabuṛ.

21:1 Mi nemfaṛaq yid-sen, nerkeb lbabuṛ nṛuḥ qbala ɣer temdint n Kus, azekka-nni ɣer temdint n Ṛudus ; syenna nkemmel abrid ɣer temdint n Batara.

21:2 Mi nufa lbabuṛ ara izegren ɣer tmurt n Finisya, nerkeb deg-s nṛuḥ.

21:3 Mi nqubel tigzirt n Qubṛus, neǧǧa-ț ɣer tama tazelmaṭ, nkemmel abrid nneɣ ɣer tmurt n Surya, newweḍ ɣer temdint n Sur imi dinna ara yessers lbabuṛ sselɛa-s.

21:4 Mi nufa dinna inelmaden, nesɛedda sebɛa wussan yid-sen ; s Ṛṛuḥ iqedsen, inelmaden-nni nnan-as i Bulus : Ur țțali ara ɣer temdint n Lquds.

21:5 Mi ɛeddan wussan-nni, nekker a nṛuḥ, ddan yid-nneɣ armi neffeɣ i temdint, nutni ț-țilawin-nsen d warraw-nsen. Mi newweḍ ɣer rrif n lebḥeṛ nuɣal ɣef tgecrar, nedɛa ɣer Sidi Ṛebbi.

21:6 Mi nemsalam yid-sen, nerkeb di lbabuṛ nṛuḥ ; nutni uɣalen ɣer yexxamen-nsen,

21:7 nukkni nkemmel abrid s lbabuṛ si temdint n Sur armi ț-țamdint n Bṭulimays, neẓra dinna atmaten, neqqim yid-sen yiwen wass.

21:8 Azekka-nni nṛuḥ, newweḍ ɣer Qiṣarya ; nekcem ɣer wexxam n Filbas, win ițbecciṛen lexbaṛ n lxiṛ, nețța yellan d yiwen si sebɛa-nni ițwaxtaṛen di temdint n Lquds ; neqqim ɣuṛ-es.

21:9 Filbas-nni, ɣuṛ-es ṛebɛa yessi-s urɛad zwiǧent, țxebbiṛent-ed s wayen i tent-id-ițțasen s ɣuṛ Sidi Ṛebbi.

21:10 Ɛeddan kra n wussan, mi d yewweḍ yiwen seg watmaten isem-is Agabus, ula d nețța ițxebbiṛ-ed s ɣuṛ Sidi Ṛebbi, iṣubb-ed si tmurt n Yahuda, yusa-d ɣuṛ-nneɣ.

21:11 Iddem tabagust n Bulus, icudd idaṛṛen-is d ifassen-is, yenna : Atan wayen i d-iqqaṛ Ṛṛuḥ iqedsen : akka ara cidden wat Isṛail di temdint n Lquds bab n tbagust-agi, a t-sellmen ger ifassen n ikafriwen.

21:12 Mi nesla annect-agi, nukkni d watmaten n Sizari nḥellel Bulus iwakken ur ițțali ara ɣer temdint n Lquds.

21:13 Dɣa Bulus yerra-yasen : Acu i kkun-yuɣen mi tețrum akka armi tgezmem tasa-w ? Nekk qebleɣ mačči kan ad țwacuddeɣ, lameɛna qebleɣ ula d lmut di temdint n Lquds ɣef ddemma n yisem n Sidna Ɛisa.

21:14 Imi yugi ad yaɣ awal-nneɣ, ur nketteṛ ara fell-as lehduṛ nenna : A neǧǧ lebɣi n Sidi Ṛebbi ad yedṛu !

21:15 Akken ɛeddan wussan-nni, nhegga iman-nneɣ a nali ɣer temdint n Lquds.

21:16 Kra inelmaden n Qiṣarya ddan yid-nneɣ, wwin-aɣ ɣer yiwen wergaz wuɣuṛ ara nili, isem-is Mansun, n tegzirt n Qubṛus, d yiwen seg inelmaden imezwura.

21:17 Mi newweḍ ɣer temdint n Lquds, sṭreḥben yis-nneɣ watmaten s lfeṛḥ.

21:18 Azekka-nni, yedda yid-nneɣ Bulus ɣer wexxam n Yeɛqub anda nnejmaɛen imeqqranen n tejmaɛt.

21:19 Mi gsellem fell-asen, Bulus yeḥka-yasen ayen akk ixdem yis Sidi Ṛebbi ger leǧnas ur nelli ara n wat Isṛail.

21:20 Mi slan ayagi, ḥemmden Sidi Ṛebbi nnan-as : Twalaḍ a gma-tneɣ acḥal n luluf n wat Isṛail i gumnen s Sidna Ɛisa yerna ṭṭfen akk di ccariɛa n Musa.

21:21 Atan slan wat Isṛail n temdint n Lquds belli tesselmadeḍ i wat Isṛail yellan ger leǧnas ad ǧǧen ccariɛa n Musa ; slan daɣen teqqaṛeḍ asen : « ur seḍhaṛet ara i warrac nwen, ur ttabaɛet ara leɛwayed i ɣ-d-ǧǧan lejdud ».

21:22 Acu ara nexdem tura ? Mbla ccekk ad slen belli tusiḍ-ed ;

21:23 ihi exdem ayen ara k-d-nini : llan gar-aneɣ ṛebɛa yergazen i gefkan lemɛahda i Sidi Ṛebbi,

21:24 awi-ten yid-ek, ssizdeg iman-ik yid-sen, seṛṛef fell-asen iwakken ad seṭṭlen iqeṛṛay-nsen ; s wakkagi ad ẓren belli ayen akk slan fell-ak d lekdeb yerna ad walin mazal teṭṭfeḍ di ccariɛa n Musa.

21:25 Ma d at leǧnas i gumnen s Sidna Ɛisa, nura-yasen : Ad ṭṭixṛen i wučči n weksum ițțunefken d asfel i lmeṣnuɛat d ssadat, i tissit n idammen, i lmal yemmuṛḍsen akk-d yir tikli.

21:26 Azekka-nni, Bulus yewwi yid-es ṛebɛa yergazen-nni, issazdeg iman-is yid-sen, ikcem yid-sen ɣer wefrag n lǧameɛ iqedsen iwakken ad ixebbeṛ lmuqeddem ɣef lweqt i deg ara fakken wussan n uzizdeg-nsen, yeɛni ass i deg ara yefk mkul yiwen seg-sen lweɛda-ines.

21:27 Mi qṛib ad fakken sebɛa wussan-nni n usizdeg, at Isṛail n tmurt n Asya walan Bulus di lǧameɛ iqedsen. Sekkren ccwal ger lɣaci dɣa ṭṭfen-t.

21:28 Bdan țɛeggiḍen : Ay at Isṛail, ɛiwnet-aɣ ! Atah wergaz-nni ițbecciṛen di mkul amkan i yemdanen meṛṛa. Iheddeṛ ayen n diri ɣef wegdud-nneɣ, ɣef wemkan-agi iqedsen, isselmad ayen ur teqbil ara ccariɛa-nneɣ, yewwi-d iyunaniyen ɣer lǧameɛ, issenǧes amkan-agi iqedsen !

21:29 Nnan-d ayagi axaṭer walan yid-es Trufim di temdint, ɣilen issekcem-it ɣer daxel n lǧameɛ iqedsen.

21:30 Ikker ccwal di temdint meṛṛa ; uzzlen-d yemdanen si mkul tama. Sṭfen Bulus, zzuɣṛen-t beṛṛa n wefrag n lǧameɛ iqedsen, sekkṛen-d tiwwura.

21:31 Mi țeddun lɣaci a t-nɣen, yewweḍ lexbaṛ ɣer lqebṭan n lɛeskeṛ n Ṛuman tamdint n Lquds meṛṛa tenhewwal.

21:32 Imiren kan yessawel i ifesyanen d lɛeskeṛ-nsen, uzzlen ɣer lɣaci. Mi walan lqebṭan akk-d lɛeskeṛ-is, ttaxṛen-as i Bulus.

21:33 Dɣa lqebṭan iqeṛṛeb ɣer Bulus, yefka lameṛ a t-ṭṭfen, a t-cudden s snat snasel ; dɣa isteqsa anwa-t, d acu i gexdem.

21:34 Meɛna lɣaci-nni, kra țɛeggiḍen akka, wiyaḍ akken nniḍen. Imi ur ifhim ara wayen i s-d-nnan si leɛyaḍ-nni, yumeṛ a t-awin ɣer lbeṛj.

21:35 Mi gewweḍ Bulus ɣer tseddaṛin, refden-t lɛeskeṛ iwakken a t-sellken si ger ifassen n lɣaci.

21:36 Axaṭer lɣaci-nni ttabaɛen-t țɛeggiḍen : « Enɣ-it ! Enɣ-it !... »

21:37 Mi qṛib a t-skecmen ɣer zdaxel n lbeṛj, Bulus yenna-yas i lqebṭan : Semmeḥ-iyi, zemreɣ a k-d-iniɣ yiwen n wawal ? Yerra-yas : Tessneḍ aț-țhedṛeḍ tayunanit ?

21:38 Mačči d kečč i d amaṣri-nni i gsekkren ccwal ussan-agi iɛeddan, i gessufɣen yid-es ṛebɛa alaf n iqettalen ɣer lxali ?

21:39 Bulus yenna-yas : Xaṭi ! Nekk n wat Isṛail, luleɣ di temdint n Sars yellan di tmurt n Silisya ; d amezdaɣ n temdint mechuṛen. Di leɛnaya-k serreḥ-iyi ad hedṛeɣ i lɣaci-yagi.

21:40 Lqebṭan iɛemmed-as, dɣa Bulus ibedd ɣef tseddaṛin, iwehha i lɣaci s ufus-is, dɣa ssusmen. Yenṭeq s tɛibṛanit, yenna-yasen :

22:1 Ay atmaten akken ma tellam dagi meẓẓi meqqeṛ, semḥesset ed tura a wen-d-iniɣ ayen s wayes ara dafɛeɣ ɣef yiman-iw !

22:2 Mi slan iheddeṛ-asen-d s tɛibṛanit, ssusmen meṛṛa iwakken a s-slen. Dɣa yenna :

22:3 Nekk n wat Isṛail, ț-țideț luleɣ di temdint n Sars n tmurt n Silisya, lameɛna țțuṛebbaɣ di temdint-agi n Lquds. Lliɣ d anelmad n ccix Gamalyel s yiman-is ; yesselmed-iyi akken ilaq amek ara tebɛeɣ ccariɛa i d-ǧǧan lejdud-nneɣ. Ttabaɛeɣ seg ul-iw abrid n Sidi Ṛebbi, am kunwi ass-agi.

22:4 Qehṛeɣ armi d lmut wid ittabaɛen abrid-agi ajdid n Sidna Ɛisa, am yergazen am tlawin, țțawiɣ-ten-id ɣer leḥbus țwarzen.

22:5 Lmuqeddem ameqqran ț-țejmaɛt n lɛuqal atnan d inigan ma bɣan a d-cehden ; d nutni s yiman-nsen i yi-d-yefkan tibṛatin i watmaten n Dimecq, syenna a d-awiɣ țwarzen ɣer temdint n Lquds, wid akk yumnen s webrid-agi, iwakken a ten-ɛaqben.

22:6 Lliɣ tedduɣ deg webrid, qṛib ad awḍeɣ ɣer temdint n Dimecq. Aț-țili d țnaṣfa n wass, taswiɛt kan, yiwet n tafat tameqqrant tewwet-ed seg igenni, tețfeǧǧiǧ, tezzi-yi-d.

22:7 Ɣliɣ ɣer lqaɛa, sliɣ i yiwen n ṣṣut i yi-d-iqqaṛen : Caɛul, a Caɛul, acuɣeṛ i yi tețqehhiṛeḍ ?

22:8 Nniɣ-as : Anwa-k, a Sidi ? Yenna-yi-d : Nekk d Ɛisa Anaṣari, win akken i tețqehhiṛeḍ.

22:9 Wid yellan yid-i walan tafat-nni, lameɛna ur slin ara i ṣṣut i yi-d iheddṛen, dɣa nniɣ : Acu i tebɣiḍ a t-xedmeɣ a Sidi ?

22:10 Sidna Ɛisa yenna-yi-d : Ekker, ṛuḥ ɣer Dimecq, dinna ara k-d-inin ayen akk ilaqen a t-txedmeḍ.

22:11 Imi iyi-tesderɣel tafat-nni s ufeǧǧeǧ-ines, d imdukkal-iw i yi ṭṭfen afus armi wwḍeɣ ɣer Dimecq.

22:12 Yella dinna yiwen wergaz isem-is Ananyas, d argaz iḍuɛen Sidi Ṛebbi, yerna yettabaɛ ccariɛa ; at Isṛail meṛṛa izedɣen di temdint n Dimecq țcehhiden fell-as s lxiṛ.

22:13 Yusa-d ɣuṛ-i yenna-yi-d : A Caɛul a gma, Sidi Ṛebbi yerra-yak-d iẓri ! Di teswiɛt-nni, yuɣal-iyi-d yezṛi, dɣa walaɣ-t.

22:14 Ananyas yenna-yi-d : Sidi Ṛebbi n lejdud-nneɣ yextaṛ ik aț-țissineḍ lebɣi-ines, aț-țwaliḍ s wallen-ik win yellan d Aḥeqqi, aț-țesleḍ imeslayen seg yimi-is .

22:15 axaṭer aț-țuɣaleḍ d inigi-ines zdat yemdanen meṛṛa ɣef wayen akk i teẓriḍ d wayen iwumi tesliḍ.

22:16 Tura d acu i tețṛaǧuḍ ? Kker aț-tețwaɣeḍseḍ, aț-țizdigeḍ seg ddnubat-ik m'ara tedɛuḍ s yisem-is.

22:17 Asmi uɣaleɣ ɣer temdint n Lquds, yiwen n wass lliɣ țẓallaɣ deg wefrag n lǧameɛ iqedsen, iweḥḥa-yi-d Sidi Ṛebbi,

22:18 walaɣ Sidna Ɛisa yenna-yi-d : Ffeɣ s lemɣawla si temdint n Lquds, axaṭer ur qebblen ara ayen s wayes ara tcehdeḍ fell-i.

22:19 Nniɣ : A Sidi, ssnen-iyi ! Sran s yiman nsen belli ttabaɛeɣ si lǧameɛ ɣer wayeḍ wid yumnen yis-ek iwakken a ten-erreɣ ɣer lḥebs ad țewten.

22:20 Ula d Stifan inigi-inek, mi ssuzlen idammen-is lliɣ dinna, qebleɣ at nɣen yerna ɛusseɣ leḥwayeǧ n wid i t-yenɣan.

22:21 Dɣa Sidi Ṛebbi yenna-yi-d : Ṛuḥ a k-ceggɛeɣ ɣer lebɛid ɣer wat leǧnas ur nelli ara n wat Isṛail.

22:22 Smeḥsisen-as armi d awal-agi imiren ɛeggḍen nnan : Ekkset si ddunit argaz am wagi ! Ur t-țaǧǧat ara ad yidir !

22:23 Țɛeggiḍen, ṭeggiṛen leḥwayeǧ nsen, ssafagen aɣebbaṛ ɣer igenni.

22:24 Lqebṭan n Ṛuman yumeṛ ad skecmen Bulus ɣer lbeṛj, a t-ewten s ujelkkaḍ ( s ucelliṭ ) iwakken ad issinen sebba ɣef wacu țɛeggiḍen fell-as.

22:25 Mi t-cudden s tɣeggadin, Bulus yenna i lqebṭan yellan dinna : Eɛni iserreḥ-awen lqanun aț țewtem aṛumani s ujelkkaḍ uqbel ad iɛeddi di ccṛeɛ ?

22:26 Ɣef yimeslayen-agi, lqebṭan-nni iṛuḥ ɣer wemdebbar ameqqran yenna-yas : Ḥader aț-țewteḍ argaz-agi ! D aṛumani !

22:27 Amdebbar ameqqran mi d-yusa ɣer Bulus, yenna-yas : Ț-țideț kečč d aṛumani ? Ih, d aṛumani i lliɣ !

22:28 Amdebbar ameqqran yenna : Nekk aṭas n yedrimen i fkiɣ iwakken ad uɣaleɣ d aṛumani. Bulus yerra-yas : Nekk, seg wasmi i d-luleɣ i rseɣ di lekwaɣeḍ d aṛumani.

22:29 Imiren widak i d-ikkren a t wten, ṭṭaxṛen-as. Lqebṭan ameqqran s yiman-is yuggad mi geẓra Bulus d aṛumani yerna icudd-it.

22:30 Azekka-nni, imi i gebɣa ad yissin wemdebbar ameqqran ɣef wacu i d-ccetkan fell-as wat Isṛail, iserreḥ i Bulus, yumeṛ i lmuqedmin akk-d usqamu n ccṛeɛ ad nnejmaɛen. Dɣa yewwi-d Bulus isbedd-it-id zdat-nsen.

23:2 Lmuqeddem ameqqran Ananyas yumeṛ i wid yellan zdat-es, a t-wten ɣer yimi.

23:3 Dɣa Bulus yenna : A k-id-iwet Ṛebbi a bu sin wudmawen ! Teqqimeḍ aț-țḥekmeḍ fell-i s ccariɛa, lameɛna kečč s yiman-ik txulfeḍ-ț imi tumreḍ ad iyi wten.

23:4 Widak yellan zdat-es nnan-as : Tregmeḍ lmuqeddem ameqqran n Ṛebbi !

23:5 Bulus yenna-yasen : Ur ẓriɣ ara ay atmaten belli d nețța i d lmuqeddem ameqqran, axaṭer yura di ccariɛa : Ur țekkes ara leqdeṛ ɣef lḥakem n wegdud-ik.

23:6 Bulus, mi gwala deg unejmaɛ-nni azgen deg-sen d isaduqiyen, azgen nniḍen d ifariziyen, iɛeggeḍ gar-asen, yenna : Ay atmaten, nekk d afarizi, d mmi-s n ufarizi ; ɣef ddemma n usirem i nesɛa di ḥeggu n lmegtin i d-beddeɣ ɣer ccṛeɛ ass-agi !

23:7 Mi d-yenna ayagi, ikker umennuɣ ger ifariziyen d isaduqiyen ; dɣa yebḍa unejmaɛ-nni.

23:8 Axaṭer isaduqiyen qqaṛen ulac ḥeggu, ulac lmalayekkat, ulac leṛwaḥ ; ma d ifariziyen, nutni umnen s wannect agi.

23:9 Ikker umennuɣ gar-asen ; kra n lɛulama n ccariɛa n tejmaɛt n ifariziyen kkren iwakken ad ḥudden Bulus, nnan : Ur nufi ara ayen n diri deg wemdan-agi. Yezmer lḥal ihdeṛ-as-d Ṛebbi neɣ lmelk !

23:10 Imi ițțimɣuṛ umenɣi, lqebṭan ameqqran yugad ad feṛqen Bulus d iftaten. Yumeṛ i lɛeskeṛ ad ṛuḥen a t-id-kksen si tlemmast-nsen, a t awin ɣer lbeṛj.

23:11 Azekka-nni deg iḍ, Sidna Ɛisa iḍheṛ-as-d i Bulus yenna-yas : Ur țțaggad ara, axaṭer am akken tcehdeḍ fell-i di temdint n Lquds, ilaq daɣen aț-țcehdeḍ fell-i di temdint n Ṛuma.

23:12 Mi guli wass, at Isṛail xedmen tiḥileț, msefhamen, gullen ur ččin ur swin alamma nɣan Bulus.

23:13 Wid akk i s-ixedmen taxazabit agi, llan akteṛ n ṛebɛin yid-sen.

23:14 Ṛuḥen ad ẓren imeqqranen d lmuqedmin akk-d lɛuqal nnan asen : Neggul s limin ameqqran, ur nečči acemma alamma nenɣa Bulus.

23:15 Tura ihi, init-as i lqebṭan ameqqran a t-id-yawi ɣer unejmaɛ n ccṛeɛ zdat-wen, amzun tebɣam a s-tɛiwdem abeḥḥet ; ma d nukkni, a nheggi iman-nneɣ a t-nenɣ uqbel a d-yaweḍ ɣer dagi.

23:16 Lameɛna mmi-s n weltma-s n Bulus imi gesla s tḥileț-nni i s heggan, iṛuḥ ɣer lbeṛj anda yețwaḥbes Bulus, a s-yessiweḍ lexbaṛ.

24:4 lameɛna iwakken ur k-nețɛeṭṭil ara aṭas, di leɛnaya-k semḥess-ed i kra n yimeslayen-agi ara k-d-nini :

24:18 Atan wamek i yi-d-ufan deg wefrag n lǧameɛ iqedsen : ufan-iyi-d akken kan i ssezdegeɣ iman-iw akken i d-tenna ccariɛa; ur iyi-d zzin imdanen ur yelli ccwal

26:1 Aɣribas yenna i Bulus : -- Atan ɣuṛ-ek wawal, tzemreḍ aț-țdafɛeḍ ɣef yiman-ik. Bulus yerfed afus-is iwakken ad idafeɛ ɣef yiman-is, yenna :
