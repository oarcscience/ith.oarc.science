Kabyle Bible: Colossians

1:1 Nekk Bulus, amceggeɛ n Ɛisa Lmasiḥ s lebɣi n Sidi Ṛebbi akk-d gma-tneɣ Timuti,

1:2 i watmaten yellan di temdint n Kulus, wid iteddun s ṣṣfa deg webrid n Lmasiḥ. Ṛṛeḥma ț-țalwit a wen-d-țțunefkent s ɣuṛ Ṛebbi Baba-tneɣ !

1:3 Nețḥemmid Sidi Ṛebbi Baba-s n Ssid-nneɣ Ɛisa Lmasiḥ yerna ndeɛɛu fell-awen ;

1:4 axaṭer nesla s liman i tesɛam di Ɛisa Lmasiḥ d wamek i tḥemmlem agdud n Sidi Ṛebbi,

1:5 ɣef ddemma n usirem i wen-ițțuheggan deg igenwan, asirem nni i wen-d-ițțuxebbṛen s wawal n tideț n lexbaṛ n lxiṛ.

1:6 Lexbeṛ-agi n lxiṛ yewḍen armi d ɣuṛ-wen, yețnerni di ddunit meṛṛa ; akken daɣen i gețnerni deg-wen ula d kunwi seg wasmi i teslam yerna tessnem tideț ɣef ṛṛeḥma n Sidi Ṛebbi,

1:7 akken i wen-ț-isselmed Ibafras, aṛfiq-nneɣ eɛzizen di leqdic ; nețța iqeddcen ɣef Lmasiḥ s wul yeṣfan,

1:8 ixebbeṛ-aɣ-ed ɣef leḥmala s wayes i kkun-yeččuṛ Ṛṛuḥ iqedsen.

1:9 Daymi nezga ndeɛɛu fell-awen ɣer Sidi Ṛebbi seg wasmi nesla yis-wen, nessutur-as a wen-d-yefk amek ara tissinem akken ilaq lebɣi ines, s lɛeqliya d lefhama i d-ițțak Ṛṛuḥ iqedsen.

1:10 S wakka, aț-țleḥḥum s tikli yebɣa Sidi Ṛebbi iwakken a s tɛeǧbem di kullec, a d-tefkem lfakya yelhan, yerna aț-țnernim di tmusni n Sidi Ṛebbi,

1:11 Ndeɛɛu ɣer Ṛebbi akken a kkun-isseǧhed s tezmert-is tameqqrant iwakken aț-tṣebṛem i kullec alamma ț-țaggara.

1:12 Ḥemmdet, cekkṛet s lfeṛḥ Sidi Ṛebbi Baba-tneɣ i wen-d-yefkan amur di lweṛt i d-ihegga i warraw n tafat.

1:13 Ikkes-aɣ-d si tezmert n ṭṭlam, iwakken a ɣ-yessiweḍ ɣer tgelda n Mmi-s eɛzizen,

1:14 s wayes nețwasellek yerna nesɛa leɛfu n ddnubat-nneɣ.

1:15 Lmasiḥ, d ṣṣifa n Sidi Ṛebbi ur nezmir a nwali, d nețța i d amezwaru n txelqit meṛṛa.

1:16 Yis i gețwaxleq wayen nețwali d wayen ur nețwali ara ama deg igenwan ama di lqaɛa, tigeldiwin d igelliden, lḥekmat akk-d tezmar yellan deg igenwan ; kullec yețwaxleq yis, kullec yețwaxleq i nețța.

1:17 Yella-d uqbel a d-yețwaxleq kra yellan, d nețța i d lsas n txelqit,

1:18 d aqeṛṛuy n tejmaɛt n imasiḥiyen yellan d lǧețța-s. Yella si tazwara, d nețța i d amenzu i d-iḥyan si ger lmegtin iwakken ad yili d amezwaru di kullec.

1:19 Axaṭer Sidi Ṛebbi yufa lfeṛḥ-is s lekmal di Mmi-s,

1:20 s Lmasiḥ i gemṣalaḥ akk-d txelqit, yessers-ed lehna ama di lqaɛa ama deg igenwan s idammen n Mmi-s yuzzlen ɣef wumidag.

1:21 Kunwi yellan zik-nni d ibeṛṛaniyen, d iɛdawen n Sidi Ṛebbi s ixemmimen-nwen d yir lecɣal nwen

1:22 Tura issemṣaleḥ-ikkun yid-es s lmut n Mmi-s iwakken a d-tbanem zdat-es teṣfam, mbla lɛib.

1:23 Ma yella teṭṭfem di liman-nwen s tideț, ma yella tbeddem, tṛeṣṣam. Ɣuṛ-wat aț-țțwexxṛem ɣef wusirem i wen-d-yewwi lexbaṛ n lxiṛ iwumi teslam, lexbaṛ-agi n lxiṛ i gețțubeccṛen i yemdanen n ddunit meṛṛa, i ɣef uɣaleɣ nekk d aqeddac.

1:24 Tura feṛḥeɣ aṭas imi nɛețțabeɣ fell-awen, axaṭer țkemmileɣ ayen ixuṣṣen i leɛtab n Lmasiḥ ɣef ddemma n imasiḥiyen meṛṛa, yellan d lǧețța-s, i ɣef uɣaleɣ d aqeddac.

1:25 Iceggeɛ-iyi-d Sidi Ṛebbi iwakken a wen-d-beccṛeɣ awal-is s lekmal.

1:26 Lbaḍna yeffren si zik, si lǧil ɣer lǧil, ibeggen-iț-id tura i wid yellan d ayla-s.

1:27 Yebɣa a d-yesken ccan n lbaḍna agi ifazen i leǧnas nniḍen ur nelli ara n wat Isṛail, iwakken ad walin Lmasiḥ deg-wen, nețța yellan d lsas n lɛaḍima i ɣer nessaram.

1:28 D Lmasiḥ agi i nețbecciṛ i yemdanen, nețweṣṣi-ten, nesselmad iten s tmusni d leɛqel, iwakken yal amdan ad innekmal di Lmasiḥ zdat Sidi Ṛebbi.

1:29 Ɣef wayagi i qeddceɣ s tezmert n Lmasiḥ yellan deg-i.

2:1 Bɣiɣ aț-țeẓrem acḥal meqqeṛ imenɣi i deg lliɣ ɣef ddemma nwen, ɣef watmaten n temdint n Ludikus akk-d wid meṛṛa i geslan yis-i meɛna leɛmeṛ ur ẓrin udem-iw.

2:2 Qebleɣ imenɣi-agi iwakken ad theddnen wulawen-nsen, ad ddukklen s leḥmala, ad ččaṛen d lefhama iwakken ad issinen lbaḍna n Sidi Ṛebbi Ṛebbi yellan di Lmasiḥ ;

2:3 deg-s i ffren igerrujen n lefhama ț-țmusni.

2:4 Nniɣ-d akka iwakken ur kkun ițkellix yiwen s imeslayen yețɣuṛṛun.

2:5 Ɣas akken ur lliɣ ara yid-wen, leɛqel-iw atan ɣuṛ-wen ; feṛḥeɣ aṭas mi ẓriɣ acḥal i teṭṭfem di liman-nwen di Lmasiḥ yerna tsedduyem kullec akken ilaq.

2:6 Ihi imi tqeblem Ɛisa Lmasiḥ d Ssid-nwen, ddut di lebɣi-ines.

2:7 Sbeddet lsas n tudert-nwen fell-as, sǧehdet liman-nwen akken i kkun nesselmed, țḥemmidet daymen Sidi Ṛebbi.

2:8 Ḥadret a kkun-kellxen wid ițfelsifen, wid yessexdamen tiḥila ț-țiḥeṛci ; ayagi d lekdubat i gebnan ɣef leɛwayed n yemdanen akk ț-țizemmar n ddunit, mačči ɣef uselmed n Lmasiḥ.

2:9 Axaṭer ayen akk yellan di Sidi Ṛebbi yezdeɣ s lekmal di Lmasiḥ,

2:10 kunwi tesɛam kullec deg-s, nețța yellan sennig lḥekmat ț-țzemmar.

2:11 Sṭhaṛa-nwen tekka-d si tikli nwen akk-d Lmasiḥ, mačči am tin yețwaxedmen s ufus n wemdan, lameɛna s ɣuṛ Lmasiḥ i d-tekka iwakken a kkun-isellek si tezmert n lǧețța tamednubt.

2:12 S weɣḍas-nwen tețwameḍlem akk-d Lmasiḥ, lameɛna teḥyam-d daɣen yid-es si lmut imi tumnem s tezmert n Sidi Ṛebbi i t-id-isseḥyan si ger lmegtin.

2:13 Kunwi ur neḍhiṛ ara, tellam temmutem ɣef ddemma n ddnubat nwen ; tura Sidi Ṛebbi yerra-kkun-id ɣer tudert s Lmasiḥ. Sidi Ṛebbi yeɛfa-yaɣ ddnubat-nneɣ meṛṛa.

2:14 Yemḥa leɛqed-nni i ɣ-isseḍlamen, imi ur nezmir ara a nexdem ayen d-teqqaṛ ccariɛa ; yemḥa-t mi t-isemmeṛ ɣef wumidag.

2:15 Sidi Ṛebbi yekkes tazmert i lḥekmat ț-țzemmar n igenwan, ikcef itent-id zdat txelqit meṛṛa, iɣleb-itent s lmut n Mmi-s ɣef wumidag.

2:16 Ihi ur țțaǧat ara win ara kkun-iḥasben ɣef wayen ara teččem d wayen ara teswem neɣ imi ur tettabaɛem ara ayen yeɛnan leɛyudat, talalit n waggur d wussan n westeɛfu ;

2:17 ayagi akk yezwar-ed d lemtel kan n wayen i d-iteddun, axaṭer tideț di Lmasiḥ i tella.

2:18 Ɣuṛ-wat a wen-kellxen yemdanen ițțarran iman-nsen d wid yunzen, țɛebbiden lmalayekkat, a kkun-id-awin ɣer webrid-nsen iwakken ur tețțawḍem ara ɣer lmeqṣud-nwen ; imdanen am wigi ttabaɛen kan ixemmimen-nsen yerna xeddmen ccan i yiman-nsen.

2:19 Ur qbilen ara ad ṭṭfen di Lmasiḥ yellan d aqeṛṛuy, i gesdukklen lǧețța s lemfaṣel meṛṛa d iẓuran i d-icudden ɣuṛ-es ; yis i tețnerni lǧețța, i tețțimɣuṛ s wakken yella di lebɣi n Sidi Ṛebbi.

2:20 Ma yella temmutem akk-d Lmasiḥ, teṭṭaxṛem ɣef wayen akk yeɛnan tudert n ddunit, acimi ihi am akken di ddunit i tellam, tqebblem ad ḥețțmen fell-awen leɛwayed n ddunit m'ara wen-qqaṛen :

2:21 « Ur teddmet ara neɣ ur ɛeṛṛdet ara neɣ ur țnalet ara !»

2:22 Lumuṛat-agi meṛṛa i nxeddem d ayen ara yefnun, d leqwanen d uselmed n yemdanen ;

2:23 ț-țideț, lumuṛat-agi sselmaden leɛbada, annuz akk-d uɛețțeb n lǧețțat nneɣ, țbanen-d amakken d ayen yelhan, lameɛna d ayen ur nezmir ad iɛawen bunadem ad iṭṭef iman-is.

3:1 Kunwi ihi, imi i d-teḥyam akk-d Lmasiḥ, nadit ɣef leḥwayeǧ n igenwan anda yella Lmasiḥ i gḥekmen ɣer tama tayeffust n Sidi Ṛebbi.

3:2 Țḥebbiṛet ɣef wayen yellan deg igenwan mačči ɣef wayen yellan di ddunit-agi,

3:3 axaṭer ɣer ddunit tețțusemmam temmutem, meɛna ɣer Sidi Ṛebbi tudert-nwen teffer akk-d Lmasiḥ.

3:4 M'ara d-iban Lmasiḥ yellan ț-țudert-nwen, kunwi daɣen a d tbanem yid-es.

3:5 Ɣef ddemma n wayagi, xḍut i yir tikli n ddunit-agi s wayes tețțeddum zik : leḥṛam, ayen iɛefnen, lebɣi n tnefsit, ccehwat n diri akk-d ṭṭmeɛ, ayagi meṛṛa am win ițɛebbiden ayen ixulfen Sidi Ṛebbi.

3:6 D annect-agi i d-isseɣlayen urrif n Sidi Ṛebbi ɣef wid i t-ițɛaṣin.

3:7 Ula d kunwi akka i tellam zik-nni mi tețɛicim di ddnub.

3:8 Meɛna tura sbeɛdet iman-nwen ɣef wannect-agi meṛṛa : urrif, zzɛaf, ddɣel ; ḥadret a d-ffɣen seg imawen-nwen rregmat neɣ yir lehduṛ.

3:9 Ur skiddibet ara wway gar-awen imi tura teǧǧam yir tikli-nwen n zik.

3:10 Tuɣalem d ijdiden, tețnernim di tmusni n Sidi Ṛebbi iwakken aț țemcabim ɣuṛ-es, nețța i kkun-d-ixelqen.

3:11 Di tudert-agi tajdiṭ ulac lxilaf ger wat Isṛail d leǧnas nniḍen, ger win iḍehṛen d win ur neḍhiṛ, ger wid itqeddmen d wid ur netqeddem, neɣ ger wakli d uḥeṛṛi, imi d Lmasiḥ i gellan d kullec di mkul yiwen.

3:12 Sidi Ṛebbi iḥemmel-ikkun, yextaṛ-ikkun aț-țilim d agdud-is. Sɛut ihi ṛṛeḥma deg wulawen-nwen, leḥnana, lxiṛ, ṣṣbeṛ d wannuz.

3:13 Țemyawit wway gar-awen, ma yella win yesɛan accetki ɣef wayeḍ msamaḥet axiṛ wway gar-awen, akken i wen-isemmeḥ Lmasiḥ, msamaḥet ula d kunwi.

3:14 Nnig n wayagi meṛṛa sɛut leḥmala, ț-țin i gesdukkulen ulawen.

3:15 Sɛut lehna n Lmasiḥ deg ulawen nwen iwakken aț-țilim d yiwen, ḥemmdet Sidi Ṛebbi imi ɣer lehna-agi i wen-d-yessawel.

3:16 Awal n Lmasiḥ ad izdeɣ gar-awen s lbaṛakat-is meṛṛa iwakken wa ad isselmad wa, wa ad inehhu wayeḍ s ṣṣwab. Ccnut s wulawen-nwen, ḥemmdet Sidi Ṛebbi s isefra n Zabuṛ akk-d ccnawi i d-ițțak Ṛṛuḥ iqedsen.

3:17 Ama deg imeslayen ama di lecɣal ara txedmem, xedmet kullec s yisem n Ssid-nneɣ Ɛisa, s yisem-is țḥemmidet Sidi Ṛebbi baba-tneɣ.

3:18 A tilawin, ḍuɛemt irgazen-nkunt, ayagi d ayen ilaqen i tid i gețḍuɛun Sidi Ṛebbi.

3:19 Ma d kunwi ay irgazen, ilaq aț-țḥemmlem tilawin-nwen, ur seɛɛut ara ulawen iqesḥen ɣuṛ-sent.

3:20 Ay arrac țțaɣet awal i imawlan nwen di kullec, axaṭer akka ara tɛeǧbem i Sidi Ṛebbi.

3:21 Lameɛna kunwi ay imawlan ur sserfuyet ara dderya-nwen neɣ m'ulac ad feclen.

3:22 Ma d kunwi ay iqeddacen, țțaɣet awal i imɛellmen-nwen di ddunit-agi mačči kan zdat-nsen akken a sen tɛeǧbem meɛna ɣas ulac-iten xedmet ccɣel-nwen seg wul, am akken i Sidi Ṛebbi.

3:23 Kra n wayen ara txedmem, xedmet-eț s wul yeṣfan am akken i Sidi Ṛebbi i t-txeddmem mačči i yemdanen,

3:24 mmektit-ed lweṛt i kkun ițṛaǧun s ɣuṛ Sidi Ṛebbi akken i t-id-yewɛed ; axaṭer amɛellem n tideț i ɣef tqeddcem, d Lmasiḥ.

3:25 Win ixeddmen cceṛ a d-iɣellet ayen yexdem ; axaṭer ɣer Sidi Ṛebbi ulac lxilaf.

4:1 Ay imɛellmen ddut s lḥeqq, ur xeddmet ara lxilaf ger iqeddacen nwen, axaṭer teẓram belli ula d kunwi tesɛam amɛellem deg igenwan.

4:2 Ur ɣefflet ara, meɛna deɛɛut țḥemmidet daymen Sidi Ṛebbi.

4:3 Deɛɛut daɣen fell-aɣ ɣer Sidi Ṛebbi iwakken a ɣ-d-ildi tiwwura a nbecceṛ awal-is, a nxebbeṛ ɣef lbaḍna n Lmasiḥ i ɣef lliɣ di lḥebs,

4:4 dɛut iwakken ad sfehmeɣ akken ilaq lbaḍna-agi i yemdanen.

4:5 Ddut s ṣṣwab zdat wid ur numin ara s Ɛisa Lmasiḥ ; faṛset tagniț, țbecciṛet awal n Ɛisa Lmasiḥ.

4:6 Sɛut iles aẓidan, imeslayen yesɛan lfayda, iwakken aț-țețțarram awal i yal yiwen s ṣṣwab.

4:7 Tucik, gma-tneɣ eɛzizen i ɣef țekleɣ, aṛfiq-nneɣ di leqdic ɣef Ssid-nneɣ, a wen-n-yawi lexbaṛ-iw.

4:8 Ɣef wannect-agi i wen-t-in ceggɛeɣ iwakken a wen-n-yawi lexbaṛ-nneɣ yerna ad ihedden ulawen-nwen.

4:9 A d-yas nețța d gma-tneɣ eɛzizen Unizim i ɣef nețkel, yellan d yiwen seg-wen ; d nutni ara wen-n-yawin lexbaṛ ɣef wayen meṛṛa ideṛṛun dagi.

4:10 Ițsellim-ed fell-awen Arisṭark aṛfiq-iw yellan yid-i di lḥebs akk-d Maṛqus yețțilin i Barnabas, win i ɣef kkun-weṣṣaɣ aț-țesterḥbem yis ma yusa-d ɣuṛ-wen.

4:11 Ițsellim-ed daɣen fell-awen Ɛisa iwumi qqaṛen Yustus. Ger wat Isṛail anagar tlata watmaten-agi i gxeddmen yid-i i tgeldit n Sidi Ṛebbi, d nutni kan i yufiɣ ɣer tama-w.

4:12 Gma-tneɣ Ibafras n tmurt-nwen, yellan d aqeddac n Ɛisa Lmasiḥ ițsellim-ed fell-awen ; yezga ideɛɛu ɣer Ṛebbi fell-awen iwakken aț-țeṭṭfem, aț-țennernim di liman yerna aț-țilim twejdem aț-țxedmem lebɣi n Sidi Ṛebbi.

4:13 A d-cehdeɣ fell-as acḥal inɛețțab fell-awen, ɣef watmaten n tejmaɛt n Ludikus akk-d wid n Yirabulis.

4:14 Luqa aḥbib-nneɣ eɛzizen yellan d ṭṭbib akk-d Dimas, țsellimen-d fell-awen.

4:15 Sellmet ɣef watmaten yellan di Ludikus, ɣef Nimfa akk d wid ițnejmaɛen deg wexxam-is.

4:16 M'ara teɣṛem tabṛaț-agi, fket-eț i watmaten n tejmaɛt n Ludikus a ț-ɣṛen, kunwi daɣen ɣṛet tabṛaț ara kkun-id-yawḍen s ɣuṛ-sen.

4:17 Init i Arcibus ad ibedd i ccɣel nni i ɣef t-iwekkel Ssid-nneɣ, a t-ixdem akken ilaq.

4:18 Nekk Bulus, s ufus-iw i wen-n uriɣ. Sslam fell-awen ! Ur tețțut ara belli d ameḥbus i lliɣ. Ṛṛeḥma n Sidi Ṛebbi aț-țili yid-wen.
