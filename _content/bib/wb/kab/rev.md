<span id="an_9:11">9:11</span> Agellid-nsen, d lmelk-nni iḥekkmen ɣef

<span id="an_9:12">9:12</span> Tawaɣit tamezwarut tɛedda. Mazal snat n

<span id="an_9:13">9:13</span> Lmelk wis sețța iwwet lbuq. Sliɣ i yiwet

n wasif ameqqran, iwumi qqaṛen « Lufrat ».

<span id="an_9:15">9:15</span> Ṛebɛa lmalayekkat-nni i gețțuheggan i

<span id="an_9:16">9:16</span> Sliɣ i leḥsab n yemnayen, llan di mitin

iɛewdiwen-nni am akken d iqeṛṛay n izmawen ; seg imawen-nsen
tețțeffeɣ-ed tmes, dexxan akk-d ukubri.

<span id="an_9:18">9:18</span> ?elt n yemdanen mmuten s tlata n
twaɣyin-agi, s tmes, s dexxan akk-d ukubri-nni i d-ițeffɣen seg
imawen-nsen.

<span id="an_9:19">9:19</span> Axaṭer tazmert n iɛewdiwen-nni tella deg

<span id="an_9:20">9:20</span> Imdanen nniḍen ur nemmut ara s
twaɣyin-agi, ur ndimen ara seg yir tikli-nsen ; kemmlen ɛebbden leǧnun
țṛuḥun ɣer ssadaț anda ɛebbden lmeṣnuɛat yețwanejṛen s ufus n wemdan, s
ddheb, s lfeṭṭa akk-d nnḥas, s wezṛu akk-d wesɣaṛ. ?as akken
lmeṣnuɛat-nni ur țwalin, ur sellen, ur leḥḥun,

<span id="an_9:21">9:21</span> imdanen-agi ur ndimen ara ɣef tmegṛaḍ i
neqqen, ɣef ssḥur ɣef tukeṛdiwin akk țecmatin i xeddmen, lameɛna
țkemmilen di lecɣal-agi.

<span id="an_10:1">10:1</span> Walaɣ daɣen yiwen lmelk nniḍen iǧehden,
iṣubb-ed seg igenni, tețțel-it-id tagut ; tislit n wanẓar tezzi ɣef
wuqeṛṛuy-is, udem-is yețfeǧǧiǧ am yiṭij, iḍaṛṛen-is am iḥeǧǧuǧa n tmes.

<span id="an_10:2">10:2</span> Yeṭṭef deg ufus-is yiwet tektabt
tamecṭuḥt yeldin. Yessers aḍar-is ayeffus ɣef lebḥeṛ, ma d aḍar azelmaḍ
yessers-it ɣef lqaɛa,

<span id="an_10:3">10:3</span> iɛeggeḍ s taɣect ɛlayen am akken d izem i
gesreɛreɛen. Mi gesreɛreɛ lmelk-nni, nesla i useqseq n tuɣac n sebɛa
ṛṛɛudat.

<span id="an_10:4">10:4</span> Mi sliɣ i tuɣac n sebɛa ṛɛudat nni țedduɣ
ad aruɣ ayen i d-nnan, lameɛna sliɣ i yiwet taɣect i d-yekkan seg igenni
tenna-yi-d : Eǧǧ di lbaḍna ayen akka i d-nnan sebɛa ṛɛudat-agi, ur
tețțaru ara.

<span id="an_10:5">10:5</span> Lmelk-nni i walaɣ ibedd ɣef lebḥeṛ akk-d
lqaɛa, yerfed afus-is ayeffus ɣer igenni,

<span id="an_10:6">10:6</span>  yeggul s Win yeddren si lǧil ɣer lǧil ,
Win i d-ixelqen igenni, lqaɛa d lebḥeṛ d wayen yellan deg-sen, a d-yedṛu
kullec mbla leɛḍil.

<span id="an_10:7">10:7</span> M'ara d-yaweḍ wass i deg ara nsel i ṣṣut
n lbuq n lmelk wis sebɛa, imiren kra wayen yellan di lbaḍna n Sidi Ṛebbi
ad yețwakemmel akken i t-id yenna i lenbiya.

<span id="an_10:8">10:8</span> Taɣect-nni i d-yekkan seg igenni, terna
tenna-yi-d : Ṛuḥ, eddem-ed taktabt tamecṭuḥt yeldin deg ufus n lmelk-nni
ibedden ɣef lqaɛa d lebḥeṛ.

<span id="an_10:9">10:9</span> Ṛuḥeɣ ɣer lmelk-nni, ssutreɣ-as a
yi-d-yefk taktabt-nni tamecṭuḥt. Yenna-yi-d : Eddem-iț, sebleɛ-iț ; deg
imi-k aț-țili ț-țaẓidant am tammemt, meɛna m'ara taweḍ ɣer uɛebbuḍ-ik,
aț-țuɣal ț-țarẓagant.

<span id="an_10:10">10:10</span>  Eddmeɣ-d taktabt-nni tamecṭuḥt seg
ufus n lmelk iwakken aț-țeččeɣ, mi ț-greɣ deg imi-w ufiɣ-ț ț-țaẓidant,
lameɛna mi tewweḍ ɣer uɛebbuḍ-iw, ččuṛen iẓerman-iw ț-țeṛzeg.

<span id="an_10:11">10:11</span> Nnan-iyi-d : Ilaq aț-țcireḍ daɣen ɣef
wayen twalaḍ i waṭas n yegduden d leǧnas d igelliden n mkul tutlayt.

<span id="an_11:1">11:1</span> Fkan-iyi-d aɣanim s wayes țektilin,
nnan-iyi-d : -- Ekker fell-ak, ektil lǧameɛ iqedsen n Sidi Ṛebbi, ktil
daɣen adekkan i deg srusun iseflawen, tḥesbeḍ wid yețɛebbiden di
lǧameɛ-agi.

<span id="an_11:2">11:2</span> Lameɛna afrag n lǧameɛ iqedsen, eǧǧ-it ur
t-țektili ara, axaṭer yețțunefk i leǧnas ur nelli ara n wat Isṛail, ara
iṛekḍen tamdint n Lquds azal n tnin uṛebɛin wagguren.

<span id="an_11:3">11:3</span> Ad ceggɛeɣ sin inigan-iw ad sburren
ticekkaṛin, iwakken ad ciren s wawal n Ṛebbi azal n tnin uṛebɛin
wagguren.

<span id="an_11:4">11:4</span> Sin inigan-agi d snat n tzemrin akk-d sin
n lemṣabeḥ i gbedden zdat Yillu Bab n lqaɛa n ddunit.

<span id="an_11:5">11:5</span> Ma yeɛṛeḍ yiwen a ten-iḍuṛṛ, tețțeffeɣ-ed
tmes seg imawen-nsen tețțeț iɛdawen-nsen, akka ara yemmet kra win ara
iɛerḍen a ten-iḍuṛṛ.

<span id="an_11:6">11:6</span> Sin inigan-agi sɛan tazmert ad ɣelqen
tiwwura igenwan iwakken ur d-yekkat ara ugeffur deg ussan i deg ara
țcirin awal n Ṛebbi ; sɛan daɣen tazmert ad rren aman d idammen, ad
sexḍen lqaɛa s mkul lebla mkul tikkelt i deg ara bɣun.

<span id="an_11:7">11:7</span> M'ara faken aciri nsen, llafɛa ara
d-yalin si tesraft lqayen, aț-țennaɣ yid-sen a ten teɣleb, yerna a
ten-tneɣ.

<span id="an_11:8">11:8</span> Lǧețțat-nsen ad qqiment ḍelqent deg ubṛaḥ
n temdint tameqqrant, i gețțumetlen s temdint n Sudum ț-țmurt n Maṣer,
tamdint anda yețwaṣleb Ssid-nsen.

<span id="an_11:9">11:9</span> Lɣaci n mkul agdud, n mkul lǧens, n mkul
tutlayt, n mkul tamurt, ad țfeṛṛiǧen deg-sen tlata wussan d wezgen, ur
qebblen ara ad țwameḍlen.

<span id="an_11:10">11:10</span> Imdanen n ddunit meṛṛa ad ilin di lfeṛḥ
ameqqran imi mmuten. Ad țemceggaɛen tirezfin wway gar-asen, axaṭer sin
inigan-agi ḍuṛṛen-ten aṭas.

<span id="an_11:11">11:11</span> Mi ɛeddan tlata wussan-nni d wezgen,
Ṛṛuḥ n Ṛebbi yesseḥya-d lǧețțat n lmegtin-nni, bedden ɣef yiḍaṛṛen-nsen.
Kra wid yellan țfeṛṛiǧen deg-sen, tekcem-iten lxelɛa ț-țameqqrant.

<span id="an_11:12">11:12</span> Yiwet n taɣect ɛlayen i d-yekkan seg
igenni tenna i sin inigan-nni : Alit-ed ɣer dagi ! Ulin deg usigna ɣer
igenni, iɛdawen-nsen țmuqulen deg-sen.

<span id="an_11:13">11:13</span> Di teswiɛt-nni tewwet zzelzla
tameqqrant, amur wis ɛecṛa n temdint yegrurej, mmuten sebɛa alaf
yemdanen. Wid i gmenɛen si zzelzla-nni tekcem-iten tugdi, imiren
setɛeṛfen s tezmert n Sidi Ṛebbi Bab n igenwan, ḥemden-t.

<span id="an_11:14">11:14</span> Tawaɣit tis snat tɛedda, ataya
tețțeddu-d twaɣit tis tlata.

<span id="an_11:15">11:15</span> Lmelk wis sebɛa yewwet lbuq. Kra n
tuɣac ɛlayen țɛeggiḍent deg igenni qqaṛent-ed : Lḥekma n ddunit tuɣal
ɣer ifassen n Sidi Ṛebbi akk-d Lmasiḥ ines ; ad yeḥkem si lǧil ɣer lǧil.

<span id="an_11:16">11:16</span> Ṛebɛa uɛecrin-nni n lecyux yellan zdat
Sidi Ṛebbi ɣef yimukan nsen n lḥekma, țseǧǧiden zdat Sidi Ṛebbi,
țɛebbiden-t

<span id="an_11:17">11:17</span> qqaṛen : Nḥemmed-ik a Sidi Ṛebbi, a Bab
n tezmert yellan si tazwara, yellan ar ass-a, imi tesxedmeḍ tazmert-ik
tameqqrant yerna tesbeddeḍ lḥekma-inek.

<span id="an_11:18">11:18</span> Leǧnas llan kkren-d ɣuṛ-ek, lameɛna
tura d urrif-ik i d-yewḍen, yusa-d lweqt i deg ara tḥasbeḍ lmegtin, ass
i deg ara tkafiḍ iqeddacen-ik lenbiya, wid textaṛeḍ akk-d wid yețqadaṛen
isem-ik, ama d imeẓyanen ama d imeqqranen s wayen uklalen ; ma d wid
yessexṛaben ddunit a ten-tesnegreḍ.

<span id="an_11:19">11:19</span> Lǧameɛ iqedsen n Sidi Ṛebbi yellan deg
igenwan yeldi-d, dɣa iban-ed zdaxel-is usenduq n leɛqed. Tebṛeq, terɛeḍ,
nesla i tuɣac, tezlez lqaɛa, yeɣli-d webruri.

<span id="an_12:1">12:1</span> Tban-ed licaṛa ț-țameqqrant deg igenni :
walaɣ yiwet tmeṭṭut tesburr-ed iṭij am llebsa, aggur seddaw iḍaṛṛen-is,
tesɛa ɣef wuqeṛṛuy-is taɛeṣṣabt n tnac yitran.

<span id="an_12:2">12:2</span> Tella s tadist, yewweḍ-ed lweqt i deg ara
d-tarew, tebda tețɛeggiḍ seg weqṛaḥ n tarrawt.

<span id="an_12:3">12:3</span> Tban-ed licaṛa nniḍen deg igenni : walaɣ
yiwet llafɛa tazeggaɣt am tmes, tesɛa sebɛa iqeṛṛay akk-d ɛecṛa
wacciwen, mkul aqeṛṛuy ters fell-as tɛeṣṣabt.

<span id="an_12:4">12:4</span> Ajelkkiḍ-is yezzuɣeṛ-ed țelt n yitran n
igenni, ideggeṛ-iten-id ɣer ddunit. Llafɛa-nni tbedd zdat tmeṭṭut-nni
ara d-yarwen, iwakken aț-țessebleɛ lṭufan-ines akken kan ara d-ilal.

<span id="an_12:5">12:5</span> Tesɛa-d aqcic ara yeksen leǧnas meṛṛa s
tɛekkazt n wuzzal. Aqcic-nni yețwarfed ɣer Sidi Ṛebbi, ɣer wemkan n
lḥekma-ines.

<span id="an_12:6">12:6</span> Tameṭṭut-nni terwel ɣer lexla anda i
s-ihegga Sidi Ṛebbi amkan iwakken aț-țečč azal n walef umitin usețțin n
wussan.

<span id="an_12:7">12:7</span> Yedṛa-d umenɣi deg igenni, Mixayel akk-d
lmalayekkat-is nnuɣen akk-d llafɛa-nni. Llafɛa-nni d lmalayekkat-is
nnuɣen-ten ula d nutni,

<span id="an_12:8">12:8</span> lameɛna ur ten-ɣliben ara, daymi i
sen-yețwakkes wemkan-nsen deg igenni.

<span id="an_12:9">12:9</span> Llafɛa-nni tameqqrant tețțuḍeggeṛ ɣer
ddunit nețțat d lmalayekkat-is, tin akken yellan d azrem-nni n zik
yețțusemman Iblis, Cciṭan, tin i gxedɛen ddunit meṛṛa.

<span id="an_12:10">12:10</span> Sliɣ deg igenni i yiwen ṣṣut ɛlayen
yeqqaṛ : tura yewweḍ-ed leslak, Nḥemmed-ik a Sidi Ṛebbi, lḥekma
ț-țezmert i Sidi Ṛebbi Illu-nneɣ, lḥekma n ddunit tuɣal ɣer ifassen n
Lmasiḥ-ines, axaṭer win i gesseḍlamen atmaten-nneɣ, win yețcektayen
fell-asen iḍ d wass zdat Sidi Ṛebbi Illu-nneɣ, yețțuḍeggeṛ seg igenni.

<span id="an_12:11">12:11</span> ?elben-t watmaten-nneɣ s idammen n
Izimer i ten-id-yefdan akk-d cchada-nsen fell-as yerna ur ɛuzzen ara
tudert-nsen lameɛna sebblen-ț fell-as.

<span id="an_12:12">12:12</span> ?ef wayagi feṛḥet ay igenwan, a wid i
gzedɣen igenwan ! A nnger-im a lqaɛa ! A nnger-ik a lebḥeṛ ! Imi i
d-iṣubb ɣuṛ-wen Cciṭan s wurrif ameqqran, axaṭer yeẓra drus n lweqt i
s-d-iqqimen.

<span id="an_12:13">12:13</span> Llafɛa-nni, mi twala iman-is
tețwaḍeggeṛ ɣer lqaɛa, tetbeɛ tameṭṭut nni i d-yesɛan aqcic.

<span id="an_12:14">12:14</span> Imiren kan țțunefken-as i tmeṭṭut-nni
sin wafriwen n lbaz ameqqran iwakken aț-țafeg ɣer unezṛuf anda ara tɛic
tlata iseggasen d wezgen, mebɛid ɣef wezrem.

<span id="an_12:15">12:15</span> Azrem-nni yessufeɣ-ed seg uqemmuc-is
aman, feggḍen-d am asif deffir n tmeṭṭut-nni iwakken aț-țyawi uḥemmal.

<span id="an_12:16">12:16</span> Lameɛna lqaɛa tsellek tameṭṭut nni,
tessebleɛ aman-nni i d-ifeggḍen seg uqemmuc n llafɛa-nni.

<span id="an_12:17">12:17</span> Llafɛa terfa ɣef tmeṭṭut-nni, tṛuḥ
aț-țeskker imenɣi akk-d wid i d yeqqimen si dderya-s, wid i gḍuɛen lumuṛ
n Sidi Ṛebbi yerna ṭṭfen di tideț n Sidna Ɛisa.

<span id="an_12:18">12:18</span> Llafɛa-nni tbedd ɣef ṛṛmel n lebḥeṛ.

<span id="an_13:1">13:1</span> Walaɣ yiwet leɛqiṛa tuli-d si lebḥeṛ,
tesɛa sebɛa iqeṛṛay d ɛecṛa wacciwen, tesɛa ɣef wacciwen-is ɛecṛa
tɛeṣṣabin, ma ɣef yiqeṛṛay-ines uran yismawen n rregmat.

<span id="an_13:2">13:2</span> Leɛqiṛa i walaɣ tecba aɣilas, iḍaṛṛen-is
am wid n wersel, axenfuc is am uxenfuc n yizem. Llafɛa-nni tefka-yas
tazmert aț-țeḥkem deg umkan-is s lḥekma tameqqrant.

<span id="an_13:3">13:3</span> Walaɣ yiwen seg iqeṛṛay-ines am akken
yețwamzel, lameɛna lǧerḥ n tmezliwt-nni yeḥla. Imezdaɣ n ddunit meṛṛa
dehcen ɣef wannect-agi, dɣa tebɛen leɛqiṛa-nni.

<span id="an_13:4">13:4</span> ?seǧǧiden-as i llafɛa-nni, axaṭer tefka
lḥekma-s i leɛqiṛa ; țseǧǧiden daɣen zdat leɛqiṛa qqaṛen : anwa yellan
am leɛqiṛa, anwa i gzemren ad yennaɣ yid-es ?

<span id="an_13:5">13:5</span> Yețțunefk-as uxenfuc i seg i d-țeffɣen
imeslayen iqebḥen akk-d rregmat ɣef Sidi Ṛebbi. Tețțunefk-as tezmert
aț-țeḥkem azal n tnin uṛebɛin n wagguren.

<span id="an_13:6">13:6</span> Teldi axenfuc-is, tețqabaḥ Sidi Ṛebbi,
treggem isem-is d wemkan i deg yețțili akk-d kra wid i gzedɣen deg
igenni.

<span id="an_13:7">13:7</span>  Serrḥen-as aț-țeskker imenɣi akk-d wid
yextaṛ Sidi Ṛebbi yerna a ten-teɣleb. Tețțunefk-as lḥekma ɣef mkul
lɛeṛc, ɣef mkul agdud, d mkul lǧens n mkul tutlayt.

<span id="an_13:8">13:8</span> Kra win ur nețwajerred ara di tektabt n
tudert yellan seg wasmi tebda ddunit, di tektabt n Izimer immezlen, ad
seǧǧden zdat-es.

<span id="an_13:9">13:9</span> Win yesɛan imeẓẓuɣen isel-ed !

<span id="an_13:10">13:10</span> Win i gețṛaǧu nnfu, ad yețwanfu ; ma d
win yețṛaǧu ujenwi, ad yemmet s ujenwi. Daymi i glaq i wid yextaṛ Sidi
Ṛebbi ad ṭṭfen di liman-nsen, ad sɛun ṣṣbeṛ.

<span id="an_13:11">13:11</span> Walaɣ daɣen yiwet leɛqiṛa nniḍen
teffeɣ-ed seddaw n tmurt. Tesɛa sin wacciwen am wid n izimer, tețmeslay
am llafɛa.

<span id="an_13:12">13:12</span> Tessexdam tazmert n leɛqiṛa tamezwarut
ɣer tama-ines, teṣṣaweḍ armi terra imezdaɣ n ddunit meṛṛa țseǧǧiden zdat
leɛqiṛa-nni tamezwarut i geḥlan si lǧerḥ-is.

<span id="an_13:13">13:13</span> Txeddem lbeṛhanat imeqqranen zdat
yemdanen, tesseɣlay-ed times seg igenni ɣer lqaɛa.

<span id="an_13:14">13:14</span> Tețɣuṛṛu imezdaɣ n ddunit s lbeṛhanat i
s-yețțunefken a ten texdem seddaw lḥekma n leɛqiṛa-nni tamezwarut,
teqqaṛ asen : sbeddet yiwen lmeṣnuɛ ɣef ṣṣifa n leɛqiṛa-nni yețțewten s
ujenwi tuɣal teḥla.

<span id="an_13:15">13:15</span> Leɛqiṛa tis snat, tețțunefk-as tezmert
a s-tefk ṛṛuḥ i lmeṣnuɛ n leɛqiṛa-nni tamezwarut, iwakken lmeṣnuɛ-nni ad
yizmir a d-yenṭeq, yerna ad ineɣ meṛṛa wid yugin a t-ɛebbden.

<span id="an_13:16">13:16</span> Leɛqiṛa-nni tḥețțem ɣef yemdanen, meẓẓi
neɣ meqqeṛ, d igellil neɣ d ameṛkanti, d akli neɣ d aḥeṛṛi, ad wwten
ticṛaḍ deg ufus ayeffus neɣ deg unyir-nsen,

<span id="an_13:17">13:17</span> yerna ula d yiwen ur izemmer ad yaɣ neɣ
ad yezzenz, anagar win yesɛan ticṛaḍ-agi n yisem n leɛqiṛa neɣ numṛu n
yisem-is.

<span id="an_13:18">13:18</span> Win iḥeṛcen yesɛan tamusni d leɛqel, a
d-yaf lmeɛna n numṛu n leɛqiṛa yellan d isem n yiwen wergaz, numṛu-agi :
d sețțemeyya usețța usețțin.

<span id="an_14:1">14:1</span> Walaɣ Izimer-nni ibedd ɣef wedrar n
Siyun, llan yid-es meyya uṛebɛa uṛebɛin alef n yemdanen yesɛan isem n
Izimer-nni akk-d isem n Baba-s țwaketben ɣef yinyiren-nsen.

<span id="an_14:2">14:2</span> Sliɣ i ṣṣut n tuɣac i d-yekkan seg
igenni, am akken d ṣṣut n lebḥeṛ m'ara yekker, am tiyita n ṛṛɛud ; ṣṣut
n tuɣac-nni, yecba ṣṣut i d-ițeffɣen seg waṭas n snitrat.

<span id="an_14:3">14:3</span> ?ɣennin yiwen ccna ajdid zdat wemkan n
lḥekma, zdat ṛebɛa lexluq zdat lecyux-nni. Yiwen ur izmir ad yeḥfeḍ
ccna-nni anagar meyya uṛebɛa uṛebɛin alef-nni i d-yețțuselken si ddunit.

<span id="an_14:4">14:4</span> Wigi, d wid ur nezni ara, qqimen ṣfan ;
anda iṛuḥ Izimer ad ddun yid-es. ?țuselken si ger yemdanen iwakken ad
țțuqeddmen d imezwura i Sidi Ṛebbi akk-d Izimer.

<span id="an_14:5">14:5</span> Ur d-yeffiɣ lekdeb seg imawen nsen, ur
yelli deg-sen wayen ara sen-d yețwassukksen.

<span id="an_14:6">14:6</span> Walaɣ yiwen lmelk yețțafgen deg igenni ;
yewwi-d yid-es yiwen lexbaṛ n lxiṛ yețdumun i dayem iwakken a t-ibecceṛ
i imezdaɣ n ddunit, i mkul lǧens, i mkul tamurt, i mkul agdud n mkul
tutlayt.

<span id="an_14:7">14:7</span> Yeqqaṛ ed s taɣect ɛlayen : Aggadet
Ṛebbi, ḥemmdet-eț axaṭer yewweḍ-ed wass n lḥisab. Seǧǧdet, ɛebdet Win i
d-ixelqen igenni d lqaɛa, lebḥeṛ akk-d leɛwanseṛ n waman.

<span id="an_14:8">14:8</span> Lmelk wis sin yetbeɛ amezwaru yeqqaṛ :
Teɣli, teɣli temdint-nni n Babilun tameqqrant, tin i gesderɛen (
yessenṭḍen ) i leǧnas meṛṛa leḥṛam ines.

<span id="an_14:9">14:9</span> Lmelk wis tlata, yetbeɛ-iten yeqqaṛ s
ṣṣut ɛlayen : Win ara iseǧden zdat leɛqiṛa akk-d lmeṣnuɛ-ines yerna
yeqbel ticṛeṭ ɣef wunyir-is neɣ ɣef wufus-is,

<span id="an_14:10">14:10</span> a d-yemmir fell-as wurrif n Sidi Ṛebbi,
ad isew lemṛaṛ, ad inɛețțab di tmes d ukubri zdat lmalayekkat iqedsen,
zdat Izimer.

<span id="an_14:11">14:11</span> Dexxan n tmes s wayes ara nɛețțaben ad
yeqqim si lǧil ɣer lǧil ; wid yețseǧǧiden zdat leɛqiṛa d lmeṣuɛ ines
akk-d kra win iqeblen ticṛeṭ n isem-is, ur steɛfayen ara iḍ d wass.

<span id="an_14:12">14:12</span> Deg wannect-agi i d-yețban ṣṣbeṛ n wid
yextaṛ Sidi Ṛebbi, wid i gḍuɛen lumuṛ-is yerna ṭṭfen di liman i sɛan di
Lmasiḥ.

<span id="an_14:13">14:13</span> Sliɣ i yiwet taɣect i d-yekkan seg
igenni teqqaṛ : Aru ayagi, d iseɛdiyen si tura wid ara yemten yili nutni
umnen s Lmasiḥ ! Ṛṛuḥ iqedsen yenna : ?-țideț, tura zemren ad steɛfun si
leɛtab-nsen axaṭer lecɣal-nni yelhan i xedmen a ten-afen.

<span id="an_14:14">14:14</span> Walaɣ asigna d amellal, ɣef wusigna-nni
yeqqim yiwen lxelq ițemcabi i wemdan. Yesɛa taɛeṣṣabt n ddheb ɣef
wuqeṛṛuy-is, yeṭṭef amger iqeḍɛen deg ufus-is.

<span id="an_14:15">14:15</span> Yiwen lmelk nniḍen yeffeɣ-ed si lǧameɛ
iqedsen, yețɛeggiḍ s ṣṣut ɛlayen i win yeqqimen ɣef wusigna yeqqaṛ-as :
Ssexdem amger-ik tmegreḍ, axaṭer yewweḍ-ed lweqt n tmegra, imi ddunit
tewjed i tmegra.

<span id="an_14:16">14:16</span> Win akken yeqqimen ɣef wusigna yesɛedda
amger-is ɣef ddunit dɣa ddunit meṛṛa tețwamger.

<span id="an_14:17">14:17</span> Yeffeɣ-ed lmelk nniḍen si lǧameɛ
iqedsen yellan deg igenni ; ula d nețța yesɛa amger iqeḍɛen.

<span id="an_14:18">14:18</span> Lmelk nniḍen iḥekkmen ɣef tmes,
yeffeɣ-ed seg wemkan anda yella udekkan n iseflawen, yenna s ṣṣut ɛlayen
i lmelk-nni yesɛan amger iqeḍɛen : Ssexdem amger-ik tgezmeḍ iguza n
tfeṛṛant n ddunit, axaṭer wwant tẓuṛin.

<span id="an_14:19">14:19</span>  Lmelk-nni yesɛedda amger-is ɣef
ddunit, igzem iguza n tfeṛṛant n ddunit, iḍeggeṛ tizuṛin-nni zdaxel n
teḥḍunt n wurrif n Sidi Ṛebbi, wwin-ț ɣer beṛṛa n temdint, iwakken ad
ṛekḍen tizuṛin-nni, ffɣen-d seg-s idammen, ulin lqedd n lmitra d wezgen,
uzzlen d leḥmali azal n telt-meyya n kilumitrat.

<span id="an_15:1">15:1</span> Walaɣ daɣen deg igenni licaṛa nniḍen
tameqqrant yesteɛǧiben ; sebɛa n lmalayekkat ṭṭfent sebɛa n lmuṣibat
tineggura, axaṭer yis-sent ara innekmal wurrif n Sidi Ṛebbi.

<span id="an_15:2">15:2</span> Walaɣ am akken d lebḥeṛ n djaj ixelḍen
akk-d tmes, wid iɣelben leɛqiṛa d lmeṣnuɛ-is akk-d numṛu n isem-is,
bedden ɣef lebḥeṛ-nni n djaj. ?ṭfen snitrat i sen-d-yefka Sidi Ṛebbi,

<span id="an_15:3">15:3</span> țcewwiqen acewwiq n Sidna Musa, aqeddac n
Sidi Ṛebbi akk-d ucewwiq n Izimer, qqaṛen : Lecɣal-ik meqqeṛ-it
steɛǧiben a Sidi Ṛebbi a Bab n tezmert ! Iberdan-ik, d iberdan n lḥeqq
ț-țideț ay Agellid n leǧnas.

<span id="an_15:4">15:4</span> Anwa ur nețțaggad, ur nețḥemmid isem-ik a
Sidi Ṛebbi ? Axaṭer d kečč kan i d imqeddes. Leǧnas meṛṛa a d-asen ad
seǧǧden zdat-ek, imi tḥekkmeḍ s lḥeqq a Bab n lḥeqq.

<span id="an_15:5">15:5</span> Deffir wayagi, walaɣ lǧameɛ iqedsen anda
yella uqiḍun n temlilit yeldi deg igenni.

<span id="an_15:6">15:6</span> Sebɛa n lmalayekkat-nni yeṭṭfen sebɛa n
lmuṣibat, ffɣent-ed si lǧameɛ-nni s llebsa n leḥrir yeṣfan yețfeǧǧiǧen,
beggsent-ed s wagusen n ddheb ɣef yidmaren-nsent.

<span id="an_15:7">15:7</span> Yiwen si ṛebɛa lxuluq-nni, yefka i
sebɛa-nni n lmalayekkat sebɛa n teqbucin n ddheb ččuṛent s wurrif n Sidi
Ṛebbi yeddren si lǧil ɣer lǧil.

<span id="an_15:8">15:8</span> Lɛaḍima akk-d lḥekma n Sidi Ṛebbi ččuṛen
lǧameɛ iqedsen s dexxan, yiwen ur izmir ad yekcem ɣer lǧameɛ-nni alamma
nnekmalent lmuṣibat n sebɛa lmalayekkat-nni.

<span id="an_16:1">16:1</span> Sliɣ i yiwen ṣṣut ɛlayen i d-yekkan si
lǧameɛ iqedsen yeqqaṛ i sebɛa-nni n lmalayekkat : -- Ṛuḥemt, smiremt ɣef
ddunit sebɛa teqbucin n wurrif n Sidi Ṛebbi !

<span id="an_16:2">16:2</span> Lmelk amezwaru iṛuḥ, yesmar taqbuct-is
ɣef ddunit. Yiwen udeddi iweɛṛen iqerriḥen, yenṭeḍ imdanen meṛṛa yesɛan
ticṛeṭ n leɛqiṛa yerna țseǧǧiden zdat lmeṣnuɛ-ines.

<span id="an_16:3">16:3</span> Lmelk wis sin yesmar taqbuct-is ɣer
lebḥeṛ, yuɣal d idammen am idammen n lmegget, imiren kra wayen yețɛicen
di lebḥeṛ yemmut.

<span id="an_16:4">16:4</span> Lmelk wis tlata yesmar taqbuct-is ɣef
yisaffen d leɛwanṣer n waman, uɣalen meṛṛa d idammen.

<span id="an_16:5">16:5</span> Sliɣ i lmelk iḥekkmen ɣef waman yeqqaṛ :
Ay Imqeddes, a Bab n lḥeqq, kečč yellan si zik, yellan ass-a, tḥekkmeḍ s
lḥeqq.

<span id="an_16:6">16:6</span> Axaṭer nutni ssazlen idammen n yimqedsen
akk-d lenbiya, daymi i sen-tefkiḍ ula d kečč ad swen idammen, axaṭer d
ayen uklalen.

<span id="an_16:7">16:7</span> Sliɣ i yiwet taɣect i d-yekkan seg
udekkan n iseflawen teqqaṛ : Anɛam a Sidi Ṛebbi a Bab n tezmert,
lḥekma-inek ț-țideț yerna d lḥeqq.

<span id="an_16:8">16:8</span> Lmelk wis ṛebɛa yesmar taqbuct-is ɣef
yiṭij, yețțunefk-as i yiṭij ad yesseṛɣ imdanen,

<span id="an_16:9">16:9</span> dɣa imdanen meṛṛa ṛɣan s yiṭij. Reggmen
Sidi Ṛebbi i gḥekmen ɣef lmuṣibat-agi, ugin ad tuben, ad setɛeṛfen yis,
akken a t-ḥemmden.

<span id="an_16:10">16:10</span> Lmelk wis xemsa yesmar taqbuct-is ɣef
wukersi n lḥekma n leɛqiṛa-nni. Yeɣli-d ṭṭlam ɣef tgelda-s ; imdanen
țkerricen ilsawen-nsen seg weqṛaḥ,

<span id="an_16:11">16:11</span> reggmen Sidi Ṛebbi Bab n igenwan ɣef
lqeṛḥ d ideddiyen-nsen, ur bɣin ara ad tuben seg yir lecɣal-nsen.

<span id="an_16:12">16:12</span> Lmelk wis sețța yesmar taqbuct is ɣef
wasif ameqqran n Lufrat. Yesɣeṛ asif-nni iwakken ad iheggi abrid i
igelliden i d-ițeddun si cceṛq.

<span id="an_16:13">16:13</span> Walaɣ ffɣen-d seg uxenfuc n llafɛa, seg
uxenfuc n leɛqiṛa, seg yimi n nnbi n lekdeb, tlata leṛwaḥ iɛefnen i
gțemcabin ɣer imqerqaṛ.

<span id="an_16:14">16:14</span> Widak-nni, d leṛwaḥ n leǧnun i gxeddmen
licaṛat, țṛuḥun ɣer igelliden n ddunit meṛṛa, iwakken a ten-id-snejmaɛen
i wumenɣi n wass ameqqran n Sidi Ṛebbi Bab n tezmert.

<span id="an_16:15">16:15</span> Aql-in a n-aseɣ am umakar. D aseɛdi win
iɛussen ɣef yiman-is ur yekkis ara llebsa-ines, iwakken ur ițeddu ara
ɛeryan, ur ițneḥcam ara m'ara t-id-walin !

<span id="an_16:16">16:16</span> Leṛwaḥ-nni n leǧnun snejmaɛen-d
igelliden n ddunit deg yiwen wemkan iwumi qqaṛen s tɛibṛanit «
Harmagidun ».

<span id="an_16:17">16:17</span> Lmelk wis sebɛa yesmar taqbuct is ger
igenni d lqaɛa. Teffeɣ-ed si lǧameɛ iqedsen yiwet taɣect ɛlayen, i
d-yekkan seg ukersi n lḥekma teqqaṛ : Dayen kullec yețwakemmel !

<span id="an_16:18">16:18</span> Tebṛeq, terɛeḍ, nesla i tuɣac, tezlez
lqaɛa, seg wasmi i d-țwaxelqen imdanen ɣer ddunit ur d-tedṛi zzelzla
tameqqrant am tagi.

<span id="an_16:19">16:19</span> Tamdint tameqqrant tebḍa ɣef tlata.
Timdinin n tmura ɣlint, Sidi Ṛebbi yemmekta-d Babilun tameqqrant iwakken
a s-yefk aț-țsew lemṛaṛ si teqbuct n wurrif-is ameqqran.

<span id="an_16:20">16:20</span> Tigzirin zedrent meṛṛa, ula d idurar ur
d-banen ara.

<span id="an_16:21">16:21</span> Yeɣli-d igedrez d ameqqran ɣef
yemdanen, mkul aɛeqqa yewzen azal n ṛebɛin kilu ; imdanen regmen Sidi
Ṛebbi ɣef lmuṣiba n igedrez-nni, axaṭer tegla-d s waṭas n lexṣaṛa.

<span id="an_17:1">17:1</span> Mbeɛd ayagi, yiwen si sebɛa
lmalayekkat-nni yeṭṭfen sebɛa teqbucin yusa-d yenna-yi-d : -- As-ed a
k-d-sneɛteɣ amek ara yețwaḥkem ɣef tmenɛult tameqqrant i gebnan ɣef
yisaffen imeqqranen.

<span id="an_17:2">17:2</span> Yid-es i zennun igelliden n ddunit, ɣef
ddemma-s i ffɣen imezdaɣ n ddunit seg ubrid n Sidi Ṛebbi.

<span id="an_17:3">17:3</span> Deg uweḥḥi ufiɣ-ed iman-iw deg unezṛuf.
Walaɣ yiwet tmeṭṭut teqqim ɣef yiwen lweḥc, rric-is d azeggaɣ, țwaketben
fell-as yismawen n rregmat, yesɛa sebɛa iqeṛṛay akk-d ɛecṛa wacciwen.

<span id="an_17:4">17:4</span> Tameṭṭut-nni telsa llebsa ifazen
țțazeggaɣt, tcebbeḥ iman-is s ddheb d yedɣaɣen ɣlayen akk-d iɛeqcan.
Teṭṭef deg ufus-is taqbuct n ddheb yeččuṛen ț-țicmatin, yeɛni ayen
ineǧsen d wayen iɛefnen n yir lecɣal is.

<span id="an_17:5">17:5</span> Yețwakteb ɣef wunyir-is yiwen yisem iwumi
yețwaffer lmeɛna-s : Babilun tucmiț tameqqrant i seg d-kkant tecmatin
akk-d leḥṛamat meṛṛa n ddunit.

<span id="an_17:6">17:6</span> Walaɣ tameṭṭut-nni teskeṛ s idammen n wid
yextaṛ Sidi Ṛebbi, d idammen n inigan n Lmasiḥ. Mi ț-walaɣ, wehmeɣ
deg-s.

<span id="an_17:7">17:7</span> Yekker lmelk-nni yenna-yi-d : Acuɣeṛ i
twehmeḍ ? Atan a k-d sfehmeɣ lbaḍna n tmeṭṭut-agi akk-d lweḥc i
ț-yețbibbin, win akken yesɛan sebɛa iqeṛṛay d ɛecṛa wacciwen.

<span id="an_17:8">17:8</span> Lweḥc-nni twalaḍ, zik yella yedder, tura
ulac-it. A d-yali si tesraft lqayen lameɛna iwakken ad iṛuḥ ɣer nnger.
Imezdaɣ n ddunit iwumi ur yețwajerred ara yisem-nsen seg wasmi i d-tebda
ddunit di tektabt n tudert, ad tɛeǧǧben m'ara walin lweḥc-nni, axaṭer
lweḥc-nni yellan zik yuɣal ulac-it, ițeddu a d-yuɣal.

<span id="an_17:9">17:9</span> Dagi, tlaq tmusni d lefhama iwakken ad
țwafehment lbaḍnat-agi : sebɛa iqeṛṛay-nni țmettilen-d sebɛa idurar i
ɣef tesbedd lḥekma-s tmeṭṭut-nni ; țmettilen-d daɣen sebɛa igelliden :

<span id="an_17:10">17:10</span> xemsa deg-sen ɣlin, yiwen mazal-it,
wayeḍ urɛad ur d-yusi, yerna asm'ara d-yas ad yeḥkem kra n lweqt kan.

<span id="an_17:11">17:11</span> Lweḥc-nni yellan zik yuɣal ulac-it, d
nețța i d agellid wis tmanya, yella daɣen d yiwen si sebɛa-nni ara
iṛuḥen ɣer nnger.

<span id="an_17:12">17:12</span> Ɛecṛa wacciwen nni i twalaḍ,
țmettilen-d ɛecṛa igelliden urɛad nesɛi lḥekma, lameɛna a d-yas lweqt i
deg ara ḥekmen azal n ssaɛa akk-d lweḥc.

<span id="an_17:13">17:13</span> Msefhamen s yiwen ṛṛay, ad ɛawnen lweḥc
s tezmert-nsen akk-d lḥekma-nsen.

<span id="an_17:14">17:14</span> Ad sekkren imenɣi akk-d Izimer meɛna a
ten-yeɣleb imi d nețța i d Ssid n Ssyadi, i d agellid n igelliden ; wid
i gextaṛ daɣen iwumi i d-yessawel, i t-iḍuɛen a ten-ɣelben ula d nutni.

<span id="an_17:15">17:15</span> Lmelk yenna-yi-d daɣen : Aman-agi n
isaffen i twalaḍ i ɣef tesbedd lḥekma-s tmeṭṭut-nni tucmiț, țmettilen-d
igduden, lɣaci d leǧnas n mkul tutlayt.

<span id="an_17:16">17:16</span> Ɛecṛa wacciwen-nni i twalaḍ akk-d
lweḥc, ad uɣalen ad keṛhen tameṭṭut-nni tucmiț, a ț-ɛerrin ad ččen
aksum-is, imiren ad sseṛɣen ayen i d-yeqqimen seg-s.

<span id="an_17:17">17:17</span> D Sidi Ṛebbi i ten-iwellhen akken ad
xedmen lebɣi-s, ddukklen ɣef yiwen ṛṛay iwakken ad fken lḥekma i lweḥc
alamma yedṛa-d lebɣi n Sidi Ṛebbi.

<span id="an_17:18">17:18</span> Ma ț-țameṭṭut-nni tucmiț i twalaḍ,
tețmettil-ed tamdint tameqqrant i gḥekmen ɣef yigelliden n ddunit.

<span id="an_18:1">18:1</span> Deffir wayagi walaɣ lmelk nniḍen i
d-yusan seg igenni, yesɛa lḥekma tameqqrant, ddunit meṛṛa tfeǧǧeǧ s
tmanegt-is.

<span id="an_18:2">18:2</span> Iɛeggeḍ s ṣṣut ɛlayen yenna : Teɣli !
Teɣli temdint-nni tameqqrant n Babilun ! Tuɣal ț-țamezduɣt n ccwaṭen, d
amkan anda țnejmaɛen leṛwaḥ imeɛfan, anda xeddmen leɛcuc-nsen leṭyuṛ
iɛefnen.

<span id="an_18:3">18:3</span> Axaṭer tewwi leǧnas meṛṛa ɣer leǧhel s
yir tikli-ines, igelliden n ddunit jehlen yid-es, wid yețțaɣen znuzun
uɣalen d imeṛkantiyen s sselɛa ifazen i s-znuzun.

<span id="an_18:4">18:4</span> Sliɣ i yiwet taɣect i d-yekkan seg igenni
teqqaṛ : Effɣet seg-s ay agdud-iw, iwakken ur tɣellim ara di ddnub-is,
iwakken ur kkun-id-țawḍent ara lmuṣibat ara d-yeɣlin fell-as.

<span id="an_18:5">18:5</span> Axaṭer ddnubat-is ɛeddan tilas wwḍen ɣer
igenni, amek ara yețțu Sidi Ṛebbi lbaṭel i texdem.

<span id="an_18:6">18:6</span> Xedmet as akken i wen-txeddem, ɛețțbet-eț
akteṛ wayen i kkun-tɛețțeb, taqbuct nni n lemṛaṛ i tessew i wiyaḍ,
ɛemmṛet-as-ț d lemṛaṛ yugaren win tessew nețțat.

<span id="an_18:7">18:7</span> Akken tefka i yiman-is ccan, tɛac di

<span id="an_18:8">18:8</span> Daymi, deg yiwen wass ara d-ɣlint fell-as

Ṛebbi Bab n tezmert i ț-iɛuqben.

<span id="an_18:9">18:9</span> Igelliden n ddunit meṛṛa i gɛacen deg-s

<span id="an_18:10">18:10</span> Imi uggaden a ten-taweḍ txeṣṣart,

tamdint tameqqrant yesɛan tazmert, deg yiwet n ssaɛa kan yeɣli-d fell-am
lɛiqab.

<span id="an_18:11">18:11</span> Wid yețțaɣen znuzun ad ilin di leḥzen

<span id="an_18:12">18:12</span> sselɛa-nni n ddheb d lfeṭṭa, idɣaɣen
ɣlayen d iɛeqcan, lkețțan n lfina akk-d lkețțan azeggaɣ d leḥrir, sselɛa
n yal asɣaṛ i d-yețțaken rriḥa taẓiḍant, leḥwayeǧ akk ineǧṛen s wacciwen
n lfil, s wesɣaṛ ɣlayen d wayen yețwaxedmen s nnḥas d wuzzal akk-d eṛxem
;

<span id="an_18:13">18:13</span> ulac daɣen win ara yaɣen lqeṛfa,
leɛqaqeṛ, leɛṭeṛ, lmuṛ, lebxuṛ, ccṛab, zzit, awren eṛqiqen akk-d yirden,
izgaren, ulli, iɛewdiwen, ikeṛṛusen n umenɣi akk-d yemdanen i rran d
aklan.

<span id="an_18:14">18:14</span> A tamdint n Babilun, ayen akken
tețmenniḍ iṛuḥ-am ibɛed fell-am, lerbaḥ-im d wayen akk mucaɛen ɣuṛ-em,
ṛuḥen-am ur tețțuɣaleḍ ara a ten-tafeḍ.

<span id="an_18:15">18:15</span> Wid yețțaɣen znuzun leḥwayeǧ di
temdint-agi yuɣalen d imeṛkantiyen, ad qqimen mebɛid ad aggaden a
ten-taweḍ twaɣit, ad țrun, ad ilin di leḥzen ;

<span id="an_18:16">18:16</span> ad inin : A tawaɣit ! A tawaɣit !

tazeggaɣt ifazen, yețɛelliqen iɛeqcan ɣlayen, idɣaɣen n lyaman n mkul
ṣṣenf akk-d ddheb,

<span id="an_18:17">18:17</span> deg yiwet n ssaɛa kan, kullec yuɣal d
iɣed ! Lqebṭanat n lbabuṛat d ixeddamen nsen, ibeḥṛiyen akk-d wid
yețsafaṛen di lebḥuṛ ɣer tmurt-nni qqimen mebɛid,

<span id="an_18:18">18:18</span> țɛeggiḍen mi walan dexxan n tmes yeččan
tamdint-nni, qqaṛen : Anta tamdint yecban tamdint agi ?

<span id="an_18:19">18:19</span>  Bdan ṭeggiṛen aɣebbaṛ ɣef
yiqeṛṛay-nsen, țrun țmeǧǧiden, țɛeggiḍen qqaṛen : A tawaɣit-im ! A
tawaɣit-im a tamdint tameqqrant, isrebḥen wid akk yesɛan lbabuṛat di
lebḥur, deg yiwet teswiɛt kan tnegreḍ tuɣaleḍ d iɣed.

<span id="an_18:20">18:20</span> A wid yextaṛ Sidi Ṛebbi, a wid i
d-iceggeɛ ad beccṛen awal-is, a lenbiya, ay igenwan, ilit di lfeṛḥ ɣef
wayen yedṛan yid-es, axaṭer Sidi Ṛebbi iɛuqeb-iț ɣef wayen i wen texdem.

<span id="an_18:21">18:21</span> Imiren yiwen lmelk yesɛan tazmert
tameqqrant, yeddem-ed tablaṭ annect n uɣaṛef n tsirt, iḍeggeṛ-iț ɣer
lebḥeṛ yenna : Akka ara tețțuḍeggeṛ temdint tameqqrant n Babilun, ur
tețțuɣal a d-tban ;

<span id="an_18:22">18:22</span> ur nețțuɣal a nsel i ṣṣut n wid
yekkaten snitra d wid i gcennun akk-d wid yekkaten ajewwaq d lbuq. Ur
d-yețɣimi deg-em ṣṣaneɛ, ur nețțɣal a nsel i iɣuṛaf n tessirt.

<span id="an_18:23">18:23</span> Ur tețțuɣal a d-tban deg-em tafat, ur
nețțuɣal a nsel i taɣect n yesli ț-țeslit. Aț-țwaɛaqbeḍ a tamdint n
Babilun imi imestujaṛ-im, d nutni i gellan d imeqqranen di ddunit ;
leǧnas meṛṛa țwaxedɛen s ssḥur i txeddmeḍ ;

<span id="an_18:24">18:24</span> axaṭer deg-em i yuzzlen idammen n
lenbiya d wid yextaṛ Sidi Ṛebbi akk-d wid meṛṛa yemmezlen di ddunit.

<span id="an_19:1">19:1</span> Imiren sliɣ i yiwen ṣṣut eɛlayen deg
igenni, d ṣṣut n luluf n yemdanen, qqaṛen : Halliluya lḥemd i Ṛebbi !
Leslak, tamanegt ț-țezmert i Sidi Ṛebbi Illu-nneɣ,

<span id="an_19:2">19:2</span> axaṭer yeḥkem s lḥeqq ț-țideț ɣef
tucmiț-nni tameqqrant i gɣuṛṛen imezdaɣ n ddunit, yerra-d țțaṛ n
iqeddacen-is yemmezlen.

<span id="an_19:3">19:3</span> Nnan daɣen : Halliluya !... Times
yesseṛɣen tamdint tameqqrant ur txețți ara, aț-țeqqim i dayem.

<span id="an_19:4">19:4</span> Ṛebɛa uɛecrin-nni lecyux akk-d ṛebɛa
lxuluq seǧǧden zdat Sidi Ṛebbi i geqqimen ɣef wukersi n lḥekma,
țcekkiṛen-t qqaṛen : Amin ! Halliluya !

<span id="an_19:5">19:5</span> Yeffeɣ-ed yiwen ṣṣut seg ukersi nni n
lḥekma, yeqqaṛ-ed : ?emmdet Sidi Ṛebbi Illu-nneɣ, kunwi meṛṛa yellan d
iqeddacen-is, kunwi i t-ițḍuɛun am umeqqran am umeẓyan.

<span id="an_19:6">19:6</span> Rniɣ sliɣ i luluf n yemdanen am akken d
ṣṣut n lebḥeṛ m'ara yekker, am tiyita n ṛṛɛud, i geqqaṛen : Halliluya !
Sidi Ṛebbi Illu-nneɣ Bab n tezmert yesbedd lḥekma-s.

<span id="an_19:7">19:7</span> A nezhu, a nefṛeḥ, a t-neḥmed axaṭer
tewweḍ-ed tmeɣṛa n Izimer, tislit thegga iman-is,

<span id="an_19:8">19:8</span> yețțunefk-as aț-țels lkețțan aṛqaq
yețfeǧǧiǧen. Lkețțan-agi imettel-ed lecɣal yelhan n wid yextaṛ Sidi
Ṛebbi.

<span id="an_19:9">19:9</span> Lmelk yenna-yi-d : Aru ayagi : d
iseɛdiyen wid i d-yețwaɛerḍen ɣer imensi n tmeɣṛa n Izimer. Yenna-yi-d
daɣen : Imeslayen-agi, d imeslayen n tideț n Sidi Ṛebbi.

<span id="an_19:10">19:10</span> Dɣa ɣliɣ ɣer iḍaṛṛen-is iwakken a
t-ɛebdeɣ, meɛna yenna-yi-d : ?ader a d-seǧǧdeḍ zdat-i, axaṭer nekk d
aqeddac n Sidi Ṛebbi am kečč am atmaten-ik i geṭṭfen di lexbaṛ n lxiṛ n
Ɛisa Lmasiḥ, d Sidi Ṛebbi kan ara tɛebdeḍ. Tideț-agi i d-ibeggen Ɛisa
Lmasiḥ, ț-țin i gețwellihen lenbiya.

<span id="an_19:11">19:11</span> Walaɣ daɣen igenni yeldi, iban-ed yiwen
uɛewdiw d amellal. Win i t-irekben isem-is « Aḥeqqi ț-țideț », iḥekkem s
lḥeqq, yețnaɣ ɣef tideț.

<span id="an_19:12">19:12</span> Allen-is am uḥeǧǧaǧu n tmes ; yesɛa
aṭas n tɛeṣṣabin ɣef wuqeṛṛuy-is. Yesɛa yiwen yisem yiwen ur t-yessin
anagar nețța.

<span id="an_19:13">19:13</span> Yelsa abeṛnus yeččuṛ d idammen. Ma d
nețța isem-is : « Awal n Ṛebbi » .

<span id="an_19:14">19:14</span> Lɛeskeṛ n igenwan lsan llebsa tamellalt
n lfina yețfeǧǧiǧen yerna ttabaɛen-t ɣef yiɛewdiwen imellalen.

<span id="an_19:15">19:15</span> Yeffeɣ-ed seg imi-s yiwen ujenwi
qeṭṭiɛen, iwakken ad yewwet leǧnas ; ad yeḥkem fell-asen s tɛekkazt n
wuzzal. A ten-yeṛkeḍ am akken ṛekkḍen tiẓurin zdaxel n lemɛinsṛa, akka
ara d-ibeggen urrif n Sidi Ṛebbi Bab n tezmert.

<span id="an_19:16">19:16</span> Yura ɣef wubeṛnus-is akk-d umeṣṣaḍ-is :
« Agellid n igelliden, Ssid n Ssyadi. »

<span id="an_19:17">19:17</span> Walaɣ daɣen yiwen lmelk ibedd ɣef
yiṭij, yessawel s taɣect eɛlayen i leḍyuṛ meṛṛa yețferfiren deg igenni :
Aset-ed, nnejmaɛet-ed aț țeččem imensi ameqqran i d-ihegga Sidi Ṛebbi,

<span id="an_19:18">19:18</span> iwakken aț-țeččem aksum n igelliden,
aksum n lɛeskeṛ d wid iḥekkmen fell-asen, aksum n iɛewdiwen d wid i
ten-irekben, aksum n yemdanen, ama d iḥeṛṛiyen ama d aklan, s umeqqran s
umeẓyan.

<span id="an_19:19">19:19</span> Walaɣ leɛqiṛa akk-d igelliden n ddunit
snejmaɛen-d lɛeskeṛ-nsen iwakken ad nnaɣen d Win irekben ɣef wuɛewdiw
akk-d lɛeskeṛ-is.

<span id="an_19:20">19:20</span> Leɛqiṛa tețwaṭṭef nețțat d nnbi nni n
lekdeb i gxeddmen lbeṛhanat seddaw n lḥekma-s, s wayes ikellex imdanen
meṛṛa yețwaḍebɛen s ṭṭabeɛ n leɛqiṛa yerna țseǧǧiden zdat lmeṣnuɛ ines.
?waḍeggṛen i sin d imuddiren ɣer temda tameqqrant n tmes i deg iṛeqq
ukubri.

<span id="an_19:21">19:21</span> Wiyaḍ mmuten s ujenwi-nni i d-yeffɣen
seg yimi n Win irekben aɛewdiw, leḍyuṛ n igenni ččan seg uksum-nsen armi
ṛwan.

<span id="an_20:1">20:1</span> Walaɣ daɣen yiwen lmelk iṣubb-ed seg
igenni, yewwi-d deg ufus-is tasaruț n tesraft lqayen akk-d yiwet ssnesla
tameqqrant.

<span id="an_20:2">20:2</span> yeṭṭef-ed llafɛa yellan d azrem nni n
zik, yellan d Cciṭan d Iblis, yeqqen-iț s ssnesla iwakken a t yeḥbes
alef iseggasen.

<span id="an_20:3">20:3</span> Iḍeggeṛ-it ɣer tesraft-nni lqayen,
yerra-yas aɣummu izemmem-iț iwakken ur ițkellix ara leǧnas alamma ɛeddan
walef iseggasen, imiren a d-yețwaserreḥ i kra n lweqt.

<span id="an_20:4">20:4</span> Mbeɛd ayagi, walaɣ ikursiyen n lḥekma,
yețțunefk i wid yeqqimen fell-asen ad ḥasben ; walaɣ leṛwaḥ n wid iwumi
țwakksen iqeṛṛay ɣef ddemma n tideț n Ɛisa Lmasiḥ akk-d wawal n Sidi
Ṛebbi. Walaɣ daɣen wid ur neqbil ara ad seǧǧden zdat leɛqiṛa akk-d
lmeṣnuɛ-is, wid ur neqbil ara ticṛeṭ-ines ɣef wunyir-nsen neɣ ɣef
yifassen-nsen. Uɣalen-d ɣer tudert, ad ḥekmen akk-d Lmasiḥ alef
iseggasen.

<span id="an_20:5">20:5</span> D wagi i d ḥeggu amezwaru. Ma d lmegtin
nniḍen ur d-țuɣalen ara ɣer tudert alamma ɛeddan walef iseggasen.

<span id="an_20:6">20:6</span> Amarezg-nsen wid ara d-iḥyun d imezwura ;
lmut tis snat ur sen tzemmer ara, ad ilin d lmuqedmin n Sidi Ṛebbi akk-d
Lmasiḥ, ad ḥekmen akk-d Lmasiḥ alef n iseggasen.

<span id="an_20:7">20:7</span> M'ara ɛeddin walef iseggasen agi, Cciṭan
a d-yețwaserreḥ si lḥebs,

<span id="an_20:8">20:8</span> iwakken ad ikellex leǧnas meṛṛa yellan di
ṛebɛa yixfawen n ddunit, Haǧuǧ Maǧuǧ yeɛni wid yellan d iɛdawen n Ṛebbi,
a ten-id-ijmeɛ i umenɣi ț-țirebbaɛ ur nezmir a neḥseb, am ṛṛmel n
lebḥeṛ.

<span id="an_20:9">20:9</span> Steɛmṛen akk ddunit, zzin i wegdud n Sidi
Ṛebbi ț-țemdint i gḥemmel. Lameɛna yeɣli-d fell-asen uḥeǧǧaǧu n tmes seg
igenni yečča ten.

<span id="an_20:10">20:10</span> Imiren yețwaḍeggeṛ Cciṭan-nni i
ten-iɣuṛṛen ɣer daxel n temda-nni n tmes d ukubri anda tella leɛqiṛa d
nnbi n lekdeb, ad țwaɛedben am yiḍ am wass i dayem.

<span id="an_20:11">20:11</span> Walaɣ daɣen yiwen ukersi n lḥekma
ɛlayen d amellal akk-d Win yeqqimen fell-as, igenni d lqaɛa rewlen
zdat-es ur sen-d-yeqqim ara wemkan.

<span id="an_20:12">20:12</span> Dɣa walaɣ lmegtin bedden zdat
ukersi-nni n lḥekma am umeqqran am umeẓyan, ldint tektabin, teldi daɣen
tektabt nniḍen yellan ț-țaktabt n tudert. Lmegtin țțuḥasben mkul yiwen
ɣef leḥsab n wayen yexdem d wayen yuran di tektabin-agi.

<span id="an_20:13">20:13</span> Lebḥeṛ iḍeggeṛ-ed akk wid yečča, lmut d
laxeṛt serrḥent-ed akk i lmegtin. Mkul yiwen yețțuḥaseb akken llan
lecɣal-is.

<span id="an_20:14">20:14</span> Lmut d laxeṛt țwaḍeggṛent ɣer temda n
tmes, țamda-agi n tmes i d lmut tis snat.

<span id="an_20:15">20:15</span>  Kra n win ur nețwajerred ara di
tektabt n tudert yețwaḍeggeṛ ɣer temda n tmes.

<span id="an_21:1">21:1</span>  Walaɣ igenni d lqaɛa ijdiden, igenni
aqdim d lqaɛa taqdimt xfan, lebḥur ur d-qqimen ara.

<span id="an_21:2">21:2</span> Walaɣ tamdint iqedsen tṣubb-ed seg igenni
tekka-d s ɣuṛ Sidi Ṛebbi, ț-țagi i ț-țamdint n Lquds tajdiṭ, thegga am
teslit icebbḥen iman-is i yesli.

<span id="an_21:3">21:3</span> Sliɣ i yiwet taɣect tenṭeq-ed s ṣṣut
eɛlayen seg ukersi-nni n lḥekma tenna-d : Tura Sidi Ṛebbi yezdeɣ ger
yemdanen, nutni ad ilin d agdud-is nețța s yiman-is ad yili yid-sen.

<span id="an_21:4">21:4</span>  A sen-yesfeḍ imeṭṭawen, ur tețțili lmut,
ur yețțili leḥzen, ur yețțili umeǧǧed ur yețțili lqeṛḥ axaṭer leḥwayeǧ
timezwura ɛeddant ur țɛawadent ara a d-țuɣalent.

<span id="an_21:5">21:5</span> Win akken yeqqimen ɣef wukersi n lḥekma
yenna : Tura rriɣ-ed kullec d ajdid. Yenna daɣen : Aru imeslayen-agi,
axaṭer imeslayen-iw ṣeḥḥan yerna deg-sen lețkal.

<span id="an_21:6">21:6</span> Yerna yenna-yi-d : Dayen ! Kullec
yețwakemmel ! D nekk i d alfa d nekk i d umiga yeɛni d nekk i d lḥeṛf
amezwaru, d nekk i d lḥeṛf aneggaru. Lliɣ si tazwara alamma ț-țaggara.
Win iffuden, a s-d-fkeɣ aman si lɛinseṛ n tudert yerna baṭel.

<span id="an_21:7">21:7</span> Win yeṭṭfen deg-i ur yeɣli ara a t-kafiɣ
s wayen yuklal ; nekk ad iliɣ d Illu-ines, nețța ad yili d mmi ;

<span id="an_21:8">21:8</span> ma d ixeddaɛen, wid ur nețțamen ara,
imejhal akk-d wid ineqqen timegṛaḍ, wid ixeddmen ticmatin akk d
iseḥḥaren, wid iɛebden lmeṣnuɛat d ssadaț, imeskaddaben, wigi meṛṛa ad
yili umur-nsen di temda n tmes ireqqen d ukebri. ?-țagi i d lmut tis
snat.

<span id="an_21:9">21:9</span> Yiwen si sebɛa lmalayekkat-nni yeṭṭfen
sebɛa teqbucin yeččuṛen d sebɛa lmuṣibat tineggura yusa-d yenna-yi-d :
Eyya-d a k-d-sekneɣ tislit, tameṭṭut n Izimer.

<span id="an_21:10">21:10</span> Imiren irfed-iyi Ṛṛuḥ n Sidi Ṛebbi,
ufiɣ-ed iman-iw ɣef yiwen wedrar ɛlayen. Isken-iyi-d tamdint n Lquds
tețṣubbu-d seg igenni s ɣuṛ Sidi Ṛebbi.

<span id="an_21:11">21:11</span> Tecceɛceɛ am lyaman yeṣfan yețṛeqṛiqen
am djaj, ț-țamanegt n Sidi Ṛebbi i gersen fell-as.

<span id="an_21:12">21:12</span> Zzin-as-d leswaṛ ɛlayen yesɛan tnac
tewwura, mkul yiwet ibedd zdat-es yiwen lmelk, uran ɣef tewwura-agi
yismawen n tnac leɛṛac n wat Isṛail.

<span id="an_21:13">21:13</span> Tlata tewwura wehhant ɣer cceṛq, tlata
ɣer lɣeṛb, tlata ɣer usammer, tlata nniḍen ɣer umalu.

<span id="an_21:14">21:14</span> Leswaṛ-nni ersen ɣef tnac lsisan i ɣef
uran tnac yismawen n ṛṛusul n Izimer n Ṛebbi.

<span id="an_21:15">21:15</span> Win akken i yi-d-ițmeslayen yeṭṭef
aɣanim n ddheb iwakken ad yektil tamdint, tiwwura-ines d leswaṛ-is.

<span id="an_21:16">21:16</span> Tuɣzi ț-țehri n leḥyuḍ n temdint-nni
ɛedlen. Yektal tamdint-nni, yufa lqis n tnac alef n wannaren ama di
tuɣzi-ines, ama di tehri-ines, ama di leɛli-ines.

<span id="an_21:17">21:17</span> Lmelk yektal daɣen leɛli n leswaṛ n
temdint s uɣanim-nni, yufa meyya uṛebɛa uṛebɛin alef n iɣallen.

<span id="an_21:18">21:18</span> Leswaṛ-is bnan s lyaman, ma ț-țamdint
tebna s ddheb iṣfan yecban djaj yețṛeqṛiqen.

<span id="an_21:19">21:19</span> Lsisan n leswaṛ-is ṛeqqmen s yezṛa
ɣlayen n mkul ṣṣenf. Amezwaru s Lyaman n Jasb, wis sin n Safir, wis

<span id="an_21:23">21:23</span> Tamdint-agi ur teḥwaǧ iṭij, ur teḥwaǧ

<span id="an_22:1">22:1</span> Lmelk-nni yesken-iyi-d daɣen asif anda

<span id="an_22:2">22:2</span> Di tlemmast n ubṛaḥ n temdint tella

<span id="an_22:5">22:5</span> Ur yețțuɣal a d-yili ṭṭlam ur țeḥwiǧin

yiman-is ara ten-inewwṛen s tafat-is, yerna ad ḥekmen yid-es i dayem.

<span id="an_22:6">22:6</span> Lmelk-nni yenna-yi-d : Imeslayen-agi, d

ara a d-yedṛu.

zdat-i axaṭer nekk d aqeddac n Sidi Ṛebbi am kečč am atmaten-ik lenbiya
akk-d wid iḥerzen imeslayen n tektabt-agi, d Sidi Ṛebbi kan ara tɛebdeḍ
!

<span id="an_22:10">22:10</span> Yenna-d daɣen : Ur țeffer ara imeslayen

<span id="an_22:11">22:11</span> Win ixeddmen lbaṭel ad ikemmel di

<span id="an_22:12">22:12</span> Atan ihi qṛib a n-aseɣ, ad fkeɣ i mkul
