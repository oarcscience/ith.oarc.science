### Amharic Bible: 1 Timothy

<span id="an_1:1">1:1</span> መድኃኒታችን እግዚአብሔር ተስፋችንም ኢየሱስ ክርስቶስ እንዳዘዘው
ትእዛዝ የኢየሱስ ክርስቶስ ሐዋርያ የሆነ ጳውሎስ፥

<span id="an_1:2">1:2</span> በእምነት እውነተኛ ልጄ ለሆነ ለጢሞቴዎስ፤ ከእግዚአብሔር ከአባታችን
ከክርስቶስ ኢየሱስም ከጌታችን ጸጋና ምሕረት ሰላምም ይሁን።

<span id="an_1:3">1:3</span> ወደ መቄዶንያ በሄድሁ ጊዜ፥ አንዳንዶች ልዩ ትምህርትን
እንዳያስተምሩና ወደ ተረት መጨረሻም ወደሌለው ወደ ትውልዶች ታሪክ እንዳያደምጡ ልታዛቸው በኤፌሶን ትቀመጥ ዘንድ
ለመንሁህ፤ እንደ እነዚህ ያሉ ነገሮች ክርክርን ያመጣሉና በእምነት ግን ላለ ለእግዚአብሔር መጋቢነት አይጠቅሙም።

<span id="an_1:5">1:5</span> የትእዛዝ ፍጻሜ ግን ከንጹሕ ልብና ከበጎ ሕሊና ግብዝነትም ከሌለበት
እምነት የሚወጣ ፍቅር ነው፤

<span id="an_1:6">1:6</span> ከእነዚህም አንዳንዶች ስተው፥ የሚሉትን ወይም ስለ እነርሱ አስረግጠው
የሚናገሩትን ሳያስተውሉ፥ የሕግ አስተማሪዎች ሊሆኑ እየወደዱ፥ ወደ ከንቱ ንግግር ፈቀቅ ብለዋል።

<span id="an_1:8">1:8</span> ነገር ግን ሰው እንደሚገባ ቢሰራበት ሕግ መልካም እንደ ሆነ
እናውቃለን፤

<span id="an_1:9">1:9</span> ይኸውም፥ ለበደለኞችና ለማይታዘዙ፥ ለዓመጸኞችና ለኃጢአተኞች፥ ቅድስና
ለሌላቸውና ለርኵሳን፥ አባትና እናትን ለሚገድሉ፥ ለነፍሰ ገዳዮችና ለሴሰኞች፥ ከወንድ ጋርም ለሚተኙ፥ በሰዎችም
ለሚነግዱ፥ ለውሸተኞችም በውሸትም ለሚምሉ፥ የብሩክ እግዚአብሔርንም ክብር እንደሚገልጥ ለእኔ አደራ እንደ ተሰጠኝ
ወንጌል የሆነውን ደኅና ትምህርት ለሚቃወም ለሌላ ነገር ሁሉ ሕግ እንደ ተሰራ እንጂ ለጻድቅ እንዳልተሰራ ሲያውቅ
ነው።

<span id="an_1:12">1:12</span> ለአገልግሎቱ ሾሞኝ ታማኝ አድርጎ ስለ ቈጠረኝ፥ ኃይል የሰጠኝን
ክርስቶስ ኢየሱስን ጌታችንን አመሰግናለሁ፤

<span id="an_1:13">1:13</span> አስቀድሞ ተሳዳቢና አሳዳጅ አንገላችም ምንም ብሆን፥ ይህን
አደረገልኝ፤ ነገር ግን ሳላውቅ ባለማመን ስላደረግሁት ምህረትን አገኘሁ፥

<span id="an_1:14">1:14</span> የጌታችንም ጸጋ በክርስቶስ ኢየሱስ ካሉ ከእምነትና ከፍቅር ጋር
አብልጦ በዛ።

<span id="an_1:15">1:15</span> ኃጢአተኞችን ሊያድን ክርስቶስ ኢየሱስ ወደ ዓለም መጣ የሚለው ቃል
የታመነና ሁሉ እንዲቀበሉት የተገባ ነው፤ ከኃጢአተኞችም ዋና እኔ ነኝ፤

<span id="an_1:16">1:16</span> ስለዚህ ግን፥ የዘላለምን ሕይወት ለማግኘት በእርሱ ያምኑ ዘንድ
ላላቸው ሰዎች ምሳሌ እንድሆን፥ ኢየሱስ ክርስቶስ ዋና በምሆን በእኔ ላይ ትዕግስቱን ሁሉ ያሳይ ዘንድ፥ ምህረትን
አገኘሁ።

<span id="an_1:17">1:17</span> ብቻውን አምላክ ለሚሆን ለማይጠፋው ለማይታየውም ለዘመናት ንጉሥ
ምስጋናና ክብር እስከ ዘላለም ድረስ ይሁን፤ አሜን።

<span id="an_1:18">1:18</span> ጢሞቴዎስ ልጄ ሆይ፥ አስቀድሞ ስለ አንተ እንደ ተነገረው ትንቢት፥
በእርሱ መልካም ጦርነት ትዋጋ ዘንድ ይህችን ትእዛዝ አደራ እሰጥሃለሁ፤

<span id="an_1:19">1:19</span> እምነትና በጎ ሕሊና ይኑርህ፥ አንዳንዶች ሕሊናን ጥለው፥ መርከብ
አለ መሪ እንደሚጠፋ፥ በእምነት ነገር ጠፍተዋልና፤

<span id="an_1:20">1:20</span> ከእነዚያም፥ እንዳይሳደቡ ይማሩ ዘንድ ለሰይጣን አሳልፌ
የሰጠኋቸው፥ ሄሜኔዎስና እስክንድሮስ ናቸው።

<span id="an_2:1">2:1</span> እንግዲህ እግዚአብሔርን በመምሰልና በጭምትነት ሁሉ ጸጥና ዝግ ብለን
እንድንኖር፥ ልመናና ጸሎት ምልጃም ምስጋናም ስለ ሰዎች ሁሉ ስለ ነገሥታትና ስለ መኳንንትም ሁሉ እንዲደረጉ ከሁሉ
በፊት እመክራለሁ።

<span id="an_2:3">2:3</span> ሰዎች ሁሉ ሊድኑና እውነቱን ወደ ማወቅ ሊደርሱ በሚወድ በእግዚአብሔር
በመድኃኒታችን ፊት መልካምና ደስ የሚያሰኝ ይህ ነው።

<span id="an_2:5">2:5</span> አንድ እግዚአብሔር አለና፥ በእግዚአብሔርና በሰውም መካከል ያለው
መካከለኛው ደግሞ አንድ አለ፥ እርሱም ሰው የሆነ ክርስቶስ ኢየሱስ ነው፤

<span id="an_2:6">2:6</span> ራሱንም ለሁሉ ቤዛ ሰጠ፥ ይህም በገዛ ዘመኑ ምስክርነቱ ነበረ፤

<span id="an_2:7">2:7</span> እኔም ለዚህ ነገር አዋጅ ነጋሪና ሐዋርያ በእምነትና በእውነትም
የአሕዛብ አስተማሪ ለመሆን ተሾምሁ፤ እውነት እናገራለሁ፤ አልዋሽም።

<span id="an_2:8">2:8</span> እንግዲህ ወንዶች በስፍራ ሁሉ አለ ቍጣና አለ ክፉ አሳብ የተቀደሱትን
እጆች እያነሱ እንዲጸልዩ እፈቅዳለሁ።

<span id="an_2:9">2:9</span> እንዲሁም ደግሞ ሴቶች በሚገባ ልብስ ከእፍረትና ራሳቸውን ከመግዛት
ጋር ሰውነታቸውን ይሸልሙ፤ እግዚአብሔርን እንፈራለን ለሚሉት ሴቶች እንደሚገባ፥ መልካም በማድረግ እንጂ በሽሩባና
በወርቅ ወይም በዕንቍ ወይም ዋጋው እጅግ በከበረ ልብስ አይሸለሙ።

<span id="an_2:11">2:11</span> ሴት በነገር ሁሉ እየተገዛች በዝግታ ትማር፤

<span id="an_2:12">2:12</span> ሴት ግን በዝግታ ትኑር እንጂ ልታስተምር ወይም በወንድ ላይ
ልትሰለጥን አልፈቅድም።

<span id="an_2:13">2:13</span> አዳም ቀድሞ ተፈጥሮአልና፥ በኋላም ሔዋን ተፈጠረች።

<span id="an_2:14">2:14</span> የተታለለም አዳም አይደለም፥ ሴቲቱ ግን ተታልላ በመተላለፍ
ወደቀች፤

<span id="an_2:15">2:15</span> ነገር ግን በእምነትና በፍቅር በቅድስናም ራሳቸውን እየገዙ ቢኖሩ
በመውለድ ትድናለች።

<span id="an_3:1">3:1</span> ማንም ኤጲስ ቆጶስነትን ቢፈልግ መልካምን ሥራ ይመኛል የሚለው ቃል
የታመነ ነው።

<span id="an_3:2">3:2</span> እንግዲህ ኤጲስ ቆጶስ እንዲህ ሊሆን ይገባዋል፤ የማይነቀፍ፥ የአንዲት
ሚስት ባል፥ ልከኛ፥ ራሱን የሚገዛ፥ እንደሚገባው የሚሰራ፥ እንግዳ ተቀባይ፥ ለማስተማር የሚበቃ፥

<span id="an_3:3">3:3</span> የማይሰክር፥ የማይጨቃጨቅ ነገር ግን ገር የሆነ፥ የማይከራከር፥
ገንዘብን የማይወድ፥

<span id="an_3:4">3:4</span> ልጆቹን በጭምትነት ሁሉ እየገዛ የራሱን ቤት በመልካም የሚያስተዳድር፤

<span id="an_3:5">3:5</span> ሰው ግን የራሱን ቤት እንዲያስተዳድር ባያውቅ፥ የእግዚአብሔርን ቤተ
ክርስቲያን እንዴት ይጠብቃታል?

<span id="an_3:6">3:6</span> በትዕቢት ተነፍቶ በዲያቢሎስ ፍርድ እንዳይወድቅ፥ አዲስ ክርስቲያን
አይሁን።

<span id="an_3:7">3:7</span> በዲያቢሎስ ነቀፋና ወጥመድም እንዳይወድቅ፥ በውጭ ካሉት ሰዎች ደግሞ
መልካም ምስክር ሊኖረው ይገባዋል።

<span id="an_3:8">3:8</span> እንዲሁም ዲያቆናት ጭምቶች፥ በሁለት ቃል የማይናገሩ፥ ለብዙ ወይን
ጠጅ የማይጎመጁ፥

<span id="an_3:9">3:9</span> ነውረኛ ረብ የማይወዱ፥ በንጹሕ ሕሊና የሃይማኖትን ምሥጢር የሚይዙ
ሊሆኑ ይገባቸዋል።

<span id="an_3:10">3:10</span> እነዚህም ደግሞ አስቀድሞ ይፈተኑ፥ ከዚያም በኋላ ያለ ነቀፋ ቢሆኑ
በዲቁና ሥራ ያገልግሉ።

<span id="an_3:11">3:11</span> እንዲሁም ሴቶች ጭምቶች፥ የማያሙ፥ ልከኞች፥ በነገር ሁሉ የታመኑ
ሊሆኑ ይገባቸዋል።

<span id="an_3:12">3:12</span> ዲያቆናት ልጆቻቸውንና የራሳቸውን ቤቶች በመልካም እየገዙ
እያንዳንዳቸው የአንዲት ሴት ባሎች ይሁኑ።

<span id="an_3:13">3:13</span> በዲቁና ሥራ በመልካም ያገለገሉ ለራሳቸው ትልቅ ማዕርግና
በክርስቶስ ኢየሱስ ባለው እምነት ብዙ ድፍረት ያገኛሉ።

<span id="an_3:14">3:14</span> ፈጥኜ ወደ አንተ እንድመጣ ተስፋ አድርጌ ይህን እጽፍልሃለሁ።

<span id="an_3:15">3:15</span> ብዘገይ ግን፥ በእግዚአብሔር ማደሪያ ቤት መኖር እንዴት እንደሚገባ
ታውቅ ዘንድ እጽፍልሃለሁ፤ ቤቱም የእውነት ዓምድና መሠረት፥ የሕያው እግዚአብሔር ቤተ ክርስቲያን ነው።

<span id="an_3:16">3:16</span> እግዚአብሔርንም የመምሰል ምሥጢር ያለ ጥርጥር ታላቅ ነው፤ በሥጋ
የተገለጠ፥ በመንፈስ የጸደቀ፥ ለመላእክት የታየ፥ በአሕዛብ የተሰበከ፥ በዓለም የታመነ፥ በክብር ያረገ።

<span id="an_4:1">4:1</span> መንፈስ ግን በግልጥ። በኋለኞች ዘመናት አንዳንዶች የሚያስቱ
መናፍስትንና በውሸተኞች ግብዝነት የተሰጠውን የአጋንንትን ትምህርት እያደመጡ፥ ሃይማኖትን ይክዳሉ ይላል፤ በገዛ
ሕሊናቸው እንደሚቃጠሉ ደንዝዘው፥

<span id="an_4:3">4:3</span> እነዚህ ውሸተኞች መጋባትን ይከለክላሉ፥ አምነውም እውነትን የሚያውቁ
ከምስጋና ጋር ይቀበሉ ዘንድ እግዚአብሔር ከፈጠረው መብል እንዲርቁ ያዝዛሉ።

<span id="an_4:4">4:4</span> እግዚአብሔር የፈጠረው ሁሉ መልካም ነውና፥ ከምስጋናም ጋር ቢቀበሉት
የሚጣል ምንም የለም፤

<span id="an_4:5">4:5</span> በእግዚአብሔር ቃልና በጸሎት የተቀደሰ ነውና።

<span id="an_4:6">4:6</span> ስለዚህ ወንድሞችን ብታሳስብ፥ በእምነትና በተከተልኸው በመልካም
ትምህርት ቃል የምትመገብ የኢየሱስ ክርስቶስ መልካም አገልጋይ ትሆናለህ።

<span id="an_4:7">4:7</span> ነገር ግን ለዚህ ዓለም ከሚመችና የአሮጊቶችን ሴቶች ጨዋታ ከሚመስለው
ተረት ራቅ። እግዚአብሔርን ለመምሰል ግን ራስህን አስለምድ።

<span id="an_4:8">4:8</span> ሰውነትን ለሥጋዊ ነገር ማስለመድ ለጥቂት ይጠቅማልና፤ እግዚአብሔርን
መምሰል ግን የአሁንና የሚመጣው ሕይወት ተስፋ ስላለው፥ ለነገር ሁሉ ይጠቅማል።

<span id="an_4:9">4:9</span> ይህ ቃል የታመነ ሁሉም እንዲቀበሉት የተገባ ነው፤

<span id="an_4:10">4:10</span> ይህን ለማግኘት እንደክማለንና፥ ስለዚህም እንሰደባለን፤ ይህም
ሰውን ሁሉ ይልቁንም የሚያምኑትን በሚያድን በሕያው አምላክ ተስፋ ስለምናደርግ ነው።

<span id="an_4:11">4:11</span> ይህን እዘዝና አስተምር። በቃልና በኑሮ በፍቅርም በእምነትም
በንጽሕናም ለሚያምኑቱ ምሳሌ ሁን እንጂ፥ ማንም ታናሽነትህን አይናቀው።

<span id="an_4:13">4:13</span> እስክመጣ ድረስ ለማንበብና ለመምከር ለማስተማርም ተጠንቀቅ።

<span id="an_4:14">4:14</span> በትንቢት ከሽማግሌዎች እጅ መጫን ጋር የተሰጠህን፥ በአንተ ያለውን
የጸጋ ስጦታ ቸል አትበል።

<span id="an_4:15">4:15</span> ማደግህ በነገር ሁሉ እንዲገለጥ ይህን አስብ፥ ይህንም አዘውትር።

<span id="an_4:16">4:16</span> ለራስህና ለትምህርትህ ተጠንቀቅ፥ በእነዚህም ጽና፤ ይህን
ብታደርግ፥ ራስህንም የሚሰሙህንም ታድናለህና።

<span id="an_5:1">5:1</span> ሽማግሌ የሆነውን አትገሥጸው፥ እርሱን ግን እንደ አባት፥ ጎበዞችን
እንደ ወንድሞች፥ የሸመገሉትን ሴቶች እንደ እናቶች፥ ቆነጃጅቱን እንደ እኅቶች በፍጹም ንጽሕና ለምናቸው።

<span id="an_5:3">5:3</span> በእውነት ባልቴቶች የሆኑትን ባልቴቶች አክብር።

<span id="an_5:4">5:4</span> ማንም ባልቴት ግን ልጆች ወይም የልጅ ልጆች ቢኖሩአት፥ እነርሱ
አስቀድመው ለገዛ ቤተ ሰዎቻቸው እግዚአብሔርን መምሰል ያሳዩ ዘንድ፥ ለወላጆቻቸውም ብድራትን ይመልሱላቸው ዘንድ
ይማሩ፤ ይህ በእግዚአብሔር ፊት መልካምና የተወደደ ነውና።

<span id="an_5:5">5:5</span> ብቻዋንም ኖራ በእውነት ባልቴት የምትሆን በእግዚአብሔር ተስፋ
ታደርጋለች፥ ሌትና ቀንም በልመና በጸሎትም ጸንታ ትኖራለች፤

<span id="an_5:6">5:6</span> ቅምጥሊቱ ግን በሕይወትዋ ሳለች የሞተች ናት።

<span id="an_5:7">5:7</span> ያለ ነቀፋ እንዲሆኑ ይህን ደግሞ እዘዝ።

<span id="an_5:8">5:8</span> ነገር ግን ለእርሱ ስለ ሆኑት ይልቁንም ስለ ቤተ ሰዎቹ የማያስብ
ማንም ቢሆን፥ ሃይማኖትን የካደ ከማያምንም ሰው ይልቅ የሚከፋ ነው።

<span id="an_5:9">5:9</span> ባልቴት በመዝገብ ብትጻፍ ዕድሜዋ ከስድሳ ዓመት እንዳያንስ፥ የአንድም
ባል ሚስት የነበረች እንድትሆን ያስፈልጋል፤

<span id="an_5:10">5:10</span> ልጆችን በማሳደግ እንግዶችንም በመቀበል፥ የቅዱሳንንም እግሮች
በማጠብ፥ የተጨነቁትንም በመርዳት በጎንም ሥራ ሁሉ በመከተል፥ ይህን መልካም ሥራ በማድረግ የተመሰከረላት ልትሆን
ይገባል።

<span id="an_5:11">5:11</span> ባል የሞተባቸውን ቆነጃጅት ግን አትቀበል፤ በክርስቶስ ላይ
ተነሥተው ሲቀማጠሉ ሊያገቡ ይወዳሉና፤

<span id="an_5:12">5:12</span> የፊተኛውንም እምነታቸውን ስለ ናቁ ይፈረድባቸዋል፤

<span id="an_5:13">5:13</span> ከዚህም ጋር ቤት ለቤት እየዞሩ ሥራን መፍታት ደግሞ ይማራሉ፥
የማይገባውንም እየተናገሩ ለፍላፊዎችና በነገር ገቢዎች ይሆናሉ እንጂ፥ ሥራ ፈቶች ብቻ አይደሉም።

<span id="an_5:14">5:14</span> እንግዲህ ቆነጃጅት ሊያገቡ፥ ልጆችንም ሊወልዱ፥ ቤቶቻቸውንም
ሊያስተዳድሩ፥ ተቃዋሚውም የሚሳደብበትን አንድን ምክንያት ስንኳ እንዳይሰጡ እፈቅዳለሁ፤

<span id="an_5:15">5:15</span> ከአሁን በፊት አንዳንዶች ሰይጣንን እንዲከተሉ ፈቀቅ ብለዋልና።

<span id="an_5:16">5:16</span> ባልቴቶች ያሉት የሚያምን ቢሆን ወይም የምታምን ብትሆን፥
ይርዱአቸው፥ ቤተ ክርስቲያንም እውነተኞችን ባልቴቶች እንድትረዳ አይክበዱባት።

<span id="an_5:17">5:17</span> በመልካም የሚያስተዳድሩ ሽማግሌዎች፥ ይልቁንም በመስበክና
በማስተማር የሚደክሙት፥ እጥፍ ክብር ይገባቸዋል።

<span id="an_5:18">5:18</span> መጽሐፍ። የሚያበራየውን በሬ አፉን አትሰር፥ ደግሞ። ለሠራተኛ
ደመወዙ ይገባዋል ይላልና።

<span id="an_5:19">5:19</span> ከሁለት ወይም ከሦስት ምስክር በቀር በሽማግሌ ላይ ክስ አትቀበል።

<span id="an_5:20">5:20</span> ሌሎቹ ደግሞ እንዲፈሩ፥ ኃጢአት የሚሰሩትን በሁሉ ፊት ገሥጻቸው።

<span id="an_5:21">5:21</span> አንድን እንኳ በአድልዎ ሳታደርግ፥ እነዚህን ያለ መዘንበል
እንድትጠብቅ በእግዚአብሔርና በጌታ በኢየሱስ ክርስቶስ በተመረጡትም መላእክት ፊት እመክርሃለሁ።

<span id="an_5:22">5:22</span> በማንም ላይ ፈጥነህ እጆችህን አትጫን፥ በሌሎችም ኃጢአት
አትተባበር፤ ራስህን በንጽህና ጠብቅ።

<span id="an_5:23">5:23</span> ስለ ሆድህና ስለ በሽታህ ብዛት ጥቂት የወይን ጠጅ ጠጣ እንጂ፥
ወደ ፊት ውኃ ብቻ አትጠጣ።

<span id="an_5:24">5:24</span> የአንዳንዶች ሰዎች ኃጢአት የተገለጠ ነው ፍርድንም ያመለክታል፥
ሌሎችን ግን ይከተላቸዋል፤

<span id="an_5:25">5:25</span> እንዲሁ መልካም ሥራ ደግሞ የተገለጠ ነው፥ ያልተገለጠም ከሆነ
ሊሰወር አይችልም።

<span id="an_6:1">6:1</span> የእግዚአብሔር ስምና ትምህርቱ አንዳይሰደብ፥ ከቀንበር በታች ያሉቱ
ባሪያዎች ሁሉ ለገዙአቸው ጌቶች ክብር ሁሉ እንደ ተገባቸው ይቍጠሩአቸው።

<span id="an_6:2">6:2</span> የሚያምኑም ጌቶች ያሉአቸው፥ ወንድሞች ስለ ሆኑ አይናቁአቸው፥ ነገር
ግን በመልካም ሥራቸው የሚጠቅሙ የሚያምኑና የተወደዱ ስለ ሆኑ፥ ከፊት ይልቅ ያገልግሉ።

<span id="an_6:3">6:3</span> እነዚህን አስተምርና ምከር። ማንም ልዩ ትምህርት የሚያስተምር
ከጌታችንም ከኢየሱስ ክርስቶስ በሆነው ጤናማ ቃልና እግዚአብሔርን ለመምሰል በሚስማማ ትምህርት የማይጠጋ ቢሆን፥

<span id="an_6:4">6:4</span> በትዕቢት ተነፍቶአል አንዳችም አያውቅም፥ ነገር ግን ምርመራን በቃልም
መዋጋትን እንደ በሽተኛ ይናፍቃል፤ ከእነዚህም ቅንዓትና ክርክር ስድብም ክፉ አሳብም እርስ በርስ መናደድም ይወጣሉ፥
አእምሮአቸውም በጠፋባቸው እውነትንም በተቀሙ፥ እግዚአብሔርን መምሰል ማትረፊያ የሚሆን በመሰላቸው ሰዎች ይገኛሉ።
እንደነዚህ ካሉት ራቅ።

<span id="an_6:6">6:6</span> ኑሮዬ ይበቃኛል ለሚለው ግን እግዚአብሔርን መምሰል እጅግ ማትረፊያ
ነው፤

<span id="an_6:7">6:7</span> ወደ ዓለም ምንም እንኳ አላመጣንምና፥

<span id="an_6:8">6:8</span> አንዳችንም ልንወስድ አይቻለንም፤ ምግብና ልብስ ከኖረን ግን፥ እርሱ
ይበቃናል።

<span id="an_6:9">6:9</span> ዳሩ ግን ባለ ጠጎች ሊሆኑ የሚፈልጉ በጥፋትና በመፍረስ ሰዎችን
በሚያሰጥምና በሚያሰንፍ በሚጎዳም በብዙ ምኞትና በፈተና በወጥመድም ይወድቃሉ።

<span id="an_6:10">6:10</span> ገንዘብን መውደድ የክፋት ሁሉ ሥር ነውና፥ አንዳንዶች ይህን
ሲመኙ፥ ከሃይማኖት ተሳስተው በብዙ ሥቃይ ራሳቸውን ወጉ።

<span id="an_6:11">6:11</span> አንተ ግን፥ የእግዚአብሔር ሰው ሆይ፥ ከዚህ ሽሽ፤ ጽድቅንና
እግዚአብሔርን መምሰል እምነትንም ፍቅርንም መጽናትንም የዋህነትንም ተከታተል።

<span id="an_6:12">6:12</span> መልካሙን የእምነት ገድል ተጋደል፥ የተጠራህለትንም በብዙም
ምስክሮች ፊት በመልካም መታመን የታመንህለትን የዘላለምን ሕይወት ያዝ።

<span id="an_6:13">6:13</span> ሁሉን በሕይወት በሚጠብቅ በእግዚአብሔር ፊት፥ በጴንጤናዊውም
በጲላጦስ ዘንድ መልካሙን መታመን በመሰከረ በክርስቶስ ኢየሱስ ፊት አዝሃለሁ፤

<span id="an_6:14">6:14</span> ጌታችን ኢየሱስ ክርስቶስ እስኪገለጥ ድረስ ያለ እድፍና ያለ ነቀፋ
ሆነህ ትእዛዙን ጠብቅ፤

<span id="an_6:15">6:15</span> ያንም መገለጡን በራሱ ጊዜ ብፁዕና ብቻውን የሆነ ገዥ፥ የነገሥታት
ንጉሥና የጌቶች ጌታ፥ ያሳያል።

<span id="an_6:16">6:16</span> እርሱ ብቻ የማይሞት ነው፤ ማንም ሊቀርበው በማይችል ብርሃን
ይኖራል፤ አንድ ሰው እንኳ አላየውም ሊያይም አይቻለውም፤ ለእርሱ ክብርና የዘላለም ኃይል ይሁን፤ አሜን።

<span id="an_6:17">6:17</span> በአሁኑ ዘመን ባለ ጠጎች የሆኑት የትዕቢትን ነገር እንዳያስቡ፥
ደስም እንዲለን ሁሉን አትርፎ በሚሰጠን በሕያው እግዚአብሔር እንጂ በሚያልፍ ባለ ጠግነት ተስፋ እንዳያደርጉ
እዘዛቸው።

<span id="an_6:18">6:18</span> እውነተኛውን ሕይወት ይይዙ ዘንድ፥ ለሚመጣው ዘመን ለራሳቸው
መልካም መሠረት የሚሆንላቸውን መዝገብ እየሰበሰቡ፥ መልካምን እንዲያደርጉ በበጎም ሥራ ባለ ጠጎች እንዲሆኑ፥ ሊረዱና
ሊያካፍሉም የተዘጋጁ እንዲሆኑ ምከራቸው።

<span id="an_6:20">6:20</span> ጢሞቴዎስ ሆይ፥ በውሸት እውቀት ከተባለ ለዓለም ከሚመች ከከንቱ
መለፍለፍና መከራከር እየራቅህ፥ የተሰጠህን አደራ ጠብቅ፤

<span id="an_6:21">6:21</span> ይህ እውቀት አለን ብለው፥ አንዳንዶች ስለ እምነት ስተዋልና። ጸጋ
ከአንተ ጋር ይሁን።
