### Amharic Bible: Titus

<span id="an_1:1">1:1</span> የእግዚአብሔር ባሪያና የኢየሱስ ክርስቶስ ሐዋርያ የሆነ ጳውሎስ፥
በእግዚአብሔር ለተመረጡት እምነት እንዲሆንላቸው በዘላለምም ሕይወት ተስፋ እንደ አምልኮት ያለውን እውነት እንዲያውቁ
የተላከ፤ ስለዚህም ሕይወት የማይዋሽ እግዚአብሔር ከዘላለም ዘመናት በፊት ተስፋ ሰጠ፥

<span id="an_1:3">1:3</span> በዘመኑም ጊዜ፥ መድኃኒታችን እግዚአብሔር እንዳዘዘ፥ ለእኔ አደራ
በተሰጠኝ ስብከት ቃሉን ገለጠ፤

<span id="an_1:4">1:4</span> በሃይማኖት ኅብረት እውነተኛ ልጄ ለሚሆን ለቲቶ፤ ከእግዚአብሔር አብ
ከመድኃኒታችንም ከጌታ ከኢየሱስ ክርስቶስ ጸጋና ምሕረት ሰላምም ይሁን።

<span id="an_1:5">1:5</span> ስለዚህ ምክንያት የቀረውን እንድታደራጅ በየከተማውም፥ እኔ አንተን
እንዳዘዝሁ፥ ሽማግሌዎችን እንድትሾም በቀርጤስ ተውሁህ፤

<span id="an_1:6">1:6</span> የማይነቀፍና የአንዲት ሚስት ባል የሚሆን፥ የሚያምኑም ልጆች ያሉት፥
ስለ መዳራትም ወይም ስለ አለመታዘዝ የማይከሰስ ማንም ቢኖር፥ ሹመው።

<span id="an_1:7">1:7</span> ኤጲስ ቆጶስ፥ እንደ እግዚአብሔር መጋቢ፥ የማይነቀፍ ሊሆን
ይገባዋልና፤ የማይኮራ፥ የማይቆጣ፥ የማይሰክር፥ የማይጨቃጨቅ፥ ነውረኛ ረብ የማይወድ፥

<span id="an_1:8">1:8</span> ነገር ግን እንግዳ ተቀባይ፥ በጎ የሆነውን ነገር የሚወድ፥ ጠንቃቃ፥
ጻድቅ፥ ቅዱስ፥ ራሱን የሚገዛ ይሁን፤

<span id="an_1:9">1:9</span> ሕይወት በሚገኝበት ትምህርት ደግሞ ሊመክር ተቃዋሚዎቹንም ሊወቅስ
ይችል ዘንድ፥ እንደተማረው በታመነ ቃል ይጽና።

<span id="an_1:10">1:10</span> የማይታዘዙና ከንቱ የሚናገሩ የሚያታልሉ ይልቁንም ከተገረዙት ወገን
የሚሆኑ ብዙ ናቸውና፤

<span id="an_1:11">1:11</span> እነዚህም ስለ ነውረኛ ረብ የማይገባውን እያስተማሩ ቤቶችን በሞላው
ስለ ሚገለብጡ፥ አፋቸውን መዝጋት ይገባል።

<span id="an_1:12">1:12</span> የገዛ ራሳቸው ነቢይ የሆነ ከእነርሱ አንዱ። የቀርጤስ ሰዎች
ዘወትር ውሸተኞች፥ ክፉዎች አራዊት፥ በላተኞች፥ ሥራ ፈቶች ናቸው ብሎአል።

<span id="an_1:13">1:13</span> ይህ ምስክር እውነተኛ ነው። ስለዚህ ምክንያት የአይሁድን ተረትና
ከእውነት ፈቀቅ የሚሉትን ሰዎች ትእዛዝ ሳያዳምጡ፥ በሃይማኖት ጤናሞች እንዲሆኑ በብርቱ ውቀሳቸው።

<span id="an_1:15">1:15</span> ሁሉ ለንጹሖች ንጹሕ ነው፤ ለርኵሳንና ለማያምኑ ግን ንጹሕ የሆነ
ምንም የለም፥ ነገር ግን አእምሮአቸውም ሕሊናቸውም ረክሶአል።

<span id="an_1:16">1:16</span> እግዚአብሔርን እንዲያውቁ በግልጥ ይናገራሉ፥ ዳሩ ግን
የሚያስጸይፉና የማይታዘዙ ለበጎ ሥራም ሁሉ የማይበቁ ስለ ሆኑ፥ በሥራቸው ይክዱታል።

<span id="an_2:1">2:1</span> አንተ ግን ሕይወት ለሚገኝበት ትምህርት የሚገባውን ተናገር።

<span id="an_2:2">2:2</span> ሽማግሌዎች ልከኞች፥ ጭምቶች፥ ራሳቸውን የሚገዙ፥ በእምነትና በፍቅር
በመጽናትም ጤናሞች እንዲሆኑ ምከራቸው፤

<span id="an_2:3">2:3</span> እንዲሁም አሮጊቶች ሴቶች አካሄዳቸው ለቅዱስ አገልግሎት የሚገባ፥
የማያሙ፥ ለብዙ ወይን ጠጅ የማይገዙ፥ በጎ የሆነውን ነገር የሚያስተምሩ ይሁኑ፤

<span id="an_2:4">2:4</span> ቆነጃጅትም የእግዚአብሔር ቃል እንዳይሰደብ፥ ባሎቻቸውን የሚወዱ፥
ልጆቻቸውን የሚወዱ፥ ራሳቸውን የሚገዙ፥ ንጹሖች፥ በቤት የሚሠሩ፥ በጎዎች፥ ለባሎቻቸው የሚታዘዙ እንዲሆኑ
ይምከሩአቸው።

<span id="an_2:6">2:6</span> ጎበዞችም እንዲሁ ራሳቸውን እንዲገዙ ምከራቸው።

<span id="an_2:7">2:7</span> የሚቃወም ሰው ስለ እኛ የሚናገረውን መጥፎ ነገር ሲያጣ እንዲያፍር፥
በነገር ሁሉ መልካምን በማድረግህ ምሳሌ የሚሆን ራስህን አሳይ፤ በትምህርትህም ደኅንነትን፥ ጭምትነትን፥ ጤናማና
የማይነቀፍ ንግግርን ግለጥ።

<span id="an_2:9">2:9</span> ባሪያዎች ስለ መድኃኒታችን ስለ እግዚአብሔር የሚሆነውን ትምህርት
በሁሉ ነገር ያስመሰግኑ ዘንድ፥ ለገዛ ጌቶቻቸው እንዲገዙ በሁሉም ደስ እንዲያሰኙ፥ ሳይቃወሙና ሳይሰርቁም በጎ
ታማኝነትን ሁሉ እንዲያሳዩ ምከራቸው።

<span id="an_2:11">2:11</span> ሰዎችን ሁሉ የሚያድን የእግዚአብሔር ጸጋ ተገልጦአልና፤

<span id="an_2:12">2:12</span> ይህም ጸጋ፥ ኃጢአተኝነትንና ዓለማዊን ምኞት ክደን፥ የተባረከውን
ተስፋችንን እርሱም የታላቁን የአምላካችንንና የመድኃኒታችንን የኢየሱስ ክርስቶስን ክብር መገለጥ እየጠበቅን፥
ራሳችንን በመግዛትና በጽድቅ እግዚአብሔርንም በመምሰል በአሁኑ ዘመን እንድንኖር ያስተምረናል፤

<span id="an_2:14">2:14</span> መድኃኒታችንም ከዓመፅ ሁሉ እንዲቤዠን፥ መልካሙንም ለማድረግ
የሚቀናውን ገንዘቡም የሚሆነውን ሕዝብ ለራሱ እንዲያነጻ፥ ስለ እኛ ነፍሱን ሰጥቶአል።

<span id="an_2:15">2:15</span> ይህን በሙሉ ሥልጣን ተናገርና ምከር ገሥጽም፤ ማንም አይናቅህ።

<span id="an_3:1">3:1</span> ለገዦችና ለባለ ሥልጣኖች የሚገዙና የሚታዘዙ፥ ለበጎ ሥራ ሁሉ
የተዘጋጁ፥ ማንንም የማይሰድቡ፥ የማይከራከሩ፥ ገሮች፥ ለሰው ሁሉ የዋህነትን ሁሉ የሚያሳዩ እንዲሆኑ ኣሳስባቸው።

<span id="an_3:3">3:3</span> እኛ ደግሞ አስቀድመን የማናስተውል ነበርንና፤ የማንታዘዝ፥ የምንስት፥
ለምኞትና ለልዩ ልዩ ተድላ እንደ ባሪያዎች የምንገዛ፥ በክፋትና በምቀኝነት የምንኖር፥ የምንጣላ፥ እርስ በርሳችን
የምንጠላላ ነበርን።

<span id="an_3:4">3:4</span> ነገር ግን የመድኃኒታችን የእግዚአብሔር ቸርነትና ሰውን መውደዱ
በተገለጠ ጊዜ፥

<span id="an_3:5">3:5</span> እንደ ምሕረቱ መጠን ለአዲስ ልደት በሚሆነው መታጠብና በመንፈስ ቅዱስ
በመታደስ አዳነን እንጂ፥ እኛ ስላደረግነው በጽድቅ ስለ ነበረው ሥራ አይደለም፤

<span id="an_3:6">3:6</span> ያን መንፈስም፥ በጸጋው ጸድቀን በዘላለም ሕይወት ተስፋ ወራሾች
እንድንሆን፥ በመድኃኒታችን በኢየሱስ ክርስቶስ በእኛ ላይ አትርፎ አፈሰሰው።

<span id="an_3:8">3:8</span> ቃሉ የታመነ ነው። እግዚአብሔርንም የሚያምኑት በመልካም ሥራ በጥንቃቄ
እንዲጸኑ፥ እነዚህን አስረግጠህ እንድትናገር እፈቅዳለሁ። ይህ መልካምና ለሰዎች የሚጠቅም ነው፤

<span id="an_3:9">3:9</span> ነገር ግን ሞኝነት ካለው ምርመራና ከትውልዶች ታሪክ ከክርክርም ስለ
ሕግም ከሚሆን ጠብ ራቅ፤ የማይጠቅሙና ከንቱዎች ናቸውና፤

<span id="an_3:10">3:10</span> መለያየትን የሚያነሣ ሰው ጠማማ እንዲሆን በራሱም ላይ ፈርዶ
ኃጢአትን እንዲያደርግ አውቀህ፥ አንድ ጊዜ ሁለት ጊዜም ከገሠጽኸው በኋላ እንዲህ ከሚመስል ሰው ራቅ።

<span id="an_3:12">3:12</span> አርጢሞንን ወይም ቲኪቆስን ወደ አንተ ስልክ፥ ወደ ኒቆጵልዮን ወደ
እኔ እንድትመጣ ትጋ፤ በዚያ ልከርም ቈርጬአለሁና።

<span id="an_3:13">3:13</span> ሕግ አዋቂውን ዜማስንና አጵሎስን ምንም እንዳይጎድልባቸው
በጉዞአቸው ተግተህ እርዳ።

<span id="an_3:14">3:14</span> ከወገናችንም ያሉት ደግሞ ፍሬ ቢሶች ሆነው እንዳይቀሩ፥ የግድ
ስለሚያስፈልገው ነገር በመልካም ሥራ እንዲጸኑ ይማሩ።

<span id="an_3:15">3:15</span> ከእኔ ጋር ያሉት ሁሉ ሰላምታ ያቀርቡልሃል። በእምነት ለሚወዱን
ሰላምታ አቅርብልን። ጸጋ ከሁላችሁ ጋር ይሁን፤ አሜን።
