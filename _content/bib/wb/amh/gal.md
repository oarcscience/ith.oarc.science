### Amharic Bible: Galatians

<span id="an_1:1">1:1</span> በኢየሱስ ክርስቶስ ከሙታንም ባነሣው በእግዚአብሔር አብ ሐዋርያ የሆነ
እንጂ ከሰዎች ወይም በሰው ያልሆነ ጳውሎስ ከእኔም ጋር ያሉት ወንድሞች ሁሉ፥ ወደ ገላትያ አብያተ ክርስቲያናት፤

<span id="an_1:3">1:3</span> ከእግዚአብሔር ከአባታችን ከጌታችንም ከኢየሱስ ክርስቶስ ጸጋና ሰላም
ለእናንተ ይሁን።

<span id="an_1:4">1:4</span> ክፉ ከሆነ ከአሁኑ ዓለም ያድነን ዘንድ እንደ አምላካችንና እንደ
አባታችን ፈቃድ ስለ ኃጢአታችን ራሱን ሰጠ።

<span id="an_1:5">1:5</span> ለአብ ከዘላለም እስከ ዘላለም ክብር ይሁን፤ አሜን።

<span id="an_1:6">1:6</span> በክርስቶስ ጸጋ እናንተን ከጠራችሁ ከእርሱ ወደ ልዩ ወንጌል እንዲህ
ፈጥናችሁ እንዴት እንዳለፋችሁ እደነቃለሁ፤

<span id="an_1:7">1:7</span> እርሱ ግን ሌላ ወንጌል አይደለም፤ የሚያናውጡአችሁ የክርስቶስንም
ወንጌል ሊያጣምሙ የሚወዱ አንዳንዶች አሉ እንጂ።

<span id="an_1:8">1:8</span> ነገር ግን እኛ ብንሆን ወይም ከሰማይ መልአክ፥ ከሰበክንላችሁ ወንጌል
የሚለይ ወንጌልን ቢሰብክላችሁ፥ የተረገመ ይሁን።

<span id="an_1:9">1:9</span> አስቀድመን እንዳልን እንዲሁ አሁን ሁለተኛ እላለሁ፥ ከተቀበላችሁት
የተለየውን ማንም ቢሰብክላችሁ የተረገመ ይሁን።

<span id="an_1:10">1:10</span> ሰውን ወይስ እግዚአብሔርን አሁን እሺ አሰኛለሁን? ወይም ሰውን
ደስ ላሰኝ እፈልጋለሁን? አሁን ሰውን ደስ ባሰኝ የክርስቶስ ባሪያ ባልሆንሁም።

<span id="an_1:11">1:11</span> ወንድሞች ሆይ፥ በእኔ የተሰበከ ወንጌል እንደ ሰው እንዳይደለ
አስታውቃችኋለሁ፤

<span id="an_1:12">1:12</span> ኢየሱስ ክርስቶስ ገልጦልኛል እንጂ እኔ ከሰው አልተቀበልሁትም
አልተማርሁትምም።

<span id="an_1:13">1:13</span> በአይሁድ ሥርዓት በፊት እንዴት እንደ ኖርሁ ሰምታችኋልና፤
የእግዚአብሔርን ቤተ ክርስቲያን ያለ ልክ አሳድድና አጠፋ ነበር፥

<span id="an_1:14">1:14</span> ለአባቶችም ወግ ከመጠን ይልቅ እየቀናሁ በወገኔ ዘንድ በዘመኔ
ካሉት ብዙዎችን በአይሁድ ሥርዓት እበልጥ ነበር።

<span id="an_1:15">1:15</span> ነገር ግን በእናቴ ማኅፀን ሳለሁ የለየኝ በጸጋውም የጠራኝ
እግዚአብሔር በአሕዛብ መካከል ስለ እርሱ ወንጌልን እሰብክ ዘንድ ልጁን በእኔ ሁኔታ ሊገልጥ በወደደ ጊዜ፥ ወዲያው
ከሥጋና ከደም ጋር አልተማከርሁም፥

<span id="an_1:17">1:17</span> ከእኔም በፊት ሐዋርያት ወደ ነበሩት ወደ ኢየሩሳሌም አልወጣሁም፥
ነገር ግን ወደ ዓረብ አገር ሄድሁ እንደ ገናም ወደ ደማስቆ ተመለስሁ።

<span id="an_1:18">1:18</span> ከዚህ ወዲያ ከሦስት ዓመት በኋላ ኬፋን ልጠይቅ ወደ ኢየሩሳሌም
ወጥቼ ከእርሱ ጋር አሥራ አምስት ቀን ሰነበትሁ፤

<span id="an_1:19">1:19</span> ነገር ግን ከጌታ ወንድም ከያዕቆብ በቀር ከሐዋርያት ሌላ
አላየሁም።

<span id="an_1:20">1:20</span> ስለምፅፍላችሁም ነገር፥ እነሆ፥ በእግዚአብሔር ፊት ሐሰት
አልናገርም።

<span id="an_1:21">1:21</span> ከዚያ ወዲያ ወደ ሶርያና ወደ ኪልቅያ አገር መጣሁ።

<span id="an_1:22">1:22</span> በክርስቶስም ያሉት የይሁዳ ማኅበሮች ፊቴን አያውቁም ነበር፤

<span id="an_1:23">1:23</span> ነገር ግን። ቀድሞ እኛን ያሳድድ የነበረ፥ እርሱ በፊት ያጠፋው
የነበረውን ሃይማኖት አሁን ይሰብካል ተብሎ ሲነገር ይሰሙ ነበር፤

<span id="an_1:24">1:24</span> ስለ እኔም እግዚአብሔርን ያከብሩ ነበር።

<span id="an_2:1">2:1</span> ከዚያ ወዲያ ከአሥራ አራት ዓመት በኋላ ከበርናባስ ጋር ቲቶን ደግሞ
ይዤ ወደ ኢየሩሳሌም ሁለተኛ ወጣሁ፤

<span id="an_2:2">2:2</span> እንደ ተገለጠልኝም ወጣሁ፤ ምናልባትም በከንቱ እንዳልሮጥ ወይም
በከንቱ ሮጬ እንዳልሆን በአሕዛብ መካከል የምሰብከውን ወንጌል አስታወቅኋቸው፤ ዋኖች ግን መስለው ለሚታዩ ለብቻቸው
አስታወቅኋቸው።

<span id="an_2:3">2:3</span> ነገር ግን ከእኔ ጋር የነበረ ቲቶ እንኳ የግሪክ ሰው ሲሆን ይገረዝ
ዘንድ ግድ አላሉትም፤

<span id="an_2:4">2:4</span> ነገር ግን ባሪያዎች ሊያደርጉን በክርስቶስ ኢየሱስ ያለንን
አርነታችንን ይሰልሉ ዘንድ ሾልከው በስውር ስለ ገቡ ስለ ሐሰተኞች ወንድሞች ነበረ።

<span id="an_2:5">2:5</span> የወንጌልም እውነት በእናንተ ዘንድ ጸንቶ እንዲኖር ለአንድ ሰዓት
እንኳ ለቅቀን አልተገዛንላቸውም።

<span id="an_2:6">2:6</span> አለቆች የመሰሉት ግን፥ በፊት ማን እንደ ነበሩ አይገደኝም፤
እግዚአብሔር የሰውን ፊት አይቶ አያደላም፤ አለቆች የመሰሉት አንዳች እንኳ አልጨመሩልኝምና፥

<span id="an_2:7">2:7</span> ተመልሰው ግን ጴጥሮስ ለተገረዙት የሆነው ወንጌል አደራ እንደ ተሰጠው
እንዲሁ ለእኔ ላልተገረዙት የሆነው ወንጌል አደራ እንደ ተሰጠኝ አዩ፤

<span id="an_2:8">2:8</span> ለተገረዙት ሐዋርያ እንዲሆን ለጴጥሮስ የሠራለት፥ ለእኔ ደግሞ
ለአሕዛብ ሐዋርያ እንድሆን ሠርቶአልና።

<span id="an_2:9">2:9</span> ደግሞ የተሰጠኝን ጸጋ አውቀው፥ አዕማድ መስለው የሚታዩ ያዕቆብና ኬፋ
ዮሐንስም እኛ ወደ አሕዛብ እነርሱም ወደ ተገረዙት ይሄዱ ዘንድ ለእኔና ለበርናባስ ቀኝ እጃቸውን ሰጡን፤

<span id="an_2:10">2:10</span> ድሆችን እናስብ ዘንድ ብቻ ለመኑን፥ ይህንም ደግሞ ላደርግ ተጋሁ።

<span id="an_2:11">2:11</span> ነገር ግን ኬፋ ወደ አንጾኪያ በመጣ ጊዜ ፊት ለፊት ተቃወምሁት፥
ይፈረድበት ዘንድ ይገባ ነበርና።

<span id="an_2:12">2:12</span> አንዳንድ ከያዕቆብ ዘንድ ሳይመጡ ከአሕዛብ ጋር አብሮ ይበላ
ነበርና፤ በመጡ ጊዜ ግን ከተገረዙት ወገን ያሉትን ፈርቶ ያፈገፍግ ነበርና ከእነርሱ ተለየ።

<span id="an_2:13">2:13</span> የቀሩትም አይሁድ ደግሞ፥ በርናባስ ስንኳ በግብዝነታቸው እስከ
ተሳበ ድረስ፥ ከእርሱ ጋር አብረው ግብዞች ሆኑ።

<span id="an_2:14">2:14</span> ነገር ግን እንደ ወንጌል እውነት በቅንነት እንዳልሄዱ ባየሁ ጊዜ
በሁሉ ፊት ኬፋን። አንተ አይሁዳዊ ሳለህ በአይሁድ ኑሮ ያይደለ በአሕዛብ ኑሮ ብትኖር፥ አሕዛብ በአይሁድ ኑሮ ሊኖሩ
እንዴት ግድ አልሃቸው? አልሁት።

<span id="an_2:15">2:15</span> እኛ በፍጥረት አይሁድ ነን ኃጢአተኞችም የሆኑ አሕዛብ አይደለንም፤

<span id="an_2:16">2:16</span> ነገር ግን ሰው በኢየሱስ ክርስቶስ በማመን እንዲጸድቅ እንጂ በሕግ
ሥራ እንዳይሆን አውቀን፥ ሥጋን የለበሰ ሁሉ በሕግ ሥራ ስለማይጸድቅ፥ እኛ ራሳችን በሕግ ሥራ ሳይሆን በክርስቶስ
እምነት እንጸድቅ ዘንድ በክርስቶስ ኢየሱስ አምነናል።

<span id="an_2:17">2:17</span> ነገር ግን በክርስቶስ ልንጸድቅ የምንፈልግ ስንሆን ራሳችን ደግሞ
ኃጢአተኞች ሆነን ከተገኘን፥ እንግዲያስ ክርስቶስ የኃጢአት አገልጋይ ነዋ? አይደለም።

<span id="an_2:18">2:18</span> ያፈረስሁትን ይህን እንደ ገና የማንጽ ከሆንሁ ራሴ ሕግ ተላላፊ
እንድሆን አስረዳለሁና።

<span id="an_2:19">2:19</span> እኔ ለእግዚአብሔር ሕያው ሆኜ እኖር ዘንድ በሕግ በኩል ለሕግ ሞቼ
ነበርና።

<span id="an_2:20">2:20</span> ከክርስቶስ ጋር ተሰቅዬአለሁ፤ እኔም አሁን ሕያው ሆኜ አልኖርም
ክርስቶስ ግን በእኔ ይኖራል፤ አሁንም በሥጋ የምኖርበት ኑሮ በወደደኝና ስለ እኔ ራሱን በሰጠው በእግዚአብሔር ልጅ
ላይ ባለ እምነት የምኖረው ነው።

<span id="an_2:21">2:21</span> የእግዚአብሔርን ጸጋ አልጥልም፤ ጽድቅስ በሕግ በኩል ከሆነ
እንኪያስ ክርስቶስ በከንቱ ሞተ።

<span id="an_3:1">3:1</span> የማታስተውሉ የገላትያ ሰዎች ሆይ፥ በዓይናችሁ ፊት ኢየሱስ ክርስቶስ
እንደ ተሰቀለ ሆኖ ተሥሎ ነበር፤ ለእውነት እንዳትታዘዙ አዚም ያደረገባችሁ ማን ነው?

<span id="an_3:2">3:2</span> ይህን ብቻ ከእናንተ እንድማር እወዳለሁ፤ በሕግ ሥራ ወይም ከእምነት
ጋር በሆነ መስማት መንፈስን ተቀበላችሁን?

<span id="an_3:3">3:3</span> እንዲህን የማታስተውሉ ናችሁ? በመንፈስ ጀምራችሁ አሁን በሥጋ
ትፈጽማላችሁን?

<span id="an_3:4">3:4</span> በውኑ ከንቱ እንደ ሆነስ እንደዚህ ያለ መከራ በከንቱ ተቀበላችሁን?

<span id="an_3:5">3:5</span> እንግዲህ መንፈስን የሚሰጣችሁ በእናንተም ዘንድ ተአምራት የሚያደርግ፥
በሕግ ሥራ ነውን ወይስ ከእምነት ጋር በሆነ መስማት ነው?

<span id="an_3:6">3:6</span> እንዲሁ አብርሃም በእግዚአብሔር አመነና ጽድቅ ሆኖ ተቆጠረለት።

<span id="an_3:7">3:7</span> እንኪያስ ከእምነት የሆኑት እነዚህ የአብርሃም ልጆች እንደ ሆኑ
እወቁ።

<span id="an_3:8">3:8</span> መጽሐፍም እግዚአብሔር አሕዛብን በእምነት እንዲያጸድቅ አስቀድሞ
አይቶ። በአንተ አሕዛብ ሁሉ ይባረካሉ ብሎ ወንጌልን ለአብርሃም አስቀድሞ ሰበከ።

<span id="an_3:9">3:9</span> እንደዚህ ከእምነት የሆኑት ካመነው ከአብርሃም ጋር ይባረካሉ።

<span id="an_3:10">3:10</span> ከሕግ ሥራ የሆኑት ሁሉ በእርግማን በታች ናቸውና፤ በሕግ መጽሐፍ
በተጻፉት ሁሉ ጸንቶ የማይኖር የማያደርግም ሁሉ የተረገመ ነው ተብሎ ተጽፎአልና።

<span id="an_3:11">3:11</span> ጻድቅ በእምነት ይኖራል ተብሎአልና ማንም በእግዚአብሔር ፊት በሕግ
እንዳይጸድቅ ግልጥ ነው።

<span id="an_3:12">3:12</span> ሕግም ከእምነት አይደለም ነገር ግን። የሚያደርገው ይኖርበታል
ተብሎአል።

<span id="an_3:13">3:13</span> በእንጨት የሚሰቀል ሁሉ የተረገመ ነው ተብሎ ተጽፎአልና ክርስቶስ
ስለ እኛ እርግማን ሆኖ ከሕግ እርግማን ዋጀን፤

<span id="an_3:14">3:14</span> የመንፈስን ተስፋ በእምነት እንድንቀበል፥ የአብርሃም በረከት ወደ
አሕዛብ በኢየሱስ ክርስቶስ ይደርስላቸው ዘንድ።

<span id="an_3:15">3:15</span> ወንድሞች ሆይ፥ እንደ ሰው ልማድ እላለሁ። የሰው ስንኳ ቢሆን
እርግጠኛውን ኪዳን ማንም አይንቅም ወይም አይጨምርበትም።

<span id="an_3:16">3:16</span> ለአብርሃምና ለዘሩም የተስፋው ቃል ተነገረ። ስለ ብዙዎች
እንደሚነገር። ለዘሮቹም አይልም፤ ስለ አንድ እንደሚነገር ግን። ለዘርህም ይላል፥ እርሱም ክርስቶስ።

<span id="an_3:17">3:17</span> ይህንም እላለሁ፤ ከአራት መቶ ሠላሳ ዓመት በኋላ የተሰጠው ሕግ
አስቀድሞ በእግዚአብሔር የተረጋገጠውን ኪዳን የተስፋውን ቃል እስኪሽር ድረስ አያፈርስም።

<span id="an_3:18">3:18</span> ርስቱ በሕግ ቢሆንስ እንግዲያስ በተስፋ ቃል አይሆንም፤ በተስፋ
ቃል ግን እግዚአብሔር ለአብርሃም ሰጥቶአል።

<span id="an_3:19">3:19</span> እንግዲህ ሕግ ምንድር ነው? ተስፋው የተሰጠው ዘር እስኪመጣ ድረስ
በመካከለኛ እጅ በመላእክት በኩል ስለ ሕግ መተላለፍ ተጨመረ።

<span id="an_3:20">3:20</span> መካከለኛውም ለአንድ ብቻ አይደለም እግዚአብሔር ግን አንድ ነው።

<span id="an_3:21">3:21</span> እንግዲህ ሕግ የእግዚአብሔርን የተስፋ ቃል የሚቃወም ነውን?
አይደለም። ሕያው ሊያደርግ የሚቻለው ሕግ ተሰጥቶ ቢሆንስ በእውነት ጽድቅ ከሕግ በሆነ ነበር፤

<span id="an_3:22">3:22</span> ነገር ግን በኢየሱስ ክርስቶስ ከማመን የሆነው የተስፋ ቃል
ለሚያምኑ ይሰጥ ዘንድ መጽሐፍ ከኃጢአት በታች ሁሉን ዘግቶታል።

<span id="an_3:23">3:23</span> እምነትም ሳይመጣ ሊገለጥ ላለው እምነት ተዘግተን ከሕግ በታች
እንጠበቅ ነበር።

<span id="an_3:24">3:24</span> እንደዚህ በእምነት እንጸድቅ ዘንድ ሕግ ወደ ክርስቶስ የሚያመጣ
ሞግዚታችን ሆኖአል፤

<span id="an_3:25">3:25</span> እምነት ግን መጥቶአልና ከእንግዲህ ወዲህ ከሞግዚት በታች
አይደለንም።

<span id="an_3:26">3:26</span> በእምነት በኩል ሁላችሁ በክርስቶስ ኢየሱስ የእግዚአብሔር ልጆች
ናችሁና፤

<span id="an_3:27">3:27</span> ከክርስቶስ ጋር አንድ ትሆኑ ዘንድ የተጠመቃችሁ ሁሉ ክርስቶስን
ለብሳችኋልና።

<span id="an_3:28">3:28</span> አይሁዳዊ ወይም የግሪክ ሰው የለም፥ ባሪያ ወይም ጨዋ ሰው የለም፥
ወንድም ሴትም የለም፤ ሁላችሁ በክርስቶስ ኢየሱስ አንድ ሰው ናችሁና።

<span id="an_3:29">3:29</span> እናንተም የክርስቶስ ከሆናችሁ እንኪያስ የአብርሃም ዘር እንደ
ተስፋውም ቃል ወራሾች ናችሁ።

<span id="an_4:1">4:1</span> ነገር ግን እላለሁ፥ ወራሹ ሕፃን ሆኖ ባለበት ዘመን ሁሉ፥ ምንም
የሁሉ ጌታ ቢሆን ከቶ ከባሪያ አይለይም፥

<span id="an_4:2">4:2</span> ነገር ግን አባቱ እስከ ቀጠረለት ቀን ድረስ ከጠባቂዎችና ከመጋቢዎች
በታች ነው።

<span id="an_4:3">4:3</span> እንዲሁ እኛ ደግሞ ሕፃናት ሆነን ሳለን ከዓለም መጀመሪያ ትምህርት
በታች ተገዝተን ባሪያዎች ነበርን፤

<span id="an_4:4">4:4</span> ነገር ግን የዘመኑ ፍጻሜ በደረሰ ጊዜ እግዚአብሔር ከሴት የተወለደውን
ከሕግም በታች የተወለደውን ልጁን ላከ፤

<span id="an_4:5">4:5</span> እንደ ልጆች እንሆን ዘንድ፥ ከሕግ በታች ያሉትን ይዋጅ ዘንድ።

<span id="an_4:6">4:6</span> ልጆችም ስለ ሆናችሁ እግዚአብሔር አባ አባት ብሎ የሚጮኽ የልጁን
መንፈስ በልባችሁ ውስጥ ላከ።

<span id="an_4:7">4:7</span> ስለዚህ ከእንግዲህ ወዲህ ልጅ ነህ እንጂ ባሪያ አይደለህም፤ ልጅም
ከሆንህ ደግሞ በክርስቶስ የእግዚአብሔር ወራሽ ነህ።

<span id="an_4:8">4:8</span> ነገር ግን በዚያን ጊዜ እግዚአብሔርን ሳታውቁ በባሕርያቸው አማልክት
ለማይሆኑ ባሪያዎች ሆናችሁ ተገዛችሁ፤

<span id="an_4:9">4:9</span> አሁን ግን እግዚአብሔርን ስታውቁ ይልቅስ በእግዚአብሔር ስትታወቁ
እንደ ገና ወደ ደካማ ወደሚናቅም ወደ መጀመሪያ ትምህርት እንዴት ትመለሳላችሁ? እንደ ገና ባሪያዎች ሆናችሁ ዳግመኛ
ለዚያ ልትገዙ ትወዳላችሁን?

<span id="an_4:10">4:10</span> ቀንንና ወርን ዘመንንም ዓመትንም በጥንቃቄ ትጠብቃላችሁ።

<span id="an_4:11">4:11</span> ምናልባት በከንቱ ለእናንተ ደክሜአለሁ ብዬ እፈራችኋለሁ።

<span id="an_4:12">4:12</span> ወንድሞች ሆይ፥ እኔ ደግሞ እንደ እናንተ ሆኜአለሁና። እንደ እኔ
ሁኑ ብዬ እለምናችኋለሁ። አንዳችም አልበደላችሁኝም።

<span id="an_4:13">4:13</span> በመጀመሪያ ወንጌልን በሰበክሁላችሁ ጊዜ ከሥጋ ድካም የተነሣ እንደ
ነበረ ታውቃላችሁ፥

<span id="an_4:14">4:14</span> በሥጋዬም ፈተና የሆነባችሁን ነገር አልናቃችሁትምና
አልተጸየፋችሁትም ነገር ግን እንደ እግዚአብሔር መልአክ አዎን እንደ ክርስቶስ ኢየሱስ ተቀበላችሁኝ።

<span id="an_4:15">4:15</span> እንግዲህ ደስ ማሰኘታችሁ ወዴት አለ? ቢቻልስ ዓይኖቻችሁን
አውጥታችሁ በሰጣችሁኝ ብዬ እመሰክርላችኋለሁ።

<span id="an_4:16">4:16</span> እንኪያስ እውነቱን ስለ ነገርኋችሁ ጠላት ሆንሁባችሁን?

<span id="an_4:17">4:17</span> በብርቱ ይፈልጉአችኋል፥ ለመልካም አይደለም ነገር ግን በብርቱ
ትፈልጉአቸው ዘንድ በውጭ ሊያስቀሩአችሁ ይወዳሉ።

<span id="an_4:18">4:18</span> ነገር ግን ከእናንተ ዘንድ በምኖርበት ጊዜ ብቻ ሳይሆን ሁልጊዜ
በመልካም ነገር ተፈላጊ ብትሆኑ መልካም ነው።

<span id="an_4:19">4:19</span> ልጆቼ ሆይ፥ ክርስቶስ በእናንተ እስኪሣል ድረስ ዳግመኛ ስለ
እናንተ ምጥ ይዞኛል።

<span id="an_4:20">4:20</span> ስለ እናንተ አመነታለሁና አሁን በእናንተ ዘንድ ሆኜ ድምፄን
ልለውጥ በወደድሁ።

<span id="an_4:21">4:21</span> እናንተ ከሕግ በታች ልትኖሩ የምትወዱ፥ ሕጉን አትሰሙምን? እስኪ
ንገሩኝ።

<span id="an_4:22">4:22</span> አንዱ ከባሪያይቱ አንዱም ከጨዋይቱ የሆኑ ሁለት ልጆች ለአብርሃም
እንደ ነበሩት ተጽፎአልና።

<span id="an_4:23">4:23</span> ነገር ግን የባሪያይቱ ልጅ እንደ ሥጋ ተወልዶአል፥ የጨዋይቱ ግን
በተስፋው ቃል ተወልዶአል።

<span id="an_4:24">4:24</span> ይህም ነገር ምሳሌ ነው፤ እነዚህ ሴቶች እንደ ሁለቱ ኪዳኖች
ናቸውና። ከደብረ ሲና የሆነችው አንዲቱ ለባርነት ልጆችን ትወልዳለች፥ እርስዋም አጋር ናት።

<span id="an_4:25">4:25</span> ይህችም አጋር በዓረብ ምድር ያለችው ደብረ ሲና ናት፤ አሁንም
ያለችውን ኢየሩሳሌምን ትመስላለች፥ ከልጆችዋ ጋር በባርነት ናትና።

<span id="an_4:26">4:26</span> ላይኛይቱ ኢየሩሳሌም ግን በነጻነት የምትኖር ናት እርስዋም
እናታችን ናት።

<span id="an_4:27">4:27</span> አንቺ የማትወልጅ መካን፥ ደስ ይበልሽ፤ አንቺ አምጠሽ የማታውቂ፥
እልል በዪና ጩኺ፤ ባል ካላቱ ይልቅ የብቸኛይቱ ልጆች በዝተዋልና ተብሎ ተጽፎአል።

<span id="an_4:28">4:28</span> እኛም፥ ወንድሞች ሆይ፥ እንደ ይስሐቅ የተስፋ ቃል ልጆች ነን።

<span id="an_4:29">4:29</span> ነገር ግን እንደ ሥጋ የተወለደው እንደ መንፈስ የተወለደውን
በዚያን ጊዜ እንዳሳደደው ዛሬም እንዲሁ ነው።

<span id="an_4:30">4:30</span> ነገር ግን መጽሐፍ ምን ይላል? የባሪያይቱ ልጅ ከጨዋይቱ ልጅ ጋር
አይወርስምና ባሪያይቱን ከልጅዋ ጋር አውጣት።

<span id="an_4:31">4:31</span> ስለዚህ፥ ወንድሞች ሆይ፥ የጨዋይቱ ልጆች ነን እንጂ የባሪያይቱ
አይደለንም።

<span id="an_5:1">5:1</span> በነጻነት ልንኖር ክርስቶስ ነጻነት አወጣን፤ እንግዲህ ጸንታችሁ ቁሙ
እንደ ገናም በባርነት ቀንበር አትያዙ።

<span id="an_5:2">5:2</span> እነሆ፥ እኔ ጳውሎስ እላችኋለሁ። ብትገረዙ ክርስቶስ ምንም
አይጠቅማችሁም።

<span id="an_5:3">5:3</span> ሕግንም ሁሉ እንዲፈጽም ግድ አለበት ብዬ ለሚገረዙት ሁሉ
ለእያንዳንዶች ደግሜ እመሰክራለሁ።

<span id="an_5:4">5:4</span> በሕግ ልትጸድቁ የምትፈልጉ ከክርስቶስ ተለይታችሁ ከጸጋው ወድቃችኋል።

<span id="an_5:5">5:5</span> እኛ በመንፈስ ከእምነት የጽድቅን ተስፋ እንጠባበቃለንና።

<span id="an_5:6">5:6</span> በክርስቶስ ኢየሱስ ሆኖ፥ በፍቅር የሚሠራ እምነት እንጂ መገረዝ ቢሆን
ወይም አለመገረዝ አይጠቅምምና።

<span id="an_5:7">5:7</span> በመልካም ትሮጡ ነበር፤ ለእውነት እንዳትታዘዙ ማን ከለከላችሁ?

<span id="an_5:8">5:8</span> ይህ ማባበል ከሚጠራችሁ አልወጣም። ጥቂት እርሾ ሊጡን ሁሉ ያቦካል።

<span id="an_5:9">5:9</span> የተለየ ነገር ከቶ እንዳታስቡ እኔ በጌታ ስለ እናንተ ታምኜአለሁ፤
የሚያናውጣችሁ ማንም ቢኖር ግን ፍርዱን ሊሸከም ነው።

<span id="an_5:11">5:11</span> ነገር ግን፥ ወንድሞች ሆይ፥ እኔ ገና እስከ አሁን መገረዝን
ብሰብክ እስከ አሁን ድረስ ለምን ያሳድዱኛል? እንኪያስ የመስቀል ዕንቅፋት ተወግዶአል።

<span id="an_5:12">5:12</span> የሚያውኩአችሁ ይቆረጡ።

<span id="an_5:13">5:13</span> ወንድሞች ሆይ፥ እናንተ ለአርነት ተጠርታችኋልና፤ ብቻ አርነታችሁ
ለሥጋ ምክንያትን አይስጥ፥ ነገር ግን በፍቅር እርስ በርሳችሁ እንደ ባሪያዎች ሁኑ።

<span id="an_5:14">5:14</span> ሕግ ሁሉ በአንድ ቃል ይፈጸማልና፥ እርሱም። ባልንጀራህን እንደ
ራስህ ውደድ የሚል ነው።

<span id="an_5:15">5:15</span> ነገር ግን እርስ በርሳችሁ ብትነካከሱ ብትበላሉ እርስ በርሳችሁ
እንዳትጠፋፉ ተጠንቀቁ።

<span id="an_5:16">5:16</span> ነገር ግን እላለሁ፥ በመንፈስ ተመላለሱ፥ የሥጋንም ምኞት ከቶ
አትፈጽሙ።

<span id="an_5:17">5:17</span> ሥጋ በመንፈስ ላይ መንፈስም በሥጋ ላይ ይመኛልና፥ እነዚህም እርስ
በርሳቸው ይቀዋወማሉ፤ ስለዚህም የምትወዱትን ልታደርጉ አትችሉም።

<span id="an_5:18">5:18</span> በመንፈስ ብትመሩ ግን ከሕግ በታች አይደላችሁም።

<span id="an_5:19">5:19</span> የሥጋ ሥራም የተገለጠ ነው እርሱም ዝሙት፥ ርኵሰት፥

<span id="an_5:20">5:20</span> መዳራት፥ ጣዖትን ማምለክ፥ ምዋርት፥ ጥል፥ ክርክር፥ ቅንዓት፥
ቁጣ፥ አድመኛነት፥

<span id="an_5:21">5:21</span> መለያየት፥ መናፍቅነት፥ ምቀኝነት፥ መግደል፥ ስካር፥ ዘፋኝነት፥
ይህንም የሚመስል ነው። አስቀድሜም እንዳልሁ፥ እንደዚህ ያሉትን የሚያደርጉ የእግዚአብሔርን መንግሥት አይወርሱም።

<span id="an_5:22">5:22</span> የመንፈስ ፍሬ ግን ፍቅር፥ ደስታ፥ ሰላም፥ ትዕግሥት፥ ቸርነት፥
በጎነት፥ እምነት፥ የውሃት፥ ራስን መግዛት ነው።

<span id="an_5:23">5:23</span> እንደዚህ ያሉትን የሚከለክል ሕግ የለም።

<span id="an_5:24">5:24</span> የክርስቶስ ኢየሱስም የሆኑቱ ሥጋን ከክፉ መሻቱና ከምኞቱ ጋር
ሰቀሉ።

<span id="an_5:25">5:25</span> በመንፈስ ብንኖር በመንፈስ ደግሞ እንመላለስ።

<span id="an_5:26">5:26</span> እርስ በርሳችን እየተነሣሣንና እየተቀናናን በከንቱ አንመካ።

<span id="an_6:1">6:1</span> ወንድሞች ሆይ፥ ሰው በማናቸውም በደል ስንኳ ቢገኝ፥ መንፈሳውያን
የሆናችሁ እናንተ እንደዚህ ያለውን ሰው በየውሃት መንፈስ አቅኑት፤ አንተ ደግሞ እንዳትፈተን ራስህን ጠብቅ።

<span id="an_6:2">6:2</span> ከእናንተ እያንዳንዱ የአንዱን ሸክም ይሸከም እንዲሁም የክርስቶስን
ሕግ ፈጽሙ።

<span id="an_6:3">6:3</span> አንዱ ምንም ሳይሆን ምንም የሆነ ቢመስለው ራሱን ያታልላልና።

<span id="an_6:4">6:4</span> ነገር ግን እያንዳንዱ የገዛ ራሱን ሥራ ይፈትን፥ ከዚያም በኋላ ስለ
ሌላው ሰው ያልሆነ ስለ ራሱ ብቻ የሚመካበትን ያገኛል፤

<span id="an_6:5">6:5</span> እያንዳንዱ የገዛ ራሱን ሸክም ሊሸከም ነውና።

<span id="an_6:6">6:6</span> ነገር ግን ቃሉን የሚማር ከሚያስተምረው ጋር መልካምን ነገር ሁሉ
ይከፋፈል።

<span id="an_6:7">6:7</span> አትሳቱ፤ እግዚአብሔር አይዘበትበትም። ሰው የሚዘራውን ሁሉ ያንኑ
ደግሞ ያጭዳልና፤

<span id="an_6:8">6:8</span> በገዛ ሥጋው የሚዘራ ከሥጋ መበስበስን ያጭዳልና፥ በመንፈስ ግን
የሚዘራው ከመንፈስ የዘላለምን ሕይወት ያጭዳል።

<span id="an_6:9">6:9</span> ባንዝልም በጊዜው እናጭዳለንና መልካም ሥራን ለመሥራት አንታክት።

<span id="an_6:10">6:10</span> እንግዲያስ ጊዜ ካገኘን ዘንድ ለሰው ሁሉ ይልቁንም ለሃይማኖት ቤተ
ሰዎች መልካም እናድርግ።

<span id="an_6:11">6:11</span> እንዴት ባሉ ታላላቆች ፊደላት በእጄ እንደ ጻፍሁላችሁ እዩ።

<span id="an_6:12">6:12</span> በሥጋ መልካም ሆነው ሊታዩ የሚወዱ ሁሉ እንድትገረዙ ግድ
አሉአችሁ፥ ነገር ግን ስለ ክርስቶስ መስቀል እንዳይሰደዱ ብቻ ነው።

<span id="an_6:13">6:13</span> በሥጋችሁ እንዲመኩ ልትገረዙ ይወዳሉ እንጂ የተገረዙቱ ራሳቸው
እንኳ ሕግን አይጠብቁም።

<span id="an_6:14">6:14</span> ነገር ግን ዓለም ለእኔ የተሰቀለበት እኔም ለዓለም የተሰቀልሁበት
ከጌታችን ከኢየሱስ ክርስቶስ መስቀል በቀር ሌላ ትምክህት ከእኔ ይራቅ።

<span id="an_6:15">6:15</span> በክርስቶስ ኢየሱስ አዲስ ፍጥረት መሆን ይጠቅማል እንጂ መገረዝ
ቢሆን ወይም አለመገረዝ አይጠቅምምና።

<span id="an_6:16">6:16</span> በዚህም ሥርዓት በሚመላለሱ ሁሉ ላይ በእግዚአብሔር እስራኤልም ላይ
ሰላምና ምሕረት ይሁን።

<span id="an_6:17">6:17</span> እኔ የኢየሱስን ማኅተም በሥጋዬ ተሸክሜአለሁና ከእንግዲህ ወዲህ
አንድ ስንኳ አያድክመኝ።

<span id="an_6:18">6:18</span> ወንድሞች ሆይ፥ የጌታችን የኢየሱስ ክርስቶስ ጸጋ ከመንፈሳችሁ ጋር
ይሁን። አሜን።
