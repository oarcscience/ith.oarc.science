### Xhosa Bible: 1 Timothy

<span id="an_1:1">1:1</span> UPawulos, umpostile kaYesu Kristu,
ngokommiselo kaThixo uMsindisi wethu, neNkosi yethu uYesu Kristu,
iThemba lethu,

<span id="an_1:2">1:2</span> ubhalela uTimoti, umntwana wakhe kanye
ngokwaselukholweni; uthi, Makube lubabalo, inceba, uxolo, oluvela
kuThixo uBawo, noKristu Yesu iNkosi yethu.

<span id="an_1:3">1:3</span> Njengoko ndakuyalayo ukuba uhlale e-Efese,
ndakuba mna ndisiya kwelaseMakedoni, ndenjenjalo ukukuyala nangoku,
ukuze ubathethele abanye, bangafundisi mfundiso yimbi;

<span id="an_1:4">1:4</span> kananjalo banganyamekeli zintsomi namilibo
yakuzalwa engaphele ndawo, izinto ezo ezibanga iimbuzwano, zingahambisi
lona ulwakho lukaThixo olungokholo.

<span id="an_1:5">1:5</span> Ke intsingiselo yoyalo iluthando oluphuma
entliziyweni ehlambulukileyo, nasesazeleni esilungileyo, naselukholweni
olungahanahanisiyo;

<span id="an_1:6">1:6</span> abathe abathile, beziphosile ezo nto,
baphambukela ekuthetheni okungento yanto;

<span id="an_1:7">1:7</span> bethanda ukuba ngabafundisi bomthetho,
phofu bengaziqiqi izinto abazithethayo, kwanezo bazingqinayo.

<span id="an_1:8">1:8</span> Siyazi ke ukuba umthetho mhle, ukuba umntu
uthi awuphathe ngokomthetho;

<span id="an_1:9">1:9</span> ngokwazi oku, ukuba umthetho lo awuwiselwe
lungisa, uwiselwe ke abachasene nomthetho, nabangenakululama,
abangahloneli Thixo, nabangaboni, abangengcwele, nabahlambeli,
ababethiyise nababethi-nina, izigwinta,

<span id="an_1:10">1:10</span> abenzi bombulo, amadoda alalanayo,
amasela abantu, amaxoki, amafunga-buxoki, nokuba kukho nto yimbi
ichasene nemfundiso ephilileyo;

<span id="an_1:11">1:11</span> ngokweendaba ezilungileyo zozuko
lukaThixo osikelelekileyo, endaphathiswa zona mna.

<span id="an_1:12">1:12</span> Kananjalo ndinombulelo kowandomelezayo,
uKristu Yesu iNkosi yethu, kuba wandibalela ekuthini ndithembekile,
endimisela ebulungiselelini;

<span id="an_1:13">1:13</span> nakuba ngaphambili ndandingumnyelisi,
nomtshutshisi, nomphathi kakubi; ndesuka ndenzelwa inceba, ngokuba
ndakwenza oko ndingazi, ndisekungakholweni;

<span id="an_1:14">1:14</span> ke lwanda ngokuncamisileyo ubabalo
lweNkosi yethu, lunokholo nothando olukuKristu Yesu.

<span id="an_1:15">1:15</span> Lithembekile ilizwi eli, lifanelwe
kukwamkelwa konke, eli lokuba uKristu Yesu weza ehlabathini ukuba
asindise aboni; endingomkhulu kubo mna;

<span id="an_1:16">1:16</span> noko ke ndesuka ndenzelwa inceba ngenxa
yoku, ukuze uYesu Kristu akubonakalalisele kum, mna ungomkhulu, konke
ukuzeka kade umsindo, ndibe ngumzekelo kwabo baza kukholwa kuye, base
ebomini obungunaphakade.

<span id="an_1:17">1:17</span> Ke kaloku, kuye uKumkani wawo amaphakade,
ongenakonakala, ongenakubonakala, kuThixo olumkileyo yedwa, makubekho
imbeko nozuko, kuse emaphakadeni asemaphakadeni. Amen.

<span id="an_1:18">1:18</span> Olu yalo ndilubeka phambi kwakho, mntwana
wam Timoti, ngokweziprofeto ezandulelayo ngawe, ukuze ulwe ngazo ukulwa
okuhle,

<span id="an_1:19">1:19</span> unokholo nesazela esilungileyo, ethe
inxenye, isigibile, yaphukelwa ngumkhombe mayelana nokholo;

<span id="an_1:20">1:20</span> inoHimenayo noAlesandire, endabanikelayo
kuSathana, ukuze baqeqeshwe ukuba banganyelisi.

<span id="an_2:1">2:1</span> Ndiyavuselela ngoko kuqala kwiinto zonke,
ukuba kwenziwe izikhungo, imithandazo, izibongozo, imibulelo, ngenxa
yabantu bonke;

<span id="an_2:2">2:2</span> ngenxa yookumkani nabo bonke
abasekongameni, ukuze sihlale ngokonwaba nangokuzola, sinokuhlonela
uThixo, nokundileka konke.

<span id="an_2:3">2:3</span> Kuba oku kuhle, kwamkelekile emehlweni
oMsindisi wethu uThixo;

<span id="an_2:4">2:4</span> onga yena bangasindiswa bonke abantu,
bafikelele ekuyazini inyaniso.

<span id="an_2:5">2:5</span> Kuba mnye uThixo; mnye nomlamli phakathi
koThixo nabantu, umntu uKristu Yesu,

<span id="an_2:6">2:6</span> lowo wazinikelayo ukuba abe yintlawulelo
yokukhulula abantu bonke; ubungqina ke obo bokungqinwa ngamaxesha
angawabo;

<span id="an_2:7">2:7</span> endithe ndamiselwa bona mna, ukuba ndibe
ngumshumayeli nompostile (ndithetha inyaniso ndikuKristu, andixoki),
umfundisi weentlanga elukholweni nasenyanisweni.

<span id="an_2:8">2:8</span> Ngoko ndinga angathandaza amadoda kuzo
zonke iindawo, ephakamisa izandla ezingcwele, engenangqumbo,
engenantandabuzo.

<span id="an_2:9">2:9</span> Kwangokunjalo mabathi nabafazi banxibe
iingubo ezifanelekileyo, beneentloni, benesidima, ingabi kukuphotha
iinwele, nokunxitywa kwegolide neeperile, nesambatho esixabiso likhulu;

<span id="an_2:10">2:10</span> mabathi (ngokufanele abafazi abathi
bahlonela uThixo), bavathe imisebenzi elungileyo.

<span id="an_2:11">2:11</span> Umfazi, ezolile, makafunde ethobele
konke.

<span id="an_2:12">2:12</span> Umfazi ke andimvumeli ukuba afundise,
nokuba ayilawule indoda; makazole.

<span id="an_2:13">2:13</span> Kuba kwabunjwa uAdam kuqala, kwandula ke
ukubunjwa uEva.

<span id="an_2:14">2:14</span> Kanjalo akulukuhlwanga Adam; kulukuhlwe
umfazi, waba sekugqitheni.

<span id="an_2:15">2:15</span> Noko ke wosindiswa ngokuzala, ukuba bathe
bema elukholweni, naseluthandweni, nasebungcwaliseni, benesidima.

<span id="an_3:1">3:1</span> Lithembekile ilizwi eli lithi, Ukuba ubani
ungxamele ubuveleli, unqwenela umsebenzi omhle.

<span id="an_3:2">3:2</span> Umveleli ngoko umelwe kukuthi abe
ngongenakukhalazelwa, eyindoda yamfazi mnye, eyingcathu, enesidima,
ezibekile, ebuka iindwendwe, enokufundisa;

<span id="an_3:3">3:3</span> engageziswa yiwayini, engembethi,
engemngxameli wanzuzo imbi. Makabe ngothantamisayo, engalwi, engemthandi
wamali;

<span id="an_3:4">3:4</span> eyongamele kakuhle eyakhe indlu, abantwana
bakhe bemthobele, benokundileka konke

<span id="an_3:5">3:5</span> (kuba ukuba umntu akakwazi ukuyongamela
eyakhe indlu, angathini na ukuphatha ibandla likaThixo?);

<span id="an_3:6">3:6</span> angabi ngosandula ukungeniswa, ukuze
angathiwa lwale likratshi, eyele ekugwetyweni koMtyholi.

<span id="an_3:7">3:7</span> Kanjalo ke umelwe kukuthi abe nobungqina
obuhle obuvela kwabangaphandle, ukuze angeyeli engcikivweni
nasesibatheni soMtyholi.

<span id="an_3:8">3:8</span> Kwangokunjalo abalungiseleli bamelwe kukuba
nokundileka, bangabi ntlolela yombini, bangazinikeli kwiwayini eninzi,
bangangxameli nzuzo imbi;

<span id="an_3:9">3:9</span> mababe nemfihlo yokholo esazeleni
esihlambulukileyo.

<span id="an_3:10">3:10</span> Nabo aba ke mabacikidwe ukuqala; bandule
ke ukulungiselela, bengabangenakumangaleleka.

<span id="an_3:11">3:11</span> Kwangokunjalo abafazi bamelwe kukuba
nokundileka, bangabi ngabahlebi; mababe ziingcathu, bathembeke ezintweni
zonke.

<span id="an_3:12">3:12</span> Abalungiseleli mababe ngamadoda abafazi
ngabanye, besongamela kakuhle abantwana nezindlu ezizezabo.

<span id="an_3:13">3:13</span> Kuba abo balungiselele kakuhle,
bazizuzela isimo esihle, nokungafihlisi okukhulu ekukholweni,
okukuKristu Yesu.

<span id="an_3:14">3:14</span> Ezi zinto ndizibhala kuwe, ndithembe
ukuba ndiya kuza kuwe kamsinya.

<span id="an_3:15">3:15</span> Ukuba ke ndithe ndalibala, ndikubhalela
ukuze wazi, ukuba kumelwe kukuthiwani na ukuhanjwa endlwini kaThixo,
yona ilibandla likaThixo ophilileyo, intsika nesiseko sayo inyaniso.

<span id="an_3:16">3:16</span> Kananjalo, ngokuvunyiweyo, inkulu
imfihlelo yona leyo yokuhlonela uThixo: lowo wabonakalaliswa esenyameni,
wagwetyelwa esemoyeni, wabonakala kwizithunywa zezulu, wavakaliswa
phakathi kweentlanga, kwakholwa kuye ehlabathini, wenyuswa waba
sebuqaqawulini.

<span id="an_4:1">4:1</span> Ke kaloku uMoya uthi ngokucacileyo inxenye
iya kumka elukholweni, ngamaxesha asemva, inyamekele oomoya
abalahlekisayo, neemfundiso zeedemon;

<span id="an_4:2">4:2</span> imka nohanahaniso lwabathetha ubuxoki,
betshiswe isazela sabo ngentsimbi;

<span id="an_4:3">4:3</span> bengavumi ukuba kuzekwe, nokuba kudliwe
izinto ezadalwa nguThixo, ukuze zamkelwe ngombulelo ngabo bakholwayo,
bayaziyo inyaniso.

<span id="an_4:4">4:4</span> Ngokuba sonke isidalwa sikaThixo sihle;
asiyonto yakulahlwa, ukuba samkelwe ngombulelo;

<span id="an_4:5">4:5</span> kuba singcwaliswa ngalo ilizwi likaThixo
nangezibongozo.

<span id="an_4:6">4:6</span> Ukuba uthe ezi zinto wazibeka phambi
kwabazalwana, woba ngumlungiseleli omhle kaYesu Kristu, wondliwe
ngamazwi okholo, nawemfundiso entle oyilandelayo.

<span id="an_4:7">4:7</span> Ke zona iintsomi ezihlambelayo
zamaxhegokazi zimangale. Ziqhelise ukuhlonela uThixo.

<span id="an_4:8">4:8</span> Kuba eyeyomzimba ingqeqesho inceda
kancinane; ke kona ukuhlonela uThixo kunceda ngako konke, kunedinga
lobomi bakalokunje, nobobo buza kubakho.

<span id="an_4:9">4:9</span> Lithembekile ilizwi elo, lifanelwe
kukwamkelwa konke.

<span id="an_4:10">4:10</span> Kuba sibulalekela, singcikivelwe oko;
ngokuba sithembele kuThixo ophilileyo, onguMsindisi wabantu bonke,
ngokukodwa owabakholwayo.

<span id="an_4:11">4:11</span> Bathethele ezi zinto, ufundise zona.

<span id="an_4:12">4:12</span> Makungabikho namnye ubudelayo ubuncinane
bakho; suka ube ngumzekelo wabakholwayo ngentetho, ngehambo, ngothando,
ngokholo, ngobunyulu.

<span id="an_4:13">4:13</span> Nyamekela imfundo, uvuselelo, imfundiso,
ndide ndifike.

<span id="an_4:14">4:14</span> Ungasihilizeli isipho esingokubabalwa,
esikuwe; owasinikwayo ngesiprofeto, kunye nokubekwa izandla
yintlanganiso yamadoda amakhulu.

<span id="an_4:15">4:15</span> Zinyamekele ezo zinto, hlala kwezo zinto,
ukuze ukuhambela kwakho phambili kubonakale kwinto yonke.

<span id="an_4:16">4:16</span> Zilumkele wena nemfundiso yakho; hlala
uhleli kwezo zinto; kuba ngokwenza oko, wozisindisa wena, ubasindise
nabo bakuvayo.

<span id="an_5:1">5:1</span> Indoda enkulu musa ukuyithethisa kabukhali,
yiyale njengoyihlo; amancinane, njengabazalwana;

<span id="an_5:2">5:2</span> abafazi abakhulu, njengoonyoko; abancinane,
njengoodade, unobunyulu bonke.

<span id="an_5:3">5:3</span> Babeke abahlolokazi abangabahlolokazi
ngenene.

<span id="an_5:4">5:4</span> Ke ukuba kukho mhlolokazi uthile
unabantwana, nokuba ngabazukulwana, ukuqala mabafunde ukuhlonela elabo
ikhaya, bababuyekezele umbuyekezo ooninakhulu; kuba oko kuhle,
kwamkelekile emehlweni kaThixo.

<span id="an_5:5">5:5</span> Ke yena ongumhlolokazi ngenene, eshiywe
yedwa, uthembela kuThixo; uhlala ehleli ekukhungeni nasemithandazweni
ubusuku nemini.

<span id="an_5:6">5:6</span> Kodwa yena oxhamla iziyolo ufile, nakuba
ngathi udla ubomi.

<span id="an_5:7">5:7</span> Bathethele ezi zinto, ukuze bangabi
nakukhalazeleka.

<span id="an_5:8">5:8</span> Ukuba ke umntu akabakhathalele abakhe,
ngokukodwa abendlu yakhe, ulukhanyele ukholo, unobubi ngaphezu
kongakholwayo.

<span id="an_5:9">5:9</span> Makangabalelwa enanini labahlolokazi
umhlolokazi ongaphantsi kweminyaka emashumi mathandathu; owayengumfazi
wandoda-nye,

<span id="an_5:10">5:10</span> engqinelwe ngemisebenzi emihle, ukuba
wondle abantwana, ukuba wamkele iindwendwe, ukuba uzihlambile iinyawo
zabangcwele, ukuba ubasizile ababandezelweyo, ukuba usukelene
nemisebenzi yonke elungileyo.

<span id="an_5:11">5:11</span> Ke bona abahlolokazi abatsha bamangale;
kuba xa bathe badimala nguKristu, bothanda ukwenda;

<span id="an_5:12">5:12</span> benesigwebo sokuba balutshitshisile
ukholo lokuqala.

<span id="an_5:13">5:13</span> Kunye noko ke bafunda ukungasebenzi nto,
bethungelana nezindlu; banganeli kungasebenzi nto kodwa, basuka
kananjalo babe ngamapholopholo, bazifake kwizinto zabanye, bathethe
izinto ezingafanelekileyo.

<span id="an_5:14">5:14</span> Ngoko ndinga abahlolokazi abatsha
bangenda, bazale, babe ngabanikazi-mizi, bangamniki ochasayo nalinye
ithuba lokutshabhisa.

<span id="an_5:15">5:15</span> Kuba inxenye iseyiphambukele emva
koSathana.

<span id="an_5:16">5:16</span> Ukuba kukho ikholwa, nokuba likholwakazi,
elinabahlolokazi, malibasize, lize ibandla lingasindwa, ukuze lona
lisize abangabahlolokazi ngenene.

<span id="an_5:17">5:17</span> Amadoda amakhulu, awongamela kakuhle,
makabalelwe ekuthini afanelwe yimbeko ephindwe kabini, ngokukodwa lawo
abulalekayo elizwini nasemfundisweni.

<span id="an_5:18">5:18</span> Kuba sitsho isibhalo ukuthi, Uze
ungayibophi umlomo inkomo ibhula; nokuthi, Umsebenzi uwufanele umvuzo
wakhe.

<span id="an_5:19">5:19</span> Ukumangalelwa kwendoda enkulu musa
ukukwamkela, kungabangakho mangqina mabini nokuba mathathu.

<span id="an_5:20">5:20</span> Abo bonayo, bohlwaye emehlweni abantu
bonke, ukuze nabanye aba babe nokoyika.

<span id="an_5:21">5:21</span> Ndiyakuqononondisa emehlweni kaThixo,
neNkosi yethu uYesu Kristu, nezithunywa zezulu ezinyuliweyo, ukuba
uzigcine ezi zinto ungaqali ugwebe, ungenzi nto ngokwayama bani.

<span id="an_5:22">5:22</span> Ungangxami ubeke izandla phezu kukabani,
kanjalo ungadlelani nabanye ngezono zabo; zigcine nyulu.

<span id="an_5:23">5:23</span> Musa ukuba sasela manzi; sela intwana
yewayini, ngenxa yesisu sakho, nobulwelwe bakho obufuthi.

<span id="an_5:24">5:24</span> Izono zabanye abantu zisekuhleni kanye,
ziyandulela ukuya emgwebeni; abanye ke ziyabalandela nokubalandela.

<span id="an_5:25">5:25</span> Kwangokunjalo nemisebenzi emihle
isekuhleni kanye, naleyo ingenjalo ayinako ukufihlakala.

<span id="an_6:1">6:1</span> Onke ke amakhoboka aphantsi kwedyokhwe,
ndinga ke angababalela abaniniwo ekuthini, bafanelwe yimbeko yonke,
ukuze igama likaThixo nemfundiso inganyeliswa.

<span id="an_6:2">6:2</span> Lawo ke ankosi zikholiweyo, makangazideli
ngakuba zingabazalwana; makube kukhona azikhonzayo, ngokuba
zingamakholwa, ziziintanda ezingamadlelana ngomsebenzi wokusiza.
Bafundise ezi zinto, ubavuselele.

<span id="an_6:3">6:3</span> Ukuba umntu ufundisa mfundiso yimbi,
angavumani namazwi aphilileyo angaweNkosi yethu uYesu Kristu, nemfundiso
engokokuhlonela uThixo:

<span id="an_6:4">6:4</span> uthiwe lwale likratshi, engazi lutho
kakuhle; ufa kukukhanuka iimbuzwano namaphike amazwi, ekuthi kwezo nto
kuphume umona, iinkani, izinyeliso, izirhano ezingendawo,

<span id="an_6:5">6:5</span> iimpikiswano zochuku lwabantu abonakele
iingqiqo, abahluthwe inyaniso, beba ukuhlonela uThixo kukusekeleza
inzuzo.

<span id="an_6:6">6:6</span> Ke kona ukuhlonela uThixo, kundawonye
nokwanela, kusekeleze inzuzo enkulu.

<span id="an_6:7">6:7</span> Kuba sibe singangenanga nanto ehlabathini;
kuyabonakala kananjalo ukuba asinakuphuma nanto.

<span id="an_6:8">6:8</span> Sinento ke edliwayo neyambathwayo, masanele
zezo zinto.

<span id="an_6:9">6:9</span> Ke abo banga bangaba zizityebi, beyela
ekuhendweni, nasesibatheni, nasezinkanukweni ezininzi ezibubudenge
ezenzakalisayo; zona zibatshonisayo abantu ekonakaleni,
nasekutshabalaleni.

<span id="an_6:10">6:10</span> Kuba ukuthanda imali yingcambu yeento
zonke ezimbi: abathe abathile, ngokuzolulela kuko, balahlekana nokholo,
bazibhodloza ngobuhlungu obuninzi.

<span id="an_6:11">6:11</span> Ke wena, mntu kaThixo, zibaleke ezo
zinto; phuthuma ke ubulungisa, ukuhlonela uThixo, ukholo, uthando,
unyamezelo, ubulali.

<span id="an_6:12">6:12</span> Wuzame umzamo omhle wokholo; bubambe
ubomi obungunaphakade, owabizelwa nokubizelwa kubo, waza waluvuma uvumo
oluhle emehlweni amangqina amaninzi.

<span id="an_6:13">6:13</span> Ndiyakuthethela emehlweni kaThixo,
ozidlisa ubomi zonke izinto, nasemehlweni kaKristu Yesu, owangqina
wavuma uvumo oluhle phambi koPontiyo Pilato,

<span id="an_6:14">6:14</span> ukuba uwugcine umthetho lo, ungabi
nabala, ungabi nakukhalazelwa, kude kufike ukubonakala kweNkosi yethu
uYesu Kristu,

<span id="an_6:15">6:15</span> aya kukubonakalisa ngawakhe amaxesha;
lowo usikelelekileyo, ukuphela konamandla, uKumkani kakumkani, uNkosi
kankosi;

<span id="an_6:16">6:16</span> okuphela konokungafi, ehleli ekukhanyeni
okungenakusondelwa; ekungekho namnye kubantu owakha wambona, nonako
ukumbona; kuye makubekho imbeko namandla angunaphakade. Amen.

<span id="an_6:17">6:17</span> Abazizityebi kweli phakade langoku
bathethele bangakratshi, bangathembeli butyebini bungagqalekiyo;
mabathembele kuThixo ophilileyo, obeka phambi kwethu izinto zonke
ngokobutyebi, ukuba zidliwe;

<span id="an_6:18">6:18</span> benze okulungileyo, babe zizityebi
zemisebenzi emihle, babe nokwaba kakuhle, babe nobudlelane;

<span id="an_6:19">6:19</span> baziqwebele isiseko esihle, base exesheni
eliza kuza, ukuze babubambe ubomi obungunaphakade.

<span id="an_6:20">6:20</span> Wa, Timoti, loo nto uyiphathisiweyo
yigcine, uphambuke ezintethweni ezingezizo, ezihlambelayo, nakwiinkcaso
