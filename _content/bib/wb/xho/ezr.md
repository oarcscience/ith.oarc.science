### Xhosa Bible: Ezra

<span id="an_1:1">1:1</span> Ke kaloku ngowokuqala umnyaka kaKoreshi,
ukumkani wasePersi, ukuze kuzaliseke ilizwi likaYehova ngomlomo
kaYeremiya, uYehova wawuxhokonxa umoya kaKoreshi, ukumkani wasePersi;
wahambisa izwi ebukumkanini bakhe bonke, walibhala, esithi,

<span id="an_1:2">1:2</span> Utsho uKoreshi ukumkani wasePersi, ukuthi,
Zonke izikumkani zehlabathi uzinike mna uYehova, uThixo wamazulu. Yena
undiyalele ukuba ndimakhele indlu eYerusalem, ekwaYuda.

<span id="an_1:3">1:3</span> Ubani ophakathi kwenu, ebantwini bakhe
bonke, uThixo wakhe makabe naye, anyuke aye eYerusalem ekwaYuda, ayakhe
indlu kaYehova, uThixo kaSirayeli (nguye uThixo), eseYerusalem.

<span id="an_1:4">1:4</span> Onke amasalela, ezindaweni zonke
aphambukele kuzo, makancedwe ngamadoda aloo ndawo akuyo ngesilivere,
negolide, nempahla, neenkomo, kunye nomnikelo wabaqhutywa yintliziyo,
wendlu kaThixo eseYerusalem.

<span id="an_1:5">1:5</span> Zesuka ke iintloko zezindlu zooyise
zakwaYuda, nezakwaBhenjamin, nezababingeleli, nezabaLevi, bonke abo
uThixo wawuxhokonxayo umoya wabo, ukuba banyuke, baye kuyakha indlu
kaYehova eseYerusalem.

<span id="an_1:6">1:6</span> Bonke ababebangqongile babancedisa ngezitya
zesilivere, nangegolide, nangempahla, nangeenkomo, nangezinto
ezinqabileyo; ngaphandle kwazo zonke ezibe zinikwe ngabantu ngokuqhutywa
yintliziyo.

<span id="an_1:7">1:7</span> Ke ukumkani uKoreshi wazirhola iimpahla
zendlu kaYehova, awayezithabathe eYerusalem uNebhukadenetsare, wazibeka
endlwini yoothixo bakhe.

<span id="an_1:8">1:8</span> UKoreshi ukumkani wasePersi wazirhola,
wazinikela kuMitredati, umgcini-ndyebo, wazibalela kuSheshbhatsare,
isikhulu sakwaYuda.

<span id="an_1:9">1:9</span> Lilo eli inani lazo: izitya zegolide
zamashumi mathathu, izitya zesilivere zaliwaka, iinkamba zamashumi
mabini anesithoba, iindebe zegolide zamashumi mathathu,

<span id="an_1:10">1:10</span> iindebe zesilivere zohlobo lwesibini
zamakhulu mane anashumi linye, iimpahla ezizimbi zaliwaka.

<span id="an_1:11">1:11</span> Zonke iimpahla zegolide nezesilivere zaba
mawaka mahlanu anamakhulu amane. Zonke ezo nto wazinyusa uSheshbhatsare,
ekunyusweni kwababethinjelwe eBhabheli, beziswa eYerusalem.

<span id="an_2:1">2:1</span> Ngabo ke aba oonyana belizwe lakwaYuda
abenyukayo, kwababethinjwe basemfudusweni, awabafudusela eBhabheli
uNebhukadenetsare, ukumkani waseBhabheli, ababuyela eYerusalem
nakwaYuda, elowo wabuyela emzini wakowabo;

<span id="an_2:2">2:2</span> abeza noZerubhabheli, noYeshuwa,
noNehemiya, noSeraya, noRehelaya, noMordekayi, noBhileshan, noMispare,
noBhigwayi, noRelum, noBhahana. Inani lamadoda abantu bakwaSirayeli:

<span id="an_2:3">2:3</span> oonyana bakaParoshe ibingamawaka amabini,
anekhulu elinamanci asixhenxe anababini.

<span id="an_2:4">2:4</span> Oonyana bakaShefatiya ibingamakhulu
amathathu, anamanci asixhenxe anababini.

<span id="an_2:5">2:5</span> Oonyana baka-Ara ibingamakhulu asixhenxe,
anamanci asixhenxe anabahlanu.

<span id="an_2:6">2:6</span> Oonyana bakaPahati wakwaMowabhi, koonyana
bakaYeshuwa noYowabhi, ibingamawaka amabini anamakhulu asibhozo,
aneshumi elinye elinababini.

<span id="an_2:7">2:7</span> Oonyana bakaElam ibiliwaka elinamakhulu
amabini, anamanci mahlanu anabane.

<span id="an_2:8">2:8</span> Oonyana bakaZatu ibingamakhulu asithoba,
anamanci mane anabahlanu.

<span id="an_2:9">2:9</span> Oonyana bakaZakayi ibingamakhulu asixhenxe,
anamanci mathandathu.

<span id="an_2:10">2:10</span> Oonyana bakaBhani ibingamakhulu
amathandathu, anamanci mane anababini.

<span id="an_2:11">2:11</span> Oonyana bakaBhebhayi ibingamakhulu
amathandathu, anamanci mabini anabathathu.

<span id="an_2:12">2:12</span> Oonyana baka-Azegadi ibiliwaka
elinamakhulu amabini, anamanci mabini anababini.

<span id="an_2:13">2:13</span> Oonyana baka-Adonikam ibingamakhulu
amathandathu, anamanci mathandathu anabathandathu.

<span id="an_2:14">2:14</span> Oonyana bakaBhigwayi ibingamawaka
amabini, anamanci mahlanu anabathandathu.

<span id="an_2:15">2:15</span> Oonyana baka-Adin ibingamakhulu amane,
anamanci mahlanu anabane.

<span id="an_2:16">2:16</span> Oonyana baka-Atere kaHezekiya
ibingamashumi asithoba anesibhozo.

<span id="an_2:17">2:17</span> Oonyana bakaBhetsayi ibingamakhulu
amathathu, ana manci mabini anabathathu.

<span id="an_2:18">2:18</span> Oonyana bakaYora ibilikhulu elineshumi
elinababini.

<span id="an_2:19">2:19</span> Oonyana bakaHashum ibingamakhulu amabini,
anamanci mabini anabathathu.

<span id="an_2:20">2:20</span> Oonyana baseGibhare ibingamashumi
asithoba anabahlanu.

<span id="an_2:21">2:21</span> Oonyana baseBhetelehem ibilikhulu,
elinamanci mabini anabathathu.

<span id="an_2:22">2:22</span> Amadoda aseNetofa ibingamashumi amahlanu
anamathandathu.

<span id="an_2:23">2:23</span> Amadoda aseAnatoti ibilikhulu, elinamanci
mabini anesibhozo.

<span id="an_2:24">2:24</span> Oonyana baseAzemavete ibingamashumi amane
anababini.

<span id="an_2:25">2:25</span> Oonyana baseKiriyati-yeharim neKefira
neBheroti, ibingamakhulu asixhenxe, anamanci amane anabathathu.

<span id="an_2:26">2:26</span> Oonyana baseRama neGabha ibingamakhulu
amathandathu, anamanci mabini anamnye.

<span id="an_2:27">2:27</span> Amadoda aseMikimase ibilikhulu,
elinamanci mabini anamabini.

<span id="an_2:28">2:28</span> Amadoda aseBheteli neAyi ibingamakhulu
amabini, anamanci mabini anamathathu.

<span id="an_2:29">2:29</span> Oonyana baseNebho ibingamashumi amahlanu
anababini.

<span id="an_2:30">2:30</span> Oonyana bakaMagebhishe ibilikhulu,
elinamanci mahlanu anabathandathu.

<span id="an_2:31">2:31</span> Oonyana bomnye uElam ibiliwaka,
elinamakhulu amabini, anamanci amahlanu anabane.

<span id="an_2:32">2:32</span> Oonyana bakaHarim ibingamakhulu
amathathu, anamanci mabini.

<span id="an_2:33">2:33</span> Oonyana baseLodi neHadide neOno
ibingamakhulu asixhenxe, anamanci mabini anabahlanu.

<span id="an_2:34">2:34</span> Oonyana baseYeriko ibingamakhulu
amathathu, anamanci mane anabahlanu.

<span id="an_2:35">2:35</span> Oonyana baseSenaha ibingamawaka
amathathu, anamakhulu amathandathu, anamanci mathathu.

<span id="an_2:36">2:36</span> Ababingeleli: oonyana bakaYedaya bendlu
kaYeshuwa ibingamakhulu asithoba, anamanci asixhenxe anabathathu.

<span id="an_2:37">2:37</span> Oonyana bakaImere ibiliwaka, elinamanci
mahlanu anababini.

<span id="an_2:38">2:38</span> Oonyana bakaPashure ibiliwaka
elinamakhulu amabini, anamanci mane anesixhenxe.

<span id="an_2:39">2:39</span> Oonyana bakaHarim ibiliwaka, elineshumi
elinesixhenxe.

<span id="an_2:40">2:40</span> AbaLevi: oonyana bakaYeshuwa
noKademiyeli, koonyana bakaHodaviya, ibingamashumi asixhenxe anabane.

<span id="an_2:41">2:41</span> Iimvumi: oonyana baka-Asafu ibilikhulu,
elinamanci mabini anesibhozo.

<span id="an_2:42">2:42</span> Oonyana babamasango: oonyana bakaShalum,
oonyana baka-Atere, oonyana bakaTalemon, oonyana baka-Akubhi, oonyana
bakaHatita, oonyana bakaShobhayi, bebonke ibilikhulu, elinamanci
mathathu anesithoba.

<span id="an_2:43">2:43</span> Abakhonzi betempile: oonyana bakaTsiha,
oonyana bakaHasufa, oonyana bakaTabhayoti,

<span id="an_2:44">2:44</span> oonyana bakaKerose, oonyana bakaSiyaha,
oonyana bakaPadon,

<span id="an_2:45">2:45</span> oonyana bakaLebhana, oonyana bakaHagabha,
oonyana baka-Akubhi,

<span id="an_2:46">2:46</span> oonyana bakaHagabhi, oonyana
bakaShalemayi, oonyana bakaHanan,

<span id="an_2:47">2:47</span> oonyana bakaGidele, oonyana bakaGahare,
oonyana bakaRehaya,

<span id="an_2:48">2:48</span> oonyana bakaRetsin, oonyana bakaNekoda,
oonyana bakaGazam,

<span id="an_2:49">2:49</span> oonyana bakaUza, oonyana bakaPaseha,
oonyana bakaBhesayi,

<span id="an_2:50">2:50</span> oonyana baka-Asna, oonyana bakaMehunim,
oonyana bakaNefusim,

<span id="an_2:51">2:51</span> oonyana bakaBhakebhuki, oonyana
bakaHakufa, oonyana bakaHar-hure,

<span id="an_2:52">2:52</span> oonyana bakaBhatseluti, oonyana
bakaMehida, oonyana bakaHarsha,

<span id="an_2:53">2:53</span> oonyana bakaBharkose, oonyana bakaSisera,
oonyana bakaTama,

<span id="an_2:54">2:54</span> oonyana bakaNetsiha, oonyana bakaHatifa.

<span id="an_2:55">2:55</span> Oonyana babakhonzi bakaSolomon: oonyana
bakaSotayi, oonyana bakaSoferete, oonyana bakaPeruda,

<span id="an_2:56">2:56</span> oonyana bakaYahala, oonyana bakaDarkon,
oonyana bakaGidele,

<span id="an_2:57">2:57</span> oonyana bakaShefatiya, oonyana
bakaHatili, oonyana bakaPokerete-tsebhayim, oonyana baka-Ami.

<span id="an_2:58">2:58</span> Bonke abakhonzi betempile, noonyana
babakhonzi bakaSolomon, ibingamakhulu amathathu, anamanci asithoba
anababini.

<span id="an_2:59">2:59</span> Ngaba abenyuka, bevela eTele-mela,
naseTele-harsha, naseKerubhi-adan-imere; ke ababanga nakuzixela izindlu
zooyise, nembewu yabo, ukuba ngabakwaSirayeli na:

<span id="an_2:60">2:60</span> oonyana bakaDelaya, oonyana bakaTobhiya,
oonyana bakaNekoda, babengamakhulu omathandathu anamanci mahlanu
anababini.

<span id="an_2:61">2:61</span> Ke koonyana bababingeleli: oonyana
bakaHabhaya, oonyana bakaHakotsi, oonyana bakaBharzilayi (owazeka umfazi
ezintombini zikaBharzilayi waseGiliyadi, wabizwa ngegama lazo),

<span id="an_2:62">2:62</span> abo bakufuna ukubhalwa kwabo
kwababhalelwe emilibeni yokuzalwa, abakufumana; bangcola ke, ababa
sebubingelelini.

<span id="an_2:63">2:63</span> Irhuluneli yathi kubo, mabangadli nto
kwezingcwele kanye, kude kuvele umbingeleli oneeUrim neeTumim.

<span id="an_2:64">2:64</span> Lonke ibandla, lindawonye, ibe ingamawaka
angamashumi amane anamabini, anamakhulu mathathu, anamanci mathandathu;

<span id="an_2:65">2:65</span> ngaphandle kwezicaka zabo ezi,
nezicakakazi zabo ezi, zingamawaka asixhenxe, anamakhulu amathathu,
anamanci mathathu anesixhenxe; zineemvumi neemvumikazi ezingamakhulu
amabini.

<span id="an_2:66">2:66</span> Amahashe abo ibingamakhulu asixhenxe,
anamanci mathathu anamathandathu; oondlebende babo ibingamakhulu
amabini, anamanci mane anabahlanu;

<span id="an_2:67">2:67</span> iinkamela zabo ibingamakhulu amane,
anamanci mathathu anantlanu; amaesile ibingamawaka amathandathu,
anamakhulu asixhenxe, anamanci mabini.

<span id="an_2:68">2:68</span> Inxenye yeentloko zezindlu zooyise
ekufikeni kwayo endlwini kaYehova eseYerusalem, yayizisela indlu kaThixo
iminikelo yokuqhutywa yintliziyo, ukuba imiswe endaweni yayo.

<span id="an_2:69">2:69</span> Banikela ngoko konke ababenako
ebuncwaneni bomsebenzi: igolide yangamawaka amashumi amathandathu,
anawaka linye eedarike, nesilivere yangamawaka amahlanu eemina, nekhulu
leengubo ezingaphantsi zababingeleli.

<span id="an_2:70">2:70</span> Bahlala ke ababingeleli, nabaLevi,
nabantu inxenye, neemvumi, nabamasango, nabakonzi betempile, emizini
yakowabo; amaSirayeli onke emizini yawo.

<span id="an_3:1">3:1</span> Ithe yakufika inyanga yesixhenxe, oonyana
bakaSirayeli besemizini yabo, bahlangana abantu njengamntu mnye, beza
eYerusalem.

<span id="an_3:2">3:2</span> Wesuka uYeshuwa unyana kaYotsadaki,
nabazalwana bakhe ababingeleli, noZerubhabheli unyana kaShelatiyeli,
nabazalwana bakhe, basakha isibingelelo sikaThixo kaSirayeli, ukuba
kunyuswe phezu kwaso amadini anyukayo, ngoko kubhaliweyo emyalelweni
kaMoses umfo wakwaThixo.

<span id="an_3:3">3:3</span> Basizinzisa isibingelelo endaweni yaso;
ngokuba beboyikiswe zizizwe zala mazwe. Banyusa phezu kwaso amadini
anyukayo kuYehova, amadini anyukayo akusasa nawangokuhlwa.

<span id="an_3:4">3:4</span> Benza umthendeleko weminquba njengoko
kubahliweyo, namadini anyukayo iimini ngeemini, ngenani, ngokwesiko,
into yemini ngangemini yayo.

<span id="an_3:5">3:5</span> Emveni kwawo benza idini elinyukayo
lamaxesha onke, nawokuthwasa kwenyanga, nawamaxesha onke kaYehova
angcwalisiweyo, amisiweyo, nawabo bonke abanikela ngokuqhutywa
yintliziyo kuYehova, iminikelo yokuqhutywa yintliziyo.

<span id="an_3:6">3:6</span> Baqala ugowokuqala umhlawenyanga
yesixhenxe, baqala banyusa amadini anyukayo kuYehova; itempile kaYehova
ibe ingekasekwa.

<span id="an_3:7">3:7</span> Imali bayinika abaqingqi nabakroli; banika
amaTsidon namaTire ukudla, nento eselwayo, neoli, ukuba athabathe
imisedare eLebhanon, ayizise ngolwandle eYafo, ngokuvunyelwa kwabo
nguKoreshi ukumkani wasePersi.

<span id="an_3:8">3:8</span> Ngomnyaka wesibini wokufika kwabo endlwini
kaThixo eYerusalem, ngenyanga yesibini, baqala ooZerubhabheli unyana
kaShelatiyeli, noYeshuwa unyana kaTsadoki, nabanye abazalwana babo,
ababingeleli, nabaLevi, nabo bonke ababephume ekuthinjweni ukuza
eYerusalem, bamisa abaLevi, bathabathela kominyaka imashumi mabini
ezelwe, kwenyusa, ukuba bawongamele umsebenzi wendlu kaYehova.

<span id="an_3:9">3:9</span> Kwema ooYeshuwa noonyana bakhe nabazalwana
bakhe, uKademiyeli noonyana bakhe, oonyana bakaHodaviya, njengamntu
mnye, ukubongamela abawenzayo umsebenzi endlwini kaThixo: oonyana
bakaHenadade, oonyana babo nabazalwana babo, abaLevi.

<span id="an_3:10">3:10</span> Abakhi bayiseka itempile kaYehova; bema
ababingeleli, bevethe, benamaxilongo; nabaLevi, oonyana baka-Asafu,
benamacangci, ukumdumisa uYehova ngokolwalathiso lukaDavide ukumkani
wakwaSirayeli.

<span id="an_3:11">3:11</span> Basusela ngokudumisa, nangokubulela
kuYehova, besithi, Ulungile, inceba yakhe ingunaphakade kuSirayeli; ke
bonke abantu baduma ngodumo olukhulu, bedumisa uYehova ngenxa yokusekwa
kwendlu kaYehova.

<span id="an_3:12">3:12</span> Ke isininzi kubabingeleli, nabaLevi,
neentloko zezindlu zooyise, namadoda amakhulu, awayeyibonile indlu
yokuqala, sathi ekusekweni kwale ndlu emehlweni aso, salila ngezwi
elikhulu; ke abaninzi baduma, bavuya ngokuphakamisa ilizwi.

<span id="an_3:13">3:13</span> Abantu ababa nakulahlula ilizwi lokuduma
ngokuvuya kwilizwi lokulila kwabantu; ngokuba abantu babeduma ngodumo
olukhulu; ilizwi lavakala nakude.

<span id="an_4:1">4:1</span> Ke kaloku, bakuva ababandezeli bamaYuda
nabamaBhenjamin, ukuba oonyana basemfudusweni bamakhela itempile uYehova
uThixo wakwaSirayeli,

<span id="an_4:2">4:2</span> beza kuZerubhabheli, nakwiintloko zezindlu
zooyise, bathi kubo, Masakhe nani, ngokuba siquqela uThixo wenu
njengani; sibingelela kuye kususela kwimihla kaEsar-hadon, ukumkani
waseAsiriya, owasinyusela apha.

<span id="an_4:3">4:3</span> Wathi uZerubhabheli, noYeshuwa, nabanye
abaziintloko zezindlu zooyise bakwaSirayeli kubo, Anithe ni nathi
ekumakheleni uThixo wethu indlu; somakhela sedwa uYehova, uThixo
wakwaSirayeli, njengoko wasiwisela umthetho ngako ukumkani onguKoreshi,
ukumkani wasePersi.

<span id="an_4:4">4:4</span> Abantu belo zwe bamana ukubaphelisa amandla
abantu bakwaYuda, beboyikisa ekwakheni;

<span id="an_4:5">4:5</span> bebaqeshela amagqwetha, ukuba alishitshise
iqhinga labo, yonke imihla kaKoreshi ukumkani wasePersi, kwada kwezisa
ebukumkanini bukaDariyo ukumkani wasePersi.

<span id="an_4:6">4:6</span> Nasebukumkanini buka-Ahaswerose,
ekuqalekeni kobukumkani bakhe, babhala, bebancetheza abemi bakwaYuda
nabaseYerusalem.

<span id="an_4:7">4:7</span> Nangemihla ka-Artashashta kwabhala
uBhish-lam, noMitredati, noTabheli, nabanye abalingane babo,
kuArtashashta ukumkani wasePersi; incwadi yabhalwa ngentetho yesiAram,
yakhunyushwa ngesiAram.

<span id="an_4:8">4:8</span> OoRehum umkhumbuzi wezinto zakomkhulu,
noShimshayi umbhali, babhala incwadi ngeYerusalem kuArtashashta
ukumkani, ngolu hlobo.

<span id="an_4:9">4:9</span> Bathumela ke ooRehum umkhumbuzi wezinto
zakomkhulu, noShimshayi umbhali, nabanye abalingane babo bamaDina,
nabama-Afarsateki, nabamaTarpela, nabama-Afarsi, nabama-Arki,
nabamaBhabheli, nabamaShushanki, nabamaDehavi, nabamaElam,

<span id="an_4:10">4:10</span> nezinye izizwe, awazithimbayo uOsnapare
omkhulu obekekileyo, wazibeka emzini wakwaSamari, nakweminye nganeno
koMlambo, nakwezinye iindawo.

<span id="an_4:11">4:11</span> Yiyo le incwadi abayithumela
kuArtashashta ukumkani, ithi: Abakhonzi bakho, amadoda anganeno
koMlambo, njalonjalo.

<span id="an_4:12">4:12</span> Makwazeke kukumkani, ukuba amaYuda
anyukileyo evela kuwe, afikile kuthi eYerusalem; ayawakha umzi
ogwilikayo, ongendawo; ayazimisa iindonga zawo, ayazimbela iziseko.

<span id="an_4:13">4:13</span> Makwazeke ngoku kukumkani, ukuba othi, xa
uthe wakhiwa lo mzi, zafezwa neendonga zawo, angarholi rhafu
nasithabathaba, nalunikelo, kude kuye kuxhwaleka ookumkani.

<span id="an_4:14">4:14</span> Kaloku, ngenxa enokuba sisidla ityuwa
yasebhotwe, kwaye kungafanelekile kuthi ukubona ukumkani encitshiswa.
Ngako oko siyathumela, simazisa ukumkani,

<span id="an_4:15">4:15</span> ukuze kuphicothwe encwadini yezikhumbuzo
ezo zooyihlo, wazi ukuba lo mzi waba ngumzi ogwilikayo, owaxhwalekisa
ookumkani namazwe, kwenziwa, isiphendu phakathi kwawo kwakudala, waba
linxuwa lo mzi ngenxa yoko.

<span id="an_4:16">4:16</span> Siyamazisa ukumkani ukuba, xa uthe
wakhiwa lo mzi, zafezwa neendonga zawo, wothi ngenxa yelo thuba ungabi
nasabelo wena nganeno koMlambo.

<span id="an_4:17">4:17</span> Waza ukumkani wabathumela impendulo
ooRehum umkhumbuzi wezinto zakomkhulu, noShimshayi umbhali, nabanye
abalingane babo abahleli kwaSamari, nabanye abanganeno koMlambo, wathi:
Uxolo, njalonjalo.

<span id="an_4:18">4:18</span> Loo ncwadi nayithumelayo kuthi ilesiwe
ngokucacileyo phambi kwam.

<span id="an_4:19">4:19</span> Ndiwise umthetho, kwaphicothwa,
kwafunyanwa ukuba lo mzi kwakudala uziphakamisile kookumkani, kwenziwa
ugwiliko nesiphendu phakathi kwawo.

<span id="an_4:20">4:20</span> Kwabe kukho ookumkani abanamandla
eYerusalem, nabanegunya kwilizwe lonke eliphesheya koMlambo apho;
kunikelwa kubo irhafu, nesithabathaba, nonikelo.

<span id="an_4:21">4:21</span> Wisani ngoko umthetho wokuba abanjezelwe
loo madoda, loo mzi ungakhiwa, kude kuwiswe umthetho ndim.

<span id="an_4:22">4:22</span> Lumkani, ningakhe le nto niyiyeke;
kungani na ukuba kukhule ukonakala, baxhwaleke ookumkani?

<span id="an_4:23">4:23</span> Ithe yakuleswa incwadi ka-Artashashta
ukumkani phambi koRehum, noShimshayi umbhali, nabalingane babo, basuka,
beza ngobungxamo eYerusalem kumaYuda, bawanyanzela ngegunya nangamandla.

<span id="an_4:24">4:24</span> Waza wema umsebenzi wendlu kaThixo
eseYerusalem; wema kwada kwaba ngumnyaka wesibini wobukumkani
bukaDariyo, ukumkani wasePersi.

<span id="an_5:1">5:1</span> Ke kaloku abaprofeti, ooHagayi umprofeti,
noZekariya unyana kaIdo, baprofeta kumaYuda abekwaYuda neYerusalem,
egameni likaThixo kaSirayeli obekubo.

<span id="an_5:2">5:2</span> Baza basuka ooZerubhabheli unyana
kaShelatiyeli, noYeshuwa unyana kaYotsadaki, nabathetheli bakhe, ukuba
bayakhe indlu kaThixo eYerusalem, abaprofeti bakaThixo bekubo
bencedisana nabo.

<span id="an_5:3">5:3</span> Kwangelo xesha, weza kubo uTatenayi, ibamba
langaneno koMlambo, noShetar-bhozenayi, nabalingane babo, bathi kubo,
Niwiselwe umthetho ngubani na ukuba niyakhe le ndlu, nilwenze olu donga?

<span id="an_5:4">5:4</span> Saza senjenje ukuthetha kubo, ngoonantsi
noonantsi, amagama amadoda ayakhayo le ndlu.

<span id="an_5:5">5:5</span> Ke iliso loThixo wawo libe liphezu
kwamadoda amakhulu amaYuda, ukuba bangawabambezeli, ide loo ndawo iye
kuDariyo, bakhuphe incwadi ngayo loo ndawo.

<span id="an_5:6">5:6</span> Incwadi leyo, ooTatenayi, ibamba langaneno
koMlambo, noShetar-bhozenayi, nabalingane bakhe bama-Afarseka banganeno
koMlambo, bayithumelayo kuDariyo ukumkani,

<span id="an_5:7">5:7</span> bayithumela loo ndawo kuye; kwabhalwa
kwathiwa kuyo: KuDariyo ukumkani, uxolo luphelele.

<span id="an_5:8">5:8</span> Makwazeke kukumkani, ukuba saya ezweni
lakwaYuda, endlwini kaThixo omkhulu. Iyakhiwa ngamatye anzima, nemithi
iyafakwa ezindongeni; lo msebenzi usetyenzwa ngenyameko, uyaphumelela
ezandleni zabo.

<span id="an_5:9">5:9</span> Saza sabuza kumadoda lawo makhulu, sathi
kuwo, Niwiselwe umthetho ngubani na ukuba niyakhe le ndlu, nilwenze olu
donga?

<span id="an_5:10">5:10</span> Sabuza kuwo namagama awo, ukuba sikwazise
wena, siwabhale amagama amadoda aziintloko kuwo.

<span id="an_5:11">5:11</span> Enjenje ke ukusibuyisela impendulo, athi,
Singabakhe, singabakhonzi bakaThixo wamazulu nehlabathi. Sakha le ndlu
yakhiwayo iminyaka emininzi eyadlulayo, yakhiwa yafezwa ngukumkani
omkhulu wakwaSirayeli.

<span id="an_5:12">5:12</span> Ngokokuba oobawo bamqumbisayo uThixo
wamazulu, wabanikela esandleni sikaNebhukadenetsare ukumkani
waseBhabheli, umKaledi. Yena wayichitha le ndlu, wabafudusela eBhabheli
abantu.

<span id="an_5:13">5:13</span> Kodwa ngomnyaka wokuqala kaKoreshi
ukumkani waseBhabheli, ukumkani uKoreshi wawisa umthetho wokuba yakhiwe
le ndlu kaThixo.

<span id="an_5:14">5:14</span> Kananjalo iimpahla zendlu kaThixo
zegolide nezesilivere, awazirholayo uNebhukadenetsare etempileni
eseYerusalem, awazizisayo etempileni yaseBhabheli, ezo wazikhupha
uKoreshi ukumkani etempileni yaseBhabheli: zanikelwa kogama
linguSheshbhatsare, awammisayo ukuba abe libamba.

<span id="an_5:15">5:15</span> Wathi kuye, Thabatha ezi mpahla, hamba,
uzise etempileni eseYerusalem, indlu kaThixo yakhelwe endaweni yayo.

<span id="an_5:16">5:16</span> Waza weza lo Sheshbhatsare, wazibeka
iziseko zendlu kaThixo eseYerusalem; kususela kwelo xesha unangoku
iyakhiwa; ayikagqitywa ke.

<span id="an_5:17">5:17</span> Kaloku, ukuba kuthe kwalunga kukumkani,
makuphicothwe endlwini yendyebo yokumkani, eseBhabheli apho, kukhangelwe
ukuba kunjalo na, ukuba wawiswa na umthetho nguKoreshi ukumkani, wokuba
yakhiwe le ndlu kaThixo eYerusalem; kananjalo makuthunyelwe kuthi
ukuthanda kokumkani ngale nto.

<span id="an_6:1">6:1</span> Waza uDariyo ukumkani wawisa umthetho;
kwaphicothwa endlwini khona eBhabheli.

<span id="an_6:2">6:2</span> Kwafunyanwa komkhulu, eAmeta, eselizweni
lamaMedi, incwadi esongwayo; kwaye kubhaliwe apho isikhumbuzo, kwathiwa:

<span id="an_6:3">6:3</span> Ngowokuqala umnyaka kaKoreshi ukumkani,
wawisa umthetho uKoreshi ukumkani, wathi, Indlu kaThixo eYerusalem,
mayakhiwe indlu leyo, ibe yindawo apho kubingelelwa khona imibingelelo;
zimiswe neziseko zayo; ukuphakama kwayo kube ziikubhite ezimashumi
mathandathu, ububanzi bayo bube ziikubhite ezimashumi mathandathu.

<span id="an_6:4">6:4</span> Ize ibe ziingcamba ezintathu zamatye
anzima, nocamba lwemithi emitsha; indleko mayiphume endlwini yokumkani.

<span id="an_6:5">6:5</span> Neempahla zendlu kaThixo zegolide,
nezesilivere, awazirholayo uNebhukadenetsare etempileni yaseYerusalem,
wazizisa eBhabheli, mazibuyiswe zisiwe etempileni eseYerusalem,
ezindaweni zazo, endlwini kaThixo.

<span id="an_6:6">6:6</span> Ngoko, Tatenayi, bamba laphesheya koMlambo,
nawe Shetar-bhozenayi, nabalingane benu bama-Afarseka phesheya koMlambo,
yibani kude naloo nto;

<span id="an_6:7">6:7</span> wuyekeni umsebenzi wale ndlu kaThixo.
Ibamba lamaYuda namadoda amakhulu akwaYuda mabayakhe le ndlu kaThixo
endaweni yayo.

<span id="an_6:8">6:8</span> Ndiwisa umthetho wento enoyenza kumadoda
amakhulu aloo maYuda, ukuba yakhiwe loo ndlu kaThixo; ndithi, loo madoda
makanikwe indleko, ebutyebini bukakumkani, erhafini yaphesheya koMlambo,
kunyanyekwe, angabanjezelwa.

<span id="an_6:9">6:9</span> Nezinto ezifunekayo, iinkunzi ezintsha
zeenkomo, neenkunzi zeegusha, neemvana, zokuba ngamadini anyukayo
kuThixo wamazulu, nengqolowa, netyuwa, newayini, neoli, ngokwelizwi
lababingeleli abaseYerusalem, makazinikwe imihla ngemihla,
kungaphosiswa;

<span id="an_6:10">6:10</span> ukuze basondeze imibingelelo elivumba
elithozamisayo kuThixo wamazulu, bathandazele ubomi bokumkani noboonyana
bakhe.

<span id="an_6:11">6:11</span> Ndiwisa umthetho wokuba bonke abantu
abayijikayo loo nto, makuncothulwe umthi endlwini yabo, baxhonywe,
babethelelwe kuwo, nendlu yabo yenziwe indawo yokuya endle, ngenxa yoko.

<span id="an_6:12">6:12</span> UThixo, olibeke khona igama lakhe,
makabakhahlele phantsi bonke ookumkani, nabantu, abolula izandla zabo,
beyijika loo nto, ukuba bayitshabalalise le ndlu kaThixo eseYerusalem.
Mna Dariyo ndiwise umthetho; mawenziwe ngenyameko.

<span id="an_6:13">6:13</span> Baza ooTatenayi ibamba langaneno
koMlambo, noShetar-bhozenayi, nabalingane babo, ngenxa enokuba uDariyo
ukumkani ethumile, benjenjalo ngenyameko.

<span id="an_6:14">6:14</span> Ke amadoda amakhulu amaYuda akha,
aphumelela ngokokuprofeta kukaHagayi umprofeti, noZekariya unyana kaIdo;
akha ke agqiba, ngenxa yomthetho kaThixo kaSirayeli, nangenxa yomthetho
wooKoreshi, noDariyo, noArtashashta, ookumkani basePersi.

<span id="an_6:15">6:15</span> Yagqitywa ke le ndlu ngosuka lwesithathu
lwenyanga enguAdare; ngumnyaka wesithandathu lowo wokulawula kukaDariyo
ukumkani.

<span id="an_6:16">6:16</span> Oonyana bakaSirayeli, ababingeleli
nabaLevi nabanye oonyana basemfudusweni, bakwenza ngovuyo ukuhlaziywa
kwale ndlu kaThixo.

<span id="an_6:17">6:17</span> Basondezela ukuhlaziywa kwale ndlu
kaThixo, ikhulu leenkunzi ezintsha zeenkomo, namakhulu amabini eenkunzi
zeegusha, namakhulu amane eemvana; neenkunzi zeebhokhwe zedini lesono,
ngenxa yamaSirayeli onke, zalishumi elinambini, ngokwenani lezizwe
zakwaSirayeli.

<span id="an_6:18">6:18</span> Babamisa ababingeleli ngokwezahlulo zabo,
nabaLevi ngokwamaqela abo, phezu kwenkonzo kaThixo eseYerusalem, ngoko
kubhaliweyo encwadini kaMoses.

<span id="an_6:19">6:19</span> Oonyana basemfudusweni bayenza ipasika
ngomhla weshumi elinesine wenyanga yokuqala;

<span id="an_6:20">6:20</span> ngokuba babezihlambulule ababingeleli
nabaLevi, njengamntu mnye, bahlambuluka bonke bephela; babaxhelela
ipasika oonyana bonke basemfudusweni, nabazalwana babo ababingeleli,
nabo ngokwabo.

<span id="an_6:21">6:21</span> Badla ke oonyana bakaSirayeli ababebuye
ekufudusweni, kwanabo bonke abazahlulayo ebunqambini beentlanga zelizwe,
beza kubo ukuba bamquqele uYehova, uThixo kaSirayeli.

<span id="an_6:22">6:22</span> Benza umthendeleko wezonka ezingenagwele
imihla yasixhenxe bevuyile; ngokuba uYehova ebebavuyisile
ngokuyibuyisela kubo intliziyo yokumkani waseAsiriya, ukuze izandla zabo
zomelezwe emsebenzini wendlu kaThixo, uThixo kaSirayeli.

<span id="an_7:1">7:1</span> Emveni kwezo zinto, ebukumkanini
buka-Artashashta ukumkani wasePersi, wathi uEzra, unyana kaSeraya,
unyana ka-Azariya, unyana kaHilekiya,

<span id="an_7:2">7:2</span> unyana ka-Shalum, unyana kaTsadoki, unyana
ka-Abhitubhi,

<span id="an_7:3">7:3</span> unyana ka-Amariya, unyana ka-Azariya,
unyana kaMerayoti,

<span id="an_7:4">7:4</span> unyana kaZeraya, unyana kaUzi, unyana
kaBhuki,

<span id="an_7:5">7:5</span> unyana ka-Abhishuwa, unyana kaPinehasi,
unyana kaElazare, unyana ka-Aron umbingeleli oyintloko:

<span id="an_7:6">7:6</span> uEzra lowo wenyuka, ephuma eBhabheli. Waye
engumbhali oyinkunkqele emyalelweni kaMoses, awawuwisayo uYehova, uThixo
kaSirayeli. Ukumkani wamnika konke awakucelayo ngokwesandla sikaYehova,
uThixo wakhe, esiphezu kwakhe.

<span id="an_7:7">7:7</span> Kwenyuka naye inxenye yoonyana
bakaSirayeli, neyababingeleli, nabaLevi, neemvumi, nabamasango,
nabakhonzi betempile, besiya eYerusalem, ngomnyaka wesixhenxe
ka-Artashashta ukumkani.

<span id="an_7:8">7:8</span> Wafika eYerusalem ngenyanga yesihlanu; lowo
ke ngumnyaka wesixhenxe wokumkani.

<span id="an_7:9">7:9</span> Ngokuba ibe ingumhla wokuqala wenyanga
yokuqala, awawumisela ukunyuka ngawo, ephuma eBhabheli; wafika
eYerusalem ngomhla wokuqala wenyanga yesihlanu, ngokwesandla
esilungileyo sikaThixo wakhe, esibe siphezu kwakhe;

<span id="an_7:10">7:10</span> ngokuba uEzra intliziyo yakhe
wayeyibhekise ekuwufuneni umyalelo kaYehova, awenze, afundise ummiselo
nesiko kwaSirayeli.

<span id="an_7:11">7:11</span> Yiyo le incwadi, awathi ukumkani
uArtashashta wamnika uEzra umbingeleli, umbhali obhala amazwi emithetho
kaYehova, nemimiselo yakhe kumaSirayeli:

<span id="an_7:12">7:12</span> UArtashashta, ukumkani wookumkani, kuEzra
umbingeleli, umbhali ogqibeleleyo emthethweni kaThixo wamazulu,
njalonjalo.

<span id="an_7:13">7:13</span> Ndiwise umthetho wokuba bonke
ebukumkanini bam, phakathi kwabantu bakwaSirayeli, nababingeleli babo
nabaLevi, abaqhutywa yintliziyo ukuya eYerusalem, mabahambe nawe.

<span id="an_7:14">7:14</span> Ngokuba wena uthunywa ngukumkani
namaphakathi akhe asixhenxe, ukubuzisa ngamaYuda neYerusalem,
ngokomthetho kaThixo wakho osesandleni sakho;

<span id="an_7:15">7:15</span> uyise isilivere negolide, ayinikelayo
ukumkani namaphakathi akhe, ngokuqhutywa yintliziyo, kuThixo kaSirayeli,
okhaya liseYerusalem;

<span id="an_7:16">7:16</span> nayo yonke isilivere negolide,
owoyifumana ezweni lonke laseBhabheli, kunye nomnikelo wokuqhutywa
yintliziyo wabantu nababingeleli, abanikela ngokuqhutywa yintliziyo
ngenxa yendlu kaThixo wabo, eseYerusalem.

<span id="an_7:17">7:17</span> Ngoko ke uya kuthenga ngenyameko ngale
mali iinkunzi ezintsha zeenkomo, neenkunzi zeegusha, neemvana,
neminikelo yazo yokudla, nethululwayo; uyisondeze esibingelelweni sendlu
kaThixo wenu eseYerusalem.

<span id="an_7:18">7:18</span> Nento ethe yalunga kuwe, nakubazalwana
bakho, ukwenziwa ngesilivere nangegolide eseleyo, noyenza ngokokuthanda
kukaThixo wenu.

<span id="an_7:19">7:19</span> Ke iimpahla ozinikiweyo ngenxa yombuso
wendlu kaThixo wakho, zinikele ziphelele phambi koThixo eYerusalem.

<span id="an_7:20">7:20</span> Nezinye izinto ezifunekayo zendlu kaThixo
wakho, othi kuyafuneka ukuba zinikelwe, wozirhola endlwini yendyebo
yokumkani.

<span id="an_7:21">7:21</span> Mna Artashashta, kumkani, ndiwisa
umthetho kubo bonke abagcini-ndyebo abaphesheya koMlambo, wokuba konke
aya kukubiza kuni uEzra, umbingeleli, umbhali womthetho kaThixo
wamazulu, makwenziwe ngenyameko,

<span id="an_7:22">7:22</span> kuse ekhulwini leetalente zesilivere,
nekhulu leekore zengqolowa, nekhulu leebhate zewayini, nekhulu leebhate
zeoli, netyuwa ingalinganiselwa.

<span id="an_7:23">7:23</span> Yonke into engomthetho kaThixo wamazulu
mayenzelwe indlu kaThixo wamazulu ngokuthe nqo; kuba kungani na ukuba
bufikelwe buburhalarhume ubukumkani bokumkani noonyana bakhe?

<span id="an_7:24">7:24</span> Makwazeke kuni, ukuba bonke ababingeleli,
nabaLevi, neemvumi, nabamasango, nabakhonzi betempile, nabalungiseleleli
bale ndlu kaThixo, akukho bani unagunya lakubathwesa irhafu,
nasithabathaba, nalunikelo.

<span id="an_7:25">7:25</span> Wena Ezra, ngokobulumko bukaThixo wakho
obusesandleni sakho, misa abagwebi, nabathethi bamatyala, ebantwini
bonke abaphesheya koMlambo; ndithi, bonke abayaziyo imithetho kaThixo
wakho; nabangayaziyo nobazisa.

<span id="an_7:26">7:26</span> Ke bonke abasukuba bengawenzi umthetho
kaThixo wakho, nomthetho wokumkani, malithethwe ityala labo ngenyameko,
nokuba baya kubulawa, nokuba baya kugxothwa, nokuba baya kudliwa, nokuba
baya kukhonkxwa.

<span id="an_7:27">7:27</span> Makabongwe uYehova, uThixo woobawo,
oyinikeleyo into enjalo entliziyweni yokumkani, ukuba ayihombise indlu
kaYehova eseYerusalem;

<span id="an_7:28">7:28</span> ondithobele inceba phambi kokumkani,
namaphakathi akhe, nabathetheli bonke bokumkani abangamagorha. Mna
ndomelezeka ngokwesandla sikaYehova uThixo wam esiphezu kwam; ndazibutha
kumaSirayeli iintloko zezindlu zooyise, ukuba zinyuke nam.

nokubhalwa kwazo emilibeni yokuzalwa, ezenyuka nam, ziphuma eBhabheli,
ebukumkanini buka-Artashashta ukumkani:

<span id="an_8:2">8:2</span> koonyana bakaPinehasi nguGershom; koonyana

<span id="an_8:3">8:3</span> koonyana bakaShekaniya, koonyana
bakaParoshe, nguZekariya, enamadoda abhalwe emilibeni yokuzalwa,
alikhulu elinamanci mahlanu;

<span id="an_8:4">8:4</span> koonyana bakaPahati wakwaMowabhi
nguEliyohenayi unyana kaZeraya, enamadoda angamakhulu mabini;

<span id="an_8:5">8:5</span> koonyana bakaShekaniya, ngunyana
kaYahaziyeli, enamadoda angamakhulu amathathu;

<span id="an_8:6">8:6</span> koonyana baka-Adin nguEbhede unyana
kaYonatan, enamadoda amashumi mahlanu;

<span id="an_8:7">8:7</span> koonyana bakaElam nguYeshaya unyana
ka-Ataliya, enamadoda amashumi asixhenxe;

<span id="an_8:8">8:8</span> koonyana bakaShefatiya nguZebhadiya unyana

<span id="an_8:9">8:9</span> koonyana bakaYowabhi nguObhadiya unyana
kaYehiyeli, enamadoda angamakhulu amabini aneshumi elinesibhozo;

<span id="an_8:10">8:10</span> koonyana bakaShelomiti ngunyana
kaYosifiya, enamadoda alikhulu elinamanci mathandathu;

<span id="an_8:11">8:11</span> koonyana bakaBhebhayi nguZekariya unyana
kaBhebhayi, enamadoda amashumi mabini anesibhozo;

<span id="an_8:12">8:12</span> koonyana baka-Azegadi nguYohanan unyana
kaHakatan, enamadoda alikhulu elineshumi linye;

<span id="an_8:13">8:13</span> koonyana bokugqibela baka-Adonikam
abamagama angala: nguElifelete, noYehiyeli, noShemaya, benamadoda
amashumi mathandathu;

<span id="an_8:14">8:14</span> koonyana bakaBhigwayi ngu-Utayi,
noZabhudi, enamadoda amashumi asixhenxe.

<span id="an_8:15">8:15</span> Ndabahlanganisela emlanjeni ongumAhava;
samisa khona imihla yamithathu. Ndabagqala abantu nababingeleli,
andafumana khona namnye woonyana bakaLevi.

<span id="an_8:16">8:16</span> Ndathumela ukuba kubizwe ooEliyezere,
noAriyeli, noShemaya, noElinatan, noYaribhi, noElinatan, noNatan,
noZekariya, noMeshulam, abaziintloko; noYoyaribhi noElinatan,
abaqondisi-mthetho.

<span id="an_8:17">8:17</span> Ndabasusa baya kuIdo, oyintloko endaweni
eyiKasifiya; ndabeka amazwi emlonyeni wabo, ukuba bathethe wona kuIdo
nabazalwana bakhe, abakhonzi betempile, endaweni eyiKasifiya, ukuba
basizisele abalungiseleli bendlu kaThixo wethu.

<span id="an_8:18">8:18</span> Bathi ke, ngokwesandla sikaThixo wethu
esilungileyo esiphezu kwethu, babazisa kuthi ooIshi-sekele koonyana
bakaMahali, unyana kaLevi, unyana kaSirayeli; noSherebhiya, noonyana
bakhe nabazalwana bakhe, ishumi elinesibhozo;

<span id="an_8:19">8:19</span> noHashabhiya enoYeshaya koonyana
bakaMerari nabazalwana bakhe, noonyana babo, amashumi amabini;

<span id="an_8:20">8:20</span> nakubakhonzi betempile, ababethe ooDavide
nabathetheli babanikela ukuba bakhonze kubaLevi; abakhonzi betempile
ibingamakhulu amabini anamanci mabini, bonke bebalulwe ngamagama.

<span id="an_8:21">8:21</span> Ndamema inzilo apho emlanjeni ongumAhava,
ukuba sizithobe phambi koThixo wethu, sizifunele kuye indlela ethe tye,
thina, nentsapho yethu, nemfuyo yethu yonke.

<span id="an_8:22">8:22</span> Ngokuba ndandineentloni ukuba ndicele
impi nabamahashe kukumkani, ukuba basincede ezintshabeni endleleni;
ngokuba besesithe kukumkani, Isandla sikaThixo wethu siphezu kwabo bonke
abamfunayo, ukuba kulunge; namandla akhe nomsindo wakhe uyabachasa bonke
abamshiyayo.

<span id="an_8:23">8:23</span> Sazila ukudla ke, samkhunga ngenxa yoko
uThixo wethu, wathandazeka sithi.

<span id="an_8:24">8:24</span> Ndahlula kubathetheli bababingeleli
ishumi elinababini, ngaphandle koSherebhiya, noHashabhiya, neshumi
labazalwana babo;

<span id="an_8:25">8:25</span> ndayilinganisela kubo isilivere,
negolide, nempahla, umrhumo wendlu kaThixo wethu, abawurhumayo ukumkani,
namaphakathi akhe, nabathetheli bakhe, namaSirayeli onke awayekho apho.

<span id="an_8:26">8:26</span> Ndalinganisela esandleni sabo amakhulu
amathandathu anamanci mahlanu eetalente zesilivere, nekhulu leetalente
zeempahla zesilivere, nekhulu leetalente zegolide;

<span id="an_8:27">8:27</span> namashumi amabini eendebe zegolide
ziliwaka leedarike, neempahla ezimbini zobhedu olubengezela kakuhle,
zinqweneleka njengegolide.

<span id="an_8:28">8:28</span> Ndathi kubo, Ningcwele kuYehova, iimpahla
zingcwele; negolide nesilivere zingumnikelo wabaqhutywa yintliziyo,
kuYehova, uThixo wooyihlo;

<span id="an_8:29">8:29</span> phaphani, nizigcine, nide nizilinganisele
phambi kwabathetheli bababingeleli, nabaLevi, nabathetheli bezindlu
zooyise zamaSirayeli, eYerusalem, emagumbini endlu kaYehova.

<span id="an_8:30">8:30</span> Ababingeleli nabaLevi bawamkela ke
umlinganiso wesilivere negolide nempahla, ukuba bawuzise eYerusalem
endlwini yoThixo wethu.

<span id="an_8:31">8:31</span> Saza sanduluka emlanjeni ongumAhava
ngomhla weshumi elinesibini wenyanga yokuqala, ukuba size eYerusalem;
isandla soThixo wethu saba phezu kwethu, sasihlangula esandleni
seentshaba nesabasilaleleyo endleleni.

<span id="an_8:32">8:32</span> Safika eYerusalem, sahlala khona iintsuku
zantathu.

<span id="an_8:33">8:33</span> Ngomhla wesine yalinganiselwa isilivere
negolide neempahla, endlwini yoThixo wethu, esandleni sikaMeremoti
unyana kaUriya, umbingeleli, enoElazare unyana kaPinehasi, benoYozabhadi
unyana kaYeshuwa, noNowadiya unyana kaBhinwi, abaLevi;

<span id="an_8:34">8:34</span> zanikelwa ngokwenani ngokulinganiselwa
kwezinto zonke; konke ukulinganiselwa kwabhalwa ngelo xesha.

<span id="an_8:35">8:35</span> Oonyana basemfudusweni, abo babephume
ekuthinjweni, basondeza amadini anyukayo kuThixo kaSirayeli: iinkunzi
ezintsha zeenkomo zalishumi elinambini, ngenxa yamaSirayeli onke,
neenkunzi zeegusha zamashumi asithoba anantandathu, neemvana zamashumi
asixhenxe anesixhenxe, neenkunzi zeebhokhwe zedini lesono zalishumi
elinambini; zonke zaba lidini elinyukayo kuYehova.

<span id="an_8:36">8:36</span> Bayinikela imithetho yokumkani
kwiirhuluneli zokumkani, nakumabamba nganeno koMlambo; abo babanceda
abantu, nendlu kaThixo.

<span id="an_9:1">9:1</span> Kwakugqitywa oko, beza kum abathetheli,
bathi, Abantu bakwaSirayeli, nababingeleli, nabaLevi, abazahluli
kwizizwe zamazwe la; benza ngokwamasikizi azo, awamaKanan, namaHeti,
namaPerizi, namaYebhusi, nama-Amon, namaMowabhi, namaYiputa, nama-Amori;

<span id="an_9:2">9:2</span> ngokuba bazizekele bona noonyana babo
abafazi ezintombini zawo; yazixuba imbewu engcwele nezizwe zala mazwe;
ke kuqale isandla sabathetheli nabaphathi kobu bumenemene.

<span id="an_9:3">9:3</span> Ndathi ndakuliva elo lizwi, ndazikrazula
iingubo zam, kwaneyokwaleka, ndazidlathula iinwele zam neendevu,
ndahlala phantsi ndimangalisiwe.

<span id="an_9:4">9:4</span> Ndahlanganisela kum bonke abagubha ngamazwi
kaThixo kaSirayeli ngenxa yobumenemene babafuduswayo; ndahlala phantsi
ndimangalisiwe, kwada kwalilixa lomnikelo wokudla wangokuhlwa.

<span id="an_9:5">9:5</span> Ngelo xa lomnikelo wokudla wangokuhlwa,
ndesuka ekuzithobeni kwam, ndazikrazula iingubo zam neyokwaleka, ndaguqa
ngamadolo, ndazolulela izandla zam kuYehova uThixo wam, ndathi:

<span id="an_9:6">9:6</span> Thixo wam, ndidanile, ndineentloni,
andinakubuphakamisela kuwe ubuso bam, Thixo wam; ngokuba ubugwenxa bethu
bukhulile, baba phezu kweentloko zethu; netyala lethu likhulu, lesa
emazulwini.

<span id="an_9:7">9:7</span> Kususela kwimihla yoobawo, sisetyaleni
elikhulu unanamhla; ngenxa yobugwenxa bethu sanikelwa thina, nookumkani
bethu, nababingeleli bethu, esandleni sookumkani bala mazwe,
nasekreleni, nasekuthinjweni, nasekuphangweni, nasekudaneni kobuso,
njengoko kunjalo namhla.

<span id="an_9:8">9:8</span> Kaloku, ngokwephanyazo elincinane, kukho
ubabalo oluvela kuYehova uThixo wethu, lokusishiyela abasindileyo,
asinike isikhonkwane endaweni yengcwele yakhe, ukuze uThixo wethu
awakhanyisele amehlo ethu, asisindise kancinane ebukhobokeni bethu.

<span id="an_9:9">9:9</span> Ngokuba singamakhoboka; ke ebukhobokeni
bethu akasishiyanga uThixo wethu; usithobele inceba phambi kookumkani
basePersi, basisindisa, ukuba siphakamise indlu yoThixo wethu, silungise
inxuwa layo, basibiyele uthango kwaYuda, naseYerusalem.

<span id="an_9:10">9:10</span> Singathini na kaloku, Thixo wethu, emveni
koko? Kuba siyishiyile imithetho yakho, owayiwisayo ngabakhonzi bakho
abaprofeti, usithi,

<span id="an_9:11">9:11</span> Ilizwe eniya kulima, lilizwe lokungcola,
ngokungcola kwezizwe zala mazwe, ngamasikizi azo azalisileyo,
athabathela kwelinye icala esa kwelinye, ngobunqambi bazo.

<span id="an_9:12">9:12</span> Ke ngoko iintombi zenu musani
ukuzendisela koonyana bazo, neentombi zazo musani ukuzizekela oonyana
benu; ningalunyamekeli uxolo lwazo nokuzilungeleyo, naphakade; ukuze
nomelele, nidle ukulunga kwelizwe elo, nilinike oonyana benu libe lilifa
kude kuse ephakadeni.

<span id="an_9:13">9:13</span> Emveni kwako konke okusihleleyo, ngenxa
yezenzo zethu ezibi, nangenxa yetyala lethu elikhulu (kanti wena, Thixo
wethu, usiyekile, akwasohlwaya ngangobugwenxa bethu, wasinika
abasindileyo abangakanana),

<span id="an_9:14">9:14</span> singabuya siyaphule na imithetho yakho,
sendiselane nezizwe ezinaloo masikizi? Akuyi kusifuthela ngomsindo side
siphele na, ukuze kungabikho baseleyo, basindileyo?

<span id="an_9:15">9:15</span> Yehova, Thixo kaSirayeli, ulilungisa,
ngokuba sisele singabasindileyo, njengoko kunjalo namhla. Uyabona,
siphambi kwakho, sinalo ityala lethu, ngokuba akukho unokuma phambi
kwakho ngenxa yoko.

<span id="an_10:1">10:1</span> Ke kaloku akuthandaza uEzra, akukuxela
oku, elila, eziwisile phambi kwendlu kaThixo, kwabuthelana kuye
kumaSirayeli ibandla elikhulu kakhulu lamadoda, nabafazi, nabantwana,
ngokuba abantu batsho ngesililo esikhulu.

<span id="an_10:2">10:2</span> Waphendula uShekaniya unyana kaYehiyeli,
owoonyana bakaElam, wathi kuEzra, Senze ubumenemene kuThixo wethu,
sazeka abafazi bezinye iintlanga ezizweni zelizwe eli; kodwa kusekho
ithemba kumaSirayeli ngale nto.

<span id="an_10:3">10:3</span> Masimenzele ngoku umnqophiso uThixo
wethu, wokuba sibakhuphe bonke abafazi nabazelwe ngabo, ngokwecebo
lenkosi yam, nelabagubhela imithetho yoThixo wethu; makwenziwe
ngokomyalelo.

<span id="an_10:4">10:4</span> Vuka, ngokuba le nto iphezu kwakho;
singakuwe; yomelela, wenze.

<span id="an_10:5">10:5</span> Wavuka uEzra, wabafungisa abathetheli
bababingeleli, nababaLevi, nabamaSirayeli onke, ukuba benze ngokwelo
lizwi; bafunga ke.

<span id="an_10:6">10:6</span> Wesuka uEzra phambi kwendlu kaThixo,
wangena egumbini likaYohanan unyana kaEliyashibhi. Wathi akufika khona,
akadla sonka, akasela manzi; ngokuba ubesenza isijwili ngenxa
yobumenemene babo bafuduswayo.

<span id="an_10:7">10:7</span> Bahambisa ilizwi kwaYuda naseYerusalem,
kubo bonke oonyana basemfudusweni, lokuba babuthelane eYerusalem;

<span id="an_10:8">10:8</span> bathi, bonke abathe abeza ngesithuba
semihla emithathu, ngokwecebo labathetheli nelamadoda amakhulu,
mayisingelwe phantsi yonke impahla yabo; ke bona ngokwabo bahlulwe
nebandla lasemfudusweni.

<span id="an_10:9">10:9</span> Abuthelana eYerusalem onke amadoda
akwaYuda nawakwaBhenjamin ngeentsuku ezintathu; yaye iyinyanga
yesithoba, ngowamashumi amabini umhla wenyanga leyo. Bahlala bonke
abantu ebaleni lendlu kaThixo, bengcangcazela yile nto, naziimvula.

<span id="an_10:10">10:10</span> Wesuka wema uEzra umbingeleli, wathi
kubo, Nenze ubumenemene; nizeke abafazi bezinye iintlanga, ukuze nongeze
etyaleni lamaSirayeli.

<span id="an_10:11">10:11</span> Mzukiseni ngokunje uYehova, uThixo
wooyihlo, nenze okukholekileyo kuye, nizahlule kwizizwe zeli lizwe,
nakubafazi bezinye iintlanga.

<span id="an_10:12">10:12</span> Laphendula lonke ibandla, lathi ngezwi
elikhulu, Kufanelekile kuthi, ukuba senze ngokwelizwi lakho;

<span id="an_10:13">10:13</span> kodwa abantu baninzi, yaye ilixesha
lemvula, akunakumiwa phandle; lo msebenzi asingowemini enye,
asingoweemini ezimbini; ngokuba sone kakhulu ngale nto.

<span id="an_10:14">10:14</span> Abathetheli bethu ke mabalimele ibandla
lonke; bathi bonke emizini yethu, abazeke abafazi bezinye iintlanga,
beze ngamaxesha amisiweyo, benamadoda amakhulu emizi ngemizi, nabagwebi
bayo, kude kubuyiswe kuthi ukuvutha komsindo woThixo wethu, isekho nje
le nto.

<span id="an_10:15">10:15</span> NgooYonatan unyana ka-Asayeli, noYazeya
unyana kaTikwa, bodwa abamelana nelo lizwi; ooMeshulam, noShabhetayi
umLevi, bebancedisa.

<span id="an_10:16">10:16</span> Benjenjalo ke oonyana basemfudusweni.
Kwahlulwa ooEzra umbingeleli, namadoda aziintloko zezindlu zooyise
ngokwezindlu zooyise, bonke bephela, bebalulwe ngamagama, bahlala
phantsi ngomhla wokuqala wenyanga yeshumi ukuyibuza le nto.

<span id="an_10:17">10:17</span> Bayigqiba into yonke yamadoda abezeke
abafazi bezinye iintlanga, elusukwini lokuqala lwenyanga yokuqala.

<span id="an_10:18">10:18</span> Koonyana bababingeleli kwafunyanwa
ababezeke abafazi bezinye iintlanga: koonyana bakaYeshuwa unyana
kaYotsadaki, nabazalwana bakhe: ingooMahaseya, noEliyezere, noYaribhi,
noGedaliya.

<span id="an_10:19">10:19</span> Banika isandla sabo, ukuba bobakhupha
abafazi babo; bathi, benetyala nje, bakhuphe inkunzi yasempahleni
emfutshane ngenxa yetyala labo;

<span id="an_10:20">10:20</span> koonyana bakaImere yanguHanani
noZebhadiya;

<span id="an_10:21">10:21</span> koonyana bakaHarim yanguMahaseya,
noEliya, noShemaya, noYehiyeli, noUziya;

<span id="an_10:22">10:22</span> koonyana bakaPashure yanguEliyohenayi,

<span id="an_10:23">10:23</span> nakubaLevi; yanguYozabhadi, noShimehi,

<span id="an_10:24">10:24</span> kwiimvumi yanguEliyashibhi;

<span id="an_10:25">10:25</span> kumaSirayeli: koonyana bakaParoshe
yanguRamiya, noIziya, noMalekiya, noMiyamin, noElazare, noMalekiya,
noBhenaya;

<span id="an_10:26">10:26</span> koonyana bakaElam yanguMataniya,
noZekariya, noYehiyeli, noAbhedi, noYeremoti, noEliya;

<span id="an_10:27">10:27</span> koonyana bakaZatu yanguEliyohenayi,
noEliyashibhi, noMataniya, noYeremoti, noZabhadi, noAziza;

<span id="an_10:28">10:28</span> nakoonyana bakaBhebhayi yanguYohanan,
noHananiya, noZabhadi, noAtelayi;

<span id="an_10:29">10:29</span> koonyana bakaBhani yanguMeshulam,
noMaluki, noAdaya, noYashubhi, noShehali, noYeremoti;

<span id="an_10:30">10:30</span> koonyana bakaPahati wakwaMowabhi
yanguAdena, noKelali, noBhenaya, noMahaseya, noMataniya, noBhetsaleli,
noBhinwi, noManase;

<span id="an_10:31">10:31</span> koonyana bakaHarim yanguEliyezere,

<span id="an_10:32">10:32</span> noBhenjamin, noMaluki, noShemariya;

<span id="an_10:33">10:33</span> koonyana bakaHashum yanguMatenayi,

<span id="an_10:35">10:35</span> noBhenaya, noBhedeya, noKelwi,

<span id="an_10:39">10:39</span> noShelemiya, noNatan, noAdaya,

<span id="an_10:40">10:40</span> noMakenadebhayi, noShashayi, noSharayi,

<span id="an_10:41">10:41</span> noAzareli, noShelemiya, noShemariya,

<span id="an_10:42">10:42</span> noShalum, noAmariya, noYosefu;

<span id="an_10:43">10:43</span> koonyana bakaNabho yanguYehiyeli,

iintlanga. Bekukho phakathi kwabo abafazi ababezele oonyana.
