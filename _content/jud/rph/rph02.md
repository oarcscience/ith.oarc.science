# ERRATA

Page [66](rph10.htm#page_66), note 8. Instead ‏הקב״ת‎ read ‏הקב״ה‎.

Page [67](rph10.htm#page_67), note 10. Insert the word ‏יקר‎ before
"should have been translated."

Page [113](rph12.htm#page_113), note 66. Read ‏מהימנתא‎ instead of
‏מהימנת‎

Page [142](rph14.htm#page_142), line 11 from top. Insert the word "the"
between "than" and "events."

Page [161](rph15.htm#page_161), note 46, line 6. Read ‏וחכמתא‎ instead
of ‏וחמתא‎

Page [163](rph15.htm#page_163), line 5 from bottom. Insert after "type"
the word ‏דפום‎

Page [171](rph15.htm#page_171), note a Instead of "here shown," read
"shown at the beginning of the book."

Page [175](rph15.htm#page_175), note 78. Insert "I" after "ch."

Page [183](rph16.htm#page_183), note 19. Read ‏אדם‎ instead of ‏אדש‎

Page [184](rph16.htm#page_184), note a. Read ‏חיילין‎ instead of
‏היילין‎

Page [190](rph17.htm#page_190), note 4. Read ‏ותולעה‎ instead of
‏ותולעת‎.

Page [202](rph17.htm#page_202), note 28. Insert the word ‏ירא‎ after the
words "comes from the root."

Page [234](rph19.htm#page_234), line 20 from top. Insert (νοῦς) after
"Intelligence."

Page [265](rph20.htm#page_265), note 93. Insert ἀόρατος καὶ
ἀκατασκεύαστος.

Page [291](rph22.htm#page_291), note 23. Insert ‏נשא‎ after "sect."
