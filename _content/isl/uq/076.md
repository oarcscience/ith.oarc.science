# 76. ‏الإنسان‎

## 'l'ʔns'n

|                                                                                                                                                                      |                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| <span id="an_076_001">1</span> ‏هَلْ أَتَىٰ عَلَى ٱلْإِنسَنِ حِينٌۭ مِّنَ ٱلدَّهْرِ لَمْ يَكُن شَيْـًۭٔا مَّذْكُورًا ‎                                               | hal 'ʔatay' ʕalay 'l'ʔinsani ḥiynunm m:ina 'ld:ahri lam yakun ʃayʔanm' m:aðkuwran'                                                   |
| <span id="an_076_002">2</span> ‏إِنَّا خَلَقْنَا ٱلْإِنسَنَ مِن نُّطْفَةٍ أَمْشَاجٍۢ نَّبْتَلِيهِ فَجَعَلْنَهُ سَمِيعًۢا بَصِيرًا ‎                                  | 'ʔin:a' xalaqna' 'l'ʔinsana min n:uṭfatin 'ʔamʃa'jinm n:abtaliyhi fajaʕalnahu samiyʕanm' baṣiyran'                                   |
| <span id="an_076_003">3</span> ‏إِنَّا هَدَيْنَهُ ٱلسَّبِيلَ إِمَّا شَاكِرًۭا وَإِمَّا كَفُورًا ‎                                                                    | 'ʔin:a' hadaynahu 'ls:abiyla 'ʔim:a' ʃa'kiranm' wa'ʔim:a' kafuwran'                                                                  |
| <span id="an_076_004">4</span> ‏إِنَّآ أَعْتَدْنَا لِلْكَفِرِينَ سَلَسِلَا۟ وَأَغْلَلًۭا وَسَعِيرًا ‎                                                               | 'ʔin:a'ʔ 'ʔaʕtadna' lilkafiriyna salasila' wa'ʔaɣlalanm' wasaʕiyran'                                                                 |
| <span id="an_076_005">5</span> ‏إِنَّ ٱلْأَبْرَارَ يَشْرَبُونَ مِن كَأْسٍۢ كَانَ مِزَاجُهَا كَافُورًا ‎                                                              | 'ʔin:a 'l'ʔabra'ra yaʃrabuwna min ka'ʔsinm ka'na miza'juha' ka'fuwran'                                                               |
| <span id="an_076_006">6</span> ‏عَيْنًۭا يَشْرَبُ بِهَا عِبَادُ ٱللَّهِ يُفَجِّرُونَهَا تَفْجِيرًۭا ‎                                                                | ʕaynanm' yaʃrabu biha' ʕiba'du 'll:ahi yufaj:iruwnaha' tafjiyranm'                                                                   |
| <span id="an_076_007">7</span> ‏يُوفُونَ بِٱلنَّذْرِ وَيَخَافُونَ يَوْمًۭا كَانَ شَرُّهُۥ مُسْتَطِيرًۭا ‎                                                            | yuwfuwna bi'ln:aðri wayaxa'fuwna yawmanm' ka'na ʃar:uhuw mustaṭiyranm'                                                               |
| <span id="an_076_008">8</span> ‏وَيُطْعِمُونَ ٱلطَّعَامَ عَلَىٰ حُبِّهِۦ مِسْكِينًۭا وَيَتِيمًۭا وَأَسِيرًا ‎                                                        | wayuṭʕimuwna 'lṭ:aʕa'ma ʕalay' ḥub:ihiy miskiynanm' wayatiymanm' wa'ʔasiyran'                                                        |
| <span id="an_076_009">9</span> ‏إِنَّمَا نُطْعِمُكُمْ لِوَجْهِ ٱللَّهِ لَا نُرِيدُ مِنكُمْ جَزَآءًۭ وَلَا شُكُورًا ‎                                                | 'ʔin:ama' nuṭʕimukum liwajhi 'll:ahi la' nuriydu minkum jaza'ʔʔanm wala' ʃukuwran'                                                   |
| <span id="an_076_010">10</span> ‏إِنَّا نَخَافُ مِن رَّبِّنَا يَوْمًا عَبُوسًۭا قَمْطَرِيرًۭا ‎                                                                      | 'ʔin:a' naxa'fu min r:ab:ina' yawman' ʕabuwsanm' qamṭariyranm'                                                                       |
| <span id="an_076_011">11</span> ‏فَوَقَىٰهُمُ ٱللَّهُ شَرَّ ذَلِكَ ٱلْيَوْمِ وَلَقَّىٰهُمْ نَضْرَةًۭ وَسُرُورًۭا ‎                                                   | fawaqay'humu 'll:ahu ʃar:a ðalika 'lyawmi walaq:ay'hum naḍratanm wasuruwranm'                                                        |
| <span id="an_076_012">12</span> ‏وَجَزَىٰهُم بِمَا صَبَرُوا۟ جَنَّةًۭ وَحَرِيرًۭا ‎                                                                                  | wajazay'hum bima' ṣabaruw' jan:atanm waḥariyranm'                                                                                    |
| <span id="an_076_013">13</span> ‏مُّتَّكِـِٔينَ فِيهَا عَلَى ٱلْأَرَآئِكِ ۖ لَا يَرَوْنَ فِيهَا شَمْسًۭا وَلَا زَمْهَرِيرًۭا ‎                                      | m:ut:akiʔiyna fiyha' ʕalay 'l'ʔara'ʔyʔiki sl' la' yarawna fiyha' ʃamsanm' wala' zamhariyranm'                                        |
| <span id="an_076_014">14</span> ‏وَدَانِيَةً عَلَيْهِمْ ظِلَلُهَا وَذُلِّلَتْ قُطُوفُهَا تَذْلِيلًۭا ‎                                                               | wada'niyatan ʕalayhim ẓilaluha' waðul:ilat quṭuwfuha' taðliylanm'                                                                    |
| <span id="an_076_015">15</span> ‏وَيُطَافُ عَلَيْهِم بِـَٔانِيَةٍۢ مِّن فِضَّةٍۢ وَأَكْوَابٍۢ كَانَتْ قَوَارِيرَا۠ ‎                                                 | wayuṭa'fu ʕalayhim biʔa'niyatinm m:in fiḍ:atinm wa'ʔakwa'binm ka'nat qawa'riyra'                                                     |
| <span id="an_076_016">16</span> ‏قَوَارِيرَا۟ مِن فِضَّةٍۢ قَدَّرُوهَا تَقْدِيرًۭا ‎                                                                                 | qawa'riyra' min fiḍ:atinm qad:aruwha' taqdiyranm'                                                                                    |
| <span id="an_076_017">17</span> ‏وَيُسْقَوْنَ فِيهَا كَأْسًۭا كَانَ مِزَاجُهَا زَنجَبِيلًا ‎                                                                         | wayusqawna fiyha' ka'ʔsanm' ka'na miza'juha' zanjabiylan'                                                                            |
| <span id="an_076_018">18</span> ‏عَيْنًۭا فِيهَا تُسَمَّىٰ سَلْسَبِيلًۭا ‎                                                                                           | ʕaynanm' fiyha' tusam:ay' salsabiylanm'                                                                                              |
| <span id="an_076_019">19</span> ‏ وَيَطُوفُ عَلَيْهِمْ وِلْدَنٌۭ مُّخَلَّدُونَ إِذَا رَأَيْتَهُمْ حَسِبْتَهُمْ لُؤْلُؤًۭا مَّنثُورًۭا ‎                              | wayaṭuwfu ʕalayhim wildanunm m:uxal:aduwna 'ʔiða' ra'ʔaytahum ḥasibtahum luwʔluwʔanm' m:anθuwranm'                                   |
| <span id="an_076_020">20</span> ‏وَإِذَا رَأَيْتَ ثَمَّ رَأَيْتَ نَعِيمًۭا وَمُلْكًۭا كَبِيرًا ‎                                                                     | wa'ʔiða' ra'ʔayta θam:a ra'ʔayta naʕiymanm' wamulkanm' kabiyran'                                                                     |
| <span id="an_076_021">21</span> ‏عَلِيَهُمْ ثِيَابُ سُندُسٍ خُضْرٌۭ وَإِسْتَبْرَقٌۭ ۖ وَحُلُّوٓا۟ أَسَاوِرَ مِن فِضَّةٍۢ وَسَقَىٰهُمْ رَبُّهُمْ شَرَابًۭا طَهُورًا ‎ | ʕaliyahum θiya'bu sundusin xuḍrunm wa'ʔistabraqunm sl' waḥul:uwʔ' 'ʔasa'wira min fiḍ:atinm wasaqay'hum rab:uhum ʃara'banm' ṭahuwran' |
| <span id="an_076_022">22</span> ‏إِنَّ هَذَا كَانَ لَكُمْ جَزَآءًۭ وَكَانَ سَعْيُكُم مَّشْكُورًا ‎                                                                  | 'ʔin:a haða' ka'na lakum jaza'ʔʔanm waka'na saʕyukum m:aʃkuwran'                                                                     |
| <span id="an_076_023">23</span> ‏إِنَّا نَحْنُ نَزَّلْنَا عَلَيْكَ ٱلْقُرْءَانَ تَنزِيلًۭا ‎                                                                         | 'ʔin:a' naḥnu naz:alna' ʕalayka 'lqurʔa'na tanziylanm'                                                                               |
| <span id="an_076_024">24</span> ‏فَٱصْبِرْ لِحُكْمِ رَبِّكَ وَلَا تُطِعْ مِنْهُمْ ءَاثِمًا أَوْ كَفُورًۭا ‎                                                          | fa'ṣbir liḥukmi rab:ika wala' tuṭiʕ minhum ʔa'θiman' 'ʔaw kafuwranm'                                                                 |
| <span id="an_076_025">25</span> ‏وَٱذْكُرِ ٱسْمَ رَبِّكَ بُكْرَةًۭ وَأَصِيلًۭا ‎                                                                                     | wa'ðkuri 'sma rab:ika bukratanm wa'ʔaṣiylanm'                                                                                        |
| <span id="an_076_026">26</span> ‏وَمِنَ ٱلَّيْلِ فَٱسْجُدْ لَهُۥ وَسَبِّحْهُ لَيْلًۭا طَوِيلًا ‎                                                                     | wamina 'l:ayli fa'sjud lahuw wasab:iḥhu laylanm' ṭawiylan'                                                                           |
| <span id="an_076_027">27</span> ‏إِنَّ هَٓؤُلَآءِ يُحِبُّونَ ٱلْعَاجِلَةَ وَيَذَرُونَ وَرَآءَهُمْ يَوْمًۭا ثَقِيلًۭا ‎                                              | 'ʔin:a haʔwʔula'ʔʔi yuḥib:uwna 'lʕa'jilata wayaðaruwna wara'ʔʔahum yawmanm' θaqiylanm'                                               |
| <span id="an_076_028">28</span> ‏نَّحْنُ خَلَقْنَهُمْ وَشَدَدْنَآ أَسْرَهُمْ ۖ وَإِذَا شِئْنَا بَدَّلْنَآ أَمْثَلَهُمْ تَبْدِيلًا ‎                                | n:aḥnu xalaqnahum waʃadadna'ʔ 'ʔasrahum sl' wa'ʔiða' ʃiyʔna' bad:alna'ʔ 'ʔamθalahum tabdiylan'                                       |
| <span id="an_076_029">29</span> ‏إِنَّ هَذِهِۦ تَذْكِرَةٌۭ ۖ فَمَن شَآءَ ٱتَّخَذَ إِلَىٰ رَبِّهِۦ سَبِيلًۭا ‎                                                       | 'ʔin:a haðihiy taðkiratunm sl' faman ʃa'ʔʔa 't:axaða 'ʔilay' rab:ihiy sabiylanm'                                                     |
| <span id="an_076_030">30</span> ‏وَمَا تَشَآءُونَ إِلَّآ أَن يَشَآءَ ٱللَّهُ ۚ إِنَّ ٱللَّهَ كَانَ عَلِيمًا حَكِيمًۭا ‎                                            | wama' taʃa'ʔʔuwna 'ʔil:a'ʔ 'ʔan yaʃa'ʔʔa 'll:ahu j 'ʔin:a 'll:aha ka'na ʕaliyman' ḥakiymanm'                                         |
| <span id="an_076_031">31</span> ‏يُدْخِلُ مَن يَشَآءُ فِى رَحْمَتِهِۦ ۚ وَٱلظَّلِمِينَ أَعَدَّ لَهُمْ عَذَابًا أَلِيمًۢا ‎                                          | yudxilu man yaʃa'ʔʔu fiy raḥmatihiy j wa'lẓ:alimiyna 'ʔaʕad:a lahum ʕaða'ban' 'ʔaliymanm'                                            |
