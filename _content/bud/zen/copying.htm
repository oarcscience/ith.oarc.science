How to Copy and Keep This Text as a Document

Due to problems with encoding, you can't directly Copy and Paste the Chinese/Japanese characters in this text. If you want to save the text as a document, please try this:

1. In the File menu, choose Save As... command.

2. In the pull-down menu Format, select "HTML Source."

3. Save.

4. In your word processor (Word, etc.) choose Open in the File menu.

5. Select the file you saved and open it.

6. In the Edit menu, choose Select All command.

7. In the Edit menu, choose Copy command.

8. Open a new document and in the Edit menu, choose Paste command.

9. Save this file.
