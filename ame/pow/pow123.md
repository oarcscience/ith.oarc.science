*Pow-Wows, or Long Lost Friend*, by John George Hoffman, \[1820\], at
Intangible Textual Heritage

------------------------------------------------------------------------

AGAINST EVIL SPIRITS AND ALL MANNER OF WITCHCRAFT.

|         |     |           |
|---------|-----|-----------|
|         | I.  |           |
| N.      | I.  | R.        |
|         | I.  |           |
| SANCTUS |     | SPIRITUS. |
|         | I.  |           |
| N.      | I.  | R.        |
|         | I.  |           |

 

All this be guarded here in time, and there in eternity. Amen.

<span id="page_44">p. 44</span>

You must write all the above on a piece of white paper and carry it
about you. The characters or letters above signify: "God bless me here
in time, and there eternally."
