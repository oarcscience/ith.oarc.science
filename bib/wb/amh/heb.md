### Amharic Bible: Hebrews

<span id="an_1:1">1:1</span> ከጥንት ጀምሮ እግዚአብሔር በብዙ ዓይነትና በብዙ ጎዳና ለአባቶቻችን
በነቢያት ተናግሮ፥

<span id="an_1:2">1:2</span> ሁሉን ወራሽ ባደረገው ደግሞም ዓለማትን በፈጠረበት በልጁ በዚህ ዘመን
መጨረሻ ለእኛ ተናገረን፤

<span id="an_1:3">1:3</span> እርሱም የክብሩ መንጸባረቅና የባሕርዩ ምሳሌ ሆኖ፥ ሁሉን በስልጣኑ
ቃል እየደገፈ፥ ኃጢአታችንን በራሱ ካነጻ በኋላ በሰማያት በግርማው ቀኝ ተቀመጠ፤

<span id="an_1:4">1:4</span> ከመላእክት ይልቅ እጅግ የሚበልጥ ስምን በወረሰ መጠን እንዲሁ
ከእነርሱ አብዝቶ ይበልጣል።

<span id="an_1:5">1:5</span> ከመላእክትስ። አንተ ልጄ ነህ፥ እኔ ዛሬ ወልጄሃለሁ፥ ደግሞም። እኔ
አባት እሆነዋለሁ እርሱም ልጅ ይሆነኛል

<span id="an_1:6">1:6</span> ያለው ከቶ ለማን ነው? ደግሞም በኵርን ወደ ዓለም ሲያገባ።
የእግዚአብሔር መላእክት ሁሉም ለእርሱ ይስገዱ ይላል።

<span id="an_1:7">1:7</span> ስለ መላእክትም። መላእክቱን መናፍስት አገልጋዮቹንም የእሳት ነበልባል
የሚያደርግ

<span id="an_1:8">1:8</span> ይላል፤ ስለ ልጁ ግን። አምላክ ሆይ፥ ዙፋንህ እስከ ዘላለም ድረስ
ይኖራል፤ የመንግሥትህ በትር የቅንነት በትር ነው።

<span id="an_1:9">1:9</span> ጽድቅን ወደድህ ዓመፅንም ጠላህ፤ ስለዚህ እግዚአብሔር አምላክህ
ከጓደኞችህ ይልቅ በደስታ ዘይት ቀባህ

<span id="an_1:10">1:10</span> ይላል። ደግሞ። ጌታ ሆይ፥ አንተ ከጥንት ምድርን መሠረትህ፥
ሰማዮችም የእጆችህ ሥራ ናቸው፤

<span id="an_1:11">1:11</span> እነርሱም ይጠፋሉ አንተ ግን ጸንተህ ትኖራለህ፤ ሁሉም እንደ ልብስ
ያረጃሉ፥

<span id="an_1:12">1:12</span> እንደ መጎናጸፊያም ትጠቀልላቸዋለህ ይለወጡማል፤ አንተ ግን አንተ
ነህ፥ ዓመቶችህም ከቶ አያልቁም

<span id="an_1:13">1:13</span> ይላል። ነገር ግን ከመላእክት። ጠላቶችህን የእግርህ መረገጫ
እስካደርግልህ ድረስ በቀኜ ተቀመጥ

<span id="an_1:14">1:14</span> ከቶ ለማን ብሎአል? ሁሉ መዳንን ይወርሱ ዘንድ ስላላቸው ለማገዝ
የሚላኩ የሚያገለግሉም መናፍስት አይደሉምን?

<span id="an_2:1">2:1</span> ስለዚህ ከሰማነው ነገር ምናልባት እንዳንወሰድ፥ ለእርሱ አብልጠን
ልንጠነቀቅ ያስፈልገናል።

<span id="an_2:2">2:2</span> በመላእክት የተነገረው ቃል ጽኑ ከሆነ፥ መተላለፍና አለመታዘዝም ሁሉ
የጽድቅን ብድራት ከተቀበለ፥ እኛስ እንዲህ ያለውን ታላቅ መዳን ቸል ብንለው፥ እንዴት እናመልጣለን?

<span id="an_2:3">2:3</span> ይህ በጌታ በመጀመሪያ የተነገረ ነበርና፥ የሰሙትም ለእኛ አጸኑት፥

<span id="an_2:4">2:4</span> እግዚአብሔርም እርሱ ራሱ እንደ ፈቀደ በምልክትና በድንቅ ነገር በልዩ
ልዩ ተአምራትም፥ መንፈስ ቅዱስንም በማደል አብሮ መሰከረለት።

<span id="an_2:5">2:5</span> ስለ እርሱ የምንናገርበትን የሚመጣውን ዓለም ለመላእክት ያስገዛው
አይደለምና።

<span id="an_2:6">2:6</span> ነገር ግን አንዱ በአንድ ስፍራ። ታስበው ዘንድ ሰው ምንድር ነው?
ወይስ ትጎበኘው ዘንድ የሰው ልጅ ምንድር ነው?

<span id="an_2:7">2:7</span> ከመላእክት ይልቅ በጥቂት አሳነስኸው፤ የክብርና የምስጋና ዘውድ
ጫንህለት፥ በእጆችህም ሥራ ላይ ሾምኸው፤

<span id="an_2:8">2:8</span> ሁሉን ከእግሮቹ በታች አስገዛህለት ብሎ መሰከረ። ሁሉን ከእርሱ በታች
ባስገዛ ጊዜ ያልተገዛለት ምንም አልተወምና። አሁን ግን ሁሉ እንደ ተገዛለት ገና አናይም፤

<span id="an_2:9">2:9</span> ነገር ግን በእግዚአብሔር ጸጋ ስለ ሰው ሁሉ ሞትን ይቀምስ ዘንድ፥
ከመላእክት ይልቅ በጥቂት አንሶ የነበረውን ኢየሱስን ከሞት መከራ የተነሣ የክብርና የምስጋናን ዘውድ ተጭኖ
እናየዋለን።

<span id="an_2:10">2:10</span> ብዙ ልጆችን ወደ ክብር ሲያመጣ የመዳናቸውን ራስ በመከራ ይፈጽም
ዘንድ፥ ከእርሱ የተነሣ ሁሉ በእርሱም ሁሉ ለሆነ፥ ለእርሱ ተገብቶታልና።

<span id="an_2:11">2:11</span> የሚቀድሰውና የሚቀደሱት ሁሉ ከአንድ ናቸውና፤

<span id="an_2:13">2:13</span> ስለዚህም ምክንያት። ስምህን ለወንድሞቼ እነግራቸዋለሁ በማኅበርም
መካከል በዜማ አመሰግንሃለሁ፤ ደግሞም። እኔ በእርሱ እታመናለሁ፤ ደግሞም። እነሆኝ እኔን እግዚአብሔር የሰጠኝንም
ልጆች ሲል ወንድሞች ብሎ ሊጠራቸው አያፍርም።

<span id="an_2:14">2:14</span> እንግዲህ ልጆቹ በሥጋና በደም ስለሚካፈሉ፥ እርሱ ደግሞ በሞት ላይ
ሥልጣን ያለውን በሞት እንዲሽር፥ ይኸውም ዲያብሎስ ነው፥ በሕይወታቸውም ሁሉ ስለ ሞት ፍርሃት በባርነት ይታሰሩ
የነበሩትን ሁሉ ነጻ እንዲያወጣ፥ በሥጋና በደም እንዲሁ ተካፈለ።

<span id="an_2:16">2:16</span> የአብርሃምን ዘር ይዞአል እንጂ የያዘው የመላእክትን አይደለም።

<span id="an_2:17">2:17</span> ስለዚህ የሕዝብን ኃጢአት ለማስተስረይ፥ ለእግዚአብሔር በሆነው
ነገር ሁሉ የሚምርና የታመነ ሊቀ ካህናት እንዲሆን፥ በነገር ሁሉ ወንድሞቹን ሊመስል ተገባው።

<span id="an_2:18">2:18</span> እርሱ ራሱ ተፈትኖ መከራን ስለ ተቀበለ የሚፈተኑትን ሊረዳቸው
ይችላልና።

<span id="an_3:1">3:1</span> ስለዚህ፥ ከሰማያዊው ጥሪ ተካፋዮች የሆናችሁ ቅዱሳን ወንድሞች ሆይ፥
የሃይማኖታችንን ሐዋርያና ሊቀ ካህናት ኢየሱስ ክርስቶስን ተመልከቱ፤

<span id="an_3:2">3:2</span> ሙሴ ደግሞ በቤቱ ሁሉ የታመነ እንደሆነ፥ እርሱ ለሾመው የታመነ
ነበረ።

<span id="an_3:3">3:3</span> ቤትን የሚያዘጋጀው ከቤቱ ይልቅ የሚበልጥ ክብር እንዳለው መጠን፥
እንዲሁ እርሱ ከሙሴ ይልቅ የሚበልጥ ክብር የተገባው ሆኖ ተቆጥሮአልና።

<span id="an_3:4">3:4</span> እያንዳንዱ ቤት በአንድ ሰው ተዘጋጅቶአልና፥ ሁሉን ያዘጋጀ ግን
እግዚአብሔር ነው።

<span id="an_3:5">3:5</span> ሙሴስ በኋላ ስለሚነገረው ነገር ምስክር ሊሆን በቤቱ ሁሉ እንደ ሎሌ
የታመነ ነበረ፥ ክርስቶስ ግን እንደ ልጅ በቤቱ ላይ የታመነ ነው፤

<span id="an_3:6">3:6</span> እኛም የምንደፍርበትን የምንመካበትንም ተስፋ እስከ መጨረሻው አጽንተን
ብንጠብቅ ቤቱ ነን።

<span id="an_3:7">3:7</span> ስለዚህ መንፈስ ቅዱስ እንደሚል።

<span id="an_3:8">3:8</span> ዛሬ ድምፁን ብትሰሙት፥ አባቶቻችሁ እኔን የፈተኑበት የመረመሩበትም
አርባ ዓመትም ሥራዬን ያዩበት በምድረ በዳ በፈተና ቀን በማስመረር እንደ ሆነ፥ ልባችሁን እልከኛ አታድርጉ።

<span id="an_3:10">3:10</span> ስለዚህ ያን ትውልድ ተቆጥቼ። ዘወትር በልባቸው ይስታሉ መንገዴን
ግን አላወቁም አልሁ፤

<span id="an_3:11">3:11</span> እንዲሁ። ወደ ዕረፍቴ አይገቡም ብዬ በቍጣዬ ማልሁ።

<span id="an_3:12">3:12</span> ወንድሞች ሆይ፥ ምናልባት ሕያው እግዚአብሔርን የሚያስክዳችሁ ክፉና
የማያምን ልብ ከእናንተ በአንዳችሁ እንዳይኖር ተጠንቀቁ፤

<span id="an_3:13">3:13</span> ነገር ግን ከእናንተ ማንም በኃጢአት መታለል እልከኛ እንዳይሆን፥
ዛሬ ተብሎ ሲጠራ ሳለ፥ በእያንዳንዱ ቀን እርስ በርሳችሁ ተመካከሩ፤

<span id="an_3:14">3:14</span> የመጀመሪያ እምነታችንን እስከ መጨረሻው አጽንተን ብንጠብቅ፥
የክርስቶስ ተካፋዮች ሆነናልና፤

<span id="an_3:15">3:15</span> እየተባለ። ዛሬ ድምጹን ብትሰሙት፥ በማስመረር እንደሆነ ልባችሁን
እልከኛ አታድርጉ።

<span id="an_3:16">3:16</span> ሰምተው ያስመረሩት እነማን ነበሩ? በሙሴ ተመርተው ከግብጽ የወጡ
ሁሉ አይደሉምን?

<span id="an_3:17">3:17</span> አርባ አመትም የተቆጣባቸው እነማን ነበሩ? ሬሳቸው በምድረ በዳ
የወደቀ፥ ኃጢአትን ያደረጉት እነርሱ አይደሉምን?

<span id="an_3:18">3:18</span> ካልታዘዙትም በቀር ወደ እረፍቱ እንዳይገቡ የማለባቸው እነማን
ነበሩ?

<span id="an_3:19">3:19</span> ባለማመናቸውም ጠንቅ ሊገቡ እንዳልተቻላቸው እናያለን።

<span id="an_4:1">4:1</span> እንግዲህ ወደ ዕረፍቱ ለመግባት ተስፋ ገና ቀርቶልን ከሆነ፥ ምናልባት
ከእናንተ ማንም የማይበቃ መስሎ እንዳይታይ እንፍራ።

<span id="an_4:2">4:2</span> ለእነዚያ ደግሞ እንደ ተነገረ ለእኛ የምስራች ተሰብኮልናልና፤ ዳሩ
ግን የሰሙት ቃል ከሰሚዎቹ ጋር በእምነት ስላልተዋሐደ አልጠቀማቸውም።

<span id="an_4:3">4:3</span> ሥራው ዓለም ከተፈጠረ ጀምሮ ምንም እንኳ ቢፈጸም። እንዲህ። ወደ
ዕረፍቴ አይገቡም ብዬ በቁጣዬ ማልሁ እንዳለ፥ እኛስ ያመንን ወደ ዕረፍቱ እንገባለን።

<span id="an_4:4">4:4</span> ስለ ሰባተኛው ቀን በአንድ ስፍራ። እግዚአብሔርም በሰባተኛው ቀን
ከሥራው ሁሉ ዐረፈ ብሎአልና፤

<span id="an_4:5">4:5</span> በዚህ ስፍራም ደግሞ። ወደ ዕረፍቴ አይገቡም።

<span id="an_4:6">4:6</span> እንግዲህ አንዳንዶች በዚያ እንዲገቡ ስለ ቀሩ፥ ቀድሞም የምስራች
የተሰበከላቸው ባለመታዘዝ ጠንቅ ስላልገቡ።

<span id="an_4:7">4:7</span> ዛሬ ድምፁን ብትሰሙት ልባችሁን እልከኛ አታድርጉ በፊት እንደ ተባለ፥
ይህን ከሚያህል ዘመን በኋላ በዳዊት ሲናገር። ዛሬ ብሎ አንድ ቀን እንደ ገና ይቀጥራል።

<span id="an_4:8">4:8</span> ኢያሱ አሳርፎአቸው ኖሮ ቢሆንስ፥ ከዚያ በኋላ ስለ ሌላ ቀን
ባልተናገረ ነበር።

<span id="an_4:9">4:9</span> እንግዲያስ የሰንበት ዕረፍት ለእግዚአብሔር ሕዝብ ቀርቶላቸዋል።

<span id="an_4:10">4:10</span> ወደ ዕረፍቱ የገባ፥ እግዚአብሔር ከሥራው እንዳረፈ፥ እርሱ ደግሞ
ከሥራው አርፎአልና።

<span id="an_4:11">4:11</span> እንግዲህ እንደዚያ እንደ አለመታዘዝ ምሳሌ ማንም እንዳይወድቅ
ወደዚያ ዕረፍት ለመግባት እንትጋ።

<span id="an_4:12">4:12</span> የእግዚአብሔር ቃል ሕያው ነውና፥ የሚሠራም፥ ሁለትም አፍ ካለው
ሰይፍ ሁሉ ይልቅ የተሳለ ነው፥ ነፍስንና መንፈስንም ጅማትንና ቅልጥምንም እስኪለይ ድረስ ይወጋል፥ የልብንም ስሜትና
አሳብ ይመረምራል፤

<span id="an_4:13">4:13</span> እኛን በሚቆጣጠር በእርሱ ዓይኖቹ ፊት ሁሉ ነገር የተራቆተና
የተገለጠ ነው እንጂ፥ በእርሱ ፊት የተሰወረ ፍጥረት የለም።

<span id="an_4:14">4:14</span> እንግዲህ በሰማያት ያለፈ ትልቅ ሊቀ ካህናት የእግዚአብሔር ልጅ
ኢየሱስ ስላለን፥ ጸንተን ሃይማኖታችንን እንጠብቅ።

<span id="an_4:15">4:15</span> ከኃጢአት በቀር በነገር ሁሉ እንደ እኛ የተፈተነ ነው እንጂ፥
በድካማችን ሊራራልን የማይችል ሊቀ ካህናት የለንም።

<span id="an_4:16">4:16</span> እንግዲህ ምሕረትን እንድንቀበል በሚያስፈልገንም ጊዜ የሚረዳንን
ጸጋ እንድናገኝ ወደ ጸጋው ዙፋን በእምነት እንቅረብ።

<span id="an_5:1">5:1</span> ሊቀ ካህናት ሁሉ ስለ ኃጢአት መባንና መስዋዕትን ሊያቀርብ ከሰው
ተመርጦ ለእግዚአብሔር በሆነው ነገር ሁሉ ስለ ሰው ይሾማልና፤

<span id="an_5:2">5:2</span> እርሱ ራሱም ደግሞ ድካምን ስለሚለብስ፥ ላልተማሩትና ለሚስቱት
ሊራራላቸው ይችላል፤

<span id="an_5:3">5:3</span> በዚህም ምክንያት ስለ ሕዝብ እንደሚያቀርብ እንዲሁ ስለ ራሱ ደግሞ
መስዋዕትን ስለ ኃጢአት ሊያቀርብ ይገባዋል።

<span id="an_5:4">5:4</span> እንደ አሮንም በእግዚአብሔር ከተጠራ በቀር ማንም ክብሩን ለራሱ
የሚወስድ የለም።

<span id="an_5:5">5:5</span> እንዲሁ ክርስቶስ ደግሞ ሊቀ ካህናት ሊሆን ራሱን አላከበረም ነገር
ግን። አንተ ልጄ ነህ እኔ ዛሬ ወልጄሃለሁ ያለው እርሱ ነው፤

<span id="an_5:6">5:6</span> እንደዚህም በሌላ ስፍራ ደግሞ። አንተ እንደ መልከ ጼዴቅ ሹመት
ለዘላለም ካህን ነህ ይላል።

<span id="an_5:7">5:7</span> እርሱም በስጋው ወራት ከሞት ሊያድነው ወደሚችል ከብርቱ ጩኸትና
ከእንባ ጋር ጸሎትንና ምልጃን አቀረበ፥ እግዚአብሔርንም ስለ መፍራቱ ተሰማለት፤

<span id="an_5:8">5:8</span> ምንም ልጅ ቢሆን፥ ከተቀበለው መከራ መታዘዝን ተማረ፤

<span id="an_5:9">5:9</span> ከተፈጸመም በኋላ በእግዚአብሔር እንደ መልከ ጼዴቅ ሹመት ሊቀ ካህናት
ተብሎ ስለ ተጠራ፥ ለሚታዘዙለት ሁሉ የዘላለም መዳን ምክንያት ሆነላቸው።

<span id="an_5:11">5:11</span> ስለ እርሱም የምንናገረው ብዙ ነገር አለን፥ ጆሮቻችሁም ስለ ፈዘዙ
በቃል ልንተረጕመው ጭንቅ ነው።

<span id="an_5:12">5:12</span> ከጊዜው የተነሳ አስተማሪዎች ልትሆኑ ሲገባችሁ፥ አንድ ሰው ስለ
እግዚአብሔር ቃላት መጀመሪያ ያለውን የሕፃንነትን ትምህርት እንዲያስተምራችሁ እንደ ገና ያስፈልጋችኋልና፤
የሚያስፈልጋችሁም ወተት ነው እንጂ ጠንካራ ምግብ አይደለም።

<span id="an_5:13">5:13</span> ወተት የሚጋት ሁሉ ሕፃን ስለ ሆነ የጽድቅን ቃል አያውቅምና፤

<span id="an_5:14">5:14</span> ጠንካራ ምግብ ግን መልካሙንና ክፉውን ለመለየት በስራቸው የለመደ
ልቡና ላላቸው ለፍጹማን ሰዎች ነው።

<span id="an_6:1">6:1</span> ስለዚህ የክርስቶስን ነገር መጀመሪያ የሚናገረውን ቃል ትተን ወደ
ፍጻሜ እንሂድ፤ መሠረትን ደግመን አንመሥርት፥ እርሱም ከሞተ ሥራ ንስሐና በእግዚአብሔር እምነት፥ ስለ ጥምቀቶችና
እጆችንም ስለ መጫን ስለ ሙታንም ትንሣኤ ስለ ዘላለም ፍርድም ትምህርት ነው።

<span id="an_6:3">6:3</span> እግዚአብሔርም ቢፈቅድ ይህን እናደርጋለን።

<span id="an_6:4">6:4</span> አንድ ጊዜ ብርሃን የበራላቸውን ሰማያዊውንም ስጦታ የቀመሱትን
ከመንፈስ ቅዱስም ተካፋዮች ሆነው የነበሩትን

<span id="an_6:5">6:5</span> መልካሙንም የእግዚአብሔርን ቃልና ሊመጣ ያለውን የዓለም ኃይል
የቀመሱትን

<span id="an_6:6">6:6</span> በኋላም የካዱትን እንደገና ለንስሐ እነርሱን ማደስ የማይቻል ነው፤
ለራሳቸው የእግዚአብሔርን ልጅ ይሰቅሉታልና ያዋርዱትማልና።

<span id="an_6:7">6:7</span> ብዙ ጊዜ በእርስዋ ላይ የሚወርደውን ዝናብ የምትጠጣ መሬት፥
ለሚያርሱአትም ደግሞ የምትጠቅምን አትክልት የምታበቅል፥ ከእግዚአብሔር በረከትን ታገኛለችና፤

<span id="an_6:8">6:8</span> እሾህና ኵርንችትን ግን ብታወጣ፥ የተጣለች ናት ለመረገምም
ትቀርባለች፥ መጨረሻዋም መቃጠል ነው።

<span id="an_6:9">6:9</span> ስለ እናንተ ግን፥ ወዳጆች ሆይ፥ ምንም እንኳ እንዲሁ ብንናገር፥
አብልጦ የሚሻለውና ለመዳን የሚሆነው እንዲሆንላችሁ ተረድተናል።

<span id="an_6:10">6:10</span> እግዚአብሔር፥ ቅዱሳንን ስላገለገላችሁ እስከ አሁንም
ስለምታገለግሉአቸው፥ ያደረጋችሁትን ሥራ ለስሙም ያሳያችሁትን ፍቅር ይረሳ ዘንድ ዓመፀኛ አይደለምና።

<span id="an_6:11">6:11</span> በእምነትና በትዕግሥትም የተስፋውን ቃል የሚወርሱትን እንድትመስሉ
እንጂ ዳተኞች እንዳትሆኑ፥ ተስፋ እስኪሞላ ድረስ እያንዳንዳችሁ ያን ትጋት እስከ መጨረሻ እንድታሳዩ እንመኛለን።

<span id="an_6:13">6:13</span> እግዚአብሔርም ለአብርሃም ተስፋ በሰጠው ጊዜ። በእውነት እየባረክሁ
እባርክሃለሁ እያበዛሁም አበዛሃለሁ ብሎ፥ ከእርሱ በሚበልጥ በማንም ሊምል ስላልቻለ፥ በራሱ ማለ፤

<span id="an_6:15">6:15</span> እንዲሁም እርሱ ከታገሰ በኋላ ተስፋውን አገኘ።

<span id="an_6:16">6:16</span> ሰዎች ከእነርሱ በሚበልጠው ይምላሉና፥ ለማስረዳትም የሆነው መሐላ
የሙግት ሁሉ ፍጻሜ ይሆናል፤

<span id="an_6:17">6:17</span> ስለዚህም እግዚአብሔር፥ የተስፋውን ቃል ለሚወርሱ ፈቃዱ እንደ
ማይለወጥ አብልጦ ሊያሳያቸው ስለ ፈቀደ፥ እግዚአብሔር ሊዋሽ በማይቻል በሁለት በማይለወጥ ነገር፥ በፊታችን ያለውን
ተስፋ ለመያዝ ለሸሸን ለእኛ ብርቱ መጽናናት ይሆንልን ዘንድ፥ በመሓላ በመካከል ገባ፤

<span id="an_6:19">6:19</span> ይህም ተስፋ እንደ ነፍስ መልሕቅ አለን እርሱም እርግጥና ጽኑ
የሆነ ወደ መጋረጃውም ውስጥ የገባ ነው፤

<span id="an_6:20">6:20</span> በዚያም ኢየሱስ እንደ መልከ ጼዴቅ ሹመት ለዘላለም ሊቀ ካህናት
የሆነው፥ ስለ እኛ ቀዳሚ ሆኖ ገባ።

<span id="an_7:1">7:1</span> የሳሌም ንጉሥና የልዑል እግዚአብሔር ካህን የሆነ ይህ መልከ ጼዴቅ
አብርሃም ነገሥታትን ገድሎ ሲመለስ ከእርሱ ጋር ተገናኝቶ ባረከው፤

<span id="an_7:2">7:2</span> ለእርሱም ደግሞ አብርሃም ከሁሉ አስራትን አካፈለው። የስሙም ትርጓሜ
በመጀመሪያ የጽድቅ ንጉሥ ነው፥ ኋላም ደግሞ የሳሌም ንጉሥ ማለት የሰላም ንጉሥ ነው።

<span id="an_7:3">7:3</span> አባትና እናት የትውልድም ቍጥር የሉትም፥ ለዘመኑም ጥንት ለህይወቱም
ፍጻሜ የለውም፥ ዳሩ ግን በእግዚአብሔር ልጅ ተመስሎ ለዘላለም ካህን ሆኖ ይኖራል።

<span id="an_7:4">7:4</span> የአባቶች አለቃ አብርሃም ከዘረፋው የሚሻለውን አስራት የሰጠው ይህ
ሰው እንዴት ትልቅ እንደ ነበረ እስኪ ተመልከቱ።

<span id="an_7:5">7:5</span> ከሌዊ ልጆችም ክህነትን የሚቀበሉት ከህዝቡ ማለት ከወንድሞቻቸው፥
እነርሱ ምንም ከአብርሃም ወገብ ቢወጡ፥ ከእነርሱ አሥራትን በሕግ እንዲያስወጡ ትእዛዝ አላቸው፤

<span id="an_7:6">7:6</span> ትውልዱ ከእነርሱ የማይቈጠረው ግን ከአብርሃም አሥራትን አውጥቶአል፥
የተስፋ ቃል የነበረውንም ባርኮአል።

<span id="an_7:7">7:7</span> ትንሹም በታላቁ እንዲባረክ ክርክር የሌለበት ነገር ነው።

<span id="an_7:8">7:8</span> በዚህስ የሚሞቱ ሰዎች አሥራትን ያስወጣሉ፥ በዚያ ግን የሚያስወጣ
በሕይወት እንዲኖር የተመሰከረለት እርሱ ነው።

<span id="an_7:9">7:9</span> ይህንም ለማለት ሲፈቀድ፥ አሥራትን የሚያስወጣ ሌዊ እንኳ በአብርሃም
እጅ አሥራትን ሰጥቶአል፤

<span id="an_7:10">7:10</span> መልከ ጼዴቅ በተገናኘው ጊዜ ገና በአባቱ ወገብ ነበረና።

<span id="an_7:11">7:11</span> እንግዲህ ህዝቡ በሌዊ ክህነት የተመሠረተን ሕግ ተቀብለዋልና በዚያ
ክህነት ፍጹምነት የተገኘ ቢሆን፥ እንደ አሮን ሹመት የማይቈጠር፥ እንደ መልከ ጼዴቅ ሹመት ግን የሆነ ሌላ ካህን
ሊነሳ ወደፊት ስለ ምን ያስፈልጋል?

<span id="an_7:12">7:12</span> ክህነቱ ሲለወጥ፥ ሕጉ ደግሞ ሊለወጥ የግድ ነውና።

<span id="an_7:13">7:13</span> ይህ ነገር የተነገረለት እርሱ በሌላ ወገን ተካፍሎአልና፥ ከዚያም
መሠዊያውን ያገለገለ ማንም የለም፤

<span id="an_7:14">7:14</span> ጌታችን ከይሁዳ ነገድ እንደወጣ የተገለጠ ነውና፥ ስለዚህም ነገድ
ሙሴ ምንም እንኳ ስለ ክህነት አልተናገረም።

<span id="an_7:15">7:15</span> በማያልፍም ሕይወት ኃይል እንጂ በሥጋ ትእዛዝ ሕግ ሳይሆን ሌላ
ካህን በመልከ ጼዴቅ ምሳሌ ቢነሳ፥ ይህ እጅግ አብልጦ የሚገለጥ ነው።

<span id="an_7:17">7:17</span> አንተ እንደ መልከ ጼዴቅ ሹመት ለዘላለም ካህን ነህ ብሎ
ይመሰክራልና።

<span id="an_7:18">7:18</span> ሕጉ ምንም ፍጹም አላደረገምና፥ ስለዚህም የምትደክም የማትጠቅምም
ስለሆነች የቀደመች ትእዛዝ ተሽራለች፥ ወደ እግዚአብሔርም የምንቀርብበት የሚሻል ተስፋ ገብቶአል።

<span id="an_7:20">7:20</span> እነርሱም ያለ መሐላ ካህናት ሆነዋልና፤ እርሱ ግን። ጌታ። አንተ
እንደ መልከ ጼዴቅ ሹመት ለዘላለም ካህን ነህ ብሎ ማለ አይጸጸትም ብሎ በተናገረለት ከመሐላ ጋር ካህን ሆኖአልና
ያለ መሐላ ካህን እንዳልሆነ መጠን፥

<span id="an_7:22">7:22</span> እንዲሁ ኢየሱስ ለሚሻል ኪዳን ዋስ ሆኖአል።

<span id="an_7:23">7:23</span> እነርሱም እንዳይኖሩ ሞት ስለ ከለከላቸው ካህናት የሆኑት ብዙ
ናቸው፤

<span id="an_7:24">7:24</span> እርሱ ግን ለዘላለም የሚኖር ስለሆነ የማይለወጥ ክህነት አለው፤

<span id="an_7:25">7:25</span> ስለ እነርሱም ሊያማልድ ዘወትር በሕይወት ይኖራልና ስለዚህ ደግሞ
በእርሱ ወደ እግዚአብሔር የሚመጡትን ፈጽሞ ሊያድናቸው ይችላል።

<span id="an_7:26">7:26</span> ቅዱስና ያለ ተንኮል ነውርም የሌለበት ከኃጢአተኞችም የተለየ
ከሰማያትም ከፍ ከፍ ያለ፥ እንደዚህ ያለ ሊቀ ካህናት ይገባልና፤

<span id="an_7:27">7:27</span> እርሱም እንደነዚያ ሊቃነ ካህናት አስቀድሞ ስለ ራሱ ኃጢአት
በኋላም ስለ ሕዝቡ ኃጢአት ዕለት ዕለት መሥዋዕትን ሊያቀርብ አያስፈልገውም፤ ራሱን ባቀረበ ጊዜ ይህን አንድ ጊዜ
ፈጽሞ አድርጎአልና።

<span id="an_7:28">7:28</span> ሕጉ ድካም ያላቸውን ሰዎች ሊቀ ካህናት አድርጎ ይሾማልና፤ ከሕግ
በኋላ የመጣ የመሐላው ቃል ግን ለዘላለም ፍጹም የሆነውን ልጅ ይሾማል።

<span id="an_8:1">8:1</span> ከተናገርነውም ዋና ነገሩ ይህ ነው፤ በሰማያት በግርማው ዙፋን ቀኝ
የተቀመጠ እንዲህ ያለ ሊቀ ካህናት አለን፤

<span id="an_8:2">8:2</span> እርሱም የመቅደስና የእውነተኛይቱ ድንኳን አገልጋይ ነው፥ እርስዋም
በሰው ሳይሆን በጌታ የተተከለች ናት።

<span id="an_8:3">8:3</span> ሊቀ ካህናት ሁሉ መባንና መሥዋዕትን ሊያቀርብ ይሾማልና፤ ስለዚህም
ለዚህ ደግሞ የሚያቀርበው አንዳች ሊኖረው የግድ ነው።

<span id="an_8:4">8:4</span> እንግዲህ በምድር ቢኖርስ፥ እንደ ሕግ መባን የሚያቀርቡት ስላሉ፥
ካህን እንኳ ባልሆነም፤

<span id="an_8:5">8:5</span> እነርሱም ሙሴ ድንኳኒቱን ሊሠራ ሳለ እንደ ተረዳ፥ ለሰማያዊ ነገር
ምሳሌና ጥላ የሚሆነውን ያገለግላሉ። በተራራው እንደ ተገለጠልህ ምሳሌ ሁሉን ታደርግ ዘንድ ተጠንቀቅ ብሎት ነበርና።

<span id="an_8:6">8:6</span> አሁን ግን በሚሻል ተስፋ ቃል በተመሠረተ በሚሻል ኪዳን ደግሞ
መካከለኛ እንደሚሆን በዚያ ልክ እጅግ የሚሻል አገልግሎት አግኝቶአል።

<span id="an_8:7">8:7</span> ፊተኛው ኪዳን ነቀፋ ባይኖረው፥ ለሁለተኛው ስፍራ ባልተፈለገም ነበር።

<span id="an_8:8">8:8</span> እነርሱን እየነቀፈ ይላቸዋልና። እነሆ፥ ከእስራኤል ቤትና ከይሁዳ ቤት
ጋር አዲስ ኪዳን የምገባበት ወራት ይመጣል ይላል ጌታ፤

<span id="an_8:9">8:9</span> ከግብፅ አገር አወጣቸው ዘንድ እጃቸውን በያዝሁበት ቀን ከአባቶቻቸው
ጋር እንደገባሁት ኪዳን አይደለም፤ እነርሱ በኪዳኔ አልጸኑምና፥ እኔም ቸል አልኋቸው ይላል ጌታ።

<span id="an_8:10">8:10</span> ከዚያ ወራት በኋላ ከእስራኤል ቤት ጋር የምገባው ቃል ኪዳን ይህ
ነውና ይላል ጌታ፤ ሕጌን በልቡናቸው አኖራለሁ በልባቸውም እጽፈዋለሁ፥ እኔም አምላክ እሆንላቸዋለሁ እነርሱም ሕዝብ
ይሆኑልኛል።

<span id="an_8:11">8:11</span> እያንዳንዱም ጐረቤቱን እያንዳንዱም ወንድሙን። ጌታን እወቅ ብሎ
አያስተምርም ከታናሹ ጀምሮ እስከ ታላቁ ድረስ ሁሉ ያውቁኛልና።

<span id="an_8:12">8:12</span> ዓመፃቸውን እምራቸዋለሁና፥ ኃጢአታቸውንም ደግሜ አላስብም።

<span id="an_8:13">8:13</span> አዲስ በማለቱ ፊተኛውን አስረጅቶአል፤ አሮጌና ውራጅ የሆነውስ
ሊጠፋ ቀርቦአል።

<span id="an_9:1">9:1</span> ፊተኛይቱም ደግሞ የአገልግሎት ስርዓትና የዚህ ዓለም የሆነው መቅደስ
ነበራት።

<span id="an_9:2">9:2</span> የመጀመሪያይቱ ድንኳን ተዘጋጅታ ነበርና፥ በእርስዋም ቅድስት
በምትባለው ውስጥ መቅረዙና ጠረጴዛው የመስዋዕቱም ኅብስት ነበረባት፤

<span id="an_9:3">9:3</span> ከሁለተኛውም መጋረጃ ወዲያ ቅድስተ ቅዱሳን የምትባለው ድንኳን
ነበረች፥

<span id="an_9:4">9:4</span> በዚያም ውስጥ የወርቅ ማዕጠንት ነበረ ሁለንተናዋም በወርቅ የተለበጠች
የኪዳን ታቦት፤ በእርስዋም ውስጥ መና ያለባት የወርቅ መሶብና የበቀለች የአሮን በትር የኪዳኑም ጽላት ነበሩ፥

<span id="an_9:5">9:5</span> በላይዋም ማስተስሪያውን የሚጋርዱ የክብር ኪሩቤል ነበሩ፤ ስለ
እነዚህም ስለ እያንዳንዳቸው ልንናገር አሁን አንችልም።

<span id="an_9:6">9:6</span> ይህም እንደዚህ ተዘጋጅቶ ሳለ፥ ካህናት አገልግሎታቸውን እየፈጸሙ
ዘወትር በፊተኛይቱ ድንኳን ይገቡባታል፤

<span id="an_9:7">9:7</span> በሁለተኛይቱ ግን ሊቀ ካህናት ብቻውን በዓመት አንድ ጊዜ ይገባባታል፥
እርሱም ሰለ ራሱና ስለ ሕዝቡ ስሕተት የሚያቀርበውን ደም ሳይዝ አይገባም፤

<span id="an_9:8">9:8</span> ፊተኛይቱም ድንኳን በዚህ ገና ቆማ ሳለች፥ ወደ ቅድስት የሚወስደው
መንገድ ገና እንዳልተገለጠ መንፈስ ቅዱስ ያሳያል።

<span id="an_9:9">9:9</span> ይህም ለአሁኑ ጊዜ ምሳሌ ነው፥ እንደዚህም መባና መስዋዕት ይቀርባሉ፤
እነዚህም እስከ መታደስ ዘመን ድረስ የተደረጉ፥ ስለ ምግብና ስለ መጠጥም ስለ ልዩ ልዩ መታጠብም የሚሆኑ የሥጋ
ሥርዓቶች ብቻ ናቸውና የሚያመልከውን በህሊና ፍጹም ሊያደርጉት አይችሉም።

<span id="an_9:11">9:11</span> ነገር ግን ክርስቶስ ይመጣ ዘንድ ላለው መልካም ነገር ሊቀ ካህናት
ሆኖ፥ በምትበልጠውና በምትሻለው በእጆችም ባልተሠራች ማለት ለዚህ ፍጥረት ባልሆነች ድንኳን፥

<span id="an_9:12">9:12</span> የዘላለምን ቤዛነት አግኝቶ አንድ ጊዜ ፈጽሞ ወደ ቅድስት በገዛ
ደሙ ገባ እንጂ በፍየሎችና በጥጆች ደም አይደለም።

<span id="an_9:13">9:13</span> የኮርማዎችና የፍየሎች ደም በረከሱትም ላይ የተረጨ የጊደር አመድ
ሥጋን ለማንጻት የሚቀድሱ ከሆኑ፥

<span id="an_9:14">9:14</span> ነውር የሌለው ሆኖ በዘላለም መንፈስ ራሱን ለእግዚአብሔር ያቀረበ
የክርስቶስ ደም እንዴት ይልቅ ሕያውን እግዚአብሔርን ልታመልኩ ከሞተ ሥራ ሕሊናችሁን ያነጻ ይሆን?

<span id="an_9:15">9:15</span> ስለዚህም የፊተኛው ኪዳን ሲጸና ሕግን የተላለፉትን የሚቤዥ ሞት
ስለ ሆነ፥ የተጠሩት የዘላለምን ርስት የተስፋ ቃል እንዲቀበሉ እርሱ የአዲስ ኪዳን መካከለኛ ነው።

<span id="an_9:16">9:16</span> ኑዛዜ ያለ እንደሆነ የተናዛዡን ሞት ማርዳት የግድ ነውና፤

<span id="an_9:17">9:17</span> ሰው ሲሞት ኑዛዜው ይጸናልና፥ ተናዛዡ በሕይወት ሲኖር ግን ከቶ
አይጠቅምም።

<span id="an_9:18">9:18</span> ስለዚህም ፊተኛው ኪዳን እንኳ ያለ ደም አልተመረቀም።

<span id="an_9:19">9:19</span> ሙሴም ትእዛዛትን ሁሉ እንደ ሕጉ ለሕዝቡ ሁሉ ከተናገረ በኋላ፥
የጥጆችንና የፍየሎችን ደም ከውኃና ከቀይ የበግ ጠጕር ከሂሶጵም ጋር ይዞ። እግዚአብሔር ያዘዘላችሁ የኪዳን ደም ይህ
ነው ብሎ በመጽሐፉና በሕዝቡ ሁሉ ላይ ረጨው።

<span id="an_9:21">9:21</span> እንዲሁም በድንኳኒቱና በማገልገያው ዕቃ ሁሉ ደምን ረጨ።

<span id="an_9:22">9:22</span> እንደ ሕጉም ከጥቂቶች በቀር ነገር ሁሉ በደም ይነጻል፥ ደምም
ሳይፈስ ስርየት የለም።

<span id="an_9:23">9:23</span> እንግዲህ በሰማያት ያሉትን የሚመስለው ነገር በዚህ ሊነጻ እንጂ
በሰማያት ያሉቱ ራሳቸው ከእርሱ ይልቅ በሚበልጥ መስዋዕት ሊነጹ የግድ ነበረ።

<span id="an_9:24">9:24</span> ክርስቶስ በእጅ ወደ ተሰራች፥ የእውነተኛይቱ ምሳሌ ወደ ምትሆን
ቅድስት አልገባምና፥ ነገር ግን በእግዚአብሔር ፊት ስለ እኛ አሁን ይታይ ዘንድ ወደ እርስዋ ወደ ሰማይ ገባ።

<span id="an_9:25">9:25</span> ሊቀ ካህናትም በየዓመቱ የሌላውን ደም ይዞ ወደ ቅድስት
እንደሚገባ፥ ራሱን ብዙ ጊዜ ሊያቀርብ አልገባም፤

<span id="an_9:26">9:26</span> እንዲህ ቢሆንስ፥ ዓለም ከተፈጠረ ጀምሮ ብዙ ጊዜ መከራ ሊቀበል
ባስፈለገው ነበር፤ አሁን ግን በዓለም ፍጻሜ ራሱን በመሠዋት ኃጢአትን ሊሽር አንድ ጊዜ ተገልጦአል።

<span id="an_9:27">9:27</span> ለሰዎችም አንድ ጊዜ መሞት ከእርሱ በኋላም ፍርድ እንደ
ተመደበባቸው፥

<span id="an_9:28">9:28</span> እንዲሁ ክርስቶስ ደግሞ፥ የብዙዎችን ኃጢአት ሊሸከም አንድ ጊዜ
ከተሰዋ በኋላ፥ ያድናቸው ዘንድ ለሚጠባበቁት ሁለተኛ ጊዜ ያለ ኃጢአት ይታይላቸዋል።

<span id="an_10:1">10:1</span> ሕጉ ሊመጣ ያለው የበጎ ነገር እውነተኛ አምሳል ሳይሆን የነገር
ጥላ አለውና፥ ስለዚህም በየዓመቱ ዘወትር በሚያቀርቡት በዚያ መሥዋዕት የሚቀርቡትን ሊፈጽም ከቶ አይችልም።

<span id="an_10:2">10:2</span> እንደዚህማ ባይሆን፥ የሚያመልኩት አንድ ጊዜ ነጽተው ከዚያ በኋላ
በሕሊናቸው ኃጢአትን ስላላወቁ ማቅረብን በተዉ አልነበረምን?

<span id="an_10:3">10:3</span> ነገር ግን በዚያ መሥዋዕት በየዓመቱ የኃጢአት መታሰቢያ አለ፤

<span id="an_10:4">10:4</span> የኮርማዎችና የፍየሎች ደም ኃጢአትን እንዲያስወግድ የማይቻል
ነውና።

<span id="an_10:5">10:5</span> ስለዚህ ወደ ዓለም ሲገባ። መሥዋዕትንና መባን አልወደድህም ሥጋን
ግን አዘጋጀህልኝ፤

<span id="an_10:6">10:6</span> በሙሉ በሚቃጠል መሥዋዕትና ሰለ ኃጢአት በሚሰዋ መሥዋዕት ደስ
አላለህም።

<span id="an_10:7">10:7</span> በዚያን ጊዜ። እነሆ፥ በመጽሐፍ ጥቅልል ስለ እኔ እንደ ተጻፈ፥
አምላኬ ሆይ፥ ፈቃድህን ላደርግ መጥቼአለሁ አልሁ

<span id="an_10:8">10:8</span> ይላል። በዚህ ላይ። መሥዋዕትንና መባን በሙሉ የሚቃጠል
መሥዋዕትንም ስለ ኃጢአትም የሚሰዋ መሥዋዕትን አልወደድህም በእርሱም ደስ አላለህም ብሎ፥ እነዚህም እንደ ሕግ
የሚቀርቡት ናቸው፥

<span id="an_10:9">10:9</span> ቀጥሎ። እነሆ፥ አምላኬ ሆይ፥ ፈቃድህን ላደርግ መጥቼአለሁ
ብሎአል። ሁለተኛውንም ሊያቆም የፊተኛውን ይሽራል።

<span id="an_10:10">10:10</span> በዚህም ፈቃድ የኢየሱስ ክርስቶስን ሥጋ አንድ ጊዜ ፈጽሞ
በማቅረብ ተቀድሰናል።

<span id="an_10:11">10:11</span> ሊቀ ካህናትም ሁሉ ዕለት ዕለት እያገለገለ ኃጢአትን ሊያስወግዱ
ከቶ የማይችሉትን እነዚያን መሥዋዕቶች ብዙ ጊዜ እያቀረበ ቆሞአል፤

<span id="an_10:12">10:12</span> እርሱ ግን ስለ ኃጢአት አንድን መሥዋዕት ለዘላለም አቅርቦ
በእግዚአብሔር ቀኝ ተቀመጠ፥

<span id="an_10:13">10:13</span> ጠላቶቹም የእግሩ መረገጫ እስኪደረጉ ድረስ ወደ ፊት ይጠብቃል።

<span id="an_10:14">10:14</span> አንድ ጊዜ በማቅረብ የሚቀደሱትን የዘላለም ፍጹማን
አድርጎአቸዋልና።

<span id="an_10:15">10:15</span> መንፈስ ቅዱስም ደግሞ ስለዚህ ይመሰክርልናል፤ ከዚያ ወራት
በኋላ ከእነርሱ ጋር የምገባው ኪዳን ይህ ነው ይላል ጌታ፤ በልባቸው ሕጌን አኖራለሁ በልቡናቸውም እጽፈዋለሁ

<span id="an_10:17">10:17</span> ብሎ ከተናገረ በኋላ፥ ኃጢአታቸውንና ዓመጻቸውንም ደግሜ
አላስብም ይላል።

<span id="an_10:18">10:18</span> የእነዚህም ስርየት ባለበት ዘንድ፥ ከዚህ ወዲህ ስለ ኃጢአት
ማቅረብ የለም።

<span id="an_10:19">10:19</span> እንግዲህ፥ ወንድሞች ሆይ፥ በመረቀልን በአዲስና በሕያው መንገድ
ወደ ቅድስት በኢየሱስ ደም በመጋረጃው ማለት በሥጋው በኩል እንድንገባ ድፍረት ስላለን፥

<span id="an_10:21">10:21</span> በእግዚአብሔርም ቤት ላይ የሆነ ታላቅ ካህን ስላለን፥

<span id="an_10:22">10:22</span> ከክፉ ሕሊና ለመንጻት ልባችንን ተረጭተን ሰውነታችንንም በጥሩ
ውኃ ታጥበን በተረዳንበት እምነት በቅን ልብ እንቅረብ፤

<span id="an_10:23">10:23</span> የተስፋን ቃል የሰጠው የታመነ ነውና እንዳይነቃነቅ የተስፋችንን
ምስክርነት እንጠብቅ፤

<span id="an_10:24">10:24</span> ለፍቅርና ለመልካምም ሥራ እንድንነቃቃ እርስ በርሳችን
እንተያይ፤

<span id="an_10:25">10:25</span> በአንዳንዶችም ዘንድ ልማድ እንደ ሆነው፥ መሰብሰባችንን አንተው
እርስ በርሳችን እንመካከር እንጂ፤ ይልቁንም ቀኑ ሲቀርብ እያያችሁ አብልጣችሁ ይህን አድርጉ።

<span id="an_10:26">10:26</span> የእውነትን እውቀት ከተቀበልን በኋላ ወደን ኃጢአት ብናደርግ
ከእንግዲህ ወዲህ ስለ ኃጢአት መሥዋዕት አይቀርልንምና፥

<span id="an_10:27">10:27</span> የሚያስፈራ ግን የፍርድ መጠበቅ ተቃዋሚዎችንም ሊበላ ያለው
የእሳት ብርታት አለ።

<span id="an_10:28">10:28</span> የሙሴን ሕግ የናቀ ሰው ሁለት ወይም ሦስት ቢመሰክሩበት ያለ
ርኅራኄ ይሞታል፤

<span id="an_10:29">10:29</span> የእግዚአብሔርን ልጅ የረገጠ ያንንም የተቀደሰበትን የኪዳኑን
ደም እንደ ርኵስ ነገር የቆጠረ የጸጋውንም መንፈስ ያክፋፋ፥ እንዴት ይልቅ የሚብስ ቅጣት የሚገባው ይመስላችኋል?

<span id="an_10:30">10:30</span> በቀል የእኔ ነው፥ እኔ ብድራትን እመልሳለሁ ያለውን
እናውቃለንና፤ ደግሞም። ጌታ በሕዝቡ ይፈርዳል።

<span id="an_10:31">10:31</span> በሕያው እግዚአብሔር እጅ መውደቅ የሚያስፈራ ነው።

<span id="an_10:32">10:32</span> ነገር ግን ግማሽ በነቀፋና በጭንቅ እንደ መጫወቻ ስለ ሆናችሁ
ግማሽም እንዲህ ካሉት ጋር ስለ ተካፈላችሁ፥ ብርሃን ከበራላችሁ በኋላ መከራ በሆነበት በትልቅ ተጋድሎ የጸናችሁበትን
የቀደመውን ዘመን አስቡ።

<span id="an_10:34">10:34</span> የሚበልጥና ለዘወትር የሚኖር ገንዘብ በሰማይ ራሳችሁ እንዳላችሁ
አውቃችሁ፥ በእስራቴ ራራችሁልኝ የገንዘባችሁንም ንጥቂያ በደስታ ተቀበላችሁ።

<span id="an_10:35">10:35</span> እንግዲህ ታላቅ ብድራት ያለውን ድፍረታችሁን አትጣሉ።

<span id="an_10:36">10:36</span> የእግዚአብሔርን ፈቃድ አድርጋችሁ የተሰጣችሁን የተስፋ ቃል
እንድታገኙ መጽናት ያስፈልጋችኋልና።

<span id="an_10:37">10:37</span> ገና በጣም ጥቂት ጊዜ ነው፥ ሊመጣ ያለውም ይመጣል
አይዘገይምም፤

<span id="an_10:38">10:38</span> ጻድቅ ግን በእምነት ይኖራል ወደ ኋላም ቢያፈገፍግ፥ ነፍሴ
በእርሱ ደስ አይላትም።

<span id="an_10:39">10:39</span> እኛ ግን ነፍሳቸውን ሊያድኑ ከሚያምኑቱ ነን እንጂ ወደ ጥፋት
ከሚያፈገፍጉ አይደለንም።

<span id="an_11:1">11:1</span> እምነትም ተስፋ ስለምናደርገው ነገር የሚያስረግጥ፥ የማናየውንም
ነገር የሚያስረዳ ነው።

<span id="an_11:2">11:2</span> ለሽማግሌዎች የተመሰከረላቸው በዚህ ነውና።

<span id="an_11:3">11:3</span> ዓለሞች በእግዚአብሔር ቃል እንደ ተዘጋጁ፥ ስለዚህም የሚታየው
ነገር ከሚታዩት እንዳልሆነ በእምነት እናስተውላለን።

<span id="an_11:4">11:4</span> አቤል ከቃየል ይልቅ የሚበልጥን መሥዋዕት ለእግዚአብሔር በእምነት
አቀረበ፥ በዚህም፥ እግዚአብሔር ስለ ስጦታው ሲመሰክር፥ እርሱ ጻድቅ እንደ ሆነ ተመሰከረለት፤ ሞቶም ሳለ በመሥዋዕቱ
እስከ አሁን ይናገራል።

<span id="an_11:5">11:5</span> ሄኖክ ሞትን እንዳያይ በእምነት ተወሰደ፥ እግዚአብሔርም ስለ
ወሰደው አልተገኘም። ሳይወሰድ እግዚአብሔርን ደስ እንዳሰኘ ተመስክሮለታልና፤

<span id="an_11:6">11:6</span> ያለ እምነትም ደስ ማሰኘት አይቻልም፤ ወደ እግዚአብሔር የሚደርስ
እግዚአብሔር እንዳለ ለሚፈልጉትም ዋጋ እንዲሰጥ ያምን ዘንድ ያስፈልገዋልና።

<span id="an_11:7">11:7</span> ኖኅ ገና ስለማይታየው ነገር ተረድቶ እግዚአብሔርን እየፈራ ቤተ
ስዎቹን ለማዳን መርከብን በእምነት አዘጋጀ፥ በዚህም ዓለምን ኰነነ፥ በእምነትም የሚገኘውን ጽድቅ ወራሽ ሆነ።

<span id="an_11:8">11:8</span> አብርሃም የተባለው ርስት አድርጎ ሊቀበለው ወዳለው ስፍራ ለመውጣት
በእምነት ታዘዘ፥ ወዴትም እንደሚሄድ ሳያውቅ ወጣ።

<span id="an_11:9">11:9</span> ለእንግዶች እንደሚሆን በተስፋ ቃል በተሰጠው አገር በድንኳን ኖሮ፥
ያን የተስፋ ቃል አብረውት ከሚወርሱ ከይስሐቅና ከያዕቆብ ጋር፥ እንደ መጻተኛ በእምነት ተቀመጠ፤

<span id="an_11:10">11:10</span> መሠረት ያላትን፥ እግዚአብሔር የሠራትንና የፈጠራትን ከተማ
ይጠብቅ ነበርና።

<span id="an_11:11">11:11</span> ተስፋ የሰጠው የታመነ እንደ ሆነ ስለ ቈጠረች፥ ሣራ ራስዋ
ደግሞ ዕድሜዋ እንኳ ካለፈ በኋላ ዘርን ለመፅነስ ኃይልን በእምነት አገኘች።

<span id="an_11:12">11:12</span> ስለዚህ ደግሞ በብዛታቸው እንደ ሰማይ ኮከብ እንደማይቈጠርም
በባሕር ዳር እንዳለ አሸዋ የነበሩት የሞተን ሰው እንኳ ከመሰለው ከአንዱ ተወለዱ።

<span id="an_11:13">11:13</span> እነዚህ ሁሉ አምነው ሞቱ፥ የተሰጣቸውን የተስፋ ቃል
አላገኙምና፤ ዳሩ ግን ከሩቅ ሆነው አዩትና ተሳለሙት፥ በምድሪቱም እንግዶችና መጻተኞች እንዲሆኑ ታመኑ።

<span id="an_11:14">11:14</span> እንዲህ የሚሉት ለእነርሱ የሚሆነውን አገር እንዲፈልጉ
ያመለክታሉና።

<span id="an_11:15">11:15</span> ያን የወጡበትን አገር አስበው ቢሆኑ፥ ሊመለሱ ጊዜ በሆነላቸው
ነበር፤

<span id="an_11:16">11:16</span> አሁን ግን የሚበልጠውን እርሱም ሰማያዊ አገር ይናፍቃሉ።
ስለዚህ እግዚአብሔር አምላካቸው ተብሎ ሊጠራ በእነርሱ አያፍርም፥ ከተማን አዘጋጅቶላቸዋልና።

<span id="an_11:17">11:17</span> አብርሃም በተፈተነበት ጊዜ ይስሐቅን በእምነት አቀረበ፥
የተስፋን ቃል የተቀበለው። በይስሐቅ ዘር ይጠራልሃል የተባለለት እርሱም አንድ ልጁን አቀረበ፤

<span id="an_11:19">11:19</span> እግዚአብሔር ከሙታን እንኳ ሊያስነሣው እንዲቻለው አስቦአልና፥
ከዚያም ደግሞ በምሳሌ አገኘው።

<span id="an_11:20">11:20</span> ይስሐቅ ሊመጣ ስላለው ነገር ያዕቆብንና ዔሳውን በእምነት
ባረካቸው።

<span id="an_11:21">11:21</span> ያዕቆብ ሲሞት በእምነት የዮሴፍን ልጆች እያንዳንዳቸው
ባረካቸው፥ በዘንጉም ጫፍ ተጠግቶ ሰገደ።

<span id="an_11:22">11:22</span> ዮሴፍ ወደ ሞት ቀርቦ ሳለ በእምነት ስለ እስራኤል ልጆች
መውጣት አስታወሰ፥ ስለ አጥንቱም አዘዛቸው።

<span id="an_11:23">11:23</span> ሙሴ ከተወለደ በኋላ ወላጆቹ ያማረ ሕፃን መሆኑን አይተው
በእምነት ሦስት ወር ሸሸጉት የንጉሥንም አዋጅ አልፈሩም።

<span id="an_11:24">11:24</span> ሙሴ ካደገ በኋላ የፈርዖን የልጅ ልጅ እንዳይባል በእምነት
እምቢ አለ፤

<span id="an_11:25">11:25</span> ከግብፅም ብዙ ገንዘብ ይልቅ ስለ ክርስቶስ መነቀፍ እጅግ
የሚበልጥ ባለ ጠግነት እንዲሆን አስቦአልና ለጊዜው በኃጢአት ከሚገኝ ደስታ ይልቅ ከእግዚአብሔር ሕዝብ ጋር መከራ
መቀበልን መረጠ፤ ብድራቱን ትኵር ብሎ ተመልክቶአልና።

<span id="an_11:27">11:27</span> የንጉሡን ቍጣ ሳይፈራ የግብፅን አገር የተወ በእምነት ነበር፤
የማይታየውን እንደሚያየው አድርጎ ጸንቶአልና።

<span id="an_11:28">11:28</span> አጥፊው የበኵሮችን ልጆች እንዳይነካ ፋሲካንና ደምን መርጨትን
በእምነት አደረገ።

<span id="an_11:29">11:29</span> በደረቅ ምድር እንደሚያልፉ በኤርትራ ባሕር በእምነት ተሻገሩ፥
የግብፅ ሰዎች ግን ሲሞክሩ ተዋጡ።

<span id="an_11:30">11:30</span> የኢያሪኮ ቅጥር ሰባት ቀን ከዞሩበት በኋላ በእምነት ወደቀ።

<span id="an_11:31">11:31</span> ጋለሞታይቱ ረዓብ ሰላዮችን በሰላም ስለ ተቀበለቻቸው ከማይታዘዙ
ጋር በእምነት አልጠፋችም።

<span id="an_11:32">11:32</span> እንግዲህ ምን እላለሁ? ስለ ጌዴዎንና ስለ ባርቅ ስለ ሶምሶንም
ስለ ዮፍታሔም ስለ ዳዊትና ስለ ሳሙኤልም ስለ ነቢያትም እንዳልተርክ ጊዜ ያጥርብኛልና።

<span id="an_11:33">11:33</span> እነርሱ በእምነት መንግሥታትን ድል ነሡ፥ ጽድቅን አደረጉ፥
የተሰጠውን የተስፋ ቃል አገኙ፥

<span id="an_11:34">11:34</span> የአንበሶችን አፍ ዘጉ፥ የእሳትን ኃይል አጠፉ፥ ከሰይፍ ስለት
አመለጡ፥ ከድካማቸው በረቱ፥ በጦርነት ኃይለኞች ሆኑ፥ የባዕድ ጭፍሮችን አባረሩ።

<span id="an_11:35">11:35</span> ሴቶች ሙታናቸውን በትንሣኤ ተቀበሉ፤ ሌሎችም መዳንን ሳይቀበሉ
የሚበልጠውን ትንሣኤ እንዲያገኙ እስከ ሞት ድረስ ተደበደቡ፤

<span id="an_11:36">11:36</span> ሌሎችም መዘበቻ በመሆንና በመገረፍ ከዚህም በላይ በእስራትና
በወኅኒ ተፈተኑ፤

<span id="an_11:37">11:37</span> በድንጋይ ተወግረው ሞቱ፥ ተፈተኑ፥ በመጋዝ ተሰነጠቁ፥ በሰይፍ
ተገድለው ሞቱ፥ ሁሉን እያጡ መከራን እየተቀበሉ እየተጨነቁ የበግና የፍየል ሌጦ ለብሰው ዞሩ፤

<span id="an_11:38">11:38</span> ዓለም አልተገባቸውምና በምድረ በዳና በተራራ፥ በዋሻና በምድር
ጕድጓድ ተቅበዘበዙ።

<span id="an_11:39">11:39</span> እነዚህም ሁሉ በእምነታቸው ተመስክሮላቸው ሳሉ የተሰጠውን
የተስፋ ቃል አላገኙም፥ ያለ እኛ ፍጹማን እንዳይሆኑ እግዚአብሔር ስለ እኛ አንዳች የሚበልጥ ነገርን አስቀድሞ አይቶ
ነበርና።

<span id="an_12:1">12:1</span> እንግዲህ እነዚህን የሚያህሉ ምስክሮች እንደ ደመና በዙሪያችን
ካሉልን፥ እኛ ደግሞ ሸክምን ሁሉ ቶሎም የሚከበንን ኃጢአት አስወግደን፥ የእምነታችንንም ራስና ፈጻሚውን ኢየሱስን
ተመልክተን፥ በፊታችን ያለውን ሩጫ በትዕግሥት እንሩጥ፤ እርሱ ነውርን ንቆ በፊቱም ስላለው ደስታ በመስቀል ታግሦ
በእግዚአብሔር ዙፋን ቀኝ ተቀምጦአልና።

<span id="an_12:3">12:3</span> በነፍሳችሁ ዝላችሁ እንዳትደክሙ፥ ከኃጢአተኞች በደረሰበት እንዲህ
ባለ መቃወም የጸናውን አስቡ።

<span id="an_12:4">12:4</span> ከኃጢአት ጋር እየተጋደላችሁ ገና ደምን እስከ ማፍሰስ ድረስ
አልተቃወማችሁም፤

<span id="an_12:5">12:5</span> እንደ ልጆችም ከእናንተ ጋር። ልጄ ሆይ፥ የጌታን ቅጣት አታቅልል፥
በሚገሥጽህም ጊዜ አትድከም፤ ጌታ የሚወደውን ይቀጣዋልና፥ የሚቀበለውንም ልጅ ሁሉ ይገርፈዋል ብሎ የሚነጋገረውን
ምክር ረስታችኋል።

<span id="an_12:7">12:7</span> ለመቀጣት ታገሡ፤ እግዚአብሔር እንደ ልጆች ያደርግላችኋልና፤ አባቱ
የማይቀጣው ልጅ ማን ነው?

<span id="an_12:8">12:8</span> ነገር ግን ሁሉ የቅጣት ተካፋይ ሆኖአልና ያለ ቅጣት ብትኖሩ
ዲቃላዎች እንጂ ልጆች አይደላችሁም።

<span id="an_12:9">12:9</span> ከዚህም በላይ የቀጡን የሥጋችን አባቶች ነበሩን እናፍራቸውም
ነበር፤ እንዴትስ ይልቅ ለመናፍስት አባት አብልጠን ልንገዛና በሕይወት ልንኖር በተገባን?

<span id="an_12:10">12:10</span> እነርሱ መልካም ሆኖ እንደ ታያቸው ለጥቂት ቀን ይቀጡን
ነበርና፥ እርሱ ግን ከቅድስናው እንድንካፈል ለጥቅማችን ይቀጣናል።

<span id="an_12:11">12:11</span> ቅጣት ሁሉ ለጊዜው የሚያሳዝን እንጂ ደስ የሚያሰኝ አይመስልም፥
ዳሩ ግን በኋላ ለለመዱት የሰላምን ፍሬ እርሱም ጽድቅን ያፈራላቸዋል።

<span id="an_12:12">12:12</span> ስለዚህ የላሉትን እጆች የሰለሉትንም ጉልበቶች አቅኑ፤

<span id="an_12:13">12:13</span> ያነከሰውም እንዲፈወስ እንጂ እንዳይናጋ፥ ለእግራችሁ ቅን
መንገድ አድርጉ።

<span id="an_12:14">12:14</span> ከሰው ሁሉ ጋር ሰላምን ተከታተሉ ትቀደሱም ዘንድ ፈልጉ፥ ያለ
እርሱ ጌታን ሊያይ የሚችል የለምና።

<span id="an_12:15">12:15</span> የእግዚአብሔር ጸጋ ለማንም እንዳይጎድለው፥ ብዙዎቹም
የሚረክሱበት አንድ መራራ ሥር ወደ ላይ በቅሎ እንዳያስጨንቅ፥ ሴሰኛም የሚሆን እንዳይገኝ፥

<span id="an_12:16">12:16</span> ወይም ስለ አንድ መብል በኵርነቱን እንደ ሸጠ እንደ ዔሳው
ለዚህ ዓለም የሚመች ሰው እንዳይሆን ተጠንቀቁ።

<span id="an_12:17">12:17</span> ከዚያ በኋላ እንኳ በረከቱን ሊወርስ በወደደ ጊዜ እንደ ተጣለ
ታውቃላችሁና፤ በእንባ ተግቶ ምንም ቢፈልገው ለንስሐ ስፍራ አላገኘምና።

<span id="an_12:18">12:18</span> ሊዳሰስ ወደሚችል ወደሚቃጠልም እሳት ወደ ጭጋግም ወደ ጨለማም
ወደ ዐውሎ ነፋስም ወደ መለከት ድምፅም ወደ ቃሎችም ነገር አልደረሳችሁምና፤

<span id="an_12:19">12:19</span> ያንም ነገር የሰሙት ሌላ ቃል እንዳይጨመርባቸው ለመኑ፤

<span id="an_12:20">12:20</span> እንስሳ እንኳ ተራራውን ቢነካ ተወግሮ ይሙት የምትለውን ትእዛዝ
ሊታገሡ አልቻሉምና፤

<span id="an_12:21">12:21</span> ሙሴም። እጅግ እፈራለሁ እንቀጠቀጥማለሁ እስኪል ድረስ የሚታየው
እጅግ የሚያስፈራ ነበር፤

<span id="an_12:22">12:22</span> ነገር ግን ወደ ጽዮን ተራራና ወደ ሕያው እግዚአብሔር ከተማ
ደርሳችኋል፥ ወደ ሰማያዊቱም ኢየሩሳሌም፥ በደስታም ወደ ተሰበሰቡት ወደ አእላፋት መላእክት፥

<span id="an_12:23">12:23</span> በሰማያትም ወደ ተጻፉ ወደ በኵራት ማኅበር፥ የሁሉም ዳኛ
ወደሚሆን ወደ እግዚአብሔር፥ ፍጹማንም ወደ ሆኑት ወደ ጻድቃን መንፈሶች፥

<span id="an_12:24">12:24</span> የአዲስም ኪዳን መካከለኛ ወደሚሆን ወደ ኢየሱስ፥ ከአቤልም ደም
ይልቅ የሚሻለውን ወደሚናገር ወደ መርጨት ደም ደርሳችኋል።

<span id="an_12:25">12:25</span> ለሚናገረው እምቢ እንዳትሉ ተጠንቀቁ፤ እነዚያ በምድር
ላስረዳቸው እምቢ ባሉ ጊዜ ካላመለጡ፥ ከሰማይ ከመጣው ፈቀቅ የምንል እኛስ እንዴት እናመልጣለን?

<span id="an_12:26">12:26</span> በዚያም ጊዜ ድምፁ ምድርን አናወጠ፥ አሁን ግን። አንድ ጊዜ
ደግሜ እኔ ሰማይን አናውጣለሁ እንጂ ምድርን ብቻ አይደለም ብሎ ተስፋ ሰጥቶአል።

<span id="an_12:27">12:27</span> ዳሩ ግን። አንድ ጊዜ ደግሜ የሚል ቃል፥ የማይናወጡት ጸንተው
እንዲኖሩ፥ የሚናወጡት የተፈጠሩ እንደሚሆኑ ይለወጡ ዘንድ ያሳያል።

<span id="an_12:28">12:28</span> ስለዚህ የማይናወጥን መንግሥት ስለምንቀበል በማክበርና በፍርሃት
እግዚአብሔርን ደስ እያሰኘን የምናመልክበትን ጸጋ እንያዝ፤

<span id="an_12:29">12:29</span> አምላካችን በእውነት የሚያጠፋ እሳት ነውና።

<span id="an_13:1">13:1</span> የወንድማማች መዋደድ ይኑር። እንግዶችን መቀበል አትርሱ፤ በዚህ
አንዳንዶች ሳያውቁ መላእክትን እንግድነት ተቀብለዋልና።

<span id="an_13:3">13:3</span> ከእነርሱ ጋር እንደ ታሰረ ሆናችሁ እስሮችን አስቡ፥ የተጨነቁትንም
ራሳችሁ ደግሞ በሥጋ እንዳለ ሆናችሁ አስቡ።

<span id="an_13:4">13:4</span> መጋባት በሁሉ ዘንድ ክቡር መኝታውም ንጹሕ ይሁን፤ ሴሰኞችንና
አመንዝሮችን ግን እግዚአብሔር ይፈርድባቸዋል።

<span id="an_13:5">13:5</span> አካሄዳችሁ ገንዘብን ያለ መውደድ ይሁን፥ ያላችሁም ይብቃችሁ፤
እርሱ ራሱ። አልለቅህም ከቶም አልተውህም ብሎአልና፤

<span id="an_13:6">13:6</span> ስለዚህ በድፍረት። ጌታ ይረዳኛልና አልፈራም፤ ሰው ምን
ያደርገኛል? እንላለን።

<span id="an_13:7">13:7</span> የእግዚአብሔርን ቃል የተናገሩአችሁን ዋኖቻችሁን አስቡ፥
የኑሮአቸውንም ፍሬ እየተመለከታችሁ በእምነታቸው ምሰሉአቸው።

<span id="an_13:8">13:8</span> ኢየሱስ ክርስቶስ ትናንትና ዛሬ እስከ ለዘላለምም ያው ነው።

<span id="an_13:9">13:9</span> ልዩ ልዩ ዓይነት በሆነ በእንግዳ ትምህርት አትወሰዱ፤ ልባችሁ
በጸጋ ቢጸና መልካም ነው እንጂ በመብል አይደለም፤ በዚህ የሚሠሩባት አልተጠቀሙምና።

<span id="an_13:10">13:10</span> መሠዊያ አለን፥ ከእርሱም ሊበሉ ድንኳኒቱን የሚያገለግሉ መብት
የላቸውም።

<span id="an_13:11">13:11</span> ሊቀ ካህናት ስለ ኃጢአት ወደ ቅድስት የእንስሶችን ደም
ያቀርባልና፤ ሥጋቸው ግን ከሰፈሩ ውጭ ይቃጠላል።

<span id="an_13:12">13:12</span> ስለዚህ ኢየሱስ ደግሞ በገዛ ደሙ ሕዝቡን እንዲቀድስ ከበር ውጭ
መከራን ተቀበለ።

<span id="an_13:13">13:13</span> እንግዲህ ነቀፌታውን እየተሸከምን ወደ እርሱ ወደ ሰፈሩ ውጭ
እንውጣ፤

<span id="an_13:14">13:14</span> በዚህ የምትኖር ከተማ የለችንምና፥ ነገር ግን ትመጣ ዘንድ
ያላትን እንፈልጋለን።

<span id="an_13:15">13:15</span> እንግዲህ ዘወትር ለእግዚአብሔር የምስጋናን መሥዋዕት፥ ማለት
ለስሙ የሚመሰክሩ የከንፈሮችን ፍሬ፥ በእርሱ እናቅርብለት።

<span id="an_13:16">13:16</span> ነገር ግን መልካም ማድረግን ለሌሎችም ማካፈልን አትርሱ፤
እንዲህ ያለው መሥዋዕት እግዚአብሔርን ደስ ያሰኘዋልና።

<span id="an_13:17">13:17</span> ለዋኖቻችሁ ታዘዙና ተገዙ፤ እነርሱ ስሌትን እንደሚሰጡ
አድርገው፥ ይህንኑ በደስታ እንጂ በኃዘን እንዳያደርጉት፥ ይህ የማይጠቅማችሁ ነበርና፥ ስለ ነፍሳችሁ ይተጋሉ።

<span id="an_13:18">13:18</span> ጸልዩልን፤ በነገር ሁሉ በመልካም እንድንኖር ወደን፥ መልካም
ሕሊና እንዳለን ተረድተናልና።

<span id="an_13:19">13:19</span> ይልቁንም ፈጥኜ እንድመለስላችሁ ይህን ታደርጉ ዘንድ አጥብቄ
እለምናችኋለሁ።

<span id="an_13:20">13:20</span> በዘላለም ኪዳን ደም ለበጎች ትልቅ እረኛ የሆነውን ጌታችንን
ኢየሱስን ከሙታን ያወጣው የሰላም አምላክ፥

<span id="an_13:21">13:21</span> በኢየሱስ ክርስቶስ በኩል በፊቱ ደስ የሚያሰኘውን በእናንተ
እያደረገ፥ ፈቃዱን ታደርጉ ዘንድ በመልካም ሥራ ሁሉ ፍጹማን ያድርጋችሁ፤ ለእርሱ እስከ ዘላለም ድረስ ክብር ይሁን፤
አሜን።

<span id="an_13:22">13:22</span> ወንድሞች ሆይ፥ የምክርን ቃል እንድትታገሡ እመክራችኋለሁ፥
በጥቂት ቃል ጽፌላችኋለሁና።

<span id="an_13:23">13:23</span> ወንድማችን ጢሞቴዎስ እንደ ተፈታ እወቁ፥ ቶሎ ብሎም ቢመጣ
ከእርሱ ጋር አያችኋለሁ።

<span id="an_13:24">13:24</span> ለዋኖቻችሁ ሁሉና ለቅዱሳን ሁሉ ሰላምታ አቅርቡልኝ። ከኢጣልያ
የሆኑቱ ሰላምታ ያቀርቡላችኋል።

<span id="an_13:25">13:25</span> ጸጋ ከሁላችሁ ጋር ይሁን፤ አሜን።
