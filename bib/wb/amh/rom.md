### Amharic Bible: Romans

<span id="an_1:1">1:1</span> ሐዋርያ ሊሆን የተጠራ የኢየሱስ ክርስቶስ ባሪያ ጳውሎስ በነቢያቱ አፍ
በቅዱሳን መጻሕፍት አስቀድሞ ተስፋ ለሰጠው ለእግዚአብሔር ወንጌል ተለየ።

<span id="an_1:3">1:3</span> ይህም ወንጌል በሥጋ ከዳዊት ዘር ስለ ተወለደ እንደ ቅድስና መንፈስ
ግን ከሙታን መነሣት የተነሣ በኃይል የእግዚአብሔር ልጅ ሆኖ ስለ ተገለጠ ስለ ልጁ ነው፤ እርሱም ጌታችን ኢየሱስ
ክርስቶስ ነው።

<span id="an_1:5">1:5</span> በእርሱም ስለ ስሙ በአሕዛብ ሁሉ መካከል ከእምነት የሚነሣ መታዘዝ
እንዲገኝ ጸጋንና ሐዋርያነትን ተቀበልን፤

<span id="an_1:6">1:6</span> በእነርሱም መካከል የኢየሱስ ክርስቶስ ልትሆኑ የተጠራችሁ እናንተ
ደግሞ ናችሁ።

<span id="an_1:7">1:7</span> በእግዚአብሔር ለተወደዳችሁና ቅዱሳን ልትሆኑ ለተጠራችሁ በሮሜ
ላላችሁት ሁሉ፥ ከእግዚአብሔር ከአባታችን ከጌታም ከኢየሱስ ክርስቶስ ጸጋና ሰላም ለእናንተ ይሁን።

<span id="an_1:8">1:8</span> እምነታችሁ በዓለም ሁሉ ስለ ተሰማች አስቀድሜ ስለ ሁላችሁ አምላኬን
በኢየሱስ ክርስቶስ አመሰግናለሁ።

<span id="an_1:9">1:9</span> በልጁ ወንጌል በመንፈሴ የማገለግለው እግዚአብሔር ምስክሬ ነውና፤
ምናልባት ብዙ ቆይቼ ወደ እናንተ አሁን እንድመጣ በእግዚአብሔር ፈቃድ መንገዴን እንዲያቀናልኝ እየለመንሁ ሁልጊዜ
ስጸልይ ስለ እናንተ ሳላቋርጥ አሳስባለሁ።

<span id="an_1:11">1:11</span> ትጸኑ ዘንድ መንፈሳዊ ስጦታ እንዳካፍላችሁ ላያችሁ እናፍቃለሁና፤

<span id="an_1:12">1:12</span> ይህንም ማለቴ በመካከላችን ባለች በእናንተና በእኔ እምነት አብረን
በእናንተ እንድንጽናና ነው።

<span id="an_1:13">1:13</span> ወንድሞች ሆይ፥ በሌሎቹ አሕዛብ ደግሞ እንደ ሆነ በእናንተም ፍሬ
አገኝ ዘንድ ብዙ ጊዜ ወደ እናንተ ልመጣ እንዳሰብሁ እስከ አሁን ግን እንደ ተከለከልሁ ታውቁ ዘንድ እወዳለሁ።

<span id="an_1:14">1:14</span> ለግሪክ ሰዎችና ላልተማሩም፥ ለጥበበኞችና ለማያስተውሉም ዕዳ
አለብኝ፤

<span id="an_1:15">1:15</span> ስለዚህም በሚቻለኝ መጠን በሮሜ ላላችሁ ለእናንተ ደግሞ ወንጌልን
ልሰብክ ተዘጋጅቼአለሁ።

<span id="an_1:16">1:16</span> በወንጌል አላፍርምና፤ አስቀድሞ ለአይሁዳዊ ደግሞም ለግሪክ ሰው፥
ለሚያምኑ ሁሉ የእግዚአብሔር ኃይል ለማዳን ነውና።

<span id="an_1:17">1:17</span> ጻድቅ በእምነት ይኖራል ተብሎ እንደ ተጻፈ የእግዚአብሔር ጽድቅ
ከእምነት ወደ እምነት በእርሱ ይገለጣልና።

<span id="an_1:18">1:18</span> እውነትን በዓመፃ በሚከለክሉ ሰዎች በኃጢአተኝነታቸውና በዓመፃቸው
ሁሉ ላይ የእግዚአብሔር ቍጣ ከሰማይ ይገለጣልና፤

<span id="an_1:19">1:19</span> እግዚአብሔር ስለ ገለጠላቸው፥ ስለ እግዚአብሔር ሊታወቅ የሚቻለው
በእነርሱ ዘንድ ግልጥ ነውና።

<span id="an_1:20">1:20</span> የማይታየው ባሕርይ እርሱም የዘላለም ኃይሉ ደግሞም አምላክነቱ
ከዓለም ፍጥረት ጀምሮ ከተሠሩት ታውቆ ግልጥ ሆኖ ይታያልና፤ ስለዚህም እግዚአብሔርን እያወቁ እንደ እግዚአብሔርነቱ
መጠን ስላላከበሩትና ስላላመሰገኑት የሚያመካኙት አጡ፤ ነገር ግን በአሳባቸው ከንቱ ሆኑ የማያስተውለውም ልባቸው
ጨለመ።

<span id="an_1:22">1:22</span> ጥበበኞች ነን ሲሉ ደንቆሮ ሆኑ፥

<span id="an_1:23">1:23</span> የማይጠፋውንም የእግዚአብሔር ክብር በሚጠፋ ሰውና በወፎች አራት
እግር ባላቸውም በሚንቀሳቀሱትም መልክ መስለው ለወጡ።

<span id="an_1:24">1:24</span> ስለዚህ እርስ በርሳቸው ሥጋቸውን ሊያዋርዱ እግዚአብሔር በልባቸው
ፍትወት ወደ ርኵስነት አሳልፎ ሰጣቸው፤

<span id="an_1:25">1:25</span> ይህም የእግዚአብሔርን እውነት በውሸት ስለ ለወጡ በፈጣሪም ፈንታ
የተፈጠረውን ስላመለኩና ስላገለገሉ ነው፤ እርሱም ለዘላለም የተባረከ ነው፤ አሜን።

<span id="an_1:26">1:26</span> ስለዚህ እግዚአብሔር ለሚያስነውር ምኞት አሳልፎ ሰጣቸው፤
ሴቶቻቸውም ለባሕርያቸው የሚገባውን ሥራ ለባሕርያቸው በማይገባው ለወጡ፤

<span id="an_1:27">1:27</span> እንዲሁም ወንዶች ደግሞ ለባሕርያቸው የሚገባውን ሴቶችን መገናኘት
ትተው እርስ በርሳቸው በፍትወታቸው ተቃጠሉ፤ ወንዶችም በወንዶች ነውር አድርገው በስሕተታቸው የሚገባውን ብድራት
በራሳቸው ተቀበሉ።

<span id="an_1:28">1:28</span> እግዚአብሔርን ለማወቅ ባልወደዱት መጠን እግዚአብሔር የማይገባውን
ያደርጉ ዘንድ ለማይረባ አእምሮ አሳልፎ ሰጣቸው፤

<span id="an_1:29">1:29</span> ዓመፃ ሁሉ፥ ግፍ፥ መመኘት፥ ክፋት ሞላባቸው፤ ቅናትን፥ ነፍስ
መግደልን፥ ክርክርን፥ ተንኰልን፥ ክፉ ጠባይን ተሞሉ፤ የሚያሾከሹኩ፥

<span id="an_1:30">1:30</span> ሐሜተኞች፥ አምላክን የሚጠሉ፥ የሚያንገላቱ፥ ትዕቢተኞች፥
ትምክህተኞች፥ ክፋትን የሚፈላለጉ፥ ለወላጆቻቸው የማይታዘዙ፥

<span id="an_1:31">1:31</span> የማያስተውሉ፥ ውል የሚያፈርሱ፥ ፍቅር የሌላቸው፥ ምሕረት ያጡ
ናቸው፤

<span id="an_1:32">1:32</span> እንደነዚህ ለሚያደርጉት ሞት ይገባቸዋል የሚለውን የእግዚአብሔርን
ሕግ እያወቁ እነዚህን ከሚያደርጉ ጋር ይስማማሉ እንጂ አድራጊዎች ብቻ አይደሉም።

<span id="an_2:1">2:1</span> ስለዚህ፥ አንተ የምትፈርድ ሰው ሁሉ ሆይ፥ የምታመካኘው የለህም፤
በሌላው በምትፈርድበት ነገር ራስህን ትኰንናለህና፤ አንተው ፈራጁ እነዚያን ታደርጋለህና።

<span id="an_2:2">2:2</span> እንደዚህም በሚያደርጉት ላይ የእግዚአብሔር ፍርድ እውነተኛ እንደ ሆነ
እናውቃለን።

<span id="an_2:3">2:3</span> አንተም እንደዚህ በሚያደርጉ የምትፈርድ ያንም የምታደርግ ሰው ሆይ፥
አንተ ከእግዚአብሔር ፍርድ የምታመልጥ ይመስልሃልን?

<span id="an_2:4">2:4</span> ወይስ የእግዚአብሔር ቸርነት ወደ ንስሐ እንዲመራህ ሳታውቅ
የቸርነቱንና የመቻሉን የትዕግሥቱንም ባለጠግነት ትንቃለህን?

<span id="an_2:5">2:5</span> ነገር ግን እንደ ጥንካሬህና ንስሐ እንደማይገባ ልብህ የእግዚአብሔር
ቅን ፍርድ በሚገለጥበት በቍጣ ቀን ቍጣን በራስህ ላይ ታከማቻለህ።

<span id="an_2:6">2:6</span> እርሱ ለእያንዳንዱ እንደ ሥራው ያስረክበዋል፤

<span id="an_2:7">2:7</span> በበጎ ሥራ በመጽናት ምስጋናንና ክብርን የማይጠፋንም ሕይወት ለሚፈልጉ
የዘላለምን ሕይወት ይሰጣቸዋል፤

<span id="an_2:8">2:8</span> ለዓመፃ በሚታዘዙ እንጂ ለእውነት በማይታዘዙትና በአድመኞች ላይ ግን
ቍጣና መቅሠፍት ይሆንባቸዋል።

<span id="an_2:9">2:9</span> ክፉውን በሚያደርግ ሰው ነፍስ ሁሉ መከራና ጭንቀት ይሆንበታል፥
አስቀድሞ በአይሁዳዊ ደግሞም በግሪክ ሰው፤

<span id="an_2:10">2:10</span> ነገር ግን በጎ ሥራ ለሚያደርጉ ሁሉ ምስጋናና ክብር ሰላምም
ይሆንላቸዋል፥ አስቀድሞ ለአይሁዳዊ ደግሞም ለግሪክ ሰው።

<span id="an_2:11">2:11</span> እግዚአብሔር ለሰው ፊት አያዳላምና።

<span id="an_2:12">2:12</span> ያለ ሕግ ኃጢአት ያደረጉ ሁሉ ያለ ሕግ ደግሞ ይጠፋሉና፤ ሕግም
ሳላቸው ኃጢአት ያደረጉ ሁሉ በሕግ ይፈረድባቸዋል፤

<span id="an_2:13">2:13</span> በእግዚአብሔር ፊት ሕግን የሚያደርጉት ይጸድቃሉ እንጂ ሕግን
የሚሰሙ ጻድቃን አይደሉምና።

<span id="an_2:14">2:14</span> ሕግ የሌላቸው አሕዛብ ከባሕርያቸው የሕግን ትእዛዝ ሲያደርጉ፥
እነዚያ ሕግ ባይኖራቸው እንኳ ለራሳቸው ሕግ ናቸውና፤

<span id="an_2:15">2:15</span> እነርሱም ሕሊናቸው ሲመሰክርላቸው፥ አሳባቸውም እርስ በርሳቸው
ሲካሰስ ወይም ሲያመካኝ በልባቸው የተጻፈውን የሕግ ሥራ ያሳያሉ።

<span id="an_2:16">2:16</span> ይህም እግዚአብሔር በኢየሱስ ክርስቶስ እኔ በወንጌል እንዳስተማርሁ
በሰው ዘንድ የተሰወረውን በሚፈርድበት ቀን ይሆናል።

<span id="an_2:17">2:17</span> አንተ ግን አይሁዳዊ ብትባል በሕግም ብትደገፍ በእግዚአብሔርም
ብትመካ፥

<span id="an_2:18">2:18</span> ፈቃዱንም ብታውቅ ከሕግም ተምረህ የሚሻለውን ፈትነህ ብትወድ፤

<span id="an_2:19">2:19</span> በሕግም የእውቀትና የእውነት መልክ ስላለህ፥ የዕውሮች መሪ፥
በጨለማም ላሉ ብርሃን፥ የሰነፎችም አስተማሪ፥ የሕፃናትም መምህር እንደ ሆንህ በራስህ ብትታመን፤

<span id="an_2:21">2:21</span> እንግዲህ አንተ ሌላውን የምታስተምር ራስህን አታስተምርምን?
አትስረቅ ብለህ የምትሰብክ ትሰርቃለህን?

<span id="an_2:22">2:22</span> አታመንዝር የምትል ታመነዝራለህን? ጣዖትን የምትጸየፍ ቤተ
መቅደስን ትዘርፋለህን?

<span id="an_2:23">2:23</span> በሕግ የምትመካ ሕግን በመተላለፍ እግዚአብሔርን ታሳፍራለህን?

<span id="an_2:24">2:24</span> በእናንተ ሰበብ የእግዚአብሔር ስም በአሕዛብ መካከል ይሰደባልና
ተብሎ እንደ ተጻፈ።

<span id="an_2:25">2:25</span> ሕግን ብታደርግ መገረዝስ ይጠቅማል፤ ሕግን ተላላፊ ብትሆን ግን
መገረዝህ አለ መገረዝ ሆኖአል።

<span id="an_2:26">2:26</span> እንግዲህ ያልተገረዘ ሰው የሕግን ሥርዓት ቢጠብቅ አለመገረዙ እንደ
መገረዝ ሆኖ አይቈጠርለትምን?

<span id="an_2:27">2:27</span> ከፍጥረቱም ያልተገረዘ ሕግን የሚፈጽም ሰው የሕግ መጽሐፍና መገረዝ
ሳለህ ሕግን በምትተላለፈው በአንተ ይፈርድብሃል።

<span id="an_2:28">2:28</span> በግልጥ አይሁዳዊ የሆነ አይሁዳዊ አይደለምና፥ በግልጥ በሥጋ
የሚደረግ መገረዝም መገረዝ አይደለምና፤

<span id="an_2:29">2:29</span> ዳሩ ግን በስውር አይሁዳዊ የሆነ አይሁዳዊ ነው፥ መገረዝም
በመንፈስ የሚደረግ የልብ መገረዝ ነው እንጂ በመጽሐፍ አይደለም፤ የእርሱ ምስጋና ከእግዚአብሔር ነው እንጂ ከሰው
አይደለም።

<span id="an_3:1">3:1</span> እንግዲህ የአይሁዳዊ ብልጫው ምንድር ነው? ወይስ የመገረዝ ጥቅሙ
ምንድር ነው? በሁሉ ነገር ብዙ ነው።

<span id="an_3:2">3:2</span> አስቀድሞ የእግዚአብሔር ቃላት አደራ ተሰጡአቸው። ታዲያ ምንድር ነው?

<span id="an_3:3">3:3</span> የማያምኑ ቢኖሩ አለማመናቸው የእግዚአብሔርን ታማኝነት ያስቀራልን?

<span id="an_3:4">3:4</span> እንዲህ አይሁን፤ በቃልህ ትጸድቅ ዘንድ ወደ ፍርድ በገባህም ጊዜ
ትረታ ዘንድ ተብሎ እንደ ተጻፈ፥ ሰው ሁሉ ውሸተኛ ከሆነ እግዚአብሔር እውነተኛ ይሁን።

<span id="an_3:5">3:5</span> ነገር ግን ዓመፃችን የእግዚአብሔርን ጽድቅ የሚያስረዳ ከሆነ ምን
እንላለን? ቍጣን የሚያመጣ እግዚአብሔር ዓመፀኛ ነውን? እንደ ሰው ልማድ እላለሁ።

<span id="an_3:6">3:6</span> እንዲህ አይሁን፤ እንዲህ ቢሆን እግዚአብሔር በዓለም እንዴት
ይፈርዳል?

<span id="an_3:7">3:7</span> በእኔ ውሸት ግን የእግዚአብሔር እውነት ለክብሩ ከላቀ ስለምን በእኔ
ደግሞ እንደ ኃጢአተኛ ገና ይፈርድብኛል?

<span id="an_3:8">3:8</span> ስለ ምንስ መልካም እንዲመጣ ክፉ አናደርግም? እንዲሁ ይሰድቡናልና
አንዳንዱም እንዲሁ እንድንል ይናገራሉና። የእነርሱም ፍርድ ቅን ነው።

<span id="an_3:9">3:9</span> እንግዲህ ምን ይሁን? ከእነርሱ እንበልጣለንን? ከቶ አይደለም፤
አይሁድም የግሪክ ሰዎችም ሁሉ ከኃጢአት በታች እንደ ሆኑ አስቀድመን ከሰናቸዋልና፤

<span id="an_3:10">3:10</span> እንዲህ ተብሎ እንደ ተጻፈ።

<span id="an_3:11">3:11</span> ጻድቅ የለም አንድ ስንኳ፤ አስተዋይም የለም፤ እግዚአብሔርንም
የሚፈልግ የለም፤ ሁሉ ተሳስተዋል፥

<span id="an_3:12">3:12</span> በአንድነትም የማይጠቅሙ ሆነዋል፤ ቸርነት የሚያደርግ የለም፥ አንድ
ስንኳ የለም።

<span id="an_3:13">3:13</span> ጕሮሮአቸው እንደ ተከፈተ መቃብር ነው፥ በመላሳቸውም ሸንግለዋል፤
የእባብ መርዝ ከከንፈሮቻቸው በታች አለ፤

<span id="an_3:14">3:14</span> አፋቸውም እርግማንና መራርነት ሞልቶበታል፤

<span id="an_3:15">3:15</span> እግሮቻቸው ደምን ለማፍሰስ ፈጣኖች ናቸው፤

<span id="an_3:16">3:16</span> ጥፋትና ጉስቁልና በመንገዳቸው ይገኛል፥

<span id="an_3:17">3:17</span> የሰላምንም መንገድ አያውቁም። በዓይኖቻቸው ፊት እግዚአብሔርን
መፍራት የለም።

<span id="an_3:19">3:19</span> አፍም ሁሉ ይዘጋ ዘንድ ዓለምም ሁሉ ከእግዚአብሔር ፍርድ በታች
ይሆን ዘንድ ሕግ የሚናገረው ሁሉ ከሕግ በታች ላሉት እንዲናገር እናውቃለን፤

<span id="an_3:20">3:20</span> ይህም የሕግን ሥራ በመሥራት ሥጋ የለበሰ ሁሉ በእርሱ ፊት
ስለማይጸድቅ ነው፤ ኃጢአት በሕግ ይታወቃልና።

<span id="an_3:21">3:21</span> አሁን ግን በሕግና በነቢያት የተመሰከረለት የእግዚአብሔር ጽድቅ
ያለ ሕግ ተገልጦአል፥

<span id="an_3:22">3:22</span> እርሱም፥ ለሚያምኑ ሁሉ የሆነ፥ በኢየሱስ ክርስቶስ በማመን
የሚገኘው የእግዚአብሔር ጽድቅ ነው፤ ልዩነት የለምና፤

<span id="an_3:23">3:23</span> ሁሉ ኃጢአትን ሠርተዋልና የእግዚአብሔርም ክብር ጎድሎአቸዋል፤

<span id="an_3:24">3:24</span> በኢየሱስ ክርስቶስም በሆነው ቤዛነት በኩል እንዲያው በጸጋው
ይጸድቃሉ።

<span id="an_3:25">3:25</span> እርሱንም እግዚአብሔር በእምነት የሚገኝ በደሙም የሆነ ማስተስሪያ
አድርጎ አቆመው፤ ይህም በፊት የተደረገውን ኃጢአት በእግዚአብሔር ችሎታ ስለ መተው ጽድቁን ያሳይ ዘንድ ነው፥

<span id="an_3:26">3:26</span> ራሱም ጻድቅ እንዲሆን በኢየሱስም የሚያምነውን እንዲያጸድቅ አሁን
በዚህ ዘመን ጽድቁን ያሳይ ዘንድ ነው።

<span id="an_3:27">3:27</span> ትምክህት እንግዲህ ወዴት ነው? እርሱ ቀርቶአል። በየትኛው ሕግ
ነው? በሥራ ሕግ ነውን? አይደለም፥ በእምነት ሕግ ነው እንጂ።

<span id="an_3:28">3:28</span> ሰው ያለ ሕግ ሥራ በእምነት እንዲጸድቅ እንቆጥራለንና።

<span id="an_3:29">3:29</span> ወይስ እግዚአብሔር የአይሁድ ብቻ አምላክ ነውን? የአሕዛብስ ደግሞ
አምላክ አይደለምን? አዎን፥ የተገረዘን ስለ እምነት ያልተገረዘንም በእምነት የሚያጸድቅ አምላክ አንድ ስለ ሆነ
የአሕዛብ ደግሞ አምላክ ነው።

<span id="an_3:31">3:31</span> እንግዲህ ሕግን በእምነት እንሽራለንን? አይደለም፤ ሕግን
እናጸናለን እንጂ።

<span id="an_4:1">4:1</span> እንግዲህ በሥጋ አባታችን የሆነ አብርሃም ምን አገኘ እንላለን?

<span id="an_4:2">4:2</span> አብርሃም በሥራ ጸድቆ ቢሆን የሚመካበት አለውና፥ ነገር ግን
በእግዚአብሔር ዘንድ አይደለም።

<span id="an_4:3">4:3</span> መጽሐፍስ ምን አለ? አብርሃምም እግዚአብሔርን አመነ ጽድቅም ሆኖ
ተቈጠረለት።

<span id="an_4:4">4:4</span> ለሚሠራ ደመወዝ እንደ ዕዳ ነው እንጂ እንደ ጸጋ አይቈጠርለትም፤

<span id="an_4:5">4:5</span> ነገር ግን ለማይሠራ፥ ኃጢአተኛውንም በሚያደድቅ ለሚያምን ሰው እምነቱ
ጽድቅ ሆኖ ይቆጠርለታል።

<span id="an_4:6">4:6</span> እንደዚህ ዳዊት ደግሞ እግዚአብሔር ያለ ሥራ ጽድቅን ስለሚቆጥርለት
ስለ ሰው ብፅዕና ይናገራል እንዲህ ሲል።

<span id="an_4:7">4:7</span> ዓመፃቸው የተሰረየላቸው ኃጢአታቸውም የተከደነላቸው ብፁዓን ናቸው፤

<span id="an_4:8">4:8</span> ጌታ ኃጢአቱን የማይቆጥርበት ሰው ብፁዕ ነው።

<span id="an_4:9">4:9</span> እንግዲህ ይህ ብፅዕና ስለ መገረዝ ተነገረ? ወይስ ደግሞ ስለ
አለመገረዝ? እምነቱ ለአብርሃም ጽድቅ ሆኖ ተቆጠረለት እንላለንና።

<span id="an_4:10">4:10</span> እንዴት ተቆጠረለት? ተገርዞ ሳለ ነውን? ወይስ ሳይገረዝ?
ተገርዞስ አይደለም፥ ሳይገረዝ ነበር እንጂ።

<span id="an_4:11">4:11</span> ሳይገረዝም በነበረው እምነት ያገኘው የጽድቅ ማኅተም የሆነ
የመገረዝን ምልክት ተቀበለ፤ ይህም እነርሱ ደግሞ ጻድቃን ሆነው ይቆጠሩ ዘንድ ሳይገረዙ ለሚያምኑ ሁሉ አባት
እንዲሆን ነው፥

<span id="an_4:12">4:12</span> ለተገረዙትም አባት እንዲሆን ነው፤ ይኸውም ለተገረዙት ብቻ
አይደለም ነገር ግን አባታችን አብርሃም ሳይገረዝ የነበረውን የእምነቱን ፍለጋ ደግሞ ለሚከተሉ ነው።

<span id="an_4:13">4:13</span> የዓለምም ወራሽ እንዲሆን ለአብርሃምና ለዘሩ የተሰጠው የተስፋ ቃል
በእምነት ጽድቅ ነው እንጂ በሕግ አይደለም።

<span id="an_4:14">4:14</span> ከሕግ የሆኑትስ ወራሾች ከሆኑ እምነት ከንቱ ሆኖአል የተስፋውም
ቃል ተሽሮአል፤

<span id="an_4:15">4:15</span> ሕጉ መቅሠፍትን ያደርጋልና፤ ነገር ግን ሕግ በሌለበት መተላለፍ
የለም።

<span id="an_4:16">4:16</span> ስለዚህ ከሕግ ብቻ ሳይሆን ከአብርሃም እምነት ደግሞ ለሆነ ለዘሩ
ሁሉ የተስፋው ቃል እንዲጸና እንደ ጸጋ ይሆን ዘንድ በእምነት ነው፤ እርሱም። ለብዙ አሕዛብ አባት አደረግሁህ ተብሎ
እንደ ተጻፈ፥ ለሙታን ሕይወት በሚሰጥ የሌለውንም እንዳለ አድርጎ በሚጠራ ባመነበት በአምላክ ፊት የሁላችን አባት
ነው።

<span id="an_4:18">4:18</span> ዘርህ እንዲሁ ሊሆን ነው እንደ ተባለ፥ ተስፋ ባልሆነው ጊዜ የብዙ
አሕዛብ አባት እንዲሆን ተስፋ ይዞ አመነ።

<span id="an_4:19">4:19</span> የመቶ ዓመትም ሽማግሌ ስለ ሆነ እንደ ምውት የሆነውን የራሱን
ሥጋና የሳራ ማኅፀን ምውት መሆኑን በእምነቱ ሳይደክም ተመለከተ፤

<span id="an_4:20">4:20</span> ለእግዚአብሔርም ክብር እየሰጠ፥ የሰጠውንም ተስፋ ደግሞ ሊፈጽም
እንዲችል አጥብቆ እየተረዳ፥ በእምነት በረታ እንጂ በአለማመን ምክንያት በእግዚአብሔር ተስፋ ቃል አልተጠራጠረም።

<span id="an_4:22">4:22</span> ስለዚህ ደግሞ ጽድቅ ሆኖ ተቆጠረለት።

<span id="an_4:23">4:23</span> ነገር ግን። ተቈጠረለት የሚለው ቃል ስለ እርሱ ብቻ የተጻፈ
አይደለም፥ ስለ እኛም ነው እንጂ፤

<span id="an_4:24">4:24</span> ስለ በደላችን አልፎ የተሰጠውን እኛን ስለ ማጽደቅም የተነሣውን
ጌታንችንን ኢየሱስን ከሙታን ባስነሣው ለምናምን ለእኛ ይቈጠርልን ዘንድ አለው።

<span id="an_5:1">5:1</span> እንግዲህ በእምነት ከጸደቅን በእግዚአብሔር ዘንድ በጌታችን በኢየሱስ
ክርስቶስ ሰላምን እንያዝ፤

<span id="an_5:2">5:2</span> በእርሱም ደግሞ ወደ ቆምንበት ወደዚህ ጸጋ በእምነት መግባትን
አግኝተናል፤ በእግዚአብሔር ክብርም ተስፋ እንመካለን።

<span id="an_5:3">5:3</span> ይህም ብቻ አይደለም፥ ነገር ግን መከራ ትዕግሥትን እንዲያደርግ፥
ትዕግሥትም ፈተናን ፈተናም ተስፋን እንዲያደርግ እያወቅን፥ በመከራችን ደግሞ እንመካለን፤

<span id="an_5:5">5:5</span> በተሰጠንም በመንፈስ ቅዱስ የእግዚአብሔር ፍቅር በልባችን ስለ ፈሰሰ
ተስፋ አያሳፍርም።

<span id="an_5:6">5:6</span> ገና ደካሞች ሳለን ክርስቶስ ዘመኑ ሲደርስ ስለ ኃጢአተኞች ሞቶአልና።

<span id="an_5:7">5:7</span> ስለ ጻድቅ የሚሞት በጭንቅ ይገኛልና፤ ስለ ቸር ሰው ግን ሊሞት እንኳ
የሚደፍር ምናልባት ይገኝ ይሆናል።

<span id="an_5:8">5:8</span> ነገር ግን ገና ኃጢአተኞች ሳለን ክርስቶስ ስለ እኛ ሞቶአልና
እግዚአብሔር ለእኛ ያለውን የራሱን ፍቅር ያስረዳል።

<span id="an_5:9">5:9</span> ይልቁንስ እንግዲህ አሁን በደሙ ከጸደቅን በእርሱ ከቍጣው እንድናለን።

<span id="an_5:10">5:10</span> ጠላቶች ሳለን ከእግዚአብሔር ጋር በልጁ ሞት ከታረቅን፥ ይልቁንም
ከታረቅን በኋላ በሕይወቱ እንድናለን፤

<span id="an_5:11">5:11</span> ይህም ብቻ አይደለም፥ ነገር ግን አሁን መታረቁን ባገኘንበት
በጌታችን በኢየሱስ ክርስቶስ በኩል በእግዚአብሔር ደግሞ እንመካለን።

<span id="an_5:12">5:12</span> ስለዚህ ምክንያት ኃጢአት በአንድ ሰው ወደ ዓለም ገባ በኃጢአትም
ሞት፥ እንደዚሁም ሁሉ ኃጢአትን ስላደረጉ ሞት ለሰው ሁሉ ደረሰ፤

<span id="an_5:13">5:13</span> ሕግ እስከ መጣ ድረስ ኃጢአት በዓለም ነበረና ነገር ግን ሕግ
በሌለበት ጊዜ ኃጢአት አይቈጠርም፤

<span id="an_5:14">5:14</span> ነገር ግን በአዳም መተላለፍ ምሳሌ ኃጢአትን ባልሠሩት ላይ እንኳ፥
ከአዳም ጀምሮ እስከ ሙሴ ድረስ ሞት ነገሠ፤ አዳም ይመጣ ዘንድ ላለው ለእርሱ አምሳሉ ነውና።

<span id="an_5:15">5:15</span> ነገር ግን ስጦታው እንደ በደሉ መጠን እንደዚያው አይደለም፤
በአንድ ሰው በደል ብዙዎቹ ሞተዋልና፥ ነገር ግን የእግዚአብሔር ጸጋና በአንድ ሰው በኢየሱስ ክርስቶስ ጸጋ የሆነው
ስጦታ ከዚያ ይልቅ ለብዙዎች በዛ።

<span id="an_5:16">5:16</span> አንድ ሰውም ኃጢአትን በማድረጉ እንደ ሆነው መጠን እንደዚያው
ስጦታው አይደለም፤ ፍርድ ከአንድ ሰው ለኵነኔ መጥቶአልና፥ ስጦታው ግን በብዙ በደል ለማጽደቅ መጣ።

<span id="an_5:17">5:17</span> በአንዱም በደል ሞት በአንዱ በኩል ከነገሠ፥ ይልቁን የጸጋን
ብዛትና የጽድቅን ስጦታ ብዛት የሚቀበሉ በአንዱ በኢየሱስ ክርስቶስ በኩል በሕይወት ይነግሣሉ።

<span id="an_5:18">5:18</span> እንግዲህ በአንድ በደል ምክንያት ፍርድ ለኵነኔ ወደ ሰው ሁሉ
እንደ መጣ፥ እንዲሁም በአንድ ጽድቅ ምክንያት ስጦታው ሕይወትን ለማጽደቅ ወደ ሰው ሁሉ መጣ።

<span id="an_5:19">5:19</span> በአንዱ ሰው አለመታዘዝ ብዙዎች ኃጢአተኞች እንደ ሆኑ፥ እንዲሁ
ደግሞ በአንዱ መታዘዝ ብዙዎች ጻድቃን ይሆናሉ።

<span id="an_5:20">5:20</span> በደልም እንዲበዛ ሕግ ጭምር ገባ፤ ዳሩ ግን ኃጢአት በበዛበት፥
ኃጢአት በሞት እንደ ነገሠ፥ እንዲሁ ደግሞ ጸጋ ከጌታችን ከኢየሱስ ክርስቶስ የተነሣ በጽድቅ ምክንያት ለዘላለም
ሕይወት ይነግሥ ዘንድ፥ ጸጋ ከመጠን ይልቅ በለጠ።

<span id="an_6:1">6:1</span> እንግዲህ ምን እንላለን? ጸጋ እንዲበዛ በኃጢአት ጸንተን እንኑርን?
አይደለም።

<span id="an_6:2">6:2</span> ለኃጢአት የሞትን እኛ ወደ ፊት እንዴት አድርገን በእርሱ እንኖራለን?

<span id="an_6:3">6:3</span> ወይስ ከክርስቶስ ኢየሱስ ጋር አንድ እንሆን ዘንድ የተጠመቅን ሁላችን
ከሞቱ ጋር አንድ እንሆን ዘንድ እንደ ተጠመቅን አታውቁምን?

<span id="an_6:4">6:4</span> እንግዲህ ክርስቶስ በአብ ክብር ከሙታን እንደ ተነሣ እንዲሁ እኛም
በአዲስ ሕይወት እንድንመላለስ፥ ከሞቱ ጋር አንድ እንሆን ዘንድ በጥምቀት ከእርሱ ጋር ተቀበርን።

<span id="an_6:5">6:5</span> ሞቱንም በሚመስል ሞት ከእርሱ ጋር ከተባበርን ትንሣኤውን በሚመስል
ትንሣኤ ደግሞ ከእርሱ ጋር እንተባበራለን፤

<span id="an_6:6">6:6</span> ከእንግዲህስ ወዲያ ለኃጢአት እንዳንገዛ የኃጢአት ሥጋ ይሻር ዘንድ
አሮጌው ሰዋችን ከእርሱ ጋር እንደ ተሰቀለ እናውቃለን፤ የሞተስ ከኃጢአቱ ጸድቋልና።

<span id="an_6:7">6:7</span> ነገር ግን ከክርስቶስ ጋር ከሞትን ከእርሱ ጋር ደግሞ በሕይወት
እንድንኖር እናምናለን፤

<span id="an_6:9">6:9</span> ክርስቶስ ከሙታን ተነሥቶ ወደ ፊት እንዳይሞት ሞትም ወደ ፊት
እንዳይገዛው እናውቃለንና።

<span id="an_6:10">6:10</span> መሞትን አንድ ጊዜ ፈጽሞ ለኃጢአት ሞቶአልና፤ በሕይወት መኖርን
ግን ለእግዚአብሔር ይኖራል።

<span id="an_6:11">6:11</span> እንዲሁም እናንተ ደግሞ ለኃጢአት እንደ ሞታችሁ፥ ግን በክርስቶስ
ኢየሱስ በጌታችን ሆናችሁ ለእግዚአብሔር ሕያዋን እንደ ሆናችሁ ራሳችሁን ቈጠሩ።

<span id="an_6:12">6:12</span> እንግዲህ ለምኞቱ እንድትታዘዙ በሚሞት ሥጋችሁ ኃጢአት አይንገሥ፤

<span id="an_6:13">6:13</span> ብልቶቻችሁንም የዓመፃ የጦር ዕቃ አድርጋችሁ ለኃጢአት አታቅርቡ፥
ነገር ግን ከሙታን ተለይታችሁ በሕይወት እንደምትኖሩ ራሳችሁን ለእግዚአብሔር አቅርቡ፥ ብልቶቻችሁንም የጽድቅ የጦር
ዕቃ አድርጋችሁ ለእግዚአብሔር አቅርቡ።

<span id="an_6:14">6:14</span> ኃጢአት አይገዛችሁምና፤ ከጸጋ በታች እንጂ ከሕግ በታች
አይደላችሁምና።

<span id="an_6:15">6:15</span> እንግዲህ ምን ይሁን? ከጸጋ በታች እንጂ ከሕግ በታች ስላይደለን
ኃጢአትን እንሥራን? አይደለም።

<span id="an_6:16">6:16</span> ለመታዘዝ ባሪያዎች እንድትሆኑ ራሳችሁን ለምታቀርቡለት፥ ለእርሱ
ለምትታዘዙለት ባሪያዎች እንደ ሆናችሁ አታውቁምን? ወይም ለሞት የኃጢአት ባሪያዎች ወይም ለጽድቅ የመታዘዝ ባሪያዎች
ናችሁ።

<span id="an_6:17">6:17</span> ነገር ግን አስቀድማችሁ የኃጢአት ባሪያዎች ከሆናችሁ፥
ለተሰጣችሁለት ለትምህርት ዓይነት ከልባችሁ ስለ ታዘዛችሁ፥ ከኃጢአትም አርነት ወጥታችሁ ለጽድቅ ስለ ተገዛችሁ
ለእግዚአብሔር ምስጋና ይሁን።

<span id="an_6:19">6:19</span> ስለ ሥጋችሁ ድካም እንደ ሰው ልማድ እላለሁ። ብልቶቻችሁ ዓመፃ
ሊያደርጉ ለርኵስነትና ለዓመፃ ባሪያዎች አድርጋችሁ እንዳቀረባችሁ፥ እንደዚሁ ብልቶቻችሁ ሊቀደሱ ለጽድቅ ባሪያዎች
አድርጋችሁ አሁን አቅርቡ።

<span id="an_6:20">6:20</span> የኃጢአት ባሪያዎች ሳላችሁ ከጽድቅ ነፃ ነበራችሁና።

<span id="an_6:21">6:21</span> እንግዲህ ዛሬ ከምታፍሩበት ነገር ያን ጊዜ ምን ፍሬ ነበራችሁ?
የዚህ ነገር መጨረሻው ሞት ነውና።

<span id="an_6:22">6:22</span> አሁን ግን ከኃጢአት አርነት ወጥታችሁ ለእግዚአብሔርም ተገዝታችሁ፥
ልትቀደሱ ፍሬ አላችሁ፤ መጨረሻውም የዘላለም ሕይወት ነው።

<span id="an_6:23">6:23</span> የኃጢአት ደመወዝ ሞት ነውና፤ የእግዚአብሔር የጸጋ ስጦታ ግን
በክርስቶስ ኢየሱስ በጌታችን የዘላለም ሕይወት ነው።

<span id="an_7:1">7:1</span> ወንድሞች ሆይ፥ ሕግን ለሚያውቁ እናገራለሁና ሰው ባለበት ዘመን ሁሉ
ሕግ እንዲገዛው አታውቁምን?

<span id="an_7:2">7:2</span> ያገባች ሴት ባልዋ በሕይወት ሲኖር ከእርሱ ጋር በሕግ ታስራለችና፤
ባልዋ ቢሞት ግን ስለ ባል ከሆነው ሕግ ተፈትታለች።

<span id="an_7:3">7:3</span> ስለዚህ ባልዋ በሕይወት ሳለ ለሌላ ወንድ ብትሆን አመንዝራ ትባላለች፤
ባልዋ ቢሞት ግን ከሕጉ አርነት ወጥታለችና ለሌላ ወንድ ብትሆን አመንዝራ አይደለችም።

<span id="an_7:4">7:4</span> እንዲሁ፥ ወንድሞቼ ሆይ፥ እናንተ ደግሞ በክርስቶስ ሥጋ ለሕግ
ተገድላችኋል፤ ለእግዚአብሔር ፍሬ እንድናፈራ፥ እናንተ ለሌላው፥ ከሙታን ለተነሣው፥ ለእርሱ ትሆኑ ዘንድ።

<span id="an_7:5">7:5</span> በሥጋ ሳለን በሕግ የሚሆን የኃጢአት መሻት ለሞት ፍሬ ሊያፈራ
በብልቶቻችን ይሠራ ነበርና፤

<span id="an_7:6">7:6</span> አሁን ግን ለእርሱ ለታሰርንበት ስለ ሞትን፥ ከሕግ ተፈትተናል፥
ስለዚህም በአዲሱ በመንፈስ ኑሮ እንገዛለን እንጂ በአሮጌው በፊደል ኑሮ አይደለም።

<span id="an_7:7">7:7</span> እንግዲህ ምን እንላለን? ሕግ ኃጢአት ነውን? አይደለም፤ ነገር ግን
በሕግ ባይሆን ኃጢአትን ባላወቅሁም ነበር፤ ሕጉ። አትመኝ ባላለ ምኞትን ባላወቅሁም ነበርና።

<span id="an_7:8">7:8</span> ኃጢአት ግን ምክንያት አግኝቶ ምኞትን ሁሉ በትእዛዝ ሠራብኝ፤ ኃጢአት
ያለ ሕግ ምውት ነውና።

<span id="an_7:9">7:9</span> እኔም ዱሮ ያለ ሕግ ሕያው ነበርሁ፤ ትእዛዝ በመጣች ጊዜ ግን ኃጢአት
ሕያው ሆነ እኔም ሞትሁ፤

<span id="an_7:10">7:10</span> ለሕይወትም የተሰጠችውን ትእዛዝ እርስዋን ለሞት ሆና አገኘኋት፤

<span id="an_7:11">7:11</span> ኃጢአት ምክንያት አግኝቶ በትእዛዝ አታሎኛልና በእርስዋም
ገድሎኛል።

<span id="an_7:12">7:12</span> ስለዚህ ሕጉ ቅዱስ ነው ትእዛዚቱም ቅድስትና ጻድቅት በጎም ናት።

<span id="an_7:13">7:13</span> እንግዲህ በጎ የሆነው ነገር ለእኔ ሞት ሆነብኝን? አይደለም፥
ነገር ግን ኃጢአት ሆነ፤ ኃጢአትም በትእዛዝ ምክንያት ያለ ልክ ኃጢአተኛ ይሆን ዘንድ፥ ኃጢአትም እንዲሆን ይገለጥ
ዘንድ በጎ በሆነው ነገር ለእኔ ሞትን ይሠራ ነበር።

<span id="an_7:14">7:14</span> ሕግ መንፈሳዊ እንደ ሆነ እናውቃለንና፤ እኔ ግን ከኃጢአት በታች
ልሆን የተሸጥሁ የሥጋ ነኝ።

<span id="an_7:15">7:15</span> የማደርገውን አላውቅምና፤ የምጠላውን ያን አደርጋለሁና ዳሩ ግን
የምወደውን እርሱን አላደርገውም።

<span id="an_7:16">7:16</span> የማልወደውን ግን የማደርግ ከሆንሁ ሕግ መልካም እንደ ሆነ
እመሰክራለሁ።

<span id="an_7:17">7:17</span> እንደዚህ ከሆነ ያን የማደርገው አሁን እኔ አይደለሁም፥ በእኔ
የሚያድር ኃጢአት ነው እንጂ።

<span id="an_7:18">7:18</span> በእኔ ማለት በሥጋዬ በጎ ነገር እንዳይኖር አውቃለሁና፤ ፈቃድ
አለኝና፥ መልካሙን ግን ማድረግ የለኝም።

<span id="an_7:19">7:19</span> የማልወደውን ክፉን ነገር አደርጋለሁና ዳሩ ግን የምወደውን በጎውን
ነገር አላደርገውም።

<span id="an_7:20">7:20</span> የማልወደውን የማደርግ ከሆንሁ ግን ያን የማደርገው አሁን እኔ
አይደለሁም፥ በእኔ የሚኖር ኃጢአት ነው እንጂ።

<span id="an_7:21">7:21</span> እንግዲያስ መልካሙን አደርግ ዘንድ ስወድ በእኔ ክፉ እንዲያድርብኝ
ሕግን አገኛለሁ።

<span id="an_7:22">7:22</span> በውስጡ ሰውነቴ በእግዚአብሔር ሕግ ደስ ይለኛልና፥

<span id="an_7:23">7:23</span> ነገር ግን በብልቶቼ ከአእምሮዬ ሕግ ጋር የሚዋጋውንና በብልቶቼ
ባለ በኃጢአት ሕግ የሚማርከኝን ሌላ ሕግ አያለሁ።

<span id="an_7:24">7:24</span> እኔ ምንኛ ጎስቋላ ሰው ነኝ! ለዚህ ሞት ከተሰጠ ሰውነት ማን
ያድነኛል?

<span id="an_7:25">7:25</span> በኢየሱስ ክርስቶስ በጌታችን ለእግዚአብሔር ምስጋና ይሁን።
እንግዲያስ እኔ በአእምሮዬ ለእግዚአብሔር ሕግ፥ በሥጋዬ ግን ለኃጢአት ሕግ እገዛለሁ።

<span id="an_8:1">8:1</span> እንግዲህ በክርስቶስ ኢየሱስ ላሉት አሁን ኵነኔ የለባቸውም።

<span id="an_8:2">8:2</span> በክርስቶስ ኢየሱስ ያለው የሕይወት መንፈስ ሕግ ከኃጢአትና ከሞት ሕግ
አርነት አውጥቶኛልና።

<span id="an_8:3">8:3</span> ከሥጋ የተነሣ ስለ ደከመ ለሕግ ያልተቻለውን፥ እግዚአብሔር የገዛ
ልጁን በኃጢአተኛ ሥጋ ምሳሌ በኃጢአትም ምክንያት ልኮ አድርጎአልና፤ እንደ መንፈስ ፈቃድ እንጂ እንደ ሥጋ ፈቃድ
በባንመላለስ በእኛ የሕግ ትእዛዝ ይፈጸም ዘንድ ኃጢአትን በሥጋ ኰነነ።

<span id="an_8:5">8:5</span> እንደ ሥጋ ፈቃድ የሚኖሩ የሥጋን ነገር ያስባሉና፥ እንደ መንፈስ
ፈቃድ የሚኖሩ ግን የመንፈስን ነገር ያስባሉ።

<span id="an_8:6">8:6</span> ስለ ሥጋ ማሰብ ሞት ነውና፥ ስለ መንፈስ ማሰብ ግን ሕይወትና ሰላም
ነው።

<span id="an_8:7">8:7</span> ስለ ሥጋ ማሰብ በእግዚአብሔር ዘንድ ጥል ነውና፤ ለእግዚአብሔር ሕግ
አይገዛምና፥ መገዛትም ተስኖታል፤

<span id="an_8:8">8:8</span> በሥጋ ያሉትም እግዚአብሔርን ደስ ሊያሰኙት አይችሉም።

<span id="an_8:9">8:9</span> እናንተ ግን የእግዚአብሔር መንፈስ በእናንተ ዘንድ ቢኖር፥ በመንፈስ
እንጂ በሥጋ አይደላችሁም። የክርስቶስ መንፈስ የሌለው ከሆነ ግን ይኸው የእርሱ ወገን አይደለም።

<span id="an_8:10">8:10</span> ክርስቶስ በእናንተ ውስጥ ቢሆን ሰውነታችሁ በኃጢአት ምክንያት
የሞተ ነው፥ መንፈሳችሁ ግን በጽድቅ ምክንያት ሕያው ነው።

<span id="an_8:11">8:11</span> ነገር ግን ኢየሱስን ከሙታን ያስነሣው የእርሱ መንፈስ በእናንተ
ዘንድ ቢኖር፥ ክርስቶስ ኢየሱስን ከሙታን ያስነሣው እርሱ በእናንተ በሚኖረው በመንፈሱ፥ ለሚሞተው ሰውነታችሁ ደግሞ
ሕይወትን ይሰጠዋል።

<span id="an_8:12">8:12</span> እንግዲህ፥ ወንድሞች ሆይ፥ ዕዳ አለብን፥ እንደ ሥጋ ፈቃድ ግን
እንኖር ዘንድ ለሥጋ አይደለም።

<span id="an_8:13">8:13</span> እንደ ሥጋ ፈቃድ ብትኖሩ ትሞቱ ዘንድ አላችሁና፤ በመንፈስ ግን
የሰውነትን ሥራ ብትገድሉ በሕይወት ትኖራላችሁ።

<span id="an_8:14">8:14</span> በእግዚአብሔር መንፈስ የሚመሩ ሁሉ እነዚህ የእግዚአብሔር ልጆች
ናቸውና።

<span id="an_8:15">8:15</span> አባ አባት ብለን የምንጮኽበትን የልጅነት መንፈስ ተቀበላችሁ እንጂ
እንደገና ለፍርሃት የባርነትን መንፈስ አልተቀበላችሁምና።

<span id="an_8:16">8:16</span> የእግዚአብሔር ልጆች መሆናችንን ያ መንፈስ ራሱ ከመንፈሳችን ጋር
ይመሰክራል።

<span id="an_8:17">8:17</span> ልጆች ከሆንን ወራሾች ደግሞ ነን፤ ማለት የእግዚአብሔር ወራሾች
ነን፥ አብረንም ደግሞ እንድንከበር አብረን መከራ ብንቀበል ከክርስቶስ ጋር አብረን ወራሾች ነን።

<span id="an_8:18">8:18</span> ለእኛም ይገለጥ ዘንድ ካለው ክብር ጋር ቢመዛዘን የአሁኑ ዘመን
ሥቃይ ምንም እንዳይደለ አስባለሁ።

<span id="an_8:19">8:19</span> የፍጥረት ናፍቆት የእግዚአብሔርን ልጆች መገለጥ ይጠባበቃልና።

<span id="an_8:20">8:20</span> ፍጥረት ለከንቱነት ተገዝቶአልና፥ በተስፋ ስላስገዛው ነው እንጂ
በፈቃዱ አይደለም፤

<span id="an_8:21">8:21</span> ተስፋውም ፍጥረት ራሱ ደግሞ ከጥፋት ባርነት ነፃነት ወጥቶ
ለእግዚአብሔር ልጆች ወደሚሆን ክብር ነፃነት እንዲደርስ ነው።

<span id="an_8:22">8:22</span> ፍጥረት ሁሉ እስከ አሁን ድረስ አብሮ በመቃተትና በምጥ መኖሩን
እናውቃለንና።

<span id="an_8:23">8:23</span> እርሱም ብቻ አይደለም፥ ነገር ግን የመንፈስ በኵራት ያለን ራሳችን
ደግሞ የሰውነታችን ቤዛ የሆነውን ልጅነት እየተጠባበቅን ራሳችን በውስጣችን እንቃትታለን።

<span id="an_8:24">8:24</span> በተስፋ ድነናልና፤ ነገር ግን ተስፋ የሚደረግበቱ ነገር ቢታይ
ተስፋ አይደለም፤ የሚያየውንማ ማን ተስፋ ያደርገዋል?

<span id="an_8:25">8:25</span> የማናየውን ግን ተስፋ ብናደርገው በትዕግሥት እንጠባበቃለን።

<span id="an_8:26">8:26</span> እንዲሁም ደግሞ መንፈስ ድካማችንን ያግዛል፤ እንዴት እንድንጸልይ
እንደሚገባን አናውቅምና፥ ነገር ግን መንፈስ ራሱ በማይነገር መቃተት ይማልድልናል፤

<span id="an_8:27">8:27</span> ልብንም የሚመረምረው የመንፈስ አሳብ ምን እንደ ሆነ ያውቃል፥
እንደ እግዚአብሔር ፈቃድ ስለ ቅዱሳን ይማልዳልና።

<span id="an_8:28">8:28</span> እግዚአብሔርንም ለሚወዱት እንደ አሳቡም ለተጠሩት ነገር ሁሉ ለበጎ
እንዲደረግ እናውቃለን።

<span id="an_8:29">8:29</span> ልጁ በብዙ ወንድሞች መካከል በኵር ይሆን ዘንድ፥ አስቀድሞ
ያወቃቸው የልጁን መልክ እንዲመስሉ አስቀድሞ ደግሞ ወስኖአልና፤

<span id="an_8:30">8:30</span> አስቀድሞም የወሰናቸውን እነዚህን ደግሞ ጠራቸው፤ የጠራቸውንም
እነዚህን ደግሞ አጸደቃቸው፤ ያጸደቃቸውንም እነዚህን ደግሞ አከበራቸው።

<span id="an_8:31">8:31</span> እንግዲህ ስለዚህ ነገር ምን እንላለን? እግዚአብሔር ከእኛ ጋር
ከሆነ ማን ይቃወመናል?

<span id="an_8:32">8:32</span> ለገዛ ልጁ ያልራራለት ነገር ግን ስለ ሁላችን አሳልፎ የሰጠው ያው
ከእርሱ ጋር ደግሞ ሁሉን ነገር እንዲያው እንዴት አይሰጠንም?

<span id="an_8:33">8:33</span> እግዚአብሔር የመረጣቸውን ማን ይከሳቸዋል? የሚያጸድቅ እግዚአብሔር
ነው፥ የሚኰንንስ ማን ነው?

<span id="an_8:34">8:34</span> የሞተው፥ ይልቁንም ከሙታን የተነሣው፥ በእግዚአብሔር ቀኝ ያለው፥
ደግሞ ስለ እኛ የሚማልደው ክርስቶስ ኢየሱስ ነው።

<span id="an_8:35">8:35</span> ከክርስቶስ ፍቅር ማን ይለየናል? መከራ፥ ወይስ ጭንቀት፥ ወይስ
ስደት፥ ወይስ ራብ፥ ወይስ ራቁትነት፥ ወይስ ፍርሃት፥ ወይስ ሰይፍ ነውን?

<span id="an_8:36">8:36</span> ስለ አንተ ቀኑን ሁሉ እንገደላለን፥ እንደሚታረዱ በጎች ተቆጠርን
ተብሎ እንደ ተጻፈ ነው።

<span id="an_8:37">8:37</span> በዚህ ሁሉ ግን በወደደን በእርሱ ከአሸናፊዎች እንበልጣለን።

<span id="an_8:38">8:38</span> ሞት ቢሆን፥ ሕይወትም ቢሆን፥ መላእክትም ቢሆኑ፥ ግዛትም ቢሆን፥
ያለውም ቢሆን፥ የሚመጣውም ቢሆን፥ ኃይላትም ቢሆኑ፥

<span id="an_8:39">8:39</span> ከፍታም ቢሆን፥ ዝቅታም ቢሆን፥ ልዩ ፍጥረትም ቢሆን በክርስቶስ
ኢየሱስ በጌታችን ካለ ከእግዚአብሔር ፍቅር ሊለየን እንዳይችል ተረድቼአለሁ።

<span id="an_9:1">9:1</span> ብዙ ኀዘን የማያቋርጥም ጭንቀት በልቤ አለብኝ ስል በክርስቶስ ሆኜ
እውነትን እናገራለሁ፤ አልዋሽምም፤ ሕሊናዬም በመንፈስ ቅዱስ ይመሰክርልኛል።

<span id="an_9:3">9:3</span> በሥጋ ዘመዶቼ ስለ ሆኑ ስለ ወንድሞቼ ከክርስቶስ ተለይቼ እኔ ራሴ
የተረገምሁ እንድሆን እጸልይ ነበርና።

<span id="an_9:4">9:4</span> እነርሱ እስራኤላውያን ናቸውና፥ ልጅነትና ክብር ኪዳንም የሕግም
መሰጠት የመቅደስም ሥርዓት የተስፋውም ቃላት ለእነርሱ ናቸውና፤

<span id="an_9:5">9:5</span> አባቶችም ለእነርሱ ናቸውና፤ ከእነርሱም ክርስቶስ በሥጋ መጣ፥ እርሱም
ከሁሉ በላይ ሆኖ ለዘላለም የተባረከ አምላክ ነው፤ አሜን።

<span id="an_9:6">9:6</span> ነገር ግን የእግዚአብሔር ቃል የተሻረ አይደለም። እነዚህ ከእስራኤል
የሚወለዱ ሁሉ እስራኤል አይደሉምና፤ የአብርሃምም ዘር ስለሆኑ ሁላቸው ልጆች አይደሉም፥

<span id="an_9:7">9:7</span> ነገር ግን። በይስሐቅ ዘር ይጠራልሃል ተባለ።

<span id="an_9:8">9:8</span> ይህም፥ የተስፋ ቃል ልጆች ዘር ሆነው ይቆጠራሉ እንጂ እነዚህ የሥጋ
ልጆች የሆኑ የእግዚአብሔር ልጆች አይደሉም ማለት ነው።

<span id="an_9:9">9:9</span> ይህ። በዚህ ጊዜ እመጣለሁ ለሳራም ልጅ ይሆንላታል የሚል የተስፋ ቃል
ነውና።

<span id="an_9:10">9:10</span> ይህ ብቻ አይደለም፥ ነገር ግን ርብቃ ደግሞ ከአንዱ ከአባታችን
ከይስሐቅ በፀነሰች ጊዜ፥

<span id="an_9:11">9:11</span> ልጆቹ ገና ሳይወለዱ፥ በጎ ወይም ክፉ ምንም ሳያደርጉ፥ ከጠሪው
እንጂ ከሥራ ሳይሆን በምርጫ የሚሆን የእግዚአብሔር አሳብ ይጸና ዘንድ፥

<span id="an_9:12">9:12</span> ለእርስዋ። ታላቁ ለታናሹ ይገዛል ተባለላት።

<span id="an_9:13">9:13</span> ያዕቆብን ወደድሁ ኤሳውን ግን ጠላሁ ተብሎ እንደ ተጻፈ ነው።

<span id="an_9:14">9:14</span> እንግዲህ ምን እንላለን? በእግዚአብሔር ዘንድ ዓመፃ አለ ወይ?
አይደለም።

<span id="an_9:15">9:15</span> ለሙሴ። የምምረውን ሁሉ እምረዋለሁ ለምራራለትም ሁሉ እራራለታለሁ
ይላልና።

<span id="an_9:16">9:16</span> እንግዲህ ምሕረት ለወደደ ወይም ለሮጠ አይደለም፥ ከሚምር
ከእግዚአብሔር ነው እንጂ።

<span id="an_9:17">9:17</span> መጽሐፍ ፈርዖንን። ኃይሌን በአንተ አሳይ ዘንድ ስሜም በምድር ሁሉ
ይነገር ዘንድ ለዚህ አስነሣሁህ ይላልና።

<span id="an_9:18">9:18</span> እንግዲህ የሚወደውን ይምረዋል፥ የሚወደውንም እልከኛ ያደርገዋል።

<span id="an_9:19">9:19</span> እንግዲህ ስለ ምን እስከ አሁን ድረስ ይነቅፋል? ፈቃዱንስ
የሚቃወም ማን ነው? ትለኝ ይሆናል።

<span id="an_9:20">9:20</span> ነገር ግን፥ አንተ ሰው ሆይ፥ ለእግዚአብሔር የምትመልስ ማን ነህ?
ሥራ ሠሪውን። ስለ ምን እንዲህ አድርገህ ሠራኸኝ ይለዋልን?

<span id="an_9:21">9:21</span> ወይም ሸክላ ሠሪ ከአንዱ ጭቃ ክፍል አንዱን ዕቃ ለክብር አንዱንም
ለውርደት ሊሠራ በጭቃ ላይ ሥልጣን የለውምን?

<span id="an_9:22">9:22</span> ነገር ግን እግዚአብሔር ቍጣውን ሊያሳይ ኃይሉንም ሊገልጥ ወዶ፥
አስቀድሞ ለክብር ባዘጋጃቸው በምሕረት ዕቃዎች ላይ የክብሩን ባለ ጠግነት ይገልጥ ዘንድ፥ ለጥፋት የተዘጋጁትን የቁጣ
ዕቃዎች በብዙ ትዕግሥት ከቻለ፥ እንዴት ነው?

<span id="an_9:24">9:24</span> የምሕረቱ ዕቃዎችም ከአይሁድ ብቻ አይደሉም፥ ነገር ግን ከአሕዛብ
ደግሞ የጠራን እኛ ነን።

<span id="an_9:25">9:25</span> እንዲሁ ደግሞ በሆሴዕ። ሕዝቤ ያልሆነውን ሕዝቤ ብዬ፥
ያልተወደደችውንም የተወደደችው ብዬ እጠራለሁ፤

<span id="an_9:26">9:26</span> እናንተ ሕዝቤ አይደላችሁም በተባለላቸውም ስፍራ በዚያ የሕያው
እግዚአብሔር ልጆች ተብለው ይጠራሉ ይላል።

<span id="an_9:27">9:27</span> ኢሳይያስም። የእስራኤል ልጆች ቁጥር ምንም እንደ ባሕር አሸዋ
ቢሆን ቅሬታው ይድናል፤ ጌታ ነገሩን ፈጽሞና ቆርጦ በምድር ላይ ያደርገዋልና ብሎ ስለ እስራኤል ይጮኻል።

<span id="an_9:29">9:29</span> ኢሳይያስም እንደዚሁ። ጌታ ፀባዖት ዘር ባላስቀረልን እንደ ሰዶም
በሆንን ገሞራንም በመሰልን ነበር ብሎ አስቀድሞ ተናገረ።

<span id="an_9:30">9:30</span> እንግዲህ ምን እንላለን? ጽድቅን ያልተከተሉት አሕዛብ ጽድቅን
አገኙ፥ እርሱ ግን ከእምነት የሆነ ጽድቅ ነው፤

<span id="an_9:31">9:31</span> እስራኤል ግን የጽድቅን ሕግ እየተከተሉ ወደ ሕግ አልደረሱም።

<span id="an_9:32">9:32</span> ይህስ ስለ ምንድር ነው? በሥራ እንጂ በእምነት ጽድቅን
ስላልተከተሉ ነው፤ እነሆ፥ በጽዮን የእንቅፋት ድንጋይና የማሰናከያ ዓለት አኖራለሁ በእርሱም የሚያምን አያፍርም
ተብሎ እንደ ተጻፈ በእንቅፋት ድንጋይ ተሰናከሉ።

<span id="an_10:1">10:1</span> ወንድሞች ሆይ፥ የልቤ በጎ ፈቃድና ስለ እስራኤል ወደ እግዚአብሔር
ልመናዬ እንዲድኑ ነው።

<span id="an_10:2">10:2</span> በእውቀት አይቅኑ እንጂ ለእግዚአብሔር እንዲቀኑ
እመሰክርላቸዋለሁና።

<span id="an_10:3">10:3</span> የእግዚአብሔርን ጽድቅ ሳያውቁ የራሳቸውንም ጽድቅ ሊያቆሙ ሲፈልጉ፥
ለእግዚአብሔር ጽድቅ አልተገዙም።

<span id="an_10:4">10:4</span> የሚያምኑ ሁሉ ይጸድቁ ዘንድ ክርስቶስ የሕግ ፍጻሜ ነውና።

<span id="an_10:5">10:5</span> ሙሴ ከሕግ የሆነውን ጽድቅ የሚያደርገው ሰው በእርሱ በሕይወት
እንዲኖር ጽፎአልና።

<span id="an_10:6">10:6</span> ከእምነት የሆነ ጽድቅ ግን እንዲህ ይላል። በልብህ። ማን ወደ
ሰማይ ይወጣል? አትበል፤ ይህ ክርስቶስን ለማውረድ ነው፤

<span id="an_10:7">10:7</span> ወይም። በልብህ። ወደ ጥልቁ ማን ይወርዳል? አትበል፤ ይህ
ክርስቶስን ከሙታን ለማውጣት ነው።

<span id="an_10:8">10:8</span> ነገር ግን ምን ይላል? በአፍህ በልብህም ሆኖ ቃሉ ቀርቦልሃል፤
ይህም የምንሰብከው የእምነት ቃል ነው።

<span id="an_10:9">10:9</span> ኢየሱስ ጌታ እንደ ሆነ በአፍህ ብትመሰክር እግዚአብሔርም ከሙታን
እንዳስነሣው በልብህ ብታምን ትድናለህና፤

<span id="an_10:10">10:10</span> ሰው በልቡ አምኖ ይጸድቃልና በአፉም መስክሮ ይድናልና።

<span id="an_10:11">10:11</span> መጽሐፍ። በእርሱ የሚያምን ሁሉ አያፍርም ይላልና።

<span id="an_10:12">10:12</span> በአይሁዳዊና በግሪክ ሰው መካከል ልዩነት የለምና፤ አንዱ ጌታ
የሁሉ ጌታ ነውና፥ ለሚጠሩትም ሁሉ ባለ ጠጋ ነው፤

<span id="an_10:13">10:13</span> የጌታን ስም የሚጠራ ሁሉ ይድናልና።

<span id="an_10:14">10:14</span> እንግዲህ ያላመኑበትን እንዴት አድርገው ይጠሩታል? ባልሰሙትስ
እንዴት ያምናሉ? ያለ ሰባኪስ እንዴት ይሰማሉ?

<span id="an_10:15">10:15</span> መልካሙን የምሥራች የሚያወሩ እግሮቻቸው እንዴት ያማሩ ናቸው
ተብሎ እንደ ተጻፈ ካልተላኩ እንዴት ይሰብካሉ?

<span id="an_10:16">10:16</span> ነገር ግን ሁሉ ለምሥራቹ ቃል አልታዘዙም። ኢሳይያስ። ጌታ
ሆይ፥ ምስክርነታችንን ማን አመነ? ብሎአልና።

<span id="an_10:17">10:17</span> እንግዲያስ እምነት ከመስማት ነው መስማትም በእግዚአብሔር ቃል
ነው።

<span id="an_10:18">10:18</span> ዳሩ ግን። ባይሰሙ ነው ወይ? እላለሁ። በእውነት። ድምፃቸው
በምድር ሁሉ ላይ ቃላቸውም እስከ ዓለም ዳርቻ ወጣ።

<span id="an_10:19">10:19</span> ነገር ግን። እስራኤል ባያውቁ ነው ወይ? እላለሁ። ሙሴ
አስቀድሞ። እኔ ሕዝብ በማይሆነው አስቀናችኋለሁ በማያስተውልም ሕዝብ አስቆጣችኋለሁ

<span id="an_10:20">10:20</span> ብሎአል። ኢሳይያስም ደፍሮ። ላልፈለጉኝ ተገኘሁ፥ ላልጠየቁኝም
ተገለጥሁ አለ።

<span id="an_10:21">10:21</span> ስለ እስራኤል ግን። ቀኑን ሁሉ ወደማይታዘዝና ወደሚቃወም ሕዝብ
እጆቼን ዘረጋሁ ይላል።

<span id="an_11:1">11:1</span> እንግዲህ። እግዚአብሔር ሕዝቡን ጣላቸውን? እላለሁ። አይደለም እኔ
ደግሞ እስራኤላዊና ከአብርሃም ዘር ከብንያምም ወገን ነኝና።

<span id="an_11:2">11:2</span> እግዚአብሔር አስቀድሞ ያወቃቸውን ሕዝብ አልጣላቸውም። መጽሐፍ ስለ
ኤልያስ በተጻፈው የሚለውን፥ በእግዚአብሔር ፊት እስራኤልን እንዴት እንደሚከስ፥ አታውቁምን?

<span id="an_11:3">11:3</span> ጌታ ሆይ፥ ነቢያትህን ገደሉ መሠዊያዎችህንም አፈረሱ እኔም ብቻዬን
ቀረሁ ነፍሴንም ይሹአታል።

<span id="an_11:4">11:4</span> ነገር ግን አምላካዊ መልስ ምን አለው? ለበአል ያልሰገዱትን ሰባት
ሺህ ሰዎች ለእኔ አስቀርቼአለሁ።

<span id="an_11:5">11:5</span> እንደዚሁም በአሁን ዘመን ደግሞ በጸጋ የተመረጡ ቅሬታዎች አሉ።

<span id="an_11:6">11:6</span> በጸጋ ከሆነ ግን ከሥራ መሆኑ ቀርቶአል፤ ጸጋ ያለዚያ ጸጋ መሆኑ
ቀርቶአል።

<span id="an_11:7">11:7</span> እንግዲህ ምንድር ነው? እስራኤል የሚፈልጉትን አላገኙትም፤
የተመረጡት ግን አገኙት፤

<span id="an_11:8">11:8</span> ሌሎቹም ደነዘዙ፤ እንዲሁም። ዓይኖቻቸው እንዳያዩ ጆሮቻቸውም
እንዳይሰሙ እግዚአብሔር የእንቅልፍ መንፈስን እስከ ዛሬ ድረስ ሰጣቸው ተብሎ ተጽፎአል። ዳዊትም።

<span id="an_11:9">11:9</span> ማዕዳቸው ወጥመድና አሽክላ ማሰናከያም ፍዳም ይሁንባቸው፤

<span id="an_11:10">11:10</span> ዓይኖቻቸው እንዳያዩ ይጨልሙ፥ ጀርባቸውንም ዘወትር አጉብጥ
ብሎአል።

<span id="an_11:11">11:11</span> እንግዲህ። የተሰናከሉ እስኪወድቁ ድረስ ነውን? እላለሁ።
አይደለም፤ ነገር ግን እነርሱን ያስቀናቸው ዘንድ በእነርሱ በደል መዳን ለአሕዛብ ሆነ።

<span id="an_11:12">11:12</span> ዳሩ ግን በደላቸው ለዓለም ባለ ጠግነት መሸነፋቸውም ለአሕዛብ
ባለ ጠግነት ከሆነ፥ ይልቁንስ መሙላታቸው እንዴት ይሆን?

<span id="an_11:13">11:13</span> ለእናንተም ለአሕዛብ እናገራለሁ። እኔ የአሕዛብ ሐዋርያ በሆንሁ
መጠን ሥጋዬ የሆኑትን አስቀንቼ ምናልባት ከእነርሱ አንዳንዱን አድን እንደሆነ አገልግሎቴን አከብራለሁ።

<span id="an_11:15">11:15</span> የእነርሱ መጣል ለዓለም መታረቅ ከሆነ ከሙታን ከሚመጣ ሕይወት
በቀር መመለሳቸው ምን ይሆን?

<span id="an_11:16">11:16</span> በኵራቱም ቅዱስ ከሆነ ብሆው ደግሞ ቅዱስ ነው፤ ሥሩም ቅዱስ
ከሆነ ቅርንጫፎቹ ደግሞ ቅዱሳን ናቸው።

<span id="an_11:17">11:17</span> ነገር ግን ከቅርንጫፎች አንዳንዱ ቢሰበሩ አንተም የበረሀ ወይራ
የሆንህ በመካከላቸው ገብተህ ከእነርሱ ጋር የወይራ ዘይት ከሚወጣው ሥር ተካፋይ ከሆንህ፥ በቅርንጫፎች ላይ አትመካ፤

<span id="an_11:18">11:18</span> ብትመካባቸው ግን ሥሩ አንተን ይሸከምሃል እንጂ ሥሩን
የምትሸከም አንተ አይደለህም።

<span id="an_11:19">11:19</span> እንግዲህ። እኔ እንድገባ ቅርንጫፎች ተሰበሩ ትል ይሆናል።

<span id="an_11:20">11:20</span> መልካም፤ እነርሱ ከአለማመን የተነሣ ተሰበሩ አንተም ከእምነት
የተነሣ ቆመሃል። ፍራ እንጂ የትዕቢትን ነገር አታስብ።

<span id="an_11:21">11:21</span> እግዚአብሔር እንደ ተፈጠሩት ለነበሩት ቅርንጫፎች የራራላቸው
ካልሆነ ለአንተ ደግሞ አይራራልህምና።

<span id="an_11:22">11:22</span> እንግዲህ የእግዚአብሔርን ቸርነትና ጭከና ተመልከት፤ ጭከናው
በወደቁት ላይ ነው፥ በቸርነቱ ግን ጸንተህ ብትኖር የእግዚአብሔር ቸርነት በአንተ ላይ ነው፤ ያለዚያ አንተ ደግሞ
ትቆረጣለህ።

<span id="an_11:23">11:23</span> እነዚያም ደግሞ በአለማመናቸው ጸንተው ባይኖሩ በዛፉ ውስጥ
ይገባሉ፤ እግዚአብሔር መልሶ ሊያገባቸው ይችላልና።

<span id="an_11:24">11:24</span> አንተ በፍጥረቱ የበረሀ ከነበረ ወይራ ተቆርጠህ እንደ ፍጥረትህ
ሳትሆን በመልካም ወይራ ከገባህ፥ ይልቁንስ እነዚያ በፍጥረታቸው ያሉት ቅርንጫፎች በራሳቸው ወይራ እንዴት አይገቡም?

<span id="an_11:25">11:25</span> ወንድሞች ሆይ፥ ልባሞች የሆናችሁ እንዳይመስላችሁ ይህን ምሥጢር
ታውቁ ዘንድ እወዳለሁ፤ የአሕዛብ ሙላት እስኪገባ ድረስ ድንዛዜ በእስራኤል በአንዳንድ በኩል ሆነባቸው፤

<span id="an_11:26">11:26</span> እንደዚሁም እስራኤል ሁሉ ይድናል፤ እንዲህ ተብሎ እንደ ተጻፈ።
መድኃኒት ከጽዮን ይወጣል ከያዕቆብም ኃጢአተኛነትን ያስወግዳል።

<span id="an_11:27">11:27</span> ኃጢአታቸውንም ስወስድላቸው ከእነርሱ ጋር የምገባው ኪዳን ይህ
ነው።

<span id="an_11:28">11:28</span> በወንጌልስ በኩል ስለ እናንተ ጠላቶች ናቸው፥ በምርጫ በኩል
ግን ስለ አባቶች ተወዳጆች ናቸው፤

<span id="an_11:29">11:29</span> እግዚአብሔር በጸጋው ስጦታና በመጥራቱ አይጸጸትምና።

<span id="an_11:30">11:30</span> እናንተም ቀድሞ ለእግዚአብሔር እንዳልታዘዛችሁ፥ አሁን ግን
ከአለመታዘዛቸው የተነሣ ምሕረት እንዳገኛችሁ፥

<span id="an_11:31">11:31</span> እንዲሁ በተማራችሁበት ምሕረት እነርሱ ደግሞ ምሕረትን ያገኙ
ዘንድ እነዚህ ደግሞ አሁን አልታዘዙም።

<span id="an_11:32">11:32</span> እግዚአብሔር ሁሉን ይምር ዘንድ ሁሉን በአለመታዘዝ ዘግቶታልና።

<span id="an_11:33">11:33</span> የእግዚአብሔር ባለ ጠግነትና ጥበብ እውቀቱም እንዴት ጥልቅ
ነው፤ ፍርዱ እንዴት የማይመረመር ነው፥ ለመንገዱም ፍለጋ የለውም።

<span id="an_11:34">11:34</span> የጌታን ልብ ያወቀው ማን ነው?

<span id="an_11:35">11:35</span> ወይስ አማካሪው ማን ነበር? ወይስ ብድራቱን ይመልስ ዘንድ
ለእርሱ አስቀድሞ የሰጠው ማን ነው?

<span id="an_11:36">11:36</span> ሁሉ ከእርሱና በእርሱ ለእርሱም ነውና፤ ለእርሱ ለዘላለም ክብር
ይሁን፤ አሜን።

<span id="an_12:1">12:1</span> እንግዲህ፥ ወንድሞች ሆይ፥ ሰውነታችሁን እግዚአብሔርን ደስ
የሚያሰኝና ሕያው ቅዱስም መሥዋዕት አድርጋችሁ ታቀርቡ ዘንድ በእግዚአብሔር ርኅራኄ እለምናችኋለሁ፥ እርሱም ለአእምሮ
የሚመች አገልግሎታችሁ ነው።

<span id="an_12:2">12:2</span> የእግዚአብሔር ፈቃድ እርሱም በጎና ደስ የሚያሰኝ ፍጹምም የሆነው
ነገር ምን እንደ ሆነ ፈትናችሁ ታውቁ ዘንድ በልባችሁ መታደስ ተለወጡ እንጂ ይህን ዓለም አትምሰሉ።

<span id="an_12:3">12:3</span> እግዚአብሔር ለእያንዳንዱ የእምነትን መጠን እንዳካፈለው፥ እንደ
ባለ አእምሮ እንዲያስብ እንጂ ማሰብ ከሚገባው አልፎ በትዕቢት እንዳያስብ በመካከላችሁ ላለው ለእያንዳንዱ በተሰጠኝ
ጸጋ እናገራለሁ።

<span id="an_12:4">12:4</span> በአንድ አካል ብዙ ብልቶች እንዳሉን፥ የብልቶቹም ሁሉ ሥራ አንድ
እንዳይደለ፥

<span id="an_12:5">12:5</span> እንዲሁ ብዙዎች ስንሆን በክርስቶስ አንድ አካል ነን፥ እርስ
በርሳችንም እያንዳንዳችን የሌላው ብልቶች ነን።

<span id="an_12:6">12:6</span> እንደ ተሰጠንም ጸጋ ልዩ ልዩ ስጦታ አለን፤ ትንቢት ቢሆን እንደ
እምነታችን መጠን ትንቢት እንናገር፤

<span id="an_12:7">12:7</span> አገልግሎት ቢሆን በአገልግሎታችን እንትጋ፤ የሚያስተምርም ቢሆን
በማስተማሩ ይትጋ፤

<span id="an_12:8">12:8</span> የሚመክርም ቢሆን በመምከሩ ይትጋ፤ የሚሰጥ በልግስና ይስጥ፤
የሚገዛ በትጋት ይግዛ፤ የሚምር በደስታ ይማር።

<span id="an_12:9">12:9</span> ፍቅራችሁ ያለ ግብዝነት ይሁን። ክፉውን ነገር ተጸየፉት፤ ከበጎ
ነገር ጋር ተባበሩ፤

<span id="an_12:10">12:10</span> በወንድማማች መዋደድ እርስ በርሳችሁ ተዋደዱ፤ እርስ በርሳችሁ
ተከባበሩ፤

<span id="an_12:11">12:11</span> ለሥራ ከመትጋት አትለግሙ፤ በመንፈስ የምትቃጠሉ ሁኑ፤ ለጌታ
ተገዙ፤

<span id="an_12:12">12:12</span> በተስፋ ደስ ይበላችሁ፤ በመከራ ታገሡ፤ በጸሎት ጽኑ፤

<span id="an_12:13">12:13</span> ቅዱሳንን በሚያስፈልጋቸው እርዱ፤ እንግዶችን ለመቀበል ትጉ።

<span id="an_12:14">12:14</span> የሚያሳድዱአችሁን መርቁ፥ መርቁ እንጂ አትርገሙ።

<span id="an_12:15">12:15</span> ደስ ከሚላቸው ጋር ደስ ይበላችሁ፥ ከሚያለቅሱም ጋር አልቅሱ።

<span id="an_12:16">12:16</span> እርስ በርሳችሁ በአንድ አሳብ ተስማሙ፤ የትዕቢትን ነገር
አታስቡ፥ ነገር ግን የትሕትናን ነገር ለመሥራት ትጉ። ልባሞች የሆናችሁ አይምሰላችሁ።

<span id="an_12:17">12:17</span> ለማንም ስለ ክፉ ፈንታ ክፉን አትመልሱ፤ በሰው ሁሉ ፊት
መልካም የሆነውን አስቡ።

<span id="an_12:18">12:18</span> ቢቻላችሁስ በእናንተ በኩል ከሰው ሁሉ ጋር በሰላም ኑሩ።

<span id="an_12:19">12:19</span> ተወዳጆች ሆይ፥ ራሳችሁ አትበቀሉ፥ ለቍጣው ፈንታ ስጡ እንጂ፤
በቀል የእኔ ነው፥ እኔ ብድራቱን እመልሳለሁ ይላል ጌታ ተብሎ ተጽፎአልና።

<span id="an_12:20">12:20</span> ጠላትህ ግን ቢራብ አብላው፤ ቢጠማ አጠጣው፤ ይህን በማድረግህ
በራሱ ላይ የእሳት ፍም ትከምራለህና።

<span id="an_12:21">12:21</span> ክፉውን በመልካም አሸንፍ እንጂ በክፉ አትሸነፍ።

<span id="an_13:1">13:1</span> ነፍስ ሁሉ በበላይ ላሉት ባለ ሥልጣኖች ይገዛ። ከእግዚአብሔር
ካልተገኘ በቀር ሥልጣን የለምና፤ ያሉትም ባለ ሥልጣኖች በእግዚአብሔር የተሾሙ ናቸው።

<span id="an_13:2">13:2</span> ስለዚህ ባለ ሥልጣንን የሚቃወም የእግዚአብሔርን ሥርዓት ይቃወማል፤
የሚቃወሙትም በራሳቸው ላይ ፍርድን ይቀበላሉ።

<span id="an_13:3">13:3</span> ገዥዎች ለክፉ አድራጊዎች እንጂ መልካም ለሚያደርጉ የሚያስፈሩ
አይደሉምና። ባለ ሥልጣንን እንዳትፈራ ትወዳለህን? መልካሙን አድርግ ከእርሱም ምስጋና ይሆንልሃል፤

<span id="an_13:4">13:4</span> ለመልካም ነገር ለአንተ የእግዚአብሔር አገልጋይ ነውና። በከንቱ
ግን ሰይፍ አይታጠቅምና ክፉ ብታደርግ ፍራ፤ ቍጣውን ለማሳየት ክፉ አድራጊውን የሚበቀል የእግዚአብሔር አገልጋይ
ነውና።

<span id="an_13:5">13:5</span> ስለዚህ ስለ ቍጣው ብቻ አይደለም ነገር ግን ስለ ሕሊና ደግሞ
መገዛት ግድ ነው።

<span id="an_13:6">13:6</span> ስለዚህ ደግሞ ትገብራላችሁና፤ በዚህ ነገር የሚተጉ የእግዚአብሔር
አገልጋዮች ናቸውና።

<span id="an_13:7">13:7</span> ለሁሉ የሚገባውን አስረክቡ፤ ግብር ለሚገባው ግብርን፥ ቀረጥ
ለሚገባው ቀረጥን፥ መፈራት ለሚገባው መፈራትን፥ ክብር ለሚገባው ክብርን ስጡ።

<span id="an_13:8">13:8</span> እርስ በርሳችሁ ከመዋደድ በቀር ለማንም ዕዳ አይኑርባችሁ፥ ሌላውን
የሚወድ ሕግን ፈጽሞታልና።

<span id="an_13:9">13:9</span> አታመንዝር፥ አትግደል፥ አትስረቅ፥ በውሸት አትመስክር፥ አትመኝ
የሚለው ከሌላይቱ ትእዛዝ ሁሉ ጋር በዚህ። ባልንጀራህን እንደ ነፍስህ ውደድ በሚለው ቃል ተጠቅልሎአል።

<span id="an_13:10">13:10</span> ፍቅር ለባልንጀራው ክፉ አያደርግም፤ ስለዚህ ፍቅር የሕግ ፍጻሜ
ነው።

<span id="an_13:11">13:11</span> ከእንቅልፍ የምትነሡበት ሰዓት አሁን እንደ ደረሰ ዘመኑን
እወቁ፤ ካመንንበት ጊዜ ይልቅ መዳናችን ዛሬ ወደ እኛ ቀርቦአልና።

<span id="an_13:12">13:12</span> ሌሊቱ አልፎአል፥ ቀኑም ቀርቦአል። እንግዲህ የጨለማውን ሥራ
አውጥተን የብርሃንን ጋሻ ጦር እንልበስ።

<span id="an_13:13">13:13</span> በቀን እንደምንሆን በአገባብ እንመላለስ፤ በዘፈንና በስካር
አይሁን፥ በዝሙትና በመዳራት አይሁን፥ በክርክርና በቅናት አይሁን፤

<span id="an_13:14">13:14</span> ነገር ግን ጌታን ኢየሱስ ክርስቶስን ልበሱት፤ ምኞቱንም
እንዲፈጽም ለሥጋ አታስቡ።

<span id="an_14:1">14:1</span> በእምነት የደከመውንም ተቀበሉት፥ በአሳቡም ላይ አትፍረዱ።

<span id="an_14:2">14:2</span> ሁሉን ይበላ ዘንድ እንደ ተፈቀደለት የሚያምን አለ፥ ደካማው ግን
አትክልት ይበላል።

<span id="an_14:3">14:3</span> የሚበላ የማይበላውን አይናቀው የማይበላውም በሚበላው አይፍረድ፥
እግዚአብሔር ተቀብሎታልና።

<span id="an_14:4">14:4</span> አንተ በሌላው ሎሌ የምትፈርድ ማን ነህ? እርሱ ቢቆም ወይም
ቢወድቅ ለገዛ ጌታው ነው፤ ነገር ግን እግዚአብሔር ሊያቆመው ይችላልና ይቆማል።

<span id="an_14:5">14:5</span> ይህ ሰው አንድ ቀን ከሌላ ቀን እንዲሻል ያስባል፥ ያ ግን ቀን
ሁሉ አንድ እንደ ሆነ ያስባል፤ እያንዳንዱ በገዛ አእምሮው አጥብቆ ይረዳ።

<span id="an_14:6">14:6</span> ቀንን የሚያከብር ለጌታ ብሎ ያከብራል፤ የሚበላም እግዚአብሔርን
ያመሰግናልና ለጌታ ብሎ ይበላል፤ የማይበላም ለጌታ ብሎ አይበላም እግዚአብሔርንም ያመሰግናል።

<span id="an_14:7">14:7</span> ከእኛ አንድ ስንኳ ለራሱ የሚኖር የለምና፥ ለራሱም የሚሞት የለም፤

<span id="an_14:8">14:8</span> በሕይወት ሆነን ብንኖር ለጌታ እንኖራለንና፥ ብንሞትም ለጌታ
እንሞታለን። እንግዲህ በሕይወት ሆነን ብንኖር ወይም ብንሞት የጌታ ነን።

<span id="an_14:9">14:9</span> ስለዚህ ነገር ሙታንንም ሕያዋንንም ይገዛ ዘንድ ክርስቶስ ሞቶአልና
ሕያውም ሆኖአልና።

<span id="an_14:10">14:10</span> አንተም በወንድምህ ላይ ስለ ምን ትፈርዳለህ? ወይስ አንተ
ደግሞ ወንድምህን ስለ ምን ትንቃለህ? ሁላችን በክርስቶስ ፍርድ ወንበር ፊት እንቆማለንና።

<span id="an_14:11">14:11</span> እኔ ሕያው ነኝ፥ ይላል ጌታ፥ ጉልበት ሁሉ ለእኔ ይንበረከካል
መላስም ሁሉ እግዚአብሔርን ያመሰግናል ተብሎ ተጽፎአልና።

<span id="an_14:12">14:12</span> እንግዲያስ እያንዳንዳችን ስለ ራሳችን ለእግዚአብሔር መልስ
እንሰጣለን።

<span id="an_14:13">14:13</span> እንግዲህ ከዛሬ ጀምሮ እርስ በርሳችን አንፈራረድ፤ ይልቁን ግን
ለወንድም እንቅፋትን ወይም ማሰናከያን ማንም እንዳያኖርበት ይህን ቍረጡ።

<span id="an_14:14">14:14</span> በራሱ ርኵስ የሆነ ነገር እንደ ሌለ በጌታ በኢየሱስ ሆኜ
አውቄአለሁ ተረድቼአለሁም፤ ነገር ግን ምንም ርኵስ እንዲሆን ለሚቆጥር ለእርሱ ርኵስ ነው።

<span id="an_14:15">14:15</span> ወንድምህንም በመብል ምክንያት የምታሳዝን ከሆንህ እንግዲህ
በፍቅር አልተመላለስህም። ክርስቶስ ስለ እርሱ የሞተለትን እርሱን በመብልህ አታጥፋው።

<span id="an_14:16">14:16</span> እንግዲህ ለእናንተ ያለው መልካም ነገር አይሰደብ፤

<span id="an_14:17">14:17</span> የእግዚአብሔር መንግሥት ጽድቅና ሰላም በመንፈስ ቅዱስም የሆነ
ደስታ ናት እንጂ መብልና መጠጥ አይደለችምና።

<span id="an_14:18">14:18</span> እንደዚህ አድርጎ ለክርስቶስ የሚገዛ እግዚአብሔርን ደስ
ያሰኛልና፥ በሰውም ዘንድ የተመሰገነ ነው።

<span id="an_14:19">14:19</span> እንግዲያስ ሰላም የሚቆምበትን እርስ በርሳችንም የምንታነጽበትን
እንከተል።

<span id="an_14:20">14:20</span> በመብል ምክንያት የእግዚአብሔርን ሥራ አታፍርስ። ሁሉ ንጹሕ
ነው፥ በመጠራጠር የተበላ እንደ ሆነ ግን ለዚያ ሰው ክፉ ነው።

<span id="an_14:21">14:21</span> ሥጋን አለመብላት ወይንንም አለመጠጣት ወንድምህም
የሚሰናከልበትን አለማድረግ መልካም ነው።

<span id="an_14:22">14:22</span> ለአንተ ያለህ እምነት በእግዚአብሔር ፊት ለራስህ ይሁንልህ።
ፈትኖ መልካም እንዲሆን በሚቈጥረው ነገር በራሱ ላይ የማይፈርድ ብፁዕ ነው።

<span id="an_14:23">14:23</span> የሚጠራጠረው ግን ቢበላ በእምነት ስላልሆነ ተኮንኖአል፤
በእምነትም ያልሆነ ሁሉ ኃጢአት ነው።

<span id="an_15:1">15:1</span> እኛም ኃይለኞች የሆንን የደካሞችን ድካም እንድንሸከም ራሳችንንም
ደስ እንዳናሰኝ ይገባናል።

<span id="an_15:2">15:2</span> እያንዳንዳችን እንድናንጸው እርሱን ለመጥቀም ባልንጀራችንን ደስ
እናሰኝ።

<span id="an_15:3">15:3</span> ክርስቶስ ራሱን ደስ አላሰኘምና፤ ነገር ግን። አንተን የነቀፉበት
ነቀፋ ወደቀብኝ ተብሎ እንደ ተጻፈ ሆነበት።

<span id="an_15:4">15:4</span> በመጽናትና መጻሕፍት በሚሰጡት መጽናናት ተስፋ ይሆንልን ዘንድ
አስቀድሞ የተጻፈው ሁሉ ለትምህርታችን ተጽፎአልና።

<span id="an_15:5">15:5</span> በአንድ ልብ ሆናችሁ በአንድ አፍ እግዚአብሔርን፥ እርሱም የጌታችን
የኢየሱስ ክርስቶስ አባት፥ ታከብሩ ዘንድ፥ የትዕግሥትና የመጽናናት አምላክ እንደ ክርስቶስ ኢየሱስ ፈቃድ እርስ
በርሳችሁ በአንድ አሳብ መሆንን ይስጣችሁ።

<span id="an_15:7">15:7</span> ስለዚህ ክርስቶስ ለእግዚአብሔር ክብር እንደ ተቀበላችሁ እንዲሁ
እርስ በርሳችሁ ተቀባበሉ።

<span id="an_15:8">15:8</span> ለአባቶች የተሰጠውን የተስፋ ቃል ያጸና ዘንድ ደግሞም። ስለዚህ
በአሕዛብ መካከል አመሰግንሃለሁ ለስምህም እዘምራለሁ ተብሎ እንደ ተጻፈ፥ አሕዛብ ስለ ምሕረቱ እግዚአብሔርን ያከብሩ
ዘንድ፥ ክርስቶስ ስለ እግዚአብሔር እውነት የመገረዝ አገልጋይ ሆነ እላለሁ።

<span id="an_15:10">15:10</span> ደግሞም። አሕዛብ ሆይ፥ ከሕዝቡ ጋር ደስ ይበላችሁ ይላል።

<span id="an_15:11">15:11</span> ደግሞም። እናንተ አሕዛብ ሁላችሁ፥ ጌታን አመስግኑ ሕዝቦቹም
ሁሉ ይወድሱት ይላል።

<span id="an_15:12">15:12</span> ደግሞም ኢሳይያስ። የእሴይ ሥር አሕዛብንም ሊገዛ የሚነሣው
ይሆናል፤ በእርሱ አሕዛብ ተስፋ ያደርጋሉ ይላል።

<span id="an_15:13">15:13</span> የተስፋ አምላክም በመንፈስ ቅዱስ ኃይል በተስፋ እንድትበዙ
በማመናችሁ ደስታንና ሰላምን ሁሉ ይሙላባችሁ።

<span id="an_15:14">15:14</span> እኔም ራሴ ደግሞ፥ ወንድሞቼ ሆይ፥ በበጎነት ራሳችሁ እንደ
ተሞላችሁ፥ እውቀትም ሁሉ እንደ ሞላባችሁ፥ እርስ በርሳችሁም ደግሞ ልትገሠጹ እንዲቻላችሁ ስለ እናንተ ተረድቼአለሁ።

<span id="an_15:15">15:15</span> ነገር ግን አሕዛብ በመንፈስ ቅዱስ ተቀድሰው የተወደደ መሥዋዕት
ሊሆኑ፥ ለእግዚአብሔር ወንጌል እንደ ካህን እያገለገልሁ፥ ለአሕዛብ የክርስቶስ ኢየሱስ አገልጋይ እሆን ዘንድ
ከእግዚአብሔር በተሰጠኝ ጸጋ ምክንያት ተመልሼ ላሳስባችሁ ብዬ በአንዳንድ ቦታ በድፍረት ጻፍሁላችሁ።

<span id="an_15:17">15:17</span> እንግዲህ ከእግዚአብሔር ዘንድ በሚሆን ነገር በክርስቶስ ኢየሱስ
ትምክህት አለኝ።

<span id="an_15:18">15:18</span> አሕዛብ እንዲታዘዙ ክርስቶስ በቃልና በሥራ፥ በምልክትና በድንቅ
ነገር ኃይል፥ በመንፈስ ቅዱስም ኃይል በእኔ አድርጎ ከሠራው በቀር ምንም ልናገር አልደፍርም፤ ስለዚህ ከኢየሩሳሌም
ጀምሬ እስከ እልዋሪቆን ድረስ እየዞርሁ የክርስቶስን ወንጌል ፈጽሜ፤ ሰብኬአለሁ።

<span id="an_15:20">15:20</span> እንዲሁም በሌላው ሰው መሠረት ላይ እንዳልሠራ የክርስቶስ ስም
በተጠራበት ስፍራ ሳይሆን ወንጌልን ለመስበክ ተጣጣርሁ፤

<span id="an_15:21">15:21</span> ነገር ግን። ስለ እርሱ ያልተወራላቸው ያያሉ፥ ያልሰሙም
ያስተውላሉ ተብሎ እንደ ተጻፈ ነው።

<span id="an_15:22">15:22</span> ስለዚህ ደግሞ ወደ እናንተ እንዳልመጣ ብዙ ጊዜ ተከለከልሁ።

<span id="an_15:23">15:23</span> አሁን ግን በዚህ አገር ስፍራ ወደ ፊት ስለሌለኝ፥ ከብዙ
ዓመትም ጀምሬ ወደ እናንተ ልመጣ ናፍቆት ስላለኝ፥

<span id="an_15:24">15:24</span> ወደ እስጳንያ በሄድሁ ጊዜ ሳልፍ እናንተን እንዳይ፥ አስቀድሜም
ጥቂት ብጠግባችሁ ወደዚያ በጉዞዬ እንድትረዱኝ ተስፋ አደርጋለሁ።

<span id="an_15:25">15:25</span> አሁን ግን ቅዱሳንን ለማገልገል ወደ ኢየሩሳሌም እሄዳለሁ።

<span id="an_15:26">15:26</span> መቄዶንያና አካይያ በኢየሩሳሌም ቅዱሳን መካከል ያሉትን ድሆች
ይረዱ ዘንድ ወደዋልና።

<span id="an_15:27">15:27</span> ወደዋልና፥ የእነርሱም ባለ ዕዳዎች ናቸው፤ አሕዛብ በእነርሱ
መንፈሳዊ ነገርን ተካፋዮች ከሆኑ በሥጋዊ ነገር ደግሞ ያገለግሉአቸው ዘንድ ይገባቸዋልና።

<span id="an_15:28">15:28</span> እንግዲህ ይህን ፈጽሜ ይህን ፍሬ ካተምሁላቸው በኋላ በእናንተ
በኩል አልፌ ወደ እስጳንያ እሄዳለሁ፤

<span id="an_15:29">15:29</span> ወደ እናንተም ስመጣ በክርስቶስ በረከት ሙላት እንድመጣ
አውቃለሁ።

<span id="an_15:30">15:30</span> ወንድሞች ሆይ፥ ስለ እኔ ወደ እግዚአብሔር እየጸለያችሁ ከእኔ
ጋር ትጋደሉ ዘንድ በጌታችን በኢየሱስ ክርስቶስና በመንፈስ ፍቅር እለምናችኋለሁ፤

<span id="an_15:31">15:31</span> በእግዚአብሔር ፈቃድ በደስታ ወደ እናንተ መጥቼ ከእናንተ ጋር
እንዳርፍ፥ በይሁዳ ካሉት ከማይታዘዙ እድን ዘንድ፥ ለኢየሩሳሌምም ያለኝ አገልግሎቴ ቅዱሳንን ደስ የሚያሰኝ ይሆን
ዘንድ ጸልዩ።

<span id="an_15:33">15:33</span> የሰላምም አምላክ ከሁላችሁ ጋር ይሁን፤ አሜን።

<span id="an_16:1">16:1</span> በክንክራኦስ ባለች ቤተ ክርስቲያን አገልጋይ የምትሆን እኅታችንን
ፌቤንን አደራ ብያችኋለሁ፤

<span id="an_16:2">16:2</span> ለቅዱሳን እንደሚገባ በጌታ ተቀበሉአት፥ እርስዋ ለብዙዎች ለእኔም
ለራሴ ደጋፊ ነበረችና፥ ከእናንተም በምትፈልገው በማናቸውም ነገር እርዱአት።

<span id="an_16:3">16:3</span> በክርስቶስ ኢየሱስ አብረውኝ ለሚሠሩ ለጵርስቅላና ለአቂላ ሰላምታ
አቅርቡልኝ፤

<span id="an_16:4">16:4</span> እነርሱም ስለ ነፍሴ ነፍሳቸውን ለሞት አቀረቡ፥ የአሕዛብም አብያተ
ክርስቲያናት ሁሉ የሚያመሰግኑአቸው ናቸው እንጂ እኔ ብቻ አይደለሁም፤

<span id="an_16:5">16:5</span> በቤታቸውም ላለች ቤተ ክርስቲያን ሰላምታ አቅርቡልኝ። ከእስያ
ለክርስቶስ በኵራት ለሆነው ለምወደው ለአጤኔጦን ሰላምታ አቅርቡልኝ።

<span id="an_16:6">16:6</span> ስለ እናንተ ብዙ ለደከመች ለማርያ ሰላምታ አቅርቡልኝ።

<span id="an_16:7">16:7</span> በሐዋርያት መካከል ስመ ጥሩዎች ለሆኑ፥ ደግሞም ክርስቶስን በማመን
ለቀደሙኝ፥ አብረውኝም ለታሰሩ ለዘመዶቼ ለአንዲራኒቆንና ለዩልያን ሰላምታ አቅርቡልኝ።

<span id="an_16:8">16:8</span> በጌታ ለምወደው ለጵልያጦን ሰላምታ አቅርቡልኝ።

<span id="an_16:9">16:9</span> በክርስቶስ አብሮን ለሚሠራ ለኢሩባኖን ለምወደውም ለስንጣክን
ሰላምታ አቅርቡልኝ።

<span id="an_16:10">16:10</span> በክርስቶስ መሆኑ ተፈትኖ ለተመሰገነው ለኤጤሌን ሰላምታ
አቅርቡልኝ። ከአርስጣባሉ ቤተ ሰዎች ላሉት ሰላምታ አቅርቡልኝ።

<span id="an_16:11">16:11</span> ለዘመዴ ለሄሮድዮና ሰላምታ አቅርቡልኝ። ከንርቀሱ ቤተ ሰዎች
በጌታ ላሉት ሰላምታ አቅርቡልኝ።

<span id="an_16:12">16:12</span> በጌታ ሆነው ለሚደክሙ ለፕሮፊሞናና ለጢሮፊሞሳ ሰላምታ
አቅርቡልኝ። በጌታ እጅግ ለደከመች ለተወደደች ለጠርሲዳ ሰላምታ አቅርቡልኝ።

<span id="an_16:13">16:13</span> በጌታ ሆኖ ለታወቀ ለሩፎን ለእኔና ለእርሱም እናት ሰላምታ
አቅርቡልኝ።

<span id="an_16:14">16:14</span> ለአስቀሪጦንና ለአፍለሶንጳ ለሄሮሜንም ለጳጥሮባም ለሄርማንም
ከእነርሱም ጋር ላሉ ወንድሞች ሰላምታ አቅርቡልኝ።

<span id="an_16:15">16:15</span> ለፍሌጎንና ለዩልያ ለኔርያና ለእኅቱም ለአልንጦንም ከእነርሱ
ጋር ላሉ ቅዱሳን ሁሉ ሰላምታ አቅርቡልኝ።

<span id="an_16:16">16:16</span> በተቀደሰ አሳሳም እርስ በርሳችሁ ሰላምታ ተሰጣጡ። የክርስቶስ
አብያተ ክርስቲያናት ሁሉ ሰላምታ ያቀርቡላችኋል።

<span id="an_16:17">16:17</span> ነገር ግን፥ ወንድሞች ሆይ፥ እናንተ የተማራችሁትን ትምህርት
የሚቃወሙትን መለያየትንና ማሰናከያን የሚያደርጉትን ሰዎች እንድትመለከቱ እለምናችኋለሁ፥ ከእነርሱ ዘንድ ፈቀቅ በሉ፤

<span id="an_16:18">16:18</span> እንዲህ ያሉት ለገዛ ሆዳቸው እንጂ ለጌታችን ለኢየሱስ ክርስቶስ
አይገዙምና፥ በመልካምና በሚያቆላምጥ ንግግርም ተንኮል የሌለባቸውን ሰዎች ልብ ያታልላሉ።

<span id="an_16:19">16:19</span> መታዘዛችሁ ለሁሉ ተወርቶአልና፤ እንግዲህ በእናንተ ደስ
ይለኛል፤ ነገር ግን ለበጎ ነገር ጥበበኞች ለክፉም የዋሆች እንድትሆኑ እወዳለሁ።

<span id="an_16:20">16:20</span> የሰላምም አምላክ ሰይጣንን ከእግራችሁ በታች ፈጥኖ
ይቀጠቅጠዋል። የጌታችን የኢየሱስ ክርስቶስ ጸጋ ከእናንተ ጋር ይሁን።

<span id="an_16:21">16:21</span> አብሮኝ የሚሠራ ጢሞቴዎስ ዘመዶቼም ሉቂዮስና ኢያሶን
ሱሲጴጥሮስም ሰላምታ ያቀርቡላችኋል።

<span id="an_16:22">16:22</span> ይህን መልእክት የጻፍሁ እኔ ጤርጥዮስ በጌታ ሰላምታ
አቀርብላችኋለሁ።

<span id="an_16:23">16:23</span> የእኔና የቤተ ክርስቲያን ሁሉ አስተናጋጅ ጋይዮስ ሰላምታ
ያቀርብላችኋል። የከተማው መጋቢ ኤርስጦስ ወንድማችንም ቁአስጥሮስ ሰላምታ ያቀርቡላችኋል።

<span id="an_16:24">16:24</span> የጌታችን የኢየሱስ ክርስቶስ ጸጋ ከሁላችሁ ጋር ይሁን፤ አሜን።

<span id="an_16:25">16:25</span> እንግዲህ እንደ ወንጌሌ ስለ ኢየሱስ ክርስቶስም እንደ ተሰበከ፥
ከዘላለም ዘመንም የተሰወረው አሁን ግን የታየው በነቢያትም መጻሕፍት የዘላለም እግዚአብሔር እንደ አዘዘ ለእምነት
መታዘዝ ይሆን ዘንድ ለአሕዛብ ሁሉ የታወቀው ምሥጢር እንደ ተገለጠ መጠን ሊያበረታችሁ ለሚችለው፥

<span id="an_16:27">16:27</span> ብቻውን ጥበብ ላለው ለእግዚአብሔር በኢየሱስ ክርስቶስ እስከ
ዘላለም ድረስ ክብር ይሁን፤ አሜን።
