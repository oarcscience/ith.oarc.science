### Amharic Bible: Ephesians

<span id="an_1:1">1:1</span> በእግዚአብሔር ፈቃድ የኢየሱስ ክርስቶስ ሐዋርያ የሆነ ጳውሎስ፥
በኤፌሶን ላሉት ቅዱሳን በክርስቶስ ኢየሱስም ላሉት ምእመናን፤

<span id="an_1:2">1:2</span> ከእግዚአብሔር ከአባታችን ከጌታም ከኢየሱስ ክርስቶስ ጸጋና ሰላም
ለእናንተ ይሁን።

<span id="an_1:3">1:3</span> በክርስቶስ በሰማያዊ ስፍራ በመንፈሳዊ በረከት ሁሉ የባረከን የጌታችን
የኢየሱስ ክርስቶስ አምላክና አባት ይባረክ።

<span id="an_1:4">1:4</span> ዓለም ሳይፈጠር፥ በፊቱ ቅዱሳንና ነውር የሌለን በፍቅር እንሆን ዘንድ
በክርስቶስ መረጠን።

<span id="an_1:5">1:5</span> በበጎ ፈቃዱ እንደ ወደደ፥ በኢየሱስ ክርስቶስ ሥራ ለእርሱ ልጆች
ልንሆን አስቀድሞ ወሰነን።

<span id="an_1:6">1:6</span> በውድ ልጁም እንዲያው የሰጠን የጸጋው ክብር ይመሰገን ዘንድ ይህን
አደረገ።

<span id="an_1:7">1:7</span> በውድ ልጁም፥ እንደ ጸጋው ባለ ጠግነት መጠን፥ በደሙ የተደረገ
ቤዛነታችንን አገኘን እርሱም የበደላችን ስርየት።

<span id="an_1:8">1:8</span> ጸጋውንም በጥበብና በአእምሮ ሁሉ አበዛልን።

<span id="an_1:9">1:9</span> በክርስቶስ ለማድረግ እንደ ወደደ እንደ አሳቡ፥ የፈቃዱን ምሥጢር
አስታውቆናልና፤

<span id="an_1:10">1:10</span> በዘመን ፍጻሜ ይደረግ ዘንድ ያለው አሳቡም በሰማይና በምድር
ያለውን ሁሉ በክርስቶስ ለመጠቅለል ነው።

<span id="an_1:11">1:11</span> እንደ ፈቃዱ ምክር ሁሉን የሚሠራ እንደ እርሱ አሳብ፥ አስቀድመን
የተወሰንን በክርስቶስ ደግሞ ርስትን ተቀበልን።

<span id="an_1:12">1:12</span> ይኸውም፥ በክርስቶስ አስቀድመን ተስፋ ያደረግን እኛ ለክብሩ
ምስጋና እንሆን ዘንድ ነው።

<span id="an_1:13">1:13</span> እናንተም ደግሞ የእውነትን ቃል፥ ይኸውም የመዳናችሁን ወንጌል፥
ሰምታችሁ ደግሞም በክርስቶስ አምናችሁ፥ በተስፋው መንፈስ በመንፈስ ቅዱስ ታተማችሁ፤

<span id="an_1:14">1:14</span> እርሱም የርስታችን መያዣ ነው፥ ለእግዚአብሔር ያለውን ሁሉ
እስኪዋጅ ድረስ፥ ይህም ለክብሩ ምስጋና ይሆናል።

<span id="an_1:15">1:15</span> ስለዚህ እኔ ደግሞ በእናንተ ዘንድ ስለሚሆን በጌታ በኢየሱስ ስለ
ማመንና ለቅዱሳን ሁሉ ስለሚሆን መውደድ ሰምቼ፥

<span id="an_1:16">1:16</span> ስለ እናንተ እያመሰገንሁ ስጸልይ ስለ እናንተ ማሳሰብን አልተውም፤

<span id="an_1:17">1:17</span> የክብር አባት የጌታችን የኢየሱስ ክርስቶስ አምላክ እርሱን በማወቅ
የጥበብንና የመገለጥን መንፈስ እንዲሰጣችሁ እለምናለሁ።

<span id="an_1:18">1:18</span> ይህም የልባችሁ ዓይኖች ሲበሩ የመጥራቱ ተስፋ ምን እንዲሆን
በቅዱሳንም ዘንድ ያለው የርስት ክብር ባለ ጠግነት ምን እንዲሆን ለምናምን ከሁሉ የሚበልጥ የኃይሉ ታላቅነት ምን
እንዲሆን ታውቁ ዘንድ ነው፤

<span id="an_1:20">1:20</span> ክርስቶስንም ከሙታን ሲያስነሣው ከአለቅነትና ከሥልጣንም ከኃይልም
ከጌትነትም ሁሉ በላይና በዚህ ዓለም ብቻ ሳይሆን ነገር ግን ሊመጣ ባለው ዓለም ደግሞ ከሚጠራው ስም ሁሉ በላይ
በሰማያዊ ስፍራ በቀኙ ሲያስቀምጠው በክርስቶስ ባደረገው ሥራ የብርታቱ ጉልበት ይታያል፤

<span id="an_1:22">1:22</span> ሁሉንም ከእግሩ በታች አስገዛለት ከሁሉ በላይም ራስ እንዲሆን
ለቤተ ክርስቲያን ሰጠው።

<span id="an_1:23">1:23</span> እርስዋም አካሉና ሁሉን በሁሉ የሚሞላ የእርሱ ሙላቱ ናት።

<span id="an_2:1">2:1</span> በበደላችሁና በኃጢአታችሁ ሙታን ነበራችሁ፤ በእነርሱም፥ በዚህ ዓለም
እንዳለው ኑሮ፥ በማይታዘዙትም ልጆች ላይ አሁን ለሚሠራው መንፈስ አለቃ እንደ ሆነው በአየር ላይ ሥልጣን እንዳለው
አለቃ ፈቃድ፥ በፊት ተመላለሳችሁባቸው።

<span id="an_2:3">2:3</span> በእነዚህም ልጆች መካከል እኛ ሁላችን ደግሞ፥ የሥጋችንንና
የልቡናችንን ፈቃድ እያደረግን፥ በሥጋችን ምኞት በፊት እንኖር ነበርን እንደ ሌሎቹም ደግሞ ከፍጥረታችን የቁጣ ልጆች
ነበርን።

<span id="an_2:4">2:4</span> ነገር ግን እግዚአብሔር በምሕረቱ ባለ ጠጋ ስለ ሆነ፥

<span id="an_2:5">2:5</span> ከወደደን ከትልቅ ፍቅሩ የተነሣ በበደላችን ሙታን እንኳ በሆንን ጊዜ
ከክርስቶስ ጋር ሕይወት ሰጠን፥ በጸጋ ድናችኋልና፥

<span id="an_2:6">2:6</span> በሚመጡ ዘመናትም በክርስቶስ ኢየሱስ ለእኛ ባለው ቸርነት ከሁሉ
የሚበልጠውን የጸጋውን ባለ ጠግነት ያሳይ ዘንድ፥ ከእርሱ ጋር አስነሣን በክርስቶስ ኢየሱስም በሰማያዊ ስፍራ ከእርሱ
ጋር አስቀመጠን።

<span id="an_2:8">2:8</span> ጸጋው በእምነት አድኖአችኋልና፤ ይህም የእግዚአብሔር ስጦታ ነው እንጂ
ከእናንተ አይደለም፤

<span id="an_2:9">2:9</span> ማንም እንዳይመካ ከሥራ አይደለም።

<span id="an_2:10">2:10</span> እኛ ፍጥረቱ ነንና፤ እንመላለስበት ዘንድ እግዚአብሔር አስቀድሞ
ያዘጋጀውን መልካሙን ሥራ ለማድረግ በክርስቶስ ኢየሱስ ተፈጠርን።

<span id="an_2:11">2:11</span> ስለዚህ እናንተ አስቀድሞ በሥጋ አሕዛብ የነበራችሁ፥ በሥጋ በእጅ
የተገረዙ በተባሉት ያልተገረዙ የተባላችሁ፥ ይህን አስቡ፤

<span id="an_2:12">2:12</span> በዚያ ዘመን ከእስራኤል መንግሥት ርቃችሁ ለተስፋውም ቃል ኪዳን
እንግዶች ሆናችሁ በዚህም ዓለም ተስፋን አጥታችሁ ከእግዚአብሔርም ተለይታችሁ ያለ ክርስቶስ ነበራችሁ።

<span id="an_2:13">2:13</span> አሁን ግን እናንተ በፊት ርቃችሁ የነበራችሁ በክርስቶስ ኢየሱስ
ሆናችሁ በክርስቶስ ደም ቀርባችኋል።

<span id="an_2:14">2:14</span> እርሱ ሰላማችን ነውና፤ ሁለቱን ያዋሐደ በአዋጅ የተነገሩትንም
የትእዛዛትን ሕግ ሽሮ በመካከል ያለውን የጥል ግድግዳን በሥጋው ያፈረሰ፤ ይህም ከሁለታቸው አንድን አዲስን ሰው
በራሱ ይፈጥር ዘንድ ሰላምንም ያደርግ ዘንድ፥

<span id="an_2:16">2:16</span> ጥልንም በመስቀሉ ገድሎ በእርሱ ሁለታቸውን በአንድ አካል
ከእግዚአብሔር ጋር ያስታርቅ ዘንድ ነው።

<span id="an_2:17">2:17</span> መጥቶም ርቃችሁ ለነበራችሁ ለእናንተ ሰላምን፥ ቀርበው ለነበሩትም
ሰላምን የምስራች ብሎ ሰበከ፤

<span id="an_2:18">2:18</span> በእርሱ ሥራ ሁላችን በአንድ መንፈስ ወደ አብ መግባት አለንና።

<span id="an_2:19">2:19</span> እንግዲያስ ከእንግዲህ ወዲህ ከቅዱሳን ጋር ባላገሮችና
የእግዚአብሔር ቤተ ሰዎች ናችሁ እንጂ እንግዶችና መጻተኞች አይደላችሁም።

<span id="an_2:20">2:20</span> በሐዋርያትና በነቢያት መሠረት ላይ ታንጻችኋል፥ የማዕዘኑም ራስ
ድንጋይ ክርስቶስ ኢየሱስ ነው፤

<span id="an_2:21">2:21</span> በእርሱም ሕንጻ ሁሉ እየተጋጠመ በጌታ ቅዱስ ቤተ መቅደስ እንዲሆን
ያድጋል፤

<span id="an_2:22">2:22</span> በእርሱም እናንተ ደግሞ ለእግዚአብሔር መኖሪያ ለመሆን በመንፈስ
አብራችሁ ትሠራላችሁ።

<span id="an_3:1">3:1</span> ስለዚህም አሕዛብ ስለ ሆናችሁ ስለ እናንተ የክርስቶስ ኢየሱስ እስር
የሆንሁ እኔ ጳውሎስ ለእግዚአብሔር እንበረከካለሁ።

<span id="an_3:2">3:2</span> ለእናንተ ስለ ተሰጠኝ ስለ እግዚአብሔር ጸጋ መጋቢነት በእርግጥ
ሰምታችኋል፤

<span id="an_3:3">3:3</span> አስቀድሜ በአጭሩ እንደ ጻፍሁ፥ ይህን ምሥጢር በመግለጥ አስታወቀኝ፤

<span id="an_3:4">3:4</span> ይህንም ስታነቡ የክርስቶስን ምሥጢር እንዴት እንደማስተውል ልትመለከቱ
ትችላላችሁ፤

<span id="an_3:5">3:5</span> ይህም፥ አሕዛብ አብረው እንዲወርሱ፥ በአንድ አካልም አብረው
እንዲሆኑ፥ በወንጌልም መስበክ በክርስቶስ ኢየሱስ በሆነ የተስፋ ቃል አብረው እንዲካፈሉ፥ ለቅዱሳን ሐዋርያትና
ለነቢያት በመንፈስ አሁን እንደ ተገለጠ በሌሎቹ ትውልዶች ዘንድ ለሰው ልጆች አልታወቀም።

<span id="an_3:7">3:7</span> እንደ ኃይሉ ሥራ እንደ ተሰጠኝም እንደ እግዚአብሔር ጸጋ ስጦታ
መጠንየወንጌል አገልጋይ ሆንሁለት።

<span id="an_3:8">3:8</span> ፍለጋ የሌለውን የክርስቶስን ባለ ጠግነት ለአሕዛብ እሰብክ ዘንድ፥
ሁሉንም በፈጠረው በእግዚአብሔር ከዘላለም የተሰወረው የምሥጢር ሥርዓት ምን እንደሆነ ለሁሉ እገልጥ ዘንድ ይህ ጸጋ
ከቅዱሳን ሁሉ ይልቅ ለማንስ ለኔ ተሰጠ፤

<span id="an_3:10">3:10</span> ብዙ ልዩ ልዩ የእግዚአብሔር ጥበብ አሁን በቤተ ክርስቲያን በኩል
በሰማያዊ ስፍራ ውስጥ ላሉት አለቆችና ሥልጣናት ትታወቅ ዘንድ፤

<span id="an_3:11">3:11</span> ይህም በክርስቶስ ኢየሱስ በጌታችን የፈጸመው የዘላለም አሳብ
ነበረ፥

<span id="an_3:12">3:12</span> በእርሱም ዘንድ ባለ እምነታችን በኩል በመታመን ድፍረትና መግባት
በእርሱ አለን።

<span id="an_3:13">3:13</span> ስለዚህ ስለ እናንተ ስላለ መከራዬ እንዳትታክቱ እለምናችኋለሁ፥
ክብራችሁ ነውና።

<span id="an_3:14">3:14</span> ስለዚህ ምክንያት በሰማይና በምድር ያለ አባትነት ሁሉ ከሚሰየምበት
ከአብ ፊት እንበረከካለሁ፤

<span id="an_3:16">3:16</span> በመንፈሱ በውስጥ ሰውነታችሁ በኃይል እንድትጠነክሩ ክርስቶስም
በልባችሁ በእምነት እንዲኖር እንደ ክብሩ ባለ ጠግነት መጠን ይስጣችሁ፤ የእናንተ ሥርና መሠረት በፍቅር ይጸና
ዘንድ፥

<span id="an_3:18">3:18</span> ከቅዱሳን ሁሉ ጋር ስፋቱና ርዝመቱ ከፍታውም ጥልቅነቱም ምን ያህል
መሆኑን ለማስተዋል፥ ከመታወቅም የሚያልፈውን የክርስቶስን ፍቅር ለማወቅ ትበረቱ ዘንድ፥ እስከ እግዚአብሔርም ፍጹም
ሙላት ደርሳችሁ ትሞሉ ዘንድ።

<span id="an_3:20">3:20</span> እንግዲህ በእኛ እንደሚሠራው ኃይል መጠን ከምንለምነው ወይም
ከምናስበው ሁሉ ይልቅ እጅግ አብልጦ ሊያደርግ ለሚቻለው፥

<span id="an_3:21">3:21</span> ለእርሱ በቤተ ክርስቲያን በክርስቶስ ኢየሱስ እስከ ትውልዶች ሁሉ
ከዘላለም እስከ ዘላለም ክብር ይሁን፤ አሜን።

<span id="an_4:1">4:1</span> እንግዲህ በጌታ እስር የሆንሁ እኔ በተጠራችሁበት መጠራታችሁ
እንደሚገባ ትመላለሱ ዘንድ እለምናችኋለሁ፤

<span id="an_4:2">4:2</span> በትሕትና ሁሉና በየዋህነት በትዕግሥትም፤ እርስ በርሳችሁ በፍቅር
ታገሡ፤

<span id="an_4:3">4:3</span> በሰላም ማሰሪያ የመንፈስን አንድነት ለመጠበቅ ትጉ።

<span id="an_4:4">4:4</span> በመጠራታችሁ በአንድ ተስፋ እንደ ተጠራችሁ አንድ አካልና አንድ
መንፈስ አለ፤

<span id="an_4:5">4:5</span> አንድ ጌታ አንድ ሃይማኖት አንዲት ጥምቀት፤

<span id="an_4:6">4:6</span> ከሁሉ በላይ የሚሆን በሁሉም የሚሠራ በሁሉም የሚኖር አንድ አምላክ
የሁሉም አባት አለ።

<span id="an_4:7">4:7</span> ነገር ግን እንደ ክርስቶስ ስጦታ መጠን ለእያንዳንዳችን ጸጋ ተሰጠን።

<span id="an_4:8">4:8</span> ስለዚህ። ወደ ላይ በወጣ ጊዜ ምርኮን ማረከ ለሰዎችም ስጦታን ሰጠ
ይላል።

<span id="an_4:9">4:9</span> ወደ ምድር ታችኛ ክፍል ደግሞ ወረደ ማለት ካልሆነ፥ ይህ ወጣ ማለትስ
ምን ማለት ነው?

<span id="an_4:10">4:10</span> ይህ የወረደው ሁሉን ይሞላ ዘንድ ከሰማያት ሁሉ በላይ የወጣው
ደግሞ ያው ነው።

<span id="an_4:11">4:11</span> እርሱም አንዳንዶቹ ሐዋርያት፥ ሌሎቹም ነቢያት፥ ሌሎቹም ወንጌልን
ሰባኪዎች፥ ሌሎቹም እረኞችና አስተማሪዎች እንዲሆኑ ሰጠ፤

<span id="an_4:12">4:12</span> ሁላችን የእግዚአብሔርን ልጅ በማመንና በማወቅ ወደሚገኝ አንድነት፥
ሙሉ ሰውም ወደ መሆን፥ የክርስቶስም ሙላቱ ወደሚሆን ወደ ሙላቱ ልክ እስክንደርስ ድረስ፥ ቅዱሳን አገልግሎትን
ለመሥራትና ለክርስቶስ አካል ሕንጻ ፍጹማን ይሆኑ ዘንድ።

<span id="an_4:14">4:14</span> እንደ ስሕተት ሽንገላ ባለ ተንኮል በሰዎችም ማታለል ምክንያት
በትምህርት ነፋስ ሁሉ እየተፍገመገምን ወዲያና ወዲህም እየተንሳፈፍን ሕፃናት መሆን ወደ ፊት አይገባንም፥

<span id="an_4:15">4:15</span> ነገር ግን እውነትን በፍቅር እየያዝን በነገር ሁሉ ወደ እርሱ ራስ
ወደሚሆን ወደ ክርስቶስ እንደግ፤

<span id="an_4:16">4:16</span> ከእርሱም የተነሣ አካል ሁሉ እያንዳንዱ ክፍል በልክ እንደሚሠራ፥
በተሰጠለት በጅማት ሁሉ እየተጋጠመና እየተያያዘ፥ ራሱን በፍቅር ለማነጽ አካሉን ያሳድጋል።

<span id="an_4:17">4:17</span> እንግዲህ አሕዛብ ደግሞ በአእምሮአቸው ከንቱነት እንደሚመላለሱ
ከእንግዲህ ወዲህ እንዳትመላለሱ እላለሁ በጌታም ሆኜ እመሰክራለሁ።

<span id="an_4:18">4:18</span> እነርሱ ባለማወቃቸው ጠንቅ በልባቸውም ደንዳንነት ጠንቅ ልቡናቸው
ጨለመ፥ ከእግዚአብሔርም ሕይወት ራቁ፤

<span id="an_4:19">4:19</span> ደንዝዘውም በመመኘት ርኵሰትን ሁሉ ለማድረግ ራሳቸውን ወደ
ሴሰኝነት አሳልፈው ሰጡ።

<span id="an_4:20">4:20</span> እናንተ ግን ክርስቶስን እንደዚህ አልተማራችሁም፤

<span id="an_4:21">4:21</span> በእርግጥ ሰምታችሁታልና፥ እውነትም በኢየሱስ እንዳለ በእርሱ
ተምራችኋል፤

<span id="an_4:22">4:22</span> ፊተኛ ኑሮአችሁን እያሰባችሁ እንደሚያታልል ምኞት የሚጠፋውን
አሮጌውን ሰው አስወግዱ፥

<span id="an_4:23">4:23</span> በአእምሮአችሁም መንፈስ ታደሱ፥

<span id="an_4:24">4:24</span> ለእውነትም በሚሆኑ ጽድቅና ቅድስና እንደ እግዚአብሔር ምሳሌ
የተፈጠረውን አዲሱን ሰው ልበሱ።

<span id="an_4:25">4:25</span> ስለዚህ ውሸትን አስወግዳችሁ፥ እርስ በርሳችን ብልቶች ሆነናልና
እያንዳንዳችሁ ከባልንጀሮቻችሁ ጋር እውነትን ተነጋገሩ።

<span id="an_4:26">4:26</span> ተቆጡ ኃጢአትንም አታድርጉ፤

<span id="an_4:27">4:27</span> በቁጣችሁ ላይ ፀሐይ አይግባ፥ ለዲያብሎስም ፈንታ አትስጡት።

<span id="an_4:28">4:28</span> የሰረቀ ከእንግዲህ ወዲህ አይስረቅ፥ ነገር ግን በዚያ ፈንታ
ለጎደለው የሚያካፍለው እንዲኖርለት በገዛ እጆቹ መልካምን እየሠራ ይድከም።

<span id="an_4:29">4:29</span> ለሚሰሙት ጸጋን ይሰጥ ዘንድ፥ እንደሚያስፈልግ ለማነጽ የሚጠቅም
ማናቸውም በጎ ቃል እንጂ ክፉ ቃል ከአፋችሁ ከቶ አይውጣ።

<span id="an_4:30">4:30</span> ለቤዛም ቀን የታተማችሁበትን ቅዱሱን የእግዚአብሔርን መንፈስ
አታሳዝኑ።

<span id="an_4:31">4:31</span> መራርነትና ንዴት ቁጣም ጩኸትም መሳደብም ሁሉ ከክፋት ሁሉ ጋር
ከእናንተ ዘንድ ይወገድ።

<span id="an_4:32">4:32</span> እርስ በርሳችሁም ቸሮችና ርኅሩኆች ሁኑ፥ እግዚአብሔርም ደግሞ
በክርስቶስ ይቅር እንዳላችሁ ይቅር ተባባሉ።

<span id="an_5:1">5:1</span> እንግዲህ እንደ ተወደዱ ልጆች እግዚአብሔርን የምትከተሉ ሁኑ፥

<span id="an_5:2">5:2</span> ክርስቶስም ደግሞ እንደ ወደዳችሁ ለእግዚአብሔርም የመዓዛ ሽታ
የሚሆንን መባንና መሥዋዕትን አድርጎ ስለ እናንተ ራሱን አሳልፎ እንደ ሰጠ በፍቅር ተመላለሱ።

<span id="an_5:3">5:3</span> ለቅዱሳን እንደሚገባ ግን ዝሙትና ርኵሰት ሁሉ ወይም መመኘት በእናንተ
ዘንድ ከቶ አይሰማ፤

<span id="an_5:4">5:4</span> የሚያሳፍር ነገርም የስንፍና ንግግርም ወይም ዋዛ የማይገቡ ናቸውና
አይሁኑ፥ ይልቁን ምስጋና እንጂ።

<span id="an_5:5">5:5</span> ይህን እወቁ፤ አመንዝራም ቢሆን ወይም ርኵስ ወይም የሚመኝ እርሱም
ጣዖትን የሚያመልክ በክርስቶስና በእግዚአብሔር መንግሥት ርስት የለውም።

<span id="an_5:6">5:6</span> ከዚህ የተነሣ በማይታዘዙት ልጆች ላይ የእግዚአብሔር ቁጣ ይመጣልና
ማንም በከንቱ ንግግር አያታልላችሁ።

<span id="an_5:7">5:7</span> እንግዲህ ከእነርሱ ጋር ተካፋዮች አትሁኑ፤

<span id="an_5:8">5:8</span> ቀድሞ ጨለማ ነበራችሁና፥ አሁን ግን በጌታ ብርሃን ናችሁ፤

<span id="an_5:9">5:9</span> የብርሃኑ ፍሬ በበጎነትና በጽድቅ በእውነትም ሁሉ ነውና ለጌታ ደስ
የሚያሰኘውን እየመረመራችሁ፥ እንደ ብርሃን ልጆች ተመላለሱ፤

<span id="an_5:11">5:11</span> ፍሬም ከሌለው ከጨለማ ሥራ ጋር አትተባበሩ፥ ይልቁን ግለጡት
እንጂ፥

<span id="an_5:12">5:12</span> እነርሱ በስውር ስለሚያደርጉት መናገር እንኳ ነውር ነውና፤

<span id="an_5:13">5:13</span> ሁሉ ግን በብርሃን ሲገለጥ ይታያል፤ የሚታየው ሁሉ ብርሃን ነውና።

<span id="an_5:14">5:14</span> ስለዚህ። አንተ የምትተኛ ንቃ ከሙታንም ተነሣ ክርስቶስም
ያበራልሃል ይላል።

<span id="an_5:15">5:15</span> እንግዲህ እንደ ጥበበኞች እንጂ ጥበብ እንደሌላቸው ሳይሆን እንዴት
እንድትመላለሱ በጥንቃቄ ተጠበቁ፤

<span id="an_5:16">5:16</span> ቀኖቹ ክፉዎች ናቸውና ዘመኑን ዋጁ።

<span id="an_5:17">5:17</span> ስለዚህ የጌታ ፈቃድ ምን እንደ ሆነ አስተውሉ እንጂ ሞኞች
አትሁኑ።

<span id="an_5:18">5:18</span> መንፈስ ይሙላባችሁ እንጂ በወይን ጠጅ አትስከሩ ይህ ማባከን
ነውና፤

<span id="an_5:19">5:19</span> በመዝሙርና በዝማሬ በመንፈሳዊም ቅኔ እርስ በርሳችሁ ተነጋገሩ፤
ለጌታ በልባችሁ ተቀኙና ዘምሩ፤

<span id="an_5:20">5:20</span> ሁልጊዜ ስለ ሁሉ በጌታችን በኢየሱስ ክርስቶስ ስም አምላካችንንና
አባታችንን ስለ ሁሉ አመስግኑ።

<span id="an_5:21">5:21</span> ለእያንዳንዳችሁ በክርስቶስ ፍርሃት የተገዛችሁ ሁኑ።

<span id="an_5:22">5:22</span> ሚስቶች ሆይ፥ ለጌታ እንደምትገዙ ለባሎቻችሁ ተገዙ፤

<span id="an_5:23">5:23</span> ክርስቶስ ደግሞ የቤተ ክርስቲያን ራስ እንደ ሆነ እርሱም አካሉን
የሚያድን እንደ ሆነ ባል የሚስት ራስ ነውና።

<span id="an_5:24">5:24</span> ዳሩ ግን ቤተ ክርስቲያን ለክርስቶስ እንደምትገዛ እንዲሁ ሚስቶች
ደግሞ በሁሉ ለባሎቻቸው ይገዙ።

<span id="an_5:25">5:25</span> ባሎች ሆይ፥ ክርስቶስ ደግሞ ቤተ ክርስቲያንን እንደ ወደዳት
ሚስቶቻችሁን ውደዱ፤ በውኃ መታጠብና ከቃሉ ጋር አንጽቶ እንዲቀድሳት ስለ እርስዋ ራሱን አሳልፎ ሰጠ፤

<span id="an_5:27">5:27</span> እድፈት ወይም የፊት መጨማደድ ወይም እንዲህ ያለ ነገር ሳይሆንባት
ቅድስትና ያለ ነውር ትሆን ዘንድ ክብርት የሆነችን ቤተ ክርስቲያን ለራሱ እንዲያቀርብ ፈለገ።

<span id="an_5:28">5:28</span> እንዲሁም ባሎች ደግሞ እንደ ገዛ ሥጋቸው አድርገው የገዛ
ሚስቶቻቸውን ሊወዱአቸው ይገባቸዋል። የገዛ ሚስቱን የሚወድ ራሱን ይወዳል፤

<span id="an_5:29">5:29</span> ማንም የገዛ ሥጋውን የሚጠላ ከቶ የለምና፥ ነገር ግን የአካሉ
ብልቶች ስለሆንን፥ ክርስቶስ ደግሞ ለቤተ ክርስቲያን እንዳደረገላት፥ ይመግበዋል ይከባከበውማል።

<span id="an_5:31">5:31</span> ስለዚህ ሰው አባቱንና እናቱን ይተዋል ከሚስቱም ጋር ይተባበራል
ሁለቱም አንድ ሥጋ ይሆናሉ።

<span id="an_5:32">5:32</span> ይህ ምሥጢር ታላቅ ነው፥ እኔ ግን ይህን ስለ ክርስቶስና ስለ ቤተ
ክርስቲያን እላለሁ።

<span id="an_5:33">5:33</span> ሆኖም ከእናንተ ደግሞ እያንዳንዱ የገዛ ሚስቱን እንዲህ እንደ ራሱ
አድርጎ ይውደዳት፥ ሚስቱም ባልዋን ትፍራ።

<span id="an_6:1">6:1</span> ልጆች ሆይ፥ ለወላጆቻችሁ በጌታ ታዘዙ፥ ይህ የሚገባ ነውና።

<span id="an_6:2">6:2</span> መልካም እንዲሆንልህ ዕድሜህም በምድር ላይ እንዲረዝም አባትህንና
እናትህን አክብር፤ እርስዋም የተስፋ ቃል ያላት ፊተኛይቱ ትእዛዝ ናት።

<span id="an_6:4">6:4</span> እናንተም አባቶች ሆይ፥ ልጆቻችሁን በጌታ ምክርና በተግሣጽ
አሳድጉአቸው እንጂ አታስቆጡአቸው።

<span id="an_6:5">6:5</span> ባሪያዎች ሆይ፥ ለክርስቶስ እንደምትታዘዙ በፍርሃትና በመንቀጥቀጥ
በልባችሁ ቅንነት በሥጋ ጌቶቻችሁ ለሆኑ ታዘዙ፤

<span id="an_6:6">6:6</span> የእግዚአብሔርን ፈቃድ እንደሚያደርጉ እንደ ክርስቶስ ባሪያዎች እንጂ
ለሰው ደስ እንደምታሰኙ ለታይታ የምትገዙ አትሁኑ።

<span id="an_6:7">6:7</span> ለሰው ሳይሆን ለጌታ እንደምትገዙ በትጋትና በበጎ ፈቃድ ተገዙ፤

<span id="an_6:8">6:8</span> ባሪያ ቢሆን ወይም ጨዋ ሰው፥ እያንዳንዱ የሚያደርገውን መልካም ነገር
ሁሉ ከጌታ በብድራት እንዲቀበለው ታውቃላችሁና።

<span id="an_6:9">6:9</span> እናንተም ጌቶች ሆይ፥ ዛቻውን ትታችሁ እንዲሁ አድርጉላቸው፥
በእነርሱና በእናንተ ላይ የሚገዛው ጌታ በሰማይ እንዳለ ለሰው ፊትም እንዳያደላ ታውቃላችሁና።

<span id="an_6:10">6:10</span> በቀረውስ በጌታና በኃይሉ ችሎት የበረታችሁ ሁኑ።

<span id="an_6:11">6:11</span> የዲያብሎስን ሽንገላ ትቃወሙ ዘንድ እንዲቻላችሁ የእግዚአብሔርን
ዕቃ ጦር ሁሉ ልበሱ።

<span id="an_6:12">6:12</span> መጋደላችን ከደምና ከሥጋ ጋር አይደለምና፥ ከአለቆችና ከሥልጣናት
ጋር ከዚህም ከጨለማ ዓለም ገዦች ጋር በሰማያዊም ስፍራ ካለ ከክፋት መንፈሳውያን ሠራዊት ጋር ነው እንጂ።

<span id="an_6:13">6:13</span> ስለዚህ በክፉው ቀን ለመቃወም፥ ሁሉንም ፈጽማችሁ ለመቆም
እንድትችሉ የእግዚአብሔርን ዕቃ ጦር ሁሉ አንሡ።

<span id="an_6:14">6:14</span> እንግዲህ ወገባችሁን በእውነት ታጥቃችሁ፥ የጽድቅንም ጥሩር
ለብሳችሁ፥ በሰላም ወንጌልም በመዘጋጀት እግሮቻችሁ ተጫምተው ቁሙ፤

<span id="an_6:16">6:16</span> በሁሉም ላይ ጨምራችሁ የሚንበለበሉትን የክፉውን ፍላጻዎች ሁሉ
ልታጠፉ የምትችሉበትን የእምነትን ጋሻ አንሡ፤

<span id="an_6:17">6:17</span> የመዳንንም ራስ ቁር የመንፈስንም ሰይፍ ያዙ እርሱም የእግዚአብሔር
ቃል ነው።

<span id="an_6:18">6:18</span> በጸሎትና በልመናም ሁሉ ዘወትር በመንፈስ ጸልዩ፤ በዚህም አሳብ
ስለ ቅዱሳን ሁሉ እየለመናችሁ በመጽናት ሁሉ ትጉ፤

<span id="an_6:19">6:19</span> ደግሞ የወንጌልን ምሥጢር በግልጥ እንዳስታውቅ አፌን በመክፈት ቃል
ይሰጠኝ ዘንድ ስለ እኔ ለምኑ፤

<span id="an_6:20">6:20</span> ስለ ወንጌልም በሰንሰለት መልእክተኛ የሆንሁ፥ መናገር እንደሚገባኝ
ስለ እርሱ በግልጥ እናገር ዘንድ ለምኑ።

<span id="an_6:21">6:21</span> ነገር ግን እናንተ ደግሞ እንዴት እንዳለሁ ኑሮዬን እንድታውቁ
የተወደደ ወንድምና በጌታ የታመነ አገልጋይ ቲኪቆስ ሁሉን ያስታውቃችኋል፤

<span id="an_6:22">6:22</span> ወሬአችንን እንድታውቁና ልባችሁን እንዲያጽናና ወደ እናንተ
የምልከው ስለዚህ ምክንያት ነው።

<span id="an_6:23">6:23</span> ከእግዚአብሔር አብ ከጌታም ከኢየሱስ ክርስቶስ ሰላምና ፍቅር
ከእምነት ጋር ለወንድሞች ይሁን።

<span id="an_6:24">6:24</span> ጌታችንን ኢየሱስ ክርስቶስን ባለመጥፋት ከሚወዱ ሁሉ ጋር ጸጋ
ይሁን፤ አሜን።
