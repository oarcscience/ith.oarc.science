### Amharic Bible: James

<span id="an_1:1">1:1</span> የእግዚአብሔርና የጌታ የኢየሱስ ክርስቶስ ባሪያ ያዕቆብ ለተበተኑ
ለአሥራ ሁለቱ ወገኖች፤ ሰላም ለእናንተ ይሁን።

<span id="an_1:2">1:2</span> ወንድሞቼ ሆይ፥ የእምነታችሁ መፈተን ትዕግሥትን እንዲያደርግላችሁ
አውቃችሁ፥ ልዩ ልዩ ፈተና ሲደርስባችሁ እንደ ሙሉ ደስታ ቍጠሩት

<span id="an_1:4">1:4</span> ትዕግሥትም ምንም የሚጎድላችሁ ሳይኖር ፍጹማንና ምሉዓን ትሆኑ ዘንድ
ሥራውን ይፈጽም።

<span id="an_1:5">1:5</span> ከእናንተ ግን ማንም ጥበብ ቢጎድለው፥ ሳይነቅፍ በልግስና ለሁሉ
የሚሰጠውን እግዚአብሔርን ይለምን፥ ለእርሱም ይሰጠዋል።

<span id="an_1:6">1:6</span> ነገር ግን በምንም ሳይጠራጠር በእምነት ይለምን፤ የሚጠራጠር ሰው
በነፋስ የተገፋና የተነቃነቀ የባሕርን ማዕበል ይመስላልና።

<span id="an_1:7">1:7</span> ሁለት አሳብ ላለው በመንገዱም ሁሉ ለሚወላውል ለዚያ ሰው ከጌታ ዘንድ
አንዳች እንዲያገኝ አይምሰለው።

<span id="an_1:9">1:9</span> የተዋረደው ወንድም ግን በከፍታው፥ ባለ ጠጋም በውርደቱ ይመካ፤ እንደ
ሣር አበባ ያልፋልና።

<span id="an_1:11">1:11</span> ፀሐይ ከትኵሳት ጋር ይወጣልና፥ ሣርንም ያጠወልጋልና፥ አበባውም
ይረግፋልና፥ የመልኩም ውበት ይጠፋልና፤ እንዲሁ ደግሞ ባለ ጠጋው በመንገዱ ይዝላል።

<span id="an_1:12">1:12</span> በፈተና የሚጸና ሰው የተባረከ ነው፤ ከተፈተነ በኋላ ለሚወዱት
ተስፋ ስለ እርሱ የሰጣቸውን የሕይወትን አክሊል ይቀበላልና።

<span id="an_1:13">1:13</span> ማንም ሲፈተን። በእግዚአብሔር እፈተናለሁ አይበል፤ እግዚአብሔር
በክፉ አይፈተንምና፤ እርሱ ራሱስ ማንንም አይፈትንም።

<span id="an_1:14">1:14</span> ነገር ግን እያንዳንዱ በራሱ ምኞት ሲሳብና ሲታለል ይፈተናል።

<span id="an_1:15">1:15</span> ከዚህ በኋላ ምኞት ፀንሳ ኃጢአትን ትወልዳለች፤ ኃጢአትም ካደገች
በኋላ ሞትን ትወልዳለች።

<span id="an_1:16">1:16</span> የተወደዳችሁ ወንድሞቼ ሆይ፥ አትሳቱ።

<span id="an_1:17">1:17</span> በጎ ስጦታ ሁሉ ፍጹምም በረከት ሁሉ ከላይ ናቸው፥ መለወጥም
በእርሱ ዘንድ ከሌለ በመዞርም የተደረገ ጥላ በእርሱ ዘንድ ከሌለ ከብርሃናት አባት ይወርዳሉ።

<span id="an_1:18">1:18</span> ለፍጥረቱ የበኵራት ዓይነት እንድንሆን በእውነት ቃል አስቦ
ወለደን።

<span id="an_1:19">1:19</span> ስለዚህ፥ የተወደዳችሁ ወንድሞቼ ሆይ፥ ሰው ሁሉ ለመስማት የፈጠነ
ለመናገርም የዘገየ ለቍጣም የዘገየ ይሁን፤

<span id="an_1:20">1:20</span> የሰው ቍጣ የእግዚአብሔርን ጽድቅ አይሠራምና።

<span id="an_1:21">1:21</span> ስለዚህ ርኵሰትን ሁሉ የክፋትንም ትርፍ አስወግዳችሁ፥ ነፍሳችሁን
ማዳን የሚችለውን በውስጣችሁም የተተከለውን ቃል በየዋህነት ተቀበሉ።

<span id="an_1:22">1:22</span> ቃሉን የምታደርጉ ሁኑ እንጂ ራሳችሁን እያሳታችሁ የምትሰሙ ብቻ
አትሁኑ።

<span id="an_1:23">1:23</span> ቃሉን የሚሰማ የማያደርገውም ቢኖር የተፈጥሮ ፊቱን በመስተዋት
የሚያይን ሰው ይመስላል፤

<span id="an_1:24">1:24</span> ራሱን አይቶ ይሄዳልና፥ ወዲያውም እንደ ምን እንደ ሆነ ይረሳል።

<span id="an_1:25">1:25</span> ነገር ግን ነጻ የሚያወጣውን ፍጹሙን ሕግ ተመልክቶ የሚጸናበት፥
ሥራንም የሚሠራ እንጂ ሰምቶ የሚረሳ ያልሆነው፥ በሥራው የተባረከ ይሆናል።

<span id="an_1:26">1:26</span> አንደበቱን ሳይገታ ልቡን እያሳተ እግዚአብሔርን የሚያመልክ
የሚመስለው ማንም ቢኖር የእርሱ አምልኮ ከንቱ ነው።

<span id="an_1:27">1:27</span> ንጹሕ የሆነ ነውርም የሌለበት አምልኮ በእግዚአብሔር አብ ዘንድ
ይህ ነው፤ ወላጆች የሌላቸውን ልጆች ባልቴቶችንም በመከራቸው መጠየቅ፥ በዓለምም ከሚገኝ እድፍ ሰውነቱን መጠበቅ
ነው።

<span id="an_2:1">2:1</span> ወንድሞቼ ሆይ፥ በክብር ጌታ በጌታችን በኢየሱስ ክርስቶስ ያለውን
እምነት ለሰው ፊት በማድላት አትያዙ።

<span id="an_2:2">2:2</span> የወርቅ ቀለበት ያደረገና የጌጥ ልብስ የለበሰ ሰው ወደ ጉባኤአችሁ
ቢገባ፥ እድፍ ልብስም የለበሰ ድሀ ሰው ደግሞ ቢገባ፥

<span id="an_2:3">2:3</span> የጌጥ ልብስም የለበሰውን ተመልክታችሁ። አንተስ በዚህ በመልካም ስፍራ
ተቀመጥ ብትሉት፥ ድሀውንም። አንተስ ወደዚያ ቁም ወይም ከእግሬ መረገጫ በታች ተቀመጥ ብትሉት፥ ራሳችሁን
መለያየታችሁ አይደለምን?

<span id="an_2:4">2:4</span> ክፉ አሳብ ያላቸውም ዳኞች መሆናችሁ አይደለምን?

<span id="an_2:5">2:5</span> የተወደዳችሁ ወንድሞቼ ሆይ፥ ስሙ፤ እግዚአብሔር በእምነት ባለ ጠጎች
እንዲሆኑ ለሚወዱትም ተስፋ ስለ እርሱ የሰጣቸውን መንግሥት እንዲወርሱ የዚህን ዓለም ድሆች አልመረጠምን?

<span id="an_2:6">2:6</span> እናንተ ግን ድሆችን አዋረዳችሁ። ባለ ጠጎቹ የሚያስጨንቁአችሁ
አይደሉምን? ወደ ፍርድ ቤትም የሚጎትቱአችሁ እነርሱ አይደሉምን?

<span id="an_2:7">2:7</span> የተጠራችሁበትን መልካሙን ስም የሚሰድቡ እነርሱ አይደሉምን?

<span id="an_2:8">2:8</span> ነገር ግን መጽሐፍ። ባልንጀራህን እንደ ራስህ ውደድ እንደሚል
የንጉሥን ሕግ ብትፈጽሙ መልካም ታደርጋላችሁ፤

<span id="an_2:9">2:9</span> ለሰው ፊት ግን ብታደሉ ኃጢአትን ትሠራላችሁ፥ ሕግም እንደ ተላላፊዎች
ይወቅሳችኋል።

<span id="an_2:10">2:10</span> ሕግን ሁሉ የሚጠብቅ፥ ነገር ግን በአንዱ የሚሰናከል ማንም ቢኖር
በሁሉ በደለኛ ይሆናል፤ አታመንዝር ያለው ደግሞ። አትግደል ብሎአልና፤

<span id="an_2:11">2:11</span> ባታመነዝርም፥ ነገር ግን ብትገድል፥ ሕግን ተላላፊ ሆነሃል።

<span id="an_2:12">2:12</span> በነጻነት ሕግ ፍርድን ይቀበሉ ዘንድ እንዳላቸው ሰዎች እንዲህ
ተናገሩ፥ እንዲህም አድርጉ።

<span id="an_2:13">2:13</span> ምሕረትን ለማያደርግ ምሕረት የሌለበት ፍርድ ይሆናልና፤ ምሕረትም
በፍርድ ላይ ይመካል።

<span id="an_2:14">2:14</span> ወንድሞቼ ሆይ፥ እምነት አለኝ የሚል፥ ሥራ ግን የሌለው ሰው ቢኖር
ምን ይጠቅመዋል? እምነቱስ ሊያድነው ይችላልን?

<span id="an_2:15">2:15</span> ወንድም ወይም እኅት ራቁታቸውን ቢሆኑ የዕለት ምግብንም ቢያጡ፥

<span id="an_2:16">2:16</span> ከእናንተ አንዱም። በደኅና ሂዱ፥ እሳት ሙቁ፥ ጥገቡም ቢላቸው
ለሰውነት ግን የሚያስፈልጉትን ባትሰጡአቸው ምን ይጠቅማቸዋል?

<span id="an_2:17">2:17</span> እንደዚሁም ሥራ የሌለው እምነት ቢኖር በራሱ የሞተ ነው።

<span id="an_2:18">2:18</span> ነገር ግን አንድ ሰው። አንተ እምነት አለህ እኔም ሥራ አለኝ፤
እምነትህን ከሥራህ ለይተህ አሳየኝ፥ እኔም እምነቴን በሥራዬ አሳይሃለሁ ይላል።

<span id="an_2:19">2:19</span> እግዚአብሔር አንድ እንደ ሆነ አንተ ታምናለህ፤ መልካም
ታደርጋለህ፤ አጋንንት ደግሞ ያምናሉ ይንቀጠቀጡማል።

<span id="an_2:20">2:20</span> አንተ ከንቱ ሰው፥ እምነት ከሥራ ተለይቶ የሞተ መሆኑን ልታውቅ
ትወዳለህን?

<span id="an_2:21">2:21</span> አባታችን አብርሃም ልጁን ይስሐቅን በመሠዊያው ባቀረበ ጊዜ በሥራ
የጸደቀ አልነበረምን?

<span id="an_2:22">2:22</span> እምነት ከሥራው ጋር አብሮ ያደርግ እንደ ነበረ፥ በሥራም እምነት
እንደ ተፈጸመ ትመለከታለህን?

<span id="an_2:23">2:23</span> መጽሐፍም። አብርሃምም እግዚአብሔርን አመነ ጽድቅም ሆኖ ተቈጠረለት
ያለው ተፈጸመ፤ የእግዚአብሔርም ወዳጅ ተባለ።

<span id="an_2:24">2:24</span> ሰው በእምነት ብቻ ሳይሆን በሥራ እንዲጸድቅ ታያላችሁ።

<span id="an_2:25">2:25</span> እንደዚሁም ጋለሞታይቱ ረዓብ ደግሞ መልእክተኞቹን ተቀብላ በሌላ
መንገድ በሰደደቻቸው ጊዜ በሥራ አልጸደቀችምን?

<span id="an_2:26">2:26</span> ከነፍስ የተለየ ሥጋ የሞተ እንደ ሆነ እንዲሁ ደግሞ ከሥራ የተለየ
እምነት የሞተ ነው።

<span id="an_3:1">3:1</span> ወንድሞቼ ሆይ፥ ከእናንተ ብዙዎቹ አስተማሪዎች አይሁኑ፥ የባሰውን
ፍርድ እንድንቀበል ታውቃላችሁና።

<span id="an_3:2">3:2</span> ሁላችን በብዙ ነገር እንሰናከላለንና፤ በቃል የማይሰናከል ማንም ቢኖር
እርሱ ሥጋውን ሁሉ ደግሞ ሊገታ የሚችል ፍጹም ሰው ነው።

<span id="an_3:3">3:3</span> እነሆ፥ ፈረሶች ይታዘዙልን ዘንድ ልጓም በአፋቸው ውስጥ እናገባለን፥
ሥጋቸውንም ሁሉ እንመራለን።

<span id="an_3:4">3:4</span> እነሆ፥ መርከቦች ደግሞ ይህን ያህል ታላቅ ቢሆኑ በዐውሎ ነፋስም
ቢነዱ፥ የመሪ ፈቃድ ወደሚወደው ስፍራ እጅግ ታናሽ በሆነ መቅዘፊያ ይመራሉ።

<span id="an_3:5">3:5</span> እንዲሁም አንደበት ደግሞ ትንሽ ብልት ሆኖ በታላላቅ ነገር ይመካል።
እነሆ፥ ትንሽ እሳት እንዴት ያለ ትልቅ ጫካ ያቃጥላል።

<span id="an_3:6">3:6</span> አንደበትም እሳት ነው። አንደበት በብልቶቻችን መካከል ዓመፀኛ ዓለም
ሆኖአል፤ ሥጋን ሁሉ ያሳድፋልና፥ የፍጥረትንም ሩጫ ያቃጥላል፥ በገሃነምም ይቃጠላል።

<span id="an_3:7">3:7</span> የአራዊትና የወፎች የተንቀሳቃሾችና በባሕር ያለ የፍጥረት ወገን ሁሉ
በሰው ይገራል፥ ደግሞ ተገርቶአል፤

<span id="an_3:8">3:8</span> ነገር ግን አንደበትን ሊገራ ማንም ሰው አይችልም፤ የሚገድል መርዝ
የሞላበት ወላዋይ ክፋት ነው።

<span id="an_3:9">3:9</span> በእርሱ ጌታንና አብን እንባርካለን፤ በእርሱም እንደ እግዚአብሔር
ምሳሌ የተፈጠሩትን ሰዎች እንረግማለን፤

<span id="an_3:10">3:10</span> ከአንድ አፍ በረከትና መርገም ይወጣሉ። ወንድሞቼ ሆይ፥ ይህ
እንዲህ ሊሆን አይገባም።

<span id="an_3:11">3:11</span> ምንጭስ ከአንድ አፍ የሚጣፍጥንና የሚመርን ውኃ ያመነጫልን?

<span id="an_3:12">3:12</span> ወንድሞቼ ሆይ፥ በለስ ወይራን ወይስ ወይን በለስን ልታፈራ
ትችላለችን? ከጨው ውኃም ጣፋጭ ውኃ አይወጣም።

<span id="an_3:13">3:13</span> ከእናንተ ጥበበኛና አስተዋይ ማን ነው? በመልካም አንዋዋሩ ስራውን
በጥበብ የዋህነት ያሳይ።

<span id="an_3:14">3:14</span> ነገር ግን መራራ ቅንዓትና አድመኛነት በልባችሁ ቢኖርባችሁ፥
አትመኩ በእውነትም ላይ አትዋሹ።

<span id="an_3:15">3:15</span> ይህ ጥበብ ከላይ የሚወርድ አይደለም፤ ነገር ግን የምድር ነው፥
የሥጋም ነው፥ የአጋንንትም ነው፤

<span id="an_3:16">3:16</span> ቅንዓትና አድመኛነት ባሉበት ስፍራ በዚያ ሁከትና ክፉ ስራ ሁሉ
አሉና።

<span id="an_3:17">3:17</span> ላይኛይቱ ጥበብ ግን በመጀመሪያ ንጽሕት ናት፥ በኋላም ታራቂ፥
ገር፥ እሺ ባይ ምሕረትና በጎ ፍሬ የሞላባት፥ ጥርጥርና ግብዝነት የሌለባት ናት።

<span id="an_3:18">3:18</span> የጽድቅም ፍሬ ሰላምን ለሚያደርጉት ሰዎች በሰላም ይዘራል።

<span id="an_4:1">4:1</span> በእናንተ ዘንድ ጦርና ጠብ ከወዴት ይመጣሉ? በብልቶቻችሁ ውስጥ
ከሚዋጉ ከእነዚህ ከምቾቶቻችሁ አይደሉምን?

<span id="an_4:2">4:2</span> ትመኛላችሁ ለእናንተም አይሆንም፤ ትገድላላችሁ በብርቱም ትፈልጋላችሁ፥
ልታገኙም አትችሉም፤ ትጣላላችሁ ትዋጉማላችሁ ነገር ግን አትለምኑምና ለእናንተ አይሆንም፤

<span id="an_4:3">4:3</span> ትለምናላችሁ፥ በምቾቶቻችሁም ትከፍሉ ዘንድ በክፉ ትለምናላችሁና
አትቀበሉም።

<span id="an_4:4">4:4</span> አመንዝሮች ሆይ፥ ዓለምን መውደድ ለእግዚአብሔር ጥል እንዲሆን
አታውቁምን? እንግዲህ የዓለም ወዳጅ ሊሆን የሚፈቅድ ሁሉ የእግዚአብሔር ጠላት ሆኖአል።

<span id="an_4:5">4:5</span> ወይስ መጽሐፍ። በእኛ ዘንድ ያሳደረው መንፈስ በቅንዓት ይመኛል ያለው
በከንቱ እንደ ተናገረ ይመስላችኋልን?

<span id="an_4:6">4:6</span> ነገር ግን ጸጋን አብልጦ ይሰጣል፤ ስለዚህ። እግዚአብሔር ትዕቢተኞችን
ይቃወማል፥ ለትሑታን ግን ጸጋን ይሰጣል ይላል።

<span id="an_4:7">4:7</span> እንግዲህ ለእግዚአብሔር ተገዙ፤ ዲያብሎስን ግን ተቃወሙ ከእናንተም
ይሸሻል፤

<span id="an_4:8">4:8</span> ወደ እግዚአብሔር ቅረቡ ወደ እናንተም ይቀርባል። እናንተ ኃጢአተኞች፥
እጆቻችሁን አንጹ፤ ሁለት አሳብም ያላችሁ እናንተ፥ ልባችሁን አጥሩ።

<span id="an_4:9">4:9</span> ተጨነቁና እዘኑ አልቅሱም፤ ሳቃችሁ ወደ ኀዘን ደስታችሁም ወደ ትካዜ
ይለወጥ።

<span id="an_4:10">4:10</span> በጌታ ፊት ራሳችሁን አዋርዱ ከፍ ከፍም ያደርጋችኋል።

<span id="an_4:11">4:11</span> ወንድሞች ሆይ፥ እርስ በርሳችሁ አትተማሙ። ወንድሙን የሚያማ
በወንድሙም የሚፈርድ ሕግን ያማል በሕግም ይፈርዳል፤ በሕግም ብትፈርድ ፈራጅ ነህ እንጂ ሕግን አድራጊ አይደለህም።

<span id="an_4:12">4:12</span> ሕግን የሚሰጥና የሚፈርድ አንድ ነው፤ እርሱም ሊያድን ሊያጠፋም
የሚችል ነው፤ በሌላው ግን የምትፈርድ አንተ ማን ነህ?

<span id="an_4:13">4:13</span> አሁንም። ዛሬ ወይም ነገ ወደዚህ ከተማ እንሄዳለን በዚያም ዓመት
እንኖራለን እንነግድማለን እናተርፍማለን የምትሉ እናንተ፥ ተመልከቱ፥ ነገ የሚሆነውን አታውቁምና።

<span id="an_4:14">4:14</span> ሕይወታችሁ ምንድር ነው? ጥቂት ጊዜ ታይቶ ኋላ እንደሚጠፋ
እንፍዋለት ናችሁና።

<span id="an_4:15">4:15</span> በዚህ ፈንታ። ጌታ ቢፈቅድ ብንኖርም ይህን ወይም ያን እናደርጋለን
ማለት ይገባችኋል።

<span id="an_4:16">4:16</span> አሁን ግን በትዕቢታችሁ ትመካላችሁ፤ እንደዚህ ያለ ትምክህት ሁሉ
ክፉ ነው።

<span id="an_4:17">4:17</span> እንግዲህ በጎ ለማድረግ አውቆ ለማይሠራው ኃጢአት ነው።

<span id="an_5:1">5:1</span> አሁንም እናንተ ባለ ጠጎች፥ ስለሚደርስባችሁ ጭንቅ ዋይ ዋይ እያላችሁ
አልቅሱ።

<span id="an_5:2">5:2</span> ሀብታችሁ ተበላሽቶአል፥ ልብሳችሁም በብል ተበልቶአል።

<span id="an_5:3">5:3</span> ወርቃችሁም ብራችሁም ዝጎአል፥ ዝገቱም ምስክር ይሆንባችኋል ሥጋችሁንም
እንደ እሳት ይበላል። ለኋለኛው ቀን መዝገብን አከማችታችኋል።

<span id="an_5:4">5:4</span> እነሆ፥ እርሻችሁን ያጨዱት የሠራተኞች ደመወዝ በእናንተ ተቀምቶ
ይጮኻል፥ የአጫጆችም ድምፅ ወደ ጌታ ፀባዖት ጆሮ ገብቶአል።

<span id="an_5:5">5:5</span> በምድር ላይ ተቀማጥላችኋል በሴሰኝነትም ኖራችኋል፤ ለእርድ ቀን
እንደሚያወፍር ልባችሁን አወፍራችኋል።

<span id="an_5:6">5:6</span> ጻድቁን ኰንናችሁታል ገድላችሁትማል፤ እናንተን አይቃወምም።

<span id="an_5:7">5:7</span> እንግዲህ፥ ወንድሞች ሆይ፥ ጌታ እስኪመጣ ድረስ ታገሡ። እነሆ፥
ገበሬው የፊተኛውንና የኋለኛውን ዝናብ እስኪቀበል ድረስ እርሱን እየታገሠ የከበረውን የመሬት ፍሬ ይጠብቃል።

<span id="an_5:8">5:8</span> እናንተ ደግሞ ታገሡ፥ ልባችሁንም አጽኑ፤ የጌታ መምጣት ቀርቦአልና።

<span id="an_5:9">5:9</span> ወንድሞች ሆይ፥ እንዳይፈረድባችሁ እርስ በርሳችሁ አታጉረምርሙ፤
እነሆ፥ ፈራጅ በደጅ ፊት ቆሞአል።

<span id="an_5:10">5:10</span> ወንድሞች ሆይ፥ የመከራና የትዕግሥት ምሳሌ የሆኑትን በጌታ ስም
የተናገሩትን ነቢያትን ተመልከቱ።

<span id="an_5:11">5:11</span> እነሆ፥ በትዕግሥት የጸኑትን ብፁዓን እንላቸዋለን፤ ኢዮብ እንደ
ታገሠ ሰምታችኋል፥ ጌታም እንደ ፈጸመለት አይታችኋል፤ ጌታ እጅግ የሚምር የሚራራም ነውና።

<span id="an_5:12">5:12</span> ከሁሉም በፊት፥ ወንድሞቼ ሆይ፥ በሰማይ ቢሆን በምድርም ቢሆን
በሌላ መሐላም ቢሆን በምንም አትማሉ፤ ነገር ግን ከፍርድ በታች እንዳትወድቁ ነገራችሁ አዎን ቢሆን አዎን ይሁን፥
አይደለምም ቢሆን አይደለም ይሁን።

<span id="an_5:13">5:13</span> ከእናንተ መከራን የሚቀበል ማንም ቢኖር እርሱ ይጸልይ፤ ደስ
የሚለውም ማንም ቢኖር እርሱ ይዘምር።

<span id="an_5:14">5:14</span> ከእናንተ የታመመ ማንም ቢኖር የቤተ ክርስቲያንን ሽምግሌዎች ወደ
እርሱ ይጥራ፤ በጌታም ስም እርሱን ዘይት ቀብተው ይጸልዩለት።

<span id="an_5:15">5:15</span> የእምነትም ጸሎት ድውዩን ያድናል ጌታም ያስነሣዋል፤ ኃጢአትንም
ሠርቶ እንደ ሆነ ይሰረይለታል።

<span id="an_5:16">5:16</span> እርስ በርሳችሁ በኃጢአታችሁ ተናዘዙ። ትፈወሱም ዘንድ እያንዳንዱ
ስለ ሌላው ይጸልይ፤ የጻድቅ ሰው ጸሎት በሥራዋ እጅግ ኃይል ታደርጋለች።

<span id="an_5:17">5:17</span> ኤልያስ እንደ እኛ የሆነ ሰው ነበረ፥ ዝናብም እንዳይዘንብ አጥብቆ
ጸለየ፥ በምድርም ላይ ሦስት ዓመት ከስድስት ወር አልዘነበም፤ ሁለተኛም ጸለየ፥

<span id="an_5:18">5:18</span> ሰማዩም ዝናብን ሰጠ ምድሪቱም ፍሬዋን አበቀለች።

<span id="an_5:19">5:19</span> ወንድሞቼ ሆይ፥ ከእናንተ ማንም ከእውነት ቢስት አንዱም ቢመልሰው፥

<span id="an_5:20">5:20</span> ኃጢአተኛን ከተሳሳተበት መንገድ የሚመልሰው ነፍሱን ከሞት
እንዲያድን፥ የኃጢአትንም ብዛት እንዲሸፍን ይወቅ።
