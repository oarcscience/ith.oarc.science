### Armenian (Western): James

<span id="an_1:1">1:1</span> Յակոբոս, Աստուծոյ ու Տէր Յիսուս Քրիստոսի
ծառայ, տասներկու տոհմերուն՝ որ ցրուած են ամէն կողմ. ողջո՜յն:

<span id="an_1:2">1:2</span> Եղբայրնե՛րս, բոլորովին ուրախութի՛ւն
համարեցէք՝ երբ զանազան փորձութիւններու մէջ իյնաք,

<span id="an_1:3">1:3</span> գիտնալով թէ ձեր հաւատքին փորձը համբերութիւն
կ՚իրագործէ:

<span id="an_1:4">1:4</span> Բայց համբերութիւնը թող ունենայ իր կատարեալ
գործը, որպէսզի կատարեալ եւ ամբողջ ըլլաք, եւ ոչինչ պակսի ձեզի:

<span id="an_1:5">1:5</span> Իսկ եթէ իմաստութիւն կը պակսի ձեզմէ մէկուն,
թող խնդրէ Աստուծմէ՝ որ բոլորին կու տայ առատապէս՝ առանց կշտամբելու, եւ
պիտի տրուի անոր:

<span id="an_1:6">1:6</span> Բայց հաւատքո՛վ թող խնդրէ, առանց տատամսելու.
որովհետեւ ա՛ն որ կը տատամսի՝ նման է ծովուն հողմակոծեալ ու տատանեալ
ալիքներուն:

<span id="an_1:7">1:7</span> Այդ մարդը թող չկարծէ թէ որեւէ բան պիտի
ստանայ Տէրոջմէն.

<span id="an_1:8">1:8</span> երկմիտ մարդը անհաստատ է իր բոլոր
ճամբաներուն մէջ:

<span id="an_1:9">1:9</span> Նուաստ եղբայրը թող պարծենայ իր
բարձրացումով,

<span id="an_1:10">1:10</span> իսկ հարուստը՝ իր նուաստացումով, որովհետեւ
ինք պիտի անցնի խոտի ծաղիկին պէս.

<span id="an_1:11">1:11</span> քանի որ արեւը կը ծագի այրող տաքութեամբ ու
կը չորցնէ խոտը, անոր ծաղիկը կը թափի, եւ անոր տեսքին վայելչութիւնն ալ կը
կորսուի: Այդպէս ալ հարուստը պիտի թառամի իր ճամբաներուն մէջ:

<span id="an_1:12">1:12</span> Երանի՜ այն մարդուն՝ որ կը տոկայ
փորձութեան. որովհետեւ երբ փորձարկուի եւ ընտիր գտնուի՝ պիտի ստանայ կեանքի
պսակը, որ Տէրը խոստացաւ զինք սիրողներուն:

<span id="an_1:13">1:13</span> Ո՛չ մէկը թող ըսէ՝ երբ կը փորձուի.
«Աստուծմէ՛ կը փորձուիմ». որովհետեւ Աստուած չարէն չի փորձուիր, ո՛չ ալ
ի՛նք կը փորձէ ոեւէ մէկը:

<span id="an_1:14">1:14</span> Սակայն իւրաքանչիւրը կը փորձուի՝
հրապուրուած ու խաբուած իր ցանկութենէն.

<span id="an_1:15">1:15</span> Յետոյ՝ ցանկութիւնը յղանալով մեղք կը
ծնանի, իսկ երբ մեղքը կատարուի՝ մահ կը ծնանի:

<span id="an_1:16">1:16</span> Մի՛ մոլորիք, սիրելի՛ եղբայրներս.

<span id="an_1:17">1:17</span> ամէն բարի նուէր եւ ամէն կատարեալ պարգեւ՝
վերէն է, ու կ՚իջնէ լոյսի Հօրմէն, որուն մէջ բնա՛ւ փոփոխութիւն չկայ, ո՛չ
ալ դառնալու շուք:

<span id="an_1:18">1:18</span> Ան ի՛ր փափաքով ծնաւ մեզ՝ ճշմարտութեան
խօսքով, որպէսզի մենք երախայրիք մը ըլլանք իր արարածներուն:

<span id="an_1:19">1:19</span> Հետեւաբար, սիրելի՛ եղբայրներս, ամէն մարդ
թող արագ ըլլայ՝ լսելու մէջ, դանդաղ՝ խօսելու մէջ, դանդաղ՝ բարկանալու մէջ.

<span id="an_1:20">1:20</span> որովհետեւ մարդուն բարկութիւնը չ՚իրագործեր
Աստուծոյ արդարութիւնը:

<span id="an_1:21">1:21</span> Ուստի, թօթափելով ամէն աղտեղութիւն ու
չարամտութեան աւելցուք, հեզութեա՛մբ ընդունեցէք ձեր մէջ տնկուած խօսքը, որ
կարող է փրկել ձեր անձերը:

<span id="an_1:22">1:22</span> Սակայն խօսքը գործադրողնե՛ր եղէ՛ք, եւ ո՛չ
թէ միայն լսողներ՝ դուք ձեզ խաբելով:

<span id="an_1:23">1:23</span> Արդարեւ, եթէ մէկը միայն լսէ խօսքը ու
չգործադրէ, կը նմանի մարդու մը՝ որ հայելիի մէջ կը դիտէ իր բնական երեսը,

<span id="an_1:24">1:24</span> որովհետեւ կը դիտէ ինքզինք ու կ՚երթայ, եւ
իսկոյն կը մոռնայ թէ ի՛նչպէս էր:

<span id="an_1:25">1:25</span> Բայց ա՛ն որ ակնկառոյց կը նայի ազատութեան
կատարեալ Օրէնքին ու կը կենայ անոր մէջ,- ո՛չ թէ մոռացկոտ լսող մը ըլլալով,
հապա՝ գործը կատարող մը,- երանելի պիտի ըլլայ իր ըրածին մէջ:

<span id="an_1:26">1:26</span> Եթէ մէկը կը կարծէ կրօնասէր ըլլալ ու չի
սանձեր իր լեզուն, այլ կը խաբէ իր սիրտը, անոր կրօնասիրութիւնը փուճ է:

<span id="an_1:27">1:27</span> Աստուծոյ եւ Հօրը առջեւ՝ մաքուր ու անարատ
կրօնասիրութիւնը սա՛ է.- այցելել որբերուն եւ այրիներուն՝ իրենց տառապանքին
մէջ, ու ինքզինք անբիծ պահել աշխարհէն:

<span id="an_2:1">2:1</span> Եղբայրնե՛րս, առանց աչառութեան թող ըլլայ մեր
Տէրոջ՝ Յիսուս Քրիստոսի՝ փառքի Տէրոջ հաւատքը:

<span id="an_2:2">2:2</span> Արդարեւ եթէ ձեր ժողովարանին մէջ մտնէ մարդ
մը՝ ոսկիէ մատանիով ու փայլուն տարազով, եւ մտնէ նաեւ աղքատ մը՝ աղտոտ
տարազով,

<span id="an_2:3">2:3</span> ու նայիք անոր՝ որ փայլուն տարազ կը կրէ, եւ
ըսէք. «Հո՛ս նստէ՝ պատիւով», իսկ աղքատին ըսէք. «Հո՛ն ոտքի կայնէ, կամ հո՛ս
նստէ՝ պատուանդանիս քով»,

<span id="an_2:4">2:4</span> միթէ ձեր մէջ խտրութիւն դրած չէ՞ք ըլլար, ու
չար մտածումներու դատաւոր եղած:

<span id="an_2:5">2:5</span> Լսեցէ՛ք, սիրելի՛ եղբայրներս. միթէ Աստուած
չընտրե՞ց այս աշխարհի աղքատները՝ որոնք հարուստ են հաւատքով, ու ժառանգորդ
արքայութեան՝ որ խոստացաւ զինք սիրողներուն:

<span id="an_2:6">2:6</span> Բայց դուք անպատուեցիք աղքատները:
Հարուստները չե՞ն՝ որ կը հարստահարեն ձեզ ու դատարանները կը քաշեն:

<span id="an_2:7">2:7</span> Անոնք չե՞ն որ կը հայհոյեն այն բարի անունը՝
որով կոչուած էք:

<span id="an_2:8">2:8</span> Ուրեմն՝ եթէ դուք կը գործադրէք արքայական
Օրէնքը՝ Գիրքին ըսածին համաձայն. «Սիրէ՛ ընկերդ քու անձիդ՝՝ պէս», լա՛ւ
կ՚ընէք:

<span id="an_2:9">2:9</span> Բայց եթէ աչառութիւն կ՚ընէք՝ մե՛ղք կը
գործէք, եւ կը կշտամբուիք Օրէնքէն՝ օրինազանցներու պէս.

<span id="an_2:10">2:10</span> քանի որ ո՛վ որ պահէ ամբողջ Օրէնքը սակայն
սայթաքի մէ՛կ բանի մէջ, պարտապան կ՚ըլլայ բոլորին:

<span id="an_2:11">2:11</span> Արդարեւ ա՛ն որ ըսաւ. «Շնութիւն մի՛ ըներ»,
ըսաւ նաեւ. «Մի՛ սպաններ»: Ուստի եթէ շնութիւն չընես բայց սպաննես,
օրինազանց կ՚ըլլաս:

<span id="an_2:12">2:12</span> Ա՛յնպէս խօսեցէք եւ ա՛յնպէս գործեցէք՝
որպէս թէ պիտի դատուիք ազատութեան Օրէնքով:

<span id="an_2:13">2:13</span> Որովհետեւ դատաստանը անողորմ պիտի ըլլայ
անոր՝ որ չէ ողորմած. սակայն ողորմութիւնը կը պարծենայ դատաստանին դիմաց:

<span id="an_2:14">2:14</span> Ի՞նչ օգուտ ունի, եղբայրնե՛րս, եթէ մէկը
ըսէ թէ հաւատք ունի սակայն գործեր չունենայ. միթէ հաւատքը կրնա՞յ փրկել
զինք:

<span id="an_2:15">2:15</span> Եթէ եղբայր մը կամ քոյր մը մերկ ըլլայ եւ
օրուան կերակուրին կարօտ,

<span id="an_2:16">2:16</span> ու ձեզմէ մէկը ըսէ անոնց. «Գացէ՛ք
խաղաղութեամբ, տաքցէ՛ք եւ կշտացէ՛ք», բայց մարմինին հարկաւոր բաները չտայ
անոնց, ի՞նչ օգուտ ունի:

<span id="an_2:17">2:17</span> Ա՛յդպէս ալ հաւատքը՝ եթէ գործեր չունենայ
իրեն հետ՝ մեռած է ինքնիր մէջ:

<span id="an_2:18">2:18</span> Նոյնիսկ մէկը կրնայ ըսել. «Դուն հաւա՛տք
ունիս, ես ալ գործե՛ր ունիմ. ցո՛յց տուր ինծի քու հաւատքդ՝ առանց գործերու,
իսկ ես ցոյց պիտի տամ քեզի իմ հաւատքս՝ գործերո՛վս”:

<span id="an_2:19">2:19</span> Դուն կը հաւատաս թէ Աստուած մէկ է. լա՛ւ
կ՚ընես: Դեւե՛րն ալ կը հաւատան ու կը սարսռան:

<span id="an_2:20">2:20</span> Բայց կ՚ուզե՞ս գիտնալ, ո՛վ սնոտի մարդ, թէ
հաւատքը մեռած է առանց գործերու:

<span id="an_2:21">2:21</span> Աբրահամ՝ մեր հայրը՝ գործերով չարդարացա՞ւ,
երբ իր որդին՝ Իսահակը մատուցանեց զոհասեղանին վրայ:

<span id="an_2:22">2:22</span> Կը տեսնե՞ս թէ հաւատքը գործակից եղաւ անոր
գործերուն, ու գործերո՛վ հաւատքը կատարեալ եղաւ,

<span id="an_2:23">2:23</span> եւ իրագործուեցաւ Գիրքին խօսքը՝ որ կ՚ըսէ.
«Աբրահամ հաւատաց Աստուծոյ, ու ատիկա արդարութիւն սեպուեցաւ անոր, եւ ինք
Աստուծոյ բարեկամ կոչուեցաւ»:

<span id="an_2:24">2:24</span> Կը նկատէ՞ք թէ մարդ կ՚արդարանայ գործերո՛վ,
ու ո՛չ թէ՝ միայն հաւատքով:

<span id="an_2:25">2:25</span> Նմանապէս Ռախաբ պոռնիկն ալ չարդարացա՞ւ
գործերով, երբ ընդունեց պատգամաւորները եւ ուրիշ ճամբայով ուղարկեց զանոնք:

<span id="an_2:26">2:26</span> Արդարեւ ինչպէս մարմինը մեռած է առանց
հոգիի, այդպէս ալ հաւատքը մեռած է առանց գործերու:

<span id="an_3:1">3:1</span> Եղբայրնե՛րս, ձեր մէջ շատ վարդապետներ թող
չըլլան, գիտնալով թէ աւելի խստութեամբ պիտի դատուինք՝՝:

<span id="an_3:2">3:2</span> Որովհետեւ բոլորս ալ կը սայթաքինք շատ
բաներու մէջ. եթէ մէկը չի սայթաքիր խօսքով՝ կատարեալ մարդ է, կարող՝
սանձելու նաեւ ամբողջ մարմինը:

<span id="an_3:3">3:3</span> Ահա՛ ձիերուն բերանը սանձ կը դնենք՝ որպէսզի
հնազանդին մեզի, եւ կը կառավարենք անոնց ամբողջ մարմինը:

<span id="an_3:4">3:4</span> Ահա՛ նաւերն ալ, որ ա՛յդչափ մեծ են ու կը
քշուին սաստիկ հովերէն, կը կառավարուին ամենափոքր ղեկով մը՝ ի՛նչպէս
նաւուղիղը փափաքի:

<span id="an_3:5">3:5</span> Այդպէս ալ լեզուն պզտիկ անդամ մըն է, բայց կը
պարծենայ մեծ բաներով: Ահա՛ ո՜րչափ նիւթ կը վառուի քիչ մը կրակով:

<span id="an_3:6">3:6</span> Լեզո՛ւն ալ կրակ մըն է, անիրաւութեան աշխարհ
մը: Լեզուն ա՛յնպէս դրուած է մեր անդամներուն մէջ, որ կ՚ապականէ ամբողջ
մարմինը ու կը բռնկեցնէ բնութեան շրջանը, իսկ ինք կը բռնկի գեհենէն:

<span id="an_3:7">3:7</span> Գազաններու, թռչուններու, սողուններու եւ
ծովային արարածներու ամէն բնութիւն՝ կը նուաճուի ու նուաճուած է մարդկային
բնութենէն:

<span id="an_3:8">3:8</span> Բայց ո՛չ մէկը կրնայ նուաճել լեզուն. ան
անզուսպ չար է, լի մահաբեր թոյնով:

<span id="an_3:9">3:9</span> Անո՛վ կ՚օրհնաբանենք Աստուած ու Հայրը, եւ
անո՛վ կ՚անիծենք մարդիկ՝ որ ստեղծուած են Աստուծոյ նմանութեամբ:

<span id="an_3:10">3:10</span> Միեւնոյն բերանէն կ՚ելլեն օրհնաբանութիւն
եւ անէծք: Եղբայրնե՛րս, պէտք չէ որ այս ա՛յսպէս ըլլայ:

<span id="an_3:11">3:11</span> Միթէ աղբիւր մը միեւնոյն ակէն կը բխեցնէ՞
անոյշ ու դառն ջուր:

<span id="an_3:12">3:12</span> Եղբայրնե՛րս, թզենին կրնա՞յ ձիթապտուղ տալ,
կամ որթատունկը՝ թուզ: Նմանապէս՝ ո՛չ մէկ աղբիւր կրնայ տալ թէ՛ աղի, թէ՛
անոյշ ջուր:

<span id="an_3:13">3:13</span> Ո՛վ որ իմաստուն ու խելացի է ձեր մէջ, բարի
վարքո՛վ թող ցոյց տայ իր գործերը՝ իմաստութեան հեզութեամբ:

<span id="an_3:14">3:14</span> Բայց եթէ դառն նախանձ եւ հակառակութիւն
ունենաք ձեր սիրտերուն մէջ, ճշմարտութեան դէմ մի՛ պարծենաք ու մի՛ ստէք:

<span id="an_3:15">3:15</span> Այս իմաստութիւնը չ՚իջներ վերէն, հապա՝
երկրային, շնչաւոր եւ դիւական է:

<span id="an_3:16">3:16</span> Որովհետեւ ո՛ւր կայ նախանձ ու
հակառակութիւն, հո՛ն կայ խառնակութիւն եւ ամէն տեսակ չար բան:

<span id="an_3:17">3:17</span> Բայց վերէն եղող իմաստութիւնը՝ նախ անկեղծ
է, յետոյ՝ խաղաղարար, ազնիւ, հլու, լի ողորմութեամբ ու բարի պտուղներով,
անկողմնակալ եւ անկեղծ:

<span id="an_3:18">3:18</span> Արդարութեան պտուղը խաղաղութեամբ կը
սերմանուի անոնց միջոցով՝ որ խաղաղութիւն կ՚ընեն:

<span id="an_4:1">4:1</span> Ուրկէ՞ կը ծագին պատերազմներն ու կռիւները
ձեր մէջ: Միթէ չե՞ն ծագիր ձեր կիրքերէն, որոնք կը մարտնչին ձեր անդամներուն
մէջ:

<span id="an_4:2">4:2</span> Կը ցանկաք, բայց չունիք. կը սպաննէք ու կը
նախանձիք, սակայն չէք կրնար հասնիլ ձեր ուզածին: Կը կռուիք եւ կը
պատերազմիք, բայց չունիք՝ որովհետեւ չէք խնդրեր:

<span id="an_4:3">4:3</span> Կը խնդրէք՝ սակայն չէք ստանար, որովհետեւ կը
խնդրէք չարամտօրէն՝ որպէսզի վատնէք ձեր կիրքերուն համար:

<span id="an_4:4">4:4</span> Շնացողնե՛ր, չէ՞ք գիտեր թէ այս աշխարհի
բարեկամութիւնը՝ Աստուծոյ դէմ թշնամութիւն է. ուրեմն ո՛վ որ կը փափաքի
բարեկամ ըլլալ աշխարհի, ինքզինք թշնամի կ՚ընէ Աստուծոյ:

<span id="an_4:5">4:5</span> Կամ կը կարծէք թէ Գիրքը ընդունա՞յն կ՚ըսէ.
«Նախանձի կը տենչայ մեր մէջ բնակող հոգին»՝՝:

<span id="an_4:6">4:6</span> Սակայն աւելի մեծ շնորհք մը կու տայ, ուստի
կ՚ըսէ. «Աստուած կ՚ընդդիմանայ ամբարտաւաններուն, բայց շնորհք կու տայ
խոնարհներուն»:

<span id="an_4:7">4:7</span> Ուրեմն հպատակեցէ՛ք Աստուծոյ: Դիմադրեցէ՛ք
Չարախօսին, եւ ան պիտի փախչի ձեզմէ:

<span id="an_4:8">4:8</span> Մօտեցէ՛ք Աստուծոյ, եւ ան պիտի մօտենայ ձեզի:
Մաքրեցէ՛ք ձեր ձեռքերը, մեղաւորնե՛ր, ու սրբացուցէ՛ք ձեր սիրտերը,
երկմիտնե՛ր:

<span id="an_4:9">4:9</span> Զգացէ՛ք ձեր թշուառութիւնը, սգացէ՛ք եւ
լացէ՛ք. ձեր խնդուքը թող փոխուի սուգի, ու ձեր ուրախութիւնը՝ տրտմութեան:

<span id="an_4:10">4:10</span> Խոնարհեցէ՛ք Տէրոջ առջեւ, եւ ան պիտի
բարձրացնէ ձեզ:

<span id="an_4:11">4:11</span> Մի՛ բամբասէք զիրար, եղբայրնե՛ր: Ա՛ն որ կը
բամբասէ իր եղբայրը կամ կը դատէ իր եղբայրը, կը բամբասէ Օրէ՛նքը եւ կը դատէ
Օրէ՛նքը: Իսկ եթէ դատես Օրէնքը, ա՛լ Օրէնքը գործադրող չես, հապա՝ դատաւոր:

<span id="an_4:12">4:12</span> Միայն մէ՛կ Օրէնսդիր (ու Դատաւոր) կայ, որ
կարող է փրկել եւ կորսնցնել: Դուն ո՞վ ես՝ որ կը դատես ուրիշը:

<span id="an_4:13">4:13</span> Հիմա գա՛նք ձեզի՝ որ կ՚ըսէք. «Այսօր կամ
վաղը պիտի երթանք այսինչ քաղաքը, հոն տարի մը պիտի կենանք, առեւտուր պիտի
ընենք ու պիտի շահինք»:

<span id="an_4:14">4:14</span> (Դո՛ւք՝ որ չէք գիտեր թէ ի՛նչ պիտի ըլլայ
վաղը:

<span id="an_4:15">4:15</span> Քանի որ ի՞նչ է ձեր կեանքը. արդարեւ շոգի
մըն է, որ քիչ մը ատեն կ՚երեւնայ, յետոյ կ՚աներեւութանայ:) Փոխարէնը ըսելու
էիք. «Եթէ Տէրը կամենայ, պիտի ապրինք եւ պիտի ընենք այս կամ այն բանը»:

<span id="an_4:16">4:16</span> Բայց հիմա կը պարծենաք ձեր
պոռոտախօսութեամբ. այդպիսի ամէն պարծանք չար է:

<span id="an_4:17">4:17</span> Ուրեմն ո՛վ որ գիտէ բարին ընել՝ բայց
չ՚ըներ, ատիկա մեղք է իրեն համար:

<span id="an_5:1">5:1</span> Հիմա գա՛նք ձեզի՝ հարուստնե՛ր. լացէ՛ք եւ
վայեցէ՛ք այն թշուառութիւններուն համար՝ որ պիտի պատահին ձեզի՝՝:

<span id="an_5:2">5:2</span> Ձեր հարստութիւնը փտած է, ու ձեր հանդերձները
ցեցի կեր եղած են:

<span id="an_5:3">5:3</span> Ձեր ոսկին եւ արծաթը ժանգոտած են. անոնց
ժանգը վկայ պիտի ըլլայ ձեզի դէմ ու կրակի պէս պիտի ուտէ ձեր մարմինը: Գանձ
դիզեցիք վերջին օրերուն համար.

<span id="an_5:4">5:4</span> սակայն ահա՛ ձեր արտերը քաղող գործաւորներուն
վարձքը՝ որմէ զրկեցիք զանոնք՝ կ՚աղաղակէ, եւ հնձուորներուն կանչը հասաւ
զօրքերու Տէրոջ ականջներուն:

<span id="an_5:5">5:5</span> Հեշտանքով ու ցոփութեամբ ապրեցաք երկրի վրայ,
կերակրեցիք ձեր սիրտերը՝ որպէս թէ մորթուելու օրուան համար:

<span id="an_5:6">5:6</span> Դատապարտեցի՛ք, սպաննեցի՛ք արդարը՝ որ չէր
ընդդիմանար ձեզի:

<span id="an_5:7">5:7</span> Ուրեմն համբերատա՛ր եղէք, եղբայրնե՛ր, մինչեւ
Տէրոջ գալուստը: Ահա՛ մշակը կը սպասէ երկրի պատուական պտուղին՝ համբերատար
ըլլալով անոր համար, մինչեւ որ ստանայ առաջին անձրեւն ու վերջին անձրեւը:

<span id="an_5:8">5:8</span> Դո՛ւք ալ համբերատար եղէք եւ ամրացուցէ՛ք ձեր
սիրտերը, որովհետեւ Տէրոջ գալուստը կը մօտենայ:

<span id="an_5:9">5:9</span> Մի՛ հառաչէք իրարու դէմ, եղբայրնե՛ր, որպէսզի
չդատապարտուիք: Ահա՛ Դատաւորը կայնած է դրան առջեւ:

<span id="an_5:10">5:10</span> Եղբայրնե՛րս, չարչարանքներու եւ
համբերատարութեան համար օրինա՛կ առէք մարգարէներէն, որոնք խօսեցան Տէրոջ
անունով:

<span id="an_5:11">5:11</span> Ահա՛ երանելի կը կոչենք տոկացողները: Դուք
լսեցի՛ք Յոբի համբերութեան մասին, եւ տեսա՛ք Տէրոջ անոր շնորհած վախճանը,
որովհետեւ Տէրը բազմագութ է, եւ արգահատող:

<span id="an_5:12">5:12</span> Բայց ամէն բանէ առաջ, եղբայրնե՛րս, երդում
մի՛ ընէք, ո՛չ երկինքի վրայ, ո՛չ երկրի վրայ. ո՛չ ալ ուրիշ որեւէ երդում:
Հապա ձեր «այո՛»ն՝ այո՛ ըլլայ, ու «ո՛չ»ը՝ ո՛չ, որպէսզի չիյնաք
դատապարտութեան տակ:

<span id="an_5:13">5:13</span> Ձեզմէ մէկը կը չարչարուի՞. թող աղօթէ: Մէկը
ոգեւորուա՞ծ է. թող սաղմոս երգէ:

<span id="an_5:14">5:14</span> Ձեզմէ մէկը հիւա՞նդ է. թող կանչէ եկեղեցիին
երէցները, եւ անոնք թող աղօթեն իր վրայ՝ իւղով օծելով զինք Տէրոջ անունով:

<span id="an_5:15">5:15</span> Հաւատքով եղած աղօթքը պիտի փրկէ հիւանդը,
եւ Տէրը ոտքի պիտի հանէ զայն. ու եթէ մեղք ալ գործած է, պիտի ներուի անոր:

<span id="an_5:16">5:16</span> Խոստովանեցէ՛ք իրարու ձեր յանցանքները, եւ
աղօթեցէ՛ք իրարու համար, որպէսզի բժշկուիք: Արդարին ներգործող աղերսանքը
շատ ազդեցիկ է:

<span id="an_5:17">5:17</span> Եղիա կիրքերու ենթակայ մարդ մըն էր՝ մեզի
նման: Ան աղօթեց որ անձրեւ չտեղայ, ու երկրի վրայ անձրեւ չտեղաց երեք տարի
եւ վեց ամիս:

<span id="an_5:18">5:18</span> Ապա դարձեալ աղօթեց, ու երկինք անձրեւ
տուաւ եւ երկիր ծաղկեցուց իր պտուղը:

<span id="an_5:19">5:19</span> Եղբայրնե՛ր, եթէ ձեզմէ ոեւէ մէկը մոլորի
ճշմարտութենէն, ու մէկը վերադարձնէ զայն,

<span id="an_5:20">5:20</span> թող գիտնայ թէ ա՛ն՝ որ կը վերադարձնէ
մեղաւորը իր մոլորութեան ճամբայէն, կը փրկէ անձ մը մահէն, ու կը ծածկէ
մեղքերու բազմութիւն մը:
