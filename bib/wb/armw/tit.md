### Armenian (Western): Titus

<span id="an_1:1">1:1</span> Պօղոս, Աստուծոյ ծառայ եւ Յիսուս Քրիստոսի
առաքեալ՝ համաձայն Աստուծոյ ընտրեալներուն հաւատքին ու բարեպաշտութեամբ
եղած ճշմարտութեան գիտակցութեան,

<span id="an_1:2">1:2</span> յաւիտենական կեանքի յոյսով՝ որ անսուտն
Աստուած խոստացաւ դարերու ժամանակներէն առաջ

<span id="an_1:3">1:3</span> - բայց յատուկ ատենին իր խօսքը բացայայտեց
քարոզութեամբ՝ որ վստահուեցաւ ինծի մեր Փրկիչ Աստուծոյ հրամանով - ,

<span id="an_1:4">1:4</span> Տիտոսի՝ հարազատ զաւակիս հասարակաց հաւատքին
համեմատ. շնորհք եւ խաղաղութիւն Հայր Աստուծմէ ու Տէր Յիսուս Քրիստոսէ՝ մեր
Փրկիչէն:

<span id="an_1:5">1:5</span> Ես ձգեցի քեզ Կրետէ, որպէսզի ուղղես բոլոր
պակաս մնացած բաները ու երէցներ նշանակես ամէն քաղաքի մէջ՝ ինչպէս
պատուիրեցի քեզի.

<span id="an_1:6">1:6</span> եթէ մէկը ըլլայ անմեղադրելի, մէկ կնոջ
ամուսին, եւ ունենայ հաւատարիմ զաւակներ, որոնք անառակութեան համար
ամբաստանուած կամ ըմբոստ չըլլան:

<span id="an_1:7">1:7</span> Որովհետեւ եպիսկոպոսը՝ իբր Աստուծոյ տնտես՝
պէտք է ըլլայ անմեղադրելի. ոչ ինքնահաւան, ոչ բարկացող, ոչ գինեմոլ, ոչ
զարնող, ոչ ամօթալի շահախնդրութեան հետամուտ եղող,

<span id="an_1:8">1:8</span> հապա՝ հիւրասէր, բարեսէր, խոհեմ, արդար,
սուրբ, ժուժկալ,

<span id="an_1:9">1:9</span> հաստատ բռնելով վարդապետութեան հաւատարիմ
խօսքը, որպէսզի կարենայ թէ՛ յորդորել ողջամիտ վարդապետութեամբ, թէ՛
կշտամբել հակաճառողները:

<span id="an_1:10">1:10</span> Քանի որ շատ են ըմբոստները, դատարկախօսներն
ու հրապուրողները, մա՛նաւանդ անոնք՝ որ թլփատութենէն են:

<span id="an_1:11">1:11</span> Պէտք է պապանձեցնել զանոնք. անոնք կը
կործանեն ամբողջ տուներ, եւ կը սորվեցնեն անպատեհ բաներ՝ ամօթալի
շահախնդրութեան համար:

<span id="an_1:12">1:12</span> Անոնցմէ մէկը՝ իրենց սեփական մարգարէն՝
ըսած է. «Կրետացիները միշտ ստախօս են, չար գազաններ, դատարկապորտներ»:

<span id="an_1:13">1:13</span> Այս վկայութիւնը ճշմարիտ է. ուստի սաստիկ
կշտամբէ՛ զանոնք, որպէսզի առողջանան հաւատքին մէջ,

<span id="an_1:14">1:14</span> ուշադրութիւն չդարձնելով հրէական
առասպելներուն ու ճշմարտութենէն խոտորող մարդոց պատուէրներուն:

<span id="an_1:15">1:15</span> Մաքուրներուն՝ ամէն բան մաքուր է. բայց
պիղծերուն եւ անհաւատներուն՝ ոչինչ մաքուր է, հապա անոնց միտքն ալ,
խղճմտանքն ալ պղծուած են:

<span id="an_1:16">1:16</span> Կը դաւանին թէ կը ճանչնան Աստուած. բայց
կ՚ուրանան զայն իրենց գործերով, ըլլալով գարշելի եւ անհնազանդ, ու խոտելի՝
ամէն բարի գործի հանդէպ:

<span id="an_2:1">2:1</span> Իսկ դուն խօսէ՛ ինչ որ կը պատշաճի ողջամիտ
վարդապետութեան:

<span id="an_2:2">2:2</span> Տարեցները թող ըլլան զգաստ, պատկառելի,
խոհեմ, եւ ողջամիտ՝ հաւատքով, սիրով ու համբերութեամբ:

<span id="an_2:3">2:3</span> Տարեց կիներն ալ թող վարուին այնպէս՝ ինչպէս
կը վայլէ սրբութեան, եւ ըլլան ո՛չ չարախօս, ո՛չ գինեմոլ, հապա՝ բարի բաներ
սորվեցնող.

<span id="an_2:4">2:4</span> որպէսզի սորվեցնեն դեռահասակ կիներուն՝ սիրել
իրենց ամուսինները, սիրել իրենց զաւակները,

<span id="an_2:5">2:5</span> եւ ըլլալ խոհեմ, մաքրակենցաղ, տնաշէն,
բարեգործ, իրենց ամուսիններուն հպատակ, որ Աստուծոյ խօսքը չհայհոյուի:

<span id="an_2:6">2:6</span> Նոյնպէս յորդորէ՛ երիտասարդները որ ըլլան
խոհեմ:

<span id="an_2:7">2:7</span> Ամէն բանի մէջ դուն քեզ ցոյց տուր իբր տիպար՝
բարի գործերու. վարդապետութեան մէջ՝ ցոյց տուր անեղծութիւն, պատկառանք,

<span id="an_2:8">2:8</span> ողջամիտ եւ անպարսաւելի խօսք, որպէսզի
հակառակողը ամչնայ՝ մեր մասին չարախօսելիք ոչինչ ունենալով:

<span id="an_2:9">2:9</span> Յորդորէ՛ ստրուկները՝ որ հպատակին իրենց
տէրերուն ու հաճելի ըլլան անոնց՝ ամէն բանի մէջ. չհակաճառեն,

<span id="an_2:10">2:10</span> չխորեն, հապա բոլորովին բարի
հաւատարմութիւն ցոյց տան, որպէսզի զարդարեն մեր Փրկիչ Աստուծոյ
վարդապետութիւնը՝ ամէն բանի մէջ:

<span id="an_2:11">2:11</span> Քանի որ Աստուծոյ փրկարար շնորհքը երեւցաւ
բոլոր մարդոց,

<span id="an_2:12">2:12</span> կրթելով մեզ՝ որ ուրանանք ամբարշտութիւնն
ու աշխարհային ցանկութիւնները, եւ ապրինք խոհեմութեամբ, արդարութեամբ ու
բարեպաշտութեամբ՝ այս ներկայ աշխարհին մէջ,

<span id="an_2:13">2:13</span> սպասելով այն երանելի յոյսին եւ մեծ
Աստուծոյ ու մեր Փրկիչ Յիսուս Քրիստոսի փառքին երեւումին:

<span id="an_2:14">2:14</span> Ան ընծայեց ինքզինք մեզի համար, որպէսզի
ազատագրէ մեզ ամէն անօրէնութենէ, եւ մաքրելով՝ ընէ իր սեփական ժողովուրդը,
նախանձախնդիր բարի գործերու:

<span id="an_2:15">2:15</span> Ըսէ՛ այս բաները, յորդորէ՛ ու կշտամբէ՛
ամբողջ հեղինակութեամբ: Ո՛չ մէկը թող արհամարհէ քեզ:

<span id="an_3:1">3:1</span> Յիշեցո՛ւր անոնց՝ որ հպատակին
պետութիւններուն եւ իշխանութիւններուն, անսան պետութեան, ու պատրաստ ըլլան
ամէն բարի գործի.

<span id="an_3:2">3:2</span> ո՛չ մէկուն հայհոյեն, ըլլան ո՛չ թէ կռուազան,
հապա՝ ազնիւ, բոլորովին հեզութիւն ցոյց տալով ամէն մարդու:

<span id="an_3:3">3:3</span> Որովհետեւ մե՛նք ալ ժամանակին անմիտ էինք,
անհնազանդ, մոլորեալ, ու կը ծառայէինք զանազան ցանկութիւններու եւ
հաճոյքներու. կը վարուէինք չարամտութեամբ ու նախանձով, հակակրելի էինք եւ
կ՚ատէինք զիրար:

<span id="an_3:4">3:4</span> Բայց երբ մեր Փրկիչ Աստուծոյ քաղցրութիւնն ու
մարդասիրութիւնը երեւցան,

<span id="an_3:5">3:5</span> ան փրկեց մեզ - ո՛չ թէ մեր ըրած արդարութեան
գործերով, հապա իր ողորմութեան համաձայն - աւազանին վերստին ծնունդով եւ
Սուրբ Հոգիին նորոգութեամբ:

<span id="an_3:6">3:6</span> Զայն ճոխութեամբ թափեց մեր վրայ՝ մեր Փրկիչ
Յիսուս Քրիստոսի միջոցով,

<span id="an_3:7">3:7</span> որպէսզի՝ անոր շնորհքով արդարացած ըլլալով՝
ժառանգորդ դառնանք, յաւիտենական կեանքի յոյսին համաձայն:

<span id="an_3:8">3:8</span> Այս խօսքը վստահելի է, եւ կը փափաքիմ որ
պնդես այս բաներուն վրայ, որպէսզի Աստուծոյ հաւատացողները ճիգ թափեն
մատակարար ըլլալու բարի գործերու. այս բաները բարի են, ու մարդոց օգտակար:

<span id="an_3:9">3:9</span> Բայց հեռո՛ւ կեցիր յիմար վէճերէն,
ազգաբանութիւններէն, հակառակութիւններէն եւ Օրէնքին շուրջ եղած կռիւներէն,
որովհետեւ անօգուտ են ու փուճ:

<span id="an_3:10">3:10</span> Հրաժարէ՛ հերձուածող մարդէն՝ առաջին եւ
երկրորդ խրատէն ետք,

<span id="an_3:11">3:11</span> գիտնալով թէ այդպիսին խոտորած է ու կը
մեղանչէ՝ դատապարտելով ինքզինք:

<span id="an_3:12">3:12</span> Երբ Արտեման կամ Տիւքիկոսը ղրկեմ քեզի,
փութա՛ գալ ինծի՝ Նիկոպոլիս, որովհետեւ որոշեցի հոն ձմերել:

<span id="an_3:13">3:13</span> Փութաջանութեա՛մբ ուղարկէ Զենաս օրինականը
եւ Ապողոսը, հոգ տանելով որ ոչինչ պակսի անոնց:

<span id="an_3:14">3:14</span> Մերիննե՛րն ալ թող սորվին մատակարար ըլլալ
բարի գործերու՝ հարկաւոր կարիքներու համար, որպէսզի անպտուղ չըլլան:

<span id="an_3:15">3:15</span> Կը բարեւեն քեզ բոլոր անոնք՝ որ ինծի հետ
են. բարեւէ՛ հաւատքին մէջ մեզ սիրողները: Շնորհքը ձեր բոլորին հետ: Ամէն:
