### Armenian (Western): Mark

<span id="an_1:1">1:1</span> Աստուծոյ Որդիին՝ Յիսուս Քրիստոսի աւետարանին
սկիզբը:

<span id="an_1:2">1:2</span> Ինչպէս Մարգարէներուն մէջ գրուած է. «Ահա՛ ես
կը ղրկեմ իմ պատգամաւորս քու առջեւէդ. ան պիտի պատրաստէ ճամբադ՝ քու
առջեւդ»:

<span id="an_1:3">1:3</span> «Անապատին մէջ գոչողին ձայնը. “Պատրաստեցէ՛ք
Տէրոջ ճամբան, շտկեցէ՛ք անոր շաւիղները”»:

<span id="an_1:4">1:4</span> Յովհաննէս կը մկրտէր անապատին մէջ եւ կը
քարոզէր ապաշխարութեան մկրտութիւնը՝ մեղքերու ներումին համար:

<span id="an_1:5">1:5</span> Ամբողջ Հրէաստանի երկիրը ու բոլոր
Երուսաղեմացիները կ՚երթային անոր: Բոլորը կը մկրտուէին անկէ Յորդանան գետին
մէջ՝ իրենց մեղքերը խոստովանելով:

<span id="an_1:6">1:6</span> Յովհաննէս հագած էր ուղտի մազէ հագուստ, եւ
իր մէջքը կապած էր կաշիէ գօտի. իր կերակուրը մարախ ու վայրի մեղր էր:

<span id="an_1:7">1:7</span> Ան կը քարոզէր ու կ՚ըսէր. «Ինձմէ հզօրը կու
գայ իմ ետեւէս: Ես արժանի չեմ ծռելու եւ անոր կօշիկներուն կապերը քակելու:

<span id="an_1:8">1:8</span> Արդարեւ ես ջուրո՛վ մկրտեցի ձեզ, բայց ան
Սուրբ Հոգիո՛վ պիտի մկրտէ ձեզ»:

<span id="an_1:9">1:9</span> Այդ օրերը Յիսուս Գալիլեայի Նազարէթէն եկաւ,
ու Յովհաննէսէ մկրտուեցաւ Յորդանանի մէջ:

<span id="an_1:10">1:10</span> Իսկոյն ջուրէն դուրս ելլելով՝ տեսաւ
երկինքը բացուած, եւ Հոգին՝ որ աղաւնիի պէս կ՚իջնէր իր վրայ.

<span id="an_1:11">1:11</span> ու ձայն մը եկաւ երկինքէն՝ որ կ՚ըսէր.
«Դո՛ւն ես իմ սիրելի Որդիս՝ որուն հաճեցայ»:

<span id="an_1:12">1:12</span> Իսկոյն Հոգին անապատը մղեց զայն:

<span id="an_1:13">1:13</span> Քառասուն օր հոն էր՝ անապատին մէջ՝
Սատանայէն փորձուած. գազաններու հետ էր, եւ հրեշտակները կը սպասարկէին
իրեն:

<span id="an_1:14">1:14</span> Յովհաննէսի մատնուելէն ետք Յիսուս՝ Գալիլեա
գալով՝ Աստուծոյ արքայութեան աւետարանը կը քարոզէր

<span id="an_1:15">1:15</span> ու կ՚ըսէր. «Ժամանակը լրացած է, եւ
Աստուծոյ թագաւորութիւնը մօտեցած է. ապաշխարեցէ՛ք, ու հաւատացէ՛ք
աւետարանին»:

<span id="an_1:16">1:16</span> Երբ Գալիլեայի ծովուն եզերքը կը քալէր,
տեսաւ Սիմոնը եւ Սիմոնի եղբայրը՝ Անդրէասը, ծովը ուռկան նետած, որովհետեւ
ձկնորս էին:

<span id="an_1:17">1:17</span> Յիսուս ըսաւ անոնց. «Եկէ՛ք իմ ետեւէս, ու
մարդո՛ց որսորդ պիտի ընեմ ձեզ»:

<span id="an_1:18">1:18</span> Անոնք ալ իսկոյն թողուցին իրենց ուռկանները
եւ հետեւեցան անոր:

<span id="an_1:19">1:19</span> Անկէ քիչ մը յառաջ երթալով՝ տեսաւ Զեբեդեան
Յակոբոսն ու անոր եղբայրը՝ Յովհաննէսը, որոնք նաւուն մէջ կը կարկտնէին
իրենց ուռկանները:

<span id="an_1:20">1:20</span> Իսկոյն կանչեց զանոնք. անոնք ալ գացին անոր
ետեւէն, նաւուն մէջ ձգելով իրենց հայրը՝ Զեբեդէոսը վարձկաններուն հետ:

<span id="an_1:21">1:21</span> Մտան Կափառնայում, եւ իսկոյն Շաբաթ օրը
ժողովարանը մտնելով՝ կը սորվեցնէր:

<span id="an_1:22">1:22</span> Անոնք կ՚ապշէին անոր ուսուցումին վրայ,
որովհետեւ անոնց կը սորվեցնէր իշխանութիւն ունեցողի մը պէս, ո՛չ թէ
դպիրներուն պէս:

<span id="an_1:23">1:23</span> Անոնց ժողովարանին մէջ մարդ մը կար՝
անմաքուր ոգիով: Ան աղաղակեց.

<span id="an_1:24">1:24</span> «Թո՛ղ մեզ. դուն ի՞նչ ունիս մեզի հետ,
Յիսո՛ւս Նազովրեցի. միթէ մեզ կորսնցնելո՞ւ եկար: Գիտեմ թէ ո՛վ ես՝ Աստուծոյ
Սուրբը»:

<span id="an_1:25">1:25</span> Յիսուս զայն սաստեց՝ ըսելով. «Պապանձէ՛ ու
ելի՛ր ատկէ»:

<span id="an_1:26">1:26</span> Անմաքուր ոգին սաստիկ ցնցեց զայն,
բարձրաձայն աղաղակեց եւ ելաւ անկէ:

<span id="an_1:27">1:27</span> Բոլորը այլայլեցան, ա՛յնքան՝ որ կը
հարցնէին իրարու. «Այս ի՞նչ է. ի՞նչ նոր ուսուցում է ասիկա, որովհետեւ
անմաքուր ոգիներո՛ւն ալ կը հրամայէ իշխանութեամբ, ու կը հնազանդին իրեն»:

<span id="an_1:28">1:28</span> Եւ իսկոյն իր համբաւը տարածուեցաւ
Գալիլեայի ամբողջ շրջակայքը:

<span id="an_1:29">1:29</span> Իսկոյն ժողովարանէն ելլելով՝ Յակոբոսի ու
Յովհաննէսի հետ մտաւ Սիմոնի եւ Անդրէասի տունը:

<span id="an_1:30">1:30</span> Սիմոնի զոքանչը պառկած էր՝ տենդով
հիւանդացած. իսկոյն անոր մասին ըսին իրեն:

<span id="an_1:31">1:31</span> Յիսուս մօտեցաւ, բռնեց անոր ձեռքէն ու ոտքի
հանեց զայն. իսկոյն տենդը թողուց զայն, եւ կը սպասարկէր անոնց:

<span id="an_1:32">1:32</span> Երբ իրիկուն եղաւ, արեւին մայր մտած ատենը՝
բոլոր ախտաւորներն ու դիւահարները իրեն կը բերէին:

<span id="an_1:33">1:33</span> Ամբողջ քաղաքը դուռը հաւաքուած էր:

<span id="an_1:34">1:34</span> Շատ հիւանդներ՝ զանազան ախտերէ բուժեց, եւ
շատ դեւեր դուրս հանեց: Դեւերուն թոյլ չէր տար որ խօսին, որովհետեւ կը
ճանչնային զինք:

<span id="an_1:35">1:35</span> Առտուն՝ արշալոյսէն շատ առաջ՝ կանգնեցաւ,
դուրս ելաւ, գնաց ամայի տեղ մը, ու հոն աղօթեց:

<span id="an_1:36">1:36</span> Սիմոն եւ իրեն հետ եղողները հետապնդեցին
զայն:

<span id="an_1:37">1:37</span> Երբ գտան զայն՝ ըսին անոր. «Բոլորը կը
փնտռեն քեզ»:

<span id="an_1:38">1:38</span> Ըսաւ անոնց. «Եկէ՛ք երթանք մերձակայ
գիւղաքաղաքները, որպէսզի հոն ալ քարոզեմ, որովհետեւ եկած եմ ա՛յս
նպատակով»:

<span id="an_1:39">1:39</span> Եւ անոնց ժողովարաններուն մէջ կը քարոզէր՝
ամբողջ Գալիլեայի մէջ, ու դեւեր կը հանէր:

<span id="an_1:40">1:40</span> Բորոտ մըն ալ եկաւ. կ՚աղաչէր, կը ծնրադրէր
ու կ՚ըսէր անոր. «Եթէ ուզես՝ կրնա՛ս զիս մաքրել»:

<span id="an_1:41">1:41</span> Յիսուս գթալով՝ երկարեց ձեռքը, դպաւ անոր
եւ ըսաւ. «Կ՚ուզե՛մ, մաքրուէ՛»:

<span id="an_1:42">1:42</span> Երբ ասիկա ըսաւ անոր, իսկոյն բորոտութիւնը
գնաց անկէ, ու մաքրուեցաւ:

<span id="an_1:43">1:43</span> Իսկ ինք ազդարարեց անոր եւ իսկոյն
ուղարկեց՝ ըսելով անոր.

<span id="an_1:44">1:44</span> «Զգուշացի՛ր որ ո՛չ մէկուն բան մը ըսես.
հապա գնա՛, ցո՛յց տուր քեզ քահանային, ու մատուցանէ՛ քու մաքրուելուդ
ընծան՝ որ Մովսէս պատուիրեց, իբր վկայութիւն անոնց»:

<span id="an_1:45">1:45</span> Բայց ան դուրս ելլելով՝ սկսաւ շատ
հրապարակել եւ բանը տարածել, այնպէս որ ա՛լ ինք չէր կրնար բացայայտօրէն
քաղաք մը մտնել, հապա դուրսը՝ ամայի տեղեր էր. սակայն ամէն կողմէ կու գային
իրեն:

<span id="an_2:1">2:1</span> Դարձեալ Կափառնայում մտաւ քանի մը օր ետք, եւ
լսուեցաւ թէ ան տան մը մէջ է:

<span id="an_2:2">2:2</span> Իսկոյն շատ մարդիկ հաւաքուեցան, այնքան՝ որ
ա՛լ տեղ չկար, ո՛չ իսկ դրան շուրջ, ու կը քարոզէր անոնց Աստուծոյ խօսքը:

<span id="an_2:3">2:3</span> Մարդիկ եկան իրեն՝ բերելով չորս հոգիով
փոխադրուած անդամալոյծ մը.

<span id="an_2:4">2:4</span> բայց տեսնելով որ բազմութենէն չէին կրնար
մօտենալ անոր, քակեցին տան տանիքը՝ ուր կը գտնուէր Յիսուս, ու ծակ մը
բանալով՝ վար իջեցուցին մահիճը, որուն վրայ պառկած էր անդամալոյծը:

<span id="an_2:5">2:5</span> Յիսուս տեսնելով անոնց հաւատքը՝ ըսաւ
անդամալոյծին. «Որդեա՛կ, մեղքերդ ներուած են քեզի»:

<span id="an_2:6">2:6</span> Քանի մը դպիրներ, որ հոն նստած էին, իրենց
սիրտերուն մէջ կը մտածէին.

<span id="an_2:7">2:7</span> «Ինչո՞ւ այս մարդը կ՚ըսէ այսպիսի
հայհոյութիւններ. ո՞վ կրնայ մեղքերը ներել՝ բացի Աստուծմէ՝՝»:

<span id="an_2:8">2:8</span> Յիսուս իսկոյն ըմբռնելով իր հոգիին մէջ՝ որ
անոնք այդպէս կը մտածեն իրենք իրենց մէջ, ըսաւ անոնց. «Ինչո՞ւ այդպէս կը
մտածէք ձեր սիրտերուն մէջ:

<span id="an_2:9">2:9</span> Ո՞րը աւելի դիւրին է, “մեղքերդ ներուած են
քեզի” ըսե՞լը անդամալոյծին, թէ՝ “ոտքի՛ ելիր, ա՛ռ մահիճդ ու քալէ՛” ըսելը:

<span id="an_2:10">2:10</span> Բայց որպէսզի գիտնաք թէ մարդու Որդին
իշխանութիւն ունի երկրի վրայ մեղքերը ներելու, (ըսաւ անդամալոյծին.)

<span id="an_2:11">2:11</span> “Քեզի՛ կ՚ըսեմ. "Ոտքի՛ ելիր, ա՛ռ մահիճդ ու
գնա՛ տունդ"”»:

<span id="an_2:12">2:12</span> Ան ալ իսկոյն ոտքի ելաւ, եւ առնելով իր
մահիճը՝ դուրս ելաւ բոլորին առջեւէն, այնպէս որ բոլորը զմայլեցան,
փառաբանեցին Աստուած ու ըսին. «Այսպիսի բան ամե՛նեւին տեսած չէինք»:

<span id="an_2:13">2:13</span> Դարձեալ գնաց ծովեզերք: Ամբողջ բազմութիւնը
իրեն կու գար, եւ անոնց կը սորվեցնէր:

<span id="an_2:14">2:14</span> Երբ կ՚անցնէր՝ տեսաւ Ղեւի Ալփէոսեանը, որ
մաքս ընդունելու տեղը նստած էր, եւ ըսաւ անոր. «Հետեւէ՛ ինծի»: Ան ալ
կանգնեցաւ ու հետեւեցաւ իրեն:

<span id="an_2:15">2:15</span> Երբ սեղան նստած՝՝ էր անոր տան մէջ, շատ
մաքսաւորներ ու մեղաւորներ ալ սեղան նստած էին Յիսուսի եւ անոր
աշակերտներուն հետ. որովհետեւ շատե՛ր կը հետեւէին անոր:

<span id="an_2:16">2:16</span> Երբ դպիրներն ու Փարիսեցիները տեսան թէ ան
կ՚ուտէ մաքսաւորներու եւ մեղաւորներու հետ, ըսին անոր աշակերտներուն.
«Ինչո՞ւ կ՚ուտէ ու կը խմէ մաքսաւորներու եւ մեղաւորներու հետ»:

<span id="an_2:17">2:17</span> Երբ Յիսուս լսեց՝ ըսաւ անոնց. «Ո՛չ թէ
առողջներուն բժիշկ պէտք է, հապա՝ հիւանդներուն: Ես եկայ ո՛չ թէ արդարները
կանչելու, հապա՝ մեղաւորները»:

<span id="an_2:18">2:18</span> Յովհաննէսի ու Փարիսեցիներուն աշակերտները
ծոմ կը պահէին: Ուստի եկան եւ ըսին անոր. «Ինչո՞ւ Յովհաննէսի ու
Փարիսեցիներուն աշակերտները ծոմ կը պահեն, բայց քու աշակերտներդ ծոմ չեն
պահեր»:

<span id="an_2:19">2:19</span> Յիսուս ըսաւ անոնց. «Միթէ կարելի՞ է՝ որ
հարսնեւորները ծոմ պահեն, մինչ փեսան իրենց հետ է: Այնքան ատեն որ փեսան
իրենց հետ է՝ կարելի չէ որ ծոմ պահեն:

<span id="an_2:20">2:20</span> Բայց օրերը պիտի գան՝ երբ փեսան պիտի
վերցուի իրենցմէ. ապա ա՛յդ օրը ծոմ պիտի պահեն:

<span id="an_2:21">2:21</span> Ո՛չ մէկը կը կարէ նոր լաթի կտոր մը հին
հանդերձի վրայ. որովհետեւ այդ նորը՝ որ անոր լրութեան համար դրուած է, կը
քաշէ հինը, ու պատռուածքը աւելի գէշ կ՚ըլլայ:

<span id="an_2:22">2:22</span> Ո՛չ ալ մէկը կը դնէ նոր գինին հին տիկերու
մէջ. քանի որ նոր գինին կը պատռէ տիկերը, գինին կը թափի, եւ տիկերը կը
կորսուին: Հապա նոր գինին դրուելու է նո՛ր տիկերու մէջ»:

<span id="an_2:23">2:23</span> Շաբաթ օր մը ինք կ՚անցնէր արտերու մէջէն,
եւ իր աշակերտները՝ մինչ կ՚երթային՝ սկսան ցորենի հասկեր փրցնել:

<span id="an_2:24">2:24</span> Փարիսեցիները ըսին իրեն. «Նայէ՛, ի՛նչ
կ՚ընեն Շաբաթ օրը, որ արտօնուած չէ»:

<span id="an_2:25">2:25</span> Ինք ալ ըսաւ անոնց. «Բնաւ չէ՞ք կարդացեր
Դաւիթի ըրածը՝ երբ պէտք ունեցաւ եւ անօթեցաւ, ինք ու իրեն հետ եղողները:

<span id="an_2:26">2:26</span> Ի՛նչպէս մտաւ Աստուծոյ տունը՝ Աբիաթար
քահանայապետին օրով, եւ կերաւ առաջադրութեան հացերը,- որ ո՛չ մէկուն
արտօնուած էր ուտել՝ բացի քահանաներէն,- ու տուաւ իրեն հետ եղողներուն»:

<span id="an_2:27">2:27</span> Եւ աւելցուց. «Շաբաթը եղաւ մարդո՛ւն համար,
ո՛չ թէ մարդը՝ Շաբաթին համար:

<span id="an_2:28">2:28</span> Հետեւաբար մարդու Որդին տէրն է նաեւ
Շաբաթին»:

<span id="an_3:1">3:1</span> Դարձեալ ժողովարանը մտաւ, ուր մարդ մը կար՝
որուն ձեռքը չորցած էր:

<span id="an_3:2">3:2</span> Անոնք կը հսկէին անոր վրայ՝ տեսնելու թէ
արդեօք պիտի բուժէ՛ զայն Շաբաթ օրը, որպէսզի ամբաստանեն զինք:

<span id="an_3:3">3:3</span> Ինք ալ ըսաւ ձեռքը չորցած մարդուն. «Ելի՛ր,
կայնէ՛ մէջտեղը»:

<span id="an_3:4">3:4</span> Յետոյ ըսաւ անոնց. «Շաբաթ օրը բարի՞ք ընել
արտօնուած է՝ թէ չարիք ընել, անձ մը փրկե՞լ՝ թէ սպաննել»: Անոնք լուռ
կեցան:

<span id="an_3:5">3:5</span> Յիսուս բարկութեամբ շուրջը նայեցաւ՝ անոնց
վրայ, տրտմելով անոնց սիրտին կուրութեան համար, ապա ըսաւ այդ մարդուն.
«Երկարէ՛ ձեռքդ»: Ան ալ երկարեց, եւ անոր ձեռքը առողջացաւ:

<span id="an_3:6">3:6</span> Փարիսեցիները՝ իսկոյն դուրս ելլելով
Հերովդէսեաններուն հետ՝ խորհրդակցեցան անոր դէմ, թէ ի՛նչպէս կորսնցնեն
զայն:

<span id="an_3:7">3:7</span> Յիսուս իր աշակերտներուն հետ մեկնեցաւ
ծովեզերքը, ու մեծ բազմութիւն մը հետեւեցաւ անոր՝ Գալիլեայէն եւ
Հրէաստանէն,

<span id="an_3:8">3:8</span> Երուսաղէմէն, Եդովմէն ու Յորդանանի միւս
կողմէն. նաեւ անոնք՝ որ Տիւրոսի եւ Սիդոնի շրջակայ տեղերէն էին, մեծ
բազմութեամբ, երբ լսեցին ինչ որ կ՚ընէր՝ եկան իրեն:

<span id="an_3:9">3:9</span> Յիսուս ըսաւ իր աշակերտներուն որ նաւակ մը
տրամադրուի իրեն՝ բազմութեան պատճառով, որպէսզի չսեղմեն զինք:

<span id="an_3:10">3:10</span> Որովհետեւ բուժեց շատերը, այնպէս որ
տանջանք ունեցողներ կը խռնուէին շուրջը՝ որպէսզի դպչին իրեն:

<span id="an_3:11">3:11</span> Երբ անմաքուր ոգիները տեսնէին զինք,
կ՚իյնային իր առջեւ ու կ՚աղաղակէին.

<span id="an_3:12">3:12</span> «Դո՛ւն ես Աստուծոյ Որդին»: Իսկ ինք սաստիկ
կը հրահանգէր անոնց՝ որ չյայտնեն զինք:

<span id="an_3:13">3:13</span> Ապա լեռը ելաւ եւ իրեն կանչեց իր
ուզածները, ու քովը գացին:

<span id="an_3:14">3:14</span> Եւ ընտրեց տասներկու հոգի, որպէսզի ըլլան
իրեն հետ, ու ղրկէ զանոնք՝ քարոզելու,

<span id="an_3:15">3:15</span> նաեւ իշխանութիւն ունենալու՝ ախտերը
բուժելու եւ դեւերը դուրս հանելու համար.-

<span id="an_3:16">3:16</span> Սիմոնը՝ որ Պետրոս անուանեց,

<span id="an_3:17">3:17</span> Զեբեդեան Յակոբոսն ու Յակոբոսի եղբայրը՝
Յովհաննէսը, որոնք Բաներեգէս անուանեց, որ ըսել է. «Որոտումի որդիներ»,

<span id="an_3:18">3:18</span> Անդրէասը, Փիլիպպոսը, Բարթողոմէոսը,
Մատթէոսը, Թովմասը, Ալփէոսեան Յակոբոսը, Թադէոսը, Սիմոն Կանանացին

<span id="an_3:19">3:19</span> եւ Յուդա Իսկարիովտացին, որ մատնեց զայն:

<span id="an_3:20">3:20</span> Տուն գացին, ու դարձեալ բազմութիւնը
համախմբուեցաւ, այնպէս որ չկրցան նոյնիսկ հաց ուտել:

<span id="an_3:21">3:21</span> Երբ իրենները լսեցին՝ գացին որ բռնեն զինք,
քանի որ կ՚ըսէին. «Ցնորա՛ծ է»:

<span id="an_3:22">3:22</span> Երուսաղէմէն իջած դպիրներն ալ կ՚ըսէին.
«Անոր մէջ Բէեղզեբուղ կայ, եւ դեւերուն իշխանո՛վ կը հանէ դեւերը»:

<span id="an_3:23">3:23</span> Յիսուս կանչելով զանոնք՝ առակներով խօսեցաւ
անոնց. «Սատանան ի՞նչպէս կրնայ դուրս հանել Սատանան:

<span id="an_3:24">3:24</span> Եթէ թագաւորութիւն մը բաժնուի ինքնիր դէմ՝
այդ թագաւորութիւնը չի կրնար կենալ:

<span id="an_3:25">3:25</span> Եթէ տուն մը բաժնուի ինքնիր դէմ, այդ տունը
չի կրնար կենալ:

<span id="an_3:26">3:26</span> Եթէ Սատանան կանգնած ու բաժնուած է ինքնիր
դէմ՝ ան չի կրնար կենալ, հապա անոր վախճանը հասած է:

<span id="an_3:27">3:27</span> Ո՛չ մէկը կրնայ մտնել ուժեղ մարդու մը
տունը եւ յափշտակել անոր կարասիները, եթէ նախ չկապէ ուժեղ մարդը, եւ այ՛ն
ատեն կողոպտէ անոր տունը:

<span id="an_3:28">3:28</span> Ճշմա՛րտապէս կը յայտարարեմ ձեզի. “Մարդոց
որդիներուն պիտի ներուի ամէն մեղք, նաեւ հայհոյութիւնները՝ ո՛րքան ալ
հայհոյեն:

<span id="an_3:29">3:29</span> Բայց ո՛վ որ հայհոյէ Սուրբ Հոգիին դէմ,
յաւիտեա՛ն ներում պիտի չունենայ, հապա արժանի պիտի ըլլայ յաւիտենական
դատապարտութեան”»:

<span id="an_3:30">3:30</span> Որովհետեւ կ՚ըսէին. «Անմաքուր ոգի կայ անոր
մէջ»:

<span id="an_3:31">3:31</span> Յետոյ անոր մայրն ու եղբայրները եկան, եւ
դուրսը կայնելով՝ մարդ ղրկեցին ու կանչեցին զայն.

<span id="an_3:32">3:32</span> բազմութիւնն ալ նստած էր անոր շուրջը: Երբ

<span id="an_5:9">5:9</span> Հարցուց անոր. «Ի՞նչ է անունդ»: Ան ալ
պատասխանեց. «Անունս Լեգէոն է, որովհետեւ բազմաթիւ ենք»:

<span id="an_5:10">5:10</span> Շատ կ՚աղաչէին անոր՝ որ այդ երկրէն դուրս
չղրկէ զիրենք:

<span id="an_5:11">5:11</span> Հոն, լերան մօտ, խոզերու մեծ երամակ մը
կար՝ որ կ՚արածէր:

<span id="an_5:12">5:12</span> Բոլոր դեւերը աղաչեցին անոր եւ ըսին. «Մեզ
խոզերո՛ւն ղրկէ, որպէսզի անոնց մէջ մտնենք»:

<span id="an_5:13">5:13</span> Յիսուս իսկոյն արտօնեց անոնց: Երբ անմաքուր
ոգիները դուրս ելան ու խոզերուն մէջ մտան, երամակը գահավէժ տեղէն ծովը
վազեց (երկու հազարի չափ կային), եւ ծովուն մէջ խեղդուեցան:

<span id="an_5:14">5:14</span> Խոզարածները փախան, ու պատմեցին քաղաքին եւ
արտերուն մէջ, ու մարդիկ դուրս ելան՝ տեսնելու թէ ի՛նչ էր պատահածը:

<span id="an_5:15">5:15</span> Եկան Յիսուսի քով, տեսան լեգէոն ունեցող
դիւահարը, որ նստած էր՝ հագուած եւ սթափած, ու վախցան:

<span id="an_5:16">5:16</span> Անոնք որ տեսեր էին՝ պատմեցին իրենց թէ
ի՛նչ պատահեցաւ դիւահարին, նաեւ խոզերուն մասին.

<span id="an_5:17">5:17</span> իրենք ալ սկսան աղաչել անոր, որպէսզի իրենց
հողամասէն երթայ:

<span id="an_5:18">5:18</span> Երբ ան նաւ մտաւ, դիւահարը կ՚աղաչէր անոր՝
որ ըլլայ անոր հետ:

<span id="an_5:19">5:19</span> Բայց Յիսուս թոյլ չտուաւ անոր, հապա ըսաւ
անոր. «Գնա՛ տունդ՝ քուկիններուդ քով, ու պատմէ՛ անոնց ամէն ինչ որ Տէրը
ըրաւ քեզի, եւ ի՛նչպէս ողորմեցաւ քեզի»:

<span id="an_5:20">5:20</span> Ան ալ գնաց, եւ սկսաւ Դեկապոլիսի մէջ
հրապարակել ամէն ինչ որ Յիսուս ըրաւ իրեն. ու բոլորը կը զարմանային:

<span id="an_5:21">5:21</span> Երբ Յիսուս դարձեալ նաւով անցաւ ծովուն
միւս եզերքը, մեծ բազմութիւն մը հաւաքուեցաւ անոր քով, եւ ինք ծովեզերքն
էր:

<span id="an_5:22">5:22</span> Ժողովարանի պետերէն մէկը եկաւ, որուն
անունը Յայրոս էր, ու երբ տեսաւ զայն՝ ինկաւ անոր ոտքը,

<span id="an_5:23">5:23</span> շատ աղաչեց անոր եւ ըսաւ. «Աղջիկս մեռնելու
մօտ է. եկո՛ւր, դի՛ր ձեռքդ անոր վրայ՝ որպէսզի բուժուի, ու պիտի ապրի»:

<span id="an_5:24">5:24</span> Ան ալ գնաց անոր հետ: Մեծ բազմութիւն մը
իրեն կը հետեւէր, եւ կը սեղմէր զինք:

<span id="an_5:25">5:25</span> Կին մը՝ որ արիւնահոսութիւն ունէր
տասներկու տարիէ ի վեր,

<span id="an_5:26">5:26</span> շատ չարչարանք կրած էր բազմաթիւ բժիշկներէ,
եւ իր ամբողջ ունեցածը ծախսած էր բայց բնա՛ւ չէր օգտուած, այլ փոխարէնը
վիճակը կը վատթարանար:

<span id="an_5:27">5:27</span> Երբ լսեց Յիսուսի մասին, ետեւէն եկաւ՝
բազմութեան մէջ, ու դպաւ անոր հանդերձին.

<span id="an_5:28">5:28</span> քանի որ կ՚ըսէր. «Եթէ միա՛յն անոր
հանդերձներուն դպչիմ՝ պիտի բժշկուիմ»:

<span id="an_5:29">5:29</span> Իսկոյն իր արիւնին աղբիւրը ցամքեցաւ, եւ իր
մարմինին մէջ գիտցաւ թէ տանջանքէն բժշկուեցաւ:

<span id="an_5:30">5:30</span> Յիսուս իսկոյն գիտցաւ ինքնիր մէջ թէ
զօրութիւն մը ելաւ իրմէ դուրս, դարձաւ բազմութեան ու ըսաւ. «Ո՞վ դպաւ իմ
հանդերձներուս»:

<span id="an_5:31">5:31</span> Իր աշակերտները ըսին իրեն. «Կը տեսնե՛ս թէ
բազմութիւնը կը սեղմէ քեզ, եւ կ՚ըսես. “Ո՞վ դպաւ ինծի”»:

<span id="an_5:32">5:32</span> Իր շուրջը կը նայէր՝ որպէսզի տեսնէ այս
բանը ընողը:

<span id="an_5:33">5:33</span> Կինը վախցաւ ու դողաց, որովհետեւ գիտէր թէ
ի՛նչ պատահեցաւ իրեն. եկաւ, ինկաւ անոր առջեւ, եւ պատմեց անոր ամբողջ
ճշմարտութիւնը:

<span id="an_5:34">5:34</span> Յիսուս ըսաւ անոր. «Աղջի՛կ, հաւատքդ բժշկեց
քեզ. գնա՛ խաղաղութեամբ, ու բժշկուա՛ծ եղիր քու տանջանքէդ»:

<span id="an_5:35">5:35</span> Երբ ինք դեռ կը խօսէր, ժողովարանի պետին
տունէն ոմանք եկան եւ ըսին. «Աղջիկդ մեռաւ, ա՛լ ինչո՞ւ կ՚անհանգստացնես
վարդապետը»:

<span id="an_5:36">5:36</span> Իսկ Յիսուս, երբ լսեց ըսուած խօսքը, իսկոյն
ըսաւ ժողովարանի պետին. «Մի՛ վախնար, միա՛յն հաւատա»:

<span id="an_5:37">5:37</span> Ու ո՛չ մէկուն թոյլ տուաւ որ իրեն ուղեկցի,
բայց միայն Պետրոսի, Յակոբոսի եւ Յակոբոսի եղբօր՝ Յովհաննէսի:

<span id="an_5:38">5:38</span> Ժողովարանի պետին տունը հասնելով՝ նշմարեց
աղմուկը, նաեւ անոնք՝ որ շատ կու լային ու կը հեծեծէին:

<span id="an_5:39">5:39</span> Ներս մտնելով՝ ըսաւ անոնց. «Ինչո՞ւ իրար
անցած էք եւ կու լաք. մանուկը մեռած չէ, հապա կը քնանայ»:

<span id="an_5:40">5:40</span> Անոնք ալ զինք ծաղրեցին. բայց ինք բոլորը
դուրս հանելով՝ իրեն հետ առաւ մանուկին հայրն ու մայրը, եւ իրեն հետ
եղողները, ու մտաւ հոն՝ ուր մանուկը պառկած էր:

<span id="an_5:41">5:41</span> Մանուկին ձեռքէն բռնելով՝ ըսաւ անոր.
«Տալիթա՛, կումի՛», որ կը թարգմանուի. «Աղջի՛կ, (քեզի՛ կ՚ըսեմ,) ոտքի՛
ելիր»:

<span id="an_5:42">5:42</span> Իսկոյն աղջիկը կանգնեցաւ եւ քալեց,
որովհետեւ տասներկու տարեկան էր. ու տեսնողները հիացումէն զմայլեցան:

<span id="an_5:43">5:43</span> Յիսուս սաստիկ պատուիրեց անոնց՝ որ ո՛չ
մէկը գիտնայ այս բանը, եւ ըսաւ՝ որ ուտելիք տան անոր:

<span id="an_6:1">6:1</span> Անկէ մեկնելով՝ գնաց իր բնագաւառը, եւ իր
աշակերտները հետեւեցան անոր:

<span id="an_6:2">6:2</span> Երբ Շաբաթ օրը հասաւ, սկսաւ ժողովարանին մէջ
սորվեցնել: Շատեր լսելով կ՚ապշէին ու կ՚ըսէին. «Ասիկա ուրկէ՞ ունի այս
բաները. այս ի՞նչ իմաստութիւն է՝ իրեն տրուած, որ այսպիսի հրաշքներ կը
գործուին իր ձեռքով:

<span id="an_6:3">6:3</span> Ասիկա հիւսնը չէ՞, Մարիամի որդին, եւ
Յակոբոսի, Յովսէսի, Յուդայի ու Սիմոնի եղբայրը. եւ իր քոյրերը հոս՝ մեր քով
չե՞ն»: Ուստի անոր պատճառով կը գայթակղէին:

<span id="an_6:4">6:4</span> Սակայն Յիսուս ըսաւ անոնց. «Մարգարէ մը առանց
պատիւի չէ, բացի իր բնագաւառին, իր ազգականներուն եւ իր տան մէջ»:

<span id="an_6:5">6:5</span> Հոն չէր կրնար հրաշք գործել. բայց միայն քանի
մը հիւանդի վրայ ձեռք դրաւ ու բուժեց զանոնք:

<span id="an_6:6">6:6</span> Եւ անոնց անհաւատութեան վրայ կը զարմանար, ու
շրջակայ գիւղերը երթալով՝ կը սորվեցնէր:

<span id="an_6:7">6:7</span> Իրեն կանչելով տասներկուքը՝ երկու-երկու
ղրկեց զանոնք, եւ իշխանութիւն տուաւ անոնց՝ անմաքուր ոգիներուն վրայ:

<span id="an_6:8">6:8</span> Պատուիրեց անոնց՝ որ ոչինչ առնեն
ճամբորդութեան համար, բայց միայն գաւազան մը. ո՛չ պարկ, ո՛չ հաց, ո՛չ դրամ՝
գօտիներուն մէջ,

<span id="an_6:9">6:9</span> հապա հողաթափ հագնին: Ու ըսաւ. «Կրկին բաճկոն
մի՛ հագնիք»,

<span id="an_6:10">6:10</span> եւ աւելցուց. «Ո՛ր տունը որ մտնէք, հո՛ն
մնացէք՝ մինչեւ որ անկէ մեկնիք:

<span id="an_6:11">6:11</span> Իսկ անոնք որ չեն ընդունիր ձեզ ու մտիկ չեն
ըներ ձեզի, երբ անկէ մեկնիք՝ թօթուեցէ՛ք ձեր ոտքերուն ներքեւի փոշին, իբր
վկայութիւն անոնց: Ճշմա՛րտապէս կը յայտարարեմ ձեզի. “Սոդոմացիներուն եւ
Գոմորացիներուն աւելի՛ դիւրին պիտի ըլլայ դատաստանին օրը՝ քան այդ
քաղաքին”՝՝»:

<span id="an_6:12">6:12</span> Անոնք ալ մեկնելով՝ կը քարոզէին, որ մարդիկ
ապաշխարեն:

<span id="an_6:13">6:13</span> Շատ դեւեր դուրս կը հանէին, շատ հիւանդներ
իւղով կ՚օծէին ու զանոնք կը բուժէին:

<span id="an_6:14">6:14</span> Երբ Հերովդէս թագաւորը լսեց, (քանի որ անոր
անունը յայտնի եղաւ,) ըսաւ. «Յովհաննէս Մկրտիչ մեռելներէն յարութիւն առած
է. ուստի հրաշքներ կը գործուին անով»:

<span id="an_6:15">6:15</span> Ուրիշներ կ՚ըսէին. «Եղիա՛ն է», ուրիշներ ալ
կ՚ըսէին. «Մարգարէ մըն է, կամ մարգարէներէն մէկուն պէս»:

<span id="an_6:16">6:16</span> Բայց երբ Հերովդէս լսեց՝ ըսաւ. «Ասիկա
Յովհաննէ՛սն է՝ որ ես գլխատեցի. մեռելներէն յարութիւն առած է»:

<span id="an_6:17">6:17</span> Արդարեւ Հերովդէս՝ ի՛նք մարդ ղրկելով բռներ
էր Յովհաննէսը, ու կապելով բանտը դրեր էր, իր եղբօր՝ Փիլիպպոսի կնոջ՝
Հերովդիայի պատճառով, քանի որ անոր հետ ամուսնացած էր.

<span id="an_6:18">6:18</span> որովհետեւ Յովհաննէս կ՚ըսէր Հերովդէսի.
«Քեզի արտօնուած չէ եղբօրդ կինը առնել»:

<span id="an_6:19">6:19</span> Հերովդիա ոխ ունէր անոր դէմ եւ կ՚ուզէր
զայն սպաննել, բայց չէր կրնար.

<span id="an_6:20">6:20</span> որովհետեւ Հերովդէս կը վախնար Յովհաննէսէ,
գիտնալով թէ ան արդար ու սուրբ մարդ մըն է, եւ կը դիտէր զայն: Անկէ լսելով՝
շատ բաներ կ՚ընէր, ու հաճոյքով անոր մտիկ կ՚ընէր:

<span id="an_6:21">6:21</span> Պատեհ օր մը եկաւ, երբ Հերովդէս իր ծնունդի
տարեդարձին առիթով ընթրիք մը սարքած էր իր մեծամեծներուն, հազարապետներուն
եւ Գալիլեայի գլխաւորներուն:

<span id="an_6:22">6:22</span> Հերովդիայի աղջիկը՝ հանդէսին սրահը
մտնելով՝ պարեց, ու հաճեցուց Հերովդէսը եւ իրեն հետ սեղան նստողները:
Թագաւորը ըսաւ աղջիկին. «Խնդրէ՛ ինձմէ ի՛նչ որ կ՚ուզես, ու պիտի տամ քեզի»:

<span id="an_6:23">6:23</span> Եւ անոր երդում ըրաւ՝ ըսելով. «Ի՛նչ որ
խնդրես ինձմէ՝ պիտի տամ քեզի, մինչեւ թագաւորութեանս կէսը»:

<span id="an_6:24">6:24</span> Ան դուրս ելլելով՝ հարցուց իր մօր. «Ի՞նչ
ուզեմ»: Ան ալ ըսաւ. «Յովհաննէս Մկրտիչին գլուխը»:

<span id="an_6:25">6:25</span> Իսկոյն փութալով թագաւորին քով մտաւ ու
խնդրեց՝ ըսելով. «Կ՚ուզեմ որ անյապաղ, ափսէի մը վրայ, Յովհաննէս Մկրտիչին
գլուխը տաս ինծի»:

<span id="an_6:26">6:26</span> Թագաւորը չափազանց տրտմեցաւ. բայց
երդումներուն եւ իրեն հետ սեղան նստողներուն պատճառով՝ չուզեց մերժել անոր
խնդրանքը՝՝:

<span id="an_6:27">6:27</span> Ուստի թագաւորը իսկոյն դահիճ մը ղրկեց, ու
հրամայեց բերել անոր գլուխը: Ան ալ գնաց, բանտին մէջ գլխատեց զայն,

<span id="an_6:28">6:28</span> ափսէի մը վրայ բերաւ անոր գլուխը եւ տուաւ
աղջիկին. աղջիկն ալ տուաւ զայն իր մօր:

<span id="an_6:29">6:29</span> Երբ անոր աշակերտները լսեցին՝ եկան,
վերցուցին անոր մարմինը եւ դրին գերեզմանի մը մէջ:

<span id="an_6:30">6:30</span> Առաքեալները հաւաքուեցան Յիսուսի քով ու
պատմեցին անոր ամէն բան, թէ՛ ինչ որ ըրին, թէ՛ ինչ որ սորվեցուցին:

<span id="an_6:31">6:31</span> Ան ալ ըսաւ անոնց. «Դո՛ւք առանձին եկէ՛ք
ամայի տեղ մը, եւ հանգչեցէ՛ք քիչ մը». քանի որ շատեր կու գային ու
կ՚երթային, եւ հաց ուտելու իսկ ժամանակ չէին ձգեր:

<span id="an_6:32">6:32</span> Ուստի նաւով ամայի տեղ մը գացին՝ առանձին:

<span id="an_6:33">6:33</span> Բազմութիւնը տեսաւ զանոնք՝ որ կ՚երթային:
Շատեր ճանչցան զայն, ու ոտքով՝ բոլոր քաղաքներէն հոն վազեցին, եւ անոնցմէ
առաջ հասնելով՝ անոր քով համախմբուեցան:

<span id="an_6:34">6:34</span> Յիսուս, երբ դուրս ելաւ, մեծ բազմութիւն մը
տեսնելով՝ գթաց անոնց վրայ, որովհետեւ հովիւ չունեցող ոչխարներու պէս էին.
ու շատ բաներ սորվեցուց անոնց:

<span id="an_6:35">6:35</span> Երբ շատ ժամեր անցան, իր աշակերտները եկան
իրեն եւ ըսին. «Հոս ամայի տեղ մըն է, ու ժամանակը արդէն շատ ուշ է:

<span id="an_6:36">6:36</span> Արձակէ՛ ժողովուրդը, որպէսզի երթան շրջակայ
արտերն ու գիւղերը եւ իրենց հաց գնեն, որովհետեւ ոչինչ ունին ուտելու»:

<span id="an_6:37">6:37</span> Բայց ինք պատասխանեց անոնց. «Դո՛ւք տուէք
ատոնց՝ որ ուտեն»: Ըսին իրեն. «Երթանք գնե՞նք երկու հարիւր դահեկանի հաց ու
տա՞նք ատոնց՝ որ ուտեն»:

<span id="an_6:38">6:38</span> Ըսաւ անոնց. «Քանի՞ նկանակ ունիք. գացէ՛ք՝
նայեցէ՛ք»: Երբ գիտցան՝ ըսին. «Հինգ, եւ երկու ձուկ»:

<span id="an_6:39">6:39</span> Հրամայեց անոնց, որ բոլորը կանաչ խոտին
վրայ՝ խումբ-խումբ նստեցնեն.

<span id="an_6:40">6:40</span> ու նստան՝ հարիւրական եւ յիսունական
շարքերով:

<span id="an_6:41">6:41</span> Հինգ նկանակներն ու երկու ձուկերը առաւ,
դէպի երկինք նայելով՝ օրհնեց, մանրեց նկանակները եւ տուաւ իր
աշակերտներուն, որպէսզի հրամցնեն անոնց. երկու ձուկերն ալ բաժնեց բոլորին:

<span id="an_6:42">6:42</span> Բոլորը կերան, կշտացան,

<span id="an_6:43">6:43</span> եւ վերցնելով բեկորներն ու ձուկերը՝
տասներկու կողով լեցուցին:

<span id="an_6:44">6:44</span> Այդ նկանակներէն ուտողները՝ հինգ հազարի
չափ այր մարդիկ էին:

<span id="an_6:45">6:45</span> Իսկոյն իր աշակերտները հարկադրեց՝ որ նաւ
մտնեն, ու իրմէ առաջ անցնին միւս եզերքը՝ Բեթսայիդա, մինչ ինք կ՚արձակէր
բազմութիւնը:

<span id="an_6:46">6:46</span> Եւ անոնցմէ հրաժեշտ առնելէ ետք՝ լեռը գնաց
աղօթելու:

<span id="an_6:47">6:47</span> Երբ իրիկուն եղաւ, նաւը ծովուն մէջտեղն էր,
իսկ ինք ցամաքին վրայ էր՝ մինակ:

<span id="an_6:48">6:48</span> Տեսաւ զանոնք՝ որ կը տանջուէին թի վարելով,
որովհետեւ հովը իրենց հակառակ էր: Գիշերուան չորրորդ պահուն ատենները՝
ծովուն վրայ քալելով գնաց անոնց, եւ կ՚ուզէր անցնիլ անոնց քովէն:

<span id="an_6:49">6:49</span> Երբ անոնք տեսան՝ որ ծովուն վրայ կը քալէր,
կարծեցին թէ աչքի երեւոյթ մըն է. ուստի աղաղակեցին,

<span id="an_6:50">6:50</span> քանի որ բոլորն ալ տեսան զայն ու
վրդովեցան: Եւ ինք իսկոյն խօսեցաւ անոնց հետ ու ըսաւ անոնց.
«Քաջալերուեցէ՛ք, ե՛ս եմ, մի՛ վախնաք»:

<span id="an_6:51">6:51</span> Եւ նաւ ելաւ անոնց քով, ու հովը դադրեցաւ:
Անոնք իրենք իրենց մէջ չափազանց զմայլած էին եւ կը զարմանային,

<span id="an_6:52">6:52</span> որովհետեւ նկանակներուն հրաշքն իսկ չէին
նկատած՝ քանի իրենց սիրտը թմրած էր:

<span id="an_6:53">6:53</span> Միւս կողմը անցնելով՝ հասան Գեննեսարէթի
երկիրը, ու նաւը տեղաւորեցին:

<span id="an_6:54">6:54</span> Երբ նաւէն ելան՝ այդ տեղի մարդիկը իսկոյն
ճանչցան զայն,

<span id="an_6:55">6:55</span> եւ ամբողջ շրջակայքը հոս-հոն վազելով՝
սկսան մահիճներով ախտաւորներ բերել հո՛ն՝ ո՛ւր կը լսէին թէ կը գտնուէր:

<span id="an_6:56">6:56</span> Ո՛ւր որ մտնէր, գիւղերը, քաղաքները կամ
արտերը, հրապարակներուն վրայ կը դնէին հիւանդները, ու կ՚աղաչէին իրեն՝ որ
գոնէ իր հանդերձին քղանցքին դպչին: Եւ անոնք որ դպան՝ բժշկուեցան:

<span id="an_7:1">7:1</span> Փարիսեցիներէն ու դպիրներէն ոմանք՝ որ եկած
էին Երուսաղէմէն՝ իր քով հաւաքուեցան:

<span id="an_7:2">7:2</span> Երբ տեսան թէ անոր աշակերտներէն ոմանք՝ պիղծ,
այսինքն՝ անլուայ ձեռքերով հաց կ՚ուտեն, մեղադրեցին.

<span id="an_7:3">7:3</span> որովհետեւ Փարիսեցիներն ու բոլոր Հրեաները
հաց չեն ուտեր՝ մինչեւ որ լա՛ւ մը լուան իրենց ձեռքերը, քանի որ
նախնիքներուն աւանդութիւնը կը պահեն:

<span id="an_7:4">7:4</span> Շուկայէն եկած ատեն եթէ չլուացուին՝ չեն
ուտեր: Ասոնց նման շատ բաներ կան, որոնք ընդունած են ու կը պահեն, ինչպէս՝
գաւաթներու, փարչերու, պղինձէ անօթներու եւ սեղաններու լուացումը:

<span id="an_7:5">7:5</span> Հետեւաբար Փարիսեցիներն ու դպիրները
հարցուցին անոր. «Ինչո՞ւ քու աշակերտներդ նախնիքներուն աւանդութեան
համաձայն չեն շարժիր, եւ անլուայ ձեռքերով հաց կ՚ուտեն»:

<span id="an_7:6">7:6</span> Ինք ալ պատասխանեց անոնց. «Եսայի ճի՛շդ
մարգարէացաւ ձեր մասին, կեղծաւորնե՛ր, ինչպէս գրուած է. “Այս ժողովուրդը
միայն շրթունքո՛վ կը պատուէ զիս, բայց իրենց սիրտը հեռու է ինձմէ.

<span id="an_7:7">7:7</span> ու պարապ տեղը կը պաշտեն զիս՝ մարդոց
պատուէրները իբր վարդապետութիւն սորվեցնելով”:

<span id="an_7:8">7:8</span> Որովհետեւ Աստուծոյ պատուիրանը ձգելով՝
մարդո՛ց աւանդութիւնը կը պահէք, ինչպէս՝ փարչերու եւ գաւաթներու լուացումը,
ու ասոնց նման շատ ուրիշ բաներ կ՚ընէք»:

<span id="an_7:9">7:9</span> Եւ ըսաւ անոնց. «Լա՛ւ կ՚անարգէք Աստուծոյ

<span id="an_7:10">7:10</span> Քանի որ Մովսէս ըսաւ. “Պատուէ՛ հայրդ ու

<span id="an_7:11">7:11</span> Բայց դուք կ՚ըսէք. “Եթէ մէկը իր հօր կամ
մօր ըսէ. "Ի՛նչ օգուտ որ ինձմէ պիտի ստանաս՝ Աստուծոյ տալիք կորբան է", (որ
կը նշանակէ՝ ընծայ,) ան ազատ կ՚ըլլայ”,

<span id="an_7:12">7:12</span> եւ ա՛լ թոյլ չէք տար անոր՝ բա՛ն մը ընել իր
հօր կամ մօր:

<span id="an_7:13">7:13</span> Անվաւեր կը դարձնէք Աստուծոյ խօսքը՝ ձեր
փոխանցած աւանդութեամբ, ու շատ ուրիշ բաներ կ՚ընէք՝ ասոր նման»:

<span id="an_7:14">7:14</span> Ամբողջ բազմութիւնը իրեն կանչելով՝ ըսաւ
անոնց. «Բոլո՛րդ ալ մտիկ ըրէք ինծի, ու հասկցէ՛ք:

<span id="an_7:15">7:15</span> Ոչինչ կայ՝ որ դուրսէն մտնէ մարդուն մէջ ու
կարող ըլլայ պղծել զայն. հապա ինչ որ դուրս կ՚ելլէ անկէ, անիկա՛ է որ կը
պղծէ մարդը:

<span id="an_7:16">7:16</span> Ո՛վ որ ականջ ունի լսելու՝ թող լսէ»:

<span id="an_7:17">7:17</span> Երբ ինք՝ զատուելով այդ բազմութենէն՝ տուն
մտաւ, իր աշակերտները հարցուցին իրեն այդ առակին մասին:

<span id="an_7:18">7:18</span> Ըսաւ անոնց. «Դո՞ւք ալ այդպէս անխելք էք.
չէ՞ք ըմբռներ թէ ամէն ինչ՝ որ դուրսէ՛ն կը մտնէ մարդուն մէջ, չի կրնար
պղծել զայն,

<span id="an_7:19">7:19</span> որովհետեւ ո՛չ թէ անոր սիրտին մէջ կը մտնէ՝
հապա փորը, եւ արտաքնոցը դուրս կ՚ելլէ. ուստի բոլոր կերակուրները մաքուր
են՝՝»:

<span id="an_7:20">7:20</span> Նաեւ ըսաւ. «Ինչ որ կ՚ելլէ մարդուն ներսէն,
անիկա՛ կը պղծէ մարդը:

<span id="an_7:21">7:21</span> Որովհետեւ ներսէ՛ն, մարդոց սիրտէ՛ն,
կ՚ելլեն չար մտածումներ, շնութիւններ, պոռնկութիւններ, սպանութիւններ,

<span id="an_7:22">7:22</span> գողութիւններ, ագահութիւններ,

ամբարտաւանութիւն, անմտութիւն:

<span id="an_7:23">7:23</span> Այս բոլոր չար բաները ներսէ՛ն կ՚ելլեն ու
կը պղծեն մարդը»:

<span id="an_7:24">7:24</span> Կանգնելով՝ անկէ գնաց Տիւրոսի եւ Սիդոնի
հողամասերը, ու տուն մը մտնելով՝ կ՚ուզէր որ ո՛չ մէկը գիտնայ. բայց չկրցաւ
թաքուն մնալ:

<span id="an_7:25">7:25</span> Որովհետեւ կին մը, որուն աղջիկը անմաքուր
ոգի ունէր իր մէջ, լսեց անոր մասին, եկաւ եւ ինկաւ անոր ոտքը:

<span id="an_7:26">7:26</span> Այդ կինը հեթանոս էր, փիւնիկ-ասորի ցեղէն,
ու կը թախանձէր անոր՝ որ հանէ դեւը իր աղջիկէն:

<span id="an_7:27">7:27</span> Յիսուս ըսաւ անոր. «Թո՛յլ տուր՝ որ նախ
զաւակնե՛րը կշտանան, քանի որ լաւ չէ առնել զաւակներուն հացը եւ նետել
շուներուն»:

<span id="an_7:28">7:28</span> Ան ալ պատասխանեց անոր. «Այո՛, Տէ՛ր.
սակայն շուներն ալ՝ սեղանին ներքեւ՝ կը կերակրուին զաւակներուն
փշրանքներէն»:

<span id="an_7:29">7:29</span> Յիսուս ըսաւ անոր. «Այդ խօսքիդ համար՝
գնա՛, դեւը դուրս ելաւ աղջիկէդ»:

<span id="an_7:30">7:30</span> Կինը իր տունը երթալով՝ գտաւ որ դեւը ելած
էր, իսկ աղջիկը՝ պառկած մահիճին վրայ:

<span id="an_7:31">7:31</span> Դարձեալ Տիւրոսի եւ Սիդոնի հողամասերէն
մեկնելով՝ գնաց Գալիլեայի ծովեզերքը, Դեկապոլիսի հողամասին մէջէն:

<span id="an_7:32">7:32</span> Մէկը բերին անոր առջեւ, խուլ ու
դժուարախօս, եւ կ՚աղաչէին՝ որ ձեռք դնէ անոր վրայ:

<span id="an_7:33">7:33</span> Բազմութենէն մէկդի առնելով զայն՝ իր
մատները մխեց անոր ականջներուն մէջ ու թքնեց, եւ դպաւ անոր լեզուին.

<span id="an_7:34">7:34</span> ապա երկինք նայելով՝ հառաչեց ու ըսաւ անոր.
«Եփփաթա՛», (որ ըսել է՝ բացուէ՛.)

<span id="an_7:35">7:35</span> իսկոյն բացուեցան անոր ականջները եւ
քակուեցաւ անոր լեզուին կապը, ու շիտակ կը խօսէր:

<span id="an_7:36">7:36</span> Պատուիրեց անոնց՝ որ ո՛չ մէկուն ըսեն.
սակայն ինք ո՛րքան կը պատուիրէր անոնց, անոնք ա՛լ աւելի կը հրապարակէին:

<span id="an_7:37">7:37</span> Չափազանց կ՚ապշէին ու կ՚ըսէին. «Ան ամէն
ինչ լաւ կ՚ընէ. խուլերուն լսել կու տայ, եւ համրերուն՝ խօսիլ»:

<span id="an_8:1">8:1</span> Այդ օրերը, բազմութիւնը յոյժ շատ ըլլալով եւ
ուտելու ոչինչ ունենալով, Յիսուս իրեն կանչեց իր աշակերտները ու ըսաւ
անոնց.

<span id="an_8:2">8:2</span> «Կը գթամ այդ բազմութեան վրայ, որովհետեւ
արդէն երեք օր է՝ որ քովս են, եւ ոչինչ ունին ուտելու:

<span id="an_8:3">8:3</span> Եթէ անօթի ուղարկեմ զանոնք իրենց տուները՝
պիտի պարտասին ճամբան, որովհետեւ ատոնցմէ ոմանք հեռաւոր տեղէ եկած են»:

<span id="an_8:4">8:4</span> Իր աշակերտները պատասխանեցին իրեն. «Այս
անապատին մէջ ուրկէ՞ կրնայ մէկը հացով կշտացնել ատոնք»:

<span id="an_8:5">8:5</span> Հարցուց անոնց. «Քանի՞ նկանակ ունիք»: Անոնք
ըսին. «Եօթը»:

<span id="an_8:6">8:6</span> Հրամայեց բազմութեան՝ որ նստի գետինին վրայ.
եւ առաւ եօթը նկանակները, շնորհակալ եղաւ, կտրեց ու տուաւ աշակերտներուն,
որպէսզի հրամցնեն բազմութեան. եւ հրամցուցին:

<span id="an_8:7">8:7</span> Քանի մը մանր ձուկ ալ ունէին: Օրհնեց ու
ըսաւ՝ որ զանոնք ալ հրամցնեն անոնց:

<span id="an_8:8">8:8</span> Կերան, կշտացան, եւ աւելցած բեկորներէն եօթը
զամբիւղ վերցուցին:

<span id="an_8:9">8:9</span> Ուտողները չորս հազարի չափ էին: Արձակեց
զանոնք,

<span id="an_8:10">8:10</span> եւ իսկոյն նաւ մտնելով՝ աշակերտներուն հետ
գնաց Դաղմանութայի կողմերը:

<span id="an_8:11">8:11</span> Փարիսեցիները ելլելով՝ սկսան անոր հետ
վիճաբանիլ, ու զայն փորձելով՝ կը խնդրէին իրմէ նշան մը երկինքէն:

<span id="an_8:12">8:12</span> Ան ալ իր հոգիին մէջ հառաչելով՝ ըսաւ.
«Ինչո՞ւ այս սերունդը նշան կը խնդրէ: Ճշմա՛րտապէս կը յայտարարեմ ձեզի թէ
այս սերունդին նշան պիտի չտրուի»:

<span id="an_8:13">8:13</span> Եւ զանոնք թողուց, դարձեալ նաւ մտաւ ու
գնաց ծովուն միւս եզերքը:

<span id="an_8:14">8:14</span> Հաց առնել մոռցած էին. բայց նաւուն մէջ
իրենց հետ միայն մէկ նկանակ ունէին:

<span id="an_8:15">8:15</span> Պատուիրեց անոնց ու ըսաւ. «Ուշադի՛ր եղէք,
զգուշացէ՛ք Փարիսեցիներու խմորէն ու Հերովդէսի խմորէն»:

<span id="an_8:16">8:16</span> Իսկ անոնք իրարու հետ կը մտածէին եւ
կ՚ըսէին. «Որովհետեւ հաց չունինք»:

<span id="an_8:17">8:17</span> Յիսուս գիտնալով անոնց մտածումը՝ ըսաւ
անոնց. «Ինչո՞ւ կը մտածէք թէ հաց չունիք. տակաւին չէ՞ք ըմբռներ ու չէ՞ք
հասկնար. տակաւին ձեր սիրտը կարծրացա՞ծ է:

<span id="an_8:18">8:18</span> Աչքեր ունիք եւ չէ՞ք տեսներ, ականջներ
ունիք ու չէ՞ք լսեր, եւ չէ՞ք յիշեր:

<span id="an_8:19">8:19</span> Երբ հինգ նկանակները կտրեցի հինգ հազարին,
բեկորներով լեցուն քանի՞ կողով վերցուցիք»: Ըսին անոր. «Տասներկու»:

<span id="an_8:20">8:20</span> «Ու երբ եօթը նկանակները կտրեցի չորս
հազարին, բեկորներով լեցուն քանի՞ զամբիւղ վերցուցիք»: Անոնք ալ ըսին.
«Եօթը»:

<span id="an_8:21">8:21</span> Եւ ըսաւ անոնց. «Հապա ի՞նչպէս չէք
հասկնար»:

<span id="an_8:22">8:22</span> Երբ գնաց Բեթսայիդա՝ կոյր մը բերին անոր
առջեւ, ու կ՚աղաչէին իրեն՝ որ դպչի անոր:

<span id="an_8:23">8:23</span> Ան ալ՝ բռնելով կոյրին ձեռքէն՝ դուրս տարաւ
գիւղէն, եւ թքնելով անոր աչքերուն՝ դրաւ ձեռքը անոր վրայ, ու հարցուց անոր
թէ կը տեսնէ՛ բան մը:

<span id="an_8:24">8:24</span> Ան ալ նայեցաւ եւ ըսաւ. «Կը տեսնեմ մարդիկ՝
ծառերու պէս, որ կը քալեն»:

<span id="an_8:25">8:25</span> Դարձեալ դրաւ ձեռքը անոր աչքերուն վրայ ու
նայիլ տուաւ անոր. եւ առողջացաւ, ու յստակօրէն կը տեսնէր բոլորը:

<span id="an_8:26">8:26</span> Յետոյ՝ ղրկելով զայն իր տունը՝ ըսաւ. «Մի՛
մտներ գիւղը, ո՛չ ալ մէկու մը ըսէ գիւղին մէջ»:

<span id="an_8:27">8:27</span> Յիսուս գնաց Փիլիպպոսի Կեսարիային գիւղերը՝
իր աշակերտներուն հետ: Ճամբան հարցուց իր աշակերտներուն. «Մարդիկ ինծի
համար ի՞նչ կ՚ըսեն, ո՞վ եմ»:

<span id="an_8:28">8:28</span> Անոնք պատասխանեցին. «Ոմանք՝ Յովհաննէս
Մկրտիչը, ուրիշներ՝ Եղիան, ուրիշներ ալ՝ մարգարէներէն մէկը»:

<span id="an_8:29">8:29</span> Եւ ինք ըսաւ անոնց. «Իսկ դո՛ւք ինծի համար
ի՞նչ կ՚ըսէք, ո՞վ եմ»: Պետրոս պատասխանեց անոր. «Դուն Քրիստո՛սն ես»:

<span id="an_8:30">8:30</span> Սաստիկ հրահանգեց անոնց՝ որ ո՛չ մէկուն
խօսին իր մասին:

<span id="an_8:31">8:31</span> Ապա սկսաւ սորվեցնել անոնց. «Պէտք է որ
մարդու Որդին շատ չարչարանքներ կրէ, մերժուի երէցներէն, քահանայապետներէն
եւ դպիրներէն, սպաննուի, ու երեք օրէն ետք յարութիւն առնէ»:

<span id="an_8:32">8:32</span> Այս խօսքը ըսաւ բացորոշապէս: Ուստի Պետրոս
մէկդի առաւ զինք եւ սկսաւ յանդիմանել:

<span id="an_8:33">8:33</span> Իսկ ինք դարձաւ, իր աշակերտներուն նայելով՝
յանդիմանեց Պետրոսը եւ ըսաւ. «Ետե՛ւս գնա, Սատանա՛յ, որովհետեւ դուն ո՛չ թէ
Աստուծոյ բաները կը մտածես, հապա՝ մարդոց բաները»:

<span id="an_8:34">8:34</span> Երբ իրեն կանչեց բազմութիւնը՝ իր
աշակերտներուն հետ, ըսաւ անոնց. «Ո՛վ որ ուզէ գալ իմ ետեւէս, թող ուրանայ
ինքզինք, վերցնէ իր խաչը, ու հետեւի ինծի:

<span id="an_8:35">8:35</span> Որովհետեւ ո՛վ որ ուզէ փրկել իր անձը՝ պիտի
կորսնցնէ զայն, իսկ ո՛վ որ կորսնցնէ իր անձը՝ ինծի ու աւետարանին համար,
պիտի փրկէ զայն:

<span id="an_8:36">8:36</span> Քանի որ ի՞նչ օգուտ է մարդուն՝ որ շահի
ամբողջ աշխարհը, բայց կորսնցնէ իր անձը.

<span id="an_8:37">8:37</span> կամ՝ մարդ ի՞նչ փրկանք պիտի տայ իր անձին
փոխարէն:

<span id="an_8:38">8:38</span> Որովհետեւ ո՛վ որ ամօթ սեպէ զիս եւ իմ
խօսքերս դաւանիլը՝ այս շնացող ու մեղաւոր սերունդին մէջ, մարդու Որդին ալ
ամօթ պիտի սեպէ զայն դաւանիլը՝ երբ գայ իր Հօր փառքով սուրբ հրեշտակներուն
հետ»:

<span id="an_8:39">8:39</span> Եւ ըսաւ անոնց. «Ճշմա՛րտապէս կը յայտարարեմ
ձեզի. “Հոս ներկայ եղողներէն կան ոմանք՝ որ մահ պիտի չհամտեսեն, մինչեւ որ
տեսնեն Աստուծոյ թագաւորութիւնը՝ զօրութեամբ եկած”»:

<span id="an_9:1">9:1</span> Վեց օր ետք՝ Յիսուս առաւ իրեն հետ Պետրոսը,
Յակոբոսն ու Յովհաննէսը, հանեց զանոնք բարձր լեռ մը՝ առանձին, եւ
այլակերպեցաւ անոնց առջեւ:

<span id="an_9:2">9:2</span> Իր հանդերձները փայլուն, չափազանց ճերմակ
եղան՝ ձիւնի պէս, այնպէս որ երկրի վրայ ո՛չ մէկ թափիչ կրնար այդպէս
ճերմկցնել:

<span id="an_9:3">9:3</span> Ու Եղիա երեւցաւ անոնց՝ Մովսէսի հետ, եւ կը
խօսակցէին Յիսուսի հետ:

<span id="an_9:4">9:4</span> Ապա Պետրոս ըսաւ Յիսուսի. «Ռաբբի՛, լաւ է որ
կենանք հոս եւ շինենք երեք վրան, մէկը՝ քեզի, մէկը՝ Մովսէսի ու մէկը՝
Եղիայի».

<span id="an_9:5">9:5</span> որովհետեւ չէր գիտեր թէ ի՛նչ կը խօսի, քանի
որ զարհուրած էին:

<span id="an_9:6">9:6</span> Ամպ մը հովանի եղաւ անոնց վրայ, եւ ձայն մը
եկաւ ամպէն՝ ըսելով. «Ա՛յս է իմ սիրելի Որդիս, անո՛ր մտիկ ըրէք»:

<span id="an_9:7">9:7</span> Յանկարծ, երբ իրենց շուրջը նայեցան, ուրիշ
ո՛չ մէկը տեսան, հապա միայն Յիսուսը՝ իրենց հետ:

<span id="an_9:8">9:8</span> Երբ վար կ՚իջնէին լեռնէն, պատուիրեց անոնց՝
որ ո՛չ մէկուն պատմեն իրենց տեսածը, բայց միայն՝ մարդու Որդիին մեռելներէն
յարութիւն առնելէն ետք:

<span id="an_9:9">9:9</span> Անոնք պահեցին այդ խօսքը, ու կը հարցնէին
իրարու թէ ի՛նչ կը նշանակէ “մեռելներէն յարութիւն առնել”:

<span id="an_9:10">9:10</span> Եւ հարցուցին իրեն. «Ինչո՞ւ դպիրները
կ՚ըսեն թէ “պէտք է որ նախ Եղիա գայ”»:

<span id="an_9:11">9:11</span> Ան ալ պատասխանեց անոնց. «Արդարեւ նախ Եղիա
կու գայ ու կը վերահաստատէ ամէն բան. ի՛նչպէս գրուած է մարդու Որդիին
մասին՝ թէ պէտք է շատ չարչարանքներ կրէ եւ անարգուի:

<span id="an_9:12">9:12</span> Բայց կ՚ըսեմ ձեզի թէ Եղիա եկաւ ալ, ու ինչ
որ ուզեցին՝ ըրին անոր, ինչպէս գրուած էր իր մասին»:

<span id="an_9:13">9:13</span> Աշակերտներուն քով գալով՝ տեսաւ մեծ
բազմութիւն մը անոնց շուրջը, ու դպիրներ՝ որ կը վիճաբանէին անոնց հետ:

<span id="an_9:14">9:14</span> Իսկոյն ամբողջ բազմութիւնը՝ երբ տեսաւ
զայն՝ շատ այլայլեցաւ, եւ յառաջ վազելով զայն կը բարեւէր:

<span id="an_9:15">9:15</span> Յիսուս հարցուց դպիրներուն. «Ի՞նչ բանի
մասին կը վիճաբանէիք անոնց հետ:

<span id="an_9:16">9:16</span> Բազմութենէն մէկը պատասխանեց. «Վարդապե՛տ,
որդիս՝ որ համր ոգի ունի՝ քեզի բերի:

<span id="an_9:17">9:17</span> Ո՛ւր որ ալ բռնէ զայն՝ գետին կը զարնէ՝՝
զայն. ինք ալ կը փրփրի, իր ակռաները կը կրճտէ եւ կը ցամքի: Քու
աշակերտներուդ ըսի որ հանեն զայն, բայց չկրցան»:

<span id="an_9:18">9:18</span> Ինք ալ պատասխանեց անոնց. «Ո՛վ անհաւատ
սերունդ, մինչեւ ե՞րբ պիտի ըլլամ ձեզի հետ, մինչեւ ե՞րբ պիտի հանդուրժեմ
ձեզի. ինծի՛ բերէք զայն»:

<span id="an_9:19">9:19</span> Իրեն բերին զայն: Երբ տղան տեսաւ զինք,
իսկոյն չար ոգին ցնցեց զայն, եւ գետին ինկած՝ կը թաւալէր ու կը փրփրէր:

<span id="an_9:20">9:20</span> Յիսուս հարցուց անոր հօր. «Ո՞րչափ ատեն է՝
որ այդ բանը պատահեցաւ անոր»: Ան ալ ըսաւ. «Մանկութենէն ի վեր:

<span id="an_9:21">9:21</span> Յաճախ կրակի ու ջուրի մէջ կը նետէ զայն՝
որպէսզի կորսնցնէ. բայց եթէ կարողութիւն ունիս՝ օգնէ՛ մեզի, գթալով մեր
վրայ»:

<span id="an_9:22">9:22</span> Յիսուս ըսաւ անոր. «Եթէ կրնաս հաւատալ.
ամէն բան կարելի է անոր՝ որ կը հաւատայ»:

<span id="an_9:23">9:23</span> Իսկոյն մանուկին հայրը աղաղակեց եւ
արցունքով ըսաւ. «Կը հաւատա՛մ, Տէ՛ր. օգնէ՛ իմ անհաւատութեանս»:

<span id="an_9:24">9:24</span> Յիսուս՝ երբ տեսաւ թէ բազմութիւնը կը
խռնուի շուրջը, սաստեց անմաքուր ոգին եւ ըսաւ անոր. «Հա՛մր ու խո՛ւլ ոգի,
կը հրամայեմ քեզի, ելի՛ր ատկէ, եւ անգա՛մ մըն ալ մի՛ մտներ անոր մէջ»:

<span id="an_9:25">9:25</span> Ոգին աղաղակելով՝ սաստիկ ցնցեց զայն ու
ելաւ. եւ տղան մեռելի պէս եղաւ, այնպէս որ շատեր ըսին. «Մեռա՛ւ»:

<span id="an_9:26">9:26</span> Իսկ Յիսուս անոր ձեռքէն բռնելով՝ ոտքի
հանեց զայն, ան ալ կանգնեցաւ:

<span id="an_9:27">9:27</span> Երբ ինք տուն մտաւ, իր աշակերտները առանձին
հարցուցին իրեն. «Մե՛նք ինչո՞ւ չկրցանք հանել զայն»:

<span id="an_9:28">9:28</span> Ըսաւ անոնց. «Այդ տեսակը կարելի չէ ուրիշ
բանով հանել, բայց միայն աղօթքով ու ծոմապահութեամբ»:

<span id="an_9:29">9:29</span> Անկէ մեկնելով՝ Գալիլեայի մէջէն կ՚անցնէին,
եւ չէր ուզեր որ մէ՛կը գիտնայ.

<span id="an_9:30">9:30</span> որովհետեւ կը սորվեցնէր իր աշակերտներուն
ու կ՚ըսէր անոնց. «Մարդու Որդին պիտի մատնուի մարդոց ձեռքը, պիտի սպաննեն
զայն, ու երբ սպաննուի՝ յարութիւն պիտի առնէ երրորդ օրը»:

<span id="an_9:31">9:31</span> Անոնք չէին հասկնար այդ խօսքը, եւ կը
վախնային հարցնել իրեն:

<span id="an_9:32">9:32</span> Եկաւ Կափառնայում. ու երբ տուն մտաւ՝
հարցուց անոնց. «Ի՞նչ բանի մասին կը մտածէիք ճամբան՝ իրարու հետ»:

<span id="an_9:33">9:33</span> Անոնք լուռ կեցան, որովհետեւ ճամբան
վիճաբանած էին իրարու հետ՝ թէ ո՛վ է մեծագոյնը:

<span id="an_9:34">9:34</span> Երբ նստաւ, կանչեց տասներկուքը եւ ըսաւ
անոնց. «Եթէ մէկը ուզէ առաջին ըլլալ, բոլորին յետի՛նը թող ըլլայ, ու
բոլորին սպասարկուն»:

հետեւելով՝ կը վախնային: Դարձեալ քովը առնելով տասներկուքը, սկսաւ ըսել

ապրուստը»:

աշակերտներէն մէկը ըսաւ իրեն. «Վարդապե՛տ, տե՛ս ինչպիսի՜ քարեր են, եւ ի՛նչ
տեսակ շէնքեր»:

<span id="an_13:2">13:2</span> Յիսուս պատասխանեց անոր. «Կը տեսնե՞ս այդ
մեծ շէնքերը. քար քարի վրայ պիտի չմնայ, ամէնը պիտի քակուի»:

<span id="an_13:3">13:3</span> Մինչ ան նստած էր Ձիթենիներու լեռը՝
տաճարին դիմաց, Պետրոս, Յակոբոս, Յովհաննէս ու Անդրէաս հարցուցին իրեն՝
առանձին.

<span id="an_13:4">13:4</span> «Ըսէ՛ մեզի, ե՞րբ պիտի ըլլայ ատիկա, եւ
ի՞նչ նշան պիտի ըլլայ այդ բոլորին կատարուելու ատենը»:

<span id="an_13:5">13:5</span> Յիսուս պատասխանելով անոնց՝ սկսաւ ըսել.
«Զգուշացէ՛ք որ ո՛չ մէկը մոլորեցնէ ձեզ.

<span id="an_13:6">13:6</span> որովհետեւ շատե՜ր պիտի գան իմ անունովս՝
ըսելով. “Ե՛ս եմ Քրիստոսը”, ու պիտի մոլորեցնեն շատերը:

<span id="an_13:7">13:7</span> Բայց երբ լսէք պատերազմներու մասին, եւ
պատերազմներու տարաձայնութիւններ՝ մի՛ վրդովիք, որովհետեւ պէտք է որ այս
ամէնը ըլլայ, բայց դեռ վախճանը չէ:

<span id="an_13:8">13:8</span> Արդարեւ ազգ ազգի դէմ պիտի ելլէ, ու
թագաւորութիւն՝ թագաւորութեան դէմ, եւ տեղ-տեղ երկրաշարժներ պիտի ըլլան.
սովեր ու խառնակութիւններ ալ պիտի ըլլան: Ասոնք ցաւերուն սկիզբն են:

<span id="an_13:9">13:9</span> Զգուշացէ՛ք դուք ձեզի համար. քանի որ
ատեաններու պիտի մատնեն ձեզ, ժողովարաններու մէջ պիտի ծեծեն ձեզ, ու
կառավարիչներու եւ թագաւորներու առջեւ պիտի կենաք՝ իմ պատճառովս, անոնց
վկայութիւն ըլլալու համար:

<span id="an_13:10">13:10</span> Բայց նախ պէտք է որ աւետարանը քարոզուի
բոլոր ազգերուն:

<span id="an_13:11">13:11</span> Իսկ երբ մատնելու տանին ձեզ, նախապէս մի՛
մտահոգուիք թէ ի՛նչ պիտի խօսիք, ո՛չ ալ խոկացէք. հապա այդ ժամուն ինչ որ
ձեզի տրուի՝ զա՛յն խօսեցէք. որովհետեւ ո՛չ թէ դուք էք որ պիտի խօսիք, հապա
Սուրբ Հոգին»:

<span id="an_13:12">13:12</span> «Եղբայրը մահուան պիտի մատնէ եղբայրը, ու
հայրը՝ զաւակը. եւ զաւակներ պիտի ելլեն իրենց ծնողներուն դէմ ու մեռցնեն
զանոնք:

<span id="an_13:13">13:13</span> Բոլորին ատելի պիտի ըլլաք իմ անունիս
համար: Բայց ո՛վ որ տոկայ մինչեւ վախճանը՝ անիկա՛ պիտի փրկուի:

<span id="an_13:14">13:14</span> Երբ տեսնէք թէ աւերողին պղծութիւնը՝՝ -

պէտք չէ ըլլար, (ո՛վ որ կարդայ՝ թող հասկնայ,) այն ատեն Հրէաստանի մէջ
եղողները լեռնե՛րը թող փախչին:

<span id="an_13:15">13:15</span> Ա՛ն որ տանիքին վրայ է՝ թող չիջնէ տունը,
ու չմտնէ իր տունէն բան մը առնելու.

<span id="an_13:16">13:16</span> եւ ա՛ն որ արտին մէջ է՝ թող չվերադառնայ
իր հանդերձը առնելու:

<span id="an_13:17">13:17</span> Բայց վա՜յ այդ օրերը յղի եղողներուն ու
ծիծ տուողներուն:

<span id="an_13:18">13:18</span> Աղօթեցէ՛ք՝ որ ձեր փախուստը ձմեռը
չըլլայ:

<span id="an_13:19">13:19</span> Որովհետեւ այդ օրերը այնպիսի՛ տառապանք
պիտի ըլլայ, որուն նմանը՝ Աստուծոյ ստեղծած աշխարհի սկիզբէն մինչեւ հիմա
եղած չէ, ո՛չ ալ պիտի ըլլայ:

<span id="an_13:20">13:20</span> Եթէ Տէրը այդ օրերը չկարճեցնէր, ո՛չ մէկ
մարմին պիտի փրկուէր. բայց այդ օրերը կարճեցուց ընտրեալներուն համար՝ որ
ընտրեց:

<span id="an_13:21">13:21</span> Այն ատեն եթէ մէկը ըսէ ձեզի. “Ահա՛ հո՛ս
է Քրիստոսը”, կամ. “Ահա՛ հո՛ն է”, մի՛ հաւատաք:

<span id="an_13:22">13:22</span> Որովհետեւ սուտ քրիստոսներ եւ սուտ
մարգարէներ պիտի ելլեն, ու ցոյց պիտի տան նշաններ եւ սքանչելիքներ, որպէսզի
եթէ կարելի ըլլայ՝ մոլորեցնեն նոյնիսկ ընտրեալնե՛րը:

<span id="an_13:23">13:23</span> Իսկ դուք զգուշացէ՛ք. ահա՛ նախապէս ըսի
ձեզի ամէն ինչ»:

<span id="an_13:24">13:24</span> «Բայց այդ օրերը, այդ տառապանքէն ետք,
արեւը պիտի խաւարի եւ լուսինը իր փայլը պիտի չտայ.

<span id="an_13:25">13:25</span> աստղերը երկինքէն պիտի իյնան, ու երկինքի
մէջ եղող զօրութիւնները պիտի սարսին:

<span id="an_13:26">13:26</span> Այն ատեն պիտի տեսնեն մարդու Որդին,
ամպերու մէջ եկած՝ մեծ զօրութեամբ ու փառքով:

<span id="an_13:27">13:27</span> Այն ատեն պիտի ղրկէ իր հրեշտակները եւ
պիտի հաւաքէ իր ընտրեալները չորս հովերէն, երկրի ծայրէն մինչեւ երկինքի
ծայրը»:

<span id="an_13:28">13:28</span> «Թզենիէ՛ն սորվեցէք առակ մը. երբ անոր
ոստերը կակուղնան ու տերեւները ցցուին, կը հասկնաք թէ ամառը մօտ է:

<span id="an_13:29">13:29</span> Դուք ալ՝ երբ եղած տեսնէք ասոնք, գիտցէ՛ք
թէ մօտ է՝ դռներուն քով:

<span id="an_13:30">13:30</span> Ճշմա՛րտապէս կը յայտարարեմ ձեզի. “Այս
սերունդը պիտի չանցնի, մինչեւ որ այս բոլորը պատահին”:

<span id="an_13:31">13:31</span> Երկինք ու երկիր պիտի անցնին, բայց իմ
խօսքերս բնա՛ւ պիտի չանցնին»:

<span id="an_13:32">13:32</span> «Իսկ այդ օրն ու ժամը՝ ո՛չ մէկ մարդ
գիտէ, ո՛չ ալ երկինքի մէջ եղող հրեշտակները, ո՛չ իսկ Որդին, հապա՝ միա՛յն
Հայրը:

<span id="an_13:33">13:33</span> Զգուշացէ՛ք, հսկեցէ՛ք եւ աղօթեցէ՛ք,
որովհետեւ չէք գիտեր թէ ատենը ե՛րբ է:

<span id="an_13:34">13:34</span> Ինչպէս ճամբորդող մարդ մը կը թողու իր
տունը, իրաւասութիւն կու տայ իր ծառաներուն, իւրաքանչիւրին՝ իր գործը, ու
դռնապանին կը հրամայէ որ արթուն կենայ:

<span id="an_13:35">13:35</span> Ուրեմն արթո՛ւն կեցէք, որովհետեւ չէք
գիտեր թէ տան տէրը ե՛րբ պիտի գայ, իրիկո՞ւնը՝ թէ կէս գիշերին,
աքլորականչի՞ն՝ թէ առտուն:

<span id="an_13:36">13:36</span> Թերեւս յանկարծ գալով՝ ձեզ գտնէ քունի
մէջ:

<span id="an_13:37">13:37</span> Եւ ինչ որ կ՚ըսեմ ձեզի՝ կ՚ըսեմ բոլորին.
“Արթո՛ւն կեցէք”»:

<span id="an_14:1">14:1</span> Երկու օր ետք՝ Զատիկ ու Բաղարջակերք էր. եւ
քահանայապետներն ու դպիրները կը փնտռէին թէ ի՛նչպէս վարպետութեամբ բռնեն
զայն եւ սպաննեն:

<span id="an_14:2">14:2</span> Բայց կ՚ըսէին. «Ո՛չ տօնին ատենը, որպէսզի
ժողովուրդին մէջ աղմուկ չըլլայ»:

<span id="an_14:3">14:3</span> Երբ ինք Բեթանիա էր, սեղան նստած՝ բորոտ
Սիմոնի տունը, կին մը եկաւ՝ որ ունէր ալապաստրէ շիշ մը, լեցուած մեծածախս
նարդոսի օծանելիքով. կոտրեց այդ շիշը ու թափեց օծանելիքը անոր գլուխին
վրայ:

<span id="an_14:4">14:4</span> Ոմանք ընդվզեցան իրենք իրենց մէջ եւ ըսին.
«Ինչո՞ւ այդ օծանելիքը վատնուեցաւ.

<span id="an_14:5">14:5</span> քանի որ ատիկա կրնար ծախուիլ երեք հարիւր
դահեկանէն աւելի, ու տրուիլ աղքատներուն». եւ սաստիկ կը սրդողէին իրեն դէմ:

<span id="an_14:6">14:6</span> Բայց Յիսուս ըսաւ. «Թողուցէ՛ք զայն, ինչո՞ւ
կ՚անհանգստացնէք զինք. ան լաւ գործ մը ըրաւ ինծի:

<span id="an_14:7">14:7</span> Որովհետեւ ամէ՛ն ատեն ունիք աղքատները ձեզի
հետ, ու ե՛րբ ուզէք՝ կրնաք բարիք ընել անոնց. բայց ամէն ատեն չունիք զիս:

<span id="an_14:8">14:8</span> Ան ըրաւ ի՛նչ որ կրնար. մարմինս նախապէս
օծեց՝ իմ թաղումիս համար:

<span id="an_14:9">14:9</span> Ճշմա՛րտապէս կը յայտարարեմ ձեզի. “Ամբողջ
աշխարհի մէջ՝ ո՛ւր որ այս աւետարանը քարոզուի, անոր ըրածն ալ պիտի պատմուի՝
իր յիշատակին համար”»:

<span id="an_14:10">14:10</span> Յուդա Իսկարիովտացին, տասներկուքէն մէկը,
գնաց քահանայապետներուն՝ որ զայն մատնէ անոնց:

<span id="an_14:11">14:11</span> Անոնք ալ լսելով՝ ուրախացան, եւ իրեն
դրամ ալ խոստացան: Ուստի կը փնտռէր պատեհութիւն մը՝ որ մատնէր զայն:

<span id="an_14:12">14:12</span> Բաղարջակերքի առաջին օրը, երբ կը մորթէին
զատիկը, անոր աշակերտները ըսին իրեն. «Ո՞ւր կ՚ուզես որ երթանք պատրաստենք,
որպէսզի ուտես զատիկը»:

<span id="an_14:13">14:13</span> Իր աշակերտներէն երկուքը ղրկեց եւ ըսաւ
անոնց. «Գացէ՛ք քաղաքը, ու ձեզի պիտի հանդիպի մարդ մը՝ որ ուսին վրայ կը

<span id="an_14:17">14:17</span> Երբ իրիկուն եղաւ՝ եկաւ տասներկուքին
հետ:

կ՚ուտէ, պիտի մատնէ զիս»:

<span id="an_14:21">14:21</span> Արդարեւ մարդու Որդին կ՚երթայ՝ ինչպէս
գրուած է իրեն համար. բայց վա՜յ այն մարդուն, որուն միջոցով կը մատնուի

<span id="an_14:24">14:24</span> Ըսաւ անոնց. «Ա՛յս է իմ արիւնս, նո՛ր

<span id="an_14:27">14:27</span> Յիսուս ըսաւ անոնց. «Այս գիշեր բոլորդ ալ

ոչխարները պիտի ցրուին”:

<span id="an_14:28">14:28</span> Բայց յարութիւն առնելէս ետք՝ ձեզմէ առաջ
պիտի երթամ Գալիլեա»:

<span id="an_14:29">14:29</span> Պետրոս ըսաւ անոր. «Թէեւ բոլորն ալ
գայթակղին, ես՝ բնա՛ւ՝՝»:

<span id="an_14:30">14:30</span> Յիսուս ըսաւ անոր. «Ճշմա՛րտապէս կը

<span id="an_14:34">14:34</span> Հետեւաբար ըսաւ անոնց. «Իմ անձս չափազանց

<span id="an_14:35">14:35</span> Քիչ մը առջեւ երթալով՝ ինկաւ գետին, իր
երեսին վրայ, եւ աղօթեց, որ եթէ կարելի է՝ այդ ժամը անցնի իրմէ:

հապա՝ ինչպէս դո՛ւն կ՚ուզես»:

<span id="an_14:37">14:37</span> Երբ եկաւ ու քնացած գտաւ զանոնք՝ ըսաւ
Պետրոսի. «Սիմո՛ն, կը քնանա՞ս. չկրցա՞ր ժա՛մ մը արթուն մնալ:

<span id="an_14:38">14:38</span> Արթո՛ւն մնացէք եւ աղօթեցէ՛ք, որպէսզի

<span id="an_14:40">14:40</span> Ապա վերադարձաւ եւ դարձեալ քնացած գտաւ
զանոնք. որովհետեւ անոնց աչքերը ծանրացած էին, ու չէին գիտեր ի՛նչ
պատասխանել անոր:

<span id="an_14:41">14:41</span> Երրորդ անգամ ալ եկաւ եւ ըսաւ անոնց.

Որդին պիտի մատնուի մեղաւորներուն ձեռքը:

երէցներէն ղրկուած մեծ բազմութիւն մը, սուրերով եւ բիրերով:

<span id="an_14:44">14:44</span> Զայն մատնողը նշան մը տուեր էր անոնց՝

ապահովութեամբ»:

<span id="an_14:45">14:45</span> Երբ ինք եկաւ՝ իսկոյն մօտեցաւ անոր եւ
ըսաւ. «Ռաբբի՛, Ռաբբի՛», ու համբուրեց զայն:

<span id="an_14:46">14:46</span> Եւ անոնք ձեռք բարձրացուցին անոր վրայ ու
բռնեցին զայն:

<span id="an_14:47">14:47</span> Քովը կայնողներէն մէկը՝ քաշեց սուրը,
զարկաւ քահանայապետին ծառային եւ խլեց անոր ականջը:

<span id="an_14:48">14:48</span> Յիսուս ըսաւ անոնց. «Իբր թէ աւազակի՞ դէմ
եկաք՝ սուրերով ու բիրերով բռնելու զիս:

<span id="an_14:49">14:49</span> Ամէ՛ն օր ձեր քով էի եւ տաճարին մէջ կը
սորվեցնէի, ու չբռնեցիք զիս. բայց այս բոլոր բաները եղան՝ որպէսզի
գրուածները իրագործուին»:

<span id="an_14:50">14:50</span> Բոլոր աշակերտներն ալ թողուցին զայն եւ
փախան:

<span id="an_14:51">14:51</span> Երիտասարդ մը անոր կը հետեւէր՝ մերկ
մարմինին վրայ կտաւ մը առած, սակայն երիտասարդները բռնեցին զայն:

<span id="an_14:52">14:52</span> Ան ալ ձգեց կտաւը, եւ մերկ փախաւ
անոնցմէ:

<span id="an_14:53">14:53</span> Տարին Յիսուսը քահանայապետին, եւ անոր
քով համախմբուեցան բոլոր քահանայապետները, երէցներն ու դպիրները:

<span id="an_14:54">14:54</span> Պետրոս հեռուէն հետեւեցաւ անոր մինչեւ
ներսը՝ քահանայապետին գաւիթը. նստաւ սպասաւորներուն հետ, ու կը տաքնար
կրակին քով:

<span id="an_14:55">14:55</span> Քահանայապետները եւ ամբողջ ատեանը
վկայութիւն կը փնտռէին Յիսուսի դէմ՝ որպէսզի մեռցնեն զայն, ու չէին գտներ.

<span id="an_14:56">14:56</span> որովհետեւ շատեր սուտ վկայութիւն կու
տային անոր դէմ, բայց վկայութիւնները համանման չէին:

<span id="an_14:57">14:57</span> Ոմանք ալ կանգնեցան եւ սուտ վկայութիւն
տուին անոր դէմ՝ ըսելով.

<span id="an_14:58">14:58</span> «Մենք լսեցինք ատկէ՝ որ կ՚ըսէր. “Ես պիտի
քակեմ այդ ձեռակերտ տաճարը, եւ պիտի կառուցանեմ ուրիշ մը՝ անձեռակերտ՝ երեք
օրուան մէջ”»:

<span id="an_14:59">14:59</span> Բայց ա՛յդպէս ալ իրենց վկայութիւնը
համանման չէր:

<span id="an_14:60">14:60</span> Քահանայապետը՝ մէջտեղ կանգնելով՝ հարցուց
Յիսուսի. «Ոչի՞նչ կը պատասխանես. ի՞նչ կը վկայեն ասոնք քեզի դէմ»:

<span id="an_14:61">14:61</span> Բայց ան լուռ կը կենար ու ոչինչ կը
պատասխանէր: Քահանայապետը դարձեալ հարցուց անոր. «Դո՞ւն ես Քրիստոսը,
օրհնեա՜լ Աստուծոյ Որդին»:

<span id="an_14:62">14:62</span> Յիսուս ըսաւ. «Ե՛ս եմ. եւ պիտի տեսնէք
մարդու Որդին՝ բազմած Զօրութեան աջ կողմը, ու եկած երկինքի ամպերով»:

<span id="an_14:63">14:63</span> Քահանայապետը պատռեց իր հանդերձները եւ
ըսաւ. «Ա՛լ ի՞նչ պէտք ունինք վկաներու:

<span id="an_14:64">14:64</span> Լսեցի՛ք ատոր հայհոյութիւնը. ի՞նչ կը

<span id="an_14:65">14:65</span> Ոմանք սկսան թքնել անոր վրայ, ծածկել
անոր երեսը, կռփահարել զայն եւ ըսել անոր. «Մարգարէացի՛ր»: Իսկ
սպասաւորները ապտակ կը զարնէին անոր:

<span id="an_14:66">14:66</span> Պետրոս վարը՝ գաւիթն էր: Քահանայապետին
աղախիններէն մէկը եկաւ,

<span id="an_14:67">14:67</span> ու տեսնելով Պետրոսը՝ որ կը տաքնար,
նայեցաւ անոր եւ ըսաւ. «Դո՛ւն ալ Յիսուս Նազովրեցիին հետ էիր»:

<span id="an_14:68">14:68</span> Բայց ան ուրացաւ՝ ըսելով. «Չեմ գիտեր, ու
չեմ հասկնար թէ ի՛նչ կը խօսիս»: Երբ դուրս ելաւ՝ նախագաւիթը, աքաղաղը
կանչեց:

<span id="an_14:69">14:69</span> Աղախինը դարձեալ տեսաւ զայն, եւ սկսաւ
ըսել շուրջը կայնողներուն. «Ասիկա՛ ալ անոնցմէ է»:

<span id="an_14:70">14:70</span> Ան ալ դարձեալ ուրացաւ: Քիչ մը ատենէն
ետք՝ դարձեալ շուրջը կայնողները ըսին Պետրոսի. «Ճշմա՛րտապէս դուն անոնցմէ՛
ես, որովհետեւ Գալիլեացի ես, եւ խօսուածքդ ալ կը նմանի»:

<span id="an_14:71">14:71</span> Բայց ան սկսաւ նզովել ու երդում ընել՝
ըսելով. «Չեմ ճանչնար այդ մարդը՝ որուն մասին կը խօսիք»:

<span id="an_14:72">14:72</span> Իսկոյն երկրորդ անգամ աքաղաղը կանչեց:
Պետրոս յիշեց այն խօսքը՝ որ Յիսուս ըսեր էր իրեն. «Դեռ աքաղաղը երկու անգամ
չկանչած՝ դուն երե՛ք անգամ պիտի ուրանաս զիս». եւ սկսաւ լալ:

<span id="an_15:1">15:1</span> Առտուն՝ իսկոյն քահանայապետները,
երէցներուն ու դպիրներուն հետ, նաեւ ամբողջ ատեանը՝ խորհրդակցեցան, կապեցին
Յիսուսը ու տարին մատնեցին Պիղատոսի:

<span id="an_15:2">15:2</span> Պիղատոս հարցուց անոր. «Դո՞ւն ես
Հրեաներուն թագաւորը»: Ան ալ պատասխանեց անոր. «Դո՛ւն կ՚ըսես»:

<span id="an_15:3">15:3</span> Իսկ քահանայապետները կ՚ամբաստանէին զինք
շատ բաներով:

<span id="an_15:4">15:4</span> Պիղատոս դարձեալ հարցուց անոր. «Ոչի՞նչ կը
պատասխանես. նայէ՛, ո՜րքան կը վկայեն քեզի դէմ»:

<span id="an_15:5">15:5</span> Յիսուս տակաւին ոչինչ կը պատասխանէր,
այնպէս որ Պիղատոս զարմացաւ:

<span id="an_15:6">15:6</span> Տօնին ատենը բանտարկեալ մը կ՚արձակէր
անոնց, ո՛վ որ ուզէին:

<span id="an_15:7">15:7</span> Մէկը կար՝ Բարաբբա կոչուած, կապուած իրեն
հետ ապստամբողներուն հետ, որոնք մարդասպանութիւն գործած էին ապստամբութեան
ատեն:

<span id="an_15:8">15:8</span> Բազմութիւնը՝ գոռալով՝ սկսաւ խնդրել, որ
ընէ՝ ինչպէս միշտ կ՚ընէր իրենց:

<span id="an_15:9">15:9</span> Պիղատոս պատասխանեց անոնց. «Կ՚ուզէ՞ք որ
արձակեմ ձեզի Հրեաներուն թագաւորը».

<span id="an_15:10">15:10</span> որովհետեւ գիտէր թէ քահանայապետները
նախանձի՛ համար մատնած էին զայն:

<span id="an_15:11">15:11</span> Բայց քահանայապետները գրգռեցին
բազմութիւնը՝ որ ան փոխարէնը Բարաբբա՛ն արձակէ իրենց:

<span id="an_15:12">15:12</span> Դարձեալ Պիղատոս ըսաւ անոնց. «Հապա ի՞նչ
կ՚ուզէք որ ընեմ անոր՝ որ կը կոչէք Հրեաներուն թագաւորը»:

<span id="an_15:13">15:13</span> Անոնք դարձեալ աղաղակեցին. «Խաչէ՛ զայն»:

<span id="an_15:14">15:14</span> Պիղատոս ըսաւ անոնց. «Բայց ի՞նչ չարիք
ըրած է»: Անոնք ա՛լ աւելի կ՚աղաղակէին. «Խաչէ՛ զայն»:

<span id="an_15:15">15:15</span> Պիղատոս ալ՝ փափաքելով գոհացնել
բազմութիւնը՝ Բարաբբան արձակեց անոնց, ու խարազանելով Յիսուսը՝ յանձնեց
անոնց որպէսզի խաչուէր:

<span id="an_15:16">15:16</span> Զինուորները ներս տարին զայն՝ գաւիթը,
այսինքն՝ կառավարիչին պալատը, եւ հաւաքեցին ամբողջ գունդը:

<span id="an_15:17">15:17</span> Ծիրանի հագցուցին անոր, ու հիւսելով
փուշէ պսակ մը՝ դրին անոր գլուխը.

<span id="an_15:18">15:18</span> եւ սկսան բարեւել զայն ու ըսել.
«Ողջո՜յն, Հրեաներո՛ւ թագաւոր»:

<span id="an_15:19">15:19</span> Եղէգով կը զարնէին անոր գլուխին, կը
թքնէին անոր վրայ, եւ ծնրադրելով՝ կ՚երկրպագէին անոր:

<span id="an_15:20">15:20</span> Երբ ծաղրեցին զայն, հանեցին վրայէն
ծիրանին, հագցուցին անոր իր հանդերձները, ու դուրս տարին զայն՝ որպէսզի
խաչեն:

<span id="an_15:21">15:21</span> Ստիպեցին մէկը՝ Սիմոն Կիւրենացին,
Աղեքսանդրոսի ու Ռուփոսի հայրը, որ ատկէ կ՚անցնէր՝ արտէն գալով, որպէսզի
վերցնէ անոր խաչը:

<span id="an_15:22">15:22</span> Եւ տարին զայն Գողգոթա կոչուած տեղը, (որ
կը թարգմանուի՝ Գանկի տեղ,)

<span id="an_15:23">15:23</span> ու տուին անոր զմուռսով խառնուած գինի՝
որ խմէ. բայց ինք չառաւ:

<span id="an_15:24">15:24</span> Երբ խաչեցին զայն, բաժնեցին անոր
հանդերձները՝ վիճակ ձգելով անոնց վրայ, թէ ո՛վ՝ ի՛նչ պիտի առնէ:

<span id="an_15:25">15:25</span> Երրորդ ժամն՝՝ էր՝ երբ խաչեցին զայն:

<span id="an_15:26">15:26</span> Անոր ամբաստանագիրին վրայ գրուած էր.
«Հրեաներուն թագաւորը»:

<span id="an_15:27">15:27</span> Երկու աւազակներ ալ խաչեցին անոր հետ,
մէկը՝ անոր աջ կողմը, ու միւսը՝ ձախ կողմը:

<span id="an_15:28">15:28</span> Այսպէս իրագործուեցաւ գրուածը՝ որ կ՚ըսէ.
«Անօրէններու հետ սեպուեցաւ»:

<span id="an_15:29">15:29</span> Անոնք որ կ՚անցնէին՝ կը հայհոյէին անոր,
կը շարժէին իրենց գլուխը եւ կ՚ըսէին. «Աւա՜ղ, որ կը քակէիր տաճարը, ու կը
կառուցանէիր երեք օրուան մէջ.

<span id="an_15:30">15:30</span> փրկէ՛ դուն քեզ եւ իջի՛ր խաչէն»:

<span id="an_15:31">15:31</span> Նմանապէս քահանայապետներն ալ՝ դպիրներուն
հետ՝ իրենց մէջ ծաղրելով կ՚ըսէին. «Ուրիշները փրկեց, ինքզի՛նք չի կրնար
փրկել:

<span id="an_15:32">15:32</span> Այդ Քրիստոսը՝ Իսրայէլի թագաւորը՝ հի՛մա
թող իջնէ խաչէն, որպէսզի տեսնենք ու հաւատանք»: Իրեն հետ խաչուածներն ալ կը
նախատէին զայն:

<span id="an_15:33">15:33</span> Երբ վեցերորդ ժամը հասաւ՝ խաւար եղաւ
ամբողջ երկրին վրայ, մինչեւ իններորդ ժամը:

<span id="an_15:34">15:34</span> Իններորդ ժամուն՝ Յիսուս բարձրաձայն
գոչեց. «Էլոհի՜, Էլոհի՜, լամա՞ սաբաքթանի», որ կը թարգմանուի. «Իմ
Աստուա՜ծս, իմ Աստուա՜ծս, ինչո՞ւ լքեցիր զիս»:

<span id="an_15:35">15:35</span> Քովը կայնողներէն ոմանք, երբ լսեցին՝
ըսին. «Եղիա՛ն կը կանչէ»:

<span id="an_15:36">15:36</span> Ու մէկը վազեց, լեցուց սպունգ մը
քացախով, անցուց եղէգի մը եւ տուաւ անոր որ խմէ՝ ըսելով. «Թողուցէ՛ք,
տեսնենք թէ Եղիա պիտի գա՞յ՝ իջեցնելու զայն»:

<span id="an_15:37">15:37</span> Յիսուս բարձր ձայն մը արձակեց ու հոգին
տուաւ:

<span id="an_15:38">15:38</span> Տաճարին վարագոյրը երկուքի պատռեցաւ՝
վերէն վար:

<span id="an_15:39">15:39</span> Հարիւրապետը՝ որ կեցած էր անոր դիմաց,
տեսնելով որ ա՛յսպէս աղաղակելով հոգին տուաւ՝ ըսաւ. «Ճշմա՛րտապէս այս մարդը
Աստուծոյ Որդին էր»:

<span id="an_15:40">15:40</span> Կիներ ալ կային, որոնք հեռուէն կը
նայէին: Անոնց մէջ էին Մարիամ Մագդաղենացին, Կրտսեր Յակոբոսի ու Յովսէսի
մայրը՝ Մարիամ, եւ Սողովմէ,

<span id="an_15:41">15:41</span> որոնք՝ երբ Յիսուս Գալիլեա էր՝ հետեւեր
ու սպասարկեր էին անոր: Ուրիշ շատ կիներ ալ կային՝ որոնք անոր հետ ելած էին
Երուսաղէմ:

<span id="an_15:42">15:42</span> Երբ արդէն իրիկուն եղաւ, քանի որ այդ օրը
Ուրբաթ էր, այսինքն՝ Շաբաթին նախորդ օրը,

<span id="an_15:43">15:43</span> Յովսէփ Արիմաթեացին, մեծայարգ խորհրդական
մը՝ որ ի՛նք ալ կը սպասէր Աստուծոյ թագաւորութեան, եկաւ ու յանդգնութեամբ
մտաւ Պիղատոսի քով, եւ խնդրեց Յիսուսի մարմինը:

<span id="an_15:44">15:44</span> Պիղատոս զարմացաւ որ արդէն մեռած է:
Կանչելով հարիւրապետը՝ հարցուց անոր թէ շատո՛նց մեռած է:

<span id="an_15:45">15:45</span> Ու երբ ստոյգը գիտցաւ հարիւրապետէն,
պարգեւեց մարմինը Յովսէփի:

<span id="an_15:46">15:46</span> Յովսէփ կտաւ գնեց, եւ իջեցնելով զայն՝
փաթթեց կտաւով, դրաւ գերեզմանի մը մէջ՝ որ փորուած էր ժայռի մէջ, ու քար մը
գլորեց գերեզմանին դուռը:

<span id="an_15:47">15:47</span> Մարիամ Մագդաղենացին, եւ Յովսէսի մայրը՝
Մարիամ, կը նայէին թէ ո՛ւր կը դնէին զայն:

<span id="an_16:1">16:1</span> Երբ Շաբաթը անցաւ, Մարիամ Մագդաղենացին,
Յակոբոսի մայրը՝ Մարիամ, ու Սողովմէ բոյրեր գնեցին, որպէսզի երթան եւ օծեն
զայն:

<span id="an_16:2">16:2</span> Մէկշաբթի առտուն՝ շատ կանուխ, երբ արեւը կը
ծագէր, գացին գերեզմանը,

<span id="an_16:3">16:3</span> ու կ՚ըսէին իրարու. «Մեզի համար ո՞վ պիտի
գլորէ քարը՝ գերեզմանին դռնէն»,

<span id="an_16:4">16:4</span> որովհետեւ շատ մեծ էր: Բայց նայելով՝ տեսան
թէ քարը գլորուած էր:

<span id="an_16:5">16:5</span> Մտնելով գերեզմանէն ներս՝ տեսան երիտասարդ
մը, որ նստած էր աջ կողմը՝ ճերմակ պարեգօտ հագած, եւ զարհուրեցան:

<span id="an_16:6">16:6</span> Ան ալ ըսաւ անոնց. «Մի՛ զարհուրիք. Յիսուս
Նազովրեցի՛ն կը փնտռէք՝ որ խաչուեցաւ: Ան յարութի՛ւն առաւ, հոս չէ: Ահա՛
այն տեղը՝ ուր դրին զայն:

<span id="an_16:7">16:7</span> Ուրեմն գացէ՛ք, ըսէ՛ք անոր աշակերտներուն
ու Պետրոսի՝ թէ ահա՛ ձեզմէ առաջ կ՚երթայ Գալիլեա. հո՛ն պիտի տեսնէք զայն՝
ինչպէս ըսաւ ձեզի»:

<span id="an_16:8">16:8</span> Անոնք՝ դողով եւ հիացումով համակուած՝
դուրս ելան՝՝ ու փախան գերեզմանէն, եւ ո՛չ մէկ բան ըսին ոեւէ մէկուն, քանի
որ կը վախնային:

<span id="an_16:9">16:9</span> Յիսուս՝ Մէկշաբթի առտուն յարութիւն առած՝
նախ երեւցաւ Մարիամ Մագդաղենացիին, որմէ հաներ էր եօթը դեւ:

<span id="an_16:10">16:10</span> Ինք ալ գնաց եւ պատմեց անոնց՝ որոնք անոր
հետ էին, մինչ կը սգային ու կու լային:

<span id="an_16:11">16:11</span> Իսկ երբ անոնք լսեցին թէ ան ողջ է եւ ինք
տեսած է զայն, չհաւատացին:

<span id="an_16:12">16:12</span> Յետոյ ուրիշ կերպարանքով երեւցաւ անոնցմէ
երկուքին՝ որոնք քալելով արտը կ՚երթային:

<span id="an_16:13">16:13</span> Իրենք ալ գացին ու պատմեցին միւսներուն.
բայց իրե՛նց ալ չհաւատացին:

<span id="an_16:14">16:14</span> Յետոյ երեւցաւ տասնմէկին՝ երբ սեղան
նստած էին, ու կշտամբեց անոնց անհաւատութիւնը եւ սիրտի կարծրութիւնը,
որովհետեւ չհաւատացին անոնց՝ որոնց ինք երեւցաւ՝ մեռելներէն յարութիւն
առած:

<span id="an_16:15">16:15</span> Նաեւ ըսաւ անոնց. «Գացէ՛ք ամբողջ
աշխարհը, ու քարոզեցէ՛ք աւետարանը բոլոր արարածներուն:

<span id="an_16:16">16:16</span> Ա՛ն որ հաւատայ եւ մկրտուի՝ պիտի փրկուի.
ա՛ն որ չհաւատայ՝ պիտի դատապարտուի:

<span id="an_16:17">16:17</span> Սա՛ նշանները պիտի հետեւին անոնց՝ որ կը
հաւատան.- դեւե՛ր դուրս պիտի հանեն իմ անունովս, նո՛ր լեզուներ պիտի խօսին,

<span id="an_16:18">16:18</span> օձե՛ր պիտի առնեն իրենց ձեռքերուն մէջ,
ու եթէ խմեն մահառիթ դեղ մը՝ պիտի չվնասէ իրենց. ձեռք պիտի դնեն
հիւանդներու վրայ, եւ անոնք պիտի առողջանան»:

<span id="an_16:19">16:19</span> Այսպէս՝ անոնց հետ խօսելէն ետք՝ Տէրը
համբարձաւ երկինք, ու բազմեցաւ Աստուծոյ աջ կողմը.

<span id="an_16:20">16:20</span> իսկ անոնք գացին եւ կը քարոզէին
ամէնուրեք: Տէրը անոնց գործակից էր, ու կը հաստատէր խօսքը՝ ընկերացող
նշաններով: Ամէն:
