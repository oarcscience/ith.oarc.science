### Aramaic Bible: 1 Thessalonians

<span id="an_1:1">1:1</span> ܦܘܠܘܤ ܘܤܠܘܢܘܤ ܘܛܝܡܬܐܤ ܠܥܕܬܐ ܕܬܤܠܘܢܝܩܝܐ
ܕܒܐܠܗܐ ܐܒܐ ܘܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܀

<span id="an_1:2">1:2</span> ܡܘܕܝܢܢ ܠܐܠܗܐ ܒܟܠܙܒܢ ܥܠ ܟܠܟܘܢ ܘܡܬܕܟܪܝܢܢ ܠܟܘܢ
ܒܨܠܘܬܢ ܐܡܝܢܐܝܬ ܀

<span id="an_1:3">1:3</span> ܘܥܗܕܝܢܢ ܩܕܡ ܐܠܗܐ ܐܒܐ ܥܒܕܐ ܕܗܝܡܢܘܬܟܘܢ ܘܥܡܠܐ
ܕܚܘܒܟܘܢ ܘܡܤܝܒܪܢܘܬܐ ܕܤܒܪܟܘܢ ܕܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:4">1:4</span> ܝܕܥܝܢܢ ܓܝܪ ܓܒܝܘܬܟܘܢ ܐܚܝ ܚܒܝܒܘܗܝ ܕܐܠܗܐ ܀

<span id="an_1:5">1:5</span> ܡܛܠ ܕܡܤܒܪܢܘܬܢ ܠܐ ܗܘܐ ܒܡܠܐ ܒܠܚܘܕ ܗܘܬ ܠܘܬܟܘܢ
ܐܠܐ ܐܦ ܒܚܝܠܐ ܘܒܪܘܚܐ ܕܩܘܕܫܐ ܘܒܦܝܤܐ ܫܪܝܪܐ ܐܦ ܐܢܬܘܢ ܝܕܥܝܢ ܐܢܬܘܢ ܐܝܟܢܐ ܗܘܝܢ
ܒܝܢܬܟܘܢ ܡܛܠܬܟܘܢ ܀

<span id="an_1:6">1:6</span> ܘܐܢܬܘܢ ܒܢ ܐܬܕܡܝܬܘܢ ܘܒܡܪܢ ܕܩܒܠܬܘܢ ܡܠܬܐ
ܒܐܘܠܨܢܐ ܪܒܐ ܘܒܚܕܘܬܐ ܕܪܘܚܐ ܕܩܘܕܫܐ ܀

<span id="an_1:7">1:7</span> ܘܗܘܝܬܘܢ ܕܡܘܬܐ ܠܟܠܗܘܢ ܡܗܝܡܢܐ ܕܐܝܬ ܒܡܩܕܘܢܝܐ
ܘܒܐܟܐܝܐ ܀

<span id="an_1:8">1:8</span> ܡܢܟܘܢ ܓܝܪ ܐܫܬܡܥܬ ܡܠܬܗ ܕܡܪܢ ܠܐ ܒܠܚܘܕ
ܒܡܩܕܘܢܝܐ ܘܒܐܟܐܝܐ ܐܠܐ ܒܟܠ ܐܬܪ ܗܝܡܢܘܬܟܘܢ ܕܒܐܠܗܐ ܐܫܬܡܥܬ ܐܝܟܢܐ ܕܠܐ ܢܤܬܢܩ
ܕܢܐܡܪ ܥܠܝܟܘܢ ܡܕܡ ܀

<span id="an_1:9">1:9</span> ܗܢܘܢ ܓܝܪ ܡܫܬܥܝܢ ܐܝܢܐ ܡܥܠܢܐ ܗܘܐ ܠܢ ܠܘܬܟܘܢ
ܘܐܝܟܢܐ ܐܬܦܢܝܬܘܢ ܠܘܬ ܐܠܗܐ ܡܢ ܕܚܠܬ ܦܬܟܪܐ ܕܬܦܠܚܘܢ ܠܐܠܗܐ ܚܝܐ ܘܫܪܝܪܐ ܀

<span id="an_1:10">1:10</span> ܟܕ ܡܤܟܝܬܘܢ ܠܒܪܗ ܡܢ ܫܡܝܐ ܠܝܫܘܥ ܗܘ ܕܐܩܝܡ ܡܢ
ܒܝܬ ܡܝܬܐ ܕܗܘ ܡܦܨܐ ܠܢ ܡܢ ܪܘܓܙܐ ܕܐܬܐ ܀

<span id="an_2:1">2:1</span> ܘܐܢܬܘܢ ܝܕܥܝܢ ܐܢܬܘܢ ܐܚܝ ܕܡܥܠܢܢ ܕܠܘܬܟܘܢ ܠܐ
ܗܘܐ ܤܪܝܩܐܝܬ ܗܘܐ ܀

<span id="an_2:2">2:2</span> ܐܠܐ ܠܘܩܕܡ ܚܫܢܢ ܘܐܨܛܥܪܢܢ ܐܝܟ ܕܝܕܥܝܬܘܢ
ܒܦܝܠܝܦܘܤ ܘܗܝܕܝܢ ܒܐܓܘܢܐ ܪܒܐ ܡܠܠܢ ܥܡܟܘܢ ܒܦܪܗܤܝܐ ܕܐܠܗܢ ܤܒܪܬܗ ܕܡܫܝܚܐ ܀

<span id="an_2:3">2:3</span> ܒܘܝܐܢ ܓܝܪ ܠܐ ܗܘܐ ܡܢ ܛܥܝܘܬܐ ܐܦܠܐ ܡܢ ܛܢܦܘܬܐ
ܐܦܠܐ ܒܢܟܠܐ ܀

<span id="an_2:4">2:4</span> ܐܠܐ ܐܝܟ ܡܐ ܕܒܩܝܢܢ ܠܐܠܗܐ ܕܬܬܗܝܡܢ ܤܒܪܬܗ ܗܟܢܐ
ܡܡܠܠܝܢܢ ܠܐ ܐܝܟ ܕܠܒܢܝ ܐܢܫܐ ܢܫܦܪ ܐܠܐ ܠܐܠܗܐ ܗܘ ܕܒܩܐ ܠܒܘܬܢ ܀

<span id="an_2:5">2:5</span> ܐܦܠܐ ܓܝܪ ܡܡܬܘܡ ܐܬܚܫܚܢ ܒܡܡܠܠܐ ܫܕܠܐ ܐܝܟ
ܕܝܕܥܝܢ ܐܢܬܘܢ ܘܠܐ ܒܥܠܬܐ ܕܝܥܢܘܬܐ ܐܠܗܐ ܤܗܕ ܀

<span id="an_2:6">2:6</span> ܘܠܐ ܒܥܝܢ ܫܘܒܚܐ ܡܢ ܒܢܝܢܫܐ ܠܐ ܡܢܟܘܢ ܘܠܐ ܡܢ
ܐܚܪܢܐ ܟܕ ܡܫܟܚܝܢ ܗܘܝܢ ܝܩܝܪܐ ܠܡܗܘܐ ܐܝܟ ܫܠܝܚܐ ܕܡܫܝܚܐ ܀

<span id="an_2:7">2:7</span> ܐܠܐ ܗܘܝܢ ܒܝܢܬܟܘܢ ܡܟܝܟܐ ܘܐܝܟ ܡܪܒܝܢܝܬܐ ܕܡܚܒܒܐ
ܒܢܝܗ ܀

<span id="an_2:8">2:8</span> ܗܟܢܐ ܐܦ ܚܢܢ ܡܚܒܒܝܢܢ ܘܡܤܘܚܝܢܢ ܠܡܬܠ ܠܟܘܢ ܠܐ
ܒܠܚܘܕ ܤܒܪܬܗ ܕܐܠܗܐ ܐܠܐ ܐܦ ܢܦܫܢ ܡܛܠ ܕܚܒܝܒܝܢ ܐܢܬܘܢ ܀

<span id="an_2:9">2:9</span> ܥܗܕܝܢ ܐܢܬܘܢ ܓܝܪ ܐܚܝܢ ܕܠܐܝܢ ܗܘܝܢ ܘܥܡܠܝܢ ܒܥܒܕ
ܐܝܕܝܢ ܒܠܠܝܐ ܘܒܐܝܡܡܐ ܕܥܠ ܚܕ ܡܢܟܘܢ ܠܐ ܢܐܩܪ ܀

<span id="an_2:10">2:10</span> ܐܢܬܘܢ ܤܗܕܝܢ ܘܐܠܗܐ ܐܝܟܢܐ ܐܟܪܙܢ ܠܟܘܢ ܤܒܪܬܗ
ܕܐܠܗܐ ܕܟܝܐܝܬ ܘܟܐܢܐܝܬ ܘܕܠܐ ܪܫܝܢ ܗܘܝܢ ܠܘܬ ܟܠܗܘܢ ܡܗܝܡܢܐ ܀

<span id="an_2:11">2:11</span> ܐܝܟܢܐ ܕܝܕܥܝܢ ܐܢܬܘܢ ܕܡܢ ܚܕ ܚܕ ܡܢܟܘܢ ܐܝܟ
ܐܒܐ ܡܢ ܒܢܘܗܝ ܒܥܝܢ ܗܘܝܢ ܘܡܠܝܢ ܗܘܝܢ ܒܠܒܟܘܢ ܘܡܤܗܕܝܢܢ ܠܟܘܢ ܀

<span id="an_2:12">2:12</span> ܕܬܗܠܟܘܢ ܐܝܟ ܕܝܐܐ ܠܐܠܗܐ ܗܘ ܕܩܪܟܘܢ ܠܡܠܟܘܬܗ
ܘܠܫܘܒܚܗ ܀

<span id="an_2:13">2:13</span> ܡܛܠ ܗܢܐ ܐܦ ܚܢܢ ܡܘܕܝܢܢ ܐܡܝܢܐܝܬ ܠܐܠܗܐ ܕܡܠܬܐ
ܕܐܠܗܐ ܕܩܒܠܬܘܢ ܡܢܢ ܠܐ ܗܘܐ ܐܝܟ ܡܠܬܐ ܕܒܢܝܢܫܐ ܩܒܠܬܘܢܗ ܐܠܐ ܐܝܟܢܐ ܕܐܝܬܝܗ
ܫܪܝܪܐܝܬ ܡܠܬܐ ܕܐܠܗܐ ܘܗܝ ܒܥܒܕܐ ܡܤܬܥܪܐ ܒܟܘܢ ܒܐܝܠܝܢ ܕܡܗܝܡܢܝܢ ܀

<span id="an_2:14">2:14</span> ܐܢܬܘܢ ܕܝܢ ܐܚܝ ܐܬܕܡܝܬܘܢ ܠܥܕܬܐ ܕܐܠܗܐ ܕܒܝܗܘܕ
ܗܠܝܢ ܕܒܝܫܘܥ ܡܫܝܚܐ ܕܗܟܢ ܤܝܒܪܬܘܢ ܐܦ ܐܢܬܘܢ ܡܢ ܒܢܝ ܫܪܒܬܟܘܢ ܐܝܟܢܐ ܕܐܦ ܗܢܘܢ ܡܢ
ܝܗܘܕܝܐ ܀

<span id="an_2:15">2:15</span> ܗܢܘܢ ܕܠܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܩܛܠܘ ܘܠܢܒܝܐ ܕܡܢܗܘܢ
ܘܠܢ ܪܕܦܘ ܘܠܐܠܗܐ ܠܐ ܫܦܪܝܢ ܘܥܒܝܕܝܢ ܤܩܘܒܠܐ ܠܟܠܗܘܢ ܒܢܝ ܐܢܫܐ ܀

<span id="an_2:16">2:16</span> ܕܟܠܝܢ ܠܢ ܕܢܡܠܠ ܥܡ ܥܡܡܐ ܕܢܚܘܢ ܠܡܫܠܡܘ
ܚܛܗܝܗܘܢ ܒܟܠܙܒܢ ܡܛܝ ܕܝܢ ܥܠܝܗܘܢ ܪܘܓܙܐ ܥܕܡܐ ܠܚܪܬܐ ܀

<span id="an_2:17">2:17</span> ܚܢܢ ܕܝܢ ܐܚܝܢ ܗܘܝܢ ܝܬܡܐ ܡܢܟܘܢ ܙܒܢܐ ܕܫܥܬܐ
ܒܐܦܝܢ ܘܠܐ ܒܠܒܢ ܘܝܬܝܪܐܝܬ ܐܬܚܦܛܢ ܠܡܚܙܐ ܐܦܝܟܘܢ ܒܚܘܒܐ ܤܓܝܐܐ ܀

<span id="an_2:18">2:18</span> ܘܨܒܝܢ ܕܢܐܬܐ ܠܘܬܟܘܢ ܐܢܐ ܦܘܠܘܤ ܚܕܐ ܙܒܢ
ܘܬܪܬܝܢ ܘܥܘܟܢܝ ܤܛܢܐ ܀

<span id="an_2:19">2:19</span> ܐܝܢܘ ܓܝܪ ܤܒܪܢ ܘܚܕܘܬܢ ܘܟܠܝܠܐ ܕܫܘܒܗܪܢ ܐܠܐ
ܐܢ ܐܢܬܘܢ ܩܕܡ ܡܪܢ ܝܫܘܥ ܒܡܐܬܝܬܗ ܀

<span id="an_2:20">2:20</span> ܐܢܬܘܢ ܐܢܘܢ ܓܝܪ ܬܫܒܘܚܬܢ ܘܚܕܘܬܢ ܀

<span id="an_3:1">3:1</span> ܘܡܛܠ ܕܠܐ ܚܡܤܢܢ ܨܒܝܢ ܠܡܦܫ ܒܐܬܢܘܤ ܒܠܚܘܕܝܢ ܀

<span id="an_3:2">3:2</span> ܘܢܫܕܪ ܠܘܬܟܘܢ ܠܛܝܡܬܐܘܤ ܐܚܘܢ ܘܡܫܡܫܢܐ ܕܐܠܗܐ
ܘܡܥܕܪܢܢ ܒܤܒܪܬܗ ܕܡܫܝܚܐ ܕܢܫܪܪܟܘܢ ܘܢܒܥܐ ܡܢܟܘܢ ܥܠ ܗܝܡܢܘܬܟܘܢ ܀

<span id="an_3:3">3:3</span> ܕܠܐ ܐܢܫ ܡܢܟܘܢ ܬܬܩܛܥ ܠܗ ܒܐܘܠܨܢܐ ܗܠܝܢ ܐܢܬܘܢ
ܓܝܪ ܝܕܥܝܢ ܐܢܬܘܢ ܕܠܗܕܐ ܗܘ ܤܝܡܝܢܢ ܀

<span id="an_3:4">3:4</span> ܐܦ ܟܕ ܠܘܬܟܘܢ ܗܘܝܢ ܓܝܪ ܩܕܡܢ ܐܡܪܢ ܠܟܘܢ
ܕܥܬܝܕܝܢܢ ܠܡܬܐܠܨܘ ܐܝܟܢܐ ܕܝܕܥܝܢ ܐܢܬܘܢ ܕܗܘܐ ܀

<span id="an_3:5">3:5</span> ܡܛܠ ܗܢܐ ܐܦ ܐܢܐ ܠܐ ܚܡܤܢܬ ܥܕܡܐ ܕܫܕܪܬ ܕܐܕܥ
ܗܝܡܢܘܬܟܘܢ ܕܕܠܡܐ ܢܢܤܝܟܘܢ ܡܢܤܝܢܐ ܘܢܗܘܐ ܥܡܠܢ ܤܪܝܩܐ ܀

<span id="an_3:6">3:6</span> ܗܫܐ ܕܝܢ ܡܢ ܕܐܬܐ ܠܘܬܢ ܛܝܡܬܐܘܤ ܡܢ ܨܐܕܝܟܘܢ
ܘܤܒܪܢ ܥܠ ܗܝܡܢܘܬܟܘܢ ܘܥܠ ܚܘܒܟܘܢ ܘܕܐܝܬ ܠܘܬܟܘܢ ܥܘܗܕܢܢ ܛܒܐ ܒܟܠ ܥܕܢ ܘܤܘܝܢ
ܐܢܬܘܢ ܠܡܚܙܝܢ ܐܝܟܢܐ ܕܐܦ ܚܢܢ ܠܟܘܢ ܀

<span id="an_3:7">3:7</span> ܡܛܠ ܗܢܐ ܐܬܒܝܐܢ ܒܟܘܢ ܐܚܝܢ ܥܠ ܟܠܗܝܢ ܥܩܬܢ
ܘܐܘܠܨܢܝܢ ܡܛܠ ܗܝܡܢܘܬܟܘܢ ܀

<span id="an_3:8">3:8</span> ܘܗܫܐ ܗܘ ܚܝܝܢܢ ܐܢ ܐܢܬܘܢ ܬܬܩܝܡܘܢ ܒܡܪܢ ܀

<span id="an_3:9">3:9</span> ܐܝܕܐ ܓܝܪ ܬܘܕܝܬܐ ܡܫܟܚܝܢܢ ܠܡܦܪܥ ܚܠܦܝܟܘܢ ܠܐܠܗܐ
ܥܠ ܟܠ ܚܕܘܬܐ ܕܚܕܝܢܢ ܡܛܠܬܟܘܢ ܀

<span id="an_3:10">3:10</span> ܐܠܐ ܕܩܕܡ ܐܠܗܐ ܝܬܝܪܐܝܬ ܢܬܟܫܦ ܒܠܠܝܐ ܘܒܐܝܡܡܐ
ܕܢܚܙܐ ܐܦܝܟܘܢ ܘܢܓܡܘܪ ܡܐ ܕܚܤܝܪܐ ܗܝܡܢܘܬܟܘܢ ܀

<span id="an_3:11">3:11</span> ܗܘ ܕܝܢ ܐܠܗܐ ܐܒܘܢ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܢܬܪܘܨ
ܐܘܪܚܢ ܠܘܬܟܘܢ ܀

<span id="an_3:12">3:12</span> ܘܢܤܓܐ ܘܢܝܬܪ ܚܘܒܟܘܢ ܕܚܕ ܠܘܬ ܚܕ ܘܕܠܘܬ ܟܠܢܫ
ܐܝܟܢܐ ܕܚܢܢ ܡܚܒܝܢܢ ܠܟܘܢ ܀

<span id="an_3:13">3:13</span> ܘܢܩܝܡ ܠܒܘܬܟܘܢ ܕܠܐ ܪܫܝܢ ܒܩܕܝܫܘܬܐ ܩܕܡ ܐܠܗܐ
ܐܒܘܢ ܒܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܥܡ ܟܠܗܘܢ ܩܕܝܫܘܗܝ ܀

<span id="an_4:1">4:1</span> ܡܟܝܠ ܐܚܝ ܒܥܝܢܢ ܡܢܟܘܢ ܘܡܬܟܫܦܝܢܢ ܠܟܘܢ ܒܡܪܢ
ܝܫܘܥ ܕܐܝܟ ܕܩܒܠܬܘܢ ܡܢܢ ܐܝܟܢ ܘܠܐ ܠܟܘܢ ܕܬܗܠܟܘܢ ܘܬܫܦܪܘܢ ܠܐܠܗܐ ܕܝܬܝܪܐܝܬ
ܬܘܤܦܘܢ ܀

<span id="an_4:2">4:2</span> ܝܕܥܝܢ ܐܢܬܘܢ ܓܝܪ ܐܝܠܝܢ ܦܘܩܕܢܐ ܝܗܒܢ ܠܟܘܢ ܒܡܪܢ
ܝܫܘܥ ܀

<span id="an_4:3">4:3</span> ܗܢܘ ܓܝܪ ܨܒܝܢܗ ܕܐܠܗܐ ܩܕܝܫܘܬܟܘܢ ܘܕܬܗܘܘܢ
ܦܪܝܩܝܢ ܡܢ ܟܠ ܙܢܝܘܬܐ ܀

<span id="an_4:4">4:4</span> ܘܢܗܘܐ ܝܕܥ ܐܢܫ ܐܢܫ ܡܢܟܘܢ ܠܡܩܢܐ ܡܐܢܗ ܒܩܕܝܫܘܬܐ
ܘܒܐܝܩܪܐ ܀

<span id="an_4:5">4:5</span> ܘܠܐ ܒܚܫܐ ܕܪܓܬܐ ܐܝܟ ܫܪܟܐ ܕܥܡܡܐ ܗܢܘܢ ܕܠܐ
ܝܕܥܝܢ ܠܐܠܗܐ ܀

<span id="an_4:6">4:6</span> ܘܠܐ ܬܗܘܘܢ ܡܫܝܚܝܢ ܠܡܥܒܪ ܘܠܡܥܠܒ ܐܢܫ ܠܐܚܘܗܝ
ܒܗܕܐ ܨܒܘܬܐ ܡܛܠ ܕܡܪܢ ܗܘ ܬܒܘܥܐ ܥܠ ܗܠܝܢ ܟܠܗܝܢ ܐܝܟܢܐ ܕܐܦ ܡܢ ܩܕܡ ܙܒܢܐ ܐܡܪܢ
ܠܟܘܢ ܘܤܗܕܢ ܀

<span id="an_4:7">4:7</span> ܠܐ ܓܝܪ ܩܪܟܘܢ ܐܠܗܐ ܠܛܢܦܘܬܐ ܐܠܐ ܠܩܕܝܫܘܬܐ ܀

<span id="an_4:8">4:8</span> ܡܟܝܠ ܡܢ ܕܛܠܡ ܠܐ ܗܘܐ ܠܒܪܢܫܐ ܛܠܡ ܐܠܐ ܠܐܠܗܐ ܗܘ
ܕܝܗܒ ܒܟܘܢ ܪܘܚܗ ܩܕܝܫܬܐ ܀

<span id="an_4:9">4:9</span> ܥܠ ܚܘܒܐ ܕܝܢ ܕܐܚܐ ܠܐ ܤܢܝܩܝܬܘܢ ܠܡܟܬܒ ܠܟܘܢ
ܐܢܬܘܢ ܓܝܪ ܩܢܘܡܟܘܢ ܡܠܦܐ ܐܢܬܘܢ ܕܐܠܗܐ ܕܬܚܒܘܢ ܚܕ ܠܚܕ ܀

<span id="an_4:10">4:10</span> ܐܦ ܥܒܕܝܢ ܐܢܬܘܢ ܗܟܢܐ ܠܟܠܗܘܢ ܐܚܐ ܕܒܟܠܗ
ܡܩܕܘܢܝܐ ܒܥܐ ܐܢܐ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܕܬܬܝܬܪܘܢ ܀

<span id="an_4:11">4:11</span> ܘܬܬܚܦܛܘܢ ܕܬܗܘܘܢ ܫܠܝܢ ܘܥܢܝܢ ܒܤܘܥܪܢܝܟܘܢ
ܘܗܘܝܬܘܢ ܦܠܚܝܢ ܒܐܝܕܝܟܘܢ ܐܝܟܢܐ ܕܦܩܕܢܟܘܢ ܀

<span id="an_4:12">4:12</span> ܕܬܗܘܘܢ ܡܗܠܟܝܢ ܒܐܤܟܡܐ ܠܘܬ ܒܪܝܐ ܘܥܠ ܐܢܫ ܠܐ
ܬܤܬܢܩܘܢ ܀

<span id="an_4:13">4:13</span> ܨܒܐ ܐܢܐ ܕܝܢ ܕܬܕܥܘܢ ܐܚܝ ܕܥܠ ܐܝܠܝܢ ܕܕܡܟܝܢ
ܠܐ ܬܗܘܐ ܟܪܝܐ ܠܟܘܢ ܐܝܟ ܫܪܟܐ ܕܐܢܫܐ ܕܤܒܪܐ ܠܝܬ ܠܗܘܢ ܀

<span id="an_4:14">4:14</span> ܐܢ ܓܝܪ ܡܗܝܡܢܝܢܢ ܕܝܫܘܥ ܡܝܬ ܘܩܡ ܗܟܢܐ ܐܦ
ܐܠܗܐ ܠܐܝܠܝܢ ܕܕܡܟܘ ܒܝܫܘܥ ܡܝܬܐ ܥܡܗ ܀

<span id="an_4:15">4:15</span> ܗܕܐ ܕܝܢ ܐܡܪܝܢܢ ܠܟܘܢ ܒܡܠܬܗ ܕܡܪܢ ܕܚܢܢ ܐܝܠܝܢ
ܕܡܫܬܚܪܝܢܢ ܒܡܐܬܝܬܗ ܕܡܪܢ ܕܚܝܝܢܢ ܠܐ ܢܕܪܟ ܠܐܝܠܝܢ ܕܕܡܟܘ ܀

<span id="an_4:16">4:16</span> ܡܛܠ ܕܗܘ ܡܪܢ ܒܦܘܩܕܢܐ ܘܒܩܠܐ ܕܪܝܫ ܡܠܐܟܐ
ܘܒܩܪܢܐ ܕܐܠܗܐ ܢܚܬ ܡܢ ܫܡܝܐ ܘܡܝܬܐ ܕܒܡܫܝܚܐ ܢܩܘܡܘܢ ܠܘܩܕܡ ܀

<span id="an_4:17">4:17</span> ܘܗܝܕܝܢ ܚܢܢ ܐܝܠܝܢ ܕܡܫܬܚܪܝܢܢ ܕܚܝܝܢܢ ܢܬܚܛܦ
ܥܡܗܘܢ ܐܟܚܕܐ ܒܥܢܢܐ ܠܐܘܪܥܗ ܕܡܪܢ ܒܐܐܪ ܘܗܟܢܐ ܒܟܠܙܒܢ ܥܡ ܡܪܢ ܢܗܘܐ ܀

<span id="an_4:18">4:18</span> ܗܘܘ ܗܟܝܠ ܡܒܝܐܝܢ ܚܕ ܠܚܕ ܒܡܠܐ ܗܠܝܢ ܀

<span id="an_5:1">5:1</span> ܥܠ ܙܒܢܐ ܕܝܢ ܘܥܕܢܐ ܐܚܝ ܠܐ ܤܢܝܩܝܬܘܢ ܠܡܟܬܒ
ܠܟܘܢ ܀

<span id="an_5:2">5:2</span> ܐܢܬܘܢ ܓܝܪ ܫܪܝܪܐܝܬ ܝܕܥܝܢ ܐܢܬܘܢ ܕܝܘܡܗ ܕܡܪܢ
ܐܝܟ ܓܢܒܐ ܒܠܠܝܐ ܗܟܢܐ ܐܬܐ ܀

<span id="an_5:3">5:3</span> ܟܕ ܢܐܡܪܘܢ ܕܫܠܡܐ ܗܘ ܘܫܝܢܐ ܘܗܝܕܝܢ ܡܢܫܠܝܐ ܢܩܘܡ
ܥܠܝܗܘܢ ܐܒܕܢܐ ܐܝܟ ܚܒܠܐ ܥܠ ܒܛܢܬܐ ܘܠܐ ܢܡܕܘܢ ܀

<span id="an_5:4">5:4</span> ܐܢܬܘܢ ܕܝܢ ܐܚܝ ܠܐ ܗܘܝܬܘܢ ܒܚܫܘܟܐ ܕܗܘ ܝܘܡܐ ܐܝܟ
ܓܢܒܐ ܢܕܪܟܟܘܢ ܀

<span id="an_5:5">5:5</span> ܟܠܟܘܢ ܓܝܪ ܒܢܝ ܢܘܗܪܐ ܐܢܬܘܢ ܘܒܢܝ ܐܝܡܡܐ ܘܠܐ
ܗܘܝܬܘܢ ܒܢܝ ܠܠܝܐ ܘܠܐ ܒܢܝ ܚܫܘܟܐ ܀

<span id="an_5:6">5:6</span> ܠܐ ܢܕܡܟ ܗܟܝܠ ܐܝܟ ܫܪܟܐ ܐܠܐ ܗܘܝܢ ܥܝܪܝܢ
ܘܡܗܘܢܝܢ ܀

<span id="an_5:7">5:7</span> ܐܝܠܝܢ ܓܝܪ ܕܕܡܟܝܢ ܒܠܠܝܐ ܗܘ ܕܡܟܝܢ ܘܐܝܠܝܢ
ܕܪܘܝܢ ܒܠܠܝܐ ܗܘ ܪܘܝܢ ܀

<span id="an_5:8">5:8</span> ܚܢܢ ܕܝܢ ܕܒܢܝ ܐܝܡܡܐ ܚܢܢ ܗܘܝܢ ܥܝܪܝܢ ܒܪܥܝܢܢ
ܘܠܒܝܫܝܢ ܫܪܝܢܐ ܕܗܝܡܢܘܬܐ ܘܕܚܘܒܐ ܘܢܤܝܡ ܤܢܘܪܬܐ ܕܤܒܪܐ ܕܚܝܐ ܀

<span id="an_5:9">5:9</span> ܡܛܠ ܕܠܐ ܤܡܢ ܐܠܗܐ ܠܪܘܓܙܐ ܐܠܐ ܠܩܢܝܢܐ ܕܚܝܐ
ܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_5:10">5:10</span> ܗܘ ܕܡܝܬ ܥܠ ܐܦܝܢ ܕܐܢ ܥܝܪܝܢܢ ܘܐܢ ܕܡܟܝܢܢ
ܐܟܚܕܐ ܥܡܗ ܢܚܐ ܀

<span id="an_5:11">5:11</span> ܡܛܠ ܗܢܐ ܒܝܐܘ ܚܕ ܠܚܕ ܘܒܢܘ ܚܕ ܠܚܕ ܐܝܟܢܐ ܕܐܦ
ܥܒܕܝܬܘܢ ܀

<span id="an_5:12">5:12</span> ܒܥܝܢܢ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܕܗܘܝܬܘܢ ܝܕܥܝܢ ܠܐܝܠܝܢ
ܕܠܐܝܢ ܒܟܘܢ ܘܩܝܡܝܢ ܒܐܦܝܟܘܢ ܒܡܪܢ ܘܡܠܦܝܢ ܠܟܘܢ ܀

<span id="an_5:13">5:13</span> ܕܗܘܘ ܚܫܝܒܝܢ ܠܟܘܢ ܒܚܘܒܐ ܝܬܝܪܐ ܘܡܛܠ ܥܒܕܗܘܢ
ܐܫܬܝܢܘ ܥܡܗܘܢ ܀

<span id="an_5:14">5:14</span> ܒܥܝܢܢ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܪܕܘ ܠܡܤܟܠܢܐ ܘܠܒܒܘ
ܠܙܥܘܪܝ ܢܦܫܐ ܘܤܒܘ ܛܥܢܐ ܕܡܚܝܠܐ ܘܐܓܪܘ ܪܘܚܟܘܢ ܠܘܬ ܟܠܢܫ ܀

<span id="an_5:15">5:15</span> ܘܐܙܕܗܪܘ ܕܠܡܐ ܐܢܫ ܡܢܟܘܢ ܒܝܫܬܐ ܚܠܦ ܒܝܫܬܐ
ܢܦܪܘܥ ܐܠܐ ܒܟܠܙܒܢ ܗܪܛܘ ܒܬܪ ܛܒܬܐ ܠܘܬ ܚܕܕܐ ܘܠܘܬ ܟܠܢܫ ܀

<span id="an_5:16">5:16</span> ܗܘܘ ܚܕܝܢ ܒܟܠܙܒܢ ܀

<span id="an_5:17">5:17</span> ܘܡܨܠܝܢ ܕܠܐ ܫܠܘܐ ܀

<span id="an_5:18">5:18</span> ܘܒܟܠ ܡܕܡ ܗܘܘ ܡܘܕܝܢ ܗܢܘ ܓܝܪ ܨܒܝܢܗ ܕܐܠܗܐ
ܒܝܫܘܥ ܡܫܝܚܐ ܒܟܘܢ ܀

<span id="an_5:19">5:19</span> ܪܘܚܐ ܠܐ ܬܕܥܟܘܢ ܀

<span id="an_5:20">5:20</span> ܢܒܝܘܬܐ ܠܐ ܬܤܠܘܢ ܀

<span id="an_5:21">5:21</span> ܟܠܡܕܡ ܒܩܘ ܘܕܫܦܝܪ ܐܚܘܕܘ ܀

<span id="an_5:22">5:22</span> ܘܡܢ ܟܠ ܨܒܘ ܒܝܫܐ ܥܪܘܩܘ ܀

<span id="an_5:23">5:23</span> ܗܘ ܕܝܢ ܐܠܗܐ ܕܫܠܡܐ ܢܩܕܫܟܘܢ ܓܡܝܪܐܝܬ ܠܟܠܟܘܢ
ܘܟܠܗ ܪܘܚܟܘܢ ܘܢܦܫܟܘܢ ܘܦܓܪܟܘܢ ܢܢܛܪ ܕܠܐ ܪܫܝܢ ܠܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_5:24">5:24</span> ܡܗܝܡܢ ܗܘ ܡܢ ܕܩܪܟܘܢ ܕܗܘ ܢܥܒܕ ܀

<span id="an_5:25">5:25</span> ܐܚܝ ܨܠܘ ܥܠܝܢ ܀

<span id="an_5:26">5:26</span> ܫܐܠܘ ܒܫܠܡܐ ܕܐܚܝܢ ܟܠܗܘܢ ܒܢܘܫܩܬܐ ܩܕܝܫܬܐ ܀

<span id="an_5:27">5:27</span> ܡܘܡܐ ܐܢܐ ܠܟܘܢ ܒܡܪܢ ܕܬܬܩܪܐ ܐܓܪܬܐ ܗܕܐ
ܠܟܠܗܘܢ ܐܚܐ ܩܕܝܫܐ ܀

<span id="an_5:28">5:28</span> ܛܝܒܘܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡܟܘܢ ܐܡܝܢ ܀
