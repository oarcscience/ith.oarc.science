### Aramaic Bible: James

<span id="an_1:1">1:1</span> ܝܥܩܘܒ ܥܒܕܗ ܕܐܠܗܐ ܘܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܠܬܪܬܥܤܪܐ
ܫܪܒܢ ܕܙܪܝܥܢ ܒܥܡܡܐ ܫܠܡ ܀

<span id="an_1:2">1:2</span> ܟܠ ܚܕܘܐ ܬܗܘܐ ܠܟܘܢ ܐܚܝ ܟܕ ܬܥܠܘܢ ܠܢܤܝܘܢܐ
ܡܫܚܠܦܐ ܘܤܓܝܐܐ ܀

<span id="an_1:3">1:3</span> ܝܕܥܝܢ ܐܢܬܘܢ ܓܝܪ ܕܒܘܩܝܐ ܕܗܝܡܢܘܬܐ ܡܩܢܐ ܠܟܘܢ
ܡܤܝܒܪܢܘܬܐ ܀

<span id="an_1:4">1:4</span> ܠܗ ܕܝܢ ܠܡܤܝܒܪܢܘܬܐ ܢܗܘܐ ܠܗ ܥܒܕܐ ܡܫܡܠܝܐ
ܕܬܗܘܘܢ ܓܡܝܪܝܢ ܘܡܫܠܡܢܝܢ ܘܒܡܕܡ ܠܐ ܗܘܝܬܘܢ ܚܤܝܪܝܢ ܀

<span id="an_1:5">1:5</span> ܐܢ ܐܢܫ ܕܝܢ ܡܢܟܘܢ ܚܤܝܪ ܡܢ ܚܟܡܬܐ ܢܫܐܠ ܡܢ ܐܠܗܐ
ܕܝܗܒ ܠܟܠ ܦܫܝܛܐܝܬ ܘܠܐ ܡܚܤܕ ܘܡܬܝܗܒܐ ܠܗ ܀

<span id="an_1:6">1:6</span> ܢܫܐܠ ܕܝܢ ܒܗܝܡܢܘܬܐ ܟܕ ܠܐ ܡܬܦܠܓ ܗܘ ܓܝܪ ܕܡܬܦܠܓ
ܕܡܐ ܠܓܠܠܐ ܕܝܡܐ ܕܫܓܫܐ ܠܗܘܢ ܪܘܚܐ ܀

<span id="an_1:7">1:7</span> ܘܠܐ ܢܤܒܪ ܗܘ ܒܪܢܫܐ ܕܢܤܒ ܡܕܡ ܡܢ ܡܪܝܐ ܀

<span id="an_1:8">1:8</span> ܐܝܢܐ ܕܦܠܝܓ ܒܪܥܝܢܗ ܘܫܓܝܫ ܒܟܠܗܝܢ ܐܘܪܚܬܗ ܀

<span id="an_1:9">1:9</span> ܢܫܬܒܗܪ ܕܝܢ ܐܚܐ ܡܟܝܟܐ ܒܪܘܡܪܡܗ ܀

<span id="an_1:10">1:10</span> ܘܥܬܝܪܐ ܒܡܘܟܟܗ ܡܛܠ ܕܐܝܟ ܗܒܒܐ ܕܥܤܒܐ ܗܟܢܐ
ܥܒܪ ܀

<span id="an_1:11">1:11</span> ܕܢܚ ܓܝܪ ܫܡܫܐ ܒܚܘܡܗ ܘܡܘܒܫ ܠܗ ܠܥܤܒܐ ܘܗܒܒܗ
ܢܦܠ ܘܫܘܦܪܐ ܕܚܙܘܗ ܐܒܕ ܗܟܢܐ ܐܦ ܥܬܝܪܐ ܚܡܐ ܒܗܘܦܟܘܗܝ ܀

<span id="an_1:12">1:12</span> ܛܘܒܘܗܝ ܠܓܒܪܐ ܕܡܤܝܒܪ ܢܤܝܘܢܐ ܕܡܐ ܕܐܬܒܚܪ ܢܤܒ
ܟܠܝܠܐ ܕܚܝܐ ܗܘ ܕܡܠܟ ܐܠܗܐ ܠܐܝܠܝܢ ܕܪܚܡܝܢ ܠܗ ܀

<span id="an_1:13">1:13</span> ܠܐ ܢܐܡܪ ܐܢܫ ܟܕ ܡܬܢܤܐ ܕܡܢ ܐܠܗܐ ܡܬܢܤܐ ܐܢܐ
ܐܠܗܐ ܓܝܪ ܠܐ ܡܢܤܝ ܒܒܝܫܬܐ ܘܗܘ ܠܐܢܫ ܠܐ ܡܢܤܐ ܀

<span id="an_1:14">1:14</span> ܐܠܐ ܐܢܫ ܐܢܫ ܡܢ ܪܓܬܗ ܗܘ ܡܬܢܤܐ ܘܡܬܪܓܪܓ
ܘܡܬܢܓܕ ܀

<span id="an_1:15">1:15</span> ܘܗܕܐ ܪܓܬܐ ܒܛܢܐ ܘܝܠܕܐ ܚܛܝܬܐ ܚܛܝܬܐ ܕܝܢ ܡܐ
ܕܐܬܓܡܪܬ ܝܠܕܐ ܡܘܬܐ ܀

<span id="an_1:16">1:16</span> ܠܐ ܬܛܥܘܢ ܐܚܝ ܚܒܝܒܐ ܀

<span id="an_1:17">1:17</span> ܟܠ ܡܘܗܒܬܐ ܛܒܬܐ ܘܡܫܡܠܝܬܐ ܡܢ ܠܥܠ ܢܚܬܐ ܡܢ
ܐܒܐ ܕܢܗܝܪܐ ܗܘ ܕܠܝܬ ܠܘܬܗ ܫܘܚܠܦܐ ܡܕܡ ܐܦܠܐ ܛܠܢܝܬܐ ܕܫܘܓܢܝܐ ܀

<span id="an_1:18">1:18</span> ܗܘܝܘ ܨܒܐ ܘܝܠܕܢ ܒܡܠܬܐ ܕܩܘܫܬܐ ܕܢܗܘܐ ܪܫܝܬܐ
ܕܒܪܝܬܗ ܀

<span id="an_1:19">1:19</span> ܘܐܢܬܘܢ ܐܚܝ ܚܒܝܒܐ ܟܠܢܫ ܡܢܟܘܢ ܢܗܘܐ ܡܤܪܗܒ
ܠܡܫܡܥ ܘܡܘܚܪ ܠܡܡܠܠܘ ܘܡܘܚܪ ܠܡܪܓܙ ܀

<span id="an_1:20">1:20</span> ܪܘܓܙܗ ܓܝܪ ܕܓܒܪܐ ܙܕܝܩܘܬܐ ܕܐܠܗܐ ܠܐ ܥܒܕ ܀

<span id="an_1:21">1:21</span> ܡܛܠ ܗܢܐ ܐܪܚܩܘ ܡܢܟܘܢ ܟܠܗ ܛܢܦܘܬܐ ܘܤܘܓܐܐ
ܕܒܝܫܘܬܐ ܘܒܡܟܝܟܘܬܐ ܩܒܠܘ ܡܠܬܐ ܕܢܨܝܒܐ ܒܟܝܢܢ ܕܗܝ ܡܫܟܚܐ ܕܬܚܐ ܐܢܝܢ ܢܦܫܬܟܘܢ ܀

<span id="an_1:22">1:22</span> ܗܘܘ ܕܝܢ ܥܒܘܕܐ ܕܡܠܬܐ ܘܠܐ ܫܡܘܥܐ ܒܠܚܘܕ ܘܠܐ
ܬܛܥܘܢ ܢܦܫܬܟܘܢ ܀

<span id="an_1:23">1:23</span> ܐܢ ܐܢܫ ܓܝܪ ܢܗܘܐ ܫܡܘܥܗ ܕܡܠܬܐ ܘܠܐ ܥܒܘܕܗ ܗܢܐ
ܕܡܐ ܠܗܘ ܕܚܙܐ ܐܦܘܗܝ ܒܡܚܙܝܬܐ ܀

<span id="an_1:24">1:24</span> ܚܙܐ ܓܝܪ ܢܦܫܗ ܘܥܒܪ ܘܛܥܐ ܕܐܝܟܢܐ ܗܘܐ ܀

<span id="an_1:25">1:25</span> ܟܠ ܕܝܢ ܕܚܪ ܒܢܡܘܤܐ ܡܫܠܡܢܐ ܕܚܐܪܘܬܐ ܘܩܘܝ ܒܗ
ܠܐ ܗܘܐ ܫܡܘܥܐ ܕܫܡܥܐ ܕܡܬܛܥܐ ܐܠܐ ܥܒܘܕܐ ܕܥܒܕܐ ܘܗܢܐ ܛܘܒܢܐ ܢܗܘܐ ܒܥܒܕܗ ܀

<span id="an_1:26">1:26</span> ܘܐܢ ܐܢܫ ܤܒܪ ܕܡܫܡܫ ܠܐܠܗܐ ܘܠܐ ܐܚܕ ܠܫܢܗ ܐܠܐ
ܡܛܥܐ ܠܗ ܠܒܗ ܕܗܢܐ ܤܪܝܩܐ ܗܝ ܬܫܡܫܬܗ ܀

<span id="an_1:27">1:27</span> ܬܫܡܫܬܐ ܓܝܪ ܕܟܝܬܐ ܘܩܕܝܫܬܐ ܩܕܡ ܐܠܗܐ ܐܒܐ ܗܕܐ
ܗܝ ܠܡܤܥܪ ܝܬܡܐ ܘܐܪܡܠܬܐ ܒܐܘܠܨܢܝܗܘܢ ܘܠܡܛܪ ܐܢܫ ܢܦܫܗ ܡܢ ܥܠܡܐ ܕܠܐ ܛܘܠܫܐ ܀

<span id="an_2:1">2:1</span> ܐܚܝ ܠܐ ܒܡܤܒ ܒܐܦܐ ܬܗܘܘܢ ܐܚܝܕܝܢ ܠܗܝܡܢܘܬܐ
ܕܬܫܒܘܚܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_2:2">2:2</span> ܐܢ ܓܝܪ ܢܥܘܠ ܠܟܢܘܫܬܟܘܢ ܐܢܫ ܕܥܙܩܬܗ ܕܕܗܒܐ ܐܘ
ܕܡܐܢܘܗܝ ܫܦܝܪܐ ܘܢܥܘܠ ܡܤܟܢܐ ܒܡܐܢܐ ܨܐܐ ܀

<span id="an_2:3">2:3</span> ܘܬܚܘܪܘܢ ܒܗܘ ܕܠܒܝܫ ܡܐܢܐ ܫܦܝܪܐ ܘܬܐܡܪܘܢ ܠܗ ܐܢܬ
ܗܪܟܐ ܬܒ ܫܦܝܪ ܘܠܡܤܟܢܐ ܬܐܡܪܘܢ ܠܗ ܐܢܬ ܩܘܡ ܠܗܠ ܐܘ ܬܒ ܠܟ ܗܪܟܐ ܩܕܡ ܟܘܒܫܐ
ܕܪܓܠܝܢ ܀

<span id="an_2:4">2:4</span> ܠܐ ܗܐ ܐܬܦܠܓܬܘܢ ܠܟܘܢ ܒܢܦܫܟܘܢ ܘܗܘܝܬܘܢ ܡܦܪܫܢܐ
ܕܡܚܫܒܬܐ ܒܝܫܬܐ ܀

<span id="an_2:5">2:5</span> ܫܡܥܘ ܐܚܝ ܚܒܝܒܐ ܠܐ ܗܘܐ ܠܡܤܟܢܐ ܕܥܠܡܐ ܥܬܝܪܐ
ܕܝܢ ܒܗܝܡܢܘܬܐ ܓܒܐ ܐܠܗܐ ܕܢܗܘܘܢ ܝܪܬܐ ܒܡܠܟܘܬܐ ܗܝ ܕܡܠܟ ܐܠܗܐ ܠܐܝܠܝܢ ܕܪܚܡܝܢ ܠܗ
܀

<span id="an_2:6">2:6</span> ܐܢܬܘܢ ܕܝܢ ܫܛܬܘܢܝܗܝ ܠܡܤܟܢܐ ܠܐ ܗܐ ܥܬܝܪܐ
ܡܫܬܥܠܝܢ ܥܠܝܟܘܢ ܘܗܢܘܢ ܢܓܕܝܢ ܠܟܘܢ ܠܒܝܬ ܕܝܢܐ ܀

<span id="an_2:7">2:7</span> ܠܐ ܗܐ ܗܢܘܢ ܡܓܕܦܝܢ ܥܠ ܫܡܐ ܛܒܐ ܕܐܬܩܪܝ ܥܠܝܟܘܢ
܀

<span id="an_2:8">2:8</span> ܘܐܢ ܢܡܘܤܐ ܕܐܠܗܐ ܒܗܕܐ ܡܫܠܡܝܬܘܢ ܐܝܟ ܕܟܬܝܒ
ܕܬܪܚܡ ܠܩܪܝܒܟ ܐܝܟ ܢܦܫܟ ܫܦܝܪ ܥܒܕܝܢ ܐܢܬܘܢ ܀

<span id="an_2:9">2:9</span> ܐܢ ܕܝܢ ܒܐܦܐ ܢܤܒܝܢ ܐܢܬܘܢ ܚܛܝܬܐ ܥܒܕܝܢ ܐܢܬܘܢ
ܘܡܬܟܘܢܝܢ ܐܢܬܘܢ ܡܢ ܢܡܘܤܐ ܐܝܟ ܥܒܪܝ ܥܠ ܢܡܘܤܐ ܀

<span id="an_2:10">2:10</span> ܐܝܢܐ ܓܝܪ ܕܟܠܗ ܢܡܘܤܐ ܢܛܪ ܘܒܚܕܐ ܫܪܥ ܠܟܠܗ
ܢܡܘܤܐ ܐܬܚܝܒ ܀

<span id="an_2:11">2:11</span> ܗܘ ܓܝܪ ܕܐܡܪ ܕܠܐ ܬܓܘܪ ܗܘ ܐܡܪ ܕܠܐ ܬܩܛܘܠ ܐܢ
ܕܝܢ ܠܐ ܓܐܪ ܐܢܬ ܐܠܐ ܩܛܠ ܐܢܬ ܗܘܝܬ ܠܟ ܥܒܪ ܥܠ ܢܡܘܤܐ ܀

<span id="an_2:12">2:12</span> ܗܟܢܐ ܗܘܝܬܘܢ ܡܡܠܠܝܢ ܘܗܟܢܐ ܗܘܝܬܘܢ ܤܥܪܝܢ ܐܝܟ
ܐܢܫܐ ܕܒܢܡܘܤܐ ܕܚܐܪܘܬܐ ܥܬܝܕܝܢ ܐܢܬܘܢ ܠܡܬܕܢܘ ܀

<span id="an_2:13">2:13</span> ܕܝܢܐ ܓܝܪ ܗܘܐ ܕܠܐ ܪܚܡܐ ܥܠ ܗܘ ܕܠܐ ܥܒܕ ܪܚܡܐ
ܡܫܬܥܠܝܢ ܐܢܬܘܢ ܒܪܚܡܐ ܥܠ ܕܝܢܐ ܀

<span id="an_2:14">2:14</span> ܡܢܐ ܗܢܝܢܐ ܐܚܝ ܐܢ ܐܢܫ ܐܡܪ ܕܐܝܬ ܠܝ ܗܝܡܢܘܬܐ
ܘܥܒܕܐ ܠܝܬ ܠܗ ܕܠܡܐ ܡܫܟܚܐ ܗܝܡܢܘܬܗ ܕܬܚܝܘܗܝ ܀

<span id="an_2:15">2:15</span> ܘܐܢ ܐܚܐ ܐܘ ܚܬܐ ܢܗܘܘܢ ܥܪܛܠܝܝܢ ܘܚܤܝܪܝܢ
ܤܝܒܪܬܐ ܕܝܘܡܐ ܀

<span id="an_2:16">2:16</span> ܘܢܐܡܪ ܠܗܘܢ ܐܢܫ ܡܢܟܘܢ ܙܠܘ ܒܫܠܡܐ ܫܚܢܘ ܘܤܒܥܘ
ܘܠܐ ܬܬܠܘܢ ܠܗܘܢ ܤܢܝܩܘܬܗ ܕܦܓܪܐ ܡܢܐ ܗܢܝܢܐ ܀

<span id="an_2:17">2:17</span> ܗܟܢܐ ܐܦ ܗܝܡܢܘܬܐ ܕܠܐ ܥܒܕܐ ܡܝܬܐ ܗܝ ܒܠܚܘܕܝܗ
܀

<span id="an_2:18">2:18</span> ܐܡܪ ܓܝܪ ܐܢܫ ܠܟ ܐܝܬ ܠܟ ܗܝܡܢܘܬܐ ܘܠܝ ܐܝܬ ܠܝ
ܥܒܕܐ ܚܘܢܝ ܗܝܡܢܘܬܟ ܕܠܐ ܥܒܕܐ ܘܐܢܐ ܡܚܘܐ ܐܢܐ ܠܟ ܗܝܡܢܘܬܝ ܡܢ ܥܒܕܝ ܀

<span id="an_2:19">2:19</span> ܡܗܝܡܢ ܐܢܬ ܕܚܕ ܗܘ ܐܠܗܐ ܫܦܝܪ ܥܒܕ ܐܢܬ ܐܦ
ܫܐܕܐ ܡܗܝܡܢܝܢ ܘܪܥܠܝܢ ܀

<span id="an_2:20">2:20</span> ܨܒܐ ܐܢܬ ܕܝܢ ܕܬܕܥ ܐܘ ܒܪܢܫܐ ܚܠܫܐ ܕܗܝܡܢܘܬܐ
ܕܠܐ ܥܒܕܐ ܡܝܬܐ ܗܝ ܀

<span id="an_2:21">2:21</span> ܐܒܘܢ ܐܒܪܗܡ ܠܐ ܗܘܐ ܡܢ ܥܒܕܐ ܐܙܕܕܩ ܕܐܤܩ
ܠܐܝܤܚܩ ܒܪܗ ܥܠ ܡܕܒܚܐ ܀

<span id="an_2:22">2:22</span> ܚܙܐ ܐܢܬ ܕܗܝܡܢܘܬܗ ܤܝܥܬ ܠܥܒܕܘܗܝ ܘܡܢ ܥܒܕܐ
ܗܝܡܢܘܬܗ ܐܬܓܡܪܬ ܀

<span id="an_2:23">2:23</span> ܘܫܠܡ ܟܬܒܐ ܕܐܡܪ ܕܗܝܡܢ ܐܒܪܗܡ ܒܐܠܗܐ ܘܐܬܚܫܒܬ
ܠܗ ܠܙܕܝܩܘ ܘܪܚܡܐ ܕܐܠܗܐ ܐܬܩܪܝ ܀

<span id="an_2:24">2:24</span> ܚܙܐ ܐܢܬ ܕܡܢ ܥܒܕܐ ܡܙܕܕܩ ܒܪܢܫܐ ܘܠܐ ܗܘܐ ܡܢ
ܗܝܡܢܘܬܐ ܒܠܚܘܕ ܀

<span id="an_2:25">2:25</span> ܗܟܢܐ ܐܦ ܪܚܒ ܙܢܝܬܐ ܠܐ ܗܘܐ ܡܢ ܥܒܕܐ ܐܙܕܕܩܬ
ܕܩܒܠܬ ܠܓܫܘܫܐ ܘܒܐܘܪܚܐ ܐܚܪܬܐ ܐܦܩܬ ܐܢܘܢ ܀

ܝܕܥܝܢ ܕܕܝܢܐ ܝܬܝܪܐ ܚܝܒܝܢܢ ܀

<span id="an_3:16">3:16</span> ܐܝܟܐ ܓܝܪ ܕܐܝܬ ܚܤܡܐ ܘܚܪܝܢܐ ܬܡܢ ܐܦ ܕܠܘܚܝܐ

<span id="an_3:18">3:18</span> ܦܐܪܐ ܕܝܢ ܕܙܕܝܩܘܬܐ ܒܫܝܢܐ ܡܙܕܪܥܝܢ ܠܐܝܠܝܢ
ܕܥܒܕܝܢ ܫܠܡܐ ܀

ܫܐܠܝܢ ܐܢܬܘܢ ܐܝܟ ܕܬܬܪܤܘܢ ܪܓܝܓܬܟܘܢ ܀

ܒܥܠܕܒܒܘܬܐ ܗܝ ܕܐܠܗܐ ܐܝܢܐ ܗܟܝܠ ܕܨܒܐ ܕܢܗܘܐ ܪܚܡܐ ܕܥܠܡܐ ܗܢܐ ܒܥܠܕܒܒܐ ܗܘܐ ܠܐܠܗܐ
܀

<span id="an_4:5">4:5</span> ܐܘ ܕܠܡܐ ܤܪܝܩܐܝܬ ܤܒܪܝܢ ܐܢܬܘܢ ܕܐܡܪ ܟܬܒܐ
ܕܒܛܢܢܐ ܪܓܐ ܪܘܚܐ ܕܥܡܪܐ ܒܢ ܀

<span id="an_4:6">4:6</span> ܛܝܒܘܬܐ ܕܝܢ ܝܬܝܪܬܐ ܝܗܒ ܠܢ ܡܪܢ ܡܛܠ ܗܕܐ ܐܡܪ
ܕܐܠܗܐ ܡܡܟܟ ܠܪܡܐ ܘܠܡܟܝܟܐ ܝܗܒ ܛܝܒܘܬܐ ܀

<span id="an_4:7">4:7</span> ܐܫܬܥܒܕܘ ܗܟܝܠ ܠܐܠܗܐ ܘܩܘܡܘ ܠܘܩܒܠ ܤܛܢܐ ܘܥܪܩ

<span id="an_4:8">4:8</span> ܘܩܪܘܒܘ ܠܘܬ ܐܠܗܐ ܘܢܬܩܪܒ ܠܟܘܢ ܕܟܘ ܐܝܕܝܟܘܢ

ܡܚܪ ܐܙܠܝܢܢ ܠܡܕܝܢܬܐ ܐܝܕܐ ܕܗܝ ܘܥܒܕܝܢܢ ܬܡܢ ܫܢܬܐ ܚܕܐ ܘܡܬܬܓܪܝܢܢ ܘܝܬܪܝܢܢ ܀

ܐܠܐ ܐܢ ܠܗܓܐ ܕܩܠܝܠ ܡܬܚܙܐ ܘܛܠܩ ܘܡܘܦܐ ܀

<span id="an_4:15">4:15</span> ܚܠܦ ܕܢܐܡܪܘܢ ܕܐܢ ܡܪܝܐ ܢܨܒܐ ܘܢܚܐ ܥܒܕܝܢܢ ܗܕܐ
