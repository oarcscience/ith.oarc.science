### Aramaic Bible: 1 John

<span id="an_1:1">1:1</span> ܡܤܒܪܝܢܢ ܠܟܘܢ ܠܗܘ ܕܐܝܬܘܗܝ ܗܘܐ ܡܢ ܒܪܫܝܬ ܗܘ
ܕܫܡܥܢ ܘܚܙܝܢܝܗܝ ܒܥܝܢܝܢ ܚܙܝܢ ܘܓܫܢ ܒܐܝܕܝܢ ܗܘ ܕܐܝܬܘܗܝ ܡܠܬܐ ܕܚܝܐ ܀

<span id="an_1:2">1:2</span> ܘܚܝܐ ܐܬܓܠܝܘ ܘܚܙܝܢ ܘܤܗܕܝܢܢ ܘܡܟܪܙܝܢܢ ܠܟܘܢ ܚܝܐ
ܕܠܥܠܡ ܗܢܘܢ ܕܐܝܬܝܗܘܢ ܗܘܘ ܠܘܬ ܐܒܐ ܘܐܬܓܠܝܘ ܠܢ ܀

<span id="an_1:3">1:3</span> ܘܡܕܡ ܕܚܙܝܢ ܘܫܡܥܢ ܡܘܕܥܝܢܢ ܐܦ ܠܟܘܢ ܕܬܗܘܐ ܠܟܘܢ
ܫܘܬܦܘܬܐ ܥܡܢ ܘܫܘܬܦܘܬܢ ܕܝܢ ܕܝܠܢ ܐܝܬܝܗ ܥܡ ܐܒܐ ܘܥܡ ܒܪܗ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:4">1:4</span> ܘܗܠܝܢ ܟܬܒܝܢܢ ܠܟܘܢ ܕܚܕܘܬܢ ܕܒܟܘܢ ܬܗܘܐ ܡܫܡܠܝܐ
܀

<span id="an_1:5">1:5</span> ܘܗܕܐ ܗܝ ܤܒܪܬܐ ܕܫܡܥܢ ܡܢܗ ܘܡܤܒܪܝܢܢ ܠܟܘܢ ܕܐܠܗܐ
ܢܘܗܪܐ ܗܘ ܘܚܫܘܟܐ ܟܠ ܟܠܗ ܠܝܬ ܒܗ ܀

<span id="an_1:6">1:6</span> ܘܐܢ ܐܡܪܝܢܢ ܕܫܘܬܦܘܬܐ ܐܝܬ ܠܢ ܥܡܗ ܘܡܗܠܟܝܢܢ
ܒܚܫܘܟܐ ܕܓܠܐ ܚܢܢ ܘܠܘ ܒܫܪܪܐ ܪܕܝܢܢ ܀

<span id="an_1:7">1:7</span> ܐܢ ܕܝܢ ܒܢܘܗܪܐ ܡܗܠܟܝܢܢ ܐܝܟ ܕܗܘ ܐܝܬܘܗܝ ܒܢܘܗܪܐ
ܐܝܬ ܠܢ ܫܘܬܦܘܬܐ ܥܡ ܚܕܕܐ ܘܕܡܐ ܕܝܫܘܥ ܒܪܗ ܡܕܟܐ ܠܢ ܡܢ ܟܠܗܘܢ ܚܛܗܝܢ ܀

<span id="an_1:8">1:8</span> ܘܐܢ ܢܐܡܪ ܕܠܝܬ ܠܢ ܚܛܝܬܐ ܢܦܫܢ ܡܛܥܝܢܢ ܘܫܪܪܐ
ܠܝܬ ܒܢ ܀

<span id="an_1:9">1:9</span> ܐܢ ܕܝܢ ܡܘܕܝܢܢ ܒܚܛܗܝܢ ܡܗܝܡܢ ܗܘ ܘܙܕܝܩ ܕܢܫܒܘܩ
ܠܢ ܚܛܗܝܢ ܘܢܕܟܝܢ ܡܢ ܟܠܗ ܥܘܠܢ ܀

<span id="an_1:10">1:10</span> ܘܐܢ ܐܡܪܝܢܢ ܕܠܐ ܚܛܝܢ ܥܒܕܝܢܢ ܠܗ ܕܓܠܐ ܘܡܠܬܗ
ܠܝܬ ܠܘܬܢ ܀

<span id="an_2:1">2:1</span> ܒܢܝ ܗܠܝܢ ܟܬܒ ܐܢܐ ܠܟܘܢ ܕܠܐ ܬܚܛܘܢ ܘܐܢ ܐܢܫ
ܢܚܛܐ ܐܝܬ ܠܢ ܦܪܩܠܛܐ ܠܘܬ ܐܒܐ ܝܫܘܥ ܡܫܝܚܐ ܙܕܝܩܐ ܀

<span id="an_2:2">2:2</span> ܗܘܝܘ ܓܝܪ ܚܘܤܝܐ ܕܥܠ ܐܦܝ ܚܛܗܝܢ ܘܠܘ ܚܠܦܝܢ ܕܝܠܢ
ܒܠܚܘܕ ܐܠܐ ܐܦ ܚܠܦ ܟܠܗ ܥܠܡܐ ܀

<span id="an_2:3">2:3</span> ܘܒܗܕܐ ܡܪܓܫܝܢܢ ܕܝܕܥܢܝܗܝ ܐܢ ܢܛܪܝܢܢ ܦܘܩܕܢܘܗܝ ܀

<span id="an_2:4">2:4</span> ܗܘ ܓܝܪ ܕܐܡܪ ܕܝܕܥܬܗ ܘܦܘܩܕܢܘܗܝ ܠܐ ܢܛܪ ܕܓܠܐ ܗܘ
ܘܫܪܪܐ ܠܝܬ ܒܗ ܀

<span id="an_2:5">2:5</span> ܗܘ ܕܝܢ ܕܢܛܪ ܡܠܬܗ ܒܗܢܐ ܡܫܠܡ ܫܪܝܪܐܝܬ ܚܘܒܗ
ܕܐܠܗܐ ܒܗܕܐ ܓܝܪ ܝܕܥܝܢܢ ܕܒܗ ܚܢܢ ܀

<span id="an_2:6">2:6</span> ܗܘ ܕܐܡܪ ܕܒܗ ܐܝܬܝ ܘܠܐ ܠܗ ܕܐܝܟ ܗܠܟܬܗ ܕܝܠܗ
ܢܗܠܟ ܀

<span id="an_2:7">2:7</span> ܚܒܝܒܝ ܠܘ ܦܘܩܕܢܐ ܚܕܬܐ ܟܬܒ ܐܢܐ ܠܟܘܢ ܐܠܐ
ܦܘܩܕܢܐ ܥܬܝܩܐ ܗܘ ܕܐܝܬ ܗܘܐ ܠܟܘܢ ܡܢ ܒܪܫܝܬ ܦܘܩܕܢܐ ܕܝܢ ܥܬܝܩܐ ܐܝܬܘܗܝ ܡܠܬܐ ܗܝ
ܕܫܡܥܬܘܢ ܀

<span id="an_2:8">2:8</span> ܬܘܒ ܦܘܩܕܢܐ ܚܕܬܐ ܟܬܒ ܐܢܐ ܠܟܘܢ ܗܘ ܕܐܝܬܘܗܝ
ܫܪܝܪܐ ܒܗ ܘܒܟܘܢ ܕܚܫܘܟܐ ܥܒܪ ܠܗ ܘܢܘܗܪܐ ܫܪܝܪܐ ܫܪܝ ܡܬܚܙܐ ܀

<span id="an_2:9">2:9</span> ܡܢ ܕܐܡܪ ܗܟܝܠ ܕܐܝܬܘܗܝ ܒܢܘܗܪܐ ܘܤܢܐ ܠܐܚܘܗܝ
ܒܚܫܘܟܐ ܐܝܬܘܗܝ ܥܕܡܐ ܠܗܫܐ ܀

<span id="an_2:10">2:10</span> ܗܘ ܕܝܢ ܕܡܚܒ ܠܐܚܘܗܝ ܒܢܘܗܪܐ ܡܟܬܪ ܘܟܫܠܐ ܠܝܬ
ܒܗ ܀

<span id="an_2:11">2:11</span> ܗܘ ܕܝܢ ܕܤܢܐ ܠܐܚܘܗܝ ܒܚܫܘܟܐ ܐܝܬܘܗܝ ܘܒܚܫܘܟܐ
ܡܗܠܟ ܘܠܐ ܝܕܥ ܠܐܝܟܐ ܐܙܠ ܡܛܠ ܕܚܫܘܟܐ ܤܡܝ ܐܢܝܢ ܥܝܢܘܗܝ ܀

<span id="an_2:12">2:12</span> ܟܬܒ ܐܢܐ ܠܟܘܢ ܒܢܝܐ ܕܐܫܬܒܩܘ ܠܟܘܢ ܚܛܗܝܟܘܢ
ܡܛܠ ܫܡܗ ܀

<span id="an_2:13">2:13</span> ܟܬܒ ܐܢܐ ܠܟܘܢ ܐܒܗܐ ܕܝܕܥܬܘܢܝܗܝ ܠܗܘ ܕܐܝܬܘܗܝ
ܗܘܐ ܡܢ ܒܪܫܝܬ ܟܬܒ ܐܢܐ ܠܟܘܢ ܥܠܝܡܐ ܕܙܟܝܬܘܢܝܗܝ ܠܒܝܫܐ ܟܬܒܬ ܠܟܘܢ ܛܠܝܐ ܕܝܕܥܬܘܢ
ܠܐܒܐ ܀

<span id="an_2:14">2:14</span> ܟܬܒܬ ܠܟܘܢ ܐܒܗܐ ܕܝܕܥܬܘܢ ܠܗܘ ܕܡܢ ܒܪܫܝܬ ܟܬܒܬ
ܠܟܘܢ ܥܠܝܡܐ ܕܚܝܠܬܢܐ ܐܢܬܘܢ ܘܡܠܬܗ ܕܐܠܗܐ ܫܪܝܐ ܒܟܘܢ ܘܙܟܝܬܘܢܝܗܝ ܠܒܝܫܐ ܀

<span id="an_2:15">2:15</span> ܠܐ ܬܪܚܡܘܢ ܠܥܠܡܐ ܘܠܐ ܠܡܕܡ ܕܐܝܬ ܒܗ ܡܢ ܓܝܪ
ܕܪܚܡ ܠܥܠܡܐ ܚܘܒܗ ܕܐܒܐ ܠܝܬ ܒܗ ܀

<span id="an_2:16">2:16</span> ܟܠ ܡܕܡ ܓܝܪ ܕܐܝܬ ܒܗ ܒܥܠܡܐ ܪܓܬܐ ܗܘ ܕܦܓܪܐ
ܘܪܓܬܐ ܕܥܝܢܐ ܘܫܘܒܗܪܐ ܕܥܠܡܐ ܗܠܝܢ ܕܠܐ ܗܘܝ ܡܢ ܐܒܐ ܐܠܐ ܡܢܗ ܐܢܝܢ ܡܢ ܥܠܡܐ ܀

<span id="an_2:17">2:17</span> ܘܥܒܪ ܥܠܡܐ ܗܘ ܘܪܓܬܗ ܗܘ ܕܝܢ ܕܥܒܕ ܨܒܝܢܗ
ܕܐܠܗܐ ܡܩܘܐ ܠܥܠܡ ܀

<span id="an_2:18">2:18</span> ܒܢܝ ܙܒܢܐ ܗܘ ܐܚܪܝܐ ܘܐܝܟ ܡܕܡ ܕܫܡܥܬܘܢ ܕܐܬܐ
ܡܫܝܚܐ ܕܓܠܐ ܘܗܫܐ ܗܘܘ ܠܗܘܢ ܤܓܝܐܐ ܡܫܝܚܐ ܕܓܠܐ ܘܡܢ ܗܕܐ ܝܕܥܝܢܢ ܕܙܒܢܐ ܗܘ ܐܚܪܝܐ
܀

<span id="an_2:19">2:19</span> ܡܢܢ ܢܦܩܘ ܐܠܐ ܠܘ ܡܢܢ ܗܘܘ ܐܠܘ ܓܝܪ ܡܢܢ ܗܘܘ
ܠܘܬܢ ܡܟܬܪܝܢ ܗܘܘ ܐܠܐ ܢܦܩܘ ܡܢܢ ܕܬܬܝܕܥ ܕܠܘ ܡܢܢ ܗܘܘ ܀

<span id="an_2:20">2:20</span> ܘܐܢܬܘܢ ܡܫܝܚܘܬܐ ܐܝܬ ܠܟܘܢ ܡܢ ܩܕܝܫܐ ܘܦܪܫܝܢ
ܐܢܬܘܢ ܗܘ ܠܟܠ ܐܢܫ ܀

<span id="an_2:21">2:21</span> ܠܐ ܟܬܒܬ ܠܟܘܢ ܕܠܐ ܝܕܥܝܢ ܐܢܬܘܢ ܠܗ ܠܫܪܪܐ ܐܠܐ
ܕܝܕܥܝܢ ܐܢܬܘܢ ܠܗ ܘܕܟܠܗ ܕܓܠܘܬܐ ܠܐ ܗܘܬ ܡܢܗ ܡܢ ܫܪܪܐ ܀

<span id="an_2:22">2:22</span> ܡܢܘ ܕܓܠܐ ܐܠܐ ܐܢ ܐܝܢܐ ܕܟܦܪ ܕܝܫܘܥ ܠܐ ܗܘܐ
ܡܫܝܚܐ ܗܢܐ ܗܘ ܡܫܝܚܐ ܕܓܠܐ ܗܘ ܕܟܦܪ ܒܐܒܐ ܟܦܪ ܐܦ ܒܒܪܐ ܀

<span id="an_2:23">2:23</span> ܘܗܘ ܕܟܦܪ ܒܒܪܐ ܐܦܠܐ ܒܐܒܐ ܡܗܝܡܢ ܡܢ ܕܡܘܕܐ
ܒܒܪܐ ܐܦ ܒܐܒܐ ܡܘܕܐ ܀

<span id="an_2:24">2:24</span> ܘܐܢܬܘܢ ܡܕܡ ܕܫܡܥܬܘܢ ܡܢ ܩܕܝܡ ܢܩܘܐ ܠܘܬܟܘܢ ܐܢ
ܓܝܪ ܡܩܘܐ ܠܘܬܟܘܢ ܡܕܡ ܕܫܡܥܬܘܢ ܡܢ ܩܕܝܡ ܐܦ ܐܢܬܘܢ ܡܩܘܝܢ ܐܢܬܘܢ ܒܐܒܐ ܘܒܒܪܐ ܀

<span id="an_2:25">2:25</span> ܘܗܢܘ ܫܘܘܕܝܐ ܕܐܫܬܘܕܝ ܠܢ ܚܝܐ ܕܠܥܠܡ ܀

<span id="an_2:26">2:26</span> ܗܠܝܢ ܕܝܢ ܟܬܒܬ ܠܟܘܢ ܡܛܠ ܐܝܠܝܢ ܕܡܛܥܝܢ ܠܟܘܢ
܀

<span id="an_2:27">2:27</span> ܘܐܦ ܐܢܬܘܢ ܐܢ ܬܩܘܐ ܠܘܬܟܘܢ ܡܫܝܚܘܬܐ ܕܩܒܠܬܘܢ
ܡܢܗ ܠܐ ܤܢܝܩܝܢ ܐܢܬܘܢ ܕܐܢܫ ܢܠܦܟܘܢ ܐܠܐ ܐܝܟ ܡܫܝܚܘܬܐ ܕܐܝܬܝܗ ܡܢ ܐܠܗܐ ܗܝ ܡܠܦܐ
ܠܟܘܢ ܥܠ ܟܠܡܕܡ ܘܫܪܝܪܐ ܗܝ ܘܠܝܬ ܒܗ ܕܓܠܘܬܐ ܘܐܝܟ ܕܐܠܦܟܘܢ ܩܘܘ ܒܗ ܀

<span id="an_2:28">2:28</span> ܘܗܫܐ ܒܢܝ ܩܘܘ ܒܗ ܕܡܐ ܕܡܬܓܠܐ ܠܐ ܢܒܗܬ ܡܢܗ
ܐܠܐ ܬܗܘܐ ܠܢ ܓܠܝܘܬ ܐܦܐ ܒܡܐܬܝܬܗ ܀

<span id="an_2:29">2:29</span> ܐܢ ܝܕܥܬܘܢ ܕܙܕܝܩ ܗܘ ܕܥܘ ܕܐܦ ܟܠ ܕܥܒܕ
ܙܕܝܩܘܬܐ ܡܢܗ ܐܝܬܘܗܝ ܀

<span id="an_3:1">3:1</span> ܘܚܙܘ ܕܟܡܐ ܤܓܝ ܚܘܒܗ ܕܐܒܐ ܠܘܬܢ ܕܒܢܝܐ ܩܪܢ ܐܦ
ܥܒܕܢ ܡܛܠ ܗܢܐ ܥܠܡܐ ܠܐ ܝܕܥ ܠܢ ܡܛܠ ܕܐܦܠܐ ܠܗ ܝܕܥ ܀

<span id="an_3:2">3:2</span> ܚܒܝܒܝ ܗܫܐ ܒܢܘܗܝ ܚܢܢ ܕܐܠܗܐ ܘܠܐ ܐܬܓܠܝ ܥܕܡܐ
ܠܗܫܐ ܡܢܐ ܥܬܝܕܝܢܢ ܕܢܗܘܐ ܝܕܥܝܢܢ ܕܝܢ ܕܡܐ ܕܐܬܓܠܝ ܒܕܡܘܬܗ ܗܘܝܢܢ ܘܚܙܝܢܢ ܠܗ ܐܝܟ
ܡܕܡ ܕܐܝܬܘܗܝ ܀

<span id="an_3:3">3:3</span> ܘܟܠ ܐܝܢܐ ܕܐܝܬ ܠܗ ܥܠܘܗܝ ܗܢܐ ܤܒܪܐ ܡܕܟܐ ܢܦܫܗ
ܐܝܟ ܕܗܘ ܕܟܐ ܗܘ ܀

<span id="an_3:4">3:4</span> ܡܢ ܕܝܢ ܕܥܒܕ ܚܛܝܬܐ ܥܘܠܐ ܤܥܪ ܚܛܝܬܐ ܓܝܪ ܟܠܗ
ܥܘܠܐ ܗܝ ܀

<span id="an_3:5">3:5</span> ܘܝܕܥܝܢ ܐܢܬܘܢ ܕܗܘ ܕܐܬܓܠܝ ܕܢܤܒ ܚܛܗܝܢ ܘܚܛܝܬܐ
ܒܗ ܠܝܬ ܀

<span id="an_3:6">3:6</span> ܘܟܠ ܕܒܗ ܡܩܘܐ ܠܐ ܚܛܐ ܘܟܠ ܕܚܛܐ ܠܐ ܚܙܝܗܝ ܘܠܐ
ܝܕܥܗ ܀

<span id="an_3:7">3:7</span> ܒܢܝ ܠܐ ܐܢܫ ܢܛܥܝܟܘܢ ܗܘ ܕܥܒܕ ܙܕܝܩܘܬܐ ܙܕܝܩܐ ܗܘ
ܐܝܟ ܕܐܦ ܗܘ ܡܫܝܚܐ ܐܝܬܘܗܝ ܙܕܝܩܐ ܀

<span id="an_3:8">3:8</span> ܗܘ ܕܤܥܪ ܚܛܝܬܐ ܡܢ ܤܛܢܐ ܗܘ ܡܛܠ ܕܡܢ ܪܫܝܬܐ ܗܘ

ܐܝܬܘܗܝ ܘܠܐ ܡܫܟܚ ܠܡܚܛܐ ܕܡܢ ܐܠܗܐ ܐܬܝܠܕ ܀

<span id="an_3:10">3:10</span> ܒܗܕܐ ܡܬܦܪܫܝܢ ܒܢܘܗܝ ܕܐܠܗܐ ܡܢ ܒܢܘܗܝ ܕܤܛܢܐ
ܟܠ ܕܠܐ ܥܒܕ ܙܕܝܩܘܬܐ ܘܠܐ ܡܚܒ ܠܐܚܘܗܝ ܠܝܬܘܗܝ ܡܢ ܐܠܗܐ ܀
