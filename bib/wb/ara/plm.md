### Aramaic Bible: Philemon

<span id="an_1:1">1:1</span> ܦܘܠܘܤ ܐܤܝܪܗ ܕܝܫܘܥ ܡܫܝܚܐ ܘܛܝܡܬܐܘܤ ܐܚܐ
ܠܦܝܠܡܘܢ ܚܒܝܒܐ ܘܦܠܚܐ ܕܥܡܢ ܀

<span id="an_1:2">1:2</span> ܘܠܐܦܝܐ ܚܒܝܒܬܢ ܘܠܐܪܟܝܦܘܤ ܦܠܚܐ ܕܥܡܢ ܘܠܥܕܬܐ
ܕܒܒܝܬܟ ܀

<span id="an_1:3">1:3</span> ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ
ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:4">1:4</span> ܡܘܕܐ ܐܢܐ ܠܐܠܗܝ ܒܟܠܙܒܢ ܘܡܬܕܟܪ ܐܢܐ ܠܟ ܒܨܠܘܬܝ
܀

<span id="an_1:5">1:5</span> ܗܐ ܡܢ ܕܫܡܥܬ ܗܝܡܢܘܬܟ ܘܚܘܒܐ ܕܐܝܬ ܠܟ ܠܘܬ ܡܪܢ
ܝܫܘܥ ܘܠܘܬ ܟܠܗܘܢ ܩܕܝܫܐ ܀

<span id="an_1:6">1:6</span> ܕܬܗܘܐ ܫܘܬܦܘܬܐ ܕܗܝܡܢܘܬܟ ܝܗܒܐ ܦܐܪܐ ܒܥܒܕܐ
ܘܒܝܕܥܬܐ ܕܟܠ ܛܒܢ ܕܐܝܬ ܠܟܘܢ ܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:7">1:7</span> ܚܕܘܬܐ ܓܝܪ ܤܓܝܐܬܐ ܐܝܬ ܠܢ ܘܒܘܝܐܐ ܕܒܝܕ ܚܘܒܟ
ܐܬܬܢܝܚܘ ܪܚܡܐ ܕܩܕܝܫܐ ܀

<span id="an_1:8">1:8</span> ܡܛܠ ܗܕܐ ܦܪܗܤܝܐ ܤܓܝܐܬܐ ܐܝܬ ܠܝ ܒܡܫܝܚܐ ܕܐܦܩܘܕ
ܠܟ ܐܝܠܝܢ ܕܙܕܩܢ ܀

<span id="an_1:9">1:9</span> ܡܛܠ ܚܘܒܐ ܕܝܢ ܡܒܥܐ ܗܘ ܒܥܐ ܐܢܐ ܡܢܟ ܐܢܐ ܦܘܠܘܤ
ܕܐܝܬܝ ܤܒܐ ܐܝܟ ܕܝܕܥ ܐܢܬ ܗܫܐ ܕܝܢ ܐܦ ܐܤܝܪܐ ܕܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:10">1:10</span> ܘܒܥܐ ܐܢܐ ܡܢܟ ܥܠ ܒܪܝ ܐܝܢܐ ܕܝܠܕܬ ܒܐܤܘܪܝ
ܐܢܤܝܡܘܤ ܀

<span id="an_1:11">1:11</span> ܗܘ ܕܒܙܒܢ ܠܝܬ ܗܘܐ ܠܟ ܒܗ ܚܫܚܘ ܗܫܐ ܕܝܢ ܐܦ ܠܟ
ܐܦ ܠܝ ܛܒ ܚܫܚ ܀

<span id="an_1:12">1:12</span> ܘܫܕܪܬܗ ܠܟ ܐܢܬ ܕܝܢ ܐܝܟ ܕܠܝܠܕܐ ܕܝܠܝ ܗܟܢܐ
ܩܒܠܝܗܝ ܀

<span id="an_1:13">1:13</span> ܨܒܐ ܗܘܝܬ ܓܝܪ ܕܠܘܬܝ ܐܚܕܝܘܗܝ ܕܢܗܘܐ ܡܫܡܫ ܠܝ
ܚܠܦܝܟ ܒܐܤܘܪܐ ܕܐܘܢܓܠܝܘܢ ܀

<span id="an_1:14">1:14</span> ܒܠܥܕ ܡܠܟܟ ܕܝܢ ܠܐ ܨܒܝܬ ܡܕܡ ܠܡܥܒܕ ܕܠܐ ܐܝܟ
ܕܒܩܛܝܪܐ ܬܗܘܐ ܛܒܬܟ ܐܠܐ ܒܨܒܝܢܟ ܀

<span id="an_1:15">1:15</span> ܟܒܪ ܕܝܢ ܐܦ ܡܛܠ ܗܕܐ ܫܢܝ ܕܫܥܬܐ ܕܠܥܠܡ
ܬܐܚܕܝܘܗܝ ܀

<span id="an_1:16">1:16</span> ܠܐ ܡܟܝܠ ܐܝܟ ܥܒܕܐ ܐܠܐ ܝܬܝܪ ܡܢ ܥܒܕܐ ܐܚܐ
ܚܒܝܒܐ ܕܝܠܝ ܚܕ ܟܡܐ ܕܝܠܟ ܘܒܒܤܪ ܘܒܡܪܢ ܀

<span id="an_1:17">1:17</span> ܐܢ ܗܟܝܠ ܐܝܬܝܟ ܠܝ ܫܘܬܦܐ ܩܒܠܝܗܝ ܐܝܟ ܕܠܝ ܀

<span id="an_1:18">1:18</span> ܘܐܢ ܡܕܡ ܚܤܪܟ ܐܘ ܚܝܒ ܗܕܐ ܥܠܝ ܚܫܘܒ ܀

<span id="an_1:19">1:19</span> ܐܢܐ ܦܘܠܘܤ ܟܬܒܬ ܒܐܝܕܝ ܐܢܐ ܦܪܥ ܐܢܐ ܕܠܐ ܐܡܪ
ܠܟ ܕܐܦ ܢܦܫܟ ܚܝܒ ܐܢܬ ܠܝ ܀

<span id="an_1:20">1:20</span> ܐܝܢ ܐܚܝ ܐܢܐ ܐܬܬܢܝܚ ܒܟ ܒܡܪܢ ܐܢܝܚ ܪܚܡܝ
ܒܡܫܝܚܐ ܀

<span id="an_1:21">1:21</span> ܡܛܠ ܕܬܟܝܠ ܐܢܐ ܕܫܡܥ ܐܢܬ ܠܝ ܟܬܒܬ ܠܟ ܘܝܕܥ
ܐܢܐ ܕܝܬܝܪ ܡܢ ܡܐ ܕܐܡܪ ܐܢܐ ܥܒܕ ܐܢܬ ܀

<span id="an_1:22">1:22</span> ܒܚܕܐ ܕܝܢ ܐܦ ܛܝܒ ܠܝ ܒܝܬ ܡܫܪܝܐ ܡܤܒܪ ܐܢܐ ܓܝܪ
ܕܒܨܠܘܬܟܘܢ ܡܬܝܗܒ ܐܢܐ ܠܟܘܢ ܀

<span id="an_1:23">1:23</span> ܫܐܠ ܒܫܠܡܟ ܐܦܦܪܐ ܫܒܝܐ ܕܥܡܝ ܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:24">1:24</span> ܘܡܪܩܘܤ ܘܐܪܤܛܪܟܘܤ ܘܕܡܐ ܘܠܘܩܐ ܡܥܕܪܢܝ ܀

<span id="an_1:25">1:25</span> ܛܝܒܘܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡ ܪܘܚܟܘܢ ܐܡܝܢ ܀
