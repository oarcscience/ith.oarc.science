### Aramaic Bible: 2 Timothy

<span id="an_1:1">1:1</span> ܦܘܠܘܤ ܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܒܨܒܝܢܗ ܕܐܠܗܐ
ܘܒܡܘܠܟܢܐ ܕܚܝܐ ܕܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:2">1:2</span> ܠܛܝܡܬܐܘܤ ܒܪܐ ܚܒܝܒܐ ܛܝܒܘܬܐ ܘܪܚܡܐ ܘܫܠܡܐ ܡܢ
ܐܠܗܐ ܐܒܐ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:3">1:3</span> ܡܘܕܐ ܐܢܐ ܠܐܠܗܐ ܗܘ ܕܠܗ ܡܫܡܫ ܐܢܐ ܡܢ ܐܒܗܬܝ
ܒܬܐܪܬܐ ܕܟܝܬܐ ܕܐܡܝܢܐܝܬ ܡܬܕܟܪ ܐܢܐ ܠܟ ܒܨܠܘܬܝ ܕܠܠܝܐ ܘܕܐܝܡܡܐ ܀

<span id="an_1:4">1:4</span> ܘܤܘܐ ܐܢܐ ܠܡܚܙܝܟ ܘܡܬܕܟܪ ܐܢܐ ܕܡܥܝܟ ܕܐܬܡܠܐ
ܚܕܘܬܐ ܀

<span id="an_1:5">1:5</span> ܒܥܘܗܕܢܐ ܕܗܘܐ ܠܝ ܒܗܝܡܢܘܬܟ ܫܪܝܪܬܐ ܗܝ ܕܫܪܬ
ܠܘܩܕܡ ܒܐܡܐ ܕܐܡܟ ܠܘܐܝܤ ܘܒܐܡܟ ܐܘܢܝܩܐ ܡܦܤ ܐܢܐ ܕܝܢ ܕܐܦ ܒܟ ܀

<span id="an_1:6">1:6</span> ܡܛܠ ܗܢܐ ܡܥܗܕ ܐܢܐ ܠܟ ܕܬܥܝܪ ܡܘܗܒܬܐ ܕܐܠܗܐ ܗܝ
ܕܐܝܬ ܒܟ ܒܤܝܡ ܐܝܕܝ ܀

<span id="an_1:7">1:7</span> ܠܐ ܓܝܪ ܝܗܒ ܠܢ ܐܠܗܐ ܪܘܚܐ ܕܕܚܠܬܐ ܐܠܐ ܕܚܝܠܐ
ܘܕܚܘܒܐ ܘܕܡܪܬܝܢܘܬܐ ܀

<span id="an_1:8">1:8</span> ܠܐ ܗܟܝܠ ܬܒܗܬ ܒܤܗܕܘܬܗ ܕܡܪܢ ܐܦܠܐ ܒܝ ܒܐܤܝܪܗ
ܐܠܐ ܫܩܘܠ ܒܝܫܬܐ ܥܡ ܐܘܢܓܠܝܘܢ ܒܚܝܠܐ ܕܐܠܗܐ ܀

<span id="an_1:9">1:9</span> ܗܘ ܕܐܚܝܢ ܘܩܪܢ ܒܩܪܝܢܐ ܩܕܝܫܐ ܠܘ ܐܝܟ ܥܒܕܝܢ ܐܠܐ
ܐܝܟ ܨܒܝܢܗ ܘܛܝܒܘܬܗ ܗܝ ܕܐܬܝܗܒܬ ܠܢ ܒܝܫܘܥ ܡܫܝܚܐ ܡܢ ܩܕܡ ܙܒܢܐ ܕܥܠܡܐ ܀

<span id="an_1:10">1:10</span> ܘܐܬܓܠܝܬ ܗܫܐ ܒܓܠܝܢܗ ܕܡܚܝܢܢ ܝܫܘܥ ܡܫܝܚܐ ܕܒܛܠ
ܠܡܘܬܐ ܘܚܘܝ ܚܝܐ ܘܠܐ ܡܬܚܒܠܢܘܬܐ ܒܐܘܢܓܠܝܘܢ ܀

<span id="an_1:11">1:11</span> ܗܘ ܕܐܬܬܤܝܡܬ ܒܗ ܐܢܐ ܟܪܘܙܐ ܘܫܠܝܚܐ ܘܡܠܦܢܐ
ܕܥܡܡܐ ܀

<span id="an_1:12">1:12</span> ܡܛܠ ܗܕܐ ܡܤܝܒܪ ܐܢܐ ܗܠܝܢ ܘܠܐ ܒܗܬ ܐܢܐ ܝܕܥ
ܐܢܐ ܓܝܪ ܒܡܢ ܗܝܡܢܬ ܘܡܦܤ ܐܢܐ ܕܡܛܝܐ ܒܐܝܕܘܗܝ ܓܘܥܠܢܝ ܠܡܛܪ ܠܝ ܠܝܘܡܐ ܗܘ ܀

<span id="an_1:13">1:13</span> ܢܗܘܝܢ ܠܟ ܚܘܪܐ ܡܠܐ ܚܠܝܡܬܐ ܕܫܡܥܬ ܡܢܝ
ܒܗܝܡܢܘܬܐ ܘܒܚܘܒܐ ܕܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:14">1:14</span> ܓܘܥܠܢܐ ܛܒܐ ܛܪ ܒܪܘܚܐ ܕܩܘܕܫܐ ܗܝ ܕܥܡܪܬ ܒܢ ܀

<span id="an_1:15">1:15</span> ܝܕܥ ܐܢܬ ܗܕܐ ܕܗܦܟܘ ܡܢܝ ܟܠܗܘܢ ܗܠܝܢ ܕܒܐܤܝܐ
ܕܐܝܬܝܗܘܢ ܡܢܗܘܢ ܦܘܓܠܘܤ ܘܗܪܡܓܢܘܤ ܀

<span id="an_1:16">1:16</span> ܢܬܠ ܡܪܢ ܪܚܡܐ ܠܒܝܬܗ ܕܐܢܤܝܦܘܪܘܤ ܕܙܒܢܝܢ
ܤܓܝܐܢ ܐܢܝܚܢܝ ܘܒܫܫܠܬܐ ܕܐܤܘܪܝ ܠܐ ܒܗܬ ܀

<span id="an_1:17">1:17</span> ܐܠܐ ܐܦ ܟܕ ܐܬܐ ܠܪܗܘܡܝ ܒܚܦܝܛܘܬܐ ܒܥܢܝ
ܘܐܫܟܚܢܝ ܀

<span id="an_1:18">1:18</span> ܢܬܠ ܠܗ ܡܪܢ ܕܢܫܟܚ ܪܚܡܐ ܠܘܬ ܡܪܢ ܒܝܘܡܐ ܗܘ
ܘܟܡܐ ܕܫܡܫܢܝ ܒܐܦܤܘܤ ܝܬܝܪܐܝܬ ܝܕܥ ܐܢܬ ܀

<span id="an_2:1">2:1</span> ܐܢܬ ܗܟܝܠ ܒܪܝ ܐܬܚܝܠ ܒܛܝܒܘܬܐ ܕܒܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_2:2">2:2</span> ܘܐܝܠܝܢ ܕܫܡܥܬ ܡܢܝ ܒܝܕ ܤܗܕܐ ܤܓܝܐܐ ܗܢܝܢ ܐܓܥܠ
ܠܐܢܫܐ ܡܗܝܡܢܐ ܐܝܠܝܢ ܕܡܛܝܐ ܒܐܝܕܝܗܘܢ ܐܦ ܠܐܚܪܢܐ ܠܡܠܦܘ ܀

<span id="an_2:3">2:3</span> ܘܤܝܒܪ ܒܝܫܬܐ ܐܝܟ ܦܠܚܐ ܛܒܐ ܕܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_2:4">2:4</span> ܠܐ ܐܢܫ ܦܠܚ ܘܡܬܦܟܪ ܒܨܒܘܬܐ ܕܥܠܡܐ ܕܢܫܦܪ ܠܗܘ ܡܢ
ܕܓܒܝܗܝ ܀

<span id="an_2:5">2:5</span> ܘܐܢ ܡܬܟܬܫ ܐܢܫ ܠܐ ܡܬܟܠܠ ܐܢ ܒܢܡܘܤܗ ܠܐ ܡܬܟܬܫ ܀

<span id="an_2:6">2:6</span> ܠܐܟܪܐ ܕܠܐܐ ܘܠܐ ܠܗ ܕܩܕܡܐ ܡܢ ܦܐܪܘܗܝ ܢܤܬܝܒܪ ܀

<span id="an_2:7">2:7</span> ܐܤܬܟܠ ܡܕܡ ܕܐܡܪ ܐܢܐ ܢܬܠ ܠܟ ܡܪܢ ܚܟܡܬܐ ܒܟܠ ܡܕܡ
܀

<span id="an_2:8">2:8</span> ܐܬܕܟܪ ܠܝܫܘܥ ܡܫܝܚܐ ܕܩܡ ܡܢ ܒܝܬ ܡܝܬܐ ܗܘ
ܕܐܝܬܘܗܝ ܡܢ ܙܪܥܐ ܕܕܘܝܕ ܐܝܟ ܐܘܢܓܠܝܘܢ ܕܝܠܝ ܀

<span id="an_2:9">2:9</span> ܕܒܗ ܤܒܠ ܐܢܐ ܒܝܫܬܐ ܥܕܡܐ ܠܐܤܘܪܐ ܐܝܟ ܥܒܕ ܒܝܫܬܐ
ܐܠܐ ܡܠܬܗ ܕܐܠܗܐ ܠܐ ܐܤܝܪܐ ܀

<span id="an_2:10">2:10</span> ܡܛܠ ܗܢܐ ܟܠܡܕܡ ܡܤܝܒܪ ܐܢܐ ܡܛܠ ܓܒܝܐ ܕܐܦ ܗܢܘܢ
ܢܫܟܚܘܢ ܚܝܐ ܕܒܝܫܘܥ ܡܫܝܚܐ ܥܡ ܫܘܒܚܐ ܕܠܥܠܡ ܀

<span id="an_2:11">2:11</span> ܡܗܝܡܢܐ ܗܝ ܡܠܬܐ ܐܢ ܓܝܪ ܡܝܬܢ ܥܡܗ ܐܦ ܥܡܗ ܢܚܐ
܀

<span id="an_2:12">2:12</span> ܘܐܢ ܢܤܝܒܪ ܐܦ ܢܡܠܟ ܥܡܗ ܐܢ ܕܝܢ ܢܟܦܘܪ ܒܗ ܐܦ
ܗܘ ܢܟܦܘܪ ܒܢ ܀

<span id="an_2:13">2:13</span> ܘܐܢ ܠܐ ܢܗܝܡܢ ܒܗ ܗܘ ܒܗܝܡܢܘܬܗ ܡܩܘܐ ܕܢܟܦܘܪ
ܓܝܪ ܒܢܦܫܗ ܠܐ ܡܫܟܚ ܀

<span id="an_2:14">2:14</span> ܗܠܝܢ ܗܘܝܬ ܡܥܗܕ ܠܗܘܢ ܘܡܤܗܕ ܩܕܡ ܡܪܢ ܕܠܐ
ܢܗܘܘܢ ܡܬܚܪܝܢ ܒܡܠܐ ܕܠܐ ܝܘܬܪܢ ܠܤܘܚܦܐ ܕܐܝܠܝܢ ܕܫܡܥܝܢ ܠܗܝܢ ܀

<span id="an_2:15">2:15</span> ܘܢܬܒܛܠ ܠܟ ܕܬܩܝܡ ܢܦܫܟ ܓܡܝܪܐܝܬ ܩܕܡ ܐܠܗܐ
ܦܠܚܐ ܕܠܐ ܒܗܬܬܐ ܕܡܟܪܙ ܬܪܝܨܐܝܬ ܡܠܬܐ ܕܫܪܪܐ ܀

<span id="an_2:16">2:16</span> ܘܡܢ ܡܠܐ ܤܪܝܩܬܐ ܕܠܝܬ ܒܗܝܢ ܚܫܚܘ ܐܫܬܐܠ
ܝܬܝܪܐܝܬ ܓܝܪ ܢܘܤܦܘܢ ܥܠ ܪܘܫܥܗܘܢ ܗܢܘܢ ܕܥܢܝܢ ܒܗܝܢ ܀

<span id="an_2:17">2:17</span> ܘܡܠܬܗܘܢ ܐܝܟ ܚܠܕܝܬܐ ܢܘܡܐ ܬܐܚܘܕ ܒܤܓܝܐܐ
ܐܝܬܘܗܝ ܕܝܢ ܚܕ ܡܢܗܘܢ ܗܘܡܢܐܘܤ ܘܐܚܪܢܐ ܦܝܠܛܘܤ ܀

<span id="an_2:18">2:18</span> ܗܠܝܢ ܕܛܥܘ ܡܢ ܫܪܪܐ ܟܕ ܐܡܪܝܢ ܕܩܝܡܬܐ ܕܡܝܬܐ
ܗܘܬ ܠܗ ܘܗܝܡܢܘܬܐ ܕܐܢܫ ܐܢܫ ܡܗܦܟܝܢ ܀

<span id="an_2:19">2:19</span> ܫܬܐܤܬܐ ܕܝܢ ܫܪܝܪܬܐ ܕܐܠܗܐ ܩܝܡܐ ܗܝ ܘܐܝܬ ܠܗ
ܚܬܡܐ ܗܢܐ ܘܝܕܥ ܡܪܝܐ ܠܐܝܠܝܢ ܕܕܝܠܗ ܐܢܘܢ ܘܢܦܪܘܩ ܡܢ ܥܘܠܐ ܟܠ ܕܩܪܐ ܠܫܡܗ ܕܡܪܝܐ ܀

<span id="an_2:20">2:20</span> ܒܒܝܬܐ ܕܝܢ ܪܒܐ ܠܐ ܗܘܐ ܡܐܢܐ ܕܕܗܒܐ ܒܠܚܘܕ ܐܘ
ܕܤܐܡܐ ܐܝܬ ܒܗ ܐܠܐ ܐܦ ܕܩܝܤܐ ܐܦ ܕܦܚܪܐ ܡܢܗܘܢ ܠܐܝܩܪܐ ܘܡܢܗܘܢ ܠܨܥܪܐ ܀

<span id="an_2:21">2:21</span> ܐܢ ܐܢܫ ܗܟܝܠ ܢܕܟܐ ܢܦܫܗ ܡܢ ܗܠܝܢ ܗܘܐ ܡܐܢܐ
ܕܟܝܐ ܠܐܝܩܪܐ ܕܥܗܢ ܠܚܘܫܚܐ ܕܡܪܗ ܘܡܛܝܒ ܠܟܠ ܥܒܕ ܛܒ ܀

<span id="an_2:22">2:22</span> ܡܢ ܟܠܗܝܢ ܪܓܝܓܬܐ ܕܛܠܝܘܬܐ ܥܪܘܩ ܘܗܪܛ ܒܬܪ
ܟܐܢܘܬܐ ܘܗܝܡܢܘܬܐ ܘܚܘܒܐ ܘܫܠܡܐ ܥܡ ܐܝܠܝܢ ܕܩܪܝܢ ܠܡܪܢ ܒܠܒܐ ܕܟܝܐ ܀

<span id="an_2:23">2:23</span> ܚܪܝܢܐ ܤܟܠܐ ܐܝܠܝܢ ܕܕܠܐ ܡܪܕܘ ܐܢܘܢ ܐܫܬܐܠ
ܡܢܗܘܢ ܝܕܥ ܐܢܬ ܓܝܪ ܕܬܟܬܘܫܐ ܡܘܠܕܝܢ ܀

<span id="an_2:24">2:24</span> ܥܒܕܗ ܕܝܢ ܕܡܪܢ ܠܐ ܚܝܒ ܠܡܬܟܬܫܘ ܐܠܐ ܕܢܗܘܐ
ܡܟܝܟ ܠܘܬ ܟܠܢܫ ܘܡܠܦܢ ܘܢܓܝܪܐ ܪܘܚܗ ܀

<span id="an_2:25">2:25</span> ܕܢܗܘܐ ܪܕܐ ܠܐܝܠܝܢ ܕܡܬܚܪܝܢ ܠܩܘܒܠܗ ܒܡܟܝܟܘܬܐ
ܕܠܡܐ ܢܬܠ ܠܗܘܢ ܐܠܗܐ ܬܝܒܘܬܐ ܘܢܕܥܘܢ ܫܪܪܐ ܀

<span id="an_2:26">2:26</span> ܘܢܥܗܕܘܢ ܢܦܫܗܘܢ ܘܢܦܪܩܘܢ ܡܢ ܦܚܗ ܕܤܛܢܐ ܕܒܗ
ܐܬܬܨܝܕܘ ܠܨܒܝܢܗ ܀

<span id="an_3:1">3:1</span> ܗܕܐ ܕܝܢ ܗܘܝܬ ܝܕܥ ܕܒܝܘܡܬܐ ܐܚܪܝܐ ܢܐܬܘܢ ܙܒܢܐ
ܩܫܝܐ ܀

<span id="an_3:2">3:2</span> ܘܢܗܘܘܢ ܒܢܝܢܫܐ ܪܚܡܝ ܢܦܫܗܘܢ ܘܪܚܡܝ ܟܤܦܐ ܫܒܗܪܢܐ
ܪܡܐ ܡܓܕܦܢܐ ܕܠܐܢܫܝܗܘܢ ܠܐ ܡܬܛܦܝܤܝܢ ܟܦܪܝ ܒܛܝܒܘܬܐ ܪܫܝܥܐ ܀

<span id="an_3:3">3:3</span> ܐܟܠܝ ܩܪܨܐ ܡܫܥܒܕܝ ܠܪܓܬܐ ܒܥܪܝܪܝܐ ܤܢܝܝ ܛܒܬܐ ܀

<span id="an_3:4">3:4</span> ܡܫܠܡܢܐ ܡܤܪܗܒܐ ܚܬܝܪܐ ܪܚܡܝ ܪܓܝܓܬܐ ܛܒ ܡܢ ܚܘܒܗ
ܕܐܠܗܐ ܀

<span id="an_3:5">3:5</span> ܕܐܝܬ ܠܗܘܢ ܐܤܟܡܐ ܕܕܚܠܬ ܐܠܗܐ ܘܡܢ ܚܝܠܗ ܪܚܝܩܝܢ
ܐܝܠܝܢ ܕܗܟܢܐ ܐܢܘܢ ܕܚܘܩ ܐܢܘܢ ܡܢܟ ܀

<span id="an_3:6">3:6</span> ܡܢܗܘܢ ܐܢܘܢ ܓܝܪ ܗܠܝܢ ܕܡܚܠܕܝܢ ܒܝܬ ܒܬܐ ܘܫܒܝܢ
ܢܫܐ ܕܛܡܝܪܢ ܒܚܛܗܐ ܘܡܬܕܒܪܢ ܠܪܓܝܓܬܐ ܡܫܚܠܦܬܐ ܀

<span id="an_3:7">3:7</span> ܕܒܟܠܙܒܢ ܝܠܦܢ ܘܡܡܬܘܡ ܠܝܕܥܬܐ ܕܫܪܪܐ ܠܡܐܬܐ ܠܐ
ܡܫܟܚܝܢ ܀

<span id="an_3:8">3:8</span> ܐܝܟܢܐ ܕܝܢ ܕܝܢܤ ܘܝܡܒܪܝܤ ܩܡܘ ܠܘܩܒܠ ܡܘܫܐ ܗܟܢܐ
ܐܦ ܗܢܘܢ ܩܝܡܝܢ ܠܘܩܒܠ ܫܪܪܐ ܐܢܫܐ ܕܡܚܒܠ ܪܥܝܢܗܘܢ ܘܤܠܝܢ ܡܢ ܗܝܡܢܘܬܐ ܀

<span id="an_3:9">3:9</span> ܐܠܐ ܠܐ ܢܐܬܘܢ ܠܩܕܡܝܗܘܢ ܫܛܝܘܬܗܘܢ ܓܝܪ ܡܬܝܕܥܐ
ܗܝ ܠܟܠܢܫ ܐܝܟܢܐ ܕܐܦ ܕܗܢܘܢ ܐܬܝܕܥܬ ܀

<span id="an_3:10">3:10</span> ܐܢܬ ܕܝܢ ܐܬܝܬ ܒܬܪ ܝܘܠܦܢܝ ܘܒܬܪ ܕܘܒܪܝ ܘܒܬܪ
ܨܒܝܢܝ ܘܒܬܪ ܗܝܡܢܘܬܝ ܘܒܬܪ ܢܓܝܪܘܬ ܪܘܚܝ ܘܒܬܪ ܚܘܒܝ ܘܒܬܪ ܡܤܝܒܪܢܘܬܝ ܀

<span id="an_3:11">3:11</span> ܘܒܬܪ ܪܕܝܦܘܬܝ ܘܒܬܪ ܚܫܝ ܘܝܕܥ ܐܢܬ ܐܝܠܝܢ
ܤܝܒܪܬ ܒܐܢܛܝܟܝܐ ܘܒܐܝܩܢܘܢ ܘܒܠܘܤܛܪܐ ܐܝܕܐ ܪܕܝܦܘܬܐ ܤܝܒܪܬ ܘܡܢ ܟܠܗܝܢ ܦܨܝܢܝ ܡܪܝ
܀

<span id="an_3:12">3:12</span> ܘܟܠܗܘܢ ܕܝܢ ܐܝܠܝܢ ܕܨܒܝܢ ܒܕܚܠܬ ܐܠܗܐ ܕܢܚܘܢ
ܒܝܫܘܥ ܡܫܝܚܐ ܡܬܪܕܦܝܢ ܀

<span id="an_3:13">3:13</span> ܒܢܝܢܫܐ ܕܝܢ ܒܝܫܐ ܘܡܛܥܝܢܐ ܢܘܤܦܘܢ ܥܠ
ܒܝܫܘܬܗܘܢ ܟܕ ܛܥܝܢ ܘܡܛܥܝܢ ܀

<span id="an_3:14">3:14</span> ܐܢܬ ܕܝܢ ܩܘܐ ܒܐܝܠܝܢ ܕܝܠܦܬ ܘܐܫܬܪܪܬ ܝܕܥ ܐܢܬ
ܓܝܪ ܡܢ ܡܢܘ ܝܠܦܬ ܀

<span id="an_3:15">3:15</span> ܘܕܡܢ ܛܠܝܘܬܟ ܤܦܪܐ ܩܕܝܫܐ ܝܠܝܦ ܐܢܬ ܕܡܫܟܚܝܢ
ܕܢܚܟܡܘܢܟ ܠܚܝܐ ܒܗܝܡܢܘܬܐ ܕܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_3:16">3:16</span> ܟܠ ܟܬܒ ܕܒܪܘܚܐ ܐܬܟܬܒ ܡܘܬܪܢܐ ܗܘ ܠܝܘܠܦܢܐ
ܘܠܟܘܘܢܐ ܘܠܬܘܪܨܐ ܘܠܡܪܕܘܬܐ ܕܒܟܐܢܘܬܐ ܀

<span id="an_3:17">3:17</span> ܕܢܗܘܐ ܓܡܝܪ ܒܪܢܫܐ ܕܐܠܗܐ ܘܠܟܠ ܥܒܕ ܛܒ ܡܫܠܡܢ
܀

<span id="an_4:1">4:1</span> ܡܤܗܕ ܐܢܐ ܠܟ ܩܕܡ ܐܠܗܐ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܗܘ
ܕܥܬܝܕ ܠܡܕܢ ܚܝܐ ܘܡܝܬܐ ܒܓܠܝܢܐ ܕܡܠܟܘܬܗ ܀

<span id="an_4:2">4:2</span> ܐܟܪܙ ܡܠܬܐ ܘܩܘܡ ܒܚܦܝܛܘܬܐ ܒܙܒܢܐ ܘܕܠܐ ܙܒܢܐ ܐܟܤ
ܘܟܘܢ ܒܟܠܗ ܡܓܪܬ ܪܘܚܐ ܘܝܘܠܦܢܐ ܀

<span id="an_4:3">4:3</span> ܢܗܘܐ ܓܝܪ ܙܒܢܐ ܕܠܝܘܠܦܢܐ ܚܠܝܡܐ ܠܐ ܢܫܬܡܥܘܢ ܐܠܐ
ܐܝܟ ܪܓܝܓܬܗܘܢ ܢܤܓܘܢ ܠܢܦܫܗܘܢ ܡܠܦܢܐ ܒܚܘܬܚܬܐ ܕܡܫܡܥܬܗܘܢ ܀

<span id="an_4:4">4:4</span> ܘܡܢ ܫܪܪܐ ܢܗܦܟܘܢ ܐܕܢܗܘܢ ܠܫܘܥܝܬܐ ܕܝܢ ܢܤܛܘܢ ܀

<span id="an_4:5">4:5</span> ܐܢܬ ܕܝܢ ܗܘܝܬ ܥܝܪ ܒܟܠܡܕܡ ܘܤܝܒܪ ܒܝܫܬܐ ܘܥܒܕܐ
ܥܒܕ ܕܡܤܒܪܢܐ ܘܬܫܡܫܬܟ ܫܠܡ ܀

<span id="an_4:6">4:6</span> ܐܢܐ ܓܝܪ ܡܟܝܠ ܡܬܢܩܐ ܐܢܐ ܘܙܒܢܐ ܕܐܫܬܪܐ ܡܛܐ ܀

<span id="an_4:7">4:7</span> ܐܓܘܢܐ ܫܦܝܪܐ ܐܬܟܬܫܬ ܘܪܗܛܝ ܫܠܡܬ ܘܗܝܡܢܘܬܝ ܢܛܪܬ
܀

<span id="an_4:8">4:8</span> ܘܡܢ ܗܫܐ ܢܛܝܪ ܠܝ ܟܠܝܠܐ ܕܟܐܢܘܬܐ ܕܢܦܪܥܝܘܗܝ ܠܝ
ܡܪܝ ܒܝܘܡܐ ܗܘ ܕܗܘܝܘ ܕܝܢܐ ܟܐܢܐ ܠܐ ܕܝܢ ܒܠܚܘܕ ܠܝ ܐܠܐ ܐܦ ܠܐܝܠܝܢ ܕܐܚܒܘ ܠܓܠܝܢܗ
܀

<span id="an_4:9">4:9</span> ܢܬܒܛܠ ܠܟ ܕܬܐܬܐ ܠܘܬܝ ܒܥܓܠ ܀

<span id="an_4:10">4:10</span> ܕܡܐ ܓܝܪ ܫܒܩܢܝ ܘܐܚܒ ܥܠܡܐ ܗܢܐ ܘܐܙܠ ܠܗ
ܠܬܤܠܘܢܝܩܐ ܩܪܤܩܘܤ ܠܓܠܛܝܐ ܛܛܘܤ ܠܕܠܡܛܝܐ ܀

<span id="an_4:11">4:11</span> ܠܘܩܐ ܗܘ ܒܠܚܘܕܘܗܝ ܥܡܝ ܠܡܪܩܘܤ ܕܒܪ ܘܐܝܬܝܗܝ
ܥܡܟ ܥܗܢ ܠܝ ܓܝܪ ܠܬܫܡܫܬܐ ܀

<span id="an_4:12">4:12</span> ܠܛܘܟܝܩܘܤ ܕܝܢ ܫܕܪܬ ܠܐܦܤܘܤ ܀

<span id="an_4:13">4:13</span> ܒܝܬ ܟܬܒܐ ܕܝܢ ܕܫܒܩܬ ܒܛܪܘܐܘܤ ܠܘܬ ܩܪܦܘܤ ܡܐ
ܕܐܬܐ ܐܢܬ ܐܝܬܝܗܝ ܘܟܬܒܐ ܝܬܝܪܐܝܬ ܟܪܟܐ ܕܡܓܠܐ ܀

<span id="an_4:14">4:14</span> ܐܠܟܤܢܕܪܤ ܩܝܢܝܐ ܒܝܫܬܐ ܤܓܝܐܬܐ ܚܘܝܢܝ ܦܪܥ ܠܗ
ܡܪܢ ܐܝܟ ܥܒܕܘܗܝ ܀

<span id="an_4:15">4:15</span> ܐܦ ܐܢܬ ܕܝܢ ܐܙܕܗܪ ܡܢܗ ܛܒ ܓܝܪ ܙܩܝܦ ܠܘܩܒܠ
ܡܠܝܢ ܀

<span id="an_4:16">4:16</span> ܒܡܦܩ ܒܪܘܚܝ ܩܕܡܝܐ ܠܐ ܐܢܫ ܗܘܐ ܥܡܝ ܐܠܐ ܟܠܗܘܢ
ܫܒܩܘܢܝ ܠܐ ܬܬܚܫܒ ܠܗܘܢ ܗܕܐ ܀

<span id="an_4:17">4:17</span> ܡܪܝ ܕܝܢ ܩܡ ܠܝ ܘܚܝܠܢܝ ܕܒܝ ܟܪܘܙܘܬܐ ܬܫܬܠܡ
ܘܢܫܡܥܘܢ ܥܡܡܐ ܟܠܗܘܢ ܕܐܬܦܨܝܬ ܡܢ ܦܘܡܐ ܕܐܪܝܐ ܀

<span id="an_4:18">4:18</span> ܘܢܦܨܝܢܝ ܡܪܝ ܡܢ ܟܠ ܥܒܕ ܒܝܫ ܘܢܚܝܢܝ ܒܡܠܟܘܬܗ
ܕܒܫܡܝܐ ܕܠܗ ܫܘܒܚܐ ܠܥܠܡ ܥܠܡܝܢ ܐܡܝܢ ܀

<span id="an_4:19">4:19</span> ܗܒ ܫܠܡܐ ܠܦܪܝܤܩܠܐ ܘܠܐܩܠܘܤ ܘܠܒܝܬܐ
ܕܐܢܤܝܦܘܪܘܤ ܀

<span id="an_4:20">4:20</span> ܐܪܤܛܘܤ ܦܫ ܠܗ ܒܩܘܪܢܬܘܤ ܛܪܘܦܝܡܘܤ ܕܝܢ ܫܒܩܬܗ
ܟܕ ܟܪܝܗ ܒܡܝܠܝܛܘܤ ܡܕܝܢܬܐ ܀

<span id="an_4:21">4:21</span> ܢܬܒܛܠ ܠܟ ܕܩܕܡ ܤܬܘܐ ܬܐܬܐ ܫܐܠ ܒܫܠܡܟ ܐܘܒܘܠܘܤ
ܘܦܘܕܤ ܘܠܝܢܘܤ ܘܩܠܘܕܝܐ ܘܐܚܐ ܟܠܗܘܢ ܀

<span id="an_4:22">4:22</span> ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡ ܪܘܚܟ ܛܝܒܘܬܐ ܥܡܟ ܐܡܝܢ ܀
