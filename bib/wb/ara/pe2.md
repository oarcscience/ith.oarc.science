### Aramaic Bible: 2 Peter

<span id="an_1:1">1:1</span> ܫܡܥܘܢ ܦܛܪܘܤ ܥܒܕܐ ܘܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܠܐܝܠܝܢ
ܕܠܗܝܡܢܘܬܐ ܫܘܝܬ ܒܐܝܩܪܐ ܥܡܢ ܐܫܬܘܝܘ ܒܙܕܝܩܘܬܐ ܕܡܪܢ ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:2">1:2</span> ܛܝܒܘܬܐ ܘܫܠܡܐ ܢܤܓܐ ܠܟܘܢ ܒܫܘܘܕܥܐ ܕܡܪܢ ܝܫܘܥ
ܡܫܝܚܐ ܀

<span id="an_1:3">1:3</span> ܐܝܟ ܡܢ ܕܟܠܗܝܢ ܐܝܠܝܢ ܕܐܝܬܝܗܝܢ ܕܚܝܠܐ ܐܠܗܝܐ
ܠܘܬ ܚܝܐ ܘܕܚܠܬ ܐܠܗܐ ܝܗܒ ܒܝܕ ܫܘܘܕܥܐ ܕܗܘ ܕܩܪܐ ܠܢ ܒܬܫܒܘܚܬܐ ܕܝܠܗ ܘܡܝܬܪܘܬܐ ܀

<span id="an_1:4">1:4</span> ܕܒܐܝܕܝܗܝܢ ܫܘܘܕܝܐ ܪܘܪܒܐ ܘܝܩܝܪܐ ܠܟܘܢ ܝܗܒ ܕܒܝܕ
ܗܠܝܢ ܬܗܘܘܢ ܫܘܬܦܐ ܕܟܝܢܐ ܐܠܗܝܐ ܟܕ ܥܪܩܝܢ ܐܢܬܘܢ ܡܢ ܚܒܠܐ ܕܪܓܝܓܬܐ ܕܒܥܠܡܐ ܀

<span id="an_1:5">1:5</span> ܘܗܝ ܕܝܢ ܗܕܐ ܟܕ ܫܩܠܛܥܢܐ ܟܠܗ ܡܥܠܝܢ ܐܢܬܘܢ
ܐܘܤܦܘ ܥܠ ܗܝܡܢܘܬܟܘܢ ܡܝܬܪܘܬܐ ܥܠ ܕܝܢ ܡܝܬܪܘܬܐ ܝܕܥܬܐ ܀

<span id="an_1:6">1:6</span> ܥܠ ܕܝܢ ܝܕܥܬܐ ܡܚܡܤܢܢܘܬܐ ܥܠ ܕܝܢ ܡܚܡܤܢܢܘܬܐ
ܡܤܝܒܪܢܘܬܐ ܥܠ ܕܝܢ ܡܤܝܒܪܢܘܬܐ ܕܚܠܬ ܐܠܗܐ ܀

<span id="an_1:7">1:7</span> ܥܠ ܕܝܢ ܕܚܠܬ ܐܠܗܐ ܪܚܡܬ ܐܚܘܬܐ ܥܠ ܕܝܢ ܪܚܡܬ
ܐܚܘܬܐ ܚܘܒܐ ܀

<span id="an_1:8">1:8</span> ܗܠܝܢ ܓܝܪ ܟܕ ܫܟܝܚܢ ܠܟܘܢ ܘܝܬܝܪܢ ܠܐ ܗܘܐ ܒܛܝܠܐ
ܐܦܠܐ ܕܠܐ ܦܐܪܐ ܡܩܝܡܢ ܠܟܘܢ ܒܫܘܘܕܥܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:9">1:9</span> ܗܘ ܓܝܪ ܕܠܐ ܫܟܝܚܢ ܠܗ ܗܠܝܢ ܤܡܝܐ ܐܝܬܘܗܝ ܕܠܐ
ܚܙܐ ܕܛܥܐ ܕܘܟܝܐ ܕܚܛܗܘܗܝ ܩܕܡܝܐ ܀

<span id="an_1:10">1:10</span> ܘܥܠ ܗܝ ܝܬܝܪܐܝܬ ܐܚܝ ܝܨܦܘ ܕܒܝܕ ܥܒܕܝܟܘܢ ܛܒܐ
ܩܪܝܬܟܘܢ ܘܓܒܝܬܟܘܢ ܡܫܪܪܬܐ ܬܥܒܕܘܢ ܟܕ ܓܝܪ ܗܠܝܢ ܥܒܕܝܢ ܐܢܬܘܢ ܠܐ ܡܬܘܡ ܡܫܬܪܥܝܢ
ܐܢܬܘܢ ܀

<span id="an_1:11">1:11</span> ܗܟܢܐ ܓܝܪ ܥܬܝܪܐܝܬ ܡܬܝܗܒܐ ܠܟܘܢ ܡܥܠܬܐ
ܕܡܠܟܘܬܐ ܕܠܥܠܡ ܕܡܪܢ ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܀

<span id="an_1:12">1:12</span> ܘܥܠ ܗܕܐ ܠܐ ܡܡܐܢ ܐܢܐ ܡܢ ܕܡܥܗܕ ܐܢܐ ܠܟܘܢ
ܐܡܝܢܐܝܬ ܥܠ ܗܠܝܢ ܟܕ ܛܒ ܐܦ ܝܕܥܝܢ ܐܢܬܘܢ ܘܤܡܝܟܝܢ ܐܢܬܘܢ ܥܠ ܫܪܪܐ ܗܢܐ ܀

<span id="an_1:13">1:13</span> ܕܟܐܢܐ ܕܝܢ ܐܤܬܒܪܬ ܠܝ ܕܟܡܐ ܕܐܝܬܝ ܒܦܓܪܐ ܗܢܐ
ܐܥܝܪܟܘܢ ܒܥܘܗܕܢܐ ܀

<span id="an_1:14">1:14</span> ܟܕ ܝܕܥ ܐܢܐ ܕܥܘܢܕܢܐ ܕܦܓܪܝ ܒܥܓܠ ܗܘܐ ܐܝܟܢܐ
ܕܐܦ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܘܕܥܢܝ ܀

<span id="an_1:15">1:15</span> ܝܨܦ ܐܢܐ ܕܝܢ ܕܐܦ ܐܡܝܢܐܝܬ ܗܘܐ ܐܝܬ ܠܟܘܢ ܕܐܦ
ܡܢ ܒܬܪ ܡܦܩܢܐ ܕܝܠܝ ܥܘܗܕܢܐ ܕܗܠܝܢ ܬܗܘܘܢ ܥܒܕܝܢ ܀

<span id="an_1:16">1:16</span> ܠܐ ܗܘܐ ܓܝܪ ܟܕ ܒܬܪ ܡܬܠܐ ܕܥܒܝܕܝܢ ܒܐܘܡܢܘܬܐ
ܐܙܠܝܢܢ ܐܘܕܥܢܟܘܢ ܚܝܠܗ ܘܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܠܐ ܟܕ ܚܙܝܐ ܗܘܝܢ ܕܪܒܘܬܐ
ܕܝܠܗ ܀

<span id="an_1:17">1:17</span> ܟܕ ܓܝܪ ܢܤܒ ܡܢ ܐܠܗܐ ܐܒܐ ܐܝܩܪܐ ܘܬܫܒܘܚܬܐ ܟܕ
ܩܠܐ ܐܬܐ ܠܗ ܕܐܝܟ ܗܢܐ ܡܢ ܬܫܒܘܚܬܐ ܦܐܝܬ ܒܪܒܘܬܗ ܕܗܢܘ ܒܪܝ ܗܘ ܚܒܝܒܐ ܗܘ ܕܒܗ
ܐܨܛܒܝܬ ܀

<span id="an_1:18">1:18</span> ܐܦ ܚܢܢ ܠܗ ܠܗܢܐ ܩܠܐ ܫܡܥܢܢ ܡܢ ܫܡܝܐ ܕܐܬܐ ܠܗ
ܟܕ ܐܝܬܝܢ ܗܘܝܢ ܥܡܗ ܒܛܘܪܐ ܩܕܝܫܐ ܀

<span id="an_1:19">1:19</span> ܘܐܝܬ ܠܢ ܕܫܪܝܪܐ ܐܦ ܡܠܬܐ ܕܢܒܝܘܬܐ ܗܝ ܕܫܦܝܪ
ܥܒܕܝܢ ܐܢܬܘܢ ܟܕ ܒܗ ܚܝܪܝܢ ܐܢܬܘܢ ܐܝܟ ܕܒܫܪܓܐ ܕܡܢܗܪ ܒܐܬܪܐ ܥܡܘܛܐ ܥܕܡܐ ܕܐܝܡܡܐ
ܢܢܗܪ ܘܫܡܫܐ ܢܕܢܚ ܒܠܒܘܬܟܘܢ ܀

<span id="an_1:20">1:20</span> ܟܕ ܗܕܐ ܠܘܩܕܡ ܝܕܥܝܢ ܐܢܬܘܢ ܕܟܠ ܢܒܝܘܬܐ ܫܪܝܐ
ܕܟܬܒܐ ܕܝܠܗ ܠܐ ܗܘܝܐ ܀

<span id="an_1:21">1:21</span> ܠܐ ܗܘܐ ܓܝܪ ܒܨܒܝܢܐ ܕܒܪܢܫܐ ܐܬܬ ܡܢ ܡܬܘܡ
ܢܒܝܘܬܐ ܐܠܐ ܟܕ ܡܢ ܪܘܚܐ ܩܕܝܫܐ ܡܬܢܓܕܝܢ ܡܠܠܘ ܩܕܝܫܐ ܒܢܝܢܫܐ ܕܐܠܗܐ ܀

<span id="an_2:1">2:1</span> ܗܘܘ ܕܝܢ ܐܦ ܢܒܝܐ ܕܓܠܐ ܒܥܡܐ ܐܝܟ ܕܐܦ ܒܟܘܢ
ܢܗܘܘܢ ܡܠܦܢܐ ܕܓܠܐ ܐܝܠܝܢ ܕܡܥܠܝܢ ܗܪܤܝܤ ܕܐܒܕܢܐ ܘܒܡܪܐ ܕܙܒܢ ܐܢܘܢ ܟܦܪܝܢ ܟܕ
ܡܝܬܝܢ ܥܠ ܢܦܫܗܘܢ ܐܒܕܢܐ ܡܤܪܗܒܐ ܀

<span id="an_2:2">2:2</span> ܘܤܓܝܐܐ ܢܐܙܠܘܢ ܒܬܪ ܛܢܦܘܬܐ ܕܝܠܗܘܢ ܗܠܝܢ
ܕܡܛܠܬܗܘܢ ܐܘܪܚܐ ܕܫܪܪܐ ܬܬܓܕܦ ܀

<span id="an_2:3">2:3</span> ܘܒܥܠܘܒܘܬܐ ܘܒܡܠܐ ܕܒܕܝܐ ܢܬܬܓܪܘܢ ܒܟܘܢ ܗܠܝܢ
ܕܕܝܢܗܘܢ ܡܢ ܩܕܝܡ ܠܐ ܒܛܠ ܘܐܒܕܢܗܘܢ ܠܐ ܢܐܡ ܀

<span id="an_2:4">2:4</span> ܐܢ ܓܝܪ ܐܠܗܐ ܥܠ ܡܠܐܟܐ ܕܚܛܘ ܠܐ ܚܤ ܐܠܐ ܒܫܫܠܬܐ
ܕܥܡܛܢܐ ܥܓܢ ܐܢܘܢ ܒܬܚܬܝܬܐ ܘܐܫܠܡ ܐܢܘܢ ܕܢܬܢܛܪܘܢ ܠܕܝܢܐ ܕܫܘܢܩܐ ܀

<span id="an_2:5">2:5</span> ܘܥܠ ܥܠܡܐ ܩܕܡܝܐ ܠܐ ܚܤ ܐܠܐ ܠܢܘܚ ܕܬܡܢܝܐ ܟܪܘܙܐ
ܕܟܐܢܘܬܐ ܢܛܪ ܟܕ ܛܘܦܢܐ ܥܠ ܥܠܡܐ ܕܪܫܝܥܐ ܐܝܬܝ ܀

<span id="an_2:6">2:6</span> ܘܟܕ ܠܡܕܝܢܬܐ ܕܤܕܘܡ ܘܕܥܡܘܪܐ ܐܘܩܕ ܘܒܗܦܘܟܝܐ ܚܝܒ
ܐܢܝܢ ܟܕ ܬܚܘܝܬܐ ܠܪܫܝܥܐ ܕܥܬܝܕܝܢ ܕܢܗܘܘܢ ܤܡ ܀

<span id="an_2:7">2:7</span> ܐܦ ܠܠܘܛ ܙܕܝܩܐ ܕܡܬܩܦܚ ܗܘܐ ܡܢ ܗܘܦܟܐ ܕܒܛܢܦܘܬܐ
ܕܗܠܝܢ ܕܕܠܐ ܢܡܘܤܐ ܐܝܬܝܗܘܢ ܗܘܘ ܦܨܝ ܀

<span id="an_2:8">2:8</span> ܒܚܙܬܐ ܓܝܪ ܘܒܫܡܥܐ ܟܕ ܥܡܪ ܗܘܐ ܟܐܢܐ ܗܘ ܒܝܢܬܗܘܢ
ܝܘܡ ܡܢ ܝܘܡ ܠܢܦܫܗ ܙܕܝܩܬܐ ܒܥܒܕܐ ܕܠܐ ܢܡܘܤ ܡܫܢܩ ܗܘܐ ܀

<span id="an_2:9">2:9</span> ܝܕܥ ܗܘ ܡܪܝܐ ܕܢܦܪܘܩ ܡܢ ܐܘܠܨܢܐ ܠܐܝܠܝܢ ܕܕܚܠܝܢ
ܠܗ ܠܥܘܠܐ ܕܝܢ ܠܝܘܡܐ ܕܕܝܢܐ ܟܕ ܡܫܬܢܩܝܢ ܢܛܪ ܀

<span id="an_2:10">2:10</span> ܝܬܝܪܐܝܬ ܕܝܢ ܠܐܝܠܝܢ ܕܒܬܪ ܒܤܪܐ ܒܪܓܬܐ
ܕܛܡܐܘܬܐ ܐܙܠܝܢ ܘܥܠ ܡܪܘܬܐ ܡܒܤܪܝܢ ܡܪܚܐ ܘܡܫܩܠܐ ܕܡܢ ܬܫܒܘܚܬܐ ܠܐ ܙܝܥܝܢ ܟܕ
ܡܓܕܦܝܢ ܀

<span id="an_2:11">2:11</span> ܐܝܟܐ ܕܡܠܐܟܐ ܕܒܚܝܠܐ ܘܒܥܘܫܢܐ ܪܘܪܒܝܢ ܡܢܗܘܢ
ܠܐ ܡܝܬܝܢ ܥܠܝܗܘܢ ܡܢ ܡܪܝܐ ܕܝܢܐ ܕܓܘܕܦܐ ܀

<span id="an_2:12">2:12</span> ܗܠܝܢ ܕܝܢ ܕܐܝܟ ܚܝܘܬܐ ܚܪܫܬܐ ܗܘܘ ܒܟܝܢܐ ܠܚܪܒܐ
ܘܠܚܒܠܐ ܟܕ ܒܐܝܠܝܢ ܕܠܐ ܝܕܥܝܢ ܡܓܕܦܝܢ ܒܚܒܠܐ ܕܝܠܗܘܢ ܢܬܚܒܠܘܢ ܀

<span id="an_2:13">2:13</span> ܟܕ ܐܝܠܝܢ ܕܗܘܐ ܒܗܘܢ ܥܘܠܐ ܐܓܪܐ ܕܥܘܠܐ ܗܢܝܘܬܐ
ܚܫܝܒ ܠܗܘܢ ܒܘܤܡܐ ܕܗܘܐ ܒܐܝܡܡܐ ܡܟܬܡܐ ܘܡܠܝܝ ܡܘܡܐ ܕܟܕ ܡܬܒܤܡܝܢ ܒܢܝܚܬܗܘܢ
ܡܬܦܢܩܝܢ ܀

<span id="an_2:14">2:14</span> ܟܕ ܥܝܢܐ ܐܝܬ ܠܗܘܢ ܕܡܠܝܢ ܓܘܪܐ ܘܚܛܗܐ ܕܠܐ
ܡܘܦܝܢ ܟܕ ܡܫܕܠܝܢ ܠܢܦܫܬܐ ܕܠܐ ܤܡܝܟܢ ܘܠܒܐ ܕܡܕܪܫ ܒܥܠܘܒܘܬܐ ܐܝܬ ܠܗܘܢ ܒܢܝܐ
ܕܠܘܛܬܐ ܀

<span id="an_2:15">2:15</span> ܕܟܕ ܫܒܩܘ ܐܘܪܚܐ ܬܪܝܨܬܐ ܫܓܘ ܘܐܙܠܘ ܒܐܘܪܚܐ
ܕܒܠܥܡ ܒܪ ܒܥܘܪ ܗܘ ܕܐܓܪܐ ܕܥܘܠܐ ܐܚܒ ܀

<span id="an_2:16">2:16</span> ܡܟܤܢܘܬܐ ܕܝܢ ܕܗܘܬ ܠܗ ܠܡܬܥܒܪܢܘܬܗ ܐܬܢܐ ܕܠܐ
ܩܠܐ ܕܒܩܠܐ ܕܒܢܝܢܫܐ ܡܠܠܬ ܟܠܬ ܠܫܛܝܘܬܗ ܕܢܒܝܐ ܀

<span id="an_2:17">2:17</span> ܗܠܝܢ ܐܢܘܢ ܡܥܝܢܐ ܕܠܐ ܡܝܐ ܥܢܢܐ ܕܡܢ ܥܠܥܠܐ
ܡܬܪܕܦܢ ܗܠܝܢ ܕܥܡܛܢܐ ܕܚܫܘܟܐ ܢܛܝܪ ܠܗܘܢ ܀

<span id="an_2:18">2:18</span> ܟܕ ܓܝܪ ܓܘܢܚܐ ܕܤܪܝܩܘܬܐ ܡܡܠܠܝܢ ܡܫܕܠܝܢ
ܒܪܓܝܓܬܐ ܛܢܦܬܐ ܕܒܤܪܐ ܠܐܝܠܝܢ ܕܒܡܠܐ ܩܠܝܠ ܥܪܩܝܢ ܡܢ ܐܝܠܝܢ ܕܒܛܥܝܘܬܐ ܡܬܗܦܟܝܢ ܀

<span id="an_2:19">2:19</span> ܘܚܐܪܘܬܐ ܠܗܘܢ ܡܫܬܘܕܝܢ ܟܕ ܗܢܘܢ ܥܒܕܐ ܐܝܬܝܗܘܢ
ܕܚܒܠܐ ܠܗܘ ܓܝܪ ܡܕܡ ܕܐܢܫ ܙܟܐ ܠܗ ܠܗܢܐ ܐܦ ܡܫܥܒܕ ܀

<span id="an_2:20">2:20</span> ܐܢ ܓܝܪ ܟܕ ܥܪܩܘ ܡܢ ܛܢܦܘܬܗ ܕܥܠܡܐ ܒܫܘܘܕܥܗ
ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܦܪܘܩܢ ܒܗܝܢ ܒܗܠܝܢ ܟܕ ܡܬܥܪܙܠܝܢ ܬܘܒ ܡܙܕܟܝܢ ܗܘܬ ܠܗ ܚܪܬܗܘܢ
ܕܒܝܫܐ ܡܢ ܩܕܡܝܬܐ ܀

<span id="an_2:21">2:21</span> ܦܩܚ ܗܘܐ ܠܗܘܢ ܓܝܪ ܕܠܐ ܢܫܬܘܕܥܘܢ ܠܐܘܪܚܐ
ܕܙܕܝܩܘܬܐ ܐܘ ܕܟܕ ܐܫܬܘܕܥܘ ܠܒܬܪܐ ܢܗܦܟܘܢ ܡܢ ܦܘܩܕܢܐ ܩܕܝܫܐ ܕܐܫܬܠܡ ܠܗܘܢ ܀

<span id="an_2:22">2:22</span> ܓܕܫ ܠܗܘܢ ܕܝܢ ܗܠܝܢ ܕܡܬܠܐ ܫܪܝܪܐ ܕܟܠܒܐ ܕܗܦܟ
ܥܠ ܬܝܘܒܗ ܘܚܙܝܪܬܐ ܕܤܚܬ ܒܥܘܪܓܠܐ ܕܤܝܢܐ ܀

<span id="an_3:1">3:1</span> ܗܕܐ ܡܢ ܟܕܘ ܚܒܝܒܝ ܐܓܪܬܐ ܕܬܪܬܝܢ ܟܬܒ ܐܢܐ ܠܟܘܢ
ܗܠܝܢ ܕܒܗܝܢ ܡܥܝܪ ܐܢܐ ܒܥܘܗܕܢܐ ܠܪܥܝܢܟܘܢ ܫܦܝܐ ܀

<span id="an_3:2">3:2</span> ܕܬܬܥܗܕܘܢ ܠܡܠܐ ܕܩܕܡ ܐܬܐܡܪ ܡܢ ܢܒܝܐ ܩܕܝܫܐ
ܘܠܦܘܩܕܢܗ ܕܡܪܢ ܘܦܪܘܩܢ ܕܒܝܕ ܫܠܝܚܐ ܀

<span id="an_3:3">3:3</span> ܟܕ ܗܕܐ ܠܘܩܕܡ ܝܕܥܝܢ ܐܢܬܘܢ ܕܢܐܬܘܢ ܒܚܪܬܐ
ܕܝܘܡܬܐ ܡܒܙܚܢܐ ܕܡܒܙܚܝܢ ܟܕ ܐܝܟ ܪܓܝܓܬܐ ܕܝܠܗܘܢ ܡܗܠܟܝܢ ܀

<span id="an_3:4">3:4</span> ܘܐܡܪܝܢ ܕܐܝܟܐ ܐܝܬܘܗܝ ܡܘܠܟܢܐ ܕܡܐܬܝܬܗ ܡܢ ܟܕ
ܓܝܪ ܐܒܗܬܢ ܫܟܒܘ ܟܠܡܕܡ ܗܟܢܐ ܡܟܬܪ ܡܢ ܫܘܪܝܗ ܕܒܪܝܬܐ ܀

<span id="an_3:5">3:5</span> ܛܥܝܐ ܠܗܘܢ ܓܝܪ ܗܕܐ ܟܕ ܨܒܝܢ ܕܫܡܝܐ ܐܝܬܝܗܘܢ ܗܘܘ
ܡܢ ܩܕܝܡ ܘܐܪܥܐ ܡܢ ܡܝܐ ܘܒܝܕ ܡܝܐ ܩܡܬ ܒܡܠܬܐ ܕܐܠܗܐ ܀

<span id="an_3:6">3:6</span> ܗܠܝܢ ܕܒܐܝܕܝܗܘܢ ܥܠܡܐ ܕܗܝܕܝܢ ܛܦ ܒܡܝܐ ܘܐܒܕ ܀

<span id="an_3:7">3:7</span> ܫܡܝܐ ܕܝܢ ܕܗܫܐ ܘܐܪܥܐ ܒܡܠܬܐ ܕܝܠܗ ܐܤܝܢܝܢ ܟܕ
ܠܢܘܪܐ ܡܬܢܛܪܝܢ ܠܝܘܡܐ ܕܕܝܢܐ ܘܕܐܒܕܢܐ ܕܒܢܝܢܫܐ ܪܫܝܥܐ ܀

<span id="an_3:8">3:8</span> ܗܕܐ ܕܝܢ ܚܕܐ ܠܐ ܬܛܥܝܟܘܢ ܚܒܝܒܝ ܕܚܕ ܝܘܡܐ ܠܡܪܝܐ
ܐܝܟ ܐܠܦ ܫܢܝܢ ܐܝܬܘܗܝ ܘܐܠܦ ܫܢܝܢ ܐܝܟ ܝܘܡܐ ܚܕ ܀

<span id="an_3:9">3:9</span> ܠܐ ܡܫܬܘܚܪ ܡܪܝܐ ܒܡܘܠܟܢܘܗܝ ܐܝܟ ܕܐܢܫܝܢ ܫܘܘܚܪܐ
ܡܤܒܪܝܢ ܐܠܐ ܡܓܪ ܪܘܚܗ ܡܛܠܬܟܘܢ ܒܕܠܐ ܨܒܐ ܕܐܢܫ ܢܐܒܕ ܐܠܐ ܟܠܢܫ ܠܬܝܒܘܬܐ ܢܐܬܐ ܀

<span id="an_3:10">3:10</span> ܐܬܐ ܕܝܢ ܝܘܡܗ ܕܡܪܝܐ ܐܝܟ ܓܢܒܐ ܗܘ ܕܒܗ ܫܡܝܐ
ܡܢ ܫܠܝ ܥܒܪܝܢ ܐܤܛܘܟܤܐ ܕܝܢ ܟܕ ܝܩܕܝܢ ܢܫܬܪܘܢ ܘܐܪܥܐ ܘܥܒܕܐ ܕܒܗ ܬܫܬܟܚ ܀

<span id="an_3:11">3:11</span> ܟܕ ܗܟܝܠ ܗܠܝܢ ܟܠܗܘܢ ܡܫܬܪܝܢ ܕܐܝܟ ܐܝܠܝܢ ܙܕܩ
ܠܟܘܢ ܕܬܗܘܘܢ ܒܗܘܦܟܝܟܘܢ ܩܕܝܫܐ ܘܒܕܚܠܬ ܐܠܗܐ ܀

<span id="an_3:12">3:12</span> ܟܕ ܡܤܟܝܢ ܐܢܬܘܢ ܘܤܘܚܝܢ ܐܢܬܘܢ ܠܡܐܬܝܬܐ ܕܝܘܡܗ
ܕܐܠܗܐ ܗܘ ܕܒܗ ܫܡܝܐ ܟܕ ܡܬܒܚܪܝܢ ܒܢܘܪܐ ܢܫܬܪܘܢ ܘܐܤܛܘܟܤܐ ܟܕ ܝܩܕܝܢ ܢܫܘܚܘܢ ܀

<span id="an_3:13">3:13</span> ܠܫܡܝܐ ܕܝܢ ܚܕܬܐ ܘܐܪܥܐ ܚܕܬܐ ܐܝܟ ܡܘܠܟܢܐ ܕܝܠܗ
ܡܤܟܝܢܢ ܗܠܝܢ ܕܒܗܘܢ ܙܕܝܩܘܬܐ ܥܡܪܐ ܀

<span id="an_3:14">3:14</span> ܡܛܠ ܗܢܐ ܚܒܝܒܝ ܟܕ ܠܗܠܝܢ ܡܤܟܝܢ ܐܢܬܘܢ ܝܨܦܘ
ܕܕܠܐ ܟܘܬܡܐ ܘܕܠܐ ܡܘܡܐ ܠܗ ܬܫܬܟܚܘܢ ܒܫܠܡܐ ܀

<span id="an_3:15">3:15</span> ܘܠܡܓܪܬ ܪܘܚܗ ܕܡܪܝܐ ܦܘܪܩܢܐ ܬܚܫܒܘܢ ܐܝܟܢܐ ܕܐܦ
ܐܚܘܢ ܚܒܝܒܐ ܦܘܠܘܤ ܐܝܟ ܚܟܡܬܐ ܕܐܬܝܗܒܬ ܠܗ ܟܬܒ ܠܟܘܢ ܀

<span id="an_3:16">3:16</span> ܐܝܟܢܐ ܕܒܟܠܗܝܢ ܐܓܪܬܗ ܡܠܠ ܒܗܝܢ ܥܠ ܗܠܝܢ
ܐܝܠܝܢ ܕܐܝܬ ܒܗܝܢ ܡܕܡ ܕܥܤܝܩ ܠܤܘܟܠܐ ܗܠܝܢ ܕܐܝܠܝܢ ܕܕܠܐ ܝܘܠܦܢܐ ܐܢܘܢ ܘܠܐ ܤܡܝܟܝܢ
ܡܥܩܡܝܢ ܠܗܝܢ ܐܝܟ ܕܐܦ ܠܗܠܝܢ ܟܬܒܐ ܕܫܪܟܐ ܠܘܬ ܐܒܕܢܐ ܕܝܠܗܘܢ ܀

<span id="an_3:17">3:17</span> ܐܢܬܘܢ ܗܟܝܠ ܚܒܝܒܝ ܟܕ ܩܕܡܝܢ ܐܢܬܘܢ ܝܕܥܝܢ ܛܪܘ
ܢܦܫܟܘܢ ܕܕܠܡܐ ܟܕ ܐܙܠܝܢ ܐܢܬܘܢ ܒܬܪ ܛܥܝܘܬܐ ܕܐܝܠܝܢ ܕܕܠܐ ܢܡܘܤ ܐܢܘܢ ܬܦܠܘܢ ܡܢ
ܤܡܟܐ ܕܝܠܟܘܢ ܀

<span id="an_3:18">3:18</span> ܗܘܝܬܘܢ ܕܝܢ ܡܬܪܒܝܢ ܒܛܝܒܘܬܐ ܘܒܝܕܥܬܐ ܕܡܪܢ
ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܘܕܐܠܗܐ ܐܒܐ ܕܠܗ ܬܫܒܘܚܬܐ ܐܦ ܗܫܐ ܘܒܟܠܙܒܢ ܘܠܝܘܡܝ ܥܠܡܐ ܐܡܝܢ
܀
