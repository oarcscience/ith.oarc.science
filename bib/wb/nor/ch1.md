### Norwegian Bible: 1 Chronicles

<span id="an_1:1">1:1</span> Adam, Set, Enos,

<span id="an_1:2">1:2</span> Kenan, Mahalalel, Jared,

<span id="an_1:3">1:3</span> Enok, Metusalah, Lamek,

<span id="an_1:4">1:4</span> Noah, Sem, Kam og Jafet.

<span id="an_1:5">1:5</span> Jafets sønner var Gomer og Magog og Madai
og Javan og Tubal og Mesek og Tiras.

<span id="an_1:6">1:6</span> Og Gomers sønner var Askenas og Difat og
Togarma.

<span id="an_1:7">1:7</span> Og Javans sønner var Elisa og Tarsisa,
Kittim og Rodanim.

<span id="an_1:8">1:8</span> Kams sønner var Kus og Misra'im, Put og
Kana'an.

<span id="an_1:9">1:9</span> Og Kus' sønner var Seba og Havila og Sabta
og Raema og Sabteka; og Raemas sønner var Sjeba og Dedan.

<span id="an_1:10">1:10</span> Og Kus fikk en sønn som hette Nimrod; han
var den første som fikk velde på jorden.

<span id="an_1:11">1:11</span> Og Misra'im var stamfar til luderne og
anamerne og lehaberne og naftuherne

<span id="an_1:12">1:12</span> og patruserne og kasluherne, som
filistrene stammer fra, og kaftorerne.

<span id="an_1:13">1:13</span> Og Kana'an blev far til Sidon - det var
hans førstefødte - og Het

<span id="an_1:14">1:14</span> og jebusittene og amorittene og
girgasittene

<span id="an_1:15">1:15</span> og hevittene og arkittene og sinittene

<span id="an_1:16">1:16</span> og arvadittene og semarittene og
hamatittene.

<span id="an_1:17">1:17</span> Sems sønner var Elam og Assur og Arpaksad
og Lud og Aram og Us og Hul og Geter og Mesek.

<span id="an_1:18">1:18</span> Og Arpaksad fikk sønnen Salah, og Salah
fikk Eber.

<span id="an_1:19">1:19</span> Og Eber fikk to sønner; den ene hette
Peleg, for i hans dager blev menneskene spredt over jorden, og hans bror
hette Joktan.

<span id="an_1:20">1:20</span> Og Joktan blev far til Almodad og Salef
og Hasarmavet og Jarah

<span id="an_1:21">1:21</span> og Hadoram og Usal og Dikla

<span id="an_1:22">1:22</span> og Ebal og Abimael og Sjeba

<span id="an_1:23">1:23</span> og Ofir og Havila og Jobab; alle disse
var Joktans sønner.

<span id="an_1:24">1:24</span> Sem, Arpaksad, Salah,

<span id="an_1:25">1:25</span> Eber, Peleg, Re'u,

<span id="an_1:26">1:26</span> Serug, Nakor, Tarah,

<span id="an_1:27">1:27</span> Abram, det er Abraham.

<span id="an_1:28">1:28</span> Abrahams sønner var Isak og Ismael.

<span id="an_1:29">1:29</span> Dette er deres ætter: Nebajot, Ismaels
førstefødte, og Kedar og Adbe'el og Mibsam.

<span id="an_1:30">1:30</span> Misma og Duma, Massa, Hadad og Tema,

<span id="an_1:31">1:31</span> Jetur, Nafis og Kedma; dette var Ismaels
venner.

<span id="an_1:32">1:32</span> Og de sønner som Abrahams medhustru
Ketura fødte, var: Simran og Joksan og Medan og Midian og Jisbak og
Suah; og Joksans sønner var Sjeba og Dedan;

<span id="an_1:33">1:33</span> og Midians sønner var Efa og Efer og
Hanok og Abida og Elda'a. Alle disse var Keturas sønner.

<span id="an_1:34">1:34</span> Og Abraham fikk sønnen Isak. Isaks sønner
var Esau og Israel.

<span id="an_1:35">1:35</span> Esaus sønner var Elifas, Re'uel og Je'us
og Jaelam og Korah.

<span id="an_1:36">1:36</span> Elifas' sønner var Teman og Omar, Sefi og
Gatam, Kenas og Timna og Amalek.

<span id="an_1:37">1:37</span> Re'uels sønner var Nahat, Serah, Samma og
Missa.

<span id="an_1:38">1:38</span> Og Se'irs sønner var Lotan og Sobal og
Sibeon og Ana og Dison og Eser og Disan.

<span id="an_1:39">1:39</span> Og Lotans sønner var Hori og Homam; og
Lotans søster var Timna.

<span id="an_1:40">1:40</span> Sobals sønner var Aljan og Manahat og
Ebal, Sefi og Onam; og Sibeons sønner var Aja og Ana.

<span id="an_1:41">1:41</span> Anas sønn var Dison; og Disons sønner var
Hamran og Esban og Jitran og Keran.

<span id="an_1:42">1:42</span> Esers sønner var Bilhan og Sa'avan og
Ja'akan; Disans sønner var Us og Aran.

<span id="an_1:43">1:43</span> Og dette var de konger som regjerte i
Edom-landet før det regjerte nogen konge over Israels barn: Bela, Beors
sønn; hans by hette Dinhaba.

<span id="an_1:44">1:44</span> Da Bela døde, blev Jobab, Serahs sønn,
fra Bosra, konge i hans sted.

<span id="an_1:45">1:45</span> Da Jobab døde, blev Husam fra
temanittenes land konge i hans sted.

<span id="an_1:46">1:46</span> Da Husam døde, blev Hadad, Bedads sønn,
konge i hans sted; det var han som slo midianittene på Moabs mark; hans
by hette Ajut.

<span id="an_1:47">1:47</span> Da Hadad døde, blev Samla fra Masreka
konge i hans sted.

<span id="an_1:48">1:48</span> Da Samla døde, blev Saul fra Rehobot ved
elven konge i hans sted.

<span id="an_1:49">1:49</span> Da Saul døde, blev Ba'al-Hanan, Akbors
sønn, konge i hans sted.

<span id="an_1:50">1:50</span> Da Ba'al-Hanan døde, blev Hadad konge i
hans sted; hans by hette Pa'i, og hans hustru hette Mehetabel, datter av
Matred, Mesahabs datter.

<span id="an_1:51">1:51</span> Og Hadad døde. Og Edoms stammefyrster
var: stammefyrsten Timna, stammefyrsten Alja, stammefyrsten Jetet,

<span id="an_1:52">1:52</span> stammefyrsten Oholibama, stammefyrsten
Ela, stammefyrsten Pinon,

<span id="an_1:53">1:53</span> stammefyrsten Kenas, stammefyrsten Teman,
stammefyrsten Mibsar,

<span id="an_1:54">1:54</span> stammefyrsten Magdiel, stammefyrsten
Iram; dette var Edoms stammefyrster.

<span id="an_2:1">2:1</span> Dette var Israels sønner: Ruben, Simeon,
Levi og Juda, Issakar og Sebulon,

<span id="an_2:2">2:2</span> Dan, Josef og Benjamin, Naftali, Gad og
Aser.

<span id="an_2:3">2:3</span> Judas sønner var Er og Onan og Sela; disse
tre fikk han med Suas datter, kana'anittinnen; men Er, Judas
førstefødte, mishaget Herren, og han lot ham dø.

<span id="an_2:4">2:4</span> Og hans sønnekone Tamar fødte ham Peres og
Serah, så at Judas sønner i alt var fem.

<span id="an_2:5">2:5</span> Peres' sønner var Hesron og Hamul.

<span id="an_2:6">2:6</span> Og Serahs sønner var Simri\* og Etan og
Heman og Kalkol og Dara, tilsammen fem. /

<span id="an_2:7">2:7</span> Og Karmis\* sønn var Akar\*\*, som førte
Israel i ulykke, fordi han bar sig troløst at med det bannlyste gods. /
/

<span id="an_2:8">2:8</span> Og Etans sønn var Asarja.

<span id="an_2:9">2:9</span> Og de sønner som Hesron fikk, var
Jerahme'el og Ram og Kelubai\*. /

<span id="an_2:10">2:10</span> Og Ram fikk sønnen Amminadab, og
Amminadab fikk sønnen Nahson, høvding for Judas barn,

<span id="an_2:11">2:11</span> og Nahson fikk sønnen Salma, og Salma
fikk sønnen Boas,

<span id="an_2:12">2:12</span> og Boas fikk sønnen Obed, og Obed fikk
sønnen Isai.

<span id="an_2:13">2:13</span> Og Isai fikk Eliab, som var hans
førstefødte sønn, og Abinadab, hans annen sønn, og Simea, den tredje,

<span id="an_2:14">2:14</span> Netanel, den fjerde, Raddai, den femte,

<span id="an_2:15">2:15</span> Osem, den sjette, David, den syvende.

<span id="an_2:16">2:16</span> Og deres søstre var Seruja og Abiga'il.
Og Serujas sønner var Absai og Joab og Asael, tre i tallet.

<span id="an_2:17">2:17</span> Og Abiga'il fødte Amasa, og Amasas far
var ismaelitten Jeter.

<span id="an_2:18">2:18</span> Og Kaleb, Hesrons sønn, fikk barn med sin
hustru Asuba og med Jeriot; og dette var hennes\* sønner: Jeser og Sobab
og Ardon. /

<span id="an_2:19">2:19</span> Da Asuba døde, tok Kaleb Efrat til
hustru, og med henne fikk han sønnen Hur,

<span id="an_2:20">2:20</span> og Hur fikk sønnen Uri, og Uri fikk
sønnen Besalel.

<span id="an_2:21">2:21</span> Siden giftet Hesron sig med en datter av
Gileads far Makir; han tok henne til hustru da han var seksti år gammel,
og fikk med henne sønnen Segub.

<span id="an_2:22">2:22</span> Og Segub fikk sønnen Ja'ir; han hadde tre
og tyve byer i Gileads land.

<span id="an_2:23">2:23</span> Men gesurittene og arameerne tok Ja'irs
byer fra dem og likeså Kenat med tilhørende landsbyer - i alt seksti
byer. Alle disse\* var sønner av\*\* Gileads far Makir. / /

<span id="an_2:24">2:24</span> Og efterat Hesron var død i Kaleb-Efrata,
fødte hans hustru Abia ham Ashur, Tekoas\* far. /

<span id="an_2:25">2:25</span> Og sønnene til Jerahme'el, Hesrons
førstefødte, var Ram, hans førstefødte, og Buna og Oren og Osem; dem
hadde han med Akia.

<span id="an_2:26">2:26</span> Jerahme'el hadde også en annen hustru,
som hette Atara; hun var mor til Onam.

<span id="an_2:27">2:27</span> Og sønnene til Ram, Jerahme'els
førstefødte, var Ma'as og Jamin og Eker.

<span id="an_2:28">2:28</span> Og Onams sønner var Sammai og Jada. Og
Sammais sønner var Nadab og Abisur.

<span id="an_2:29">2:29</span> Abisurs hustru hette Abiha'il, og med
henne fikk han Akban og Molid.

<span id="an_2:30">2:30</span> Og Nadabs sønner var Seled og Appa'im;
Seled døde uten sønner.

<span id="an_2:31">2:31</span> Og Appa'ims sønn var Jisi; og Jisis sønn
var Sesan; og Sesans barn\* var Aklai. /

<span id="an_2:32">2:32</span> Og sønnene til Sammais bror Jada var
Jeter og Jonatan. Jeter døde uten sønner.

<span id="an_2:33">2:33</span> Og Jonatans sønner var Pelet og Sasa.
Dette var Jerahme'els sønner.

<span id="an_2:34">2:34</span> Sesan hadde ingen sønner, bare døtre. Men
Sesan hadde en egyptisk tjener, som hette Jarha,

<span id="an_2:35">2:35</span> og han lot sin tjener Jarha få sin datter
til hustru, og med henne fikk han sønnen Attai.

<span id="an_2:36">2:36</span> Og Attai fikk sønnen Natan, og Natan fikk
sønnen Sabad,

<span id="an_2:37">2:37</span> og Sabad fikk sønnen Eflal, og Eflal fikk
sønnen Obed,

<span id="an_2:38">2:38</span> og Obed fikk sønnen Jehu, og Jehu fikk
sønnen Asarja,

<span id="an_2:39">2:39</span> og Asarja fikk sønnen Hales, og Hales
fikk sønnen Elasa,

<span id="an_2:40">2:40</span> og Elasa fikk sønnen Sismai, og Sismai
fikk sønnen Sallum,

<span id="an_2:41">2:41</span> og Sallum fikk sønnen Jekamja, og Jekamja
fikk sønnen Elisama.

<span id="an_2:42">2:42</span> Og sønnene til Jerahme'els bror Kaleb var
Mesa hans førstefødte - han var far til Sif - og sønnene til Hebrons far
Maresa.

<span id="an_2:43">2:43</span> Og Hebrons sønner var Korah og Tappuah og
Rekem og Sema.

<span id="an_2:44">2:44</span> Og Sema fikk sønnen Raham, far til
Jorkeam, og Rekem fikk sønnen Sammai.

<span id="an_2:45">2:45</span> Og Sammais sønn var Maon, og Maon var far
til Betsur.

<span id="an_2:46">2:46</span> Og Kalebs medhustru Efa fødte Haran og
Mosa og Gases; og Haran fikk sønnen Gases.

<span id="an_2:47">2:47</span> Og Jehdais sønner var Regem og Jotam og
Gesan og Pelet og Efa og Sa'af.

<span id="an_2:48">2:48</span> Kalebs medhustru Ma'aka fødte Seber og
Tirhana;

<span id="an_2:49">2:49</span> hun fødte også Sa'af, far til Madmanna,
og Seva, far til Makbena og Gibea; og Kalebs datter var Aksa.

<span id="an_2:50">2:50</span> Dette var Kalebs sønner: Sobal, sønn til
Efratas førstefødte sønn Hur og far til Kirjat-Jearim,

<span id="an_2:51">2:51</span> Salma, far til Betlehem, Haref, far til
Betgader.

<span id="an_2:52">2:52</span> Og sønnene til Kirjat-Jearims far Sobal
var Haroe\* og halvdelen av Hammenuhot-ætten. /

<span id="an_2:53">2:53</span> Og Kirjat-Jearims ætter var jitrittene og
putittene og sumatittene og misra'ittene; fra dem stammer soratittene og
estaolittene.

<span id="an_2:54">2:54</span> Salmas sønner var Betlehem og
netofatittene, Atrot-Bet-Joab og halvdelen av manahtittene, sorittene.

<span id="an_2:55">2:55</span> Og de skriftlærdes ætter, deres som bodde
i Jabes, var tiratittene, simatittene, sukatittene; dette er de
kinnitter som nedstammer fra Hammat, far til Rekabs hus.

<span id="an_3:1">3:1</span> Dette var Davids sønner som han fikk i
Hebron: Den førstefødte, Amnon, fikk han med Akinoam fra Jisre'el; den
annen, Daniel\*, med Abiga'il fra Karmel; /

<span id="an_3:2">3:2</span> den tredje, Absalom, var sønn av Ma'aka, en
datter av kongen i Gesur Talmai; den fjerde, Adonja, var sønn av Haggit;

<span id="an_3:3">3:3</span> den femte, Sefatja, fikk han med Abital;
den sjette, Jitream, med sin hustru Egla.

<span id="an_3:4">3:4</span> Disse seks fikk han i Hebron; der regjerte
han i syv år og seks måneder, og i tre og tretti år regjerte han i
Jerusalem.

<span id="an_3:5">3:5</span> Dette er de sønner han fikk i Jerusalem:
Simea og Sobab og Natan og Salomo - disse fire fikk han med Batsua\*,
Ammiels datter - /

<span id="an_3:6">3:6</span> og Jibhar og Elisama og Elifelet

<span id="an_3:7">3:7</span> og Nogah og Nefeg og Jafia

<span id="an_3:8">3:8</span> og Elisama og Eljada og Elifelet, ni i
tallet.

<span id="an_3:9">3:9</span> Alle disse var Davids sønner foruten de
sønner han fikk med sine medhustruer; og Tamar var deres søster.

<span id="an_3:10">3:10</span> Og Salomos sønn var Rehabeam; hans sønn
Abia; hans sønn Asa; hans sønn Josafat;

<span id="an_3:11">3:11</span> hans sønn Joram; hans sønn Akasja; hans
sønn Joas,

<span id="an_3:12">3:12</span> hans sønn Amasja; hans sønn Asarja; hans
sønn Jotam;

<span id="an_3:13">3:13</span> hans sønn Akas; hans sønn Esekias; hans
sønn Manasse;

<span id="an_3:14">3:14</span> hans sønn Amon; hans sønn Josias.

<span id="an_3:15">3:15</span> Og Josias' sønner var Johanan, som var
hans førstefødte sønn, Jojakim, hans annen sønn, Sedekias, den tredje,
Sallum, den fjerde.

<span id="an_3:16">3:16</span> Og Jojakims sønner var hans sønn Jekonja

<span id="an_3:17">3:17</span> Og den fangne Jekonjas sønner var hans
sønn Sealtiel

<span id="an_3:18">3:18</span> og Malkiram og Pedaja og Senassar,
Jekamja, Hosama og Nedabja.

<span id="an_3:19">3:19</span> Og Pedajas sønner var Serubabel og
Sime'i; og Serubabels sønner var Mesullam og Hananja - deres søster var
Selomit -

<span id="an_3:20">3:20</span> og Hasuba og Ohel og Berekja, Hasadja og
Jusab-Hesed, fem i tallet.

<span id="an_3:21">3:21</span> Og Hananjas sønner var Pelatja og Jesaja,

<span id="an_3:22">3:22</span> Og Sekanjas sønn var Semaja; og Semajas
sønner var Hattus og Jigal og Bariah og Nearja og Safat, seks i tallet.

<span id="an_3:23">3:23</span> Og Nearjas sønner var Eljoenai og Hiskia
og Asrikam, tre i tallet.

<span id="an_3:24">3:24</span> Og Eljoenais sønner var Hodaiva og
Eljasib og Pelaja og Akkub og Johanan og Delaja og Anani, syv i tallet.

<span id="an_4:1">4:1</span> Judas sønner var Peres, Hesron og Karmi og
Hur og Sobal.

<span id="an_4:2">4:2</span> Og Reaja, Sobals sønn, fikk sønnen Jahat,
og Jahat fikk sønnene Ahumai og Lahad; dette var soratittenes ætter.

<span id="an_4:3">4:3</span> Og dette var Abi-Etams \[ætter\]: Jisre'el
og Jisma og Jidbas, og deres søster hette Haslelponi.

<span id="an_4:4">4:4</span> Og Pnuel var far til Gedor, og Eser var far
til Husa; de var sønner av Hur, Efratas førstefødte sønn og far til
Betlehem.

<span id="an_4:5">4:5</span> Og Tekoas far Ashur hadde to hustruer, Hela
og Na'ara.

<span id="an_4:6">4:6</span> Med Na'ara fikk han Ahussam og Hefer og
Temeni og Ahastari; dette var Na'aras sønner.

<span id="an_4:7">4:7</span> Og Helas sønner var Seret, Jishar og Etnan.

<span id="an_4:8">4:8</span> Og Kos blev far til Anub og Hassobeba og
Aharhels, Harums sønns ætter.

<span id="an_4:9">4:9</span> Jabes var æret fremfor sine brødre; hans
mor gav ham navnet Jabes\*, idet hun sa: Jeg har født ham med smerte. /

<span id="an_4:10">4:10</span> Og Jabes påkalte Israels Gud og sa: Å, om
du vilde velsigne mig og utvide mine landemerker og la din hånd være med
mig, og om du vilde lage det så at det ingen ulykke kom, og jeg blev fri
for smerte! Og Gud lot det komme som han hadde bedt om.

<span id="an_4:11">4:11</span> Og Suhas bror Kelub fikk sønnen Mehir;
han var far til Eston.

<span id="an_4:12">4:12</span> Og Eston var far til Bet-Rafa og Paseah
og Tehinna, Ir-Nahas' far dette var mennene fra Reka.

<span id="an_4:13">4:13</span> Og Kenas' sønner var Otniel og Seraja, og
Otniels sønn var Hatat.

<span id="an_4:14">4:14</span> Og Meonotai fikk sønnen Ofra, og Seraja
fikk sønnen Joab, som var far til ætten i Tømmermannsdalen; for de var
tømmermenn.

<span id="an_4:15">4:15</span> Og sønnene til Kaleb, Jefunnes sønn, var
Iru, Ela og Na'am, og Elas sønn var Kenas.

<span id="an_4:16">4:16</span> Og Jehallelels sønner var Sif og Sifa,
Tirja og Asarel.

<span id="an_4:17">4:17</span> Og Esras sønner var Jeter og Mered og
Efer og Jalon; og hun\* blev fruktsommelig og fødte Mirjam og Sammai og
Jisbah, far til Estemoa. /

<span id="an_4:18">4:18</span> Og hans jødiske hustru fødte Jered, far
til Gedor, og Heber, far til Soko, og Jekutiel, far til Sanoah; men de
andre\* var sønner av Faraos datter Bitja, som Mered hadde tatt til
hustru. /

<span id="an_4:19">4:19</span> Og sønnene av Hodijas hustru, Nahams
søster, var Ke'ilas far, garmitten, og Estemoa, ma'akatitten.

<span id="an_29:21">29:21</span> Så bar de slaktoffere frem for Herren;
og dagen efter ofret de brennoffere til Herren: tusen okser, tusen
værer, tusen lam, med tilhørende drikkoffere, og slaktoffere i mengde
for hele Israel.

<span id="an_29:22">29:22</span> Og den dag åt og drakk de for Herrens
åsyn med stor glede, og de gjorde annen gang\* Davids sønn Salomo til
konge og salvet ham til en Herrens fyrste og Sadok til prest. /

<span id="an_29:23">29:23</span> Og Salomo satt på Herrens trone som
konge i sin far Davids sted; han hadde lykken med sig, og hele Israel
var ham lydig,

<span id="an_29:24">29:24</span> og alle høvdingene og heltene og likeså
alle kong Davids sønner hyldet kong Salomo.

<span id="an_29:25">29:25</span> Og Herren lot Salomo bli overmåte stor
for hele Israels øine og gav ham en kongelig herlighet som ingen konge i
Israel før ham hadde hatt.

<span id="an_29:26">29:26</span> David, Isais sønn, hadde vært konge
over hele Israel.

<span id="an_29:27">29:27</span> Den tid han var konge over Israel, var
firti år; i Hebron regjerte han i syv år, og i Jerusalem regjerte han i
tre og tretti år.

<span id="an_29:28">29:28</span> Og han døde i høi alder, mett av dager,
rikdom og ære, og hans sønn Salomo blev konge i hans sted.

<span id="an_29:29">29:29</span> Hvad som er å fortelle om kong David,
både i hans første og i hans senere dager, det er opskrevet i seeren
Samuels krønike og i profeten Natans krønike og i seeren Gads krønike;

<span id="an_29:30">29:30</span> der er og fortalt om hele hans
regjering og hans store gjerninger og om de tider som kom over ham og
Israel og over alle andre land og riker.
