### Kabyle Bible: John

<span id="an_1:1">1:1</span> Di tazwara yella win yellan d Awal n Ṛebbi,
yella akk-d Ṛebbi, d nețța i d Ṛebbi.

<span id="an_1:2">1:2</span> Si tazwara, yella akk-d Ṛebbi,

<span id="an_1:3">1:3</span> kullec yețwaxleq seg-s ; deg wayen meṛṛa
yețwaxelqen, ulac ayen ur d-nekki ara s ɣuṛ-es.

<span id="an_1:4">1:4</span> Deg-s i tella tudert ; tudert-agi tella
ț-țafat i yemdanen.

<span id="an_1:5">1:5</span> Tafat tețfeǧǧiǧ di ṭṭlam, yerna ṭṭlam ur
ț-iqbil ara.

<span id="an_1:6">1:6</span> Yella yiwen wergaz yețwaceggeɛ ed s ɣuṛ
Ṛebbi, argaz-agi isem-is Yeḥya ;

<span id="an_1:7">1:7</span> yusa-d ad yili d inigi n tafat iwakken
imdanen meṛṛa ad amnen s cchada-ines.

<span id="an_1:8">1:8</span> Mačči d nețța i ț-țafat, nețța yusa-d d
inigi n tafat.

<span id="an_1:9">1:9</span> Tafat-agi ț-țafat n tideț, tusa-d ɣer
ddunit iwakken aț-țecṛeq ɣef yal amdan.

<span id="an_1:10">1:10</span> Win yellan d Awal n Ṛebbi yella yakan si
tazwara di ddunit, imi d nețța i ț-ixelqen lameɛna ddunit ur t- teɛqil
ara.

<span id="an_1:11">1:11</span> Yusa-d ɣer lumma-s lameɛna nețțat ur
t-teqbil ara.

<span id="an_1:12">1:12</span> Ma d wid i t-iqeblen yumnen yis, widak
iɛuzz-iten, uɣalen d arraw n Ṛebbi,

<span id="an_1:13">1:13</span> mačči seg wemdan i d-lulen, mačči s lebɣi
n tnefsit neɣ s lebɣi n wemdan, lameɛna d Ṛebbi i sen-yefkan tudert
tajdiṭ.

<span id="an_1:14">1:14</span> Win akken yellan d Awal, yuɣal-ed d
amdan, iɛac gar-aneɣ, yeččuṛ ț-țideț akk-d ṛṛeḥma. Nețweḥḥid di lqeḍra i
d-yefka Baba Ṛebbi i Mmi-s awḥid.

<span id="an_1:15">1:15</span> Yeḥya iched-ed fell-as zdat lɣaci yeqqaṛ
: D nețța ɣef i wen-d-nniɣ : win ara d-yasen deffir-iw yezwar-iyi-d
axaṭer yella uqbel ad iliɣ.

<span id="an_1:16">1:16</span> Yesseṛwa-yaɣ s lxiṛat-is, lbaṛaka ɣef
tayeḍ.

<span id="an_1:17">1:17</span> ccariɛa tusa-d s ufus n Sidna Musa, ma d
ṛṛeḥma ț-țideț usant-ed s Sidna Ɛisa Lmasiḥ

<span id="an_1:18">1:18</span> Yiwen ur yeẓri Sidi Ṛebbi, Mmi-s awḥid
yellan ɣuṛ-es, d nețța i ɣ-t-id isbeggnen.

<span id="an_1:19">1:19</span> Atah wamek i geched Yeḥya asmi i
d-ceggɛen ɣuṛes lḥekkam n wudayen si temdint n Lquds. Ceggɛen-d ɣuṛes
tajmaɛt n lmu-qedmin akk-d yemṛabḍen n at Lewwi akken a t-steqsin,
nnan-as : Anwa-k keččini ?

<span id="an_1:20">1:20</span> Yenna-yasen tideț mbla tuffra : Mačči d
nekk i d Lmasiḥ !

<span id="an_1:21">1:21</span> Rnan steqsan-t : Anwa-k ihi ? D kečč i d
nnbi Ilyas ? Yeḥya yenna-yasen : Xaṭi mačči d nekk. Nutni rnan nnan-as :
D nnbi i telliḍ ? Nețța yerra-yasen : Xaṭi !

<span id="an_1:22">1:22</span> Rnan ḥeṛsen-t nnan-as : Anwa-k ihi ? Ilaq
a nerr lexbaṛ i wid i ɣ-d-iceggɛen. D acu i teqqaṛeḍ ɣef yiman-ik ?

<span id="an_1:23">1:23</span> Yeḥya yenna-yasen : Nekk d win akken ɣef
i d-yenna nnbi Iceɛya : ț-țaɣect tețɛeggiḍ deg unezṛuf : Heggit abrid i
Sidi Ṛebbi !

<span id="an_1:24">1:24</span> Wid i d-yusan ɣuṛ-es i gellan di tejmaɛt
n ifariziyen,

<span id="an_1:25">1:25</span> rnan steqsan-t nnan-as : Imi ur telliḍ
ara d Lmasiḥ neɣ d nnbi Ilyas neɣ d nnbi nniḍen, iwacu ihi i tesseɣḍaseḍ
lɣaci deg waman ?

<span id="an_1:26">1:26</span> Yeḥya yerra-yasen : Nekk sseɣḍaseɣ-kkun
deg waman ; yella gar-awen yiwen ur t-tessinem ara,

<span id="an_1:27">1:27</span> usiɣ-ed uqbel-is, lameɛna ur uklaleɣ ara
ad fsiɣ ulamma d lexyuḍ n warkasen-is.

<span id="an_1:28">1:28</span> Annect-agi meṛṛa yedṛa di taddart n
Bitanya ɣer tama n cceṛq n wasif n Urdun anda i gesseɣḍas Yeḥya lɣaci.

<span id="an_1:29">1:29</span> Azekka-nni, Yeḥya iwala Sidna Ɛisa
iteddu-d ɣuṛ-es, yenṭeq s ṣṣut ɛlayen : Atan Izimer n Ṛebbi, d win
ițekksen ddnub n ddunit.

<span id="an_1:30">1:30</span> Fell-as i wen-d-nniɣ : « a d-yas yiwen,
zwareɣ-t-id meɛna yella uqbel a d iliɣ.»

<span id="an_1:31">1:31</span> Nekk s yiman-iw ur ẓriɣ ara anwa-t meɛna
usiɣ-ed ad sɣeḍseɣ deg waman akken a t-issinen wat Isṛail .

<span id="an_1:32">1:32</span> Atan wayen i d-yenna Yeḥya Aɣeṭṭas
fell-as : Walaɣ Ṛṛuḥ iqedsen iṣubb-ed seg igenni s ṣṣifa n tetbirt
yers-ed fell-as.

<span id="an_1:33">1:33</span> Nekk ur ẓriɣ ara belli d nețța, meɛna
Sidi Ṛebbi i yi-d iceggɛen iwakken ad sɣeḍseɣ deg waman, yenna-yi-d : «
Aț-țwaliḍ Ṛṛuḥ iṣubb-ed seg igenni, a d-yers ɣef yiwen wergaz, d nețța
ara yesɣeḍsen imdanen s Ṛṛuḥ iqedsen. »

<span id="an_1:34">1:34</span> Ayagi d ayen i ẓrant wallen-iw, daymi i
d-cehdeɣ fell-as qqaṛeɣ : « Argaz-agi d Mmi-s n Ṛebbi.»

<span id="an_1:35">1:35</span> Azekka-nni, Yeḥya mazal-it dinna nețța d
sin seg inelmaden-is.

<span id="an_1:36">1:36</span> Akken i gwala Sidna Ɛisa iɛedda-d,
yenna-yasen : Atan izimer n Ṛebbi !

<span id="an_1:37">1:37</span> Mi slan i imeslayen-agi, inelmaden-nni
tebɛen Sidna Ɛisa.

<span id="an_1:38">1:38</span> Sidna Ɛisa mi geẓra tebɛen-t-id, yezzi-d
ɣuṛ-sen, yenna-yasen : D acu tebɣam ? Nutni nnan-as : Ṛabi ( yeɛni : a
Sidi ), anda tzedɣeḍ ?

<span id="an_1:39">1:39</span> Yenna-yasen : Ddut yid-i aț-țeẓrem.
Inelmaden-nni ddan yid-es, ẓran anda yezdeɣ. Aț-țili d ṛebɛa n tmeddit,
qqimen yid-es armi yekfa wass.

<span id="an_1:40">1:40</span> Andriyus, gma-s n Semɛun Buṭrus d yiwen
seg inelmaden-nni i gtebɛen Sidna Ɛisa m'akken i sen-d-yehdeṛ fell-as
Yeḥya ;

<span id="an_1:41">1:41</span> iṛuḥ uqbel ɣer gma-s Semɛun, yenna-yas :
Nufa Lmasiḥ ! Win i gextaṛ Ṛebbi.

<span id="an_1:42">1:42</span> Andriyus yewwi Semɛun ɣer Sidna Ɛisa.
Akken i t-iwala Sidna Ɛisa, imuqel-it yenna-yas : Kečč i d Semɛun mmi-s
n Yunes. Sya d asawen aț-tețțusemmiḍ Sifas yeɛni : Buṭrus.

<span id="an_1:43">1:43</span> Azekka-nni, Sidna Ɛisa yeqsed ad iṛuḥ ɣer
tmurt n Jlili. Deg ubrid yemmuger-ed Filibus, yenna-yas : Ddu-d yid-i !

<span id="an_1:44">1:44</span> Filibus, d yiwen si taddart n Bitsayda,
anda akken zedɣen Buṭrus d Andriyus.

<span id="an_1:45">1:45</span> Syenna Filibus yeẓra-d Natanahil,
yenna-yas : Nufa win ɣef i d-yehdeṛ Sidna Musa di ccariɛa, win akken i
ɣef d-xebbṛen lenbiya. D Ɛisa, mmi-s n Yusef n taddart n Naṣaret.

<span id="an_1:46">1:46</span> Natanahil yerra-yas : Amek, yezmer a
d-iffeɣ wayen yelhan si taddart n Naṣaret ? Filibus yerra-yas : Eyya-d
aț-țwaliḍ s wallen-ik !

<span id="an_1:47">1:47</span> Sidna Ɛisa mi gwala Natanahil iteddu-d
ɣuṛ-es, yenna : Atah yiwen n wat Isṛail n tideț, ulac deg-s leɣdeṛ.

<span id="an_1:48">1:48</span> Natanahil yenna i Sidna Ɛisa : Anda i
yi-tesneḍ ? Sidna Ɛisa yerra-yas : Uqbel a k-d-yessiwel Filibus, mi
telliḍ ddaw n tneqleț, walaɣ-k-in !

<span id="an_1:49">1:49</span> Dɣa Natanahil yenna-yas : A Sidi, kečč d
Mmi-s n Ṛebbi ! D agellid n wat Isṛail !

<span id="an_1:50">1:50</span> Sidna Ɛisa yerra-yas : Tumneḍ imi i
k-d-nniɣ walaɣ k-in seddaw n tneqleț ; sya ɣer zdat aț-țeẓred ayen
yugaren annect-a !

<span id="an_1:51">1:51</span> S tideț qqaṛeɣ-awen : aț-țwalim igenni
yeldi, lmalayekkat n Sidi Ṛebbi țțalint țṣubbunt sennig Mmi-s n bunadem.

<span id="an_2:1">2:1</span> Mi ɛeddan sin wussan, tella-d tmeɣṛa di
tmurt n Jlili di taddart Kana, tella dinna yemma-s n Sidna Ɛisa.

<span id="an_2:2">2:2</span> Ula d Sidna Ɛisa yețwaɛṛed nețța d
inelmaden-is ɣer tmeɣṛa-nni.

<span id="an_2:3">2:3</span> Imi i ten-ixuṣṣ ccṛab, yemma-s n Sidna Ɛisa
tusa-d ɣuṛ-es tenna-yas : Ifuk-asen ccṛab !

<span id="an_2:4">2:4</span> Sidna Ɛisa yerra-yas : D nekk neɣ d kemm i
teɛna temsalt-agi a tameṭṭut ? Taswiɛt-iw urɛad i d-tusi.

<span id="an_2:5">2:5</span> Dɣa yemma-s tenna i iqeddacen : Xedmet ayen
akk ara wen-d yini !

<span id="an_2:6">2:6</span> Llant dinna sețța n tsebbalin ( icmuxen )
yețwaxedmen s wedɣaɣ, țțawint si tmanyin ar meyya uɛecrin n litrat,
seg-sent i d-țɛemmiṛen wat Isṛail aman i luḍu.

<span id="an_2:7">2:7</span> Sidna Ɛisa yenna i iqeddacen nni : ?čaṛet
ticbayliyin-agi d aman. ?čuṛen-tent armi d imi.

<span id="an_2:8">2:8</span> Dɣa yenna-asen : Ɛemmṛet-ed tura kra n
waman seg-sent tawim-ten i bab n tmeɣṛa. Kkren xedmen akken i
sen-d-yenna.

<span id="an_2:9">2:9</span> Win iḍebbṛen di leqdic n tmeɣṛa yeɛṛeḍ
aman-nni i guɣalen d ccṛab. Nețța ur yeẓri ansi i d-yekka ccṛab-agi, ma
d iqeddacen-is ẓran axaṭer d nutni i ten-id-yewwin. Iṛuḥ yessawel i
yesli,

<span id="an_2:10">2:10</span> yenna-yas : Di tmeɣṛiwin, medden srusun-d
uqbel ccṛab yelhan, mi ḥman inebgawen rennun-asen-d win ṛqiqen, kečč
teǧǧiḍ ccṛab yelhan armi ț-țura !

<span id="an_2:11">2:11</span> D wagi i d lbeṛhan amezwaru i gexdem
Sidna Ɛisa di taddart n Kana di tmurt n Jlili. S wakka i d-ibeggen
tamanegt-is i inelmaden-is, dɣa umnen yis.

<span id="an_2:12">2:12</span> Deffir wayagi Sidna Ɛisa iṣubb ɣer
temdint n Kafernaḥum. Tedda yid-es yemma-s, atmaten-is akk-d
inelmaden-is ; qqimen dinna anagar kra n wussan.

<span id="an_2:13">2:13</span> Akken i d-yeqṛeb wass n Tfaska n izimer n
leslak yellan d lɛid n wat Isṛail, Sidna Ɛisa yuli ɣer temdint n Lquds.

<span id="an_2:14">2:14</span> Yufa deg wefrag n lǧameɛ iqedsen wid
yeznuzun izgaren, ulli d yetbiren, yufa daɣen dinna wid yețbeddilen
idrimen ; rran lǧameɛ iqedsen d ssuq.

<span id="an_2:15">2:15</span> Ixdem-ed ajelkwaḍ ( acelliṭ ) s wemrar,
yebda itellif-iten s nutni s wulli-nsen akk-d yezgaren-nsen, iqleb-asen
( ițți-yasen ) daɣen ṭṭwabel i wid yețbeddilen idrimen, iḍeggeṛ asen
kullec ɣer lqaɛa, issufeɣ-iten meṛṛa seg ufrag n lǧameɛ iqedsen ,

<span id="an_2:16">2:16</span> yerna yenna i wid yeznuzun itbiren :
Kkset syagi ssuq-nwen ! Wagi d axxam n Baba, ur t-țarrat ara d ssuq !

<span id="an_2:17">2:17</span> inelmaden-is mmektan-d ayen yuran di
tektabt n ?abur : Leḥmala i sɛiɣ ɣef wexxam-ik am tmes itețțen ul-iw !

<span id="an_2:18">2:18</span> Kkren wudayen nnan-as : D acu n lbeṛhan
ara ɣ-d-tbeggneḍ akken a neɛqel lḥekma s wayes i txeddmeḍ annect-agi ?

<span id="an_2:19">2:19</span> Sidna Ɛisa yerra-yasen : Huddet
lǧameɛ-agi iqedsen, a t-id-sbeddeɣ di tlata wussan.

<span id="an_2:20">2:20</span> At Isṛail nnan-as : Wid yebnan lǧameɛ-agi
qqimen deg-s sețța uṛebɛin iseggasen, kečč tzemreḍ a s-tɛiwdeḍ lebni di
tlata wussan !

<span id="an_2:21">2:21</span> Lameɛna « Lǧameɛ » i ɣef d-yehdeṛ Sidna
Ɛisa d lǧețța-s.

<span id="an_2:22">2:22</span> Asmi i d-yeḥya Sidna Ɛisa si lmut ɣer
tudert, inelmaden-is mmektan-d awal-agi i d-yenna, dɣa umnen s wayen
yuran di tira iqedsen d wayen i d-yenna Sidna Ɛisa.

<span id="an_2:23">2:23</span> Mi gella Sidna Ɛisa di temdint n Lquds di
lweqt n lɛid n Tfaska, aṭas i gumnen yis mi ẓran lbeṛhanat i gxeddem.

<span id="an_2:24">2:24</span> Lameɛna Sidna Ɛisa ur yesɛi ara deg-sen
laman axaṭer yessen-iten akken ma llan.

<span id="an_2:25">2:25</span> Ur yeḥwaǧ ara a s-d-ḥkun ɣef yemdanen imi
yessen ayen yellan deg wulawen-nsen.

<span id="an_3:1">3:1</span> Yella yiwen wergaz isem-is Nikudem i gellan
si tejmaɛt n ifariziyen, d yiwen seg imeqqranen n wat Isṛail.

<span id="an_3:2">3:2</span> Iṛuḥ-ed deg iḍ ɣer Sidna Ɛisa yenna-yas : A
Sidi neẓra belli d Sidi Ṛebbi i k-id-iceggɛen iwakken a ɣ-teslemdeḍ,
axaṭer yiwen ur izmir ad yexdem lbeṛhanat i txeddmeḍ m'ur yelli ara
Ṛebbi yid-es.

<span id="an_3:3">3:3</span> Sidna Ɛisa yerra-yas : S tideț a k-d-iniɣ :
yiwen ur izmir ad iẓer tagelda n Ṛebbi m'ur d-iɛawed ara talalit.

<span id="an_3:4">3:4</span> Nikudem yenna-yas : Amek i gezmer yiwen a d
iɛiwed talalit ma yella d amɣaṛ ? Izmer ad yuɣal ɣer tɛebbuṭ n yemma-s a
d iɛiwed talalit ?

<span id="an_3:5">3:5</span> Sidna Ɛisa yerra-yas : Nniɣ-ak s tideț :
m'ur d-ilul ara yiwen seg waman akk-d Ṛṛuḥ, ur ițțizmir ara ad ikcem
tagelda n Ṛebbi.

<span id="an_3:6">3:6</span> Win i d-ilulen seg wemdan ț-țaṛwiḥt n
wemdan, win i d-ilulen s Ṛṛuḥ iqedsen d Ṛṛuḥ.

<span id="an_3:7">3:7</span> Ur țțawham ara imi i k-d-nniɣ « ilaq a
d-tɛiwdem talalit.»

<span id="an_3:8">3:8</span> Aḍu yețsuḍu ɣer wanda i s-yehwa, tselleḍ i
ṣṣut-is lameɛna ur tẓerreḍ ara ansi d-yekka neɣ anda iteddu. Akka daɣen
i gella ɣef kra n win i d-ilulen s Ṛṛuḥ iqedsen.

<span id="an_3:9">3:9</span> Nikudem yenna-yas : Amek ara yedṛu wayagi ?

<span id="an_3:10">3:10</span> Sidna Ɛisa yerra-yas : Tesselmadeḍ
ccariɛa i wat Isṛail ur tessineḍ ara annect-agi ?

<span id="an_3:11">3:11</span> A k- iniɣ tideț: neqqaṛ-ed ayen kan i
nessen, nețcehhid-ed s wayen i nwala, lameɛna kunwi ur teqbilem ara
cchada-nneɣ.

<span id="an_3:12">3:12</span> M'ur tuminem ara mi wen-d-hedṛeɣ ɣef
temsalin n ddunit-agi amek ara tamnem s tid n igenwan ?

<span id="an_3:13">3:13</span> Yiwen ur yuli ɣer igenwan anagar Mmi-s n
bunadem i d-iṣubben seg igenwan.

<span id="an_3:14">3:14</span> Akken i gɛelleq Musa azrem n nnḥas ɣef
tgejdit deg unezṛuf, i glaq ad ițțuɛelleq daɣen Mmi-s n bunadem

<span id="an_3:15">3:15</span> iwakken kra n win yumnen yis, ad yesɛu
tudert n dayem.

<span id="an_3:16">3:16</span> Sidi Ṛebbi iḥemmel imdanen n ddunit meṛṛa
armi i d-yefka Mmi-s awḥid iwakken kra n win yumnen yis ur yețmețțat ara
meɛna ad yesɛu tudert n dayem.

<span id="an_3:17">3:17</span> Sidi Ṛebbi ur d-iceggeɛ ara Mmi-s ɣer
ddunit iwakken ad iḥaseb imdanen meɛna iwakken a ten-isellek.

<span id="an_3:18">3:18</span> Win yumnen yis ur yețțuḥaseb ara, ma d
win ur numin ara atan yețțuḥaseb yakan, imi ur yumin ara s Mmi-s awḥid n
Sidi Ṛebbi.

<span id="an_3:19">3:19</span> Atah lḥisab yellan fell-as : « tafat
tusa-d ɣer ddunit meɛna imdanen ḥemmlen ṭṭlam axiṛ n tafat, axaṭer
lecɣal-nsen diri-ten. »

<span id="an_3:20">3:20</span> Kra n win ixeddmen cceṛ, yekṛeh tafat
yerna ur ițqeṛṛib ara ɣuṛ-es, ițțaggad a d-tessedheṛ yir lecɣal-is.

<span id="an_3:21">3:21</span> Ma d win iteddun di tideț yețțas-ed ɣer
tafat akken a d-iban belli kra n wayen ixeddem, ixeddem it di lebɣi n
Sidi Ṛebbi.

<span id="an_3:22">3:22</span> Syenna Sidna Ɛisa d inelmaden is ṛuḥen
ɣer tmurt n Yahuda, qqimen dinna kra n wussan, sseɣḍasen lɣaci deg
waman.

<span id="an_3:23">3:23</span> Yeḥya daɣen yella yesseɣḍas lɣaci di
