### Kabyle Bible: 1 Peter

<span id="an_1:1">1:1</span> Nekk Buṭrus, amceggeɛ n Ɛisa Lmasiḥ, i
yeɣṛiben ițɛicin di tmurt n Galasya, n Kafadusya, n Asya, n Bitinya
akk-d wid yellan di tmurt n tqenṭert.

<span id="an_1:2">1:2</span> Akken i t-iqsed Baba Ṛebbi, tețwaxtaṛem si
tazwara s tezmert n Ṛṛuḥ iqedsen iwakken aț-țḍuɛem Ɛisa Lmasiḥ, a
kkun-iṣfu s idammen-is yuzzlen. Ṛṛeḥma ț-țalwit a wen-d tețțunefkent s
tugeț.

<span id="an_1:3">1:3</span> Ad ițțubarek Ṛebbi, Baba-s n Ssid-nneɣ Ɛisa
Lmasiḥ i ɣ-d-ifkan s ṛṛeḥma-ines tudert tajḍiṭ s ḥeggu n Ssid-nneɣ Ɛisa
Lmasiḥ si ger lmegtin, s wakka i nesɛa asirem ameqqran,

<span id="an_1:4">1:4</span> iwakken a newṛet ayen ur nrekku, ur
nețțames, ur nfennu. D annect-agi i kkun-ițṛaǧun deg igenwan kunwi
yumnen yis.

<span id="an_1:5">1:5</span> Sidi Ṛebbi ițḥadar fell-awen s tezmert-is
alamma iban-ed leslak nwen di taggara.

<span id="an_1:6">1:6</span> D ayagi i kkun-issefṛaḥen ɣas akken tura
sseḥzanent-kkun teswiɛin agi deg ilaq a d-tɛeddim seg ujeṛṛeb.

<span id="an_1:7">1:7</span> D uguren-agi ara d-isbeggnen lǧehd n
liman-nwen i gesɛan azal akteṛ n ddheb ; ɣas akken ddheb-agi ifennu,
ilaq-as ad iɛeddi di tmes iwakken ad iṣfu. Akken daɣen ula d liman-nwen
ilaq ad ițțujeṛṛeb ; s wakka a wen-yili d sebba n ccan, n ucekkeṛ, akk-d
tmanegt asm'ara d-yuɣal Ɛisa Lmasiḥ.

<span id="an_1:8">1:8</span> Tḥemmlem-t mbla ma twalam-t, tumnem yis ɣas
akken ar tura ur t-teẓrim ara. Daymi lfeṛḥ-nwen d ameqqran armi ur
tezmirem ara a t-id-tesfehmem s imeslayen.

<span id="an_1:9">1:9</span> Axaṭeṛ lfayda n liman-nwen, d leslak n
teṛwiḥin-nwen.

<span id="an_1:10">1:10</span>  Lenbiya caren-d ɣef wayen yeɛnan leslak
i wen-d-ihegga Sidi Ṛebbi. Meyzen, qellben a d-afen lawan d wamek ara
d-idṛu wayen i d-ixebbeṛ Ṛṛuḥ n Lmasiḥ yellan deg-sen ; axaṭer d
Ṛṛuḥ-agi n Lmasiḥ i d-ixebbṛen ɣef wayen ara inneɛtab Lmasiḥ akk-d
tmanegt ara d-yasen deffir leɛtab-agi.

<span id="an_1:12">1:12</span> Ițțubeggen-asen-d belli mačči i nutni,
meɛna i kunwi i d-xebbṛen ayagi. D ayen i wen-d-xebbṛen tura wid
ițbecciṛen lexbaṛ n lxiṛ, s tezmert n Ṛṛuḥ iqedsen i d-ițțucegɛen seg
igenni. Ula d lmalayekkat bɣant ad walint annect-agi.

<span id="an_1:13">1:13</span> Daymi, sǧehdet leɛqel-nwen, ur ɣefflet
ara, ɛasset iman-nwen, sɛut asirem iṣeḥḥan di leslak ara wen
d-ițțunefken asm' ara d-yuɣal Ɛisa Lmasiḥ.

<span id="an_1:14">1:14</span> Ilit am arrac ițțaɣen awal, ur xeddmet
ara ayen akk i txeddmem asm' akken werɛad i kkun-tekcim tmusni,

<span id="an_1:15">1:15</span> meɛna ilit teṣfam di tikli-nwen am akken
iṣfa Sidi Ṛebbi i wen-d issawlen.

<span id="an_1:16">1:16</span> Imi yura : Ilit teṣfam axaṭer nekk ṣfiɣ.

<span id="an_1:17">1:17</span> Ma tnedhem ɣer Sidi Ṛebbi am akken d
Baba-twen, sbeggnet-ed di tudert-nwen belli tḍuɛem-t s leqdeṛ ameqqran,
deg wussan i wen-d iqqimen di ddunit-agi ; nețța ur nxeddem lxilaf ger
yemdanen, yețḥasaben yal yiwen s wakken llan lecɣal-is.

<span id="an_1:18">1:18</span> Teẓram belli mačči s idrimen neɣ s ddheb
neɣ s lfeṭṭa ur nețdum ara i tețwacafɛem si yir tikli i wen-d-ǧǧan
lejdud-nwen,

<span id="an_1:19">1:19</span> meɛna s idammen ɣlayen n Lmasiḥ, i gefkan
iman-is d asfel am izimer ur nesɛi lɛib, ur nesɛi ccama.

<span id="an_1:20">1:20</span> Ihegga-t-id Sidi Ṛebbi uqbel a d-texleq
ddunit, meɛna armi d ussan-agi ineggura i t-id-isbeggen ɣef ddemma-nwen.

<span id="an_1:21">1:21</span> Imi yis i tesɛam liman ɣer Sidi Ṛebbi i
t-id-isseḥyan si ger lmegtin, i s-ifkan tamanegt tameqqrant, iwakken
lsas n usirem-nwen akk-d liman-nwen ad ilin di Sidi Ṛebbi.

<span id="an_1:22">1:22</span> Imi tessazdgem ulawen-nwen, teddam di
tideț, akken aț-țḥemmlem seg wul atmaten-nwen, myeḥmalet ihi wway
gar-awen s wul yeṣfan

<span id="an_1:23">1:23</span> axaṭer tɛawdem-d talalit ɣer tudert
tajḍiṭ. Talalit-agi mačči seg wemdan ițmețțaten i d-tekka, meɛna seg
wawal n tudert ițdumun n Sidi Ṛebbi.

<span id="an_1:24">1:24</span> Akken yella di tira iqedsen : Amdan am
leḥcic, + ccan-is meṛṛa am ujeǧǧig n lexla,+ leḥcic ițɣaṛ, ajeǧǧig
iɣelli,+

<span id="an_1:25">1:25</span> ma d awal n Sidi Ṛebbi+ ițdumu i dayem+ .
Awal-agi, d awal n lexbaṛ n lxiṛ i wen-d-ițțubecṛen.

<span id="an_2:1">2:1</span> Kkset seg ulawen-nwen cceṛ, tiḥila, tismin
; ur seɛɛut ara sin wudmawen, ur țmeslayet ara ayen n diri ɣef wiyaḍ.

<span id="an_2:2">2:2</span> Am llufan illuẓen ayefki n yemma-s, nadit
ula d kunwi aț-țeṛwum awal n Sidi Ṛebbi iwakken aț-țimɣuṛem,
aț-tețțuselkem,

<span id="an_2:3">2:3</span> ma yella tjeṛbem acḥal yelha Sidi Ṛebbi !

<span id="an_2:4">2:4</span> Qeṛṛbet ɣuṛ-es nețța yellan d-azṛu nni
yeddren i ḍeggṛen yemdanen, meɛna Sidi Ṛebbi ixtaṛ-it, axaṭer azal-is
meqqeṛ.

<span id="an_2:5">2:5</span> Ula d kunwi yellan am izṛa yeddren, dduklet
iwakken ad ibedd wexxam i deg ara izdeɣ Ṛṛuḥ n Sidi Ṛebbi. Aț-țilim
ț-țajmaɛt n lmuqedmin ara iqeddmen s yisem n Ssid-nneɣ Ɛisa Lmasiḥ,
iseflawen iqedsen ara iɛeǧben i Sidi Ṛebbi.

<span id="an_2:6">2:6</span> Axaṭer atan wayen yuran di tira iqedsen :
Xtaṛeɣ-ed azṛu+ yesɛan azal d ameqqran+ , sserseɣ-t deg udrar n Siyun,+
d nețța i geṭṭfen lebni,+ win ara yamnen yis ur ineddem ara.+

<span id="an_2:7">2:7</span> Amarezg-nwen kunwi yumnen yis. Ma d wid ur
numin ara : Adɣaɣ-nni ḍeggṛen wid ibennun,+ d nețța i guɣalen d azṛu
alemmas,+ Win yeṭṭfen lebni. +

<span id="an_2:8">2:8</span> D wagi i d azṛu + i guɣalen d ugur i
yemdanen, + d azṛu i ten-isseɣlayen+ . Imi ugin ad amnen s wawal n Sidi
Ṛebbi, ayen i ten-ițṛaǧun d ugur-agi ara ten-iɣeḍlen.

<span id="an_2:9">2:9</span> Kunwi d arraw n laṣel ițwaxtaṛen, d
lmuqedmin n ugellid, d lǧens iṣfan, d agdud i d-ițwacefɛen. Tețwaxtaṛem
iwakken aț-țbecṛem lecɣal yelhan n Sidi Ṛebbi, i kkun-id issufɣen si
ṭṭlam yewwi-kkun-id ɣer tafat-is isteɛǧiben.

<span id="an_2:10">2:10</span> Zik-nni ur tewwiḍem ara ula d lǧens, ma
ț-țura tuɣalem d agdud n Sidi Ṛebbi ; tellam zik-nni ur tesɛim ara
ṛṛeḥma n Sidi Ṛebbi ma ț-țura tețțunefk-awen-d.

<span id="an_2:11">2:11</span> A wid eɛzizen ! Kunwi yellan d
ibeṛṛaniyen, d imsebriden di ddunit a kkun-nhuɣ aț-țettixṛem i lebɣi n
tnefsit i gețnaɣen d ṛṛuḥ-nwen.

<span id="an_2:12">2:12</span> Sɛut tikli ilhan ger wid ur nessin ara
Sidi Ṛebbi, iwakken ɣas a d-skidben fell-awen, neɣ xedmen awen cceṛ ; m'
ara walin lecɣal-nwen ilhan ad ḥemden Sidi Ṛebbi asm' ara d-yerzu
ɣuṛ-sen.

<span id="an_2:13">2:13</span> Ɣef ddemma n Sidi Ṛebbi qadṛet wid
iḥekmen. Qadṛet agellid imi d nețța i gḥekmen,

<span id="an_2:14">2:14</span> qadṛet ineɣlafen-is imi d nețța i
ten-isbedden iwakken ad ɛaqben imcumen, ad cekkṛen wid i gțeddun s
lḥeqq.

<span id="an_2:15">2:15</span> Xedmet ayen yelhan iwakken aț-țerrem
takmamt i imejhal ur nefhim ; d wagi i d lebɣi n Sidi Ṛebbi.

<span id="an_2:16">2:16</span> Ilit d ilelliyen, meɛna ur ssexdamet ara
tilelli-nwen iwakken aț-țɣummem cceṛ i txeddmem, ilit d iqeddacen n Sidi
Ṛebbi.

<span id="an_2:17">2:17</span> Qadṛet imdanen meṛṛa ; ḥemmlet atmaten
nwen, ḍuɛet Sidi Ṛebbi, qadṛet agellid.

<span id="an_2:18">2:18</span> Ay iqeddacen ! Ilaq aț-qadṛem imɛellmen
nwen, aɣet-asen awal ama i wid ilhan ḥninen, ama i wid iweɛṛen.

<span id="an_2:19">2:19</span> D ayen ilhan i win iṣebbṛen m' ara iɛeddi
fell-as lbaṭel ilmend n ṭṭaɛa n Sidi Ṛebbi.

<span id="an_2:20">2:20</span> Ma tețwaɛaqbem imi txedmem kra n diri, ma
tṣebṛem i wannect-agi, anwa ara kkun-icekkṛen ? Meɛna ma iɛedda
fell-awen lbaṭel, mi txedmem ayen yelhan, tṣebṛem, d ayen yelhan ɣer
Sidi Ṛebbi.

<span id="an_2:21">2:21</span> Ɣer wannect-agi i wen-d-issawel Sidi
Ṛebbi, axaṭer Lmasiḥ s yiman-is inneɛtab fell-awen, iǧǧa-yawen-d lemtel
iwakken aț țeddum di lateṛ-is.

<span id="an_2:22">2:22</span> Nețța ur nednib, leɛmeṛ ur d-iffiɣ lekdeb
seg imi-s.

<span id="an_2:23">2:23</span> Nețța ițwaregmen ur d-yerri rregmat,
qehṛen-t ur d-irfid afus-is ɣer yiwen ; lameɛna yerra kullec ger ifassen
n win i gḥekmen s lḥeqq.

<span id="an_2:24">2:24</span> Nețța i gɛebban ddnubat-nneɣ, ițwasemmeṛ
ɣef wumidag iwakken a nili am wid yemmuten ɣef wayen yeɛnan ddnub axaṭer
ddnub ur isɛi ara tazmert ɣef wid immuten. S wakka, tura ur nețɛici ara
di ddnub meɛna di lḥeqq, akken yura : Di leǧruḥ-is i tufam ḥellu. +

<span id="an_2:25">2:25</span> Zik tellam am ulli iɛeṛqen, ma ț-țura,
tuɣalem ɣer umeksa ițɛassan ɣef teṛwiḥin-nwen.

<span id="an_3:1">3:1</span> Kunemti daɣen a tilawin ḍuɛemt
irgazen-nkunt, iwakken ɣas ma llan kra n yergazen ur uminen ara s wawal
n Sidi Ṛebbi, s tikli ilhan n tilawin-nsen,

<span id="an_3:2">3:2</span> ad amnen mbla imeslayen m' ara walin
tikli-nsent teṣfa, tesɛa leqdeṛ.

<span id="an_3:3">3:3</span> Ur țzewwiqemt ara iman-nkunt s ufella kan,
ama s ddheb, neɣ s llebsa ifazen, neɣ s wemcaḍ n ucekkuɛ nkunt,

<span id="an_3:4">3:4</span> meɛna ccbaḥa-nkunt ilaq aț-țili deg
wulawen-nkunt ; axaṭer ayen yesɛan azal ameqqran ɣer Sidi Ṛebbi :
ț-țezdeg n wul, lehna d leḥnana, imi ayagi d ayen ițdumun.

<span id="an_3:5">3:5</span> Akka i țcebbiḥent iman-nsent tilawin iḍuɛen
Sidi Ṛebbi, yețqadaṛen irgazen-nsent ;

<span id="an_3:6">3:6</span> am Saṛa iḍuɛen argaz-is Ibṛahim,
tessawal-as « A Sidi. » Aț-țuɣalemt d yessi-s ma txeddmemt ayen yelhan
mbla tugdi.

<span id="an_3:7">3:7</span> Ma d kunwi ay irgazen, ḥadret tilawin-nwen
imi teẓram ur ǧhident ara am kunwi. Ilaq a tent-tqadṛem axaṭer nutenti
daɣen ad weṛtent yid-wen ṛṛeḥma n Sidi Ṛebbi ; s wakka ur d-ițili ara
ẓẓeṛb i tẓallit-nwen.

<span id="an_3:8">3:8</span> Ihi sɛut akk yiwet n lɛeqliya, myeḥmalet
wway gar-awen. Sɛut tagmaț, iḥninet, ddut s wannuz.

<span id="an_3:9">3:9</span> Ur țțarat ara țțaṛ, ur țarrat ara rregmat i
wid i kkun-iregmen ; meɛna ddɛut s lxiṛ i wiyaḍ axaṭer d annect-agi i
gețṛaǧu deg-wen Sidi Ṛebbi, iwakken aț-țweṛtem lbaṛaka-ines.

<span id="an_3:10">3:10</span>  Win iḥemmlen tudert, + ibɣan ad yili d
aseɛdi deg ussan-is :+ + + Ad iḥader iles-is ɣef wayen n diri+ d yimi-s
seg yimeslayen ițɣuṛṛun+ .

<span id="an_3:11">3:11</span> Ad issebɛed iman-is ɣef cceṛ, + ad
ixeddem ayen yelhan,+ ad ițnadi amek ara yili dayem di talwit,+ +

<span id="an_3:12">3:12</span> axaṭer Sidi Ṛebbi ṛeṣṣant wallen-is ɣef
yiḥeqqiyen, + + isell-ed i tiɣṛi-nsen ; + meɛna ițeffer udem-is + ɣef
wid ixeddmen cceṛ.

<span id="an_3:13">3:13</span> Anwa ara wen-ibɣun cceṛ ma yella tḥemmlem
aț-țxedmem anagar ayen yelhan ?

<span id="an_3:14">3:14</span> Yerna ma yella tețwaḍelmem ɣef lḥeqq,
feṛḥet ! Ur țțagadet ara imdanen, ur tḥeyyiṛet ara.

<span id="an_3:15">3:15</span> Ɛuzzet deg ulawen-nwen Lmasiḥ yellan d
Ssid-nwen. Ilit dayem twejdem aț-țǧawbem s leɛqel d leqdeṛ i kra n wid i
kkun-id-isteqsan ɣef wusirem nwen di Sidi Ṛebbi, iwakken ad ithedden
wul-nwen ;

<span id="an_3:16">3:16</span> imiren wid i d-iskiddiben ɣef tikli-nwen
ilhan deg ubrid n Lmasiḥ, ad nneḥcamen.

<span id="an_3:17">3:17</span> Axiṛ aț-țenṭaṛṛem ma txedmem ayen yelhan
di lebɣi n Sidi Ṛebbi, wala aț-țenṭaṛṛem ma txedmem ayen n diri.

<span id="an_3:18">3:18</span> Lmasiḥ s yiman-is immut ; s lmut-agi-ines
i gefra tamsalt n ddnub n yemdanen. Nețța yellan d aḥeqqi, immut ɣef
ddemma n imednuben meṛṛa iwakken a kkun-yawi ɣer Sidi Ṛebbi. Nɣan
lgețța-s meɛna yuɣal-ed ɣer tudert s tezmert n Ṛṛuḥ iqedsen.

<span id="an_3:19">3:19</span> S tezmert n Ṛṛuḥ iqedsen i gṛuḥ ad
ibecceṛ i leṛwaḥ ițwaḥebsen,

<span id="an_3:20">3:20</span> yugin ad amnen, asm' akken i ten-ițṛaǧu
Sidi Ṛebbi s ṣṣbeṛ akken a d-uɣalen ɣer webrid-is ; asmi i gexdem Sidna
Nuḥ lbabuṛ i ɣer kecmen anagar tmanya yemdanen i gmenɛen si lḥemla n
waman.

<span id="an_3:21">3:21</span> Aman-agi, d lemtel n weɣḍas i
kkun-ițselliken ass-a ula d kunwi. Aɣḍas-agi n waman mačči d ayen i
gessizdigen lǧețța n wemdan, lameɛna ițbeggin-ed belli nefka tudert-nneɣ
i Sidi Ṛebbi s wul yeṣfan. Ayagi yezmer a d-yedṛu i lmend n ḥeggu n
Sidna Ɛisa Lmasiḥ

<span id="an_3:22">3:22</span> i gulin ɣer igenni, yeṭṭef amkan n lḥekma
ɣer tama tayeffust n Sidi Ṛebbi, sennig n lmalayekkat, sennig n tezmar d
lḥekmat meṛṛa.

<span id="an_4:1">4:1</span> Am akken inneɛtab Lmasiḥ di lǧețța-s, kunwi
daɣen ilaq-awen aț-țheggim iman-nwen ; axaṭer win i genneɛtaben di
lǧețța-s iṭṭaxeṛ ɣef ddnub ;

<span id="an_4:2">4:2</span> iwakken ussan i s-d-iqqimen di tudert-is,
ad iddu s lebɣi n Sidi Ṛebbi, mačči akken ibɣa nețța.

<span id="an_4:3">4:3</span> Beṛka-kkun tura ! Tellam tețțeddum d wid
ijehlen, ur nessin Sidi Ṛebbi : tellam d isekṛanen, tețțeddum deg webrid
n lefsad d zzhu, tɛebdem lemqamat, ulawen nwen ččuṛen d ṭṭmeɛ.

<span id="an_4:4">4:4</span> Tura wehmen imi teṭṭaxṛem fell-asen ur
tețțeddum ara deg webrid nsen i gețțawin ɣer nnger, daymi i kkaten
deg-wen.

<span id="an_4:5">4:5</span> Lameɛna a ten-iḥaseb Sidi Ṛebbi, nețța
yețḥasaben wid iddren d wid yemmuten.

<span id="an_4:6">4:6</span> S wakka, lexbaṛ n lxiṛ ițțubecceṛ ula i wid
yemmuten iwakken ɣas țțuḥasben am imdanen n ddunit meṛṛa, ad idiren s
Ṛṛuḥ iqedsen akken ibɣa Sidi Ṛebbi.

<span id="an_4:7">4:7</span> Taggara n ddunit tețțeddu-d ! Ɣef
wannect-a, sɛut leɛqel, ɛasset ɣef yiman-nwen iwakken aț-țizmirem
aț-țeẓẓallem.

<span id="an_4:8">4:8</span> Lḥaǧa tamezwarut, myeḥmalet wway gar-awen
seg wul, axaṭer leḥmala tețsamaḥ aṭas n ddnubat.

<span id="an_4:9">4:9</span> Țmesṭerḥabet wway gar-awen s wul iṣfan.

<span id="an_4:10">4:10</span> Mkul yiwen deg-wen d acu n tikci i
s-d-ifka Sidi Ṛebbi, ihi sxedmet-eț iwakken ad sfaydin wiyaḍ seg-s.

<span id="an_4:11">4:11</span> Win i wumi i d-yețțunefk ad ihḍeṛ, ilaq
ad ihḍeṛ s imeslayen i s-d-ițțasen s ɣuṛ Sidi Ṛebbi. Akken daɣen ma
yella wayeḍ ițțunefk-as-ed ad iqdec, ad iqdec ɣef wiyaḍ s tezmert i
s-d-ifka Sidi Ṛebbi. Iwakken di yal ccɣel, ad ițwaḥmed Sidi Ṛebbi s
yisem n Ɛisa Lmasiḥ, bab n tezmert ț-țmanegt si lǧil ɣer lǧil. Amin !

<span id="an_4:12">4:12</span> A wid eɛzizen ! Ur wehhmet ara ɣef
wujeṛṛeb i d-iɣellin fell-awen am tmes, ur xellɛet ara, axaṭer ayagi
ilaq a d-idṛu.

<span id="an_4:13">4:13</span> Lameɛna feṛḥet imi i tețțekkim di leɛtab
n Lmasiḥ, axaṭer asm' ara d-tban tmanegt-is, aț-țfeṛḥem yid-es lfeṛḥ d
ameqqran.

<span id="an_4:14">4:14</span> Amarezg-nwen ma yella iɛedda fell-awen
lbaṭel imi tumnem s Lmasiḥ ! Axaṭer Ṛṛuḥ iqedsen iččuṛen ț-țamanegt,
Ṛṛuḥ n Sidi Ṛebbi yella deg-wen !

<span id="an_4:15">4:15</span> Ḥadret a d-yili gar-awen win ara
ițwaqehṛen axaṭer yenɣa tamgeṛṭ neɣ yuker, neɣ iger iman-is di lecɣal n
wiyaḍ ;

<span id="an_4:16">4:16</span> lameɛna ma yella ițwaqheṛ imi yumen s
Lmasiḥ, ur ilaq ara ad inneḥcam ; ilaq ad iḥmed Sidi Ṛebbi ɣef
yisem-agi.

<span id="an_4:17">4:17</span> Yewweḍ-ed lweqt n lḥisab n Sidi Ṛebbi ;
yerna ad ibdu seg wat wexxam-is. Ihi ma yella seg-neɣ ara ibdu lḥisab,
amek ara tili taggara n wid ur numin ara s lexbaṛ n lxiṛ n Sidi Ṛebbi ?

<span id="an_4:18">4:18</span> Ma yella iwɛeṛ-as i uḥeqqi iwakken ad
ițwasellek, amek ara tedṛu ihi d wemcum ur numin ara s Sidi Ṛebbi ?

<span id="an_4:19">4:19</span> Ma yella s lebɣi n Sidi Ṛebbi i genneɛtab
yiwen ; ilaq ad yerr kullec ger ifassen n win i t-id-ixelqen, ad ițkel
fell-as, ad ikemmel di lecɣal-is ilhan.

<span id="an_5:1">5:1</span> Ad nhuɣ tura imeqqranen di liman yellan
gar-awen. Nekk yellan d ameqqran am nutni, nekk iḥedṛen i leɛtab n
Lmasiḥ, ițekkan di tmanegt-is ara d-ibanen ; a ten weṣṣiɣ :

<span id="an_5:2">5:2</span> Ḥadret wid i wen-d-iwekkel Sidi Ṛebbi akken
ițḥadar umeksa ɣef qeḍɛit-is ; lhit ihi yid-sen s lebɣi-nwen, mačči
sennig n wulawen-nwen, xedmet ayagi ɣef wudem n Sidi Ṛebbi mačči s
ṭṭmeɛ.

<span id="an_5:3">5:3</span> Ur ssimɣuṛet ara iman-nwen ɣef wid i
wen-d-iwekkel Sidi Ṛebbi, meɛna ilit d lemtel ara tebɛen.

<span id="an_5:4">5:4</span> Imiren asm' ara d-iban win yellan sennig
imeqqranen meṛṛa, a wen d-tețțunefk tɛeṣṣabt n tmanegt ur nfennu.

<span id="an_5:5">5:5</span> Ula d kunwi daɣen ay ilmeẓyen, qadṛet
imeqqranen di liman ! Ddut s leqdeṛ d wannuz, axaṭer Sidi Ṛebbi ur
iqebbel ara wid ițzuxxun, meɛna ițțak ṛṛeḥma-s i wid ițțanzen.

<span id="an_5:6">5:6</span> Anzet ihi seddaw ufus iǧehden n Sidi Ṛebbi,
iwakken a kkun-irfed di teswiɛt ilaqen.

<span id="an_5:7">5:7</span> Rret akk iɣeblan-nwen ger ifassen-is,
axaṭer ițḥebbiṛ fell-awen.

<span id="an_5:8">5:8</span> Ḥadret, Ɛasset iman-nwen axaṭer aɛdaw-nwen
d Cciṭan, ițezzi iṛeɛɛed am yizem, ițqellib anwa ara issebleɛ.

<span id="an_5:9">5:9</span> Ur s-țaǧǧat ara abrid, sɛut liman iǧehden ;
axaṭer teẓram belli atmaten nwen di ddunit meṛṛa, akka i tdeṛṛu yid-sen
am kunwi.

<span id="an_5:10">5:10</span> M'ara tenneɛtabem kra n wussan, Sidi
Ṛebbi Bab n mkul ṛṛeḥma i kkun-ixtaṛen s Ɛisa Lmasiḥ iwakken aț-țilim di
tmanegt-is ițdumun, a kkun-iheggi, a kkun iseggem, a kkun-isseǧhed,
iwakken ur tɣellim ara.

<span id="an_5:11">5:11</span> I nețța lḥekma si lǧil ɣer lǧil. Amin !

<span id="an_5:12">5:12</span> S ufus n gma-tneɣ eɛzizen Silwan i ɣef
nețkel, i wen-d-uriɣ kra n imeslayen-agi iwakken a kkun-nhuɣ yerna ad
cehdeɣ belli ț-țagi i d ṛṛeḥma n tideț n Sidi Ṛebbi s wayes tumnem.

<span id="an_5:13">5:13</span> Tajmaɛt n watmaten yumnen s Lmasiḥ,
yellan di temdint n Babilun țsellimen-d fell-awen ; ula d mmi Maṛqus
ițsellim fell-awen.

<span id="an_5:14">5:14</span> Msalamet wway gar-awen am atmaten ;
talwit a d-ters fell-awen kunwi yumnen s Lmasiḥ.
