### Kabyle Bible: 1 Corinthians

<span id="an_1:1">1:1</span> Nekk Bulus, iwumi i d-issawel Sidi Ṛebbi s
lebɣi-ines iwakken ad iliɣ d amceggeɛ n Ɛisa Lmasiḥ akk-d gma-tneɣ
Sustin,

<span id="an_1:2">1:2</span> i tejmaɛt n Sidi Ṛebbi i gellan di temdint
n Kurintus, i wid i d-yextaṛ Sidi Ṛebbi s Ɛisa Lmasiḥ ad ilin d
imqedsen, d wid akk yellan di mkul amkan ineddhen s yisem n Sidna Ɛisa
Lmasiḥ, i gellan d Ssid-nsen d Ssid nneɣ :

<span id="an_1:3">1:3</span> ṛṛeḥma ț-țalwit a wen-d țțunefkent s ɣuṛ
Ṛebbi Baba-tneɣ akk-d Sidna Ɛisa Lmasiḥ.

<span id="an_1:4">1:4</span> Zgiɣ țḥemmideɣ Sidi Ṛebbi fell-awen ɣef
ṛṛeḥma i wen-d-yefka s Ɛisa Lmasiḥ,

<span id="an_1:5">1:5</span> axaṭer yis i kkun-id yeṛzeq s mkul lxiṛ,
ama s wawal-is ama s tmusni-ines,

<span id="an_1:6">1:6</span> imi cchada ɣef Lmasiḥ tṛeṣṣa nezzeh
deg-wen.

<span id="an_1:7">1:7</span> Daymi ur kkun-ixuṣṣ wacemma n lxiṛ s ɣuṛ
Ṛebbi, kunwi yețṛaǧun ass i deg ara d-yuɣal Ssid-nneɣ.

<span id="an_1:8">1:8</span> A kkun isseǧhed alamma ț-țaggara iwakken a
d-tbanem mbla lɛib ass n tuɣalin n Ɛisa Lmasiḥ.

<span id="an_1:9">1:9</span> Sidi Ṛebbi yețțaṭaf deg wawal-is ; d nețța
i wen-d-issawlen iwakken aț-țɛicem s tdukli akk-d Mmi-s, Ɛisa Lmasiḥ
Ssid-nneɣ.

<span id="an_1:10">1:10</span> Ay atmaten, a kkun-nhuɣ s yisem n
Ssid-nneɣ Ɛisa Lmasiḥ : msefhamet wway gar-awen, ur ilaq ara a d-yili
beṭṭu gar-awen, ddukklet, sɛut yiwen uxemmem d yiwen n ṛṛay.

<span id="an_1:11">1:11</span> Axaṭer ay atmaten, irgazen n wexxam n
Kluwi ṣṣawḍen-iyi-d lexbaṛ belli ițțili-d umennuɣ gar-awen.

<span id="an_1:12">1:12</span> Ayen bɣiɣ a d-iniɣ : sliɣ belli llan
gar-awen wid yeqqaṛen :« nekk ddiɣ d Bulus », ma d wayeḍ yeqqaṛ : « nekk
țekkiɣ d Abulus », wayeḍ daɣen : « nekk ddiɣ d Buṭrus », llan daɣen wid
yeqqaṛen : « nekk țekkiɣ di Lmasiḥ ».

<span id="an_1:13">1:13</span> Amek akka, tɣilem Lmasiḥ yebḍa ? Eɛni d
Bulus i gețwasemmṛen fell-awen neɣ s yisem n Bulus i tețwaɣeḍsem ?

<span id="an_1:14">1:14</span> Ad ḥemmdeɣ Ṛebbi imi ur sɣeḍseɣ ula d
yiwen deg-wen anagar Krisbus d Gayus.

<span id="an_1:15">1:15</span> Akka, yiwen deg-wen ur izmir a d-yini
belli yețwaɣḍes s yisem-iw.

<span id="an_1:16">1:16</span> Tella daɣen twacult n Stifanas, d nekk i
ț-isɣeḍsen meɛna ur ḥsiɣ ara sɣeḍseɣ yiwen nniḍen dinna.

<span id="an_1:17">1:17</span> Lmasiḥ ur iyi-d-iceggeɛ ara iwakken ad
sɣeḍseɣ meɛna iceggeɛ iyi-d ad beccṛeɣ lexbaṛ n lxiṛ ; mačči s
yimeslayen i d-yekkan si lefhama n wemdan iwakken ur tekkseɣ ara lqima i
lmut n Lmasiḥ ɣef wumidag.

<span id="an_1:18">1:18</span> Ț-țideț, abecceṛ n lmut n Lmasiḥ ɣef
wumidag ț-țidderwect i wid iḍaɛen ; meɛna ɣuṛ-nneɣ nukni yellan deg
webrid n leslak, ț-țazmert n Ṛebbi.

<span id="an_1:19">1:19</span> Atan wayen i d-nnant tira iqedsen : Ad
snegreɣ tamusni n imusnawen, ad rreɣ d ula+ c lefhama n wid ifehmen+ .

<span id="an_1:20">1:20</span> Anda-t uɛeqli ? Anda-t umusnaw ? Anda-t
uḥṛic ara d-isfehmen ayen ideṛṛun di lǧil-agi ? Ur d-isbeggen ara Sidi
Ṛebbi belli tamusni n ddunit-agi d lehbala ?

<span id="an_1:21">1:21</span> Imi at ddunit s lefhama-nsen ur ɛqilen
ara tamusni n Sidi Ṛebbi, nețța yebɣa ad isellek wid yumnen yis s
lexbaṛ-agi n lxiṛ i ḥesben wiyaḍ d lehbala.

<span id="an_1:22">1:22</span> At Isṛail ssuturen lbeṛhanat, iyunaniyen
țnadin tamusni ;

<span id="an_1:23">1:23</span> ma d nukni nețbecciṛ Lmasiḥ, win akken
yețțusemmṛen ɣef wumidag, yellan d țbehdila i wat Isṛail, d lehbala ɣer
iyunaniyen.

<span id="an_1:24">1:24</span> Meɛna wid iwumi i d-yessawel Sidi Ṛebbi,
ama d at Isṛail ama d iyunaniyen, umnen belli d nețța i d Lmasiḥ, i
ț-țazmert n Ṛebbi akk-d tmusni-ines.

<span id="an_1:25">1:25</span> Axaṭer ayen akken ḥesben yemdanen d
lehbala n Sidi Ṛebbi, ț-țamusni iṣeḥḥan akteṛ n tmusni n yemdanen, ayen
ḥesben daɣen d feccal n Sidi Ṛebbi, yeɣleb lǧehd n yemdanen.

<span id="an_1:26">1:26</span> Walit ay atmaten, kunwi iwumi d-yessawel
Sidi Ṛebbi : ur llin gar-awen aṭas n wid yesɛan tamusni n ddunit, ur
llin aṭas n wid yesɛan imukan ɛlayen, ur llin daɣen aṭas inaṣliyen
yesɛan lqima tameqqrant ;

<span id="an_1:27">1:27</span> meɛna Sidi Ṛebbi yextaṛ ayen yellan d
lehbala ɣer wemdan iwakken ad isneḥcam imusnawen, yextaṛ ayen ur nesɛi
tazmert di ddunit iwakken ad isneḥcam wid iǧehden.

<span id="an_1:28">1:28</span> Sidi Ṛebbi yextaṛ ayen ur nesɛi azal,
yețwaḥeqṛen di ddunit akk-d wayen yețwaḥesben d ulac, iwakken ad isseɣli
lqima n wayen yesɛan azal ɣer yemdanen.

<span id="an_1:29">1:29</span> Akka, yiwen ur izmir ad izuxx zdat Sidi
Ṛebbi.

<span id="an_1:30">1:30</span> Imi d Sidi Ṛebbi i kkun-isdukklen d Ɛisa
Lmasiḥ, d nețța i ɣ-yuɣalen d lefhama i d yekkan s ɣuṛ Sidi Ṛebbi. Yis i
ɣ yerra Sidi Ṛebbi d iḥeqqiyen, iṣeffa tikli-nneɣ, yefda-yaɣ-d si ddnub.

<span id="an_1:31">1:31</span> Am akken yura di tira iqedsen : Win
yebɣan ad izuxx, ad izuxx s lecɣal n Ṛebbi.

<span id="an_2:1">2:1</span> Daymi ay atmaten mi n-ṛuḥeɣ ɣuṛ wen iwakken
a wen mmeslayeɣ ɣef lbaḍna n Sidi Ṛebbi, ur a wen mmeslayeɣ ara s
yimeslayen yeččuṛen d zzux neɣ s tmusni yessewhamen.

<span id="an_2:2">2:2</span> Axaṭer ur nudaɣ ara a d mmeslayeɣ gar-awen
ɣef wayen nniḍen anagar ɣef Ɛisa Lmasiḥ ; Ɛisa Lmasiḥ i gețwaṣemmṛen ɣef
wumidag.

<span id="an_2:3">2:3</span> Ula d nekk asmi lliɣ ɣuṛ-wen, lliɣ fecleɣ,
kukraɣ yerna tekcem-iyi tugdi.

<span id="an_2:4">2:4</span> Ayen i wen-slemdeɣ d wayen i wen-beccṛeɣ ur
yebni ara ɣef tmusni n wemdan, lameɛna d ayen yebnan ɣef tezmert n Ṛṛuḥ
iqedsen.

<span id="an_2:5">2:5</span> Akka, liman-nwen ur yețrus ara ɣef tmusni n
yemdanen, meɛna ɣef tezmert n Sidi Ṛebbi.

<span id="an_2:6">2:6</span> Nesselmad tamusni-agi i wid yesɛan liman
iǧehden ; tamusni-agi, mačči ț-țamusni n ddunit-agi neɣ ț-țin n tezmar n
igelliden n ddunit-agi iteddun ɣer nnger.

<span id="an_2:7">2:7</span> Nețbecciṛ tamusni n Sidi Ṛebbi yellan d
lbaḍna yeffren ɣef yemdanen, i ghegga si zik i lɛezz-nneɣ.

<span id="an_2:8">2:8</span> Ula d yiwen deg igelliden n zzman-agi ur
ț-yessin, lemmer ssnen-ț tili ur semmṛen ara Bab n tmanegt ɣef wumidag.

<span id="an_2:9">2:9</span> Am akken i t-id-nnant tira iqedsen : Ayen
ur twala tiṭ, ayen ur yesli umeẓẓuɣ+ , ayen ɣef ur ixemmem ara wemdan,+
d ayen i ghegga Sidi Ṛebbi i wid i t-iḥemmlen+ .

<span id="an_2:10">2:10</span> Sidi Ṛebbi ibeggen-aɣ-t-id i nukni s Ṛṛuḥ
iqedsen. Axaṭer Ṛṛuḥ iqedsen yezmer ad iẓer ulamma d lecɣal yeffren n
Sidi Ṛebbi.

<span id="an_2:11">2:11</span> Anwa i geẓran ayen yellan deg wul n
wemdan ? Anagar ṛṛuḥ-ines i geẓran ayen yellan deg wul-is. Yiwen ur
yessin daɣen ixemmimen n Ṛebbi anagar Ṛṛuḥ n Sidi Ṛebbi.

<span id="an_2:12">2:12</span> Nukni mačči d ṛṛuḥ n ddunit i neqbel,
meɛna d Ṛṛuḥ i d-yekkan s ɣuṛ Sidi Ṛebbi, iwakken a nissin ayen akk i
ɣ-d-yefka s ṛṛeḥma-ines.

<span id="an_2:13">2:13</span> Nukni ur nețmeslay ara s yimeslayen i
tesselmad tmusni n wemdan, meɛna nețmeslay s yimeslayen i ɣ-isselmad
Ṛṛuḥ n Sidi Ṛebbi. Akka i nessefham tideț n Ṛṛuḥ iqedsen i wid yesɛan
Ṛṛuḥ-agi.

<span id="an_2:14">2:14</span> Amdan ițeddun s lɛeqliya n ddunit ur
iqebbel ara ayen i d-ițțasen s ɣuṛ Ṛṛuḥ n Sidi Ṛebbi imi ɣuṛ-es ayagi d
lehbala ; ur yezmir ara a t-yissin axaṭer s Ṛṛuḥ n Sidi Ṛebbi i nezmer a
t-nefhem.

<span id="an_2:15">2:15</span> Amdan yesɛan Ṛṛuḥ iqedsen n Sidi Ṛebbi
yezmer ad ifhem kullec, ma d nețța, yiwen ur izmir a t-iḥaseb.

<span id="an_2:16">2:16</span> Akken yura di tira iqedsen : Anwa i
gessnen ixemmimen n+ Sidi Ṛebbi,+ anwa i gzemren a t-yenhu + ? Ma d
nukni, nesɛa axemmem n Lmasiḥ.

<span id="an_3:1">3:1</span> Ay atmaten, ur ufiɣ ara amek ara
wen-mmeslayeɣ am akken țmeslayeɣ i yemdanen yesɛan Ṛṛuḥ iqedsen, meɛna
mmeslayeɣ-awen am yemdanen n ddunit, am wid yellan d arrac deg ubrid n
Lmasiḥ.

<span id="an_3:2">3:2</span> Fkiɣ-awen ayefki mačči d aɣṛum axaṭer aɣṛum
ur as-tezmirem ara, yerna ar tura ur as-tezmirem ara, mazal-ikkun
tețɛicim s lebɣi n tnefsit nwen.

<span id="an_3:3">3:3</span> Imi mazal gar-awen tismin d umceččew,
tbeggnem-d belli mazal ikkun tețɛicim s lɛeqliya-nwen yerna tleḥḥum am
yemdanen n ddunit.

<span id="an_3:4">3:4</span> M'ara yini yiwen deg-wen : « nekk ddiɣ d
Bulus », wayeḍ ad yini : « Nekk țekkiɣ d Abulus » ; eɛni ayagi mačči d
lɛeqliya n ddunit ?

<span id="an_3:5">3:5</span> D acu-t Abulus, d acu-t Bulus ? D iqeddacen
kan i kkun-id-yewwin aț-țamnem s Lmasiḥ ! Yal yiwen deg-nneɣ ixeddem
ayen i s-d-yefka Sidi Ṛebbi a t-yexdem.

<span id="an_3:6">3:6</span> Nekk ẓẓiɣ imɣi, Abulus issew-it, meɛna d
Sidi Ṛebbi i t-issegman.

<span id="an_3:7">3:7</span> Ihi mačči d win ițeẓẓun neɣ d win yesswayen
i gesɛan azal meɛna d Sidi Ṛebbi yessegmuyen.

<span id="an_3:8">3:8</span> Win ițeẓẓun d win yesswayen ɛedlen ; mkul
yiwen a s-yerr Sidi Ṛebbi s wakken yuklal di lxedma-s.

<span id="an_3:9">3:9</span> Axaṭer aql-aɣ neddukkel d iqeddacen ɣef
lecɣal n Sidi Ṛebbi. Kunwi d iger i gxeddem Sidi Ṛebbi, d axxam i
gbennu.

<span id="an_3:10">3:10</span> S ṛṛeḥma i yi-d-yefka Sidi Ṛebbi, am
umusnaw n lebni, nekk sserseɣ lsas, wayeḍ yebna fell-as ; meɛna yal
yiwen ad iḥader amek ara yebnu.

<span id="an_3:11">3:11</span> Yiwen ur izmir ad issers lsas nniḍen deg
wemkan n win yersen yakan : Ɛisa Lmasiḥ.

<span id="an_3:12">3:12</span> Ma yella win yebnan ɣef lsas-agi ama s
ddheb, ama s lfeṭṭa, ama s yeẓra ɣlayen, ama s wesɣaṛ neɣ s wusaɣuṛ d
walim,

<span id="an_3:13">3:13</span> mkul yiwen a d-iban ccɣel-is ass n
lḥisab, axaṭer mkul ccɣel ad iɛeddi di tmes ; ț-țimes ara d-ibeggnen
lecɣal n yal yiwen ma seḥḥan neɣ ur ṣeḥḥan ara.

<span id="an_3:14">3:14</span> Win yebnan ɣef lsas, ur t-tečči ara tmes
wayen akken yebna, a t-id-yaweḍ lxiṛ ameqqran ;

<span id="an_3:15">3:15</span> ma d win iwumi tečča tmes wayen yexdem, a
s-iṛuḥ lxiṛ-is ; ma d nețța ad ițțusellek, am win izegren times imneɛ.

<span id="an_3:16">3:16</span> Ur teẓrim ara belli kunwi d lǧameɛ n Sidi
Ṛebbi i tellam ? Ur teẓrim ara belli Ṛṛuḥ-is yezdeɣ deg-wen ?

<span id="an_3:17">3:17</span> Ihi win yesseɣlin axxam n Sidi Ṛebbi,
Sidi Ṛebbi a t-isseɣli axaṭer axxam n Sidi Ṛebbi d imqeddes, d kunwi i d
axxam-is.

<span id="an_3:18">3:18</span> Ḥadret ad yili win ara iɣuṛṛen iman-is !
Ma yella gar-awen win iḥesben iman-is d aɛeqli di ddunit-a, ilaq ad
yuɣal d amehbul iwakken ad yili d aɛeqli.

<span id="an_3:19">3:19</span> Axaṭer tamusni n ddunit-agi, ɣer Sidi
Ṛebbi d lehbala. Tira iqedsen nnant-ed : Sidi Ṛebbi yețțeṭṭef imusnawen
di tḥila-nsen.

<span id="an_3:20">3:20</span> Yura daɣen : Sidi Ṛebbi yessen ixemmimen
n imusnawen, yeẓra belli d ixemmimen ur nesɛi azal.

<span id="an_3:21">3:21</span> Ḥadret ad yili win ara izuxxen s
yemdanen, axaṭer kullec d ayla nwen ;

<span id="an_3:22">3:22</span> Bulus, Abulus neɣ Buṭrus, ddunit, tudert,
lmut, lweqt-a d lweqt i d-iteddun, kullec d ayla-nwen ;

<span id="an_3:23">3:23</span> kunwi d ayla n Lmasiḥ, Lmasiḥ d ayla n
Sidi Ṛebbi.

<span id="an_4:1">4:1</span> Ilaq a ɣ-ḥesben yemdanen belli d iqeddacen
kan n Lmasiḥ i nella, d wid i gwekkel Sidi Ṛebbi i wessefhem n
lbaḍna-ines.

<span id="an_4:2">4:2</span> Acu i nețṛaǧu deg win i nwekkel ɣef lḥaǧa ?
Nețṛaǧu a nețkel fell-as !

<span id="an_4:3">4:3</span> Nekk, ur iyi-d-tewqiɛ ara ma yella
tesḍelmem-iyi kunwi neɣ ma isseḍlem-iyi ccṛeɛ n yemdanen ; nekk daɣen ur
sḍelmeɣ ara iman-iw,

<span id="an_4:4">4:4</span> ɣas akken ur iyi-isseḍlem ara wul-iw, mačči
d ayagi ara d-isbeggnen belli d aḥeqqi i lliɣ, axaṭer d Sidi Ṛebbi ara
yi-ḥasben.

<span id="an_4:5">4:5</span> Daymi ur țḥasabet ara uqbel lweqt, uqbel
tuɣalin n Ssid-nneɣ ; d nețța ara d-isbeggnen s tafat-is ayen yeffren di
ṭṭlam, ara d-ibeggnen daɣen ayen yeffren deg wulawen. Imiren yal yiwen
ad isɛu ccan yuklal ɣer Sidi Ṛebbi.

<span id="an_4:6">4:6</span> Ay atmaten, ayen akka i d-nniɣ fell-i akk-d
Abulus nniɣ-t-id iwakken aț-țfehmem lmeɛna n lemtel-agi : « Ur țɛeddit
ara akkin i tilas n wayen yuran » ; ilaq yiwen ur ixeddem lxilaf ger
yemdanen, iwakken ad izuxx s yiwen deg-sen.

<span id="an_4:7">4:7</span> Ihi anwa i k-yessulin ccan sennig wiyaḍ ?
Acu ur k-d-nekki ara s ɣuṛ Ṛebbi deg wayen akk tesɛiḍ ? Imi d nețța i
k-d-yefkan, acuɣeṛ i tețcekkiṛeḍ iman-ik am akken mačči s ɣuṛ-es i
k-d-yekka ?

<span id="an_4:8">4:8</span> Atan teṛwam yakan, tuɣalem d imeṛkantiyen !
Tuɣalem d imḍebbṛen mbla nukni, acḥal bɣiɣ lemmer ț-țideț tzemrem i
yiman-nwen iwakken a nḍebbeṛ s tdukli !

<span id="an_4:9">4:9</span> Nukni yellan d imceggɛen n Lmasiḥ,
yefka-yaɣ Ṛebbi imukan ineggura ; aql-aɣ am wid ibuṣan, am wid i
tețɛassa lmut, tfeṛṛeǧ deg-nneɣ ddunit meṛṛa, am lmalayekkat, am
yemdanen.

<span id="an_4:10">4:10</span> Nukni d imehbal ɣef ddemma n Lmasiḥ, ma d
kunwi d iɛeqliyen di tikli-nwen akk-d Lmasiḥ ! Nukni nefcel, kunwi
tesɛam tazmert ; nukni nețwaḥqeṛ ma d kunwi tesɛam ccan !

<span id="an_4:11">4:11</span> Ar tura, aql-aɣ neqqim i laẓ d fad,
neɛra, nețwet, nețmenṭar si tama ɣer tayeḍ ;

<span id="an_4:12">4:12</span> nețɛețțib iman nneɣ a nexdem iwakken a
d-nawi aɣṛum-nneɣ s ifassen-nneɣ. M'ara ɣ-reggmen nukni nețbarak-iten,
m'ara ɣ-țqehhiṛen nukni nețțawi-yasen,

<span id="an_4:13">4:13</span> m'ara d-inin ayen n diri fell-aneɣ
nenehhu-ten. Ar tura ḥesben-aɣ am iḍumman n ddunit, am ineggura n
yemdanen.

<span id="an_4:14">4:14</span> Ur wen-d-uriɣ ara ayagi iwakken a
kkun-sneḥcameɣ, meɛna iwakken a kkun-sfehmeɣ am arraw iw i ḥemmleɣ.

<span id="an_4:15">4:15</span> Ɣas ma llan ɛecṛa alaf yemdanen i kkun
isselmaden deg webrid n Lmasiḥ, yiwen n baba-twen kan i tesɛam di
Lmasiḥ, imi d nekk i kkun-id-yewwin ɣer webrid n Lmasiḥ s Lexbaṛ n lxiṛ.

<span id="an_4:16">4:16</span> A kkun-weṣṣiɣ ihi aț-țeddum am akken ddiɣ
nekkini.

<span id="an_4:17">4:17</span> Ɣef wayagi i wen-ceggɛeɣ Timuti, mmi
eɛzizen di Lmasiḥ, iḍuɛen Sidi Ṛebbi iwakken a wen-d-yesmekti amek i
leḥḥuɣ deg webrid n Lmasiḥ, akken i t-sselmadeɣ di mkul tajmaɛt n
watmaten.

<span id="an_4:18">4:18</span> Kra deg-wen ikcem-iten zzux, ɣilen ur
n-țṛuḥuɣ ara a kkun-ẓreɣ ?

<span id="an_4:19">4:19</span> Meɛna ma yebɣa Sidi Ṛebbi, qṛib a n-aseɣ
ɣuṛ-wen. Ad ẓreɣ widak-agi yețzuxxun, mačči d imeslayen-nsen kan iwumi
ara ḥesseɣ meɛna ad waliɣ ayen iwumi izemren.

<span id="an_4:20">4:20</span> Axaṭer tazmert n tgeldit n Sidi Ṛebbi
mačči d imeslayen kan ara ț-id-isbeggnen, meɛna d lecɣal.

<span id="an_4:21">4:21</span> D acu i tebɣam ? A n-aseɣ ɣuṛ wen s
uɛekkaz, neɣ s wul yeččuṛen d leḥmala akk-d leḥnana ?

<span id="an_5:1">5:1</span> Di mkul amkan nesla yedṛa leḥṛam d ameqqran
gar-awen, d ayen ur nețțaf ara ulamma ger leǧnas nniḍen, tessawḍem armi
yiwen deg-wen yețɛic di leḥṛam akk ț-țmeṭṭut n baba-s !

<span id="an_5:2">5:2</span> Yerna mazal-ikkun tețzuxxum ! Ilaq-awen
aț-țilim di leḥzen, argaz i gxeddmen annect-agi ilaq a t tessufɣem si
gar-awen ;

<span id="an_5:3">5:3</span> ma d nekk, ɣas beɛdeɣ fell-awen leɛqel-iw
yella yid-wen ; atan sḍelmeɣ win ixedmen lɛaṛ-agi am akken lliɣ yid-wen.

<span id="an_5:4">5:4</span> S yisem n Sidna Ɛisa, s tezmert is,
nnejmaɛet aql-i yid-wen :

n tuɣalin n Ssid-nneɣ Ɛisa.

<span id="an_5:6">5:6</span> A wen-d-iniɣ : zzux-agi-nwen ur imɛin ara !
Eɛni ur tessinem ara lemtel i d-yeqqaṛen belli cwiṭ n yiɣes n temtunt
yessalay arukti meṛṛa ?

<span id="an_5:7">5:7</span> Ssizdeget iman-nwen, kkset si gar-awen
tamtunt ( iɣes ) taqdimt n ddnub iwakken aț-țilim d arukti ajdid yeṣfan.
Ț-țideț d arukti ajdid yeṣfan i tellam, axaṭer Tafaska-nneɣ tebda, imi
Lmasiḥ yellan d izimer nneɣ n leslak yefka iman-is d asfel.

<span id="an_5:8">5:8</span> Tura ihi a nkemmel lɛid weɣṛum mbla tamtunt
( iɣes ), mačči s temtunt-nni ixesṛen yellan d ddnub akk-d cceṛ, meɛna s
weɣṛum ur nesɛi tamtunt ( iɣes ) yellan d ṣṣfa akk-d tideț.

<span id="an_5:9">5:9</span> Di tebṛaț-nni i wen-n-uriɣ, nniɣ awen-d ur
țeddut ara d wid yețɛicin di leḥṛam.

<span id="an_5:10">5:10</span> Ur wen-d-nniɣ ara annect-agi ɣef wat
ddunit yețɛicin di leḥṛam, yeččuṛen d ṭṭmeɛ, d lexdeɛ ț-țukeṛda,
yețɛebbiden lmeṣnuɛat am Ṛebbi, neɣ m'ulac ilaq-awen aț-țeffɣem si
ddunit-agi.

<span id="an_5:11">5:11</span> Meɛna nniɣ-awen-d : ur țemxalaḍet ara d
win yețțarran iman-is d amasiḥi yili nețța yețɛici di leḥṛam, yeččuṛ d
ṭṭmeɛ, d lexdeɛ, iheddeṛ ɣef wiyaḍ, iɛebbed lmeṣnuɛat, d asekṛan, d
amakar. Ur țɣimit ara d wergaz am agi ulamma ɣer lqut.

<span id="an_5:12">5:12</span> Acu ara ɣ-yawin a nḥaseb wid n beṛṛa ?
Eɛni mačči d wid yellan gar-awen i glaq aț-țḥasbem ?

<span id="an_5:13">5:13</span> Wid n beṛṛa d Ṛebbi ara ten-iḥasben.
Meɛna akken yura di tira iqedsen : Stixṛet amcum si gar-awen.

<span id="an_6:1">6:1</span> Ma yella yiwen deg-wen yebɣa ad iccetki ɣef
gma-s, amek ara iṛuḥ a t-icaṛeɛ zdat wid ur numin ara s Lmasiḥ, mačči
zdat wid yumnen am nețța ?

<span id="an_6:2">6:2</span> Eɛni ur teẓrim ara d wid yumnen s Lmasiḥ
ara iḥasben at ddunit ? Ma yella ihi d kunwi ara iḥasben at ddunit, amek
ur tețțizmirem ara aț-țefrum timsalin timecṭuḥin ?

<span id="an_6:3">6:3</span> Ur teẓrim ara belli a nḥaseb lmalayekkat ?
Anda ara d-banent temsalin n ddunit-agi !

<span id="an_6:4">6:4</span> Ma d kunwi m'ara yili lxilaf gar-awen,
tețṛuḥum ɣer wid ur numin ara s Lmasiḥ iwakken ad frun tilufa-nwen !

<span id="an_6:5">6:5</span> Ur tețneḥcamem ara ! Eɛni ulac ula d yiwen
n wergaz ifehmen gar awen, i gzemren ad ifru timsalin ger watmaten ?

<span id="an_6:6">6:6</span> Meɛna kunwi atan wa yețcetki ɣef wa, yerna
ɣer wid ur numin ara s Lmasiḥ !

<span id="an_6:7">6:7</span> Ihi amennuɣ i d-yețțilin gar awen,
ițbeggin-ed lɛib-nwen. Acuɣeṛ ur tṣebbṛem ara axiṛ ad iɛeddi fell-awen
lbaṭel, acuɣeṛ ur tețțaǧǧam ara iman-nwen aț-țețțuɛerrim ?

<span id="an_6:8">6:8</span> Meɛna d kunwi i d imesbaṭliyen, tețɛerrim
wiyaḍ ɣas akken d atmaten-nwen !

<span id="an_6:9">6:9</span> Eɛni ur teẓrim ara belli wid ixeddmen
lbaṭel ur keččmen ara ɣer tgelda n Sidi Ṛebbi ? Ur țɣuṛṛut ara iman-nwen
axaṭer kra n win yețɛicin di leḥṛam neɣ wid iɛebbden lmeṣnuɛat, wid
ixeddmen zzna, at leɛṛuṛ,

<span id="an_6:10">6:10</span> wid yețțakren, wid itețțen ayla n medden,
wid ireggmen, ixeddaɛen akk-d isekṛanen, ur keččmen ara tagelda n Sidi
Ṛebbi.

<span id="an_6:11">6:11</span> Akka i llan yakan kra deg-wen uqbel ad
amnen. Meɛna ddnub-nwen yurad, tețwaxtaṛem iwakken aț-țilim d ayla n
Sidi Ṛebbi, tuɣalem d iḥeqqiyen zdat Ṛebbi s yisem n Ssid-nneɣ Ɛisa
Lmasiḥ akk-d Ṛṛuḥ iqedsen n Sidi Ṛebbi, Illu-nneɣ.

<span id="an_6:12">6:12</span> Kra deg-wen qqaṛen : « Ɣuṛ-i kullec d
leḥlal », meɛna mačči d kullec i kkun-inefɛen ; zemreɣ a d-iniɣ : kullec
d leḥlal ɣuṛ-i, meɛna ur țțaǧaɣ ara lḥaǧa a yi-temlek.

<span id="an_6:13">6:13</span> Lqut yella-d i wɛebbuḍ, aɛebbuḍ yella-d i
lqut ; a d-yas wass i deg ara ten-yessenger Sidi Ṛebbi i sin. Meɛna
lǧețța ur d-tețwaxleq ara i yir tikli, tețwaxleq-ed iwakken aț-țqeddec
ɣef Sidi Ṛebbi i ț-isedduyen.

<span id="an_6:14">6:14</span> Sidi Ṛebbi i d-yessekren Sidna Ɛisa si
ger lmegtin, a ɣ-d-isseḥyu ula d nukni s tezmert-is.

<span id="an_6:15">6:15</span> Ur teẓrim ara belli lǧețțat-nwen d
lemfaṣel n Lmasiḥ ? Eɛni zemreɣ a d-ddmeɣ lemfaṣel n Lmasiḥ iwakken a
ten-erreɣ d lemfaṣel n tmeṭṭut ixeddmen asekkak yeɛni zzna ? Ur zmireɣ
ara !

<span id="an_6:16">6:16</span> Ur teẓrim ara belli win ara yeznun nețța
akk-d yir tameṭṭut ad yuɣal yid-es d yiwen ? Akken yura di tira iqedsen
: Di sin ad uɣalen d yiwet n lǧețța.

<span id="an_6:17">6:17</span> Akka daɣen win ara iḥemmlen Lmasiḥ ad
yili yid-es s yiwen n Ṛṛuḥ.

<span id="an_6:18">6:18</span> Xḍut i yir tikli ! Kra n ddnub i gezmer a
t-ixdem wemdan ițɣimi beṛṛa n lǧețța-s, ma d win ara ixedmen zzna
yessewsex lǧețța-s.

<span id="an_6:19">6:19</span> Ur teẓrim ara daɣen belli Ṛṛuḥ iqedsen
yezdeɣ di lǧețțat-nwen ? Ṛṛuḥ-nni i wen-d-ițțunefken s ɣuṛ Sidi Ṛebbi ?
Ur teẓrim ara belli ur tețțalasem ara deg yiman-nwen ?

<span id="an_6:20">6:20</span> Axaṭer yella win i kkun-id-ifdan s wazal
d ameqqran, ɣlayen. Ɛuzzet ihi Sidi Ṛebbi di lǧețțat-nwen.

<span id="an_7:1">7:1</span> A nɛeddit tura ɣer isteqsiyen i yi-d-tefkam
di tebṛaț-nwen teqqaṛem : « D ayen yelhan i wergaz m'ur yezwiǧ ara ».

<span id="an_7:2">7:2</span> Ț-țideț, meɛna iwakken ur d-yețțili ara
leḥṛam, axiṛ-as i wergaz ad yesɛu tameṭṭut-is, axiṛ-as i tmeṭṭut
aț-țesɛu argaz-is.

<span id="an_7:3">7:3</span> Argaz ilaq ad ixdem lebɣi i tmeṭṭut-is,
tameṭṭut daɣen ilaq aț-țexdem lebɣi i wergaz-is.

<span id="an_7:4">7:4</span> Tameṭṭut ur tezmir ara aț-țexdem ayen tebɣa
s lǧețța-s, axaṭer lǧețța-s d ayla n wergaz-is ; argaz daɣen ur yezmir
ara ad yexdem ayen i gebɣa s lǧețța-s, axaṭer lǧețța-s d ayla n
tmeṭṭut-is.

<span id="an_7:5">7:5</span> Ur țemyagit ara gar-awen anagar ma yella
temsefhamem iwakken aț-țeǧǧem kra n wussan i tẓallit ; imiren ddukklet
daɣen ; ḥadret a wen-yaf Cciṭan abrid a kkun-iɣuṛṛ ma yella ur tezmirem
ara aț-țeṭṭfem iman-nwen.

<span id="an_7:6">7:6</span> Ayagi mačči d lameṛ i wen-fkiɣ, meɛna d
aweṣṣi kan i kkun-weṣṣaɣ.

<span id="an_7:7">7:7</span> Lemmer ufiɣ medden akk ad ilin am nekk,
meɛna mkul yiwen d acu i s-d-ifka Sidi Ṛebbi, wa yefka-yas-d aya, wayeḍ
yefka-yas-d ayen nniḍen.

<span id="an_7:8">7:8</span> I wid ur nezwiǧ ara akk-d tuǧǧal a sen-iniɣ
: yelha-yasen ma qqimen am nekk.

<span id="an_7:9">7:9</span> Meɛna m'ur zmiren ara ad ṭṭfen iman-nsen,
ad zewǧen axiṛ. Axiṛ-asen ad zewǧen wala aț-țecɛel tmes deg
wulawen-nsen.

<span id="an_7:10">7:10</span> A d-fkeɣ lameṛ i imasiḥiyen izewǧen ;
lameṛ-agi mačči s ɣuṛ-i i d-yekka meɛna s ɣuṛ Sidi Ṛebbi : tameṭṭut
izewǧen ur ilaq ara aț-țeǧǧ argaz-is :

<span id="an_7:11">7:11</span> ma yella teǧǧa-t, ur s-ilaq ara aț-țɛawed
zzwaǧ. Neɣ ma tebɣa, tezmer aț-țemṣalaḥ yid-es aț-țuɣal ɣuṛ-es. Argaz
daɣen ur s-ilaq ara ad yebru i tmeṭṭut-is.

<span id="an_7:12">7:12</span> Ɣef wayen yeɛnan wiyaḍ atan wayen zemreɣ
a d-iniɣ ( ayagi s ɣuṛ-i mačči s ɣuṛ Sidi Ṛebbi ) : ma yella yiwen seg
watmaten yesɛa tameṭṭut ur numin ara s Lmasiḥ yili teqbel aț-țeqqim d
wergaz-is, ur ilaq ara a s-yebru ;

<span id="an_7:13">7:13</span> akken daɣen, tameṭṭut tamasiḥit yesɛan
argaz ur numin ara s Lmasiḥ yili yeqbel ad iɛic yid-es, ur ilaq ara
aț-țerwel fell-as.

<span id="an_7:14">7:14</span> Axaṭer argaz ur numin ara s Lmasiḥ yella
seddaw leɛnaya n Sidi Ṛebbi s tmeṭṭut-is yumnen ; daɣen tameṭṭut ur
numin ara s Lmasiḥ, tella seddaw leɛnaya n Sidi Ṛebbi s wergaz-is yumnen
; neɣ m'ulac dderya-nsen ad țwaḥesben d dderya n leḥṛam nutni yellan d
dderya n leḥlal.

<span id="an_7:15">7:15</span> Ma yella win ur numin ara s Lmasiḥ yebɣa
ad imfaṛaq akk-d tmeṭṭut-is i gumnen, neɣ tameṭṭut ur numin ara s Lmasiḥ
tebɣa aț-țemfaṛaq akk-d wergaz-is i gumnen, zemren ad mfaṛaqen ;
argaz-nni neɣ tameṭṭut-nni yellan d imasiḥiyen ur d-yeqqim wayen i
ten-icerken, axaṭer Sidi Ṛebbi yessawel aɣ-d iwakken a nɛic di lehna.

<span id="an_7:16">7:16</span> Yerna amek i teẓriḍ kemm a tameṭṭut
tamasiḥit belli tzemṛeḍ aț-țselkeḍ argaz-im ? Neɣ amek i teẓriḍ kečč ay
argaz amasiḥi belli tzemreḍ aț-țselkeḍ tameṭṭut-ik ?

<span id="an_7:17">7:17</span> Ilaq yal yiwen ad yelḥu s wayen i
s-d-yefka Sidi Ṛebbi, yerna ad yeqqim di lḥala i deg yella asmi i
s-d-yessawel. Akka i țweṣṣiɣ tijmuyaɛ n watmaten meṛṛa.

<span id="an_7:18">7:18</span> Ma yella walebɛaḍ yuɣ-it lḥal yexten mi
s-d-yessawel Sidi Ṛebbi, ilaq ad yeqqim akken yella ; ma yella walebɛaḍ
ur yextin ara asmi i s-d-yessawel Sidi Ṛebbi, ur ilaq ara ad yexten.

<span id="an_7:19">7:19</span> Yexten walebɛaḍ neɣ ur yextin ara ulac
deg-s ; ayen yesɛan azal d ṭṭaɛa n lumuṛat n Sidi Ṛebbi.

<span id="an_7:20">7:20</span> Ilaq yal yiwen ad iqqim akken i t-id-yufa
lḥal asmi i s-d-yessawel Sidi Ṛebbi.

<span id="an_7:21">7:21</span> D akli i telliḍ asmi i k-d yessawel Sidi
Ṛebbi ? Ur țxemmim ara, meɛna ma tusa-yak-ed teswiɛt i deg tzemreḍ
aț-țesɛuḍ tilelli, faṛes-iț.

<span id="an_7:22">7:22</span> Axaṭer akli iwumi d-yessawel Sidi Ṛebbi
yuɣal d win yesɛan tilelli ɣer Sidi Ṛebbi ; daɣen amdan aḥuṛṛi iwumi
d-yessawel Sidi Ṛebbi yuɣal d akli n Lmasiḥ.

<span id="an_7:23">7:23</span> Yella win ixelṣen fell-awen iwakken
aț-țilim d iḥuṛṛiyen, ihi ur țțuɣalet ara d aklan n yemdanen.

<span id="an_7:24">7:24</span> Ihi ay atmaten, ilaq yal yiwen ad yeqqim
zdat Ṛebbi di lḥala i deg yella asmi s-d-yessawel.

<span id="an_7:25">7:25</span> Ɣef wayen yeɛnan wid ur nezwiǧ ara, Sidi
Ṛebbi ur d-yenni kra fell-asen, meɛna a wen-d-fkeɣ ṛṛay-iw, nekk yerra
Sidi Ṛebbi s ṛṛeḥma-ines tameqqrant d argaz yuklalen lețkal.

<span id="an_7:26">7:26</span> Ɣuṛ-i d ayen yelhan i wergaz ma yeqqim
akken yella ɣef ddemma n lweqt-agi iweɛṛen.

<span id="an_7:27">7:27</span> Ma txeḍbeḍ tameṭṭut, ur țnadi ara a
ț-teǧǧeḍ ; m'ur tezwiǧeḍ ara, ur țnadi ara aț-țzewǧeḍ.

<span id="an_7:28">7:28</span> Tura ma yella ilemẓi neɣ tilemẓit zewǧen,
ur wwin ara ddnub ; meɛna wid ara izewǧen ad ṛwun lḥif di tudert-nsen ;
nekk mačči d ayen i wen-mennaɣ.

<span id="an_7:29">7:29</span> Atan wayen ara d-iniɣ ay atmaten : lweqt
ițțazal, drus i d iqqimen, wid izewǧen ilaq ad ilin am akken ur zwiǧen
ara,

<span id="an_7:30">7:30</span> wid yețrun ad ilin am akken ur țrun ara,
wid ifeṛḥen am akken ur fṛiḥen ara, wid yețțaɣen am akken ur sɛin ara.

<span id="an_7:31">7:31</span> Wid yesɛan nnfeɛ di ddunit-agi am akken
ur t-sɛin ara axaṭer ddunit-agi ur tețdum ara.

<span id="an_7:32">7:32</span> Ur bɣiɣ ara aț-țesɛum iɣeblan. Win ur
nezwiǧ ara ițxemmim i lecɣal n Sidi Ṛebbi, yețnadi kan ad yeɛǧeb Sidi
Ṛebbi ;

<span id="an_7:33">7:33</span> meɛna win izewǧen yețxemmim ɣef lecɣal n
ddunit, yețnadi ad yeɛǧeb i tmeṭṭut-is ; ul-is yebḍa ɣef sin.

<span id="an_7:34">7:34</span> Akken daɣen tameṭṭut ur nezwiǧ ara akk-d
țlemẓit ; țxemmiment kan ɣef lecɣal n Sidi Ṛebbi iwakken ad ilint
ț-țiqeddacin-ines s lǧețțat-nsent d ṛṛuḥ-nsent ; meɛna tin izewǧen
tețxemmim kan ɣef lecɣal n ddunit d wamek ara teɛǧeb i wergaz-is.

<span id="an_7:35">7:35</span> Ur wen-d-nniɣ ara ayagi iwakken a
kkun-kellxeɣ meɛna nniɣ awen-t-id i nnfeɛ-nwen axaṭer bɣiɣ aț-țeddum s
tikli yelhan, yeṣfan iwakken aț-țeṭṭfem deg webrid n Lmasiḥ mačči deg
wayen nniḍen.

<span id="an_7:36">7:36</span> Ma yella yiwen deg-wen ixḍeb tilemẓit,
iḥar ur yezmir ara ad yeṛǧu ibɣa ad yezweǧ, mačči d ddnub ma yella
yezweǧ.

<span id="an_7:37">7:37</span> Ma d win iqesden s lebɣi-ines ur izeggej
ara, ma yella yeẓra iman-is yezmer ad iṣbeṛ, d ayen yelhan i gexdem m'ur
yezwiǧ ara.

<span id="an_7:38">7:38</span> Ihi win izewǧen d ayen yelhan i gexdem,
win ur nezwiǧ ara d ayen yelhan nezzeh.

<span id="an_7:39">7:39</span> Tameṭṭut izewǧen ur tezmir ara aț-țeǧǧ
argaz-is skud mazal-it yedder, ma yella yemmut, tezmer aț-țɛiwed zzwaǧ d
win tebɣa, acu kan ilaq ad yili d win yumnen s Lmasiḥ am nețțat.

<span id="an_7:40">7:40</span> Meɛna ɣef wakken walaɣ ma teqqim akken
tella axiṛ-as, akken ara tesɛu lehna. Wagi d ṛṛay-iw nekk, ẓriɣ ula d
nekk sɛiɣ Ṛṛuḥ n Sidi Ṛebbi.

<span id="an_8:1">8:1</span> Ɣef wayen yeɛnan aksum n iseflawen i zellun
i ssadaț, ț-țideț nesɛa akk tamusni, d acu kan tamusni tețțawi-d zzux i
wemdan ma d leḥmala tesnernay deg-nneɣ laman ɣer Sidi Ṛebbi.

<span id="an_8:2">8:2</span> Win i gɣilen yessen kra, urɛad yessin akken
i glaq ad yissin.

<span id="an_8:3">8:3</span> Meɛna win iḥemmlen Sidi Ṛebbi, Sidi Ṛebbi
yessen-it.

<span id="an_8:4">8:4</span> Ɣef wayen yeɛnan učči n weksum n iseflawen
immezlen i lmeṣnuɛat, neẓra belli lmeṣnuɛat-agi ț-țikellax di ddunit
axaṭer anagar yiwen n Ṛebbi i gellan.

<span id="an_8:5">8:5</span> Ɣas qqaṛen aṭas n iṛebbiten i gellan ama
deg igenni ama di lqaɛa, ( axaṭer ț-țideț ! Ɣuṛ-sen aṭas n iṛebbiten
akk-d ssadaț i gellan)

<span id="an_8:6">8:6</span> meɛna ɣuṛ-nneɣ nukni yiwen kan n Ṛebbi i
gellan, d baba-tneɣ Ṛebbi i d-ixelqen kullec, ula d nukni d ayla-s.
Anagar yiwen n Ssid i gellan : d Ɛisa Lmasiḥ, yis i d-yella kullec, yis
daɣen i nedder ula d nukni.

<span id="an_8:7">8:7</span> Aṭas ur nessin ara tideț-agi. Kra deg-sen i
guɣen tannumi tețțen asfel n ssadaț, tura m'ara tețțen aksum-nni mazal
ḥesben-t belli ț-țideț d asfel i ssadaț, neyya-nsen tenquqel, ḥussen am
akken denben.

<span id="an_8:8">8:8</span> Mačči d ayen yețmaččan ara ɣ-iqeṛṛben ɣer
Ṛebbi. Ma nečča ur nerbiḥ acemma, m'ur nečči ara, ur aɣ-iṛuḥ wacemma.

<span id="an_8:9">8:9</span> Ihi, ɛasset iman-nwen, ɣuṛ-wat tilelli-agi
i tesɛam aț-țili d ugur n uɣelluy i wid ur neǧhid ara di liman.

<span id="an_8:10">8:10</span> Axaṭer ma iwala-k-id walebɛaḍ ur neǧhid
ara di liman, teqqimeḍ aț-țeččeḍ di lemqam kečč yessnen, amek tebɣiḍ ur
yețṛuḥu ara ula d nețța ad yečč seg iseflawen-agi yemmezlen i ssadaț ?

<span id="an_8:11">8:11</span> S wakka ihi, gma-k-agi ur neǧhid ara di
liman, i ɣef yemmut Lmasiḥ , ɣef ddemma n tmusni-nni inek i geɣli.

<span id="an_8:12">8:12</span> Imi tețțawim ddnub ɣef watmaten-nwen,
tessexṛabem neyya-nsen, tdenbem ɣer Lmasiḥ.

<span id="an_8:13">8:13</span> Ihi ma yella ayen ara ččeɣ izmer ad
isseɣli gma di ddnub, ur țțuɣaleɣ ara maḍi ad ččeɣ lmakla-nni iwakken ur
iɣelli ara.

<span id="an_9:1">9:1</span> Eɛni ur zmireɣ ara ad xedmeɣ akken bɣiɣ ?
Mačči d amceggeɛ n Lmasiḥ i lliɣ ? Ur ẓriɣ ara Sidna Ɛisa s wallen-iw ?
Mačči d nekk i kkun-id-yewwin ɣer webrid n Lmasiḥ ?

<span id="an_9:2">9:2</span> Ɣas wiyaḍ ur iyi-ḥsiben ara d amceggeɛ n
Lmasiḥ, kunwi ur tezmirem ara a yi-tnekkṛem axaṭer ț-țikli-nwen akk-d
Lmasiḥ i d țbut belli d Lmasiḥ i yi-d-iceggɛen.

<span id="an_9:3">9:3</span> Atan wamek țțaraɣ i wid i yi-sseḍlamen :

<span id="an_9:4">9:4</span> Eɛni ur aɣ-d-iṣaḥ ara a nečč, a nsew nukni
s imceggɛen n Lmasiḥ ?

<span id="an_9:5">9:5</span> Ur sɛiɣ ara lḥeqq ad aɣeɣ tameṭṭut yumnen
aț-țețțeddu yid-i am akken xeddmen imceggɛen nniḍen n Lmasiḥ d watmaten
n Ssid-nneɣ akk-d Buṭrus ?

<span id="an_9:6">9:6</span> Neɣ d nekk kan akk-d Barnabas i glaqen a
nexdem iwakken a d-nawi aɣṛum nneɣ ?

<span id="an_9:7">9:7</span> Yella wass i deg teslam belli aɛeskṛi ilaq
ad ixelleṣ m'ara d-isɛeddi lɛeskeṛ ? Anwa ara yeẓẓun tara ur itețț ara
si tẓuṛin-is ? Ulac ameksa ara yeksen taqeḍɛit n lmal ur itess ara seg
uyefki-ines !

<span id="an_9:8">9:8</span> Tɣilem ayen akka i d-nniɣ d lɛaddat kan n
yemdanen ? Ula d ccariɛa n Musa tenna-t-id !

<span id="an_9:9">9:9</span> Axaṭer yura deg-s : Ur țțarat ara takmamt i
wezger isserwaten nneɛma. Eɛni ɣef yezgaren i gḥebbeṛ Sidi Ṛebbi ?

<span id="an_9:10">9:10</span> Mačči ɣef ddemma-nneɣ i d-yenna akka ?
Ț-țideț, d nukni iwumi i d-yenna imeslayen-agi ! Ilaq win ikerrzen akk-d
win yesserwaten ad ssirmen ad sɛun amur di lɣella nsen.

<span id="an_9:11">9:11</span> Ma yella nezreɛ gar-awen lbaṛakat n Sidi
Ṛebbi, d acu yellan ma nemger-ed s ɣuṛ-wen tamɛict nneɣ ?

<span id="an_9:12">9:12</span> Ma twalam d lḥeqq ad meggren wiyaḍ s
ɣuṛ-wen, eɛni mačči d nukni i gezwaren ? Meɛna nukni ur d-newwi acemma s
ɣuṛ-wen, neṣbeṛ i kullec iwakken ur d-yețțili ara wugur i lexbaṛ n lxiṛ
n Lmasiḥ.

<span id="an_9:13">9:13</span> Ur teẓrim ara belli wid ixeddmen di
lǧameɛ țțawin-d lqut-nsen si lǧameɛ ? Ur teẓrim ara daɣen lmuqeddem
izellun i Ṛebbi iseflawen i d-țțawin medden, yețțawi amur-is seg-sen ?

<span id="an_9:14">9:14</span> Akka daɣen i d-yenna Sidi Ṛebbi ɣef wid
yețbecciṛen lexbaṛ n lxiṛ : di lexbaṛ n lxiṛ ara afen tamɛict-nsen.

<span id="an_9:15">9:15</span> Nekk ur d-wwiɣ acemma s ɣuṛ wen, yerna ur
d-uriɣ ara tabṛaț-agi iwakken a d-ssutreɣ lḥeqq-iw. Axiṛ-iyi lmut wala a
yi-tețwakkes sebba-agi s wayes zemreɣ ad zuxxeɣ !

<span id="an_9:16">9:16</span> Ur țzuxxuɣ ara s yiman-iw imi țbecciṛeɣ
lexbaṛ n lxiṛ, axaṭer țwaḥeṛseɣ ad beccṛeɣ. A nnger-iw ma yella ur
beccṛeɣ ara lexbaṛ-agi n lxiṛ.

<span id="an_9:17">9:17</span> Lemmer xedmeɣ ayagi s lebɣi w sɛiɣ lḥeqq
ad ssutreɣ lexlaṣ, meɛna d Ṛebbi i gḥettmen fell-i ccɣel-agi.

<span id="an_9:18">9:18</span> Ihi, d acu i d lexlaṣ-iw ? D lfeṛḥ i
seɛɛuɣ m'ara țbecciṛeɣ lexbaṛ n lxiṛ mbla lexlaṣ, mbla ma ssutreɣ
lḥeqq-iw.

<span id="an_9:19">9:19</span> Ɣas akken ur iyi-ițțalas yiwen kra ; rriɣ
iman-iw d aqeddac n mkul yiwen iwakken a d-rebḥeɣ aṭas n yemdanen ɣer
webrid n Lmasiḥ.

<span id="an_9:20">9:20</span> Ɣer wat Isṛail rriɣ iman-iw am nutni
iwakken a ten-id-rebḥeɣ ; ɣer wid yettabaɛen ccariɛa n Musa tebɛeɣ-ț am
nutni iwakken a ten id rebḥeɣ, ɣas akken nekk ur lliɣ ara seddaw leḥkum
n ccariɛa.

<span id="an_9:21">9:21</span> Akka daɣen, m'ara iliɣ d wid ur nesɛi ara
ccariɛa, țțaraɣ iman-iw am akken ur sɛiɣ ara ccariɛa iwakken a
ten-id-rebḥeɣ, ɣas akken nekk sɛiɣ ccariɛa n Ṛebbi imi d Lmasiḥ i d
ccariɛa-inu.

<span id="an_9:22">9:22</span> Akk-d wid ur neǧhid ara deg webrid n
Ṛebbi, uɣaleɣ am nutni, iwakken a ten-id-rebḥeɣ. Ddiɣ di lebɣi i mkul
yiwen iwakken a d-sellkeɣ ulamma kra deg-sen.

<span id="an_9:23">9:23</span> Ayagi akk xedmeɣ-t ɣef ddemma n lexbaṛ n
lxiṛ iwakken ad sɛuɣ yid-sen amur.

<span id="an_9:24">9:24</span> Di temzizla, teẓram belli deg wid akk
yețțazalen deg wannar, yiwen kan ara irebḥen. Azzlet ihi iwakken
aț-țrebḥem.

<span id="an_9:25">9:25</span> Wid akk yețțazzalen, m'ara heggin
iman-nsen i tazzla, țɛețțiben iman-nsen ; nutni țenɛețțaben iwakken a
d-rebḥen cciɛa ur nețdum ara ; lameɛna nukni, nețɛețțib iman-nneɣ
iwakken a nerbeḥ cciɛa ara idumen.

<span id="an_9:26">9:26</span> Nekk ur țțazzaleɣ ara am win ur nesɛi ara
lmeqsud, ur kkateɣ ara deg ubeḥri m'ara țnaɣeɣ ;

<span id="an_9:27">9:27</span> meɛna țɛețțibeɣ lǧețța-inu, țɛassaɣ ɣef
yiman-iw, axaṭer uggadeɣ nekk i gbeccṛen i wiyaḍ ad ɣliɣ, ad uɣaleɣ ɣer
deffir.

<span id="an_10:2">10:2</span> Țwaɣeḍsen meṛṛa di tagut-nni akk-d lebḥeṛ
mi ddan d Sidna Musa,

<span id="an_10:3">10:3</span> ččan meṛṛa seg yiwen lqut n Sidi Ṛebbi,

<span id="an_10:4">10:4</span> swan meṛṛa si lɛinseṛ n Sidi Ṛebbi i
d-iteffɣen seg wezṛu iteddun yid-sen ; azṛu-agi, d Lmasiḥ.

<span id="an_10:5">10:5</span> Ɣas akken, aṭas seg-sen ur neɛǧib ara i
Sidi Ṛebbi ; daymi i ten-tesseɣli lmut yiwen yiwen mi teddun deg
unezṛuf.

<span id="an_10:6">10:6</span> Ayen akken yedṛan yid-sen yeqqim-ed d
lemtel i nukni, iwakken ur neṭṭamaɛ ara ayen n diri am akken i
t-ṭṭamaɛen nutni.

<span id="an_10:7">10:7</span> Ur țțilit ara am wid iɛebbden ssadaț
akken xeddmen kra seg-sen, axaṭer am akken yura di tira iqedsen : Yeqqim
wegdud ad yečč ad isew, mi gfukk yekker ad yezhu, ad yecḍeḥ.

<span id="an_10:8">10:8</span> Ur ilaq ara daɣen a nxeddem leɛṛuṛ akken
xeddmen kra seg-sen ; daymi deg yiwen n wass, lmut tečča tlata uɛecrin
alef deg-sen.

<span id="an_10:9">10:9</span> Ur ilaq ara a njeṛṛeb Sidi Ṛebbi akken i
xedmen kra seg-sen, dɣa qqsen-ten izerman mmuten.

<span id="an_10:10">10:10</span> Ur ilaq ara daɣen aț-țesmermugem ɣef
wiyaḍ akken i smermugen kra deg-sen armi i ten-issenger lmelk n lmut.

<span id="an_10:11">10:11</span> Timsalin-agi akk i sen-yedṛan,
qqiment-ed d lemtel, uran-tent di tektabin, iwakken a nelmed nukni
ițɛicin di taggara n lweqt.

<span id="an_10:12">10:12</span> Ihi win iḥesben iman-is ibedd, ad iɛass
iman-is ur iɣelli ara !

<span id="an_10:13">10:13</span> Ayen akk s wayes i tețwajeṛbem, d ayen
iwumi yezmer wemdan. Sidi Ṛebbi d bab n lḥeqq, ur yețɛemmid ara
aț-țenɛețțabem sennig n tezmert nwen ; lameɛna m'ara tețwajeṛbem, a
wen-d-yefk tazmert s wayes ara tṣebṛem d wamek ara teffɣem si tegniț n
leɛtab.

<span id="an_10:14">10:14</span> A wid eɛzizen fell-i, xḍut i weɛbad n
ssadaț.

<span id="an_10:15">10:15</span> Țmeslayeɣ awen-d am akken țmeslayen i
wid ifehmen ; meyyzet kunwi s yiman nwen ɣef wayen i d-qqaṛeɣ.

<span id="an_10:16">10:16</span> Taqbuct n lbaṛaka i ɣef nețḥemmid Sidi
Ṛebbi, eɛni ur aɣ-tecrik ara d idammen n Lmasiḥ ? Neɣ aɣṛum i nbeṭṭu, ur
aɣ-yecrik ara d lǧețța n Lmasiḥ ?

<span id="an_10:17">10:17</span> Imi yiwen n weɣṛum kan i gellan, ula d
nukni nuɣal d yiwet n lǧețța ɣas akken deg waṭas yid-nneɣ i nella,
axaṭer ntețț meṛṛa seg yiwen n weɣṛum.

<span id="an_10:18">10:18</span> Walit at Isṛail : wid itețțen seg
weksum n iseflawen s weɛbad-nsen, eɛni ur criken ara d wemkan-nni anda i
zellun iseflawen ?

<span id="an_10:19">10:19</span> Eɛni bɣiɣ a d-iniɣ belli asfel nni i
zellun i ssadaț neɣ ssadaț-nni i țɛebbiden, sɛan lqima ? Xaṭi !

<span id="an_10:20">10:20</span> Meɛna ayen akka i zellun, zellun-t i
leǧnun mačči i Ṛebbi ; nekk ur bɣiɣ ara aț-țcerkem akk-d leǧnun.

<span id="an_10:21">10:21</span> Ur tezmirem ara aț-țeswem si teqbuct n
Sidi Ṛebbi ma yella tețțessem si țeqbuct n leǧnun ; ur tezmirem ara
daɣen aț-țeččem seg weɣṛum n Sidi Ṛebbi ma yella tețțețțem seg weɣṛum n
leǧnun.

<span id="an_10:22">10:22</span> Eɛni nebɣa a nejreḥ ul n Sidi Ṛebbi a
d-neskker tismin-is ? Tɣilem tazmert-nneɣ teɣleb tazmert-is ?

<span id="an_10:23">10:23</span> Llan wid i d-iqqaṛen : kullec d leḥlal
fell-aɣ ; ț-țideț, lameɛna ur aɣ-yenfiɛ ara kullec. Kullec
yețțusemmeḥ-aɣ a t-nexdem, meɛna mačči d kullec i ɣ-yesnernayen di
liman.

<span id="an_10:24">10:24</span> Ur țnadit ara ɣef nnfeɛ-nwen kan kunwi,
meɛna nadit ɣef nnfeɛ n wiyaḍ.

<span id="an_10:25">10:25</span> Tzemrem aț-țeččem kra n wayen i
gețnuzun di ssuq mbla asteqsi, mbla ma tḥebbṛem,

<span id="an_10:26">10:26</span> axaṭer ddunit d wayen akk yellan deg-s
d ayla n Sidi Ṛebbi.

<span id="an_10:27">10:27</span> Ma yeɛṛeḍ-ikkun walebɛad ur numin ara s
Lmasiḥ, ma tṛuḥem, ččet s neyya mbla asteqsi seg wayen akk ara
wen-d-issers.

<span id="an_10:28">10:28</span> Lameɛna ma yella win i wen-d yennan :
ayagi yețțunefk d asfel i ssadaț, ur tețțet ara imi i kkun-id ixebbeṛ
iwakken ad yethedden wul nwen.

<span id="an_10:29">10:29</span> Mačči ɣef neyya-nwen i d-nniɣ akka,
lameɛna ɣef neyya n win i kkun-iɛerḍen. Ihi, acimi ara ǧǧeɣ wayeḍ ad
idebbeṛ deg ixemmimen-iw ?

<span id="an_10:30">10:30</span> Ma yella ḥemdeɣ Sidi Ṛebbi ɣef wayen i
tețțeɣ, acimi ara ǧǧeɣ iman-iw ad wwten deg-i ɣef lqut i ɣef ḥemdeɣ Sidi
Ṛebbi ?

<span id="an_10:31">10:31</span> Ama teččam neɣ teswam, neɣ kra n wayen
ara txedmem, xedmet-eț i tmanegt n Sidi Ṛebbi.

<span id="an_10:32">10:32</span> Ur țțilit ara d sebba n uɣelluy ama i
wat Isṛail, ama i iyunaniyen, ama i tejmaɛt n Sidi Ṛebbi,

<span id="an_10:33">10:33</span> am akken xeddmeɣ nekk, țnadiɣ a
wen-ɛeǧbeɣ meṛṛa di kullec, ur țqellibeɣ ara ɣef nnfeɛ-inu, meɛna
țqellibeɣ ɣef nnfeɛ n umur ameqqran deg-wen iwakken ad țwaselken.

<span id="an_11:1">11:1</span> Ɛandet-iyi am akken țɛanadeɣ Lmasiḥ.

<span id="an_11:2">11:2</span> A kkun-ibarek Ṛebbi imi i
yi-d-tețmektayem di yal taswiɛt yerna teṭṭafaṛem lewṣayat-iw akken i
wen-tent-slemdeɣ.

<span id="an_11:3">11:3</span> Meɛna, tura bɣiɣ aț-țissinem ayagi :
Lmasiḥ d aqeṛṛuy ɣef yemdanen meṛṛa, argaz d aqeṛṛuy ɣef tmeṭṭut-is, ma
d Ṛebbi d aqeṛṛuy ɣef Lmasiḥ.

<span id="an_11:4">11:4</span> Ihi argaz ara yesburren ɣef wuqeṛṛuy-is
m'ara ideɛɛu ɣer Sidi Ṛebbi neɣ m'ara d-yețxebbiṛ ayen i s-d-yusan s ɣuṛ
Sidi Ṛebbi, yekkes lḥeṛma ɣef Ssid-is.

<span id="an_11:5">11:5</span> Lameɛna tameṭṭut ara ideɛɛun ɣer Sidi
Ṛebbi neɣ ara yețxebbiṛen s wayen i s-d-yețțasen s ɣuṛ Ṛebbi aqeṛṛuy-is
ɛeryan, tekkes lḥeṛma ɣef wergaz-is, am akken d aqeṛṛuy-is i tseṭṭel.

<span id="an_11:6">11:6</span> Axaṭer ma yella tameṭṭut ur tesburr ara,
aț-țegzem kan ihi acebbub-is am wergaz ; lameɛna ma yella d lɛib i
tmeṭṭut aț-țegzem neɣ aț-țṣeṭṭel acebbub-is, ilaq ihi aț-țɣum
aqeṛṛuy-is.

<span id="an_11:7">11:7</span> Argaz ur ilaq ara ad yesburr i wqeṛṛuy-is
axaṭer d nețța i gmetlen Sidi Ṛebbi d lɛaḍima-ines ; ma ț-țameṭṭut d
nețțat i d lḥeṛma n wergaz.

<span id="an_11:8">11:8</span> Axaṭer mačči d argaz i d- yețwaxelqen si
tmeṭṭut meɛna ț- țameṭṭut i d-ițwaxelqen seg wergaz.

<span id="an_11:9">11:9</span> Daɣen argaz ur d-yețwaxleq ara i tmeṭṭut,
lameɛna ț-țameṭṭut i d yețwaxelqen i wergaz.

<span id="an_11:10">11:10</span> Daymi ɣef ddemma n lmalayekkat,
tameṭṭut ilaq aț-țesbur i uqeṛṛuy-is iwakken a d-tbeggen belli seddaw
lḥekma n wergaz-is i tella.

<span id="an_11:11">11:11</span> Ɣer Sidi Ṛebbi, ur d-tețțili tmeṭṭut
mbla argaz, ur d-yețțili wergaz mbla tameṭṭut.

<span id="an_11:12">11:12</span> Axaṭer akken i d-tețwaxleq tmeṭṭut seg
wergaz, akken daɣen i d-yețlal wergaz si tmeṭṭut, ayagi akk s ɣuṛ Sidi
Ṛebbi i d-yekka.

<span id="an_11:13">11:13</span> Meyzet kunwi s yiman-nwen : eɛni
yelha-yas i tmeṭṭut aț-țdeɛɛu ɣer Sidi Ṛebbi mbla ma tesburr ɣef
wuqeṛṛuy-is ?

<span id="an_11:14">11:14</span> Ula d lɛadda n yemdanen tesselmed awen
belli d lɛib i wergaz ma iṛebba acebbub-is.

<span id="an_11:15">11:15</span> Lameɛna d ccan ameqqran i tmeṭṭut ma
tṛebba acebbub-is axaṭer yețțunefk-as-d iwakken a s-yili d asburru.

<span id="an_11:16">11:16</span> Ma yella win yebɣan ad iɣanen ɣef
wannect-agi, ilaq ad iẓer belli nukni ur nesɛi ara tannumi-agi, ur telli
ara daɣen di tejmuyaɛ n Sidi Ṛebbi.

<span id="an_11:17">11:17</span> A nɛeddi tura ɣer wayen nniḍen. Ɣef

<span id="an_11:18">11:18</span> Sliɣ belli m'ara tețnejmaɛem yețțili

<span id="an_11:20">11:20</span> M'ara tedduklem ɣer imensi n usmekti,

kkun cekkṛeɣ ? Ur kkun-țcekkiṛeɣ ara ɣef wayagi !

<span id="an_11:23">11:23</span> D ayen i yi-d-ițțunefken s ɣuṛ Sidi

yeddem-ed aɣṛum,

Wagi d lgețța-w yețțunefken fell awen, xeddmet ayagi iwakken a yi-d
țmektayem. »

<span id="an_11:25">11:25</span> Daɣen, mi fukken učči, yeddem ed

ayagi iwakken a yi-d-tețmektayem. »

<span id="an_11:26">11:26</span> Axaṭer kra n tikkelt i deg ara teččem

Lmasiḥ alamma d ass i deg ara d-yuɣal.

<span id="an_11:27">11:27</span> Daymi win ara yeččen seg weɣṛum-agi,
ara yeswen si teqbuct-agi n usmekti n Lmasiḥ, m'ur sen-yefki ara lqima,
yewwi ddnub ɣer yiri-s ɣef ddemma n lǧețța akk-d idammen n Lmasiḥ.

<span id="an_11:28">11:28</span> Ihi mkul yiwen ad ixemmem deg yiman-is
uqbel ad yečč seg weɣṛum-nni neɣ ad isew si teqbuct-nni.

<span id="an_11:29">11:29</span> Axaṭer ma yella yiwen deg-wen yečča neɣ
yeswa tiremt n usmekti n Lmasiḥ, mbla ma yefka lqima i lǧețța n Lmasiḥ,
yewwi-d lɛiqab i yiman-is.

<span id="an_11:30">11:30</span> Ɣef wayagi i llan gar-awen aṭas n wid
ihelken akk-d ineɛyuba, yerna aṭas deg-sen i gemmuten.

<span id="an_11:31">11:31</span> Lemmer nețḥadar ɣef yiman nneɣ, tili ur
d-nețțawi ara lɛiqab n Sidi Ṛebbi fell-aneɣ.

<span id="an_11:32">11:32</span> S wakka m'ara ɣ-iɛaqeb Sidi Ṛebbi,
yețṛebbi-yaɣ iwakken ur aɣ yețḥasab ara d wat n ddunit.

<span id="an_11:33">11:33</span> Ihi ay atmaten, asmi ara tennejmaɛem
iwakken aț-țeččem, myeṛǧut wway gar-awen.

<span id="an_11:34">11:34</span> Ma llan wid yelluẓen, ad ččen uqbel deg
wexxam-nsen iwakken ur d-tesseɣlayem ara lɛiqab n Sidi Ṛebbi fell-awen
m'ara tennejmaɛem. Timsalin nniḍen a tent-fruɣ m'ara n-aseɣ.

<span id="an_12:1">12:1</span> Ay atmaten, ur bɣiɣ ara aț-țeqqimem mbla
tamusni ɣef wayen yeɛnan ayen i d-yețțak Ṛṛuḥ iqedsen.

<span id="an_12:2">12:2</span> Teẓram belli asm'akken ur tuminem ara s
Lmasiḥ, țțawwin kkun wiyaḍ tɛebbdem lmeṣnuɛat ur neneṭṭeq ara.

<span id="an_12:3">12:3</span> Tura ilaq aț-țeẓrem belli yiwen ur izmir
a d-yini : « ad yețwanɛel Ɛisa Lmasiḥ » ma yella s Ṛṛuḥ iqedsen i
gețmeslay. Daɣen yiwen ur izmir ad iched belli Ɛisa Lmasiḥ d Ssid-is, ma
yella mačči s Ṛṛuḥ iqedsen i t-id-yenna.

<span id="an_12:4">12:4</span> Llan aṭas n leṣnaf n tukciwin n Ṛebbi,
lameɛna yiwen n Ṛṛuḥ kan i gellan ;

<span id="an_12:5">12:5</span> aṭas n leṣnaf n leqdic i gellan meɛna
yiwen n Ssid-nneɣ kan i nesɛa ;

<span id="an_12:6">12:6</span> aṭas n leṣnaf n lecɣal i gellan, meɛna
Ṛebbi d yiwen, d nețța i ten-isedduyen akk deg-nneɣ meṛṛa.

<span id="an_12:7">12:7</span> Yal yiwen tețțunefk-as-d tukci n Ṛṛuḥ
iqedsen i nnfeɛ-nneɣ meṛṛa.

<span id="an_12:8">12:8</span> Ihi Ṛṛuḥ iqedsen yefka-d i yiwen ad
immeslay s ṣṣwab d lefhama, yefka-d i wayeḍ ad immeslay s tmusni.

<span id="an_12:9">12:9</span> Ṛṛuḥ-agi yefka daɣen i wayeḍ liman
iǧehden, i wayeḍ tazmert s wayes ara yesseḥlu imuḍan ;

<span id="an_12:10">12:10</span> yefka daɣen i wayeḍ tazmert ad ixdem
lbeṛhanat, ma d wayeḍ ad ițxebbiṛ s wayen i s-d-yețțassen s ɣuṛ Sidi
Ṛebbi ; wayeḍ daɣen yefka-yas-ed tazmert s wayes ara yeɛqel ayen i
d-ițțasen s ɣuṛ Ṛṛuḥ n Sidi Ṛebbi d wayen i d-ițțasen s ɣuṛ leṛwaḥ n
diri ; wayeḍ yețțunefk-as-d ad immeslay timeslayin ur nețwassen ara ;
wayeḍ daɣen yețțunefk-as-d a d-yessefhem timeslayin-agi ur nețwassen
ara.

<span id="an_12:11">12:11</span> Ayagi akk itekk-ed seg yiwen n Ṛṛuḥ.
Ṛṛuḥ iqedsen yețțak-ed ayagi meṛṛa akken yebɣa, i yal yiwen deg nneɣ.

<span id="an_12:12">12:12</span> Lǧețța n wemdan d yiwet, ɣas akken
tesɛa aṭas n lemfaṣel. Lemfaṣel-agi ɣas deg waṭas yid-sen, d yiwet n
lǧețța i llan ; akken daɣen i gella Lmasiḥ.

<span id="an_12:13">12:13</span> Axaṭer s yiwen n Ṛṛuḥ kan i nețwaɣḍes
meṛṛa iwakken a nili d yiwet n lǧețța ; ama d at Isṛail neɣ d
iyunaniyen, ama d aklan neɣ d iḥeṛṛiyen, seg yiwen n Ṛṛuḥ i neswa meṛṛa.

<span id="an_12:14">12:14</span> Lǧețța ur tebni ara ɣef yiwen n
lmefṣel, lameɛna tebna ɣef waṭas n lemfaṣel.

<span id="an_12:15">12:15</span> Ma yenna-d uḍar : nekk mačči d afus ur
țekkiɣ ara di lǧețța, eɛni imi ur yelli ara d afus ur ițekka ara ula d
nețța di lǧețța ?

<span id="an_12:16">12:16</span> Ma yella yenna-d umeẓẓuɣ : nekk mačči
ț-țiṭ, ur țekkiɣ ara di lǧețța, eɛni ț-țideț m'ur yelli ara ț-țiṭ ur
ițekka ara ula d nețța di lǧețța ?

<span id="an_12:17">12:17</span> Ma yella lǧețța meṛṛa ț-țiṭ, amek ara
tili tmesliwt ? Neɣ ma yella d ameẓẓuɣ kan i tella, amek ara yili
wesriḥi ?

<span id="an_12:18">12:18</span> Sidi Ṛebbi yebna mkul lmefṣel di lǧețța
akken i s-yehwa.

<span id="an_12:19">12:19</span> Ma yella lǧețța meṛṛa d yiwen n
lmefṣel, anda-ț ihi lǧețța ?

<span id="an_12:20">12:20</span> S wakka ihi aṭas n lemfaṣel i gellan,
meɛna lǧețța yiwet.

<span id="an_12:21">12:21</span> Tiṭ ur tezmir ara aț-țini i ufus : ur
k-uḥwaǧeɣ ara ! Aqeṛṛuy ur yezmir ara ad yini i uḍaṛ : ur k-uḥwaǧeɣ ara.

<span id="an_12:22">12:22</span> Lemfaṣel-nni n lǧețța i d-yețbanen d
imecṭaḥ, nefɛen aṭas.

<span id="an_12:23">12:23</span> Lemfaṣel n lǧețța iwumi ur nefki ara
lqima, d nutni i nețḥadar akteṛ n wiyaḍ ; ihi d lemfaṣel ur nezmir ara a
d-nebder i nețḥadar aṭas,

<span id="an_12:24">12:24</span> ma d wid yesɛan lqima, ur uḥwaǧen ara
lemḥadra. Sidi Ṛebbi yebna lǧețța akken ara izegged lqima i lemfaṣel
ixuṣṣen,

<span id="an_12:25">12:25</span> iwakken ur d-yețțili ara lxilaf ger
lemfaṣel, meɛna ad țemḥadaren wway gar-asen.

<span id="an_12:26">12:26</span> Ma yella yiwen n lmefṣel yețwaqṛeḥ,
wiyaḍ meṛṛa ad țwaqeṛḥen yid-es ; ma yella yețțunefk-as ccan i wayeḍ,
wiyaḍ akk ad feṛḥen yid-es.

<span id="an_12:27">12:27</span> Kunwi meṛṛa d lǧețța n Lmasiḥ, yal
yiwen deg-wen d lmefṣel di lǧețța-s, mkul yiwen deg umkan-is.

<span id="an_12:28">12:28</span> Di tejmaɛt n Lmasiḥ, Sidi Ṛebbi yesbedd
uqbel ṛṛusul, deffir-nsen wid i d-yețxebbiṛen s ɣuṛ Sidi Ṛebbi, rnan-d
wid yesselmaden, rnan-d daɣen wid yesɛan tazmert ad xedmen lbeṛhanat,
wid yesɛan tazmert ad sseḥlun imuḍan, wid ara iɛiwnen wiyaḍ, wid ara
yilin d imdebbṛen akk-d wid ara yemmeslayen timeslayin ur nețwassen ara.

<span id="an_12:29">12:29</span> Eɛni țțuceggɛen-d meṛṛa d ṛṛusul ? Neɣ
țxebbiṛen-d akk s ɣuṛ Sidi Ṛebbi ? Sselmaden meṛṛa ? Xeddmen meṛṛa
lbeṛhanat ?

<span id="an_12:30">12:30</span> Eɛni sɛan meṛṛa tazmert ad sseḥlun
imuḍan, neɣ ad mmeslayen meṛṛa timeslayin ur nețwassen ara ? Neɣ zemren
akk a d-sfehmen timeslayin-agi ?

<span id="an_12:31">12:31</span> Ihi a wen-d-iniɣ : nadit axiṛ ɣef
tikciwin-nni inefɛen aṭas. A wen d-mleɣ tura abrid yifen akk iberdan.

<span id="an_13:1">13:1</span> Ɣas zemreɣ ad mmeslayeɣ timeslayin n

ad iliɣ am ṭṭbel yeddendunen neɣ nnaqus yeṭṭenṭunen.

<span id="an_13:2">13:2</span> Ɣas ma yețțunefk-iyi-d ad țxebbiṛeɣ s ɣuṛ

ad sɛuɣ liman s wayes i zemreɣ ad iniɣ i wedrar qleɛ iman-ik syagi
tṛuḥeḍ ɣer wemkan ihina, m'ur sɛiɣ ara leḥmala ɣer wiyaḍ, nekk d ulac.

<span id="an_13:3">13:3</span> Ɣas daɣen ad feṛqeɣ ayen akk sɛiɣ i wid
yelluẓen, ad rnuɣ ad sebbleɣ tudert-iw alamma d lmut ɣef ddemma n wiyaḍ,
m'ur sɛiɣ ara leḥmala deg ul-iw, ayagi ur iyi-infiɛ ara.

<span id="an_13:4">13:4</span> Leḥmala n tideț tețțawi-d ṣṣbeṛ, leḥmala
tețɛawan, leḥmala ur tesɛi ara tismin, ur tesɛi ara zzux, win yesɛan
leḥmala deg ul-is ur yețcekkiṛ ara iman-is,

<span id="an_13:5">13:5</span> Win yesɛan leḥmala n tideț ur ițnadi ara
ad ixdem cceṛ, ur yețnadi ara ɣef nnfeɛ-ines, ur izeɛɛef ur yețțaṭṭaf
cceḥnat ;

<span id="an_13:6">13:6</span> Win yesɛan leḥmala ur ifeṛṛeḥ ara s
lbaṭel meɛna ifeṛṛeḥ s lḥeqq;

<span id="an_13:7">13:7</span> leḥmala tețsamaḥ kullec, tețțamen,
tessaram tsebbeṛ i kullec.

<span id="an_13:8">13:8</span> Axebbeṛ n wayen i d-yețțasen s ɣuṛ Ṛebbi
ad ifak, timeslayin ur nețwassen ara ad ḥebsent, tamusni aț-țefnu, ma d
leḥmala n tideț werǧin aț-țfak ;

<span id="an_13:9">13:9</span> axaṭer tamusni-nneɣ txuṣṣ, ayen nețbecciṛ
s ɣuṛ Sidi Ṛebbi ur innekmal ara,

<span id="an_13:10">13:10</span> meɛna m'ara d-yaweḍ wayen innekmalen,
ayen ixuṣṣen meṛṛa ad imḥu.

<span id="an_13:11">13:11</span> Asmi lliɣ d aqcic, țmeslayeɣ am weqcic,
țxemmimeɣ, țmeyyizeɣ slɛeqliya n weqcic ; asmi uɣaleɣ d argaz, ǧǧiɣ
lɛeqliya n weqcic.

<span id="an_13:12">13:12</span> Ass-agi, nețwali am akken di lemri
yumsen, meɛna imiren a nuɣal a nwali akken ilaq ; ass-agi tamusni inu
mazal txuṣṣ lameɛna ad uɣaleɣ ad issineɣ akken i yi-issen Sidi Ṛebbi.

<span id="an_13:13">13:13</span> Tura ihi tlata n leḥwayeǧ-agi i gesɛan
lqima : liman, asirem d leḥmala ; lameɛna di tlata-agi, d lehmala i
ț-țameqqrant.

<span id="an_14:1">14:1</span> Qbel kullec nadit aț-țesɛum leḥmala n
tideț. Nadit daɣen ɣef tukciwin n Ṛṛuḥ iqedsen, lameɛna nadit axiṛ a
d-țxebbiṛem s wayen i d-ițțasen s ɣuṛ Sidi Ṛebbi.

<span id="an_14:2">14:2</span> Win yețmeslayen tutlayin ur nețwassen
ara, mačči i yemdanen i gețmeslay lameɛna i Ṛebbi, imi yiwen ur ifehhem
acu i d-yeqqaṛ ; axaṭer s tezmert n Ṛṛuḥ iqedsen i d-iqqaṛ ayen ur
nezmir a t-nefhem.

<span id="an_14:3">14:3</span> Ma d win i d-ițxebbiṛen s ɣuṛ Sidi Ṛebbi,
ițmeslay-ed i yemdanen, yesnernay liman-nsen, inehhu-ten,
yețṣebbiṛ-iten.

<span id="an_14:4">14:4</span> Win yețmeslayen tutlayin ur nețwassen
ara, yesseǧhad iman-is di liman, ma d win i d-ițxebbiṛen s wayen i
d-ițțasen s ɣuṛ Sidi Ṛebbi, yesseǧhad tajmaɛt n Sidi Ṛebbi meṛṛa.

<span id="an_14:5">14:5</span> Bɣiɣ aț-țemmeslayem meṛṛa s tutlayin ur
nețwassen ara, lameɛna bɣiɣ axiṛ ma tețxebbiṛem s ɣuṛ Sidi Ṛebbi. Win
yețxebbiṛen s ɣuṛ Sidi Ṛebbi yenfeɛ akteṛ n win yețmeslayen
tutlayin-agi, anagar ma yella izmer a d-yessefhem ayen i d-yenna,
iwakken tajmaɛt meṛṛa aț-țestenfeɛ.

<span id="an_14:6">14:6</span> Tura ay atmaten, lemmer a n-aseɣ ɣuṛ-wen
a wen-d mmeslayeɣ s tutlayt ur nețwassen ara, deg wacu ara kkun-nefɛeɣ ?
A kkun-nefɛeɣ ma wwiɣ-awen-d ayen i yi-d-ițțuweḥḥan d wayen i
yi-d-yețțuxebṛen s ɣuṛ Sidi Ṛebbi, neɣ țamusni akk-d uselmed !

<span id="an_14:7">14:7</span> A d-nefk lemtel n dduzan n lmusiqa am
tjewwaqt d snitra : ma yella ur yemxalaf ara ṣṣut-nsent, amek ara neɛqel
ṣṣut n tjewaqt neɣ win n snitra ?

<span id="an_14:8">14:8</span> Neɣ daɣen m'ur neɛqil ara mliḥ ṣṣut n
lbuq s wayes i țheggin imnayen iman-nsen ɣer imenɣi, anwa ara iheggin
iman-is ɣer umenɣi ?

<span id="an_14:9">14:9</span> Kunwi daɣen, ma yella tneṭqem-d s tutlayt
ur nețwassen ara, amek ara nefhem d acu i d-teqqaṛem ? Am akken
tețmeslayem i waḍu !

<span id="an_14:10">14:10</span> Llant aṭas n tutlayin yemxalafen di
ddunit, lameɛna sɛant akk lmeɛna.

<span id="an_14:11">14:11</span> Ma yella ur fhimeɣ ara tameslayt n
walebɛaḍ, ad iliɣ d abeṛṛani ar ɣuṛ-es, nețța daɣen ad yili ɣuṛ-i d
abeṛṛani.

<span id="an_14:12">14:12</span> Kunwi daɣen, imi tețnadim aț-țesɛum
tukciwin i d-yețțak Ṛṛuḥ iqedsen, nadit axiṛ aț-țesɛum ayen ara inefɛen
tajmaɛt n Sidi Ṛebbi.

<span id="an_14:13">14:13</span> Daymi, win yețmeslayen tutlayt ur
netwassen ara ilaq ad issuter i Ṛebbi a s-d-yefk ad yessefhem ayen i
d-iqqaṛ.

<span id="an_14:14">14:14</span> Axaṭer ma yella deɛɛuɣ s tutlayt ur
nețwassen ara, d ṛṛuḥ-iw i gdeɛɛun ɣer Sidi Ṛebbi mačči d lefhama-inu.

<span id="an_14:15">14:15</span> D acu ara xedmeɣ ihi ? Ad dɛuɣ s ṛṛuḥ,
lameɛna ad dɛuɣ daɣen s lefhama-inu ; ad cnuɣ s ṛṛuḥ, ad cnuɣ daɣen s
lefhama-inu.

<span id="an_14:16">14:16</span> Axaṭer ma yella tḥemdeḍ Ṛebbi s ṛṛuḥ-ik
kan s tutlayt ur nețwassen ara, win yeqqimen gar-awen, ur nessin ara,
amek ara d-yini « amin » i yimeslayen-ik, m'ur yefhim ara d acu i
d-teqqaṛeḍ ?

<span id="an_14:17">14:17</span> Ɣas yimeslayen-ik s wayes i tḥemdeḍ
Ṛebbi gerrzen aṭas, ur nfiɛen ara win ur nefhim ara d acu i d-teqqaṛeḍ.

<span id="an_14:18">14:18</span> Ad ḥemdeɣ Ṛebbi imi țmeslayeɣ tutlayin
ur nețwassen ara akteṛ-nwen meṛṛa.

<span id="an_14:19">14:19</span> Meɛna di tejmaɛt, axiṛ-iyi a d-iniɣ
xemsa yimeslayen yețwafehmen iwakken ad lemden wiyaḍ, wala ɛecṛa alaf
imeslayen s tutlayt ur nețwassen ara.

<span id="an_14:20">14:20</span> Ay atmaten, ur țțilit ara am arrac
m'ara tmeyyzem ɣef kra, lameɛna sɛut lɛeqliya n imeqqranen ; ma ɣef
wayen yeɛnan cceṛ, ilit am arrac.

<span id="an_14:21">14:21</span> Atan wayen yuran di tira : Sidi Ṛebbi
yenna-d : A d-mmeslayeɣ i wegdud-agi+ seg imawen n ibeṛṛaniyen + s
