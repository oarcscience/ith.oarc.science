### Kabyle Bible: Jude

<span id="an_1:1">1:1</span> Nekk Yuda yellan d gma-s n Yeɛqub, aqqedac
n Ɛisa Lmasiḥ, i wid i wumi d-issawel Sidi Ṛebbi Baba-tneɣ, yerna
ihegga-ten i Ɛisa Lmasiḥ ;

<span id="an_1:2">1:2</span> ṛṛeḥma, talwit d leḥmala a
wen-d-tețțunefkent s tugeț !

<span id="an_1:3">1:3</span> A wid eɛzizen, acḥal aya segmi i bɣiɣ a
wen-n-aruɣ ɣef wayen yeɛnan leslak i ɣ-icerken, walaɣ belli ilaq-iyi a
kkun-nhuɣ iwakken aț-țesɛum tazmert aț-țḥadrem ɣef liman, liman i
d-yețțunefken yiwet n tikkelt i wid i gextaṛ Sidi Ṛebbi.

<span id="an_1:4">1:4</span> Axaṭer kra n yemdanen skecmen-d iman-nsen
gar-awen, sseɛwajen lmeɛna n ṛṛeḥma n Sidi Ṛebbi iwakken ad ffren yir
tikli-nsen, nekṛen ula d Ɛisa Lmasiḥ, win yellan d Ssid-nneɣ ; lɛiqab i
ten-yețṛaǧun d ameqqran akken i d-ițwaxebbeṛ si zik di tira iqedsen.

<span id="an_1:5">1:5</span> ?as akken teẓram ayagi meṛṛa, bɣiɣ a
kkun-id-smektiɣ belli asmi i d-isellek Sidi Ṛebbi agdud-is si tmurt n
Maṣer yessenger gar-asen wid akk yugin ad amnen yis ;

<span id="an_1:6">1:6</span> daɣen, lmalayekkat-nni ur neṭṭif ara di
ccan i sent-id-ițțunefken, meɛna ǧǧant imukan-nsent eɛlayen, Sidi Ṛebbi
yeqqen-itent di ṭṭlam i dayem alamma d ass ameqqran n lḥisab.

<span id="an_1:7">1:7</span> Akken daɣen i tedṛa d imezdaɣ n temdinin n
Sudum d Gumuṛ akk-d tudrin i sent-id-yezzin, ixeddmen anagar ticmatin d
zzna, qqiment-ed d lemtel mi tent-yessenger Sidi Ṛebbi s tmes yecban
times n ǧahennama ;

<span id="an_1:8">1:8</span> imdanen-agi daɣen i d-ikecmen gar-aneɣ
xeddmen am nutni, tirga-nsen ssawaḍent-țen ad xedmen ticmatin, ur
setɛeṛfen ara s lḥekma n Sidi Ṛebbi, reggmen ula d lmalayekkat yellan
deg igenwan.

<span id="an_1:9">1:9</span> Ula d lmelk Mixayel ur yexdim ara
annect-agi asm'akken i gennuɣ akk-d Cciṭan ɣef lǧețța n Sidna Musa ;
yegguma a t-yergem ula d rregmat meɛna yenna-yas kan : a k-iɛaqeb Sidi
Ṛebbi.

<span id="an_1:10">1:10</span> Meɛna imdanen-agi reggmen ayen ur ssinen
ara, ula d ayen akken ssnen i gessen ula d lmal ur nefhim ara,
yețțawi-ten anagar ɣer nnger-nsen.

<span id="an_1:11">1:11</span> A tawaɣit-nsen ! Axaṭer tebɛen lǧeṛṛa n
Kahin, sṛuḥen iman-nsen di leɣlaḍ, yewwi-ten ṭṭmeɛ n yedrimen am Belɛam,
negren am akken yenger Kuri asm'akken i d-yekker ɣer ccwal.

<span id="an_1:12">1:12</span> D ugur ameqqran m'ara ṭṭfen imukan
gar-awen ɣer imensi n tegmaț, zehhun mbla lḥecma, țxemmimen kan ɣef
yiman-nsen. Cban asigna yețțawi waḍu, ur d-yețțak lehwa ; cban ttjuṛ ur
d-nețțak ara lfakya di lawan-nsent, mmutent, qquṛent dɣa țwaqelɛent.

<span id="an_1:13">1:13</span> Akken i d-ṭṭeggiṛent lemwaji timeqqranin
tikufta-nsent n lewsex, i d-ssufuɣen nutni tikufta n leɛṛuṛ-nsen. Cban
itran ixețțin, iɣellin ɣer dderb n ṭṭlam i sen ițwaheggan.

<span id="an_1:14">1:14</span> Fell-asen i d-icar Hinux iɛacen di lǧil
wis sebɛa deffir Adem, mi d-yenna : atan a d-yas Sidi Ṛebbi nețța d
luluf n lmalayekkat-is

<span id="an_1:15">1:15</span> iwakken ad iḥaseb imdanen meṛṛa, ad
iɛaqeb imcumen ur ten-tḍuɛ ara, ɣef yir lecɣal-nsen, ɣef rregmat i
d-ssufuɣen yemcumen-agi seg yimawen-nsen .

<span id="an_1:16">1:16</span> Imdanen-agi ur steqniɛen ara deg wayen
xeddmen, teddun s lebɣi n tnefsit-nsen ; imeslayen-nsen ččuṛen d zzux,
sqizziben i yemdanen ɣef ddemma n nnfeɛ-nsen.

<span id="an_1:17">1:17</span> Ma d kunwi a wid eɛzizen, mmektit-ed ayen
i wen-d-nnan yakan ṛṛusul n Ssid-nneɣ Ɛisa Lmasiḥ

<span id="an_1:18">1:18</span> asm'akken i wen-d-qqaṛen : di zzman
aneggaru ad ilin wid ara yețmesxiṛen fell-awen, ara ilḥun s lebɣi n
tnefsit-nsen.

<span id="an_1:19">1:19</span> Ihi, d wigi i d-yețțawin lxilaf ger
yemdanen, țxemmimen anagar ɣef wayen yeɛnan ddunit, ur sɛin ara Ṛṛuḥ
iqedsen n Sidi Ṛebbi deg wulawen-nsen.

<span id="an_1:20">1:20</span> Ma d kunwi ay iḥbiben-iw, snernit
liman-nwen yeṣfan, deɛɛut s tezmert n Ṛṛuḥ iqedsen.

<span id="an_1:21">1:21</span> ?ṭfet di leḥmala n Sidi Ṛebbi, țeklet ɣef
ṛṛeḥma n Sidna Ɛisa Lmasiḥ, ara wen-d-yefken tudert yețdumun.

<span id="an_1:22">1:22</span> ?unnet ɣef wid yețcukkun sfehmetțen

<span id="an_1:23">1:23</span> iwakken ad țțuselken, jebdet-țen-id si
tmes. Llan daɣen wiyaḍ ilaq aț-țḥunnem fell-asen meɛna ɣuṛ-wat
aț-țennalem ulamma d llebsa-nsen i tessenǧes lǧețța-nsen.

<span id="an_1:24">1:24</span> Tanemmirt tameqqrant i win yesɛan tazmert
ad iḥader fell-awen iwakken ur tețțecḍem ara, a kkun-id isbedd zdat-es
mbla lɛib di tmanegt- ines s lfeṛḥ ameqqran.

<span id="an_1:25">1:25</span> Tamanegt tameqqrant i Sidi Ṛebbi awḥid i
gellan d amsellek-nneɣ s Ɛisa Lmasiḥ Ssid-nneɣ ; i Yillu nneɣ lhiba,
tazmert, lḥekma akk-d lɛaḍima, uqbel a d-texleq ddunit, di lweqt-agi
akk-d lweqat i d-iteddun ! Amin !
