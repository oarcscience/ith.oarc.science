### Kabyle Bible: 1 Thessalonians

<span id="an_1:1">1:1</span> Nekk Bulus, Silwan akk-d Timuti, i tejmaɛt
n watmaten yellan di temdint n Tiṣalunik yumnen s Sidi Ṛebbi baba-tneɣ
akk-d Ssid-nneɣ Ɛisa lmasiḥ. Ṛṛeḥma ț-țalwit a d-ersent fell-awen.

<span id="an_1:2">1:2</span> Yal ass nețḥemmid Sidi Ṛebbi fell-awen ;
mkul m'ara nedɛu ɣuṛ-es, nețțader-ikkun-id di tẓallit-nneɣ.

<span id="an_1:3">1:3</span> Nețmektay-ikkun-id zdat Sidi Ṛebbi
baba-tneɣ, ɣef wayen akk txeddmem seg wasmi tumnem s Ssid-nneɣ Ɛisa
Lmasiḥ ; tqeblem aț-țenneɛtabem ɣef ddemma n yisem-is imi tḥemmlem-t
yerna lețkal-nwen fell-as d ameqqran.

<span id="an_1:4">1:4</span> Ay atmaten eɛzizen ɣer Sidi Ṛebbi, neẓra
belli d nețța i kkun-id ixtaṛen.

<span id="an_1:5">1:5</span> Lexbeṛ-agi n lxiṛ i wen-d newwi mačči d
imeslayen kan, lameɛna s țțeḥqiq i wen-d-nbecceṛ lexbaṛ-agi n tideț, s
tezmert n Ṛṛuḥ iqedsen. Teẓram amek i nedda gar-awen, asm'akken i nella
ɣuṛ-wen ; nexdem ayagi iwakken a kkun-nɛiwen di tikli-nwen.

<span id="an_1:6">1:6</span> Ula d kunwi tuɣalem tețțeddum am nukni deg
webrid n Ssid-nneɣ, imi di teswiɛt iweɛṛen i tqeblem awal-is s lfeṛḥ i
d-ițțak Ṛṛuḥ iqedsen.

<span id="an_1:7">1:7</span> S wakka tuɣalem d lemtel i wid akk yumnen,
i gellan di tmura n Masidunya akk-d Akaya.

<span id="an_1:8">1:8</span> Axaṭer s ɣuṛ-wen, lexbaṛ n lxiṛ iwweḍ mačči
kan ɣer tmura-agi, meɛna slan s liman-nwen di Sidi Ṛebbi di mkul amkan.
Daymi fiḥel a d-nemmeslay fell-awen.

<span id="an_1:9">1:9</span> Imdanen meṛṛa ḥekkun amek i tesṭerḥbem
yis-nneɣ, amek i d-tuɣalem ɣer webrid n Sidi Ṛebbi, teǧǧam aɛbad n
ssadaț, iwakken aț-țḍuɛem Sidi Ṛebbi n tideț, Sidi Ṛebbi yeddren.

<span id="an_1:10">1:10</span> Yerna tețṛaǧum a d-yuɣal Ssid nneɣ Ɛisa
seg igenwan, Mmi-s-nni i d-isseḥya si ger lmegtin ; d nețța ara
ɣ-isellken ass n lḥisab.

<span id="an_2:1">2:1</span> Ay atmaten, kunwi s yiman-nwen teẓram mačči
mbla lmeɛna i d-nusa ɣuṛ-wen.

<span id="an_2:2">2:2</span> Teẓram daɣen nenneɛtab, nețwargem di
temdint n Filibus, meɛna neṭṭef di liman-nneɣ di Sidi Ṛebbi, nkemmel
abecceṛ n wawal-is gar-awen ɣas akken nețwaqehheṛ.

<span id="an_2:3">2:3</span> Lexbeṛ-agi n lxiṛ i wen-d newwi ur ibni ara
ɣef lekdeb neɣ ɣef tḥila d lexdeɛ.

<span id="an_2:4">2:4</span> Sidi Ṛebbi ițkel fell-aɣ, daymi i ɣ-iceggeɛ
iwakken a nbecceṛ awal-is. Axaṭer iẓra nețmeslay mačči iwakken a neɛǧeb
i yemdanen, meɛna iwakken a s-neɛǧeb i nețța yessnen ulawen-nneɣ.

<span id="an_2:5">2:5</span> Teẓram belli leɛmeṛ ur nemmeslay s usqizzeb
neɣ neḍmeɛ kra deg yiwen, atan Sidi Ṛebbi d inigi !

<span id="an_2:6">2:6</span> Leɛmeṛ ur nqelleb ad yali ccan-nneɣ ama
ɣuṛ-wen ama ɣer wiyaḍ ; yerna lemmer nebɣi, nezmer a nḥettem iman-nneɣ
fell-awen imi d imceggɛen n Lmasiḥ i nella !

<span id="an_2:7">2:7</span> Meɛna asmi nella gar-awen, neččuṛ d leḥnana
am tyemmaț ițḥadaren ɣef warraw-is.

<span id="an_2:8">2:8</span> Imi tellam deg wulawen-nneɣ, mačči kan d
lexbaṛ n lxiṛ n Sidi Ṛebbi i nebɣa a wen-nbecceṛ, meɛna nezmer a nsebbel
ula ț-țiṛwiḥin-nneɣ fell-awen, axaṭer teɛzizem fell-aɣ aṭas.

<span id="an_2:9">2:9</span> Ma tecfam ay atmaten ɣef wussan nni mi i
wen-d-newwi lexbaṛ n lxiṛ, nenneɛtab, nxeddem iḍ d wass akken ur nețțili
ț-țaɛkumt ula i yiwen deg-wen.

<span id="an_2:10">2:10</span> Ṛebbi yeẓra, kunwi teẓram belli
tikli-nneɣ teṣfa, nedda s lḥeqq, yiwen ur aɣ-d-issukkes lɛib gar-awen
kunwi s wid yumnen.

<span id="an_2:11">2:11</span> Teẓram daɣen nella i mkul yiwen deg-wen
am ubabat ɣer warraw-is.

<span id="an_2:12">2:12</span> Nenha-kkun, nesseǧhed-ikkun,
nḥellel-ikkun iwakken aț-țeddum akken yebɣa Sidi Ṛebbi i wen-d issawlen
ɣer tgeldit-is d lɛaḍima-s.

<span id="an_2:13">2:13</span> S wakka nețḥemmid dayem Sidi Ṛebbi imi
tqeblem awal-is i wen-d newwi. Tqeblem-t mačči am akken d awal n wemdan,
meɛna tqeblem-t imi d awal n Sidi Ṛebbi. D awal-agi i gxeddmen deg-wen
tura, kunwi s wid yumnen.

<span id="an_2:14">2:14</span> Ay atmaten, tedṛa yid-wen am tejmuyaɛ n
imasiḥiyen yellan di tmurt n Yahuda : akken i kkun-qehṛen warraw n
tmurt-nwen akken daɣen i ten-qehṛen wudayen.

<span id="an_2:15">2:15</span> D udayen-agi i genɣan Ssid nneɣ Ɛisa
Lmasiḥ akk-d lenbiya, d nutni daɣen i ɣ-iqehṛen. Ɛuṣan Sidi Ṛebbi yerna
d iɛdawen n yemdanen.

<span id="an_2:16">2:16</span> Rnan kemmlen di ddnubat-nsen, axaṭer ur
ɣ-ǧǧin ara a nbecceṛ leslak n Sidi Ṛebbi i leǧnas nniḍen ur nelli ara n
wat Isṛail. S wakka i d-yeɣli lɛiqab n Sidi Ṛebbi fell-asen.

<span id="an_2:17">2:17</span> Ma d nukni ay atmaten seg wasmi i d-nṛuḥ
s ɣuṛ-wen, ɣas akken ur kkun-ẓerrent ara wallen, tellam deg wulawen-nneɣ
; nxaq fell-awen aṭas, daymi i neɛṛeḍ a d-nas iwakken a kkun-nẓer.

<span id="an_2:18">2:18</span> Acḥal n tikkal i nebɣa a nuɣal ɣuṛ-wen,
nekk s yiman-iw, yiwet neɣ snat n tikkal i ɛerḍeɣ a n-ruḥeɣ, meɛna
Cciṭan ur aɣ yeǧǧi ara, iqḍeɛ-aɣ abrid.

<span id="an_2:19">2:19</span> D kunwi i d sebba n usirem d lfeṛḥ-nneɣ
ɣer Sidi Ṛebbi, yis-wen ara nzuxx zdat Ssid-nneɣ Ɛisa asm'ara d-yuɣal.

<span id="an_2:20">2:20</span> Ț-țideț, d kunwi i d can nneɣ i d
lfeṛḥ-nneɣ.

<span id="an_3:1">3:1</span> Imi ur nezmir ara a neṣbeṛ fell-awen nnig
wannect-a, nufa-d yiwen ṛṛay : nukni a neqqim di temdint n Atinya,

<span id="an_3:2">3:2</span> ma d gma-tneɣ Timuti, aqeddac n Sidi Ṛebbi,
ițbecciṛen lexbaṛ n lxiṛ n Lmasiḥ, a wen-t-in-nceggeɛ a kkun isseǧhed
yerna a kkun-inhu di liman nwen,

<span id="an_3:3">3:3</span> iwakken yiwen ur ifeččel di teswiɛt-agi n
uqehheṛ. Teẓram belli ayagi d ayen i ɣ-ițṛaǧun.

<span id="an_3:4">3:4</span> Asmi nella yid-wen nenna-yawen yakan belli
a d-yas wass i deg ara nețwaqehheṛ. Atan tura twalam belli d ayen
ideṛṛun.

<span id="an_3:5">3:5</span> Imi xaqeɣ fell-awen, ur zmireɣ ara ad
ṣebṛeɣ, ceggɛeɣ-awen gma tneɣ Timuti iwakken a yi-d-yawi lexbaṛ ma mazal
teṭṭfem di liman nwen ; axaṭer uggadeɣ a kkun iɣuṛṛ Cciṭan, aț-țețțum
ayen akk i wen-nesselmed, uggadeɣ lxedma nneɣ aț-țuɣal d ulac.

<span id="an_3:6">3:6</span> Mi d-yuɣal Timuti s ɣuṛ-wen, yewwi-yaɣ-d
lexbaṛ yelhan fell-awen : iḥka-yaɣ-d amek tḍuɛem Sidi Ṛebbi, d leḥmala
yellan wway gar-awen ; yenna-yaɣ-d daɣen amek i ɣ-d țmektayem, ur
aɣ-tețțum ara, tḥarem melmi ara ɣ-teẓrem am akken i nxaq fell-awen ula d
nukni.

<span id="an_3:7">3:7</span> Ay atmaten, ɣas akka nella di lmeḥna d
iɣeblan, theddnen wulawen nneɣ ɣef wayen yeɛnan liman-nwen.

<span id="an_3:8">3:8</span> Tura yuɣal-aɣ-d ṛṛuḥ imi neẓra teṭṭfem di
Ssid-nneɣ Ɛisa Lmasiḥ.

<span id="an_3:9">3:9</span> Lfeṛḥ-nneɣ d ameqqran zdat Sidi Ṛebbi ɣef
ddemma-nwen. Akken nebɣu neḥmed-it fell-awen drus.

<span id="an_3:10">3:10</span> Nețḥellil Sidi Ṛebbi am iḍ am ass, a
ɣ-d-yefk tagniț i deg ara kkun-nẓer, iwakken a wen-nkemmel aselmed n
wayen i kkun-ixuṣṣen di liman-nwen.

<span id="an_3:11">3:11</span> Nessutur Sidi Ṛebbi baba-tneɣ d Ssid-nneɣ
Ɛisa a ɣ-d-ldin iberdan iwakken a n-nas ɣuṛ-wen.

<span id="an_3:12">3:12</span> Ad isnerni Ṛebbi leḥmala i tesɛam wway
gar-awen, aț-țḥemmlem daɣen imdanen meṛṛa am akken i kkun nḥemmel nukni.

<span id="an_3:13">3:13</span> Ad isseǧhed Ṛebbi ulawen nwen, iwakken
aț-țeṣfum, aț-țilim mbla lɛib zdat Sidi Ṛebbi baba-tneɣ i wass i deg ara
d-yuɣal Ssid-nneɣ Ɛisa Lmasiḥ nețța d lmalayekkat.

<span id="an_4:1">4:1</span> Ay atmaten, tlemdem s ɣuṛ-nneɣ amek ara
teddum iwakken aț-țɛeǧbem i Sidi Ṛebbi ; a kkun nenhu s yisem n Sidna
Ɛisa Lmasiḥ kemmlet ɣer zdat iwakken aț țimɣuṛem.

<span id="an_4:2">4:2</span> Ay atmaten teẓram belli lewṣayat i
wen-nesselmed, kkant-ed s ɣuṛ Sidna Ɛisa Lmasiḥ.

<span id="an_4:3">4:3</span> D ṣṣfa n wulawen-nwen i gebɣa Sidi Ṛebbi ;
yebɣa aț-țḥadrem iman nwen si lefsad,

<span id="an_4:4">4:4</span> yal yiwen deg-wen ur ițțak ara lebɣi i
tnefsit-is deg wayen icemten ; ad iddu s ṣṣfa d lḥeṛma,

<span id="an_4:5">4:5</span> ur ițmenni ara ayen icemten am akken
xeddmen imejhal ur nessin ara Sidi Ṛebbi.

<span id="an_4:6">4:6</span> Deg wannect-agi ilaq yiwen ur ițkellix
gma-s di tlufa yeɛnan idrimen. Acḥal n tikkal i kkun-nweṣṣa iwakken
aț-țḥadrem iman-nwen axaṭer Sidi Ṛebbi ad yerr țțar akken i
wen-t-id-nenna !

<span id="an_4:7">4:7</span> Sidi Ṛebbi ur aɣ-d-inni ara a neddu di
lefsad meɛna iweṣṣa-yaɣ-d a neddu s ṣṣfa.

<span id="an_4:8">4:8</span> S wakka win ur neqbil ara lewṣayat-agi,
mačči d amdan i gɛuṣa, meɛna iɛuṣa Sidi Ṛebbi i wen-d-ițțaken Ṛṛuḥ-is
iqedsen.

<span id="an_4:9">4:9</span> Ɣef wayen yeɛnan tagmaț, fiḥel ma
nemmeslay-awen-d fell-as, imi temyeḥmalem wway gar-awen akken i tlemdem
s ɣuṛ Sidi Ṛebbi.

<span id="an_4:10">4:10</span> Akken daɣen i tḥemmlem atmaten yellan di
tmurt n Masidunya meṛṛa. Di leɛnaya nwen ay atmaten, kemmlet deg
webrid-agi.

<span id="an_4:11">4:11</span> Nadit aț-țɛicem di talwit, ur țɛassat ara
d acu i xeddmen wiyaḍ. Xedmet s ifassen-nwen, ur ṭṭamaɛet ara aɣṛum-nwen
deg wiyaḍ, akken i kkun-nweṣṣa yakan

<span id="an_4:12">4:12</span> iwakken a kkun-qadṛen medden ɣef ddemma n
tikli nwen yeṣfan, yerna ur tețțilim ț-țaɛkumt ula i yiwen.

<span id="an_4:13">4:13</span> Ay atmaten, ur nebɣi ara a kkun-neǧǧ di
ccekk ɣef wayen yeɛnan wid yemmuten, iwakken ur tețțilim ara di leḥzen
am wid ur nesɛi ara asirem n leslak n Sidna Ɛisa.

<span id="an_4:14">4:14</span> Axaṭer akken numen belli Sidna Ɛisa
yemmut yuɣal yeḥya-d, numen daɣen belli Sidi Ṛebbi a d-isseḥyu wid
yemmuten, imi umnen s Sidna Ɛisa.

<span id="an_4:15">4:15</span> Atah wayen ara wen-d-nini akken i
t-id-isselmed Ssid-nneɣ : ma yella mazal-aɣ nedder asm'ara d-yuɣal, ur
nețțili ara d imezwura ɣef wid yemmuten.

<span id="an_4:16">4:16</span> Axaṭer m'ara d-iffeɣ lameṛ, m'ara
d-tenṭeq taɣect n lmelk akk-d ṣṣut n lbuq n Sidi Ṛebbi, Sidna Ɛisa s
yiman-is a d-iṣubb seg igenni, imiren widak-nni yemmuten, llan umnen
yis, a d-ḥyun d imezwura si ger lmegtin.

<span id="an_4:17">4:17</span> Ma d nukni ma yella mazal-aɣ nedder, a
nețwarfed yid-sen ɣef wusigna, a nemmager Sidna Ɛisa deg igenwan,
iwakken a nili yid-es i dayem.

<span id="an_4:18">4:18</span> Ihi mseǧhadet, mṣebbaṛet wway gar-awen s
yimeslayen-agi.

<span id="an_5:1">5:1</span> Ɣef wayen yeɛnan ussan akk d lweqt, fiḥel
ma nura-yawen fell-asen.

<span id="an_5:2">5:2</span> Teẓram s yiman-nwen belli ass-nni, Sidna
Ɛisa a d-yuɣal am umakar deg iḍ.

<span id="an_5:3">5:3</span> M'ara yinin yemdanen : « aql-aɣ di lehna
ț-țalwit, » ass-nni ur țfiqen ara alamma yeɣli-d fell-asen nnger, am
lewjaɛ m'ara ṭṭfen tameṭṭut yellan s tadist, yernu yiwen ur imenneɛ.

<span id="an_5:4">5:4</span> Ma d kunwi ay atmaten, ur tellim ara di
ṭṭlam iwakken a d-iɣli wass-nni fell-awen am umakar ;

<span id="an_5:5">5:5</span> axaṭer kunwi d arraw n tafat, tețțeddum deg
wass ; nukni ur nelli ara d arraw n ṭṭlam iteddun deg yiḍ.

<span id="an_5:6">5:6</span> Ihi ur ilaq ara a neɣfel am wiyaḍ, meɛna
ilaq a nɛass iman-nneɣ yerna a nɛiwez.

<span id="an_5:7">5:7</span> Wid yegganen gganen deg yiḍ, wid iṣekkren
țṣekkiren deg yiḍ.

<span id="an_5:8">5:8</span> Ma d nukni yellan d arraw n tafat, a nɛass
ɣef yiman-nneɣ : a nesɛu liman d leḥmala am llebsa n wuzzal ițlusu
uɛeskṛi, a nesɛu asirem n leslak, am tcacit n wuzzal ɣef wuqeṛṛuy.

<span id="an_5:9">5:9</span> Axaṭer Sidi Ṛebbi ur aɣ-ixtaṛ ara iwakken a
nețwaɛaqeb, meɛna yebɣa a nesɛu leslak s Ssid-nneɣ Ɛisa Lmasiḥ

<span id="an_5:10">5:10</span> i gemmuten fell-aɣ, iwakken ama nedder
ama nemmut a nili yid-es.

<span id="an_5:11">5:11</span> S wakka ihi, țemseǧhadet, țemṣebbaṛet
wway gar-awen akken i txeddmem yakan.

<span id="an_5:12">5:12</span> Di leɛnaya-nwen ay atmaten, qadṛet wid
iqeddcen gar-awen, axaṭer s yisem n Sidna Ɛisa Lmasiḥ i kkun nehhun, i
kkun-sselmaden.

<span id="an_5:13">5:13</span> Ɛuzzet țen tḥemmlem-ten ɣef ddemma n
lxedma-nsen, ɛicet di lehna wway gar-awen.

<span id="an_5:14">5:14</span> Di leɛnaya-nwen ay atmaten : nhut wid
iteddun s yir tikli, ṣebbṛet wid iwumi ɣlin ifadden, refdet wid ifeclen,
swesɛet ulawen-nwen ɣuṛ-sen meṛṛa.

<span id="an_5:15">5:15</span> Ḥadret iman-nwen ur țțarat ara țțaṛ,
qellbet ɣef wayen yelhan ama gar-awen ama ɣer imdanen meṛṛa.

<span id="an_5:16">5:16</span> Ilit dayem di lfeṛḥ,

<span id="an_5:17">5:17</span> yal ass deɛɛut ɣer Sidi Ṛebbi,

<span id="an_5:18">5:18</span> ḥemmdet-eț ɣef kullec. D ayagi i gețṛaǧu
Sidi Ṛebbi deg-wen, di tikli-nwen akk-d Ɛisa Lmasiḥ.

<span id="an_5:19">5:19</span> Ur snusut ara tafat n Ṛṛuḥ iqedsen ;

<span id="an_5:20">5:20</span> ur ḥeqqṛet ara ayen i d-ițweḥḥi Ṛṛuḥ
iqedsen i yemdanen.

<span id="an_5:21">5:21</span> Meɛna meyzet, qeblet ayen yelhan,

<span id="an_5:22">5:22</span> ḥadret iman-nwen ɣef wayen n diri.

<span id="an_5:23">5:23</span> Sidi Ṛebbi bab n talwit, ad iṣeffi
ṛṛuḥ-nwen, lǧețțat nwen ț-țeṛwiḥin-nwen, iwakken aț țeqqimem d izedganen
i wass i deg ara d-yuɣal Ssid-nneɣ Ɛisa Lmasiḥ.

<span id="an_5:24">5:24</span> Win i kkun-ixtaṛen, deg-s laman, d nețța
ara kkun-iṣṣiwden ɣer wannect agi.

<span id="an_5:25">5:25</span> Ay atmaten, deɛɛut ɣer Sidi Ṛebbi
fell-aneɣ.

<span id="an_5:26">5:26</span> Sṣiwḍet sslam-iw i watmaten meṛṛa.

<span id="an_5:27">5:27</span> Di leɛnaya-nwen, s yisem n Ssid-nneɣ,
sseɣṛet tabṛaț-agi i watmaten meṛṛa.

<span id="an_5:28">5:28</span> Ṛṛeḥma n Ssid-nneɣ a d-ters fell-awen.
