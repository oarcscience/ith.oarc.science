yemdanen ɣer lɛaḍima-s ;

Cciṭan, win akken iḥekkmen ɣef lmut,

<span id="an_2:15">2:15</span> ad ikkes lxuf i wid akk yezgan d aklan

<span id="an_2:17">2:17</span> Daymi i s-ilaq a d-yas s ṣṣifa n wemdan

<span id="an_3:7">3:7</span> ?ef wannect-a i d-yenna Ṛṛuḥ iqedsen :

<span id="an_3:8">3:8</span>  ur sɣaṛayet ara ulawen-nwen am wasmi i

Malxisadeq mačči am Haṛun n at Lewwi ?

<span id="an_7:13">7:13</span> Axaṭer Ssid-nneɣ, ur d-yekki ara seg

ula d yiwen n lmuqeddem i gqedcen deg wudekkan n iseflawen.

<span id="an_7:14">7:14</span> Ihi neẓra belli Ssid-nneɣ yeffeɣ ed seg

seg wedrum agi.

<span id="an_7:15">7:15</span> Yella wayen i d-ibeggnen annect agi,
axaṭer yeffeɣ-ed lmuqeddem nniḍen yecban Malxisadeq.

<span id="an_7:16">7:16</span> Mačči d ccariɛa i t-yerran d lmuqeddem

<span id="an_7:17">7:17</span> imi yura fell-as : Kečč d lmuqeddem i

<span id="an_7:18">7:18</span> Lameṛ aqdim yeɛnan lmuqedmin yețwabeddel,

<span id="an_8:5">8:5</span> Ccɣel i xeddmen lmuqedemin agi d lemtel, d

idammen n iseflawen ɣef ddemma n ddnubat-is akk-d wid n wegdud.

ɣer wemkan iqedsen nezzeh skud mazal aqiḍun n temlilit.

ad ssiwḍen ɣer ṣṣfa n wul.

<span id="an_9:10">9:10</span> D ayen yeɛnan kan lɛaddat n ccariɛa ɣef

yid-es lameɛna d idammen-is nețța, s wannect-a i ɣ-d-yefda i dayem.

iqelwacen akk-d yiɣed n tgenduzt yeṛɣan d asfel țṣeffin lǧețța n wid
idenben m'ara ten ṛuccen yis-sen,

d-yedṛan, lmut-is i gtekksen ɣef yemdanen ddnubat-nsen n wasmi i llan
seddaw n leɛqed amezwaru.

<span id="an_9:16">9:16</span> Ma yella walebɛaḍ yeǧǧa-d lewṣaya weqbel
ad yemmet akken ad feṛqen ayla-s, ilaq ad aṛǧun alamma yemmut.

<span id="an_9:17">9:17</span> Lewṣaya-agi ur tṣeḥḥa ara ma yella mazal
bab-is yedder lameɛna m'ara yemmet aț-țuɣal tṣeḥḥa.

<span id="an_9:18">9:18</span> Daymi leɛqed amezwaru daɣen yebda s

iqedsen akk-d wegdud meṛṛa,

s idammen, ulac leɛfu mbla tazzla n idammen.

igenwan ilaq ad ițwaṣeffi s idammen n wesfel yifen iseflawen imezwura.

<span id="an_9:24">9:24</span> Axaṭer Lmasiḥ ur yekcim ara ɣer wemkan
iqedsen i bnan ifasen n yemdanen, yellan d lemtel n win n tideț, meɛna
yuli ɣer igenni anda yella tura d amcafeɛ-nneɣ zdat Sidi Ṛebbi.

<span id="an_9:25">9:25</span> Lmuqeddem ameqqran ikeččem mkul aseggas
ɣer wemkan iqedsen nezzeh, iwakken ad iqeddem idammen n lmal, ma d
Lmasiḥ ur ikcim ara iwakken ad isebbel iman-is aṭas n tikkal ;

seg wasmi i d-texleq ddunit. Lameɛna yusa-d di zzman-agi aneggaru yiwet
n tikkelt kan, isebbel iman-is d asfel iwakken ad ikkes ddnub.

<span id="an_9:27">9:27</span> Lmut tețțawi imdanen yiwet n tikkelt kan
imiren a d-yili lḥisab.

<span id="an_9:28">9:28</span> Ula d Lmasiḥ yefka iman-is yiwet n

iwakken ad isellek wid i t-yețṛaǧun

țqeddimen seg useggas ɣer useggas.

<span id="an_10:2">10:2</span> Ihi tura lemmer wid yețqeddimen

<span id="an_10:3">10:3</span> Meɛna s iseflawen-agi țmektayen-d yal
aseggas belli mazal-iten d imednuben.

<span id="an_10:4">10:4</span> Imi d lmuḥal idammen n iɛejmiyen d

<span id="an_10:5">10:5</span> Daymi mi i gțeddu a d-yas Lmasiḥ ɣer

<span id="an_10:6">10:6</span>  Ur teqbileḍ iseflawen n lmal i tețțeț

<span id="an_10:8">10:8</span> Teslam d acu i d-yenna Lmasiḥ ! Yenna :
