### Kabyle Bible: Ephesians

<span id="an_1:1">1:1</span> Nekk Bulus yellan d amceggeɛ n Ɛisa Lmasiḥ
s lebɣi n Sidi Ṛebbi, uriɣ-ed tabṛaț-agi i watmaten yellan di temdint n
Ifasus i gumnen s Ɛisa Lmaṣiḥ.

<span id="an_1:2">1:2</span> Ṛṛeḥma d lehna a wen-d țțunefkent s ɣuṛ
Sidi Ṛebbi baba-tneɣ akk-d Ssid-nneɣ Ɛisa Lmasiḥ.

<span id="an_1:3">1:3</span> Ad ițțubarek Sidi Ṛebbi, baba-s n Ssid-nneɣ
Ɛisa Lmasiḥ, i ɣ-d-ifkan deg igenwan mkul lbaṛakat n Ṛṛuḥ iqedsen, s
Ɛisa Lmasiḥ.

<span id="an_1:4">1:4</span> Yis i ɣ-d-ixtaṛ Sidi Ṛebbi uqbel a d-ixleq
ddunit, iwakken a nili zdat-es d izedganen mbla lɛib.

<span id="an_1:5">1:5</span> Sidi Ṛebbi iḥemmel-aɣ, daymi i geqsed si
zik s lebɣi-ines a ɣ-yerr d arraw-is s Ɛisa Lmasiḥ,

<span id="an_1:6">1:6</span> iwakken a t-neḥmed a t-ncekkeṛ ɣef ṛṛeḥma-s
tameqqrant i ɣ-d-ifka s Mmi-s eɛzizen.

<span id="an_1:7">1:7</span> S lmut n Sidna Ɛisa i ɣ-d-icufeɛ Sidi
Ṛebbi, s idammen-is yuzzlen i ɣ-țwaɛfan ddnubat-nneɣ ; s wakka
isbeggen-aɣ-d acḥal meqqṛet ṛṛeḥma ines

<span id="an_1:8">1:8</span> i d-issers fell-aɣ s tugeț yerna ifka-yaɣ-d
lefhama ț-țmusni.

<span id="an_1:9">1:9</span> Sidi Ṛebbi isbeggen-aɣ-ed lebɣi-s yellan d
lbaḍna, lemziya-nni i geqsed a ț-ixdem si tazwara s Lmasiḥ,

<span id="an_1:10">1:10</span> ara yessiweḍ ɣer lekmal-is di lweqt i
gextaṛ : ad isdukkel ayen akk yellan ama deg igenwan ama di lqaɛa,
seddaw lḥekma n Lmasiḥ.

<span id="an_1:11">1:11</span> S Ɛisa Lmasiḥ i ɣ-yerra Sidi Ṛebbi d
ayla-s, am akken i t-iqsed si tazwara di lebɣi-ines,

<span id="an_1:12">1:12</span> iwakken nukni yessarmen di Lmasiḥ d
imezwura, a neḥmed Sidi Ṛebbi ɣef tmanegt-is.

<span id="an_1:13">1:13</span> Kunwi daɣen mi teslam s lexbaṛ n lxiṛ i
wen-d-yewwin leslak, tumnem s Lmasiḥ ; dɣa Sidi Ṛebbi iḍbeɛ-ikkun s
ṭṭabeɛ-ines yellan d Ṛṛuḥ iqedsen i gewɛed a wen-t-id yefk.

<span id="an_1:14">1:14</span> Ṛṛuḥ iqedsen d aɛeṛbun n lweṛt-nneɣ,
alamma yewweḍ ɣer lekmal-is leslak n wid akk yextaṛ Sidi Ṛebbi. A neḥmed
ihi Sidi Ṛebbi ɣef tmanegt-agi-ines ( lɛaḍima-s ).

<span id="an_1:15">1:15</span> Daymi mi sliɣ s liman-nwen yeṛsan di Ɛisa
Lmasiḥ d wamek i tḥemmlem wid akk i gextaṛ Sidi Ṛebbi,

<span id="an_1:16">1:16</span> kull ass țḥemmideɣ rennuɣ Sidi Ṛebbi ɣef
ddemma-nwen, țțadreɣ-kkun-id dayem di tẓallit-iw

<span id="an_1:17">1:17</span> iwakken Sidi Ṛebbi, Illu n Ssid nneɣ Ɛisa
Lmasiḥ yellan d bab n tmanegt, a wen-d-yefk s Ṛṛuḥ iqedsen tamusni d
lefhama s wayes ara t-tissinem akken ilaq ;

<span id="an_1:18">1:18</span> a wen-ildi leɛqel iwakken aț țfehmem d
acu n usirem i ɣer wen-d issawel, d acu n lbaṛakat timeqqranin i ghegga
i wid yumnen yis ;

<span id="an_1:19">1:19</span> aț-țwalim tazmert-is tameqqrant i
ɣ-d-ibeggen i nukni s wid yumnen yis.

<span id="an_1:20">1:20</span> Tazmert nni i d-ibeggen s Ɛisa Lmasiḥ, mi
t-id-yesseḥya si ger lmegtin, yerna yesɣim-it ɣer tama-s tayeffust deg
igenwan iwakken ad yeḥkem,

<span id="an_1:21">1:21</span> sennig igelliden meṛṛa n ddunit, sennig
imdebbṛen ț-țnezmarin, sennig tgeldiwin akk-d yismawen meṛṛa yellan
mačči kan di lǧil-agi meɛna ula di lǧil i d-iteddun.

<span id="an_1:22">1:22</span> Sidi Ṛebbi yerra kullec seddaw iḍaṛṛen n
Lmasiḥ, yerra-t daɣen d aqeṛṛuy n tejmaɛt n imasiḥiyen

<span id="an_1:23">1:23</span> yellan d lǧețța n Lmasiḥ ; deg-s i
gețțili yerna d nețța i d kullec di ddunit meṛṛa.

<span id="an_2:1">2:1</span> Zik-nni, tellam am wid yemmuten ɣef ddemma
n tuccḍa-nwen akk-d ddnubat-nwen ;

<span id="an_2:2">2:2</span> tețɛicim di ddnub, tewwi-kkun ddunit,
tḍefṛem Cciṭan, sselṭan n ddunit-agi i gțekellixen ar tura wid ur nḍuɛ
ara Sidi Ṛebbi.

<span id="an_2:3">2:3</span> Nukni daɣen, zik-nni am nutni i nella ;
nettabaɛ ixemmimen-nneɣ, ccehwat-nneɣ, nxeddem lebɣi n tnefsit-nneɣ,
nella nuklal urrif n Sidi Ṛebbi am wiyaḍ.

<span id="an_2:4">2:4</span> Lameɛna imi i geččuṛ Sidi Ṛebbi d ṛṛeḥma,
yerna leḥmala-ines ɣuṛ-nneɣ ț-țameqqrant,

<span id="an_2:5">2:5</span> yerra-yaɣ-d ɣer tudert akk-d Lmasiḥ, nukni
yellan am wid yemmuten ɣef sebba n ddnubat-nneɣ. S ṛṛeḥma n Sidi Ṛebbi i
tețwasellkem.

<span id="an_2:6">2:6</span> S tdukli-nneɣ akk-d Ɛisa Lmasiḥ, Sidi Ṛebbi
isseḥya-ɣ-ed akk-d Lmasiḥ yerna yesɣim-aɣ yid-es deg igenwan,

<span id="an_2:7">2:7</span> iwakken di leqṛun i d-iteddun, a d-tban
acḥal meqqṛit ṛṛeḥma-ines akk-d leḥnana-s i ɣ-d-isbeggen s Ɛisa Lmasiḥ.

<span id="an_2:8">2:8</span> S ṛṛeḥma n Sidi Ṛebbi kan i tețwasellkem,
mi tumnem. Ayagi mačči d ayen i d-ikan s ɣuṛ-wen lameɛna ț-țukci n Sidi
Ṛebbi.

<span id="an_2:9">2:9</span> Mačči s lḥasanat-nwen i tețwasellkem akken
yiwen ur ițzuxxu s yiman is ;

<span id="an_2:10">2:10</span> axaṭer ayen i nella tura s ɣuṛ Sidi Ṛebbi
i ɣ-d-yekka, ixleq-aɣ-ed di Ɛisa Lmasiḥ iwakken a nexdem di tudert-nneɣ
lecɣal yelhan i-ɣ-d ihegga si tazwara.

<span id="an_2:11">2:11</span> Kunwi ur nelli ara seg wegdud n wat
Isṛail, iwumi qqaṛen wat Isṛail : « A wid ur neḍhiṛ » axaṭer nutni
ḥesben iman-nsen ḍehṛen ɣef ddemma n ṭṭhaṛa-nni i sɛan di lɛadda-nsen,

<span id="an_2:12">2:12</span> mmektit-ed amek i tellam zik : di
lweqt-nni tellam tbeɛdem ɣef Lmasiḥ, d ibeṛṛaniyen ur nelli ara seg
ugdud i gextaṛ Sidi Ṛebbi, ur tețțekkam di lemɛahdat i sen-iwɛed, ur
tesɛim asirem, ur tessinem Sidi Ṛebbi.

<span id="an_2:13">2:13</span> Meɛna imi tumnem s Ɛisa Lmasiḥ, kunwi
yellan zik tbeɛdem fell-as, tura tqeṛṛbem-d ɣuṛ-es s idammen-is yuzzlen
ɣef ddemma-nwen.

<span id="an_2:14">2:14</span> Axaṭer d Lmasiḥ s yiman-is i d
lehna-nneɣ, imi i gesdukkel at Isṛail d leǧnas nniḍen yerra-ten d yiwen
n wegdud. Mi gefka iman-is d asfel, ihudd lḥiḍ-nni i ten-ifeṛqen, i
ten-yerran d iɛdawen ;

<span id="an_2:15">2:15</span> yesseɣli tazmert n ccariɛa yerna lewṣayat
akk-d leqwanen-ines, iwakken a d-issufeɣ si snat ccetlat-agi, yiwen kan
n wegdud ajdid ara yamnen yis ; s wakka i d-issers lehna.

<span id="an_2:16">2:16</span> S lmut-is ɣef wumidag i ten-issemṣaleḥ d
Sidi Ṛebbi, ikkes tiɛdawit yellan gar-asen, isdukkel sin leǧnas-agi
yerra-ten d yiwen wegdud.

<span id="an_2:17">2:17</span> Yusa-d a wen-ibecceṛ lexbaṛ n lxiṛ ,
yewwi-awen-d lehna ama i kunwi yellan tbeɛdem ɣef Ṛebbi, ama i wat
Isṛail iqeṛben ɣuṛ-es ;

<span id="an_2:18">2:18</span> axaṭer yis i nezmer am kunwi am nukni, a
nqeṛṛeb meṛṛa ɣer Baba Ṛebbi, s yiwen n Ṛṛuḥ iqedsen.

<span id="an_2:19">2:19</span> Ihi tura, kunwi ur nelli ara seg wat
Isṛail, ur tellim ara d ibeṛṛaniyen neɣ d imsebriden, meɛna tețțekkam di
tejmaɛt n wegdud n Sidi Ṛebbi, tuɣalem d iɛeyyalen n twacult-is.

<span id="an_2:20">2:20</span> Tuɣalem d axxam yebnan ɣef lsas i ssersen
ṛṛusul akk-d lenbiya, Ɛisa Lmasiḥ s yiman-is i d azṛu alemmas yeṭṭfen
lebni.

<span id="an_2:21">2:21</span> D nețța i gessalayen lebni, yesseǧhad-it
iwakken ad yuɣal d lǧameɛ iqedsen n Sidi Ṛebbi.

<span id="an_2:22">2:22</span> S tdukli-nwen akk-d Lmasiḥ, ula d kunwi
tețțekkam di lebni-agi, iwakken aț-țilim d axxam anda yezdeɣ Sidi Ṛebbi
s Ṛṛuḥ-is.

<span id="an_3:1">3:1</span> Daymi nekk Bulus, aql-i d ameḥbus n Lmasiḥ
ɣef ddemma nwen kunwi ur nelli ara n wat Isṛail.

<span id="an_3:2">3:2</span> Ahat teslam s lxedma i yi-d ițțunefken s
ṛṛeḥma n Sidi Ṛebbi ɣef wayen i kkun-yeɛnan.

<span id="an_3:3">3:3</span> Sidi Ṛebbi ildi-yi-d allen iwakken ad
fehmeɣ lbaḍna-ines, am akken i t-id-uriɣ s kra imeslayen ;

<span id="an_3:4">3:4</span> m'ara ten-teɣṛem, aț-țeẓrem ayen fehmeɣ ɣef
lbaḍna n Lmasiḥ.

<span id="an_3:5">3:5</span> Sidi Ṛebbi ur d-ibeggen ara lbaḍna-agi i
lejdud-nneɣ, lameɛna tura ibeggen-iț-id s Ṛṛuḥ-is iqedsen i
imceggɛen-ines akk-d lenbiya.

<span id="an_3:6">3:6</span> Ațah lbaḍna-agi : imi umnen leǧnas ur nelli
ara n wat Isṛail s lexbaṛ n lxiṛ n Lmasiḥ i sen-d ițțubeccṛen, ad țekkin
di lɛahed i d-ițțunefken i wat Isṛail, ad weṛten yid-sen lbaṛakat yerna
ad uɣalen meṛṛa d yiwet n lǧețța.

<span id="an_3:7">3:7</span> Uɣaleɣ d aqeddac n lexbaṛ-agi n lxiṛ n
Lmasiḥ s ṛṛeḥma n Sidi Ṛebbi ț-țezmert-is ixeddmen deg-i ;

<span id="an_3:8">3:8</span> nekk yellan d ulac ger imasiḥiyen meṛṛa,
yețțunefk-iyi-d s ṛṛeḥma n Sidi Ṛebbi ad beccṛeɣ i wid ur nelli ara n
wat Isṛail, lexbaṛ n lxiṛ yeɛnan lbaṛakat ur nkeffu yellan di Lmasiḥ

<span id="an_3:9">3:9</span> yerna a d-sfehmeɣ lmeɛna n lbaḍna agi i
geffer Sidi Ṛebbi si lebda, nețța i gxelqen kullec.

<span id="an_3:10">3:10</span> S wakka, lḥekmat ț-țnezmarin n igenwan ad
issinent tamusni n Sidi Ṛebbi di mkul ṣṣifat-ines, s tejmaɛt n
imasiḥiyen

<span id="an_3:11">3:11</span> akken yella di leqsed n Sidi Ṛebbi si
lebda ; leqsed-agi yessaweḍ-it ɣer lekmal-is s Ɛisa Lmasiḥ Ssid-nneɣ.

<span id="an_3:12">3:12</span> S liman-nneɣ di Sidna Ɛisa i nezmer a
nqeṛṛeb ɣer Sidi Ṛebbi s lețkal.

<span id="an_3:13">3:13</span> Daymi i wen-qqaṛeɣ tura : ur țțaǧat ara
uguren i d-țmagareɣ ɣef ddemma-nwen a kkun-sfeclen axaṭer d ayen ara
kkun-iqeṛṛben ɣer Sidi Ṛebbi.

<span id="an_3:14">3:14</span> Ɣef wayagi i kerfeɣ tigecrar, seǧǧdeɣ
zdat Sidi Ṛebbi baba-tneɣ,

<span id="an_3:15">3:15</span> seg i d-tekka yal tawacult yellan ama deg
igenwan ama di lqaɛa,

<span id="an_3:16">3:16</span> ṭṭalabeɣ-as iwakken s lɛaḍima-s
tameqqrant, a kkun-isseǧhed s tezmert n Ṛṛuḥ-is iqedsen.

<span id="an_3:17">3:17</span> Ssutureɣ-as ad izdeɣ Lmasiḥ deg
wulawen-nwen s liman, aț-țesɛum izuṛan ara iṛeṣṣin di leḥmala,

<span id="an_3:18">3:18</span> iwakken kunwi akk-d wid yumnen meṛṛa,
aț-țzemrem aț-țfehmem acḥal hraw, acḥal ɣezzif, acḥal lqay, acḥal ɛlay
leḥmala n Lmasiḥ,

<span id="an_3:19">3:19</span> yerna aț-țissinem leḥmala-agi yugaren yal
tamusni yellan di ddunit, alamma yezdeɣ Ṛṛuḥ n Sidi Ṛebbi deg-wen s
lekmal.

<span id="an_3:20">3:20</span> Sidi Ṛebbi ixeddem deg-nneɣ s tezmert-is,
yețțak-aɣ-d akteṛ n wayen akk i s-neṭṭalab d wayen akk i nețxemmim ;

<span id="an_3:21">3:21</span> ad ițțubarek yisem-is si lǧil ɣer lǧil di
tejmaɛt n imasiḥiyen akk-d Ɛisa Lmasiḥ i dayem, Amin !

<span id="an_4:1">4:1</span> Nekk yellan d aqeddac n Lmasiḥ, a kkun-nhuɣ
ihi aț-țeddum akken yebɣa Sidi Ṛebbi mi wen-d-issawel.

<span id="an_4:2">4:2</span> Ddut s lḥenna d wannuz akk-d ṣṣbeṛ,
țemyawit wway gar-awen s lmaḥibba.

<span id="an_4:3">4:3</span> Ḥeṛset iman-nwen aț-țeṭṭfem di tdukkli n
Ṛṛuḥ iqedsen, s lehna wway gar-awen.

<span id="an_4:4">4:4</span> Yiwet n lǧețța kan akk-d yiwen n Ṛṛuḥ
iqedsen i gellan, am akken daɣen i wen-d-issawel Sidi Ṛebbi ɣer yiwen
usirem kan.

<span id="an_4:5">4:5</span> Yiwen kan i d Ssid-nneɣ, yiwen n liman
akk-d yiwen n weɣḍas i gellan,

<span id="an_4:6">4:6</span> yiwen n Ṛebbi kan i gellan, d nețța i d
baba-tneɣ meṛṛa ; yella sennig kullec, yella ger yemdanen meṛṛa, yella
daɣen deg-sen meṛṛa.

<span id="an_4:7">4:7</span> Meɛna mkul yiwen deg-nneɣ tețțunefk-as-d
ṛṛeḥma n Sidi Ṛebbi s wakken yella umur i s-d-yefka Lmasiḥ.

<span id="an_4:8">4:8</span> Akken yura di tira iqedsen+ : Mi yuli ɣer
igenwan, + yewwi yid-es imeḥbas, ifka-d tikciwin i yemdanen.+ +

<span id="an_4:9">4:9</span> D acu i d lmeɛna : « Yuli ɣer igenni ? »
Ayagi, yeɛni : uqbel ad yali Lmasiḥ ɣer igenni, iṣubb ɣer imukan lqayen
n ddaw tmurt.

<span id="an_4:10">4:10</span> D win akken iṣubben, i gulin daɣen ɣer
igenwan iwakken ad yeɛmeṛ ddunit meṛṛa ;

<span id="an_4:11">4:11</span> d nețța i d-ifeṛqen tukciwin i yemdanen :
kra yefka-yasen ad ilin d ṛṛusul, kra d lenbiya, kra nniḍen ad beccṛen
lexbaṛ n lxiṛ, wiyaḍ d imeksawen n tejmaɛt n imasiḥiyen neɣ d wid
yesselmaden agdud n Sidi Ṛebbi.

<span id="an_4:12">4:12</span> S tukciwin-agi, Sidna Ɛisa ad iheggi wid
yextaṛ Sidi Ṛebbi, ad sɛun tazmert s wayes ara xedmen ccɣel i sen-d-ifka
iwakken aț-țegmu tejmaɛt n watmaten yellan d lǧețța n Lmasiḥ ;

<span id="an_4:13">4:13</span> s wakka a ndukkel deg yiwen n liman akk-d
yiwet n tmusni ɣef wayen yeɛnan Mmi-s n Sidi Ṛebbi, alamma nuɣal d
imeqqranen di liman, imiren s tdukli-nneɣ a naweḍ ɣer lekmal yellan di
Lmasiḥ.

<span id="an_4:14">4:14</span> S wakka ur nețɣimi ara d arrac imeẓyanen
i țțawint lemwaji, i gekkat waḍu ɣer mkul lǧiha, iwumi țkellixen
yemdanen s tḥila-nsen d uselmed-nsen.

<span id="an_4:15">4:15</span> Meɛna m'ara d-nini tideț s lmaḥibba, a
nennerni di kullec di tikli-nneɣ akk-d Lmasiḥ yellan d aqeṛṛuy n tejmaɛt
n watmaten.

<span id="an_4:16">4:16</span> Lǧețța tebna s leɛḍam i cuddent lemfaṣel
yal yiwen deg umkan-is ; s Lmasiḥ i tețțimɣuṛ tețnerni di lmaḥibba m'ara
xeddmen leɛḍam ccɣel-nsen akken ilaq.

<span id="an_4:17">4:17</span> Atah ihi wayen i d-nniɣ d wayen i
d-qqaṛeɣ s yisem n Lmasiḥ : ur țɛicit ara am wid ur nessin ara Sidi
Ṛebbi, imi nutni ttabaɛen ixemmimen-nsen ur nesɛi lmeɛna.

<span id="an_4:18">4:18</span> Lefhama-nsen iɣumm-iț ṭṭlam, sbeɛden
iman-nsen ɣef tudert i d ițțak Sidi Ṛebbi axaṭer jehlen, qquṛen
wulawen-nsen.

<span id="an_4:19">4:19</span> Ur țsetḥin ur țneḥcamen, xeddmen anagar
leɛṛuṛ yerna ur sɛin ara leqniɛa.

<span id="an_4:20">4:20</span> Meɛna kunwi, mačči d ayagi i tlemdem ɣef
wayen yeɛnan Lmasiḥ.

<span id="an_4:21">4:21</span> Ma yella d ṣṣeḥ teslam yis yerna slemden
awen tideț yeɛnan Lmasiḥ,

<span id="an_4:22">4:22</span> eǧǧet ihi tikli d lɛeqliyat-nwen tiqdimin
i ssefsadent ccehwat-nni yețɣuṛṛun,

<span id="an_4:23">4:23</span> iwakken aț-țesɛum lefhama tajdiṭ i
d-yețțak Ṛṛuḥ iqedsen ;

<span id="an_4:24">4:24</span> aț-țuɣalem d imdanen ijdiden akken yella
di lebɣi n Sidi Ṛebbi, iwakken aț-țeddum deg webrid n lḥeqq akk-d ṣṣfa i
ɣer tessufuɣ tideț.

<span id="an_4:25">4:25</span> Daymi, xḍut i lekdeb, ilaq mkul yiwen
deg-wen ad immeslay s tideț i lɣiṛ-is, axaṭer d leɛḍam n yiwet n lǧețța
i nella.

<span id="an_4:26">4:26</span> Ma yella terfam ur dennbet ara, ilaq a
wen-yekkes wurrif uqbel ad yeɣli yiṭij ;

<span id="an_4:27">4:27</span> ur țțaǧat ara abrid i Cciṭan ad ikcem
ulawen nwen.

<span id="an_4:28">4:28</span> Win yellan yețțakar ilaq ad yeǧǧ tukerḍa,
ad ixdem s iɣallen-is iwakken a d-isewweṛ aɣṛum-is s leḥlal, s wakka ad
izmir ad iɛiwen wid yeḥwaǧen.

<span id="an_4:29">4:29</span> Ur ilaq ara a d-iffeɣ yir wawal seg imi
nwen, meɛna init-ed kan imeslayen iṣelḥen s wayes ara sfaydin wid ara
sen-islen yerna ad nnernin di liman.

<span id="an_4:30">4:30</span> Ur sseḥzanet ara Ṛṛuḥ iqedsen, axaṭer yis
i kkun-iḍbeɛ Sidi Ṛebbi i wass i deg ara innekmal leslak-nwen.

<span id="an_4:31">4:31</span> Ekkset si gar-awen : ddɣel, taɛdawit,
zzɛaf, urrif, leɛyaḍ, rregmat d wayen akk n diri.

<span id="an_4:32">4:32</span> Ilit d iḥninen msamaḥet wway gar-awen,
akken i wen-isemmeḥ Sidi Ṛebbi s Lmasiḥ.

<span id="an_5:1">5:1</span> Cbut Sidi Ṛebbi baba-twen imi tellam d
arraw-is eɛzizen ;

<span id="an_5:2">5:2</span> ddut di tudert-nwen s leḥmala am akken i
ɣ-iḥemmel Lmasiḥ i gefkan iman-is ɣef ddemma-nneɣ d asfel i Sidi Ṛebbi,
am lweɛda yesɛan rriḥa iɛeǧben i Sidi Ṛebbi.

<span id="an_5:3">5:3</span> Kunwi s imasiḥiyen xḍut i zzna, i wayen ur
neṣfi ara, i ṭṭmeɛ ; ayagi meṛṛa ur ilaq ara a d-ițwabdar gar-awen
ulamma d abdar ;

<span id="an_5:4">5:4</span> ur qebbḥet, ur d-qqaṛet imeslayen ur nesɛi
lmeɛna, ur stehzayet ara ɣef wiyaḍ, ayagi d ayen n diri ; meɛna d
imeslayen ara iḥemden Sidi Ṛebbi i glaqen a d-ffɣen seg imi-nwen ;

<span id="an_5:5">5:5</span> axaṭer ilaq aț-țeẓrem belli wid izennun neɣ
wid ixeddmen ayen ur neṣfi akk-d wid ur nesteqniɛ ara, imi wid ur
nesteqniɛ ara cban wid iɛebbden lmeṣnuɛat, widak-agi meṛṛa ur țekkayen
ara di tgeldit n Lmasiḥ akk-d Sidi Ṛebbi.

<span id="an_5:6">5:6</span> Ḥadret ad yili win ara kkun-ikellxen s
lehduṛ ur nesɛi lmeɛna ; axaṭer ɣef wayagi i d-ițrusu wurrif n Sidi
Ṛebbi ɣef wid i t ițɛuṣun.

<span id="an_5:7">5:7</span> Ɣuṛ-wat ad yili wayen ara kkun icerken
yid-sen.

<span id="an_5:8">5:8</span> Ț-țideț, zik-nni tellam di ṭṭlam, meɛna
tura s tikli nwen deg ubrid n Lmasiḥ, teffɣem ɣer tafat. Ddut ihi di
tafat imi tellam d arraw n tafat ;

<span id="an_5:9">5:9</span> axaṭer tafat tețțawi ɣer wayen yelhan, ɣer
lḥeqq akk-d tideț.

<span id="an_5:10">5:10</span> Meyyzet aț-țfehmem ayen iɛeǧben i Sidi
Ṛebbi ;

<span id="an_5:11">5:11</span> ur țekkayet ara di lecɣal n ṭṭlam ur
nesɛi lfayda, meɛna kecfet-ed lecɣal-agi ɣer tafat.

<span id="an_5:12">5:12</span> S tideț ayen xeddmen imdanen agi s
tuffra, d lḥecma a t-id-nebder ;

<span id="an_5:13">5:13</span> lameɛna m'ara d-nekcef ayen xeddmen,
lecɣal-nsen țbanen-d di tafat imi anda tella tafat, kullec yețban di
tafat.

<span id="an_5:14">5:14</span> Axaṭer yura : Ddekwal a kečč yeṭṭsen,
ekker-ed si ger lmegtin, imiren a d-tecṛeq fell-ak tafat n Lmasiḥ.+

<span id="an_5:15">5:15</span> Ihi ḥadret mliḥ tikli-nwen, ur leḥḥut ara
am imehbal, meɛna am wid itɛeqlen ;

<span id="an_5:16">5:16</span> faṛset lweqt axaṭer ussan d ixeddaɛen.

<span id="an_5:17">5:17</span> Daymi, ur smuhbulet ara, meɛna nadit
aț-țissinem lebɣi n Sidi Ṛebbi.

<span id="an_5:18">5:18</span> Ur sekkṛet ara, axaṭer ssekṛan yessufuɣ i
webrid, lameɛna ǧǧet Ṛṛuḥ iqedsen ad iččaṛ ulawen-nwen ;

<span id="an_5:19">5:19</span> țemseǧhadet wway-gar-awen s isefra n
Zabuṛ, cnut ɣef Sidi Ṛebbi, țḥemmidet-eț s wulawen-nwen.

<span id="an_5:20">5:20</span> Ḥemmdet Sidi Ṛebbi baba-tneɣ ɣef kullec,
di mkul lweqt, s yisem n Sidna Ɛisa Lmasiḥ.

<span id="an_5:21">5:21</span> Țemqadaṛet wway gar-awen i lmend n
Lmasiḥ.

<span id="an_5:22">5:22</span> A tilawin, ḍuɛemt irgazen-nkunt am akken
i glaq aț țḍuɛemt Sidi Ṛebbi !

<span id="an_5:23">5:23</span> Axaṭer argaz d aqeṛṛuy ɣef tmeṭṭut-is, am
akken yella Lmasiḥ d aqeṛṛuy ɣef tejmaɛt n imasiḥiyen ; Lmasiḥ d
amsellek n tejmaɛt-agi yellan d lǧețța-s.

<span id="an_5:24">5:24</span> Am akken i tețḍuɛu tejmaɛt n imasiḥiyen
Lmasiḥ, tilawin daɣen ilaq ad ḍuɛent irgazen-nsent di kullec.

<span id="an_5:25">5:25</span> Ay irgazen, mkul yiwen deg-wen ilaq ad
iḥemmel tameṭṭut-is akken iḥemmel Lmasiḥ tajmaɛt n imasiḥiyen yerna
isebbel tudert-is fell-as ;

<span id="an_5:26">5:26</span> s weɣḍas n waman d wawal n Sidi Ṛebbi
yerra-ț teṣfa iwakken a ț-iqeddem i Sidi Ṛebbi ;

<span id="an_5:27">5:27</span> yebɣa a d-tban tejmaɛt-agi n imasiḥiyen
zdat-es, teṣfa mbla ccwami, mbla akmac, mbla lɛib, aț-țili ț țimqeddest,
ulac deg-s ayen ara s-d-nessukkes.

<span id="an_5:28">5:28</span> Akken daɣen, irgazen ilaq ad ḥemmlen
tilawin-nsen am lǧețțat-nsen. Win iḥemmlen tameṭṭut-is d iman-is i
gḥemmel.

<span id="an_5:29">5:29</span> Ulac win ikeṛhen lǧețța-s, lameɛna
ițɛeyyic-iț, ițḥadar-iț am akken i gxeddem Lmasiḥ i tejmaɛt n
imasiḥiyen,

<span id="an_5:30">5:30</span> axaṭer d leɛḍam n lǧețța-s i nella ; am
akken yura :

<span id="an_5:31">5:31</span> daymi argaz ilaq ad yeǧǧ baba-s d yemma-s
iwakken ad iɛic ț-țmeṭṭut-is, dɣa di sin yid-sen ad uɣalen d yiwen.

<span id="an_5:32">5:32</span> Bɣiɣ a wen-d-iniɣ belli ayagi d lbaḍna
tameqqrant yeɛnan Lmasiḥ akk-d tejmaɛt n imasiḥiyen.

<span id="an_5:33">5:33</span> Ihi, mkul yiwen deg-wen ilaq ad iḥemmel
tameṭṭut-is am yiman-is, ma ț-țameṭṭut ilaq aț-țqadeṛ argaz-is.

<span id="an_6:1">6:1</span> A dderya, ḍuɛet imawlan-nwen akken yebɣa
Sidi Ṛebbi, axaṭer ayagi d ayen yellan d lḥeqq.

<span id="an_6:2">6:2</span> Di ccariɛa i d-yefka Sidi Ṛebbi i Sidna
Musa, d wagi i d lameṛ amezwaru yesɛan lemɛahda :

<span id="an_6:3">6:3</span> Qadeṛ baba-k d yemma-k iwakken aț-țɛiceḍ di
lehna yerna ad yiɣzif leɛmeṛ-ik di ddunit.+

<span id="an_6:4">6:4</span> Ma d kunwi ay imawlan, ur sserfayet ara
dderya-nwen, meɛna ṛebbit-țen akken i kkun-id-iweṣṣa Sidi Ṛebbi.

<span id="an_6:5">6:5</span> Ay aklan, ḍuɛet imɛellmen n ddunit-agi, s
leqdeṛ, s wul yeṣfan d neyya am akken d Lmasiḥ i tețḍuɛum .

<span id="an_6:6">6:6</span> Ayagi ur t-xeddmet ara kan m'ara
kkun-țɛassan am akken tețnadim aț-țɛeǧbem i yemdanen, lameɛna ilit am
iqeddacen n Lmasiḥ i gxeddmen lebɣi n Sidi Ṛebbi seg ul.

<span id="an_6:7">6:7</span> Qedcet fell-asen am akken d Lmasiḥ s
yiman-is i ɣef tqeddcem mačči ɣef yemdanen.

<span id="an_6:8">6:8</span> Teẓram belli win ixedmen lxiṛ, ama d akli
neɣ d aḥeṛṛi, a t-yaf ɣer Sidi Ṛebbi.

<span id="an_6:9">6:9</span> Ma d kunwi ay imɛellmen, xedmet daɣen i
iqeddacen-nwen akken yella di lebɣi n Sidi Ṛebbi ; ur xeddmet ara
deg-sen lbaṭel, imi teẓram belli amɛellem-nsen yellan daɣen d
amɛellem-nwen, deg igenwan i gella yerna nețța ur ixeddem ara lxilaf ger
yemdanen.

<span id="an_6:10">6:10</span> Sǧehdet ihi iman-nwen s tezmert
tameqqrant n Sidi Ṛebbi.

<span id="an_6:11">6:11</span> Refdet leslaḥ meṛṛa i d-yețțak Sidi
Ṛebbi, iwakken ur kkun-iɣelleb ara Cciṭan s tḥila-ines ;

<span id="an_6:12">6:12</span> axaṭer imenɣi-nneɣ mačči akk d yemdanen i
gella, lameɛna nețnaɣ ț-țgeldiwin akk-d tnezmarin n ṭṭlam iḥekmen di
ddunit-agi, akk-d leṛwaḥ imcumen yellan deg igenwan.

<span id="an_6:13">6:13</span> Ɣef wayagi, refdet leslaḥ n Sidi Ṛebbi,
iwakken aț-țizmirem aț-țqablem tiswiɛin iweɛṛen, s wakka ɣer taggara n
umenuɣ a kkun-id-yaf lḥal mazal tbeddem.

<span id="an_6:14">6:14</span> Heggit iman-nwen am iɛsekṛiwen, beggset
tideț, lset lḥeqq ɣef yidmaren-nwen am tidemmar n wuzzal.

<span id="an_6:15">6:15</span> Sɛut zzwaṛa am warkasen deg
yidaṛṛen-nwen, daymen theggam i ubecceṛ n lexbaṛ n lxiṛ.

<span id="an_6:16">6:16</span> Di yal taswiɛt, sɛut liman am tseddarit s
wayes ara tizmirem aț-țsensem isufa n tmes n Cciṭan ;

<span id="an_6:17">6:17</span> cuddet leslak am tcacit n wuzzal ɣef
wuqeṛṛuy-nwen, sxedmet asekkin n Ṛṛuḥ iqedsen yellan d awal n Sidi
Ṛebbi.

<span id="an_6:18">6:18</span> Di yal taswiɛt deɛɛut ɣer Sidi Ṛebbi s
Ṛṛuḥ iqedsen s yal taẓallit d uḥellel. Ɛiwzet ihi, kettṛet di tẓallit,
deɛɛut ɣer Sidi Ṛebbi ɣef watmaten meṛṛa ;

<span id="an_6:19">6:19</span> deɛɛut daɣen fell-i iwakken a yi-d-yefk
Sidi Ṛebbi imeslayen s wayes ara beccṛeɣ lbaḍna n lexbaṛ n lxiṛ mbla
tugdi.

<span id="an_6:20">6:20</span> Țțuceggɛeɣ-d ad beccṛeɣ lexbaṛ n lxiṛ ;
ɣef ddemma-s i lliɣ d ameḥbus ; dɛut ihi iwakken a t-beccṛeɣ akken ilaq,
mbla tugdi.

<span id="an_6:21">6:21</span> Bɣiɣ aț-țeẓrem ula d kunwi ayen i
yi-yeɛnan d wayen i xeddmeɣ, daymi i n-ceggɛeɣ gma-tneɣ ameɛzuz Tucik
iqeddcen seg ul ɣef Sidna Ɛisa, d nețța ara kkun-isɛelmen ɣef kullec.

<span id="an_6:22">6:22</span> Ceggɛeɣ-t-in iwakken aț-țeẓrem ayen akk i
ɣ-iɛnan, yerna ad isseǧhed ulawen-nwen.

<span id="an_6:23">6:23</span> Ay atmaten, lehna, leḥmala akk-d liman a
wen-d-țțunefken s ɣuṛ Sidi Ṛebbi baba-tneɣ akk-d Sidna Ɛisa Lmasiḥ.

<span id="an_6:24">6:24</span> Ṛṛeḥma n Sidi Ṛebbi a d-ters ɣef wid akk
iḥemmlen Sidna Ɛisa Lmasiḥ s lmaḥibba ur nfennu !
