i nebɣa a nesseɣli, meɛna d lqima-ines n ṣṣeḥ i s-nefka.

<span id="an_4:1">4:1</span> D acu ara d-nini ihi ɣef Sidna Ibṛahim,
jeddi-tneɣ ? Amek i tedṛa yid-es ?

<span id="an_4:2">4:2</span> Lemmer yețwaḥseb d aḥeqqi s lxiṛ yexdem,
tili yesɛa sebba s wayes ara izuxx, meɛna ur yezmir ara ad izuxx zdat
Sidi Ṛebbi.

<span id="an_4:3">4:3</span> Akken yura di tira iqedsen : Sidna Ibṛahim
yețkel ɣef Ṛebbi, daymi i gețwaḥseb d aḥeqqi .

<span id="an_4:4">4:4</span> Axeddam yuklal lexlaṣ-is, lexlaṣ ines ur
s-ițwaḥseb ara d lemziya meɛna d lḥeqq-is.

<span id="an_4:5">4:5</span> Akken daɣen win ur nețkil ara ɣef lxiṛ i
gxeddem, meɛna yețkel ɣef Ṛebbi i gețțarran amednub d aḥeqqi, ad
yețwaḥseb d aḥeqqi ɣef ddemma n liman-is.

<span id="an_4:6">4:6</span> Akka i gessefra Sidna Dawed ɣef wemdan i
gḥețțeb Sidi Ṛebbi d aḥeqqi mbla ma yemmuqel ɣer lefɛayel-is :

<span id="an_4:7">4:7</span>  D iseɛdiyen wid iwumi țwasemḥent
seyyat-nsen, iwumi țwamḥan ddnubat -nsen !

<span id="an_4:8">4:8</span>  D aseɛdi wemdan iwumi ur iḥsib ara Sidi
Ṛebbi ddnub-is !

<span id="an_4:9">4:9</span> Lfeṛḥ-agi, yella i wid iḍehṛen kan neɣ
yella daɣen i wid ur neḍhiṛ-ara ? Nenna-d belli : Ṛebbi iḥseb Sidna
Ibṛahim d aḥeqqi ɣef ddemma n liman-ines.

<span id="an_4:10">4:10</span> Melmi ițwaḥseb Sidna Ibṛahim d aḥeqqi,
uqbel ad yeḍheṛ neɣ armi geḍheṛ ? Mačči armi geḍheṛ meɛna uqbel !

<span id="an_4:11">4:11</span> ?ṭhaṛa tusa-yas-ed d țbut, d ccama i
d-isbeggnen belli Sidi Ṛebbi yerra-t d aḥeqqi ɣef ddemma n liman-ines
uqbel ad yeḍheṛ. S wakka Sidna Ibṛahim yuɣal d jedd n wid akk yumnen, i
gețwaḥesben d iḥeqqiyen ɣer Sidi Ṛebbi ɣas ur ḍhiṛen ara.

<span id="an_4:12">4:12</span> Yella daɣen d jedd n wid akk iḍehṛen, ur
nețkil ara ɣef ṭṭhaṛa-nsen kan meɛna tebɛen lateṛ-is, sɛan liman i gesɛa
uqbel ad yeḍheṛ.

<span id="an_4:13">4:13</span> Ayen akk i gewɛed Sidi Ṛebbi d lweṛt di
ddunit i Sidna Ibṛahim akk-d dderya n dderya-s, ur s-t-iwɛid ara imi i
getbeɛ ccariɛa, meɛna imi i gețkel fell-as. Daymi i t-yeḥseb Sidi Ṛebbi
d aḥeqqi.

<span id="an_4:14">4:14</span> Lemmer d wid ittabaɛen ccariɛa ara
iweṛten, iwumi ihi liman, iwumi lweɛd n Sidi Ṛebbi.

<span id="an_4:15">4:15</span> Axaṭer d ccariɛa i d-ițawin urrif n
Ṛebbi, lemmer ulac ccariɛa ur d-tețțili ara tṛuẓi n ccariɛa.

<span id="an_4:16">4:16</span> Lweɛd i d-yefka Sidi Ṛebbi i Sidna
Ibṛahim, yebna ɣef liman iwakken ad yili d lemziya, ad țekkin deg-s
dderya n dderya-s mačči d wid kan yessnen ccariɛa lameɛna wid akk yumnen
am nețța, yuɣal d jeddi-tneɣ nukni akk s wid yumnen.

<span id="an_4:17">4:17</span> Am akken yura di Tawṛat n Sidna Musa :
Sbeddeɣ-k iwakken aț-țiliḍ d jedd n waṭas n leǧnas. Yuɣal d jeddi-tneɣ
zdat Ṛebbi s wayes i gumen, i d-iḥeggun lmegtin, i d-ixelqen ayen ur
nelli ara.

<span id="an_4:18">4:18</span> ?as akken ur d-iqqim usirem, Sidna
Ibṛahim yuṛǧa s lețkal d ameqqran armi yuɣal d jedd n waṭas n yegduden
akken i s-yenna Sidi Ṛebbi : Akka ara tili dderya n dderya-k.

<span id="an_4:19">4:19</span> Liman n Sidna Ibṛahim ur yenqis ara ɣas
akken qṛib ad issiweḍ meyya iseggasen, iwala iman-is dayen ur yezmir ara
ad isɛu dderya, yerna tameṭṭut-is Saṛa teǧǧa-ț dderya.

<span id="an_4:20">4:20</span> Yețkel ɣef Sidi Ṛebbi, ur t-yekcim ara
ccekk ɣef wayen i s-yewɛed. D liman yesɛa i t-isǧehden, s wakka i
gețḥemmid Sidi Ṛebbi Bab n tezmert.

<span id="an_4:21">4:21</span> Yețkel belli Sidi Ṛebbi yezmer a s-d-yefk
ayen i s-yewɛed.

<span id="an_4:22">4:22</span> ?ef wannect-agi i t-yeḥseb Sidi Ṛebbi d
aḥeqqi.

<span id="an_4:23">4:23</span> Mačči fell-as kan i gura di tira iqedsen
: Yețwaḥseb d aḥeqqi,

<span id="an_4:24">4:24</span> lameɛna fell-aɣ ula d nukni, imi numen s
win i d-isseḥyan Ɛisa Ssid nneɣ si ger lmegtin.

<span id="an_4:25">4:25</span> Yețțunefk ad immet ɣef ddemma n ddnubat
nneɣ, isseḥya-t-id Sidi Ṛebbi si ger lmegtin iwakken a nuɣal d
iḥeqqiyen.

<span id="an_5:1">5:1</span> Tura imi nuɣal d iḥeqqiyen s liman, ters-ed
lehna gar-aɣ d Ṛebbi s Ssid-nneɣ Ɛisa Lmasiḥ.

<span id="an_5:2">5:2</span> Yis daɣen i nufa abrid s liman ɣer ṛṛeḥma n
Sidi Ṛebbi i deg nella tura. Lfeṛḥ-nneɣ, d asirem nesɛa belli a nțekki
di lɛaḍima n Sidi Ṛebbi.

<span id="an_5:3">5:3</span> Ula di lmeḥnat, nețțili di lfeṛḥ axaṭer
neẓra lmeḥna tețțawi-d ṣṣbeṛ,

<span id="an_5:4">5:4</span> ṣṣbeṛ isseǧhad deg ujeṛṛeb, lǧehd deg
ujeṛṛeb ițțawi-d asirem.

<span id="an_5:5">5:5</span> Asirem-agi ulac deg-s lexdeɛ, axaṭer
leḥmala n Sidi Ṛebbi teẓẓa deg ulawen-nneɣ s Ṛṛuḥ iqedsen i
ɣ-d-yețțunefken ;

<span id="an_5:6">5:6</span> asm'akken i ɣ-iɣleb ddnub ur s-nezmir ara,
Lmasiḥ yemmut ɣef ddemma n imednuben di lweqt ilaqen.

<span id="an_5:7">5:7</span> Iwɛeṛ i wemdan ad iqbel ad immet ɣef ddemma
n uḥeqqi ; ahat ad yili win ara iqeblen ad immet ɣef ddemma n wemdan
yelhan ;

<span id="an_5:8">5:8</span> ma d Lmasiḥ, nețța yemmut fell-aɣ m'akken
nella d imednuben ; s wakka i ɣ-d-ibeggen Sidi Ṛebbi leḥmala-ines.

<span id="an_5:9">5:9</span> Imi nuɣal tura d iḥeqqiyen ɣer Sidi Ṛebbi s
idammen n Lmasiḥ yuzzlen fell-aɣ, yis daɣen ara nețțusellek seg urrif n
Sidi Ṛebbi.

<span id="an_5:10">5:10</span> Asm'akken nella d iɛdawen-is, yefka-d
Mmi-s iwakken s lmut-is a ɣ-issemṣaleḥ yid-es. Tura imi nemṣalaḥ d Sidi
Ṛebbi, a nețțusellek daɣen s tudert n Mmi-s.

<span id="an_5:11">5:11</span> Mačči d ayagi kan, lameɛna tura
lfeṛḥ-nneɣ atan di Sidi Ṛebbi, s Sidna Ɛisa Lmasiḥ i ɣ-issemṣalaḥen
yid-es.

<span id="an_5:12">5:12</span> S wakka ihi, ddnub ikcem-ed ɣer ddunit s
yiwen wemdan, amdan agi d Adem ; ddnub-agi yewwi-d lmut, lmut tețțawi
akk imdanen, axaṭer imdanen meṛṛa denben.

<span id="an_5:13">5:13</span> Uqbel a d-yefk Sidi Ṛebbi ccariɛa i Musa,
ddnub yella yakan di ddunit ; meɛna imdanen ur țțuḥasben ara s ccariɛa
imi urɛad i d-tusi.

<span id="an_5:14">5:14</span> ?as akka, si zzman n Adem armi d zzman n
Sidna Musa lmut teḥkem ula ɣef yemdanen ur nɛuṣa ara Ṛebbi am akken i
t-iɛuṣa Adem, nețța yellan d lemtel n win ara d-yasen.

<span id="an_5:15">5:15</span> Ur nezmir ara a nmettel ddnub n Adem ɣer
lemziya d ṛṛeḥma n Sidi Ṛebbi ; Adem iɛuṣa Sidi Ṛebbi yiwet n tikkelt
yewwi-d lmut i yemdanen meṛṛa, acḥal ihi meqqṛet ṛṛeḥma n Sidi Ṛebbi i
d-yețțunefken i yemdanen s Ɛisa Lmasiḥ.

<span id="an_5:16">5:16</span> Ṛṛeḥma-agi n Sidi Ṛebbi mačči am ddnub n
yiwen wemdan. Axaṭer yiwen n ddnub yewwi-d lḥisab d lɛiqab, ma d ṛṛeḥma
n Sidi Ṛebbi, tețțekkes lḥisab d lɛiqab ɣef waṭas n ddnubat.

<span id="an_5:17">5:17</span> Ma yella lmut teḥkem ɣef yemdanen meṛṛa s
ddnub n yiwen wemdan, ihi wid iwumi d-tețțunefk ṛṛeḥma d lḥeqq, ad
ḥekmen yerna ad idiren s Ɛisa Lmasiḥ.

<span id="an_5:18">5:18</span> S wakka, ddnub n yiwen wemdan yesseɣli-d
lɛiqab ɣef yemdanen meṛṛa, akken daɣen lḥeqq i gexdem yiwen wemdan,
yewwi-d leslak i gessidiren imdanen meṛṛa.

<span id="an_5:19">5:19</span> Ihi am akken ɛeṣṣu n yiwen wemdan, yerra
aṭas n yemdanen d imednuben, akken daɣen ṭṭaɛa n yiwen wemdan terra aṭas
n yemdanen d iḥeqqiyen.

<span id="an_5:20">5:20</span> Ccariɛa tger-ed iman-is iwakken a
d-tbeggen acḥal meqqeṛ ddnub, lameɛna anda yekteṛ ddnub, ṛṛeḥma n Sidi
Ṛebbi meqqṛet akteṛ.

<span id="an_5:21">5:21</span> S wakka ddnub yeḥkem s lmut, ma d ṛṛeḥma
n Sidi Ṛebbi teḥkem s lḥeqq, tețțawi ɣer tudert n dayem s Ɛisa Lmasiḥ
Ssid-nneɣ.

<span id="an_6:1">6:1</span> Acu ara d-nini ihi ? A nkemmel a nețɛici di
ddnub iwakken ṛṛeḥma n Sidi Ṛebbi aț-țimɣuṛ ?

<span id="an_6:2">6:2</span> Ala ! Nukni i gemmuten ɣef ddemma n ddnub,
amek ara nkemmel a nețɛici deg-s ?

<span id="an_6:3">6:3</span> Neɣ ur teẓrim ara belli nukni akk i
gețwaɣeḍsen iwakken a neddukel d Ɛisa Lmasiḥ, nețwaɣḍeṣ yid-es di
lmut-is ?

<span id="an_6:4">6:4</span> S weɣḍas, nețwamḍel yid-es iwakken, nukni
daɣen a d-neḥyu akken i d-yeḥya Lmasiḥ si ger lmegtin s tezmert n Ṛebbi,
a nekcem di tudert tajḍiṭ.

<span id="an_6:5">6:5</span> Axaṭer ma necrek yid-es di lmut-is, a
necrek daɣen yid-es di ḥeggu-ines.

<span id="an_6:6">6:6</span> Ilaq-aɣ a nẓer belli amdan-nni i nella zik,
yemmut, ițwasemmeṛ akk-d Lmasiḥ iwakken aț-țemmet lǧețța yeččuṛen d
ddnub, ur nețțili ara d aklan n ddnub.

<span id="an_6:7">6:7</span> Axaṭer win yemmuten islek si ddnub,

<span id="an_6:8">6:8</span> imi nemmut akk-d Lmasiḥ, numen daɣen belli
a nidir yid-es.

<span id="an_6:9">6:9</span> Neẓra belli Lmasiḥ seg wasmi i d-yeḥya si
ger lmegtin, ur yețțuɣal ara ad immet, lmut ur tezmir ara aț-țeḥkem
fell-as.

<span id="an_6:10">6:10</span> Axaṭer taluft n ddnub yefra-ț ɣef yiwet n
tikkelt i dayem, s lmut-is. Tura atan d lḥey, idder i Sidi Ṛebbi.

<span id="an_6:11">6:11</span> Ula d kunwi ḥesbet iman-nwen am akken
temmutem ɣef wayen yeɛnan ddnub. Ɛicet i Ṛebbi di tdukkli akk-d Ɛisa
Lmasiḥ.

<span id="an_6:12">6:12</span> Tura ur țțaǧat ara ddnub ad iḥkem
fell-awen, ur xeddmet ara ccehwat n tnefsit-nwen am zik.

<span id="an_6:13">6:13</span> Ur țțaǧat ara lemfaṣel n lǧețțat-nwen ad
ɣlint di ddnub, iwakken ad ixdem yis-sen cceṛ ; lameɛna qeddmet
iman-nwen i Ṛebbi am win i d-iḥyan si lmut iwakken aț-țqeddcem fell-as a
ț-țxeddmem anagar lxiṛ.

<span id="an_6:14">6:14</span> Ddnub ur yețțuɣal ara ad iḥkem fell-awen
axaṭer ur tellim ara seddaw n ccariɛa lameɛna seddaw n ṛṛeḥma n Sidi
Ṛebbi.

<span id="an_6:15">6:15</span> Acu ara d-nini ihi ? A neǧǧ ddnub ad
idebbeṛ fell-aɣ tura imi nella seddaw n ṛṛeḥma n Sidi Ṛebbi mačči seddaw
n ccariɛa ? A ɣ-imneɛ Ṛebbi deg wannect-agi !

<span id="an_6:16">6:16</span> Ur teẓrim ara belli ma tuɣalem d aklan n
yiwen iwakken ad idebbeṛ fell-awen, ilaq a s-taɣem awal ama d ddnub
yețțawin ɣer lmut neɣ d ṭṭaɛa n Ṛebbi yețțawin ɣer tudert taḥeqqit.

<span id="an_6:17">6:17</span> Lameɛna a neḥmed Ṛebbi imi zik tellam d
aklan n ddnub, ma tura tḍuɛem s wulawen-nwen awal i tlemdem.

<span id="an_6:18">6:18</span> Tețțusellkem-d si ddnub tuɣalem d aklan n
lḥeqq.

<span id="an_6:19">6:19</span> Tura a wen-d-hedṛeɣ s lɛeqliya nwen
iwakken aț-țfehmem axaṭer annect-agi yewɛeṛ i lefhama. Am akken tsellmem
iman-nwen zik d aklan i lɛaṛ iwakken aț-țxedmem lɛaṛ, tura sellmet
iman-nwen d aklan i lḥeqq d ṣṣwab iwakken aț-țeṣfu tudert-nwen.

<span id="an_6:20">6:20</span> Asmi tellam d aklan n ddnub texḍam i
lḥeqq.

<span id="an_6:21">6:21</span> Acu i trebḥem s lecɣal-nni s wayes
tessetḥam tura, lecɣal-nni i gețțawin ɣer lmut.

<span id="an_6:22">6:22</span> Tura tețwaselkem si ddnub, tuɣalem d
iqeddacen n Ṛebbi, lfayda nwen ț-țikli deg webrid yeṣfan akk-d tudert n
dayem.

<span id="an_6:23">6:23</span> Ddnub yețțawi ɣer lmut, ma d ṛṛeḥma n
Sidi Ṛebbi tewwi-yaɣ-d tudert n dayem di tikli-nneɣ akk-d Ɛisa Lmasiḥ
Ssid-nneɣ.

<span id="an_7:1">7:1</span> Ay atmaten ur teẓrim ara belli ccariɛa
teḥkem ɣef wemdan skud mazal-it idder ? Heddṛeɣ-ed akka i wid yessnen
ccariɛa.

<span id="an_7:2">7:2</span> A wen-d-fkeɣ yiwen n lemtel : tameṭṭut
tețwarez ɣer wergaz-is s ccariɛa skud mazal-it yedder, meɛna ma yemmut
wergaz-is tserreḥ-as ccariɛa ma tebɣa aț-țɛiwed zzwaǧ ;

<span id="an_7:3">7:3</span> ma tuɣ wayeḍ skud mazal argaz-is idder,
yețțusemma tezna ; meɛna ma yemmut wergaz-is tserreḥ-as ccariɛa
aț-țɛiwed zzwaǧ, ayagi ur yețțusemma ara d leḥṛam.

<span id="an_7:4">7:4</span> Ula d kunwi ay atmaten, temmutem ɣef wayen
yeɛnan ccariɛa s lmut n Lmasiḥ, akken aț-țeṭṭfem deg win i d-yeḥyan si
ger lmegtin iwakken aț-țxedmem ayen i s-iɛeǧben i Sidi Ṛebbi.

<span id="an_7:5">7:5</span> Axaṭer d ccariɛa i d-isbeggnen amek i
nțeddu s lebɣi n tnefsit-nneɣ d ccehwat-nneɣ n diri ; d nețțat daɣen i
d-isbeggnen amek yexdem ddnub-agi deg-nneɣ, armi i ɣ-yeṣṣaweḍ ɣer lmut.

<span id="an_7:6">7:6</span> Lameɛna tura nețțusellek si lḥekma n
ccariɛa imi nețțusemma nemmut ɣef wayen yeɛnan ccariɛa nni i ɣ-yurzen, s
wakka i nezmer tura a nḍuɛ Sidi Ṛebbi s lɛeqliya tajḍiṭ n Ṛṛuḥ iqedsen,
mačči am zik s lɛeqliya-nni taqdimt mi nella seddaw n ccariɛa.

<span id="an_7:7">7:7</span> Acu ara d-nini ihi, eɛni ccariɛa deg-s
ddnub ? Xaṭi ! Lameɛna lemmer ulac ccariɛa tili ur țissineɣ ara d acu i
d ddnub. Axaṭer lemmer ur d-tenni ara ccariɛa : ur ṭṭamaɛ ara , tilli ur
ẓriɣ ara belli ṭṭmeɛ d leḥṛam ?

<span id="an_7:8">7:8</span> Ddnub ifuṛes tagniț s lameṛ n ccariɛa ;
yesnulfa-d deg-i mkul ṣṣenf n ṭṭmeɛ n diri ; axaṭer mbla ccariɛa ur
d-ițban ara d acu i d ddnub.

<span id="an_7:9">7:9</span> Zik, asmi ulac ccariɛa lliɣ țɛiciɣ s nneya,
mi d-iffeɣ lameṛ n ccariɛa, ddnub yuki-d deg-i, yeṣṣaweḍ-iyi ɣer lmut.

<span id="an_7:10">7:10</span> S wakka, lameṛ i glaqen ad iyi-ssiweḍ ɣer
tudert, yewwi-yi ɣer lmut

<span id="an_7:11">7:11</span> axaṭer ddnub yufa abrid, s lameṛ-agi n
ccariɛa dɣa ixdeɛ-iyi, yewwi-yi ɣer lmut.

<span id="an_7:12">7:12</span> Ihi, ccariɛa s yiman-is teṣfa, lumuṛat
yellan deg-s ṣfan, lhan yerna d lḥeqq.

<span id="an_7:13">7:13</span> Eɛni yuɣal-iyi wayen yelhan d sebba n
lmut ? Xaṭi ! D ddnub i yi-ṣṣawḍen ɣer lmut s wayen yelhan, iwakken s
ccariɛa, a d-tban acḥal meqqṛet tezmert n ddnub.

<span id="an_7:14">7:14</span> Neẓra belli ccariɛa n Musa tețțunefk-ed s
Ṛṛuḥ n Sidi Ṛebbi. Ma d nekk, d amdan kan i lliɣ yerna imlek-iyi ddnub.

<span id="an_7:15">7:15</span> Ur ẓriɣ ara acu i xeddmeɣ, ayen bɣiɣ a
t-xedmeɣ ur t-xeddmeɣ ara lameɛna d ayen akken keṛheɣ i xeddmeɣ.

<span id="an_7:16">7:16</span> Ma yella ayen i xeddmeɣ d ayen akken ur
bɣiɣ ara a t-xedmeɣ, tḥeqqeqeɣ belli ccariɛa telha.

<span id="an_7:17">7:17</span> Tura ihi mačči d nekk i gxeddmen ayagi,
meɛna d ddnub izedɣen deg-i.

Sidi Ṛebbi, meɛna s ṭṭbiɛa n wemdan yellan deg-i, d akli n ddnub i lliɣ.

s ṭṭbiɛa-nneɣ.

<span id="an_8:7">8:7</span> Axaṭer wid ittabaɛen lebɣi n wemdan uɣalen
d iɛdawen n Sidi Ṛebbi imi ur ḍuɛen ara ccariɛa-ines, axaṭer ṭṭbiɛa n
wemdan ur tezmir ara aț-țḍuɛ ccariɛa n Sidi Ṛebbi.

<span id="an_8:8">8:8</span> Wid temlek ṭṭbiɛa-nsen ulamek ara ɛeǧben i
Sidi Ṛebbi.

Ṛebbi izdeɣ deg-wen. Win deg ur yezdiɣ ara Ṛṛuḥ n Lmasiḥ ur yelli ara d
ayla-s.

ger lmegtin, yezmer daɣen s Ṛṛuḥ-is izedɣen deg-wen, a d-yerr tudert i
lǧețța-nwen ara yemten.

<span id="an_8:12">8:12</span> S wakka ihi ay atmaten lebɣi n
ṭṭbiɛa-nneɣ ur yeṭebbiṛ ara fell-aɣ tura iwakken a nexdem lebɣi-ines.

<span id="an_8:13">8:13</span> Ma yella tɛacem s lebɣi n ṭṭbiɛa nwen
atan aț-țemtem, lameɛna ma tɣelbem lebɣi n ṭṭbiɛa-nwen s Ṛṛuḥ iqedsen,
aț-țidirem.

<span id="an_8:14">8:14</span> Axaṭer wid ițeddun s Ṛṛuḥ n Sidi Ṛebbi, d
nutni i d arraw n Ṛebbi.

<span id="an_8:15">8:15</span> Ṛṛuḥ i wen-d-ițțunefken mačči d Ṛṛuḥ ara
kkun-irren d aklan iwakken aț-țɛicem di tugdi, meɛna d Ṛṛuḥ iqedsen i
kkun-irran d arraw n Ṛebbi, yis i nezmer a nessiwel i Sidi Ṛebbi : « A
baba-tneɣ ! »

<span id="an_8:16">8:16</span> D Ṛṛuḥ n Sidi Ṛebbi s yiman-is i
d-iḥeqqeqen deg wulawen-nneɣ belli nuɣal d arraw n Sidi Ṛebbi.

<span id="an_8:17">8:17</span> Ihi ma yella d arraw-is i nella, d nukni

ma nenɛețțab yid-es, a nili daɣen yid-es di lɛaḍima-ines.

<span id="an_8:18">8:18</span> Ihi leɛtab i nesɛedday tura, ur nezmir
ara a t-nmettel ɣer lɛaḍima ara d-ibeggen Sidi Ṛebbi deg-nneɣ.

<span id="an_8:19">8:19</span> Taxelqit meṛṛa tḥar melmi ara d-banen
warraw n Sidi Ṛebbi.

<span id="an_8:21">8:21</span> Meɛna taxelqit-agi tessaram
aț-țețwasellek si twaɣit i ț-isfesden, iwakken aț-țekki di lɛaḍima n
tlelli n warraw n Sidi Ṛebbi.

tenɛețțab, tețnazaɛ am tmeṭṭut yeṭṭfen addud.

<span id="an_8:23">8:23</span> Mačči ț-țaxelqit kan, meɛna ula d nukni
yesɛan Ṛṛuḥ iqedsen, yellan d amur amezwaru n tikciwin i ɣ-d-iwɛed Sidi
Ṛebbi, nețnazaɛ deg iman-nneɣ, nețṛaǧu a nban s tideț d arraw n Ṛebbi i
nella m'ara icafeɛ deg-nneɣ s lekmal.

<span id="an_8:24">8:24</span> Axaṭer leslak nneɣ d ayen i nessaram ; ma
yella nesɛa ayen i nessaram, ayagi mačči d asirem : anwa ara yessirmen
ayen yesɛa ?

<span id="an_8:25">8:25</span> Meɛna ma nessaram ayen ur nesɛi,
nețṛaǧu-t s ṣṣbeṛ.

<span id="an_8:26">8:26</span> Akken daɣen Ṛṛuḥ iqedsen ițțas-ed a ɣ
iɛiwen nukni ixuṣṣen di tezmert, axaṭer ur nessin ara a neẓẓal akken
ilaq ; lameɛna Ṛṛuḥ iqedsen s yiman-is ideɛɛu ɣer Ṛebbi ɣef ddemma-nneɣ,
s nnehtat iwumi ur yezmir yimi a tent-id-yini s imeslayen ;

<span id="an_8:27">8:27</span> imiren Sidi Ṛebbi yessnen ulawen, yeẓra d
acu i gessutur Ṛṛuḥ iqedsen, imi s lebɣi n Sidi Ṛebbi i gdeɛɛu ɣef wid
yextaṛ.

<span id="an_8:28">8:28</span> Neẓra belli Sidi Ṛebbi issufuɣ ɣer lxiṛ
ayen akk i d-ideṛṛun d wid i t-iḥemmlen, wid yextaṛ s lebɣi-ines.

<span id="an_8:29">8:29</span> Axaṭer wid akk i gextaṛ Sidi Ṛebbi si
zik, iqsed si tazwara a ten-yerr am Mmi-s iwakken Mmi-s agi ad yili d
amenzu n waṭas n watmaten.

<span id="an_8:30">8:30</span> Wid i gextaṛ Sidi Ṛebbi si zik, d wid
iwumi i d-issawel ; wid-nni iwumi yessawel yerra-ten d iḥeqqiyen
zdat-es, wid yerra d iḥeqqiyen yefka-yasen amkan di lɛaḍima-s.

<span id="an_8:31">8:31</span> Acu ara d-nernu ihi ? Anwa ara ɣ-izemren
ma yella Sidi Ṛebbi yid-nneɣ ?

<span id="an_8:32">8:32</span> Nețța iqeblen ad isebbel Mmi-s eɛzizen,
yefka-t d asfel fell-aɣ meṛṛa, amek ur ɣ-d-yețțak ara daɣen kullec
yid-es ?

<span id="an_8:33">8:33</span> Anwa ara icetkin ɣef wid yextaṛ Ṛebbi ? D
Sidi Ṛebbi s yiman-is i ten-iḥesben d iḥeqqiyen !

<span id="an_8:34">8:34</span> Anwa i gzemren ad iḥkem fell asen ? Ɛisa
Lmasiḥ d win yemmuten yerna iḥya-d si ger lmegtin, yuli ɣer tama
tayeffust n Sidi Ṛebbi iwakken ad icafeɛ fell-aɣ !

<span id="an_8:35">8:35</span> Anwa i gzemren a ɣ-ifṛeq ɣef leḥmala n

ayen yessaggaden neɣ d ajenwi ?

<span id="an_8:36">8:36</span> Am akken yura di tektabt n ?abur : ?ef
ddemma-inek a Sidi Ṛebbi i nețqabal lmut mkul ass, ḥesben-aɣ am ulli i
țțawin ɣer tmezliwt.

<span id="an_8:37">8:37</span> Meɛna deg wannect-agi meṛṛa d nukni i
gɣelben s Lmasiḥ i ɣ- iḥemmlen.

<span id="an_8:38">8:38</span> Axaṭer țekleɣ belli ulac d acu i gzemren
a ɣ-ifṛeq ɣef leḥmala n Sidi Ṛebbi s Ɛisa Lmasiḥ Ssid-nneɣ : ama d lmut
neɣ ț-țudert, ama d lmalayekkat, ama ț-țigeldiwin, ama d leḥwayeǧ n tura
neɣ ț-țid ara d-yasen, ama d ayen ɛlayen, ama d ayen lqayen, ama
ț-țizmar n igenwan neɣ ț-țid n lqaɛa neɣ ayen akk yellan di txelqit.

iqedsen i yi-sedduyen :

<span id="an_9:2">9:2</span> leḥzen d uɣilif zgan deg ul-iw.

<span id="an_9:3">9:3</span> Zemreɣ ad qebleɣ ad țțuneɛleɣ neɣ ad
țțuḥeṛṛmeɣ ɣef Lmasiḥ ɣef ddemma n watmaten-iw akk-d lumma-inu ;

<span id="an_9:4">9:4</span> nutni at Isṛail i gextaṛ Sidi Ṛebbi d

ccariɛa d wamek ara t-ɛebden, yerna-yasen-d lemɛahdat.

<span id="an_9:5">9:5</span> Seg-sen i d-ffɣen lejdud-nneɣ i gellan d
iqeddacen imezwura n Sidi Ṛebbi, ula d Lmasiḥ seg-sen i d-iffeɣ mi
d-ilul s ṣṣifa n wemdan, nețța i gellan sennig kra yellan, d Illu
yețwaḥemden i dayem. Amin !

<span id="an_9:6">9:6</span> Acu ara d-nini ? Sidi Ṛebbi yendem deg

Isṛail i gellan n wat Isṛail i gextaṛ Sidi Ṛebbi.

<span id="an_9:7">9:7</span> Daɣen mačči d wid meṛṛa i d-yeǧǧa Sidna
Ibṛahim i gellan d arraw-is n tideț, axaṭer Sidi Ṛebbi inna-yas-d : Seg
Isḥaq ara d-teffeɣ dderya i k-weɛdeɣ.

<span id="an_9:8">9:8</span> S wakka ihi nezmer ad ini mačči d dderya i
d ilulen s lebɣi n wemdan ara ițțusemmin d arraw n Sidi Ṛebbi meɛna
anagar wid i d-ilulen s lebɣi n Sidi Ṛebbi akken i t-id-yewɛed.

<span id="an_9:9">9:9</span> Axaṭer atnan imeslayen s wayes i gɛuhed
Sidi Ṛebbi : A d-uɣaleɣ aseggas i d-iteddun di lweqt-agi, ?ara a d-tesɛu
aqcic.

<span id="an_9:10">9:10</span> Mačči d ayagi kan : arraw n Rbiḥa daɣen i
gellan d akniwen, sɛan yiwen n baba-tsen yellan d jeddi-tneɣ Isḥaq.

<span id="an_9:11">9:11</span> Meɛna imi Sidi Ṛebbi ițextiri imdanen
mačči ɣef ddemma n lecɣal-nsen meɛna akken yella lebɣi-s d usiwel-ines ;
mazal urɛad i d-lulen warraw n Rbiḥa, ur xdimen cceṛ, ur xdimen lxiṛ, mi
s-d-yenna Sidi Ṛebbi : Amenzu deg warraw-im ad yuɣal d aqeddac ɣef
wumecṭuḥ.

<span id="an_9:13">9:13</span> Akken i gura daɣen deg umkan nniḍen n
tira iqedsen : ?emmleɣ Yeɛqub, keṛheɣ Icaw.

<span id="an_9:14">9:14</span> D acu ara d-nini deg wannect-a ? Eɛni
Sidi Ṛebbi ixeddem lbaṭel ? D lmuḥal !

<span id="an_9:15">9:15</span> Axaṭer yenna-yas i Sidna Musa : A
d-sserseɣ ṛṛeḥma-inu ɣef win i bɣiɣ, ad ḥunneɣ daɣen ɣef win i bɣiɣ.

<span id="an_9:16">9:16</span> Lbaṛaka n Sidi Ṛebbi ur d-tețțas ara s
lefɛayel n wemdan ama yuzzel ama yeqqim meɛna kullec yețțas-ed s ɣuṛ
Sidi Ṛebbi i d-yețțaken ṛṛeḥma ines akken yebɣa.

<span id="an_9:17">9:17</span> Anda nniḍen di tira iqedsen Sidi Ṛebbi

d-sbeggneɣ tazmert-iw deg-ek, s wakka di ddunit meṛṛa ad issinen belli d
nekk i d Illu.

<span id="an_9:18">9:18</span> Akka ihi, Sidi Ṛebbi yesrusu-d ṛṛeḥma-s
ɣef win yebɣa, isɣaṛay daɣen ul n win yebɣa.

<span id="an_9:19">9:19</span> Ahat ad yili win ara d-yinin : imi akka i
gella lḥal, acu i mazal ițlumu deg-nneɣ ? Anwa i gzemren a t-iɛaṣi ?

<span id="an_9:20">9:20</span> I kečč ay amdan ur nesɛi azal, d acu i
tḥesbeḍ iman-ik iwakken aț-țḥasbeḍ Sidi Ṛebbi ? Eɛni tacmuxt ițwameslen
s wakal tezmer aț-țini i win i ț-imeslen acuɣer i yi-tmesleḍ s
ṣṣifa-yagi ?

<span id="an_9:21">9:21</span> Win imesslen afexxaṛ yezmer ad ixdem ayen
i s-ihwan s wakal ; seg wakal-nni yezmer ad imsel tacmuxt izewqen neɣ
tin i sseqdacen kull ass.

<span id="an_9:22">9:22</span> Eɛni Sidi Ṛebbi ur yezmir ara a d-ibeggen
urrif-is akk-d tezmert-is ? Iɣaḍ-ik lḥal ma yella yeṣber s ṣṣber
ameqqran i wid yuklalen urrif-is yerna teddun ɣer nnger ?

<span id="an_9:23">9:23</span> Deg wannect-agi, Sidi Ṛebbi yebɣa a
d-isbeggen acḥal meqqṛet tmanegt-is i nukni i ɣef iḥunn, i ghegga si zik
iwakken a nili yid-es di tmanegt-is ;

<span id="an_9:24">9:24</span> nukni iwumi i d-yessawel Sidi Ṛebbi mačči
seg wat Isṛail kan, meɛna ula si ger leǧnas nniḍen ;

<span id="an_9:25">9:25</span> akken i t-id-yenna Sidi Ṛebbi di tektabt
n nnbi Hucaɛ : Lǧens-nni ur nelli ara d agdud-iw a s-semmiɣ agdud-iw,
win akken ur nețwaḥemmel ara a s-semmiɣ win eɛzizen fell-i.

<span id="an_9:26">9:26</span> Anda akken i sen-qqaṛen : « Ur tellim ara
d agdud n Sidi Ṛebbi», dinna ara țțusemmin d arraw n Sidi Ṛebbi yeddren.

<span id="an_9:27">9:27</span> Si lǧiha-s, nnbi Iceɛya icar-ed ɣef wayen
yeɛnan agdud n wat Isṛail : ?as ad ketṛen wat Isṛail am ṛṛmel ɣef yiri n
lebḥeṛ, kra deg-sen kan ara yețțuselken.

<span id="an_9:28">9:28</span>  Axaṭer Sidi Ṛebbi ur ițɛeṭṭil ara ad
ixdem di ddunit ayen akken i d-yenna deg wawal-is.

<span id="an_9:29">9:29</span>  Yura di tira iqedsen : Lemmer Sidi Ṛebbi

temdint n Sudum, aț-țeḍru yid-nneɣ am temdint n Gumuṛ inegren. Ayagi
yenna-t-id nnbi Iceɛya uqbel a d-yedṛu.

<span id="an_9:30">9:30</span> D acu ara d-nini ihi ? Igduden nniḍen ur
nețnadi ara ɣef lḥeqq uɣalen d iḥeqqiyen zdat Sidi Ṛebbi, uɣalen d
iḥeqqiyen imi umnen yis.

<span id="an_9:31">9:31</span> Ma d at Isṛail yețnadin ad uɣalen d
iḥeqqiyen s ccariɛa, ur wwiḍen ara ɣer lmeqṣud-nsen s ccariɛa.

<span id="an_9:32">9:32</span> Acuɣer ? Axaṭer ur qellben ara ad ɛeǧben
i Sidi Ṛebbi s liman, meɛna nutni ɣilen ad uɣalen d iḥeqqiyen s
lefɛayel-nsen. Mmugren-d adɣaɣ i sen-yellan d ugur, isseɣli-ten.

<span id="an_9:33">9:33</span> Am akken yura di tektabt iqedsen : Atan
ad sserseɣ ɣef wedrar n Siyun adɣaɣ, d adɣaɣ i ɣef yekkat wugur, d azṛu
yesseɣlayen, meɛna win yumnen yis weṛǧin ad yenneḥcam.

<span id="an_10:1">10:1</span> Ay atmaten, ayen i țmenniɣ deg ul-iw, d
leslak n wat Isṛail meṛṛa, acḥal i ssutreɣ di Ṛebbi ɣef ddemma-nsen
iwakken ad țțuselken.

<span id="an_10:2">10:2</span> Axaṭer zemreɣ ad cehdeɣ fell-asen belli
