### Kabyle Bible: 2 John

<span id="an_1:1">1:1</span> Nekk yellan d ameqqran n tejmaɛt, i Kirya i
gextaṛ Sidi Ṛebbi akk-d warraw-is i ḥemmleɣ s tideț. Mačči d nekk kan i
kkun iḥemmlen, meɛna ula d wid akk yesnen tideț ;

<span id="an_1:2">1:2</span> ɣef ddemma n tideț yellan deg-nneɣ ara
yeqqimen deg-nneɣ i dayem.

<span id="an_1:3">1:3</span> Ssmaḥ, ṛṛeḥma ț-țalwit a ɣ-d țțunefken s
ɣuṛ Baba Ṛebbi d Mmi-s Ɛisa Lmasiḥ, iwakken a nɛic di tideț akk-d
leḥmala.

<span id="an_1:4">1:4</span> Acḥal feṛḥeɣ imi ufiɣ kra seg warraw-im
teddun deg webrid n tideț, akken i ɣ-t-id-yumeṛ Baba Ṛebbi.

<span id="an_1:5">1:5</span> Ihi tura a tajmaɛt n Kirya, ayen i nețṛaǧu
deg-wen : a nemyeḥmal wway gar-aneɣ, mačči d lameṛ ajdid i wen-d-wwiɣ
lameɛna d lameṛ yellan yakan si tazwara.

<span id="an_1:6">1:6</span> Leḥmala, d m'ara nțeddu deg webrid n Sidi
Ṛebbi nḥerrez lumuṛat-is. S lameṛ-agi i teslam si tazwara i glaq
aț-țeddum.

<span id="an_1:7">1:7</span> Aṭas n wid yesselmaden lekdeb i d-innulfan
di ddunit, ur setɛeṛfen ara belli Ɛisa Lmasiḥ yusa-d ɣer ddunit s ṣṣifa
n wemdan. D wigi i gețɣuṛṛun, i d iɛdawen n Lmasiḥ.

<span id="an_1:8">1:8</span> ?adret iman-nwen iwakken ur a wen-yețṛuḥu
ara wayen akk i wen-nesselmed lameɛna aț-țɣelltem lfayda tameqqrant.

<span id="an_1:9">1:9</span> Kra n win ara iɛeddin tilas deg wayen
yesselmed Lmasiḥ, ur yeṭṭif ara di Ṛebbi ; ma d win yeṭṭfen deg uselmed
n Lmasiḥ yeṭṭef di Baba Ṛebbi akk-d Mmi-s.

<span id="an_1:10">1:10</span> Kra n win ara d-yasen ɣuṛ-wen m'ur
wen-d-yewwi ara aselmed-agi n Lmasiḥ ur sṭerḥibet ara yis ɣer
yexxamen-nwen, ur țsellimet ara ula d sslam fell-as !

<span id="an_1:11">1:11</span> Axaṭer win i gesṭerḥben yis, yețțekki di
lecɣal-is n diri.

<span id="an_1:12">1:12</span> Mazal-iyi aṭas n lecɣal i ɣef bɣiɣ a
wen-d-hedṛeɣ lameɛna mačči di tebṛaț, ssarameɣ a n-aseɣ ɣuṛ-wen, a
wen-mmeslayeɣ s yimi-w, iwakken lfeṛḥ-nneɣ ad innekmal.

<span id="an_1:13">1:13</span> Arraw n weltma-m yextaṛ Ṛebbi, țsellimen
fell-am.
