### Kabyle Bible: Luke

<span id="an_1:1">1:1</span> Imi aṭas i gebdan tira ɣef wayen yedṛan
gar-aneɣ,

<span id="an_1:2">1:2</span> akken i ɣ-t-id ɛawden wid yellan d inigan

Sidi Ṛebbi.

<span id="an_1:3">1:3</span> Steqsaɣ ula d nekk akken ilaq ɣef wayen akk
yedṛan seg wass amezwaru, ufiɣ d ayen yelhan a k-t-id aruɣ a Tawfilus,

<span id="an_1:4">1:4</span> iwakken aț-țissineḍ tideț ɣef wayen i
tesliḍ meṛṛa.

<span id="an_1:5">1:5</span> Di zzman n Hiṛudus, agellid n tmurt n
Yahuda, yella yiwen n lmuqeddem si tejmaɛt n Abya, isem-is Zakarya.
Tameṭṭut-is Ilicaba, seg izuṛan n Haṛun.

<span id="an_1:6">1:6</span> D iḥeqqiyen i llan i sin, ḥerzen yerna
ḍuɛen awal n Sidi Ṛebbi.

<span id="an_1:7">1:7</span> Ur sɛin ara dderya, meqqṛit i sin di leɛmeṛ

<span id="an_1:8">1:8</span> Mi d-tewweḍ nnuba n leqdic n wedrum n

<span id="an_1:9">1:9</span> akken i tella di lɛadda n lmuqedmin, gren

lebxuṛ.

<span id="an_1:10">1:10</span> Mi gella isseṛɣay lebxuṛ di lǧameɛ, lɣaci
meṛṛa yeqqimen di beṛṛa deɛɛun ɣer Sidi Ṛebbi.

<span id="an_1:11">1:11</span> Iḍheṛ-as-d yiwen lmelk n Sidi Ṛebbi,
ibedd ɣer tama tayeffust n udekkan-nni anda sseṛɣayen lebxuṛ.

<span id="an_1:12">1:12</span> Mi t-iwala Zakarya, yedhec ikcem-it lxuf.

<span id="an_1:13">1:13</span> Lmelk-nni yenna yas -ed : Ur țțagad ara a
Zakarya, ayen i tessutreḍ ɣer Sidi Ṛebbi yețwaqbel. Tameṭṭut-ik Ilicaba
ad a k-d-tesɛu aqcic, a s-tsemmiḍ Yeḥya.

<span id="an_1:14">1:14</span> Aț-țfeṛḥeḍ yis, a k-yili d sebba n sseɛd
; imdanen meṛṛa ad feṛḥen s tlalit-is.

<span id="an_1:15">1:15</span> Ad yesɛu ccan d ameqqran ɣer Sidi Ṛebbi,
a d-iččaṛ d Ṛṛuḥ iqedsen si tɛebbuṭ n yemma-s, ur itess ccṛab, ur itess
ayen nniḍen isekkṛen.

<span id="an_1:16">1:16</span> A d-yerr aṭas n wat Isṛail ɣer webrid n
Sidi Ṛebbi.

<span id="an_1:17">1:17</span> Ad ilḥu s leɛnaya n Sidi Ṛebbi, s ṛṛuḥ
ț-țezmert n nnbi Ilyas. Ad issemlil imawlan d warraw-nsen, a d-yerr wid
ijehlen ɣer webrid n iḥeqqiyen, akken ad iheggi i Sidi Ṛebbi agdud ara
t-iḍuɛen.

<span id="an_1:18">1:18</span> Zakarya yenna i lmelk : Amek ara ɛeqleɣ
belli ayen akka i d-tenniḍ ț-țideț ? Nekk d amɣaṛ, ula ț-țameṭṭut-iw
meqqṛet di leɛmeṛ !

<span id="an_1:19">1:19</span> Lmelk-nni yerra-yas-d : Nekk d Jebrayil,

n lxiṛ.

<span id="an_1:20">1:20</span> Lameɛna imi ur tumineḍ ara s

alamma d asmi ara d-yedṛu wannect-agi. Meɛna ḥṣu belli ayen akka i
k-d-nniɣ ad yedṛu deg wass-is.

<span id="an_1:21">1:21</span> Lɣaci-nni yețṛaǧun Zakarya deg ufrag,
wehmen acuɣeṛ iɛeṭṭel deg umkan iqedsen n lǧameɛ.

<span id="an_1:22">1:22</span> Mi d-iffeɣ, yeggugem, ur izmir ara a

Yebda ițwehhi-yasen-d s ifassen-is, yeqqim d agugam.

<span id="an_1:23">1:23</span> Mi kfan wussan n lxedma-s di lǧameɛ,
Zakarya yuɣal ɣer wexxam-is.

<span id="an_1:24">1:24</span> Ɛeddan kra n wussan, tameṭṭut-is Ilicaba
terfed tadist ; dɣa tesseḥjeb iman-is xemsa wagguren, teqqaṛ :

<span id="an_1:25">1:25</span> Atan Sidi Ṛebbi yerra-d ddehn-is ɣuṛ-i,
ikkes-iyi lɛib zdat yemdanen.

<span id="an_1:26">1:26</span> Aggur wis sețța, Sidi Ṛebbi iceggeɛ-d
lmelk Jebrayil ɣer taddart n Naṣaret, di tmurt n Jlili.

<span id="an_1:27">1:27</span> Iceggeɛ-it-id ɣer yiwet n tlemẓit i
gexḍeb yiwen wergaz isem-is Yusef, yellan seg izuṛan n Sidna Dawed.
Tilemẓit-agi isem-is Meryem.

<span id="an_1:28">1:28</span> Ikcem lmelk ɣuṛ-es yenna-yas : Sslam
fell-am a tin iburek Sidi Ṛebbi ! Sidi Ṛebbi yella yid-em.

<span id="an_1:29">1:29</span> Meryem tedhec, tenna : D acu i d lmeɛna n
imeslayen agi ?

<span id="an_1:30">1:30</span> Lmelk yenna-yas : Ur țțaggad ara a Meryem
! S ṛṛeḥma-s tameqqrant, Sidi Ṛebbi yextaṛ-ikem.

<span id="an_1:31">1:31</span> Atan aț-țrefdeḍ tadist, a d-tesɛuḍ aqcic,

<span id="an_1:32">1:32</span> Ad yesɛu ccan d ameqqran, ad ițțusemmi

Sidna Dawed, yiwen si lejdud-is.

<span id="an_1:33">1:33</span> Ad iḥkem ɣef tarwa n Yeɛqub i dayem,
tageldit-is ur tfennu ara.

<span id="an_1:34">1:34</span> Meryem tenna i lmelk-nni : Amek ara
d-idṛu yid-i wayagi nekk ur nezwiǧ ara ?

<span id="an_1:35">1:35</span> Lmelk yerra-yas : Atan a d-yers fell-am
Ṛṛuḥ iqedsen ; tazmert n Sidi Ṛebbi ɛlayen a kem-tɣumm am tili, daymi
aqcic-agi imqeddes ara d-ilalen ad ițțsemmi « Mmi-s n Ṛebbi ».

<span id="an_1:36">1:36</span> Ilicaba i m-ițțilin, ula d nețțat ațan s
tadist a d-tesɛu aqcic ɣas akken meqqṛet di leɛmeṛ. Tin akken iwumi
semman tiɛiqeṛt, ațan deg waggur wis sețța,

<span id="an_1:37">1:37</span> imi ulac wayen iwumi ur izmir Sidi Ṛebbi.

<span id="an_1:38">1:38</span> Meryem tenna : Aql-i ț-țaqeddact n Sidi
Ṛebbi, ad idṛu yid-i wayen i d-tenniḍ. Imiren Lmelk-nni yeǧǧa-ț, iṛuḥ.

<span id="an_1:39">1:39</span> Deg wussan-nni, Meryem tekker tṛuḥ s
lemɣawla ɣer yiwet n taddart yellan deg idurar n tmurt n Yahuda.

<span id="an_1:40">1:40</span> Tekcem ɣer wexxam n Zakarya, tsellem ɣef
Ilicaba.

<span id="an_1:41">1:41</span> Akken i tesla Ilicaba i sslam n Meryem,
yefrawes llufan yellan di tɛebbuṭ-is ; imiren teččuṛ d Ṛṛuḥ iqedsen,

<span id="an_1:42">1:42</span> tɛeggeḍ tenna : A Meryem, tețțubarkeḍ ger
tilawin meṛṛa, ițțubarek daɣen win yellan di tɛebbuṭ-im.

<span id="an_1:43">1:43</span> D acu-yi, iwakken a d-terzef ɣuṛ-i
yemma-s n Ssid-iw ?

<span id="an_1:44">1:44</span> Akken kan i sliɣ i sslam-im, llufan
yellan di tɛebbuṭ-iw yefrawes s lfeṛḥ.

<span id="an_1:45">1:45</span> ?-țaseɛdit kemm yumnen s wayen i
m-d-yenna Sidi Ṛebbi !

<span id="an_1:46">1:46</span> Meryem tenna : Taṛwiḥt-iw teḥmed Sidi
Ṛebbi,

<span id="an_1:47">1:47</span> ul-iw yeččuṛ d lfeṛḥ imi d nețța i d
amsellek-iw,

<span id="an_1:48">1:48</span> axaṭer iwala-yi-d nekk taqeddact-is

ț-țaseɛdit »,

<span id="an_1:49">1:49</span> imi Sidi Ṛebbi Bab n tezmert, yexdem
yis-i ayen issewhamen. Isem-is d imqeddes.

<span id="an_1:50">1:50</span>  Ṛṛeḥma-s tețdum si lǧil ɣer lǧil ɣef wid
akk i t-iḍuɛen.

<span id="an_1:51">1:51</span> Isken-ed tazmert n yiɣil-is, isseɛṛeq
iberdan i wid yețzuxxun ;

<span id="an_1:52">1:52</span>  yessers-ed imeqqranen seg imukan
eɛlayen, yerfed wid yețwaḥeqṛen ;

<span id="an_1:53">1:53</span>  yesseṛwa-yasen leṛẓaq i wid yelluẓen, ma
d imeṛkantiyen yerra-ten ifassen d ilmawen.

<span id="an_1:54">1:54</span>  Isellek tarwa n wat Isṛail yellan d
iqeddacen-is, ur ițțu ara Ṛṛeḥma-s,

<span id="an_1:55">1:55</span> i gewɛed i Sidna Ibṛahim d warraw-is i
dayem akken i t-yenna i lejdud-nneɣ.

<span id="an_1:56">1:56</span> Meryem teqqim ɣer Ilicaba azal n tlata

<span id="an_1:57">1:57</span> Yewweḍ-ed lweqt i deg ara d-tarew
Ilicaba, tesɛa-d aqcic.

<span id="an_1:58">1:58</span> Imawlan-is d lǧiran-is feṛḥen aṭas mi
slan s ṛṛehma i d-issers fell-as Sidi Ṛebbi.

<span id="an_1:59">1:59</span> Mi wwḍen tmanya n wussan ɣef weqcic-nni,
usan-d a s-sḍehṛen, bɣan a s-semmin s yisem n baba-s : Zakarya.

<span id="an_1:60">1:60</span> Dɣa tenṭeq-ed yemma-s tenna-yasen : Xaṭi
! A s-nsemmi Yeḥya !

<span id="an_1:61">1:61</span> Nnan-as : Ula d yiwen deg wedrum-nwen ur
yesɛi isem-agi.

<span id="an_1:62">1:62</span> Steqsan baba-s s uwehhi amek yebɣa ad
isemmi i mmi-s.

<span id="an_1:63">1:63</span> Issuter asen-d talwiḥt, yura-yasen-d
deg-s : « Yeḥya i d isem-is ». Dɣa wehmen meṛṛa.

<span id="an_1:64">1:64</span> Imiren kan iserreḥ yiles-is, yebda

<span id="an_1:65">1:65</span> Imezdaɣ n lǧiha-nni meṛṛa wehmen,

<span id="an_1:66">1:66</span> Wid akk yeslan imeslayen-nni, ḥerzen-ten

afus n Sidi Ṛebbi yella fell-as.

<span id="an_1:67">1:67</span> Zakarya, baba-s n weqcic-nni, iččuṛ d

<span id="an_1:69">1:69</span> Yefka-yaɣ-d yiwen wemsellek d ameqqran

<span id="an_1:70">1:70</span> am akken i t-id-ixebbeṛ Sidi Ṛebbi si

yiɛdawen-nneɣ, si ger ifassen n wid akk i ɣ-ikeṛhen.

<span id="an_1:72">1:72</span> Isbeggen ṛṛeḥma-s i lejdud-nneɣ,
yemmekta-d lemɛahda-s iqedsen

<span id="an_1:73">1:73</span> i gefka i jeddi-tneɣ Sidna Ibṛahim.

<span id="an_1:74">1:74</span> M'ara nețțusellek seg ufus n
yeɛdawen-nneɣ, a neɛbed Sidi Ṛebbi mbla lxuf,

<span id="an_1:75">1:75</span> s lḥeqq, s ṭṭaɛa d wannuz deg wussan n
ddunit-nneɣ meṛṛa.

<span id="an_1:76">1:76</span> Ma d kečč a mmi, aț-țețțusemmiḍ d nnbi n

iberdan-is,

<span id="an_1:77">1:77</span> aț-țesfehmeḍ agdud-is ɣef leslak n Sidi
Ṛebbi s leɛfu n ddnubat-nsen.

<span id="an_1:78">1:78</span> Axaṭer Ṛebbi-nneɣ d aḥnin, s Ṛṛeḥma-ines
a d-icṛeq fell-aɣ tafat-is am tin n yiṭij seg yigenwan.

<span id="an_1:79">1:79</span>  Tafat-is a d-tecṛeq ɣef wid yellan di
ṭṭlam n lmut, ad aɣ-yawi deg webrid n lehna.

<span id="an_1:80">1:80</span> Aqcic-nni ițțimɣuṛ yețnerni di leɛqel.

<span id="an_2:1">2:1</span> Deg ussan-nni, yeffeɣ-ed yiwen n lqanun s

tmura-ines.

<span id="an_2:2">2:2</span> Ajerred-agi amezwaru yedṛa asmi yella

<span id="an_2:3">2:3</span> Imdanen meṛṛa ṛuḥen ad țțujerden mkul yiwen

<span id="an_2:4">2:4</span> Yusef yellan seg izuṛan n Sidna Dawed, yuli

lejdud-is yellan di tmurt n Yahuda.

<span id="an_2:5">2:5</span> Iṛuḥ ad ițțujerred nețța ț-țxedibt-is

<span id="an_2:6">2:6</span> Mi ten-yuɣ lḥal dinna, yewweḍ-d lweqt i deg

<span id="an_2:7">2:7</span> Tesɛa-d mmi-s amenzu, tețțel-it,

<span id="an_2:9">2:9</span> Ataya lmelk n Sidi Ṛebbi iḍheṛ-asen-d,

ameqqran.

<span id="an_2:10">2:10</span> Lmelk yenna-yasen : Ur țțagadet ! Aql-i

<span id="an_2:11">2:11</span> Ass-agi di taddart n Sidna Dawed,
ilul-awen-d yiwen wemsellek, d nețța i d Lmasiḥ n Sidi Ṛebbi !

<span id="an_2:12">2:12</span> Atan wamek ara t-tɛeqlem : aț-țafem

<span id="an_2:13">2:13</span> Imiren kan, ț-țirebbaɛ n lmalayekkat i

<span id="an_2:14">2:14</span> Lḥemd i Sidi Ṛebbi deg imukan eɛlayen,
lehna di ddunit ɣef yemdanen eɛzizen fell-as.

<span id="an_2:15">2:15</span> Akken uɣalent lmalayekkat-nni ɣer
yigenni, imeksawen nnan wway gar-asen : Kkret a nṛuḥet ɣer Bitelḥem, a
nẓer ayen i gedṛan, d wayen akka s wayes i ɣ-d-ixebbeṛ Sidi Ṛebbi.

<span id="an_2:16">2:16</span> Imiren kan ṛuḥen, ufan dinna Meryem d
Yusef akk-d llufan-nni yeṭṭes di lmedwed.

<span id="an_2:17">2:17</span> Mi walan aqcic-nni, ḥkan-d wayen i
sen-d-yenna lmelk fell-as.

<span id="an_2:18">2:18</span> Wid akk yeslan s wayen i d-ḥkan
imeksawen-nni, dehcen.

<span id="an_2:19">2:19</span> Meryem teḥrez imeslayen-nni deg wul-is,
tețxemmim fell-asen.

<span id="an_2:20">2:20</span> Mi uɣalen imeksawen-nni, țḥemmiden
țcekkiṛen Sidi Ṛebbi ɣef wayen slan d wayen ẓran, imi ufan kullec akken
i sen-d-ițțuxebbeṛ.

<span id="an_2:21">2:21</span> Yewweḍ-ed wass wis tmanya i deg ara
s-sḍehṛen i weqcic-nni ; semman-as Ɛisa, isem-nni i s-yefka lmelk uqbel
ad yili di tɛebbuṭ n yemma-s.

<span id="an_2:22">2:22</span> Asmi wwḍen ṛebɛin wussan ɣef tlalit n
weqcic-nni, Yusef d Meryem wwin-t ɣer temdint n Lquds iwakken ad
ițțuqeddem i Sidi Ṛebbi,

<span id="an_2:23">2:23</span> akken yura di ccariɛa n Sidna Musa : yal
aqcic amenzu, ilaq ad ițțuqeddem i Sidi Ṛebbi.

<span id="an_2:24">2:24</span> Yerna ṛuḥen iwakken ad fken d asfel snat
n tmilliwin neɣ sin yetbiren imeẓyanen akken yella di ccariɛa n Sidna
Musa.

<span id="an_2:25">2:25</span> Di temdint n Lquds yella yiwen wergaz
isem-is Semɛun. Argaz-agi d aḥeqqi, iḍuɛ Sidi Ṛebbi, yețṛaǧu amsellek n
wat Isṛail, Ṛṛuḥ iqedsen yețțili yid-es.

<span id="an_2:26">2:26</span> Iweḥḥa-yas-ed belli ur ițmețțat ara
alamma yeẓra Lmasiḥ n Sidi Ṛebbi.

<span id="an_2:27">2:27</span> Ṛṛuḥ n Sidi Ṛebbi iwelleh Semɛun ad iṛuḥ
ɣer lǧameɛ iqedsen.Atnaya kecmen-d Yusef d Meryem, wwin-d llufan-nni
iwakken ad xedmen wayen i d-tenna ccariɛa n Musa fell-as.

<span id="an_2:28">2:28</span> Semɛun yeṭṭef aqcic-nni ger ifassen-is,
iḥmed Sidi Ṛebbi yenna :

<span id="an_2:29">2:29</span> Tura a Sidi Ṛebbi ɣas ad mmteɣ aql-i di

di ddnub. A nnger n win ara yilin d sebba n uɣelluy ;

<span id="an_17:3">17:3</span> ?adret ɣef yiman-nwen ! Ma yeḍlem gma-k

<span id="an_17:6">17:6</span> Sidna Ɛisa yerra-yasen : Lemmer tesɛam

aț-țeččeḍ aț-țesweḍ ula d keččini. »

<span id="an_17:12">17:12</span> Mi gekcem ɣer yiwet n taddart, mmugren

ḥunn fell-aɣ !

Ṛuḥet a kkun-walin lmuqedmin, a d-cehden belli teḥlam. Deg webrid-nsen
mi ṛuḥen, akken walan iman-nsen ḥlan ṣfan,

<span id="an_17:16">17:16</span> Yusa-d yeɣli ɣer iḍaṛṛen n Sidna Ɛisa,

<span id="an_17:17">17:17</span> Sidna Ɛisa yenna i lɣaci : Eɛni ur ḥlin

yeḥmed Ṛebbi !

uɣal ɣer wexxam ik, imi tumneḍ yis-i, tețțusellkeḍ.

Sidi Ṛebbi mačči d ayen țwalint wallen.

<span id="an_17:21">17:21</span> Ur nezmir ara a d-nini : ațan dagi neɣ
ațan dihin, axaṭer lḥekma n Sidi Ṛebbi ațan gar-wen.

<span id="an_17:22">17:22</span> Sidna Ɛisa yenna i inelmaden-is : A

wass lameɛna ur kkun-id-ițṣaḥ ara.

<span id="an_17:23">17:23</span> Imiren ma nnan-awen-d : atah dagi neɣ

<span id="an_17:25">17:25</span> Tazwara ilaq ad yeɛteb aṭas, yerna

<span id="an_17:27">17:27</span> Imdanen llan tețțen, țessen,
țemyezwaǧen, zeggjen i dderya-nsen, armi d asmi yekcem Nuḥ ɣer lbabuṛ ;
yewwet-ed uḥemmal, neflen-d waman ɣef ddunit, imdanen meṛṛa mmuten.

<span id="an_17:28">17:28</span> D ayen yedṛan di zzman n Luṭ ara yedṛun
daɣen : imdanen tețțen tessen, țțaɣen znuzun, țeẓẓun, bennun ;

<span id="an_17:29">17:29</span> asmi d-yeffeɣ Luṭ si temdint n ?udum,
teɣli-d seg igenni tmes d ukebri i ten-isnegren akk.

<span id="an_17:30">17:30</span> Akken ara d-tedṛu asm'ara d-iban Mmi-s
n bunadem.

<span id="an_17:31">17:31</span> Ass-nni win ara yilin sufella n wexxam
ur ilaq ara ad yekcem a d-iddem lqecc-is ; win i d-yufa lḥal di lexla ur
ilaq ara ad yuɣal ɣer deffir ;

<span id="an_17:32">17:32</span> mmektit-ed tameṭṭut n Luṭ.

<span id="an_17:33">17:33</span> Win ara yebɣun ad isellek tudert-is ad
a s-tṛuḥ, win iwumi ara tṛuḥ ɣef ddemma n yisem-iw a ț-isellek.

<span id="an_17:34">17:34</span> Qqaṛeɣ-awen : deg iḍ-nni, ger sin ara
yilin deg yiwen wusu, yiwen ad ițwarfed wayeḍ a d-yeqqim.

<span id="an_17:35">17:35</span> Ger snat n tilawin yeẓẓaden, yiwet
aț-țețwarfed tayeḍ a d-teqqim.

<span id="an_17:36">17:36</span> Ger sin yergazen ara yilin di lexla,
yiwen ad ițwarfed wayeḍ a d-yeqqim.

<span id="an_17:37">17:37</span> Inelmaden-is steqsan-t nnan-as : Anda

d-nejmaɛen igudar.

<span id="an_18:1">18:1</span> Sidna Ɛisa yewwi-yasen-d lemtel-agi
iwakken ad deɛɛun daymen ɣer Sidi Ṛebbi ur feččlen ara.

<span id="an_18:2">18:2</span> Yenna-yasen : Deg yiwet n temdint yella
yiwen lqaḍi, ur ițțagad Ṛebbi ur yețqadaṛ imdanen.

<span id="an_18:3">18:3</span> Di temdint-nni tella daɣen yiwet n
taǧǧalt tețṛuḥu ɣuṛ-es, teqqaṛ-as : « eḥkem s lḥeqq gar-i d wexṣim-iw. »

<span id="an_18:4">18:4</span> Aṭas i geqqim yegguma a s-yefk lḥeqq,
meɛna ixemmem deg yiman-is yenna : « taǧǧalt-agi eɛyiɣ deg-es ! ?as ur
țțaggadeɣ Ṛebbi ur ḥsibeɣ yiwen

<span id="an_18:5">18:5</span> a s-fkeɣ lḥeqq akken ur d-tețțuɣal ara a
yi-teṛṛeẓ aqeṛṛuy-iw. »

<span id="an_18:6">18:6</span> Dɣa Sidna Ɛisa yenna : Twalam ayen i
d-yenna lqaḍi agi amesbaṭli !

<span id="an_18:7">18:7</span> Amek Sidi Ṛebbi ur d-yețțak ara lḥeqq i

yețɛeṭṭil ara iwakken a ten-iɛiwen, a sen-yefk lḥeqq.

<span id="an_18:8">18:8</span> Meɛna Mmi-s n bunadem asm'ara d-yas,
wissen ma d-yaf liman di ddunit ?

<span id="an_18:9">18:9</span> Yenna-d daɣen lemtel-agi ɣef kra n
yemdanen iḥesben iman-nsen d iḥeqqiyen ɣer Ṛebbi, iḥeqqṛen wiyaḍ :

<span id="an_18:10">18:10</span> Sin yergazen ulin ɣer lǧameɛ ad ẓallen
; yiwen (d ccix n lǧameɛ) n ukabar n ifariziyen, wayeḍ d amekkas
(ixeddmen ɣer iṛumaniyen istɛemṛen tamurt).

<span id="an_18:11">18:11</span> Afarizi ibedd yețẓalla yeqqaṛ deg
iman-is : a k-ḥemdeɣ a k-cekkṛeɣ a Sidi Ṛebbi imi ur lliɣ ara am wid
yellan d imcumen, d ixeddaɛen, yețɛicin di zzna ; a k-cekkṛeɣ daɣen imi
ur lliɣ ara am umekkas-agi.

<span id="an_18:12">18:12</span> ?țuẓumeɣ sin wussan di dduṛt, țțakeɣ
leɛcuṛ seg wayen sɛiɣ.

<span id="an_18:13">18:13</span> Amekkas-nni ibedd mebɛid, issetḥa ad
yerfed ula d allen-is ɣer igenni, yekkat deg idmaren-is yeqqaṛ : « a
Sidi Ṛebbi, ḥunn fell-i nekk yellan d amednub ! »

<span id="an_18:14">18:14</span> A wen-d-iniɣ : amekkas-nni yuɣal ɣer
wexxam-is di lehna, yețwaqbel ɣer Sidi Ṛebbi, ma d afarizi-nni ur
yețwaqbel ara. Kra n win yessimɣuṛen iman-is a t-id-isres Sidi Ṛebbi ;
ma d win yessimẓiyen iman-is a t-issaɛli.

<span id="an_18:15">18:15</span> Lɣaci țțawin-as-ed ula d llufanat
iwakken ad issers afus-is fell-asen a ten-ibarek. Inelmaden-is mi walan
ayagi, țnaɣen-ten.

<span id="an_18:16">18:16</span> Sidna Ɛisa issawel asen-d yenna :
Anfet-asen i warrac imeẓyanen a d-asen ɣuṛ-i ur ten-țțarrat ara, axaṭer
tagelda n Ṛebbi tețwahegga i wid yellan am nutni.

<span id="an_18:17">18:17</span> A wen-iniɣ tideț : kra n win ur neqbil
ara tageldit n Sidi Ṛebbi am weqcic ameẓyan, ur ț-ikeččem ara.

<span id="an_18:18">18:18</span> Yiwen umeṛkanti isteqsa Sidna Ɛisa
yenna-yas : Ay argaz n lɛali, d acu i yi ilaqen a t-xedmeɣ iwakken ad
sɛuɣ tudert n dayem ?

<span id="an_18:19">18:19</span> Sidna Ɛisa yenna-yas : Acuɣeṛ i
yi-d-tenniḍ « ay argaz lɛali » ? Ulac win yellan d lɛali-t anagar Sidi
Ṛebbi !

<span id="an_18:20">18:20</span> Tessneḍ ayen i d-tenna ccariɛa n Musa :
ur xeddem ara zzna, ur neqq ara, ur țțaker ara, ur țcehhid ara s ẓẓur,
qadeṛ baba-k d yemma-k.

<span id="an_18:21">18:21</span> Argaz-nni yenna-yas : Ayagi akk
xeddmeɣ-t seg wasmi lliɣ d ameẓyan.

<span id="an_18:22">18:22</span> Mi gesla Sidna Ɛisa i imeslayen agi,
yenna-yas : Ihi txuṣṣ-ik yiwet lḥaǧa kan : ṛuḥ zzenz ayen akk tesɛiḍ,
seddeq-it i yigellilen iwakken aț-țesɛuḍ agerruj deg igenni, tuɣaleḍ-ed
aț-țedduḍ yid-i.

<span id="an_18:23">18:23</span> Argaz-nni mi gesla i yimeslayen-agi
yeɣli-d fell-as leḥzen imi d ameṛkanti i gella.

<span id="an_18:24">18:24</span> Mi t-iwala yeḥzen, Sidna Ɛisa yenna :
Acḥal i sen-yewɛeṛ i wid yesɛan cci ad kecmen ɣer tgelda n Sidi Ṛebbi !

<span id="an_18:25">18:25</span> Ishel i welɣem ad iɛeddi di tiṭ n
tissegnit wala i umeṛkanti ad yekcem ɣer tgelda n Sidi Ṛebbi.

<span id="an_18:26">18:26</span> Wid i s-yeslan, nnan-as : Anwa i
gzemren ad ițțusellek ?

<span id="an_18:27">18:27</span> Sidna Ɛisa yerra-yasen : Ayen yellan d
lmuḥal i wemdan yeshel ɣer Sidi Ṛebbi.

<span id="an_18:28">18:28</span> Buṭrus yenna-yas : I nukkni yeǧǧan
kullec iwakken a neddu yid-ek ?

<span id="an_18:29">18:29</span> Sidna Ɛisa yenna-yasen : A wen-iniɣ
tideț : kra n win yeǧǧan axxam-is, tameṭṭut-is, ayetma- s, imawlan-is
neɣ arraw-is ɣef ddemma n tgeldit n Ṛebbi,

<span id="an_18:30">18:30</span> a s-d-ițunefk di ddunit-agi akteṛ n
wayen yeǧǧa yerna ɣer zdat, ad yesɛu tudert n dayem.

<span id="an_18:31">18:31</span> Sidna Ɛisa ijmeɛ-ed ɣuṛ-es tnac-nni
inelmaden-is yenna-yasen : Aql-aɣ a nali ɣer temdint n Lquds, ad idṛu
wayen akk i d-uran lenbiya ɣef wayen yeɛnan Mmi-s n bunadem.

<span id="an_18:32">18:32</span> Ad ițțusellem ger ifassen n lkeffaṛ, ad
ḍsen fell-as, ad kksen fell-as sser, a t-ssusfen.

<span id="an_18:33">18:33</span> A t-wwten s ujelkkaḍ ( s ucelliṭ ), a
t-enɣen ; ass wis tlata a d-iḥyu si ger lmegtin.

<span id="an_18:34">18:34</span> Inelmaden-is ur fhimen acemma deg wayen
i sen-d-yenna, lmeɛna n imeslayen-agi tețwaffer fell-asen. Ur ẓrin ara d
acu i gebɣa a t-id-yini.

<span id="an_18:35">18:35</span> Akken qṛib ad yaweḍ Sidna Ɛisa ɣer
temdint n Yiriku, yiwen uderɣal yeqqim rrif n webrid yessutur tin n
Ṛebbi.

<span id="an_18:36">18:36</span> Mi gesla i lɣaci țɛeddayen, yesteqsa d
acu i gedṛan ;

<span id="an_18:37">18:37</span> nnan-as belli d Ɛisa Anaṣari i
d-iɛeddan.

<span id="an_18:38">18:38</span> Dɣa iɛeggeḍ yenna : A Ɛisa, a mmi-s n
Sidna Dawed, ḥunn fell-i !

<span id="an_18:39">18:39</span> Wid yezwaren țdemmiren-t ɣer deffir

mmi-s n Sidna Dawed, ḥunn fell-i !

<span id="an_18:40">18:40</span> Sidna Ɛisa yeḥbes, yefka lameṛ a
s-t-id-awin. Mi d-yewweḍ ɣuṛ-es, yenna-yas :

<span id="an_18:41">18:41</span> D acu tebɣiḍ a k-t-xedmeɣ ? Yerra-yas-d
: A Sidi, bɣiɣ a yi-d-yuɣal yeẓri !

<span id="an_18:42">18:42</span> Sidna Ɛisa yenna-yas : Wali ! Imi
tumneḍ yis-i atan tețțusellkeḍ.

<span id="an_18:43">18:43</span> Imiren kan yuɣal-it-id yeẓri yețwali,
yedda d Sidna Ɛisa, iḥemmed Ṛebbi. Mi walan ayagi, lɣaci meṛṛa ḥemden
Ṛebbi.

<span id="an_19:3">19:3</span> Yebɣa ad iẓer anwa i d Sidna Ɛisa lameɛna

<span id="an_19:4">19:4</span> Yuzzel ɣer zdat, yuli ɣef ttejṛa iwakken

<span id="an_19:5">19:5</span> Mi gewweḍ Sidna Ɛisa ɣer wemkan-nni,

nseɣ ass-agi.

<span id="an_19:6">19:6</span> Zaci yers-ed s lemɣawla ad yesteṛḥeb s
Sidna Ɛisa.

<span id="an_19:7">19:7</span> Lɣaci mi walan ayagi smermugen wway gar

<span id="an_19:8">19:8</span> Zaci iqeṛṛeb-ed ɣer Sidna Ɛisa, yenna-yas
: A Sidi, atan ad fkeɣ i igellilen azgen n wayen sɛiɣ, yerna ma ḍelmeɣ

<span id="an_19:9">19:9</span> Sidna Ɛisa yenna-yas : Ass-agi leslak

<span id="an_19:10">19:10</span> Axaṭer Mmi-s n bunadem yusa-d ad inadi

<span id="an_19:11">19:11</span> Sidna Ɛisa yerna-d yiwen n lemtel i wid

imiren kan ara d-tas tgeldit n Sidi Ṛebbi.

<span id="an_19:12">19:12</span> Yenna yasen : Yiwen wergaz imawlan-is
muceɛen, ițeddu ad iṛuḥ ɣer yiwet n tmurt ibeɛden iwakken a t-sbedden d
agellid ɣef tmurt-is , imiren a d-yuɣal.

<span id="an_19:13">19:13</span> Qbel ad iṛuḥ, yessawel i ɛecṛa seg

<span id="an_19:14">19:14</span> Arraw n tmurt-is i t-ikeṛhen, ceggɛen
deffir-es kra n yemdanen ara yinin : « ur nebɣi ara ad yili wergaz agi d
agellid fell-aneɣ. »

<span id="an_19:15">19:15</span> Mi t-sbedden d agellid, yuɣal ed ɣer
tmurt-is ; yessawel i iqeddacen nni iwumi yefka tiwiztin iwakken ad
isteqsi mkul yiwen deg-sen d acu i d-yerbeḥ yis-sent.

<span id="an_19:16">19:16</span> Amezwaru yusa-d, yenna : A Sidi, s
twizeț-nni i yi-d-tefkiḍ rebḥeɣ-ed ɛecṛa nniḍen.

<span id="an_19:17">19:17</span> Agellid yenna-yas : D ayen yelhan ay
aqeddac lɛali, imi deg-ek laman ɣef lḥaǧa tamecṭuḥt, a k-fkeɣ aț-țḥekmeḍ
ɣef ɛecṛa n tudrin.

<span id="an_19:18">19:18</span> Aqeddac wis sin yusa-d yenna : A Sidi,
rebḥeɣ-ed xemsa n twiztin s twizeț i yi-d-tefkiḍ.

<span id="an_19:19">19:19</span> Agellid yenna-yas : Ula d kečč a
k-sbeddeɣ d lḥakem ɣef xemsa tudrin.

<span id="an_19:20">19:20</span> Yusa-d uqeddac wis tlata yenna : A
Sidi, ațah twizeț-inek ! Freɣ-ț deg yiwen uceṭṭiḍ ;

<span id="an_23:45">23:45</span> Leḥjab yellan deg umkan iqedsen di

<span id="an_23:47">23:47</span> Lqebṭan mi geẓra ayen yedṛan yețweḥḥid

d aḥeqqi, yețṛaǧu tageldit n Ṛebbi.
