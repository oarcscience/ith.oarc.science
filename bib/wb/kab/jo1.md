### Kabyle Bible: 1 John

meṛṛa.

<span id="an_2:3">2:3</span> Ihi ma nețțaɣ awal i Sidi Ṛebbi yerna

<span id="an_2:4">2:4</span> Win ara yinin : ssneɣ Ṛebbi yili ur yeṭṭif

wen-d-nniɣ yakan ay ilmeẓyen, uriɣ-awen axaṭer tesnem Baba Ṛebbi.

<span id="an_2:14">2:14</span> Uriɣ-awen ay ibabaten, axaṭer tesnem win

<span id="an_2:17">2:17</span> Ddunit d wayen yellan deg-s ad iɛeddi,

<span id="an_2:18">2:18</span> Ay arrac imeẓyanen, ț-țaswiɛt taneggarut

<span id="an_2:21">2:21</span> Uriɣ-awen mačči imi ur tessinem ara

<span id="an_2:22">2:22</span> Anwa i d akeddab anagar win inekkṛen

<span id="an_2:23">2:23</span> Win ara inekṛen Mmi-s n Ṛebbi yenkeṛ

<span id="an_2:24">2:24</span> Ma d kunwi, ayen teslam si tazwara ilaq

aț-țilim di tdukkli akk-d Baba Ṛebbi d Mmi-s.

<span id="an_2:25">2:25</span> Atan lweɛd i ɣ-d-ifka Lmasiḥ : ț-țudert

<span id="an_2:26">2:26</span> Uriɣ-awen ayagi ɣef wid yețnadin a

<span id="an_2:27">2:27</span> Ma d kunwi, Ṛṛuḥ iqedsen i d-yesmar

<span id="an_3:2">3:2</span> A wid eɛzizen, aql-aɣ tura d arraw n Ṛebbi,

<span id="an_3:3">3:3</span> Kra n win yessaramen deg-s ad yeṣfu akken

<span id="an_3:4">3:4</span> Kra n win idenben yerẓa ccariɛa n Sidi

<span id="an_3:10">3:10</span> Atah wayen i d-ițbegginen lxilaf yellan

<span id="an_3:11">3:11</span> Atan lameṛ i wumi teslam si tazwara :

<span id="an_4:15">4:15</span> Kra n win ara icehden belli Ɛisa d Mmi-s

<span id="an_4:16">4:16</span> Nukni nessen leḥmala i ɣ-iḥemmel Sidi

<span id="an_4:18">4:18</span> Win yesɛan leḥmala n tideț ur t-tkeččem

<span id="an_4:21">4:21</span> Atan ihi lameṛ i ɣ-d-iǧǧa Lmasiḥ : win

Ɛisa d Mmi-s n Ṛebbi.

<span id="an_5:6">5:6</span> Ɛisa Lmasiḥ, d win akken i d-yusan ɣer
ddunit s waman d idammen, mačči kan s waman n weɣḍas-is lameɛna s waman
akk-d idammen-is yuzzlen mi gemmut fell-aneɣ ; yerna d Ṛṛuḥ iqedsen i
d-icehden fell-as axaṭer Ṛṛuḥ iqedsen, d Ṛṛuḥ n tideț.

<span id="an_5:7">5:7</span>  Ṛṛuḥ iqedsen, aman d idammen, di tlata
yid-sen beggnen-d tideț-agi.

<span id="an_5:9">5:9</span> Ma yella cchada n yemdanen tețwaqbel,
cchada n Sidi Ṛebbi tugar tin n yemdanen ; axaṭer d Sidi Ṛebbi s
yiman-is i d-icehden ɣef Mmi-s.

<span id="an_5:10">5:10</span> Win yumnen s Mmi-s n Ṛebbi, yesɛa țbut
deg wul-is, ma d win ur numin ara, yerra Sidi Ṛebbi d akeddab imi ur
yumin ara s wayen i d-iched Ṛebbi ɣef Mmi-s.

<span id="an_5:11">5:11</span> Cchada-yagi ț-țudert n dayem i ɣ-d-yefka
Sidi Ṛebbi, tudert-agi di Mmi-s i tella.

<span id="an_5:12">5:12</span> Win yumnen s Mmi-s n Ṛebbi yesɛa tudert ;
win ur numin ara s Mmi-s n Ṛebbi ur yesɛi ara tudert.

<span id="an_5:13">5:13</span> Ayagi uriɣ-awen-t-id iwakken aț-țeẓrem
belli tesɛam tudert n ṣṣeḥ yețdumun, kunwi yumnen s Mmi-s n Ṛebbi.

<span id="an_5:14">5:14</span> Tura atah lețkal i nesɛa ɣuṛ-es : ma
nessuter-as kra yellan di lebɣi-s, yețḥessis-aɣ-d ;

<span id="an_5:15">5:15</span> ma yella neẓra belli yețḥessis-aɣ-d, kra
n wayen ara s nessuter, neḥsa belli yewweḍ-aɣ-d.

<span id="an_5:16">5:16</span> Ma yella win yeẓran gma-s yedneb yili
ddnub-is ur yețțawi ara ɣer lmut, ilaq ad yedɛu ɣer Sidi Ṛebbi, nețța a
s-d-yefk tudert ; nniɣ-ed ayagi ɣef wid ixeddmen ddnub ur nețțawi ara
ɣer lmut. Yella ddnub yețțawin ɣer lmut lameɛna mačči ɣef wid ixeddmen
ddnub-agi i wen-d-nniɣ aț-țedɛum.

<span id="an_5:17">5:17</span> Kra yellan d lbaṭel, d ddnub, meɛna llan
ddnubat ur nețțawi ara ɣer lmut.

<span id="an_5:18">5:18</span> Neẓra belli kra n wid yellan d arraw n
Sidi Ṛebbi ur țɛicin ara di ddnub ; meɛna Mmi-s n Ṛebbi yețɛassa
fell-asen yerna Cciṭan ur yezmir ara a ten-yennal.

<span id="an_5:19">5:19</span> Neẓra belli d arraw n Sidi Ṛebbi i nella,
ma d ddunit teɣli ger ifassen n wemcum.

<span id="an_5:20">5:20</span> Neẓra daɣen Mmi-s n Ṛebbi yusa-d ɣer
ddunit yefka-yaɣ-d lefhama s wayes ara nissin Ṛebbi n țideț. Deg-s i
neṭṭef s yisem n Mmi-s Ɛisa Lmasiḥ ; d nețța i d Ṛebbi n ṣṣeḥ, i
ț-țudert yețdumun.

<span id="an_5:21">5:21</span> Ay ilmeẓyen, ḥadret aț-țɛebdem iṛebbiten
n lekdeb.
