### Modern Hebrew Bible: Judges

<span id="an_1:1">1:1</span> ויהי אחרי מות יהושע וישאלו בני ישראל ביהוה
לאמר מי יעלה לנו אל הכנעני בתחלה להלחם בו׃

<span id="an_1:2">1:2</span> ויאמר יהוה יהודה יעלה הנה נתתי את הארץ
בידו׃

<span id="an_1:3">1:3</span> ויאמר יהודה לשמעון אחיו עלה אתי בגורלי
ונלחמה בכנעני והלכתי גם אני אתך בגורלך וילך אתו שמעון׃

<span id="an_1:4">1:4</span> ויעל יהודה ויתן יהוה את הכנעני והפרזי בידם
ויכום בבזק עשרת אלפים איש׃

<span id="an_1:5">1:5</span> וימצאו את אדני בזק בבזק וילחמו בו ויכו את
הכנעני ואת הפרזי׃

<span id="an_1:6">1:6</span> וינס אדני בזק וירדפו אחריו ויאחזו אתו
ויקצצו את בהנות ידיו ורגליו׃

<span id="an_1:7">1:7</span> ויאמר אדני בזק שבעים מלכים בהנות ידיהם
ורגליהם מקצצים היו מלקטים תחת שלחני כאשר עשיתי כן שלם לי אלהים ויביאהו
ירושלם וימת שם׃

<span id="an_1:8">1:8</span> וילחמו בני יהודה בירושלם וילכדו אותה ויכוה
לפי חרב ואת העיר שלחו באש׃

<span id="an_1:9">1:9</span> ואחר ירדו בני יהודה להלחם בכנעני יושב ההר
והנגב והשפלה׃

<span id="an_1:10">1:10</span> וילך יהודה אל הכנעני היושב בחברון ושם
חברון לפנים קרית ארבע ויכו את ששי ואת אחימן ואת תלמי׃

<span id="an_1:11">1:11</span> וילך משם אל יושבי דביר ושם דביר לפנים
קרית ספר׃

<span id="an_1:12">1:12</span> ויאמר כלב אשר יכה את קרית ספר ולכדה ונתתי
לו את עכסה בתי לאשה׃

<span id="an_1:13">1:13</span> וילכדה עתניאל בן קנז אחי כלב הקטן ממנו
ויתן לו את עכסה בתו לאשה׃

<span id="an_1:14">1:14</span> ויהי בבואה ותסיתהו לשאול מאת אביה השדה
ותצנח מעל החמור ויאמר לה כלב מה לך׃

<span id="an_1:15">1:15</span> ותאמר לו הבה לי ברכה כי ארץ הנגב נתתני
ונתתה לי גלת מים ויתן לה כלב את גלת עלית ואת גלת תחתית׃

<span id="an_1:16">1:16</span> ובני קיני חתן משה עלו מעיר התמרים את בני
יהודה מדבר יהודה אשר בנגב ערד וילך וישב את העם׃

<span id="an_1:17">1:17</span> וילך יהודה את שמעון אחיו ויכו את הכנעני
יושב צפת ויחרימו אותה ויקרא את שם העיר חרמה׃

<span id="an_1:18">1:18</span> וילכד יהודה את עזה ואת גבולה ואת אשקלון
ואת גבולה ואת עקרון ואת גבולה׃

<span id="an_1:19">1:19</span> ויהי יהוה את יהודה וירש את ההר כי לא
להוריש את ישבי העמק כי רכב ברזל להם׃

<span id="an_1:20">1:20</span> ויתנו לכלב את חברון כאשר דבר משה ויורש
משם את שלשה בני הענק׃

<span id="an_1:21">1:21</span> ואת היבוסי ישב ירושלם לא הורישו בני בנימן
וישב היבוסי את בני בנימן בירושלם עד היום הזה׃

<span id="an_1:22">1:22</span> ויעלו בית יוסף גם הם בית אל ויהוה עמם׃

<span id="an_1:23">1:23</span> ויתירו בית יוסף בבית אל ושם העיר לפנים
לוז׃

<span id="an_1:24">1:24</span> ויראו השמרים איש יוצא מן העיר ויאמרו לו
הראנו נא את מבוא העיר ועשינו עמך חסד׃

<span id="an_1:25">1:25</span> ויראם את מבוא העיר ויכו את העיר לפי חרב
ואת האיש ואת כל משפחתו שלחו׃

<span id="an_1:26">1:26</span> וילך האיש ארץ החתים ויבן עיר ויקרא שמה
לוז הוא שמה עד היום הזה׃

<span id="an_1:27">1:27</span> ולא הוריש מנשה את בית שאן ואת בנותיה ואת
תענך ואת בנתיה ואת ישב דור ואת בנותיה ואת יושבי יבלעם ואת בנתיה ואת
יושבי מגדו ואת בנותיה ויואל הכנעני לשבת בארץ הזאת׃

<span id="an_1:28">1:28</span> ויהי כי חזק ישראל וישם את הכנעני למס
והוריש לא הורישו׃

<span id="an_1:29">1:29</span> ואפרים לא הוריש את הכנעני היושב בגזר וישב
הכנעני בקרבו בגזר׃

<span id="an_1:30">1:30</span> זבולן לא הוריש את יושבי קטרון ואת יושבי
נהלל וישב הכנעני בקרבו ויהיו למס׃

<span id="an_1:31">1:31</span> אשר לא הוריש את ישבי עכו ואת יושבי צידון
ואת אחלב ואת אכזיב ואת חלבה ואת אפיק ואת רחב׃

<span id="an_1:32">1:32</span> וישב האשרי בקרב הכנעני ישבי הארץ כי לא
הורישו׃

<span id="an_1:33">1:33</span> נפתלי לא הוריש את ישבי בית שמש ואת ישבי
בית ענת וישב בקרב הכנעני ישבי הארץ וישבי בית שמש ובית ענת היו להם למס׃

<span id="an_1:34">1:34</span> וילחצו האמרי את בני דן ההרה כי לא נתנו
לרדת לעמק׃

<span id="an_1:35">1:35</span> ויואל האמרי לשבת בהר חרס באילון ובשעלבים
ותכבד יד בית יוסף ויהיו למס׃

<span id="an_1:36">1:36</span> וגבול האמרי ממעלה עקרבים מהסלע ומעלה׃

<span id="an_2:1">2:1</span> ויעל מלאך יהוה מן הגלגל אל הבכים ויאמר אעלה
אתכם ממצרים ואביא אתכם אל הארץ אשר נשבעתי לאבתיכם ואמר לא אפר בריתי אתכם
לעולם׃

<span id="an_2:2">2:2</span> ואתם לא תכרתו ברית ליושבי הארץ הזאת
מזבחותיהם תתצון ולא שמעתם בקלי מה זאת עשיתם׃

<span id="an_2:3">2:3</span> וגם אמרתי לא אגרש אותם מפניכם והיו לכם
לצדים ואלהיהם יהיו לכם למוקש׃

<span id="an_2:4">2:4</span> ויהי כדבר מלאך יהוה את הדברים האלה אל כל
בני ישראל וישאו העם את קולם ויבכו׃

<span id="an_2:5">2:5</span> ויקראו שם המקום ההוא בכים ויזבחו שם ליהוה׃

<span id="an_2:6">2:6</span> וישלח יהושע את העם וילכו בני ישראל איש
לנחלתו לרשת את הארץ׃

<span id="an_2:7">2:7</span> ויעבדו העם את יהוה כל ימי יהושע וכל ימי
הזקנים אשר האריכו ימים אחרי יהושוע אשר ראו את כל מעשה יהוה הגדול אשר עשה
לישראל׃

<span id="an_2:8">2:8</span> וימת יהושע בן נון עבד יהוה בן מאה ועשר
שנים׃

<span id="an_2:9">2:9</span> ויקברו אותו בגבול נחלתו בתמנת חרס בהר אפרים
מצפון להר געש׃

<span id="an_2:10">2:10</span> וגם כל הדור ההוא נאספו אל אבותיו ויקם דור
אחר אחריהם אשר לא ידעו את יהוה וגם את המעשה אשר עשה לישראל׃

<span id="an_2:11">2:11</span> ויעשו בני ישראל את הרע בעיני יהוה ויעבדו
את הבעלים׃

<span id="an_2:12">2:12</span> ויעזבו את יהוה אלהי אבותם המוציא אותם
מארץ מצרים וילכו אחרי אלהים אחרים מאלהי העמים אשר סביבותיהם וישתחוו להם
ויכעסו את יהוה׃

<span id="an_2:13">2:13</span> ויעזבו את יהוה ויעבדו לבעל ולעשתרות׃

<span id="an_2:14">2:14</span> ויחר אף יהוה בישראל ויתנם ביד שסים וישסו
אותם וימכרם ביד אויביהם מסביב ולא יכלו עוד לעמד לפני אויביהם׃

<span id="an_2:15">2:15</span> בכל אשר יצאו יד יהוה היתה בם לרעה כאשר
דבר יהוה וכאשר נשבע יהוה להם ויצר להם מאד׃

<span id="an_2:16">2:16</span> ויקם יהוה שפטים ויושיעום מיד שסיהם׃

<span id="an_2:17">2:17</span> וגם אל שפטיהם לא שמעו כי זנו אחרי אלהים
אחרים וישתחוו להם סרו מהר מן הדרך אשר הלכו אבותם לשמע מצות יהוה לא עשו
כן׃

<span id="an_2:18">2:18</span> וכי הקים יהוה להם שפטים והיה יהוה עם השפט
והושיעם מיד איביהם כל ימי השופט כי ינחם יהוה מנאקתם מפני לחציהם ודחקיהם׃

<span id="an_2:19">2:19</span> והיה במות השופט ישבו והשחיתו מאבותם ללכת
אחרי אלהים אחרים לעבדם ולהשתחות להם לא הפילו ממעלליהם ומדרכם הקשה׃

<span id="an_2:20">2:20</span> ויחר אף יהוה בישראל ויאמר יען אשר עברו
הגוי הזה את בריתי אשר צויתי את אבותם ולא שמעו לקולי׃

<span id="an_2:21">2:21</span> גם אני לא אוסיף להוריש איש מפניהם מן
הגוים אשר עזב יהושע וימת׃

<span id="an_2:22">2:22</span> למען נסות בם את ישראל השמרים הם את דרך
יהוה ללכת בם כאשר שמרו אבותם אם לא׃

<span id="an_2:23">2:23</span> וינח יהוה את הגוים האלה לבלתי הורישם מהר
ולא נתנם ביד יהושע׃

<span id="an_3:1">3:1</span> ואלה הגוים אשר הניח יהוה לנסות בם את ישראל
את כל אשר לא ידעו את כל מלחמות כנען׃

<span id="an_3:2">3:2</span> רק למען דעת דרות בני ישראל ללמדם מלחמה רק
אשר לפנים לא ידעום׃

<span id="an_3:3">3:3</span> חמשת סרני פלשתים וכל הכנעני והצידני והחוי
ישב הר הלבנון מהר בעל חרמון עד לבוא חמת׃

<span id="an_3:4">3:4</span> ויהיו לנסות בם את ישראל לדעת הישמעו את מצות
יהוה אשר צוה את אבותם ביד משה׃

<span id="an_3:5">3:5</span> ובני ישראל ישבו בקרב הכנעני החתי והאמרי
והפרזי והחוי והיבוסי׃

<span id="an_3:6">3:6</span> ויקחו את בנותיהם להם לנשים ואת בנותיהם נתנו
לבניהם ויעבדו את אלהיהם׃

<span id="an_3:7">3:7</span> ויעשו בני ישראל את הרע בעיני יהוה וישכחו את
יהוה אלהיהם ויעבדו את הבעלים ואת האשרות׃

<span id="an_3:8">3:8</span> ויחר אף יהוה בישראל וימכרם ביד כושן רשעתים
מלך ארם נהרים ויעבדו בני ישראל את כושן רשעתים שמנה שנים׃

<span id="an_3:9">3:9</span> ויזעקו בני ישראל אל יהוה ויקם יהוה מושיע
לבני ישראל ויושיעם את עתניאל בן קנז אחי כלב הקטן ממנו׃

<span id="an_3:10">3:10</span> ותהי עליו רוח יהוה וישפט את ישראל ויצא
למלחמה ויתן יהוה בידו את כושן רשעתים מלך ארם ותעז ידו על כושן רשעתים׃

<span id="an_3:11">3:11</span> ותשקט הארץ ארבעים שנה וימת עתניאל בן קנז׃

<span id="an_3:12">3:12</span> ויספו בני ישראל לעשות הרע בעיני יהוה
ויחזק יהוה את עגלון מלך מואב על ישראל על כי עשו את הרע בעיני יהוה׃

<span id="an_3:13">3:13</span> ויאסף אליו את בני עמון ועמלק וילך ויך את
ישראל ויירשו את עיר התמרים׃

<span id="an_3:14">3:14</span> ויעבדו בני ישראל את עגלון מלך מואב שמונה
עשרה שנה׃

<span id="an_3:15">3:15</span> ויזעקו בני ישראל אל יהוה ויקם יהוה להם
מושיע את אהוד בן גרא בן הימיני איש אטר יד ימינו וישלחו בני ישראל בידו
מנחה לעגלון מלך מואב׃

<span id="an_3:16">3:16</span> ויעש לו אהוד חרב ולה שני פיות גמד ארכה
ויחגר אותה מתחת למדיו על ירך ימינו׃

<span id="an_3:17">3:17</span> ויקרב את המנחה לעגלון מלך מואב ועגלון איש
בריא מאד׃

<span id="an_3:18">3:18</span> ויהי כאשר כלה להקריב את המנחה וישלח את
העם נשאי המנחה׃

<span id="an_3:19">3:19</span> והוא שב מן הפסילים אשר את הגלגל ויאמר דבר
סתר לי אליך המלך ויאמר הס ויצאו מעליו כל העמדים עליו׃

<span id="an_3:20">3:20</span> ואהוד בא אליו והוא ישב בעלית המקרה אשר לו
לבדו ויאמר אהוד דבר אלהים לי אליך ויקם מעל הכסא׃

<span id="an_3:21">3:21</span> וישלח אהוד את יד שמאלו ויקח את החרב מעל
ירך ימינו ויתקעה בבטנו׃

<span id="an_3:22">3:22</span> ויבא גם הנצב אחר הלהב ויסגר החלב בעד הלהב
כי לא שלף החרב מבטנו ויצא הפרשדנה׃

<span id="an_3:23">3:23</span> ויצא אהוד המסדרונה ויסגר דלתות העליה בעדו
ונעל׃

<span id="an_3:24">3:24</span> והוא יצא ועבדיו באו ויראו והנה דלתות
העליה נעלות ויאמרו אך מסיך הוא את רגליו בחדר המקרה׃

<span id="an_3:25">3:25</span> ויחילו עד בוש והנה איננו פתח דלתות העליה
ויקחו את המפתח ויפתחו והנה אדניהם נפל ארצה מת׃

<span id="an_3:26">3:26</span> ואהוד נמלט עד התמהמהם והוא עבר את הפסילים
וימלט השעירתה׃

<span id="an_3:27">3:27</span> ויהי בבואו ויתקע בשופר בהר אפרים וירדו
עמו בני ישראל מן ההר והוא לפניהם׃

<span id="an_3:28">3:28</span> ויאמר אלהם רדפו אחרי כי נתן יהוה את

איש לעבר׃

<span id="an_3:29">3:29</span> ויכו את מואב בעת ההיא כעשרת אלפים איש כל
שמן וכל איש חיל ולא נמלט איש׃

<span id="an_3:30">3:30</span> ותכנע מואב ביום ההוא תחת יד ישראל ותשקט
הארץ שמונים שנה׃

<span id="an_3:31">3:31</span> ואחריו היה שמגר בן ענת ויך את פלשתים שש
