### Xhosa Bible: 2 Thessalonians

<span id="an_1:1">1:1</span> UPawulos noSilvano noTimoti babhalela
ibandla labaseTesalonika, elikuThixo uBawo wethu, neNkosi yethu uYesu
Kristu;

<span id="an_1:2">1:2</span> bathi, Makube lubabalo kuni, noxolo oluvela
kuThixo uBawo, neNkosi yethu uYesu Kristu.

<span id="an_1:3">1:3</span> Simelwe kukuhlala sibulela kuThixo ngenxa
yenu, bazalwana, njengokuba kufanelekile, ngokuba lukhula
ngokugqithiseleyo ukholo lwenu, lusanda uthandano lwenu nonke ngabanye;

<span id="an_1:4">1:4</span> ngokokude thina ngokwethu siqhayise ngani
emabandleni kaThixo, ngenxa yonyamezelo lwenu, nokholo lwenu,
ezintshutshisweni zenu zonke, nasezimbandezelweni enizinyamezelayo;

<span id="an_1:5">1:5</span> umbonakaliso ke lowo womgwebo onobulungisa
kaThixo, ukuze nibalelwe ekuthini nibufanele ubukumkani bukaThixo,
enithi ngenxa yabo nive nokuva ubunzima;

<span id="an_1:6">1:6</span> ukuba kanti yinto ebubulungisa kuThixo
ukubabuyekeza ngembandezelo abanibandezelayo;

<span id="an_1:7">1:7</span> athi, nina babandezelwayo, anibuyekeze
ngokuniphumza kunye nathi, ekutyhilekeni kweNkosi uYesu, ivela
emazulwini inezithunywa zamandla ayo;

<span id="an_1:8">1:8</span> inomlilo odangazelayo, ibaphindezela
abangamaziyo uThixo, nabangazilulameliyo iindaba ezilungileyo zeNkosi
yethu uYesu Kristu;

<span id="an_1:9">1:9</span> bona abo baya kuva ukubethwa, intshabalalo
engunaphakade ke leyo, bemke ebusweni beNkosi, nasebuqaqawulini
bokomelela kwayo;

<span id="an_1:10">1:10</span> xa ithe yeza kuzukiswa kubangcwele bayo,
imangaliseke kubo bonke abakholwayo (ngokuba nakholwa bubungqina bethu
kuni), ngaloo mini.

<span id="an_1:11">1:11</span> Ngoko ke sihlala sithandaza nokuthandaza
ngenxa yenu, ukuba uThixo wethu anibalele ekuthini nilufanele ubizo
lwenu, azalise ngokunamandla konke ukukholiswa kukulunga, nomsebenzi
wokholo;

<span id="an_1:12">1:12</span> ukuze lizukiswe igama leNkosi yethu uYesu
Kristu kuni, nani nizukiswe kuye, ngokobabalo lukaThixo wethu, neNkosi
yethu uYesu Kristu.

<span id="an_2:1">2:1</span> Ke kaloku siyanicela, bazalwana, ngenxa
yokufika kweNkosi yethu uYesu Kristu, nokuhlanganiselwa kwethu kuyo,

<span id="an_2:2">2:2</span> ukuba ningazanyazanyiswa kwakamsinya,
nahlukane nengqiqo; ukuba ningabi navuso ngamoya, ngalizwi, ngancwadi
ingathi, ivela kuthi, ngokungathi iselingene imini leyo kaKristu.

<span id="an_2:3">2:3</span> Makungabikho bani unilukuhlayo nangalunye
uhlobo; ngokuba ayinakufika yona, lungakhanga lufike ulahlo kuqala,
atyhileke umntu wesono, unyana wotshabalalo;

<span id="an_2:4">2:4</span> lowo uchasayo, uziphakamisayo ngaphezu
kwento yonke ekuthiwa nguThixo, nehlonelwayo, ngokokude azihlalise
etempileni kaThixo, ezenza uThixo, ezibonisa ukuba unguThixo.

<span id="an_2:5">2:5</span> Anikhumbuli na, ukuba oko bendisekuni
ndanixelela ezi zinto?

<span id="an_2:6">2:6</span> Nangoku niyakwazi okubambezelayo, ukuze
atyhileke ngelakhe ixesha.

<span id="an_2:7">2:7</span> Kuba imfihlelo yokuchasana nomthetho
iselisebenza; kodwa kulindwe ukuba ade lowo ubambezelayo ngoku asuswe
kanye.

<span id="an_2:8">2:8</span> Andule ukutyhileka ongenamthetho; lowo iya
kumtshisa iNkosi ngomoya womlomo wayo, imbangise ngokubonakala kokufika
kwayo:

<span id="an_2:9">2:9</span> kwalowo, ukufika kungokokusebenza
kukaSathana, ngayo imisebenzi yonke yamandla, nemiqondiso, nezimanga
zobuxoki,

<span id="an_2:10">2:10</span> nangako ukulukuhla konke kokungalungisi
kwabo batshabalalayo; ngenxa enokuba bengakwamkelanga ukuyithanda
inyaniso, ukuze basindiswe.

<span id="an_2:11">2:11</span> Nangenxa yoko, uThixo uya kubathumela
ulahlekiso olusebenza ngamandla, ukuze bakholwe bubuxoki;

<span id="an_2:12">2:12</span> ukuze bagwetywe bonke abangakholwanga
yiyo inyaniso, basuka bakholiswa kuko ukungalungisi.

<span id="an_2:13">2:13</span> Ke thina simelwe kukuhlala sibulela kuye
uThixo ngenxa yenu, bazalwana bathandiweyo yiNkosi, ukuba uThixo
kwasekuqalekeni waninyulela ukusindiswa, ngokungcwalisa koMoya
nokukholwa yinyaniso;

<span id="an_2:14">2:14</span> awanibizela kuko oko ngazo iindaba
ezilungileyo zethu, ukuba nizuze uzuko lweNkosi yethu uYesu Kristu.

<span id="an_2:15">2:15</span> Ngoko ke, bazalwana, yimani, nizibambe
izithethe enazifundiswayo, nokuba kungelizwi, nokuba kungencwadi yethu.

<span id="an_2:16">2:16</span> Ke kaloku, iNkosi yethu uYesu Kristu
ngokwayo, noThixo uBawo wethu owasithandayo, wasinika intuthuzelo
engunaphakade, nethemba elilungileyo ngobabalo,

<span id="an_2:17">2:17</span> wanga angazithuthuzela iintliziyo zenu,
anizimase kuwo onke amazwi nemisebenzi elungileyo.

<span id="an_3:1">3:1</span> Elokugqiba, sithandazeleni, bazalwana,
ukuba ilizwi leNkosi likhawuleze, lizukiswe, njengoko lenjenjalo nakuni;

<span id="an_3:2">3:2</span> nokuba sihlangulwe ebantwini abangendawo,
abakhohlakeleyo; kuba ukholo alubantwini bonke.

<span id="an_3:3">3:3</span> Ithembekile ke yona iNkosi, eya kunizimasa,
inigcine kuyo inkohlakalo.

<span id="an_3:4">3:4</span> Sikholose ngani ke ngokwaseNkosini, ukuba
izinto esinithethela zona niyazenza, nabe nisaya kuzenza.

<span id="an_3:5">3:5</span> Yanga ke iNkosi ingathi iintliziyo zenu
izikhokelele eluthandweni lukaThixo, naselunyamezelweni lukaKristu.

<span id="an_3:6">3:6</span> Ke kaloku siyanithethela, bazalwana,
egameni leNkosi yethu uYesu Kristu, ukuba nikhwebuke kubo bonke
abazalwana abahamba ngokunxaxha, bengahambi ngokwesithethe abasamkelayo
kuthi.

<span id="an_3:7">3:7</span> Kuba niyazi ngokwenu, ukuba nimelwe
kukuthini na ukusilinganisa; ngokuba asinxaxhanga phakathi kwenu.

<span id="an_3:8">3:8</span> Asidlanga sonka samntu ngesisa; sidle
ngokubulaleka nangemigu, sisebenza ubusuku nemini, ukuze singabi bunzima
nakubani kuni;

<span id="an_3:9">3:9</span> kungengakuba sasingenagunya; senzela ukuze
sizinikele sibe ngumfuziselo kuni, nize nina nixelise thina.

<span id="an_3:10">3:10</span> Kuba naxenikweni besikuni, sanithethela
ngelithi, Ukuba umntu akathandi kusebenza, makangadli nokudla.

<span id="an_3:11">3:11</span> Kuba siyeva ukuba inxenye phakathi kwenu
ihamba ngokunxaxha, ingasebenzi nento, izifaka kwizinto zabanye.

<span id="an_3:12">3:12</span> Abanjalo ke siyabathethela, sibavuselela
ngayo iNkosi yethu uYesu Kristu, ukuba bathi, besebenza ngokuzola, badle
esisesabo isonka.

<span id="an_3:13">3:13</span> Ke nina, bazalwana, musani ukutha amandla
ekwenzeni okulungileyo.

<span id="an_3:14">3:14</span> Ukuba ke kukho bani ungalilulameliyo
ilizwi lethu ngencwadi le, mqondeni lowo, ningazidibanisi naye, ukuze
adane.

<span id="an_3:15">3:15</span> Noko ke ze ningambaleli ekuthini
ulutshaba; manimluleke njengomzalwana.

<span id="an_3:16">3:16</span> Yanga ke iNkosi yoxolo ngokwayo
inganinika uxolo ngamaxa onke, ngeendlela zonke. INkosi mayibe nani
nonke.

<span id="an_3:17">3:17</span> Umbuliso ngesam isandla, mna Pawulos.
Nguwo lowo umqondiso kuzo zonke iincwadi; ndenjenjalo ukubhala.

<span id="an_3:18">3:18</span> Ubabalo lweNkosi yethu uYesu Kristu
malube nani nonke. Amen.
