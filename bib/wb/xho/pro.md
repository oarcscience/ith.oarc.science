### Xhosa Bible: Proverbs

<span id="an_1:1">1:1</span> Imizekeliso kaSolomon unyana kaDavide,
ukumkani wamaSirayeli, Yokuba kwaziwe ubulumko noqeqesho;

<span id="an_1:2">1:2</span> Yokuba kuqondwe iintetho zengqondo;

<span id="an_1:3">1:3</span> Yokuba kwamkelwe uqeqesho lwengqiqo,
Ubulungisa, nesiko, nokuthe tye;

<span id="an_1:4">1:4</span> Yokuba izidenge zinikwe ubuqili, Ulutsha
lunikwe ukwazi nokunkqangiyela;

<span id="an_1:5">1:5</span> Ukuze eve osisilumko, aqokele afunde,
Oqondayo azuze amacebo obulumko;

<span id="an_1:6">1:6</span> Yokuba kuqondwe umzekeliso nezafobe
zokuthetha, Amazwi ezilumko nezintsonkotha zazo.

<span id="an_1:7">1:7</span> Ukoyika uYehova kukuqala kokwazi; Ubulumko
noqeqesho izimathane ziludelile.

<span id="an_1:8">1:8</span> Nyana wam, yiva uqeqesho lukayihlo,
Ungawulahli umyalo kanyoko.

<span id="an_1:9">1:9</span> Kuba ezo zinto zisisangqawe esihle
entlokweni yakho, Nezambalo emqaleni wakho.

<span id="an_1:10">1:10</span> Nyana wam, ukuba aboni bathe bakuhenda,
Uze ungavumi ke.

<span id="an_1:11">1:11</span> Ukuba bathe, Hamba nathi, Masilalele
igazi, Masibalalele abamsulwa bengenatyala;

<span id="an_1:12">1:12</span> Masibaginye njengelabafileyo behleli,
Begqibelele, njengabehla baye emhadini;

<span id="an_1:13">1:13</span> Sofumana ubuncwane obunqabileyo,
Sozizalisa izindlu zethu ngamaxhoba;

<span id="an_1:14">1:14</span> Wena ke woliphosa phakathi kwethu
iqashiso lakho, Yoba nye ingxowa yethu sonke:

<span id="an_1:15">1:15</span> Nyana wam, musa ukuhamba ngendlela nabo;
Lunqande unyawo lwakho emendweni wabo.

<span id="an_1:16">1:16</span> Kuba iinyawo zabo zigidimela ububi,
Zikhawulezela ukuphalaza igazi;

<span id="an_1:17">1:17</span> Kuba ufumana usanekwa umnatha Emehlweni
eentaka ezi zonke;

<span id="an_1:18">1:18</span> Ke bona balalela elabo igazi, Bazonda
owabo umphefumlo.

<span id="an_1:19">1:19</span> Unjalo umendo wabo bonke abangxamele
inzuzo embi; Imka nomphefumlo womniniyo.

<span id="an_1:20">1:20</span> Ubulumko bumemeza ngaphandle, Buvakalisa
izwi labo ezitratweni.

<span id="an_1:21">1:21</span> Bumemeza emantloko eendawo zembutho;
Ekungeneni kwamasango, Phakathi komzi, buthetha iintetho zabo, busithi:

<span id="an_1:22">1:22</span> Kunini na, ziyathandini, nithanda
ubuyatha, Abagxeki befuna ukugxeka, Izinyabi zithiya ukwazi?

<span id="an_1:23">1:23</span> Nithe nabuyela ekohlwayeni kwam,
Niyabona, ndowumpompozisela kuni umoya wam, Ndonazisa amazwi am.

<span id="an_1:24">1:24</span> Ngokokuba ndinibizile, anavuma, Ndolule
isandla sam, akwabakho ukhathalayo;

<span id="an_1:25">1:25</span> Nesuka naliyeka necebo lam lonke, Anavuma
ndakunohlwaya.

<span id="an_1:26">1:26</span> Nam ke ndonihleka ekusindekeni kwenu;
Ndogculela ekufikeni kokunkwantyiswa kwenu;

<span id="an_1:27">1:27</span> Ekufikeni kokunkwantyiswa kwenu
njengesithonga, Ekuzeni kokusindeka kwenu njengesaqhwithi, Ekufikelweni
kwenu yimbandezelo nenkcutheko.

<span id="an_1:28">1:28</span> Baya kwandula bandibize, ndingaphenduli,
Bandifune kwakusasa, bangandifumani;

<span id="an_1:29">1:29</span> Ngethuba lokuba bakuthiyile ukwazi,
Abakunyula ukoyika uYehova;

<span id="an_1:30">1:30</span> Abalivuma icebo lam, Basigiba sonke
isohlwayo sam:

<span id="an_1:31">1:31</span> Baya kudla ke isiqhamo sendlela yabo,
Bahluthe ngamacebo abo.

<span id="an_1:32">1:32</span> Kuba ukuphamba kweziyatha kuyazi bulala,
Nobunqobo bezinyabi buyazitshabala lisa.

<span id="an_1:33">1:33</span> Ke yena ondivayo uhleli ekholosile,
Wonwabile ekunkwantyiseni kobubi.

<span id="an_2:1">2:1</span> Nyana wam, ukuba uthe wazamkela iintetho
zam, Wayiqwebela kuwe imithetho yam,

<span id="an_2:2">2:2</span> Ukuze uyibazele ubulumko indlebe yakho,
Uyithobele ukuqonda intliziyo yakho;

<span id="an_2:3">2:3</span> Ewe, ukuba uthe wayibiza ingqondo, Walisa
ekuqondeni ilizwi lakho;

<span id="an_2:4">2:4</span> Ukuba uthe wabufuna njengesilivere, Wabumba
njengobutyebi obuselelweyo:

<span id="an_2:5">2:5</span> Uya kwandula ukukuqonda ukoyika uYehova,
Ukufumane ukumazi uThixo,

<span id="an_2:6">2:6</span> Ngokuba uYehova upha ubulumko; Emlonyeni
wakhe kuphuma ukwazi nokuqonda.

<span id="an_2:7">2:7</span> Ubaqwebela uzimaso abathe tye,
Uyingweletshetshe kwabahamba ngengqibelelo,

<span id="an_2:8">2:8</span> Ukuze alondoloze umendo wesiko, Agcine
nendlela yabakhe benceba.

<span id="an_2:9">2:9</span> Uya kwandula uqonde ubulungisa, Nesiko,
nokuthe tye, nayo yonke imikhondo yokulungileyo;

<span id="an_2:10">2:10</span> Kuba ubulumko buya kungena entliziyweni
yakho, Ukwazi kwenze mnandi umphefumlo wakho.

<span id="an_2:11">2:11</span> Ukunkqangiyela kokugcina, Ukuqonda
kokulondoloza;

<span id="an_2:12">2:12</span> Ukuba kukuhlangule endleleni embi,
Kubantu abathetha impenduka;

<span id="an_2:13">2:13</span> Kwabashiya umendo wokuthe tye, Ukuze
bahambe ngeendlela zobumnyama;

<span id="an_2:14">2:14</span> Kwabavuyela ukwenza ububi, Bagcobe
ngeempenduka zabenza ububi;

<span id="an_2:15">2:15</span> Abamendo ujibilizayo, Abathi gu bucala
ngemikhondo yabo;

<span id="an_2:16">2:16</span> Ukuze uhlangulwe kumkamntu, Kowolunye
uhlanga, ontetho yakhe igudileyo;

<span id="an_2:17">2:17</span> Lowo ushiya ikholwane lobutsha bakhe,
Awulibale umnqophiso woThixo wakhe.

<span id="an_2:18">2:18</span> Kuba utshona ekufeni nendlu yakhe,
Nemikhondo yakhe itshonela kwabangasekhoyo;

<span id="an_2:19">2:19</span> Bonke abangenayo kuye ababuyi, Abafiki
kumendo wobomi:

<span id="an_2:20">2:20</span> Ukuze uhambe ngendlela yabalungileyo,
Ugcine umendo wamalungisa.

<span id="an_2:21">2:21</span> Ngokuba abathe tye baya kulima ilizwe,
Basale kulo abagqibeleleyo,

<span id="an_2:22">2:22</span> Banqunyulwe ezweni abangendawo,
Abanginizayo bancothulwe kulo.

<span id="an_3:1">3:1</span> Nyana wam, musa ukuwulibala umyalelo wam;
Intliziyo yakho mayiyibambe imithetho yam;

<span id="an_3:2">3:2</span> Kuba iya kukongeza imihla emide, Neminyaka
yobomi, noxolo.

<span id="an_3:3">3:3</span> Inceba nenyaniso mayingakushiyi; Zinxibe
kuwe emqaleni; Zibhale elucwecweni lwentliziyo yakho,

<span id="an_3:4">3:4</span> Ubabalwe ube nengqiqo entle Emehlweni
kaThixo nawabantu.

<span id="an_3:5">3:5</span> Kholosa ngoYehova ngentliziyo yakho yonke,
Ungayami okwakho ukuqonda.

<span id="an_3:6">3:6</span> Uze umazi yena ezindleleni zakho zonke;
Wowulungelelanisa umendo wakho.

<span id="an_3:7">3:7</span> Musa ukuba sisilumko kwawakho amehlo;
Yoyika uYehova usuke entweni embi.

<span id="an_3:8">3:8</span> Ukuphila kobakho enkabeni yakho, Nokuqina
emathanjeni akho.

<span id="an_3:9">3:9</span> Mzukise uYehova ngobuncwane bakho,
Nangentlahlela yongeniselo lwakho lonke:

<span id="an_3:10">3:10</span> Ozala amaqonga akho bubuninzi, Nemikhombe
yakho yokukhongozela iphuphume iwayini entsha.

<span id="an_3:11">3:11</span> Uqeqesho lukaYehova, nyana wam, musa
ukulucekisa, Ungakruquki sisohlwayo sakhe;

<span id="an_3:12">3:12</span> Kuba lo amthandayo uYehova uyamohlwaya,
Njengoyise esohlwaya unyana akholwayo nguye.

<span id="an_3:13">3:13</span> Hayi, uyolo lomntu ofumene ubulumko,
Lomntu ozuze ingqondo!

<span id="an_3:14">3:14</span> Kuba lulungile urhwebo lwabo ngaphezu
korhwebo lwesilivere, Nongeniselo lwabo ngaphezu kwegolide embiweyo.

<span id="an_3:15">3:15</span> Bunqabile bona ngaphezu kwekorale,
Nezinto zakho zonke ezinqwenelekayo azilingani nabo.

<span id="an_3:16">3:16</span> Imihla emide isesandleni sabo sokunene,
Kwesokhohlo sabo bubutyebi nozuko.

<span id="an_3:17">3:17</span> Iindlela zabo ziindlela ezimnandi, Wonke
umendo wabo uluxolo.

<span id="an_3:18">3:18</span> Bungumthi wobomi kwababambelela kubo,
Unoyolo obubambayo.

<span id="an_3:19">3:19</span> UYehova waliseka ihlabathi ngobulumko,
Walizinzisa izulu ngengqondo.

<span id="an_3:20">3:20</span> Agqobhoka amanzi anzongonzongo ngokwazi
kwakhe, Savuza umbethe isibhakabhaka.

<span id="an_3:21">3:21</span> Nyana wam, ezi nto mazingemki emehlweni
akho: Londoloza uzimaso nomnkqangiyelo.

<span id="an_3:22">3:22</span> Zoba bubomi emphefumlweni, Nento entle
emqaleni wakho.

<span id="an_3:23">3:23</span> Uya kwandula uhambe ngendlela yakho
ukholosile. Lungabetheki unyawo lwakho.

<span id="an_3:24">3:24</span> Ukuba uthe walala, akuyi kunkwantya;
Wolala, bube mnandi ubuthongo bakho.

<span id="an_3:25">3:25</span> Musa ukoyika lunkwantyiso luzayo
ngesiquphe, Nasithonga sabangendawo, sakufika.

<span id="an_3:26">3:26</span> Kuba uYehova uya kuba yinqolonci yakho,
Alugcine unyawo lwakho lungabanjiswa.

<span id="an_3:27">3:27</span> Musa ukwala nento elungileyo kolunge
nayo, Kwakuba kusemandleni esandla sakho ukuyenza.

<span id="an_3:28">3:28</span> Musa ukuthi kummelwane wakho, Hamba,
wobuya uze, Ndokunika ngomso; Kanti unayo.

<span id="an_3:29">3:29</span> Musa ukuyila ububi ngommelwane wakho,
Ehleli nawe yena ekholosile.

<span id="an_3:30">3:30</span> Musa ukubambana nomntu kungekho nto,
Engakuphathanga kakubi nje.

<span id="an_3:31">3:31</span> Musa ukuyimonela indoda egonyamelayo,
Unganyuli nanye kwiindlela zayo.

<span id="an_3:32">3:32</span> Kuba othi gu bucala ulisikizi kuYehova;
Ke lunabathe tye ucweyo lwakhe.

<span id="an_3:33">3:33</span> Ingqalekiso kaYehova isendlwini
yongendawo; Ke isithili samalungisa uyasisikelela.

<span id="an_3:34">3:34</span> Ngokubhekisele kubagxeki, uyabagxeka; Ke
abalulamileyo uyabababala.

<span id="an_3:35">3:35</span> Izilumko ziya kuludla ilifa uzuko; Ke
izinyabi zamkela umrhumo wokucukucezwa.

<span id="an_4:1">4:1</span> Luveni, bonyana, uqeqesho lukayihlo,
Nizibaze iindlebe ukuze nikwazi ukuqonda.

<span id="an_4:2">4:2</span> Kuba ndininika imfundiso elungileyo; Musani
ukuwushiya umyalelo wam.

<span id="an_4:3">4:3</span> Kuba ndandingunyana kubawo, Ndithambile,
ndimnye phambi koma.

<span id="an_4:4">4:4</span> Wandiyala ke, wathi kum, Intliziyo yakho
mayiwabambe amazwi am; Gcina imithetho yam, uphile,

<span id="an_4:5">4:5</span> Rhweba ubulumko, rhweba ingqondo; Musa
ukulibala, ungathi gu bucala entethweni yomlomo wam.

<span id="an_4:6">4:6</span> Musa ukubushiya, bokugcina; Buthande,
bokulondoloza.

<span id="an_4:7">4:7</span> Ingqalo yobulumko ithi, Rhweba ubulumko,
Ngako konke ukurhweba kwakho urhwebe ingqondo.

<span id="an_4:8">4:8</span> Bunonelele, bokuphakamisa; Bokuzukisa, xa
uthe wabuwola.

<span id="an_4:9">4:9</span> Boyinika intloko yakho isangqawe esihle,
Bokupha isithsaba sokuhomba.

<span id="an_4:10">4:10</span> Yiva, nyana wam, uyamkele intetho yam,
Iba mininzi kuwe iminyaka yobomi.

<span id="an_4:11">4:11</span> Ndikuyalele indlela yobulumko,
Ndikunyathelise emkhondweni wokuthe tye.

<span id="an_4:12">4:12</span> Ekuhambeni kwakho akayi kuxinana amabanga
akho; Nokuba uthe wagidima, akuyi kukhubeka.

<span id="an_4:13">4:13</span> Bambelela eluqeqeshweni, musa ukuluyeka;
Lulondoloze, ngokuba lububomi bakho.

<span id="an_4:14">4:14</span> Musa ukungena emendweni wabangendawo,
Unganyatheli endleleni yabanobubi.

<span id="an_4:15">4:15</span> Yiyeke; musa ukuya kugqitha kuyo;
Yicezele, ugqithe.

<span id="an_4:16">4:16</span> Kuba abalali, ukuba abathanga benze
ububi; Bemkelwa bubuthongo ukuba abathanga bakhubekise abanye.

<span id="an_4:17">4:17</span> Kuba badla isonka sokungendawo, Basele
iwayini yogonyamelo.

<span id="an_4:18">4:18</span> Ke wona umendo wamalungisa
unjengokukhanya komso, Okuya kuba mhlophe ngokuba mhlophe, kude kube
semini enkulu.

<span id="an_4:19">4:19</span> Indlela yabangendawo injengesithokothoko;
Abayazi into abakhubeka kuyo.

<span id="an_4:20">4:20</span> Nyana wam, wabazele indlebe amazwi am;
Intetho yam yithobele indlebe yakho.

<span id="an_4:21">4:21</span> Mayingemki emehlweni akho; Yigcine
entliziyweni yakho ngaphakathi.

<span id="an_4:22">4:22</span> Kuba ibubomi kwabayifumanayo, Nempiliso
enyameni yabo yonke.

<span id="an_4:23">4:23</span> Nazintweni zonke zigcinwayo, londoloza
intliziyo yakho; Kuba aphuma kuyo amathende obomi.

<span id="an_4:24">4:24</span> Kususe kuwe ukujibiliza komlomo, Nokuthi
gu bucala komlomo kudedise kuwe.

<span id="an_4:25">4:25</span> Amehlo akho makakhangele athi gca,
Neenkophe zakho zithi nzo kokuphambi kwakho.

<span id="an_4:26">4:26</span> Lungelelanisa umkhondo wonyawo lwakho,
Zonke iindlela zakho zizimaseke.

<span id="an_4:27">4:27</span> Musa ukuthi gu bucala ngasekunene
nangasekhohlo; Susa unyawo lwakho entweni embi.

<span id="an_5:1">5:1</span> Nyana wam, bubazele indlebe ubulumko bam,
Ingqondo yam uyithobele indlebe yakho;

<span id="an_5:2">5:2</span> Ukuze ugcine iminkqangiyelo, Ulondoloze
ukwazi umlomo wakho.

<span id="an_5:3">5:3</span> Ngokuba ivuza incindi yobusi imilebe
yomlomo womkamntu, Libuthelezi ngaphezu kweoli ikhuhlangubo lakhe;

<span id="an_5:4">5:4</span> Ke ukuphela kwakhe kukrakra
njengomhlonyane, Kubukhali njengekrele elintlangothi mbini.

<span id="an_5:5">5:5</span> Iinyawo zakhe zihla ziye ekufeni, Amabanga
akhe abambelele kwelabafileyo.

<span id="an_5:6">5:6</span> Akangeze wahamba ngomendo wobomi; Imikhondo
yakhe iyajikajika, engazi.

<span id="an_5:7">5:7</span> Ke ngoku, bonyana, ndiveni, Ningemki
entethweni yomlomo wam.

<span id="an_5:8">5:8</span> Mayibe kude kuye indlela yakho, Ungasondeli
emnyango wendlu yakhe;

<span id="an_5:9">5:9</span> Hleze uyinike abanye intlahla yakho,
Uyinike isijorha iminyaka yakho;

<span id="an_5:10">5:10</span> Hleze kuhluthe abasemzini ngamandla akho,

<span id="an_5:11">5:11</span> Ugcume ekupheleni kwakho, Ekudlekeni
kwenyama yakho nezihlunu zakho;

<span id="an_5:12">5:12</span> Uthi, Bendiluthiyeleni na uqeqesho,
Yasigiba isohlwayo intliziyo yam?

<span id="an_5:13">5:13</span> Andaliphulaphula izwi labayali bam;
Andayithobela indlebe yam kubafundisi bam!

<span id="an_5:14">5:14</span> Ndaphantsa ukuba sebubini bonke
Esazulwini sesikhungu nebandla.

<span id="an_5:15">5:15</span> Sela amanzi kwelakho iqula, Nemiqukuqela
yamanzi ephuma emthonjeni wakho.

<span id="an_5:16">5:16</span> Mayiphalale phandle imithombo yakho, Ibe
yimijelo yamanzi ezitratweni.

<span id="an_5:17">5:17</span> Mayibe yeyakho, yeyakho wedwa, Ingabi
yeyabangengabo abakowenu kunye nawe.

<span id="an_5:18">5:18</span> Malisikelelwe ithende lakho, Uvuye
ngumfazi wobutsha bakho;

<span id="an_5:19">5:19</span> Makabe yimazi yexhama elithandekayo,
neyegogo elihle; Amabele akhe akukholise ngamaxesha onke, Uyotywe
luthando lwakhe kuphele.

<span id="an_5:20">5:20</span> Yini na, nyana wam, ukuba uyotywe
ngumkamntu, Uwole isifuba sowolunye uhlanga?

<span id="an_5:21">5:21</span> Kuba iindlela zomntu ziphambi kwamehlo
kaYehova, Ayilungelelanise imikhondo yakhe.

<span id="an_5:22">5:22</span> Ubugwenxa bakhe bumthi hlasi ongendawo,
Abanjwe ziimbophelelo zesono sakhe.

<span id="an_5:23">5:23</span> Yena ufa kukuswela uqeqesho, Ayobe
ebuninzini bokumatha kwakhe.

<span id="an_6:1">6:1</span> Nyana wam, ukuba uthe wammela ummelwane
wakho, Wabambana nomnye ngesandla:

<span id="an_6:2">6:2</span> Urhintyelwe ngentetho yomlomo wakho, Uthiwe
hlasi ngentetho yomlomo wakho.

<span id="an_6:3">6:3</span> Yenza le nto ngoku, nyana wam, uzihlangule;
Kuba uzifake esandleni sommelwane wakho; Yiya, ziwise phantsi, umzame
ummelwane wakho.

<span id="an_6:4">6:4</span> Musa ukuwanika ubuthongo amehlo akho,
Ungaziniki kozela iinkophe zakho;

<span id="an_6:5">6:5</span> Zihlangule esandleni njengebhadi,
Njengentaka esandleni somthiyeli.

<span id="an_6:6">6:6</span> Yiya embovaneni, vilandini; Khangela
iindlela zayo, ulumke:

<span id="an_6:7">6:7</span> Yona ithi, ingenamphathi, Namongameli,
namlawuli,

<span id="an_6:8">6:8</span> Isuke isilungise isonka sayo ehlotyeni,
Ikubuthe ukudla kwayo ngexa lokuvuna.

<span id="an_6:9">6:9</span> Kunini na ulele, vilandini? Uya kuvuka nini
na ebuthongweni bakho?

<span id="an_6:10">6:10</span> Xa uthe, Intwana yobuthongo, intwana

sangukumkani; Nesidenge, xa sithe sahlutha sisonka;

<span id="an_30:23">30:23</span> Ngaphantsi kwesigele sentombi, xa sithe
senda; Nesicakakazi xa sithe sadla ilifa lenkosikazi yaso.

<span id="an_30:24">30:24</span> Izinto zine ezincinane emhlabeni, Kanti
zilumke kakhulu:

<span id="an_30:25">30:25</span> Iimbovane ngabantu abangenamandla Kanti
zilungisa ukudla kwazo ehlotyeni;

<span id="an_30:26">30:26</span> Iimbila ngabantu abangomeleleyo, Kanti
zimisa izindlu zazo emaweni;

<span id="an_30:27">30:27</span> Iinkumbi azinakumkani, Kanti ziphuma
zingamaqela ngamaqela zonke;

<span id="an_30:28">30:28</span> Icilitshe unako ukulibamba ngezandla,
Kanti lisemabhotweni ookumkani.

<span id="an_30:29">30:29</span> Izinto zintathu ezihamba ngokunyalasa,
Hayi, zine ezilungisayo ukuhamba:

<span id="an_30:30">30:30</span> Yingonyama, igorha phakathi kwezinto
ezizitho zine, Elingabuyi umva phambi kweento zonke;

<span id="an_30:31">30:31</span> Yinja elingesi nokuba yinkunzi
yebhokhwe; Ngukumkani enomkhosi wakhe.

<span id="an_30:32">30:32</span> Ukuba uthe wenza ngobudenge
ngokuziphakamisa, Nokuba uthe wankqangiyela, Beka isandla emlonyeni.

<span id="an_30:33">30:33</span> Kuba ukucofa ubisi kuvelisa amasi,
Ukucofa impumlo kuvelisa igazi, Ukucofa umsindo kuvelisa imbambano.

<span id="an_31:1">31:1</span> Amazwi kaLemuweli, ukumkani waseMasa,
awamqeqesha ngawo unina.

<span id="an_31:2">31:2</span> Yini, nyana wam? yini, nyana wesizalo
sam? Yini, nyana wezibhambathiso zam?

<span id="an_31:3">31:3</span> Amandla akho musa ukuwanika amankazana,
Iindlela zakho uzinike abo batshabalalisa ookumkani.

<span id="an_31:4">31:4</span> Asiyindawo yokumkani, Lemuweli,
asiyindawo yokumkani ukusela iwayini; Asiyindawo yezidwangube ukuthi,
Siphi na isiselo esinxilisayo?

<span id="an_31:5">31:5</span> Hleze basele bakulibale okumiselweyo,
Balijike ityala loonyana bonke beentsizi.

<span id="an_31:6">31:6</span> Isiselo esinxilisayo sinikeni
abatshabalalayo, Newayini yinikeni abamphefumlo ukrakra.

<span id="an_31:7">31:7</span> Mabasele abo, babulibale ubuhlwempu babo,
Bangabi sabukhumbula ububi babo.

<span id="an_31:8">31:8</span> Wuvule umlomo wakho ngenxa yosisidenge,
Ngenxa yetyala labo bonke abadlulayo.

<span id="an_31:9">31:9</span> Vula umlomo wakho, gweba ngobulungisa,
Ugwebe ityala lolusizana nolihlwempu.

<span id="an_31:10">31:10</span> Umfazi onesidima ngubani na
ongamfumanayo? Lingaphezulu nakwikorale ixabiso lakhe.

<span id="an_31:11">31:11</span> Ikholose ngaye intliziyo yendoda yakhe,
Ayisayi kuswela nto.

<span id="an_31:12">31:12</span> Uyiphatha kakuhle, kungabi kakubi,
Yonke imihla yobomi bakhe.

<span id="an_31:13">31:13</span> Ufuna uboya begusha neflakisi, Asebenze
ethumekelele ngezandla zakhe.

<span id="an_31:14">31:14</span> Unjengeenqanawa zabarhwebi, Ukuya
kuthabatha kude ukudla kwakhe.

<span id="an_31:15">31:15</span> Uvuka kwasebusuku, Ayinike into
edliwayo indlu yakhe, Awunike umsebenzi wawo umthinjana wakhe.

<span id="an_31:16">31:16</span> Unkqangiyela intsimi ayizuze;
Ngeziqhamo zezandla zakhe utyala isidiliya.

<span id="an_31:17">31:17</span> Ubhinqa amandla emanqeni akhe,
Ayomeleze imikhono yakhe.

<span id="an_31:18">31:18</span> Uyakuva ukuba kulungile ukurhweba
kwakhe; Isibane sakhe asicimi ebusuku.

<span id="an_31:19">31:19</span> Uzibeka izandla zakhe entongeni
yokuluka, Neminwe yakhe ibamba isikhanda sento yokuluka.

<span id="an_31:20">31:20</span> Isandla sakhe usitwabululela olusizana,
Azolulele izandla zakhe kumahlwempu.

<span id="an_31:21">31:21</span> Indlu yakhe akayoyikeli, mhlana
ngekhephu, Kuba yonke indlu yakhe yambethe ezisithileyo.

<span id="an_31:22">31:22</span> Uzenzela imiqamelo; Yilinen
ecikizekileyo kwanemfusa isambatho sakhe.

<span id="an_31:23">31:23</span> Iyaziwa indoda yakhe emasangweni,
Ekubutheni kwayo namadoda amakhulu elizwe.

<span id="an_31:24">31:24</span> Wenza iingubo zelinen entle, Athengise
ngazo, athengise ngemibhinqo kumaKanan.

<span id="an_31:25">31:25</span> Ngamandla nobungangamela isambatho
sakhe; Usuka ahleke ngemini ezayo.

<span id="an_31:26">31:26</span> Umlomo wakhe uwuvula ngobulumko;
Umyalelo onenceba uselulwimini lwakhe.

<span id="an_31:27">31:27</span> Uyayondela ihambo yendlu yakhe,
Angasidli isonka sobuvila.

<span id="an_31:28">31:28</span> Bayasuka oonyana bakhe, bathi unoyolo;
Iyasuka indoda yakhe imdumise;

<span id="an_31:29">31:29</span> Ithi, Zininzi iintombi ezenza
ngokunesidima; Ke wena uzidlule zonke ziphela.

<span id="an_31:30">31:30</span> Bubuxoki ubuhle, ngamampunge
ubunzwakazi; Ngumfazi owoyika uYehova yedwa oya kudunyiswa.

<span id="an_31:31">31:31</span> Mnikeni kwiziqhamo zezandla zakhe,
Zimdumise emasangweni izenzo zakhe.
