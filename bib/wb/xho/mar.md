### Xhosa Bible: Mark

<span id="an_1:1">1:1</span> Ukuqaleka kwayo ivangeli kaYesu Kristu,
uNyana kaThixo.

<span id="an_1:2">1:2</span> Njengoko kubhaliweyo ebaprofetini,
kwathiwa, Yabona, mna ndithuma umthunywa wam ngaphambi kobuso bakho, Oya
kuyilungisa indlela yakho phambi kwakho.

<span id="an_1:3">1:3</span> Izwi lodandulukayo entlango, lisithi,
Wulungiseni umendo weNkosi; Zityenenezeni iindlela zayo!

<span id="an_1:4">1:4</span> Kwaye kukho uYohane, ebhaptiza entlango,
evakalisa ubhaptizo lwenguquko, ukuze kuxolelwe izono.

<span id="an_1:5">1:5</span> Laye liphuma lisiya kuye lonke ilizwe
lakwaYuda nabaseYerusalem; baye bonke bebhaptizwa nguye eYordan
emlanjeni, bezixela izono zabo.

<span id="an_1:6">1:6</span> Waye ke kaloku uYohane ambethe uboya
benkamela, enombhinqo wentlonze esinqeni sakhe, abe ke esidla iinkumbi
nobusi basendle.

<span id="an_1:7">1:7</span> Wayememeza esithi, Emva kwam kuza lowo
unamandla kunam, endingafanelekileyo ukuba ndithobe phantsi,
ndiwuthukulule nomtya weembadada zakhe.

<span id="an_1:8">1:8</span> Mna okunene ndanibhaptiza ngamanzi, ke yena
uya kunibhaptiza ngoMoya oyiNgcwele.

<span id="an_1:9">1:9</span> Kwathi ngaloo mihla kweza uYesu, evela
eNazarete yelaseGalili, wabhaptizwa nguYohane eYordan apho.

<span id="an_1:10">1:10</span> Uthe kwaoko, akuba enyuka ephuma
emanzini, walibona izulu likrazulwe, noMoya enjengehobe, esihla phezu
kwakhe.

<span id="an_1:11">1:11</span> Kwabakho nezwi eliphuma ezulwini,
lisithi, Wena unguNyana wam oyintanda, endikholisiweyo nguye.

<span id="an_1:12">1:12</span> Uthi kwaoko uMoya amkhuphele entlango.

<span id="an_1:13">1:13</span> Waye esentlango apho iintsuku ezimashumi
mane, ehendwa nguSathana; waye endawonye namarhamncwa; zaye izithunywa
zezulu zimlungiselela.

<span id="an_1:14">1:14</span> Ke kaloku emveni kokufakwa kukaYohane
entolongweni, weza uYesu kwelaseGalili, evakalisa iindaba ezilungileyo
zobukumkani bukaThixo,

<span id="an_1:15">1:15</span> esithi, Lizalisekile ixesha, nobukumkani
bukaThixo busondele; guqukani, nikholwe kuzo iindaba ezilungileyo ezi.

<span id="an_1:16">1:16</span> Uthe ke, ehamba ngaselwandle lwaseGalili,
wabona uSimon, noAndreya umzalwana wakhe, bephosa umnatha elwandle;
ngokuba babengababambisi beentlanzi.

<span id="an_1:17">1:17</span> Wathi kubo uYesu, Yizani apha
nindilandele; ndonenza nibe ngababambisi babantu.

<span id="an_1:18">1:18</span> Bayishiya kwaoko iminatha yabo,
bamlandela.

<span id="an_1:19">1:19</span> Uhambele phambili kancinane esuka apho,
wabona uYakobi, lo kaZebhedi, noYohane, umzalwana wakhe, besemkhombeni
wabo, beyilungisa iminatha yabo.

<span id="an_1:20">1:20</span> Wababiza kwaoko; bamshiya uyise uZebhedi
emkhombeni enabaqeshwa, besuka bamlandela.

<span id="an_1:21">1:21</span> Bangena eKapernahum. Engene kwaoko
ngesabatha endlwini yesikhungu, wafundisa.

<span id="an_1:22">1:22</span> Bakhwankqiswa kukufundisa kwakhe; ngokuba
ebebafundisa njengonegunya, engafundisi njengabo ababhali.

<span id="an_1:23">1:23</span> Kwaye kukho endlwini yabo yesikhungu
umntu onomoya ongcolileyo; wakhala,

<span id="an_1:24">1:24</span> esithi, Yoo! Yini na enathi nawe, Yesu
waseNazarete? Uze kusitshabalalisa na? Ndiyakwazi wena, lowo unguye;
ungoyiNgcwele kaThixo.

<span id="an_1:25">1:25</span> Wawukhalimela uYesu, esithi, Yithi tu,
uphume kuye.

<span id="an_1:26">1:26</span> Uthe umoya ongcolileyo wambetha wee
xhwenene; wakhala ngezwi elikhulu, waphuma kuye.

<span id="an_1:27">1:27</span> Baqhiphuka umbilini bonke, ngokokude
babuzisane, besithi, Yintoni na le nto? Le mfundiso intsha yiyiphi na,
le nto noomoya abangcolileyo aba abathethela ngegunya, bamlulamele?

<span id="an_1:28">1:28</span> Lwaphuma ke kwaoko udumo lwakhe; lwaya
kuwo wonke ummandla welaseGalili.

<span id="an_1:29">1:29</span> Kwaoko, bakuphuma endlwini yesikhungu,
bangena endlwini kaSimon noAndreya, benoYakobi noYohane.

<span id="an_1:30">1:30</span> Ke kaloku umkhwekazi kaSimon wayelele,
esifa yicesina; bamxelela ngaye kwaoko.

<span id="an_1:31">1:31</span> Weza ke kuye, wamvusa embambe ngesandla;
yamyeka kwaoko icesina, wabalungiselela.

<span id="an_1:32">1:32</span> Ke kaloku, kwakuhlwa, xa litshonayo
ilanga, bazisa kuye bonke ababesifa, nababephethwe ziidemon.

<span id="an_1:33">1:33</span> Waye umzi wonke uhlanganisene ndawonye
emnyango apho.

<span id="an_1:34">1:34</span> Waphilisa babaninzi ababesifa zizifo
ngezifo; wakhupha needemon ezininzi; waye engazivumeli iidemon ukuba
zithethe, kuba bezimazi.

<span id="an_1:35">1:35</span> Uthe kusasa, kusemnyama kakhulu, wavuka,
waphuma waya endaweni eyintlango, wathandaza khona apho.

<span id="an_1:36">1:36</span> Bamphuthuma ooSimon nabo babenaye.

<span id="an_1:37">1:37</span> Bemfumene bathi kuye, Bonke bayakufuna.

<span id="an_1:38">1:38</span> Athi kubo, Masihambe siye emizini
ekufuphi, ukuze nakhona ndivakalise iindaba ezilungileyo, kuba
ndiphumele oko.

<span id="an_1:39">1:39</span> Wayevakalisa ke ezindlwini zabo
zesikhungu kulo lonke elaseGalili, ezikhupha needemon.

<span id="an_1:40">1:40</span> Kweza kuye oneqhenqa, embongoza, esiwa
ngamadolo phambi kwakhe, esithi kuye, Ukuba uyathanda, unako
ukundihlambulula.

<span id="an_1:41">1:41</span> UYesu, esikwe yimfesane ke, wasolula
isandla, wamchukumisa, wathi kuye, Ndiyathanda; hlambuluka.

<span id="an_1:42">1:42</span> Akutsho, lemka kwaoko kuye iqhenqa,
wahlambuluka.

<span id="an_1:43">1:43</span> Akuba emyalile kalukhuni, wamkhupha
kwaoko.

<span id="an_1:44">1:44</span> Athi kuye, Lumka ungaxeleli bani nento
le; hamba uye kuzibonisa kumbingeleli, usondeze ngenxa yokuhlanjululwa
kwakho ezo zinto wamisa zona uMoses, zibe bubungqina kubo.

<span id="an_1:45">1:45</span> Ke yena, akuphuma, waqala ukuyivakalisa
kakhulu nokuyibhengeza loo ndawo, ngokokude angabi saba nako uYesu
ukungena mzini ekuhleni; waba ngaphandle ezindaweni eziziintlango;
babesithi beze kuye bevela ngeenxa zonke.

<span id="an_2:1">2:1</span> Kwakuba ntsuku, wabuya wangena eKapernahum;
kwavakala ukuba usekhaya,

<span id="an_2:2">2:2</span> Kwaoko kwahlanganisana into eninzi yabantu,
ngokokude kungabi sabakho kukhwelelana, nakule mimandla isemnyango;
wayelithetha ke kubo ilizwi.

<span id="an_2:3">2:3</span> Kuthi kufike kuye abantu bezisa onedumbe,
ethwelwe ngabane.

<span id="an_2:4">2:4</span> Bakuba bengenakusondela kuye ngenxa
yendimbane, banqika entungo apho abekhona; bakuba begqobhozile,
baluhlisa ukhuko abelele kulo lowo unedumbe.

<span id="an_2:5">2:5</span> Akulubona ke uYesu ukholo lwabo, wathi
konedumbe, Mntwana wam, uzixolelwe izono zakho.

<span id="an_2:6">2:6</span> Kwaye ke kukho apho ababhali abathile
behleli, becamanga ezintliziyweni zabo, besithi,

<span id="an_2:7">2:7</span> Yini na ukuba lo enjenje ukuthetha
izinyeliso? Ngubani na onako ukuxolela izono, mnye kwaphela, nguThixo?

<span id="an_2:8">2:8</span> Eqiqe kwaoko uYesu ngomoya wakhe, ukuba
benjenjalo ukucamanga ngaphakathi kwabo, wathi kubo, Yini na ukuba
nicamange ezi zinto ezintliziyweni zenu?

<span id="an_2:9">2:9</span> Okhona kulula kukuphi na: kukho ukuthi
konedumbe, Uzixolelwe izono; kukho nokuthi, Vuka, uthabathe ukhuko
lwakho, uhambe?

<span id="an_2:10">2:10</span> Ke ukuze nazi ukuba unalo igunya uNyana
woMntu lokuxolela izono emhlabeni apha (uthi konedumbe),

<span id="an_2:11">2:11</span> Ndithi kuwe, Vuka, uthabathe ukhuko
lwakho, uye endlwini yakho.

<span id="an_2:12">2:12</span> Wavuka kwaoko, waluthabatha ukhuko,
waphuma phambi kwabo bonke, ngokokude bathi nqa bonke, bamzukise uThixo,
besithi, Asizanga sikubone okunje.

<span id="an_2:13">2:13</span> Wabuya waphuma waya ngaselwandle, saye
sonke isihlwele sisiza kuye; wayebafundisa.

<span id="an_2:14">2:14</span> Wathi edlula, wabona uLevi, lo
ka-Alifeyu, ehleli endaweni yerhafu; athi kuye, Ndilandele. Wesuka
wamlandela.

<span id="an_2:15">2:15</span> Kwathi ekuhlaleni kwakhe ngasesithebeni
endlwini yakhe, kwahlala nento eninzi yababuthi berhafu naboni ndawonye
noYesu nabafundi bakhe; kuba babebaninzi, baye ke bemlandela.

<span id="an_2:16">2:16</span> Baye ababhali kwanabaFarisi, bakumbona
esidla ndawonye nababuthi berhafu naboni, besithi kubafundi bakhe. Yini
na ukuba adle asele nababuthi berhafu naboni?

<span id="an_2:17">2:17</span> Ekuvile ke oko uYesu, wathi kubo,
Akufuneki gqirha kwabaphili leyo lifuneka kwabafayo. Andize kubiza
malungisa, ndize kubiza aboni, ukuba baguquke.

<span id="an_2:18">2:18</span> Baye abafundi bakaYohane nababaFarisi
bezila ukudla; beze bathi kuye, Kungani na ukuba abafundi bakaYohane
nababaFarisi bazile ukudla, kanti ke abakho abafundi abakuzili bona?

<span id="an_2:19">2:19</span> Wathi uYesu kubo, Banako yini na abegumbi
lomtshakazi ukuzila ukudla, xa anabo umyeni? Lo gama wonke asekubo
umyeni, abanako ukuzila ukudla.

<span id="an_2:20">2:20</span> Ke kofika mihla awoza asuswe kubo umyeni;
baya kwandula ke ukuzila ngaloo mihla.

<span id="an_2:21">2:21</span> Akukho uthungela isiziba somchako omtsha
engutyeni endala; okanye ke eso sifakiweyo, sitsha, sokrazula kuyo
endala, sibe sibi ngokunye isikroba.

<span id="an_2:22">2:22</span> Kanjalo akukho utha iwayini entsha
ezintsubeni ezindala; okanye iwayini, leyo intsha, yozigqabhuza
iintsuba, iphalale iwayini, zitshabalale neentsuba; yona iwayini entsha
imelwe kukuthiwa ezintsubeni ezintsha.

<span id="an_2:23">2:23</span> Kwathi ehamba ecanda emasimini
ngesabatha, baqala abafundi bakhe ukuhamba besikha izikhwebu.

<span id="an_2:24">2:24</span> Baye besithi kuye abaFarisi, Khangela,
yini na ukuba benze ngemini yesabatha oko kungavumelekileyo?

<span id="an_2:25">2:25</span> Wathi yena kubo; Anizanga niyilese na
into awayenzayo uDavide, oko wayeswele, elambile nokulamba, yena nabo
babenaye?

<span id="an_2:26">2:26</span> Okokuba wangena endlwini kaThixo ngexesha
lika-Abhiyatare umbingeleli omkhulu, wazidla izonka ezibekwe phambi
koThixo, ezo kungavumelekileyo ukuba zidliwe ngabo bangebabingeleli,
wanika nabo babenaye?

<span id="an_2:27">2:27</span> Wayesithi kubo, Isabatha le yabakho
ngenxa yomntu, asingumntu owabakhoyo ngenxa yesabatha.

<span id="an_2:28">2:28</span> Ngoko ke uNyana woMntu lo uyiNkosi yayo
nesabatha.

<span id="an_3:1">3:1</span> Wabuya wangena endlwini yesikhungu. Kwaye
kukho apho umntu onesandla esomileyo.

<span id="an_3:2">3:2</span> Baye bemlalele, ukuba womphilisa na
ngesabatha, ukuze bammangalele.

<span id="an_3:3">3:3</span> Athi kuloo mntu unesandla esomileyo,
Phakama uze phakathi.

<span id="an_3:4">3:4</span> Athi kubo, Okuvumelekileyo ngesabatha
kukwenza okulungileyo, kukwenza okubi, kusini na? Kukusindisa,
kukubulala, kusini na? Ke bahlala bathi tu.

<span id="an_3:5">3:5</span> Akuba ebasingasingile ngengqumbo, ebuhlungu
ngenxa yokuqaqadeka kwentliziyo yabo, uthi kumntu lowo, Solule isandla
sakho. Wasolula, sabuya saphila ke isandla sakhe, saba kwanjengesinye.

<span id="an_3:6">3:6</span> Baphuma abaFarisi, benza kwaoko ibhunga
namaHerode ngaye, lokuba bamtshabalalise.

<span id="an_3:7">3:7</span> Wemka uYesu nabafundi bakhe, waya elwandle;
yamlandela inkitha eninzi, ivela kwelaseGalili, nakwelakwaYuda,

<span id="an_3:8">3:8</span> naseYerusalem, nakwelakwaIdume, naphesheya
kweYordan; nabo bangaseTire nabaseTsidon, inkitha eninzi; bathi, bakuva
izinto ezinkulu abezenza, beza kuye.

<span id="an_3:9">3:9</span> Wathi kubafundi bakhe, umkhonjana mawuhlale
umlindile ngenxa yezihlwele, ukuze bangamxini.

<span id="an_3:10">3:10</span> Kuba waphilisa babaninzi, ngokokude bathi
ngunga phezu kwakhe, ukuze bamchukumise bonke ababenezibetho;

<span id="an_3:11">3:11</span> nabo oomoya abangcolileyo babesithi, xa
sukuba bembona, bawe phantsi kuye, bakhale, besithi, Unguye wena uNyana
kaThixo.

<span id="an_3:12">3:12</span> Wayebakhalimela kakhulu ukuba
bangamtyhili.

<span id="an_3:13">3:13</span> Uthi ke anyuke intaba, abizele kuye abo
abethanda ukubabiza yena ngokwakhe; besuka beza kuye.

<span id="an_3:14">3:14</span> Wadala balishumi elinababini, ukuze babe
naye, nokuze abathume baye kuvakalisa ubukumkani bukaThixo,

<span id="an_3:15">3:15</span> babe negunya lokuziphilisa izifo,
nelokuzikhupha iidemon.

<span id="an_3:16">3:16</span> YanguSimon, awamthiya igama elinguPetros;

<span id="an_3:17">3:17</span> yanguYakobi lo kaZebhedi, noYohane
umzalwana kaYakobi, awathi bona wabathiya amagama okuba ngooBhowanerges,
oko kukuthi, ngabeendudumo;

<span id="an_3:18">3:18</span> yanguAndreya, noFilipu, noBhartolomeyu,
noMateyu, noTomas, noYakobi, lo ka-Alifeyu, noTadeyu, noSimon Kanan,

<span id="an_3:19">3:19</span> noYuda Sikariyoti, owamngcatshayo
nokumngcatsha.

<span id="an_3:20">3:20</span> Bathi beze ekhaya; kubuye kubuthelane
izihlwele, ngokokude bangabi nako nokudla isonka.

<span id="an_3:21">3:21</span> Bevile abakowabo, baphuma beya kumbamba;
kuba babesithi, Uyageza.

<span id="an_3:22">3:22</span> Baye bona ababhali, ababehle bevela
eYerusalem, besithi, UnoBhelezebhule; besithi, Iidemon ezi uzikhupha
ngaye umphathi weedemon.

<span id="an_3:23">3:23</span> Uthe ebabizele kuye, wathetha nabo
ngemizekeliso, esithi, Angathini na uSathana ukukhupha uSathana?

<span id="an_3:24">3:24</span> Nobukumkani, ukuba buthe bahlulelana
bodwa, abunako ukuma obo bukumkani;

<span id="an_3:25">3:25</span> nendlu, ukuba ithe yahlulelana yodwa,
ayinako ukuma loo ndlu;

<span id="an_3:26">3:26</span> noSathana, ukuba uthe wazichasa
ngokwakhe, wahlulelana yedwa, akanako ukuma, uphelile.

<span id="an_3:27">3:27</span> Akukho unako ukuthi impahla yegorha
angene endlwini yalo ayiphange, ukuba akathanga tanci alibophe igorha
elo, andule ukuyiphanga indlu yalo.

<span id="an_3:28">3:28</span> Inene ndithi kuni, Zonke izono baya
kuzixolelwa oonyana babantu, kwanezinyeliso abasukuba benyelisa ngazo;

<span id="an_3:29">3:29</span> ke yena othe wanyelisa uMoya oyiNgcwele,
akanako ukuxolelwa naphakade; unetyala lokugwetywa okungunaphakade.

<span id="an_3:30">3:30</span> Watsho kuba besithi, Unomoya ongcolileyo.

<span id="an_3:31">3:31</span> Bafika ke kaloku abazalwana bakhe nonina;
bathi, bemi ngaphandle, bathumela kuye bembiza.

<span id="an_3:32">3:32</span> Zaye izihlwele zihleli zimphahlile; bathi
kuye, Nanku unyoko nabazalwana bakho ngaphandle, bekufuna.

<span id="an_3:33">3:33</span> Wabaphendula esithi, Ngubani na uma?
Ngoobani na nabazalwana bam?

<span id="an_3:34">3:34</span> Akuba ebasingasingile ngeenxa zonke
ababehleli bemphahlile, uthi, Nanku uma nabazalwana bam.

<span id="an_3:35">3:35</span> Kuba othe wakwenza ukuthanda kukaThixo,
lowo ngumzalwana wam, nodade wethu, noma.

<span id="an_4:1">4:1</span> Wabuya waqala ukufundisa ngaselwandle.
Kwahlanganisana kuye isihlwele esikhulu, ngokokude aye kungena
emkhombeni, ahlale elwandle yena; sabe sonke isihlwele sisemhlabeni,
sibheke elwandle apho.

<span id="an_4:2">4:2</span> Wayebafundisa iindawo ezininzi
ngemizekeliso. Ubesithi kubo ekubafundiseni kwakhe,

<span id="an_4:3">4:3</span> Phulaphulani; yabonani, umhlwayeli waphuma
waya kuhlwayela;

<span id="an_4:4">4:4</span> kwathi ekuhlwayeleni kwakhe, inxalenye
yembewu yawa ngasendleleni, zeza iintaka zezulu zayidla zayigqiba.

<span id="an_4:5">4:5</span> Enye yawa endaweni elulwalwa, apho
ibingenamhlaba mninzi; yantshula ke kwaoko, ngenxa yokuba
ibingenabunzulu bamhlaba;

<span id="an_4:6">4:6</span> ithe ke, lakuphuma ilanga, yatsha, yathi
ngenxa yokuba ingenangcambu, yoma kwaphela.

<span id="an_4:7">4:7</span> Yimbi yawa emithaneni enameva; yakhula
imithana enameva yayiminxa; ayanika siqhamo.

<span id="an_4:8">4:8</span> Yimbi yawa kuwo wona umhlaba omhle, yanika
isiqhamo esakhulayo sachuma; yathwala amashumi amathathu enye, namashumi
amathandathu enye, nekhulu enye.

<span id="an_4:9">4:9</span> Wathi kubo, Lowo uneendlebe zokuva makeve.

<span id="an_4:10">4:10</span> Ke kaloku, xa abeyedwa, abo
babemphahlile, ndawonye nabalishumi elinababini, bambuza umzekeliso
lowo.

<span id="an_4:11">4:11</span> Wathi kubo, Nikuphiwe nina ukuyazi
imfihlelo yobukumkani bukaThixo; ke kwabo bangaphandle zonke izinto ezi
zibakho ngemizekeliso;

<span id="an_4:12">4:12</span> ukuze bebona nje, babone bangaqiqi;
besiva nje, beve bangaqondi; hleze babuye, zebaxolelwe izono zabo.

<span id="an_4:13">4:13</span> Athi kubo, Aniwazi na lo mzekeliso?
Nothini na ke ukuyazi imizekeliso yonke?

<span id="an_4:14">4:14</span> Umhlwayeli lo uhlwayela ilizwi.

<span id="an_4:15">4:15</span> Abo ke bangasendleni, apho ilizwi
