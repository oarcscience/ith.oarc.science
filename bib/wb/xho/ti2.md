### Xhosa Bible: 2 Timothy

<span id="an_1:1">1:1</span> UPawulos, umpostile kaYesu Kristu
ngokuthanda kukaThixo, ngokwedinga lobomi obukuKristu Yesu,

<span id="an_1:2">1:2</span> ubhalela uTimoti, umntwana wakhe oyintanda;
uthi, Makube lubabalo, nenceba, noxolo oluvela kuThixo uYise, noKristu
Yesu iNkosi yethu.

<span id="an_1:3">1:3</span> Ndinokubulela kuye uThixo, endamkhonzayo
kwakoomawokhulu ndinesazela esihlambulukileyo, ndikukhumbula
njengokungapheliyo, ekukhungeni kwam ubusuku nemini;

<span id="an_1:4">1:4</span> ndilangazelela ukunga ndingakubona,
ndizikhumbula nje iinyembezi zakho, ukuze ndizaliswe luvuyo;

<span id="an_1:5">1:5</span> ndikhunjuzwa nje ukholo olukuwe
olungenakuhanahanisa, lona lwahlalayo kuqala kunyokokhulu uLoyisi,
nakunyoko uYunike, endinyanisileyo ke ukuthi luhleli nakuwe.

<span id="an_1:6">1:6</span> Endithi ngenxa yelo thuba ndikukhumbuze,
ukuba usikhwezele isipho sokubabalwa kwakho nguThixo, esikhoyo kuwe
ngokubekwa kwezandla zam phezu kwakho.

<span id="an_1:7">1:7</span> Kuba uThixo akasinikanga moya wabugwala;
wasinika owamandla, nowothando, nowesidima.

<span id="an_1:8">1:8</span> Ngoko ke musa ukuba nazintloni ngabo
ubungqina beNkosi yethu, nangam, mna mbanjwa wayo; suka uve ububi kunye
nam ngenxa yeendaba ezilungileyo, ngokwamandla kaThixo;

<span id="an_1:9">1:9</span> owasisindisayo, wasibiza ngobizo
olungcwele, engenzi ngokwemisebenzi yethu, esenza ngokweyakhe ingqibo,
nobabalo, esababalwa ngalo ngoKristu Yesu phambi kwamaxesha aphakade;

<span id="an_1:10">1:10</span> lwabonakaliswa ke ngoku ngako ukubonakala
koMsindisi wethu uYesu Kristu, owakubhangisayo okunene ukufa, kodwa
wabukhanyisa ubomi nokungonakali, ngazo iindaba ezilungileyo ezi;

<span id="an_1:11">1:11</span> endamiselwa zona mna, ndaba ngumvakalisi
nompostile nomfundisi weentlanga;

<span id="an_1:12">1:12</span> endithi kananjalo ngenxa yeli thuba ndive
obu bunzima. Noko ke andinazintloni; kuba ndiyamazi endikholiweyo nguye;
kanjalo ndithi, ndeyisekile ukuba unako ukuyigcina into
endimphathisileyo, ese kuloo mini.

<span id="an_1:13">1:13</span> Umzekelo wamazwi aphilileyo owawevayo
kum, wubambe elukholweni naseluthandweni olukuKristu Yesu.

<span id="an_1:14">1:14</span> Loo nto intle uyiphathisiweyo, yigcine
ngoMoya oyiNgcwele ohleliyo ngaphakathi kwethu.

<span id="an_1:15">1:15</span> Oku uyakwazi, ukuba bandinikele umva
bonke abaseAsiya; abanoFigelo noHermogene.

<span id="an_1:16">1:16</span> Yanga iNkosi ingayenzela inceba indlu
kaOnesiforo; ngokuba kwaba futhi endiphumza, akaba nazintloni nguwo
umxokelelwane wam.

<span id="an_1:17">1:17</span> Wathi akuba eseRoma, wandifuna
ngokukhuthala okukhulu, wandifumana.

<span id="an_1:18">1:18</span> Yanga iNkosi ingamfumanisa inceba
eNkosini ngaloo mini. Nguwena waziyo kakuhle oko zingako izinto
awalungiselela ngazo e-Efese.

<span id="an_2:1">2:1</span> Wena ngoko, mntwana wam, yomelela
elubabalweni olukuKristu Yesu;

<span id="an_2:2">2:2</span> uthi nezinto owazivayo kum ngamangqina
amaninzi, ezo uzibeke phambi kwabantu abathembekileyo, bona baya
kukulingana ukufundisa nabanye.

<span id="an_2:3">2:3</span> Wena ngoko bunyamezele ububi,
njengomphumi-mkhosi olungileyo kaKristu Yesu.

<span id="an_2:4">2:4</span> Akukho namnye uthi, ephuma umkhosi,
azibhijele ngemicimbi yobu bomi; ukuze amkholise lowo umnyulele ukuphuma
umkhosi.

<span id="an_2:5">2:5</span> Nakuba ke ubani esithi azame emdlalweni,
akathweswa sithsaba, ukuba akathanga azame ngokomthetho.

<span id="an_2:6">2:6</span> Umlimi lowo ubulalekayo, umelwe kukwamkela
kuqala kwiziqhamo.

<span id="an_2:7">2:7</span> Ziqiqe izinto endizithethayo; iNkosi
yokupha ke ingqondo ezintweni zonke.

<span id="an_2:8">2:8</span> Mkhumbule uYesu Kristu, engovukileyo
kwabafileyo, engowembewu kaDavide, ngokweendaba zam ezilungileyo;

<span id="an_2:9">2:9</span> endiva ububi ngenxa yazo, kude kuse
ezintanjeni, njengomenzi wobubi; ke lona ilizwi likaThixo alibotshiwe.

<span id="an_2:10">2:10</span> Ngenxa yoko izinto zonke ndiyazinyamezela
ngenxa yabo abanyulwa, ukuze nabo bamkele usindiso olukuKristu Yesu,
ndawonye nozuko olungunaphakade.

<span id="an_2:11">2:11</span> Lithembekile ilizwi eli: Kuba xa safa
naye, sodla ubomi kwandawonye naye;

<span id="an_2:12">2:12</span> ukuba siyanyamezela, solawula kwandawonye
naye; ukuba siyamkhanyela, naye wosikhanyela;

<span id="an_2:13">2:13</span> ukuba asithembekile, yena uhlala
ethembekile; akanako ukuzikhanyeza.

<span id="an_2:14">2:14</span> Bakhumbuze ezo zinto, uqononondise
emehlweni eNkosi, ukuba kungenziwa mfazwe yamazwi angancedi lutho, kuze
kubhukuqeke abaphulaphuli.

<span id="an_2:15">2:15</span> Kukhuthalele ukuzimisa phambi koThixo
ucikidekile, ungumsebenzi ongenazintloni, ulungisa ukulahlula ilizwi
lenyaniso.

<span id="an_2:16">2:16</span> Ke zona iintetho ezingento,
ezihlambelayo, zinxweme; kuba ziya kuhambela phambili kokukhona
kungahloneli Thixo.

<span id="an_2:17">2:17</span> Nelizwi labo liya kudla njengesilonda
esiqhenqethayo; abanoHimenayo noFileto,

<span id="an_2:18">2:18</span> abathe bona bayiphosa inyaniso, besithi
uvuko seluthe lwakho, kananjalo belubhukuqa ukholo lwenxenye.

<span id="an_2:19">2:19</span> Noko ke limi, lithe nkqi, ilitye lesiseko
likaThixo, linambhalo uthi, INkosi iyabazi abangabayo; nothi, Mabasuke
kuko ukungalungisi, bonke abalibizayo igama likaKristu.

<span id="an_2:20">2:20</span> Ke endlwini enkulu akubikho zitya
zagolide nazasilivere zodwa; kubakho nezomthi nezomdongwe. Inxenye ke
zezembeko, inxenye azizambeko.

<span id="an_2:21">2:21</span> Ukuba ngoko umntu uthe wazikhuphulula
kwezo zinto, woba sisitya sembeko, singcwalisiwe, simlungele kakuhle
umninindlu, silungiselwe wonke umsebenzi olungileyo.

<span id="an_2:22">2:22</span> Ke zibaleke iinkanuko zobutsha; phuthuma
ke ubulungisa, ukholo, uthando, uxolo nabo bayinqulayo iNkosi
ngokwasentliziyweni ehlambulukileyo.

<span id="an_2:23">2:23</span> Yinxweme imibuzwana yobudenge
engenangqeqesho, usazi ukuba izala imfazwe.

<span id="an_2:24">2:24</span> Ke umkhonzi weNkosi akamele kulwa; umele
ukuba nokuthantamisa kubantu bonke, enokufundisa, enyamezela ububi,

<span id="an_2:25">2:25</span> ebaqeqesha ngobulali abo baphikayo; ukuba
akangethi na kanti uThixo abaphe inguquko esa ekuyazini inyaniso,

<span id="an_2:26">2:26</span> babuye baqabuke, baphume esibatheni sakhe
uMtyholi, ababanjiselwe ukuthanda kwakhe nguye.

<span id="an_3:1">3:1</span> Ke, kwazi oku, ukuba ngemihla yokugqibela
kuya kufika amaxesha anomngcipheko.

<span id="an_3:2">3:2</span> Kuba abantu baya kuba ngabazithandayo,
abathandi bemali, abaqhankqalazi, abakratshi, abanyelisi, abangeva
bazali,

<span id="an_3:3">3:3</span> abangabuleliyo, abangengcwele,
abangenabubele, abangenatarhu, abatyholi, abangenakuzeyisa, izibhoja,
abangenakuthanda kulunga,

<span id="an_3:4">3:4</span> abancethezi, izityhuthutyhuthu, abathiwe
lwale likratshi, abathandi beziyolo kunokuba bathande uThixo;

<span id="an_3:5">3:5</span> benokumila kokuhlonela uThixo, ukanti
bayawakhanyela amandla ako. Kwabo ke khwebuka.

<span id="an_3:6">3:6</span> Kuba baphuma kwaba, abo banyebelezela
ezindlwini, bathimbe abafazazana abazono ziziimfumba, bephethwe
ziinkanuko ngeenkanuko,

<span id="an_3:7">3:7</span> behlala befunda, bangaze phofu babe nako
ukufika ekuyazini inyaniso.

<span id="an_3:8">3:8</span> Kwangolo hlobo ke, bathi ooYanes noYambres
bamchasa uMoses, ngokunjalo nabo aba bayayichasa inyaniso. Bangabantu
abonakele iingqiqo, abangenakucikideka ngakulo ukholo.

<span id="an_3:9">3:9</span> Abayi kuhambela phambili ngakumbi; kuba
ukungabi nangqiqo kwabo kuya kubonakala kubantu bonke, njengokuba kwaba
njalo okwabaya.

<span id="an_3:10">3:10</span> Ke wena uyilandile eyam imfundiso,
ihambo, ingqibo, ukholo, ukuzeka kade umsindo, uthando, unyamezelo,

<span id="an_3:11">3:11</span> iintshutshiso, iintlungu ezinjengezo
zandihlelayo kwa-Antiyokwe, eIkoniyo, eListra; ukutshutshiswa
okunjengoko ndakuthwalayo, yaza iNkosi yandihlangula kuko konke.

<span id="an_3:12">3:12</span> Nabo bonke ke abathanda ukuhlala
ngokuhlonela uThixo kuKristu Yesu baya kutshutshiswa.

<span id="an_3:13">3:13</span> Ke bona abantu abangendawo noosiyazi,
kukhona baya kuhambela phambili ebubini, belahlekisa, belahlekiswa.

<span id="an_3:14">3:14</span> Ke wena yima ezintweni owazifundayo,
waqiniselwa kuzo, umazi owazifunda kuye;

<span id="an_3:15">3:15</span> nokuba uzaze kwasebuntwaneni izibhalo
ezingcwele, ezinako ukukulumkisela elusindisweni ngokukholwa kuKristu
Yesu.

<span id="an_3:16">3:16</span> Sonke isibhalo, siphefumlelwe nguThixo
nje, sikwancedela ukufundisa, ukohlwaya, ukululeka, ukuqeqesha
okusebulungiseni,

<span id="an_3:17">3:17</span> ukuze umntu kaThixo afaneleke, exhobele
wonke umsebenzi olungileyo.

<span id="an_4:1">4:1</span> Ndiyaqononondisa mna ngoko emehlweni
kaThixo, neNkosi yethu uYesu Kristu, oza kugweba abaphilileyo
nabafileyo, kwanangokubonakala kwakhe nobukumkani bakhe:

<span id="an_4:2">4:2</span> livakalise ilizwi; zingisa ukwenjenjalo
ngamathuba alungileyo naxakileyo; yohlwaya, khalimela, vuselela, unako
konke ukuzeka kade umsindo, nokufundisa.

<span id="an_4:3">4:3</span> Kuba kuya kubakho ixesha abangayi
kuyinyamezela imfundiso ephilileyo; baya kusuka ngokweenkanuko zabo
bazifumbele abafundisi, bebaba iindlebe;

<span id="an_4:4">4:4</span> babeya kuzisonga zimke iindlebe kuyo
inyaniso, baphambukele ke kuzo iintsomi.

<span id="an_4:5">4:5</span> Ke wena yiba yingcathu ezintweni zonke,
nyamezela ububi, umsebenzi womshumayeli weendaba ezilungileyo wenze,
buzalise ubulungiseleli bakho.

<span id="an_4:6">4:6</span> Kuba mna sendithululwa ngokomnikelo; nalo
ixesha lokunduluka kwam likufuphi.

<span id="an_4:7">4:7</span> Umzamo omhle ndiwuzamile, ugqatso
ndilufezile, ukholo ndilugcinile.

<span id="an_4:8">4:8</span> Okokugqibela ndibekelwe isithsaba
sobulungisa, eya kundibuyisela ngaso ngaloo mini iNkosi, umgwebi
olilungisa; ingabi ndim ndedwa ke, ibe ngabo bonke abakuthandayo
ukubonakala kwayo.

<span id="an_4:9">4:9</span> Khawuleza uze kum kamsinya;

<span id="an_4:10">4:10</span> kuba uDemasi wandishiya ngokuthanda
iphakade langoku, waya eTesalonika; uKreske uye kwelaseGalati; uTito uye
kwelaseDalmati.

<span id="an_4:11">4:11</span> NguLuka yedwa onam. Thabatha uMarko uze
naye; kuba unokundinceda kakuhle kumsebenzi wokulungiselela.

<span id="an_4:12">4:12</span> Ke uTikiko ndamthumela e-Efese.

<span id="an_4:13">4:13</span> Ingubo endayishiya eTrowa kuKarpo, uze
uyiphathe ekuzeni kwakho, neencwadi ezo, ngokukodwa ezezintsu.

<span id="an_4:14">4:14</span> UAlesandire, umkhandi wobhedu, wandenza
izinto ezininzi ezimbi; iNkosi iya kumbuyekeza ngokwemisebenzi yakhe.

<span id="an_4:15">4:15</span> Mlumkele nawe; kuba wawachasa kunene
amazwi ethu.

<span id="an_4:16">4:16</span> Kokokuqala ukuziphendulela kwam,
akubangakho namnye unam, basuka bonke bandishiya; kwanga akungebalelwe
kubo oko.

<span id="an_4:17">4:17</span> Noko ke iNkosi yandinceda, yandomeleza;
ukuze ngam uvakaliso lweendaba ezilungileyo luzaliseke, neentlanga zonke
zive; ndaza ndahlangulwa emlonyeni wengonyama.

<span id="an_4:18">4:18</span> Isaya kundihlangula yona iNkosi kuwo
wonke umsebenzi ongendawo, indisindisele ebukumkanini bayo
obusemazulwini; kuyo makubekho uzuko, kuse emaphakadeni asemaphakadeni.
Amen.

<span id="an_4:19">4:19</span> Bulisa kuPriska noAkwila, nakwindlu
kaOnesiforo.

<span id="an_4:20">4:20</span> UErasto wasala eKorinte. Ke uTrofimo
ndamshiya eMileto esifa.

<span id="an_4:21">4:21</span> Khawuleza uze, kungekabi sebusika. Uya
kubulisa uYubhulo, noPudens, noLino, noKlawudiya, nabo abazalwana bonke.

<span id="an_4:22">4:22</span> INkosi uYesu Kristu mayibe nomoya wakho.
Ubabalo malube nani. Amen.
