### Georgian Bible: Mark

<span id="an_1:1">1:1</span> დასაბამი სახარებისაჲ იესუ ქრისტესი, ძისა
ღმრთისაჲ.

<span id="an_1:2">1:2</span> ვითარცა წერილ არს ესაია წინაწარმეტყუელსა:
აჰა ესერა მე წარვავლინო ანგელოზი ჩემი წინაშე პირსა შენსა, რომელმან
განჰმზადნეს გზანი შენნი წინაშე შენსა.

<span id="an_1:3">1:3</span> ჴმაჲ ღაღადებისაჲ უდაბნოსა: განჰმზადენით
გზანი უფლისანი და წრფელ ყვენით ალაგნი მისნი.

<span id="an_1:4">1:4</span> იყო იოვანე უდაბნოსა და ნათელ-სცემდა და
ქადაგებდა ნათლის-ცემასა სინანულისასა მისატევებელად ცოდვათა.

<span id="an_1:5">1:5</span> და განვიდოდა მისა ყოველი სოფლები
ჰურიასტანისაჲ და იერუსალჱმელნი და ნათელს-იღებდეს მისგან ყოველნი
იორდანესა მდინარესა და აღუვარებდეს ცოდვათა მათთა.

<span id="an_1:6">1:6</span> ხოლო ემოსა იოვანეს სამოსლად თმისაგან
აქლემისა, და სარტყელი ტყავისაჲ წელთა მისთა, და ჭამდა იგი მკალსა და
თაფლსა ველურსა,

<span id="an_1:7">1:7</span> ქადაგებდა და იტყოდა: ესერა მოვალს უძლიერესი
ჩემსა შემდგომად ჩემსა, რომლისა ვერ ღირს ვარ განჴსნად საბელსა ჴამლთა
მისთასა.

<span id="an_1:8">1:8</span> მე ნათელ-გცემ თქუენ წყლითა, ხოლო მან
ნათელ-გცეს თქუენ სულითა წმიდითა.

<span id="an_1:9">1:9</span> და იყო მათ დღეთა შინა მოვიდა იესუ ნაზარეთით
გალილეაჲსაჲთ და ნათელ-იღო იოვანესგან იორდანესა.

<span id="an_1:10">1:10</span> და მეყსეულად აღმოსლვასა მისსა მიერ წყლით
იხილნა ცანი განხმულნი და სული ღმრთისაჲ გარდამომავალი, ვითარცა ტრედი, მის
ზედა.

<span id="an_1:11">1:11</span> და ჴმაჲ იყო ზეცით: შენ ხარ ძე ჩემი
საყუარელი, შენ სათნო-გიყავ.

<span id="an_1:12">1:12</span> და მეყსეულად სულმან განიყვანა იგი
უდაბნოდ.

<span id="an_1:13">1:13</span> და იყო მუნ ორმეოც დღე და გამოიცადებოდა
ეშმაკისაგან და იყო მჴეცთა თანა, და ანგელოზნი ჰმსახურებდეს მას.

<span id="an_1:14">1:14</span> და შემდგომად მიცემისა იოვანესა მოვიდა
იესუ გალილეად, ქადაგებდა სახარებასა სასუფეველისა ღმრთისასა

<span id="an_1:15">1:15</span> და იტყოდა, ვითარმედ: აღსრულებულ არს ჟამი
და მოახლებულ არს სასუფეველი ღმრთისაჲ; შეინანეთ და გრწმენინ სახარებისაჲ.

<span id="an_1:16">1:16</span> ხოლო ვიდოდა რაჲ ზღჳს-კიდესა მას
გალილეაჲსასა, იხილნა სიმონ და ანდრეა, ძმაჲ მისი, ითხევლიდეს რაჲ ზღუასა
მას ზედა, რამეთუ იყვნეს იგინი მესათხევლე.

<span id="an_1:17">1:17</span> და ჰრქუა მათ იესუ: მოვედით და შემომიდეგით
მე, და გყვნე თქუენ მესათხევლე კაცთა.

<span id="an_1:18">1:18</span> ხოლო მათ მეყსეულად დაუტევნეს ბადენი მათნი
და მისდევდეს მას.

<span id="an_1:19">1:19</span> და მცირედ წარვიდა და იხილა იაკობ ზებედესი
და იოვანე, ძმაჲ მისი, და იგინი განაგებდეს ბადეთა ნავსა შინა.

<span id="an_1:20">1:20</span> და უწოდა მათ. ხოლო მათ მეყსეულად დაუტევეს
ზებედე, მამაჲ მათი, ნავსა შინა მუშაკთა მისთა თანა და მისდევდეს მას.

<span id="an_1:21">1:21</span> და შევიდეს კაპერნაუმდ. და მეყსეულად დღესა
შაბათსა შევიდა შესაკრებელსა მათსა და ასწავებდა მათ.

<span id="an_1:22">1:22</span> და განუკჳრდებოდა ერსა მას მოძღურებაჲ იგი
მისი, რამეთუ ასწავებდა მათ, ვითარცა-იგი ვის აქუნ ჴელმწიფებაჲ, და არა
ვითარ-იგი მწიგნობარნი მათნი.

<span id="an_1:23">1:23</span> და იყო კრებულსა მას შინა მათსა კაცი,
რომლისა თანა იყო სული არაწმიდაჲ, ღაღატ-ყო

<span id="an_1:24">1:24</span> და თქუა: ეჰა, რაჲ არს ჩუენი და შენი, იესუ
ნაზარეველო? მოსრულ ხარ წარწყმედად ჩუენდა. გიცით შენ, ვინ ხარ, წმიდაჲ ეგე
ღმრთისაჲ.

<span id="an_1:25">1:25</span> შეჰრისხნა მას იესუ და ჰრქუა: პირი დაიყავ
და განვედ მაგისგან!

<span id="an_1:26">1:26</span> და დასცა იგი სულმან მან არაწმიდამან, და
ჴმა-ყო ჴმითა დიდითა და განვიდა მისგან.

<span id="an_1:27">1:27</span> და დაუკჳრდებოდა ყოველთა, ვიდრე
გამოძიებადმდე ურთიერთას, და იტყოდეს: რაჲ-მე არს მოძღურებაჲ ესე ახალი,
რამეთუ ჴელმწიფებით სულთაცა არაწმიდათა უბრძანებს, და ერჩიან მას?

<span id="an_1:28">1:28</span> და განჴდა ჰამბავი მისი ყოველსა სოფლებსა
გალილეაჲსასა.

<span id="an_1:29">1:29</span> და მიერ შესაკრებელით გამოვიდა და მოვიდა
სახლსა სიმონისსა და ანდრეაჲსსა იოვანეს და იაკობის თანა.

<span id="an_1:30">1:30</span> ხოლო სიდედრი სიმონისი დავრდომილ იყო
მჴურვალებითა. და უთხრეს იესუს მისთჳს.

<span id="an_1:31">1:31</span> და მოვიდა და უპყრა ჴელი მისი და აღადგინა
იგი; და მეყსეულად დაუტევა იგი მჴურვალებამან მან, და აღდგა და ჰმსახურებდა
მათ.

<span id="an_1:32">1:32</span> და ვითარ შემწუხრდებოდა, და მზე დაჰვიდოდა,
მოჰყვანდეს მისა ყოველნი სნეულნი და ეშმაკეულნი

<span id="an_1:33">1:33</span> და ყოველი ქალაქი შეკრებულ იყო წინაშე
კართა.

<span id="an_1:34">1:34</span> და განკურნნა ყოველნი ბოროტთაგან სენთა და
თითო-სახეთაგან სნეულებათა და ეშმაკნი მრავალნი განასხნა და არა უტევებდა
სიტყუად ეშმაკთა მათ, რამეთუ იცოდეს იგი, ვითარმედ ქრისტე არს.

<span id="an_1:35">1:35</span> და განთიად რიჟურაჟუს ოდენ აღდგა და
გამოვიდა მიერ და განვიდა უდაბნოსა ადგილსა და მუნ ილოცვიდა.

<span id="an_1:36">1:36</span> და მისდევდეს მას სიმონ და მისთანანი.

<span id="an_1:37">1:37</span> და პოვეს იგი და ჰრქუეს მას, ვითარმედ:
ყოველნი გეძიებენ შენ.

<span id="an_1:38">1:38</span> ხოლო იესუ ჰრქუა მათ: მოვედით და წარვიდეთ
მახლობელსა დაბნებსა და ქალაქებსა, რაჲთა მუნ უქადაგოთ, რამეთუ ამისთჳს
მოვივლინე.

<span id="an_1:39">1:39</span> და ქადაგებდა შესაკრებელთა მათთა და
ყოველსა გალილეასა და ეშმაკთა განასხმიდა.

<span id="an_1:40">1:40</span> მაშინ მოუჴდა მას ერთი განკეთროვნებული,
ევედრებოდა მას და მუჴლნი დაიდგინა და ეტყოდა: უფალო, უკუეთუ გინდეს,
ძალ-გიც განწმედად ჩემდა.

<span id="an_1:41">1:41</span> ხოლო იესუს შეეწყალა იგი, განყო ჴელი,
შეახო მას და ჰრქუა: მნებავს, განწმიდენი!

<span id="an_1:42">1:42</span> და ვითარცა ესე თქუა, მეყსეულად განეყარა
მას კეთროვნებაჲ იგი, და განწმიდნა იგი

<span id="an_1:43">1:43</span> და შეჰრისხნა მას და მეყსეულად გამოაძო იგი

<span id="an_1:44">1:44</span> და ჰრქუა მას: იხილე, ნუვის უთხრობ, არამედ
მივედ და უჩუენე თავი შენი მღდელსა და შეწირე განწმედისა შენისათჳს,
რომელი-იგი ბრძანა მოსე საწამებელად მათა.

<span id="an_1:45">1:45</span> ხოლო იგი გამოვიდა მიერ და იწყო ქადაგებად
ფრიად და მიმოდადებად სიტყჳსა მის, ვითარმედ ვერღარა ჴელ-ეწიფა მას ცხადად
ქალაქად შესლვად, არამედ უდაბნოთა ადგილთა იყოფვოდა. და მოვიდოდეს მისა
ყოვლით კერძო.

<span id="an_2:1">2:1</span> და შევიდა კუალად კაპერნაუმდ შემდგომად
მცირედთა დღეთა; და ესმა, რამეთუ სახლსა შინა არს.

<span id="an_2:2">2:2</span> და მეყსეულად შეკრბა მუნ სიმრავლე ერისაჲ,
ვიდრემდე ვერღარა იტევდა წინაშე კართა მათ, და ეტყოდა მათ სიტუასა.

<span id="an_2:3">2:3</span> და მოვიდეს მისა და მოაქუნდა განრღუეული
ზე-კიდებული ოთხთაგან.

<span id="an_2:4">2:4</span> და ვითარ ვერ შეუძლეს მიახლებად მისა ერისა
მისგან, აღსძარცუეს სართული სახლისაჲ მის, სადა-იგი იყო, და აღიღეს და
გარდაუტევეს ცხედარი იგი, რომელსა ზედა იდვა განრღუეული იგი.

<span id="an_2:5">2:5</span> და იხილა იესუ სარწმუნოებაჲ იგი მათი და
ჰრქუა განრღუეულსა მას: შვილო, მიგეტევნენ შენ ცოდვანი შენნი!

<span id="an_2:6">2:6</span> ხოლო იყვნეს ვინმე მუნ მსხდომარენი
მწიგნობართაგანნი და ზრახვიდეს გულთა შინა მათთა და იტყოდეს,

<span id="an_2:7">2:7</span> ვითარმედ: ესე გმობასა იტყჳს; ვის
ჴელ-ეწიფების მიტევებად ცოდვათა, გარნა მხოლოსა ღმერთსა?

<span id="an_2:8">2:8</span> ხოლო იესუ მეყსეულად გულისჴმა-ყო სულითა,
რამეთუ ესრეთ ზრახვენ გულთა შინა მათთა, და ჰრქუა მათ: რასა მაგას ჰზრახავთ
გულთა შინა თქუენთა?

<span id="an_2:9">2:9</span> რაჲ უადვილეს არს სიტყუად განრღუეულისა ამის:
მიგეტევნენ ცოდვანი შენნი, ანუ თქუმად: აღდეგ, აღიღე ცხედარი შენი და
ვიდოდე?

<span id="an_2:10">2:10</span> ხოლო რაჲთა უწყოდით, რამეთუ ჴელმწიფებაჲ
აქუს ძესა კაცისასა ქუეყანასა ზედა მიტევებად ცოდვათა, და ჰრქუა განღუეულსა
მას:

<span id="an_2:11">2:11</span> შენ გეტყჳ: აღდეგ და აღიღე ცხედარი შენი და
წარვედ სახიდ შენდა!

<span id="an_2:12">2:12</span> და მეყსეულად აღდგა იგი და აღიღო ცხედარი
