### Georgian Bible: James

<span id="an_1:1">1:1</span> იაკობ, ღმრთისა და უფლისა იესუ ქრისტეს
მონაჲ, ათორმეტთა ნათესავთა, - გიხაროდენ!

<span id="an_1:2">1:2</span> ყოველივე სიხარულად შეჰრაცხეთ, ძმანო ჩემნო,
რაჟამს განსაცდელსა შესცჳვეთ პირად-პირადსა;

<span id="an_1:3">1:3</span> უწყოდეთ, რამეთუ გამოცდილებამან მან
სარწმუნოებისა თქუენისამან შექმნის მოთმინებაჲ,

<span id="an_1:4">1:4</span> ხოლო მოთმინებასა მას საქმე სრული აქუნდინ,
რაჲთა იყვნეთ სრულ და ყოვლად ცოცხალ და არარაჲთ ნაკლულევან.

<span id="an_1:5">1:5</span> ხოლო უკუეთუ ვინმე თქუენგანი ნაკლულევან არს
სულიერითა სიბრძნითა, ითხოვენ ღმრთისაგან, რომელმან-იგი მოსცის ყოველთა
უხუად და არავის აყუედრის, და მოსცეს მას.

<span id="an_1:6">1:6</span> ითხოვდინ ხოლო სარწმუნოებით და ნუ ორგულებით,
რამეთუ რომელი ორგულებდეს, მსგავს არს იგი ღელვასა ზღჳსასა, ქარითა
აღძრულსა და მიმოტაცებულსა.

<span id="an_1:7">1:7</span> ნუ ჰგონებნ კაცი იგი, ვითარმედ მო-რაჲმე-იღოს
ღმრთისაგან.

<span id="an_1:8">1:8</span> კაცი ორგული დაუდგრომელ არს ყოველთა გზათა
მისთა.

<span id="an_1:9">1:9</span> იქადოდენ ძმაჲ იგი მდაბალი ამაღლებასა თჳსსა,

<span id="an_1:10">1:10</span> ხოლო მდიდარი - დამდაბლებასა თჳსსა, რამეთუ
ვითარცა ყუავილი თივისაჲ, ეგრეთ წარჴდეს.

<span id="an_1:11">1:11</span> რამეთუ აღმოჰჴდა მზე სიცხითა თჳსითა და
განაჴმო თივაჲ იგი და ყუავილი მისი დასცჳვა და შუენიერებაჲ პირისა მისისაჲ
წარჴდა. ეგრეცა მდიდარი სლვასა შინა თჳსსა დაჭნეს.

<span id="an_1:12">1:12</span> ნეტარ არს კაცი იგი, რომელმან დაუთმოს
განსაცდელსა, რამეთუ გამოცდილ იქნეს და მოიღოს გჳრგჳნი ცხორებისაჲ, რომელი
აღუთქუა ღმერთმან მოყუარეთა მისთა.

<span id="an_1:13">1:13</span> ნუვინ განცდილთაგანი იტყჳნ, ვითარმედ
ღმრთისა მიერ განვიცადები, რამეთუ ღმერთი გამოუცდელ არს ბოროტისა და არავის
განსცდის იგი.

<span id="an_1:14">1:14</span> არამედ კაცად-კაცადი განიცადების თჳსისაგან
გულის თქუმისა, მიიზიდვის და სცთების.

<span id="an_1:15">1:15</span> მერმე გულის თქუმაჲ იგი მიუდგის და შვის
ცოდვაჲ და ცოდვაჲ იგი სრულ-იქმნის და შვის სიკუდილი.

<span id="an_1:16">1:16</span> ნუ სცთებით, ძმანო ჩემნო საყუარელნო!

<span id="an_1:17">1:17</span> ყოველივე მოცემული კეთილი და ყოველივე ნიჭი
სრული ზეგარდამო არს გარდამოსრულ მამისაგან ნათლისა, რომლისა თანა არა არს
ცვალება, გინა თუ ქცევისა აჩრდილ.

<span id="an_1:18">1:18</span> მას უნდა და გუშვნა ჩუენ სიტყჳთა
ჭეშმარიტებისაჲთა, რაჲთა ვიყვნეთ ჩუენ დასაბამ რაჲმე მისთა დაბადებულთა. აწ
უკუე, ძმანო ჩემნო საყუარელნო, იყავნ ყოველი კაცი მოსწრაფე სმენად, მძიმე
სიტყუად, მძიმე გულის წყრომად.

<span id="an_1:18">1:18</span> მას უნდა და გუშვნა ჩუენ სიტყჳთა
ჭეშმარიტებისაჲთა, რაჲთა ვიყვნეთ ჩუენ დასაბამ რაჲმე მისთა დაბადებულთა. აწ
უკუე, ძმანო ჩემნო საყუარელნო, იყავნ ყოველი კაცი მოსწრაფე სმენად, მძიმე
სიტყუად, მძიმე გულის წყრომად.

<span id="an_1:20">1:20</span> რამეთუ გულის წყრომამან კაცისამან სიმართლე
ღმრთისაჲ არა ქმნის.

<span id="an_1:21">1:21</span> ამისთჳს განიშორეთ ყოველი მწიკულევანებაჲ
და მეტი იგი სიბოროტისაჲ, სიმშჳდით შეიწყნარეთ ნერგი იგი სიტყჳსაჲ, რომელი
შემძლებელ არს ცხორებად სულთა თქუენთა.

<span id="an_1:22">1:22</span> და იყვენით მყოფელ სიტყჳსა მის და ნუ ხოლო
მსმენელ და შეურაცხის-მყოფელ თავთა თჳსთა.

<span id="an_1:23">1:23</span> უკუეთუ ვინმე მსმენელ ხოლო იყოს სიტყჳსა
მის და მყოფელ არა, ესევითარი იგი მსგავს არს კაცისა მის, რომელმან
განიცადის პირი ქმნულებისა თჳსისაჲ სარკითა.

<span id="an_1:24">1:24</span> რამეთუ განიცადა თავი თჳსი და წარვიდა და
მეყსეულად დაავიწყდა, რაბამ-რაჲ იყო.

<span id="an_1:25">1:25</span> ხოლო რომელმან მიჰხედა სჯულსა მას სრულსა
აზნაურებისასა და დაადგრა მას, ესე არა მსმენელ და დამვიწყებელ იქმნა,
არამედ მყოფელ საქმისა; ესე ნეტარ იყოს ქმნითა მით მისითა.

<span id="an_1:26">1:26</span> რომელსა ეგონოს თქუენ შორის, ვითარმედ
ღმრთის მსახურ არს და არა აღჳრ-ასხმიდეს ენასა თჳსსა, არამედ აცთუნებდეს
გულსა თჳსსა, ამისი ამაო არს მსახურებაჲ.

<span id="an_1:27">1:27</span> მსახურებაჲ წმიდაჲ და შეუგინებელი ღმრთისა
მიერ და მამისა ესე არს: მიხედვაჲ ობოლთა და ქურივთაჲ ჭირსა შინა მათსა და
შეუგინებელად დაცვაჲ თავისა თჳსისაჲ ამის სოფლისაგან.

<span id="an_2:1">2:1</span> ძმანო ჩემნო, ნუ თუალთ-ღებით, გაქუნ
სარწმუნოებაჲ უფლისა ჩუენისა იესუ ქრისტეს დიდებისაჲ.

<span id="an_2:2">2:2</span> რამეთუ, უკუეთუ შევიდეს შესაკრებელსა თქუენსა
კაცი ოქროჲსა ბეჭდითა და სამოსლითა ბრწყინვალითა, და შევიდეს გლახაკიცა
სამოსელითა შეურაცხითა.

<span id="an_2:3">2:3</span> და მიჰხედოთ მას, რომელსა-იგი ემოსოს
სამოსელი ბრწყინვალე, და ჰრქუათ მას: შენ დაჯედ აქა კეთილად, და გლახაკსა
მას ჰრქუათ: შენ დეგ იქი, ანუ დაჯედ აქა ქუეშე ფერჴთა ჩუენთა

<span id="an_2:4">2:4</span> და არა განიკითხენით თავთა შორის თქუენთა და
იქმენით მსაჯულ გულის სიტყჳთა ბოროტითა?

<span id="an_2:5">2:5</span> ისმინეთ, ძმანო ჩემნო საყუარელნო: ანუ არა
გლახაკნი სოფლისანი გამოირჩინაა ღმერთმან, მდიდარნი სარწმუნოებითა და
მკჳდრნი სასუფეველისანი, რომელ-იგი აღუთქუა მოყუარეთა თჳსთა?

<span id="an_2:6">2:6</span> ხოლო თქუენ შეურაცხ-ჰყავთ გლახაკი. ანუ არა
მდიდარნი გმძლავრობენა თქუენ, და იგინივე მიგიზიდვენ თქუენ სამსჯავროდ?

<span id="an_2:7">2:7</span> არა იგინი ჰგმობენა კეთილსა მას სახელსა,
რომელი წოდებულ არს თქუენ ზედა?

<span id="an_2:8">2:8</span> უკუეთუ სჯულსა აღსრულებთ სამეუფოსა, მსგავსად
წერილისა: შეიყუარო მოყუასი შენი, ვითარცა თავი თჳსი, კეთილად ჰყოფთ.

<span id="an_2:9">2:9</span> ხოლო უკუეთუ თუალთ-ღებით, ცოდვასა იქმთ და
იმხილებით სჯულისაგან, ვითარცა გარდამავალნი სჯულისანი.

<span id="an_2:10">2:10</span> რამეთუ რომელმან ყოველი სჯული დაიმარხოს და
სცთეს ერთითა, იქმნა იგი ყოვლისავე თანამდებ.

<span id="an_2:11">2:11</span> რამეთუ რომელმან-იგი თქუა: ნუ იმრუშებ,
მანვე თქუა: ნუ კაც-ჰკლავ, ხოლო უკუეთუ იმრუშო არა და კაც-ჰკლა, იქმენ
გარდამავალ სჯულისა.

<span id="an_2:12">2:12</span> ესრეთ იტყოდეთ და ესრეთ ჰყოფდით, ვითარცა
სჯულისაგან აზნაურებისა განსჯადნი.

<span id="an_2:13">2:13</span> რამეთუ სასჯელი უწყალო არს მათთჳს, რომელთა
არა ყონ წყალობაჲ; იქადოდეთ წყალობასა სასჯელისასა.

<span id="an_2:14">2:14</span> რაჲ სარგებელ არს, ძმანო ჩემნო, უკუეთუ
ვინმე თქუას: სარწმუნოებაჲ მაქუს, და საქმენი არა ჰქონდინ? ნუ ძალ-უც-მეა
სარწმუნოებასა ხოლო ცხოვნებად მისა?

<span id="an_2:15">2:15</span> რამეთუ უკუეთუ ძმაჲ გინა დაჲ შიშველ იყვნენ
და ნაკლულევან მდღევრისა საზრდელისა

<span id="an_2:16">2:16</span> და ჰრქუას ვინმე თქუენგანმან: წარვედით
მშჳდობით, განძეღით და განტეფით, და არა სცეთ მათ საჴმარი ჴორცითაჲ, რაჲ
სარგებელ არს?

<span id="an_2:17">2:17</span> ეგრეთცა სარწმუნოებაჲ, უკუეთუ არა აქუნდენ
საქმენი, მკუდარ არს იგი ხოლო.

<span id="an_2:18">2:18</span> არამედ თქუას თუ ვინმე: შენ სარწმუნოებაჲ
გაქუს და მე საქმენი მქონან, მიჩუენე მე სარწმუნოებაჲ შენი საქმეთაგან
შენთა, და მე გიჩუნო შენ საქმეთაგან ჩემთა სარწმუნოებაჲ ჩემი.

<span id="an_2:19">2:19</span> შენ გრწამს, ვითარმედ ერთ არს ღმერთი,
კეთილად იქმ, და ეშმაკთაცა ჰრწამს და ძრწიან მისგან.

<span id="an_2:20">2:20</span> გნებავსა ცნობად, ჵ კაცო ამაოო, რამეთუ
სარწმუნოებაჲ თჳნიერ საქმეთასა მკუდარ არს?

<span id="an_2:21">2:21</span> აბრაჰამ, მამაჲ ჩუენი, არა საქმეთაგან
განმართლდაა, რამეთუ შეწირა ისაკი, ძე თჳსი, საკურთხეველსა ზედა?

<span id="an_2:22">2:22</span> ჰხედავა, რამეთუ სარწმუნოებაჲ შეეწია
საქმეთა მისთა და საქმეთაგან სარწმუნოებაჲ იგი სრულ იქმნა?

<span id="an_2:23">2:23</span> და აღესრულა წერილი იგი, რომელი იტყჳს:
ჰრწმენა აბრაჰამს ღმერთი, და შეერაცხა მას სიმართლედ, და მეგობარ ღმრთისა
იწოდა.

<span id="an_2:24">2:24</span> ჰხედავთა, რამეთუ საქმეთაგან განმართლდების
კაცი და არა სარწმუნოებისაგან ხოლო?
