### Georgian Bible: John

<span id="an_1:1">1:1</span> პირველითგან იყო სიტყუაჲ, და სიტყუაჲ იგი იყო
ღმრთისა თანა, და ღმერთი იყო სიტყუაჲ იგი.

<span id="an_1:2">1:2</span> ესე იყო პირველითგან ღმრთისა თანა.

<span id="an_1:3">1:3</span> ყოველივე მის მიერ შეიქმნა, და თჳნიერ მისა
არცა ერთი რაჲ იქმნა, რაოდენი-რაჲ იქმნა.

<span id="an_1:4">1:4</span> მის თანა ცხორებაჲ იყო და ცხორებაჲ იგი იყო
ნათელ კაცთა.

<span id="an_1:5">1:5</span> და ნათელი იგი ბნელსა შინა ჩანს, და ბნელი
იგი მას ვერ ეწია.

<span id="an_1:6">1:6</span> იყო კაცი მოვილინებული ღმრთისა მიერ, და
სახელი მისი იოვანე.

<span id="an_1:7">1:7</span> ესე მოვიდა მოწამედ, რაჲთა წამოს ნათლისა
მისთჳს, რაჲთა ყოველთა ჰრწმენეს მისგან.

<span id="an_1:8">1:8</span> არა თუ იგი იყო ნათელი, არამედ რაჲთა წამოს
ნათლისა მისთჳს.

<span id="an_1:9">1:9</span> იყო ნათელი ჭეშმარიტი, რომელი განანათლებს
ყოველსა კაცსა, მომავალსა სოფლად.

<span id="an_1:10">1:10</span> სოფელსა შინა იყო, და სოფელი მის მიერ
შეიქმნა, და სოფელმან იგი ვერ იცნა.

<span id="an_1:11">1:11</span> თჳსთა თანა მოვიდა, და თჳსთა იგი არა
შეიწყნარეს.

<span id="an_1:12">1:12</span> ხოლო რაოდენთა-იგი შეიწყნარეს, მოსცა მათ
ჴელმწიფებაჲ შვილ ღმრთისა ყოფად, რომელთა ჰრწამს სახელი მისი,

<span id="an_1:13">1:13</span> რომელნი არა სისხლთაგან, არცა ნებითა
ჴორცთაჲთა, არცა ნებითა მამაკაცისაჲთა, არამედ ღმრთისაგან იშვნეს.

<span id="an_1:14">1:14</span> და სიტყუაჲ იგი ჴორციელ იქმნა და დაემკჳდრა
ჩუენ შორის, და ვიხილეთ დიდებაჲ მისი, დიდებაჲ ვითარცა მხოლოდ შობილისაჲ
მამისა მიერ, სავსე მადლითა და ჭეშმარიტებითა.

<span id="an_1:15">1:15</span> იოვანე წამა მისთჳს, ღაღატ-ყო და თქუა: ესე
არს, რომლისათჳს-იგი ვთქუ: რომელი ჩემსა შემდგომად მოვალს, ჩემსა უწინარეს
იყო, რამეთუ პირველ ჩემსა არს.

<span id="an_1:16">1:16</span> და სავსებისაგან მისისა ჩუენ ყოველთა
მოვიღეთ, და მადლი მადლისა წილ.

<span id="an_1:17">1:17</span> რამეთუ სჯული მოსესგან მოეცა, ხოლო მადლი
და ჭეშმარიტებაჲ ქრისტე იესუჲს მიერ იქმნა.

<span id="an_1:18">1:18</span> ღმერთი არავინ სადა იხილა; მხოლოდ-შობილმან
ძემან, რომელი იყო წიაღთა მამისათა, მან გამოთქუა.

<span id="an_1:19">1:19</span> და ესე არს წამებაჲ იოვანესი, ოდეს
მიავლინნეს მისა ჰურიათა იერუსალჱმით მღდელნი და ლევიტელნი, რაჲთა ჰკითხონ
მას, ვითარმედ: შენ ვინ ხარ?

<span id="an_1:20">1:20</span> და აღიარა და არა უვარ-ყო; აღიარა,
ვითარმედ: მე არა ვარ ქრისტე.

<span id="an_1:21">1:21</span> და მათ ჰკითხეს მას: და უკუე ვინ ხარ შენ?
ელია ხარა? და მან თქუა: არა ვარ; წინაწარმეტყუელნი იგი ხარა შენ? და
მიუგო: არა.

<span id="an_1:22">1:22</span> და ჰრქუეს: გუარქუ ჩუენ, ვინ ხარ შენ,
რაჲთა სიტყუაჲ მივართუათ მომავლინებელთა ჩუენთა, რასა იტყჳ შენთჳს?

<span id="an_1:23">1:23</span> და თქუა: მე - ჴმაჲ ღაღადებისაჲ უდაბნოსა:
განჰმზადენით გზანი უფლისანი, ვითარცა თქუა ესაია წინაწარმეტყუელმან.

<span id="an_1:24">1:24</span> და მივლინებულნი იგი იყვნეს
ფარისეველთაგანნი.

<span id="an_1:25">1:25</span> ჰკითხეს მას და ჰრქუეს: და რაჲსათჳს
ნათელ-სცემ, უკუეთუ არა ხარ შენ ქრისტე, არცა ელია, არცა წინაწარმეტყუელი
იგი?

<span id="an_1:26">1:26</span> მიუგო იოვანე და ჰრქუა მათ: მე ნათელ-ვსცემ
წყლითა, ხოლო შორის თქუენსა დგას, რომელი თქუენ არა იცით,

<span id="an_1:27">1:27</span> რომელი ჩემსა შემდგომად მოსლვად არს,
რომელი პირველ ჩემსა იყო, რომლისა არა ღირს ვარ მე, რაჲთა განვჰჴსნნე
საბელნი ჴამლთა მისთანი.

<span id="an_1:28">1:28</span> ესე ბეთაბრას იყო, წიაღ-იორდანესა,
სადა-იგი იყო იოვანე და ნეთელ-სცემდა.

<span id="an_1:29">1:29</span> ხვალისაგან იხილა იოვანე იესუ, მომავალი
მისა, და თქუა: აჰა ტარიგი ღმრთისაჲ, რომელმან აიხუნეს ცოდვანი სრფლისანი.

<span id="an_1:30">1:30</span> ესე არს, რომლისათჳს-იგი ვთქუ: შემდგომად
ჩემსა მოვალს კაცი, რომელი პირველ ჩემსა იყო, რამეთუ უპირატეს ჩემსა არს.

<span id="an_1:31">1:31</span> და მე არა ვიცოდე იგი, არამედ რაჲთა
გამოეცხადოს ისრაჱლსა, ამისთჳს მოვედ მე წყლითა ნათლის-ცემად.

<span id="an_1:32">1:32</span> და წამა იოვანე და თქუა, რამეთუ: ვიხილე
სული ღმრთისაჲ ვითარცა ტრედი გარდამომავალი ზეცით, და დაადგრა მას ზედა.

<span id="an_1:33">1:33</span> და მე არა ვიცოდე იგი, არამედ რომელმან
მომავლინა მე ნათლის-ცემად წყლითა, მან მრქუა მე: რომელსა ზედა იხილო სული
გარდამომავალი და დადგრომილი მის ზედა, იგი არს, რომელმან ნათელ-გცეს
სულითა წმიდითა.

<span id="an_1:34">1:34</span> და მე ვიხილე და ვწამე, რამეთუ ესე არს ძე
ღმრთისაჲ.

<span id="an_1:35">1:35</span> ხვალისა დღე დგა კუალად იოვანე და მოწაფეთა
მისთაგანნი ორნი.

<span id="an_1:36">1:36</span> და მიჰხედა იესუს, ვიდოდა რაჲ იგი, და
თქუა: აჰა ტარიგი ღმრთისაჲ.

<span id="an_1:37">1:37</span> და ესმა მისი ორთა მათ მოწაფეთა, იტყოდა
რაჲ ამას, და მისდევდეს იესუს.

<span id="an_1:38">1:38</span> მოექცა იესუ და იხილნა იგინი,
მი-რაჲ-სდევდეს მას, და ჰრქუა მათ: რასა ეძიებთ? ხოლო მათ ჰრქუეს მას: რაბი
(რომელი ითარგმანების ?მოძღუარ?), სადა იყოფი?

<span id="an_1:39">1:39</span> ჰრქუა მათ: მოვედით და იხილეთ. და მივიდეს
და იხილეს, სადა იყოფოდა, და მის თანა დაადგრეს მას დღესა შინა. ჟამი იყო
ვითარ მეათე.

<span id="an_1:40">1:40</span> იყო ანდრეა, ძმაჲ სიმონ-პეტრესი, ერთი
ორთაგანი, რომელთა ესმა იოვანესგან, და მისდევდეს მას.

<span id="an_1:41">1:41</span> პოვა ამან პირველად ძმაჲ თჳსი სიმონ და
ჰრქუა მას: ვპოეთ მესია, რომელ არს თარგმანებით: ?ქრისტე?.

<span id="an_1:42">1:42</span> ამან მოიყვანა იგი იესუჲსა. მიხედა მას
იესუ და ჰრქუა: შენ ხარ სიმონ, ძე იონაჲსი, შენ გეწადოს კეფა, რომელი
ითარგმანების ?კლდე?.

<span id="an_1:43">1:43</span> ხვალისაგან უნდა იესუს განსლვად გალილეად
და პოვა ფილიპე და ჰრქუა მას იესუ: მომდევდი მე.

<span id="an_1:44">1:44</span> ხოლო იყო ფილიპე ბეთსაიდაჲთ, ქალაქისაგან
ანდრეაჲსა და პეტრესა.

<span id="an_1:45">1:45</span> პოვა ფილიპე ნათანაელ და ჰრქუა მას:
რომელი-იგი დაწერა მოსე სჯულსა და წინაწარმეტყუელთა, ვპოეთ: იესუ, ძე
იოსებისი, ნაზარეთით.

<span id="an_1:46">1:46</span> და ჰრქუა მას ნათანაელ: ნაზარეთით
შესაძლებელ არსა კეთილისა რაჲსამე ყოფად? ჰრქუა მას ფილიპე: მოვედ და
იხილე.

<span id="an_1:47">1:47</span> იხილა რაჲ იესუ ნათანაელ, მომავალი მისა,
და თქუა მისთჳს: აჰა ჭეშმარიტად ისრაიტელი, რომლისა თანა ზაკუვაჲ არა არს.

<span id="an_1:48">1:48</span> ჰრქუა მას ნათანაელ: ვინაჲ მიცი მე? მიუგო
იესუ და ჰრქუა მას: პირველ ფილიპეს ჴმობადმდე შენდა, იყავ რაჲ ლეღუსა
ქუეშე, გიხილე შენ.

<span id="an_1:49">1:49</span> მიუგო მას ნათანაელ და ჰრქუა: რაბი, შენ
ხარ ძე ღმრთისაჲ, შენ ხარ მეუფჱ ისრაჱლისაჲ.

<span id="an_1:50">1:50</span> მიუგო იესუ და ჰრქუა მას, რამეთუ: გარქუ
შენ, ვითარმედ: გიხილე შენ ლეღუსა ქუეშე, და გრწამს; უფროჲსი ამათსა იხილო.

<span id="an_1:51">1:51</span> და ჰრქუა მათ: ამენ, ამენ გეტყჳ თქუენ:
ამიერითგან იხილნეთ ცანი განხუმულნი და ანგელოზნი ღმრთისანი, აღმავალნი და
გარდამომავალნი ძესა ზედა კაცისასა.

გალილეაჲსასა, და მუნ იყო დედაჲ იესუჲსი.

<span id="an_2:2">2:2</span> მიჰხადეს იესუს და მოწაფეთა მისთა ქორწილსა
მას.

<span id="an_2:3">2:3</span> და ვითარ მოაკლდა ღჳნოჲ, ჰრქუა იესუს დედამან
მისმან: ღჳნოჲ არა აქუს.

<span id="an_2:4">2:4</span> ჰრქუა მას იესუ: რაჲ არს ჩემდა და შენდა,

<span id="an_2:5">2:5</span> ჰრქუა დედამან მისმან მსახურთა მათ,
ვითარმედ: რაჲცა გრქუას თქუენ, ყავთ!

<span id="an_2:6">2:6</span> იყვნეს მუნ სარწყულნი ქვისანი ექუსნი,
მდგომარენი, მსგავსად განწმენდისა მის ჰურიათაჲსა. რომელთა შთაისხიან
საწყაული ორ-ორი გინა სამ-სამი.

<span id="an_2:7">2:7</span> და ჰრქუა მათ იესუ: აღავსენით სარწყულნი ეგე
წყლითა! და მათ აღავსნეს იგინი პირისპირ.

<span id="an_2:8">2:8</span> და ჰრქუა მათ: აღმოავსეთ აწ და მოართჳთ
პურისა უფალსა! ხოლო მათ მოართუეს.

<span id="an_2:9">2:9</span> და ვითარცა გემოჲ იხილა პურისა უფალმან წყალი
იგი, რომელი ღჳნო იქმნა, და არა უწყოდა, ვინაჲ იყო, გარნა მსახურთა მათ
