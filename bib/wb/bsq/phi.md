### Basque Bible: Philippians

<span id="an_1:1">1:1</span> PAVLEC eta Timotheoc Iesus Christen
cerbitzariéc Iesus Christ Iaunean Philippesen diraden saindu guciey
Ipizpicuequin batean eta Diacrequin,

<span id="an_1:2">1:2</span> Gratia dela çuequin eta baquea Iainco
Aitaganic, eta Iesus Christ Iaunaganic.

<span id="an_1:3">1:3</span> Esquerrac emaiten drauzquiot neure
Iaincoari çueçazco orhoitzapen handirequin,

<span id="an_1:4">1:4</span> (Bethiere neure oratione gucietan çuen
guciongatic bozcariorequin othoitze eguiten dudala)

<span id="an_1:5">1:5</span> Ceren Euangelioren communionera ethorri
içan çareten, lehen egunetic oraindrano:

<span id="an_1:6">1:6</span> Huneçaz seguratzen naicela, ecen obra ona
çuetan hassi vkan duenac acabaturen duela Iesus Christen egunerano:

<span id="an_1:7">1:7</span> Nola raçoina baita nic haur estima deçadan
çueçaz gucioz, ceren bihotzean baitzadutzatet, eta ene estecaduretan,
eta defensionean eta Euangelioaren confirmationean çuec gucioc enequin
baitzarete participant ene gratian.

<span id="an_1:8">1:8</span> Ecen Iaincoa dut testimonio cein cinez
desiratzen çaituztedan çuec gucioc Iesus Christen affectione cordialez:

<span id="an_1:9">1:9</span> Eta hunez othoiztez nago, çuen charitatea
oraino guehiago eta guehiago abunda dadin eçagutzerequin eta iugemendu
gucirequin:

<span id="an_1:10">1:10</span> Gauça contrarioac discerni ditzaçuençát:
pur eta trebucu gabe çaretençát Iesus Christen egunerano:

<span id="an_1:11">1:11</span> Iesus Christez Iaincoaren gloriatan eta
laudoriotan diraden fructuz betheac çaretelaric.

<span id="an_1:12">1:12</span> Eta nahi dut daquiçuen, anayeác, ecen
niri heldu çaizquidan gauçác Euangelioaren auançamendutarago ethorri
içan diradela:

<span id="an_1:13">1:13</span> Hambat non ene estecadurác famatu içan
baitirade Christ Iaunean Pretorio gucian eta berce leku gucietan:

<span id="an_1:14">1:14</span> Eta anayetaric anhitz gure Iaunean ene
estecaduréz asseguraturic, ausartquiago hitzaz minçatzera hauçu
baitirade.

<span id="an_1:15">1:15</span> Eta batzuc behinçát inuidiaz eta
contentionez, eta bercéc vorondate onez-ere Christ predicatzen duté:

<span id="an_1:16">1:16</span> Batzuc, diot, contentionez Christ
denuntiatzen duté ez purqui, ene estecadurác afflictionez emendatu
vstez:

<span id="an_1:17">1:17</span> Baina bercéc charitatez, daquitelaric
ecen Euangelioaren defensionetan eçarri içan naicela.

<span id="an_1:18">1:18</span> Cer bada? nola-ere baita, bada fictionez,
bada eguiaz, Christ predicatzen da: eta hunetan alegueratzen naiz, eta
alegueraturen-ere.

<span id="an_1:19">1:19</span> Ceren baitaquit, ecen haur ethorriren
çaitadala saluamendutara çuen orationéz, eta Iesus Cristen Spirituaren
aiutáz,

<span id="an_1:20">1:20</span> Ene iguriquite fermuaren eta sperançaren
araura, ecen deusetan eznaicela confus içanen, aitzitic libertate
gucirequin bethiere beçala, orain-ere magnificaturen dela Christ ene
gorputzean, vicitzez bada hiltzez bada.

<span id="an_1:21">1:21</span> Ecen niri Christ irabaizte çait vicitzera
eta hiltzera.

<span id="an_1:22">1:22</span> Ala haraguian vicitzera probetchu
dudanez, eta cer hauta deçaquedan, eztaquit.

<span id="an_1:23">1:23</span> Ecen hertsen naiz alde bietaric, desir
dudalaric partitzera eta Christequin içatera: ecen haur vnguiz hobeago
dut:

<span id="an_1:24">1:24</span> Baina haraguian egon nadin necessarioago
da çuengatic.

<span id="an_1:25">1:25</span> Eta haur seguratuqui badaquit, ecen
egonen naicela eta perseueraturen dudala çuequin gucioquin çuen
auançamendutan, eta çuen fedearen bozcariotan.

<span id="an_1:26">1:26</span> Abunda dadinçát çuen gloriationea Iesus
Christean niçaz, ene berriz çuetaraco ethorteaz.

<span id="an_1:27">1:27</span> Solament Christen Euangelioari dagocan
beçala conuersa eçaçue, bada ethor nadin eta ikus çaitzatedan, bada
absent naicén, adi deçadan çuen eguitecoez, ecen perseueratzen duçuela
spiritu batetan, gogo batez elkarrequin Euangelioaren fedeaz combatitzen
çaretelaric, eta contrastez deusetan icitzen etzaretelaric:

<span id="an_1:28">1:28</span> Cein baita hayén perditionetaco seignale,
eta çuen saluamendutaco, eta haur Iaincoaganic.

<span id="an_1:29">1:29</span> Ecen çuey eman içan çaiçue Christgatic,
ez solament hura baithan sinhestea, baina harengatic suffritzea-ere:

<span id="an_1:30">1:30</span> Nitan ikussi duçuen, eta orain ençuten
duçuen combat bera duçuela.

<span id="an_2:1">2:1</span> Bada baldin consolationeric batre bada
Christean, baldin charitatezco solageamenduric batre, baldin spirituzco
communioneric batre, baldin affectione cordialic eta misericordiaric
batre bada,

<span id="an_2:2">2:2</span> Compli eçaçue ene bozcarioa, sendimendu
ber-bat duçuelaric, charitate ber-bat duçuelaric, gogo ber-batetaco eta
consentimendu ber-batetaco çaretelaric.

<span id="an_2:3">2:3</span> Deus eztadila contentionez edo vana gloriaz
eguin, baina bihotzetico humilitatez batac bercea bere buruä baino
excellentago estima beça.

<span id="an_2:4">2:4</span> Etzaudetela batbedera cein çuen
commoditatetara behá, baina bercerenetara-ere.

<span id="an_2:5">2:5</span> Bada biz çuetan Iesus Christ baithan-ere
içan den affectionea.

<span id="an_2:6">2:6</span> Ceinec Iaincoaren formán celaric ezpaitu
estimatu harrapatze Iaincoaren bardin içatea:

<span id="an_2:7">2:7</span> Badaric-ere bere buruä ezdeustu vkan du,
cerbitzari forma harturic, guiçonén irudico eguin içanic, eta formaz
eriden içanic guiçon beçala:

<span id="an_2:8">2:8</span> Bere buruä beheratu vkan du obedient eguin
içanic heriorano, are herio crutzecorano.

<span id="an_2:9">2:9</span> Halacotz Iaincoac-ere hura subiranoqui
goratu vkan du, eta icen-bat eman vkan drauca icen gucién gaineco denic:

<span id="an_2:10">2:10</span> Iesusen icenean ceruètacoén eta lurrecoén
eta luppecoén belhaun gucia gur dadinçát:

<span id="an_2:11">2:11</span> Eta mihi guciac confessa deçan ecen Iaun
dela Iesus Christ, Iainco Aitaren glorián.

<span id="an_2:12">2:12</span> Hunegatic, ene maiteác, bethiere obeditu
vkan duçuen beçala, ez ene presentián beçala solament, baina orain
vnguiz guehiago ene absentián, beldurrequin eta ikararequin emplega
çaitezte çuen saluamenduan.

<span id="an_2:13">2:13</span> Ecen Iaincoa da eguiten duena çuetan eta
nahi vkaitea eta eguitea, bere placer onaren araura.

<span id="an_2:14">2:14</span> Gauça guciac eguin itzaçue murmuratzeric
eta questioneric gabe:

<span id="an_2:15">2:15</span> Çaretençát reprotchu baguetaco eta
simple, Iaincoaren haour irreprehensible, natione bihurriaren eta
gaichtoaren artean, ceinén artean argui eguiçue torchoéc beçala munduan,
vicitzetaco hitza aitzinera ekarten dutenén ançora:

<span id="an_2:16">2:16</span> Gloria ahal nadinçát Christen egunean,
ecen eztudala alfer laster eguin, ez alfer lan eguin:

<span id="an_2:17">2:17</span> Eta baldin çuen fedearen sacrificio eta
cerbitzu gainera sacrifica banadi-ere, aleguera naiz eta çuen gucion
onaz aleguera naiz.

<span id="an_2:18">2:18</span> Çuec-ere halaber çareten aleguera, eta
aleguera çaitezte ene onaz.

<span id="an_2:19">2:19</span> Eta sperança dut Iesus Iaunean, Timotheo
sarri igorriren drauçuedala, nic-ere gogo on dudançát çuen eguitecoac
eçaguturic.

<span id="an_2:20">2:20</span> Ecen eztut nehor hain gogo
bardinetacoric, eguiaz çuen eguitecoéz arrangura duqueenic.

<span id="an_2:21">2:21</span> Ecen guciac bere gaucey iarreiquiten
çaizté, ez Iesus Christeney.

<span id="an_2:22">2:22</span> Baina harçazco experientiá eçagutzen
duçue, nola, semeac aitá cerbitzatzen duen beçala, enequin cerbizatzen
ari içan den Euangelioan.

<span id="an_2:23">2:23</span> Hura bada sperança dut igorriren
drauçuedala, neure eguitecoey dreçu eman drauqueedan sarrienaz.

<span id="an_2:24">2:24</span> Eta seguratzen naiz Iaunean, ecen
neuror-ere sarri ethorriren naicela çuetara.

<span id="an_2:25">2:25</span> Baina necessario cela estimatu vkan dut
çuetara igortera Epaphrodite anayea eta aiutaçalea eta guerlaco neure
laguna, çuen Apostolu-ere dena, eta nic behar vkan dudanaren
administraçalea.

<span id="an_2:26">2:26</span> Ecen harc çuec gucioc desiratzen
centuzten, eta guciz desplacer çuen ceren eri içan cela aditu vkan
baitzenduten.

<span id="an_2:27">2:27</span> Eta segur eri içan da hiltzeco statuan:
baina Iaincoac misericordia eguin vkan drauca, eta ez hari solament
baina niri-ere, tristitia tristitiaren gainera eznuençát.

<span id="an_2:28">2:28</span> Bada, hambat affectione handizago hura
igorri vkan dut, hura ikussiric berriz aleguera çaiteztençát, eta nic
hambat tristitia gutiago dudançát.

<span id="an_2:29">2:29</span> Recebi eçaçue bada hura gure Iaunean
bozcario gucirequin: eta halaco diradenac estimatan eduquitzaçue.

<span id="an_2:30">2:30</span> Ecen Christen obragatic heriorano
hurbildu içan da, bere vicia abandonnaturic, suppli leçançát çuen ene
baitharaco cerbitzu peitua.

<span id="an_3:1">3:1</span> Gaineracoaz den becembatean, ene anayeác,
aleguera çaitezte Iaunean: Gauça berén çuey scribatzera niri etzait
herabe, eta çuen segurançá da.

<span id="an_3:2">3:2</span> Gogoa eyeçue orey, gogoa eyeçue obrero
gaichtoey, gogoa emoçue concisioneari.

<span id="an_3:3">3:3</span> Ecen gu gara Circoncisionea, Iaincoa
spirituz cerbitzatzen dugunoc, eta gloriatzen garenoc Iesus Christean
eta haraguian confidançaric eztugunoc:

<span id="an_3:4">3:4</span> Nic cerçaz haraguian-ere confidança dudan
badut-ere. Baldin cembeit bercec vste badu confidançaric duela
haraguian, nic guehiago.

<span id="an_3:5">3:5</span> Bainaiz çortzi garreneco egunean
circonciditua, bainaiz Israel arraçatic, Beniaminen leinutic, Hebraico
Hebraicoetaric, religionez Phariseu:

<span id="an_3:6">3:6</span> Zeloz den becembatean Eliçaren
persecutaçale: Leguean den iustitiaz den becembatean, estacuru gabe.

<span id="an_3:7">3:7</span> Baina irabacitan nadutzan gauçác, caltetaco
estimatu vkan ditut Christen amorecatic.

<span id="an_3:8">3:8</span> Bayeta segur gauça guciac calte diradela
estimatzén dut Iesus Christ neure Iaunaren eçagutzearen
excellentiagatic: ceinagatic gauça hauçaz gucioçaz neure buruä billuci
vkan baitut, eta gorotz beçala eduquiten ditut, Christ irabaz
deçadançát.

<span id="an_3:9">3:9</span> Eta eriden nadin hartan, ez neure iustitia
Legueaz dena dudalaric, baina Christen fedeaz dena, cein baita,
Iaincoaganic fedez den iustitiá:

<span id="an_3:10">3:10</span> Eta eçagut deçadançát, hura, eta haren
resurrectionearen verthutea, eta haren afflictionén participationea,
haren herioaren conforme eguin içanic:

<span id="an_3:11">3:11</span> Eya cerbait maneraz hel ahal naitenez
hilen resurrectionera:

<span id="an_3:12">3:12</span> Ez ia ardietsi dudalacotz, edo ia perfect
naicelacotz: baina banarreió, eya are ardiets ahal deçaquedanez, eta
causa hunegatic ardietsi içan naiz Iesus Christez.

<span id="an_3:13">3:13</span> Anayeác, niçaz den becembatean eztut
neure buruäz estimatzen oraino ardietsi vkan dudala.

<span id="an_3:14">3:14</span> Baina gauçabat eguiten dut guibeletic
diraden gauçác ahanzten ditudala, eta aitzinetic diraden gaucetara
auançatzen naicela, chedeari narrayó, Iaincoaren garaitico vocationearen
preçioari, Iesus Christ Iaunean.

<span id="an_3:15">3:15</span> Bada, perfect garen gucióc dugun
sendimendu haour: eta baldin cerbait bercela senditzen baduçue, hura-ere
Iaincoac reuelaturen drauçue.

<span id="an_3:16">3:16</span> Guciagatic-ere certara heldu içan
baicara, hartan regla ber-batez ebil gaitecen, eta garen sendimendu
ber-batetaco.

<span id="an_3:17">3:17</span> Anayeác, çareten gogo batez ene
imitaçale, eta consideraitzaçue hala dabiltzanac, nola gu baiquaituçue
exemplutan.

<span id="an_3:18">3:18</span> Ecen anhitz dirade dabiltzanac ceineçaz
anhitzetan erran vkan baitrauçuet, eta orain-ere nigarrez nagoela
erraiten, Christen crutzearen etsay diradela:

<span id="an_3:19">3:19</span> Ceinén fina baita perditionea, ceinén
Iaincoa, sabela, eta gloriá hayén confusionerequin, pensatzen dutelaric
lurreco gaucetan.

<span id="an_3:20">3:20</span> Baina gure conuersationea ceruètan da:
nondic Saluadorearen-ere beguira baicaude, cein baita Iesus Christ
Iauna,

<span id="an_3:21">3:21</span> Ceinec transformaturen baitu gure gorputz
ezdeusa, haren gorputz gloriosoaren conforme eguin dadinçát, gauça
guciac-ere bere suiet eguin ahal ditzaqueen verthutearen araura.

<span id="an_4:1">4:1</span> Halacotz, ene anaye maiteác, eta desiratuác
ene bozcarioa eta coroá, hunela egon çaitezte gure Iaunean, ene maiteác.

<span id="an_4:2">4:2</span> Euodiari othoiztez nagoca, eta Synticheri
othoiztez nagoca diraden affectione ber-batetaco Iaunean.

<span id="an_4:3">4:3</span> Eta hiri-ere othoitz eguiten drauat, neure
eguiazco laguná, aiuta ditzan emazte enequin batean bataillatu diradenac
Euangelioan, eta Clementequin eta ene berce aiutaçalequin, ceinén icenac
baitirade vicitzeco liburuän.

<span id="an_4:4">4:4</span> Aleguera çaitezte Iaunean bethiere: berriz
diot, aleguera çaitezte.

<span id="an_4:5">4:5</span> Çuen modestiá eçagut bedi guiçon guciéz:
Iauna hurbil da.

<span id="an_4:6">4:6</span> Eztuçuela deusez ansiaric: baina gauça
orotan othoitzaz eta supplicationez esquer emaiterequin çuen requestác
notifica bequizquió Iaincoari.

<span id="an_4:7">4:7</span> Eta Iaincoaren baque adimendu gucia
iragaiten duenac beguiraturen ditu çuen bihotzac eta çuen adimenduac
Iesus Christean.

<span id="an_4:8">4:8</span> Gaineracoaz, anayeác, gauça eguiazco
diraden guciac gauça honest guciac, iustoac, purac, amoriotacoac, icen
onetacoac, baldin edocein verthute bada eta edocein laudorio, gauça
hautan pensa eçaçue:

<span id="an_4:9">4:9</span> Cein ikassi-ere baitituçue, eta recebitu,
eta ençun, eta ikussi-ere nitan. Gauça hauc bada eguin itzaçue, eta
baquezco Iaincoa içanen da çuequin.

<span id="an_4:10">4:10</span> Alegueratu içan naiz Iaunean haguitz,
ceren ia azquenecotz arrapherdatu çareten çuen niçazco ansián: hartan
pensatzen-ere bacendutén, baina commoditateric etzindutén

<span id="an_4:11">4:11</span> Eznaiz minço deusen peitu içanez: ecen
nic ikassi dut erideiten naicenaren araura content içaten.

<span id="an_4:12">4:12</span> Eta badaquit behera içaten, badaquit
abundant içaten-ere: leku gucietan eta gauça gucietan ikassi dut asse
içaten eta gosse içaten, eta abundant içaten eta peitu içaten.

<span id="an_4:13">4:13</span> Gauça guciac ahal ditzaquet Christ
fortificatzen nauenaz:

<span id="an_4:14">4:14</span> Badaric-ere vngui eguin duçue ceren
elkarrequin ene afflictionera communicatu vkan baituçue.

<span id="an_4:15">4:15</span> Eta çuec-ere badaquiçue, Philippianoác,
ecen Euangelioaren predicatione hatsean, Macedoniaric partitu nincenean,
eceinere Eliçac etzarautala deus communicatu har eta emanezco
beharquian, çuec ceuroc baicen.

<span id="an_4:16">4:16</span> Ecen Thessalonican nincenean, behin eta
berriz, behar nuena igorri vkan drautaçue.

<span id="an_4:17">4:17</span> Ez emaitzeren galdez nagoelacotz: baina
galdeguiten dut fructua eçagut daquiçuen contura çuen abantailletan.

<span id="an_4:18">4:18</span> Bada recebitu vkan ditut gauça guciac,
eta abundantia dut: complitu içan naicela, diot, Epaphroditeganic
recebituric çueçaz igorriac, vrrin onetaco vssain anço, Iaincoaren
gogaraco eta placentiataco sacrificio.

<span id="an_4:19">4:19</span> Ene Iaincoac-ere supplituren du behar
vkanen duçuen gucia, bere abrastassunaren araura gloriarequin Iesus
Christean.

<span id="an_4:20">4:20</span> Bada gure Iainco eta Aitari dela gloria
secula seculacotz. Amen.

<span id="an_4:21">4:21</span> Salutaitzaçue saindu guciac Iesus
Christean Salutatzen çaituztez enequin diraden anayéc.

<span id="an_4:22">4:22</span> Salutatzen çaituztéz saindu guciéc, eta
principalqui Cesaren etchecoéc.

<span id="an_4:23">4:23</span> Iesus Christ gure Iaunaren gratiá dela
çuequin gucioquin. Amen.
