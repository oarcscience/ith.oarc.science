### Basque Bible: 2 Thessalonians

<span id="an_1:1">1:1</span> PAVLEC eta Siluanoc eta Timotheoc gure
Iainco Aitan eta Iesus Christ gure Iaunean Thessaloniceanoen den
Eliçari:

<span id="an_1:2">1:2</span> Gratia dela çuequin eta baquea gure Iainco
Aitaganic eta Iesus Christ Iaunaganic.

<span id="an_1:3">1:3</span> Esquerrac eman behar drauzquiogu Iaincoari
bethiere çuengatic, anayeác, raçoina den beçala, ceren haguitz handitzen
den çuen fedea eta ceren abundatzen den çuen guciotaric batbederaren
elkarganaco charitatea:

<span id="an_1:4">1:4</span> Hala non gueuror çuengatic gloriatzen
baicara Iaincoaren Elicetan, çuen persecutione eta iragaiten dituçuen
afflictione gucietaco patientiáz eta fedeaz:

<span id="an_1:5">1:5</span> Cein baitirade Iaincoaren iugemendu
iustoaren seignale, Iaincoaren resumaren digne eguin çaiteztençát,
ceinagatic suffritzen-ere baituçue:

<span id="an_1:6">1:6</span> Ecen gauça iustoa da Iaincoa baithan, çuec
affligitzen çaituzteney, afflictione renda diecén

<span id="an_1:7">1:7</span> Eta çuey affligitzen çaretenoy solageamendu
gurequin, aguer dadinean Iesus Iauna cerutic bere puissanciaco
Aingueruèquin,

<span id="an_1:8">1:8</span> Suzco garrequin, mendecatzen delaric
Iaincoa eçagutzen eztutenéz, eta Iesus Christ gure Iaunaren Euangelioari
behatzen etzaizquionéz:

<span id="an_1:9">1:9</span> Cein punituren baitirade punitione
eternalez, condemnaturic Iaunaren presentiáz eta haren botherearen
gloriáz:

<span id="an_1:10">1:10</span> Dathorrenean glorificatu dençát bere
sainduetan, eta miragarri eguin dadinçat sinhesten duten gucietan (ceren
gure çuec baitharaco testimoniagea sinhetsi içan baita) egun hartan:

<span id="an_1:11">1:11</span> Causa hunegatic-ere othoitz eguiten dugu
bethiere çuengatic, çuec vocationearen digne gure Iaincoac eguin
çaitzatençat, eta compli deçan bere ontassunaren placer on gucia, eta
fedearen obrá botheretsuqui:

<span id="an_1:12">1:12</span> Glorifica dadinçát Iesus Christ gure
Iaunaren icena çuetan, eta çuec hartan, gure Iaincoaren eta Iesus Christ
Iaunaren gratiaren araura.

<span id="an_2:1">2:1</span> Othoitz eguiten drauçuegu bada, anayeác,
Iesus Christ gure Iaunaren aduenimenduaz eta gure harenganaco
biltzarreaz.

<span id="an_2:2">2:2</span> Etzaitezten bertan adimenduz erauz, eta
etzaitezten trubla ez spirituz, ez hitzez, ez epistolaz, guçaz scribatua
baliz beçala, Christen eguna bertan baliz beçala.

<span id="an_2:3">2:3</span> Nehorc etzaitzatela seduci eceinere
maneraz: ecen egun hura ezta ethorriren non reuoltamendua lehen ethor
eztadin, eta manifesta eztadin bekatutaco guiçona, perditionezco semea,

<span id="an_2:4">2:4</span> Cein opposatzen eta alchatzen baita Iainco
erraiten eta adoratzen den guciren contra: hala non hura iainco beçala,
Iaincoaren templean iarten baita, bere buruä eracusten duela ecen iainco
dela.

<span id="an_2:5">2:5</span> Ala etzarete orhoit ecen oraino çuequin
nincela, gauça hauc erraiten nerauzquiçuela?

<span id="an_2:6">2:6</span> Eta orain cerc daducan badaquiçue, hura
manifesta dadinçat bere demborán.

<span id="an_2:7">2:7</span> Ecen ia eguiten da iniquitatezco mysterioa:
solament orain daducanac, eduquiren du ken daiteno artetic.

<span id="an_2:8">2:8</span> Eta orduan manifestaturen da gaichtoa, cein
Iaunac deseguinen baitu bere ahoco Spirituaz, eta bere aduenimenduco
claretateaz abolituren.

<span id="an_2:9">2:9</span> Hura diot ceinen ethortea baita Satanen
operationearen araura, puissança gucirequin eta signorequin eta
gueçurrezco miraculurequin:

<span id="an_2:10">2:10</span> Eta iniquitatezco abusione gucirequin
galtzen diradenetan, ceren eguiaren amorioa ezpaitute recebitu vkán
saluatu içateco.

<span id="an_2:11">2:11</span> Eta halacotz igorriren draue Iaincoac
abusionezco operationea, gueçurra sinhets deçatençát:

<span id="an_2:12">2:12</span> Condemna ditecençat eguia sinhetsi vkan
eztuten guciac, baina iniquitatean placer hartu vkan dutenac.

<span id="an_2:13">2:13</span> Baina guc, Iaunaz onhetsi anayeac, behar
drauzquiogu bethi esquerrac eman Iaincoari çueçaz, ceren hatseandanic
elegitu baitzaituzte Iaincoac saluamendutara Spirituaren
sanctificationez, eta eguiazco fedeaz:

<span id="an_2:14">2:14</span> Ceinetara deithu vkan baitzaituzte gure
Euangelioaz Iesus Christ gure Iaunaren gloriaren acquisitionetacotz.

<span id="an_2:15">2:15</span> Halacotz, anayeác, çaudete fermu eta
educaçue ikassi duçuen doctriná, bada gure hitzez bada epistolaz.

<span id="an_2:16">2:16</span> Bada Iesus Christ gure Iaun berac, eta
gure Iainco eta Aita onhetsi gaituenac, eta eman consolatione eternala
eta sperança ona gratiaz,

<span id="an_2:17">2:17</span> Consola ditzala çuen bihotzac, eta
confirma çaitzatela hitz eta obra on orotan.

<span id="an_3:1">3:1</span> Gaineracoaz, anayeác, othoitz eguiçue
guregatic, Iaunaren hitzac laster daguiançat, eta glorifica dadin çuec
baithan-ere beçala:

<span id="an_3:2">3:2</span> Eta gu deliura gaitecen gende desordenatu
eta gaichto hautaric: ecen fedea ezta gucién.

<span id="an_3:3">3:3</span> Baina fidel da Iauna, ceinec confirmaturen
baitzaituzte eta beguiraturen gaichtoaganic.

<span id="an_3:4">3:4</span> Asseguratzen-ere bagara Iaunean çueçaz,
ecen denuntiatzen drauzquiçuegun gauçác eguiten-ere badituçuela eta
eguinen.

<span id="an_3:5">3:5</span> Bada Iaunac chuchent ditzala çuen bihotzac
Iaincoaren amoriora, eta Christen beguira egoitera.

<span id="an_3:6">3:6</span> Anayeác, denuntiatzen drauçuegu halaber
Iesus Christ gure Iaunaren icenean, separa çaitezten anaye
desordenatuqui eta ez gureganic recebitu duen doctrinaren araura ebilten
den orotaric.

<span id="an_3:7">3:7</span> Ecen ceuroc badaquiçue nola imitatu behar
gaituçuen: ecen ezgara desordenatuqui maneyatu içan çuen artean.

<span id="an_3:8">3:8</span> Eta eztugu nehoren oguia dohainic ian:
baina trabaillurequin eta nequerequin gau eta egun languiten ari
guinadela, çuetaric nehoren phorogu ez guinadençát.

<span id="an_3:9">3:9</span> Ez bothere eztugunez, baina gure buruén
çuey exemplutan emaiteagatic, imita gaitzaçuençat.

<span id="an_3:10">3:10</span> Ecen çuequin guinadenean-ere haour
denuntiatzen guendrauçuen, ecen baldin cembeit trabaillatu nahi ezpada,
ian-ere ezteçan.

<span id="an_3:11">3:11</span> Ecen ençuten dugu çuen artean badiradela
batzu desordenatuqui dabiltzanic, deus ari eztiradelaric, baina
curiosqui vici diradelaric.

<span id="an_3:12">3:12</span> Bada halaco diradeney denuntiatzen
drauegu, eta othoitz eguiten Iesus Christ gure Iaunaz sossegurequin
trabaillatzen ari diradela bere oguia ian deçaten.

<span id="an_3:13">3:13</span> Eta çuec, anayeác, etzaiteztela enoya
vngui eguitez.

<span id="an_3:14">3:14</span> Eta baldin cembeitec gure hitza obeditzen
ezpadu, epistolaz hura nota eçaçue: eta ezteçaçuela conuersa harequin,
ahalque duençát:

<span id="an_3:15">3:15</span> Guciagatic-ere etsay beçala ezteçaçuela
eduqui, baina admonesta eçaçue anaye anço.

<span id="an_3:16">3:16</span> Bada Iaun baquezcoac bethiere diçuela
baquea manera gucian. Iauna dela çuequin gucioquin.

<span id="an_3:17">3:17</span> Salutationea, ene Paulen escuz, baita
seignale ene epistola gucietan: hunela scribatzen dut.

<span id="an_3:18">3:18</span> Iesus Christ gure Iaunaren gratia dela
çuequin.
