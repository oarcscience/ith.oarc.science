H5869  
עין  
עַיִן ‎ ‛ayin  
*ah‘-yin*  
Probably a primitive word; an *eye* (literally or figuratively); by
analogy a *fountain* (as the *eye* of the landscape): - affliction,
outward appearance, + before, + think best, colour, conceit, + be
content, countenance, + displease, eye (\[-brow\], \[-d\], -sight),
face, + favour, fountain, furrow \[from the margin\], X him, + humble,
knowledge, look, (+ well), X me, open (-ly), + (not) please, presence, +
regard, resemblance, sight, X thee, X them, + think, X us, well, X you
(-rselves).
