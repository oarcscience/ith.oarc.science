208.

THE MIRROR.

Experiments with the mirror (208-210).If the illuminated object is of the same size as the luminous body and as that in which the light is reflected, the amount of the reflected light will bear the same proportion to the intermediate light as this second light will bear to the first, if both bodies are smooth and white.
