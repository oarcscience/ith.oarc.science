### HYMN LXX

#### A benediction on cow and calf

<span class="margnote"><span id="an_0607001">1</span></span>As wine
associates with flesh, as dice attend the gaming-board,  
  As an enamoured man's desire is firmly set upon a dame,  
  So let thy heart and soul, O Cow, be firmly set upou thy calf.  
<span class="margnote"><span id="an_0607002">2</span></span>As the male
elephant pursues with eager step his female's track,  
  As an enamoured man's desire is firmly set upon a dame,  
  So let thy heart and soul, O Cow, be firmly set upon the calf.  
<span class="margnote"><span id="an_0607003">3</span></span>Close as the
felly and the spoke, fixt as the wheel-rim on the  
   nave,  
  As an enamoured man's desire is firmly set upon a dame,  
  So let thy heart and soul, O Cow, be firmly set upon thy calf.
