<span id="v.iii.i-p1"></span> *Retr.* ii. 23. “After I had written ‘on
the Good of Marriage,’ it was expected that I should write on Holy
Virginity; and I did not delay to do so: and that it is God’s gift, and
how great a gift, and with what humility to be guarded, so far as I was
able I set forth in one volume. This book begins,” &c.
