<span id="xiii.vii-p1"></span> <span class="c2"><span
id="xiii.vii-p1.1"></span>(3.) The Third Breviate.</span>

<span id="xiii.vii-p2"></span>III. “Again we must ask,” he says, “what
sin is,—natural? or accidental? If natural, it is not sin; if
accidental, it is separable; <span id="fr_1368"></span>[1369](#fn_1368.md)
and if it is separable, it can be avoided; and because it can be
avoided, man can be without that which can be avoided.” The answer to
this is, that sin is not natural; but nature (especially in that corrupt
state from which we have become by nature “children of wrath” <span
id="fr_1369"></span>[1370](#fn_1369) .md) has too little determination of
will to avoid sin, unless assisted and healed by God’s grace through
Jesus Christ our Lord.
