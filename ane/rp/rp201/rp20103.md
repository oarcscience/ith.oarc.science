### EQUIVALENTS OF THE HEBREW LETTERS IN THE TRANSLITERATION OF ASSYRIAN NAMES MENTIONED IN THESE VOLUMES.

<div align="center">

|     |        |                              |     |         |
|-----|--------|------------------------------|-----|---------|
| ‏א‎ | *a, ’* |                              | ‏ל‎ | *l*     |
| ‏ב‎ | *b*    |                              | ‏מ‎ | *m*     |
| ‏ג‎ | *g*    |                              | ‏נ‎ | *n*     |
| ‏ד‎ | *d*    |                              | ‏ס‎ | *’s, s* |
| ‏ה‎ | *h*    |                              | ‏ע‎ | *e*     |
| ‏ו‎ | *u, v* |                              | ‏פ‎ | *p*     |
| ‏ז‎ | *z*    |                              | ‏צ‎ | *ts*    |
| ‏ח‎ | *kh*   |                              | ‏ק‎ | *q*     |
| ‏ת‎ | *dh*   |                              | ‏ר‎ | *r*     |
| ‏י‎ | *i, y* |                              | ‏ש‎ | *s, sh* |
| ‏כ‎ | *k*    |                              | ‏ט‎ | *t*     |

</div>

 

N.B.—Those Assyriologists who transcribe ‏ש‎ by *sh* use *s* for ‏ס‎.
The Assyrian *e* represents a diphthong as well as ‏ע‎.
